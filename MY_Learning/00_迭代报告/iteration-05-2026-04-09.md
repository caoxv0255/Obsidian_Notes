### 2. 蒙特卡洛 Control Variates

**新建笔记**：`Year_2/01_Math_Advanced/1.5_随机过程/06_蒙特卡洛ControlVariates.md`

| 内容 | 覆盖情况 |
|------|---------|
| Control Variates 原理：以已知解析解函数作为控制变量 | ✅ |
| 最常用：BS 解析解作为 CV（量化金融行业标准）| ✅ |
| 最优 $\theta^*$ 推导：$\theta^* = \text{Cov}(X,Y)/\text{Var}(Y)$ | ✅ |
| 调整后估计量：$Z = C_{\text{MC}} - \theta(C_{\text{BS}} - E[C_{\text{BS}}])$ | ✅ |
| 方差缩减效果：$\text{Var}(Z^*)/\text{Var}(X) = 1 - \rho^2$ | ✅ |
| $\rho = 0.99$ 时相当于 **50 倍**路径数的精度 | ✅ |
| Python 实现：完整 CV 代码 + 标准误对比 | ✅ |
| CV + Antithetic Variates 叠加原理（互补、乘法效果）| ✅ |
| 理解程度自评 | ✅ **4/5** |

**关键成果**：
- CV 本质是利用 $\rho^2$ 决定方差缩减，$\rho \to 1$ 时方差可接近 0
- $\theta=1$ 时 CV 估计量是无偏的（使用 BS 精确期望），实际中可简化为 $\theta=1$
- Antithetic + CV 叠加是量化实操标准组合，两者从不同维度缩减方差
- $\theta$ 的数据驱动估计（从模拟数据）vs 理论最优值的权衡
