# Year 3 概览 - 双轨专业化与研究启动

> **角色定位**：研究者/教育者视角。大三是从"学习者"到"初级研究者"转变的关键一年——量化交易和计算神经科学两条主线都要从知识积累走向专题深化与原创问题。

## 🧭 与总纲的关系

- [[../00_双路径评估与重构规划]]：Year 3 的定位依据，负责把双轨基础转成研究切口与申请叙事。
- [[../00_双路径能力矩阵]]：Year 3 的执行坐标，研究切口、方法和产出都要回到这张矩阵里检查。
- Year 3 不再是“选一个方向”，而是把两条主线都推进到可研究、可复现、可叙述的状态。

## 🎯 Year 3 核心目标

1. **专业化深度**：量化交易和计算神经科学两条主线都进入专题深化
2. **科研启动**：参与至少1个研究项目（课程研究 or 独立研究），并明确一个主攻问题
3. **研究方法掌握**：能独立做文献综述、设计研究方案
4. **保持 GPA 竞争力**（3.8+，核心专业课）

## 📌 本年产出要求

- 量化侧：至少形成一个策略/因子/风险主题的可复用研究笔记或原型代码。
- 神经侧：至少形成一个神经建模或神经数据处理的 mini project。
- 方法侧：形成文献综述、研究设计和实验记录的统一模板。
- 叙事侧：写出可延续到 Year 4 的研究故事草稿。

## ⚠️ Year 3 与前两年的关键区别

- **Year 1-2**：知识的**宽度**积累（共享基础 + 双线入门）
- **Year 3**：知识的**深度**转化（双线专题深化 + 原创产出）
- 泛读时代结束，精读+批判时代开始
- 这一年的核心不是补课，而是把双轨学习转成研究证据链。

## 📅 推荐时间线

| 学期 | 重点 |
|------|------|
| 大三秋 | 锁定双轨研究切口 + 找研究组（RA/课程项目） |
| 大三冬 | 深耕两条主线核心知识 + 文献综述 + 研究设计 |
| 大三春 | 开始研究 + 论文写作准备 + 竞赛（可选） |
| 暑期 | 研究冲刺（海外科研/远程RA）或 实习 |
