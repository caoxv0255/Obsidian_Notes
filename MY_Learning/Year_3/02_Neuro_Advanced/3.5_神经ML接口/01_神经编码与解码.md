# 神经编码与解码

## 概述

神经编码（Neuronal Encoding）与神经解码（Neuronal Decoding）是计算神经科学的核心问题之一。前者研究神经系统如何将外部刺激或内部状态转化为神经活动的模式，后者则研究如何从记录到的神经信号推断所编码的信息。这两个方向共同构成了理解大脑信息处理的桥梁。

本章将系统介绍神经编码的主要理论框架——Rate Code 与 Temporal Code，感受野与响应场的概念，信息论视角下的神经编码容量，以及多种神经解码方法。最后通过 Python 代码模拟神经元群的 Poisson 发放过程并计算互信息，巩固对理论的理解。

---

## 1. Rate Code vs Temporal Code

### 1.1 频率编码（Rate Code）

频率编码是神经科学中最经典的信息编码假说。其核心思想是：**信息主要通过神经元的发放频率（firing rate）来传递**。

$$r = \frac{n}{T}$$

其中 $r$ 为发放频率（spikes/s），$n$ 为时间窗口 $T$ 内的动作电位数量。

**历史背景：** Adrian（1926）在青蛙肌肉实验中首次提出这一观点。他发现刺激强度与感觉神经的发放频率呈正相关，而非取决于单个动作电位的具体形状。

**频率编码的优点：**
- 对噪声具有较强的鲁棒性
- 时间窗口内的平均操作天然具有低通滤波效果
- 实验测量相对简单（传统胞外电极即可）

**局限性：**
- 无法解释快速行为响应（< 50ms）
- 忽略了 spike timing 中的精细结构
- 脑区中普遍存在的相位进动（phase precession）现象难以用纯频率编码解释

### 1.2 时间编码（Temporal Code）

时间编码主张：**信息不仅存在于发放频率中，更编码在动作电位的精确时间结构中**。

时间编码的典型特征包括：

- **相位编码（Phase Coding）：** 神经元在其振荡周期中的特定相位发放，如海马区 place cell 的相位进动现象
- **时间精确编码（Temporal Precision）：** 刺激信息在 < 10ms 的时间尺度上被编码
- **时空模式（Spatio-temporal Pattern）：** 多个神经元的协同发放模式携带信息

### 1.3 两种编码框架的融合

现代研究越来越认识到，频率编码与时间编码并非互斥关系，而是适用于不同脑区和不同认知任务的不同层面。

| 特征 | 频率编码 | 时间编码 |
|------|----------|----------|
| 信息载体 | 平均发放率 | Spike 精确时间 |
| 时间尺度 | 50-500ms 窗口 | < 50ms |
| 适用脑区 | 皮层广泛区域 | 海马、听觉皮层等 |
| 对噪声鲁棒性 | 高 | 中等 |
| 信息容量 | 中等 | 高 |

## 2. 响应场与感受野

### 2.1 基本定义

**感受野（Receptive Field，RF）** 是计算神经科学中最核心的概念之一。它指的是：对于一个特定神经元，能够引起其反应（激活或抑制）的全部刺激空间。

对于初级视觉皮层（V1）神经元：
$$RF(x, y) = \text{空间位置 }(x, y) \text{ 的加权感受区域}$$

### 2.2 视觉系统的经典感受野

Hubel 和 Wiesel（1959, 1962）的开创性工作揭示了视觉皮层神经元的感受野特性：

**简单细胞（Simple Cells）：**
- 具有明确的 ON 区和 OFF 区（空间分离）
- 对特定方向的边缘/条形反应最强
- 可用 Gabor 函数描述：
$$G(x, y) = A \exp\left(-\frac{x'^2 + y'^2}{2\sigma^2}\right) \cos(2\pi f x' + \phi)$$
其中 $x' = x\cos\theta + y\sin\theta$ 为旋转后的坐标

**复杂细胞（Complex Cells）：**
- ON 区和 OFF 区重叠，不再有空间分离
- 对运动方向敏感
- 对精确位置不敏感（位置不变性）

### 2.3 感受野的数学描述

视觉皮层简单细胞的感受野可以用 Gabor 函数精确建模。Gabor 函数结合了高斯包络与正弦载波，能够同时描述空间频率选择性和方向选择性。

### 2.4 感受野映射方法

实验上测量感受野的经典方法包括：

1. **Reverse Correlation（反向相关）：** 向被试呈现白噪声刺激序列，记录神经响应，通过互相关计算 RF
2. **Spike-Triggered Average（STA）：** 对 spike 时刻之前的刺激取平均，得到等效感受野
3. **系统探测刺激（Probing Stimuli）：** 使用特定几何形状（边缘、条形）系统探测 RF

## 3. 信息论视角下的神经编码

### 3.1 熵与神经信息

信息论为神经编码提供了严格的定量框架。Shannon（1948）的开创性工作定义了信息的基本量度：

**香农熵（Shannon Entropy）：**
$$H(X) = -\sum_{x \in \mathcal{X}} p(x) \log_2 p(x) \quad \text{[bits]}$$

熵衡量了随机变量 $X$ 的不确定性。对于神经编码，熵表示神经 response $R$ 在不掌握任何刺激知识时的总信息量。

**互信息（Mutual Information）：**
$$I(S; R) = H(R) - H(R|S) = \sum_{s, r} p(s, r) \log_2 \frac{p(r|s)}{p(r)}$$

互信息直接量化了神经系统从刺激 $S$ 到响应 $R$ 传递了多少信息——这正是神经编码的核心问题。

**神经信息编码容量：**

在 Rate Code 框架下，单神经元的信息容量约为：
$$C_{\text{rate}} \approx \log_2(1 + \text{SNR}) \quad \text{[bits/spike]}$$

## 4. 神经解码方法

### 4.1 维纳滤波（Wiener Filter）

维纳滤波是最经典的线性解码方法之一。它寻找一个线性滤波器 $w$，使得解码信号 $\hat{s}(t) = w * r(t)$ 与真实刺激 $s(t)$ 的均方误差最小：

$$\min_w \mathbb{E}\left[(s(t) - w * r(t))^2\right]$$

在频域中，维纳滤波器的解为：
$$W(\omega) = \frac{P_{sr}(\omega)}{P_{rr}(\omega)}$$

其中 $P_{sr}$ 是刺激与响应的互功率谱，$P_{rr}$ 是响应的自功率谱。

### 4.2 最大似然解码（Maximum Likelihood Decoding）

最大似然解码假设神经响应服从某种统计分布（通常是 Poisson 分布），然后寻找最可能产生该响应的刺激：

$$\hat{s} = \arg\max_s \log p(r|s)$$

对于 Poisson 模型：
$$\log p(r|s) = \sum_i \left[r_i \log f_i(s) - f_i(s)\right] + \text{const}$$

### 4.3 Population Vector Coding

Georgopoulos 等人（1986）在研究运动皮层时提出了 Population Vector 方法。给定每个神经元的偏好方向 $\mathbf{c}_i$ 和响应 $r_i$，解码出的运动方向为：

$$\mathbf{v} = \sum_i r_i \mathbf{c}_i$$

这个方法直观地体现了"投票"机制：每个神经元为其偏好方向"投票"，权重为其发放率。

def simulate_rate_coding(stimulus_strength, duration=1.0, dt=0.001, base_rate=10.0):
    """
    模拟频率编码：发放率正比于刺激强度
    
    参数:
        stimulus_strength: 刺激强度，范围 [0, 1]
        duration: 模拟时长（秒）
        dt: 时间步长（秒）
        base_rate: 基础发放率 (Hz)
    
    返回:
        spikes: 二值 spike 序列
        rate: 理论发放率 (Hz)
    """
    n_steps = int(duration / dt)
    rate = base_rate + 50 * stimulus_strength  # 线性调制
    spike_prob = rate * dt
    spikes = np.random.rand(n_steps) < spike_prob
    return spikes, rate


def simulate_temporal_coding(stimulus_strength, duration=1.0, dt=0.001):
    """
    模拟时间编码：高刺激强度时 spike 更集中在特定时间窗口
    
    参数:
        stimulus_strength: 刺激强度，范围 [0, 1]
        duration: 模拟时长（秒）
        dt: 时间步长（秒）
    
    返回:
        spikes: 二值 spike 序列
        t_info: 包含时间信息的字典
    """
    n_steps = int(duration / dt)
    base_rate = 20 * stimulus_strength
    concentration = 2 + 5 * stimulus_strength  # 集中度参数
    
    # 时间权重：使用 von Mises 分布近似
    t = np.arange(n_steps) * dt
    time_weights = np.exp(concentration * np.cos(2 * np.pi * t / duration - np.pi))
    time_weights = time_weights / time_weights.sum()
    
    spike_prob = base_rate * dt * time_weights
    spikes = np.random.rand(n_steps) < spike_prob
    
    return spikes, {"time_weights": time_weights, "base_rate": base_rate}

def gabor_function(x, y, sigma=20.0, theta=0.0, f=0.1, phi=0.0):
    """
    生成 Gabor 函数（简单细胞感受野模型）
    
    参数:
        x, y: 空间坐标（像素）
        sigma: 高斯包络标准差
        theta: 方向（弧度）
        f: 正弦载波频率
        phi: 正弦相位偏移
    
    返回:
        gabor: Gabor 函数值
    """
    x_theta = x * np.cos(theta) + y * np.sin(theta)
    y_theta = -x * np.sin(theta) + y * np.cos(theta)
    gaussian = np.exp(-(x_theta**2 + y_theta**2) / (2 * sigma**2))
    sinusoidal = np.cos(2 * np.pi * f * x_theta + phi)
    return gaussian * sinusoidal
