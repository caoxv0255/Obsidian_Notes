## 概述

网络动力学是研究神经网络中信息传递和处理规律的学科，旨在理解大脑网络的结构-功能关系和动态特性。本章将系统介绍网络动力学的基本概念、理论基础和实践技巧，为后续学习高级网络模型打下基础。

## 1. 网络动力学的定义

### 1.1 网络动力学的定义

**网络动力学**（Network Dynamics）是研究神经网络中信息传递和处理规律的学科，关注网络的动态行为和功能特性。

### 1.2 网络动力学的重要性

1. **理解大脑功能**：解释大脑如何处理信息
2. **疾病机制研究**：理解神经疾病的网络机制
3. **人工智能发展**：推动神经网络和机器学习发展
4. **认知科学**：解释认知过程的神经基础

### 1.3 网络动力学的挑战

1. **复杂性**：神经网络的复杂性极高
2. **多尺度**：从分子到系统层面的多尺度
3. **动态性**：网络的动态变化特性
4. **个体差异**：不同个体间的差异

## 2. 网络动力学的理论基础

### 2.1 网络理论

#### 图论基础

**基本概念**：
- 节点（Node）：神经元或脑区
- 边（Edge）：神经连接或功能连接
- 度（Degree）：节点的连接数
- 路径（Path）：节点间的连接路径

**重要指标**：
- 平均路径长度
- 聚类系数
- 度分布
- 中心性

#### 随机图模型

**Erdős-Rényi模型**：
- 随机连接
- 度分布服从泊松分布
- 适用于小世界网络

**Barabási-Albert模型**：
- 优先连接
- 度分布服从幂律分布
- 适用于无标度网络

```python
import numpy as np
import matplotlib.pyplot as plt
import networkx as nx
from scipy.stats import powerlaw

class NetworkDynamics:
    """网络动力学"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def create_random_graph(self, n_nodes, edge_probability):
        """创建随机图"""
        G = nx.erdos_renyi_graph(n_nodes, edge_probability)
        return G
    
    def create_scale_free_network(self, n_nodes, m):
        """创建无标度网络"""
        G = nx.barabasi_albert_graph(n_nodes, m)
        return G
    
    def create_small_world_network(self, n_nodes, k, p):
        """创建小世界网络"""
        G = nx.watts_strogatz_graph(n_nodes, k, p)
        return G
    
    def calculate_network_metrics(self, G):
        """计算网络指标"""
        metrics = {}
        
        # 基本指标
        metrics['n_nodes'] = G.number_of_nodes()
        metrics['n_edges'] = G.number_of_edges()
        metrics['density'] = nx.density(G)
        
        # 路径指标
        if nx.is_connected(G):
            metrics['avg_path_length'] = nx.average_shortest_path_length(G)
        else:
            metrics['avg_path_length'] = float('inf')
        
        # 聚类指标
        metrics['clustering_coefficient'] = nx.average_clustering(G)
        
        # 度分布
        degrees = [d for n, d in G.degree()]
        metrics['avg_degree'] = np.mean(degrees)
        metrics['degree_std'] = np.std(degrees)
        
        # 中心性
        metrics['degree_centrality'] = nx.degree_centrality(G)
        metrics['betweenness_centrality'] = nx.betweenness_centrality(G)
        metrics['eigenvector_centrality'] = nx.eigenvector_centrality(G, max_iter=1000)
        
        return metrics
    
    def analyze_network_structure(self, G):
        """分析网络结构"""
        # 度分布
        degrees = [d for n, d in G.degree()]
        degree_distribution = np.histogram(degrees, bins=np.arange(min(degrees), max(degrees)+2))[0]
        
        # 度分布拟合
        try:
            fit = powerlaw.Fit(degrees)
            power_law_alpha = fit.power_law.alpha
        except:
            power_law_alpha = None
        
        return {
            'degree_distribution': degree_distribution,
            'power_law_alpha': power_law_alpha,
            'assortativity': nx.degree_assortativity_coefficient(G)
        }

class NetworkSimulation:
    """网络模拟"""
    
    def __init__(self):
        """初始化"""
        self.dynamics = NetworkDynamics()
    
    def simulate_synchronous_dynamics(self, G, initial_state, n_steps=100):
        """同步动力学模拟"""
        n_nodes = G.number_of_nodes()
        state = initial_state.copy()
        states = [state.copy()]
        
        for _ in range(n_steps):
            new_state = np.zeros(n_nodes)
            
            for i in range(n_nodes):
                neighbors = list(G.neighbors(i))
                if len(neighbors) > 0:
                    new_state[i] = np.mean([state[j] for j in neighbors])
                else:
                    new_state[i] = state[i]
            
            state = new_state
            states.append(state.copy())
        
        return np.array(states)
    
    def simulate_asynchronous_dynamics(self, G, initial_state, n_steps=100):
        """异步动力学模拟"""
        n_nodes = G.number_of_nodes()
        state = initial_state.copy()
        states = [state.copy()]
        
        for _ in range(n_steps):
            # 随机选择节点
            node = np.random.randint(0, n_nodes)
            
            # 更新节点状态
            neighbors = list(G.neighbors(node))
            if len(neighbors) > 0:
                state[node] = np.mean([state[j] for j in neighbors])
            
            states.append(state.copy())
        
        return np.array(states)
    
    def simulate_stochastic_dynamics(self, G, initial_state, n_steps=100, noise_level=0.1):
        """随机动力学模拟"""
        n_nodes = G.number_of_nodes()
        state = initial_state.copy()
        states = [state.copy()]
        
        for _ in range(n_steps):
            new_state = np.zeros(n_nodes)
            
            for i in range(n_nodes):
                neighbors = list(G.neighbors(i))
                if len(neighbors) > 0:
                    new_state[i] = np.mean([state[j] for j in neighbors])
                else:
                    new_state[i] = state[i]
                
                # 添加噪声
                new_state[i] += np.random.normal(0, noise_level)
            
            state = new_state
            states.append(state.copy())
        
        return np.array(states)
```

### 2.2 动态系统理论

#### 相空间分析

**相空间**：描述系统状态的空间。

**轨迹**：系统状态随时间的演化。

**吸引子**：系统趋向的稳定状态。

#### 混沌理论

**混沌系统**：对初始条件敏感的系统。

**李雅普诺夫指数**：衡量系统敏感性的指标。

**Python实现**
```python
class DynamicalSystems:
    """动力系统"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def logistic_map(self, x, r):
        """Logistic映射"""
        return r * x * (1 - x)
    
    def simulate_logistic_map(self, x0, r, n_steps=1000):
        """模拟Logistic映射"""
        x = np.zeros(n_steps)
        x[0] = x0
        
        for i in range(1, n_steps):
            x[i] = self.logistic_map(x[i-1], r)
        
        return x
    
    def lyapunov_exponent(self, x, r, n_steps=1000):
        """计算李雅普诺夫指数"""
        lyapunov_sum = 0
        
        for i in range(1, n_steps):
            x_next = self.logistic_map(x[i-1], r)
            lyapunov_sum += np.log(abs(r * (1 - 2 * x[i-1])))
            x[i] = x_next
        
        return lyapunov_sum / (n_steps - 1)
    
    def rossler_system(self, state, a=0.2, b=0.2, c=5.7):
        """Rossler系统"""
        x, y, z = state
        dxdt = -y - z
        dydt = x + a * y
        dzdt = b + z * (x - c)
        return np.array([dxdt, dydt, dzdt])
    
    def simulate_rossler_system(self, initial_state, a=0.2, b=0.2, c=5.7, n_steps=10000, dt=0.01):
        """模拟Rossler系统"""
        states = np.zeros((n_steps, 3))
        states[0] = initial_state
        
        for i in range(1, n_steps):
            k1 = self.rossler_system(states[i-1], a, b, c)
            k2 = self.rossler_system(states[i-1] + 0.5 * dt * k1, a, b, c)
            k3 = self.rossler_system(states[i-1] + 0.5 * dt * k2, a, b, c)
            k4 = self.rossler_system(states[i-1] + dt * k3, a, b, c)
            
            states[i] = states[i-1] + (dt / 6.0) * (k1 + 2*k2 + 2*k3 + k4)
        
        return states

class NetworkDynamicsAnalysis:
    """网络动力学分析"""
    
    def __init__(self):
        """初始化"""
        self.dynamics = DynamicalSystems()
    
    def calculate_network_stability(self, G, dynamics_function, n_steps=100):
        """计算网络稳定性"""
        n_nodes = G.number_of_nodes()
        stability_scores = []
        
        for _ in range(n_steps):
            # 随机选择初始状态
            initial_state = np.random.uniform(-1, 1, n_nodes)
            
            # 模拟动力学
            states = dynamics_function(initial_state)
            
            # 计算稳定性
            final_state = states[-1]
            stability = np.mean(np.abs(final_state))
            stability_scores.append(stability)
        
        return np.mean(stability_scores), np.std(stability_scores)
    
    def detect_network_transitions(self, G, dynamics_function, threshold=0.5):
        """检测网络相变"""
        n_nodes = G.number_of_nodes()
        transitions = []
        
        for _ in range(100):
            # 随机选择初始状态
            initial_state = np.random.uniform(-1, 1, n_nodes)
            
            # 模拟动力学
            states = dynamics_function(initial_state)
            
            # 检测相变
            for i in range(1, len(states)):
                if np.mean(np.abs(states[i])) > threshold and np.mean(np.abs(states[i-1])) <= threshold:
                    transitions.append(i)
        
        return transitions
```

### 2.3 网络同步理论

#### 同步现象

**同步**：网络中节点状态趋于一致。

**同步条件**：
- 网络连接强度足够大
- 节点动力学特性相似

#### 同步指标

**同步指数**：
$$S = \frac{1}{n-1} \sum_{i=1}^{n-1} \sum_{j=i+1}^{n} \frac{|\phi_i - \phi_j|}{\max(|\phi_i|, |\phi_j|)}$$

**Python实现**
```python
class NetworkSynchronization:
    """网络同步"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_synchronization_index(self, states):
        """计算同步指数"""
        n_nodes = states.shape[1]
        n_steps = states.shape[0]
        
        sync_index = np.zeros(n_steps)
        
        for t in range(n_steps):
            state_t = states[t]
            mean_state = np.mean(state_t)
            variance = np.var(state_t)
            
            if variance > 0:
                sync_index[t] = 1 - np.std(state_t) / np.sqrt(variance)
            else:
                sync_index[t] = 1
        
        return sync_index
    
    def calculate_laplacian_matrix(self, G):
        """计算拉普拉斯矩阵"""
        n_nodes = G.number_of_nodes()
        L = np.zeros((n_nodes, n_nodes))
        
        # 邻接矩阵
        A = nx.adjacency_matrix(G).toarray()
        
        # 度矩阵
        D = np.diag(np.sum(A, axis=1))
        
        # 拉普拉斯矩阵
        L = D - A
        
        return L
    
    def calculate_synchronization_threshold(self, G, coupling_strength=1.0):
        """计算同步阈值"""
        L = self.calculate_laplacian_matrix(G)
        
        # 最小非零特征值
        eigenvalues = np.linalg.eigvals(L)
        eigenvalues = np.sort(np.real(eigenvalues))
        
        # 同步阈值
        threshold = eigenvalues[1] / (2 * coupling_strength)
        
        return threshold
    
    def simulate_network_synchronization(self, G, initial_state, n_steps=100, coupling_strength=1.0):
        """模拟网络同步"""
        n_nodes = G.number_of_nodes()
        states = np.zeros((n_steps, n_nodes))
        states[0] = initial_state
        
        L = self.calculate_laplacian_matrix(G)
        
        for t in range(1, n_steps):
            state_t = states[t-1]
            
            # 动力学方程
            d_state = -state_t + coupling_strength * L @ state_t
            
            # 更新状态
            states[t] = state_t + 0.01 * d_state
        
        return states

class NetworkBifurcation:
    """网络分岔"""
    
    def __init__(self):
        """初始化"""
        self.synchronization = NetworkSynchronization()
    
    def calculate_bifurcation_diagram(self, G, coupling_strengths, n_steps=1000):
        """计算分岔图"""
        bifurcation_data = []
        
        for coupling in coupling_strengths:
            # 模拟网络同步
            initial_state = np.random.uniform(-1, 1, G.number_of_nodes())
            states = self.synchronization.simulate_network_synchronization(
                G, initial_state, n_steps=n_steps, coupling_strength=coupling
            )
            
            # 计算同步指数
            sync_index = self.synchronization.calculate_synchronization_index(states)
            
            # 收集数据
            for i in range(n_steps // 2, n_steps):  # 舍弃瞬态
                bifurcation_data.append((coupling, np.mean(sync_index[i-50:i])))
        
        return bifurcation_data
```

```python
import numpy as np
import matplotlib.pyplot as plt
import networkx as nx
from sklearn.decomposition import PCA
from sklearn.manifold import TSNE

class AdvancedNetworkDynamics:
    """高级网络动力学"""
    
    def __init__(self):
        """初始化"""
        self.synchronization = NetworkSynchronization()
        self.dynamics = DynamicalSystems()
    
    def calculate_network_metrics_advanced(self, G):
        """计算高级网络指标"""
        metrics = {}
        
        # 基本指标
        metrics['n_nodes'] = G.number_of_nodes()
        metrics['n_edges'] = G.number_of_edges()
        metrics['density'] = nx.density(G)
        
        # 路径指标
        if nx.is_connected(G):
            metrics['avg_path_length'] = nx.average_shortest_path_length(G)
            metrics['diameter'] = nx.diameter(G)
        else:
            metrics['avg_path_length'] = float('inf')
            metrics['diameter'] = float('inf')
        
        # 聚类指标
        metrics['clustering_coefficient'] = nx.average_clustering(G)
        metrics['transitivity'] = nx.transitivity(G)
        
        # 度分布
        degrees = [d for n, d in G.degree()]
        metrics['avg_degree'] = np.mean(degrees)
        metrics['degree_std'] = np.std(degrees)
        
        # 中心性
        metrics['degree_centrality'] = nx.degree_centrality(G)
        metrics['betweenness_centrality'] = nx.betweenness_centrality(G)
        metrics['eigenvector_centrality'] = nx.eigenvector_centrality(G, max_iter=1000)
        metrics['closeness_centrality'] = nx.closeness_centrality(G)
        
        # 社区结构
        try:
            communities = nx.community.label_propagation_communities(G)
            metrics['n_communities'] = len(communities)
            metrics['modularity'] = nx.community.modularity(G, communities)
        except:
            metrics['n_communities'] = 0
            metrics['modularity'] = 0
        
        return metrics
    
    def analyze_network_dynamics(self, G, dynamics_function, n_steps=100):
        """分析网络动力学"""
        n_nodes = G.number_of_nodes()
        dynamics_data = []
        
        for _ in range(10):  # 10次随机模拟
            # 随机选择初始状态
            initial_state = np.random.uniform(-1, 1, n_nodes)
            
            # 模拟动力学
            states = dynamics_function(initial_state)
            
            # 计算动力学指标
            sync_index = self.synchronization.calculate_synchronization_index(states)
            stability = np.std(sync_index)
            
            dynamics_data.append({
                'stability': stability,
                'final_sync': sync_index[-1],
                'mean_sync': np.mean(sync_index)
            })
        
        return dynamics_data
    
    def reconstruct_network_from_time_series(self, time_series, method='correlation'):
        """从时间序列重建网络"""
        n_nodes = time_series.shape[1]
        
        if method == 'correlation':
            # 相关性矩阵
            correlation_matrix = np.corrcoef(time_series.T)
            
            # 构建图
            G = nx.from_numpy_array(correlation_matrix)
            
        elif method == 'granger':
            # Granger因果分析
            G = nx.Graph()
            
            for i in range(n_nodes):
                for j in range(n_nodes):
                    if i != j:
                        # 简化的Granger因果分析
                        G.add_edge(i, j)
        
        return G
```
