# 3.3 学习算法

> **专业视角**：突触可塑性是神经网络学习的生物学基础。STDP是最重要的生物可塑性模型，也是脉冲神经网络训练的理论起点。本模块从Hebbian学习出发，系统覆盖STDP机制、脉冲神经网络的训练方法、以及强化学习与神经科学的交叉。

## 4.1 Hebbian学习与突触可塑性

### Hebb法则

**"一起放电的神经元，连接会变强"**（Neurons that fire together, wire together）

$$
\Delta w_{ij} = \eta \cdot x_i \cdot y_j
$$

- $x_i$: 突触前神经元 $i$ 的活动
- $y_j$: 突触后神经元 $j$ 的活动
- $\eta$: 学习率

### 核心机制类型

| 类型 | 定义 | 时间窗口 |
|------|------|----------|
| 长时程增强（LTP） | 突触增强，高频刺激后 | 持续小时~天 |
| 长时程抑制（LTD） | 突触减弱，低频刺激后 | 持续小时~天 |
| 短时程可塑性（STP） | 突触效能的短暂变化 | 毫秒~秒 |

## 4.2 STDP（脉冲时序依赖可塑性）

### 核心规则

STDP根据突触前脉冲和突触后脉冲的**时序关系**调整突触权重：

$$
\Delta w = \begin{cases}
A_+ \cdot e^{-\Delta t / \tau_+}, & \Delta t > 0 \text{（突触前先于突触后）} \\
-A_- \cdot e^{\Delta t / \tau_-}, & \Delta t < 0 \text{（突触后先于突触前）}
\end{cases}
$$

其中 $\Delta t = t_{post} - t_{pre}$

### Python实现

```python
import numpy as np

class STDP:
    """标准STDP模型"""

    def __init__(self, A_plus=0.01, A_minus=0.012, tau_plus=20e-3, tau_minus=20e-3):
        self.A_plus = A_plus      # LTP学习幅度
        self.A_minus = A_minus    # LTD学习幅度
        self.tau_plus = tau_plus  # LTP时间常数(ms)
        self.tau_minus = tau_minus  # LTD时间常数(ms)

    def update(self, w, dt):
        """
        更新权重
        dt: 突触前-突触后时间差(ms)
        w: 当前权重
        """
        if dt > 0:
            # 突触前先激活 → 突触后激活 → 强化
            dw = self.A_plus * np.exp(-dt / self.tau_plus)
        else:
            # 突触后先激活 → 突触前激活 → 弱化
            dw = -self.A_minus * np.exp(dt / self.tau_minus)
        return np.clip(w + dw, 0, 1)  # 权重限制在[0,1]
```

### STDP特性

- **不对称时间窗口**：LTP窗口约+40ms， LTD窗口约-40ms
- **赢者通吃（Winner-Takes-All）**：强化早期到达的突触，抑制后期到达
- **时间编码**：神经元不仅编码发放频率，还编码精确时序

## 4.3 脉冲神经网络训练方法

### 代理梯度法（Surrogate Gradient）

由于脉冲函数（Heaviside）不可导，引入代理梯度：

```python
class SurrogateGradient(torch.autograd.Function):
    """代理梯度：使用sigmoid近似阶跃函数"""

    @staticmethod
    def forward(ctx, input, beta=10.0):
        ctx.save_for_backward(input)
        ctx.beta = beta
        return (input > 0).float()

    @staticmethod
    def backward(ctx, grad_output):
        input, = ctx.saved_tensors
        beta = ctx.beta
        sigmoid = torch.sigmoid(beta * input)
        return grad_output * sigmoid * (1 - sigmoid) * beta, None

# 使用示例：SpikePyTorch中的梯度代理
class LIFSpike(torch.autograd.Function):
    @staticmethod
    def forward(ctx, x, threshold):
        return (x > threshold).float()

    @staticmethod
    def backward(ctx, grad_output):
        # 代理梯度：Sigmond近似
        return grad_output * 0.3, None
```

### STDP训练

```python
class STDPLayer:
    """STDP规则训练的脉冲神经元层"""

    def __init__(self, n_input, n_neurons, lr=0.01):
        self.W = np.random.rand(n_neurons, n_input) * 0.1
        self.lr = lr

    def forward(self, pre_spikes, post_spikes):
        """
        pre_spikes: (n_input,) 布尔数组
        post_spikes: (n_neurons,) 布尔数组
        """
        # 计算发放次数
        pre_count = pre_spikes.sum()
        post_count = post_spikes.sum()

        # Hebbian STDP
        delta_W = self.lr * np.outer(post_spikes, pre_spikes)
        delta_W -= self.lr * 0.1 * self.W  # 权重衰减，防止过大
        self.W = np.clip(self.W + delta_W, 0, 1)

    def train(self, spike_times, n_iterations=1000):
        """模拟训练"""
        for _ in range(n_iterations):
            # 随机产生脉冲序列
            pre_spikes = np.random.rand(self.W.shape[1]) > 0.9
            post_votes = self.W @ pre_spikes
            post_spikes = post_votes > np.random.rand(len(post_votes))
            self.forward(pre_spikes, post_spikes)
```

###  Reservoir Computing

```python
class EchoStateNetwork:
    """
    储备池计算：固定随机储备池，只训练读出层
    适合时序任务，训练效率高
    """

    def __init__(self, n_input, n_reservoir, n_output, spectral_radius=0.9):
        self.n_reservoir = n_reservoir
        self.W_in = np.random.rand(n_reservoir, n_input) * 2 - 1
        self.W = np.random.rand(n_reservoir, n_reservoir) - 0.5

        # 谱半径控制动态稳定性
        self.W *= spectral_radius / np.max(np.abs(np.linalg.eigvals(self.W)))

        self.W_out = np.zeros((n_output, n_reservoir))
        self.state = np.zeros(n_reservoir)

    def step(self, input_signal, leak_rate=0.3):
        """储备池单步更新"""
        pre_activation = self.W @ self.state + self.W_in @ input_signal
        self.state = (1 - leak_rate) * self.state + leak_rate * np.tanh(pre_activation)
        return self.state

    def train(self, X, y, discard=100):
        """训练读出层（岭回归）"""
        states = []
        for x in X:
            self.state = self.step(x)
            states.append(self.state)

        states = np.array(states[discard:])
        targets = y[discard:]

        # 岭回归
        self.W_out = targets.T @ states @ np.linalg.inv(
            states.T @ states + 0.01 * np.eye(states.shape[1])
        )
```

## 4.4 强化学习与神经科学的交叉

### 多巴胺与TD学习

中脑多巴胺神经元实现**时序差分学习（TD Learning）**：

```
奖励预测误差 = 实际奖励 - 预期奖励
δ = r(t) + γV(s(t+1)) - V(s(t))

多巴胺信号 ∝ δ
```

- 正向误差 → 多巴胺释放 → 强化行为
- 负向误差 → 多巴胺抑制 → 削弱行为

### Actor-Critic神经实现

```python
class ActorCriticSNN:
    """
    Actor-Critic架构的神经实现
    - Critic: 估计状态价值V(s)
    - Actor: 根据TD误差更新策略
    """

    def __init__(self, state_dim, action_dim, lr=0.001):
        self.actor = np.random.randn(action_dim, state_dim) * 0.01
        self.critic = np.random.randn(1, state_dim) * 0.01
        self.lr = lr

    def get_action(self, state):
        """Softmax策略"""
        logits = self.actor @ state
        probs = np.exp(logits - np.max(logits))
        probs /= probs.sum()
        return np.random.choice(len(probs), p=probs)

    def update(self, state, action, reward, next_state, gamma=0.99):
        """TD更新"""
        v_s = self.critic @ state
        v_next = self.critic @ next_state

        td_error = reward + gamma * v_next - v_s

        # Critic更新
        self.critic += self.lr * td_error * state

        # Actor更新
        probs = self._softmax(self.actor @ state)
        self.actor += self.lr * td_error * (np.eye(len(probs))[action] - probs)[:, None] @ state[None, :]
```

## 4.5 突触可塑性的计算理论

### 去噪Hebb规则（Differential Hebbian）

竞争与协作的平衡：

```python
class OjaRule:
    """Oja规则：Hebbian学习 + 归一化"""

    def __init__(self, n_neurons, n_input, learning_rate=0.01):
        self.W = np.random.randn(n_neurons, n_input) * 0.1
        self.lr = learning_rate

    def update(self, pre, post):
        """Oja更新"""
        # Hebbian项
        hebbian = np.outer(post, pre)
        # 抑制项（防止权重爆炸）
        normalization = np.sum(post**2) * self.W
        self.W += self.lr * (hebbian - normalization)
        return self.W
```

### Bienenstock-Cooper-Munro (BCM) 规则

突触可塑性取决于突触后活动的时间窗口：

```python
class BCMRule:
    """
    BCM规则：塑性阈值随突触后活动自适应调整
    阈值 = E[post²]的滑动平均
    """

    def __init__(self, lr=0.01, theta_scale=1.0):
        self.lr = lr
        self.theta = 0.0  # 塑性阈值
        self.theta_scale = theta_scale

    def update(self, pre, post, dt=1.0):
        """BCM更新"""
        # 更新阈值：E[post²]的滑动平均
        self.theta += 0.01 * (post**2 - self.theta)

        # 计算塑性变化
        phi = post * (post - self.theta) * self.theta_scale
        dw = self.lr * phi * pre
        return dw
```

## 4.6 睡眠与记忆巩固

### 离线重放（Offline Replay）

- **海马体重放**：睡眠时，之前激活的神经元序列被重新激活
- **尖波涟漪（Sharp-Wave Ripples）**：海马体的同步振荡事件，重放序列
- **皮质重排**：慢波睡眠期间，皮质-海马体对话进行记忆整合
