# 01-投资组合优化基础

## 概述

投资组合优化是量化金融中的核心环节，旨在通过数学方法构建最优投资组合，在给定风险水平下最大化收益，或在给定收益水平下最小化风险。本章将系统介绍投资组合优化的基本概念、理论基础和实践技巧。

## 1. 投资组合优化的定义

### 1.1 投资组合优化的定义

**投资组合优化**（Portfolio Optimization）是运用数学方法构建最优投资组合的过程，旨在实现风险与收益的最优平衡。

### 1.2 投资组合优化的重要性

1. **风险控制**：通过分散化降低投资组合风险
2. **收益最大化**：在风险可控的前提下追求最大收益
3. **效率提升**：提高投资组合的夏普比率
4. **决策支持**：为投资决策提供量化依据

### 1.3 投资组合优化的挑战

1. **参数估计**：期望收益和协方差矩阵的估计误差
2. **约束条件**：投资限制、合规要求等
3. **非线性优化**：优化问题的复杂性
4. **动态调整**：市场变化下的组合再平衡

## 2. 投资组合优化的理论基础

### 2.1 马科维茨均值-方差模型

#### 基本假设

1. **投资者厌恶风险**：在相同收益下选择风险更小的组合
2. **收益服从正态分布**：投资组合收益服从正态分布
3. **投资者只关心均值和方差**：完全由均值和方差决定投资决策

#### 优化目标

**最小方差组合**：
$$\min_{w} \quad w^T \Sigma w$$
$$\text{s.t.} \quad w^T \mu = \bar{\mu}, \quad \sum_{i=1}^n w_i = 1$$

**最大夏普比率组合**：
$$\max_{w} \quad \frac{w^T \mu - r_f}{\sqrt{w^T \Sigma w}}$$
$$\text{s.t.} \quad \sum_{i=1}^n w_i = 1$$

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from scipy.optimize import minimize

class PortfolioOptimization:
    """投资组合优化"""
    
    def __init__(self, risk_free_rate=0.02):
        """初始化"""
        self.risk_free_rate = risk_free_rate
    
    def calculate_portfolio_return(self, weights, returns):
        """计算投资组合收益率"""
        return np.dot(weights, returns)
    
    def calculate_portfolio_variance(self, weights, cov_matrix):
        """计算投资组合方差"""
        return np.dot(weights.T, np.dot(cov_matrix, weights))
    
    def calculate_portfolio_volatility(self, weights, cov_matrix):
        """计算投资组合波动率"""
        return np.sqrt(self.calculate_portfolio_variance(weights, cov_matrix))
    
    def calculate_portfolio_sharpe_ratio(self, weights, returns, cov_matrix):
        """计算投资组合夏普比率"""
        portfolio_return = self.calculate_portfolio_return(weights, returns)
        portfolio_volatility = self.calculate_portfolio_volatility(weights, cov_matrix)
        sharpe_ratio = (portfolio_return - self.risk_free_rate) / portfolio_volatility
        return sharpe_ratio
    
    def calculate_efficient_frontier(self, expected_returns, cov_matrix, n_portfolios=100):
        """计算有效前沿"""
        # 生成随机权重
        random_weights = np.random.dirichlet(np.ones(len(expected_returns)), n_portfolios)
        
        # 计算投资组合收益和风险
        portfolio_returns = np.array([self.calculate_portfolio_return(w, expected_returns) 
                                    for w in random_weights])
        portfolio_volatilities = np.array([self.calculate_portfolio_volatility(w, cov_matrix) 
                                        for w in random_weights])
        
        # 计算夏普比率
        sharpe_ratios = portfolio_returns / portfolio_volatilities
        
        return portfolio_returns, portfolio_volatilities, sharpe_ratios
    
    def optimize_min_variance(self, expected_returns, cov_matrix):
        """最小方差投资组合"""
        n_assets = len(expected_returns)
        
        # 定义优化变量
        weights = np.array([1/n_assets] * n_assets)
        
        # 定义目标函数
        def objective(weights):
            return self.calculate_portfolio_variance(weights, cov_matrix)
        
        # 定义约束条件
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
            {'type': 'ineq', 'fun': lambda w: w}
        ]
        
        # 优化
        result = minimize(objective, weights, method='SLSQP', constraints=constraints)
        
        if result.success:
            return result.x
        else:
            return None

            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
            {'type': 'ineq', 'fun': lambda w: w}

    """Black-Litterman优化"""
    
    def __init__(self, risk_free_rate=0.02):
        """初始化"""
        self.risk_free_rate = risk_free_rate
    
    def calculate_implied_returns(self, weights, cov_matrix, tau=0.025):
        """计算隐含收益"""
        implied_returns = tau * np.dot(cov_matrix, weights)
        return implied_returns
    
    def calculate_black_litterman_returns(self, expected_returns, cov_matrix, 
                                         views, tau=0.025):
        """计算Black-Litterman收益"""
        # 计算隐含收益
        implied_returns = self.calculate_implied_returns(weights, cov_matrix, tau)
        
        # 计算后验收益
        posterior_returns = (1 + tau) * expected_returns + tau * implied_returns
        
        return posterior_returns

### 2.2 现代投资组合理论

#### 多因子模型

**基本形式**：
$$R_i = \alpha_i + \sum_{j=1}^k \beta_{ij} F_j + \epsilon_i$$

其中：
- $R_i$：资产$i$的收益率
- $\alpha_i$：特定收益率
- $\beta_{ij}$：资产$i$对因子$j$的敏感性
- $F_j$：第$j$个因子收益率
- $\epsilon_i$：特定收益率

#### 因子模型优化

**优化目标**：
$$\min_{w} \quad w^T \Sigma w$$
$$\text{s.t.} \quad w^T \mu = \bar{\mu}, \quad \sum_{i=1}^n w_i = 1$$

**约束条件**：
- 因子暴露限制
- 行业权重限制
- 最大单资产权重限制

### 2.3 风险平价模型

#### 风险平价原理

**核心思想**：每个资产对投资组合总风险的贡献相等。

#### 数学表达

**风险贡献**：
$$\text{RC}_i = w_i \cdot \frac{\partial \sigma_p}{\partial w_i}$$

**风险平价约束**：
$$\text{RC}_i = \frac{1}{n} \sum_{j=1}^n \text{RC}_j$$

## 3. 投资组合优化的实践技巧

### 3.1 参数估计

#### 期望收益估计

**历史平均法**：
$$\hat{\mu}_i = \frac{1}{T} \sum_{t=1}^T r_{i,t}$$

**指数加权移动平均法**：
$$\hat{\mu}_i = \sum_{t=1}^T \lambda^{T-t} r_{i,t}$$

#### 协方差矩阵估计

**样本协方差矩阵**：
$$\hat{\Sigma}_{ij} = \frac{1}{T-1} \sum_{t=1}^T (r_{i,t} - \hat{\mu}_i)(r_{j,t} - \hat{\mu}_j)$$

**收缩估计**：
$$\hat{\Sigma} = \lambda \hat{\Sigma}_{sample} + (1-\lambda) \hat{\Sigma}_{target}$$

### 3.2 约束条件处理

#### 线性约束

**等式约束**：
$$A w = b$$

**不等式约束**：
$$C w \leq d$$

#### 非线性约束

**二次约束**：
$$w^T \Sigma w \leq \sigma_{target}^2$$

**绝对值约束**：
$$|w_i| \leq w_{max,i}$$

### 3.3 优化算法

#### 二次规划

**标准形式**：
$$\min_{w} \quad \frac{1}{2} w^T H w + f^T w$$
$$\text{s.t.} \quad A w = b, \quad C w \leq d$$

#### 迭代优化

**梯度下降法**：
$$w_{k+1} = w_k - \alpha_k \nabla f(w_k)$$

**牛顿法**：
$$w_{k+1} = w_k - [\nabla^2 f(w_k)]^{-1} \nabla f(w_k)$$

        """计算夏普比率"""
        portfolio_return = np.dot(weights, returns)
        portfolio_volatility = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
        sharpe_ratio = (portfolio_return - self.risk_free_rate) / portfolio_volatility
        return sharpe_ratio
    
    def calculate_var(self, returns, weights, cov_matrix, confidence_level=0.05):
        """计算VaR"""
        portfolio_return = np.dot(weights, returns)
        portfolio_volatility = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
        var = portfolio_return - np.percentile(np.random.normal(portfolio_return, portfolio_volatility, 10000), confidence_level * 100)
        return var
    
    def calculate_cvar(self, returns, weights, cov_matrix, confidence_level=0.05):
        """计算CVaR"""
        portfolio_return = np.dot(weights, returns)
        portfolio_volatility = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
        var = portfolio_return - np.percentile(np.random.normal(portfolio_return, portfolio_volatility, 10000), confidence_level * 100)
        cvar = np.mean(np.random.normal(portfolio_return, portfolio_volatility, 10000)[np.random.normal(portfolio_return, portfolio_volatility, 10000) <= var])
        return cvar
    
    def optimize_portfolio(self, returns, cov_matrix, risk_tolerance=0.05, 
                          target_return=None, max_volatility=None):
        """优化投资组合"""
        n_assets = len(returns)
        
        # 定义优化变量
        weights = np.array([1/n_assets] * n_assets)
        
        # 定义目标函数
        def objective(weights):
            portfolio_volatility = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
            portfolio_return = np.dot(weights, returns)
            
            # 风险惩罚
            risk_penalty = risk_tolerance * portfolio_volatility
            
            # 收益惩罚
            if target_return:
                return_penalty = 1000 * (portfolio_return - target_return) ** 2
            
            # 波动率惩罚
            if max_volatility:
                volatility_penalty = 1000 * (portfolio_volatility - max_volatility) ** 2
            
            return portfolio_volatility + risk_penalty + return_penalty + volatility_penalty
        
        # 定义约束条件
        constraints = [
            {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
            {'type': 'ineq', 'fun': lambda w: w}
        ]
        
        if target_return:
            constraints.append({'type': 'eq', 'fun': lambda w: np.dot(weights, returns) - target_return})
        
        if max_volatility:
            constraints.append({'type': 'ineq', 'fun': lambda w: max_volatility - np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))})
        
        # 优化
        result = minimize(objective, weights, method='SLSQP', constraints=constraints)
        
        if result.success:
            return result.x
        else:
            return None
    
    def calculate_portfolio_risk_metrics(self, weights, returns, cov_matrix):
        """计算投资组合风险指标"""
        portfolio_return = np.dot(weights, returns)
        portfolio_volatility = np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
        sharpe_ratio = self.calculate_sharpe_ratio(returns, weights, cov_matrix)
        var = self.calculate_var(returns, weights, cov_matrix)
        cvar = self.calculate_cvar(returns, weights, cov_matrix)
        
        return {
            'return': portfolio_return,
            'volatility': portfolio_volatility,
            'sharpe_ratio': sharpe_ratio,
            'var': var,
            'cvar': cvar,
            'weights': weights
        }

class MultiPeriodPortfolioOptimization:
    """多期投资组合优化"""
    
    def __init__(self, risk_free_rate=0.02):

    def optimize_multi_period(self, returns, cov_matrix, periods=5, 
                             target_returns=None, max_volatilities=None):
        """多期投资组合优化"""
        n_assets = len(returns)
        n_periods = periods
        
        if target_returns is None:
            target_returns = [0.05] * n_periods
        
        if max_volatilities is None:
            max_volatilities = [0.1] * n_periods
        
        # 生成权重矩阵
        weights = np.zeros((n_periods, n_assets))
        
        # 优化每期权重
        for i in range(n_periods):
            # 定义优化变量
            w = np.array([1/n_assets] * n_assets)
            
            # 定义目标函数
            def objective(w):
                portfolio_volatility = np.sqrt(np.dot(w.T, np.dot(cov_matrix, w)))
                portfolio_return = np.dot(w, returns)
                
                # 风险惩罚
                risk_penalty = 0.01 * portfolio_volatility
                
                # 收益惩罚
                return_penalty = 1000 * (portfolio_return - target_returns[i]) ** 2
                
                # 波动率惩罚
                volatility_penalty = 1000 * (portfolio_volatility - max_volatilities[i]) ** 2
                
                return portfolio_volatility + risk_penalty + return_penalty + volatility_penalty
            
            # 定义约束条件
            constraints = [
                {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},
                {'type': 'ineq', 'fun': lambda w: w}
            ]
            
            # 优化
            result = minimize(objective, w, method='SLSQP', constraints=constraints)
            
            if result.success:
                weights[i] = result.x
            else:
                weights[i] = w
        
        return weights
    
    def calculate_multi_period_metrics(self, weights, returns, cov_matrix):
        """计算多期投资组合指标"""
        n_periods = len(weights)
        period_metrics = []
        
        for i in range(n_periods):
            portfolio_return = np.dot(weights[i], returns)
            portfolio_volatility = np.sqrt(np.dot(weights[i].T, np.dot(cov_matrix, weights[i])))
            sharpe_ratio = (portfolio_return - self.risk_free_rate) / portfolio_volatility
            
            period_metrics.append({
                'period': i+1,
                'return': portfolio_return,
                'volatility': portfolio_volatility,
                'sharpe_ratio': sharpe_ratio
            })
        
        return period_metrics
