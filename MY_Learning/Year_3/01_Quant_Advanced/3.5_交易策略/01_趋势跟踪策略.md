# 趋势跟踪策略

## 核心概念

趋势跟踪（Trend Following）是量化交易中最经典、实践最广泛的一类策略。其核心理念是**"趋势是你的朋友"**（The Trend is Your Friend）——资产价格一旦形成趋势，往往会延续一段时间，策略通过识别并跟随趋势获利。

趋势跟踪的理论基础来自行为金融学：**锚定效应**使投资者对新信息反应不足，导致价格逐步调整形成趋势；**羊群效应**使趋势形成后吸引更多追随者，强化趋势延续。学术界对趋势的成因存在争议，但趋势策略在长期实践中证明了其有效性。

趋势跟踪通常被视为**相对价值策略**的对面——不试图预测资产的绝对价值，而是顺应市场方向。在资产配置中，趋势跟踪策略与传统股债组合的**相关性较低**，具有显著的**分散化收益**，这也是CTA（Commodity Trading Advisor）策略在组合中配置价值的原因。

## 方法论

### 趋势识别方法

| 方法 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| 移动平均交叉 | 短期MA上穿/下穿长期MA | 简单直观 | 滞后严重 |
| 趋势线突破 | 价格突破趋势线时入场 | 直观 | 主观性强 |
| 波动率通道 | 价格突破布林带/肯特纳通道 | 自适应 | 参数敏感 |
| 价格动量 | ROC、ADX等动量指标 | 响应快 | 震荡市亏损 |
| 多时间框架 | 日线确认方向，小时线入场 | 综合信息 | 复杂度高 |

### 核心指标

1. **ADX（Average Directional Index）**：衡量趋势强度的指标，ADX > 25通常表示趋势明确
2. **移动平均线**：SMA、EMA、WMA等，用于判断趋势方向
3. **ATR（Average True Range）**：衡量波动率，用于动态止损
4. **动量指标**：ROC、RSI等，衡量价格变化速度

### 趋势策略的收益来源

```
趋势收益 = 趋势段利润 - 震荡段亏损 - 交易成本
```

趋势策略在趋势市场中获利，在震荡市场中反复止损。**胜率通常较低**（30-40%），但**盈亏比高**（>2:1），依靠少数大盈利覆盖多数小亏损。

## 代码实现

```python
import numpy as np
import pandas as pd
from dataclasses import dataclass
from typing import Optional, Tuple

@dataclass
class TrendSignal:
    timestamp: pd.Timestamp
    symbol: str
    direction: int  # 1: long, -1: short, 0: flat
    strength: float  # 0-1
    entry_price: float
    stop_loss: float
    take_profit: Optional[float] = None

class TrendFollowingStrategy:
    """双EMA交叉趋势跟踪策略"""
    
    def __init__(self, fast_period: int = 10, slow_period: int = 50,
                 atr_period: int = 14, atr_multiplier: float = 2.0,
                 adx_threshold: float = 25.0):
        self.fast_period = fast_period
        self.slow_period = slow_period
        self.atr_period = atr_period
        self.atr_multiplier = atr_multiplier
        self.adx_threshold = adx_threshold
    
    def calculate_ema(self, prices: pd.Series, period: int) -> pd.Series:
        return prices.ewm(span=period, adjust=False).mean()
    
    def calculate_atr(self, high: pd.Series, low: pd.Series, 
                     close: pd.Series, period: int) -> pd.Series:
        tr1 = high - low
        tr2 = abs(high - close.shift())
        tr3 = abs(low - close.shift())
        tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
        atr = tr.rolling(window=period).mean()
        return atr
    
    def calculate_adx(self, high: pd.Series, low: pd.Series,
                     close: pd.Series, period: int) -> pd.Series:
        plus_dm = high.diff()
        minus_dm = -low.diff()
        
        plus_dm[plus_dm < 0] = 0
        minus_dm[minus_dm < 0] = 0
        
        tr = self.calculate_atr(high, low, close, period)
        
        plus_di = 100 * (plus_dm.rolling(period).mean() / tr)
        minus_di = 100 * (minus_dm.rolling(period).mean() / tr)
        
        dx = 100 * abs(plus_di - minus_di) / (plus_di + minus_di)
        adx = dx.rolling(window=period).mean()
        
        return adx
    
    def generate_signals(self, data: pd.DataFrame) -> pd.DataFrame:
        """生成交易信号"""
        df = data.copy()
        
        # 计算指标
        df['ema_fast'] = self.calculate_ema(df['close'], self.fast_period)
        df['ema_slow'] = self.calculate_ema(df['close'], self.slow_period)
        df['atr'] = self.calculate_atr(df['high'], df['low'], df['close'], self.atr_period)
        df['adx'] = self.calculate_adx(df['high'], df['low'], df['close'], self.atr_period)
        
        # 趋势方向信号
        df['trend_long'] = df['ema_fast'] > df['ema_slow']
        df['trend_strength'] = df['adx'] / 100.0  # 归一化
        
        # 生成入场信号
        df['signal'] = 0
        df.loc[(df['trend_long']) & (df['adx'] > self.adx_threshold), 'signal'] = 1
        df.loc[(~df['trend_long']) & (df['adx'] > self.adx_threshold), 'signal'] = -1
        
        # 入场点：EMA交叉且ADX确认
        df['entry'] = (df['signal'].diff() != 0) & (df['signal'] != 0)
        
        return df
    
    def calculate_position_size(self, price: float, atr: float, 
                                capital: float, risk_pct: float = 0.02) -> int:
        """基于ATR的头寸计算"""
        risk_amount = capital * risk_pct
        stop_distance = atr * self.atr_multiplier
        share_size = risk_amount / stop_distance
        return int(share_size)
    
    def backtest(self, data: pd.DataFrame, initial_capital: float = 1000000,
                 risk_pct: float = 0.02) -> dict:
        """简化的回测"""
        df = self.generate_signals(data)
        capital = initial_capital
        position = 0
        entry_price = 0
        trades = []
        
        for i, row in df.iterrows():
            if row['entry'] and position == 0:
                # 开仓
                position_size = self.calculate_position_size(
                    row['close'], row['atr'], capital, risk_pct
                )
                if position_size > 0:
                    position = position_size * row['signal']
                    entry_price = row['close']
                    capital -= position_size * entry_price
                    trades.append({'entry': row['close'], 'size': position_size, 'side': row['signal']})
            
            elif position != 0:
                # 止损/止盈检查
                stop_loss = entry_price - row['atr'] * self.atr_multiplier if position > 0 else entry_price + row['atr'] * self.atr_multiplier
                
                should_exit = False
                if position > 0 and row['low'] <= stop_loss:
                    should_exit = True
                    exit_price = stop_loss
                elif position < 0 and row['high'] >= stop_loss:
                    should_exit = True
                    exit_price = stop_loss
                
                if should_exit:
                    capital += abs(position) * exit_price
                    trades[-1]['exit'] = exit_price
                    trades[-1]['pnl'] = (exit_price - entry_price) * position
                    position = 0
        
        total_pnl = sum(t.get('pnl', 0) for t in trades)
        win_rate = len([t for t in trades if t.get('pnl', 0) > 0]) / max(len(trades), 1)
        
        return {
            'final_capital': capital + position * df.iloc[-1]['close'],
            'total_pnl': total_pnl,
            'num_trades': len(trades),
            'win_rate': win_rate,
            'trades': trades
        }
```

## 应用与局限

### 应用场景

1. **商品期货**：趋势策略在农产品、金属、能源等期货上应用广泛（CTA策略）
2. **全球宏观**：外汇、国债、股指期货的趋势跟踪
3. **组合分散化**：与股票策略低相关性，适合作为组合补充

### 局限性

1. **震荡市表现差**：趋势策略在低波动、震荡市场中频繁止损
2. **滞后性**：移动平均等指标天然滞后，趋势反转时造成较大回撤
3. **参数敏感**：策略表现对参数（周期、止损倍数）敏感
4. **趋势反转风险**：趋势反转时可能遭受巨大亏损
