# 交易风险管理

## 核心概念

交易风险管理（Trading Risk Management）是量化交易系统中保障资本安全、防止极端损失的核心模块。无论策略多么精密，一旦风险控制失效，可能在短时间内造成毁灭性损失。历史上著名的量化灾难——长期资本管理（LTCM，1998）、骑士资本（Knight Capital，2012）——都与风险管理系统失效密切相关。

风险管理涵盖**事前（Pre-trade）**、**事中（Real-time）**、**事后（Post-trade）**三个维度：事前风险检查在订单发送前评估风险敞口；事中风险监控实时追踪持仓与盈亏；事后风险分析用于策略评估与改进。

对于申请CMU/MIT等顶尖项目的学生而言，风险管理不仅关乎资金安全，更是理解金融市场风险本质的重要知识体系。

## 方法论

### 核心风险指标

| 指标 | 定义 | 警戒阈值 |
|------|------|---------|
| VaR（风险价值） | 给定置信水平下的最大损失 | 2% capital/day |
| CVaR/ES | 超过VaR的平均损失 | 3% capital/day |
| 最大回撤 | 峰值到谷底的最大跌幅 | 15-20% |
| 杠杆率 | 总敞口/净资产 | <3x |
| 波动率 | 收益率标准差年化 | <25% |
| 夏普比率 | (超额收益/波动率) | >1.5 |

### 风险管理体系

```
┌─────────────────────────────────────────────────┐
│                   风险委员会                     │
├─────────────────────────────────────────────────┤
│  风控层                                            │
│  ├── 战略级风控（最大敞口、地区限制）                  │
│  ├── 组合级风控（VaR、波动率、杠杆）                   │
│  └── 订单级风控（单笔限额、价格偏离）                  │
├─────────────────────────────────────────────────┤
│  执行层                                            │
│  ├── 断路器（自动停止交易）                          │
│  ├── 仓位监控（实时P&L追踪）                        │
│  └── 流动性管理（成交量限制）                        │
└─────────────────────────────────────────────────┘
```

## 代码实现

```python
import asyncio
from dataclasses import dataclass, field
from typing import Dict, List, Optional
from datetime import datetime
import numpy as np

@dataclass
class RiskLimits:
    max_position_per_symbol: float = 100000     # 单标的最大仓位
    max_portfolio_value: float = 1000000       # 组合最大市值
    max_daily_loss: float = 20000              # 日内最大亏损
    max_leverage: float = 3.0                 # 最大杠杆
    max_volatility: float = 0.25              # 最大年化波动率
    var_confidence: float = 0.95               # VaR置信度
    var_lookback_days: int = 252              # VaR回溯天数

class Position:
    def __init__(self, symbol: str, quantity: float, avg_price: float):
        self.symbol = symbol
        self.quantity = quantity
        self.avg_price = avg_price
        self.current_price = avg_price
    
    @property
    def market_value(self) -> float:
        return abs(self.quantity) * self.current_price
    
    @property
    def unrealized_pnl(self) -> float:
        return (self.current_price - self.avg_price) * self.quantity

class RiskManager:
    def __init__(self, limits: RiskLimits):
        self.limits = limits
        self.positions: Dict[str, Position] = {}
        self.daily_pnl = 0.0
        self.daily_turnover = 0.0
        self.enabled = True
        self._trade_log: List[dict] = []
    
    def update_position(self, symbol: str, quantity: float, price: float):
        if symbol in self.positions:
            pos = self.positions[symbol]
            # 更新平均成本
            new_qty = pos.quantity + quantity
            if new_qty != 0:
                pos.avg_price = (pos.quantity * pos.avg_price + quantity * price) / new_qty
            pos.quantity = new_qty
            pos.current_price = price
        else:
            self.positions[symbol] = Position(symbol, quantity, price)
    
    def update_prices(self, prices: Dict[str, float]):
        for symbol, price in prices.items():
            if symbol in self.positions:
                self.positions[symbol].current_price = price
    
    def update_daily_pnl(self, pnl: float):
        self.daily_pnl += pnl
        if self.daily_pnl < -self.limits.max_daily_loss:
            self.trigger_circuit_breaker(f"Daily loss {self.daily_pnl} exceeds limit")
    
    def check_order_risk(self, symbol: str, quantity: float, price: float, side: str) -> tuple:
        """检查订单风险，返回 (approved: bool, reason: str)"""
        if not self.enabled:
            return True, ""
        
        # 检查交易使能
        if self.daily_pnl < -self.limits.max_daily_loss * 0.5:
            return False, "Daily loss warning threshold reached"
        
        # 检查单标仓位
        current_pos = self.positions.get(symbol)
        current_qty = current_pos.quantity if current_pos else 0
        new_qty = current_qty + quantity
        
        if abs(new_qty) * price > self.limits.max_position_per_symbol:
            return False, f"Position limit exceeded for {symbol}"
        
        # 检查组合总市值
        total_exposure = sum(p.market_value for p in self.positions.values())
        new_exposure = total_exposure + abs(quantity * price)
        
        if new_exposure > self.limits.max_portfolio_value:
            return False, "Portfolio exposure limit exceeded"
        
        # 检查杠杆
        total_equity = self._estimate_equity()
        if total_equity > 0 and new_exposure / total_equity > self.limits.max_leverage:
            return False, "Leverage limit exceeded"
        
        return True, "Approved"
    
    def calculate_var(self, returns: np.ndarray) -> float:
        """计算参数VaR（假设正态分布）"""
        if len(returns) < 2:
            return 0.0
        mu = np.mean(returns)
        sigma = np.std(returns)
        z = 1.645 if self.limits.var_confidence == 0.95 else 2.326  # 95% or 99%
        return mu + z * sigma
    
    def calculate_portfolio_var(self) -> float:
        """计算组合VaR"""
        if len(self._trade_log) < self.limits.var_lookback_days:
            return 0.0
        returns = np.array([t['return'] for t in self._trade_log[-self.limits.var_lookback_days:]])
        return self.calculate_var(returns)
    
    def trigger_circuit_breaker(self, reason: str):
        """触发断路器，停止所有交易"""
        self.enabled = False
        print(f"CIRCUIT BREAKER TRIGGERED: {reason}")
        print(f"Time: {datetime.now()}")
        print(f"All trading halted until manual review")
    
    def _estimate_equity(self) -> float:
        """估算权益（简化版）"""
        unrealized = sum(p.unrealized_pnl for p in self.positions.values())
        return 1000000 + self.daily_pnl + unrealized  # 假设初始资本100万
    
    def get_risk_report(self) -> dict:
        """生成风险报告"""
        total_exposure = sum(p.market_value for p in self.positions.values())
        equity = self._estimate_equity()
        leverage = total_exposure / equity if equity > 0 else 0
        
        return {
            'total_exposure': total_exposure,
            'equity': equity,
            'leverage': leverage,
            'daily_pnl': self.daily_pnl,
            'position_count': len(self.positions),
            'var_95': self.calculate_portfolio_var(),
            'trading_enabled': self.enabled,
            'largest_position': max((p.market_value for p in self.positions.values()), default=0)
        }
```

## 应用与局限

### 应用

1. **系统化风控**：规则驱动的风控减少人为干预，避免情绪化交易决策
2. **实时监控**：秒级风险指标更新，快速响应市场异常
3. **压力测试**：模拟极端市场情景（1987股灾、2008金融危机）评估组合脆弱性

### 局限性

1. **模型风险**：VaR等指标依赖收益率分布假设，黑天鹅事件往往超出模型预测
2. **相关性崩溃**：压力情景下资产相关性急剧上升，分散化效果消失
3. **流动性风险**：VaR不考虑流动性不足时的平仓成本
4. **人机协同**：完全自动化的风控可能错失市场机会，需要动态调整
