# 3.2 文献综述

> **专业视角**：文献综述是研究的起点，也是终点（发表时）。它不是简单的文献堆砌，而是对领域发展的系统性批判性分析。本模块覆盖文献检索策略、批判性阅读方法、文献综述的结构框架、以及如何识别研究gap。

## 4.1 文献检索策略

### 核心数据库

| 数据库 | 覆盖范围 | 特点 |
|--------|---------|------|
| **Google Scholar** | 全学科 | 引用追踪强，适合找高引论文 |
| **PubMed** | 生物医学 | MeSH主题词，专业过滤 |
| **arXiv** | 物理/数学/CS/统计 | 预印本，更新快 |
| **Semantic Scholar** | 全学科 | AI驱动的引用分析 |
| **Web of Science** | 全学科 | 引文索引，筛选高引期刊 |
| **Scopus** | 全学科 | 检索结果更广 |

### 检索词构建

```python
def build_search_query(components, operator="AND"):
    """
    构建系统性检索词
    components: [{term: "machine learning", synonyms: ["ML", "deep learning"]}, ...]
    """
    queries = []
    for comp in components:
        synonyms = [comp["term"]] + comp.get("synonyms", [])
        syn_group = " OR ".join([f'"{s}" OR {s}' for s in synonyms])
        queries.append(f"({syn_group})")
    return f" {operator} ".join(queries)


# 示例：量化金融领域机器学习文献检索
query = build_search_query([
    {"term": "machine learning", "synonyms": ["ML", "deep learning", "neural network"]},
    {"term": "quantitative finance", "synonyms": ["algorithmic trading", "financial prediction"]},
    {"term": "time series", "synonyms": ["forecasting", "returns"]}
], operator="AND")

# 生成: ("machine learning" OR "ML" OR "deep learning" OR "neural network")
# AND ("quantitative finance" OR "algorithmic trading" OR "financial prediction")
# AND ("time series" OR "forecasting" OR "returns")
```

### 检索流程

```
确定研究问题
    ↓
构建检索词（关键词 + 同义词 + MeSH）
    ↓
初步检索（宽泛）
    ↓
筛选标题和摘要（快速浏览）
    ↓
全文评估（关键文献）
    ↓
追踪参考文献（滚雪球）
    ↓
追踪引用该文献的文献（反向滚雪球）
```

## 4.2 批判性阅读框架

### 批判性阅读三问

**1. 作者想解决什么问题？（研究问题）**
- 研究问题是清晰、具体、可检验的吗？
- 研究gap是否被明确指出？

**2. 他们怎么解决的？（方法）**
- 方法是否适合研究问题？
- 样本量/数据量是否足够？
- 是否有对照组/基准对比？
- 可重复性：代码/数据是否公开？

**3. 结论是否被数据支持？（结果）**
- 统计检验是否合理？
- 效应量（Effect Size）是否报告？
- 是否有替代解释（Confound）？
- 在更广泛人群/情境中是否成立？

## 4.3 文献综述结构

### 经典结构（叙事性综述）

```
1. 引言
   - 研究领域的背景
   - 综述的目的和范围
   - 搜索策略（可选）

2. 正文
   - 按主题/时间/方法论组织
   - 每段：先总结主要发现，再进行批判性分析

3. 讨论
   - 综合主要发现
   - 识别研究gap
   - 提出未来研究方向

4. 结论
   - 研究领域的现状
   - 核心共识与争议
```

### 系统性综述（Systematic Review）

```
1. Protocol（方案）
   - 研究问题（PICO格式）
   - 检索策略（预注册）
   - 纳入/排除标准

2. 检索与筛选
   - PRISMA流程图
   - 筛选一致性（Kappa统计）

3. 质量评估
   - Cochrane RoB工具（随机对照试验）
   - STROBE（观察性研究）
   - QUADAS-2（诊断准确性研究）

4. 数据综合
   - 定性综合（叙事分析）
   - 定量综合（Meta分析）

5. 报告
   - PRISMA checklist
```

### PICO格式（医学/临床研究）

| 要素 | 定义 | 示例 |
|------|------|------|
| **P**opulation | 研究对象 | 2型糖尿病患者 |
| **I**ntervention | 干预措施 | 使用强化学习辅助治疗方案 |
| **C**omparison | 对照组 | 传统治疗方案 |
| **O**utcome | 结局指标 | 血糖控制水平 |

import requests

def get_zotero_items(api_key, library_id, library_type='user'):
    """获取Zotero文献库中的条目"""
    headers = {"Zotero-API-Key": api_key}

### 从arXiv生成BibTeX

```python
import arxiv

def fetch_arxiv_bibtex(arxiv_id):
    """获取arXiv论文的BibTeX引用"""
    search = arxiv.Search(id_list=[arxiv_id])
    paper = next(search.results())

    bibtex = f"""@article{{{paper.get_short_id()},
    author = {{{paper.authors[0].name} and others}},
    title = {{{paper.title}}},
    year = {{{paper.published.year}}},
    eprint = {{{paper.entry_id.split('/')[-1]}}},
    archivePrefix = {{arXiv}},
    primaryClass = {{{paper.primary_category}}},
    url = {{{paper.entry_id}}}
}}"""
    return bibtex

# 使用示例
print(fetch_arxiv_bibtex("2301.00001"))
```

## 4.5 识别研究Gap

### Gap类型

| 类型 | 描述 | 例子 |
|------|------|------|
| 方法gap | 现有方法未应用于某领域 | 将强化学习用于期权定价（以前未有人做过） |
| 理论gap | 理论预测未经验证 | STDP规则在特定网络结构下的预测未测试 |
| 应用gap | 技术未转化为实际应用 | 理论模型未在真实市场数据上验证 |
| 情境gap | 方法在新技术/条件下未测试 | 大语言模型在金融文本上的应用 |
| 样本gap | 特定人群/数据未被研究 | 研究主要在发达国家，中国市场未覆盖 |

### Gap分析框架

```
文献A：做了X，用了方法M1
文献B：做了Y，用了方法M2
文献C：做了Z，用了方法M1
文献D：做了X、Y、Z（综述）

Gap识别：
- 没有人用M2方法做X → 方法gap
- X只在发达国家测试， 中国未测试 → 情境gap
- X、Y、Z的结合从未被研究 → 综合gap
```

import numpy as np
from sklearn.manifold import TSNE
from sklearn.cluster import KMeans

def bibliometric_map(papers):
    """
    基于引用关系和关键词的文献地图
    papers: [{title, abstract, citations, keywords}, ...]
    """
    # 1. 构建引用网络
    citation_matrix = build_citation_matrix(papers)

    # 2. 关键词共现矩阵
    keyword_matrix = build_keyword_cooccurrence(papers)

    # 3. t-SNE降维可视化

    # 4. 聚类
