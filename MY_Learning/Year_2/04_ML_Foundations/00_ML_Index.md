# 机器学习基础 · 课程索引

> 本模块为 Year 2 机器学习基础，面向有微积分、线性代数、概率论基础的学习者。
>
> **前置知识**：Year 1 数学（微积分 + 线性代数 + 概率论）
> **后续衔接**：Year 3 量化金融（因子挖掘、组合优化）、计算神经科学（神经-ML接口）

---

## 地图

```
04_ML_Foundations/
├── 1.1_监督学习基础/      回归与分类、损失函数、梯度下降
├── 1.2_无监督学习/         聚类、降维、密度估计
├── 1.3_模型评估与验证/    偏差-方差、过拟合、交叉验证、评估指标
├── 1.4_特征工程/          标准化、编码、特征选择、合成特征
├── 1.5_经典算法/          LR / SVM / 决策树 / RF / KNN / NB
└── 1.6_ML工作流/          Pipeline、数据划分、模型选择、超参调优
```

---

## 学习目标

- [ ] 理解监督/无监督/半监督学习的区别与适用场景
- [ ] 掌握线性回归、逻辑回归的原理与实现
- [ ] 理解模型评估指标体系（AUC/Recall/Precision/F1）
- [ ] 能够独立构建一个完整的 ML Pipeline（数据→特征→模型→评估）
- [ ] 为 Year 3 量化/神经方向的应用做好准备
