## 1. 理论基础

### 1.1 问题的提出

**均值方差组合的问题**：
- 对期望收益估计极其敏感
- 估计误差导致权重不稳定
- 可能导致极度集中的组合

**风险平价的动机**：
> 让每个资产对组合总风险的贡献相等

### 1.2 风险贡献公式

资产 $i$ 对组合的风险贡献：

$$RC_i = w_i \cdot \frac{\partial \sigma_p}{\partial w_i} = \frac{w_i (\boldsymbol{\Sigma} \mathbf{w})_i}{\sigma_p}$$

展开：
$$RC_i = \frac{w_i \sum_{j=1}^{n} \sigma_{ij} w_j}{\sigma_p}$$

### 1.3 风险平价条件

$$\frac{w_1 \sigma_1^2 + w_1 \sigma_{12} + ...}{\sigma_p} = \frac{w_2 \sigma_{21} + w_2 \sigma_2^2 + ...}{\sigma_p} = ... = \frac{\sigma_p}{n}$$

## 2. Python实现

```python
import numpy as np
from scipy.optimize import minimize

def risk_contribution(weights, cov_matrix):
    """计算各资产的风险贡献"""
    portfolio_vol = np.sqrt(weights @ cov_matrix @ weights)
    marginal_contrib = cov_matrix @ weights
    risk_contrib = weights * marginal_contrib / portfolio_vol
    return risk_contrib

def risk_parity_target(target_risk, n_assets):
    """风险平价目标函数"""
    return target_risk / n_assets

def risk_parity_optimization(cov_matrix, risk_budget=None):
    """
    风险平价优化
    
    Parameters:
    -----------
    cov_matrix : np.ndarray
        协方差矩阵
    n_assets : int
        资产数量
    risk_budget : np.ndarray, optional
        风险预算，默认等权
        
    Returns:
    --------
    weights : np.ndarray
        最优权重
    """
    n_assets = len(cov_matrix)
    
    if risk_budget is None:
        risk_budget = np.ones(n_assets) / n_assets
    
    def objective(weights):
        """目标函数：最小化风险贡献与目标偏差"""
        rc = risk_contribution(weights, cov_matrix)
        total_risk = np.sqrt(weights @ cov_matrix @ weights)
        target_rc = risk_budget * total_risk
        return np.sum((rc - target_rc) ** 2)
    
    constraints = [
        {'type': 'eq', 'fun': lambda w: np.sum(w) - 1},  # 权重和为1
        {'type': 'ineq', 'fun': lambda w: w}  # 非负约束
    ]
    
    init_weights = np.ones(n_assets) / n_assets
    
    result = minimize(
        objective,
        init_weights,
        method='SLSQP',
        constraints=constraints
    )
    
    return result.x

# 使用示例
if __name__ == "__main__":
    # 假设三资产协方差矩阵
    cov = np.array([
        [0.0400, 0.0100, 0.0050],
        [0.0100, 0.0900, 0.0150],
        [0.0050, 0.0150, 0.1600]
    ])
    
    weights = risk_parity_optimization(cov)
    rc = risk_contribution(weights, cov)
    total_vol = np.sqrt(weights @ cov @ weights)
    
    print(f"最优权重: {weights}")
    print(f"各资产风险贡献: {rc}")
    print(f"总风险: {total_vol:.4f}")
    print(f"风险贡献占比: {rc / total_vol}")
```

## 3. 风险平价与杠杆

### 3.1 无风险资产假设下的风险平价

风险平价组合通常**不包含无风险资产**，而是使用**杠杆**调整风险：

```
风险平价债券组合:
- 60% 长期国债
- 40% 短期国债
- 使用杠杆放大到目标风险水平
```

### 3.2 全天候策略 (All Weather)

桥水基金的全天候策略基于风险平价思想：

| 资产 | 风险贡献 | 预期角色 |
|------|----------|----------|
| 股票 | 30% | 增长资产 |
| 长期国债 | 40% | 通胀保护 |
| 短期国债 | 15% | 抗衰退 |
| 黄金 | 7.5% | 商品 |
| 大宗商品 | 7.5% | 通胀保护 |

## 4. 与均值方差比较

| 特征 | 均值方差 | 风险平价 |
|------|----------|----------|
| 对收益估计的敏感性 | 高 | 低 |
| 对协方差估计的敏感性 | 高 | 中 |
| 分散化程度 | 取决于参数 | 自然分散 |
| 典型杠杆 | 无/高 | 通常加杠杆 |
| 再平衡频率 | 较低 | 较高 |
