## 1. 期权基础

### 1.1 基本术语

| 术语 | 定义 |
|------|------|
| **Call Option** | 买入期权，赋予持有者在到期日以行权价买入标的资产的权利 |
| **Put Option** | 卖出期权，赋予持有者在到期日以行权价卖出标的资产的权利 |
| **Strike Price (K)** | 行权价，期权合约中规定的买入/卖出价格 |
| **Spot Price (S)** | 标的资产当前市场价格 |
| **Expiration Date** | 到期日 |
| **Premium** | 期权费，买方支付的权利金 |

### 1.2 期权类型

```
期权分类
├── 按权利方向
│   ├── Call (看涨期权)
│   └── Put (看跌期权)
├── 按到期时间
│   ├── American (美式) - 到期日前任何时间可行权
│   └── European (欧式) - 仅到期日可行权
└── 按价值状态
    ├── ITM (In The Money) 实值
    ├── OTM (Out of The Money) 虚值
    └── ATM (At The Money) 平值
```

### 1.3 期权价值状态

| 期权类型 | ITM条件 | ATM条件 | OTM条件 |
|---------|---------|---------|---------|
| Call | S > K | S ≈ K | S < K |
| Put | S < K | S ≈ K | S > K |

## 2. 期权 payoff 图

### 2.1 买入 Call (Long Call)

```python
import numpy as np
import matplotlib.pyplot as plt

def long_call_payoff(S, K, premium):
    """
    买入Call期权的Payoff
    S: 到期日标的价格
    K: 行权价
    premium: 期权费
    """
    payoff = np.maximum(S - K, 0) - premium
    return payoff

# 绘图
K = 100  # 行权价
premium = 5  # 期权费
S_range = np.linspace(70, 130, 100)

payoff = long_call_payoff(S_range, K, premium)

plt.figure(figsize=(10, 6))
plt.plot(S_range, payoff, 'b-', linewidth=2)
plt.axhline(y=0, color='gray', linestyle='--')
plt.axvline(x=K, color='r', linestyle='--', label=f'行权价 K={K}')
plt.xlabel('标的资产价格 S')
plt.ylabel('盈亏')
plt.title(f'买入Call期权 Payoff (K={K}, Premium={premium})')
plt.grid(True, alpha=0.3)
plt.legend()
plt.show()
```

**Payoff 图表**：
```
        盈亏
         │
    25   │         *
         │        /
    15   │       /
         │      /
     5   │* * *--------------  ← 最大亏损 = -premium
         │/
    -5   │
         └────────────────────→ 标的资产价格
              70   100  130
              K    损益点
```

### 2.2 买入 Put (Long Put)

```python
def long_put_payoff(S, K, premium):
    """买入Put期权的Payoff"""
    payoff = np.maximum(K - S, 0) - premium
    return payoff
```

### 2.3 卖出 Call (Short Call)

```python
def short_call_payoff(S, K, premium):
    """卖出Call期权的Payoff"""
    payoff = -np.maximum(S - K, 0) + premium
    return payoff
```

## 3. 期权价格结构

### 3.1 内在价值 vs 时间价值

$$期权价格 = 内在价值 + 时间价值$$

**内在价值 (Intrinsic Value)**：
- Call: $\max(S - K, 0)$
- Put: $\max(K - S, 0)$

**时间价值 (Time Value)**：
$$TimeValue = 期权价格 - 内在价值$$

### 3.2 时间价值的来源

1. **波动性价值**：标的资产价格可能向有利方向移动
2. **利率价值**：资金的时间价值
3. **提前行权价值**（美式期权）：提前获得标的资产的价值

### 3.3 时间价值衰减

```
时间价值
    │
  5 │  ●
    │ ● ●
  4 │●   ●
    │     ●
  3 │       ●
    │         ●
  2 │           ●
    │             ●
  1 │               ●
    │                 ●
  0 │__________________●_______
    0    30   60   90   120  到期日
              时间
```

**关键规律**：时间价值随到期日临近而加速衰减（Theta为负且绝对值递增）

## 4. 期权核心参数 (Greeks概述)

### 4.1 五大希腊字母

| Greek | 定义 | 物理意义 |
|-------|------|----------|
| **Delta (Δ)** | $\frac{\partial V}{\partial S}$ |标的资产价格变化对期权的影响 |
| **Gamma (Γ)** | $\frac{\partial^2 V}{\partial S^2}$ | Delta对标的资产价格的变化率 |
| **Theta (Θ)** | $\frac{\partial V}{\partial t}$ | 时间流逝对期权价值的影响 |
| **Vega (ν)** | $\frac{\partial V}{\partial \sigma}$ | 波动率变化对期权价值的影响 |
| **Rho (ρ)** | $\frac{\partial V}{\partial r}$ | 利率变化对期权价值的影响 |

### 4.2 快速记忆

| 期权类型 | Delta范围 | Gamma | Theta | Vega |
|---------|-----------|-------|-------|------|
| Long Call | (0, 1) | 正 | 负 | 正 |
| Short Call | (-1, 0) | 负 | 正 | 负 |
| Long Put | (-1, 0) | 正 | 负 | 正 |
| Short Put | (0, 1) | 负 | 正 | 负 |

## 5. 基本交易策略

### 5.1 保护性 Put (Protective Put)

**构建**：持有标的资产 + 买入 Put

```python
def protective_put(S, S0, K, premium):
    """
    Protective Put 策略 Payoff
    S0: 买入标的资产价格
    """
    stock_payoff = S - S0
    put_payoff = np.maximum(K - S, 0) - premium
    total = stock_payoff + put_payoff
    return total
```

**用途**：对冲下行风险，类似"保险"

### 5.2 备兑 Call (Covered Call)

**构建**：持有标的资产 + 卖出 Call

**用途**：增加收入（牺牲上行潜力）

### 5.3 价差策略 (Spreads)

| 策略 | 构建 | 预期 |
|------|------|------|
| Bull Call Spread | 买入低行权价Call + 卖出高行权价Call | 温和看涨 |
| Bear Put Spread | 买入高行权价Put + 卖出低行权价Put | 温和看跌 |
| Long Straddle | 买入Call + 买入Put (同K) | 大幅波动 |
| Long Strangle | 买入Call(K高) + 买入Put(K低) | 大幅波动 |

## 6. 期权定价核心思想

### 6.1 定价方法分类

```
期权定价方法
├── 解析方法
│   ├── Black-Scholes-Merton
│   └── Black76 (期货期权)
├── 数值方法
│   ├── 二叉树 (Binomial)
│   ├── 三叉树 (Trinomial)
│   ├── 蒙特卡洛模拟
│   └── 有限差分法
└── 近似方法
    ├── Barone-Adesi Whaley
    └── Bjerksund-Stensland
```

### 6.2 BSM模型核心假设

1. 标的资产价格服从几何布朗运动
2. 无风险利率已知且恒定
3. 市场无摩擦（无交易成本、税收）
4. 标的资产无分红
5. 期权为欧式

### 6.3 BSM定价公式（高级预览）

$$C = S_0 N(d_1) - Ke^{-rT} N(d_2)$$

其中：
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$
$$d_2 = d_1 - \sigma\sqrt{T}$$

**详细推导见 [[Q1.10_Black-Scholes模型]]**
