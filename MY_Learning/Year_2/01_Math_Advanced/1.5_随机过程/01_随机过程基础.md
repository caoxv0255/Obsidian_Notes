# 01-随机过程基础

## 概述

随机过程是研究随时间演进的随机现象的数学工具，在金融工程、信号处理和物理学等领域有广泛应用。本章将系统介绍随机过程的基本概念、分类和性质，为后续学习金融数学和时间序列分析打下基础。

## 1. 随机过程的基本概念

### 1.1 随机过程的定义

**定义**：随机过程是定义在某个概率空间上的随机变量族，通常用$X(t, \omega)$表示，其中：
- $t$：时间参数（连续或离散）
- $\omega$：样本点

**离散时间随机过程**：
$$X = \{X_n, n \in \mathbb{Z}^+\}$$

**连续时间随机过程**：
$$X = \{X(t), t \in [0, \infty)\}$$

### 1.2 随机过程的分类

#### 按时间参数分类

**离散时间过程**：
- 时间点：$t = 0, 1, 2, \ldots$
- 例子：ARIMA模型、GARCH模型

**连续时间过程**：
- 时间点：$t \in [0, \infty)$
- 例子：布朗运动、泊松过程

#### 按状态空间分类

**离散状态过程**：
- 状态空间：有限或可数集合
- 例子：马尔可夫链、泊松过程

**连续状态过程**：
- 状态空间：连续集合
- 例子：布朗运动、几何布朗运动

#### 按性质分类

**平稳过程**：
- 统计性质不随时间改变
- 严平稳：联合分布不随时间改变
- 宽平稳：均值、方差和自协方差不随时间改变

**非平稳过程**：
- 统计性质随时间改变
- 例子：趋势过程、季节性过程

## 2. 随机过程的性质

### 2.1 期望和方差

#### 期望函数

**定义**：随机过程的期望函数为：
$$m_X(t) = E[X(t)]$$

**性质**：
- $m_X(t)$是时间的函数
- 描述过程的平均水平

#### 方差函数

**定义**：随机过程的方差函数为：
$$\gamma_X(t,t) = \text{Var}[X(t)] = E[(X(t) - m_X(t))^2]$$

**性质**：
- $\gamma_X(t,t) \geq 0$
- 描述过程的波动性

### 2.2 自协方差函数

**定义**：随机过程的自协方差函数为：
$$\gamma_X(t_1, t_2) = E[(X(t_1) - m_X(t_1))(X(t_2) - m_X(t_2))]$$

**性质**：
- $\gamma_X(t_1, t_2) = \gamma_X(t_2, t_1)$（对称性）
- $\gamma_X(t, t) = \gamma_X(t,t)$（方差函数）

### 2.3 自相关函数

**定义**：随机过程的自相关函数为：
$$\rho_X(t_1, t_2) = \frac{\gamma_X(t_1, t_2)}{\sqrt{\gamma_X(t_1, t_1)\gamma_X(t_2, t_2)}}$$

**性质**：
- $-1 \leq \rho_X(t_1, t_2) \leq 1$
- $\rho_X(t, t) = 1$

## 3. 常见随机过程

### 3.1 泊松过程

#### 泊松过程的定义

**泊松过程**是计数过程，满足：
1. $N(0) = 0$
2. 独立增量：$N(t_2) - N(t_1)$与$N(t_1) - N(t_0)$独立
3. 平稳增量：$N(t_2) - N(t_1)$仅依赖于$t_2 - t_1$
4. 泊松分布：$N(t) \sim P(\lambda t)$

#### 泊松过程的性质

**概率质量函数**：
$$P(N(t) = k) = \frac{(\lambda t)^k}{k!}e^{-\lambda t}$$

**期望和方差**：
$$E[N(t)] = \text{Var}[N(t)] = \lambda t$$

**到达时间**：
- 到达时间间隔：$T_i \sim \text{Exp}(\lambda)$
- 第$n$个到达时间：$S_n = \sum_{i=1}^n T_i \sim \text{Gamma}(n, \lambda)$

#### 应用

**排队论**：顾客到达
**可靠性理论**：故障发生
**金融工程**：交易次数

### 3.2 马尔可夫链

#### 马尔可夫链的定义

**马尔可夫链**是具有马尔可夫性质的随机过程：
$$P(X_{n+1} = j | X_n = i, X_{n-1} = i_{n-1}, \ldots, X_0 = i_0) = P(X_{n+1} = j | X_n = i)$$

#### 转移概率

**转移概率**：$p_{ij} = P(X_{n+1} = j | X_n = i)$

**转移概率矩阵**：
$$P = \begin{pmatrix}
p_{11} & p_{12} & \cdots & p_{1n} \\
p_{21} & p_{22} & \cdots & p_{2n} \\
\vdots & \vdots & \ddots & \vdots \\
p_{n1} & p_{n2} & \cdots & p_{nn}
\end{pmatrix}$$

#### 初值分布

**初始分布**：$\pi = (\pi_1, \pi_2, \ldots, \pi_n)$，其中$\pi_i = P(X_0 = i)$

#### $n$步转移概率

** Chapman-Kolmogorov方程**：
$$P^{(n)} = P^n$$

其中$P^{(n)} = (p_{ij}^{(n)})$，$p_{ij}^{(n)} = P(X_n = j | X_0 = i)$

#### 稳态分布

**平稳分布**：$\pi = \pi P$，且$\sum_{i=1}^n \pi_i = 1$

**渐近性质**：
- 如果链是不可约且正常返的，则存在唯一的平稳分布
- $\lim_{n \to \infty} P^{(n)} = \pi$

### 3.3 布朗运动

#### 布朗运动的定义

**布朗运动**（Wiener过程）是连续时间随机过程，满足：
1. $W(0) = 0$
2. 独立增量：$W(t_2) - W(t_1)$与$W(t_1) - W(t_0)$独立
3. 正态增量：$W(t) - W(s) \sim N(0, t-s)$
4. 路径连续：$W(t)$是连续函数

#### 布朗运动的性质

**期望和方差**：
$$E[W(t)] = 0, \quad \text{Var}[W(t)] = t$$

**自相关函数**：
$$E[W(s)W(t)] = \min(s, t)$$

**独立增量**：
$$E[(W(t_2) - W(t_1))(W(t_4) - W(t_3))] = 0$$

#### 应用

**金融工程**：几何布朗运动
**物理学**：粒子运动
**随机微积分**：伊藤积分

### 3.4 几何布朗运动

#### 几何布朗运动的定义

**几何布朗运动**是满足以下SDE的随机过程：
$$dS_t = \mu S_t dt + \sigma S_t dW_t$$

其中：
- $S_t$：股票价格
- $\mu$：漂移系数
- $\sigma$：波动率
- $W_t$：布朗运动

#### 解

**解为**：
$$S_t = S_0 \exp\left(\left(\mu - \frac{\sigma^2}{2}\right)t + \sigma W_t\right)$$

#### 性质

**期望**：
$$E[S_t] = S_0 e^{\mu t}$$

**方差**：
$$\text{Var}[S_t] = S_0^2 e^{2\mu t}(e^{\sigma^2 t} - 1)$$

**对数正态分布**：
$$\ln S_t \sim N\left(\left(\mu - \frac{\sigma^2}{2}\right)t, \sigma^2 t\right)$$

#### 应用

**Black-Scholes模型**：期权定价
**风险模型**：资产定价

### 4.1 泊松过程模拟

```python
import numpy as np
import matplotlib.pyplot as plt

class PoissonProcess:
    """泊松过程类"""
    
    def __init__(self, rate=1.0):
        """初始化"""
        self.rate = rate
    
    def generate_arrival_times(self, T):
        """生成到达时间"""
        arrival_times = []
        current_time = 0
        
        while current_time < T:
            # 生成指数分布的到达间隔
            interval = np.random.exponential(1/self.rate)
            current_time += interval
            if current_time <= T:
                arrival_times.append(current_time)
        
        return np.array(arrival_times)
    
    def generate_counts(self, T, num_simulations=1000):
        """生成计数过程"""
        counts = []
        for _ in range(num_simulations):
            arrival_times = self.generate_arrival_times(T)
            counts.append(len(arrival_times))
        
        return np.array(counts)
    
    def simulate(self, T, num_paths=5):
        """模拟过程"""
        fig, ax = plt.subplots(figsize=(10, 6))
        
        for i in range(num_paths):
            arrival_times = self.generate_arrival_times(T)
            
            # 绘制过程
            times = np.concatenate([[0], arrival_times, [T]])
            counts = np.concatenate([[0], np.cumsum(np.ones_like(arrival_times)), [len(arrival_times)]])
            
            ax.step(times, counts, label=f'路径{i+1}')
        
        ax.set_xlabel('时间')
        ax.set_ylabel('到达次数')
        ax.set_title(f'泊松过程模拟 (λ={self.rate})')
        ax.legend()
        ax.grid(True)
        
        plt.show()

# 使用示例
if __name__ == "__main__":
    # 创建泊松过程对象
    pp = PoissonProcess(rate=2.0)
    
    # 生成到达时间
    T = 10
    arrival_times = pp.generate_arrival_times(T)
    print(f"到达时间: {arrival_times}")
    
    # 生成计数过程
    counts = pp.generate_counts(T, 1000)
    print(f"平均到达次数: {np.mean(counts):.2f}")
    print(f"到达次数标准差: {np.std(counts):.2f}")
    
    # 模拟过程
    pp.simulate(T, 5)
```

### 4.2 马尔可夫链模拟

```python
class MarkovChain:
    """马尔可夫链类"""
    
    def __init__(self, transition_matrix, initial_state=0):
        """初始化"""
        self.P = np.array(transition_matrix)
        self.current_state = initial_state
        self.states = list(range(len(transition_matrix)))
    
    def step(self):
        """一步转移"""
        probabilities = self.P[self.current_state]
        self.current_state = np.random.choice(self.states, p=probabilities)
        return self.current_state
    
    def simulate(self, num_steps):
        """模拟过程"""
        states = [self.current_state]
        for _ in range(num_steps):
            states.append(self.step())
        return np.array(states)
    
    def simulate_multiple(self, num_paths, num_steps):
        """模拟多条路径"""
        paths = []
        for _ in range(num_paths):
            path = self.simulate(num_steps)
            paths.append(path)
        return np.array(paths)
    
    def get_stationary_distribution(self, num_steps=10000):
        """估计平稳分布"""
        states = self.simulate(num_steps)
        unique, counts = np.unique(states, return_counts=True)
        return counts / num_steps

# 使用示例
if __name__ == "__main__":
    # 定义转移矩阵
    P = [[0.7, 0.3],
         [0.4, 0.6]]
    
    # 创建马尔可夫链对象
    mc = MarkovChain(P, initial_state=0)
    
    # 模拟过程
    states = mc.simulate(100)
    print(f"状态序列: {states}")
    
    # 估计平稳分布
    stationary_dist = mc.get_stationary_distribution(10000)
    print(f"估计的平稳分布: {stationary_dist}")
```

### 4.3 布朗运动模拟

```python
class BrownianMotion:
    """布朗运动类"""
    
    def __init__(self, T=1.0, num_steps=1000):
        """初始化"""
        self.T = T
        self.num_steps = num_steps
        self.dt = T / num_steps
        self.times = np.linspace(0, T, num_steps + 1)
    
    def generate_path(self):
        """生成单条路径"""
        # 生成增量
        increments = np.random.normal(0, np.sqrt(self.dt), self.num_steps)
        
        # 累积路径
        paths = np.concatenate([[0], np.cumsum(increments)])
        
        return self.times, paths
    
    def simulate_multiple(self, num_paths=5):
        """模拟多条路径"""
        fig, ax = plt.subplots(figsize=(10, 6))
        
        for i in range(num_paths):
            times, path = self.generate_path()
            ax.plot(times, path, label=f'路径{i+1}')
        
        ax.set_xlabel('时间')
        ax.set_ylabel('位置')
        ax.set_title('布朗运动模拟')
        ax.legend()
        ax.grid(True)
        
        plt.show()
    
    def simulate_with_mean(self, num_paths=5):
        """模拟带均值的布朗运动"""
        fig, ax = plt.subplots(figsize=(10, 6))
        
        for i in range(num_paths):
            times, path = self.generate_path()
            ax.plot(times, path, label=f'路径{i+1}')
        
        # 添加均值线
        ax.axhline(y=0, color='r', linestyle='--', label='均值')
        
        ax.set_xlabel('时间')
        ax.set_ylabel('位置')
        ax.set_title('布朗运动模拟 (带均值)')
        ax.legend()
        ax.grid(True)
        
        plt.show()

# 使用示例
if __name__ == "__main__":
    # 创建布朗运动对象
    bm = BrownianMotion(T=1.0, num_steps=1000)
    
    # 生成单条路径
    times, path = bm.generate_path()
    print(f"时间点数: {len(times)}")
    print(f"路径起点: {path[0]}")
    print(f"路径终点: {path[-1]}")
    
    # 模拟多条路径
    bm.simulate_multiple(5)
    
    # 模拟带均值的路径
    bm.simulate_with_mean(5)
```

## 5. 随机过程在金融中的应用

### 5.1 Black-Scholes模型

**Black-Scholes模型**是期权定价的经典模型，假设股票价格服从几何布朗运动：

**模型假设**：
1. 无风险利率$\mu$为常数
2. 波动率$\sigma$为常数
3. 无交易成本
4. 无套利机会

**期权定价**：
- **看涨期权**：$C = S_0 N(d_1) - Ke^{-rT} N(d_2)$
- **看跌期权**：$P = Ke^{-rT} N(-d_2) - S_0 N(-d_1)$

其中：
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$$
$$d_2 = d_1 - \sigma\sqrt{T}$$

### 5.2 GARCH模型

**GARCH模型**（广义自回归条件异方差）用于建模金融时间序列的波动率：

**模型形式**：
$$\sigma_t^2 = \omega + \alpha_1 \epsilon_{t-1}^2 + \beta_1 \sigma_{t-1}^2$$

其中：
- $\sigma_t^2$：条件方差
- $\epsilon_t$：残差
- $\omega, \alpha_1, \beta_1$：参数

### 5.3 随机微积分

**伊藤积分**：
$$\int_0^T f(t) dW_t = \lim_{n \to \infty} \sum_{i=0}^{n-1} f(t_i)(W_{t_{i+1}} - W_{t_i})$$

**伊藤引理**：
$$df(X_t) = f'(X_t)dX_t + \frac{1}{2}f''(X_t)(dX_t)^2$$

## 6. Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

class FinancialModels:
    """金融模型类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def black_scholes_call(self, S, K, T, r, sigma):
        """Black-Scholes看涨期权定价"""
        d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
        d2 = d1 - sigma*np.sqrt(T)
        
        call = S*norm.cdf(d1) - K*np.exp(-r*T)*norm.cdf(d2)
        return call
    
    def black_scholes_put(self, S, K, T, r, sigma):
        """Black-Scholes看跌期权定价"""
        d1 = (np.log(S/K) + (r + 0.5*sigma**2)*T) / (sigma*np.sqrt(T))
        d2 = d1 - sigma*np.sqrt(T)
        
        put = K*np.exp(-r*T)*norm.cdf(-d2) - S*norm.cdf(-d1)
        return put
    
    def garch_simulation(self, omega, alpha, beta, T, num_paths=1000):
        """GARCH模型模拟"""
        sigma = np.zeros(T)
        sigma[0] = 1.0
        
        for i in range(1, T):
            sigma[i] = np.sqrt(omega + alpha * np.random.normal(0, 1)**2 + beta * sigma[i-1]**2)
        
        return sigma

class RandomProcessSimulator:
    """随机过程模拟器"""
    
    def __init__(self):
        """初始化"""
        self.models = FinancialModels()
    
    def simulate_geometric_brownian_motion(self, S0, mu, sigma, T, num_steps=1000):
        """几何布朗运动模拟"""
        dt = T / num_steps
        times = np.linspace(0, T, num_steps + 1)
        
        # 生成布朗运动增量
        increments = np.random.normal(0, np.sqrt(dt), num_steps)
        W = np.concatenate([[0], np.cumsum(increments)])
        
        # 计算几何布朗运动
        S = S0 * np.exp((mu - 0.5*sigma**2)*times + sigma*W)
        
        return times, S
    
    def plot_geometric_brownian_motion(self, num_paths=5):
        """绘制几何布朗运动"""
        fig, ax = plt.subplots(figsize=(10, 6))
        
        for i in range(num_paths):
            times, S = self.simulate_geometric_brownian_motion(
                S0=100, mu=0.05, sigma=0.2, T=1.0
            )
            ax.plot(times, S, label=f'路径{i+1}')
        
        ax.set_xlabel('时间')
        ax.set_ylabel('价格')
        ax.set_title('几何布朗运动模拟')
        ax.legend()
        ax.grid(True)
        
        plt.show()
    
    def option_pricing_example(self):
        """期权定价示例"""
        print("=== Black-Scholes期权定价 ===")
        
        # 参数
        S = 100  # 股票价格
        K = 100  # 行权价格
        T = 1    # 到期时间
        r = 0.05 # 无风险利率
        sigma = 0.2 # 波动率
        
        # 定价
        call_price = self.models.black_scholes_call(S, K, T, r, sigma)
        put_price = self.models.black_scholes_put(S, K, T, r, sigma)
        
        print(f"看涨期权价格: {call_price:.4f}")
        print(f"看跌期权价格: {put_price:.4f}")
        
        # 画图
        K_values = np.linspace(50, 150, 50)
        call_prices = [self.models.black_scholes_call(S, K, T, r, sigma) for K in K_values]
        put_prices = [self.models.black_scholes_put(S, K, T, r, sigma) for K in K_values]
        
        plt.figure(figsize=(10, 6))
        plt.plot(K_values, call_prices, label='看涨期权')
        plt.plot(K_values, put_prices, label='看跌期权')
        plt.axvline(x=S, color='r', linestyle='--', label='股票价格')
        plt.xlabel('行权价格')
        plt.ylabel('期权价格')
        plt.title('期权价格曲线')
        plt.legend()
        plt.grid(True)
        plt.show()

# 使用示例
if __name__ == "__main__":
    # 创建模拟器
    simulator = RandomProcessSimulator()
    
    # 几何布朗运动模拟
    simulator.plot_geometric_brownian_motion(5)
    
    # 期权定价
    simulator.option_pricing_example()
    
    # GARCH模型模拟
    garch = FinancialModels()
    sigma = garch.garch_simulation(omega=0.01, alpha=0.1, beta=0.8, T=100)
    print(f"GARCH模拟 - 平均波动率: {np.mean(sigma):.4f}")
```

## 7. 常见问题

### Q1: 什么是随机过程？

**A**: 随机过程是定义在概率空间上的随机变量族，用于描述随时间演进的随机现象。

### Q2: 泊松过程和布朗运动有什么区别？

**A**: 泊松过程是离散状态的计数过程，布朗运动是连续状态的路径过程。

### Q3: 马尔可夫链的马尔可夫性质是什么？

**A**: 马尔可夫性质是指未来状态仅依赖于当前状态，与过去历史无关。
