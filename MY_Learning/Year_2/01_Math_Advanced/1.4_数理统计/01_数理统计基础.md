# 01-数理统计基础

## 概述

数理统计是概率论与实际应用之间的桥梁，是量化金融和计算神经科学的重要数学基础。本章将系统介绍数理统计的基本概念、理论和方法，为后续学习统计推断和机器学习打下坚实基础。

## 1. 数理统计的基本概念

### 1.1 总体与样本

#### 总体（Population）

**定义**：研究对象的全体称为总体。

**特点**：
- 研究对象的集合
- 包含所有个体
- 通常不可完全观察

**分类**：
- 有限总体：个体数量有限
- 无限总体：个体数量无限

#### 样本（Sample）

**定义**：从总体中抽取的一部分个体称为样本。

**特点**：
- 随机抽取
- 可观测
- 代表性

**样本容量**：样本中个体的数量。

### 1.2 统计量

#### 统计量的定义

**定义**：不包含未知参数的样本函数称为统计量。

**特点**：
- 样本的函数
- 不含未知参数
- 可观测

#### 常见统计量

**样本均值**：
$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$$

**样本方差**：
$$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$$

**样本标准差**：
$$S = \sqrt{S^2}$$

**样本中位数**：将样本按大小顺序排列，中间的数值。

**样本极差**：
$$R = \max(X_1, X_2, \ldots, X_n) - \min(X_1, X_2, \ldots, X_n)$$

## 2. 抽样分布

### 2.1 抽样分布的定义

**定义**：统计量的分布称为抽样分布。

### 2.2 常见抽样分布

#### 正态分布

**定义**：若随机变量X服从正态分布，则记为$X \sim N(\mu, \sigma^2)$。

**概率密度函数**：
$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

**性质**：
- 对称分布
- 均值为$\mu$，方差为$\sigma^2$
- 标准正态分布：$Z \sim N(0,1)$

#### 卡方分布

**定义**：若$Z_1, Z_2, \ldots, Z_n$相互独立且$Z_i \sim N(0,1)$，则：
$$\chi^2 = \sum_{i=1}^n Z_i^2 \sim \chi^2(n)$$

**性质**：
- 自由度为$n$
- 均值为$n$
- 方差为$2n$

#### t分布

**定义**：若$Z \sim N(0,1)$，$U \sim \chi^2(n)$，且$Z$与$U$独立，则：
$$T = \frac{Z}{\sqrt{U/n}} \sim t(n)$$

**性质**：
- 自由度为$n$
- 对称分布
- 当$n \to \infty$时，$t(n) \to N(0,1)$

#### F分布

**定义**：若$U \sim \chi^2(n_1)$，$V \sim \chi^2(n_2)$，且$U$与$V$独立，则：
$$F = \frac{U/n_1}{V/n_2} \sim F(n_1, n_2)$$

**性质**：
- 自由度为$(n_1, n_2)$
- 均值为$\frac{n_2}{n_2-2}$（当$n_2 > 2$时）
- 方差为$\frac{2n_2^2(n_1+n_2-2)}{n_1(n_2-2)^2(n_2-4)}$（当$n_2 > 4$时）

## 3. 参数估计

### 3.1 点估计

#### 矩估计法

**原理**：用样本矩估计总体矩。

**步骤**：
1. 求总体的$k$阶矩
2. 用样本$k$阶矩替换
3. 解方程组得到参数估计

**示例**：
设$X_1, X_2, \ldots, X_n \sim N(\mu, \sigma^2)$，求$\mu$和$\sigma^2$的矩估计。

- $E(X) = \mu$，用$\bar{X}$估计$\mu$
- $E(X^2) = \mu^2 + \sigma^2$，用$\frac{1}{n}\sum_{i=1}^n X_i^2$估计$\mu^2 + \sigma^2$

#### 最大似然估计法

**定义**：使似然函数最大的参数值称为最大似然估计。

**似然函数**：
$$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$$

**对数似然函数**：
$$\ell(\theta) = \ln L(\theta) = \sum_{i=1}^n \ln f(x_i; \theta)$$

**步骤**：
1. 构造似然函数
2. 取对数
3. 求导并令导数为0
4. 解方程得到$\hat{\theta}$

### 3.2 区间估计

#### 置信区间

**定义**：设$\theta$为总体参数，若随机区间$(\hat{\theta}_1, \hat{\theta}_2)$满足：
$$P(\hat{\theta}_1 \leq \theta \leq \hat{\theta}_2) = 1 - \alpha$$
则称$(\hat{\theta}_1, \hat{\theta}_2)$为$\theta$的$1-\alpha$置信区间。

#### 常见置信区间

**正态总体均值置信区间**（已知$\sigma^2$）：
$$\bar{X} \pm z_{\alpha/2} \frac{\sigma}{\sqrt{n}}$$

**正态总体均值置信区间**（未知$\sigma^2$）：
$$\bar{X} \pm t_{\alpha/2}(n-1) \frac{S}{\sqrt{n}}$$

**正态总体方差置信区间**：
$$\left(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right)$$

## 4. 假设检验

### 4.1 假设检验的基本概念

#### 原假设与备择假设

**原假设**（$H_0$）：通常表示无差异或无效应的假设。

**备择假设**（$H_1$）：与原假设对立的假设。

#### 假设检验的类型

**双侧检验**：
- $H_0: \theta = \theta_0$
- $H_1: \theta \neq \theta_0$

**单侧检验**：
- 左侧检验：$H_0: \theta \geq \theta_0$，$H_1: \theta < \theta_0$
- 右侧检验：$H_0: \theta \leq \theta_0$，$H_1: \theta > \theta_0$

#### 检验统计量与拒绝域

**检验统计量**：用于检验假设的统计量。

**拒绝域**：拒绝原假设的区域。

**接受域**：接受原假设的区域。

### 4.2 常见假设检验

#### 正态总体均值检验

**已知$\sigma^2$**：
- 检验统计量：$Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$
- 拒绝域：$|Z| > z_{\alpha/2}$

**未知$\sigma^2$**：
- 检验统计量：$T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$
- 拒绝域：$|T| > t_{\alpha/2}(n-1)$

#### 正态总体方差检验

**检验统计量**：
$$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}$$

**拒绝域**：
- 左侧检验：$\chi^2 > \chi^2_{1-\alpha}(n-1)$
- 右侧检验：$\chi^2 < \chi^2_{\alpha}(n-1)$
- 双侧检验：$\chi^2 < \chi^2_{\alpha/2}(n-1)$或$\chi^2 > \chi^2_{1-\alpha/2}(n-1)$

#### 两个正态总体方差比检验

**检验统计量**：
$$F = \frac{S_1^2}{S_2^2}$$

**拒绝域**：
- $F > F_{\alpha/2}(n_1-1, n_2-1)$
- $F < F_{1-\alpha/2}(n_1-1, n_2-1)$

## 5. 回归分析

### 5.1 一元线性回归

#### 模型建立

**模型**：
$$Y_i = \beta_0 + \beta_1 X_i + \epsilon_i$$

其中：
- $Y_i$：因变量
- $X_i$：自变量
- $\beta_0$：截距
- $\beta_1$：斜率
- $\epsilon_i$：随机误差

#### 参数估计

**最小二乘估计**：
$$\hat{\beta}_1 = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sum_{i=1}^n (X_i - \bar{X})^2}$$
$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1 \bar{X}$$

#### 模型检验

**相关系数**：
$$r = \frac{\sum_{i=1}^n (X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum_{i=1}^n (X_i - \bar{X})^2 \sum_{i=1}^n (Y_i - \bar{Y})^2}}$$

**回归方程的显著性检验**：
- $F$检验：检验整个回归方程的显著性
- $t$检验：检验回归系数的显著性

### 5.2 多元线性回归

#### 模型建立

**模型**：
$$Y_i = \beta_0 + \beta_1 X_{i1} + \beta_2 X_{i2} + \cdots + \beta_p X_{ip} + \epsilon_i$$

#### 参数估计

**矩阵形式**：
$$\hat{\beta} = (X^T X)^{-1} X^T Y$$

#### 模型检验

**决定系数**：
$$R^2 = 1 - \frac{SSE}{SST}$$

**调整的决定系数**：
$$R^2_{adj} = 1 - \frac{SSE/(n-p-1)}{SST/(n-1)}$$

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from scipy import stats
from scipy.stats import t, chi2, f
import statsmodels.api as sm
from statsmodels.formula.api import ols

class StatisticalInference:
    """统计推断类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def sample_mean(self, data):
        """计算样本均值"""
        return np.mean(data)
    
    def sample_variance(self, data):
        """计算样本方差"""
        return np.var(data, ddof=1)
    
    def sample_std(self, data):
        """计算样本标准差"""
        return np.std(data, ddof=1)
    
    def sample_median(self, data):
        """计算样本中位数"""
        return np.median(data)
    
    def sample_range(self, data):
        """计算样本极差"""
        return np.max(data) - np.min(data)
    
    def confidence_interval_mean(self, data, confidence=0.95):
        """计算均值置信区间"""
        n = len(data)
        mean = np.mean(data)
        std = np.std(data, ddof=1)
        
        # t分布临界值
        alpha = 1 - confidence
        t_critical = t.ppf(1 - alpha/2, n-1)
        
        margin_error = t_critical * std / np.sqrt(n)
        
        return (mean - margin_error, mean + margin_error)
    
    def confidence_interval_variance(self, data, confidence=0.95):
        """计算方差置信区间"""
        n = len(data)
        var = np.var(data, ddof=1)
        
        alpha = 1 - confidence
        chi2_lower = chi2.ppf(alpha/2, n-1)
        chi2_upper = chi2.ppf(1 - alpha/2, n-1)
        
        lower = (n-1) * var / chi2_upper
        upper = (n-1) * var / chi2_lower
        
        return (lower, upper)
    
    def hypothesis_test_mean(self, data, mu0, alternative='two-sided', alpha=0.05):
        """均值假设检验"""
        n = len(data)
        mean = np.mean(data)
        std = np.std(data, ddof=1)
        
        # t统计量
        t_stat = (mean - mu0) / (std / np.sqrt(n))
        
        # p值
        if alternative == 'two-sided':
            p_value = 2 * (1 - t.cdf(abs(t_stat), n-1))
        elif alternative == 'greater':
            p_value = 1 - t.cdf(t_stat, n-1)
        else:
            p_value = t.cdf(t_stat, n-1)
        
        # 拒绝域
        reject = p_value < alpha
        
        return {
            't_statistic': t_stat,
            'p_value': p_value,
            'reject': reject,
            'critical_value': t.ppf(1 - alpha/2, n-1) if alternative == 'two-sided' else t.ppf(1 - alpha, n-1)
        }
    
    def regression_analysis(self, X, y):
        """回归分析"""
        # 添加常数项
        X = sm.add_constant(X)
        
        # 拟合模型
        model = sm.OLS(y, X).fit()
        
        return model

class RegressionAnalysis:
    """回归分析类"""
    
    def __init__(self):
        """初始化"""
        self.inference = StatisticalInference()
    
    def simple_linear_regression(self, x, y):
        """简单线性回归"""
        n = len(x)
        
        # 计算回归系数
        x_mean = np.mean(x)
        y_mean = np.mean(y)
        
        numerator = np.sum((x - x_mean) * (y - y_mean))
        denominator = np.sum((x - x_mean) ** 2)
        
        beta1 = numerator / denominator
        beta0 = y_mean - beta1 * x_mean
        
        # 计算拟合值
        y_pred = beta0 + beta1 * x
        
        # 计算残差
        residuals = y - y_pred
        
        # 计算R平方
        ss_res = np.sum(residuals ** 2)
        ss_tot = np.sum((y - y_mean) ** 2)
        r_squared = 1 - (ss_res / ss_tot)
        
        # 计算标准误差
        mse = ss_res / (n - 2)
        var_beta1 = mse / denominator
        var_beta0 = mse * (1/n + x_mean**2 / denominator)
        
        se_beta1 = np.sqrt(var_beta1)
        se_beta0 = np.sqrt(var_beta0)
        
        # t统计量
        t_beta1 = beta1 / se_beta1
        t_beta0 = beta0 / se_beta0
        
        # p值
        p_beta1 = 2 * (1 - t.cdf(abs(t_beta1), n-2))
        p_beta0 = 2 * (1 - t.cdf(abs(t_beta0), n-2))
        
        return {
            'coefficients': {'intercept': beta0, 'slope': beta1},
            'r_squared': r_squared,
            'mse': mse,
            'se': {'intercept': se_beta0, 'slope': se_beta1},
            't_stats': {'intercept': t_beta0, 'slope': t_beta1},
            'p_values': {'intercept': p_beta0, 'slope': p_beta1},
            'y_pred': y_pred,
            'residuals': residuals
        }
    
    def multiple_linear_regression(self, X, y):
        """多元线性回归"""
        # 添加常数项
        X = np.column_stack([np.ones(len(X)), X])
        
        # 计算回归系数
        beta = np.linalg.inv(X.T @ X) @ X.T @ y
        
        # 计算拟合值
        y_pred = X @ beta
        
        # 计算残差
        residuals = y - y_pred
        
        # 计算R平方
        ss_res = np.sum(residuals ** 2)
        ss_tot = np.sum((y - np.mean(y)) ** 2)
        r_squared = 1 - (ss_res / ss_tot)
        
        # 计算调整的R平方
        n = len(y)
        p = X.shape[1] - 1
        r_squared_adj = 1 - (1 - r_squared) * (n - 1) / (n - p - 1)
        
        # 计算方差-协方差矩阵
        mse = ss_res / (n - p - 1)
        var_beta = mse * np.linalg.inv(X.T @ X)
        
        # 标准误差
        se_beta = np.sqrt(np.diag(var_beta))
        
        # t统计量
        t_stats = beta / se_beta
        
        # p值
        p_values = 2 * (1 - t.cdf(np.abs(t_stats), n - p - 1))
        
        return {
            'coefficients': beta,
            'r_squared': r_squared,
            'r_squared_adj': r_squared_adj,
            'mse': mse,
            'se': se_beta,
            't_stats': t_stats,
            'p_values': p_values,
            'y_pred': y_pred,
            'residuals': residuals
        }

class HypothesisTesting:
    """假设检验类"""
    
    def __init__(self):
        """初始化"""
        self.inference = StatisticalInference()
    
    def t_test(self, x, mu0, alternative='two-sided', alpha=0.05):
        """t检验"""
        n = len(x)
        x_mean = np.mean(x)
        s = np.std(x, ddof=1)
        
        # t统计量
        t_stat = (x_mean - mu0) / (s / np.sqrt(n))
        
        # p值
        if alternative == 'two-sided':
            p_value = 2 * (1 - t.cdf(abs(t_stat), n-1))
        elif alternative == 'greater':
            p_value = 1 - t.cdf(t_stat, n-1)
        else:
            p_value = t.cdf(t_stat, n-1)
        
        # 拒绝域
        reject = p_value < alpha
        
        return {
            't_statistic': t_stat,
            'p_value': p_value,
            'reject': reject,
            'critical_value': t.ppf(1 - alpha/2, n-1) if alternative == 'two-sided' else t.ppf(1 - alpha, n-1),
            'alternative': alternative
        }
    
    def chi2_test(self, x, var0, alternative='two-sided', alpha=0.05):
        """卡方检验"""
        n = len(x)
        s2 = np.var(x, ddof=1)
        
        # 卡方统计量
        chi2_stat = (n - 1) * s2 / var0
        
        # p值
        if alternative == 'two-sided':
            p_value = 2 * min(chi2.cdf(chi2_stat, n-1), 1 - chi2.cdf(chi2_stat, n-1))
        elif alternative == 'greater':
            p_value = 1 - chi2.cdf(chi2_stat, n-1)
        else:
            p_value = chi2.cdf(chi2_stat, n-1)
        
        # 拒绝域
        reject = p_value < alpha
        
        return {
            'chi2_statistic': chi2_stat,
            'p_value': p_value,
            'reject': reject,
            'critical_value': chi2.ppf(1 - alpha/2, n-1) if alternative == 'two-sided' else chi2.ppf(1 - alpha, n-1),
            'alternative': alternative
        }
    
    def f_test(self, x1, x2, alpha=0.05):
        """F检验"""
        n1 = len(x1)
        n2 = len(x2)
        s12 = np.var(x1, ddof=1)
        s22 = np.var(x2, ddof=1)
        
        # F统计量
        f_stat = s12 / s22
        
        # p值
        p_value = 2 * min(f.cdf(f_stat, n1-1, n2-1), 1 - f.cdf(f_stat, n1-1, n2-1))
        
        # 拒绝域
        reject = p_value < alpha
        
        return {
            'f_statistic': f_stat,
            'p_value': p_value,
            'reject': reject,
            'critical_value': f.ppf(1 - alpha/2, n1-1, n2-1),
            'alternative': 'two-sided'
        }

    print(f"拒绝原假设: {chi2_result['reject']}")
    
    # F检验
    print("\n=== F检验 ===")
    f_result = hypothesis_test.f_test(x1, x2, 0.05)
    print(f"F统计量: {f_result['f_statistic']:.4f}")
    print(f"p值: {f_result['p_value']:.4f}")
    print(f"拒绝原假设: {f_result['reject']}")
```

def hypothesis_testing_example():
    """假设检验示例"""
    # 生成模拟数据
    np.random.seed(42)
    data = np.random.normal(100, 15, 100)
    
    # 创建假设检验对象
    hypothesis_test = HypothesisTesting()

    print("=== t检验示例 ===")
    result = hypothesis_test.t_test(data, 100, 'two-sided', 0.05)
    print(f"假设: H0: μ = 100 vs H1: μ ≠ 100")
    print(f"样本均值: {np.mean(data):.4f}")
    print(f"样本标准差: {np.std(data, ddof=1):.4f}")
    print(f"t统计量: {result['t_statistic']:.4f}")
    print(f"p值: {result['p_value']:.4f}")
    print(f"显著性水平: 0.05")
    print(f"结论: {'拒绝H0' if result['reject'] else '接受H0'}")
    
    # 卡方检验
    print("\n=== 卡方检验示例 ===")
    x1 = np.random.normal(100, 10, 50)
    x2 = np.random.normal(100, 15, 50)
    result = hypothesis_test.chi2_test(x1, 100, 'two-sided', 0.05)
    print(f"假设: H0: σ² = 100 vs H1: σ² ≠ 100")
    print(f"样本方差: {np.var(x1, ddof=1):.4f}")
    print(f"卡方统计量: {result['chi2_statistic']:.4f}")
    print(f"p值: {result['p_value']:.4f}")
    print(f"显著性水平: 0.05")
    print(f"结论: {'拒绝H0' if result['reject'] else '接受H0'}")
```
