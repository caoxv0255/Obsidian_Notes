## 1. 神经系统的层级组织

### 1.1 神经系统的两大分支

```
神经系统
├── 中枢神经系统（CNS）  ← 大脑 + 脊髓
└── 外周神经系统（PNS）  ← 躯体神经 + 自主神经
```

**中枢神经系统**是信息处理的核心：
- **大脑**（Cerebrum）：高级认知功能
- **小脑**（Cerebellum）：运动协调、学习
- **脑干**（Brain Stem）：生命维持基本功能
- **脊髓**（Spinal Cord）：感觉传导、运动反射

### 1.2 大脑的主要分区

从**背面**俯视大脑皮层，可见四大脑叶：

```
            额叶（Frontal Lobe）
           ┌─────────────────┐
           │                 │
           │    顶叶         │ ← 感觉整合
           │  (Parietal)     │
    左半球  │                 │  右半球
           │    颞叶         │ ← 听觉、记忆
           │  (Temporal)     │
           │                 │
           └─────────────────┘
              枕叶（Occipital） ← 视觉
                    ↑
               初级视觉皮层 V1
```

| 脑叶 | 主要功能 | 关键区域 |
|------|----------|----------|
| **额叶** | 计划、决策、执行控制、工作记忆 | 前额叶皮层（PFC）、运动皮层 |
| **顶叶** | 空间感知、注意力、感觉整合 | 躯体感觉皮层、顶叶联合皮层 |
| **颞叶** | 听觉处理、物体识别、记忆 | 听觉皮层、海马体、杏仁核 |
| **枕叶** | 视觉信息处理 | 初级视觉皮层（V1）、视觉联合皮层 |

## 2. 大脑皮层的六层架构

### 2.1 层状结构概述

新皮层（Neocortex）具有**六层**（Layer I-VI）的层状结构，每层有独特的细胞组成和连接模式：

```
Layer I（分子层）：   稀疏神经元，主要是树突和轴突末梢
         ↓
Layer II/III（外颗粒层）：小型锥体神经元，水平连接
         ↓
Layer IV（内颗粒层）：  接收丘脑特异性输入（感觉信息入口）
         ↓
Layer V（内锥体层）：  大型锥体神经元，传出皮层输出
         ↓
Layer VI（多形层）：   多形神经元，传出到丘脑
```

### 2.2 各层的连接特性

### 2.3 皮层柱（Cortical Column）—— 功能基本单元

**皮层柱**是皮层信息处理的基本功能单元：

- 直径约 50-100 μm
- 包含约 10,000-20,000 个神经元
- 同一柱内神经元有相似的感受野/功能偏好
- 柱之间通过水平连接形成**功能网络**

> **计算神经科学意义**：皮层柱的层状结构是"**反向工程**"的目标——我们需要理解每一层的计算功能，才能建立真实的神经网络模型。

## 3. 主要皮层区域的功能定位

### 3.1 视觉通路（从眼到枕叶）

```
视网膜 → 视神经 → 视交叉 → 外侧膝状体（LGN）→ 初级视觉皮层 V1
                                                        ↓
                                        V2 → V3 → V4（物体识别）
                                                        ↓
                                              MT（运动检测）→ MST
```

| 区域 | 功能 | 损伤结果 |
|------|------|----------|
| V1 | 初级视觉：边缘、方向、空间频率 | 皮质盲 |
| V2 | 整合视野信息、深度感知 | 视觉忽视 |
| V3 | 形状和运动整合 | 运动感知障碍 |
| V4 | 颜色感知、物体识别 | 色盲、物体失认 |
| MT/MST | 运动检测、前庭整合 | 运动盲 |

### 3.2 听觉通路

```
耳蜗 → 听神经 → 脑干（上橄榄复合体）→ 下丘 → 听皮层（A1）
```

- **A1**（初级听觉皮层）：音调图谱（Tonotopic Map）
- 层级处理：从简单频率 → 复杂声音特征 → 语音识别

### 3.3 躯体感觉皮层

**体感皮层**（S1）的**侏儒人**（Homunculus）图：

```
       足部 ────────────────→ 半球内侧（中央沟后）
       腿部
       躯干        ← 中央后回（躯体感觉皮层）
       手部（放大）← Hand representation
       面部 ────────────────→ 半球外侧
```

> **关键洞察**：皮层表征遵循**资源分配原则**——手指和面部的感受器密度高，对应更大的皮层表征区。这是"**神经拥挤**"（Neural Crowding）效应的体现。

### 3.4 前额叶皮层（PFC）—— 执行功能的神经基础

**前额叶皮层**是**计算神经科学**最重要的研究区域之一：

| 子区域 | 功能 |
|--------|------|
| **背外侧PFC**（DLPFC） | 工作记忆、认知控制、计划 |
| **腹内侧PFC**（VMPFC） | 风险评估、情绪调节、决策 |
| **前扣带回**（ACC） | 冲突监控、错误检测 |

PFC 通过**长程连接**整合来自所有其他皮层的信息，是**注意力和工作记忆**的核心。

## 4. 皮层下结构

### 4.1 基底节（Basal Ganglia）

```
皮层 ──兴奋性投射──→ 基底节 ──抑制性投射──→ 丘脑 ──兴奋性投射──→ 皮层
                                ↑
                          黑质（SNc）
                          （多巴胺调节）
```

**功能**：**动作选择**和**强化学习**
- 直接通路：促进目标运动
- 间接通路：抑制不需要的运动
- 多巴胺信号调节学习的权重

### 4.2 海马体（HippoCampus）

```
海马体结构：
齿状回（DG）→ CA3 → CA1 → 穹窿 → 下丘脑
           ↑           ↓
         苔藓纤维   Schaffer侧支
```

**功能**：
- **空间导航**（Place Cells：位置细胞）
- **情景记忆**形成
- **记忆巩固**（从短时 → 长时）

> **计算神经科学意义**：海马体的**稀疏编码**（Sparse Coding）机制是理解记忆存储的关键。

### 4.3 丘脑（Thalamus）—— 皮层门户

丘脑不仅是"中继站"，更是**主动的信息门控**（Gating）系统：

- **特异性丘脑核**（如 LGN）：传递感觉信息到对应皮层区
- **髓板内核**（Intralaminar）：调节皮层整体兴奋性（觉醒）
- **丘脑-皮层回路**：皮层和丘脑双向连接，形成**功能柱**（Functional Loops）

> **关键认知**：丘脑-皮层系统的同步活动（Synchronization）是**意识**的神经基础之一。

## 5. Python 实践：脑区连接可视化

```python
import numpy as np
import matplotlib.pyplot as plt

# 简化的脑区连接矩阵
regions = ['V1', 'V2', 'MT', 'A1', 'S1', 'PFC', 'DLPFC', 'Hippo', 'BG']
n = len(regions)

# 连接强度（0-1），1=强连接
connections = np.zeros((n, n))

# 视觉通路
connections[0, 1] = 0.9  # V1 → V2
connections[1, 2] = 0.8  # V2 → MT

# 跨通道整合
connections[1, 5] = 0.6  # V2 → PFC

# 前额叶内部
connections[5, 6] = 0.9  # PFC ↔ DLPFC

# 海马-皮层
connections[7, 5] = 0.5  # Hippo → PFC
connections[5, 7] = 0.4  # PFC → Hippo（反馈）

# 基底节-丘脑-皮层回路
connections[8, 5] = 0.7  # BG → PFC

# 可视化
fig, ax = plt.subplots(figsize=(8, 6))
im = ax.imshow(connections, cmap='Blues', vmin=0, vmax=1)
ax.set_xticks(range(n))
ax.set_yticks(range(n))
ax.set_xticklabels(regions, rotation=45)
ax.set_yticklabels(regions)
ax.set_title('Brain Region Connectivity Matrix')
plt.colorbar(im, label='Connection Strength')
plt.tight_layout()
plt.savefig('brain_connectivity.png', dpi=150)
plt.show()
```

| 概念 | 定义 | 计算意义 |
