# 约束条件处理

## 约束条件体系

投资组合优化中的约束条件是实际可投资性的关键保障。本项目系统性地处理了各类约束。

## 约束分类

### 1. 硬约束 (Hard Constraints)
必须满足的约束，违反则组合不可执行。

```python
class HardConstraints:
    """硬约束集合"""
    
    @staticmethod
    def weight_sum_constraint(weights):
        """权重和为1"""
        return np.isclose(weights.sum(), 1.0, atol=1e-6)
    
    @staticmethod
    def long_only_constraint(weights):
        """只允许做多"""
        return np.all(weights >= -1e-6)
    
    @staticmethod
    def max_weight_constraint(weights, max_w=0.15):
        """单个资产最大权重"""
        return np.all(weights <= max_w + 1e-6)
```

### 2. 软约束 (Soft Constraints)
希望满足但不强制，违反会产生惩罚成本。

```python
class SoftConstraints:
    """软约束及惩罚函数"""
    
    @staticmethod
    def sector_concentration_penalty(weights, sector_mapping, max_sector_weight=0.30):
        """行业集中度惩罚"""
        penalty = 0
        sectors = set(sector_mapping.values())
        
        for sector in sectors:
            sector_assets = [a for a, s in sector_mapping.items() if s == sector]
            sector_weight = sum(weights[i] for i in sector_assets if i < len(weights))
            
            if sector_weight > max_sector_weight:
                penalty += (sector_weight - max_sector_weight) ** 2
        
        return penalty * 100  # 惩罚系数
    
    @staticmethod
    def turnover_penalty(old_weights, new_weights, penalty_rate=0.001):
        """换手率惩罚：避免频繁交易"""
        turnover = np.abs(new_weights - old_weights).sum() / 2
        return turnover * penalty_rate
```

## 约束实现

### 交易成本约束
```python
class TransactionCostConstraints:
    """交易成本相关约束"""
    
    @staticmethod
    def calculate_net_return(old_weights, new_weights, returns, commission_rate=0.0003):
        """计算扣费后的净收益"""
        turnover = np.abs(new_weights - old_weights).sum() / 2
        
        # 交易成本
        transaction_cost = turnover * commission_rate * 2
        
        # 组合收益
        portfolio_return = (new_weights * returns).sum()
        
        return portfolio_return - transaction_cost
    
    @staticmethod
    def liquidity_constraint(weights, volumes, min_daily_volume=5000000):
        """流动性约束：确保可交易"""
        # 单日最大成交量占比不超过10%
        max_position_value = weights * volumes * 0.1
        return max_position_value
```

### 行业/风格约束
```python
class SectorConstraints:
    """行业暴露约束"""
    
    def __init__(self, sector_mapping, factor_exposure):
        self.sector_mapping = sector_mapping
        self.factor_exposure = factor_exposure
    
    def enforce_sector_limits(self, weights, max_sector_exposure=0.30):
        """强制行业权重上限"""
        adjusted_weights = weights.copy()
        
        for sector in set(self.sector_mapping.values()):
            sector_assets = [i for i, s in enumerate(self.sector_mapping) if s == sector]
            sector_weight = sum(adjusted_weights[i] for i in sector_assets)
            
            if sector_weight > max_sector_exposure:
                excess = sector_weight - max_sector_exposure
                # 等比例缩减该行业权重
                for i in sector_assets:
                    reduction = excess * (adjusted_weights[i] / sector_weight)
                    adjusted_weights[i] -= reduction
        
        return adjusted_weights
    
    def factor_exposure_constraint(self, weights, max_beta=1.2, max_size_exposure=1.5):
        """风格因子暴露约束"""
        portfolio_beta = (weights @ self.factor_exposure['beta']).sum()
        portfolio_size = (weights @ self.factor_exposure['size']).sum()
        
        return {
            'beta_ok': portfolio_beta <= max_beta,
            'size_ok': portfolio_size <= max_size_exposure,
            'current_beta': portfolio_beta,
            'current_size': portfolio_size
        }
```

## 约束处理策略

### 投影法
当优化结果违反约束时，将其投影到可行域：

```python
class ConstraintProjection:
    """约束投影：将不满足约束的解投影到可行域"""
    
    @staticmethod
    def project_to_simplex(weights):
        """投影到单纯形（权重和为1，非负）"""
        weights = np.maximum(weights, 0)  # 确保非负
        
        # Sorted L1 Norm Minimization Projection
        n = len(weights)
        u = np.sort(weights)[::-1]
        cssv = np.cumsum(u)
        
        for i in range(n):
            if u[i] + (1 - cssv[i]) / (i + 1) > 0:
                rho = i + 1
                theta = (1 - cssv[i]) / rho
                break
        
        return np.maximum(weights - theta, 0)
    
    @staticmethod
    def project_with_max_weight(weights, max_w=0.15):
        """权重裁剪到上限"""
        return np.minimum(np.maximum(weights, 0), max_w)
```

### 约束松弛
对于难以严格满足的约束，采用惩罚函数：

```python
class ConstraintRelaxation:
    """约束松弛：将硬约束转化为惩罚项"""
    
    def __init__(self, constraints):
        self.constraints = constraints
    
    def add_penalty_to_objective(self, objective_func, weights, penalty_weight=1000):
        """在目标函数中加入惩罚项"""
        penalty = 0
        
        # 权重和惩罚
        weight_sum_penalty = (sum(weights) - 1) ** 2
        
        # 非负惩罚
        negative_penalty = np.sum(np.minimum(weights, 0) ** 2)
        
        # 权重上限惩罚
        max_penalty = np.sum(np.maximum(weights - 0.15, 0) ** 2)
        
        total_penalty = penalty_weight * (weight_sum_penalty + negative_penalty + max_penalty)
        
        return objective_func(weights) + total_penalty
```

## 实际约束案例

### 案例：涨跌停无法交易
```python
def handle_limit_locked(date, signal_weights, price_data):
    """处理涨跌停导致的无法交易"""
    adjusted_weights = signal_weights.copy()
    
    for code in adjusted_weights.index:
        if price_data.loc[date, code, 'limit_up']:
            # 涨停日无法买入，将该权重重新分配
            locked_weight = adjusted_weights.loc[code]
            # 从其他可交易股票中等比例分配
            available_codes = [
                c for c in adjusted_weights.index 
                if c != code and not price_data.loc[date, c, 'limit_down']
            ]
            if available_codes:
                boost = locked_weight / len(available_codes)
                for c in available_codes:
                    adjusted_weights.loc[c] += boost
            adjusted_weights.loc[code] = 0
        
        elif price_data.loc[date, code, 'limit_down']:
            # 跌停日无法卖出，但保留原有持仓
            pass
    
    return adjusted_weights
```

### 案例：持仓停牌
```python
def handle_suspended(old_positions, new_weights, suspended_codes):
    """处理持仓股票停牌"""
    final_weights = new_weights.copy()
    
    for code in suspended_codes:
        if code in old_positions:
            # 保留停牌持仓，无法调整
            suspended_weight = old_positions[code] * old_positions.get_price(code) / total_portfolio_value
            final_weights.loc[code] = final_weights.get(code, 0) + suspended_weight
    
    # 归一化
    final_weights = final_weights / final_weights.sum()
    
    return final_weights
```

## 约束对组合表现的影响

| 约束条件 | 对收益的影响 | 对风险的影响 |
|----------|--------------|--------------|
| 权重上限15% | -1.2% | -0.8% |
| 行业上限30% | -0.5% | -0.3% |
| 换手率约束 | -0.3% | +0.1% |
| 流动性约束 | -0.8% | -0.4% |

**结论**: 约束会降低收益，但通常也降低了风险。对于追求稳健的组合，约束是必要的。
