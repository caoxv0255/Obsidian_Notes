## 因子分类体系

### 1. 估值因子 (Valuation Factors)
衡量公司相对价值的因子，反映市场对公司基本面的定价。

```python
class ValuationFactors:
    """估值类因子计算"""
    
    @staticmethod
    def pe_ratio(market_cap, net_income):
        """市盈率倒数 (E/P)"""
        return net_income / market_cap
    
    @staticmethod
    def pb_ratio(price, book_per_share):
        """市净率倒数 (B/P)"""
        return book_per_share / price
    
    @staticmethod
    def pc_ratio(cash_flow, market_cap):
        """市现率 (CF/P)"""
        return cash_flow / market_cap
    
    @staticmethod
    def ps_ratio(revenue, market_cap):
        """市销率 (S/P)"""
        return revenue / market_cap
    
    @staticmethod
    def ev_ebitda(enterprise_value, ebitda):
        """EV/EBITDA 企业价值倍数"""
        return enterprise_value / ebitda
```

### 2. 动量因子 (Momentum Factors)
衡量股票过去价格表现对未来收益的预测能力。

```python
class MomentumFactors:
    """动量类因子计算"""
    
    def __init__(self, price_data):
        self.price = price_data
    
    def return_(self, window):
        """特定窗口期收益率"""
        return self.price.pct_change(window)
    
    def rank_return(self, window):
        """窗口期内收益率排名（横截面动量）"""
        return self.price.rank(axis=1, pct=True).pct_change(window)
    
    def reversal(self, short_window=5, long_window=20):
        """均值回归因子"""
        short_term = self.price.pct_change(short_window)
        long_term = self.price.pct_change(long_window)
        return long_term - short_term
    
    def momentum_acceleration(self, window=20):
        """动量加速度：收益的二阶导"""
        returns = self.price.pct_change()
        return returns.rolling(window).apply(lambda x: np.polyfit(range(len(x)), x, 1)[0])
```

### 3. 质量因子 (Quality Factors)
衡量公司盈利质量、运营效率、财务健康度。

```python
class QualityFactors:
    """质量类因子计算"""
    
    @staticmethod
    def roa(net_income, total_assets):
        """资产收益率"""
        return net_income / total_assets
    
    @staticmethod
    def roe(net_income, equity):
        """净资产收益率"""
        return net_income / equity
    
    @staticmethod
    def gross_margin(revenue, cost):
        """毛利率"""
        return (revenue - cost) / revenue
    
    @staticmethod
    def asset_turnover(revenue, total_assets):
        """资产周转率"""
        return revenue / total_assets
    
    @staticmethod
    def debt_to_equity(total_debt, equity):
        """资产负债率"""
        return total_debt / equity
    
    @staticmethod
    def accruals(net_income, operating_cash_flow):
        """应计项目：衡量盈利质量
        高应计意味着盈利主要由应计项目构成，未来可能反转
        """
        return (net_income - operating_cash_flow) / total_assets
```

## 因子处理与分析

### 因子标准化
```python
def standardize_factor(factor_matrix, method='zscore'):
    """因子标准化处理"""
    if method == 'zscore':
        # Z-score标准化
        return (factor_matrix - factor_matrix.mean()) / factor_matrix.std()
    elif method == 'rank':
        # 排名标准化（对极端值更稳健）
        return factor_matrix.rank(axis=1, pct=True)
    elif method == 'winsorize':
        # 去极值处理
        q1 = factor_matrix.quantile(0.01, axis=1)
        q99 = factor_matrix.quantile(0.99, axis=1)
        return factor_matrix.clip(lower=q1, upper=q99, axis=0)
```

### 因子有效性检验
```python
def factor_validation(factor, returns, top_quantile=0.2, bottom_quantile=0.2):
    """因子有效性全面检验"""
    
    results = {}
    
    # 1. IC分析 (Information Coefficient)
    ic_series = factor.corrwith(returns, axis=1)
    results['IC_mean'] = ic_series.mean()
    results['IC_std'] = ic_series.std()
    results['IC_IR'] = ic_series.mean() / ic_series.std() if ic_series.std() != 0 else 0
    
    # 2. 分组回测
    long_short = []
    for date in factor.index:
        f = factor.loc[date].dropna()
        r = returns.loc[date].reindex(f.index).dropna()
        
        if len(f) < 10:
            continue
            
        # 分5组
        groups = pd.qcut(f, 5, labels=False)
        long_return = r[groups == 4].mean()  # 最高因子值组
        short_return = r[groups == 0].mean() # 最低因子值组
        long_short.append(long_return - short_return)
    
    results['long_short_mean'] = np.mean(long_short)
    results['long_short_t'] = ttest_1samp(long_short, 0)[0]  # t统计量
    
    # 3. 衰减分析
    for delay in range(1, 6):
        delayed_returns = returns.shift(delay)
        ic_delayed = factor.corrwith(delayed_returns, axis=1)
        results[f'IC_delay_{delay}'] = ic_delayed.mean()
    
    return pd.DataFrame([results])
```

## 因子组合策略

### 单因子策略框架
```python
class SingleFactorStrategy:
    """单因子选股策略"""
    
    def __init__(self, factor_name, rebalance_freq='W'):
        self.factor_name = factor_name
        self.rebalance_freq = rebalance_freq
        self.position = {}
    
    def generate_signals(self, factor_data, date):
        """生成交易信号"""
        factor = factor_data.loc[date].dropna()
        # 做多因子值最高的前20%，做空最低的后20%
        signals = pd.Series(0, index=factor.index)
        signals[factor > factor.quantile(0.8)] = 1
        signals[factor < factor.quantile(0.2)] = -1
        return signals
    
    def rebalance(self, signals, current_positions):
        """根据信号调仓"""
        trades = signals - current_positions
        return trades
```

## 关键发现

### 1. 估值因子的周期性
- **发现**: 估值因子在市场风格切换时有明显失效期
- **解释**: 2019-2021年成长风格下，PE类因子表现差；2022年后价值因子回归
- **应用**: 加入市场风格判断因子，动态调整因子权重

### 2. 动量因子的"趋势-反转"双面性
- **短期动量(1-4周)**: 呈现趋势延续特性
- **中长期动量(3-12月)**: 呈现均值回归特性
- **应用**: 构建多周期动量综合因子

### 3. 质量因子的长周期有效性
- ROE、资产周转率等质量因子在全市场范围内长期有效
- 但需注意行业间可比性问题
- **应用**: 使用行业相对质量因子（行业内排名）

## 因子挖掘的思考

因子挖掘的本质是**寻找市场定价的规律**。好的因子需要满足：
1. **逻辑可解释**: 能讲清楚背后的经济学逻辑
2. **历史稳健**: 在不同市场周期、不同板块都有效
3. **难以套利**: 不是很容易被市场其他参与者抹平

实践中，90%的想法会在验证阶段被淘汰。保持开放的思维、严谨的验证、持续的迭代是关键。
