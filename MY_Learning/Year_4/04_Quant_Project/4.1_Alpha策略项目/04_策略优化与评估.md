## 策略优化框架

策略优化不是简单地调参数，而是找到在风险和收益之间最优平衡的参数组合。本项目采用了多种优化方法。

## 参数优化方法

### 1. 网格搜索 (Grid Search)
```python
def grid_search(strategy_class, param_grid, data_train, data_test):
    """网格搜索找最优参数"""
    results = []
    
    # 生成参数组合
    param_names = list(param_grid.keys())
    param_values = list(param_grid.values())
    
    for combo in itertools.product(*param_values):
        params = dict(zip(param_names, combo))
        
        # 训练集表现
        strategy = strategy_class(**params)
        train_metrics = run_backtest(strategy, data_train)
        
        # 测试集表现
        test_metrics = run_backtest(strategy, data_test)
        
        results.append({
            'params': params,
            'train_sharpe': train_metrics['sharpe'],
            'test_sharpe': test_metrics['sharpe'],
            'train_return': train_metrics['annual_return'],
            'test_return': test_metrics['annual_return']
        })
    
    return pd.DataFrame(results)
```

### 2. Walk-Forward Analysis
解决过拟合的核心方法：在历史窗口上优化参数，在未来窗口上验证。

```python
class WalkForwardOptimizer:
    """Walk-Forward 优化"""
    
    def __init__(self, train_window, test_window, step_size):
        self.train_window = train_window   # 如252天（1年）
        self.test_window = test_window     # 如63天（1季度）
        self.step_size = step_size         # 如21天（1个月）
    
    def run(self, dates, param_grid):
        """
        1. 用[t, t+train_window]的数据优化参数
        2. 用[t+train_window, t+train_window+test_window]的数据测试
        3. 滚动到下一个窗口
        """
        results = []
        
        t = dates[0]
        while t + self.train_window + self.test_window <= dates[-1]:
            train_start = t
            train_end = t + self.train_window
            test_start = train_end
            test_end = test_start + self.test_window
            
            # 在训练集上优化
            best_params = self.optimize_on_train(
                param_grid, train_start, train_end
            )
            
            # 在测试集上验证
            test_metrics = self.validate(
                best_params, test_start, test_end
            )
            
            results.append({
                'period': f"{train_start}-{train_end}",
                'params': best_params,
                **test_metrics
            })
            
            t += self.step_size
        
        return pd.DataFrame(results)
```

## 策略评估维度

### 收益类指标
| 指标 | 计算方法 | 说明 |
|------|----------|------|
| 年化收益率 | $(1+R_{total})^{1/n}-1$ | 排除资金时间价值 |
| 超额收益 | $R_p - R_b$ | 相对于基准的收益 |
| 信息比率 | $IR = \frac{R_p - R_b}{\sigma(R_p - R_b)}$ | 单位主动风险带来的超额收益 |

### 风险类指标
| 指标 | 计算方法 | 说明 |
|------|----------|------|
| 最大回撤 | $\max_{t_1<t_2} (V_{t_1} - V_{t_2}) / V_{t_1}$ | 历史最大亏损 |
| 波动率 | $\sigma(R_p)$ | 收益率标准差 |
| VaR | $VaR_\alpha = -\inf\{x: P(R>x) \leq \alpha\}$ | 风险价值 |

### 风险调整收益
| 指标 | 公式 | 说明 |
|------|------|------|
| 夏普比率 | $SR = \frac{R_p - R_f}{\sigma_p}$ | 单位总风险的超额收益 |
| Sortino比率 | $Sortino = \frac{R_p - R_f}{DR}$ | 只考虑下行风险 |
| Calmar比率 | $Calmar = \frac{R_p}{MaxDD}$ | 收益与最大回撤比 |

### 2. 蒙特卡洛模拟
```python
def monte_carlo_bootstrap(returns, n_simulations=10000):
    """Bootstrap蒙特卡洛：评估策略稳健性"""
    simulated_sharpes = []
    
    for _ in range(n_simulations):
        # 有放回抽样
        sample = np.random.choice(returns, size=len(returns), replace=True)
        simulated_sharpes.append(sample.mean() / sample.std() * np.sqrt(252))
    
    return {
        'mean_sharpe': np.mean(simulated_sharpes),
        'std_sharpe': np.std(simulated_sharpes),
        'confidence_interval': np.percentile(simulated_sharpes, [5, 95])
    }
```

def momentum_overshoot(factor, threshold=0.3):
    """检测动量是否过度拉伸"""
    # 因子值处于历史极值时，未来反转概率高
    percentile = (factor - factor.mean()) / factor.std()
    if abs(percentile) > threshold:
        return True  # 存在反转风险
    return False
