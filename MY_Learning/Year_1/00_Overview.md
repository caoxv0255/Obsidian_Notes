## 📖 学习路径

```
Year_1/
├── 1.1_微积分/          ← 单变量微积分（上课同步）
├── 1.2_线性代数/         ← 向量空间→矩阵→特征值（核心！）
├── 1.3_概率论/           ← 概率基础，为随机过程打基础
├── 2.1_Python/           ← 语法 → 科学计算 → 数据分析
├── 2.2_数据结构/          ← 竞赛基础：数组、链表、树、图
├── 2.3_编程基础/          ← C++语法（竞赛用）
├── 3.1_金融市场简介/      ← 了解金融基本概念
├── 3.2_量化是什么/        ← 量化交易是什么？有哪些策略？
└── 4.1_基础算法/          ← 排序、查找、基础数据结构应用
```

## 📚 核心参考教材

- **微积分**：Thomas' Calculus / 同济《高等数学》
- **线性代数**：Gilbert Strang, *Introduction to Linear Algebra*
- **概率论**：Ross, *A First Course in Probability*
- **数据结构**：C++版《数据结构》/ LeetCode Hot 100
- **Python**：官方文档 + *Python for Data Science*

## ✅ 验收标准

- [ ] 微积分：能计算单变量函数的极限/导数/积分，应用题会做
- [ ] 线性代数：理解向量空间、矩阵分解、特征值几何意义
- [ ] 概率论：掌握常见分布、二项/正态/泊松，能算期望方差
- [ ] Python：用pandas/matplotlib完成数据分析小项目
- [ ] 数据结构：能用数组/链表/树实现常见算法
- [ ] C++：能读懂竞赛代码，能自己写简单算法
- [ ] 双路径入门：能分别解释量化交易和计算神经科学的基本问题
