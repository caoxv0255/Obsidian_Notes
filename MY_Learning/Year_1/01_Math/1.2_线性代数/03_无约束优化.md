# 梯度下降与优化算法（Gradient Descent and Optimization Algorithms）

> 机器学习和深度学习中最核心的优化方法

## 3.1 梯度下降基础

### 3.1.1 基本思想

梯度下降是一种迭代优化算法，用于寻找函数的最小值。

**核心思想**：沿着函数梯度的反方向移动，逐步逼近最小值。

### 3.1.2 数学形式

对于目标函数 $f(\mathbf{x})$，梯度下降的更新规则：

$$\mathbf{x}_{t+1} = \mathbf{x}_t - \alpha \nabla f(\mathbf{x}_t)$$

其中：
- $\mathbf{x}_t$：第 $t$ 步的参数值
- $\alpha$：学习率（步长）
- $\nabla f(\mathbf{x}_t)$：梯度

### 3.1.3 几何直觉

**梯度**：
- 梯度 $\nabla f$ 指向函数值增长最快的方向
- 梯度的反方向 $-\nabla f$ 指向函数值下降最快的方向
- 梯度的模 $||\nabla f||$ 反映了函数在该点的陡峭程度

**学习率的影响**：
- 学习率过大：可能跳过最小值，甚至发散
- 学习率过小：收敛速度慢
- 学习率适中：平稳收敛

---

## 3.2 收敛性分析

### 3.2.1 凸函数的收敛性

**假设**：
1. $f$ 是凸函数
2. $f$ 是 $L$-光滑的（梯度Lipschitz连续）
3. 存在全局最小值 $f^*$

**收敛速度**：

$$f(\mathbf{x}_T) - f^* \leq \frac{||\mathbf{x}_0 - \mathbf{x}^*||^2}{2\alpha T}$$

**结论**：梯度下降在凸函数上的收敛速度是 $O(1/T)$。

### 3.2.2 强凸函数的收敛性

**强凸性条件**：存在 $\mu > 0$，使得

$$f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^\top(\mathbf{y}-\mathbf{x}) + \frac{\mu}{2}||\mathbf{y}-\mathbf{x}||^2$$

**收敛速度**（线性收敛）：

$$||\mathbf{x}_T - \mathbf{x}^*||^2 \leq \left(1 - \frac{\mu}{L}\right)^T ||\mathbf{x}_0 - \mathbf{x}^*||^2$$

## 3.3 梯度下降的变种

### 3.3.1 批量梯度下降（BGD）

使用全部数据计算梯度：

$$\nabla f(\mathbf{x}) = \frac{1}{n}\sum_{i=1}^{n}\nabla f_i(\mathbf{x})$$

**优点**：
- 收敛稳定
- 可以利用矩阵运算加速

**缺点**：
- 大数据集上速度慢
- 不能在线学习

### 3.3.2 随机梯度下降（SGD）

每次使用一个样本：

$$\mathbf{x}_{t+1} = \mathbf{x}_t - \alpha \nabla f_{i_t}(\mathbf{x}_t)$$

**优点**：
- 计算速度快
- 可以在线学习
- 有机会跳出局部最小值

**缺点**：
- 收敛不稳定
- 需要逐渐减小学习率

### 3.3.3 小批量梯度下降（Mini-batch GD）

折中方案，每次使用 $m$ 个样本：

$$\mathbf{x}_{t+1} = \mathbf{x}_t - \alpha \frac{1}{m}\sum_{j=1}^{m}\nabla f_{i_j}(\mathbf{x}_t)$$

**优点**：
- 平衡了稳定性和速度
- 可以利用GPU并行计算

**典型批量大小**：32, 64, 128, 256

## 3.4 动量法（Momentum）

### 3.4.1 动量法原理

**问题**：SGD在峡谷型损失函数中震荡严重

**解决方案**：引入动量，累积历史梯度

**更新规则**：

$$\mathbf{v}_{t+1} = \beta \mathbf{v}_t + \nabla f(\mathbf{x}_t)$$
$$\mathbf{x}_{t+1} = \mathbf{x}_t - \alpha \mathbf{v}_{t+1}$$

其中 $\beta$ 是动量系数（通常取 0.9）。

### 3.4.2 物理直觉

- $\mathbf{v}$ 类似速度
- $\nabla f$ 类似加速度
- $\beta$ 类似摩擦系数

**效果**：
- 在一致的梯度方向上加速
- 在不一致的方向上减速
- 减少震荡

### 3.4.3 Python实现

```python
import numpy as np

class SGDWithMomentum:
    def __init__(self, learning_rate=0.01, momentum=0.9):
        self.lr = learning_rate
        self.momentum = momentum
        self.velocity = None
    
    def update(self, params, grads):
        if self.velocity is None:
            self.velocity = np.zeros_like(params)
        
        # 更新速度
        self.velocity = self.momentum * self.velocity + grads
        # 更新参数
        params -= self.lr * self.velocity
        
        return params
```

## 3.5 自适应学习率方法

### 3.5.1 AdaGrad

**思想**：根据历史梯度调整每个参数的学习率

**更新规则**：

$$\mathbf{s}_{t+1} = \mathbf{s}_t + \nabla f(\mathbf{x}_t)^2$$
$$\mathbf{x}_{t+1} = \mathbf{x}_t - \frac{\alpha}{\sqrt{\mathbf{s}_{t+1} + \epsilon}} \odot \nabla f(\mathbf{x}_t)$$

**优点**：
- 自动调整学习率
- 适合稀疏数据

**缺点**：
- 学习率单调递减
- 可能过早停止学习

### 3.5.2 RMSprop

**改进**：使用指数移动平均代替累积和

$$\mathbf{s}_{t+1} = \beta \mathbf{s}_t + (1-\beta) \nabla f(\mathbf{x}_t)^2$$
$$\mathbf{x}_{t+1} = \mathbf{x}_t - \frac{\alpha}{\sqrt{\mathbf{s}_{t+1} + \epsilon}} \nabla f(\mathbf{x}_t)$$

**优点**：
- 解决了AdaGrad学习率过早减小的问题
- 适合非平稳目标

### 3.5.3 Adam

**结合动量和自适应学习率**

**更新规则**：

$$\mathbf{m}_{t+1} = \beta_1 \mathbf{m}_t + (1-\beta_1) \nabla f(\mathbf{x}_t)$$  (一阶矩估计)
$$\mathbf{v}_{t+1} = \beta_2 \mathbf{v}_t + (1-\beta_2) \nabla f(\mathbf{x}_t)^2$$  (二阶矩估计)
$$\hat{\mathbf{m}}_{t+1} = \frac{\mathbf{m}_{t+1}}{1-\beta_1^{t+1}}$$  (偏差修正)
$$\hat{\mathbf{v}}_{t+1} = \frac{\mathbf{v}_{t+1}}{1-\beta_2^{t+1}}$$  (偏差修正)
$$\mathbf{x}_{t+1} = \mathbf{x}_t - \alpha \frac{\hat{\mathbf{m}}_{t+1}}{\sqrt{\hat{\mathbf{v}}_{t+1}} + \epsilon}$$

**超参数**：
- $\alpha = 0.001$：学习率
- $\beta_1 = 0.9$：一阶矩衰减率
- $\beta_2 = 0.999$：二阶矩衰减率
- $\epsilon = 10^{-8}$：数值稳定项

### 3.5.4 Python实现Adam

```python
import numpy as np

class Adam:
    def __init__(self, learning_rate=0.001, beta1=0.9, beta2=0.999, epsilon=1e-8):
        self.lr = learning_rate
        self.beta1 = beta1
        self.beta2 = beta2
        self.epsilon = epsilon
        self.m = None  # 一阶矩
        self.v = None  # 二阶矩
        self.t = 0     # 时间步
    
    def update(self, params, grads):
        if self.m is None:
            self.m = np.zeros_like(params)
            self.v = np.zeros_like(params)
        
        self.t += 1
        
        # 更新一阶矩和二阶矩
        self.m = self.beta1 * self.m + (1 - self.beta1) * grads
        self.v = self.beta2 * self.v + (1 - self.beta2) * (grads ** 2)
        
        # 偏差修正
        m_hat = self.m / (1 - self.beta1 ** self.t)
        v_hat = self.v / (1 - self.beta2 ** self.t)
        
        # 更新参数
        params -= self.lr * m_hat / (np.sqrt(v_hat) + self.epsilon)
        
        return params
```

## 3.6 学习率调度

### 3.6.1 学习率衰减

**阶梯衰减**：

$$\alpha_t = \alpha_0 \times \gamma^{\lfloor t/T \rfloor}$$

```python
def step_decay(epoch, initial_lr, drop_rate=0.5, epochs_drop=10):
    return initial_lr * (drop_rate ** (epoch // epochs_drop))
```

**指数衰减**：

$$\alpha_t = \alpha_0 e^{-kt}$$

```python
def exponential_decay(epoch, initial_lr, k=0.1):
    return initial_lr * np.exp(-k * epoch)
```

### 3.6.2 学习率预热（Warmup）

**问题**：训练初期参数随机，梯度不稳定

**解决方案**：前几个epoch使用较小学习率

```python
def warmup_decay(epoch, warmup_epochs, initial_lr, target_lr):
    if epoch < warmup_epochs:
        return initial_lr + (target_lr - initial_lr) * epoch / warmup_epochs
    else:
        return target_lr * (0.95 ** (epoch - warmup_epochs))
```

### 3.6.3 余弦退火

$$\alpha_t = \alpha_{min} + \frac{1}{2}(\alpha_{max} - \alpha_{min})\left(1 + \cos\left(\frac{t\pi}{T}\right)\right)$$

```python
def cosine_annealing(epoch, T_max, eta_min=0, eta_max=0.1):
    return eta_min + (eta_max - eta_min) * (1 + np.cos(np.pi * epoch / T_max)) / 2
```

## 3.7 应用场景

### 3.7.1 量化金融应用

#### 策略参数优化

```python
import numpy as np

class StrategyOptimizer:
    def __init__(self, strategy_func, data):
        self.strategy = strategy_func
        self.data = data
        self.optimizer = Adam(learning_rate=0.01)
    
    def objective(self, params):
        """
        计算策略的负夏普比率（目标是最小化）
        """
        returns = self.strategy(self.data, params)
        sharpe = np.mean(returns) / np.std(returns) * np.sqrt(252)
        return -sharpe
    
    def optimize(self, initial_params, epochs=1000):
        params = initial_params.copy()
        
        for epoch in range(epochs):
            # 计算梯度（数值梯度）
            grads = self.numerical_gradient(params)
            
            # 更新参数
            params = self.optimizer.update(params, grads)
            
            if epoch % 100 == 0:
                obj_value = self.objective(params)
                print(f"Epoch {epoch}, Objective: {obj_value:.4f}")
        
        return params
    
    def numerical_gradient(self, params, eps=1e-5):
        """
        数值梯度计算
        """
        grads = np.zeros_like(params)
        for i in range(len(params)):
            params_plus = params.copy()
            params_plus[i] += eps
            params_minus = params.copy()
            params_minus[i] -= eps
            
            grads[i] = (self.objective(params_plus) - self.objective(params_minus)) / (2 * eps)
        
        return grads

# 示例：优化移动平均策略参数
def ma_strategy(data, params):
    short_window, long_window = int(params[0]), int(params[1])
    # 简化实现：返回策略收益率
    return np.random.randn(252) * 0.01

# 优化
optimizer = StrategyOptimizer(ma_strategy, data=None)
initial_params = np.array([10.0, 30.0])
optimal_params = optimizer.optimize(initial_params)
```

### 3.7.2 计算神经科学应用

#### 神经网络训练

```python
import torch
import torch.nn as nn
import torch.optim as optim

class NeuralDecoder:
    def __init__(self, input_dim, hidden_dim, output_dim):
        self.model = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, output_dim)
        )
    
    def train(self, X_train, y_train, epochs=100, batch_size=32, lr=0.001):
        """
        训练神经解码器
        """
        criterion = nn.MSELoss()
        optimizer = optim.Adam(self.model.parameters(), lr=lr)
        
        # 学习率调度器
        scheduler = optim.lr_scheduler.CosineAnnealingLR(optimizer, T_max=epochs)
        
        dataset = torch.utils.data.TensorDataset(
            torch.FloatTensor(X_train), 
            torch.FloatTensor(y_train)
        )
        dataloader = torch.utils.data.DataLoader(dataset, batch_size=batch_size, shuffle=True)
        
        losses = []
        for epoch in range(epochs):
            epoch_loss = 0
            for batch_X, batch_y in dataloader:
                optimizer.zero_grad()
                
                # 前向传播
                outputs = self.model(batch_X)
                loss = criterion(outputs, batch_y)
                
                # 反向传播
                loss.backward()
                optimizer.step()
                
                epoch_loss += loss.item()
            
            scheduler.step()
            avg_loss = epoch_loss / len(dataloader)
            losses.append(avg_loss)
            
            if epoch % 10 == 0:
                print(f"Epoch {epoch}, Loss: {avg_loss:.6f}")
        
        return losses
    
    def predict(self, X):
        with torch.no_grad():
            return self.model(torch.FloatTensor(X)).numpy()

# 示例：训练运动解码器
# 假设：神经信号 -> 手部运动方向
input_dim = 100  # 100个神经元
hidden_dim = 64
output_dim = 2   # x, y方向

decoder = NeuralDecoder(input_dim, hidden_dim, output_dim)

# 模拟数据
X_train = np.random.randn(1000, input_dim)
y_train = np.random.randn(1000, output_dim)

# 训练
losses = decoder.train(X_train, y_train)
```

## 3.8 优化算法选择指南

### 3.8.1 选择流程图

```
数据规模?
├─ 小 (<10000样本)
│  └─ 使用BGD + L-BFGS
├─ 中 (10000-100000)
│  └─ 使用Mini-batch GD + Adam
└─ 大 (>100000)
   └─ 使用SGD + Momentum
```

### 3.8.2 各算法对比

| 算法 | 收敛速度 | 内存占用 | 适用场景 | 推荐度 |
|------|---------|---------|---------|--------|
| SGD | 慢 | 低 | 大数据、在线学习 | ⭐⭐⭐ |
| Momentum | 中 | 低 | 非平稳目标 | ⭐⭐⭐⭐ |
| AdaGrad | 快 | 中 | 稀疏数据 | ⭐⭐⭐ |
| RMSprop | 快 | 中 | 非平稳、RNN | ⭐⭐⭐⭐ |
| Adam | 快 | 中 | 通用场景 | ⭐⭐⭐⭐⭐ |

### 3.8.3 超参数调优建议

**学习率调优**：
1. 从小值开始（0.0001）
2. 逐步增大，观察损失曲线
3. 找到"刚好发散"的临界值
4. 使用该值的 1/10 到 1/2

**Adam超参数**：
- 学习率：通常 0.001
- $\beta_1$：通常 0.9（很少需要调整）
- $\beta_2$：通常 0.999（很少需要调整）

## 3.9 常见问题与解决方案

### Q1：损失不下降怎么办？

**可能原因**：
1. 学习率过大 → 减小学习率
2. 学习率过小 → 增大学习率或使用Adam
3. 数据未归一化 → 标准化输入
4. 梯度消失/爆炸 → 使用BatchNorm

### Q2：训练震荡怎么办？

**解决方案**：
1. 减小学习率
2. 使用动量
3. 增加批量大小
4. 梯度裁剪

### Q3：收敛太慢怎么办？

**解决方案**：
1. 使用Adam或RMSprop
2. 学习率预热
3. 更好的初始化
4. BatchNorm
