# 01-实数与极限

## 概述

实数与极限是微积分的基础，是整个数学分析的基石。本章将系统介绍实数的性质、极限的严格定义和计算方法，为后续学习微积分奠定坚实基础。

## 1. 实数的性质

### 1.1 实数集的定义

**实数集** $\mathbb{R}$ 是满足以下公理的集合：

1. **加法公理**：
   - 封闭性：$a + b \in \mathbb{R}$，对所有 $a, b \in \mathbb{R}$
   - 交换律：$a + b = b + a$
   - 结合律：$(a + b) + c = a + (b + c)$
   - 零元：存在 $0 \in \mathbb{R}$，使得 $a + 0 = a$
   - 负元：对每个 $a \in \mathbb{R}$，存在 $-a \in \mathbb{R}$，使得 $a + (-a) = 0$

2. **乘法公理**：
   - 封闭性：$ab \in \mathbb{R}$，对所有 $a, b \in \mathbb{R}$
   - 交换律：$ab = ba$
   - 结合律：$(ab)c = a(bc)$
   - 单位元：存在 $1 \in \mathbb{R}$，使得 $a \cdot 1 = a$
   - 逆元：对每个 $a \neq 0$，存在 $a^{-1} \in \mathbb{R}$，使得 $a \cdot a^{-1} = 1$

3. **分配律**：$a(b + c) = ab + ac$

### 1.2 实数的序结构

实数集 $\mathbb{R}$ 上定义了序关系 $\leq$，满足：
- **三歧性**：对任意 $a, b \in \mathbb{R}$，$a \leq b$ 或 $b \leq a$
- **传递性**：若 $a \leq b$ 且 $b \leq c$，则 $a \leq c$
- **自反性**：$a \leq a$

### 1.3 上确界和下确界

**上确界**（上确界）：集合 $S$ 的上确界 $\sup S$ 是满足以下条件的实数：
1. $\sup S$ 是 $S$ 的上界
2. 对任意 $M < \sup S$，存在 $s \in S$ 使得 $s > M$

**下确界**（下确界）：集合 $S$ 的下确界 $\inf S$ 是满足以下条件的实数：
1. $\inf S$ 是 $S$ 的下界
2. 对任意 $m > \inf S$，存在 $s \in S$ 使得 $s < m$

## 2. 极限的严格定义

### 2.1 数列的极限

**数列** $\{a_n\}$ 是一个函数 $a: \mathbb{N} \rightarrow \mathbb{R}$。

**数列的极限**：$\lim_{n \to \infty} a_n = L$，如果对于任意 $\varepsilon > 0$，存在 $N \in \mathbb{N}$，使得当 $n > N$ 时，$|a_n - L| < \varepsilon$。

### 2.2 函数的极限

**函数的极限**：$\lim_{x \to a} f(x) = L$，如果对于任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $0 < |x - a| < \delta$ 时，$|f(x) - L| < \varepsilon$。

### 2.3 单侧极限

**右极限**：$\lim_{x \to a^+} f(x) = L$，如果对于任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $a < x < a + \delta$ 时，$|f(x) - L| < \varepsilon$。

**左极限**：$\lim_{x \to a^-} f(x) = L$，如果对于任意 $\varepsilon > 0$，存在 $\delta > 0$，使得当 $a - \delta < x < a$ 时，$|f(x) - L| < \varepsilon$。

### 2.4 无穷极限

**正无穷极限**：$\lim_{x \to a} f(x) = +\infty$，如果对于任意 $M > 0$，存在 $\delta > 0$，使得当 $0 < |x - a| < \delta$ 时，$f(x) > M$。

**负无穷极限**：$\lim_{x \to a} f(x) = -\infty$，如果对于任意 $M < 0$，存在 $\delta > 0$，使得当 $0 < |x - a| < \delta$ 时，$f(x) < M$。

## 3. 极限的性质

### 3.1 唯一性

极限如果存在，则是唯一的。

### 3.2 四则运算

如果 $\lim_{x \to a} f(x) = L$，$\lim_{x \to a} g(x) = M$，则：

1. **和**：$\lim_{x \to a} [f(x) + g(x)] = L + M$
2. **差**：$\lim_{x \to a} [f(x) - g(x)] = L - M$
3. **积**：$\lim_{x \to a} [f(x) g(x)] = LM$
4. **商**：$\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}$（$M \neq 0$）

### 3.3 复合函数极限

如果 $\lim_{x \to a} g(x) = b$，$\lim_{y \to b} f(y) = L$，且存在 $\delta_0 > 0$，使得当 $0 < |x - a| < \delta_0$ 时，$g(x) \neq b$，则：

$$\lim_{x \to a} f(g(x)) = L$$

### 3.4 夹逼定理

如果对于所有 $x$ 在 $a$ 的某个去心邻域内（可能 $x = a$ 除外），有：

$$f(x) \leq g(x) \leq h(x)$$

且 $\lim_{x \to a} f(x) = \lim_{x \to a} h(x) = L$，则：

$$\lim_{x \to a} g(x) = L$$

## 4. 极限的计算

### 4.1 基本极限

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

$$\lim_{x \to 0} \frac{e^x - 1}{x} = 1$$

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

### 4.2 洛必达法则

对于 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型未定式，如果 $\lim_{x \to a} \frac{f'(x)}{g'(x)}$ 存在，则：

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

### 4.3 重要极限

**重要极限1**：

$$\lim_{x \to 0} (1 + x)^{1/x} = e$$

**重要极限2**：

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

## 5. Python实现

```python
import numpy as np
import matplotlib.pyplot as plt
from sympy import symbols, limit, oo, sin, exp, log

class RealAnalysis:
    """实分析类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def limit_numerical(self, func, x0, h=1e-6):
        """数值极限"""
        return (func(x0 + h) - func(x0 - h)) / (2 * h)
    
    def limit_left(self, func, x0, h=1e-6):
        """左极限"""
        return (func(x0 - h) - func(x0 - 2 * h)) / h
    
    def limit_right(self, func, x0, h=1e-6):
        """右极限"""
        return (func(x0 + 2 * h) - func(x0 + h)) / h
    
    def limit_at_infinity(self, func, h=1e-6):
        """无穷极限"""
        x = symbols('x')
        return limit(func(x), x, oo)
    
    def squeeze_theorem_demo(self):
        """夹逼定理演示"""
        x = np.linspace(0.01, 1, 100)
        
        # 创建三个函数，满足夹逼条件
        f = np.sin(x) / x  # 夹逼函数1
        g = x  # 夹逼函数2
        h = np.sin(x) / x  # 夹逼函数3
        
        # 修正夹逼函数2和3
        g = np.ones_like(x) * 0.8
        h = np.ones_like(x) * 1.2
        
        plt.figure(figsize=(10, 6))
        plt.plot(x, f, 'b-', linewidth=2, label='sin(x)/x')
        plt.plot(x, g, 'r--', linewidth=2, label='0.8')
        plt.plot(x, h, 'g--', linewidth=2, label='1.2')
        plt.axhline(y=1, color='k', linestyle='-', linewidth=2, label='极限=1')
        plt.xlabel('x')
        plt.ylabel('y')
        plt.title('夹逼定理演示')
        plt.legend()
        plt.grid(True, alpha=0.3)
        plt.show()
        
        # 验证夹逼条件
        print("夹逼函数1: sin(x)/x")
        print("夹逼函数2: 0.8")
        print("夹逼函数3: 1.2")
        print("当x→0时，夹逼函数2和3的极限都是1")
        print("根据夹逼定理，sin(x)/x的极限也是1")

class Calculus:
    """微积分类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def derivative_numerical(self, func, x, h=1e-6):
        """数值导数"""
        return (func(x + h) - func(x - h)) / (2 * h)
    
    def derivative_forward(self, func, x, h=1e-6):
        """前向导数"""
        return (func(x + h) - func(x)) / h
    
    def derivative_backward(self, func, x, h=1e-6):
        """后向导数"""
        return (func(x) - func(x - h)) / h
    
    def integral_numerical(self, func, a, b, n=1000):
        """数值积分（梯形法则）"""
        x = np.linspace(a, b, n + 1)
        y = func(x)
        
        # 梯形法则
        integral = (y[0] + y[-1]) / 2 + np.sum(y[1:-1])
        integral *= (b - a) / n
        
        return integral
    
    def integral_simpson(self, func, a, b, n=1000):
        """数值积分（Simpson法则）"""
        if n % 2 == 1:
            n += 1  # 确保n为偶数
            
        x = np.linspace(a, b, n + 1)
        y = func(x)
        
        # Simpson法则
        integral = y[0] + y[-1]
        integral += 4 * np.sum(y[1:-1:2])  # 奇数索引
        integral += 2 * np.sum(y[2:-2:2])  # 偶数索引
        integral *= (b - a) / (3 * n)
        
        return integral

# 使用示例
if __name__ == "__main__":
    # 创建实分析对象
    ra = RealAnalysis()
    
    # 测试夹逼定理
    print("=== 夹逼定理演示 ===")

    # 创建微积分对象
    calc = Calculus()
    
    # 测试数值微积分
    print("\n=== 数值微积分演示 ===")
    
    # 定义函数
    f = lambda x: x**2
    df = lambda x: 2 * x  # 导数
    
    # 数值导数
    x_test = 2.0
    numerical_derivative = calc.derivative_numerical(f, x_test)
    analytical_derivative = df(x_test)
    
    print(f"函数 f(x) = x² 在 x = {x_test} 处的导数:")
    print(f"数值导数: {numerical_derivative:.6f}")
    print(f"解析导数: {analytical_derivative:.6f}")
    print(f"误差: {abs(numerical_derivative - analytical_derivative):.2e}")
    
    # 数值积分
    a, b = 0, 2
    numerical_integral = calc.integral_numerical(f, a, b)
    analytical_integral = (b**3 - a**3) / 3
    
    print(f"\n函数 f(x) = x² 在 [{a}, {b}] 上的积分:")
    print(f"数值积分: {numerical_integral:.6f}")
    print(f"解析积分: {analytical_integral:.6f}")
    print(f"误差: {abs(numerical_integral - analytical_integral):.2e}")
```

## 6. 应用示例

### 6.1 物理学中的应用

在物理学中，极限的概念用于定义速度、加速度等基本概念：

```python
def physics_application():
    """物理学应用示例"""
    # 位置函数
    def position(t):
        return 0.5 * 9.8 * t**2  # 自由落体运动
    
    # 速度定义（位置的导数）
    def velocity(t):
        return 9.8 * t
    
    # 加速度定义（速度的导数）
    def acceleration(t):
        return 9.8
    
    # 测试
    t = 2.0
    print(f"时间 t = {t} 秒:")
    print(f"位置: {position(t):.2f} 米")
    print(f"速度: {velocity(t):.2f} m/s")
    print(f"加速度: {acceleration(t):.2f} m/s²")
```

### 6.2 经济学中的应用

在经济学中，极限用于定义边际概念：

```python
def economics_application():
    """经济学应用示例"""
    # 成本函数
    def cost(q):
        return 0.1 * q**3 - 2 * q**2 + 10 * q + 50
    
    # 边际成本（成本的导数）
    def marginal_cost(q):
        return 0.3 * q**2 - 4 * q + 10
    
    # 测试
    q = 10
    print(f"产量 q = {q}:")
    print(f"总成本: {cost(q):.2f}")
    print(f"边际成本: {marginal_cost(q):.2f}")
```

## 7. 常见问题

### Q1: 如何证明一个数列收敛？

**A**: 使用极限的定义，找到合适的 $N$，使得当 $n > N$ 时，$|a_n - L| < \varepsilon$。

### Q2: 极限不存在的情况有哪些？

**A**: 振荡、趋于无穷、左右极限不相等等。

### Q3: 洛必达法则的使用条件是什么？

**A**: 需要是 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型未定式，且导数存在。
