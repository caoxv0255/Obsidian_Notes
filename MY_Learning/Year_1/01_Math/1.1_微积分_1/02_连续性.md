# 02-连续性

## 概述

连续性是函数的一个重要性质，描述了函数在某点附近的行为。本章将系统介绍连续性的定义、性质和应用，为微积分的学习奠定基础。

## 1. 连续性的定义

### 1.1 函数在一点的连续性

**函数在一点的连续性**：函数 $f(x)$ 在点 $x_0$ 连续，如果：

$$\lim_{x \to x_0} f(x) = f(x_0)$$

这等价于：
1. $f(x_0)$ 存在
2. $\lim_{x \to x_0} f(x)$ 存在
3. $\lim_{x \to x_0} f(x) = f(x_0)$

### 1.2 左连续和右连续

**左连续**：$\lim_{x \to x_0^-} f(x) = f(x_0)$

**右连续**：$\lim_{x \to x_0^+} f(x) = f(x_0)$

### 1.3 间断点

**间断点**：函数在某点不连续的点。

**间断点的分类**：

1. **可去间断点**：$\lim_{x \to x_0} f(x)$ 存在但不等于 $f(x_0)$
2. **跳跃间断点**：左极限和右极限都存在但不相等
3. **无穷间断点**：至少有一个单侧极限为无穷
4. **振荡间断点**：极限不存在但不为无穷

## 2. 连续函数的性质

### 2.1 连续函数的四则运算

如果 $f(x)$ 和 $g(x)$ 在点 $x_0$ 连续，则：
- $f(x) \pm g(x)$ 在 $x_0$ 连续
- $f(x) \cdot g(x)$ 在 $x_0$ 连续
- $\frac{f(x)}{g(x)}$ 在 $x_0$ 连续（$g(x_0) \neq 0$）

### 2.2 复合函数的连续性

如果 $g(x)$ 在 $x_0$ 连续，$f(y)$ 在 $y_0 = g(x_0)$ 连续，则复合函数 $f(g(x))$ 在 $x_0$ 连续。

### 2.3 连续函数的有界性

**局部有界性**：如果 $f(x)$ 在 $x_0$ 连续，则在 $x_0$ 的某个邻域内有界。

### 2.4 连续函数的最值性

**有界闭区间上的连续函数**：在闭区间 $[a, b]$ 上连续的函数必有最大值和最小值。

## 3. 闭区间上连续函数的性质

### 3.1 有界性定理

**有界性定理**：在闭区间 $[a, b]$ 上连续的函数必有界。

### 3.2 最值定理

**最值定理**：在闭区间 $[a, b]$ 上连续的函数必有最大值和最小值。

### 3.3 零点定理

**零点定理**：如果 $f(x)$ 在闭区间 $[a, b]$ 上连续，且 $f(a) \cdot f(b) < 0$，则存在 $c \in (a, b)$，使得 $f(c) = 0$。

### 3.4 介值定理

**介值定理**：如果 $f(x)$ 在闭区间 $[a, b]$ 上连续，$f(a) \neq f(b)$，则对于任意 $k$ 介于 $f(a)$ 和 $f(b)$ 之间，存在 $c \in (a, b)$，使得 $f(c) = k$。

## 4. 一致连续性

### 4.1 一致连续的定义

函数 $f(x)$ 在集合 $S$ 上一致连续，如果对于任意 $\varepsilon > 0$，存在 $\delta > 0$，使得对于所有 $x, y \in S$，当 $|x - y| < \delta$ 时，$|f(x) - f(y)| < \varepsilon$。

### 4.2 一致连续与连续的关系

**定理**：在闭区间 $[a, b]$ 上连续的函数必一致连续。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.optimize import bisect

class Continuity:
    """连续性类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def is_continuous_at_point(self, func, x0, h=1e-6):
        """检查函数在某点是否连续"""
        try:
            # 检查函数值是否存在
            f_x0 = func(x0)
            
            # 检查极限是否存在
            left_limit = self.left_limit(func, x0, h)
            right_limit = self.right_limit(func, x0, h)
            
            # 检查极限是否等于函数值
            if np.isnan(left_limit) or np.isnan(right_limit):
                return False
            
            return np.isclose(left_limit, right_limit) and np.isclose(left_limit, f_x0)
        except:
            return False
    
    def left_limit(self, func, x0, h=1e-6):
        """左极限"""
        try:
            return func(x0 - h)
        except:
            return np.nan
    
    def right_limit(self, func, x0, h=1e-6):
        """右极限"""
        try:
            return func(x0 + h)
        except:
            return np.nan
    
    def check_discontinuity_type(self, func, x0, h=1e-6):
        """检查间断点类型"""
        try:
            f_x0 = func(x0)
            left_limit = self.left_limit(func, x0, h)
            right_limit = self.right_limit(func, x0, h)
            
            if np.isnan(left_limit) or np.isnan(right_limit):
                return "无穷间断点"
            
            if np.isclose(left_limit, right_limit):
                if np.isclose(left_limit, f_x0):
                    return "连续点"
                else:
                    return "可去间断点"
            else:
                if np.isclose(left_limit, f_x0) or np.isclose(right_limit, f_x0):
                    return "跳跃间断点"
                else:
                    return "跳跃间断点"
        except:
            return "其他间断点"
    
    def find_zero_point(self, func, a, b, tol=1e-6):
        """寻找零点（零点定理）"""
        if func(a) * func(b) >= 0:
            print("零点定理不适用：f(a)和f(b)同号")
            return None
        
        # 使用二分法
        try:
            zero = bisect(func, a, b)
            return zero
        except:
            print("二分法失败")
            return None
    
    def demonstrate_intermediate_value_theorem(self):
        """演示介值定理"""
        # 定义一个函数
        def f(x):
            return x**3 - x - 2
        
        # 在区间[1, 2]上寻找零点
        a, b = 1, 2
        zero = self.find_zero_point(f, a, b)
        
        if zero is not None:
            print(f"函数 f(x) = x³ - x - 2 在 [{a}, {b}] 上的零点: {zero:.6f}")
            print(f"验证: f({zero:.6f}) = {f(zero):.2e}")
        else:
            print("未找到零点")

class ContinuousFunctions:
    """连续函数类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def boundedness_theorem_demo(self):
        """演示有界性定理"""
        # 定义一个在闭区间上连续的函数
        def f(x):
            return x**2 + np.sin(x)
        
        # 在区间[-2, 2]上
        x = np.linspace(-2, 2, 1000)
        y = f(x)
        
        # 找到最大值和最小值
        max_val = np.max(y)
        min_val = np.min(y)
        
        plt.figure(figsize=(10, 6))
        plt.plot(x, y, 'b-', linewidth=2, label='f(x) = x² + sin(x)')
        plt.axhline(y=max_val, color='r', linestyle='--', label=f'最大值: {max_val:.4f}')
        plt.axhline(y=min_val, color='g', linestyle='--', label=f'最小值: {min_val:.4f}')
        plt.xlabel('x')
        plt.ylabel('f(x)')
        plt.title('有界性定理演示')
        plt.legend()
        plt.grid(True, alpha=0.3)
        plt.show()
        
        print(f"在区间[-2, 2]上，函数 f(x) = x² + sin(x) 的最大值: {max_val:.4f}")
        print(f"在区间[-2, 2]上，函数 f(x) = x² + sin(x) 的最小值: {min_val:.4f}")
        print("验证：函数在闭区间上有界")

class Discontinuity:
    """间断点类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def demonstrate_discontinuities(self):
        """演示不同类型的间断点"""
        # 定义不同类型的函数
        def f1(x):
            return 1/x if x != 0 else np.nan
        
        def f2(x):
            return np.sin(1/x) if x != 0 else np.nan
        
        def f3(x):
            return 1 if x >= 0 else -1
        
        # 在x=0处检查间断点
        x0 = 0
        h = 1e-6
        
        print("=== 间断点演示 ===")
        print("函数 f(x) = 1/x 在 x=0 处:")
        print(f"左极限: {self.left_limit(f1, x0, h)}")
        print(f"右极限: {self.right_limit(f1, x0, h)}")
        print(f"函数值: {f1(x0)}")
        print("类型: 无穷间断点\n")
        
        print("函数 f(x) = sin(1/x) 在 x=0 处:")
        print(f"左极限: {self.left_limit(f2, x0, h)}")
        print(f"右极限: {self.right_limit(f2, x0, h)}")
        print(f"函数值: {f2(x0)}")
        print("类型: 振荡间断点\n")
        
        print("函数 f(x) = sign(x) 在 x=0 处:")
        print(f"左极限: {self.left_limit(f3, x0, h)}")
        print(f"右极限: {self.right_limit(f3, x0, h)}")
        print(f"函数值: {f3(x0)}")
        print("类型: 跳跃间断点")
    
    def left_limit(self, func, x0, h):
        """左极限"""
        try:
            return func(x0 - h)
        except:
            return np.nan
    
    def right_limit(self, func, x0, h):
        """右极限"""
        try:
            return func(x0 + h)
        except:
            return np.nan
```

## 6. 应用示例

### 6.1 物理学中的应用

在物理学中，连续性概念用于描述物理量的变化：

```python
def physics_application():
    """物理学应用示例"""
    # 温度分布函数
    def temperature(x):
        return 20 + 10 * np.sin(np.pi * x / 10)
    
    # 在x=5处检查连续性
    x0 = 5
    print(f"温度分布函数在 x = {x0} 处的值: {temperature(x0):.2f}°C")
    
    # 检查极限
    h = 1e-6
    left_limit = temperature(x0 - h)
    right_limit = temperature(x0 + h)
    
    print(f"左极限: {left_limit:.2f}°C")
    print(f"右极限: {right_limit:.2f}°C")
    print("结论: 温度分布函数在x=5处连续")
```

### 6.2 经济学中的应用

在经济学中，连续性概念用于描述成本函数和收益函数：

```python
def economics_application():
    """经济学应用示例"""
    # 成本函数
    def cost(q):
        return 0.1 * q**2 + 5 * q + 100
    
    # 收益函数
    def revenue(q):
        return 10 * q - 0.05 * q**2
    
    # 利润函数
    def profit(q):
        return revenue(q) - cost(q)
    
    # 在q=50处检查连续性
    q0 = 50
    print(f"利润函数在 q = {q0} 处的值: {profit(q0):.2f}")
    
    # 检查边际成本和边际收益
    def marginal_cost(q):
        return 0.2 * q + 5
    
    def marginal_revenue(q):
        return 10 - 0.1 * q
    
    print(f"边际成本: {marginal_cost(q0):.2f}")
    print(f"边际收益: {marginal_revenue(q0):.2f}")
    
    # 寻找最大利润点
    from scipy.optimize import minimize_scalar
    result = minimize_scalar(lambda q: -profit(q), bounds=(0, 100), method='bounded')
    optimal_q = result.x
    max_profit = profit(optimal_q)
    
    print(f"最大利润点: q = {optimal_q:.2f}")
    print(f"最大利润: {max_profit:.2f}")
```

## 7. 常见问题

### Q1: 如何判断函数在某点是否连续？

**A**: 检查函数值是否存在、极限是否存在、极限是否等于函数值。

### Q2: 连续函数是否一定有界？

**A**: 在闭区间上连续的函数必有界，但在开区间上不一定。

### Q3: 如何证明函数在某个区间上一致连续？

**A**: 使用一致连续的定义，证明对于任意ε>0，存在δ>0使得满足条件。
