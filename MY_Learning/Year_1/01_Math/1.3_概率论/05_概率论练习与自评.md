### 第一部分：古典概型与条件概率

**T1.1（摸球 + 条件概率）**
袋中有 5 个红球、3 个白球，每次不放回摸取 2 个。设 $A$ = "第一个是红球"，$B$ = "第二个是白球"。
求：$P(A)$，$P(B|A)$，$P(A\cap B)$，$P(B)$（用全概率公式验证）。

<details>
<summary>答案</summary>

$$P(A) = \frac{5}{8} = 0.625$$

$$P(B|A) = \frac{3}{7} \quad \text{（摸走一个红球，剩7球其中3白）}$$

$$P(A\cap B) = P(A) \cdot P(B|A) = \frac{5}{8} \times \frac{3}{7} = \frac{15}{56}$$

全概率验证：
$$P(B) = P(A)P(B|A) + P(A^c)P(B|A^c) = \frac{5}{8}\cdot\frac{3}{7} + \frac{3}{8}\cdot\frac{2}{7} = \frac{15+6}{56} = \frac{21}{56} = \frac{3}{8} \checkmark$$

</details>

**T1.2（贝叶斯 + 医学检测 · 量化场景）**
某种疾病在人群中的患病率是 1%。有一种检测：
- 患病者 95% 呈阳性（灵敏度）
- 健康者 90% 呈阴性（特异度）

某人检测呈阳性，求他真正患病的概率。

<details>
<summary>答案</summary>

设 $D$ = 患病，$T$ = 阳性

$$P(D|T) = \frac{P(D) \cdot P(T|D)}{P(D)P(T|D) + P(D^c)P(T|D^c)} = \frac{0.01 \times 0.95}{0.01 \times 0.95 + 0.99 \times 0.10} = \frac{0.0095}{0.0095 + 0.099} \approx 8.8\%$$

> **启示**：即使检测"准确"，先验概率低时阳性预测值也很小。这在金融信用评分、医学检测中都极其重要。

```python
# Python验证
p_d = 0.01
p_t_given_d = 0.95
p_t_given_not_d = 0.10
p_d_given_t = (p_d * p_t_given_d) / (p_d * p_t_given_d + (1-p_d) * p_t_given_not_d)
print(f"P(患病|阳性) = {p_d_given_t:.4f}")  # 0.0872
```

</details>

### 第二部分：随机变量与分布

**T2.1（离散分布 · 泊松 + 二项）**
量化场景：某股票每天的涨跌次数近似服从泊松分布，均值 $\lambda = 2$（每天平均涨跌2次）。

（1）求某天恰好涨跌 3 次的概率；
（2）若观察 10 天，每天涨跌次数独立，以 $X$ 表示总涨跌次数，求 $P(X \leq 15)$。

<details>
<summary>答案</summary>

（1）$P(X=3) = \frac{e^{-2} \cdot 2^3}{3!} \approx 0.180$

```python
from scipy.stats import poisson
print(poisson.pmf(3, 2))  # 0.180
```

（2）10天总涨跌次数 $X \sim \text{Poisson}(10 \times 2) = \text{Poisson}(20)$
$P(X \leq 15) = F(15; 20)$

```python
from scipy.stats import poisson
print(poisson.cdf(15, 20))  # ≈ 0.1565
```

</details>

**T2.2（正态分布 · 金融应用）**
假设某股票日对数收益率 $r \sim N(0.0005, 0.02^2)$（均值 0.05%/天，标准差 2%/天）。

（1）求日收益率超过 3% 的概率；
（2）求 VaR（99% 单日风险价值），即 $P(r < -VaR) = 1\%$；
（3）若交易 252 天，投资组合收益近似服从什么分布？求 99% 年化 VaR。

<details>
<summary>答案</summary>

```python
import scipy.stats as stats
import numpy as np

mu = 0.0005
sigma = 0.02

# (1)
p_exceed_3pct = 1 - stats.norm.cdf(0.03, mu, sigma)
print(f"P(r > 3%) = {p_exceed_3pct:.4f}")  # ≈ 0.0668

# (2) VaR: P(r < -VaR) = 0.01
z_99 = stats.norm.ppf(0.01)  # -2.326
daily_var = -(mu + z_99 * sigma)
print(f"单日99% VaR = {daily_var:.4f}")  # ≈ 0.0465 (4.65%)

# (3) 252天收益
mu_yr = mu * 252
sigma_yr = sigma * np.sqrt(252)
print(f"年化均值={mu_yr:.4f}, 年化波动率={sigma_yr:.4f}")

annual_var = -(mu_yr + z_99 * sigma_yr)
print(f"年化99% VaR = {annual_var:.4f}")  # ≈ 0.737 (73.7%)
```

> **注**：这是简化版 VaR 计算（不考虑收益率的厚尾性）。真实量化中还需考虑 GARCH 模型、Historical Simulation 等方法。

</details>

### 第三部分：大数定律与中心极限定理

**T3.1（CLT 验证 · 量化场景）**
模拟：用 Python 生成 10000 组样本，每组 $n=30$ 个来自指数分布 $Exp(1)$ 的随机数，计算每组的均值。验证：30个指数分布样本的均值近似服从 $N(1, 1/30)$。

```python
import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt

np.random.seed(42)
n = 30
k = 10000
sample_means = [np.mean(np.random.exponential(1, n)) for _ in range(k)]

# 理论分布
mu, se = 1, 1/np.sqrt(30)

# 画直方图 + 理论正态曲线
fig, ax = plt.subplots(figsize=(10, 6))
ax.hist(sample_means, bins=50, density=True, alpha=0.6, label='样本均值分布')
x = np.linspace(0.5, 1.5, 200)
ax.plot(x, stats.norm.pdf(x, mu, se), 'r-', lw=2, label=f'N(μ={mu}, σ²={1/n:.4f})')
ax.set_title('CLT验证：30个指数分布样本的均值分布')
ax.legend()
plt.savefig('CLT_verification.png', dpi=150)
plt.show()
```

**T3.2（大数定律 · 赌徒破产场景）**
某赌徒每局以概率 $p=0.5$ 赢 1 元，以概率 $q=0.5$ 输 1 元。初始资金 $X_0 = 10$ 元。玩 1000 局后资金 $X_{1000}$ 的期望是多少？若 $p=0.49$ 呢？

<details>
<summary>分析</summary>

每局净收益 $Y_i$：$P(Y_i=+1)=p$，$P(Y_i=-1)=q$，$E[Y_i] = p-q$

- $p=0.5$：$E[Y_i]=0$，$E[X_{1000}] = 10 + 1000 \times 0 = 10$
- $p=0.49$：$E[Y_i] = -0.02$，$E[X_{1000}] = 10 - 20 = -10$（破产！）

```python
import numpy as np
np.random.seed(42)

def gambler(p, x0, n):
    return x0 + np.sum(np.random.choice([1, -1], size=n, p=[p, 1-p]))

# 模拟10000次
results_50 = [gambler(0.50, 10, 1000) for _ in range(10000)]
results_49 = [gambler(0.49, 10, 1000) for _ in range(10000)]

print(f"p=0.50: 均值={np.mean(results_50):.2f}, 破产率={np.mean(np.array(results_50)<=0):.3f}")
print(f"p=0.49: 均值={np.mean(results_49):.2f}, 破产率={np.mean(np.array(results_49)<=0):.3f}")
```

> **关键洞察**：期望值为正不代表你更可能赢钱——这是一个经典随机游走问题。弱大数定律告诉我们均值会收敛到期望，但强大数定律告诉我们 sample path 仍然会有巨大的波动。这对理解"久赌必输"非常重要。

</details>
