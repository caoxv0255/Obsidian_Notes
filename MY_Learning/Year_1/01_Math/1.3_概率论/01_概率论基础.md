# 01 概率论基础

## 1.1 概率空间

### 1.1.1 基本概念

**随机试验（Random Experiment）**：在相同条件下可以重复进行，但结果事先不可知的试验。记作 $E$。

- $E_1$：抛一枚硬币，观察正面还是反面
- $E_2$：掷一颗骰子，观察点数
- $E_3$：记录某路口一天内的交通事故数

**样本空间（Sample Space）**：随机试验所有可能结果的集合，记作 $\Omega$。

- 抛硬币：$\Omega = \{正面, 反面\}$ 或 $\Omega = \{H, T\}$
- 掷骰子：$\Omega = \{1, 2, 3, 4, 5, 6\}$
- 交通事故数：$\Omega = \{0, 1, 2, 3, \cdots\}$

**样本点（Sample Point）**：样本空间中的每一个元素，记作 $\omega$。

**随机事件（Random Event）**：样本空间的任意子集，记作 $A, B, C\cdots$。当随机试验的结果属于该子集时，称事件发生。

- 必然事件：$\Omega$ 本身
- 不可能事件：$\emptyset$

### 1.1.2 事件的关系与运算

| 关系/运算 | 记号 | 含义 |
|-----------|------|------|
| 包含 | $A \subset B$ | $A$ 发生则 $B$ 必然发生 |
| 相等 | $A = B$ | $A \subset B$ 且 $B \subset A$ |
| 和事件 | $A \cup B$ | $A$ 或 $B$ 至少有一个发生 |
| 积事件 | $A \cap B$ 或 $AB$ | $A$ 和 $B$ 同时发生 |
| 差事件 | $A - B$ | $A$ 发生而 $B$ 不发生 |
| 互斥 | $A \cap B = \emptyset$ | $A$ 和 $B$ 不能同时发生 |
| 对立 | $B = \Omega - A$ | $A$ 和 $B$ 互斥且 $A \cup B = \Omega$ |

> **德摩根律**：$\overline{A \cup B} = \bar{A} \cap \bar{B}$，$\overline{A \cap B} = \bar{A} \cup \bar{B}$

### 1.1.3 概率的公理化定义（Kolmogorov 公理）

设 $\Omega$ 为样本空间，$\mathcal{F}$ 为事件域（$\Omega$ 的子集构成的 $\sigma$-代数），概率 $P$ 是定义在 $\mathcal{F}$ 上的函数，满足：

1. **非负性**：$P(A) \geq 0$，对任意 $A \in \mathcal{F}$
2. **规范性**：$P(\Omega) = 1$
3. **可列可加性**：若 $A_1, A_2, \cdots$ 两两互斥，则 $P\left(\bigcup_{i=1}^{\infty} A_i\right) = \sum_{i=1}^{\infty} P(A_i)$

由此可推出概率的基本性质：
- $P(\emptyset) = 0$
- $P(\bar{A}) = 1 - P(A)$
- 若 $A \subset B$，则 $P(A) \leq P(B)$
- **有限可加性**：若 $A_1, \cdots, A_n$ 两两互斥，则 $P(A_1 \cup \cdots \cup A_n) = \sum_{i=1}^{n} P(A_i)$
- $P(A \cup B) = P(A) + P(B) - P(AB)$（容斥原理）

## 1.2 古典概型

### 1.2.1 定义

若随机试验满足：
1. 样本空间 $\Omega$ 只包含有限个样本点：$|\Omega| = n < \infty$
2. 每个基本事件（每个样本点）发生的可能性相同

则称这类问题为**古典概型**（Classical Probability Model）。

在这种模型下，事件 $A$ 的概率为：
$$P(A) = \frac{|A|}{|\Omega|} = \frac{\text{有利于 } A \text{ 的基本事件数}}{\text{基本事件总数}}$$

### 1.2.2 排列与组合

**排列（有序）**：
- 不放回排列：$P_n^k = \frac{n!}{(n-k)!}$
- 全排列：$n!$

**组合（无序）**：
$$C_n^k = \binom{n}{k} = \frac{n!}{k!(n-k)!}$$

> **注意区分**：摸球时"不放回"用组合，"放回摸球"（每次摸完放回再摸）则用排列。

### 1.2.3 例题

**例题 1**（摸球问题）：口袋中有 5 个白球和 3 个黑球，不放回地摸取 3 个球。求：
（1）全是白球的概率；
（2）恰好有 2 个白球的概率。

**解**：
总的基本事件数：从 8 个球中任取 3 个，$C_8^3 = 56$。

（1）全是白球：$C_5^3 = 10$ 种，$P = \frac{C_5^3}{C_8^3} = \frac{10}{56} = \frac{5}{28}$

（2）恰好 2 个白球：先选 2 个白球 $C_5^2$，再选 1 个黑球 $C_3^1$，
$$P = \frac{C_5^2 \cdot C_3^1}{C_8^3} = \frac{10 \times 3}{56} = \frac{30}{56} = \frac{15}{28}$$

---

**例题 2**（生日问题）：一个班有 30 名学生，求至少有两个人的生日在同一天的概率。

**解**：先求所有人生日都不同的概率。
全年按 365 天计算（不考虑闰年），基本事件总数：$365^{30}$。

有利事件：$A_{365}^{30} = \frac{365!}{335!}$。

$$P(\text{至少一对同生日}) = 1 - P(\text{全不同}) = 1 - \frac{365 \times 364 \times \cdots \times 336}{365^{30}}$$

用 Python 验证：
```python
import math
from fractions import Fraction

# 精确计算
n = 30
prob_all_different = 1
for i in range(n):
    prob_all_different *= (365 - i) / 365

prob_at_least_one_match = 1 - prob_all_different
print(f"30人中至少两人同生日的概率: {prob_at_least_one_match:.4f}")
# 输出：0.7063（约 70.6%）
```

> 惊讶吗？仅仅 30 人，就有超过 70% 的概率存在同生日！这就是**生日悖论（Birthday Paradox）**。

## 1.3 条件概率

### 1.3.1 定义

设 $A, B$ 为同一概率空间中的两个事件，且 $P(B) > 0$，则在事件 $B$ 已发生的条件下，事件 $A$ 发生的**条件概率**为：
$$P(A \mid B) = \frac{P(AB)}{P(B)}$$

类似地，若 $P(A) > 0$，则：
$$P(B \mid A) = \frac{P(AB)}{P(A)}$$

### 1.3.2 乘法定理

由条件概率定义直接得到：
$$P(AB) = P(A) \cdot P(B \mid A) = P(B) \cdot P(A \mid B)$$

**推广（乘法公式）**：对 $n$ 个事件 $A_1, A_2, \cdots, A_n$，
$$P(A_1 A_2 \cdots A_n) = P(A_1) \cdot P(A_2 \mid A_1) \cdot P(A_3 \mid A_1 A_2) \cdots P(A_n \mid A_1 \cdots A_{n-1})$$

### 1.3.3 全概率公式

设 $\Omega$ 的一个**划分**为 $B_1, B_2, \cdots, B_n$（两两互斥，且 $\bigcup_i B_i = \Omega$，$P(B_i) > 0$），则对任意事件 $A$：
$$P(A) = \sum_{i=1}^{n} P(B_i) \cdot P(A \mid B_i)$$

**理解**：把复杂事件 $A$ 拆分成若干互不相容的"路径"，分别计算各路径贡献的概率后求和。

### 1.3.4 例题

**例题**：某工厂有三条生产线：
- 生产线甲：占总产品的 50%，次品率 2%
- 生产线乙：占总产品的 30%，次品率 3%
- 生产线丙：占总产品的 20%，次品率 1%

从全部产品中随机抽取一件，求该产品为次品的概率。

**解**：设 $B_1, B_2, B_3$ 分别表示"甲、乙、丙"生产的产品，$A$ 表示"次品"。
$$P(A) = P(B_1)P(A \mid B_1) + P(B_2)P(A \mid B_2) + P(B_3)P(A \mid B_3)$$
$$= 0.5 \times 0.02 + 0.3 \times 0.03 + 0.2 \times 0.01 = 0.01 + 0.009 + 0.002 = 0.021$$

```python
# Python 验证
p_a = 0.5 * 0.02 + 0.3 * 0.03 + 0.2 * 0.01
print(f"次品率: {p_a:.3f}")  # 输出: 0.021
```

## 1.4 贝叶斯公式

### 1.4.1 公式推导

由条件概率和乘法定理可直接推出**贝叶斯公式**：

$$P(B_i \mid A) = \frac{P(B_i) \cdot P(A \mid B_i)}{\sum_{j=1}^{n} P(B_j) \cdot P(A \mid B_j)}$$

其中 $B_1, \cdots, B_n$ 是样本空间的一个划分。

### 1.4.2 先验概率与后验概率

- **先验概率（Prior Probability）**：在获得新信息之前，对事件概率的初始判断 $P(B_i)$
- **后验概率（Posterior Probability）**：在获得新信息（事件 $A$ 发生）之后，对 $P(B_i \mid A)$ 的更新

贝叶斯公式的本质就是**从结果倒推原因**。

### 1.4.3 例题

**例题（续）**：已知抽到了一件次品，求该次品来自生产线甲的概率。

**解**：$$P(\text{甲} \mid \text{次品}) = \frac{P(\text{甲}) \cdot P(\text{次品} \mid \text{甲})}{P(\text{次品})}$$
$$= \frac{0.5 \times 0.02}{0.021} = \frac{0.01}{0.021} \approx 0.4762$$

```python
# Python 验证
p_b1_a = (0.5 * 0.02) / 0.021
print(f"P(甲|次品) = {p_b1_a:.4f}")  # 输出: 0.4762
```

> 发现：虽然生产线甲产量最大（50%），但在已知"次品"这个信息后，它贡献的次品比例约为 47.6%——这比它 50% 的产量占比略低，说明甲的次品率（2%）其实低于平均水平。

## 1.5 独立性

### 1.5.1 两个事件的独立性

若 $P(AB) = P(A) \cdot P(B)$，则称事件 $A$ 与 $B$ **相互独立**。

可以证明：若 $A$ 与 $B$ 独立，则 $A$ 与 $\bar{B}$、$\bar{A}$ 与 $B$、$\bar{A}$ 与 $\bar{B}$ 也相互独立。

> **注意**：互斥与独立是不同的概念！若 $P(A) > 0, P(B) > 0$：
> - 互斥：$P(AB) = 0$，但 $P(A)P(B) > 0$，所以不可能独立
> - 独立：$P(AB) = P(A)P(B) > 0$，所以不可能互斥

### 1.5.2 多个事件的独立性

对 $n$ 个事件 $A_1, \cdots, A_n$：

1. **两两独立**：对任意 $i \neq j$，$P(A_i A_j) = P(A_i) P(A_j)$
2. **相互独立**：对任意 $k \geq 2$ 和任意 $1 \leq i_1 < \cdots < i_k \leq n$，
$$P(A_{i_1} A_{i_2} \cdots A_{i_k}) = P(A_{i_1}) P(A_{i_2}) \cdots P(A_{i_k})$$

### 1.5.3 例题

**例题**：甲、乙两人同时向同一目标射击，已知甲的命中率为 0.8，乙的命中率为 0.7。求：
（1）目标被命中的概率；
（2）已知目标被命中，恰好是甲命中的概率。

**解**：
设 $A$：甲命中，$B$：乙命中，$C$：目标被命中。

（1）目标被命中 = $A \cup B$，且 $A, B$ 相互独立：
$$P(C) = P(A \cup B) = P(A) + P(B) - P(AB) = 0.8 + 0.7 - 0.8 \times 0.7 = 0.94$$

（2）$$P(A \mid C) = \frac{P(A)}{P(C)} = \frac{0.8}{0.94} \approx 0.8511$$

注意：这里用到了 $P(AC) = P(A)$（因为 $A \subset C$），即甲命中时目标必然被命中。

## 1.6 小结

| 概念 | 核心公式 |
|------|---------|
| 条件概率 | $P(A \mid B) = P(AB) / P(B)$ |
| 乘法公式 | $P(AB) = P(A)P(B \mid A)$ |
| 全概率公式 | $P(A) = \sum P(B_i)P(A \mid B_i)$ |
| 贝叶斯公式 | $P(B_i \mid A) = \dfrac{P(B_i)P(A \mid B_i)}{\sum P(B_j)P(A \mid B_j)}$ |
| 独立性 | $P(AB) = P(A)P(B)$ |
