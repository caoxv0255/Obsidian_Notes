## 1. 离散分布与量化金融

### 1.1 伯努利分布 Bernoulli(p)

**定义**：一次试验，成功概率 $p$，失败概率 $1-p$。
$$P(X=1) = p,\quad P(X=0) = 1-p$$

**量化场景**：股价涨跌判断（上涨=1，下跌=0）；信用违约（违约=1，未违约=0）

```python
import numpy as np
from scipy import stats

# 模拟股价涨跌路径
p_up = 0.52  # 假设有轻微优势（现实往往接近0.5）
n_days = 252  # 一年交易日
n_simulations = 10000

# 用伯努利分布模拟每日涨跌
daily_returns = np.where(
    np.random.random((n_simulations, n_days)) < p_up,
    1.0,   # 上涨
    -1.0   # 下跌
)

# 一年后累计收益
annual_returns = np.prod(1 + daily_returns * 0.01, axis=1)  # 假设每日波动1%

print(f"年均收益 > 0 的比例: {np.mean(annual_returns > 1):.2%}")
# 期望：一年的漂移 = (p_up - (1-p_up)) * n_days * 0.01 = 正
# 实际：因为方差大，胜率52%并不能保证稳定盈利
```

### 1.2 二项分布 Binomial(n, p)

**定义**：$n$ 次独立伯努利试验，成功次数 $k$ 的概率：
$$P(X=k) = C_n^k p^k(1-p)^{n-k}$$

**量化场景**：
- 期权定价（二叉树模型）：股票价格在每个时间步上涨或下跌
- 信用风险：$n$ 个债券中违约的数量

```python
# 二叉树期权定价（简化版）
def binomial_option_price(S, K, T, r, sigma, n=100, option_type='call'):
    """
    S: 标的资产当前价格
    K: 行权价
    T: 到期时间（年）
    r: 无风险利率
    sigma: 波动率
    n: 时间步数
    """
    dt = T / n
    u = np.exp(sigma * np.sqrt(dt))    # 上涨因子
    d = 1 / u                           # 下跌因子
    p = (np.exp(r * dt) - d) / (u - d) # 风险中性概率
    
    # 股票价格树（只记录最终节点）
    k = np.arange(n + 1)
    stock_prices = S * (u ** (n - k)) * (d ** k)
    
    # 期权 payoff
    if option_type == 'call':
        option_values = np.maximum(stock_prices - K, 0)
    else:
        option_values = np.maximum(K - stock_prices, 0)
    
    # 倒推计算期权价格
    for i in range(n - 1, -1, -1):
        option_values = np.exp(-r * dt) * (p * option_values[:-1] + (1-p) * option_values[1:])
    
    return option_values[0]

# 示例：Apple 150美元行权价，1年期限
price = binomial_option_price(S=150, K=150, T=1, r=0.05, sigma=0.25, n=100)
print(f"看涨期权价格: ${price:.2f}")
```

### 1.3 泊松分布 Poisson(λ)

**定义**：单位时间/空间内随机事件发生次数的分布。
$$P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$$

**量化场景**：
- 稀有事件建模：市场崩盘、突发信用违约
- 跳扩散模型（Jump Diffusion）：Merton模型中股票价格的跳跃部分

```python
# 泊松过程模拟：市场崩盘次数
def simulate_jump_diffusion(S0=100, mu=0.0001, sigma=0.02, 
                            lambda_jump=2, jump_mean=-0.05, jump_std=0.1,
                            n_days=252):
    """
    跳扩散过程模拟（Merton模型）
    dS/S = (mu - lambda*jump_mean) * dt + sigma * dW + q * dN
    q ~ N(jump_mean, jump_std^2), N ~ Poisson(lambda)
    """
    prices = [S0]
    S = S0
    
    for day in range(n_days):
        # 扩散部分（几何布朗运动）
        Z = np.random.standard_normal()
        dS_diffusion = (mu - 0.5 * sigma**2) + sigma * Z
        
        # 跳跃部分
        N = np.random.poisson(lambda_jump / n_days)  # 每天的跳跃次数期望
        q = 0
        if N > 0:
            q = np.sum(np.random.normal(jump_mean, jump_std, N))
        
        S = S * np.exp(dS_diffusion + q)
        prices.append(S)
    
    return np.array(prices)

# 模拟并可视化
prices = simulate_jump_diffusion()
# 注意：有跳跃的日子价格会有不连续的大幅变动
```

## 2. 连续分布与量化金融

### 2.1 正态分布 N(μ, σ²) — 核心地位

正态分布在量化金融中无处不在（尽管实际数据往往有厚尾）：

**经典应用**：
- 股价收益率假设：$r_t \sim N(\mu, \sigma^2)$
- 投资组合理论：多资产收益率的联合分布
- Black-Scholes 推导核心假设

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 股价的几何布朗运动模拟
def simulate_gbm(S0, mu, sigma, T=1, n_steps=252, n_paths=5):
    """几何布朗运动模拟股价路径"""
    dt = T / n_steps
    t = np.linspace(0, T, n_steps + 1)
    
    # 生成随机增量
    Z = np.random.standard_normal((n_paths, n_steps))
    
    # 累计路径（使用Brownian桥的性质）
    increments = (mu - 0.5 * sigma**2) * dt + sigma * np.sqrt(dt) * Z
    log_prices = np.log(S0) + np.cumsum(increments, axis=1)
    prices = S0 * np.exp(np.hstack([np.zeros((n_paths, 1)), np.exp(log_prices)]))
    
    return t, prices

t, paths = simulate_gbm(S0=100, mu=0.1, sigma=0.2, n_paths=5)
for i in range(5):
    plt.plot(t, paths[i], alpha=0.7)
plt.title('几何布朗运动模拟股价路径')
plt.xlabel('时间（年）')
plt.ylabel('股价')
plt.show()

# 正态分布检验：收益率的正态性
returns = np.diff(np.log(paths[0]))  # 对数收益率
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
axes[0].hist(returns, bins=30, density=True, alpha=0.6)
x = np.linspace(-0.05, 0.05, 100)
axes[0].plot(x, stats.norm.pdf(x, np.mean(returns), np.std(returns)), 'r-', lw=2)
axes[0].set_title('收益率直方图 vs 正态分布拟合')

stats.probplot(returns, dist="norm", plot=axes[1])
axes[1].set_title('Q-Q图（检验正态性）')
plt.tight_layout()
plt.show()

# Q-Q图判断：如果数据偏离直线，说明非正态
# 金融数据通常在尾部偏离（厚尾现象）
```

### 2.2 对数正态分布 — 股价模型

股价 $S_t$ 不能为负，假设 $S_t$ 服从**对数正态**而非正态：
$$S_t = S_0 e^{X_t}, \quad X_t \sim N(\mu t, \sigma^2 t)$$

**为什么用对数正态？**
- $S_t > 0$ 自动保证
- 收益率 $r_t = \ln(S_t/S_{t-1}) \sim N$ 符合直观
- 与 Black-Scholes 模型兼容

```python
# 对数正态分布的性质
S0 = 100
mu_log = 0.08   # 年化对数收益率均值
sigma = 0.20    # 年化波动率
T = 1           # 1年

# 1年后股价分布参数
mu_S = S0 * np.exp(mu_log * T)  # 期望
var_S = S0**2 * np.exp(2*mu_log*T) * (np.exp(sigma**2 * T) - 1)
std_S = np.sqrt(var_S)

print(f"1年后股价期望: ${mu_S:.2f}")
print(f"1年后股价标准差: ${std_S:.2f}")
print(f"概率 S_T < 0（对数正态保证为0）")
```

### 2.3 t分布 — 金融厚尾建模

金融收益率的尾部比正态分布"更厚"（极端事件发生概率更高）。t分布比正态有更大的尾部，适合描述收益率。

```python
# t分布 vs 正态分布：厚尾对比
fig, ax = plt.subplots(1, 1, figsize=(10, 5))
x = np.linspace(-5, 5, 1000)

for df in [3, 5, 10, 30]:
    ax.plot(x, stats.t.pdf(x, df), label=f't(df={df})', alpha=0.8)

ax.plot(x, stats.norm.pdf(x), 'k--', lw=2, label='正态 N(0,1)')
ax.set_title('t分布 vs 正态分布（厚尾现象）')
ax.set_xlabel('x')
ax.set_ylabel('概率密度')
ax.legend()
ax.set_xlim(-5, 5)
ax.grid(True, alpha=0.3)
plt.show()

# 5%分位数对比（VaR计算）
print(f"5%分位数对比（左侧风险）：")
for df in [3, 5, 10, 30]:
    t_5 = stats.t.ppf(0.05, df)
    norm_5 = stats.norm.ppf(0.05)
    print(f"  t(df={df}): {t_5:.4f}  vs  正态: {norm_5:.4f}  → t分布尾部更左（风险更大）")
```

## 3. 随机变量变换

### 3.1 简单收益率 vs 对数收益率

```python
prices = np.array([100, 110, 105, 115, 112])
simple_returns = np.diff(prices) / prices[:-1]
log_returns = np.diff(np.log(prices))

print("简单收益率:", simple_returns)
print("对数收益率:", log_returns)
print("差异：简单收益率有'算术叠加'性质，对数收益率有'时间可加'性质")
print(f"\n1年总简单收益率: {np.prod(1+simple_returns) - 1:.4f}")
print(f"1年总对数收益率: {np.sum(log_returns):.4f}")
```

## 4. 与两个方向的关联

### 量化金融方向

| 分布 | 金融应用 |
|------|----------|
| Bernoulli | 股价涨跌、信用违约 |
| Binomial | 二叉树期权定价、信用组合 |
| Poisson | 稀有事件、跳扩散模型 |
| 正态分布 | 布朗运动、Black-Scholes |
| 对数正态 | 股价建模 |
| t分布 | 收益率厚尾建模 |
| 指数分布 | 等待时间建模（期权到期） |

### 计算神经科学方向

| 分布 | 神经科学应用 |
|------|-------------|
| Poisson | 神经元放电建模（最核心！） |
| 正态分布 | 神经噪声、感觉编码 |
| Beta分布 | 神经反应的概率建模 |
| Gamma分布 | 神经元不应期建模 |
| t分布 | 神经数据的厚尾噪声 |

> **共性洞见**：Poisson 分布同时是量化金融（信用违约、跳扩散）和计算神经科学（神经元放电）的核心分布——这是两个方向交汇的数学基础之一。
