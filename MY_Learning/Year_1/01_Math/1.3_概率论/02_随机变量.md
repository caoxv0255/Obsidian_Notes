# 02 随机变量

## 2.1 随机变量的概念

随机变量是将随机试验的结果映射为实数的函数 X: Omega -> R。它的引入让我们能用微积分和代数工具系统地研究概率问题。

随机变量分为两大类：
- **离散随机变量**：取值有限个或可列无限个（能逐个列举）
- **连续随机变量**：取值充满某个区间或实数集（不可列）

常见离散随机变量：抛硬币（0/1）、掷骰子（1~6）、一小时内到达的电话数。
常见连续随机变量：测量误差、人的身高、液体温度。

## 2.2 离散随机变量

### 分布律（概率质量函数）

设离散随机变量 X 的所有可能取值为 x1, x2, ..., 则分布律为 P(X = xk) = pk，必须满足 pk >= 0 且 sum(pk) = 1。

通常用表格表示：

| X | x1 | x2 | x3 | ... |
|---|---|---|---|---|
| P | p1 | p2 | p3 | ... |

### 分布函数

F(x) = P(X <= x) = sum(xk <= x) pk。

性质：单调非递减、右连续，lim F(x) = 0 (x->-inf), lim F(x) = 1 (x->+inf), P(a < X <= b) = F(b) - F(a)。

### 常见离散分布

#### （1）两点分布 Bernoulli(p)

P(X=1) = p, P(X=0) = 1-p。期望 E(X) = p, 方差 D(X) = p(1-p)。

应用：一次伯努利试验（抛一次硬币、一次产品抽检）。

```python
from scipy.stats import bernoulli
p = 0.3
mean, var = bernoulli.mean(p), bernoulli.var(p)
print(f'期望={mean}, 方差={var}')
print(f'P(X=1)={bernoulli.pmf(1, p)}, P(X=0)={bernoulli.pmf(0, p)}')
# 输出：期望=0.3, 方差=0.21
# P(X=1)=0.3, P(X=0)=0.7
```

#### （2）二项分布 B(n, p)

n 次独立重复伯努利试验中成功次数的分布。记作 X ~ B(n, p)。

P(X=k) = C(n,k) * p^k * (1-p)^(n-k), k = 0,1,...,n

性质：
- E(X) = np
- D(X) = np(1-p)
- 可加性：独立二项分布相加仍为二项分布

**例题**：产品合格率为95%，抽10件。求恰有2件不合格的概率和至多1件不合格的概率。

解：n=10, p=0.05

（1）P(X=2) = C(10,2) * 0.05^2 * 0.95^8 = 45 * 0.0025 * 0.6634 = 0.0746

（2）P(X<=1) = P(X=0) + P(X=1) = 0.9138

```python
from scipy.stats import binom
n, p = 10, 0.05
print(f'P(X=2) = {binom.pmf(2, n, p):.4f}')   # 0.0746
print(f'P(X<=1) = {binom.cdf(1, n, p):.4f}')  # 0.9138
```

> **注意**：当 n 很大、p 很小时，二项分布可以用泊松分布近似（见下文泊松定理）。

#### （3）泊松分布 P(lambda)

P(X=k) = lambda^k * e^(-lambda) / k!, k = 0,1,2,...

记作 X ~ P(lambda)，lambda > 0 为参数。

性质：
- E(X) = lambda
- D(X) = lambda（均值和方差相等！）
- 可加性：独立泊松分布相加仍为泊松分布

**泊松定理（二项分布的泊松近似）**：当 n -> inf, p -> 0, 且 np = lambda 常数时：
C(n,k) * p^k * (1-p)^(n-k) -> lambda^k * e^(-lambda) / k!

应用场景：稀有事件——单位时间内到达的顾客数、放射性衰变数、布匹上的瑕疵数。

**例题**：10000人参保，每人年死亡率0.002，死亡人数不超过15的概率。

解：n=10000, p=0.002, lambda = np = 20。

```python
from scipy.stats import poisson
lambda_ = 20
prob = poisson.cdf(15, lambda_)
print(f'P(X<=15) = {prob:.4f}')  # 0.1565
```

#### （4）几何分布 Geom(p)

首次成功出现时的试验次数。记作 X ~ Geom(p)。

P(X=k) = (1-p)^(k-1) * p, k = 1,2,3,...

性质：
- E(X) = 1/p
- D(X) = (1-p)/p^2
- 无记忆性：P(X>m+n | X>m) = P(X>n)

```python
from scipy.stats import geom
p = 0.3
print(f'E(X) = {geom.mean(p):.2f}')       # 3.33
print(f'D(X) = {geom.var(p):.4f}')        # 1.1111
print(f'P(X=5) = {geom.pmf(5, p):.4f}')  # 0.0720
```

## 2.3 连续随机变量

### 概率密度函数（PDF）

存在非负函数 f(x)，使得 P(a <= X <= b) = integral_a^b f(x) dx。

性质：
1. f(x) >= 0，对所有 x
2. integral_-inf^+inf f(x) dx = 1
3. 对任意 x, P(X=x) = 0（点概率为零！）

> **重要**：连续随机变量在单点处的概率恒为零，因此 P(a < X < b) = P(a <= X <= b)。

### 分布函数（CDF）

F(x) = P(X <= x) = integral_-inf^x f(t) dt。

性质：F(x) 单调递增, F prime(x) = f(x)（在 f 连续处），P(a < X <= b) = F(b) - F(a)。

### 常见连续分布

#### （1）均匀分布 U(a, b)

f(x) = 1/(b-a), a <= x <= b; 0 otherwise

F(x) = 0 (x<a), (x-a)/(b-a) (a<=x<b), 1 (x>=b)

性质：E(X) = (a+b)/2, D(X) = (b-a)^2/12。

```python
from scipy.stats import uniform
a, b = 2, 5
print(f'E(X) = {uniform.mean(loc=a, scale=b-a):.2f}')  # 3.50
print(f'D(X) = {uniform.var(loc=a, scale=b-a):.4f}')   # 0.7500
```

#### （2）指数分布 Exp(lambda)

f(x) = lambda * e^(-lambda*x), x >= 0
F(x) = 1 - e^(-lambda*x), x >= 0

性质：
- E(X) = 1/lambda
- D(X) = 1/lambda^2
- 无记忆性：P(X > s+t | X > s) = P(X > t)

应用：元器件寿命、等待时间（与泊松过程密切相关）。

```python
from scipy.stats import expon
lambda_ = 2
print(f'E(X) = {expon.mean(scale=1/lambda_):.2f}')         # 0.50
print(f'P(X<=0.5) = {expon.cdf(0.5, scale=1/lambda_):.4f}') # 0.6321
```

#### （3）正态分布 N(mu, sigma^2)（最重要！）

f(x) = (1/sqrt(2*pi*sigma^2)) * exp(-(x-mu)^2 / (2*sigma^2))

记作 X ~ N(mu, sigma^2)。

性质：
- 图像关于 x=mu 对称，峰值在 mu 处，拐点在 mu +- sigma 处
- E(X) = mu, D(X) = sigma^2

重要数值（标准正态 Z ~ N(0,1)）：

| k | P(|Z|<k) |
|---|----------|
| 1 | 68.26% |
| 2 | 95.44% |
| 3 | 99.74% |

**标准化变换**：任何正态分布可通过 Z = (X - mu) / sigma ~ N(0,1) 转化为标准正态分布。

P(a < X < b) = Phi((b-mu)/sigma) - Phi((a-mu)/sigma)

**例题**：成年男性身高 X ~ N(170, 8^2)。求：
（1）身高在160-180cm之间的概率；
（2）身高超过185cm的概率。

解：（1）P(160<X<180) = Phi(1.25) - Phi(-1.25) = 2*Phi(1.25) - 1
（2）P(X>185) = 1 - Phi(1.875)

```python
from scipy.stats import norm
mu, sigma = 170, 8

prob1 = norm.cdf(180, mu, sigma) - norm.cdf(160, mu, sigma)
prob2 = 1 - norm.cdf(185, mu, sigma)
print(f'P(160<X<180) = {prob1:.4f}')  # 0.7888
print(f'P(X>185) = {prob2:.4f}')       # 0.0304
```

## 2.4 小结

| 分布 | 记号 | 期望 | 方差 | 适用场景 |
|------|------|------|------|----------|
| 两点分布 | B(1,p) | p | p(1-p) | 一次伯努利试验 |
| 二项分布 | B(n,p) | np | np(1-p) | n次独立重复试验 |
| 几何分布 | Geom(p) | 1/p | (1-p)/p^2 | 首次成功试验次数 |
| 泊松分布 | P(lambda) | lambda | lambda | 稀有事件计数 |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)^2/12 | 取值均匀 |
| 指数分布 | Exp(lambda) | 1/lambda | 1/lambda^2 | 寿命、等待时间 |
| 正态分布 | N(mu,sigma^2) | mu | sigma^2 | 大量独立因素叠加 |

## 附：常用分布的 Python 速查

```python
from scipy import stats

# 二项分布 B(10, 0.3)
print(f"B(10,0.3): mean={stats.binom.mean(10,0.3):.2f}, P(X=3)={stats.binom.pmf(3,10,0.3):.4f}")
# 输出: B(10,0.3): mean=3.00, P(X=3)=0.2668

# 泊松分布 P(4)
print(f"P(4): mean={stats.poisson.mean(4):.2f}, P(X=2)={stats.poisson.pmf(2,4):.4f}")
# 输出: P(4): mean=4.00, P(X=2)=0.1465

# 正态分布 N(0,1)
print(f"N(0,1): Phi(1)={stats.norm.cdf(1):.4f}, phi(0)={stats.norm.pdf(0):.4f}")
# 输出: N(0,1): Phi(1)=0.8413, phi(0)=0.3989

# 指数分布 Exp(1/3)
print(f"Exp(1/3): E(X)={stats.expon.mean(scale=3):.2f}, P(X<=2)={stats.expon.cdf(2,scale=3):.4f}")
# 输出: Exp(1/3): E(X)=3.00, P(X<=2)=0.4866
```
