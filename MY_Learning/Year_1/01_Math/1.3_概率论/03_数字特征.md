# 03 数字特征

## 3.1 数学期望

### 3.1.1 定义

**离散随机变量的期望**：
若 X 的分布律为 P(X = xk) = pk，k = 1, 2, ...，则
$$E(X) = \sum_{k} x_k \cdot p_k$$
要求级数绝对收敛，即 \sum |xk| * pk < ∞。

**连续随机变量的期望**：
若 X 的概率密度为 f(x)，则
$$E(X) = \int_{-\infty}^{+\infty} x \cdot f(x) \, dx$$
要求积分绝对收敛。

> **为什么要求"绝对收敛"？** 期望是随机变量"长期平均"的数学抽象。如果级数/积分条件收敛但不绝对收敛，则求和/积分的次序会影响结果，这与期望的客观含义相冲突。

### 3.1.2 期望的性质（对所有随机变量类型通用）

设 X, Y 为随机变量，c 为常数：

1. **线性性**：E(cX + Y) = cE(X) + E(Y)
2. **常数不变**：E(c) = c
3. **若 X >= 0，则 E(X) >= 0**
4. **若 X 与 Y 独立，则 E(XY) = E(X) * E(Y)**

> 注意：第4条中"独立"条件不可省！不独立的随机变量乘积期望不等于各自期望的乘积。

### 3.1.3 随机变量函数的期望

不需要知道 X 的分布，直接通过变换求 Y = g(X) 的期望：

**离散型**：E(g(X)) = \sum g(xk) * pk
**连续型**：E(g(X)) = \int_{-\infty}^{+\infty} g(x) * f(x) dx

### 3.1.4 例题

**例题 1**：投掷一枚均匀骰子，求点数 X 的期望。

解：X ~ 均匀分布 {1,2,3,4,5,6}，每个点数概率 = 1/6
E(X) = 1*(1/6) + 2*(1/6) + ... + 6*(1/6) = 3.5

```python
# 验证
X_vals = [1, 2, 3, 4, 5, 6]
probs = [1/6] * 6
E_X = sum(x * p for x, p in zip(X_vals, probs))
print(f'E(X) = {E_X:.2f}')  # 3.50
```

**例题 2**：设 X ~ N(mu, sigma^2)，求 E(X^2)。

解：利用 E(X^2) = D(X) + [E(X)]^2 = sigma^2 + mu^2

```python
from scipy.stats import norm
import numpy as np

mu, sigma = 3, 2
# 方法1：直接积分
x = np.linspace(mu - 4*sigma, mu + 4*sigma, 10000)
dx = x[1] - x[0]
E_X2_direct = np.sum(x**2 * norm.pdf(x, mu, sigma)) * dx
print(f'E(X^2) 直接积分: {E_X2_direct:.4f}')

# 方法2：用公式
E_X2_formula = sigma**2 + mu**2
print(f'E(X^2) 公式: {E_X2_formula:.4f}')
# 输出：13.0000
```

## 3.2 方差

### 3.2.1 定义

方差描述随机变量 X 围绕其期望 E(X) 的离散程度（波动大小）。

$$D(X) = \text{Var}(X) = E\left[(X - E(X))^2\right] = E(X^2) - [E(X)]^2$$

后一个等式称为**方差计算公式**，使用更方便。

方差的平方根 \sqrt{D(X)} 称为**标准差**，记作 \sigma(X)。

### 3.2.2 方差的性质

设 X, Y 为随机变量，c 为常数：

1. **D(c) = 0**（常数的波动为零）
2. **D(cX) = c^2 * D(X)**
3. **D(X + c) = D(X)**（平移不改变方差）
4. **若 X 与 Y 独立**：D(X + Y) = D(X) + D(Y)

> **注意第4条**：独立条件下方差可加！注意与期望线性性的区别（期望的线性性不需要独立条件）。

**重要推论**：若 X1, X2, ..., Xn 两两独立，Xi 的方差为 Di，则
$$D\left(\sum_{i=1}^{n} X_i\right) = \sum_{i=1}^{n} D(X_i)$$

### 3.2.3 常见分布的期望与方差

| 分布 | 记号 | E(X) | D(X) |
|------|------|------|------|
| 两点分布 | B(1,p) | p | p(1-p) |
| 二项分布 | B(n,p) | np | np(1-p) |
| 几何分布 | Geom(p) | 1/p | (1-p)/p^2 |
| 泊松分布 | P(lambda) | lambda | lambda |
| 均匀分布 | U(a,b) | (a+b)/2 | (b-a)^2/12 |
| 指数分布 | Exp(lambda) | 1/lambda | 1/lambda^2 |
| 正态分布 | N(mu,sigma^2) | mu | sigma^2 |

### 3.2.4 例题

**例题**：设 X 服从参数为 p 的几何分布，求 D(X)。

解：已知 E(X) = 1/p。先求 E(X^2)。

几何分布：P(X=k) = (1-p)^(k-1) * p, k = 1, 2, ...

利用幂级数求和：
E(X(X-1)) = p * (1-p) * 2! / p^3 = 2(1-p)/p^2
E(X^2) = E(X(X-1)) + E(X) = 2(1-p)/p^2 + 1/p = (2-2p+p)/p^2 = (2-p)/p^2

D(X) = E(X^2) - [E(X)]^2 = (2-p)/p^2 - 1/p^2 = (1-p)/p^2

```python
from scipy.stats import geom
import numpy as np

for p in [0.1, 0.3, 0.5]:
    mean = geom.mean(p)
    # 用 E(X^2) - [E(X)]^2 计算方差
    # E(X^2) = sum k^2 * P(X=k)
    k = np.arange(1, 1000)
    E_X2 = np.sum(k**2 * geom.pmf(k, p))
    var = E_X2 - mean**2
    print(f'p={p}: E(X)={mean:.4f}, D(X)={var:.4f}, (1-p)/p^2={(1-p)/p**2:.4f}')
# 输出验证：var = (1-p)/p^2
```

**例题**：设 X ~ N(mu, sigma^2)，求 D(X)。

解：这是正态分布的基本性质，D(X) = sigma^2，证略。

## 3.3 协方差与相关系数

### 3.3.1 为什么要引入协方差？

方差只描述单个随机变量的波动。对于两个随机变量 X 和 Y，我们需要一个量来描述它们**一起波动**的趋势——这就是协方差。

### 3.3.2 协方差的定义

$$\text{Cov}(X, Y) = E\left[(X - E(X))(Y - E(Y))\right] = E(XY) - E(X)E(Y)$$

**协方差为零**：称 X 与 Y **不相关**。

> **注意区分**：
> - 独立 -> 一定不相关（独立 => Cov=0）
> - 不相关 -> 不一定独立（Cov=0 不一定独立）
> - 只有在**正态分布**情况下，不相关等价于独立

### 3.3.3 协方差的性质

1. **对称性**：Cov(X, Y) = Cov(Y, X)
2. **自协方差**：Cov(X, X) = D(X)
3. **线性性**：Cov(aX + b, Y) = a * Cov(X, Y)
4. **协方差加性**：Cov(X + Y, Z) = Cov(X, Z) + Cov(Y, Z)
5. **独立时**：若 X 与 Y 独立，则 Cov(X, Y) = 0

### 3.3.4 相关系数

协方差的大小受 X 和 Y 量纲的影响。为了消除这个影响，引入**相关系数**（Pearson 相关系数）：

$$\rho_{XY} = \frac{\text{Cov}(X, Y)}{\sqrt{D(X)} \cdot \sqrt{D(Y)}}$$

性质：
- \rho_{XY} \in [-1, 1]
- |\rho_{XY}| = 1 当且仅当 X 与 Y 存在严格线性关系（Y = aX + b）
- \rho_{XY} > 0 称正相关，\rho_{XY} < 0 称负相关

### 3.3.5 例题

**例题**：设 (X, Y) 的联合分布律为：

| X \ Y | 0 | 1 |
|--------|---|---|
| 0 | 0.1 | 0.2 |
| 1 | 0.3 | 0.4 |

求 Cov(X, Y) 和 \rho_{XY}。

解：
先求边缘分布：
P(X=0) = 0.1 + 0.2 = 0.3, P(X=1) = 0.3 + 0.4 = 0.7
P(Y=0) = 0.1 + 0.3 = 0.4, P(Y=1) = 0.2 + 0.4 = 0.6

E(X) = 0*0.3 + 1*0.7 = 0.7
E(Y) = 0*0.4 + 1*0.6 = 0.6

E(XY) = 0*0*0.1 + 0*1*0.2 + 1*0*0.3 + 1*1*0.4 = 0.4

Cov(X, Y) = E(XY) - E(X)E(Y) = 0.4 - 0.7*0.6 = 0.4 - 0.42 = -0.02

D(X) = E(X^2) - [E(X)]^2 = 0.7 - 0.49 = 0.21
D(Y) = 0.6 - 0.36 = 0.24

\rho_{XY} = -0.02 / sqrt(0.21 * 0.24) = -0.02 / 0.2245 = -0.0891

```python
# Python 验证
import numpy as np

# 联合分布
p = np.array([[0.1, 0.2], [0.3, 0.4]])
x_vals = np.array([0, 1])
y_vals = np.array([0, 1])

# 边缘分布
p_X = p.sum(axis=1)  # [0.3, 0.7]
p_Y = p.sum(axis=0)  # [0.4, 0.6]

E_X = np.sum(x_vals * p_X)
E_Y = np.sum(y_vals * p_Y)
print(f'E(X)={E_X:.2f}, E(Y)={E_Y:.2f}')

# E(XY)
E_XY = np.sum(x_vals[:, None] * y_vals[None, :] * p)
print(f'E(XY)={E_XY:.2f}')

Cov = E_XY - E_X * E_Y
print(f'Cov(X,Y)={Cov:.4f}')

D_X = np.sum(x_vals**2 * p_X) - E_X**2
D_Y = np.sum(y_vals**2 * p_Y) - E_Y**2
rho = Cov / np.sqrt(D_X * D_Y)
print(f'rho_XY={rho:.4f}')
```

**结论**：\rho_{XY} \approx -0.089$，接近于 0，说明 X 和 Y 几乎无线性相关性。

## 3.4 协方差矩阵与多元扩展

对于 n 维随机变量 (X1, X2, ..., Xn)，定义协方差矩阵：

$$\Sigma = (\sigma_{ij})_{n \times n}, \quad \sigma_{ij} = \text{Cov}(X_i, X_j)$$

协方差矩阵是对称半正定矩阵，其对角线元素是各分量的方差，非对角线元素是分量间的协方差。

**多元正态分布**：n 维正态分布 N(\mu, \Sigma)，其概率密度由均值向量 \mu 和协方差矩阵 \Sigma 完全确定。

## 3.5 小结

| 概念 | 公式 | 核心性质 |
|------|------|----------|
| 数学期望 | E(X) | 线性性、绝对收敛 |
| 方差 | D(X) = E(X^2) - [E(X)]^2 | 非负性、平移不变 |
| 协方差 | Cov(X,Y) = E(XY) - E(X)E(Y) | 对称性、独立则为零 |
| 相关系数 | \rho = Cov / (\sigma_X \sigma_Y) | [-1,1]，衡量线性相关程度 |

```python
import numpy as np
from scipy import stats

# 1. 二项分布的期望和方差验证
n, p = 10, 0.4
print(f'B({n},{p}): E(X)={stats.binom.mean(n,p):.2f}, D(X)={stats.binom.var(n,p):.2f}')

# 2. 正态分布的期望和方差验证
mu, sigma = 5, 2
print(f'N({mu},{sigma**2}): E(X)={stats.norm.mean(mu,sigma):.2f}, D(X)={stats.norm.var(mu,sigma):.2f}')

# 3. 协方差计算
X = np.array([1, 2, 3, 4, 5])
Y = np.array([2, 4, 5, 4, 5])
print(f'Cov(X,Y)={np.cov(X,Y,bias=True)[0,1]:.4f}')
print(f'rho={np.corrcoef(X,Y)[0,1]:.4f}')

# 4. 生成随机样本验证大数定律（采样均值趋近期望）
samples = stats.norm.rvs(loc=5, scale=2, size=10000)
print(f'样本均值={samples.mean():.4f}, 理论期望=5')
print(f'样本方差={samples.var():.4f}, 理论方差=4')
```
