## 1. 大数定律

大数定律是概率论中最为核心的定理之一，它描述了**大量重复试验的平均结果趋于稳定**这一直观现象。无论单个随机变量如何波动，当样本量足够大时，它们的平均值会高度集中于期望值附近。

> **生活类比**：抛一枚均匀硬币，抛10次可能7次正面，但抛10000次，正面比例几乎必然接近50%。这就是大数定律在起作用。

### 1.1 切比雪夫不等式

**切比雪夫不等式**是大数定律的先导工具，它给出了随机变量偏离期望值的概率上界。

**定理**：设随机变量 $X$ 的期望 $E(X) = \mu$，方差 $D(X) = \sigma^2$，则对任意 $\varepsilon > 0$：

$$P(|X - \mu| \geq \varepsilon) \leq \frac{\sigma^2}{\varepsilon^2}$$

等价形式：

$$P(|X - \mu| < \varepsilon) \geq 1 - \frac{\sigma^2}{\varepsilon^2}$$

**直观理解**：无论分布形状如何，只要知道方差，就能bound住偏离期望的概率。方差越小，偏离的可能性越低。

**Python验证**：

```python
import numpy as np

# 验证切比雪夫不等式：对任意分布都成立
def verify_chebyshev(n trials=100000, eps=3):
    """对不同分布验证切比雪夫不等式"""
    distributions = {
        '均匀分布': np.random.uniform(-1, 1, trials),
        '指数分布': np.random.exponential(2, trials),
        '二项分布': np.random.binomial(20, 0.3, trials),
        '泊松分布': np.random.poisson(5, trials)
    }
    
    for name, samples in distributions.items():
        theoretical_bound = 1 / (eps**2)  # sigma^2=1 (标准化后)
        empirical_prob = np.mean(np.abs(samples - np.mean(samples)) >= eps)
        print(f"{name}: 理论上界={theoretical_bound:.4f}, 经验概率={empirical_prob:.4f}")

# 运行验证
# verify_chebyshev()
# 均匀分布: 理论上界=0.1111, 经验概率≈0.11
# 指数分布: 理论上界=0.1111, 经验概率≈0.05（更紧，因为尾部更轻）
```

**应用场景**：切比雪夫不等式在质量控制和风险评估中常用于建立置信区间，给出最坏情况的概率保证。

### 1.2 弱大数定律

**弱大数定律（Weak Law of Large Numbers, WLLN）**：

设 $X_1, X_2, \ldots, X_n$ 是独立同分布的随机变量序列，$E(X_i) = \mu$，则：

$$\frac{1}{n} \sum_{i=1}^{n} X_i \xrightarrow{P} \mu \quad (n \to \infty)$$

即样本均值依概率收敛于期望。

**证明思路（基于切比雪夫）**：

1. 记 $\bar{X}_n = \frac{1}{n}\sum_{i=1}^{n} X_i$
2. $E(\bar{X}_n) = \mu$
3. $D(\bar{X}_n) = \frac{D(X_1)}{n} = \frac{\sigma^2}{n}$
4. 由切比雪夫不等式：$P(|\bar{X}_n - \mu| \geq \varepsilon) \leq \frac{\sigma^2}{n\varepsilon^2} \to 0$

**直观**：样本均值的方差随 $n$ 增大而减小，因此样本均值越来越集中于 $\mu$。

---

### 1.3 强大数定律

**强大数定律（Strong Law of Large Numbers, SLLN）**：

设 $X_1, X_2, \ldots, X_n$ 是独立同分布的随机变量，$E(X_i) = \mu$，则：

$$\frac{1}{n} \sum_{i=1}^{n} X_i \xrightarrow{a.s.} \mu \quad (n \to \infty)$$

即样本均值**几乎必然**（almost surely）收敛于期望。

**弱大数 vs 强大数**：

| 收敛类型 | 含义 | 强度 |
|----------|------|------|
| 依概率收敛 ($xrightarrow{P}$) | 对于任意 $\varepsilon$，当 $n$ 大时，偏离概率任意小 | 较弱 |
| 几乎必然收敛 ($xrightarrow{a.s.$) | 样本路径上，随着 $n$ 增大，平均值一定趋于 $\mu$ | 强 |

> 强大数定律的"强大"在于它对**每一条**样本路径都保证收敛，而不仅仅是概率意义上。

## 2. 中心极限定理

中心极限定理（Central Limit Theorem, CLT）是概率论中最令人惊叹的结果之一：**大量独立随机变量的和近似服从正态分布**，无论原始分布是什么。

### 2.1 独立同分布中心极限定理

**定理（林德伯格-莱维 CLT）**：

设 $X_1, X_2, \ldots, X_n$ 是独立同分布的随机变量，$E(X_i) = \mu$，$D(X_i) = \sigma^2 < \infty$，则：

$$\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} = \frac{\sum_{i=1}^{n} X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0, 1)$$

即**标准化后的样本均值渐近服从标准正态分布**。

**等价形式**：当 $n$ 足够大时：

$$\bar{X}_n \approx N\left(\mu, \frac{\sigma^2}{n}\right)$$

### 2.2 棣莫弗-拉普拉斯定理

**二项分布的正态近似**（CLT的特殊情况）：

设 $X \sim B(n, p)$，则：

$$\frac{X - np}{\sqrt{np(1-p)}} \xrightarrow{d} N(0, 1)$$

这意味着**二项分布可以用正态分布来近似**，当 $n$ 足够大时。

```python
import numpy as np
from scipy import stats
import matplotlib.pyplot as plt

# 演示中心极限定理：从不同分布抽样，样本均值分布都趋向正态
def demonstrate_clt(dist='uniform', n_samples=10000, sample_size=30):
    """
    从指定分布抽样，计算样本均值，观察其分布
    """
    if dist == 'uniform':
        # 均匀分布 [0, 1]
        raw_data = np.random.uniform(0, 1, (sample_size, n_samples))
    elif dist == 'exponential':
        # 指数分布 lambda=1
        raw_data = np.random.exponential(1, (sample_size, n_samples))
    elif dist == 'bernoulli':
        # 伯努利分布 p=0.3
        raw_data = np.random.binomial(1, 0.3, (sample_size, n_samples))
    
    # 每列是一个样本，计算每个样本的均值
    sample_means = np.mean(raw_data, axis=0)
    
    # 绘制样本均值的直方图
    plt.figure(figsize=(10, 6))
    plt.hist(sample_means, bins=50, density=True, alpha=0.6, label=f'{dist}分布的样本均值')
    
    # 叠加理论正态曲线
    if dist == 'uniform':
        mu, sigma = 0.5, 1/np.sqrt(12*sample_size)
    elif dist == 'exponential':
        mu, sigma = 1, 1/np.sqrt(sample_size)
    elif dist == 'bernoulli':
        mu, sigma = 0.3, np.sqrt(0.3*0.7/sample_size)
    
    x = np.linspace(mu - 4*sigma, mu + 4*sigma, 100)
    plt.plot(x, stats.norm.pdf(x, mu, sigma), 'r-', lw=2, label=f'正态 N({mu:.3f}, {sigma:.4f})')
    plt.legend()
    plt.title(f'中心极限定理演示：{dist}分布的样本均值趋于正态')
    plt.show()

# 演示不同分布的样本均值都趋向正态
for dist in ['uniform', 'exponential', 'bernoulli']:
    demonstrate_clt(dist=dist)
```

**CLT的意义**：

1. **解释了为什么正态分布在自然界如此普遍** — 大量微小独立因素叠加的结果
2. **统计推断的基础** — t检验、ANOVA等都以CLT为理论支撑
3. **工程与金融中的误差分析** — 测量误差通常是大量微小误差之和

## 3. Python模拟验证

### 3.1 大数定律模拟

```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_law_of_large_numbers(true_mean=0.5, n_trials=100):
    """
    模拟大数定律：样本均值随样本量增加趋近于真实均值
    使用均匀分布 [0, 1]，理论均值 = 0.5
    """
    sample_sizes = np.arange(1, n_trials + 1)
    cumulative_means = []
    
    running_sum = 0
    for n in sample_sizes:
        # 每次抽取一个样本
        x = np.random.uniform(0, 1)
        running_sum += x
        cumulative_means.append(running_sum / n)
    
    plt.figure(figsize=(12, 6))
    plt.plot(sample_sizes, cumulative_means, 'b-', lw=1.5, label='样本均值')
    plt.axhline(y=true_mean, color='r', linestyle='--', lw=2, label=f'真实均值 = {true_mean}')
    plt.xlabel('样本量 n')
    plt.ylabel('样本均值')
    plt.title('大数定律演示：样本均值收敛于真实均值')
    plt.legend()
    plt.grid(True, alpha=0.3)
    plt.show()

simulate_law_of_large_numbers()
```

### 3.2 中心极限定理模拟

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

def simulate_clt(original_dist='exponential', n_samples=10000, sample_sizes=[1, 5, 30, 100]):
    """
    演示中心极限定理：不同样本量下，样本均值的分布趋近正态
    """
    fig, axes = plt.subplots(2, 2, figsize=(14, 10))
    axes = axes.flatten()
    
    for idx, n in enumerate(sample_sizes):
        if original_dist == 'exponential':
            samples = np.random.exponential(scale=2, size=(n, n_samples))
        elif original_dist == 'uniform':
            samples = np.random.uniform(0, 1, (n, n_samples))
        
        sample_means = np.mean(samples, axis=0)
        mu = 2 if original_dist == 'exponential' else 0.5
        sigma = (2 / np.sqrt(n)) if original_dist == 'exponential' else (1/np.sqrt(12*n))
        
        ax = axes[idx]
        ax.hist(sample_means, bins=50, density=True, alpha=0.6, label=f'n={n}')
        x = np.linspace(mu - 4*sigma, mu + 4*sigma, 200)
        ax.plot(x, stats.norm.pdf(x, mu, sigma), 'r-', lw=2, label='正态近似')
        ax.set_title(f'样本量 n={n}\n均值={mu:.3f}, 标准差={sigma:.4f}')
        ax.legend()
        ax.grid(True, alpha=0.3)
    
    plt.suptitle(f'中心极限定理演示：{original_dist}分布的样本均值', fontsize=14)
    plt.tight_layout()
    plt.show()

simulate_clt(original_dist='exponential')
simulate_clt(original_dist='uniform')
```

## 4. 直观理解与应用

### 4.1 大数定律的应用

**保险业**：大量独立个体投保，单个个体风险不确定，但总体赔付率趋于稳定 → 可以精算定价。

**选举民调**：采访少量个体时结果波动大，样本量足够大时，民调结果接近真实支持率。

**赌场经营**：单次赌局结果不确定，但大量赌局后，赌场盈利比例趋近于某个确定值（庄家优势）。

### 4.2 中心极限定理的应用

**质量控制**：产品尺寸是大量微小因素（材料、设备、操作）之和 → 服从正态分布 → 用 $\bar{X} \pm 3\sigma$ 确定控制限。

**金融建模**：股票收益率是大量信息叠加的结果 → 正态分布假设在量化金融中广泛应用（虽然实际有厚尾问题）。

**抽样推断**：总体均值估计的抽样分布 → CLT保证了大样本下估计的有效性。

### 4.3 两个定理的区别

| 定理 | 核心结论 | 收敛方向 |
|------|----------|----------|
| 大数定律 | 样本均值 $\bar{X}_n \to \mu$ | 均值本身收敛 |
| 中心极限定理 | $\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \to N(0,1)$ | 标准化后的均值分布收敛到正态 |

> **一句话总结**：大数定律告诉我们**均值趋近哪去**，中心极限定理告诉我们**它是如何围绕那个值分布的**。

## 📝 小结

1. **切比雪夫不等式**：对任意分布，给出偏离期望的概率上界
2. **弱大数定律**：样本均值依概率收敛于期望（独立同分布情形）
3. **强大数定律**：样本均值几乎必然收敛于期望（更强的结论）
4. **中心极限定理**：大量独立随机变量和的标准化形式趋近标准正态分布

> ⚠️ **注意**：大数定律和中心极限定理都要求**独立**随机变量。相关结构会显著改变结论，这也是量化金融中"尾部风险"和"波动率聚集"等现象的来源之一。
