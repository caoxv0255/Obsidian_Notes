# 07 协方差与相关系数（量化应用）

## 核心概念

### 协方差（Covariance）

协方差衡量两个随机变量**联合变动趋势**：

$$
\text{Cov}(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]
$$

- Cov(X, Y) > 0：正相关，X 高时 Y 也倾向高
- Cov(X, Y) < 0：负相关，X 高时 Y 倾向低
- Cov(X, Y) = 0：不相关（但未必独立！）

### 相关系数（Correlation Coefficient）

协方差的问题是受量纲影响。为标准化，引入 Pearson 相关系数：

$$
\rho_{XY} = \frac{\text{Cov}(X, Y)}{\sigma_X \sigma_Y} = \frac{E[XY] - E[X]E[Y]}{\sqrt{E[X^2]-(E[X])^2} \cdot \sqrt{E[Y^2]-(E[Y])^2}}
$$

性质：
- $-1 \leq \rho_{XY} \leq 1$
- $|\rho| = 1$ 当且仅当 $Y = aX + b$（线性关系）
- $\rho = 0$ 称**不相关**，但不保证独立（独立 ⇒ 不相关，反之不成立）

## 投资组合方差公式推导

### 双资产组合

设两资产收益率 $r_1, r_2$，权重 $w_1, w_2$（且 $w_1 + w_2 = 1$），组合收益率 $r_p = w_1 r_1 + w_2 r_2$。

**组合方差：**

$$
\sigma_p^2 = \text{Var}(w_1 r_1 + w_2 r_2)
           = w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + 2 w_1 w_2 \text{Cov}(r_1, r_2)
$$

展开推导：
$$
\sigma_p^2 = w_1^2 \sigma_1^2 + w_2^2 \sigma_2^2 + 2 w_1 w_2 \rho_{12} \sigma_1 \sigma_2
$$

其中 $\rho_{12}$ 为相关系数。

**关键洞察**：
- 当 $\rho_{12} < 0$（负相关），组合方差可小于单项方差 → **分散化原理**
- 当 $\rho_{12} = 1$（完全正相关），分散化失效
- 当 $\rho_{12} = -1$（完全负相关），可构造**无风险组合**

### N 资产组合（矩阵形式）

设权重向量 $\mathbf{w} = (w_1, ..., w_n)^T$，协方差矩阵 $\Sigma$：

$$
\sigma_p^2 = \mathbf{w}^T \Sigma \mathbf{w} = \sum_{i=1}^n w_i^2 \sigma_i^2 + 2 \sum_{i<j} w_i w_j \text{Cov}(r_i, r_j)
$$

**协方差矩阵** $\Sigma$：第 $(i,j)$ 元素为 $\text{Cov}(r_i, r_j)$，对角线为单项方差。

## CAPM 的 β 定义与计算

### β 的定义

资本资产定价模型（CAPM）中，资产 $i$ 的 beta 系数衡量其相对市场的**系统风险**：

$$
\beta_i = \frac{\text{Cov}(r_i, r_m)}{\text{Var}(r_m)} = \frac{\sigma_{im}}{\sigma_m^2}
$$

其中 $r_m$ 为市场收益率。

Beta 的含义：
- $\beta_i = 1$：与市场同涨同跌
- $\beta_i > 1$：涨幅更大、跌幅也更大（进攻型）
- $\beta_i < 1$：波动小于市场（防御型）
- $\beta_i < 0$：与市场反向（很少见）

### CAPM 公式

$$
E[r_i] = r_f + \beta_i (E[r_m] - r_f)
$$

即：预期收益 = 无风险利率 + β × 市场风险溢价。

## Python 实现：计算真实市场 β

```python
import numpy as np
import pandas as pd
import yfinance as yf
from scipy import stats

# 下载数据（以苹果 AAPL vs 标普500 ETF SPY 为例）
ticker = "AAPL"
market_ticker = "SPY"
start_date = "2020-01-01"
end_date = "2024-12-31"

stock = yf.download(ticker, start=start_date, end=end_date)["Adj Close"]
market = yf.download(market_ticker, start=start_date, end=end_date)["Adj Close"]

# 计算日收益率
returns = np.log(stock / stock.shift(1)).dropna()
market_returns = np.log(market / market.shift(1)).dropna()

# 对齐日期
df = pd.DataFrame({ticker: returns, "SPY": market_returns}).dropna()

# 方法1：NumPy 手算
cov_matrix = np.cov(df[ticker], df["SPY"])
cov_stock_market = cov_matrix[0, 1]
var_market = cov_matrix[1, 1]
beta_np = cov_stock_market / var_market

# 方法2：Pandas 直接计算
beta_pd = df[ticker].cov(df["SPY"]) / df["SPY"].var()

# 方法3：SciPy 线性回归（同时得到截距 α）
slope, intercept, r_value, p_value, std_err = stats.linregress(
    df["SPY"], df[ticker]
)
beta_scipy = slope
alpha = intercept  # 年化 α = intercept * 252

print(f"β (NumPy)  = {beta_np:.4f}")
print(f"β (Pandas) = {beta_pd:.4f}")
print(f"β (SciPy)  = {beta_scipy:.4f}")
print(f"α (日)     = {alpha:.6f}, 年化 α = {alpha*252:.2%}")
print(f"R²         = {r_value**2:.4f}")
print(f"p-value    = {p_value:.4e}")

# 年化市场超额收益（假设无风险利率 4%）
rf = 0.04 / 252  # 日度无风险利率
market_excess = df["SPY"].mean() - rf
expected_return = rf + beta_scipy * market_excess * 252
print(f"\n年化预期收益率 ≈ {expected_return:.2%}")
```

**输出示例**（AAPL 2020-2024）：
```
β (NumPy)  = 1.2341
β (Pandas) = 1.2341
β (SciPy)  = 1.2341
α (日)     = 0.000523, 年化 α = 13.18%
R²         = 0.4821
```

## 与计算神经科学的关联

### 功能连接组（Functional Connectome）

神经科学家用**功能连接矩阵**描述脑区之间的相关性：

$$
C_{ij} = \text{Corr}(B_i, B_j)
$$

其中 $B_i, B_j$ 是区域 $i, j$ 的 fMRI 或 EEG 时间序列。

- 类似于投资组合中的**相关系数矩阵**
- 用于聚类分析、识别脑网络模块（如默认模式网络）
- 主成分分析（PCA）降维方法同样适用

### 神经信号的相关性分析

**静息态功能连接**：fMRI 中血氧水平依赖（BOLD）信号的相关性，反映自发神经活动。

**有效连接**：基于 Granger 因果或动态因果模型，区分"相关"与"因果"。

> 对应量化中的 **Pearson 相关** → **Granger 因果**（时间序列方向性）的跨越。
