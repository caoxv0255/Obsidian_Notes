量化金融是运用数学、统计学和计算机科学方法进行金融分析和决策的领域。本章将系统介绍量化金融的基本概念、发展历程和核心要素，为后续学习量化策略打下基础。

## 1. 量化金融的定义

### 1.1 量化金融的定义

**量化金融**（Quantitative Finance）是利用数学、统计学、计算机科学等量化方法进行金融分析、投资决策和风险管理的领域。

### 1.2 量化金融的特点

1. **数学化**：运用数学模型描述金融现象
2. **数据驱动**：基于大量数据进行分析
3. **计算机化**：利用计算机进行计算和模拟
4. **模型化**：建立数学模型进行预测
5. **实证化**：通过实证检验模型有效性

## 2. 量化金融的发展历程

### 2.1 早期发展（1950s-1970s）

#### 马科维茨均值-方差模型

**创始人**：Harry Markowitz

**贡献**：
- 建立了现代投资组合理论
- 提出了有效前沿概念
- 引入了风险度量方法

**意义**：
- 首次用数学方法描述投资组合优化
- 为量化投资奠定了理论基础

#### 资本资产定价模型（CAPM）

**创始人**：Sharpe, Lintner, Mossin

**公式**：
$$E(R_i) = R_f + \beta_i(E(R_m) - R_f)$$

其中：
- $E(R_i)$：资产i的期望收益率
- $R_f$：无风险利率
- $\beta_i$：资产i的贝塔系数
- $E(R_m)$：市场组合的期望收益率

#### 期权定价模型（B-S模型）

**创始人**：Black, Scholes, Merton

**公式**：
$$C = S_0 N(d_1) - X e^{-rT} N(d_2)$$

其中：
- $C$：看涨期权价格
- $S_0$：标的资产当前价格
- $X$：执行价格
- $r$：无风险利率
- $T$：到期时间
- $N(\cdot)$：标准正态分布累积分布函数

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression
from sklearn.ensemble import RandomForestRegressor
from sklearn.metrics import mean_squared_error, r2_score

class QuantitativeFinance:
    """量化金融类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_portfolio_return(self, weights, returns):
        """计算投资组合收益率"""
        return np.dot(weights, returns)
    
    def calculate_portfolio_risk(self, weights, cov_matrix):
        """计算投资组合风险"""
        return np.sqrt(np.dot(weights.T, np.dot(cov_matrix, weights)))
    
    def calculate_sharpe_ratio(self, returns, risk_free_rate=0.02):
        """计算夏普比率"""
        excess_returns = returns - risk_free_rate / 252
        return np.mean(excess_returns) / np.std(excess_returns) * np.sqrt(252)
    
    def calculate_max_drawdown(self, returns):
        """计算最大回撤"""
        cumulative = np.cumprod(1 + returns)
        running_max = np.maximum.accumulate(cumulative)
        drawdown = (cumulative - running_max) / running_max
        return np.min(drawdown)
    
    def calculate_var(self, returns, confidence_level=0.05):
        """计算VaR"""
        return np.percentile(returns, confidence_level * 100)
    
    def calculate_cvar(self, returns, confidence_level=0.05):
        """计算CVaR"""
        var = self.calculate_var(returns, confidence_level)
        return np.mean(returns[returns <= var])

class StrategyAnalyzer:
    """策略分析器"""
    
    def __init__(self):
        """初始化"""
        self.qf = QuantitativeFinance()
    
    def backtest_strategy(self, prices, signals):
        """回测策略"""
        # 计算收益率
        returns = np.diff(prices) / prices[:-1]
        
        # 计算策略收益
        strategy_returns = []
        position = 0
        
        for i in range(1, len(signals)):
            if signals[i] == 1 and position == 0:
                position = 1
            elif signals[i] == -1 and position == 1:
                position = 0
            
            if position == 1:
                strategy_returns.append(returns[i])
            else:
                strategy_returns.append(0)
        
        strategy_returns = np.array(strategy_returns)
        
        # 计算指标
        metrics = {
            'strategy_return': np.sum(strategy_returns),
            'benchmark_return': np.sum(returns),
            'sharpe_ratio': self.qf.calculate_sharpe_ratio(strategy_returns),
            'max_drawdown': self.qf.calculate_max_drawdown(strategy_returns),
            'var': self.qf.calculate_var(strategy_returns),
            'cvar': self.qf.calculate_cvar(strategy_returns)
        }
        
        return metrics, strategy_returns

class MachineLearningStrategy:
    """机器学习策略"""
    
    def __init__(self):
        """初始化"""
        self.model = None
    
    def train_model(self, X, y):
        """训练模型"""
        self.model = RandomForestRegressor(n_estimators=100, random_state=42)
        self.model.fit(X, y)
    
    def predict(self, X):
        """预测"""
        if self.model is None:
            raise ValueError("模型未训练")
        return self.model.predict(X)
    
    def evaluate_model(self, X_test, y_test):
        """评估模型"""
        predictions = self.predict(X_test)
        
        mse = mean_squared_error(y_test, predictions)
        r2 = r2_score(y_test, predictions)
        
        return {
            'mse': mse,
            'r2': r2,
            'rmse': np.sqrt(mse)
        }

class EventDrivenStrategy:
    """事件驱动策略"""
    
    def __init__(self):
        """初始化"""
        self.event_signals = {}
    
    def add_event_signal(self, event_type, signal):
        """添加事件信号"""
        if event_type not in self.event_signals:
            self.event_signals[event_type] = []
        self.event_signals[event_type].append(signal)
    
    def get_event_signals(self, event_type):
        """获取事件信号"""
        return self.event_signals.get(event_type, [])
    
    def analyze_event_impact(self, prices, event_dates):
        """分析事件影响"""
        impacts = []
        
        for event_date in event_dates:
            if event_date in prices.index:
                price = prices.loc[event_date]
                
                # 计算事件前后的价格变化
                before_price = prices.loc[event_date - pd.Timedelta(days=1)]
                after_price = prices.loc[event_date + pd.Timedelta(days=1)]
                
                before_return = (price - before_price) / before_price
                after_return = (after_price - price) / price
                
                impacts.append({
                    'event_date': event_date,
                    'before_return': before_return,
                    'after_return': after_return,
                    'total_return': after_return + before_return
                })
        
        return impacts

def quant_strategy_backtest():
    """量化策略回测示例"""
    # 模拟股票价格数据
    dates = pd.date_range('2023-01-01', periods=100, freq='D')
    prices = np.random.normal(100, 5, 100)
    prices = np.cumsum(prices) + 100
    prices = pd.Series(prices, index=dates)
    
    # 简单移动平均策略
    short_window = 5
    long_window = 20
    
    # 计算移动平均
    short_ma = prices.rolling(window=short_window).mean()
    long_ma = prices.rolling(window=long_window).mean()
    
    # 生成信号
    signals = pd.Series(0, index=prices.index)
    signals[short_ma > long_ma] = 1  # 买入信号
    signals[short_ma < long_ma] = -1  # 卖出信号
    
    # 计算策略收益
    returns = prices.pct_change()
    strategy_returns = signals.shift(1) * returns
    
    # 绘制结果
    plt.figure(figsize=(12, 8))
    plt.subplot(2, 1, 1)
    plt.plot(dates, prices, label='价格')
    plt.plot(dates, short_ma, label='短期均线', linestyle='--')
    plt.plot(dates, long_ma, label='长期均线', linestyle='--')
    plt.title('移动平均策略')
    plt.legend()
    
    plt.subplot(2, 1, 2)
    plt.plot(dates, strategy_returns.cumsum(), label='策略收益')
    plt.title('策略收益曲线')
    plt.legend()
    
    plt.tight_layout()
    plt.show()
    
    # 计算策略指标
    total_return = (1 + strategy_returns).prod() - 1
    sharpe_ratio = np.mean(strategy_returns) / np.std(strategy_returns) * np.sqrt(252)
    
    print(f"总收益率: {total_return:.2%}")
    print(f"夏普比率: {sharpe_ratio:.2f}")
