# 面向对象编程

## 核心概念

面向对象编程（OOP）是一种程序设计范式，通过"对象"来组织代码。Python 是一种多范式语言，完全支持面向对象编程。理解 OOP 对于构建复杂的数据处理系统和机器学习管道至关重要。

**OOP 四大特性：**
- **封装（Encapsulation）**：将数据和操作组织在一起，对外隐藏实现细节
- **继承（Inheritance）**：子类复用父类的代码，实现"is-a"关系
- **多态（Polymorphism）**：同一接口不同实现，父类引用指向子类对象
- **抽象（Abstraction）**：简化复杂系统，隐藏不必要的细节

**类（Class）与对象（Object）：**
- 类是对象的蓝图/模板
- 对象是类的实例
- 类定义属性（数据）和方法（行为）

## 代码实现

### 1. 类的基础

```python
# 定义一个简单的类
class Dog:
    """狗类"""
    
    # 类属性（所有实例共享）
    species = "Canis familiaris"
    
    # 初始化方法（构造器）
    def __init__(self, name, age):
        # 实例属性
        self.name = name
        self.age = age
    
    # 实例方法
    def bark(self):
        return f"{self.name} says Woof!"
    
    # 字符串表示
    def __str__(self):
        return f"{self.name} is {self.age} years old"

# 创建对象
my_dog = Dog("Buddy", 3)
print(my_dog)              # 自动调用 __str__
print(my_dog.bark())       # Buddy says Woof!
print(Dog.species)         # 类的属性
```

### 2. 继承

```python
# 父类
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        raise NotImplementedError("子类必须实现 speak 方法")

# 子类
class Cat(Animal):
    def speak(self):
        return f"{self.name} says Meow!"

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof!"

# 多态
animals = [Cat("Whiskers"), Dog("Buddy"), Cat("Luna")]
for animal in animals:
    print(animal.speak())

# 多重继承（谨慎使用）
class A:
    def method(self):
        return "A"

class B:
    def method(self):
        return "B"

class C(A, B):  # C 继承 A 和 B
    pass

c = C()
print(c.method())  # A（左侧优先）
```

### 3. 属性访问控制

```python
class BankAccount:
    def __init__(self, balance):
        # 私有属性（以双下划线开头）
        self.__balance = balance
    
    # 公有方法访问私有属性
    def get_balance(self):
        return self.__balance
    
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
            return True
        return False
    
    def withdraw(self, amount):
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return True
        return False

account = BankAccount(1000)
print(account.get_balance())      # 1000
account.deposit(500)
print(account.get_balance())      # 1500
# account.__balance  # AttributeError：无法直接访问
```

### 4. 类方法与静态方法

```python
class MathUtils:
    # 类变量
    pi = 3.14159
    
    # 实例方法
    def double(self, x):
        return x * 2
    
    # 类方法（cls 作为第一个参数）
    @classmethod
    def circle_area(cls, radius):
        return cls.pi * radius ** 2
    
    # 静态方法（不需要 self 或 cls）
    @staticmethod
    def add(x, y):
        return x + y

# 调用类方法
print(MathUtils.circle_area(5))  # 不需要创建实例
print(MathUtils.add(3, 4))

# 调用实例方法
utils = MathUtils()
print(utils.double(10))
```

### 5. 数据科学中的 OOP 应用

```python
import numpy as np
import pandas as pd

# 数据集类
class Dataset:
    def __init__(self, name):
        self.name = name
        self.data = None
        self.shape = None
    
    def load_from_dict(self, data_dict):
        self.data = pd.DataFrame(data_dict)
        self.shape = self.data.shape
        return self
    
    def get_summary(self):
        return {
            "name": self.name,
            "shape": self.shape,
            "columns": list(self.data.columns),
            "dtypes": self.data.dtypes.to_dict()
        }
    
    def __repr__(self):
        return f"Dataset(name={self.name}, shape={self.shape})"

# 使用 Dataset
data = {
    "student": ["Alice", "Bob", "Charlie"],
    "math": [95, 88, 92],
    "physics": [90, 85, 88]
}
ds = Dataset("学生成绩")
ds.load_from_dict(data)
print(ds.get_summary())
```

## 实战技巧

**1. 使用 dataclasses 简化类定义（Python 3.7+）**
```python
from dataclasses import dataclass, field
from typing import List

@dataclass
class Student:
    name: str
    age: int
    grades: List[float] = field(default_factory=list)
    
    def average_grade(self):
        return sum(self.grades) / len(self.grades) if self.grades else 0

student = Student("Alice", 20, [95, 88, 92])
print(student.average_grade())
```

**2. 使用 __slots__ 优化内存**
```python
class Point:
    __slots__ = ['x', 'y']
    
    def __init__(self, x, y):
        self.x = x
        self.y = y

# __slots__ 可以减少内存占用，适合大量对象的场景
```

**3. property 装饰器实现 getter/setter**
```python
class Temperature:
    def __init__(self):
        self._celsius = 0
    
    @property
    def celsius(self):
        return self._celsius
    
    @celsius.setter
    def celsius(self, value):
        if value < -273.15:
            raise ValueError("温度不能低于绝对零度")
        self._celsius = value
    
    @property
    def fahrenheit(self):
        return self._celsius * 9/5 + 32

temp = Temperature()
temp.celsius = 25
print(temp.fahrenheit)  # 77.0
```

**4. 使用 __enter__ 和 __exit__ 实现上下文管理器**
```python
class FileManager:
    def __init__(self, filename, mode):
        self.filename = filename
        self.mode = mode
        self.file = None
    
    def __enter__(self):
        self.file = open(self.filename, self.mode)
        return self.file
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        if self.file:
            self.file.close()
        return False

with FileManager("test.txt", "w") as f:
    f.write("Hello!")
```

## 常见错误

**1. 修改类属性导致意外行为**
```python
class Dog:
    tricks = []  # 类属性，所有实例共享！
    
    def add_trick(self, trick):
        self.tricks.append(trick)

dog1 = Dog()
dog2 = Dog()
dog1.add_trick("sit")
print(dog2.tricks)  # ['sit'] ← 意外！两个狗共享同一个列表

# 正确做法：在 __init__ 中创建实例属性
class Dog:
    def __init__(self):
        self.tricks = []  # 实例属性
    
    def add_trick(self, trick):
        self.tricks.append(trick)
```

**2. super() 调用顺序错误**
```python
# Python 3 的 super() 可以不带参数
class Child(Parent):
    def __init__(self, name, extra):
        super().__init__(name)  # 正确
        self.extra = extra
```

**3. 混淆类变量和实例变量**
```python
# 错误示例
class Counter:
    count = 0  # 类变量
    
    def __init__(self):
        self.count += 1  # 这会创建实例变量，不修改类变量

# 正确做法
class Counter:
    count = 0
    
    def __init__(self):
        Counter.count += 1  # 明确修改类变量
```
