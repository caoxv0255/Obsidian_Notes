# 控制流

## 核心概念

控制流决定了程序执行的顺序和路径。Python 提供了三种主要的控制流结构：顺序执行、条件分支和循环迭代。掌握这些基础控制结构是编写任何程序的前提。

**条件分支：** 根据条件的真假决定执行哪段代码
- `if` / `elif` / `else`

**循环迭代：** 重复执行一段代码
- `for` 循环：已知迭代次数或可迭代对象
- `while` 循环：未知迭代次数，根据条件执行

**跳转语句：**
- `break`：跳出循环
- `continue`：跳过当前迭代，进入下一次
- `pass`：占位语句，什么都不做

**Python 与其他语言的关键区别：**
- Python 没有 `switch` 语句，用 `elif` 链实现多分支
- Python 的循环可以带 `else` 子句（循环正常结束时会执行）
- Python 使用缩进而不是花括号来划分代码块

## 代码实现

### 1. 条件语句

```python
# 基本 if 语句
age = 20

if age >= 18:
    print("成年人")
    
# if-else 语句
score = 75
if score >= 60:
    print("及格")
else:
    print("不及格")

# if-elif-else 多重条件
grade = 85
if grade >= 90:
    print("A")
elif grade >= 80:
    print("B")
elif grade >= 70:
    print("C")
elif grade >= 60:
    print("D")
else:
    print("F")

# 条件表达式（三元运算符）
age = 20
status = "成年人" if age >= 18 else "未成年人"

# 多个条件的组合
age = 25
income = 50000

if age >= 18 and income > 30000:
    print("符合申请条件")

# in 操作符检查成员关系
fruit = "apple"
if fruit in ["apple", "banana", "cherry"]:
    print(f"{fruit} 在列表中")

# 嵌套条件
x = 10
if x > 0:
    if x < 100:
        print("x 是 0-100 之间的正数")
```

### 2. for 循环

```python
# 遍历列表
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# 遍历范围
for i in range(5):
    print(i)          # 0, 1, 2, 3, 4

for i in range(2, 8):
    print(i)          # 2, 3, 4, 5, 6, 7

for i in range(0, 10, 3):
    print(i)          # 0, 3, 6, 9（步长为3）

# 遍历字符串
for char in "Python":
    print(char)

# 遍历字典
student = {"name": "Alice", "age": 20, "major": "DS"}
for key in student:
    print(f"{key}: {student[key]}")

for key, value in student.items():
    print(f"{key}: {value}")

# enumerate 同时获取索引和值
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# zip 遍历多个序列
names = ["Alice", "Bob", "Charlie"]
scores = [95, 88, 92]
for name, score in zip(names, scores):
    print(f"{name}: {score}")

# 列表推导式中的循环
squares = [x**2 for x in range(10)]
even_squares = [x**2 for x in range(10) if x % 2 == 0]
```

### 3. while 循环

```python
# 基本 while 循环
count = 0
while count < 5:
    print(count)
    count += 1

# while 循环与 break
while True:
    user_input = input("输入 'quit' 退出: ")
    if user_input == "quit":
        break
    print(f"你输入了: {user_input}")

# while 循环与 continue
i = 0
while i < 10:
    i += 1
    if i % 2 == 0:
        continue  # 跳过偶数
    print(i)  # 打印奇数: 1, 3, 5, 7, 9

# while-else 结构
count = 0
while count < 3:
    print(f"尝试 {count}")
    count += 1
else:
    print("循环正常结束")  # 如果不是被 break 终止的，执行这里
```

### 4. 循环的 else 子句

```python
# 循环 else 是 Python 的独特特性
# else 在循环正常结束时执行，break 时不执行

def find_prime_above(n):
    """找到大于 n 的第一个质数"""
    num = n + 1
    while True:
        for i in range(2, int(num**0.5) + 1):
            if num % i == 0:
                break
        else:
            return num
        num += 1

print(find_prime_above(100))  # 101

# 使用 else 的典型场景：搜索
numbers = [1, 3, 5, 7, 9]
target = 6

for num in numbers:
    if num == target:
        print(f"找到了: {target}")
        break
else:
    print(f"未找到 {target}")
```

### 5. 数据科学中的控制流应用

```python
import numpy as np
import pandas as pd

# 条件筛选数据
data = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "David"],
    "score": [95, 58, 87, 72],
    "passed": [True, False, True, True]
})

# 筛选及格学生
passed_students = data[data["score"] >= 60]
print(passed_students)

# 使用循环处理数据
for idx, row in data.iterrows():
    if row["score"] >= 90:
        grade = "A"
    elif row["score"] >= 80:
        grade = "B"
    elif row["score"] >= 70:
        grade = "C"
    elif row["score"] >= 60:
        grade = "D"
    else:
        grade = "F"
    print(f"{row['name']}: {grade}")

# 使用 np.where 进行条件赋值
data["grade"] = np.where(data["score"] >= 90, "A",
                np.where(data["score"] >= 80, "B",
                np.where(data["score"] >= 70, "C",
                np.where(data["score"] >= 60, "D", "F"))))
```

## 实战技巧

**1. 使用 any() 和 all() 进行批量条件检查**
```python
# 检查是否存在满足条件的元素
scores = [45, 67, 89, 92, 55]
has_passed = any(s >= 60 for s in scores)

# 检查是否所有元素都满足条件
all_passed = all(s >= 60 for s in scores)
```

**2. 使用 itertools 优化循环**
```python
from itertools import product, combinations, permutations

# 多层循环的优雅写法
colors = ["红", "绿", "蓝"]
sizes = ["S", "M", "L"]

for color, size in product(colors, sizes):
    print(f"{color} {size}")

# 组合和排列
comb = list(combinations([1, 2, 3], 2))  # [(1,2), (1,3), (2,3)]
perm = list(permutations([1, 2, 3], 2))  # [(1,2), (1,3), (2,1), (2,3), (3,1), (3,2)]
```

**3. 异常处理替代条件检查**
```python
# 某些情况下用 try-except 比 if-else 更高效
# 比如检查文件是否存在
try:
    with open("data.csv", "r") as f:
        data = f.read()
except FileNotFoundError:
    print("文件不存在")
```

**4. 使用 match-case（Python 3.10+）**
```python
# 模式匹配，类似于其他语言的 switch
def http_status(status):
    match status:
        case 200:
            return "OK"
        case 404:
            return "Not Found"
        case 500:
            return "Server Error"
        case _:
            return "Unknown"
```

## 常见错误

**1. 无限循环**
```python
# 错误示例
i = 0
while i < 10:
    print(i)
    # 忘记 i += 1，导致无限循环

# 正确做法
i = 0
while i < 10:
    print(i)
    i += 1
```

**2. 在循环中修改迭代对象**
```python
# 错误示例
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    if fruit == "banana":
        fruits.remove(fruit)  # 可能导致跳过元素

# 正确做法：使用列表推导式创建新列表
fruits = ["apple", "banana", "cherry"]
fruits = [f for f in fruits if f != "banana"]
```

**3. 混淆 = 和 ==**
```python
# 错误示例
if x = 10:  # SyntaxError: invalid syntax
    print("x 是 10")

# 正确做法
if x == 10:
    print("x 是 10")
```

**4. 浮点数比较**
```python
# 错误示例
result = 0.1 + 0.2
if result == 0.3:  # False！
    print("相等")

# 正确做法：使用容差比较
import math
if math.isclose(result, 0.3, rel_tol=1e-9):
    print("近似相等")
```

**5. for 循环中的变量作用域**
```python
# Python 中 for 循环的变量在循环结束后仍然保留
for i in range(5):
    pass
print(i)  # 4（最后一个值）

# 如果需要隔离，使用函数或列表推导式
[... for i in range(5)]  # i 不会泄漏到外部
```
