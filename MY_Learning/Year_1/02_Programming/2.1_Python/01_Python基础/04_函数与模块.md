# 函数与模块

## 核心概念

函数是组织代码的基本单元，可以接受输入、执行操作并返回结果。模块是 Python 代码的组织单位，一个 `.py` 文件就是一个模块。合理使用函数和模块可以实现代码复用、提高可维护性。

**函数的基本组成：**
- `def` 关键字定义函数
- 函数名（遵循标识符命名规则）
- 参数列表（可选）
- 函数体（缩进的代码块）
- `return` 语句（可选，默认为 None）

**模块的概念：**
- 每个 `.py` 文件都是独立的模块
- 模块有自己的命名空间
- 可以使用 `import` 导入模块
- Python 标准库提供了大量模块（内置模块）

**模块的搜索路径：**
1. 当前目录
2. 环境变量 PYTHONPATH 中的目录
3. Python 安装目录下的 lib
4. 第三方包目录（site-packages）

**常用标准模块：**
- `os`：操作系统接口
- `sys`：系统相关参数
- `math`：数学函数
- `datetime`：日期时间处理
- `random`：随机数生成
- `json`：JSON 序列化

## 代码实现

### 1. 函数定义与调用

```python
# 基本函数定义
def greet():
    """问候函数"""
    print("Hello, World!")

greet()  # 调用函数

# 带参数的函数
def greet_person(name):
    """带参数的问候"""
    print(f"Hello, {name}!")

greet_person("Alice")  # Hello, Alice!

# 带返回值的函数
def add(a, b):
    """返回两数之和"""
    return a + b

result = add(3, 5)  # result = 8

# 多返回值函数（实际返回元组）
def get_stats(numbers):
    """返回最小值、最大值、平均值"""
    return min(numbers), max(numbers), sum(numbers) / len(numbers)

mn, mx, avg = get_stats([1, 2, 3, 4, 5])  # 解包
```

### 2. 参数类型

```python
# 位置参数
def power(base, exponent):
    return base ** exponent

power(2, 3)  # 8

# 关键字参数
power(exponent=3, base=2)  # 8，顺序无关

# 默认参数
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

greet("Alice")  # Hello, Alice!
greet("Bob", "Hi")  # Hi, Bob!

# 可变参数 *args
def sum_all(*args):
    """接受任意数量的位置参数"""
    total = 0
    for num in args:
        total += num
    return total

sum_all(1, 2, 3, 4, 5)  # 15
sum_all(10, 20)  # 30

# 关键字参数 **kwargs
def print_info(**kwargs):
    """接受任意数量的关键字参数"""
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=20, major="DS")

# 组合使用
def func(pos1, pos2, *args, **kwargs):
    print(f"位置参数: {pos1}, {pos2}")
    print(f"可变参数: {args}")
    print(f"关键字参数: {kwargs}")
```

### 3. 特殊参数语法与 Lambda

```python
# 位置-only 参数（/ 之前的参数只能按位置传递）
def func(pos_only, /, standard, *, kw_only):
    print(f"{pos_only}, {standard}, {kw_only}")

func(1, 2, kw_only=3)  # 正确
# func(pos_only=1, standard=2, kw_only=3)  # 错误

# Lambda 表达式（匿名函数）
square = lambda x: x ** 2
print(square(5))  # 25

add = lambda x, y: x + y
print(add(3, 4))  # 7

# Lambda 与内置函数结合
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x**2, numbers))
filtered = list(filter(lambda x: x % 2 == 0, numbers))
```

### 5. 数据科学中的函数应用

```python
import numpy as np
import pandas as pd
from typing import List, Dict

# 数据清洗函数
def normalize(data: List[float]) -> List[float]:
    """将数据归一化到 0-1 范围"""
    min_val = min(data)
    max_val = max(data)
    return [(x - min_val) / (max_val - min_val) for x in data]

# 计算统计量
def descriptive_stats(df: pd.DataFrame, column: str) -> Dict[str, float]:
    """计算描述性统计"""
    col = df[column]
    return {
        "mean": col.mean(),
        "median": col.median(),
        "std": col.std(),
        "min": col.min(),
        "max": col.max(),
        "q25": col.quantile(0.25),
        "q75": col.quantile(0.75)
    }

# 示例 DataFrame
df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie", "David"],
    "math": [95, 58, 87, 72],
    "physics": [90, 65, 88, 78]
})

# 应用函数
df["math_normalized"] = normalize(df["math"].tolist())
print(descriptive_stats(df, "math"))
```

## 实战技巧

**1. 使用装饰器增强函数功能**
```python
import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} 执行时间: {end - start:.4f} 秒")
        return result
    return wrapper

@timer
def slow_function():
    time.sleep(0.5)
    return "完成"

slow_function()
```

**2. 使用 functools 缓存结果**
```python
from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci(n):
    """斐波那契数列（带缓存）"""
    if n < 2:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(100))  # 快速计算
```

**3. 创建 Python 包**
```text
mypackage/
    __init__.py      # 包初始化文件
    module1.py       # 模块1
    module2.py       # 模块2
```
`__init__.py` 可以指定包的导出内容，控制 `from mypackage import *` 的行为。

**4. __name__ == "__main__" 用法**
```python
# 当模块直接运行时执行，作为导入时不执行
if __name__ == "__main__":
    # 测试代码
    main()
```

## 常见错误

**1. 默认参数使用可变对象**
```python
# 错误示例
def add_item(item, items=[]):
    items.append(item)
    return items

print(add_item("a"))  # ['a']
print(add_item("b"))  # ['a', 'b'] ← 默认参数被保留！

# 正确做法
def add_item_fixed(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```

**2. 循环变量泄漏**
```python
# Python 中 for 循环的变量在循环结束后保留
for i in range(5):
    pass
print(i)  # 4，最后一个值

# 正确做法：在需要隔离时使用函数
def process():
    for i in range(5):
        pass
    return i
```

**3. 模块循环导入**
```python
# module_a.py
# from module_b import func_b  # 如果 module_b 也导入 module_a，会导致循环导入错误

# 解决方案：将导入移到函数内部，或重构代码结构
```

**4. 忘记 return 语句**
```python
def calculate(x):
    x ** 2  # 没有 return，返回 None

result = calculate(5)
print(result)  # None
```

**5. 导入不存在的模块或函数**
```python
# 错误
import mathh  # ModuleNotFoundError
from math import sqrttt  # ImportError

# 正确
import math
from math import sqrt
```
