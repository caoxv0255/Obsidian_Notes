# 数据类型与结构

## 核心概念

Python 内置了丰富的数据类型和数据结构，理解它们是编写高效 Python 代码的基础。数据科学家每天都在处理各种类型的数据：数值、文本、表格、时间序列等，这些最终都映射到 Python 的基本数据结构上。

**Python 内置数据类型：**
- **数值类型**：整数（int）、浮点数（float）、复数（complex）
- **布尔类型**：True / False
- **字符串类型**：str
- **None 类型**：表示空值

**Python 常用数据结构：**
- **列表（List）**：有序、可变、可重复，`[1, 2, 3]`
- **元组（Tuple）**：有序、不可变、可重复，`(1, 2, 3)`
- **字典（Dict）**：键值对，无序（Python 3.7+ 有序），`{'name': 'Alice', 'age': 20}`
- **集合（Set）**：无序、不重复，`{1, 2, 3}`

**数据类型转换：**
```python
int(3.14)    # 3，浮点转整数（截断）
float(3)     # 3.0，整数转浮点
str(3.14)    # '3.14'，转字符串
bool(0)      # False，0、空值转False
bool(1)      # True，非零转True
list((1, 2)) # [1, 2]，转列表
```

## 代码实现

### 1. 数值类型与运算

```python
# 基本数值运算
a = 17
b = 5

print(a + b)    # 22
print(a - b)    # 12
print(a * b)    # 85
print(a / b)    # 3.4（浮点除法）
print(a // b)   # 3（整除，向下取整）
print(a % b)    # 2（取余）
print(a ** b)   # 1419857（幂运算）

# 整数溢出？Python 不存在！
# Python 的 int 是任意精度整数，不像 C/Java 那样有上限
big_num = 10 ** 100
print(big_num)  # 100...0（很大很大的数）

# 科学计数法
Avogadro = 6.022e23   # 6.022 × 10²³
Planck = 6.626e-34    # 6.626 × 10⁻³⁴

# 数值精度问题（所有语言都有）
result = 0.1 + 0.2
print(result)   # 0.30000000000000004
# 数据科学中常用 decimal 模块或 NumPy 处理精度问题
```

### 2. 列表（List）

```python
# 创建列表
fruits = ["apple", "banana", "cherry"]
numbers = list(range(1, 6))    # [1, 2, 3, 4, 5]
mixed = [1, "two", 3.0, True]

# 索引和切片
print(fruits[0])       # apple（从0开始）
print(fruits[-1])      # cherry（最后一个）
print(numbers[1:4])    # [2, 3, 4]
print(numbers[::2])   # [1, 3, 5]（跳步）

# 列表操作
fruits.append("orange")        # 末尾添加
fruits.insert(1, "grape")      # 指定位置插入
fruits.remove("banana")        # 删除第一个匹配项
popped = fruits.pop()          # 弹出并返回最后一个
del fruits[0]                  # 删除指定位置

# 列表推导式（数据科学核心语法）
squares = [x**2 for x in range(10)]
evens = [x for x in range(20) if x % 2 == 0]
matrix = [[i*j for j in range(3)] for i in range(3)]

# 列表常用函数
nums = [3, 1, 4, 1, 5, 9, 2, 6]
print(len(nums))       # 8，长度
print(max(nums))       # 9，最大值
print(min(nums))       # 1，最小值
print(sum(nums))       # 31，求和
nums.sort()            # 原地排序
print(sorted(nums))    # 返回排序后的新列表
```

### 3. 元组（Tuple）

```python
# 元组创建
point = (3, 4)
single = (42,)         # 单元素元组必须加逗号
empty = ()

# 元组解包
x, y = point
a, b, c = (1, 2, 3)

# 函数返回多个值（返回的就是元组）
def get_stats(numbers):
    return min(numbers), max(numbers), sum(numbers)/len(numbers)

mn, mx, avg = get_stats([1, 2, 3, 4, 5])

# 命名元组（数据科学常用）
from collections import namedtuple
Person = namedtuple('Person', ['name', 'age', 'city'])
alice = Person("Alice", 20, "Beijing")
print(alice.name)      # Alice
print(alice.age)       # 20
```

### 4. 字典（Dict）

```python
# 创建字典
student = {
    "name": "Alice",
    "age": 20,
    "major": "Data Science",
    "gpa": 3.8
}

# 访问和修改
print(student["name"])       # Alice
student["gpa"] = 3.9         # 修改
student["year"] = 1          # 添加新键值对

# 安全访问（键不存在时不报错）
print(student.get("email", "Not provided"))  # Not provided

# 字典常用方法
print(student.keys())        # dict_keys(['name', 'age', ...])
print(student.values())      # dict_values(['Alice', 20, ...])
print(student.items())       # dict_items([('name', 'Alice'), ...])

# 字典推导式
squares = {x: x**2 for x in range(5)}
names = ["Alice", "Bob", "Charlie"]
lengths = {name: len(name) for name in names}

# 合并字典（Python 3.9+）
dict1 = {"a": 1, "b": 2}
dict2 = {"c": 3, "d": 4}
merged = dict1 | dict2
```

### 5. 集合（Set）

```python
# 创建集合
a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

# 集合运算
print(a | b)       # {1, 2, 3, 4, 5, 6, 7, 8} 并集
print(a & b)       # {4, 5} 交集
print(a - b)       # {1, 2, 3} 差集
print(a ^ b)       # {1, 2, 3, 6, 7, 8} 对称差集

# 集合方法
a.add(6)           # 添加元素
a.remove(1)        # 删除元素（不存在会报错）
a.discard(10)      # 删除元素（不存在不报错）
a.update([7, 8])   # 添加多个元素

# 去重
duplicates = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(duplicates))  # [1, 2, 3, 4]

# 判断子集/超集
set1 = {1, 2, 3}
set2 = {1, 2}
print(set2.issubset(set1))     # True
print(set1.issuperset(set2))   # True
```

### 6. 数据结构在数据科学中的应用

```python
import pandas as pd
import numpy as np

# 用字典创建 DataFrame（数据科学核心操作）
data = {
    "name": ["Alice", "Bob", "Charlie"],
    "math": [95, 88, 92],
    "physics": [90, 85, 88]
}
df = pd.DataFrame(data)
print(df)

# 用列表存储时间序列数据
import datetime
dates = [datetime.date(2024, 1, i) for i in range(1, 8)]
values = [10, 12, 15, 14, 18, 20, 22]
time_series = dict(zip(dates, values))
print(time_series)

# 用集合处理唯一值
all_cities = ["Beijing", "Shanghai", "Beijing", "Hangzhou", "Shanghai"]
unique_cities = set(all_cities)
print(f"共{len(unique_cities)}个不同城市: {unique_cities}")
```

## 实战技巧

**1. 选择正确的数据结构**
- 需要频繁增删元素 → 列表
- 需要保持元素不变 → 元组
- 需要按键快速查找 → 字典
- 需要去重或集合运算 → 集合

**2. 列表推导式替代 map/filter**
```python
# 传统方式
squares = list(map(lambda x: x**2, range(10)))

# 列表推导式（更直观）
squares = [x**2 for x in range(10)]
```

**3. 使用 enumerate 同时获取索引和值**
```python
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
```

**4. 使用 zip 合并多个列表**
```python
names = ["Alice", "Bob", "Charlie"]
scores = [95, 88, 92]
for name, score in zip(names, scores):
    print(f"{name}: {score}")
```

**5. collections 模块提供了更多数据结构**
```python
from collections import deque, Counter, defaultdict

# deque：双端队列，适合做队列和栈
queue = deque(["a", "b", "c"])
queue.append("d")
queue.popleft()

# Counter：计数统计
cnt = Counter(["a", "b", "a", "c", "a"])
print(cnt["a"])   # 3

# defaultdict：带默认值的字典
dd = defaultdict(int)
dd["key"] += 1
```

## 常见错误

**1. 修改不可变对象**
```python
# 错误示例
point = (1, 2)
point[0] = 10    # TypeError: 'tuple' object does not support item assignment

# 正确做法：创建新的元组
point = (10, 2)
```

**2. 列表切片赋值时的常见错误**
```python
# 错误示例
lst = [1, 2, 3]
lst[5] = 10   # IndexError: list assignment index out of range

# 正确做法：先扩展列表
lst.extend([0, 0])  # 或使用 append
lst[5] = 10
```

**3. 字典键不存在时的 KeyError**
```python
# 错误示例
d = {"a": 1}
print(d["b"])   # KeyError: 'b'

# 正确做法：使用 get 方法
print(d.get("b", 0))  # 0，键不存在时返回默认值
```

**4. 集合直接创建空集合**
```python
# 错误示例
empty = {}      # 这是字典，不是集合！
print(type(empty))  # <class 'dict'>

# 正确做法
empty_set = set()
```

**5. 混淆可变对象作为默认参数**
```python
# 错误示例（可变默认参数的陷阱）
def add_item(item, items=[]):
    items.append(item)
    return items

print(add_item("a"))  # ['a']
print(add_item("b"))  # ['a', 'b']  ← 默认参数被保留！

# 正确做法
def add_item_fixed(item, items=None):
    if items is None:
        items = []
    items.append(item)
    return items
```
