# 项目实战 (Projects)

> 从理论到实践，通过项目巩固知识，培养工程能力

## 模块概览

项目实战是ML学习的核心环节，通过13个分级项目，从简单的线性回归到前沿的大语言模型微调，实现理论与实践的完美结合。

**设计原则**：
- **分级递进**：Level 1入门 → Level 2进阶 → Level 3高级 → Level 4前沿
- **理论对应**：每个项目对应特定理论知识点
- **完整流程**：包含数据处理、模型训练、评估优化全流程
- **代码注释**：详细注释，便于理解复现

## 学习建议

### 初学者建议
1. **从P01开始**：不要跳过基础项目
2. **完整实现**：不要只运行代码，要理解每一步
3. **记录笔记**：记录遇到的问题和解决方案
4. **尝试改进**：尝试修改超参数，观察效果

### 进阶建议
1. **理解原理**：不仅会用，要理解为什么这样设计
2. **代码规范**：养成良好编程习惯
3. **性能优化**：学习加速训练、减少显存的技巧
4. **实验设计**：学会设计对比实验

### 高级建议
1. **阅读论文**：了解项目对应的原始论文
2. **复现SOTA**：尝试复现最先进的方法
3. **创新改进**：提出自己的改进思路
4. **工程化**：考虑模型部署、服务化
