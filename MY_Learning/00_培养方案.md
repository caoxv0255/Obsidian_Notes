# 培养方案

> 量化交易 × 计算神经科学 双路径本科→直博培养方案  
> 适用：对外经贸大学 数据科学与大数据技术  
> 目标院校：CMU / MIT / ITMO 等顶级计算金融 / 计算神经科学硕博项目

---

## 一、培养逻辑

本方案不按"年级"组织，而是按**能力域 × 阶段**组织。年级只是时间容器，能力增长才是核心。

```
共享基础 (Year 1-2 持续)
    │
    ├── 量化交易轨道 ──────┐
    │                      ├── 研究问题 → 数据验证 → 可复现产出 → 申请叙事
    └── 计算神经科学轨道 ──┘
```

三条原则：
1. **共享基础先于方向分化**：数学/编程/ML 是两条轨道的共同底座
2. **双轨并行，不提前二选一**：在大二结束前两条轨道都保持投入，直到研究问题自然收束
3. **每一阶段都要有可验证产出**：不是"学完了"，而是"产出了什么证据"

## 三、阶段划分

### Stage 1：共享基础 (Year 1 — 当前阶段)

**已完成**：
- 数学7个子领域全覆盖（微积分/线代/概率/抽象/高等/优化/离散，~134篇笔记）
- C++ 从基础语法到37个竞赛模板（91篇笔记）
- Python 科学计算与PyTorch入门（15篇笔记）
- 数据结构9章 + CLRS第1-31章（~40篇笔记）
- 曼昆宏观经济学15章

**当前重点**：
1. 经济学微观部分补充
2. 量化金融六个模块内容深化（当前偏框架，需加具体分析）
3. 计算神经科学六个模块内容深化
4. 启动 daily/weekly 复盘闭环

**验收标准**：
- GPA ≥ 3.8
- 概率论/优化理论能独立完成 C 档习题
- 完成第一个端到端数据分析项目（Python）

### Stage 2：双轨并行 (Year 2)

**量化轨道产出**：
- 第一个完整量化策略（选股→回测→风险分析→样本外验证）
- 一份策略文档 + 可复现代码
- 因子模型实战：至少完成一个多因子策略

**神经轨道产出**：
- 第一个神经建模 mini project（用Brian2/NEST实现）
- 一份建模笔记 + 参数分析
- 复现一篇经典计算神经科学论文的模型

**共享方法层**：
- ML 基础（监督/无监督/特征工程/模型评估）
- 随机过程与随机微积分（已有5篇笔记，继续深化）

**验收标准**：
- 两条轨道各有至少一个可展示的项目
- 能说清"为什么选择这个问题"和"方法为什么有效"

### Stage 3：专题深化 (Year 3)

**核心任务**：
- 从 Stage 2 的项目中识别出可深入的研究问题
- 文献综述：在选定方向上阅读 20-30 篇论文
- 确定主攻方向（量化或神经，不排除交叉）
- 开始导师联系和研究组探索

**产出**：
- 一份文献综述 + 研究提案
- 第一个小规模原创工作（不要求发表，但要求完整）

### Stage 4：研究收束 (Year 4)

**核心任务**：
- 研究深化 → 论文写作 → 投稿
- SOP / Personal Statement 撰写
- 推荐信准备
- GRE / TOEFL（如需）

**产出**：
- 至少一篇可投稿的 working paper
- 完整的申请材料包
- 清晰的研究叙事：为什么做这个 → 怎么做的 → 发现了什么 → 下一步

## 四、每周执行节奏

```
周一至周五（每天4-5小时学习）：
  - 绩点课程：~2.5h（≥70%）
  - 双路径推进：~1.5h（量化/神经交替，每天一个方向）
  - 每日记录：10分钟

周六：
  - 竞赛/项目集中：~6h
  - 量化项目 or 神经建模

周日：
  - 轻量复习：~3h
  - 周复盘 + 下周计划：30分钟
```
