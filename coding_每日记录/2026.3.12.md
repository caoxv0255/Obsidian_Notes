## 📚 模块1：分治思想直观理解

### 分治（Divide and Conquer）核心特征

1. **问题可分**：原问题能分解为规模更小的相同问题（如数组可二分）
2. **子问题独立**：各子问题的求解互不干扰（排序左右子数组互不影响）
3. **解可合并**：子问题的解能合并为原问题的解（有序子数组可合并为整体有序）

### 归并排序的"分-治-合"流程

**分（Divide）**
- 原始数组 [38, 27, 43, 3, 9, 82, 10]
- 从中间分成两半：左 [38, 27, 43, 3]，右 [9, 82, 10]
- 继续递归分割，直到每个子数组长度为1（天然有序）

**治（Conquer）**
- 递归地对每个子数组进行归并排序
- 长度为1的子数组已有序，递归到底部自动返回

**合（Combine）**
- 将两个已排序的子数组合并为一个有序数组
- 比较两个子数组的头部元素，取较小者放入结果
- 重复直到所有元素合并完毕

## ⚡ 模块2：快速排序模板

### Lomuto分区法（易于理解）

```cpp
#include <vector>
using namespace std;

// Lomuto分区函数：返回基准值的最终位置
int partition(vector<int>& nums, int left, int right) {
    int pivot = nums[right];  // 选择最右元素作为基准
    int i = left - 1;         // i指向小于基准的区域的末尾

    for (int j = left; j < right; j++) {
        if (nums[j] < pivot) {
            i++;
            swap(nums[i], nums[j]);
        }
    }
    // 将基准放到正确位置
    swap(nums[i + 1], nums[right]);
    return i + 1;
}

// 递归快速排序主函数
void quickSort(vector<int>& nums, int left, int right) {
    // 递归终止条件：子数组长度 <= 1
    if (left >= right) return;

    int pivotIndex = partition(nums, left, right);
    // 递归排序左半部分（小于基准）
    quickSort(nums, left, pivotIndex - 1);
    // 递归排序右半部分（大于等于基准）
    quickSort(nums, pivotIndex + 1, right);
}

// 对外接口
void quickSort(vector<int>& nums) {
    if (nums.empty()) return;
    quickSort(nums, 0, nums.size() - 1);
}
```

### 逐步调试示例（[3, 1, 4, 2, 5]）

**第一次调用 quickSort(nums, 0, 4)**
- pivot = nums[4] = 5
- 分区后数组变为 [3, 1, 4, 2, 5]（5已在最右，无需移动）
- pivotIndex = 4
- 递归左子数组 [3, 1, 4, 2]，右子数组为空

**第二次调用 quickSort(nums, 0, 3)**
- pivot = 2
- 分区过程：
  - j=0: 3>2，不动
  - j=1: 1<2，i=0，交换 nums[0]↔nums[1] → [1, 3, 4, 2]
  - j=2: 4>2，不动
  - j=3: 2=2，循环结束
- 交换 nums[i+1]↔nums[3] → [1, 2, 4, 3]
- pivotIndex = 1
- 递归左子数组 [1]（长度1直接返回），右子数组 [4, 3]

### 时间复杂度分析

- **平均情况**：O(n log n)，每次分区大致均匀
- **最坏情况**：O(n²)，当数组已有序且每次都选最右为基准
- **优化策略**：随机选择基准 `swap(nums[right], nums[left + rand() % (right-left+1)])`

## 🔄 模块3：归并排序模板

### 递归归并排序（自顶向下）

```cpp
#include <vector>
using namespace std;

// 合并两个有序子数组 nums[left..mid] 和 nums[mid+1..right]
void merge(vector<int>& nums, int left, int mid, int right) {
    vector<int> temp(right - left + 1);
    int i = left, j = mid + 1, k = 0;

    while (i <= mid && j <= right) {
        if (nums[i] <= nums[j]) {
            temp[k++] = nums[i++];
        } else {
            temp[k++] = nums[j++];
        }
    }

    // 复制剩余元素
    while (i <= mid) temp[k++] = nums[i++];
    while (j <= right) temp[k++] = nums[j++];

    // 将排序结果复制回原数组
    for (int idx = 0; idx < temp.size(); idx++) {
        nums[left + idx] = temp[idx];
    }
}

// 递归归并排序
void mergeSortRecursive(vector<int>& nums, int left, int right) {
    if (left >= right) return;

    int mid = left + (right - left) / 2;
    mergeSortRecursive(nums, left, mid);    // 排序左半
    mergeSortRecursive(nums, mid + 1, right); // 排序右半
    merge(nums, left, mid, right);          // 合并
}

// 对外接口
void mergeSort(vector<int>& nums) {
    if (nums.empty()) return;
    mergeSortRecursive(nums, 0, nums.size() - 1);
}
```

### 迭代归并排序（自底向上，避免递归栈）

```cpp
void mergeSortIterative(vector<int>& nums) {
    int n = nums.size();
    if (n <= 1) return;

    // 步长从1开始，每次翻倍
    for (int step = 1; step < n; step *= 2) {
        for (int left = 0; left < n; left += 2 * step) {
            int mid = min(left + step - 1, n - 1);
            int right = min(left + 2 * step - 1, n - 1);
            merge(nums, left, mid, right);
        }
    }
}
```

### 空间复杂度对比

| 写法 | 空间复杂度 | 优点 | 缺点 |
|------|-----------|------|------|
| 递归归并 | O(n)（临时数组）+ O(log n)（递归栈） | 代码直观，易于理解 | 递归深度受栈限制 |
| 迭代归并 | O(n)（仅临时数组） | 无递归栈开销，适合大规模数据 | 代码稍复杂，边界处理需小心 |

## 🎨 模块4：LeetCode 75颜色分类

### 问题描述

给定一个包含红色（0）、白色（1）和蓝色（2）的数组，原地对它们进行排序，使得相同颜色的元素相邻，且按红、白、蓝顺序排列。不允许使用库的排序函数。

### 荷兰国旗问题（三向快排）

用三个指针将数组分为三个区域：
- [0, p0)：全是0（红色）
- [p0, i)：全是1（白色）
- [i, p2]：未处理区域
- (p2, n-1]：全是2（蓝色）

### 标准解法（一次遍历，O(n)）

```cpp
void sortColors(vector<int>& nums) {
    int n = nums.size();
    int p0 = 0;      // 指向下一个0应该放的位置
    int p2 = n - 1;  // 指向下一个2应该放的位置
    int i = 0;       // 当前遍历指针

    while (i <= p2) {
        if (nums[i] == 0) {
            swap(nums[i], nums[p0]);
            p0++;
            i++;
        } else if (nums[i] == 1) {
            i++;
        } else { // nums[i] == 2
            swap(nums[i], nums[p2]);
            p2--;
            // 注意：交换后nums[i]可能是0或1，不能直接i++，需要再次判断
        }
    }
}
```

### 调试步骤示例（[2,0,2,1,1,0]）

| 步骤 | p0 | i | p2 | 数组状态 | 动作 |
|------|----|----|----|---------|------|
| 初始 | 0 | 0 | 5 | [2,0,2,1,1,0] | nums[0]=2 → 与nums[5]交换 |
| 1 | 0 | 0 | 4 | [0,0,2,1,1,2] | nums[0]=0 → 与nums[0]交换（自身），p0++, i++ |
| 2 | 1 | 1 | 4 | [0,0,2,1,1,2] | nums[1]=0 → 交换，p0++, i++ |
| 3 | 2 | 2 | 4 | [0,0,2,1,1,2] | nums[2]=2 → 与nums[4]交换 |
| 4 | 2 | 2 | 3 | [0,0,1,1,2,2] | nums[2]=1 → i++ |
| 5 | 2 | 3 | 3 | [0,0,1,1,2,2] | nums[3]=1 → i++，i>p2结束 |

## 🌳 模块5：递归树可视化

### 快速排序递归树示例（[3,1,4,2,5]）

```
第一层（根）：[3,1,4,2,5]  pivot=5
              分区后：左=[3,1,4,2] 右=[]
              /                \
第二层：   [3,1,4,2] pivot=2      []（空，直接返回）
         分区后：左=[1] 右=[4,3]
         /        \
第三层： [1]       [4,3] pivot=3
        (返回)     分区后：左=[] 右=[4]
                   /        \
第四层：         []         [4]
                           (返回)
```

**树的性质**：
- 深度：最坏情况（数组已有序）深度为n，平均深度为log n
- 每层工作量：分区操作的总和为O(n)
- 总工作量：平均O(n log n)，最坏O(n²)

### 归并排序递归树示例（同一数组）

```
第一层：      [3,1,4,2,5]
              分成 [3,1,4] 和 [2,5]
              /               \
第二层：   [3,1,4]           [2,5]
          分成 [3] [1,4]     分成 [2] [5]
         /        \         /        \
第三层： [3]      [1,4]    [2]      [5]
               分成 [1] [4]
               /        \
第四层：      [1]       [4]
```

**合并过程（自底向上）**：
- 第四层：合并[1]和[4] → [1,4]
- 第三层：合并[3]和[1,4] → [1,3,4]；合并[2]和[5] → [2,5]
- 第二层：合并[1,3,4]和[2,5] → [1,2,3,4,5]

### 递归树与时间复杂度证明

**快速排序平均情况**
- 每次分区大致均匀 → 递归树近似平衡 → 深度O(log n) × 每层O(n) = O(n log n)

**归并排序稳定O(n log n)**
- 递归树总是平衡的（固定二分） → 深度O(log n) × 每层合并总O(n) = O(n log n)
