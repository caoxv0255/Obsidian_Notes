## 📚 模块1：LeetCode 22 括号生成——DFS模拟与合法性检查

### 问题描述

数字 n 代表生成括号的对数，请你设计一个函数，生成所有可能并且有效的括号组合。

### 核心思想：DFS模拟括号添加过程

括号生成问题的本质是在每一步选择添加左括号或右括号，但必须满足以下合法性约束：

- 左括号数不能超过n：最多只能有n个左括号
- 右括号数不能超过左括号数：任意时刻，已添加的右括号数 ≤ 左括号数

### 状态空间搜索模型

- **状态**：当前已添加的左括号数 left、右括号数 right、当前括号字符串 current
- **选择**：添加左括号（如果 left < n）或添加右括号（如果 right < left）
- **结束条件**：left == n && right == n，得到一个有效括号组合

### 完整代码模板（C++）

```cpp
#include <vector>
#include <string>
using namespace std;

class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> result;
        string current;
        backtrack(n, 0, 0, current, result);
        return result;
    }
    
private:
    void backtrack(int n, int left, int right, string& current, vector<string>& result) {
        // 终止条件：左右括号都已用完
        if (left == n && right == n) {
            result.push_back(current);
            return;
        }
        
        // 选择1：添加左括号（如果还有左括号可用）
        if (left < n) {
            current.push_back('(');
            backtrack(n, left + 1, right, current, result);
            current.pop_back(); // 撤销选择
        }
        
        // 选择2：添加右括号（如果右括号数小于左括号数）
        if (right < left) {
            current.push_back(')');
            backtrack(n, left, right + 1, current, result);
            current.pop_back(); // 撤销选择
        }
    }
};
```

### 逐步调试示例（n=3）

```
初始状态: left=0, right=0, current=""
深度1: 添加左括号 → left=1, right=0, current="("
深度2: 添加左括号 → left=2, right=0, current="(("
深度3: 添加左括号 → left=3, right=0, current="((("
深度4: 添加右括号 → left=3, right=1, current="((()"
深度5: 添加右括号 → left=3, right=2, current="((())"
深度6: 添加右括号 → left=3, right=3, current="((()))"
到达终止条件 → 加入结果集: "((()))"
```

### 时间复杂度分析

- 无剪枝理论状态数：2^(2n)
- 剪枝后实际状态数：卡特兰数 C(n) = (1/(n+1)) × C(2n, n)
- 时间复杂度：O(C(n))，即 O(4^n / (n√n))
- 空间复杂度：O(n)（递归栈深度 + 当前字符串长度）

**零压力提示**：掌握DFS模板与合法性检查即达标，复杂度分析为可选深化内容。

## 📚 模块2：LeetCode 39 组合总和——无限取用回溯与剪枝策略

### 问题描述

给你一个无重复元素的整数数组 candidates 和一个目标整数 target，找出 candidates 中可以使数字和为目标数 target 的所有不同组合。candidates 中的同一个数字可以无限制重复被选取。

### 核心思想：排序+回溯+剪枝

- 排序预处理：对数组排序，方便剪枝和去重
- 状态定义：当前和 sum、当前索引 start（避免重复组合）、当前路径 path
- 选择策略：从 start 开始遍历，每个元素可以多次使用
- 剪枝优化：当 sum + candidates[i] > target 时，由于数组已排序，后续元素只会更大，可直接跳出循环

### 完整代码模板（C++）

```cpp
#include <vector>
#include <algorithm>
using namespace std;

class Solution {
public:
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        vector<vector<int>> result;
        vector<int> path;
        sort(candidates.begin(), candidates.end()); // 关键：排序为剪枝准备
        backtrack(candidates, target, 0, 0, path, result);
        return result;
    }
    
private:
    void backtrack(vector<int>& candidates, int target, int start, int sum, 
                   vector<int>& path, vector<vector<int>>& result) {
        // 终止条件：和等于目标值
        if (sum == target) {
            result.push_back(path);
            return;
        }
        
        // 遍历选择列表：从start开始，避免重复组合
        for (int i = start; i < candidates.size(); i++) {
            // 剪枝：如果加上当前元素已超目标值，由于数组已排序，后续元素只会更大
            if (sum + candidates[i] > target) {
                break; // 直接结束循环，不再尝试后续元素
            }
            
            // 做选择
            path.push_back(candidates[i]);
            // 递归：注意start仍为i（允许重复使用当前元素）
            backtrack(candidates, target, i, sum + candidates[i], path, result);
            // 撤销选择
            path.pop_back();
        }
    }
};
```

### 逐步调试示例（candidates = [2,3,6,7], target = 7）

```
初始状态: start=0, sum=0, path=[]
深度1: 选择2 → sum=2, path=[2], 递归(start=0)
深度2: 选择2 → sum=4, path=[2,2], 递归(start=0)
深度3: 选择2 → sum=6, path=[2,2,2], 递归(start=0)
深度4: 选择2 → sum=8 > target → 剪枝，回溯
深度3: 选择3 → sum=7, path=[2,2,3], 递归(start=1)
  到达终止条件 → 加入结果集: [2,2,3]
...
最终得到结果: [[2,2,3],[7]]
```

### 关键观察

- 排序后剪枝：当 sum + candidates[i] > target 时，i 之后的所有元素都会更大，可以直接 break
- 避免重复组合：通过 start 参数确保组合按非递减顺序生成
- 无限取用：递归时 start = i（不是 i+1），允许重复使用当前元素

**零压力提示**：掌握排序回溯模板与基本剪枝即达标，复杂分析为可选深化内容。

## 📚 模块3：实际问题转化——从问题描述到状态空间建模

### 通用转化步骤

任何适合用搜索算法解决的问题，都可以通过以下四步转化为状态空间模型：

1. **识别状态变量**：确定哪些信息需要记录来表示当前进展
2. **定义选择列表**：在当前状态下，有哪些合法的下一步选择
3. **设定结束条件**：何时当前路径构成一个有效解
4. **设计剪枝策略**：如何提前排除不可能到达解的分支

### 状态空间树构建示例（括号生成 n=2）

```
根节点: (0,0) ""
├─ 左分支: (1,0) "("
│   ├─ 左分支: (2,0) "(("  ❌ 无效
│   └─ 右分支: (1,1) "()"
│       ├─ 左分支: (2,1) "()("
│       │   └─ 右分支: (2,2) "()()" ✅ 解
│       └─ 右分支: (1,2) "())" ❌ 无效
└─ 右分支: (0,1) ")" ❌ 无效
```

#### LeetCode 22 括号生成（提交ID: 708276537）

**性能数据**：运行时 0 ms（击败 100% 用户），内存 9.4 MB（击败 99.16% 用户）

**通过测试用例**：8/8

**AC代码**：

```cpp
class Solution {
public:
    vector<string> generateParenthesis(int n) {
        vector<string> res;
        string cur(2*n,0);
        int left=0,right=0;
        backtracking(res, cur, n, left, right);
        return res;
    }

private:
    void backtracking(vector<string>& res, string &cur, int n, int left,
                      int right) {
        if (left == n && right == n) {
            res.push_back(cur);
            return;
        }
        if (left < n) {
            cur[left+right]='(';
            backtracking(res, cur, n, left + 1, right);
        }
        if (right < left) {
            cur[left+right]=')';
            backtracking(res, cur, n, left, right + 1);
        }
    }
};
```

**代码亮点**：
- 使用固定长度字符串 `string cur(2*n,0)` 直接在正确位置写入括号，避免 push_back/pop_back 操作
- 字符串索引 `left+right` 精确定位当前应写入的位置
- 剪枝条件正确：左括号 `< n`，右括号 `< 左括号数`

#### LeetCode 39 组合总和（提交ID: 708541063）

**性能数据**：运行时 3 ms（击败 64.99% 用户），内存 13.8 MB（击败 89.21% 用户）

**通过测试用例**：160/160

**AC代码**：

```cpp
class Solution {
public:
    vector<vector<int>> combinationSum(vector<int>& candidates, int target) {
        sort(candidates.begin(), candidates.end());
        vector<vector<int>> res;
        vector<int> cur;
        int sum = 0;
        int start = 0;
        backtracking(res, candidates, target, cur, sum, start);
        return res;
    }

private:
    void backtracking(vector<vector<int>>& res, vector<int>& candidates,
                      int target, vector<int>& cur, int sum, int start) {
        if (sum == target) {
            res.push_back(cur);
            return;
        }
        for (int i = start; i < candidates.size(); ++i) {
            if (sum + candidates[i] > target) {
                break;
            }
            cur.push_back(candidates[i]);
            backtracking(res, candidates, target, cur, sum + candidates[i], i);
            cur.pop_back();
        }
    }
};
```

**代码亮点**：
- 预处理排序，为剪枝提供单调性基础
- 剪枝条件 `sum + candidates[i] > target` + break 实现阈值剪枝
- 递归传递 `i`（而非 `i+1`），实现元素无限取用
- 通过 `start` 参数避免生成重复组合
