- ← 应用 → [[环形检测（LeetCode 141）]]
- ← 应用 → [[环入口数学（LeetCode 142）]]

#### 知识点子节点

**离散存储原理**
- 节点在内存中离散分布
- 通过指针连接
- 优点：插入删除 O(1)

**指针连接机制**
- 节点包含数据和next指针
- 最后一个节点指向null
- 支持动态扩容

**头插法反转**
- 新节点指向原头节点
- 修改head指向新节点
- 时间复杂度 O(n)

**递归反转对比**
- 递归法：代码简洁，空间O(n)
- 迭代法：空间O(1)
- 理解递归的栈帧

**环形检测（LeetCode 141）**
- 快慢指针法
- 快指针每次走2步
- 若有环必相遇

**环入口数学（LeetCode 142）** #强度渐进Day4
- 相遇点到环入口距离 = 头到环入口距离
- 数学推导证明
- 重置慢指针到头部

## 🔗 知识点关联

### 跨维度连接

**数组 ↔ 链表**
- 数组：适合随机访问，不适合频繁插入删除
- 链表：适合频繁插入删除，不适合随机访问

**栈 ↔ 队列**
- 栈：后进先出，括号匹配
- 队列：先进先出，层序遍历

**双指针法的统一思想**
- 数组：快慢指针（删除重复项）
- 链表：快慢指针（环形检测）
- 应用：一次遍历解决问题
