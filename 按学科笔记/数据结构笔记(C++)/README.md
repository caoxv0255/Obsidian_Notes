## 📚 项目简介

本笔记系统基于《数据结构（C++版）》教材编写，旨在提供一套完整、详细、实用的数据结构学习资料。每章都包含：

- ✅ **完整C++代码实现**：使用现代C++17/20特性，代码可编译、可运行
- ✅ **详细复杂度分析**：时间复杂度和空间复杂度的推导过程
- ✅ **多种实现方案**：不同方案的对比和优缺点分析
- ✅ **实际应用场景**：工业级应用案例和代码示例
- ✅ **常见陷阱分析**：易错点和调试技巧
- ✅ **优化技巧**：进阶优化方法和性能提升技巧
- ✅ **丰富练习题**：基础/进阶/挑战三个难度层次
- ✅ **知识图谱**：使用Mermaid图表展示概念关系

## 📝 笔记格式说明

### Frontmatter 元数据

每个笔记文件都包含YAML格式的元数据：

```yaml
---
type: chapter
topic: 章节主题
category: data_structures
difficulty: beginner
language: C++20
created: 2026-03-05
status: learning
acm_relevant: true
---
```

**字段说明**：
- `type`：笔记类型（chapter/concept/example）
- `topic`：主题名称
- `category`：分类（data_structures/algorithms）
- `difficulty`：难度（beginner/intermediate/advanced）
- `language`：编程语言
- `created`：创建日期
- `status`：状态（learning/completed）
- `acm_relevant`：是否与ACM/算法竞赛相关

### 代码示例格式

```cpp
/**
 * @brief 数据结构名称
 * @details 详细说明
 * 
 * @tparam T 元素类型
 */

class ClassName {
    // 实现代码
};
```
