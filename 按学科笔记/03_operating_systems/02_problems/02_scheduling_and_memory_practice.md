# 操作系统练习：调度与内存

## 练习 1：调度指标计算

给定三个进程的到达时间和运行时间，请分别计算 FCFS、RR 和 SJF 下的等待时间、周转时间与平均响应时间。

### 参考要点

- 先明确调度顺序。
- 再逐个推导完成时刻。
- 最后汇总平均指标。

## 练习 2：页面置换比较

给定访问串，请比较 FIFO、LRU 和 OPT 的缺页次数，并分析为什么某些策略会出现 Belady 现象。

### 参考要点

- FIFO 只看先后，不看使用频率。
- LRU 利用局部性。
- OPT 是理论最优基线。

## 练习 3：死锁判断

请给出一个资源分配图，判断系统是否可能死锁，并说明如果资源是单实例和多实例时结论有何不同。

### 参考要点

- 资源请求是否形成环路。
- 单实例时环路和死锁高度相关。
- 多实例时还要结合剩余资源判断。
