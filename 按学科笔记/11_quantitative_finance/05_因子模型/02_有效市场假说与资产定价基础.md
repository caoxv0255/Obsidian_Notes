## 1. 有效市场假说（EMH）

### 1.1 什么是有效市场？

**有效市场**（Efficient Market）是指资产价格**完全反映**所有可用信息的市场。在这样的市场中：

- 没有套利机会（价格等于基本面价值）
- 价格变动不可预测（新信息随机产生）
- 任何分析方法都无法持续获得超额收益

> 有效市场并非"价格永远正确"，而是"错误定价会被快速纠正"

### 1.2 EMH 的三种形式

有效市场假说分为三个层次，反映了不同的信息集：

```
信息集      →     弱式有效（Weak Form）
    ↓
    ├─ 历史价格信息
    └─ 技术分析无效

信息集      →     半强式有效（Semi-Strong Form）
    ↓
    ├─ 历史价格 + 公开信息
    └─ 基本面分析无效

信息集      →     强式有效（Strong Form）
    ↓
    ├─ 历史价格 + 公开信息 + 私有信息
    └─ 内幕交易无效
```

| 形式 | 包含信息 | 失效的分析方法 | 实证支持 |
|------|----------|----------------|----------|
| 弱式有效 | 历史价格 | 技术分析 | **强** |
| 半强式有效 | 历史价格 + 公开信息 | 基本面分析 | **中等** |
| 强式有效 | 全部信息（包括内幕） | 内幕交易 | **弱** |

### 1.3 有效市场的数学描述

在有效市场中，下一期价格满足：

$$P_{t+1} = P_t + \text{新信息} + \text{随机扰动}$$

其中"新信息"是不可预测的随机变量：

$$\text{新信息} \sim N(0, \sigma^2_{\epsilon})$$

**收益率序列的自协方差为零**：

$$\text{Cov}(r_t, r_{t-k}) = 0, \quad \forall k \neq 0$$

> 这意味着收益率服从**随机游走**（Random Walk），价格变动不可预测。

### 1.4 有效市场的局限性与反例

有效市场假说并非无懈可击，以下现象挑战了 EMH：

| 现象 | 描述 | 含义 |
|------|------|------|
| **规模效应**（Size Effect） | 小市值股票长期表现优于大市值 | 定价效率存疑 |
| **价值效应**（Value Effect） | 低估值股票（高B/P）收益高于高估值 | 市盈率预测力 |
| **动量效应**（Momentum） | 过去赢家倾向继续跑赢 | 短期趋势持续 |
| **反转效应**（Reversal） | 长期看过去输家跑赢赢家 | 过度反应修正 |
| **杠杆效应**（Leverage Effect） | 波动率与收益率负相关 | 非对称风险 |

> **关键认知**：这些"异象"（Anomalies）并非否定有效市场，而是揭示了有效市场的**边界条件**。量化交易的核心，正是在这些边界处寻找盈利机会。

## 2. 风险与收益的基本关系

### 2.1 收益率的描述性统计

对于单一资产，收益率 $r$ 的基本统计量：

```python
import numpy as np
import pandas as pd

# 假设有日收益率序列
returns = np.random.normal(0.001, 0.02, 252)  # 年化 μ=25%, σ=32%

stats = {
    '均值': np.mean(returns) * 252,           # 年化收益率
    '标准差': np.std(returns) * np.sqrt(252), # 年化波动率
    '偏度': pd.Series(returns).skew(),         # 偏度（对称性）
    '峰度': pd.Series(returns).kurtosis(),     # 超额峰度（厚尾程度）
    '夏普比率': (np.mean(returns)*252) / (np.std(returns)*np.sqrt(252)),
}

# 收益率分布检验
from scipy import stats as st
jb_stat, jb_pvalue = st.jarque_bera(returns)
print(f"Jarque-Bera检验: 统计量={jb_stat:.2f}, p值={jb_pvalue:.4f}")
# p < 0.05 → 拒绝正态分布假设
```

### 2.2 风险厌恶与无差异曲线

**风险厌恶**（Risk Aversion）是现代金融学的核心假设之一：

$$u(w) = \mathbb{E}[w] - \frac{1}{2} A \cdot \text{Var}(w)$$

- $A > 0$：风险厌恶系数
- $A = 0$：风险中性
- $A < 0$：风险偏好

投资者在**风险与收益**之间的权衡，构成了其**无差异曲线**（Indifference Curve）：

```
收益 U
  ↑        陡峭的曲线 = 高风险厌恶
  |       /
  |     /   ← 更偏好收益
  |   /
  | /        ← 更偏好低风险
  +————————————→ 风险 σ
```

### 2.3 系统性风险 vs 非系统性风险

| 风险类型 | 别名 | 来源 | 能否通过分散化消除 |
|----------|------|------|-------------------|
| **系统性风险** | 市场风险、不可分散风险 | 宏观经济、政策、利率 | ❌ 无法消除 |
| **非系统性风险** | 特异性风险、可分散风险 | 公司特有事件 | ✅ 可通过组合分散 |

```
总风险 = 系统性风险² + 非系统性风险²
σ²_total = β² · σ²_market + σ²_idiosyncratic
```

> **核心结论**：理性的投资者只需要承担**系统性风险**作为补偿，非系统性风险没有风险溢价——这就是多因子模型的起点。

## 3. 无套利原则的数学表述

### 3.1 套利（Arbitrage）的定义

**套利**是指在无风险的情况下获得正收益的交易：

1. **一价定律**（Law of One Price）：相同现金流的资产价格相同
2. **无套利条件**：不存在"空手套白狼"的机会

数学上，如果两个资产未来现金流相同，则当前价格必须相同：

$$P_A = P_B \quad \text{如果} \quad \text{CF}_A(t) = \text{CF}_B(t), \forall t$$

### 3.2 定价核（Pricing Kernel）

在无套利市场中，存在一个**定价核**（或称随机贴现因子）$M_t$，使得所有资产价格满足：

$$P_0 = \mathbb{E}[M_1 \cdot CF_1]$$

- $M_1$：从时刻 1 到时刻 0 的随机贴现因子
- 如果市场完备，$M_1$ 唯一

**股票定价**：

$$P_0 = \mathbb{E}[M_1] \cdot \mathbb{E}[P_1 + D_1] + \text{Cov}(M_1, P_1 + D_1)$$

右边第一项是**风险中性定价**（忽略风险溢价），第二项是**风险调整**。

### 3.3 风险厌恶条件下的定价

假设代表性投资者的效用函数为 $u(W)$，则定价核：

$$M_1 = \frac{u'(W_1)}{u'(W_0)}$$

对于**CARA**（常数绝对风险厌恶）效用：

$$u(W) = -\exp(-A \cdot W) \quad \Rightarrow \quad M_1 = \exp(-A \cdot W_1 + A \cdot \mathbb{E}[W_1])$$

对于**CRRA**（常数相对风险厌恶）效用：

$$u(W) = \frac{W^{1-\gamma}}{1-\gamma} \quad \Rightarrow \quad M_1 = \left(\frac{W_0}{W_1}\right)^\gamma$$

> 不同的风险厌恶假设，推导出不同的定价核，进而得到不同的资产定价模型。

## 4. 定价基本定理与资产定价

### 4.1 定价基本定理

**第一基本定理**：市场无套利 ⟺ 存在（可能非唯一）定价核 $M$

**第二基本定理**：市场完备（无冗余资产）⟺ 定价核 $M$ 唯一

$$\text{存在 } M > 0 \text{ 使得 } P = \mathbb{E}[M \cdot X], \forall \text{现金流 } X$$

### 4.2 期望收益率与定价核的关系

从 $P_0 = \mathbb{E}[M_1 \cdot X_1]$，可得：

$$1 = \mathbb{E}[M_1 \cdot (1 + r_X)] \quad \Rightarrow \quad \mathbb{E}[M_1 \cdot r_X] = 1 - \mathbb{E}[M_1]$$

**期望超额收益率**：

$$\mathbb{E}[r_i] - r_f = -\frac{\text{Cov}(M_1, r_i)}{\mathbb{E}[M_1]} = \lambda_i$$

这就是 **ICAPM**（跨期资本资产定价模型）的核心公式：风险溢价取决于资产收益率与定价核的协方差。

## 5. Python 实践：验证有效市场特征

```python
import numpy as np
import pandas as pd
from scipy import stats

# 模拟：有效市场下的日收益率（随机游走）
np.random.seed(42)
n = 1000
returns = np.random.normal(0.0005, 0.01, n)  # 日均收益0.05%，波动1%

# 1. 检验自相关性（弱式有效）
from statsmodels.stats.stattools import jarque_bera
from statsmodels.tsa.stattools import acf

acf_values = acf(returns, nlags=5)
print("自相关系数（前5阶）:", acf_values[1:])
# 如果接近0 → 符合弱式有效

# 2. 收益率分布检验
jb_stat, jb_pvalue = jarque_bera(returns)
print(f"Jarque-Bera: 统计量={jb_stat:.2f}, p值={jb_pvalue:.4f}")
# p > 0.05 → 不能拒绝正态分布假设

# 3. 运行检验（Run Test）— 检验价格变动是否随机
signs = np.sign(returns)
runs = np.sum(signs[1:] != signs[:-1]) + 1
n_pos = np.sum(signs == 1)
n_neg = np.sum(signs == -1)
expected_runs = (2 * n_pos * n_neg) / (n_pos + n_neg) + 1
print(f"实际游程数={runs}, 期望游程数={expected_runs:.1f}")
```

## 6. 章节小结

### 核心概念

| 概念 | 定义 | 应用 |
|------|------|------|
| 弱式有效 | 历史价格已反映在当前价格 | 技术分析无效 |
| 半强式有效 | 公开信息已反映在当前价格 | 基本面分析无效 |
| 系统性风险 | 无法通过分散化消除 | 风险溢价来源 |
| 无套利 | 不存在零成本获利机会 | 定价核存在 |
| 定价核 $M$ | 折现现金流的概率权重 | 所有资产统一定价 |
