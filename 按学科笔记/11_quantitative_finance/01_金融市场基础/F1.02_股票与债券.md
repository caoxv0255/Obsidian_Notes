# F1.02 股票与债券

> 本章区分两类核心金融工具：股票（权益资产）与债券（固定收益资产），建立正确的概念框架。

## 1. 股票 (Stock / Equity)

### 1.1 法律本质：权益凭证

> **股票**是股份有限公司发行的、证明股东持有公司股份的可转让凭证。

**关键特征**：

| 特征 | 含义 |
|------|------|
| **所有权** | 持有股票 = 持有公司部分所有权 |
| **剩余求偿权** | 公司清算时，股东排在债权人之后 |
| **投票权** | 普通股股东享有公司决策投票权 |
| **收益权** | 分享公司利润（股利） |
| **风险** | 无固定收益，收益不确定 |

### 1.2 普通股 vs 优先股

| 属性 | 普通股 | 优先股 |
|------|--------|--------|
| **表决权** | ✅ 有 | ❌ 无（通常） |
| **股利** | 不固定，视公司盈利 | 固定（或浮动） |
| **清偿顺序** | 最后 | 优先于普通股 |
| **累计股利** | 非累计 | 通常累计 |
| **转换** | 不可转换为普通股 | 可转换为普通股（可赎回） |

### 1.3 股票估值基础

**股利贴现模型 (DDM)**：

```python
def ddm_zero_growth(D, r):
    """
    零增长DDM：适用于稳定行业
    D: 固定股利
    r: 折现率
    """
    return D / r

def ddm_constant_growth(D0, g, r):
    """
    Gordon增长模型：股利以固定增长率g永续增长
    D0: 当前股利
    g: 增长率
    r: 折现率 (需满足 r > g)
    """
    return D0 * (1 + g) / (r - g)

# 示例
price = ddm_constant_growth(D0=1.5, g=0.05, r=0.10)
print(f"股票价值: {price:.2f} 元")  # 31.5 元
```

**市盈率 (P/E)**：

```
P/E = 市场价格 / 每股收益 (EPS)
```

低P/E可能意味着价值被低估，或市场预期低增长；高P/E可能意味着高增长预期或估值偏高。

## 2. 债券 (Bond / Fixed Income)

### 2.1 法律本质：债务凭证

> **债券**是发行人（政府、企业等）向投资者出具的、约定在一定期限内还本付息的债务融资工具。

**关键特征**：

| 特征 | 含义 |
|------|------|
| **债务关系** | 持有债券 = 对发行人的债权 |
| **固定收益** | 定期支付固定利息（票息） |
| **优先求偿权** | 公司清算时先于股东 |
| **到期还本** | 到期日偿还面值 |
| **风险** | 信用风险、利率风险、通胀风险 |

### 2.2 债券的构成要素

| 要素 | 说明 |
|------|------|
| **面值 (Face Value / Par)** | 到期偿还的金额，通常为1000元 |
| **票面利率 (Coupon Rate)** | 年利息 = 面值 × 票面利率 |
| **到期期限 (Maturity)** | 距离到期的时间 |
| **付息频率** | 年付、半年付、季度付等 |
| **发行价格** | 可能折价、溢价、平价发行 |

### 2.3 债券分类

```
债券分类
├── 按发行人
│   ├── 国债（国家信用）
│   ├── 地方政府债
│   ├── 企业债（公司债）
│   └── 金融债
├── 按票息
│   ├── 固定利率债券
│   ├── 浮动利率债券
│   └── 零息债券（折价发行）
└── 按担保
    ├── 抵押债券
    └── 信用债券（无担保）
```

### 2.4 债券估值

**基本公式**：

```python
def bond_price(face, coupon_rate, ytm, maturity, freq=1):
    """
    债券定价：现金流贴现
    face: 面值
    coupon_rate: 票面利率
    ytm: 到期收益率 (折现率)
    maturity: 到期期限（年）
    freq: 付息频率（默认年付）
    """
    coupon = face * coupon_rate / freq  # 每期利息
    periods = maturity * freq            # 总期数
    r_per = ytm / freq                  # 每期折现率
    
    # 利息现金流现值
    pv_coupons = 0
    for t in range(1, periods + 1):
        pv_coupons += coupon / (1 + r_per) ** t
    
    # 面值现值
    pv_face = face / (1 + r_per) ** periods
    
    return pv_coupons + pv_face

# 示例：面值1000，票息6%，YTN 8%，3年到期
price = bond_price(1000, 0.06, 0.08, 3)
print(f"债券价格: {price:.2f} 元")  # ≈ 948.46
```

### 2.5 债券价格与收益率的关系

**核心规律**：

| 市场利率 vs 票面利率 | 发行方式 | 价格变动 |
|---------------------|---------|----------|
| **>** (市场 > 票面) | 折价发行 | 价格 < 面值 |
| **=** (市场 = 票面) | 平价发行 | 价格 = 面值 |
| **<** (市场 < 票面) | 溢价发行 | 价格 > 面值 |

**价格-收益率曲线**：债券价格与收益率呈反向非线性关系（凸性）。

## 3. 股票 vs 债券：核心对比

### 3.1 关键区别

| 维度 | 股票 | 债券 |
|------|------|------|
| **法律性质** | 所有权凭证 | 债务凭证 |
| **收益特征** | 不固定（股利+资本利得） | 固定（票息） |
| **风险水平** | 较高 | 较低（信用风险内） |
| **清偿顺序** | 最后 | 优先 |
| **期限** | 永续（无到期日） | 有固定期限 |
| **控制权** | 有投票权 | 无投票权 |
| **税务** | 股利税后扣除 | 利息税前扣除（税收优惠） |

### 3.2 收益-风险对比

```
收益
  ↑
  │                    ★ 股票
  │                  ╱
  │                ╱
  │              ╱ ★ 可转债
  │            ╱
  │          ╱
  │        ╱ ★ 债券
  │      ╱
  │    ╱
  │  ╱
  │╱____________________________→ 风险
  └ 低 ────────────────────────→ 高
  (债券)              (股票)
```

## 4. 投资组合中的应用

### 4.1 资本配置线 (CAL)

```python
import numpy as np

def capital_allocation_line(rf, E_rp, std_p, risk_aversion):
    """
    资本配置线：风险资产组合与无风险资产的配置
    rf: 无风险利率
    E_rp: 风险资产组合预期收益
    std_p: 风险资产组合标准差
    risk_aversion: 风险厌恶系数 (A)
    
    返回：(波动率序列, 收益率序列)
    """
    sigma = np.linspace(0, 0.5, 100)
    # E(R) = Rf + A * σ² (均值-方差偏好)
    E_r = rf + risk_aversion * sigma ** 2
    return sigma, E_r

# 示例
sigma, E_r = capital_allocation_line(rf=0.03, E_rp=0.10, std_p=0.20, A=3)
```

### 4.2 股票与债券的配置逻辑

| 市场环境 | 股票配置 | 债券配置 | 逻辑 |
|----------|----------|----------|------|
| **经济扩张** | ↑ 高配 | ↓ 低配 | 企业盈利增长，利率上升 |
| **经济衰退** | ↓ 低配 | ↑ 高配 | 避险情绪，利率下降 |
| **通胀上升** | ↓ 低配 | ↓ 低配 | 现金、黄金、TIPS更优 |
