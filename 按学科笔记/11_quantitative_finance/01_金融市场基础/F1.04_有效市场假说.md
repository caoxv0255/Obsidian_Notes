## 1. 有效市场假说 (Efficient Market Hypothesis)

### 1.1 核心定义

> **法玛 (Eugene Fama, 1970)**：如果证券价格"充分反映"所有可用信息，则市场是有效的。

**"充分反映"的含义**：

| 信息类型 | 反映速度 | 价格含义 |
|----------|----------|----------|
| 历史价格 | 立即 | 反映历史信息 |
| 公开信息 | 立即 | 反映公开可用信息 |
| 所有信息 | 立即 | 反映内幕信息 |

### 1.2 三种有效市场形式

```
有效市场假说三种形式
│
├── 弱式有效 (Weak Form)
│   │  "历史价格信息已反映在当前价格中"
│   │  → 技术分析无效
│   │  → 统计检验：自相关检验、游程检验
│   ↓
├── 半强式有效 (Semi-Strong Form)
│   │  "所有公开信息已反映在当前价格中"
│   │  → 基本面分析无效
│   │  → 事件研究：盈利公告、拆股等
│   ↓
│
└── 强式有效 (Strong Form)
    "所有信息（公开+内幕）已反映在当前价格中"
    → 内幕交易也无法获利
    → 最强形式，现实中几乎不成立
```

### 1.3 形式对比

| 市场形式 | 价格反映内容 | 可预测性 | 技术分析 | 基本面分析 |
|----------|--------------|----------|----------|------------|
| **弱式** | 历史价格 | 不可预测 | ❌ 无用 | ✅ 有效 |
| **半强式** | 公开信息 | 不可预测 | ❌ 无用 | ❌ 无用 |
| **强式** | 内幕信息 | 不可预测 | ❌ 无用 | ❌ 无用 |

## 2. 理性框架：α与β

### 2.1 收益分解

在CAPM框架下，资产收益可分解为：

```
E(R_i) = R_f + β_i × [E(R_m) - R_f]
                      ↑
                  市场风险溢价

总收益 = 无风险收益 + β收益（市场系统风险） + α收益（超额收益）
 R_i     =  R_f    +    β收益              +  α
```

**三种收益来源**：

| 收益 | 符号 | 来源 | 性质 |
|------|------|------|------|
| 无风险收益 | R_f | 国债 | 确定性 |
| 市场收益 | β | 承担系统风险 | 随市场波动 |
| 超额收益 | α | 择时/选股能力 | 零和博弈 |

### 2.2 α的本质

> **α** (Alpha) 是策略相对于基准的超额收益，是量化交易的核心追求。

```python
def calculate_alpha(strategy_returns, benchmark_returns, risk_free_rate):
    """
    计算策略Alpha（詹森指数 Jensen's Alpha）
    
    α = E(R_strategy) - [Rf + β × (E(R_market) - Rf)]
    """
    # 超额收益
    excess_strategy = np.mean(strategy_returns) - risk_free_rate
    excess_market = np.mean(benchmark_returns) - risk_free_rate
    
    # Beta
    cov_matrix = np.cov(strategy_returns, benchmark_returns)
    beta = cov_matrix[0, 1] / np.var(benchmark_returns)
    
    # Alpha
    alpha = excess_strategy - beta * excess_market
    
    return alpha

# 示例
strategy_returns = np.array([0.05, 0.08, -0.02, 0.06])
benchmark_returns = np.array([0.03, 0.05, -0.01, 0.04])
alpha = calculate_alpha(strategy_returns, benchmark_returns, 0.02)
print(f"詹森Alpha: {alpha:.4f}")  # ≈ 0.01
```

### 2.3 β的意义

| β值 | 含义 | 市场敏感度 |
|-----|------|-----------|
| β > 1 | 进攻型 | 市场涨1%，该资产涨>1% |
| β = 1 | 市场中性 | 与市场同步 |
| β < 1 | 防御型 | 市场涨1%，该资产涨<1% |
| β < 0 | 反向 | 与市场负相关 |

## 3. 市场异象 (Market Anomalies)

> 有效市场假说的"裂缝"：某些可预测的收益模式挑战了市场有效性。

### 3.1 主要异象分类

```
市场异象
│
├── 定价异象（估值类）
│   ├── 市盈率效应（P/E ratio）— 低P/E股票长期跑赢
│   ├── 账面市值比效应（Book-to-Market）— HML因子
│   ├── 规模效应（Size）— 小盘股长期跑赢（SMID因子）
│   └── 股息率效应 — 高股息股票超额收益
│
├── 事件异象（事件驱动）
│   ├── 盈利公告效应 — 盈利超预期后继续上涨
│   ├── 拆股效应 — 拆股后短期上涨
│   └── 纳入指数效应 — 纳入指数后被动买入推高价格
│
├── 技术异象（动量/反转）
│   ├── 短期反转（1个月）— 涨后必跌，跌后必涨
│   ├── 动量效应（3-12个月）— 强者恒强，弱者恒弱
│   └── 长期反转（3-5年）— 跌后必涨，涨后必跌
│
└── 行为金融异象
    ├── 过度自信偏差
    ├── 损失厌恶
    ├── 羊群效应
    └── 锚定效应
```

### 3.2 动量效应与反转效应

```python
import pandas as pd
import numpy as np

def momentum_effect(prices, formation_period=6, holding_period=6):
    """
    动量效应检验：
    formation_period月动量 → holding_period月收益
    
    买入过去涨得最好的股票（赢家组合）
    卖出过去跌得最多的股票（输家组合）
    
    如果动量效应存在：赢家组合 > 输家组合
    """
    # 计算月度收益率
    returns = prices.pct_change().dropna()
    
    # 动量排序
    past_returns = returns.shift(1).rolling(formation_period).sum()
    
    # 分组
    quintiles = past_returns.apply(lambda x: pd.qcut(x, 5, labels=False), axis=1)
    
    # 持有期收益
    future_returns = returns.rolling(holding_period).sum().shift(-holding_period)
    
    # 赢家组合 vs 输家组合
    winner_returns = future_returns[quintiles == 4].mean()
    loser_returns = future_returns[quintiles == 0].mean()
    
    momentum_premium = winner_returns - loser_returns
    
    return momentum_premium

# 注：A股市场动量效应较弱，反转效应较强（与美股相反）
```

### 3.3 市盈率效应

```python
def pe_effect(returns, pe_ratios, n_quintiles=5):
    """
    市盈率效应检验：
    低P/E股票是否长期跑赢高P/E股票
    """
    # 按P/E分组
    pe_groups = pd.qcut(pe_ratios, n_quintiles, labels=['极低P/E', '低P/E', '中P/E', '高P/E', '极高P/E'])
    
    # 各组平均收益
    portfolio_returns = {}
    for i in range(n_quintiles):
        mask = pe_groups == i
        portfolio_returns[f'Q{i+1}'] = returns[mask].mean()
    
    return pd.Series(portfolio_returns)
```

## 4. 行为金融学视角

### 4.1 有限理性

有效市场假说的对立面是**有限理性**：

| 有效市场假说 | 行为金融学 |
|--------------|-------------|
| 投资者是理性的 | 投资者受认知偏差影响 |
| 价格是正确的 | 价格可能偏离基本价值 |
| 无套利机会 | 存在行为偏差导致的错误定价 |

### 4.2 核心行为偏差

| 偏差 | 定义 | 市场表现 |
|------|------|----------|
| **过度自信** | 高估自己判断准确度 | 过度交易、频繁买卖 |
| **损失厌恶** | 损失比等量收益更痛苦 | 止损过晚，止盈过早 |
| **锚定效应** | 依赖初始信息做决策 | 股价锚定历史高点 |
| **羊群效应** | 模仿他人行为 | 泡沫、暴跌 |
| **反应过度/不足** | 对信息处理偏差 | 动量/反转异象 |

### 4.3 处置效应

> **处置效应**：投资者倾向于过早卖出盈利股票而长期持有亏损股票。

```python
def disposition_effect(trades, prices):
    """
    处置效应检验：
    实现盈利次数 / 卖出总次数  vs  实现亏损次数 / 卖出总次数
    
    如果处置效应存在：
    PROFFIT = 卖出盈利次数 / 卖出总次数
    LOSS = 卖出亏损次数 / 卖出总次数
    PROFFIT > LOSS
    """
    realized_gains = 0
    realized_losses = 0
    
    for trade in trades:
        if trade['realized_pnl'] > 0:
            realized_gains += 1
        elif trade['realized_pnl'] < 0:
            realized_losses += 1
    
    profit_ratio = realized_gains / (realized_gains + realized_losses)
    loss_ratio = realized_losses / (realized_gains + realized_losses)
    
    return profit_ratio, loss_ratio, profit_ratio - loss_ratio
```

## 5. 对量化交易的启示

### 5.1 EMH vs 量化策略

| EMH立场 | 量化策略立场 |
|---------|-------------|
| 市场有效，无超额收益 | 市场存在短暂无效，可捕捉 |
| 技术分析无效 | 短期量价模式存在 |
| 基本面分析无效 | 错误定价被纠正需时间 |

### 5.2 量化策略的存在前提

```
量化策略的逻辑：
┌────────────────────────────────────────────┐
│ 假设：市场存在短期无效                       │
│     ↓                                      │
│ 发现：历史数据中的可预测模式                  │
│     ↓                                      │
│ 验证：统计检验显著 + 经济逻辑合理             │
│     ↓                                      │
│ 执行：程序化交易，快速捕捉                    │
│     ↓                                      │
│ 监控：策略衰减时及时止损                      │
└────────────────────────────────────────────┘
```

### 5.3 α的来源

| α来源 | 说明 | 检验方法 |
|--------|------|----------|
| **信息领先** | 提前获取/解读信息 | 事件研究 |
| **行为偏差** | 别人犯错我获利 | 异象研究 |
| **流动性** | 承担低流动性风险 | 流动性溢价模型 |
| **技术分析** | 量价模式存在 | 回测 |
