## 1. 货币时间价值 (Time Value of Money)

### 1.1 核心原理

> **定义**：货币的时间价值是指相同金额的货币在当前时点的价值高于未来时点的价值，原因是资金具有增值能力。

**基本公式**：

```
未来价值 (FV) = 现值 (PV) × (1 + r)^n
现值 (PV) = FV / (1 + r)^n
```

其中：
- `r` = 利率（折现率）
- `n` = 期数

### 1.2 复利与单利

| 类型 | 公式 | 特点 |
|------|------|------|
| **单利** | FV = PV × (1 + r×n) | 本金不变，利息不计入下一期 |
| **复利** | FV = PV × (1 + r)^n | 利息计入本金，滚存计息 |

**例子**：本金1000元，年利率5%，3年后：

| 方式 | 计算 | 结果 |
|------|------|------|
| 单利 | 1000 × (1 + 0.05×3) | 1150 元 |
| 复利 | 1000 × (1.05)^3 | 1157.63 元 |

### 1.3 年金 (Annuity)

**普通年金**（每期期末支付）：

```
PV = C × [1 - (1+r)^(-n)] / r
```

**先付年金**（每期期初支付）：

```
PV = C × [1 - (1+r)^(-n)] / r × (1+r)
```

## 2. 利率体系

### 2.1 名义利率 vs 实际利率

**费雪公式 (Fisher Equation)**：

```
(1 + r_nominal) = (1 + r_real) × (1 + π)
```

即：

```
r_real ≈ r_nominal - π
```

| 符号 | 含义 |
|------|------|
| r_nominal | 名义利率（银行挂牌利率） |
| r_real | 实际利率（购买力增长） |
| π | 通货膨胀率 |

**例子**：
- 名义利率 = 8%
- 通货膨胀率 = 3%
- 实际利率 ≈ 8% - 3% = **5%**

### 2.2 连续复利

当复利频率趋向无穷大时，得到**连续复利**：

```
FV = PV × e^(rt)
PV = FV × e^(-rt)
```

连续复利在金融工程中广泛应用（Black-Scholes模型等）。

### 2.3 利率期限结构

> **定义**：利率期限结构（Term Structure）描述的是无违约风险国债的到期收益率（Yield）与到期期限（Maturity）之间的关系。

**三条经典曲线形态**：

| 形态 | 含义 | 典型场景 |
|------|------|----------|
| **正常/上升** | 期限越长，收益率越高 | 经济扩张期 |
| **平坦** | 期限对收益率影响小 | 过渡期 |
| **倒挂** | 短期收益率高于长期 | 经济衰退预警 |

**重要理论**：

| 理论 | 核心观点 |
|------|----------|
| **预期理论** | 长期利率 = 短期利率的预期均值 |
| **流动性溢价理论** | 长期利率 = 短期预期 + 流动性溢价 |
| **市场分割理论** | 不同期限市场独立定价 |

## 3. 风险与收益

### 3.1 风险溢价的定义

> **风险溢价 (Risk Premium)**：投资者因承担额外风险而要求的额外收益。

```
期望收益率 = 无风险收益率 + 风险溢价
E(R) = R_f + RP
```

### 3.2 系统性风险 vs 非系统性风险

| 类型 | 特征 | 分散化效果 |
|------|------|-----------|
| **系统性风险** (β) | 影响整个市场，无法消除 | ❌ 不可分散 |
| **非系统性风险** (α) | 只影响特定公司/行业 | ✅ 可通过组合消除 |

**资本资产定价模型 (CAPM)** 中的风险溢价：

```
E(R_i) - R_f = β_i × [E(R_m) - R_f]

其中：
- E(R_i) - R_f = 资产i的风险溢价
- β_i = 资产i的贝塔系数（相对于市场组合的敏感度）
- E(R_m) - R_f = 市场风险溢价（Market Risk Premium）
```

### 3.3 风险厌恶与风险溢价

假设投资者**风险厌恶**（大多数情况），则：

```
风险溢价 > 0
```

风险厌恶程度越高，同样的风险要求更高的溢价。

## 4. 金融市场基本分类

### 4.1 按交易期限分类

| 市场 | 期限 | 主要功能 |
|------|------|----------|
| **货币市场** | < 1年 | 短期资金融通（票据、国库券） |
| **资本市场** | ≥ 1年 | 长期投资（股票、债券） |

### 4.2 按交易对象分类

```
金融市场
├── 权益市场（股票）
├── 债务市场（债券）
├── 衍生品市场（期权、期货）
├── 外汇市场
└── 商品市场
```

### 4.3 一级市场 vs 二级市场

| 类型 | 定义 | 特点 |
|------|------|------|
| **一级市场** | 发行市场，IPO，增发 | 资金流向发行人 |
| **二级市场** | 流通市场，交易现有证券 | 提供流动性 |

## 5. 金融产品定价基础

### 5.1 债券定价原理

**贴现现金流 (DCF)**：

```
P = Σ [CF_t / (1+r)^t] + F / (1+r)^n
```

其中：
- CF_t = 第t期现金流（利息）
- F = 面值
- r = 折现率（市场利率）
- n = 到期期限

**例子**：面值为1000元，票面利率6%，每年付息，市场利率8%，剩余3年到期。

```
P = 60/(1.08) + 60/(1.08²) + 60/(1.08³) + 1000/(1.08³)
  = 55.56 + 51.44 + 47.63 + 793.83
  = 948.46 元
```

债券价格低于面值（折价），因为市场利率 > 票面利率。

### 5.2 股票定价基础

**股利贴现模型 (DDM)**：

```
P = Σ [D_t / (1+r)^t] + P_T / (1+r)^T
```

**零增长模型**：

```
P = D / r
```

其中 D 为固定股利。
