# F1.03 金融衍生品概述

> 本章介绍期权、期货、互换三大类金融衍生品，建立衍生品定价的直观框架。

## 学习目标

- [ ] 理解衍生品的定义与核心特征
- [ ] 区分期货与期权的区别
- [ ] 掌握期权的基本策略（买权、卖权）
- [ ] 了解互换的基本应用场景

---

## 1. 衍生品的定义

> **金融衍生品** (Derivative) 是一种其价值依赖于原生资产（标的资产）价格的金融合约。

**三大核心特征**：

| 特征 | 说明 |
|------|------|
| **价值派生** | 价值来源于标的资产的价格变动 |
| **杠杆效应** | 以小博大，支付少量保证金控制大额资产 |
| **双向功能** | 既可套期保值，也可投机 |

**标的资产类型**：

```
标的资产
├── 权益类（股票指数、个股）
├── 利率类（国债收益率、LIBOR）
├── 汇率类（美元/人民币等）
├── 商品类（黄金、石油、农产品）
└── 信用类（CDS指数）
```

## 2. 远期合约 (Forward Contract)

### 2.1 定义

> **远期合约**是买卖双方约定在未来某一确定时间、以确定价格买卖某种资产的合约。

**特点**：

| 属性 | 远期合约 |
|------|----------|
| **交易场所** | OTC（场外，非标准化） |
| **定价** | 双方协商 |
| **履约** | 到期必须交割 |
| **信用风险** | 双方违约风险 |

### 2.2 远期价格与价值

**远期价格** (Forward Price)：

```
F = S × e^(rT)

其中：
- S = 标的资产现货价格
- r = 无风险利率
- T = 到期期限
```

**远期价值** (Forward Value)：

到期时，多头价值 = max(S_T - F, 0)，空头价值 = max(F - S_T, 0)

## 3. 期货合约 (Futures Contract)

### 3.1 定义

> **期货合约**是在交易所交易的标准化远期合约，规定了标的资产、数量、交割日期等条款。

**与远期合约的关键区别**：

| 属性 | 远期 | 期货 |
|------|------|------|
| **交易场所** | OTC | 交易所 |
| **标准化** | 双方协商 | 标准化 |
| **保证金** | 无 | 有（每日结算） |
| **交割方式** | 可实物/现金 | 多数对冲平仓 |
| **信用风险** | 双方承担 | 清算所做担保 |

### 3.2 期货的每日结算机制

```python
class FuturesContract:
    def __init__(self, initial_price, margin_rate=0.05):
        self.initial_price = initial_price
        self.margin = initial_price * margin_rate  # 保证金
        self.maintenance_margin = self.margin * 0.75  # 维持保证金
        self.daily_pnl = 0
        self.positions = {}  # 每日盯市盈亏记录
    
    def daily_settlement(self, close_price, prev_close):
        """
        每日结算：计算盈亏，触发追保
        """
        self.daily_pnl = close_price - prev_close
        
        if self.balance < self.maintenance_margin:
            return "MARGIN CALL - 需要追加保证金"
        return "Normal"
```

### 3.3 期货的应用场景

| 用途 | 说明 | 案例 |
|------|------|------|
| **套期保值** | 对冲现货价格风险 | 航空公司买入航油期货锁定成本 |
| **投机** | 承担风险获取收益 | 看多原油，买入期货 |
| **套利** | 捕捉定价偏差 | 跨期套利、跨市场套利 |

## 4. 期权合约 (Options)

### 4.1 核心定义

> **期权**是赋予买方在特定时间（或之前）以特定价格买入或卖出标的资产权利的合约，而非义务。

**期权 vs 期货/远期**：

| 特征 | 期货/远期 | 期权 |
|------|----------|------|
| **买卖双方** | 双方都有义务 | 买方有权利，卖方有义务 |
| **最大损失** | 双方均无限 | 买方有限（期权费），卖方无限 |
| **成本** | 无前期费用 | 支付期权费 |
| **收益结构** | 线性对称 | 非线性不对称 |

### 4.2 期权的分类

```
期权分类
├── 按权利内容
│   ├── 看涨期权 (Call Option) — 买入权
│   └── 看跌期权 (Put Option) — 卖出权
├── 按行权时间
│   ├── 欧式期权 (European) — 到期日才能行权
│   └── 美式期权 (American) — 到期前任意时间可行权
└── 按标的资产
    ├── 股票期权
    ├── 指数期权
    ├── 期货期权
    └── 外汇期权
```

### 4.3 期权的到期收益

**看涨期权 (Call)**：

```python
def call_payoff(S_T, K):
    """
    看涨期权到期收益
    S_T: 到期标的资产价格
    K: 行权价
    """
    return max(S_T - K, 0)

def call_profit(S_T, K, premium):
    """
    看涨期权 profit = payoff - 期权费
    """
    return call_payoff(S_T, K) - premium
```

**看跌期权 (Put)**：

```python
def put_payoff(S_T, K):
    """
    看跌期权到期收益
    """
    return max(K - S_T, 0)

def put_profit(S_T, K, premium):
    """
    看跌期权 profit = payoff - 期权费
    """
    return put_payoff(S_T, K) - premium
```

**收益结构图示**：

```
看涨期权收益              看跌期权收益
  payoff                   payoff
    │↗                    ↗│
    │╱                     ╱│
────┼──── K          K────┼────
   ╱│                   ╱  │
  ╱ │                  ╱   │
    ↓                   ↓
   S_T                 S_T
```

### 4.4 期权的内在价值与时间价值

```
期权价格 = 内在价值 + 时间价值

┌─────────────────────────────────────┐
│ 期权价格                             │
│  ╭────────────────────────────────╮ │
│  │ 时间价值（波动率、剩余期限）     │ │
│  │  ╭────────────────────────────╮│ │
│  │  │ 内在价值（实值程度）        ││ │
│  │  │  ╭────────────────────────╮││ │
│  │  │  │                       │││ │
│  │  │  ╰────────────────────────╯││ │
│  │  ╰────────────────────────────╯│ │
│  ╰────────────────────────────────╯ │
└─────────────────────────────────────┘
```

**内在价值**：

| 类型 | 条件 | 内在价值 |
|------|------|----------|
| **实值 (ITM)** | S > K (Call) | S - K |
| **平价 (ATM)** | S ≈ K | 0 |
| **虚值 (OTM)** | S < K (Call) | 0 |

### 4.5 期权的基本策略

**买入看涨期权 (Buy Call)**：

```python
# 场景：投资者预期股价上涨，但不希望承担下跌风险
# 支付期权费，上涨收益无限

# 盈亏平衡点 = 行权价 + 期权费
break_even = K + premium
```

**保护性看跌 (Protective Put)**：

```python
# 场景：持有股票+买入看跌期权
# 相当于为股票组合购买保险
# 下跌有底，上涨空间保留

# 盈亏平衡点 = 股票买入价 - 期权费
break_even = stock_price - premium
```

## 5. 互换合约 (Swap)

### 5.1 定义

> **互换**是两个交易对手约定在一定期限内交换一系列现金流的合约。

### 5.2 利率互换 (Interest Rate Swap)

最常见的互换类型：

```
┌─────────────────────────────────────────────────────┐
│ 利率互换结构                                         │
├─────────────────────────────────────────────────────┤
│                                                     │
│  甲方向乙方支付：                                     │
│    「固定利率 × 名义本金」                            │
│                      ←──────→                       │
│              金融机构（中介）                          │
│                      ←──────→                       │
│  乙方向甲方支付：                                     │
│    「浮动利率(LIBOR+spread) × 名义本金」              │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**用途**：

| 角色 | 目的 |
|------|------|
| 固定利率支付者 | 将浮动利率债务转为固定利率 |
| 浮动利率支付者 | 将固定利率资产转为浮动利率 |

## 6. 衍生品的定价思想

### 6.1 无套利定价原理

> **无套利定价** (No-Arbitrage Pricing)：在有效市场中，不存在无风险利润。

**核心思想**：

```
复制组合：
期权价值 = 标的资产组合 + 债券的某个组合
          = Δ × 标的资产 + B × 债券

通过标的资产和债券复制期权的收益特征，
反解出期权当前的理论价格
```

### 6.2 Black-Scholes 定价公式（简介）

欧式看涨期权定价：

```
C = S × N(d₁) - K × e^(-rT) × N(d₂)

其中：
d₁ = [ln(S/K) + (r + σ²/2)T] / (σ√T)
d₂ = d₁ - σ√T
N(·) = 标准正态分布累积函数
σ = 标的资产波动率
```

**假设条件**：

| 假设 | 说明 |
|------|------|
| 股价服从几何布朗运动 | 收益率服从对数正态分布 |
| 无交易成本和税收 | 市场完美 |
| 无风险利率恒定 | r 为常数 |
| 无 dividend | 股票不分红 |
| 欧式期权 | 只能到期行权 |
