# 01-量化策略入门

## 概述

量化策略是运用数学、统计学和计算机科学方法设计和实施的金融交易策略。本章将系统介绍量化策略的基本概念、发展历程和主要类型，为后续学习具体策略实现打下基础。

## 1. 量化策略的定义

### 1.1 量化策略的定义

**量化策略**（Quantitative Strategy）是运用数学、统计学、计算机科学等量化方法进行金融分析、投资决策和风险管理的策略。

### 1.2 量化策略的特点

1. **数学化**：运用数学模型描述金融现象
2. **数据驱动**：基于大量数据进行分析
3. **计算机化**：利用计算机进行计算和模拟
4. **模型化**：建立数学模型进行预测
5. **实证化**：通过实证检验策略有效性

### 1.3 量化策略的发展历程

#### 早期发展（1950s-1970s）

**马科维茨均值-方差模型**（1952年）
- 建立了现代投资组合理论
- 引入了风险度量概念
- 为量化投资奠定了理论基础

**资本资产定价模型**（1964年）
- 解释了风险与收益的关系
- 为资产定价奠定了基础
- 促进了量化模型的发展

#### 现代发展（1980s-2000s）

**Black-Scholes期权定价模型**（1973年）
- 解决了期权定价问题
- 开创了衍生品定价理论
- 促进了量化策略的发展

**量化投资的兴起**
- 量化基金出现
- 算法交易发展
- 风险管理工具完善

#### 当代发展（2010s至今）

**机器学习在量化策略中的应用**

### 2.1 按投资风格分类

#### 价值策略

**定义**：基于基本面分析，寻找被低估的资产进行投资。

**核心思想**：
- 价值投资理论
- 低估值高增长
- 长期持有

**主要指标**：
- 市盈率（P/E）
- 市净率（P/B）
- 市销率（P/S）
- 债务权益比

**Python实现**：
```python
import numpy as np
import pandas as pd

class ValueStrategy:
    """价值策略类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_pe_ratio(self, price, eps):
        """计算市盈率"""
        return price / eps
    
    def calculate_pb_ratio(self, price, book_value_per_share):
        """计算市净率"""
        return price / book_value_per_share
    
    def calculate_ps_ratio(self, price, revenue_per_share):
        """计算市销率"""
        return price / revenue_per_share
    
    def calculate_debt_to_equity(self, total_debt, total_equity):
        """计算债务权益比"""
        return total_debt / total_equity
    
    def screen_value_stocks(self, stock_data, pe_threshold=15, pb_threshold=2.0):
        """筛选价值股票"""
        # 计算指标
        stock_data['PE_Ratio'] = stock_data['Price'] / stock_data['EPS']
        stock_data['PB_Ratio'] = stock_data['Price'] / stock_data['Book_Value_Per_Share']
        
        # 筛选条件
        value_stocks = stock_data[
            (stock_data['PE_Ratio'] <= pe_threshold) &
            (stock_data['PB_Ratio'] <= pb_threshold)
        ]
        
        return value_stocks
    
    def calculate_value_score(self, stock_data):
        """计算价值评分"""
        # 标准化指标
        stock_data['PE_Score'] = 1 / stock_data['PE_Ratio']
        stock_data['PB_Score'] = 1 / stock_data['PB_Ratio']
        
        # 计算综合评分
        stock_data['Value_Score'] = (
            stock_data['PE_Score'] * 0.4 +
            stock_data['PB_Score'] * 0.6
        )
        
        return stock_data.sort_values('Value_Score', ascending=False)

# 使用示例
if __name__ == "__main__":
    # 创建价值策略对象
    vs = ValueStrategy()
    
    # 生成模拟数据
    np.random.seed(42)
    n_stocks = 100
    stock_data = pd.DataFrame({
        'Symbol': [f'STOCK_{i}' for i in range(n_stocks)],
        'Price': np.random.uniform(10, 100, n_stocks),
        'EPS': np.random.uniform(0.5, 5, n_stocks),
        'Book_Value_Per_Share': np.random.uniform(5, 50, n_stocks),
        'Revenue_Per_Share': np.random.uniform(1, 20, n_stocks),
        'Total_Debt': np.random.uniform(100, 1000, n_stocks),
        'Total_Equity': np.random.uniform(100, 1000, n_stocks)
    })
    
    # 计算指标
    stock_data['PE_Ratio'] = vs.calculate_pe_ratio(stock_data['Price'], stock_data['EPS'])
    stock_data['PB_Ratio'] = vs.calculate_pb_ratio(stock_data['Price'], stock_data['Book_Value_Per_Share'])
    
    # 筛选价值股票
    value_stocks = vs.screen_value_stocks(stock_data, pe_threshold=20, pb_threshold=3.0)
    print(f"筛选出的价值股票数量: {len(value_stocks)}")
    
    # 计算价值评分
    scored_stocks = vs.calculate_value_score(stock_data)
    print("前5个价值评分最高的股票:")
```

```python
class GrowthStrategy:
    """成长策略类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_revenue_growth_rate(self, current_revenue, previous_revenue):
        """计算收入增长率"""
        return (current_revenue - previous_revenue) / previous_revenue
    
    def calculate_profit_growth_rate(self, current_profit, previous_profit):
        """计算利润增长率"""
        return (current_profit - previous_profit) / previous_profit
    
    def calculate_earnings_per_share_growth(self, current_eps, previous_eps):
        """计算每股收益增长率"""
        return (current_eps - previous_eps) / previous_eps
    
    def screen_growth_stocks(self, stock_data, min_growth_rate=0.1):
        """筛选成长股票"""
        # 计算增长率
        stock_data['Revenue_Growth'] = stock_data.groupby('Symbol')['Revenue'].pct_change()
        stock_data['Profit_Growth'] = stock_data.groupby('Symbol')['Net_Profit'].pct_change()
        
        # 筛选条件
        growth_stocks = stock_data[
            (stock_data['Revenue_Growth'] > min_growth_rate) &
            (stock_data['Profit_Growth'] > min_growth_rate)
        ]
        
        return growth_stocks
    
    def calculate_growth_score(self, stock_data):
        """计算成长评分"""
        # 计算增长率
        stock_data['Revenue_Growth'] = stock_data.groupby('Symbol')['Revenue'].pct_change()
        stock_data['Profit_Growth'] = stock_data.groupby('Symbol')['Net_Profit'].pct_change()
        
        # 标准化增长率
        stock_data['Revenue_Growth_Score'] = stock_data['Revenue_Growth'].rank(pct=True)
        stock_data['Profit_Growth_Score'] = stock_data['Profit_Growth'].rank(pct=True)
        
        # 计算综合评分
        stock_data['Growth_Score'] = (
            stock_data['Revenue_Growth_Score'] * 0.6 +
            stock_data['Profit_Growth_Score'] * 0.4
        )
        
        return stock_data.sort_values('Growth_Score', ascending=False)

# 使用示例
if __name__ == "__main__":
    # 创建成长策略对象
    gs = GrowthStrategy()
    
    # 生成模拟数据
    np.random.seed(42)
    n_stocks = 100
    stock_data = pd.DataFrame({
        'Symbol': [f'STOCK_{i}' for i in range(n_stocks)],
        'Revenue': np.random.uniform(100, 1000, n_stocks),
        'Net_Profit': np.random.uniform(10, 100, n_stocks),

        stock_data['Net_Profit'], stock_data['Previous_Profit']

    # 计算成长评分
```

#### 动量策略

**定义**：基于价格趋势，追涨杀跌的投资策略。

**核心思想**：

**Python实现**：
```python
class MomentumStrategy:
    """动量策略类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_price_momentum(self, prices, period=20):
        """计算价格动量"""
        return (prices - prices.shift(period)) / prices.shift(period)
    
    def calculate_volume_momentum(self, volumes, period=20):
        """计算成交量动量"""
        return (volumes - volumes.shift(period)) / volumes.shift(period)
    
    def calculate_rsi(self, prices, period=14):
        """计算RSI指标"""
        delta = prices.diff()
        gain = (delta.where(delta > 0, 0)).rolling(window=period).mean()
        loss = (-delta.where(delta < 0, 0)).rolling(window=period).mean()
        rs = gain / loss
        rsi = 100 - (100 / (1 + rs))
        return rsi
    
    def generate_momentum_signals(self, prices, volumes, period=20):
        """生成动量信号"""
        # 计算指标
        price_momentum = self.calculate_price_momentum(prices, period)
        volume_momentum = self.calculate_volume_momentum(volumes, period)
        rsi = self.calculate_rsi(prices, period)
        
        # 生成信号
        signals = pd.Series(0, index=prices.index)
        
        # 买入信号：价格动量>0且RSI<70
        signals[(price_momentum > 0) & (rsi < 70)] = 1
        
        # 卖出信号：价格动量<0且RSI>30
        signals[(price_momentum < 0) & (rsi > 30)] = -1
        
        return signals, price_momentum, volume_momentum, rsi
    
    def backtest_momentum_strategy(self, prices, volumes, initial_capital=100000):
        """回测动量策略"""
        # 生成信号
        signals, price_momentum, volume_momentum, rsi = self.generate_momentum_signals(prices, volumes)
        
        # 初始化
        portfolio_value = [initial_capital]
        position = 0
        cash = initial_capital
        
        # 回测
        for i in range(1, len(signals)):
            signal = signals.iloc[i]
            price = prices.iloc[i]
            
            if signal == 1 and position == 0:
                # 买入
                shares = cash // price
                position = shares
                cash -= shares * price
                portfolio_value.append(cash + position * price)
            elif signal == -1 and position > 0:
                # 卖出
                cash += position * price
                position = 0
                portfolio_value.append(cash)
            else:
                # 持有
                portfolio_value.append(cash + position * price)
        
        return portfolio_value

# 使用示例
if __name__ == "__main__":
    # 创建动量策略对象
    ms = MomentumStrategy()
    
    # 生成模拟价格数据
    np.random.seed(42)
    n_days = 100
    prices = np.cumsum(np.random.normal(0, 0.02, n_days)) + 100
    volumes = np.random.uniform(1000, 10000, n_days)
    
    # 生成信号
    signals, price_momentum, volume_momentum, rsi = ms.generate_momentum_signals(
        pd.Series(prices), pd.Series(volumes)
    )
    
    print("=== 动量策略示例 ===")
    print(f"交易信号数量: {np.sum(signals != 0)}")
    print(f"买入信号数量: {np.sum(signals == 1)}")
    print(f"卖出信号数量: {np.sum(signals == -1)}")
    
    # 回测
    portfolio_value = ms.backtest_momentum_strategy(
        pd.Series(prices), pd.Series(volumes)
    )
    
    print(f"初始资金: 100000")
    print(f"最终资金: {portfolio_value[-1]:.2f}")
    print(f"收益率: {(portfolio_value[-1] - 100000) / 100000:.2%}")
```

```python
def equal_weight_portfolio(strategies, weights=None):
    """等权重投资组合"""
    if weights is None:
        n_strategies = len(strategies)
        weights = np.ones(n_strategies) / n_strategies
    
    return np.average(strategies, axis=0, weights=weights)
```

```python
def risk_parity_portfolio(returns, target_vol=0.15):
    """风险平价投资组合"""
    # 计算协方差矩阵
    cov_matrix = np.cov(returns.T)
    
    # 计算风险贡献
    portfolio_vol = np.sqrt(np.sum(cov_matrix))
    risk_contribution = np.diag(cov_matrix) / portfolio_vol
    
    # 计算权重
    weights = target_vol * risk_contribution / cov_matrix
    
    return weights / np.sum(weights)
```

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.metrics import mean_squared_error

class QuantitativeStrategy:
    """量化策略类"""
    
    def __init__(self):
        """初始化"""
        pass
    
    def calculate_strategy_return(self, signals, returns):
        """计算策略收益"""
        strategy_returns = signals.shift(1) * returns
        return strategy_returns
    
    def calculate_cumulative_return(self, returns):
        """计算累计收益"""
        return (1 + returns).cumprod()
    
    def calculate_sharpe_ratio(self, returns, risk_free_rate=0.02):
        """计算夏普比率"""
        excess_returns = returns - risk_free_rate / 252
        return np.mean(excess_returns) / np.std(excess_returns) * np.sqrt(252)
    
    def calculate_max_drawdown(self, returns):
        """计算最大回撤"""
        cumulative = np.cumprod(1 + returns)
        running_max = np.maximum.accumulate(cumulative)
        drawdown = (cumulative - running_max) / running_max
        return np.min(drawdown)
    
    def calculate_var(self, returns, confidence_level=0.05):
        """计算VaR"""
        return np.percentile(returns, confidence_level * 100)
    
    def calculate_cvar(self, returns, confidence_level=0.05):
        """计算CVaR"""
        var = self.calculate_var(returns, confidence_level)
        return np.mean(returns[returns <= var])

class StrategyEvaluator:
    """策略评估器"""
    
    def __init__(self):
        """初始化"""
        self.strategy = QuantitativeStrategy()
    
    def backtest_strategy(self, prices, signals):
        """回测策略"""
        # 计算收益率

class StrategyOptimizer:
    """策略优化器"""
    
    def __init__(self):
        """初始化"""
        self.evaluator = StrategyEvaluator()
    
    def grid_search_optimization(self, prices, strategy_func, param_grid):
        """网格搜索优化"""
        best_params = None
        best_metrics = None
        best_sharpe = -np.inf
        
        # 生成参数组合
        param_combinations = []
        for param_values in param_grid.values():
            if not param_combinations:
                param_combinations = [[value] for value in param_values]
            else:
                new_combinations = []
                for combo in param_combinations:
                    for value in param_values:
                        new_combinations.append(combo + [value])
                param_combinations = new_combinations
        
        # 参数名称
        param_names = list(param_grid.keys())
        
        # 遍历参数组合
        for combo in param_combinations:
            params = dict(zip(param_names, combo))
            
            # 生成信号
            signals = strategy_func(prices, **params)
            
            # 回测
            returns = prices.pct_change()
            strategy_returns = self.evaluator.strategy.calculate_strategy_return(signals, returns)
            
            # 计算指标
            sharpe_ratio = self.evaluator.strategy.calculate_sharpe_ratio(strategy_returns)
            
            if sharpe_ratio > best_sharpe:
                best_sharpe = sharpe_ratio
                best_params = params
                best_metrics = {
                    'sharpe_ratio': sharpe_ratio,
                    'cumulative_return': self.evaluator.strategy.calculate_cumulative_return(strategy_returns).iloc[-1] - 1,
                    'max_drawdown': self.evaluator.strategy.calculate_max_drawdown(strategy_returns)
                }
        
        return best_params, best_metrics
    
    def random_search_optimization(self, prices, strategy_func, param_grid, n_iter=100):
        """随机搜索优化"""
        best_params = None
        best_metrics = None
        best_sharpe = -np.inf
        
        param_names = list(param_grid.keys())
        
        for _ in range(n_iter):
            # 随机选择参数
            params = {}
            for param_name, param_values in param_grid.items():
                params[param_name] = np.random.choice(param_values)
            
            # 生成信号
            signals = strategy_func(prices, **params)
            
            # 回测
            returns = prices.pct_change()
            strategy_returns = self.evaluator.strategy.calculate_strategy_return(signals, returns)
            
            # 计算指标
            sharpe_ratio = self.evaluator.strategy.calculate_sharpe_ratio(strategy_returns)
            
            if sharpe_ratio > best_sharpe:
                best_sharpe = sharpe_ratio
                best_params = params
                best_metrics = {
                    'sharpe_ratio': sharpe_ratio,
                    'cumulative_return': self.evaluator.strategy.calculate_cumulative_return(strategy_returns).iloc[-1] - 1,
                    'max_drawdown': self.evaluator.strategy.calculate_max_drawdown(strategy_returns)
                }
        
        return best_params, best_metrics
```

```python
def multi_strategy_portfolio_example():
    """多策略组合示例"""
    # 创建策略评估器
    evaluator = StrategyEvaluator()
    
    # 生成模拟价格数据
    np.random.seed(42)
    n_days = 100
    prices = np.cumsum(np.random.normal(0, 0.02, n_days)) + 100
    prices = pd.Series(prices, index=pd.date_range('2023-01-01', periods=n_days))
    
    # 定义多个策略
    def momentum_strategy(prices, period=20):
        """动量策略"""
        momentum = prices.pct_change(period)
        signals = pd.Series(0, index=prices.index)
        signals[momentum > 0.01] = 1
        signals[momentum < -0.01] = -1
        return signals
    
    def mean_reversion_strategy(prices, period=20):
        """均值回归策略"""
        ma = prices.rolling(window=period).mean()
        deviation = (prices - ma) / ma.std()
        signals = pd.Series(0, index=prices.index)
        signals[deviation > 1.5] = -1
        signals[deviation < -1.5] = 1
        return signals
    
    def value_strategy(prices, pe_threshold=15):
        """价值策略"""
        # 模拟PE数据
        pe = np.random.uniform(5, 25, len(prices))
        signals = pd.Series(0, index=prices.index)
        signals[pe < pe_threshold] = 1
        signals[pe > pe_threshold * 1.5] = -1
        return signals
    
    # 生成信号
    momentum_signals = momentum_strategy(prices)
    mean_reversion_signals = mean_reversion_strategy(prices)
    value_signals = value_strategy(prices)
    
    # 组合策略
    strategies_dict = {
        '动量策略': momentum_signals,
        '均值回归策略': mean_reversion_signals,
        '价值策略': value_signals
    }
    
    # 比较策略
    results = evaluator.compare_strategies(prices, strategies_dict)
    
    # 等权重组合

    equal_weight_returns = equal_weight_returns * prices.pct_change()
    
    print("\n=== 等权重组合结果 ===")
```
