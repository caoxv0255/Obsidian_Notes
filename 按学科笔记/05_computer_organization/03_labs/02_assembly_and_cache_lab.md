# 计算机组成原理实验：汇编分析与 Cache 模拟

## 实验 A：汇编分析

编译一段简单的循环或数组处理代码，观察生成的汇编，并解释寄存器分配与循环展开的影响。

### 观察重点

- 哪些高级语句被拆成了多条指令。
- 变量如何在寄存器和栈之间移动。
- 编译优化是否改变了指令数。

## 实验 B：Cache 模拟

使用模拟器或自写小程序比较不同访问模式下的 Cache 命中率，重点观察顺序访问、步长访问和随机访问的差异。

### 观察重点

- 局部性对命中率的影响。
- 缓存行大小变化后性能如何变化。
- 替换策略是否影响结果。
