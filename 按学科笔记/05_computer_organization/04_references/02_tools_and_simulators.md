# 计算机组成原理工具与模拟器

## 推荐工具类型

- 指令集模拟器：用于观察寄存器、内存和分支行为。
- 流水线模拟器：用于观察冒险、停顿和转发。
- Cache 可视化工具：用于观察命中、缺失和替换过程。
- 汇编查看工具：用于分析编译后的指令序列。

## 工具使用目标

- 不只是跑通程序，而是看懂硬件为什么这样工作。
- 不只是记录结果，而是能把结果解释成体系结构现象。

## 建议练习

- 用同一段代码分别在不同优化等级下生成汇编。
- 对比顺序访问和随机访问的 Cache 行为。
- 用流水线模拟器复现一次数据冒险和一次控制冒险。
