# 第4章：效用

> 本章学习目标：
> - 理解效用的定义
> - 掌握效用函数的性质
> - 学会分析效用函数
> - 理解效用函数的类型

---

## 4.1 效用的定义

### 4.1.1 效用的定义

**效用（Utility）**：
消费者从消费商品组合中获得的满足程度。

### 4.1.2 效用的性质

**效用的性质（Properties of Utility）**：

**1. 序数性（Ordinality）**：
- 效用是序数的，不是基数的
- 效用表示排序，不表示满足程度的大小

**2. 递增性（Increasing）**：
- 效用函数是递增的
- 效用随着消费的增加而增加

**3. 单调性（Monotonicity）**：
- 效用函数是单调的
- 多多益善

---

## 4.2 效用函数

### 4.2.1 效用函数的定义

**效用函数（Utility Function）**：
给每个商品组合分配一个效用值的函数。

### 4.2.2 效用函数的公式

**效用函数的公式（Formula of Utility Function）**：
```
u(x₁, x₂)
```

**其中**：
- u：效用
- x₁：商品1的数量
- x₂：商品2的数量

### 4.2.3 效用函数的性质

**效用函数的性质（Properties of Utility Function）**：

**1. 单调变换（Monotonic Transformation）**：
- 效用函数可以单调变换
- 单调变换不改变偏好的排序

**2. 单调变换的类型（Types of Monotonic Transformation）**：
- 正单调变换：f'(u) > 0
- 负单调变换：f'(u) < 0

### 4.2.4 效用函数的例子

**效用函数的例子（Examples of Utility Function）**：

**1. 柯布-道格拉斯效用函数（Cobb-Douglas Utility Function）**：
```
u(x₁, x₂) = x₁^a × x₂^b
```

**特征**：
- 边际替代率递减
- 无差异曲线凸向原点

**2. 完全替代品效用函数（Perfect Substitutes Utility Function）**：
```
u(x₁, x₂) = ax₁ + bx₂
```

**特征**：
- 边际替代率是常数
- 无差异曲线是直线

**3. 完全互补品效用函数（Perfect Complements Utility Function）**：
```
u(x₁, x₂) = min(ax₁, bx₂)
```

**特征**：
- 边际替代率是0或无穷大
- 无差异曲线是L形

**4. 拟线性效用函数（Quasilinear Utility Function）**：
```
u(x₁, x₂) = v(x₁) + x₂
```

**特征**：
- 无差异曲线是平行的
- 收入的变化只影响x₂

## 4.3 边际效用

### 4.3.1 边际效用的定义

**边际效用（Marginal Utility, MU）**：
增加一单位商品消费带来的效用变化。

### 4.3.2 边际效用的公式

**边际效用的公式（Formula of Marginal Utility）**：
```
MU₁ = ∂u/∂x₁
```
```
MU₂ = ∂u/∂x₂
```

**其中**：
- MU₁：商品1的边际效用
- MU₂：商品2的边际效用
- ∂u/∂x₁：效用对x₁的偏导数
- ∂u/∂x₂：效用对x₂的偏导数

### 4.3.3 边际效用递减

**边际效用递减（Diminishing Marginal Utility）**：
随着消费的增加，边际效用递减。

**公式**：
```
∂²u/∂x₁² < 0
```
```
∂²u/∂x₂² < 0
```

**含义**：
- 第1单位商品的边际效用最大
- 第2单位商品的边际效用次之
- ...

---

## 4.4 边际替代率和边际效用

### 4.4.1 边际替代率和边际效用的关系

**边际替代率和边际效用的关系（Relationship between MRS and MU）**：
```
MRS = MU₁/MU₂
```

**证明**：
- 无差异曲线：u(x₁, x₂) = 常数
- 全微分：∂u/∂x₁ × dx₁ + ∂u/∂x₂ × dx₂ = 0
- MU₁ × dx₁ + MU₂ × dx₂ = 0
- dx₂/dx₁ = -MU₁/MU₂
- MRS = -dx₂/dx₁ = MU₁/MU₂

### 4.4.2 边际替代率递减和边际效用递减

**边际替代率递减和边际效用递减（Diminishing MRS and Diminishing MU）**：

**边际替代率递减**：
- 随着x₁增加，MRS递减
- 反映无差异曲线凸向原点

**边际效用递减**：
- 随着x₁增加，MU₁递减
- 随着x₂增加，MU₂递减

**关系**：
- 如果MU₁和MU₂都递减，MRS可能递减，也可能递增
- MRS递减需要更严格的条件

## 4.5 效用函数的类型

### 4.5.1 柯布-道格拉斯效用函数

**柯布-道格拉斯效用函数（Cobb-Douglas Utility Function）**：
```
u(x₁, x₂) = x₁^a × x₂^b
```

**特征**：
- 边际替代率递减
- 无差异曲线凸向原点
- 边际效用递减

**边际效用**：
```
MU₁ = a × x₁^(a-1) × x₂^b
```
```
MU₂ = b × x₁^a × x₂^(b-1)
```

**边际替代率**：
```
MRS = MU₁/MU₂ = (a/b) × (x₂/x₁)
```

### 4.5.2 完全替代品效用函数

**完全替代品效用函数（Perfect Substitutes Utility Function）**：
```
u(x₁, x₂) = ax₁ + bx₂
```

**特征**：
- 边际替代率是常数
- 无差异曲线是直线
- 边际效用是常数

**边际效用**：
```
MU₁ = a
```
```
MU₂ = b
```

**边际替代率**：
```
MRS = MU₁/MU₂ = a/b
```

### 4.5.3 完全互补品效用函数

**完全互补品效用函数（Perfect Complements Utility Function）**：
```
u(x₁, x₂) = min(ax₁, bx₂)
```

**特征**：
- 边际替代率是0或无穷大
- 无差异曲线是L形
- 边际效用是0或无穷大

**边际效用**：
```
MU₁ = a（如果ax₁ < bx₂）
```
```
MU₁ = 0（如果ax₁ > bx₂）
```
```
MU₂ = b（如果bx₂ < ax₁）
```
```
MU₂ = 0（如果bx₂ > ax₁）
```

**边际替代率**：
```
MRS = 0（如果ax₁ < bx₂）
```
```
MRS = ∞（如果ax₁ > bx₂）
```

### 4.5.4 拟线性效用函数

**拟线性效用函数（Quasilinear Utility Function）**：
```
u(x₁, x₂) = v(x₁) + x₂
```

**特征**：
- 无差异曲线是平行的
- 收入的变化只影响x₂
- 边际效用递减

**边际效用**：
```
MU₁ = v'(x₁)
```
```
MU₂ = 1
```

**边际替代率**：
```
MRS = MU₁/MU₂ = v'(x₁)
```

## 4.6 本章总结

### 核心要点

1. **效用的定义**：
   - 效用：消费者从消费商品组合中获得的满足程度
   - 效用的性质：序数性、递增性、单调性

2. **效用函数**：
   - 效用函数：给每个商品组合分配一个效用值的函数
   - 效用函数的性质：单调变换不改变偏好的排序

3. **边际效用**：
   - 边际效用：增加一单位商品消费带来的效用变化
   - 边际效用的公式：MU₁ = ∂u/∂x₁，MU₂ = ∂u/∂x₂
   - 边际效用递减：随着消费的增加，边际效用递减

4. **边际替代率和边际效用**：
   - 边际替代率和边际效用的关系：MRS = MU₁/MU₂
   - 边际替代率递减和边际效用递减：MRS递减需要更严格的条件

5. **效用函数的类型**：
   - 柯布-道格拉斯效用函数：u(x₁, x₂) = x₁^a × x₂^b，边际替代率递减
   - 完全替代品效用函数：u(x₁, x₂) = ax₁ + bx₂，边际替代率是常数
   - 完全互补品效用函数：u(x₁, x₂) = min(ax₁, bx₂)，边际替代率是0或无穷大
   - 拟线性效用函数：u(x₁, x₂) = v(x₁) + x₂，无差异曲线是平行的
