# 第2章：预算约束

> 本章学习目标：
> - 理解预算约束的定义

---

## 2.1 预算约束的定义

### 2.1.1 预算约束的定义

**预算约束（Budget Constraint）**：
消费者能够购买的商品组合受到收入的限制。

### 2.1.2 预算约束的假设

**预算约束的假设（Assumptions of Budget Constraint）**：

1. **两种商品（Two Goods）**：
   - 商品1（Good 1）
   - 商品2（Good 2）

2. **固定价格（Fixed Prices）**：
   - 商品1的价格：p₁
   - 商品2的价格：p₂

3. **固定收入（Fixed Income）**：
   - 收入：m

4. **完全可分（Perfectly Divisible）**：
   - 商品可以按任意数量购买
   - 包括分数

---

## 2.2 预算约束的数学表述

### 2.2.1 预算约束的公式

**预算约束的公式（Budget Constraint Formula）**：
```
p₁x₁ + p₂x₂ ≤ m
```

**其中**：
- p₁：商品1的价格
- x₁：商品1的数量
- p₂：商品2的价格
- x₂：商品2的数量
- m：收入

**含义**：
- 商品1的支出：p₁x₁
- 商品2的支出：p₂x₂
- 总支出：p₁x₁ + p₂x₂
- 总支出不能超过收入：p₁x₁ + p₂x₂ ≤ m

### 2.2.2 预算线

**预算线（Budget Line）**：
预算约束的等式：p₁x₁ + p₂x₂ = m

## 2.3 预算约束的性质

### 2.3.1 预算约束的性质

**预算约束的性质（Properties of Budget Constraint）**：

1. **凸性（Convexity）**：
   - 预算集是凸集
   - 任意两个商品组合的加权平均仍在预算集内

2. **有界性（Boundedness）**：
   - 预算集是有界的
   - 商品数量不能无限增加

3. **闭合性（Closedness）**：
   - 预算集是闭合的
   - 包括边界

**商品1价格和收入同比例变化**：

## 2.5 税收和补贴

### 2.5.1 数量税

**数量税（Quantity Tax）**：
对每单位商品征收的税。

**效果**：
- 商品价格上升
- 预算线旋转

**公式**：
```
p₁' = p₁ + t
```

**其中**：
- p₁'：含税价格
- p₁：不含税价格
- t：税

### 2.5.2 数量补贴

**数量补贴（Quantity Subsidy）**：
对每单位商品给予的补贴。

**效果**：
- 商品价格下降
- 预算线旋转

**公式**：
```
p₁' = p₁ - s
```

**其中**：
- p₁'：含补贴价格
- p₁：不含补贴价格
- s：补贴

### 2.5.3 从价税

**从价税（Ad Valorem Tax）**：
按商品价格比例征收的税。

**效果**：
- 商品价格上升
- 预算线旋转

**公式**：
```
p₁' = p₁(1 + t)
```

**其中**：
- p₁'：含税价格
- p₁：不含税价格
- t：税率

### 2.5.4 从价补贴

**从价补贴（Ad Valorem Subsidy）**：
按商品价格比例给予的补贴。

**效果**：
- 商品价格下降
- 预算线旋转

**公式**：
```
p₁' = p₁(1 - s)
```

**其中**：
- p₁'：含补贴价格
- p₁：不含补贴价格
- s：补贴率

## 2.7 本章总结

### 核心要点

1. **预算约束的定义**：
   - 预算约束：消费者能够购买的商品组合受到收入的限制
   - 假设：两种商品、固定价格、固定收入、完全可分

2. **预算约束的数学表述**：
   - 预算约束的公式：p₁x₁ + p₂x₂ ≤ m
   - 预算线：p₁x₁ + p₂x₂ = m
   - 预算集：消费者能够购买的所有商品组合

3. **预算约束的性质**：
   - 凸性：预算集是凸集
   - 有界性：预算集是有界的
   - 闭合性：预算集是闭合的

4. **预算约束的变化**：
   - 收入变化：预算线平移
   - 价格变化：预算线旋转
   - 收入和价格同时变化：预算线可能平移或旋转

5. **税收和补贴**：
   - 数量税：对每单位商品征收的税，商品价格上升
   - 数量补贴：对每单位商品给予的补贴，商品价格下降
   - 从价税：按商品价格比例征收的税，商品价格上升
   - 从价补贴：按商品价格比例给予的补贴，商品价格下降

6. **配给**：
   - 配给：对商品购买数量的限制
   - 消费者不能购买超过配给数量的商品
