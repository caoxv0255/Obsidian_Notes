## 重要概念

### 基本概念
- 偏好（Preferences）
- 效用（Utility）
- 预算约束（Budget Constraint）
- 优化（Optimization）
- 均衡（Equilibrium）

### 消费者理论
- 效用最大化（Utility Maximization）
- 需求函数（Demand Function）
- 消费者剩余（Consumer Surplus）
- 显示偏好（Revealed Preference）
- 斯勒茨基方程（Slutsky Equation）

### 生产者理论
- 利润最大化（Profit Maximization）
- 成本最小化（Cost Minimization）
- 供给函数（Supply Function）
- 生产者剩余（Producer Surplus）
- 成本曲线（Cost Curves）

### 市场结构
- 完全竞争（Perfect Competition）
- 垄断（Monopoly）
- 寡头（Oligopoly）
- 博弈论（Game Theory）
- 纳什均衡（Nash Equilibrium）
