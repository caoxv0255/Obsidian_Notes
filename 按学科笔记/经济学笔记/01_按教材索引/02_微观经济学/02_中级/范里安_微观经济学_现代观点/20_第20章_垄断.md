# 第20章：垄断

> 本章学习目标：
> - 理解垄断的定义

> - 学会分析垄断的福利损失

---

## 20.1 垄断的定义

### 20.1.1 垄断的定义

**垄断（Monopoly）**：
市场上只有一个卖者。

### 20.1.2 垄断的特征

**垄断的特征（Characteristics of Monopoly）**：

**1. 唯一卖者（Single Seller）**：
- 市场上只有一个卖者
- 卖者控制供给

**2. 价格制定者（Price Maker）**：
- 垄断厂商可以制定价格
- 价格取决于产量

**3. 进入壁垒（Barriers to Entry）**：
- 其他厂商无法进入市场
- 进入壁垒包括：法律壁垒、自然壁垒、战略壁垒

**4. 产品无替代品（No Close Substitutes）**：
- 垄断厂商的产品没有接近的替代品
- 消费者无法转向其他产品

---

## 20.2 垄断厂商的定价决策

### 20.2.1 垄断厂商的利润最大化

**垄断厂商的利润最大化（Monopolist's Profit Maximization）**：
```
max π = p(y) × y - C(y)
```

**其中**：
- π：利润
- p(y)：反需求函数
- y：产量
- C(y)：成本函数

### 20.2.2 垄断厂商的利润最大化条件

**垄断厂商的利润最大化条件（Condition of Monopolist's Profit Maximization）**：
```
MR(y) = MC(y)
```

**其中**：
- MR(y)：边际收益
- MC(y)：边际成本

**边际收益的公式**：
```
MR(y) = p(y) + y × dp/dy = p(y) × (1 + 1/ε)
```

**其中**：
- p(y)：价格
- y：产量
- dp/dy：价格对产量的导数
- ε：需求价格弹性

**含义**：
- 边际收益等于边际成本
- 利润最大化的必要条件

**二阶条件**：
```
dMR/dy - dMC/dy < 0
```

**含义**：
- 边际收益递减的速度快于边际成本递增的速度
- 利润最大化的充分条件

## 20.3 垄断定价

### 20.3.1 垄断定价的公式

**垄断定价的公式（Formula of Monopoly Pricing）**：
```
p = MC(y)/(1 + 1/ε)
```

**其中**：
- p：价格
- MC(y)：边际成本
- ε：需求价格弹性

**含义**：
- 价格加成（Markup）取决于需求价格弹性
- 需求价格弹性越小，价格加成越大

### 20.3.2 勒纳指数

**勒纳指数（Lerner Index）**：
衡量垄断势力的指标。

**公式**：
```
L = (p - MC)/p = -1/ε
```

**其中**：
- L：勒纳指数
- p：价格
- MC：边际成本
- ε：需求价格弹性

**含义**：
- 勒纳指数等于价格减去边际成本除以价格
- 勒纳指数等于负的需求价格弹性的倒数
- 勒纳指数越大，垄断势力越大

## 20.4 垄断的福利损失

### 20.4.1 无谓损失

**无谓损失（Deadweight Loss, DWL）**：
垄断导致的福利损失。

**图形表示**：

```plotly
data:
  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "y"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- MC：边际成本曲线
- D：需求曲线
- 垄断产量低于完全竞争产量
- 垄断价格高于完全竞争价格
- 无谓损失：垄断导致的福利损失

### 20.4.2 无谓损失的计算

**无谓损失的计算（Calculation of Deadweight Loss）**：
```
DWL = ∫ᵧₘᵧ꜀ [p(y) - MC(y)] dy
```

**其中**：
- DWL：无谓损失
- p(y)：价格函数
- MC(y)：边际成本函数
- yₘ：垄断产量
- y꜀：完全竞争产量

**含义**：
- 无谓损失等于垄断产量和完全竞争产量之间，需求曲线和边际成本曲线之间的面积

## 20.5 价格歧视

### 20.5.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
对相同的商品或服务向不同消费者收取不同的价格。

### 20.5.2 价格歧视的条件

**价格歧视的条件（Conditions for Price Discrimination）**：

**1. 市场势力（Market Power）**：
- 垄断厂商有定价能力

**2. 消费者异质性（Consumer Heterogeneity）**：
- 消费者的支付意愿不同

**3. 防止套利（Prevent Arbitrage）**：
- 消费者无法转卖商品

### 20.5.3 价格歧视的类型

**价格歧视的类型（Types of Price Discrimination）**：

**1. 一级价格歧视（First-Degree Price Discrimination）**：
- 对每个消费者收取其支付意愿
- 垄断厂商获得全部消费者剩余
- 无谓损失为零

**2. 二级价格歧视（Second-Degree Price Discrimination）**：
- 根据购买数量定价
- 消费者选择不同的价格-数量组合
- 垄断厂商获得部分消费者剩余

**3. 三级价格歧视（Third-Degree Price Discrimination）**：
- 根据消费者群体定价
- 不同消费者群体支付不同价格
- 垄断厂商获得部分消费者剩余

## 20.6 垄断管制

### 20.6.1 垄断管制的定义

**垄断管制（Monopoly Regulation）**：
政府对垄断厂商的管制。

### 20.6.2 垄断管制的类型

**垄断管制的类型（Types of Monopoly Regulation）**：

**1. 价格管制（Price Regulation）**：
- 政府规定垄断厂商的价格
- 目标：减少垄断的无谓损失

**2. 收益管制（Rate of Return Regulation）**：
- 政府规定垄断厂商的收益率
- 目标：防止垄断厂商获得超额利润

**3. 拆分垄断（Break Up Monopoly）**：
- 政府拆分垄断厂商
- 目标：增加竞争

## 20.7 本章总结

### 核心要点

1. **垄断的定义**：
   - 垄断：市场上只有一个卖者
   - 垄断的特征：唯一卖者、价格制定者、进入壁垒、产品无替代品

2. **垄断厂商的定价决策**：
   - 垄断厂商的利润最大化：max π = p(y) × y - C(y)
   - 垄断厂商的利润最大化条件：MR(y) = MC(y)
   - 边际收益的公式：MR(y) = p(y) × (1 + 1/ε)

3. **垄断定价**：
   - 垄断定价的公式：p = MC(y)/(1 + 1/ε)
   - 勒纳指数：L = (p - MC)/p = -1/ε

4. **垄断的福利损失**：
   - 无谓损失：垄断导致的福利损失
   - 无谓损失的计算：DWL = ∫ᵧₘᵧ꜀ [p(y) - MC(y)] dy

5. **价格歧视**：
   - 价格歧视：对相同的商品或服务向不同消费者收取不同的价格
   - 价格歧视的条件：市场势力、消费者异质性、防止套利
   - 价格歧视的类型：一级价格歧视、二级价格歧视、三级价格歧视

6. **垄断管制**：
   - 垄断管制：政府对垄断厂商的管制
   - 垄断管制的类型：价格管制、收益管制、拆分垄断
