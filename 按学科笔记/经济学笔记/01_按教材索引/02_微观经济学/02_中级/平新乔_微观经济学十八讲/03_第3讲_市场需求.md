# 第三讲：市场需求

> 本章学习目标：
> - 理解个人需求
> - 掌握市场需求
> - 学会分析需求弹性

---

## 3.1 个人需求

### 3.1.1 个人需求的定义

**个人需求（Individual Demand）**：
在给定价格和收入下，消费者选择的最优消费量。

### 3.1.2 个人需求函数

**个人需求函数（Individual Demand Function）**：
表示消费者对商品的需求量与价格和收入之间的函数关系。

**公式**：
```
xᵢ = xᵢ(p₁, p₂, m)
```

**其中**：
- xᵢ：消费者对商品i的需求量
- p₁, p₂：商品1和商品2的价格
- m：收入

### 3.1.3 需求曲线

**需求曲线（Demand Curve）**：
表示需求量与价格之间关系的曲线。

**图形表示**：

```plotly
data:
  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "p"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- D：需求曲线
- 横轴：需求量x
- 纵轴：价格p
- 需求曲线向右下方倾斜

### 3.1.4 需求定律

**需求定律（Law of Demand）**：
在其他条件不变的情况下，价格上升，需求量下降。

**条件**：
- 替代效应和收入效应都为负
- 或者替代效应为负，收入效应为正但小于替代效应

## 3.2 市场需求

### 3.2.1 市场需求的定义

**市场需求（Market Demand）**：
所有消费者对商品的需求的总和。

### 3.2.2 市场需求函数

**市场需求函数（Market Demand Function）**：
表示市场总需求量与价格和总收入之间的函数关系。

**公式**：
```
X = Σᵢ xᵢ(p, mᵢ)
```

**其中**：
- X：市场总需求量
- xᵢ(p, mᵢ)：消费者i的需求函数
- p：价格
- mᵢ：消费者i的收入

### 3.2.3 市场需求曲线

**市场需求曲线（Market Demand Curve）**：
表示市场总需求量与价格之间关系的曲线。

**图形表示**：

```plotly
data:
  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "p"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true

  responsive: true
```

**分析**：
- D：市场需求曲线
- 横轴：市场总需求量X
- 纵轴：价格p
- 市场需求曲线向右下方倾斜

### 3.2.4 横向加总

**横向加总（Horizontal Summation）**：
市场需求曲线是个人需求曲线的横向加总。

**公式**：
```
X(p) = Σᵢ xᵢ(p)
```

**其中**：
- X(p)：市场需求
- xᵢ(p)：消费者i的需求

**含义**：
- 在给定价格下，市场需求等于所有消费者个人需求之和

## 3.3 需求弹性

### 3.3.1 需求价格弹性的定义

**需求价格弹性（Price Elasticity of Demand）**：
需求量对价格变化的敏感程度。

### 3.3.2 需求价格弹性的公式

**需求价格弹性的公式（Formula of Price Elasticity of Demand）**：
```
ε = (∂X/∂p) × (p/X)
```

**其中**：
- ε：需求价格弹性
- ∂X/∂p：需求对价格的导数
- p：价格
- X：需求量

### 3.3.3 需求价格弹性的分类

**需求价格弹性的分类（Classification of Price Elasticity of Demand）**：

**1. 富有弹性（Elastic）**：
- |ε| > 1
- 需求量对价格变化敏感
- 例如：奢侈品

**2. 单位弹性（Unit Elastic）**：
- |ε| = 1
- 需求量变化率等于价格变化率
- 例如：某些必需品

**3. 缺乏弹性（Inelastic）**：
- |ε| < 1
- 需求量对价格变化不敏感
- 例如：必需品

**4. 完全弹性（Perfectly Elastic）**：
- |ε| = ∞
- 需求曲线是水平的
- 例如：完全竞争市场

**5. 完全无弹性（Perfectly Inelastic）**：
- |ε| = 0
- 需求曲线是垂直的
- 例如：救命药物

### 3.3.4 需求价格弹性的决定因素

**需求价格弹性的决定因素（Determinants of Price Elasticity of Demand）**：

**1. 替代品的可获得性（Availability of Substitutes）**：
- 替代品越多，弹性越大
- 替代品越少，弹性越小

**2. 必需品与奢侈品（Necessities vs. Luxuries）**：
- 必需品弹性小
- 奢侈品弹性大

**3. 时间跨度（Time Horizon）**：
- 短期弹性小
- 长期弹性大

**4. 商品在预算中的份额（Share of Budget）**：
- 预算份额越大，弹性越大
- 预算份额越小，弹性越小

## 3.4 需求收入弹性

### 3.4.1 需求收入弹性的定义

**需求收入弹性（Income Elasticity of Demand）**：
需求量对收入变化的敏感程度。

### 3.4.2 需求收入弹性的公式

**需求收入弹性的公式（Formula of Income Elasticity of Demand）**：
```
η = (∂X/∂m) × (m/X)
```

**其中**：
- η：需求收入弹性
- ∂X/∂m：需求对收入的导数
- m：收入
- X：需求量

### 3.4.3 需求收入弹性的分类

**需求收入弹性的分类（Classification of Income Elasticity of Demand）**：

**1. 正常商品（Normal Goods）**：
- η > 0
- 收入增加，需求增加
- 例如：大多数商品

**2. 劣等商品（Inferior Goods）**：
- η < 0
- 收入增加，需求减少
- 例如：某些低档商品

**3. 奢侈品（Luxury Goods）**：
- η > 1
- 收入增加，需求增加更多
- 例如：高档商品

**4. 必需品（Necessity Goods）**：
- 0 < η < 1
- 收入增加，需求增加较少
- 例如：食物

## 3.5 本章总结

### 核心要点

1. **个人需求**：
   - 个人需求：在给定价格和收入下，消费者选择的最优消费量
   - 个人需求函数：xᵢ = xᵢ(p₁, p₂, m)
   - 需求曲线：表示需求量与价格之间关系的曲线
   - 需求定律：价格上升，需求量下降

2. **市场需求**：
   - 市场需求：所有消费者对商品的需求的总和
   - 市场需求函数：X = Σᵢ xᵢ(p, mᵢ)
   - 市场需求曲线：表示市场总需求量与价格之间关系的曲线
   - 横向加总：X(p) = Σᵢ xᵢ(p)

3. **需求弹性**：
   - 需求价格弹性：ε = (∂X/∂p) × (p/X)
   - 需求价格弹性的分类：富有弹性、单位弹性、缺乏弹性、完全弹性、完全无弹性
   - 需求价格弹性的决定因素：替代品的可获得性、必需品与奢侈品、时间跨度、商品在预算中的份额

4. **需求收入弹性**：
   - 需求收入弹性：η = (∂X/∂m) × (m/X)
   - 需求收入弹性的分类：正常商品、劣等商品、奢侈品、必需品
