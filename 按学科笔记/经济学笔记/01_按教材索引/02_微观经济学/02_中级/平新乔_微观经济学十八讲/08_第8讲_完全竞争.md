# 第八讲：完全竞争

## 8.1 完全竞争市场

### 8.1.1 完全竞争市场的定义

**完全竞争市场（Perfectly Competitive Market）**：
许多买者和卖者的市场结构。

### 8.1.2 完全竞争市场的特征

**完全竞争市场的特征（Characteristics of Perfectly Competitive Market）**：

**1. 许多买者和卖者（Many Buyers and Sellers）**：
- 市场上有许多买者和卖者
- 每个买者和卖者市场份额小

**2. 产品同质（Homogeneous Products）**
- 所有卖者生产的产品是同质的
- 买者和卖者对产品无偏好

**3. 自由进入和退出（Free Entry and Exit）**
- 企业可以自由进入和退出市场
- 没有进入壁垒

**4. 完全信息（Perfect Information）**
- 买者和卖者拥有完全信息
- 没有信息不对称

## 8.2 短期均衡

### 8.2.1 短期均衡的定义

**短期均衡（Short-Run Equilibrium）**：
在资本固定的情况下，市场达到均衡状态。

### 8.2.2 短期市场均衡条件

**短期市场均衡条件（Condition of Short-Run Market Equilibrium）**：
```
D(p) = S(p)
```

**其中**：
- D(p)：市场需求
- S(p)：市场供给
- p：价格

**含义**：
- 市场需求等于市场供给
- 市场出清

### 8.2.3 短期均衡的决定

**短期均衡的决定（Determination of Short-Run Equilibrium）**：

**图形表示**：

- S：供给曲线
- D：需求曲线
- 横轴：产量y
- 纵轴：价格p
- 均衡点：供给曲线和需求曲线的交点
- 均衡价格：p*
- 均衡产量：y*

### 8.2.4 短期企业的利润

**短期企业的利润（Short-Run Profit of Firm）**：
```
π = p*y - C(y)
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

**三种情况**：
- p > AC：正利润
- p = AC：零利润
- p < AC：负利润

## 8.3 长期均衡

### 8.3.1 长期均衡的定义

**长期均衡（Long-Run Equilibrium）**：
在所有投入都可变的情况下，市场达到均衡状态。

### 8.3.2 长期市场均衡条件

**长期市场均衡条件（Condition of Long-Run Market Equilibrium）**：
```
D(p) = S(p)
p = min AC
```

**其中**：
- D(p)：市场需求
- S(p)：市场供给
- p：价格
- AC：平均成本

**含义**：
- 市场需求等于市场供给
- 价格等于最小平均成本
- 企业获得零利润

### 8.3.3 长期均衡的决定

**长期均衡的决定（Determination of Long-Run Equilibrium）**：

**图形表示**：

**分析**：

- LMC：长期边际成本曲线
- 横轴：产量y
- 纵轴：价格p
- 均衡点：长期平均成本曲线的最低点
- 均衡价格：p* = min AC
- 均衡产量：y* = argmin AC

### 8.3.4 长期企业的利润

**长期企业的利润（Long-Run Profit of Firm）**：
```
π = p*y - C(y) = 0
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

**含义**：
- 价格等于平均成本
- 企业获得零利润
- 正常利润

## 8.4 福利分析

### 8.4.1 消费者剩余

**消费者剩余（Consumer Surplus）**：
消费者愿意支付的最高价格与实际支付价格之间的差额。

**公式**：
```
CS = ∫₀^y D(p) dp - p*y
```

**其中**：
- CS：消费者剩余
- D(p)：需求函数
- p：实际支付价格
- y：消费量

### 8.4.2 生产者剩余

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

**公式**：
```
PS = p*y - ∫₀^y S(p) dp
```

**其中**：
- PS：生产者剩余
- S(p)：供给函数
- p：实际获得价格
- y：供给量

### 8.4.3 总剩余

**总剩余（Total Surplus）**：
消费者剩余和生产者剩余的总和。

**公式**：
```
TS = CS + PS
```

**其中**：
- TS：总剩余
- CS：消费者剩余
- PS：生产者剩余

**含义**：
- 总剩余衡量社会总福利
- 总剩余最大化表示市场效率

### 8.4.4 完全竞争的效率

**完全竞争的效率（Efficiency of Perfect Competition）**：
- 完全竞争市场实现帕累托效率
- 总剩余最大化
- 消费者剩余和生产者剩余之和最大化

**福利经济学第一定理**：
任何竞争性均衡都是帕累托有效的。

## 8.5 本章总结

### 核心要点

1. **完全竞争市场**：
   - 完全竞争市场：许多买者和卖者的市场结构
   - 完全竞争市场的特征：许多买者和卖者、产品同质、自由进入和退出、完全信息

2. **短期均衡**：
   - 短期均衡：在资本固定的情况下，市场达到均衡状态
   - 短期市场均衡条件：D(p) = S(p)
   - 短期企业的利润：π = p*y - C(y)
   - 三种情况：p > AC（正利润）、p = AC（零利润）、p < AC（负利润）

3. **长期均衡**：
   - 长期均衡：在所有投入都可变的情况下，市场达到均衡状态
   - 长期市场均衡条件：D(p) = S(p)，p = min AC
   - 长期企业的利润：π = p*y - C(y) = 0
   - 企业获得零利润（正常利润）

4. **福利分析**：
   - 消费者剩余：CS = ∫₀^y D(p) dp - p*y
   - 生产者剩余：PS = p*y - ∫₀^y S(p) dp
   - 总剩余：TS = CS + PS
   - 完全竞争的效率：完全竞争市场实现帕累托效率，总剩余最大化
