# 第一讲：偏好与效用

> 本章学习目标：
> - 理解偏好的定义和性质
> - 掌握效用函数
> - 学会分析无差异曲线

---

## 1.1 偏好的定义

### 1.1.1 偏好的定义

**偏好（Preference）**：
消费者对不同商品组合的评价。

### 1.1.2 偏好的性质

**偏好的性质（Properties of Preference）**：

**1. 完备性（Completeness）**：
- 对于任意两个商品组合，消费者能够进行比较
- 对于任意x和y，有：x ≽ y 或 y ≽ x 或两者都成立

**2. 传递性（Transitivity）**：
- 如果x ≽ y且y ≽ z，则x ≽ z
- 偏好是一致的

**3. 反身性（Reflexivity）**：
- 对于任意x，有x ≽ x
- 商品组合至少和自身一样好

**4. 连续性（Continuity）**：
- 偏好关系是连续的
- 如果xₙ → x，yₙ → y，且xₙ ≽ yₙ对所有n成立，则x ≽ y

**5. 单调性（Monotonicity）**：
- 商品越多越好
- 如果x ≥ y，则x ≽ y

**6. 凸性（Convexity）**：
- 平均组合优于极端组合
- 如果x ≽ z且y ≽ z，则对于所有λ∈[0,1]，有λx + (1-λ)y ≽ z

## 1.2 效用函数

### 1.2.1 效用函数的定义

**效用函数（Utility Function）**：
表示偏好的数值函数。

### 1.2.2 效用函数的性质

**效用函数的性质（Properties of Utility Function）**：

**1. 正单调变换（Positive Monotonic Transformation）**：
- 如果f(·)是严格递增函数，则f(u(x))表示相同的偏好
- 效用函数的序数性质：只有序数意义，没有基数意义

**2. 边际效用（Marginal Utility）**：
- 效用函数对某一商品的偏导数
- MUₓ = ∂U/∂x

**3. 边际效用递减（Diminishing Marginal Utility）**：
- 随着消费量增加，边际效用递减
- ∂²U/∂x² < 0

### 1.2.3 常见的效用函数

**常见的效用函数（Common Utility Functions）**：

**1. 柯布-道格拉斯效用函数（Cobb-Douglas Utility Function）**：
```
U(x, y) = xᵃ yᵇ
```
**其中**：
- a, b > 0
- 边际效用递减
- 凸偏好

**2. 完全替代效用函数（Perfect Substitutes）**：
```
U(x, y) = ax + by
```
**其中**：
- a, b > 0
- 边际效用不变
- 线性偏好

**3. 完全互补效用函数（Perfect Complements）**：
```
U(x, y) = min{ax, by}
```
**其中**：
- a, b > 0
- 里昂惕夫偏好

**4. CES效用函数（Constant Elasticity of Substitution）**：
```
U(x, y) = [axᵖ + byᵖ]^(1/p)
```
**其中**：
- a, b > 0
- p ≤ 1
- 替代弹性为1/(1-p)

**5. 拟线性效用函数（Quasilinear Utility Function）**：
```
U(x, y) = u(x) + y
```
**其中**：
- u(·)是严格递增的凹函数
- y是计价物

## 1.3 无差异曲线

### 1.3.1 无差异曲线的定义

**无差异曲线（Indifference Curve）**：
使消费者获得相同效用的所有商品组合的集合。

### 1.3.2 无差异曲线的性质

**无差异曲线的性质（Properties of Indifference Curve）**：

**1. 向右下方倾斜（Downward Sloping）**：
- 斜率为负
- 为了保持效用不变，减少一种商品必须增加另一种商品

**2. 凸向原点（Convex to the Origin）**：
- 边际替代率递减
- 平均组合优于极端组合

**3. 不相交（Non-Intersecting）**：
- 两条无差异曲线不能相交
- 如果相交，则违反传递性

**4. 越远离原点效用越高（Higher Utility Further from Origin）**：
- 商品越多越好
- 距离原点越远的无差异曲线效用越高

### 1.3.3 边际替代率

**边际替代率（Marginal Rate of Substitution, MRS）**：
在保持效用不变的情况下，消费者愿意用一种商品替代另一种商品的比率。

**公式**：
```
MRSₓy = MUₓ/MUy = -dy/dx|U=U̅
```

**其中**：
- MRSₓy：x对y的边际替代率
- MUₓ：x的边际效用
- MUy：y的边际效用
- dy/dx|U=U̅：无差异曲线的斜率

**含义**：
- 边际替代率等于无差异曲线斜率的绝对值
- 边际替代率等于边际效用之比

### 1.3.4 边际替代率递减

**边际替代率递减（Diminishing Marginal Rate of Substitution）**：
随着x的增加，边际替代率递减。

**数学表述**：
```
∂MRSₓy/∂x < 0
```

**含义**：
- 随着x的增加，消费者愿意用x替代y的意愿递减
- 无差异曲线凸向原点

## 1.4 本章总结

### 核心要点

1. **偏好的定义**：
   - 偏好：消费者对不同商品组合的评价
   - 偏好的性质：完备性、传递性、反身性、连续性、单调性、凸性

2. **效用函数**：
   - 效用函数：表示偏好的数值函数
   - 效用函数的性质：正单调变换、边际效用、边际效用递减
   - 常见的效用函数：柯布-道格拉斯、完全替代、完全互补、CES、拟线性

3. **无差异曲线**：
   - 无差异曲线：使消费者获得相同效用的所有商品组合的集合
   - 无差异曲线的性质：向右下方倾斜、凸向原点、不相交、越远离原点效用越高
   - 边际替代率：MRSₓy = MUₓ/MUy = -dy/dx|U=U̅
   - 边际替代率递减：∂MRSₓy/∂x < 0
