## 重要概念

### 基本概念
- 供给（Supply）
- 需求（Demand）
- 均衡（Equilibrium）
- 弹性（Elasticity）
- 效用（Utility）
- 消费者均衡（Consumer Equilibrium）

### 生产者理论
- 生产函数（Production Function）
- 成本函数（Cost Function）
- 利润最大化（Profit Maximization）
- 市场结构（Market Structure）
- 完全竞争（Perfect Competition）

### 市场结构
- 垄断（Monopoly）
- 寡头（Oligopoly）
- 垄断竞争（Monopolistic Competition）
- 博弈论（Game Theory）
- 纳什均衡（Nash Equilibrium）
