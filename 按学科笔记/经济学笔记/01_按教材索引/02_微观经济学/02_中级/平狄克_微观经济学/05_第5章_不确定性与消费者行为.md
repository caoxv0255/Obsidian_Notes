## 5.1 不确定性

### 5.1.1 不确定性的定义

**不确定性（Uncertainty）**
结果不确定的情况。

### 5.1.2 概率分布

**概率分布（Probability Distribution）**
表示各种结果发生可能性的分布。

**期望值（Expected Value）**：
```
E[X] = Σₜ pₜ xₜ
```

**方差（Variance）**：
```
Var[X] = Σₜ pₜ (xₜ - E[X])²
```

---

## 5.2 期望效用理论

### 5.2.1 期望效用函数

**期望效用函数（Expected Utility Function）**：
```
U(P) = Σₜ pₜ u(xₜ)
```

**其中**：
- U(P)：概率分布P的期望效用
- pₜ：结果t的概率
- u(xₜ)：结果t的效用

### 5.2.2 风险态度

**风险态度（Risk Attitude）**：

**1. 风险规避（Risk Aversion）**：
- u(E[X]) > E[u(X)]
- 效用函数是凹的

**2. 风险中性（Risk Neutrality）**：
- u(E[X]) = E[u(X)]
- 效用函数是线性的

**3. 风险偏好（Risk Loving）**：
- u(E[X]) < E[u(X)]
- 效用函数是凸的

---

## 5.3 保险

### 5.3.1 保险的定义

**保险（Insurance）**
转移风险的机制。

### 5.3.2 保险需求

**保险需求（Insurance Demand）**：
风险规避的消费者愿意支付保险费。

**条件**：
```
u(w - p) ≥ E[u(w - L)]
```

---

## 5.4 本章总结

### 核心要点

1. **不确定性**：
   - 不确定性：结果不确定的情况
   - 概率分布：表示各种结果发生可能性的分布
   - 期望值：E[X] = Σₜ pₜ xₜ
   - 方差：Var[X] = Σₜ pₜ (xₜ - E[X])²

2. **期望效用理论**：
   - 期望效用函数：U(P) = Σₜ pₜ u(xₜ)
   - 风险规避：u(E[X]) > E[u(X)]，效用函数是凹的
   - 风险中性：u(E[X]) = E[u(X)]，效用函数是线性的
   - 风险偏好：u(E[X]) < E[u(X)]，效用函数是凸的

3. **保险**：
   - 保险：转移风险的机制
   - 保险需求：风险规避的消费者愿意支付保险费
