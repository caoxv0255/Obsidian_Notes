# 第13章：博弈论和竞争策略

## 13.1 博弈论的应用

### 13.1.1 博弈论在经济中的应用

**博弈论在经济中的应用（Applications of Game Theory in Economics）**：

**1. 寡头竞争（Oligopolistic Competition）**：
- 企业之间相互影响
- 企业需要考虑竞争对手的反应

**2. 讨价还价（Bargaining）**：
- 双方协商价格
- 双方需要考虑对方的策略

**3. 拍卖（Auction）**：
- 竞价者竞争商品
- 竞价者需要考虑其他竞价者的出价

## 13.2 重复博弈

### 13.2.1 重复博弈的定义

**重复博弈（Repeated Game）**：
同一博弈重复进行多次。

### 13.2.2 重复博弈的类型

**重复博弈的类型（Types of Repeated Game）**：

**1. 有限重复博弈（Finite Repeated Game）**：
- 博弈重复有限次
- 重复次数已知

**2. 无限重复博弈（Infinite Repeated Game）**：
- 博弈重复无限次
- 重复次数未知

### 13.2.3 重复博弈的均衡

**重复博弈的均衡（Equilibrium of Repeated Game）**：

**1. 有限重复博弈的均衡**：
- 有限重复囚徒困境的纳什均衡是不合作
- 最后一期不合作，倒推第一期也不合作

**2. 无限重复博弈的均衡**：
- 无限重复囚徒困境可能存在合作均衡
- 如果折现因子足够大，合作是纳什均衡

---

## 13.3 序贯博弈

### 13.3.1 序贯博弈的定义

**序贯博弈（Sequential Game）**：
参与人依次行动的博弈。

### 13.3.2 序贯博弈的表示

**序贯博弈的表示（Representation of Sequential Game）**：

**1. 博弈树（Game Tree）**：
- 表示序贯博弈的图形
- 节点表示决策点
- 分支表示行动选择
- 终点表示收益

**图形表示**：

```
      参与人A
        / \
      合作 不合作
      /       \
  参与人B    参与人B
    / \        / \
  合作 不合作 合作 不合作
    |    |     |    |
   -1,-1 -10,0 -10,0 -5,-5
```

**分析**：
- 参与人A先选择合作或不合作
- 参与人B后选择合作或不合作
- 终点表示收益

### 13.3.3 子博弈完美均衡

**子博弈完美均衡（Subgame Perfect Equilibrium）**：
在每个子博弈中都是纳什均衡的策略组合。

**定义**：
```
对于所有子博弈，策略组合都是纳什均衡
```

**含义**：
- 子博弈完美均衡是更强的均衡概念
- 消除不可置信的威胁

## 13.4 威胁和承诺

### 13.4.1 威胁的定义

**威胁（Threat）**：
参与人宣称如果对方采取某些行动，将采取报复行动。

### 13.4.2 威胁的可信性

**威胁的可信性（Credibility of Threat）**：
威胁是否可信取决于是否符合理性。

**条件**：
- 如果威胁符合参与人的利益，威胁是可信的
- 如果威胁不符合参与人的利益，威胁是不可信的

### 13.4.3 承诺的定义

**承诺（Commitment）**：
参与人采取行动使自己无法改变策略。

### 13.4.4 承诺的可信性

**承诺的可信性（Credibility of Commitment）**：
承诺是否可信取决于是否能够约束自己。

**条件**：
- 如果承诺能够约束自己，承诺是可信的
- 如果承诺不能约束自己，承诺是不可信的

---

## 13.5 合作策略

### 13.5.1 合作策略的定义

**合作策略（Cooperative Strategy）**：
参与人合作以获得更高收益。

### 13.5.2 合作策略的类型

**合作策略的类型（Types of Cooperative Strategy）**：

**1. 触发策略（Trigger Strategy）**：
- 如果对方合作，我也合作
- 如果对方不合作，我永远不合作

**2. 针锋相对策略（Tit-for-Tat Strategy）**：
- 第一期合作
- 后续期采取对方上一期的策略

**3. 宽恕触发策略（Forgiving Trigger Strategy）**：
- 如果对方合作，我也合作
- 如果对方不合作，我惩罚k期后重新合作

## 13.6 本章总结

### 核心要点

1. **博弈论的应用**：
   - 博弈论在经济中的应用：寡头竞争、讨价还价、拍卖

2. **重复博弈**：
   - 重复博弈：同一博弈重复进行多次
   - 重复博弈的类型：有限重复博弈、无限重复博弈
   - 重复博弈的均衡：有限重复博弈的纳什均衡是不合作；无限重复博弈可能存在合作均衡

3. **序贯博弈**：
   - 序贯博弈：参与人依次行动的博弈
   - 序贯博弈的表示：博弈树
   - 子博弈完美均衡：在每个子博弈中都是纳什均衡的策略组合

4. **威胁和承诺**：
   - 威胁：参与人宣称如果对方采取某些行动，将采取报复行动
   - 威胁的可信性：威胁是否可信取决于是否符合理性
   - 承诺：参与人采取行动使自己无法改变策略
   - 承诺的可信性：承诺是否可信取决于是否能够约束自己

5. **合作策略**：
   - 合作策略：参与人合作以获得更高收益
   - 合作策略的类型：触发策略、针锋相对策略、宽恕触发策略
