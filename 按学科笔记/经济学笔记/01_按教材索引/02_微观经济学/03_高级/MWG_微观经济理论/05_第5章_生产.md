## 5.1 生产函数

### 5.1.1 生产函数的定义

**生产函数（Production Function, f: ℝ₊ᴸ → ℝ₊）**：
表示投入与产出之间的技术关系。

**公式**：
```
y = f(z)
```
**其中**：
- y：产出
- z = (z₁, ..., zₗ)：投入向量
- f：生产函数

### 5.1.2 生产函数的性质

**生产函数的性质（Properties of Production Function）**：

1. **单调性（Monotonicity）**：
   - 如果z ≥ z'，则f(z) ≥ f(z')
   - 投入越多，产出越多

2. **凹性（Concavity）**：
   - 对于所有z, z'和λ ∈ [0,1]，有：
     f(λz + (1-λ)z') ≥ λf(z) + (1-λ)f(z')
   - 边际产量递减

3. **齐次性（Homogeneity）**：
   - 如果生产函数是r次齐次的，则f(αz) = αʳf(z)，对于所有α > 0
   - 如果r = 1，则生产函数是规模报酬不变
   - 如果r > 1，则生产函数是规模报酬递增
   - 如果r < 1，则生产函数是规模报酬递减

4. **连续性（Continuity）**：
   - 生产函数是连续的

### 5.1.3 边际产量

**边际产量（Marginal Product, MPᵢ）**：
投入i增加1单位，产出增加的量。

**公式**：
```
MPᵢ = ∂f(z)/∂zᵢ
```

**性质**：
- 正单调性 ⇒ MPᵢ > 0
- 边际产量递减：∂MPᵢ/∂zᵢ < 0

### 5.1.4 技术替代率

**技术替代率（Technical Rate of Substitution, TRSᵢⱼ）**：
保持产出不变，投入i可以替代投入j的比率。

**公式**：
```
TRSᵢⱼ = MPᵢ / MPⱼ = (∂f(z)/∂zᵢ) / (∂f(z)/∂zⱼ)
```

**性质**：
- 凹性 ⇒ 技术替代率递减
- 沿着等产量曲线，技术替代率等于等产量曲线的斜率的负值

## 5.2 成本最小化

### 5.2.1 成本最小化问题

**成本最小化问题（Cost Minimization Problem, CMP）**：
```
min w·z
s.t. f(z) ≥ q
z ≥ 0
```
**其中**：
- w：投入价格向量（w ≫ 0）
- z：投入向量
- q：产出水平

**要素需求对应（Factor Demand Correspondence, z(w, q)）**：
CMP的解集。

**要素需求函数（Factor Demand Function, z(w, q)）**：
如果CMP的解唯一，则z(w, q)是函数。

### 5.2.2 成本函数

**成本函数（Cost Function, c(w, q)）**：
CMP的最小成本。

**公式**：
```
c(w, q) = min{w·z: f(z) ≥ q, z ≥ 0}
```

**性质**：
1. **齐次性（Homogeneity）**：
   - c(αw, q) = α·c(w, q)，对于所有α > 0

2. **单调性（Monotonicity）**：
   - 如果w ≥ w'，则c(w, q) ≥ c(w', q)
   - 如果q ≥ q'，则c(w, q) ≥ c(w, q')

3. **凹性（Concavity）**：
   - c(w, q)关于w是凹函数

4. **连续性（Continuity）**：
   - c(w, q)关于w和q是连续的

5. **谢泼德引理（Shephard's Lemma）**：
   - zᵢ(w, q) = ∂c(w, q)/∂wᵢ

### 5.2.3 成本函数的结构

**成本函数的结构（Structure of Cost Function）**：

**1. 平均成本（Average Cost, AC）**：
```
AC(q) = c(w, q) / q
```

**2. 边际成本（Marginal Cost, MC）**：
```
MC(q) = ∂c(w, q) / ∂q
```

**3. 固定成本（Fixed Cost, FC）**：
```
FC = c(w, 0)
```

**4. 可变成本（Variable Cost, VC）**：
```
VC(q) = c(w, q) - FC
```

## 5.3 利润最大化

### 5.3.1 利润最大化问题

**利润最大化问题（Profit Maximization Problem, PMP）**：
```
max p·f(z) - w·z
s.t. z ≥ 0
```
**其中**：
- p：产出价格
- w：投入价格向量
- z：投入向量

**供给对应（Supply Correspondence, y(p, w)）**：
PMP的解集。

**供给函数（Supply Function, y(p, w)）**：
如果PMP的解唯一，则y(p, w)是函数。

### 5.3.2 利润函数

**利润函数（Profit Function, π(p, w)）**：
PMP的最大利润。

**公式**：
```
π(p, w) = max{p·f(z) - w·z: z ≥ 0}
```

**性质**：
1. **齐次性（Homogeneity）**：
   - π(αp, αw) = α·π(p, w)，对于所有α > 0

2. **单调性（Monotonicity）**：
   - 如果p ≥ p'，则π(p, w) ≥ π(p', w)
   - 如果w ≥ w'，则π(p, w) ≤ π(p, w')

3. **凸性（Convexity）**：
   - π(p, w)关于(p, w)是凸函数

4. **连续性（Continuity）**：
   - π(p, w)关于p和w是连续的

5. **霍特林引理（Hotelling's Lemma）**：
   - y(p, w) = ∂π(p, w)/∂p
   - -z(p, w) = ∂π(p, w)/∂w

### 5.3.3 利润最大化的一阶条件

**一阶条件（First-Order Conditions, FOCs）**：

对于内点解：
```
p·MPᵢ(z) = wᵢ, 对于所有i
```
**其中**：
- MPᵢ(z) = ∂f(z)/∂zᵢ：边际产量
- wᵢ：投入i的价格

**含义**：
- 产出的边际收益等于投入的边际成本
- 对于所有投入，边际产量等于价格比率

### 5.3.4 利润最大化的二阶条件

**二阶条件（Second-Order Conditions, SOCs）**：
海森矩阵负定：
```
H = [p·∂²f/∂zᵢ∂zⱼ] 是负定的
```

**充分条件**：
- 如果生产函数f(z)是严格凹的，则PMP的解唯一

## 5.4 供给函数

### 5.4.1 供给函数的定义

**供给函数（Supply Function, y(p, w)）**：
利润最大化问题的解。

**性质**：
1. **零次齐次性（Homogeneity of Degree Zero）**：
   - y(αp, αw) = y(p, w)，对于所有α > 0

2. **单调性（Monotonicity）**：
   - 如果p ≥ p'，则y(p, w) ≥ y(p', w)
   - 如果w ≥ w'，则y(p, w) ≤ y(p, w')

3. **连续性（Continuity）**：
   - 如果生产函数是严格凹的，则y(p, w)是连续的

4. **凸性/单值性（Convexity/Single-Valuedness）**：
   - 如果生产函数是严格凹的，则y(p, w)是单值的

### 5.4.2 供给弹性

**供给弹性（Supply Elasticity）**：
```
εᵧ = (∂y/∂p)·(p/y)
```
- εᵧ > 0：正常情况
- εᵧ > 1：富有弹性
- εᵧ < 1：缺乏弹性

### 5.4.3 供给与成本的关系

**供给与成本的关系（Relationship between Supply and Cost）**：
```
p = MC(q)
```
**其中**：
- p：价格
- MC(q)：边际成本

**含义**：
- 在竞争市场中，企业按照边际成本定价
- 供给曲线是边际成本曲线的向上倾斜部分

## 5.5 规模报酬

### 5.5.1 规模报酬的定义

**规模报酬（Returns to Scale）**：
投入按相同比例变化时，产出的变化。

### 5.5.2 规模报酬的类型

**规模报酬的类型（Types of Returns to Scale）**：

1. **规模报酬不变（Constant Returns to Scale, CRS）**：
   - f(αz) = α·f(z)，对于所有α > 0
   - 投入增加α倍，产出增加α倍

2. **规模报酬递增（Increasing Returns to Scale, IRS）**：
   - f(αz) > α·f(z)，对于所有α > 1
   - 投入增加α倍，产出增加大于α倍

3. **规模报酬递减（Decreasing Returns to Scale, DRS）**：
   - f(αz) < α·f(z)，对于所有α > 1
   - 投入增加α倍，产出增加小于α倍

### 5.5.3 规模报酬与成本

**规模报酬与成本（Returns to Scale and Costs）**：

1. **规模报酬不变（CRS）**：
   - 平均成本不变：AC(q) = 常数
   - 边际成本不变：MC(q) = 常数
   - AC(q) = MC(q)

2. **规模报酬递增（IRS）**：
   - 平均成本递减：dAC(q)/dq < 0
   - 边际成本递减：dMC(q)/dq < 0
   - MC(q) < AC(q)

3. **规模报酬递减（DRS）**：
   - 平均成本递增：dAC(q)/dq > 0
   - 边际成本递增：dMC(q)/dq > 0
   - MC(q) > AC(q)

## 5.6 本章总结

### 核心要点

1. **生产函数**：
   - 生产函数：y = f(z)
   - 生产函数的性质：单调性、凹性、齐次性、连续性
   - 边际产量：MPᵢ = ∂f(z)/∂zᵢ
   - 技术替代率：TRSᵢⱼ = MPᵢ / MPⱼ

2. **成本最小化**：
   - 成本最小化问题：min w·z s.t. f(z) ≥ q
   - 成本函数：c(w, q) = min{w·z: f(z) ≥ q}
   - 成本函数的性质：齐次性、单调性、凹性、连续性、谢泼德引理
   - 成本函数的结构：平均成本、边际成本、固定成本、可变成本

3. **利润最大化**：
   - 利润最大化问题：max p·f(z) - w·z
   - 利润函数：π(p, w) = max{p·f(z) - w·z}
   - 利润函数的性质：齐次性、单调性、凸性、连续性、霍特林引理
   - 利润最大化的一阶条件：p·MPᵢ(z) = wᵢ

4. **供给函数**：
   - 供给函数：y(p, w)
   - 供给函数的性质：零次齐次性、单调性、连续性、凸性
   - 供给弹性：εᵧ = (∂y/∂p)·(p/y)
   - 供给与成本的关系：p = MC(q)

5. **规模报酬**：
   - 规模报酬不变：f(αz) = α·f(z)
   - 规模报酬递增：f(αz) > α·f(z)
   - 规模报酬递减：f(αz) < α·f(z)
   - 规模报酬与成本的关系
