## 14.1 霍特林模型

### 14.1.1 霍特林模型的定义

**霍特林模型（Hotelling Model）**：
企业位于空间（产品空间），消费者根据距离（产品差异）选择企业。

**假设**：
1. 企业位于单位长度线段[0, 1]上
2. 企业1位于x₁，企业2位于x₂，0 ≤ x₁ < x₂ ≤ 1
3. 消费者均匀分布在[0, 1]上
4. 消费者选择距离最近的企业
5. 消费者的交通成本为t·|x - xᵢ|，其中t是单位距离的交通成本

### 14.1.2 霍特林均衡

**霍特林均衡（Hotelling Equilibrium）**：
价格组合(p*₁, p*₂)是霍特林均衡，如果：
```
π₁(p*₁, p*₂) ≥ π₁(p₁, p*₂), 对于所有p₁
π₂(p*₁, p*₂) ≥ π₂(p*₁, p₂), 对于所有p₂
```
**其中**：
- π₁(p₁, p₂)：企业1在价格组合(p₁, p₂)下的收益
- π₂(p₁, p₂)：企业2在价格组合(p₁, p₂)下的收益

### 14.1.3 霍特林均衡的计算

**霍特林均衡的计算（Computation of Hotelling Equilibrium）**：

**需求函数**：
- 企业1的需求：D₁(p₁, p₂) = x₁ + (x₂ - x₁)/2 + (p₂ - p₁)/(2t)
- 企业2的需求：D₂(p₁, p₂) = 1 - x₂ + (x₂ - x₁)/2 + (p₁ - p₂)/(2t)

**收益函数**：
- 企业1的收益：π₁(p₁, p₂) = p₁·D₁(p₁, p₂)
- 企业2的收益：π₂(p₁, p₂) = p₂·D₂(p₁, p₂)

**一阶条件**：
```
∂π₁/∂p₁ = 0
∂π₂/∂p₂ = 0
```

**解**：
```
p*₁ = p*₂ = t·(x₂ - x₁)
```
**其中**：
- t：单位距离的交通成本
- x₂ - x₁：企业之间的距离

### 14.1.4 霍特林均衡的性质

**霍特林均衡的性质（Properties of Hotelling Equilibrium）**：

1. **价格与距离正相关（Price Increases with Distance）**：
   - 企业之间的距离越大，均衡价格越高
   - 产品差异化越大，价格越高

2. **与同质产品比较（Comparison with Homogeneous Products）**：
   - 如果x₁ = x₂（企业位于同一位置），则霍特林均衡退化为伯特兰均衡
   - 如果x₁ ≠ x₂（企业位于不同位置），则霍特林均衡价格高于边际成本

3. **最小化差异化（Principle of Minimum Differentiation）**：
   - 如果企业可以选择位置，则企业会位于中心（x₁ = x₂ = 0.5）
   - 最小化差异化，最大化市场份额

## 14.2 萨洛普圆周模型

### 14.2.1 萨洛普圆周模型的定义

**萨洛普圆周模型（Salop Circle Model）**：
企业位于单位圆周上，消费者根据距离选择企业。

**假设**：
1. 企业位于单位圆周上
2. 消费者均匀分布在圆周上
3. 消费者选择距离最近的企业
4. 消费者的交通成本为t·|θ - θᵢ|，其中t是单位距离的交通成本

### 14.2.2 萨洛普均衡

**萨洛普均衡（Salop Equilibrium）**：
价格组合(p*₁, ..., p*ₙ)是萨洛普均衡，如果：
```
πᵢ(p*ᵢ, p*₋ᵢ) ≥ πᵢ(pᵢ, p*₋ᵢ), 对于所有i和所有pᵢ
```
**其中**：
- πᵢ(pᵢ, p*₋ᵢ)：企业i在价格组合(pᵢ, p*₋ᵢ)下的收益

### 14.2.3 萨洛普均衡的计算

**萨洛普均衡的计算（Computation of Salop Equilibrium）**：

**需求函数**：
- 企业i的需求：Dᵢ(pᵢ, p₋ᵢ) = (p₊ᵢ - pᵢ + p₋ᵢ - pᵢ)/(2t) + 1/n
- 其中：p₊ᵢ是相邻企业的价格，p₋ᵢ是另一相邻企业的价格

**收益函数**：
- 企业i的收益：πᵢ(pᵢ, p₋ᵢ) = pᵢ·Dᵢ(pᵢ, p₋ᵢ) - c

**一阶条件**：
```
∂πᵢ/∂pᵢ = 0
```

**解**：
```
p* = c + t/n
```
**其中**：
- c：边际成本
- t：单位距离的交通成本
- n：企业数量

### 14.2.4 萨洛普均衡的性质

**萨洛普均衡的性质（Properties of Salop Equilibrium）**：

1. **价格与企业数量负相关（Price Decreases with Number of Firms）**：
   - 企业数量越多，均衡价格越低
   - 竞争越激烈，价格越低

2. **自由进入（Free Entry）**：
   - 如果企业可以自由进入和退出，则企业数量使得利润为零
   - 均衡企业数量：n = √(t/c)

3. **与同质产品比较（Comparison with Homogeneous Products）**：
   - 如果t = 0（交通成本为零），则萨洛普均衡退化为伯特兰均衡
   - 如果t > 0（交通成本为正），则萨洛普均衡价格高于边际成本

## 14.3 产品差异化定价

### 14.3.1 产品差异化定价的定义

**产品差异化定价（Product Differentiation Pricing）**：
企业根据产品差异化程度制定价格。

### 14.3.2 横向差异化

**横向差异化（Horizontal Differentiation）**：
产品在不同维度上有差异，消费者偏好不同。

**例子**：
- 颜色、口味、品牌
- 不同消费者有不同的偏好

### 14.3.3 纵向差异化

**纵向差异化（Vertical Differentiation）**：
产品在质量上有差异，所有消费者偏好高质量产品。

**例子**：
- 质量、性能、耐用性
- 所有消费者偏好高质量产品

### 14.3.4 产品差异化定价的策略

**产品差异化定价的策略（Strategies of Product Differentiation Pricing）**：

1. **品牌溢价（Brand Premium）**：
   - 知名品牌可以收取更高的价格
   - 消费者愿意为品牌支付溢价

2. **质量差异化（Quality Differentiation）**：
   - 高质量产品收取更高的价格
   - 消费者愿意为质量支付溢价

3. **创新差异化（Innovation Differentiation）**：
   - 创新产品可以收取更高的价格
   - 消费者愿意为创新支付溢价

## 14.4 广告

### 14.4.1 广告的定义

**广告（Advertising）**：
企业通过广告提高产品知名度和需求。

### 14.4.2 广告的需求效应

**广告的需求效应（Demand Effect of Advertising）**：
- 广告增加产品知名度
- 广告增加产品需求
- 广告提高产品价格

### 14.4.3 广告的最优水平

**广告的最优水平（Optimal Level of Advertising）**：
企业选择广告水平a，实现利润最大化。

**问题**：
```
max_{p, a} (p - c)·D(p, a) - a
s.t. a ≥ 0
```
**其中**：
- p：价格
- a：广告水平
- c：边际成本
- D(p, a)：需求函数，取决于价格和广告水平

**一阶条件**：
```
∂π/∂p = 0
∂π/∂a = 0
```

### 14.4.4 广告的类型

**广告的类型（Types of Advertising）**：

1. **信息性广告（Informative Advertising）**：
   - 提供产品信息
   - 降低消费者搜索成本
   - 例如：产品说明书、比较广告

2. **说服性广告（Persuasive Advertising）**：
   - 影响消费者偏好
   - 提高产品吸引力
   - 例如：品牌形象广告、情感广告

3. **竞争性广告（Competitive Advertising）**：
   - 与竞争对手比较
   - 强调产品优势
   - 例如：比较广告、攻击性广告

## 14.5 本章总结

### 核心要点

1. **霍特林模型**：
   - 霍特林模型：企业位于空间（产品空间），消费者根据距离选择企业
   - 霍特林均衡：p*₁ = p*₂ = t·(x₂ - x₁)
   - 霍特林均衡的性质：价格与距离正相关
   - 最小化差异化：企业位于中心

2. **萨洛普圆周模型**：
   - 萨洛普圆周模型：企业位于单位圆周上，消费者根据距离选择企业
   - 萨洛普均衡：p* = c + t/n
   - 萨洛普均衡的性质：价格与企业数量负相关
   - 自由进入：企业数量使得利润为零

3. **产品差异化定价**：
   - 产品差异化定价：企业根据产品差异化程度制定价格
   - 横向差异化：产品在不同维度上有差异，消费者偏好不同
   - 纵向差异化：产品在质量上有差异，所有消费者偏好高质量产品
   - 产品差异化定价的策略：品牌溢价、质量差异化、创新差异化

4. **广告**：
   - 广告：企业通过广告提高产品知名度和需求
   - 广告的需求效应：增加产品知名度、增加产品需求、提高产品价格
   - 广告的最优水平：选择广告水平a，实现利润最大化
   - 广告的类型：信息性广告、说服性广告、竞争性广告
