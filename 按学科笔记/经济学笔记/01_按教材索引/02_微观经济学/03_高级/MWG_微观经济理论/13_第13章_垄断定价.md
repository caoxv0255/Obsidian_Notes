## 13.1 垄断定价

### 13.1.1 垄断定价的定义

**垄断定价（Monopoly Pricing）**：
垄断企业选择价格，实现利润最大化。

**假设**：
1. 垄断企业选择价格p
2. 需求函数q = D(p)
3. 成本函数c(q)
4. 收益π = p·q - c(q)

### 13.1.2 垄断均衡

**垄断均衡（Monopoly Equilibrium）**：
价格p*是垄断均衡，如果：
```
π(p*) ≥ π(p), 对于所有p
```
**其中**：
- π(p) = p·D(p) - c(D(p))：价格p下的收益

**一阶条件**：
```
∂π/∂p = D(p) + p·D'(p) - c'(D(p))·D'(p) = 0
```
**等价形式**：
```
MR = MC
```
**其中**：
- MR：边际收益
- MC：边际成本

### 13.1.3 垄断均衡的性质

**垄断均衡的性质（Properties of Monopoly Equilibrium）**：

1. **存在性（Existence）**：
   - 如果需求函数D(p)是连续的、递减的，成本函数c(q)是凸的，则垄断均衡存在

2. **唯一性（Uniqueness）**：
   - 如果需求函数D(p)是严格凹的，成本函数c(q)是严格凸的，则垄断均衡唯一

3. **与完全竞争的比较（Comparison with Perfect Competition）**：
   - 垄断价格高于竞争价格
   - 垄断产量低于竞争产量
   - 垄断利润为正
   - 无谓损失（Deadweight Loss）存在

## 13.2 价格歧视

### 13.2.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
垄断企业对不同消费者收取不同价格，以增加利润。

### 13.2.2 价格歧视的类型

**价格歧视的类型（Types of Price Discrimination）**：

1. **一级价格歧视（First-Degree Price Discrimination）**：
   - 对每个消费者收取其保留价格
   - 垄断企业获得全部消费者剩余
   - 无谓损失为零

2. **二级价格歧视（Second-Degree Price Discrimination）**：
   - 设计不同价格-数量组合，让消费者自选择
   - 消费者根据消费量选择价格
   - 例如：数量折扣

3. **三级价格歧视（Third-Degree Price Discrimination）**：
   - 对不同消费者群体收取不同价格
   - 消费者群体可以根据可观察的特征区分
   - 例如：学生折扣、老年折扣

### 13.2.3 一级价格歧视

**一级价格歧视（First-Degree Price Discrimination）**：
对每个消费者收取其保留价格。

**最优定价**：
```
pᵢ = vᵢ
```
**其中**：
- vᵢ：消费者i的保留价格（支付意愿）

**收益**：
```
π = Σᵢ vᵢ - c(Σᵢ 1)
```
**其中**：
- Σᵢ vᵢ：总收益
- c(Σᵢ 1)：总成本

**性质**：
- 垄断企业获得全部消费者剩余
- 无谓损失为零
- 配置效率（Allocative Efficiency）

### 13.2.4 二级价格歧视

**二级价格歧视（Second-Degree Price Discrimination）**：
设计不同价格-数量组合，让消费者自选择。

**方法**：
1. 设计消费计划{(qᵢ, pᵢ)}ᵢ
2. 消费者根据消费量选择价格
3. 确保消费者自选择（Self-Selection）

**自选择约束（Self-Selection Constraints）**：
- 类型为θ的消费者选择为θ设计的消费计划
- 类型为θ的消费者不会选择为其他类型θ'设计的消费计划

**公式**：
```
u(θ, (q(θ), p(θ))) ≥ u(θ, (q(θ'), p(θ'))), 对于所有θ, θ'
```
**其中**：
- u(θ, (q, p))：类型为θ的消费者选择消费计划(q, p)的效用

### 13.2.5 三级价格歧视

**三级价格歧视（Third-Degree Price Discrimination）**：
对不同消费者群体收取不同价格。

**方法**：
1. 将消费者分为不同群体（例如：学生、非学生）
2. 对每个群体设定不同的价格
3. 使得每个群体的边际收益等于边际成本

**最优定价**：
```
MR₁(p₁) = MR₂(p₂) = ... = MRₙ(pₙ) = MC
```
**其中**：
- MRᵢ(pᵢ)：群体i的边际收益
- MC：边际成本

**性质**：
- 需求弹性低的群体收取高价格
- 需求弹性高的群体收取低价格
- 无谓损失高于一级价格歧视，但低于统一垄断定价

## 13.3 两部收费

### 13.3.1 两部收费的定义

**两部收费（Two-Part Tariff）**：
价格由固定费用和可变费用组成。

**公式**：
```
T(q) = F + p·q
```
**其中**：
- F：固定费用（入场费）
- p：可变费用（边际价格）
- q：消费量

### 13.3.2 两部收费的最优设计

**两部收费的最优设计（Optimal Two-Part Tariff）**：
设计者选择固定费用F和可变费用p，使得收益最大化。

**问题**：
```
max_{F, p} N·F + (p - MC)·q(p)
s.t. u(θ, q(p)) - F ≥ u(θ, 0), 对于所有θ
```
**其中**：
- N：消费者数量
- q(p)：需求函数
- MC：边际成本
- u(θ, q(p))：类型为θ的消费者消费q(p)的效用
- u(θ, 0)：类型为θ的消费者的保留效用

**解**：
1. 可变费用p = MC（边际成本定价）
2. 固定费用F = u(θ, q(MC)) - u(θ, 0)（消费者剩余）

### 13.3.3 两部收费的性质

**两部收费的性质（Properties of Two-Part Tariff）**：

1. **效率（Efficiency）**：
   - 可变费用p = MC，实现配置效率
   - 消费量达到最优水平

2. **榨取（Extraction）**：
   - 固定费用F提取全部消费者剩余
   - 垄断企业获得全部消费者剩余

3. **与一级价格歧视的比较（Comparison with First-Degree Price Discrimination）**：
   - 两部收费与一级价格歧视等价
   - 如果可以设计两部收费，则可以实现一级价格歧视

## 13.4 捆绑销售

### 13.4.1 捆绑销售的定义

**捆绑销售（Bundling）**：
将不同产品组合销售，通常以折扣价。

### 13.4.2 纯捆绑

**纯捆绑（Pure Bundling）**：
只销售产品组合，不单独销售产品。

**例子**：
- 微软Office套装（Word + Excel + PowerPoint）
- 不单独销售Word、Excel、PowerPoint

### 13.4.3 混合捆绑

**混合捆绑（Mixed Bundling）**：
既销售产品组合，也单独销售产品。

**例子**：
- 麦当劳套餐（汉堡 + 薯条 + 饮料）
- 也可以单独购买汉堡、薯条、饮料

### 13.4.4 捆绑销售的收益

**捆绑销售的收益（Benefits of Bundling）**：
- 增加垄断企业的收益
- 消费者获得更多选择
- 价格歧视（不同消费者对捆绑的评价不同）

### 13.4.5 捆绑销售的条件

**捆绑销售的条件（Conditions for Bundling）：
1. 消费者对产品的评价负相关（Negative Correlation）
2. 需求弹性不同
3. 可以防止套利（Arbitrage）

## 13.5 本章总结

### 核心要点

1. **垄断定价**：
   - 垄断定价：垄断企业选择价格，实现利润最大化
   - 垄断均衡：MR = MC
   - 垄断均衡的性质：存在性、唯一性
   - 与完全竞争的比较：垄断价格高于竞争价格，垄断产量低于竞争产量

2. **价格歧视**：
   - 价格歧视：垄断企业对不同消费者收取不同价格
   - 一级价格歧视：对每个消费者收取其保留价格
   - 二级价格歧视：设计不同价格-数量组合，让消费者自选择
   - 三级价格歧视：对不同消费者群体收取不同价格

3. **两部收费**：
   - 两部收费：T(q) = F + p·q
   - 两部收费的最优设计：p = MC，F = 消费者剩余
   - 两部收费的性质：效率、榨取
   - 与一级价格歧视的比较：两部收费与一级价格歧视等价

4. **捆绑销售**：
   - 捆绑销售：将不同产品组合销售
   - 纯捆绑：只销售产品组合
   - 混合捆绑：既销售产品组合，也单独销售产品
   - 捆绑销售的收益：增加收益、价格歧视
   - 捆绑销售的条件：评价负相关、需求弹性不同、防止套利
