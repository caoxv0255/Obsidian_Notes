# 第4章：总体选择

> 本章学习目标：
> - 理解总体消费
> - 掌握总体消费的性质
> - 学会分析总体消费的福利分析

---

## 4.1 总体消费

### 4.1.1 总体消费的定义

**总体消费（Aggregate Consumption）**：
所有消费者消费的总和。

**公式**：
```
X = Σₖ xᵏ
```
**其中**：
- X：总体消费向量
- xᵏ：消费者k的消费向量
- k = 1, ..., K：消费者数量

### 4.1.2 总体消费的约束

**总体消费的约束（Constraints on Aggregate Consumption）**：

**1. 资源约束（Resource Constraint）**：
```
Σₖ xᵏ ≤ Σⱼ yⱼ + ω
```
**其中**：
- yⱼ：企业j的产出向量
- ω：总禀赋

**2. 禀赋约束（Endowment Constraint）**：
```
Σₖ ωᵏ = ω
```
**其中**：
- ωᵏ：消费者k的禀赋
- ω：总禀赋

### 4.1.3 总体消费的可能性

**总体消费的可能性（Feasibility of Aggregate Consumption）**：
- 总体消费必须满足资源约束
- 总体消费必须是可实现的

**可行配置（Feasible Allocation）**：
配置{xᵏ, yⱼ}ᵏⱼ是可行的，如果：
1. Σₖ xᵏ ≤ Σⱼ yⱼ + ω
2. xᵏ ∈ Xᵏ（消费集）
3. yⱼ ∈ Yⱼ（生产集）

---

## 4.2 总体消费的性质

### 4.2.1 总体消费的连续性

**总体消费的连续性（Continuity of Aggregate Consumption）**：
- 如果每个消费者的需求函数是连续的，则总体消费是连续的

**定理**：
如果对于所有k，xᵏ(p, wᵏ)是连续的，则X(p, w) = Σₖ xᵏ(p, wᵏ)是连续的。

### 4.2.2 总体消费的凸性

**总体消费的凸性（Convexity of Aggregate Consumption）**：
- 如果每个消费者的需求集是凸的，则总体消费集是凸的

**定理**：
如果对于所有k，需求集Dᵏ(p, wᵏ)是凸的，则总体需求集D(p, w) = Σₖ Dᵏ(p, wᵏ)是凸的。

### 4.2.3 总体消费的单调性

**总体消费的单调性（Monotonicity of Aggregate Consumption）**：
- 如果每个消费者的需求是单调的，则总体消费是单调的

**定理**：
如果对于所有k，需求函数xᵏ(p, wᵏ)是单调的，则X(p, w) = Σₖ xᵏ(p, wᵏ)是单调的。

## 4.3 总体消费的福利分析

### 4.3.1 帕累托最优

**帕累托最优（Pareto Optimal）**：
配置{xᵏ}ᵏ是帕累托最优的，如果不存在其他配置{x'ᵏ}ᵏ，使得：
1. 对于所有k，uᵏ(x'ᵏ) ≥ uᵏ(xᵏ)
2. 存在至少一个k，使得uᵏ(x'ᵏ) > uᵏ(xᵏ)

### 4.3.2 福利函数

**福利函数（Welfare Function）**：
```
W = Σₖ αₖ·uᵏ(xᵏ)
```
**其中**：
- W：社会福利
- αₖ：消费者k的权重（αₖ ≥ 0，Σₖ αₖ = 1）
- uᵏ(xᵏ)：消费者k的效用

**功利主义福利函数（Utilitarian Welfare Function）**：
```
W = Σₖ uᵏ(xᵏ)
```
**其中**：
- αₖ = 1/K，对于所有k

**罗尔斯福利函数（Rawlsian Welfare Function）**：
```
W = minₖ uᵏ(xᵏ)
```
**其中**：
- 关注最贫困的消费者

### 4.3.3 福利最大化

**福利最大化问题（Welfare Maximization Problem）**：
```
max W = Σₖ αₖ·uᵏ(xᵏ)
s.t. Σₖ xᵏ ≤ Σⱼ yⱼ + ω
xᵏ ∈ Xᵏ
```

**一阶条件（First-Order Conditions）**：
对于内点解：
```
αₖ·∇uᵏ(xᵏ) = λ, 对于所有k
```
**其中**：
- λ：拉格朗日乘子
- ∇uᵏ(xᵏ)：消费者k的边际效用向量

**含义**：
- 所有消费者的加权边际效用相等
- 等于商品的影子价格λ

## 4.4 社会选择理论

### 4.4.1 社会福利函数

**社会福利函数（Social Welfare Function, SWF）**：
将个体效用映射为社会福利的函数。

**公式**：
```
W = W(u¹, u², ..., uᴷ)
```
**其中**：
- W：社会福利
- uᵏ：消费者k的效用

### 4.4.2 阿罗不可能定理

**阿罗不可能定理（Arrow's Impossibility Theorem）**：

**定理**：
不存在同时满足以下四个条件的社会福利函数：
1. **无限制域（Unrestricted Domain）**：定义在所有可能的个体偏好上
2. **帕累托原则（Pareto Principle）**：如果所有人都偏好x > y，则社会偏好x > y
3. **无关选项的独立性（Independence of Irrelevant Alternatives, IIA）**：社会对x和y的偏好只取决于个体对x和y的偏好
4. **非独裁性（Non-dictatorship）**：不存在个人独裁者，其偏好决定社会偏好

### 4.4.3 社会福利函数的类型

**社会福利函数的类型（Types of Social Welfare Functions）**：

1. **功利主义社会福利函数（Utilitarian Social Welfare Function）**：
   ```
   W = Σₖ uᵏ
   ```
   - 简单加总
   - 不关注分配

2. **罗尔斯社会福利函数（Rawlsian Social Welfare Function）**：
   ```
   W = minₖ uᵏ
   ```
   - 关注最贫困者
   - 关注分配

3. **伯格森-萨缪尔森社会福利函数（Bergson-Samuelson Social Welfare Function）**：
   ```
   W = W(u¹, u², ..., uᴷ)
   ```
   - 一般形式
   - 可以关注分配

## 4.5 不平等度量

### 4.5.1 不平等的度量

**不平等的度量（Measurement of Inequality）**：

1. **基尼系数（Gini Coefficient）**：
   ```
   G = (1 / (2n²μ)) Σᵢ Σⱼ |xᵢ - xⱼ|
   ```
   **其中**：
   - G：基尼系数
   - n：人口数量
   - μ：平均收入
   - xᵢ：个体i的收入

   - G ∈ [0, 1]
   - G = 0：完全平等
   - G = 1：完全不平等

2. **泰尔指数（Theil Index）**：
   ```
   T = (1 / n) Σᵢ (xᵢ / μ)·ln(xᵢ / μ)
   ```
   **其中**：
   - T：泰尔指数
   - xᵢ：个体i的收入
   - μ：平均收入

   - T ∈ [0, ln(n)]
   - T = 0：完全平等
   - T = ln(n)：完全不平等

3. **方差（Variance）**：
   ```
   Var(x) = (1 / n) Σᵢ (xᵢ - μ)²
   ```
   **其中**：
   - Var(x)：方差
   - xᵢ：个体i的收入
   - μ：平均收入

### 4.5.2 不平等的分解

**不平等的分解（Decomposition of Inequality）**：
- 按组分解（Between-Group Inequality）
- 组内分解（Within-Group Inequality）

**泰尔指数的分解（Theil Index Decomposition）**：
```
T = T_B + T_W
```
**其中**：
- T：总体不平等
- T_B：组间不平等
- T_W：组内不平等

## 4.6 本章总结

### 核心要点

1. **总体消费**：
   - 总体消费：X = Σₖ xᵏ
   - 总体消费的约束：资源约束、禀赋约束
   - 可行配置：Σₖ xᵏ ≤ Σⱼ yⱼ + ω

2. **总体消费的性质**：
   - 连续性：如果每个消费者的需求函数是连续的，则总体消费是连续的
   - 凸性：如果每个消费者的需求集是凸的，则总体消费集是凸的
   - 单调性：如果每个消费者的需求是单调的，则总体消费是单调的

3. **总体消费的福利分析**：
   - 帕累托最优：不存在其他配置使所有人至少不差，且至少一人更好
   - 福利函数：W = Σₖ αₖ·uᵏ(xᵏ)
   - 福利最大化：αₖ·∇uᵏ(xᵏ) = λ

4. **社会选择理论**：
   - 社会福利函数：W = W(u¹, u², ..., uᴷ)
   - 阿罗不可能定理：不存在满足四个条件的社会福利函数
   - 社会福利函数的类型：功利主义、罗尔斯、伯格森-萨缪尔森

5. **不平等度量**：
   - 基尼系数：G = (1 / (2n²μ)) Σᵢ Σⱼ |xᵢ - xⱼ|
   - 泰尔指数：T = (1 / n) Σᵢ (xᵢ / μ)·ln(xᵢ / μ)
   - 方差：Var(x) = (1 / n) Σᵢ (xᵢ - μ)²
   - 不平等的分解：T = T_B + T_W
