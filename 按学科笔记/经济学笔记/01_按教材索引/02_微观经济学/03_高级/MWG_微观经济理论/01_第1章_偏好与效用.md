# 第1章：偏好与效用

> 本章学习目标：
> - 理解偏好关系
> - 掌握效用函数

---

## 1.1 偏好关系

### 1.1.1 消费集

**消费集（Consumption Set, X）**：
消费者可以选择的所有消费束的集合。

**性质**：
- 非空：X ≠ ∅
- 凸性：对于任意x, y ∈ X，对于所有λ ∈ [0,1]，λx + (1-λ)y ∈ X
- 闭集：X是闭集
- 下有界：X下有界

### 1.1.2 偏好关系的定义

**偏好关系（Preference Relation, ≿）**：
消费集X上的二元关系，表示消费者对消费束的偏好。

**性质**：
1. **完全性（Completeness）**：
   - 对于任意x, y ∈ X，要么x ≿ y，要么y ≿ x，或者两者都成立

2. **传递性（Transitivity）**：
   - 如果x ≿ y且y ≿ z，则x ≿ z

3. **理性（Rationality）**：
   - 偏好关系同时满足完全性和传递性

### 1.1.3 无差异关系

**无差异关系（Indifference Relation, ~）**：
x ~ y 当且仅当 x ≿ y 且 y ≿ x

**严格偏好（Strict Preference, ≻）**：
x ≻ y 当且仅当 x ≿ y 且 y ≿/ x

### 1.1.4 偏好的性质

**偏好的性质（Properties of Preferences）**：

**1. 单调性（Monotonicity）**：
- **弱单调性（Weak Monotonicity）**：
  - 如果x ≥ y，则x ≿ y
- **强单调性（Strong Monotonicity）**：
  - 如果x ≥ y且x ≠ y，则x ≻ y

**2. 凸性（Convexity）**：
- **凸偏好（Convex Preferences）**：
  - 如果x ≿ y，则对于所有λ ∈ [0,1]，λx + (1-λ)y ≿ y
- **严格凸偏好（Strictly Convex Preferences）**：
  - 如果x ≿ y且x ≠ y，则对于所有λ ∈ (0,1)，λx + (1-λ)y ≻ y

**3. 连续性（Continuity）**：
- 偏好关系是连续的，如果对于任意序列{xⁿ}收敛于x，{yⁿ}收敛于y，且xⁿ ≿ yⁿ对所有n成立，则x ≿ y

## 1.2 效用函数

### 1.2.1 效用函数的定义

**效用函数（Utility Function, u: X → ℝ）**：
表示偏好关系的数值函数。

**性质**：
- 对于任意x, y ∈ X，x ≿ y 当且仅当 u(x) ≥ u(y)

### 1.2.2 效用函数的存在性

**效用函数的存在性（Existence of Utility Function）**：

**定理**：
如果偏好关系≿是理性的、连续的，且消费集X是连续的，则存在连续的效用函数u: X → ℝ表示偏好关系≿。

**证明思路**：
1. 使用德布鲁定理
2. 构造效用函数
3. 证明连续性

### 1.2.3 效用函数的唯一性

**效用函数的唯一性（Uniqueness of Utility Function）**：
效用函数在正单调变换下唯一。

**定义**：
如果效用函数u(x)表示偏好关系≿，则对于任意严格递增函数φ: ℝ → ℝ，φ(u(x))也表示偏好关系≿。

### 1.2.4 拟凹性与凹性

**拟凹性（Quasiconcavity）**：
效用函数u(x)是拟凹的，如果集合{x ∈ X: u(x) ≥ t}对于所有t都是凸的。

**凹性（Concavity）**：
效用函数u(x)是凹的，如果对于所有x, y ∈ X和λ ∈ [0,1]，有：
u(λx + (1-λ)y) ≥ λu(x) + (1-λ)u(y)

**关系**：
- 凹性 ⇒ 拟凹性
- 拟凹性 ⇔ 凸偏好

## 1.3 效用函数的性质

### 1.3.1 边际效用

**边际效用（Marginal Utility, MUᵢ）**：
效用函数对第i种商品消费量的偏导数。

**公式**：
```
MUᵢ = ∂u(x)/∂xᵢ
```

**性质**：
- 正单调性 ⇒ MUᵢ > 0
- 边际效用递减：∂MUᵢ/∂xᵢ < 0

### 1.3.2 边际替代率

**边际替代率（Marginal Rate of Substitution, MRSᵢⱼ）**：
消费者愿意用第j种商品替代第i种商品的比率。

**公式**：
```
MRSᵢⱼ = MUᵢ / MUⱼ = (∂u(x)/∂xᵢ) / (∂u(x)/∂xⱼ)
```

**性质**：
- 凸偏好 ⇒ 边际替代率递减
- 沿着无差异曲线，边际替代率等于无差异曲线的斜率的负值

### 1.3.3 效用最大化

**效用最大化问题（Utility Maximization Problem, UMP）**：
```
max u(x)
s.t. p·x ≤ w
x ∈ X
```
**其中**：
- u(x)：效用函数
- p：价格向量
- w：财富
- X：消费集

**一阶条件（First-Order Conditions, FOCs）**：
对于内点解：
```
MUᵢ / pᵢ = μ, 对于所有i
```
**其中**：
- μ：拉格朗日乘子
- λᵢ = MUᵢ / pᵢ：边际效用与价格之比

**含义**：
- 对于所有商品，边际效用与价格之比相等
- 所有商品的边际效用/价格比等于拉格朗日乘子μ

## 1.4 期望效用理论

### 1.4.1 不确定性下的偏好

**不确定性下的偏好（Preferences under Uncertainty）**：
- 消费束是随机变量
- 偏好关系≿定义在彩票（Lotteries）上

**彩票（Lottery）**：
L = {(p₁, x₁), (p₂, x₂), ..., (pₙ, xₙ)}
- pᵢ：结果xᵢ发生的概率
- Σᵢ pᵢ = 1

### 1.4.2 冯·诺伊曼-摩根斯坦效用函数

**冯·诺伊曼-摩根斯坦效用函数（Von Neumann-Morgenstern Utility Function, VNM Utility Function）**：
表示不确定性下偏好的效用函数。

**公式**：
```
U(L) = Σᵢ pᵢ·u(xᵢ)
```
**其中**：
- U(L)：彩票L的期望效用
- u(xᵢ)：结果xᵢ的效用（伯努利效用函数）
- pᵢ：结果xᵢ发生的概率

### 1.4.3 风险态度

**风险态度（Risk Attitude）**：
- **风险厌恶（Risk Aversion）**：
  - u(·)是凹函数
  - 对于所有L，E[u(x)] ≤ u(E[x])
- **风险中性（Risk Neutrality）**：
  - u(·)是线性函数
  - 对于所有L，E[u(x)] = u(E[x])
- **风险偏好（Risk Loving）**：
  - u(·)是凸函数
  - 对于所有L，E[u(x)] ≥ u(E[x])

**阿罗-普拉特绝对风险厌恶系数（Arrow-Pratt Measure of Absolute Risk Aversion, ARA）**：
```
A(x) = -u''(x) / u'(x)
```
- A(x) > 0 ⇒ 风险厌恶
- A(x) = 0 ⇒ 风险中性
- A(x) < 0 ⇒ 风险偏好

**阿罗-普拉特相对风险厌恶系数（Arrow-Pratt Measure of Relative Risk Aversion, RRA）**：
```
R(x) = -x·u''(x) / u'(x) = x·A(x)
```

### 1.4.4 期望效用定理

**期望效用定理（Expected Utility Theorem）**：

**定理**：
如果偏好关系≿满足：
1. 完全性和传递性
2. 连续性
3. 独立性（如果L ≿ M，则对于任意L'和α ∈ [0,1]，αL + (1-α)L' ≿ αM + (1-α)L'）

则存在冯·诺伊曼-摩根斯坦效用函数U(·)表示偏好关系≿。

**独立性（Independence）**：
- 是期望效用理论的关键假设
- 意味着偏好不依赖于无关选项

## 1.5 本章总结

### 核心要点

1. **偏好关系**：
   - 偏好关系：消费集上的二元关系
   - 偏好的性质：完全性、传递性、理性、单调性、凸性、连续性
   - 无差异关系：x ~ y 当且仅当 x ≿ y 且 y ≿ x

2. **效用函数**：
   - 效用函数：表示偏好关系的数值函数
   - 效用函数的存在性：理性、连续的偏好关系存在连续的效用函数
   - 效用函数的唯一性：在正单调变换下唯一
   - 拟凹性与凹性：凹性 ⇒ 拟凹性

3. **效用函数的性质**：
   - 边际效用：MUᵢ = ∂u(x)/∂xᵢ
   - 边际替代率：MRSᵢⱼ = MUᵢ / MUⱼ
   - 效用最大化：MUᵢ / pᵢ = μ

4. **期望效用理论**：
   - 冯·诺伊曼-摩根斯坦效用函数：U(L) = Σᵢ pᵢ·u(xᵢ)
   - 风险态度：风险厌恶、风险中性、风险偏好
   - 阿罗-普拉特风险厌恶系数：A(x) = -u''(x) / u'(x)
   - 期望效用定理：独立性假设
