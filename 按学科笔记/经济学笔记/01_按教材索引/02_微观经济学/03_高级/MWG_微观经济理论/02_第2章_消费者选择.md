## 2.1 消费者选择问题

### 2.1.1 效用最大化问题

**效用最大化问题（Utility Maximization Problem, UMP）**：
```
max u(x)
s.t. p·x ≤ w
x ∈ X
```
**其中**：
- u(x)：效用函数
- p：价格向量（p ≫ 0）
- w：财富（w > 0）
- X：消费集（通常为ℝ₊ᴸ）

**瓦尔拉斯需求对应（Walrasian Demand Correspondence, x(p, w)）**：
UMP的解集。

**瓦尔拉斯需求函数（Walrasian Demand Function, x(p, w)）**：
如果UMP的解唯一，则x(p, w)是函数。

### 2.1.2 效用最大化的一阶条件

**一阶条件（First-Order Conditions, FOCs）**：

对于内点解，构建拉格朗日函数：
```
ℒ(x, λ) = u(x) - λ(p·x - w)
```

**一阶条件**：
```
∂ℒ/∂xᵢ = ∂u(x)/∂xᵢ - λpᵢ = 0, 对于所有i
∂ℒ/∂λ = -(p·x - w) = 0
```

**等价形式**：
```
MUᵢ / pᵢ = λ, 对于所有i
p·x = w
```
**其中**：
- λ：拉格朗日乘子（财富的边际效用）
- MUᵢ：边际效用

**含义**：
- 对于所有商品，边际效用与价格之比相等
- 等于财富的边际效用λ

### 2.1.3 效用最大化的二阶条件

**二阶条件（Second-Order Conditions, SOCs）**：
海森矩阵负定：
```
H = [∂²u/∂xᵢ∂xⱼ] 是负定的
```

**充分条件**：
- 如果效用函数u(x)是严格拟凹的，则UMP的解唯一

### 2.1.4 效用最大化的解的存在性和唯一性

**存在性（Existence）**：
- 如果效用函数u(x)是连续的，消费集X是紧集，则UMP的解存在

**唯一性（Uniqueness）**：
- 如果效用函数u(x)是严格拟凹的，则UMP的解唯一

## 2.2 对偶性

### 2.2.1 支出最小化问题

**支出最小化问题（Expenditure Minimization Problem, EMP）**：
```
min p·x
s.t. u(x) ≥ u
x ∈ X
```
**其中**：
- p：价格向量（p ≫ 0）
- u：效用水平
- X：消费集

**希克斯需求对应（Hicksian Demand Correspondence, h(p, u)）**：
EMP的解集。

**希克斯需求函数（Hicksian Demand Function, h(p, u)）**：
如果EMP的解唯一，则h(p, u)是函数。

### 2.2.2 支出函数

**支出函数（Expenditure Function, e(p, u)）**：
EMP的最小支出。

**公式**：
```
e(p, u) = min{p·x: u(x) ≥ u, x ∈ X}
```

**性质**：
1. **齐次性（Homogeneity）**：
   - e(αp, u) = α·e(p, u)，对于所有α > 0

2. **单调性（Monotonicity）**：
   - 如果p ≥ p'，则e(p, u) ≥ e(p', u)
   - 如果u ≥ u'，则e(p, u) ≥ e(p, u')

3. **凹性（Concavity）**：
   - e(p, u)关于p是凹函数

4. **连续性（Continuity）**：
   - e(p, u)关于p和u是连续的

5. **谢泼德引理（Shephard's Lemma）**：
   - hᵢ(p, u) = ∂e(p, u)/∂pᵢ

### 2.2.3 对偶性定理

**对偶性定理（Duality Theorem）**：

**定理1**：
如果x*是UMP的解，效用为u* = u(x*)，则x*是EMP的解，效用为u*。

**定理2**：
如果x*是EMP的解，支出为e* = p·x*，则x*是UMP的解，财富为w* = e*。

**对偶性关系（Duality Relations）**：
```
x(p, w) = h(p, v(p, w))
h(p, u) = x(p, e(p, u))
v(p, w) = max{u: e(p, u) ≤ w}
e(p, u) = min{w: v(p, w) ≥ u}
```
**其中**：
- v(p, w)：间接效用函数
- e(p, u)：支出函数

## 2.3 间接效用函数

### 2.3.1 间接效用函数的定义

**间接效用函数（Indirect Utility Function, v(p, w)）**：
UMP的最大效用。

**公式**：
```
v(p, w) = max{u(x): p·x ≤ w, x ∈ X}
```

### 2.3.2 间接效用函数的性质

**间接效用函数的性质（Properties of Indirect Utility Function）**：

1. **齐次性（Homogeneity）**：
   - v(αp, αw) = v(p, w)，对于所有α > 0

2. **单调性（Monotonicity）**：
   - 如果p ≥ p'，则v(p, w) ≤ v(p', w)
   - 如果w ≥ w'，则v(p, w) ≥ v(p, w')

3. **拟凸性（Quasiconvexity）**：
   - v(p, w)关于(p, w)是拟凸函数

4. **连续性（Continuity）**：
   - v(p, w)关于p和w是连续的

5. **罗伊恒等式（Roy's Identity）**：
   - xᵢ(p, w) = -[∂v(p, w)/∂pᵢ] / [∂v(p, w)/∂w]

### 2.3.3 罗伊恒等式

**罗伊恒等式（Roy's Identity）**：
从间接效用函数推导瓦尔拉斯需求函数。

**公式**：
```
xᵢ(p, w) = -[∂v(p, w)/∂pᵢ] / [∂v(p, w)/∂w]
```

**证明思路**：
1. 根据包络定理
2. 利用间接效用函数的性质
3. 推导出需求函数

## 2.4 需求函数的性质

### 2.4.1 瓦尔拉斯需求的性质

**瓦尔拉斯需求的性质（Properties of Walrasian Demand）**：

1. **齐次性（Homogeneity of Degree Zero）**：
   - x(αp, αw) = x(p, w)，对于所有α > 0
   - 意味着需求不受价格和财富同时同比例变化的影响

2. **瓦尔拉斯法则（Walras' Law）**：
   - p·x(p, w) = w
   - 消费者花光所有财富

3. **连续性（Continuity）**：
   - 如果偏好是严格凸的，则x(p, w)是连续的

4. **凸性/单值性（Convexity/Single-Valuedness）**：
   - 如果偏好是严格凸的，则x(p, w)是单值的

### 2.4.2 希克斯需求的性质

**希克斯需求的性质（Properties of Hicksian Demand）**：

1. **齐次性（Homogeneity of Degree Zero）**：
   - h(αp, u) = h(p, u)，对于所有α > 0

2. **无差异曲线性（Indifference Curve Property）**：
   - u(h(p, u)) = u
   - 消费束位于同一条无差异曲线上

3. **替代矩阵的对称性和负定性（Symmetry and Negative Semidefiniteness of Substitution Matrix）**：
   - S(p, u) = [∂hᵢ(p, u)/∂pⱼ] 是对称的和负半定的
   - S(p, u) = ∂h(p, u)/∂p = ∂²e(p, u)/∂p²

4. **补偿需求定律（Law of Compensated Demand）**：
   - ∂hᵢ(p, u)/∂pᵢ ≤ 0
   - 希克斯需求是价格的减函数

## 2.5 斯卢茨基方程

### 2.5.1 斯卢茨基方程的定义

**斯卢茨基方程（Slutsky Equation）**：
描述瓦尔拉斯需求对价格变化的分解。

**公式**：
```
∂xᵢ(p, w)/∂pⱼ = ∂hᵢ(p, u)/∂pⱼ - xⱼ(p, w)·[∂xᵢ(p, w)/∂w]
```
**其中**：
- ∂xᵢ(p, w)/∂pⱼ：瓦尔拉斯需求对价格的总效应
- ∂hᵢ(p, u)/∂pⱼ：替代效应（Substitution Effect）
- -xⱼ(p, w)·[∂xᵢ(p, w)/∂w]：收入效应（Income Effect）

### 2.5.2 替代效应与收入效应

**替代效应（Substitution Effect）**：
```
SEᵢⱼ = ∂hᵢ(p, u)/∂pⱼ = ∂xᵢ(p, w)/∂pⱼ + xⱼ(p, w)·[∂xᵢ(p, w)/∂w]
```
- 保持效用不变，价格变化对需求的影响
- 符号：SEᵢⱼ ≤ 0（当i = j时）
- 替代矩阵S(p, u)是对称的和负半定的

**收入效应（Income Effect）**：
```
IEᵢⱼ = -xⱼ(p, w)·[∂xᵢ(p, w)/∂w]
```
- 价格变化通过实际收入变化对需求的影响
- 符号取决于商品的性质（正常商品、劣等商品）

### 2.5.3 斯卢茨基方程的性质

**斯卢茨基方程的性质（Properties of Slutsky Equation）**：

1. **对称性（Symmetry）**：
   - ∂xᵢ/∂pⱼ + xⱼ·(∂xᵢ/∂w) = ∂xⱼ/∂pᵢ + xᵢ·(∂xⱼ/∂w)
   - 替代效应是对称的

2. **负定性（Negative Semidefiniteness）**：
   - 替代矩阵S(p, u)是负半定的
   - 意味着自价格效应为负（吉芬商品除外）

3. **齐次性（Homogeneity）**：
   - Σⱼ pⱼ·(∂xᵢ/∂pⱼ) + w·(∂xᵢ/∂w) = 0
   - 需求是零次齐次的

4. **加总性（Aggregation）**：
   - Σᵢ pᵢ·(∂xᵢ/∂pⱼ) = -xⱼ
   - Σᵢ pᵢ·(∂xᵢ/∂w) = 1

### 2.5.4 正常商品与劣等商品

**正常商品（Normal Goods）**：
- ∂xᵢ(p, w)/∂w > 0
- 收入增加，需求增加

**劣等商品（Inferior Goods）**：
- ∂xᵢ(p, w)/∂w < 0
- 收入增加，需求减少

**吉芬商品（Giffen Goods）**：
- 劣等商品
- 收入效应的绝对值大于替代效应
- ∂xᵢ(p, w)/∂pᵢ > 0
- 价格上升，需求增加

## 2.6 需求弹性

### 2.6.1 价格弹性

**价格弹性（Price Elasticity）**：
```
εᵢⱼ = (∂xᵢ/∂pⱼ)·(pⱼ/xᵢ)
```
- εᵢᵢ：自价格弹性（Own-Price Elasticity）
- εᵢⱼ（i ≠ j）：交叉价格弹性（Cross-Price Elasticity）

**性质**：
- εᵢᵢ < 0：正常情况
- εᵢᵢ > 0：吉芬商品
- |εᵢᵢ| > 1：富有弹性
- |εᵢᵢ| < 1：缺乏弹性

### 2.6.2 收入弹性

**收入弹性（Income Elasticity）**：
```
εᵢᵂ = (∂xᵢ/∂w)·(w/xᵢ)
```

**性质**：
- εᵢᵂ > 0：正常商品
- εᵢᵂ < 0：劣等商品
- εᵢᵂ > 1：奢侈品（Luxury Goods）
- 0 < εᵢᵂ < 1：必需品（Necessity Goods）

### 2.6.3 弹性之间的关系

**弹性之间的关系（Relations among Elasticities）**：

1. **恩格尔加总（Engel Aggregation）**：
   - Σᵢ wᵢ·εᵢᵂ = 1，其中wᵢ = pᵢ·xᵢ/w

2. **库尔诺加总（Cournot Aggregation）**：
   - Σᵢ wᵢ·εᵢⱼ = -wⱼ/w

3. **齐次性（Homogeneity）**：
   - Σⱼ εᵢⱼ + εᵢᵂ = 0

## 2.7 本章总结

### 核心要点

1. **消费者选择问题**：
   - 效用最大化问题（UMP）：max u(x) s.t. p·x ≤ w
   - 一阶条件：MUᵢ / pᵢ = λ
   - 二阶条件：海森矩阵负定
   - 解的存在性和唯一性

2. **对偶性**：
   - 支出最小化问题（EMP）：min p·x s.t. u(x) ≥ u
   - 支出函数：e(p, u)
   - 对偶性定理：UMP和EMP的解对偶
   - 对偶性关系：x(p, w) = h(p, v(p, w))

3. **间接效用函数**：
   - 间接效用函数：v(p, w)
   - 性质：齐次性、单调性、拟凸性、连续性
   - 罗伊恒等式：xᵢ(p, w) = -[∂v(p, w)/∂pᵢ] / [∂v(p, w)/∂w]

4. **需求函数的性质**：
   - 瓦尔拉斯需求：齐次性、瓦尔拉斯法则、连续性、凸性
   - 希克斯需求：齐次性、无差异曲线性、替代矩阵的对称性和负定性、补偿需求定律

5. **斯卢茨基方程**：
   - 斯卢茨基方程：∂xᵢ/∂pⱼ = ∂hᵢ/∂pⱼ - xⱼ·(∂xᵢ/∂w)
   - 替代效应与收入效应
   - 正常商品与劣等商品

6. **需求弹性**：
   - 价格弹性：εᵢⱼ = (∂xᵢ/∂pⱼ)·(pⱼ/xᵢ)
   - 收入弹性：εᵢᵂ = (∂xᵢ/∂w)·(w/xᵢ)
   - 弹性之间的关系
