## 8.1 策略型博弈

### 8.1.1 基本定义

**定义8.1.1（策略型博弈）**
策略型博弈（Strategic-form Game）或标准型博弈（Normal-form Game）由以下要素组成：

```
G = {N, {Sᵢ}, {uᵢ}}
```

其中：
- N = {1, 2, ..., n} 是参与者集合
- Sᵢ 是参与者 i 的策略集
- uᵢ: S × S × ... × S → ℝ 是参与者 i 的支付函数

**策略组合**（Strategy Profile）：s = (s₁, s₂, ..., sₙ) ∈ S₁ × S₂ × ... × Sₙ

**记号约定**：
- sᵢ：参与者 i 的策略
- s₋ᵢ：除 i 外所有参与者的策略组合，即 s₋ᵢ = (s₁, ..., sᵢ₋₁, sᵢ₊₁, ..., sₙ)
- s = (sᵢ, s₋ᵢ)：完整的策略组合

**例子8.1.1（囚徒困境）**
两个嫌疑人被捕，检察官分别审讯。

```
               囚徒2
            坦白      沉默
坦白     (-5,-5)   (0,-10)
囚徒1
沉默    (-10,0)   (-1,-1)
```

- 如果两人都坦白：各判5年
- 如果一人坦白，一人沉默：坦白者释放，沉默者判10年
- 如果两人都沉默：各判1年

**分析**：
- 囚徒1的最佳响应：
  - 如果囚徒2坦白：坦白（-5）> 沉默（-10）
  - 如果囚徒2沉默：坦白（0）> 沉默（-1）
  - 无论囚徒2如何，坦白都是最佳响应

- 囚徒1和囚徒2面临相同结构
- (坦白, 坦白) 是纳什均衡
- 但 (沉默, 沉默) 帕累托优于 (坦白, 坦白)

**例子8.1.2（性别战）**
一男一女决定约会活动，偏好略有不同。

```
               女
            足球      歌剧
足球     (2,1)   (0,0)
男
歌剧     (0,0)   (1,2)
```

- 两个纳什均衡：(足球, 足球) 和 (歌剧, 歌剧)
- 协调问题：需要事先沟通或约定

### 8.1.2 优势策略

**定义8.1.2（严格优势策略）**
策略 sᵢ' 是参与者 i 的严格优势策略（Strictly Dominant Strategy），如果对于所有 s₋ᵢ ∈ S₋ᵢ：

```
uᵢ(sᵢ', s₋ᵢ) > uᵢ(sᵢ, s₋ᵢ), ∀sᵢ ≠ sᵢ'
```

**定义8.1.3（弱优势策略）**
策略 sᵢ' 是参与者 i 的弱优势策略（Weakly Dominant Strategy），如果对于所有 s₋ᵢ ∈ S₋ᵢ：

```
uᵢ(sᵢ', s₋ᵢ) ≥ uᵢ(sᵢ, s₋ᵢ), ∀sᵢ ≠ sᵢ'
```

且至少存在一个 s₋ᵢ 使得不等式严格成立。

**定理8.1.1（优势策略均衡）**
如果每个参与者都有严格优势策略，则该策略组合构成唯一的纳什均衡。

**推论**：囚徒困境中，"坦白"是每个参与者的严格优势策略，因此 (坦白, 坦白) 是唯一均衡。

**优势策略的迭代剔除（Iterated Elimination of Dominated Strategies, IEDS）**：

**定义8.1.4（严格被占优策略）**
策略 sᵢ 被严格占优，如果存在策略 sᵢ' 使得对于所有 s₋ᵢ：

```
uᵢ(sᵢ', s₋ᵢ) > uᵢ(sᵢ, s₋ᵢ)
```

**IEDS过程**：
1. 剔除所有被严格占优的策略
2. 在剩余的策略集中，继续剔除被严格占优的策略
3. 重复直到无法再剔除

**定理8.1.2（IEDS与纳什均衡）**
在有限博弈中，如果迭代剔除严格被占优策略后只剩下一个策略组合，则该组合是唯一的纳什均衡。

**例子8.1.3（IEDS的应用）**

```
                   参与者2
                L      M      R
U      (1,0)  (1,2)  (0,1)
参与者1  D      (0,3)  (0,1)  (2,0)
```

- 参与者2：R 被 M 严格占优（1>0, 2>1）
- 剔除 R 后：
  - 参与者1：D 被 U 严格占优（1>0, 1>0）
- 剔除 D 后：
  - 参与者2：M 被 L 严格占优（0>1）
- 剩余策略：(U, L)

### 8.1.3 最佳响应

**定义8.1.5（最佳响应）**
策略 sᵢ* 是参与者 i 对 s₋ᵢ 的最佳响应（Best Response），如果：

```
uᵢ(sᵢ*, s₋ᵢ) ≥ uᵢ(sᵢ, s₋ᵢ), ∀sᵢ ∈ Sᵢ
```

最佳响应集（Best Response Correspondence）：
```
BRᵢ(s₋ᵢ) = {sᵢ ∈ Sᵢ : uᵢ(sᵢ, s₋ᵢ) ≥ uᵢ(sᵢ', s₋ᵢ), ∀sᵢ' ∈ Sᵢ}
```

**定义8.1.6（纯策略纳什均衡）**
策略组合 s* = (s₁*, s₂*, ..., sₙ*) 是纯策略纳什均衡（Pure Strategy Nash Equilibrium），如果对于所有 i：

```
sᵢ* ∈ BRᵢ(s₋ᵢ*)
```

即每个人的策略都是对其他人策略的最佳响应。

**定理8.1.3（纳什均衡的定义等价性）**
以下定义等价：
1. s* 是纳什均衡
2. 没有人有单方面偏离的动机
3. 对于所有 i，sᵢ* ∈ BRᵢ(s₋ᵢ*)

**寻找纳什均衡的方法**：

**方法1：最佳响应分析**
- 计算每个参与者的最佳响应函数
- 找到最佳响应函数的交点

**例子8.1.4（库诺特双寡头）**
两个企业同时决定产量 q₁, q₂ ≥ 0。

反需求函数：P(q₁ + q₂) = a - (q₁ + q₂)
成本函数：Cᵢ(qᵢ) = cqᵢ, 0 < c < a

企业 i 的利润：
```
πᵢ(q₁, q₂) = (a - q₁ - q₂ - c)qᵢ
```

企业1的最佳响应：
```
max π₁(q₁, q₂) = (a - q₁ - q₂ - c)q₁
```

FOC：∂π₁/∂q₁ = a - 2q₁ - q₂ - c = 0
```
q₁ = (a - c - q₂)/2
```

企业2的最佳响应：
```
q₂ = (a - c - q₁)/2
```

联立求解：
```
q₁ = (a - c)/3
q₂ = (a - c)/3
```

纳什均衡：q* = ((a-c)/3, (a-c)/3)

**方法2：检查所有策略组合**
对于有限博弈，逐一检查每个策略组合是否满足纳什均衡条件。

**例子8.1.5（石头剪刀布）**

```

参与者1  剪刀   (-1,1) (0,0)   (1,-1)
布       (1,-1) (-1,1)  (0,0)
```

没有纯策略纳什均衡（需要混合策略）。

### 8.1.4 混合策略纳什均衡

**定义8.1.7（混合策略）**
参与者 i 的混合策略 σᵢ 是纯策略集 Sᵢ 上的概率分布：
```
σᵢ: Sᵢ → [0, 1], Σ_{sᵢ∈Sᵢ} σᵢ(sᵢ) = 1
```

**记号**：
- Δ(Sᵢ)：参与者 i 的混合策略空间
- σ = (σ₁, ..., σₙ)：混合策略组合
- σᵢ(sᵢ)：混合策略 σᵢ 对纯策略 sᵢ 赋予的概率

**定义8.1.8（混合策略下的期望支付）**
给定混合策略组合 σ，参与者 i 的期望支付：
```
uᵢ(σ) = Σ_{s∈S} [Πⱼ σⱼ(sⱼ)]·uᵢ(s)
```

其中 s = (s₁, ..., sₙ) 是纯策略组合。

**定义8.1.9（混合策略纳什均衡）**
混合策略组合 σ* = (σ₁*, ..., σₙ*) 是纳什均衡，如果对于所有 i：

```
uᵢ(σᵢ*, σ₋ᵢ*) ≥ uᵢ(σᵢ, σ₋ᵢ*), ∀σᵢ ∈ Δ(Sᵢ)
```

**定理8.1.4（混合策略纳什均衡的必要条件）**
σ* 是纳什均衡的充要条件是：对于每个参与者 i，所有被 σᵢ* 赋予正概率的纯策略都产生相同的期望支付，且不小于任何其他纯策略的期望支付。

**证明**：
设 σ* 是纳什均衡。对于参与者 i：

令 Sᵢ⁺ = {sᵢ ∈ Sᵢ : σᵢ*(sᵢ) > 0} 是被赋予正概率的纯策略集合。

如果存在 sᵢ' ∈ Sᵢ⁺ 和 sᵢ'' ∈ Sᵢ 使得 uᵢ(sᵢ', σ₋ᵢ*) < uᵢ(sᵢ'', σ₋ᵢ*)，则参与者 i 可以将 sᵢ' 的概率转移到 sᵢ''，提高期望支付，这与纳什均衡矛盾。

因此：uᵢ(sᵢ, σ₋ᵢ*) = uᵢ(σᵢ*, σ₋ᵢ*), ∀sᵢ ∈ Sᵢ⁺

且对于任何 sᵢ ∉ Sᵢ⁺：uᵢ(sᵢ, σ₋ᵢ*) ≤ uᵢ(σᵢ*, σ₋ᵢ*)

证毕。

**例子8.1.6（石头剪刀布的混合策略均衡）**
设参与者1以概率 (p₁, p₂, p₃) 选择（石头, 剪刀, 布）
设参与者2以概率 (q₁, q₂, q₃) 选择（石头, 剪刀, 布）

参与者1的期望支付（针对参与者2的策略）：
```
u₁(石头, σ₂) = 0·q₁ + 1·q₂ + (-1)·q₃ = q₂ - q₃
u₁(剪刀, σ₂) = -1·q₁ + 0·q₂ + 1·q₃ = -q₁ + q₃
u₁(布, σ₂) = 1·q₁ + (-1)·q₂ + 0·q₃ = q₁ - q₂
```

在均衡中，所有被选中的纯策略期望支付相等：
```
q₂ - q₃ = -q₁ + q₃ = q₁ - q₂
```

由于 q₁ + q₂ + q₃ = 1，解得：
```
q₁ = q₂ = q₃ = 1/3
```

同理，p₁ = p₂ = p₃ = 1/3

纳什均衡：σ* = ((1/3, 1/3, 1/3), (1/3, 1/3, 1/3))

### 8.1.5 纳什均衡的存在性

**定理8.1.5（纳什均衡存在性定理）**
对于任何有限博弈（有限参与者，有限策略集），存在至少一个（纯策略或混合策略）纳什均衡。

**证明思路**：
1. 构造最佳响应映射：BR: σ → BR(σ)
2. 证明 BR 是上半连续的凸值对应
3. 应用角谷不动点定理（Kakutani Fixed Point Theorem）
4. BR 的不动点即为纳什均衡

**角谷不动点定理**：
设 X 是 ℝⁿ 的非空紧凸子集，F: X → X 是上半连续的凸值对应，则存在 x* ∈ X 使得 x* ∈ F(x*)。

**定理8.1.6（连续博弈的纳什均衡存在性）**
如果：
1. 参与者集合 N 是有限的
2. 策略集 Sᵢ 是非空紧凸集
3. 支付函数 uᵢ(s) 是连续的且对 sᵢ 是拟凹的

则存在纯策略纳什均衡。

**例子8.1.7（伯特兰双寡头）**
两个企业同时定价 p₁, p₂ ≥ 0。

需求函数：
- 如果 p₁ < p₂：企业1获得全部需求
- 如果 p₁ = p₂：平分需求
- 如果 p₁ > p₂：企业1获得零需求

成本函数：Cᵢ(qᵢ) = cqᵢ

纳什均衡：
- p₁ = p₂ = c（边际成本定价）
- 这是唯一的纯策略纳什均衡

**与库诺特比较**：
- 库诺特：选择产量，价格由市场决定
- 伯特兰：选择价格，产量由市场决定
- 伯特兰均衡更接近完全竞争结果

## 8.2 纳什均衡的精炼

### 8.2.1 多重均衡问题

许多博弈存在多个纳什均衡，需要精炼标准来选择"合理"的均衡。

**例子8.2.1（多重均衡）**

```
                    参与者2
                 L        M        R
U       (2,1)    (0,0)    (0,0)
参与者1  M       (0,0)    (1,2)    (0,0)
        D       (0,0)    (0,0)    (1,1)
```

三个纳什均衡：(U, L), (M, M), (D, R)

哪个均衡更合理？需要精炼标准。

### 8.2.2 帕累托效率

**定义8.2.1（帕累托占优均衡）**
纳什均衡 s* 帕累托占优于纳什均衡 s**，如果：

```
uᵢ(s*) ≥ uᵢ(s**), ∀i
且存在至少一个 i 使得 uᵢ(s*) > uᵢ(s**)
```

**应用**：在性别战中，两个均衡都是帕累托最优的，无法通过帕累托标准选择。

### 8.2.3 风险占优

**定义8.2.2（风险占优）**
考虑 2×2 协调博弈：

```
            参与者2
        A        B
A   (a, a)   (0, 0)
参与者1
B   (0, 0)   (b, b)
```

其中 a, b > 0。

均衡 (A, A) 风险占优于 (B, B)，如果：
```
a² > b²
```

或者等价地：
```
a > b
```

**含义**：风险占优的均衡对对方的偏离更有鲁棒性。

**例子8.2.2（猎鹿博弈）**

```
            参与者2
        鹿        兔
鹿   (4, 4)   (0, 2)
参与者1
兔   (2, 0)   (2, 2)
```

两个均衡：(鹿, 鹿) 和 (兔, 兔)

- (鹿, 鹿) 帕累托占优于 (兔, 兔)
- 但 (兔, 兔) 可能风险占优（取决于具体参数）

### 8.2.4 颤抖手精炼

**定义8.2.3（完全混合策略）**
策略 σᵢ 是完全混合的（Totally Mixed），如果 σᵢ(sᵢ) > 0 对所有 sᵢ ∈ Sᵢ。

**定义8.2.4（颤抖手完美均衡）**
纳什均衡 σ* 是颤抖手完美（Trembling-Hand Perfect）的，如果存在完全混合策略序列 {σᵉ} 收敛于 σ*，使得对于足够小的 ε > 0，σᵢ* 是对 σ₋ᵢᵉ 的最佳响应。

**直观含义**：
- 参与者可能"颤抖"（犯小错误）
- 均衡应对这些颤抖具有鲁棒性

**定理8.2.1（颤抖手完美均衡的存在性）**
每个有限博弈至少存在一个颤抖手完美均衡。

**例子8.2.3（弱优势策略与颤抖手）**

```
            参与者2
        L        R
U   (1, 1)   (0, 0)
参与者1
D   (0, 0)   (0, 0)
```

- (U, L) 是纳什均衡
- 但 (U, L) 不是颤抖手完美的
- 如果参与者2有微小概率选择 R，参与者1 应该选择 U 还是 D？无差异...

**更好的例子**：

```
            参与者2
        L        R
U   (2, 1)   (0, 0)
参与者1
D   (0, 0)   (1, 2)
```

- 两个纳什均衡：(U, L) 和 (D, R)
- 都是颤抖手完美的

### 8.2.5 恰当均衡

**定义8.2.5（恰当均衡）**
纳什均衡 σ* 是恰当的（Proper），如果存在完全混合策略序列 {σᵉ} 收敛于 σ*，使得：

对于任何两个纯策略 sᵢ, sᵢ' ∈ Sᵢ：
如果 uᵢ(sᵢ, σ₋ᵢᵉ) < uᵢ(sᵢ', σ₋ᵢᵉ)，则：
```
lim_{ε→0} σᵢᵉ(sᵢ') / σᵢᵉ(sᵢ) = ∞
```

**含义**：
- 更差的策略以更低的概率被选择
- 错误的概率与"错误成本"成反比

**定理8.2.2（恰当均衡的性质）**
- 每个恰当均衡都是颤抖手完美均衡
- 恰当均衡总是存在的

## 8.3 扩展型博弈

### 8.3.1 扩展型博弈的定义

**定义8.3.1（扩展型博弈）**
扩展型博弈（Extensive-form Game）由以下要素组成：

```
Γ = {N, H, P, (Iᵢ), (uᵢ)}
```

其中：
- N = {1, 2, ..., n} 是参与者集合
- H 是历史（History）的集合，满足：
  - ∅ ∈ H（空历史）
  - 如果 h ∈ H 且 a 是 h 之后的行动，则 (h, a) ∈ H
  - 如果 h ∈ H，则 h 的所有前缀都在 H 中
  - 如果 h ∈ H 无后继，则 h 是终点历史
- Z 是终点历史的集合
- P: H \ Z → N ∪ {N} 是参与者函数（P(h) = N 表示机会节点）
- Iᵢ 是参与者 i 的信息分割（Information Partition）
- uᵢ: Z → ℝ 是参与者 i 的支付函数

**信息集（Information Set）**：
- 参与者 i 的信息集 Iᵢ 是 H 的一个分割
- 同一信息集中的历史对参与者 i 是不可区分的
- 如果 |I| > 1，则信息集 I 是不完美的（Imperfect Information）

**定义8.3.2（博弈树）**
扩展型博弈可以用树状结构表示：
- 节点：历史
- 边：行动
- 根节点：空历史
- 叶节点：终点历史

**例子8.3.1（进入博弈）**

```
                    进入者
                  /       \
                进入      不进入
               /            \
            在位者          (1, 0)
          /      \
       容忍      斗争
      (2, 1)   (0, -1)
```

- 进入者先行动
- 如果进入，在位者选择容忍或斗争
- 支付：(进入者, 在位者)

**例子8.3.2（不完美信息博弈）**

```
                      参与者1
                    /       \
                  A          B
                 / \        / \
            参与者2     参与者2
           (信息集)    (信息集)
             / \         / \
            C   D       C   D
          (2,1)(1,2)  (1,2)(2,1)
```

- 参与者2看不到参与者1的选择
- 参与者2面临相同的信息集

### 8.3.2 策略

**定义8.3.3（策略）**
在扩展型博弈中，参与者 i 的策略是一个完全行动计划：
```
sᵢ: Hᵢ → Aᵢ
```

其中：
- Hᵢ 是参与者 i 行动的历史集合
- Aᵢ 是可行动的集合
- 对于所有 h ∈ Hᵢ 和 h' ∈ I(h)（h 所在的信息集），有 sᵢ(h) = sᵢ(h')

**含义**：策略规定了在每个信息集下选择什么行动。

**例子8.3.3（进入博弈的策略）**

进入者的策略：
- s₁ = 进入
- s₁' = 不进入

在位者的策略：
- s₂ = 容忍
- s₂' = 斗争

策略组合：
- (进入, 容忍)
- (进入, 斗争)
- (不进入, 容忍)
- (不进入, 斗争)

**注意**：虽然 (不进入, 容忍) 和 (不进入, 斗争) 在实际中无区别，但它们是不同的策略。

### 8.3.3 子博弈

**定义8.3.4（子博弈）**
扩展型博弈的子博弈（Subgame） Γ|ʰ 满足：
1. h ∈ H 是非终点历史
2. 对于任何历史 h' ∈ H，如果 h' 是 h 的后继，则 h' 的所有前缀都是 h 的后继
3. 信息集不被分割：对于任何信息集 I，如果 I ∩ H|ʰ ≠ ∅，则 I ⊆ H|ʰ

**直观含义**：子博弈是博弈中能够单独构成完整博弈的部分。

**例子8.3.4（子博弈识别）**

```
                      参与者1
                    /       \
                  A          B
                 / \        / \
            参与者2     参与者2
           /     \     /     \
          C       D   E       F
```

- 以 C 开始的部分是一个子博弈
- 以 D 开始的部分是一个子博弈
- 以 E 开始的部分是一个子博弈
- 以 F 开始的部分是一个子博弈
- 如果参与者2有一个跨越 A 和 B 的信息集，则这些都不是子博弈

### 8.3.4 子博弈完美纳什均衡

**定义8.3.5（子博弈完美纳什均衡）**
策略组合 s* 是子博弈完美纳什均衡（Subgame Perfect Nash Equilibrium, SPNE），如果对于每个子博弈 Γ|ʰ，限制在 Γ|ʰ 上的 s*|ʰ 是 Γ|ʰ 的纳什均衡。

**定理8.3.1（SPNE的性质）**
- 每个SPNE都是纳什均衡
- 逆命题不成立：存在不是SPNE的纳什均衡

**定理8.3.2（有限博弈SPNE的存在性）**
每个有限完全信息扩展型博弈至少存在一个SPNE。

**寻找SPNE的方法：逆向归纳法（Backward Induction）**

**步骤**：
1. 从最后的决策节点开始
2. 在每个决策节点，选择对参与者最优的行动
3. 用最优行动的支付替代决策节点
4. 重复直到根节点

**例子8.3.5（进入博弈的SPNE）**

```
                    进入者
                  /       \
                进入      不进入
               /            \
            在位者          (1, 0)
          /      \
       容忍      斗争
      (2, 1)   (0, -1)
```

逆向归纳：
1. 在位者在"进入"后的决策：
   - 容忍：支付 = 1
   - 斗争：支付 = -1
   - 最优选择：容忍
   - 用 (2, 1) 替代该节点

2. 进入者在根节点的决策：
   - 进入：支付 = 2
   - 不进入：支付 = 1
   - 最优选择：进入

SPNE：(进入, 容忍)

**例子8.3.6（序贯博弈）**

```
                    参与者1
                  /       \
                L          R
               /            \
            参与者2          (2, 1)
          /      \
        l        r
      (1, 2)   (3, 0)
```

逆向归纳：
1. 参与者2在 (L) 后的选择：
   - l：支付 = 2
   - r：支付 = 0
   - 最优：l
   - 用 (1, 2) 替代

2. 参与者1的选择：
   - L：支付 = 1
   - R：支付 = 2
   - 最优：R

SPNE：(R, l)

**注意**：虽然参与者2永远不会行动，但策略必须规定在每个信息集下的行动。

### 8.3.5 不完美信息博弈

**定义8.3.6（完美信息博弈）**
博弈具有完美信息（Perfect Information），如果每个信息集都是单元素集合。

**定义8.3.7（不完美信息博弈）**
博弈具有不完美信息（Imperfect Information），如果存在至少一个信息集包含多个历史。

**例子8.3.7（不完美信息博弈）**

```
                      参与者1
                    /       \
                  A          B
                 /            \
               参与者2        参与者2
              (信息集I)     (信息集I)
               /   \         /   \
              L     R       L     R
           (2,1) (0,0)   (0,0) (1,2)
```

参与者2无法区分参与者1选择了 A 还是 B。

**策略**：
- 参与者1：s₁ ∈ {A, B}
- 参与者2：s₂ ∈ {L, R}

支付矩阵：

```
         参与者2
       L        R
A   (2, 1)   (0, 0)
参与者1
B   (0, 0)   (1, 2)
```

纳什均衡：
- (A, L)：参与者1无偏离动机，参与者2在信息集 I 下无偏离动机
- (B, R)：类似

**问题**：哪个均衡更合理？

### 8.3.6 完美贝叶斯均衡

**定义8.3.8（信念）**
在不完美信息博弈中，参与者 i 在信息集 I 上的信念 μ(I) 是一个概率分布：
```
μ(I): I → [0, 1]
Σ_{h∈I} μ(h) = 1
```

信念表示参与者 i 认为自己处于哪个历史的概率。

**定义8.3.9（完美贝叶斯均衡）**
策略组合 (σ₁, ..., σₙ) 和信念系统 μ 构成完美贝叶斯均衡（Perfect Bayesian Equilibrium, PBE），如果满足：

**条件1（信念的一致性）**：
在每个信息集 I 上，信念 μ(I) 由策略 σ 使用贝叶斯法则生成。

**条件2（序贯理性）**：
在每个信息集 I 上，给定信念 μ(I)，策略 σᵢ|I 是最优的。

**条件3（不包含被占优策略的信念）**：
在某些精炼版本中，要求信念不赋予被占优策略正概率。

**例子8.3.8（信号博弈）**

发送者 S：知道类型 t ∈ {θ₁, θ₂}
接收者 R：不知道类型

博弈结构：
1. 自然选择类型，P(t = θ₁) = p, P(t = θ₂) = 1-p
2. 发送者发送信号 m ∈ {m₁, m₂}
3. 接收者根据信号采取行动 a ∈ {a₁, a₂}
4. 支付：uᵢ(t, m, a)

完美贝叶斯均衡包括：
- 发送者的策略：m(t)
- 接收者的策略：a(m)
- 接收者的信念：μ(t|m)

**均衡类型**：
1. **分离均衡（Separating Equilibrium）**：不同类型发送不同信号
2. **混同均衡（Pooling Equilibrium）**：不同类型发送相同信号
3. **半分离均衡（Semi-Separating Equilibrium）**：部分类型分离

## 8.4 不完全信息博弈

### 8.4.1 贝叶斯博弈

**定义8.4.1（贝叶斯博弈）**
贝叶斯博弈（Bayesian Game）或类型不完全信息博弈由以下要素组成：

```
G = {N, (Tᵢ), (pᵢ), (Sᵢ), (uᵢ)}
```

其中：
- N = {1, 2, ..., n} 是参与者集合
- Tᵢ 是参与者 i 的类型空间（Type Space）
- pᵢ: Tᵢ → Δ(T₋ᵢ) 是参与者 i 关于其他人类型的信念
- Sᵢ 是参与者 i 的策略空间
- uᵢ: T × S → ℝ 是支付函数

**定义8.4.2（策略）**
在贝叶斯博弈中，参与者 i 的策略是一个函数：
```
sᵢ: Tᵢ → Sᵢ
```

即策略规定了每种类型下选择什么行动。

**定义8.4.3（期望支付）**
给定策略组合 s = (s₁, ..., sₙ)，参与者 i 类型为 tᵢ 的期望支付：
```
Uᵢ(s | tᵢ) = E_{t₋ᵢ}[uᵢ(tᵢ, t₋ᵢ, sᵢ(tᵢ), s₋ᵢ(t₋ᵢ))]
```

### 8.4.2 贝叶斯纳什均衡

**定义8.4.4（贝叶斯纳什均衡）**
策略组合 s* = (s₁*, ..., sₙ*) 是贝叶斯纳什均衡（Bayesian Nash Equilibrium），如果对于所有 i 和所有 tᵢ ∈ Tᵢ：

```
Uᵢ(s* | tᵢ) ≥ Uᵢ(sᵢ, s₋ᵢ* | tᵢ), ∀sᵢ: Tᵢ → Sᵢ
```

即每种类型下，策略都是最佳响应。

**定理8.4.1（贝叶斯纳什均衡的存在性）**
在标准连续性假设下，贝叶斯博弈存在贝叶斯纳什均衡。

**例子8.4.1（密封拍卖）**
两个投标人，私有估值 v₁, v₂ ~ U[0, 1]，独立同分布。

投标人 i 的策略：bᵢ(vᵢ) = k·vᵢ

投标人1的期望支付（给定估值 v₁，出价 b₁ = k·v₁）：
```
E[π₁] = P(b₁ > b₂)·(v₁ - b₁)
      = P(k·v₁ > k·v₂)·(v₁ - k·v₁)
      = P(v₁ > v₂)·(1 - k)·v₁
      = v₁·(1 - k)·v₁
      = (1 - k)·v₁²
```

最大化：(1 - k)·v₁²

但这忽略了策略的均衡性质...

**正确分析**：
假设投标人使用线性策略 bᵢ(vᵢ) = α·vᵢ

给定投标人2的策略，投标人1的问题：
```
max_{b₁} P(b₁ > α·v₂)·(v₁ - b₁)
     = max_{b₁} P(v₂ < b₁/α)·(v₁ - b₁)
     = max_{b₁} (b₁/α)·(v₁ - b₁)  （对于 b₁ ≤ α）
```

FOC：∂/∂b₁ [(b₁/α)(v₁ - b₁)] = 0
```
(1/α)(v₁ - b₁) - (b₁/α) = 0
v₁ - 2b₁ = 0
b₁ = v₁/2
```

因此 α = 1/2

贝叶斯纳什均衡：bᵢ(vᵢ) = vᵢ/2

**例子8.4.2（不完全信息库诺特博弈）**
两个企业，成本 cᵢ 是私有信息。

c₁ ~ U[0, 1]，c₂ ~ U[0, 1]，独立。

企业 i 的策略：qᵢ(cᵢ)

企业1的问题（给定 c₁）：
```
max E[π₁(q₁ | c₁)] = E[(a - q₁ - q₂(c₂) - c₁)·q₁]
```

假设线性策略：qᵢ(cᵢ) = α - β·cᵢ

求解贝叶斯纳什均衡...

均衡：qᵢ(cᵢ) = (a - 2cᵢ)/3

### 8.4.3 海萨尼转换

**海萨尼转换（Harsanyi Transformation）**：
将不完全信息博弈转换为不完美信息博弈。

**步骤**：
1. 引入虚拟参与者"自然"（Nature）
2. 自然首先选择类型分布
3. 参与者知道自己的类型，不知道其他人的类型
4. 博弈继续，但参与者根据信念采取行动

**意义**：
- 统一了不完全信息和不完美信息的分析
- 使贝叶斯纳什均衡成为SPNE的特例

**定理8.4.2（海萨尼转换的等价性）**
贝叶斯博弈的贝叶斯纳什均衡等价于海萨尼转换后扩展型博弈的完美贝叶斯均衡。

### 8.4.4 共同先验信念

**定义8.4.5（共同先验）**
参与者对类型分布有共同先验信念，如果存在联合分布 p(t₁, ..., tₙ) 使得：

```
pᵢ(t₋ᵢ | tᵢ) = p(t₁, ..., tₙ) / Σ_{t₋ᵢ} p(t₁, ..., tₙ)
```

**含义**：
- 参与者的信念来自于同一概率分布
- 信念的差异仅源于信息不同，而非主观判断

**定理8.4.3（共同先验与贝叶斯更新）**
在共同先验下，参与者使用贝叶斯法则更新信念。

**例子8.4.3（共同先验的应用）**
两个参与者，类型 (t₁, t₂) 的联合分布：

```
          t₂ = L    t₂ = H
t₁ = L    0.3      0.2
t₁ = H    0.2      0.3
```

参与者1知道 t₁ = L 后的信念：
```
P(t₂ = L | t₁ = L) = 0.3 / (0.3 + 0.2) = 0.6
P(t₂ = H | t₁ = L) = 0.2 / (0.3 + 0.2) = 0.4
```

### 8.4.5 机制设计基础

**机制设计（Mechanism Design）**：
给定参与者类型和目标函数，设计博弈机制以实现期望结果。

**定义8.4.6（直接机制）**
直接机制要求参与者报告类型，然后根据报告分配结果。

**定义8.4.7（激励相容）**
机制是激励相容的，如果如实报告是贝叶斯纳什均衡。

**定理8.4.4（显示原理）**
对于任何机制和其贝叶斯纳什均衡，存在一个激励相容的直接机制，实现相同的结果。

**意义**：
- 可以只考虑直接机制
- 大大简化机制设计问题

**例子8.4.4（维克里拍卖）**
二价密封拍卖（Second-Price Sealed-Bid Auction）：

- 每个投标人提交出价 bᵢ
- 出价最高者获胜，支付第二高的出价

**激励相容性**：
如实出价 bᵢ = vᵢ 是贝叶斯纳什均衡。

**证明**：
考虑投标人1，真实估值 v₁，出价 b₁。

情况1：b₁ > v₁
- 如果第二高出价 b₂ ∈ (v₁, b₁)：投标人1支付 b₂ > v₁，负支付
- 如果第二高出价 b₂ ≤ v₁：与出价 v₁ 相同
- 出价 b₁ > v₁ 不优于出价 v₁

情况2：b₁ < v₁
- 如果第二高出价 b₂ ∈ (b₁, v₁)：投标人1本可以以 b₂ < v₁ 获胜，错过正支付
- 如果第二高出价 b₂ ≤ b₁：与出价 v₁ 相同
- 出价 b₁ < v₁ 不优于出价 v₁

因此，如实出价 b₁ = v₁ 是最优策略。证毕。

## 8.5 重复博弈

### 8.5.1 重复博弈的定义

**定义8.5.1（重复博弈）**
重复博弈（Repeated Game）是基础博弈（Stage Game）重复多次的博弈。

```
Gᵀ = {G, T, δ}
```

其中：
- G 是基础博弈
- T 是重复次数（T = ∞ 表示无限重复）
- δ ∈ (0, 1) 是贴现因子

**策略**：在重复博弈中，策略是从历史到行动的映射：
```
sᵢ: H → Aᵢ
```

其中 H 是所有可能历史的集合。

**例子8.5.1（重复囚徒困境）**
基础博弈（囚徒困境）重复 T 次。

如果 T 是有限的，使用逆向归纳：
- 最后一期：坦白是优势策略
- 倒数第二期：预期最后一期坦白，当前期也坦白
- ...
- 所有期都坦白

如果 T 是无限的，可能出现合作！

### 8.5.2 无限重复博弈

**定义8.5.2（贴现支付）**
在无限重复博弈中，参与者 i 的贴现支付：
```
Uᵢ = Σ_{t=1}^∞ δ^{t-1}·uᵢᵗ
```

其中 uᵢᵗ 是第 t 期的支付。

**定义8.5.3（平均支付）**
参与者 i 的平均支付：
```
\bar{u}_i = (1 - δ)·Σ_{t=1}^∞ δ^{t-1}·uᵢᵗ
```

**定义8.5.4（无名氏定理，Folk Theorem）**
在无限重复博弈中，如果 δ 足够大，任何严格个人理性的支付向量都可以作为子博弈完美均衡的支付向量。

**定义8.5.5（个人理性）**
支付向量 u 是个人理性的，如果对于所有 i：
```
uᵢ ≥ \underline{u}_i
```

其中 \underline{u}_i 是参与者 i 的最小最大支付（Minimax Payoff）：
```
\underline{u}_i = min_{s₋ᵢ} max_{sᵢ} uᵢ(sᵢ, s₋ᵢ)
```

**定理8.5.1（无名氏定理）**
在无限重复博弈中，如果 δ 足够接近 1，则任何可行且个人理性的支付向量都可以作为某个子博弈完美均衡的支付向量。

**例子8.5.2（重复囚徒困境的合作）**
囚徒困境基础博弈：

```
            参与者2
        坦白      沉默
坦白   (-5,-5)   (0,-10)
参与者1
沉默   (-10,0)   (-1,-1)
```

最小最大支付：
- 参与者1的最小最大：min_{s₂} max_{s₁} u₁(s₁, s₂)
  - 如果 s₂ = 坦白：max{-5, -10} = -5
  - 如果 s₂ = 沉默：max{0, -10} = 0
  - 最小最大：min{-5, 0} = -5

- 参与者2的最小最大：-5

合作策略（触发策略，Trigger Strategy）：
- 开始：合作（沉默）
- 如果有人背叛，之后永远背叛（坦白）

**分析**：
如果一直合作：
```
U合作 = -1 + δ·(-1) + δ²·(-1) + ... = -1/(1-δ)
```

如果第一期背叛：
```
U背叛 = 0 + δ·(-5) + δ²·(-5) + ... = 0 - 5δ/(1-δ)
```

合作的条件：U合作 ≥ U背叛
```
-1/(1-δ) ≥ -5δ/(1-δ)
-1 ≥ -5δ
δ ≥ 1/5
```

如果贴现因子 δ ≥ 0.2，合作是子博弈完美均衡！

### 8.5.3 触发策略

**定义8.5.6（触发策略）**
触发策略（Trigger Strategy）是一种重复博弈策略：
- 开始合作
- 如果有人偏离，触发惩罚（永远惩罚或暂时惩罚）

**类型**：
1. **冷酷触发（Grim Trigger）**：一旦偏离，永远惩罚
2. **宽容触发（Forgiving Trigger）**：惩罚一段时间后恢复合作
3. **针锋相对（Tit-for-Tat）**：复制对手上一期的行动

**例子8.5.3（针锋相对策略）**
在重复囚徒困境中：
- 第一期：沉默
- 之后：复制对手上一期的行动

**性质**：
- 如果对手一直沉默，你也一直沉默
- 如果对手背叛一次，你背叛一次，然后恢复合作
- 简单、宽容、报复性强

**定理8.5.2（针锋相对在进化博弈中的成功）**
在演化博弈中，针锋相对策略往往表现出色（Axelrod 锦标赛）。

### 8.5.4 有限重复博弈

**定理8.5.3（有限重复博弈的纳什均衡）**
在有限重复博弈中：
- 如果基础博弈有唯一的纳什均衡，则唯一的子博弈完美均衡是重复该均衡
- 如果基础博弈有多个纳什均衡，可能出现其他子博弈完美均衡

**例子8.5.4（有限重复性别战）**
基础博弈（性别战）有两个纳什均衡。

重复博弈中可能出现：
- 轮流选择不同均衡
- 混合策略
- 更复杂的协调

**定理8.5.4（有限重复博弈的无名氏定理）**
如果基础博弈有多个纳什均衡，且重复次数足够长，则可以近似实现任何可行且个人理性的支付向量。

## 8.6 合作博弈

### 8.6.1 合作博弈的基本概念

**定义8.6.1（合作博弈）**
合作博弈（Cooperative Game）研究参与者如何通过合作形成联盟（Coalition）并分配收益。

**定义8.6.2（联盟型博弈）**
联盟型博弈（Coalitional Game）由以下要素组成：

```
(N, v)
```

其中：
- N = {1, 2, ..., n} 是参与者集合
- v: 2ᴺ → ℝ 是特征函数（Characteristic Function）
- v(S) 是联盟 S 能获得的最大支付

**性质**：
1. **规范性**：v(∅) = 0
2. **超可加性（Superadditivity）**：对于不相交的联盟 S, T，有：
   ```
   v(S ∪ T) ≥ v(S) + v(T)
   ```

**例子8.6.1（多数投票博弈）**
3个参与者，简单多数通过。

- v({i}) = 0（单个成员无法通过提案）
- v({i, j}) = 1（两个成员可以通过提案）
- v({1, 2, 3}) = 1

### 8.6.2 核（Core）

**定义8.6.3（配置）**
配置（Imputation）是支付向量 x = (x₁, ..., xₙ) 满足：
1. **个体理性**：xᵢ ≥ v({i}), ∀i
2. **集体理性**：Σᵢ xᵢ = v(N)

**定义8.6.4（核）**
核（Core）是所有不被任何联盟否决的配置的集合：
```
C = {x: Σ_{i∈S} xᵢ ≥ v(S), ∀S ⊆ N, 且 Σᵢ xᵢ = v(N)}
```

**直观含义**：
- 没有联盟能够通过脱离大联盟而获得更好的支付
- 核是稳定配置的集合

**定理8.6.1（核的性质）**
- 核可能是空集
- 如果核非空，它是凸集
- 核是所有合作解概念的交集

**例子8.6.2（三人博弈的核）**
考虑博弈：
- v({i}) = 0
- v({1, 2}) = 6
- v({1, 3}) = 6
- v({2, 3}) = 6
- v({1, 2, 3}) = 8

核的条件：
1. x₁ + x₂ ≥ 6
2. x₁ + x₃ ≥ 6
3. x₂ + x₃ ≥ 6
4. x₁ + x₂ + x₃ = 8
5. xᵢ ≥ 0, ∀i

解：
从 x₁ + x₂ ≥ 6 和 x₃ ≥ 0，得 x₁ + x₂ + x₃ ≥ 6
但 x₁ + x₂ + x₃ = 8，所以：
```
x₃ ≤ 2
x₂ ≤ 2
x₁ ≤ 2
```

核：x = (2, 2, 2)

### 8.6.3 夏普利值（Shapley Value）

**定义8.6.5（夏普利值）**
夏普利值（Shapley Value）是参与者 i 的"平均边际贡献"：

```
φᵢ(v) = Σ_{S⊆N\{i}} [|S|!(n-|S|-1)! / n!] · [v(S ∪ {i}) - v(S)]
```

**直观含义**：
- 考虑参与者按随机顺序加入联盟
- 参与者 i 的贡献是他加入时的边际贡献
- 夏普利值是所有可能顺序下的期望边际贡献

**性质**：
1. **效率性**：Σᵢ φᵢ(v) = v(N)
2. **对称性**：如果 i 和 j 是对称的，则 φᵢ(v) = φⱼ(v)
3. **虚拟性**：如果 i 是虚拟参与者（无贡献），则 φᵢ(v) = 0
4. **可加性**：φᵢ(v + w) = φᵢ(v) + φᵢ(w)

**定理8.6.2（夏普利值的存在唯一性）**
夏普利值是唯一满足上述四个性质的解概念。

**例子8.6.3（三人博弈的夏普利值）**
考虑前面的博弈：
- v({i}) = 0
- v({1, 2}) = 6
- v({1, 3}) = 6
- v({2, 3}) = 6
- v({1, 2, 3}) = 8

计算 φ₁：

参与者1加入时的顺序（共 6 种）：
1. (1, 2, 3)：v({1}) - v(∅) = 0 - 0 = 0
2. (1, 3, 2)：v({1}) - v(∅) = 0 - 0 = 0
3. (2, 1, 3)：v({1, 2}) - v({2}) = 6 - 0 = 6
4. (2, 3, 1)：v({1, 2, 3}) - v({2, 3}) = 8 - 6 = 2
5. (3, 1, 2)：v({1, 3}) - v({3}) = 6 - 0 = 6
6. (3, 2, 1)：v({1, 2, 3}) - v({2, 3}) = 8 - 6 = 2

φ₁ = (0 + 0 + 6 + 2 + 6 + 2) / 6 = 16/6 = 8/3

同理：
φ₂ = 8/3
φ₃ = 8/3

夏普利值：(8/3, 8/3, 8/3) ≈ (2.67, 2.67, 2.67)

与核 (2, 2, 2) 不同！

### 8.6.4 核与夏普利值的关系

**定理8.6.3（核与夏普利值）**
- 夏普利值不一定在核中
- 如果博弈是凸的（Convex Game），则夏普利值在核中

**定义8.6.6（凸博弈）**
博弈 v 是凸的，如果对于所有 S, T ⊆ N：
```
v(S ∪ T) + v(S ∩ T) ≥ v(S) + v(T)
```

**含义**：
- 边际贡献递增
- 参与者越多，联盟越有效

**例子8.6.4（凸博弈）**
考虑博弈：
- v({i}) = 0
- v({1, 2}) = 4
- v({1, 3}) = 3
- v({2, 3}) = 5
- v({1, 2, 3}) = 12

检查凸性：
v({1, 2} ∪ {1, 3}) + v({1, 2} ∩ {1, 3}) = v({1, 2, 3}) + v({1})
= 12 + 0 = 12

v({1, 2}) + v({1, 3}) = 4 + 3 = 7

12 ≥ 7 ✓

### 8.6.5 其他解概念

**定义8.6.7（谈判集）**
谈判集（Bargaining Set）是满足以下条件的配置：
- 每个异议（Objection）都有反异议（Counter-Objection）

**定义8.6.8（内核）**
内核（Kernel）是谈判集的子集，满足额外的平衡条件。

**定义8.6.9（核仁）**
核仁（Nucleolus）是最小化最大不满意的配置。

**比较**：
- 核：最强条件，可能为空
- 夏普利值：唯一解，有良好性质
- 谈判集、内核、核仁：介于两者之间

## 本章总结

### 核心要点

1. **策略型博弈**
   - 纳什均衡是基本解概念
   - 纳什均衡的存在性：有限博弈总有纳什均衡（可能需要混合策略）
   - 多重均衡需要精炼标准

2. **纳什均衡的精炼**
   - 帕累托效率、风险占优
   - 颤抖手完美均衡：对小错误的鲁棒性
   - 恰当均衡：错误概率与成本成反比

3. **扩展型博弈**
   - 子博弈完美纳什均衡：每个子博弈都是纳什均衡
   - 逆向归纳法：寻找SPNE的方法
   - 完美贝叶斯均衡：处理不完美信息

4. **不完全信息博弈**
   - 贝叶斯博弈：类型私有
   - 贝叶斯纳什均衡：每种类型下的最佳响应
   - 海萨尼转换：将不完全信息转换为不完美信息

5. **重复博弈**
   - 无限重复博弈：无名氏定理
   - 触发策略：合作与惩罚
   - 贴现因子决定合作的可能性

6. **合作博弈**
   - 联盟型博弈：特征函数描述联盟价值
   - 核：稳定的配置集合
   - 夏普利值：平均边际贡献

### 重要概念表格

| 概念 | 定义 | 特点 |
|------|------|------|
| 纳什均衡 | 每个人的策略都是对其他人策略的最佳响应 | 基本解概念，可能多重 |
| 混合策略 | 纯策略上的概率分布 | 允许随机化 |
| SPNE | 每个子博弈都是纳什均衡 | 消除不可信威胁 |
| 贝叶斯纳什均衡 | 每种类型下的最佳响应 | 处理不完全信息 |
| 完美贝叶斯均衡 | 策略和信念满足序贯理性 | 处理不完美信息 |
| 无名氏定理 | δ足够大时，任何个人理性支付可实现 | 解释长期合作 |
| 触发策略 | 合作直到有人偏离，然后惩罚 | 实现合作的一种方式 |
| 夏普利值 | 平均边际贡献 | 唯一解概念 |
| 核 | 不被任何联盟否决的配置 | 稳定解概念 |

### 主要定理汇总

| 定理 | 内容 | 条件 |
|------|------|------|
| 纳什均衡存在性 | 有限博弈总有纳什均衡 | 有限参与者，有限策略 |

### 结语

Jehle & Reny的《高级微观经济理论》为理解和分析经济现象提供了严谨的理论框架。通过本书的学习，读者应该能够：

1. 运用数学工具分析经济问题
2. 理解微观经济学的基本原理
3. 掌握均衡分析方法
4. 评估经济政策的福利效应
5. 分析策略互动和博弈行为

这些理论工具不仅适用于学术研究，也为政策制定、商业决策和个人选择提供了重要指导。随着经济学的发展，微观经济理论将继续演进，为理解复杂的经济现象提供更深入的洞察。
