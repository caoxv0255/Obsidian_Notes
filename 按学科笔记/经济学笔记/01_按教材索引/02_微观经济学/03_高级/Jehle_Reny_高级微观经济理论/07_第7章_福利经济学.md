# 第7章 福利经济学

本章主要介绍福利经济学的基本理论，包括：

4. 理解社会选择理论的基本问题
5. 探讨公平与效率之间的关系

## 7.1 帕累托最优

### 7.1.1 基本定义

**定义7.1.1（帕累托优势）**
在交换经济中，配置 x = (x₁, x₂, ..., xᵢ) 称为帕累托优于配置 y = (y₁, y₂, ..., yᵢ)，如果：
- 对于所有 i ∈ {1, ..., I}，都有 xᵢ ≥ᵢ yᵢ
- 至少存在一个 i，使得 xᵢ >ᵢ yᵢ

其中 ≥ᵢ 表示弱偏好关系，>ᵢ 表示严格偏好关系。

**定义7.1.2（帕累托最优）**
配置 x = (x₁, x₂, ..., xᵢ) 称为帕累托最优，如果不存在其他可行配置 y，使得 y 帕累托优于 x。

帕累托最优是一个弱效率标准，它要求无法在不使任何人变差的情况下使至少一人变好。这个标准具有以下特点：

1. **非完全性**：并非所有配置都能进行比较
2. **非传递性**：帕累托关系可能不满足传递性
3. **弱效率**：可能存在多个帕累托最优配置

### 7.1.2 交换经济的帕累托最优

考虑一个纯交换经济（I个消费者，L种商品）：

**模型设定**
- 消费者 i 的效用函数：uᵢ(xᵢ)
- 总禀赋：ω = Σᵢ ωᵢ
- 可行配置集合：X = {x | Σᵢ xᵢ ≤ ω, xᵢ ≥ 0}

**定理7.1.1（帕累托最优的必要条件）**
在标准假设下（效用函数连续、严格拟凹、单调），如果配置 x* 是帕累托最优的，则存在标量 αᵢ > 0（i = 1, ..., I），使得对于所有 i, j 和所有 l, k：

```
αᵢ·∂uᵢ(x*ᵢ)/∂xᵢˡ / αⱼ·∂uⱼ(x*ⱼ)/∂xⱼᵏ = ∂uᵢ(x*ᵢ)/∂xᵢᵏ / ∂uⱼ(x*ⱼ)/∂xⱼˡ
```

特别地，对于每个 i, j：

```
MRSᵢˡᵏ = MRSⱼˡᵏ
```

即所有消费者的边际替代率相等。

**证明**：
配置 x* 帕累托最优意味着以下效用最大化问题有解：

```
max u₁(x₁)
s.t. uᵢ(xᵢ) ≥ uᵢ(x*ᵢ), i = 2, ..., I
    Σᵢ xᵢ = ω
    xᵢ ≥ 0
```

构建拉格朗日函数：
```
L = u₁(x₁) - Σᵢ=2ˡ λᵢ[uᵢ(x*ᵢ) - uᵢ(xᵢ)] - μ'·[Σᵢ xᵢ - ω]
```

一阶条件：
```
∂L/∂x₁ = ∇u₁(x₁) - μ = 0
∂L/∂xᵢ = λᵢ∇uᵢ(xᵢ) - μ = 0, i = 2, ..., I
```

令 α₁ = 1, αᵢ = λᵢ (i = 2, ..., I)，则：
```
αᵢ∇uᵢ(x*ᵢ) = μ, ∀i
```

因此对于所有 i, j：
```
αᵢ∇uᵢ(x*ᵢ) = αⱼ∇uⱼ(x*ⱼ)
```

这意味着：
```
αᵢ·∂uᵢ(x*ᵢ)/∂xᵢˡ / αᵢ·∂uᵢ(x*ᵢ)/∂xᵢᵏ = αⱼ·∂uⱼ(x*ⱼ)/∂xⱼˡ / αⱼ·∂uⱼ(x*ⱼ)/∂xⱼᵏ
```

即 MRSᵢˡᵏ = MRSⱼˡᵏ。证毕。

**定理7.1.2（帕累托最优配置的连续性）**
在效用函数严格单调和严格拟凹的条件下，帕累托最优配置的集合是连续的。

### 7.1.3 生产经济的帕累托最优

考虑生产经济（I个消费者，J个厂商，L种商品）：

**模型设定**
- 消费者 i：拥有禀赋 ωᵢ 和厂商份额 θᵢⱼ ≥ 0
- 厂商 j：生产可能性集 Yⱼ
- 利润函数：πⱼ(p) = max_{yⱼ∈Yⱼ} p·yⱼ
- 消费者 i 的预算约束：p·xᵢ ≤ p·ωᵢ + Σⱼ θᵢⱼπⱼ(p)

**定理7.1.3（生产经济帕累托最优的条件）**
配置 (x*, y*) 是帕累托最优的必要条件：

1. **交换效率**：对于所有 i, j：
   ```
   MRSᵢˡᵏ = MRSⱼˡᵏ
   ```

2. **生产效率**：对于所有厂商 k, l 和商品 s, t：
   ```
   MRTSₖˢᵗ = MRTSₗˢᵗ
   ```
   其中 MRTSₖˢᵗ = -∂fₖ/∂s / ∂fₖ/∂t

3. **产品混合效率**：对于所有消费者 i 和厂商 j：
   ```
   MRSᵢˡᵏ = MRTⱼˡᵏ
   ```
   其中 MRTⱼˡᵏ = -∂gⱼ/∂l / ∂gⱼ/∂k（生产可能性边界）

**证明**：
配置 (x*, y*) 帕累托最优等价于以下问题有解：

```
max u₁(x₁)
s.t. uᵢ(xᵢ) ≥ uᵢ(x*ᵢ), i = 2, ..., I
    Σᵢ xᵢ = Σⱼ yⱼ + Σᵢ ωᵢ
    yⱼ ∈ Yⱼ
    xᵢ ≥ 0
```

拉格朗日函数：
```
L = u₁(x₁) - Σᵢ=2ˡ λᵢ[uᵢ(x*ᵢ) - uᵢ(xᵢ)] - μ'·[Σᵢ xᵢ - Σⱼ yⱼ - Σᵢ ωᵢ]
```

一阶条件：
```
∂L/∂x₁ = ∇u₁(x₁) - μ = 0
∂L/∂xᵢ = λᵢ∇uᵢ(xᵢ) - μ = 0, i = 2, ..., I
∂L/∂yⱼ = μ = 0, ∀j
```

结合生产效率条件，可得到上述三个效率条件。证毕。

**图形分析：生产可能性边界与无差异曲线**

```
    商品2
      ↑
      |       PPF
      |      /
      |     /     IC
      |    /    /
      |   /    /
      |  /    /
      | /    /
      |/____/__________→ 商品1
          A
```

- PPF（生产可能性边界）：所有有效率的生产组合
- IC：社会无差异曲线（或代表性消费者的无差异曲线）
- A点：帕累托最优配置点，此时 MRT = MRS

### 7.1.4 帕累托最优与竞争均衡

**定理7.1.4（福利经济学第一定理）**
如果 (p*, x*, y*) 是瓦尔拉斯竞争均衡，且对于每个 i，x*ᵢ 是局部非饱和的，则配置 (x*, y*) 是帕累托最优的。

**证明**：
假设 (p*, x*, y*) 是瓦尔拉斯均衡，但 (x*, y*) 不是帕累托最优的。
则存在可行配置 (x', y') 使得 x' ≥ᵢ x*ᵢ 对所有 i，且至少有一个严格偏好。

由于预算平衡：
```
p*·x*ᵢ = p*·ωᵢ + Σⱼ θᵢⱼπⱼ(p*)
```

由于 x*ᵢ 是消费者效用最大化问题的解，且 x'ᵢ ≥ᵢ x*ᵢ，由局部非饱和性可得：
```
p*·x'ᵢ ≥ p*·x*ᵢ = p*·ωᵢ + Σⱼ θᵢⱼπⱼ(p*)
```

对所有 i 求和：
```
p*·Σᵢ x'ᵢ ≥ p*·Σᵢ ωᵢ + Σᵢ Σⱼ θᵢⱼπⱼ(p*) = p*·Σᵢ ωᵢ + Σⱼ πⱼ(p*)
```

由于利润最大化：
```
πⱼ(p*) = max p*·yⱼ ≥ p*·y'ⱼ
```

因此：
```
p*·Σᵢ x'ᵢ ≥ p*·Σᵢ ωᵢ + Σⱼ p*·y'ⱼ = p*·(Σⱼ y'ⱼ + Σᵢ ωᵢ)
```

由于 (x', y') 是可行的，Σᵢ x'ᵢ = Σⱼ y'ⱼ + Σᵢ ωᵢ，因此：
```
p*·Σᵢ x'ᵢ ≥ p*·Σᵢ x'ᵢ
```

这意味着所有不等式必须取等号。但若存在某个 i 使得 x'ᵢ >ᵢ x*ᵢ，由严格凸性假设会导致矛盾。证毕。

**意义**：
- 竞争市场能够自动实现帕累托最优配置
- 市场经济是有效的资源配置机制
- 不需要中央计划者

**定理7.1.5（福利经济学第二定理）**
在以下条件下，任何帕累托最优配置都可以通过适当的禀赋再分配和竞争市场机制实现：

1. 效用函数和生产集是凸的
2. 偏好是连续的、严格凸的、局部非饱和的
3. 生产集是闭的、凸的，且满足自由处置性
4. 不存在外部性

**证明**：
给定帕累托最优配置 (x*, y*)，我们需要构造禀赋配置使得 (x*, y*) 成为竞争均衡。

定义转移后的禀赋：
```
ω'ᵢ = x*ᵢ - Σⱼ θᵢⱼy*ⱼ
```

选择价格 p* 使得：
1. p*·y*ⱼ ≥ p*·yⱼ，∀yⱼ ∈ Yⱼ（利润最大化）
2. 在新禀赋下，x*ᵢ 满足预算约束：p*·x*ᵢ = p*·ω'ᵢ + Σⱼ θᵢⱼπⱼ(p*)

由帕累托最优的必要条件，存在支持价格 p* 使得上述条件成立。证毕。

**意义**：
- 效率与公平可以分离
- 通过再分配实现公平，通过市场实现效率
- 为政策干预提供理论依据

## 7.2 福利经济学基本定理

### 7.2.1 第一定理的假设与局限性

福利经济学第一定理需要以下假设：

**假设1（竞争性）**
所有市场参与者都是价格接受者（无市场势力）。

**假设2（信息完全）**
价格充分反映了所有相关信息。

**假设3（无外部性）**
消费者的效用仅取决于自己的消费，厂商的生产仅取决于自己的投入。

**假设4（公共品问题）**
所有商品都是私人物品，具有竞争性和排他性。

**假设5（偏好凸性）**
效用函数是拟凹的（凸偏好）。

**局限性分析**：

1. **市场势力**
   - 垄断导致配置无效率
   - 垄断定价使价格 > 边际成本
   - 造成无谓损失（Deadweight Loss）

   - 正外部性（教育）：社会边际收益 > 私人边际收益

3. **公共品**
   - 非竞争性：一人消费不影响他人消费
   - 非排他性：难以排除他人消费
   - 搭便车问题导致供给不足

4. **信息不对称**
   - 逆向选择（Adverse Selection）
   - 道德风险（Moral Hazard）
   - 市场失灵

**例子7.2.1（外部性导致的市场失灵）**
考虑两个企业A和B，企业A的生产对企业B产生负外部性。

企业A的利润：πₐ = p·yₐ - cₐ(yₐ)
企业B的成本：cᵦ(yₐ, yᵦ)，其中 ∂cᵦ/∂yₐ > 0

社会最优选择：
```
max p·(yₐ + yᵦ) - cₐ(yₐ) - cᵦ(yₐ, yᵦ)
```

一阶条件：
```
p = cₐ'(yₐ) + ∂cᵦ/∂yₐ  （社会最优）
p = cₐ'(yₐ)              （市场均衡）
```

市场均衡时，企业A不考虑对外部性的影响，导致 yₐ 过高。

**政策含义**：
- 庇古税（Pigouvian Tax）：对产生外部性的行为征税
- 科斯定理（Coase Theorem）：在交易成本为零时，通过产权界定可解决外部性问题
- 数量管制：设定排放上限

### 7.2.2 第二定理的实现条件

福利经济学第二定理需要更强的假设：

**定理7.2.1（第二定理的充分条件）**
给定以下条件：
1. 对于所有 i，≽ᵢ 是连续、严格凸、局部非饱和的
2. 对于所有 j，Yⱼ 是闭凸集，且满足自由处置
3. 0 ∈ Yⱼ（停止生产是可行的）

则对于任何帕累托最优配置 (x*, y*)，存在禀赋转移 (τ₁, ..., τᵢ) 使得 Σᵢ τᵢ = 0，且 (x*, y*) 成为竞争均衡。

**证明思路**：
1. 利用分离超平面定理
2. 构造支持价格向量 p*
3. 定义转移支付使预算约束成立
4. 验证均衡条件

**实现第二定理的挑战**：

1. **再分配成本**
   - 信息问题：政府不知道个人效用函数
   - 激励问题：再分配可能扭曲劳动供给
   - 行政成本：税收和转移支付的管理成本

2. **凸性要求**
   - 规模报酬递增（Increasing Returns）导致生产集非凸
   - 某些技术自然具有规模报酬递增特征

   **例子7.2.2（规模报酬递增与第二定理失效）**
   考虑单一产品生产，成本函数 c(y) = F + cy（F > 0 为固定成本）。
   
   平均成本：AC = F/y + c
   边际成本：MC = c
   
   当 y 很小时，AC > MC，市场价格无法补偿平均成本。
   此时帕累托最优配置可能无法通过竞争市场实现。

   **图形：自然垄断**
   ```
   成本/价格
     ↑
     |    AC
     |   /
     |  /   MC
     | /   /
     |/___/______→ 产量
        |
       最低点
   ```

3. **不完全市场**
   - 某些风险无法通过市场交易
   - 保险市场缺失或不完全
   - 导致帕累托最优无法实现

**定理7.2.2（不完全市场下的福利定理）**
当市场不完全时（某些商品或状态无法交易）：
- 竞争均衡可能不是帕累托最优的
- 帕累托最优可能无法通过竞争市场实现

### 7.2.3 次优理论（Theory of the Second Best）

**定理7.2.3（利普西-兰卡斯特定理）**
如果由于某种约束，某个帕累托最优条件无法满足，则最优配置通常不满足其他帕累托条件。

**含义**：
- 不能简单地"局部"修正市场失灵
- 需要整体考虑所有约束条件
- 次优（Second Best）最优不同于"第一最佳"（First Best）

**例子7.2.3（次优定价）**
考虑两个垄断部门，政府只能在一个部门实施边际成本定价。

部门1：边际成本定价，p₁ = MC₁
部门2：垄断定价，p₂ > MC₂

社会最优问题：
```
max W = CS₁(x₁) + CS₂(x₂) + π₁(x₁) + π₂(x₂)
s.t. p₁ = MC₁
```

由于部门2的价格扭曲，部门1的最优价格可能不等于边际成本。

## 7.3 社会福利函数

### 7.3.1 社会福利函数的定义

**定义7.3.1（社会福利函数）**
社会福利函数（Social Welfare Function, SWF）是一个从个体效用组合到社会福利评价值的映射：

```
W: ℝᴵ → ℝ
W(u₁, u₂, ..., uᵢ) = w
```

其中 uᵢ 是个体 i 的效用水平，w 是社会福利水平。

**基本性质**：

1. **个体主义（Individualism）**
   社会福利仅取决于个体效用，与效用来源无关。

2. **帕累托性质（Pareto Property）**
   如果 u' ≥ u（分量wise）且 u' ≠ u，则 W(u') > W(u)

3. **对称性（Symmetry）**
   对于任何排列 σ，W(u₁, ..., uᵢ) = W(uσ(₁), ..., uσ(ᵢ))

4. **凸性（Quasi-concavity）**
   W(λu + (1-λ)u') ≥ min{W(u), W(u')}

### 7.3.2 古典功利主义

**定义7.3.2（功利主义社会福利函数）**
功利主义SWF将社会福利定义为个体效用之和：

```
W(u) = Σᵢ αᵢuᵢ
```

其中 αᵢ ≥ 0 是权重，通常假设 αᵢ = 1（所有人平等）。

**特性**：
- 可加性：社会福利等于个体效用总和
- 不考虑分配：只关心总量
- 极端的效用主义：可能允许严重的不平等

**例子7.3.1（功利主义的应用）**
考虑两个个体，效用函数 u₁(x₁) = √x₁, u₂(x₂) = √x₂，总资源 x₁ + x₂ = 100。

功利主义最大化：
```
max √x₁ + √x₂
s.t. x₁ + x₂ = 100
```

拉格朗日函数：
```
L = √x₁ + √x₂ + λ(100 - x₁ - x₂)
```

一阶条件：
```
∂L/∂x₁ = 1/(2√x₁) - λ = 0
∂L/∂x₂ = 1/(2√x₂) - λ = 0
```

因此：1/(2√x₁) = 1/(2√x₂) ⇒ x₁ = x₂ = 50

**意义**：
- 在凹效用函数下，功利主义倾向于平等分配
- 效用函数的凹性反映了风险厌恶和不平等厌恶

### 7.3.3 罗尔斯主义（Rawlsian）

**定义7.3.3（罗尔斯主义社会福利函数）**
罗尔斯主义SWF关注最差境况的个体：

```
W(u) = min{u₁, u₂, ..., uᵢ}
```

这是"最大最小"（Maximin）原则的体现。

**特性**：
- 重视公平：社会福利由最差的人决定
- 强调基本权利：保障最低生活水平
- 可能牺牲效率：可能接受较低的总体福利以改善最差者

**例子7.3.2（罗尔斯主义与功利主义的比较）**
考虑两种配置：

配置A：u₁ = 10, u₂ = 10
配置B：u₁ = 15, u₂ = 5

功利主义：
- W(A) = 20
- W(B) = 20
- 两者无差异

罗尔斯主义：
- W(A) = min{10, 10} = 10
- W(B) = min{15, 5} = 5
- A优于B

**定理7.3.1（罗尔斯主义的极限性质）**
罗尔斯主义SWF可以视为以下函数的极限：

```
W(u) = lim_{ρ→-∞} [Σᵢ uᵢᵖ]¹/ᵖ
```

### 7.3.4 柏格森-萨缪尔森社会福利函数

**定义7.3.4（柏格森-萨缪尔森SWF）**
最一般形式的SWF：

```
W = W(u₁(x₁), u₂(x₂), ..., uᵢ(xᵢ))
```

其中效用函数是序数的，社会福利函数也是序数的。

**性质**：
- 允许任意形式的函数形式
- 不要求基数效用或人际可比性
- 非常灵活，但也导致缺乏可操作性

**特殊形式**：

1. **不变弹性（CES）形式**
   ```
   W(u) = [Σᵢ αᵢ uᵢ^{1-ε}]^{1/(1-ε)}, ε ≠ 1
   W(u) = Πᵢ uᵢ^{αᵢ}, ε = 1
   ```

   其中 ε 是不平等厌恶参数：
   - ε = 0：功利主义
   - ε → ∞：罗尔斯主义
   - ε = 1：纳什乘积形式

2. **加权柯布-道格拉斯形式**
   ```
   W(u) = Πᵢ uᵢ^{αᵢ}
   ```

   其中 Σᵢ αᵢ = 1, αᵢ > 0

### 7.3.5 社会福利函数与帕累托最优

**定理7.3.2（SWF最优化与帕累托最优）**
如果社会福利函数W满足帕累托性质，则任何最大化W的配置都是帕累托最优的。

**证明**：
设 x* 最大化 W(u₁(x*₁), ..., uᵢ(x*ᵢ))。

假设 x* 不是帕累托最优的，则存在可行配置 x' 使得 x' ≥ᵢ x*ᵢ 对所有 i，且至少有一个严格偏好。

由于W满足帕累托性质：
```
W(u₁(x'₁), ..., uᵢ(x'ᵢ)) > W(u₁(x*₁), ..., uᵢ(x*ᵢ))
```

这与 x* 最大化W矛盾。证毕。

**逆命题不成立**：并非所有帕累托最优配置都能由某个SWF最大化得到。

**定理7.3.3（帕累托最优与SWF）**
在凸性假设下，任何帕累托最优配置都可以通过选择适当的SWF（特别是功利主义形式）最大化得到。

**应用：社会最优竞争均衡**

问题：
```
max W(u₁(x₁), ..., uᵢ(xᵢ))
s.t. Σᵢ xᵢ = Σⱼ yⱼ + Σᵢ ωᵢ
    yⱼ ∈ Yⱼ
```

拉格朗日函数：
```
L = W(u₁(x₁), ..., uᵢ(xᵢ)) - μ'·[Σᵢ xᵢ - Σⱼ yⱼ - Σᵢ ωᵢ]
```

一阶条件：
```
∂W/∂uᵢ · ∇uᵢ(xᵢ) = μ, ∀i
μ · yⱼ ≥ μ · yⱼ, ∀yⱼ ∈ Yⱼ
```

令 p = μ/μ₀（其中 μ₀ 是任意标准化），则：
```
p = (∂W/∂uᵢ)∇uᵢ(xᵢ)
```

这意味着价格反映边际社会福利权重。

## 7.4 社会选择理论

### 7.4.1 社会选择问题

**定义7.4.1（社会选择问题）**
给定：
1. 个体集合 N = {1, 2, ..., n}
2. 备选方案集合 X（有限或无限）
3. 每个个体 i 的偏好关系 ≽ᵢ

社会选择规则 f 将个体偏好组合映射为社会偏好：
```
f: (≽₁, ≽₂, ..., ≽ₙ) → ≽
```

**基本问题**：
能否找到合理的社会选择规则，满足某些直观可接受的条件？

### 7.4.2 阿罗不可能定理

**定义7.4.2（偏好域）**
考虑所有可能的严格偏好序的集合。假设：
- 无限制域（Unrestricted Domain, UD）：任何个体偏好组合都是允许的

**社会选择规则的性质**：

**性质1（帕累托原则，P）**
如果所有个体都认为 x 优于 y（x ≻ᵢ y, ∀i），则社会也应认为 x 优于 y（x ≻ y）。

**性质2（无关方案独立性，IIA）**
x 和 y 的社会排序仅取决于个体对 x 和 y 的排序，与其他方案无关。

**性质3（非独裁性，ND）**
不存在个体 d，使得对于所有偏好组合，社会偏好完全等同于 d 的偏好。

**定理7.4.1（阿罗不可能定理）**
在以下条件下：
1. X 包含至少3个方案
2. 无限制域（UD）
3. 社会偏好是传递的
4. 满足帕累托原则（P）
5. 满足无关方案独立性（IIA）

则唯一的社会选择规则是独裁规则（违反ND）。

**证明思路**：
1. 构造关键联盟（Decisive Coalition）
2. 证明关键联盟具有聚合性质
3. 利用域限制和传递性证明关键联盟只能是单个人
4. 该个人成为独裁者

**意义**：
- 民主决策面临根本性困境
- 无法同时满足所有合理条件
- 必须在某些条件上做出让步

### 7.4.3 单峰偏好与多数决规则

**定义7.4.3（单峰偏好）**
对于方案集 X = {x₁, x₂, ..., xₖ}（按某种顺序排列），偏好 ≽ᵢ 是单峰的，如果：
- 存在"峰点" x*，使得效用 uᵢ(x) 在 x* 处达到最大
- 对于任何 x, y，如果 x* ≥ x ≥ y 或 x* ≤ x ≤ y，则 uᵢ(x) ≥ uᵢ(y)

**定理7.4.2（中位选民定理）**
如果：
1. 方案集 X = ℝ（单维）
2. 所有个体偏好是单峰的
3. 使用多数决规则

则多数决的社会偏好是传递的，且社会选择结果为中位选民的最优方案。

**定义7.4.4（中位选民）**
个体 m 是中位选民，如果满足：
- 至少半数个体偏好 ≤ xᵐ
- 至少半数个体偏好 ≥ xᵐ

**证明**：
设个体按峰点排序：x*₁ ≤ x*₂ ≤ ... ≤ x*ₙ

中位选民 m：满足 x*ₘ 是中位数。

对于任何方案 y：
- 如果 y < x*ₘ：多数人（m, m+1, ..., n）偏好 x*ₘ 优于 y
- 如果 y > x*ₘ：多数人（1, 2, ..., m）偏好 x*ₘ 优于 y
- 因此 x*ₘ 在多数决下获胜

证毕。

**意义**：
- 单峰偏好假设下，多数决规则是合理的
- 中间立场者决定社会选择

### 7.4.4 策略性投票与吉巴德-萨特斯韦特定理

**定义7.4.5（策略性投票）**
策略性投票是指投票者不根据自己的真实偏好投票，而是为了影响选举结果而进行策略性行为。

**定理7.4.3（吉巴德-萨特斯韦特定理）**
如果社会选择规则满足：
1. 无限制域
2. 非独裁性
3. 对至少3个方案做出选择
4. 激励相容（如实投票是最优策略）

则该规则必须是独裁性的。

**意义**：
- 在多数决等非独裁规则下，策略性投票是不可避免的
- 无法设计出既能保证效率又能防策略性的民主投票机制

**例子7.4.1（策略性投票）**
3个选民，3个方案 {A, B, C}，偏好：

选民1：A ≻ B ≻ C
选民2：B ≻ C ≻ A
选民3：C ≻ A ≻ B

多数决下：A vs B（1和3选A），A vs C（1和2选A）⇒ A当选
       B vs C（1和2选B），B vs A（2和3选A）⇒ A当选
       C vs A（2和3选C），C vs B（1和3选A）⇒ A当选

但如果选民3策略性地将A排在最后，投票顺序：C ≻ B ≻ A
则 A vs B（1选A），A vs C（1选A），但最终结果会变化...

### 7.4.5 放松阿罗条件的可能性

阿罗定理的前提很强，放松某些条件可能得到积极结果：

1. **放宽无限制域**
   - 单峰偏好
   - 价值限制偏好
   - 单峰+单谷偏好

2. **放宽无关方案独立性**
   - 引入基数效用或强度信息
   - 允许考虑偏好强度

3. **放宽传递性要求**
   - 允许社会偏好非传递（如循环多数）
   - 使用准传递性或非循环性

4. **放宽方案数量限制**
   - 如果只有2个方案，多数决规则满足所有条件

**定理7.4.4（森的帕累托自由不可能定理）**
即使放宽IIA，也可能导致新的不可能性。

**应用：实际社会选择机制**

1. **简单多数决**
   - 优点：简单、透明
   - 缺点：循环多数、策略性投票

2. **排序投票（Borda计数）**
   - 给每个方案赋分，求总分
   - 违反IIA

3. **认可投票（Approval Voting）**
   - 投票者认可所有可接受的方案
   - 不违反IIA，但可能产生策略性投票

4. **比例代表制**
   - 反映少数群体偏好
   - 但可能产生联合政府的不稳定性

## 7.5 公平与效率

### 7.5.1 公平的衡量

**定义7.5.1（洛伦兹曲线）**
洛伦兹曲线描述收入或财富分配的不平等程度。

设收入排序：y₁ ≤ y₂ ≤ ... ≤ yₙ

累计收入比例：
```
L(p) = (Σᵢ=1^{[pn]} yᵢ) / (Σᵢ=1ⁿ yᵢ)
```

其中 p ∈ [0, 1] 是累计人口比例。

**定义7.5.2（基尼系数）**
基尼系数是洛伦兹曲线与45度线之间的面积占45度线下总面积的比例：

```
G = 1 - 2∫₀¹ L(p)dp = 1 - 2Σᵢ (L(pᵢ) + L(pᵢ₋₁))(pᵢ - pᵢ₋₁)
```

基尼系数的取值范围：
- G = 0：完全平等
- G = 1：完全不平等（一人拥有全部收入）

**离散形式**：
```
G = 1 - (2/(n+1))·Σᵢ [Σⱼ=1ⁱ yⱼ] / (Σⱼ=1ⁿ yⱼ)
```

**其他不平等衡量指标**：

1. **变异系数（Coefficient of Variation, CV）**
   ```
   CV = σ/μ
   ```
   其中 σ 是标准差，μ 是均值

2. **泰尔指数（Theil Index）**
   ```
   T = (1/n)·Σᵢ (yᵢ/μ)·ln(yᵢ/μ)
   ```

3. **阿特金森指数（Atkinson Index）**
   ```
   A = 1 - [ (1/n)·Σᵢ (yᵢ/μ)^{1-ε} ]^{1/(1-ε)}
   ```
   其中 ε 是不平等厌恶参数

### 7.5.2 公平配置的公理化方法

**定义7.5.3（嫉妒自由配置，Envy-Free）**
配置 x 是嫉妒自由的，如果对于所有 i, j：
```
xᵢ ≥ᵢ xⱼ
```

即没有人想要别人的配置。

**定义7.5.4（平等主义配置，Egalitarian）**
配置 x 是平等主义的，如果所有个体获得相同的效用：
```
u₁(x₁) = u₂(x₂) = ... = uᵢ(xᵢ)
```

**定义7.5.5（公平配置，Fair）**
配置 x 是公平的，如果它既是嫉妒自由的，又是帕累托最优的。

**定理7.5.1（公平配置的存在性）**
在凸经济中，存在公平配置。

**证明思路**：
1. 从平等分配开始
2. 逐步进行帕累托改进
3. 在保证不产生嫉妒的前提下达到帕累托最优

**例子7.5.1（埃奇沃斯盒中的公平配置）**
考虑两个消费者，禀赋 ω = (1, 1)，效用函数：
u₁(x₁, y₁) = x₁y₁
u₂(x₂, y₂) = x₂y₂

平等分配：x₁ = x₂ = 0.5, y₁ = y₂ = 0.5
效用：u₁ = u₂ = 0.25

通过交换达到帕累托最优，同时保持效用相等。

**定理7.5.3（最优所得税理论）**
Mirrlees模型：政府通过非线性税收在激励和再分配之间权衡。

问题：

其中：
- w 是能力（工资率）
- T(w) 是税收函数
- u(c, w) 是效用函数

**结论**：
- 边际税率在顶端和底端接近0
- 中等收入者税率较高
- 最优税收取决于社会对不平等的态度（ε）

### 7.5.4 公平竞争环境

**定义7.5.6（机会均等）**
机会均等意味着：
1. 所有个体有平等的法律权利
2. 所有个体有平等的教育机会
3. 所有个体有平等的职业机会

**定理7.5.4（补偿原则，Compensation Principle）**
如果配置 y 可以通过配置 x 的补偿检验，则 y 优于 x：

**卡尔多-希克斯标准（Kaldor-Hicks Criterion）**
存在转移支付 τᵢ，使得：
- x'ᵢ = yᵢ + τᵢ ≥ᵢ xᵢ，对所有 i
- Σᵢ τᵢ = 0

**西托夫斯基标准（Scitovsky Criterion）**
配置 y 优于 x，如果：
1. 存在补偿使 y 帕累托优于 x（卡尔多标准）
2. 不存在补偿使 x 帕累托优于 y（反向标准）

**问题**：
- 传递性不满足
- 可能产生循环

**定理7.5.5（伯格森-萨缪尔森社会排序）**
使用社会福利函数可以解决补偿原则的传递性问题。

### 7.5.5 社会正义的现代观点

**森的能力方法（Capability Approach）**
关注个体的"能力"（Capabilities）和"功能"（Functionings）：

- 功能：个体实际实现的状态（如健康、教育）
- 能力：个体能够实现的功能的集合
- 自由：选择的能力

**意义**：
- 超越收入和效用的衡量
- 关注发展质量而非仅仅数量
- 人类发展指数（HDI）的应用

**定义7.5.7（人类发展指数）**
HDI = (LEI + EI + II)/3

其中：
- LEI（寿命指数）：预期寿命
- EI（教育指数）：识字率和入学率
- II（收入指数）：人均GDP（对数）

**公平-效率的现代政策启示**：

1. **教育投资**
   - 提高人力资本
   - 促进机会均等
   - 长期减少不平等

2. **累进税收**
   - 再分配收入
   - 资助公共服务
   - 但需考虑激励效应

3. **社会保障**
   - 保障基本生活水平
   - 提供风险分担
   - 减少贫困

4. **反歧视法律**
   - 保障平等权利
   - 促进机会均等
   - 消除系统性不平等

## 本章总结

### 核心要点

1. **帕累托最优**
   - 帕累托最优是基本的效率标准
   - 在竞争市场下，瓦尔拉斯均衡是帕累托最优的（第一福利定理）
   - 在凸性条件下，任何帕累托最优都可以通过适当的禀赋再分配实现（第二福利定理）

2. **福利经济学定理**
   - 第一定理强调市场效率，但需要竞争、无外部性、完全信息等强假设
   - 第二定理强调效率与公平的分离，但需要凸性等条件
   - 次优理论指出，当某些条件不满足时，最优配置不满足其他帕累托条件

3. **社会福利函数**
   - 功利主义：关注效用总和
   - 罗尔斯主义：关注最差者的福利
   - CES形式：在功利主义和罗尔斯主义之间权衡
   - 社会福利函数最大化可以得到帕累托最优配置

4. **社会选择理论**
   - 阿罗不可能定理：无法同时满足无限制域、帕累托、IIA、非独裁性
   - 单峰偏好下，多数决规则是合理的（中位选民定理）
   - 吉巴德-萨特斯韦特定理：非独裁规则下，策略性投票不可避免

5. **公平与效率**
   - 公平可以定义为嫉妒自由、平等主义等
   - 效率-公平权衡取决于社会福利函数的形式
   - 机会均等比结果平等更重要
   - 现代政策关注能力、机会、自由

### 重要概念表格

| 概念 | 定义 | 特点 |
|------|------|------|
| 帕累托最优 | 无法在不使任何人变差的情况下使至少一人变好 | 弱效率标准，可能有多个最优配置 |
| 瓦尔拉斯均衡 | 所有市场同时出清的竞争均衡 | 在标准假设下是帕累托最优的 |
| 社会福利函数 | 从个体效用到社会福利的映射 | 不同形式反映不同的公平观 |
| 功利主义SWF | W = Σᵢ uᵢ | 关注总量，不考虑分配 |
| 罗尔斯主义SWF | W = min{uᵢ} | 关注最差者，重视公平 |
| 洛伦兹曲线 | 描述收入分配不平等的曲线 | 曲线越弯曲，不平等越高 |
| 基尼系数 | 洛伦兹曲线与45度线之间的面积 | 0到1，越大越不平等 |
| 嫉妒自由配置 | 没有人想要别人的配置 | 公平的基本要求 |
| 中位选民定理 | 单峰偏好下，多数决结果为中位选民偏好 | 解释两党制政策趋同 |
| 阿罗不可能定理 | 无法同时满足所有合理的社会选择条件 | 民主决策的根本困境 |

### 主要定理汇总

| 定理 | 内容 | 条件 |
|------|------|------|
| 福利经济学第一定理 | 瓦尔拉斯均衡是帕累托最优的 | 竞争、无外部性、完全信息 |
| 福利经济学第二定理 | 帕累托最优可通过竞争市场实现 | 凸性、连续性、局部非饱和 |
| 利普西-兰卡斯特定理 | 次优最优不满足其他帕累托条件 | 某些约束无法满足 |
| 阿罗不可能定理 | 无法满足所有社会选择条件 | 无限制域、帕累托、IIA |
| 中位选民定理 | 多数决结果为中位选民偏好 | 单峰偏好 |
| 吉巴德-萨特斯韦特定理 | 非独裁规则下策略性投票不可避免 | 3个以上方案 |
| 公平配置存在性 | 存在既是嫉妒自由又是帕累托最优的配置 | 凸经济 |
