## 2.1 偏好关系

### 2.1.1 偏好的定义与性质

**定义2.1.1（消费集）**：消费集X是消费者可以选择的所有消费束的集合。典型地，X = ℝ^n_+ = {x ∈ ℝ^n | x_i ≥ 0, i = 1, ..., n}，其中x_i表示商品i的消费量。

**定义2.1.2（偏好关系）**：消费集X上的二元关系≽称为偏好关系，如果满足：
1. **完备性**：对于任意x, y ∈ X，要么x ≽ y，要么y ≽ x（或两者都成立）。
2. **自反性**：对于任意x ∈ X，x ≽ x。
3. **传递性**：对于任意x, y, z ∈ X，如果x ≽ y且y ≽ z，则x ≽ z。

**定义2.1.3（严格偏好与无差异）**：
- x ≻ y（严格偏好）：x ≽ y但y ≼ x不成立。
- x ~ y（无差异）：x ≽ y且y ≽ x。

**定理2.1.1（无差异的性质）**：~是X上的等价关系。

**证明**：
1. 自反性：x ~ x，因为x ≽ x。
2. 对称性：如果x ~ y，则x ≽ y且y ≽ x，因此y ~ x。
3. 传递性：如果x ~ y且y ~ z，则x ≽ y, y ≽ x, y ≽ z, z ≽ y。由传递性，x ≽ z且z ≽ x，因此x ~ z。□

### 2.1.2 偏好的连续性

**定义2.1.4（偏好关系的连续性）**：偏好关系≽是连续的，如果对于任意x ∈ X，上轮廓集{y ∈ X | y ≽ x}和下轮廓集{y ∈ X | y ≼ x}都是闭集。

**等价定义2.1.1**：偏好关系≽是连续的，如果对于任意收敛序列x_k → x和y_k → y，如果x_k ≽ y_k对所有k成立，则x ≽ y。

**定理2.1.2（连续偏好的性质）**：连续偏好关系的无差异类是闭集。

**证明**：设无差异类I(x) = {y ∈ X | y ~ x}。由于I(x) = {y | y ≽ x} ∩ {y | y ≼ x}是两个闭集的交集，因此是闭集。□

**例2.1.1（字典序偏好）**：定义x ≽ y当且仅当：要么x₁ > y₁；要么x₁ = y₁且x₂ ≥ y₂。这种偏好不是连续的。例如，考虑序列x^k = (1 + 1/k, 0) → (1, 0) = x，y^k = (1, 1/k) → (1, 0) = y。对于每个k，x^k ≻ y^k（因为1 + 1/k > 1），但在极限处x ~ y（因为x = y）。这违反了连续性。

**图形分析2.1.1**：
```
字典序偏好的无差异曲线:

商品2
  |
  |    y^k = (1, 1/k)
  |      •
  |      |
  |      •
  |      |

```

### 2.1.3 偏好的单调性

**定义2.1.5（单调性）**：偏好关系≽是单调的，如果对于任意x, y ∈ X，x ≥ y（即x_i ≥ y_i对所有i）且x ≠ y ⇒ x ≻ y。

**定义2.1.6（弱单调性）**：偏好关系≽是弱单调的，如果对于任意x, y ∈ X，x ≥ y ⇒ x ≽ y。

**定义2.1.7（局部非饱和性）**：偏好关系≽是局部非饱和的，如果对于任意x ∈ X和任意ε > 0，存在y ∈ X使得‖y - x‖ < ε且y ≻ x。

**定理2.1.3（单调性、弱单调性与局部非饱和性的关系）**：
单调性 ⇒ 弱单调性 ⇒ 局部非饱和性

**例2.1.2**：效用函数u(x) = -∑_{i=1}^n x_i表示消费者偏好较少的消费（厌恶商品）。这个偏好不是单调的，但满足弱单调性。如果允许商品为负（如污染），这种偏好可能成立。

### 2.1.4 偏好的凸性

**定义2.1.8（凸偏好）**：偏好关系≽是凸的，如果对于任意x ∈ X，上轮廓集{y ∈ X | y ≽ x}是凸集。

**等价定义2.1.2**：偏好关系≽是凸的，如果对于任意x, y ∈ X且y ≽ x，对于任意λ ∈ [0, 1]，有λy + (1-λ)x ≽ x。

**定义2.1.9（严格凸偏好）**：偏好关系≽是严格凸的，如果对于任意x, y ∈ X，y ≽ x且y ≠ x，对于任意λ ∈ (0, 1)，有λy + (1-λ)x ≻ x。

**图形分析2.1.2**：
```
凸偏好 vs 非凸偏好:

凸偏好:          非凸偏好:
  ●               ●
 /|\              |\
● | ●            ● | ●
 \|/              |/
  ●               ●

凸偏好：上轮廓集是凸的
非凸偏好：上轮廓集不是凸的
```

**定理2.1.4（凸偏好与边际替代率）**：如果偏好是凸的，则无差异曲线是凸向原点的，这意味着边际替代率递减。

**例2.1.3**：科布-道格拉斯效用函数u(x₁, x₂) = x₁^α x₂^β（α, β > 0）产生严格凸偏好。柯西效用函数u(x₁, x₂) = x₁ + x₂产生弱凸偏好（无差异曲线是直线）。

### 2.1.5 显示偏好理论

**定义2.1.10（显示偏好）**：给定价格p和收入m，如果消费者选择x且p·x ≤ m，则x是显示优于任何满足p·y ≤ m的消费束y。记作x R(p, m) y。

**定义2.1.11（显示偏好弱公理，WARP）**：如果x R(p, m) y且y ≠ x，则不能有y R(p', m') x，其中p'·y ≤ m'且p'·x ≤ m'。

**定义2.1.12（显示偏好强公理，SARP）**：如果x通过显示偏好链显示优于y（即x R(p¹, m¹) x¹, x¹ R(p², m²) x², ..., x^k R(p^{k+1}, m^{k+1}) y），则不能有y R(p', m') x。

**定理2.1.5（阿弗里亚特定理）**：如果需求行为满足SARP，则存在理性化该需求行为的效用函数。

**应用2.1.1（检验理性选择）**：显示偏好理论提供了检验消费者选择是否理性的方法。如果观察到违反WARP或SARP的行为，则消费者的选择不能由任何效用函数理性化。

## 2.2 效用函数

### 2.2.1 效用函数的定义

**定义2.2.1（效用函数）**：函数u: X → ℝ是偏好关系≽的效用函数表示，如果对于任意x, y ∈ X：
x ≽ y ⇔ u(x) ≥ u(y)

**定理2.2.1（效用函数的存在性）**：如果消费集X是可数的，则任何完备的偏好关系都有效用函数表示。

**证明**：构造性证明。由于X是可数的，将其元素排序为x¹, x², x³, ...。定义u(x^k) = sup{r ∈ ℚ | x^k ≽ x^j 且 r > u(x^j) 对某些j < k}。这个构造确保了偏好的序性质。□

**定理2.2.2（德布鲁定理）**：如果消费集X是ℝ^n的连通子集，且偏好关系≽是完备的、传递的和连续的，则存在连续的效用函数u: X → ℝ表示该偏好关系。

**证明概要**：
1. 在ℝ中构造稠密子集D。
2. 对于每个消费束x，定义u(x) = inf{r ∈ D | x ≽ y 对某个y ∈ I(r)}，其中I(r)是某个标量消费束的等价类。
3. 连续性保证u是连续的。□

### 2.2.2 效用函数的性质

**定理2.2.3（效用函数的唯一性）**：效用函数是序数概念。如果u(x)是偏好关系≽的效用函数，则任何严格单调递增变换v(x) = f(u(x))也是表示同一偏好的效用函数。

**证明**：设f: ℝ → ℝ是严格单调递增函数。由于x ≽ y ⇔ u(x) ≥ u(y) ⇔ f(u(x)) ≥ f(u(y)) ⇔ v(x) ≥ v(y)，因此v也是效用函数。□

**定理2.2.4（效用函数与偏好性质的关系）**：
1. 偏好是单调的 ⇔ 效用函数是严格单调递增的。
2. 偏好是凸的 ⇔ 效用函数是拟凹的。
3. 偏好是严格凸的 ⇔ 效用函数是严格拟凹的。

**证明（凸偏好 ⇒ 拟凹效用函数）**：设偏好是凸的，u是效用函数。考虑任意x, y和λ ∈ [0, 1]。由于偏好是凸的，如果u(y) ≥ u(x)，则λy + (1-λ)x ≽ x，因此u(λy + (1-λ)x) ≥ u(x)。设t = u(x)，则{x | u(x) ≤ t}是凸集，这是拟凹函数的定义。□

### 2.2.3 常见效用函数

**例2.2.1（科布-道格拉斯效用函数）**：
u(x) = ∏_{i=1}^n x_i^α_i, α_i > 0

性质：
- 严格单调递增（如果所有α_i > 0）
- 严格拟凹（因此代表严格凸偏好）
- 边际替代率：MRS_{ij} = (∂u/∂x_i)/(∂u/∂x_j) = (α_i x_j)/(α_j x_i)

**例2.2.2（CES效用函数）**：
u(x) = (∑_{i=1}^n α_i x_i^ρ)^{1/ρ}, α_i > 0, ρ ≠ 0

替代弹性：σ = 1/(1-ρ)

特殊情形：
- ρ = 1：线性效用函数（完全替代）
- ρ → 0：科布-道格拉斯效用函数（σ = 1）
- ρ → -∞：里昂惕夫效用函数（完全互补）

**例2.2.3（拟线性效用函数）**：
u(x) = v(x₁, ..., x_{n-1}) + x_n

性质：
- 商品n是计价物，其边际效用恒为1
- 对商品n的需求不依赖于收入（收入效应为零）
- 在福利分析中特别有用

### 2.2.4 效用函数的可微性

**定义2.2.3（可微效用函数）**：效用函数u: X → ℝ是可微的，如果其所有偏导数存在且连续。

**定理2.2.5（边际替代率）**：对于可微的严格拟凹效用函数，商品i对商品j的边际替代率为：
MRS_{ij} = MU_i/MU_j = (∂u/∂x_i)/(∂u/∂x_j)

在最优消费束处，MRS_{ij} = p_i/p_j。

**证明**：在最优处，无差异曲线与预算线相切。无差异曲线的斜率为-MU_i/MU_j，预算线的斜率为-p_i/p_j。相切条件给出MU_i/MU_j = p_i/p_j。□

## 2.3 消费者最优化问题

### 2.3.1 效用最大化问题（UMP）

**问题2.3.1（效用最大化问题）**：
max_{x ∈ ℝ^n_+} u(x)
s.t. p·x ≤ m

其中p ∈ ℝ^n_++是价格向量，m > 0是收入。

**定义2.3.1（马歇尔需求函数）**：效用最大化问题的解x(p, m)称为马歇尔需求函数（或瓦尔拉斯需求函数）。

**定义2.3.2（间接效用函数）**：V(p, m) = u(x(p, m))是效用最大化问题的值函数，称为间接效用函数。

**定理2.3.1（效用最大化问题的存在性）**：如果满足：
1. 效用函数u是连续的
2. 预算集合B(p, m) = {x ∈ ℝ^n_+ | p·x ≤ m}是非空紧集

则效用最大化问题有解。

**证明**：u是连续函数，B(p, m)是紧集。由极值定理，u在B(p, m)上达到最大值。□

**定理2.3.2（效用最大化问题的唯一性）**：如果效用函数u是严格拟凹的，则效用最大化问题有唯一解。

**证明**：假设有两个不同的最优解x'和x''。由于u(x') = u(x'') = V(p, m)，且u是严格拟凹的，对于λ ∈ (0, 1)，有u(λx' + (1-λ)x'') > V(p, m)。这与V(p, m)是最大值矛盾。因此，最优解唯一。□

### 2.3.2 库恩-塔克条件

**定理2.3.3（库恩-塔克条件）**：设效用函数u是连续可微的，且效用最大化问题的解x*是内点解（x* > 0）。则存在拉格朗日乘数λ ≥ 0，使得：

1. ∂u/∂x_i(x*) = λp_i, i = 1, ..., n
2. p·x* = m（瓦尔拉斯定律）
3. u(x*) = V(p, m)

**证明**：拉格朗日函数为L(x, λ) = u(x) + λ(m - p·x)。一阶条件为∂L/∂x_i = ∂u/∂x_i - λp_i = 0。如果预算约束是紧的（在最优处），则p·x* = m。□

**解释2.3.1（拉格朗日乘数的经济学含义）**：λ是收入的边际效用，即λ = ∂V/∂m。这意味着如果收入增加一单位，效用增加λ单位。

**定理2.3.4（边界解的条件）**：如果最优解x*满足x_i* = 0（即商品i不被消费），则：
∂u/∂x_i(x*) ≤ λp_i

即商品i在x*处的边际效用不超过其价格乘以收入的边际效用。

### 2.3.3 马歇尔需求函数的性质

**定理2.3.5（齐次性）**：马歇尔需求函数是零次齐次的：
x(αp, αm) = x(p, m) 对所有α > 0

**证明**：预算约束αp·x ≤ αm等价于p·x ≤ m。因此，最大化问题不变，最优解不变。□

**定理2.3.6（瓦尔拉斯定律）**：对于任何p和m，如果效用函数是局部非饱和的，则：
p·x(p, m) = m

即消费者花光所有收入。

**证明**：如果p·x(p, m) < m，由于局部非饱和性，存在y ∈ X使得p·y ≤ m且u(y) > u(x(p, m))。这与x(p, m)是最优解矛盾。因此，p·x(p, m) = m。□

**定理2.3.7（连续性）**：如果偏好是连续的且效用函数是严格拟凹的，则马歇尔需求函数x(p, m)在(p, m)上是连续的。

### 2.3.4 间接效用函数的性质

**定理2.3.8（间接效用函数的性质）**：
1. 零次齐次性：V(αp, αm) = V(p, m) 对所有α > 0
2. 对价格非增：如果p' ≥ p，则V(p', m) ≤ V(p, m)
3. 对收入非减：如果m' ≥ m，则V(p, m') ≥ V(p, m)
4. 对价格拟凸：对于任意t，{ (p, m) | V(p, m) ≤ t }是凸集
5. 罗伊恒等式：x_i(p, m) = - (∂V/∂p_i) / (∂V/∂m)

**证明（罗伊恒等式）**：由包络定理，∂V/∂p_i = -λx_i和∂V/∂m = λ，其中λ是收入的边际效用。因此，x_i = - (∂V/∂p_i) / (∂V/∂m)。□

**例2.3.1（科布-道格拉斯效用函数）**：
u(x₁, x₂) = x₁^α x₂^{1-α}, α ∈ (0, 1)

求解效用最大化问题：
1. 一阶条件：αx₁^{α-1}x₂^{1-α} = λp₁, (1-α)x₁^αx₂^{-α} = λp₂
2. 两式相除：αx₂/[(1-α)x₁] = p₁/p₂ ⇒ x₂ = [(1-α)p₁/αp₂]x₁
3. 代入预算约束：p₁x₁ + p₂[(1-α)p₁/αp₂]x₁ = p₁x₁[1 + (1-α)/α] = p₁x₁/α = m
4. 解得：x₁ = αm/p₁, x₂ = (1-α)m/p₂

马歇尔需求函数：x(p, m) = (αm/p₁, (1-α)m/p₂)

间接效用函数：V(p, m) = (αm/p₁)^α ((1-α)m/p₂)^{1-α}

## 2.4 支出最小化问题

### 2.4.1 支出最小化问题（EMP）

**问题2.4.1（支出最小化问题）**：
min_{x ∈ ℝ^n_+} p·x
s.t. u(x) ≥ ū

其中p ∈ ℝ^n_++是价格向量，ū是目标效用水平。

**定义2.4.1（希克斯需求函数）**：支出最小化问题的解h(p, u)称为希克斯需求函数（或补偿需求函数）。

**定义2.4.2（支出函数）**：e(p, u) = p·h(p, u)是支出最小化问题的值函数，称为支出函数。

**定理2.4.1（支出最小化问题的存在性）**：如果满足：
1. 效用函数u是连续的
2. 上轮廓集{ x | u(x) ≥ ū }是非空紧集

则支出最小化问题有解。

### 2.4.2 支出函数的性质

**定理2.4.2（支出函数的性质）**：
1. 对价格一次齐次：e(αp, u) = αe(p, u) 对所有α > 0
2. 对价格非减：如果p' ≥ p，则e(p', u) ≥ e(p, u)
3. 对效用非减：如果u' ≥ u，则e(p, u') ≥ e(p, u)
4. 对价格凹：e(p, u)是p的凹函数
5. 谢泼德引理：∂e/∂p_i = h_i(p, u)

**证明（谢泼德引理）**：由包络定理，e(p, u)的最优值关于p的导数等于拉格朗日函数关于p的偏导数，即∂e/∂p_i = h_i(p, u)。□

**证明（支出函数对价格凹）**：设p = λp' + (1-λ)p''，λ ∈ [0, 1]。则：
e(p, u) = p·h(p, u) = λp'·h(p, u) + (1-λ)p''·h(p, u)
≥ λe(p', u) + (1-λ)e(p'', u)
因为h(p, u)不是p'或p''的最优解，但满足约束u(h(p, u)) ≥ u。□

### 2.4.3 希克斯需求函数的性质

**定理2.4.3（希克斯需求函数的性质）**：
1. 对价格零次齐次：h(αp, u) = h(p, u) 对所有α > 0
2. 对效用非减：如果u' ≥ u，则h(p, u') ≥ h(p, u)
3. 斯卢茨基方程：∂h_i/∂p_j = ∂x_i/∂p_j + x_j (∂x_i/∂m)
4. 替代矩阵对称：∂h_i/∂p_j = ∂h_j/∂p_i
5. 替代矩阵半负定：∑_{i,j} a_i a_j (∂h_i/∂p_j) ≤ 0 对任意a ∈ ℝ^n

**证明（斯卢茨基方程）**：证明概要：
1. 由对偶性，h(p, u) = x(p, e(p, u))
2. 对p_j求导：∂h_i/∂p_j = ∂x_i/∂p_j + ∑_k (∂x_i/∂m)(∂e/∂p_k)(∂p_k/∂p_j)
3. 由谢泼德引理，∂e/∂p_j = h_j(p, u) = x_j(p, e(p, u))
4. 当p_j变化时，k = j时∂p_k/∂p_j = 1，其他为0
5. 因此，∂h_i/∂p_j = ∂x_i/∂p_j + x_j (∂x_i/∂m) □

**解释2.4.1（斯卢茨基方程的经济学含义）**：
- ∂x_i/∂p_j：价格变化的总效应
- ∂h_i/∂p_j：替代效应（保持效用不变）
- x_j (∂x_i/∂m)：收入效应

## 2.5 对偶性

### 2.5.1 效用最大化与支出最小化的对偶性

**定理2.5.1（对偶性关系）**：
1. x(p, m) = h(p, V(p, m))
2. h(p, u) = x(p, e(p, u))

即马歇尔需求可以通过希克斯需求得到，反之亦然。

**证明（第一部分）**：设x* = x(p, m)，则p·x* = m（瓦尔拉斯定律）。定义u* = u(x*) = V(p, m)。由于x*在给定价格和收入下最大化效用，它在给定效用水平u*下最小化支出（否则存在y使得p·y < m且u(y) ≥ u*，这与x*是最优解矛盾）。因此，x* = h(p, u*) = h(p, V(p, m))。□

**定理2.5.2（对偶性等式）**：
1. e(p, V(p, m)) = m
2. V(p, e(p, u)) = u

即支出函数和间接效用函数互为反函数。

**证明（第一部分）**：e(p, V(p, m)) = p·h(p, V(p, m)) = p·x(p, m) = m（由对偶性和瓦尔拉斯定律）。□

### 2.5.2 霍特林引理与谢泼德引理

**霍特林引理**：间接效用函数V(p, m)的导数给出马歇尔需求：
∂V/∂p_i = -λx_i, ∂V/∂m = λ

其中λ是收入的边际效用。

**谢泼德引理**：支出函数e(p, u)的导数给出希克斯需求：
∂e/∂p_i = h_i(p, u)

这两个引理是包络定理在消费者理论中的应用。

### 2.5.3 对偶性的应用

**应用2.5.1（福利变化测度）**：
- 补偿变化（CV）：e(p^0, V(p^1, m^1)) - e(p^0, V(p^0, m^0)) = e(p^0, u^1) - m^0
- 等价变化（EV）：e(p^1, V(p^1, m^1)) - e(p^1, V(p^0, m^0)) = m^1 - e(p^1, u^0)

其中(p^0, m^0)和(p^1, m^1)分别是初始和最终的价格收入组合，u^0 = V(p^0, m^0)，u^1 = V(p^1, m^1)。

**应用2.5.2（价格指数）**：
真实生活成本指数 = e(p^1, u^0)/e(p^0, u^0) = e(p^1, u^0)/m^0

如果真实生活成本指数 > 拉斯佩尔指数（p^1·x^0/p^0·x^0），则消费者从价格变化中受益。

### 2.5.4 需求函数的积分

**定理2.5.3（需求函数的可积性）**：需求函数x(p, m)可以由效用函数导出当且仅当：
1. 零次齐次性：x(αp, αm) = x(p, m)
2. 瓦尔拉斯定律：p·x(p, m) = m
3. 对称的斯卢茨基替代矩阵：S(p, m) = [∂h_i/∂p_j]是对称半负定的

**解释2.5.1（可积性问题的意义）**：给定需求函数，我们可以检验它是否理性（即是否由效用最大化导出）。可积性条件提供了检验方法。

### 2.5.5 替代效应与收入效应的分解

**斯卢茨基方程（另一种形式）**：
∂x_i/∂p_j = ∂h_i/∂p_j - x_j (∂x_i/∂m)

即价格变化的总效应 = 替代效应 - 收入效应

**例2.5.1（科布-道格拉斯需求的斯卢茨基分解）**：
x₁ = αm/p₁, x₂ = (1-α)m/p₂

计算∂x₁/∂p₁ = -αm/p₁²
计算x₁(∂x₁/∂m) = (αm/p₁)(α/p₁) = α²m/p₁²
计算∂h₁/∂p₁ = ∂x₁/∂p₁ + x₁(∂x₁/∂m) = -αm/p₁² + α²m/p₁² = -α(1-α)m/p₁² < 0

验证：替代效应∂h₁/∂p₁ < 0，收入效应x₁(∂x₁/∂m) > 0，总效应∂x₁/∂p₁ < 0。

### 2.5.6 需求弹性

**定义2.5.1（自价格弹性）**：ε_{ii} = (∂x_i/∂p_i)(p_i/x_i)

**定义2.5.2（交叉价格弹性）**：ε_{ij} = (∂x_i/∂p_j)(p_j/x_i), i ≠ j

**定义2.5.3（收入弹性）**：ε_{im} = (∂x_i/∂m)(m/x_i)

**定义2.5.4（斯卢茨基弹性）**：ε^s_{ij} = (∂h_i/∂p_j)(p_j/h_i)

**定理2.5.4（弹性形式的斯卢茨基方程）**：
ε_{ij} = ε^s_{ij} - w_j ε_{im}

其中w_j = p_j x_j/m是商品j的支出份额。

**定理2.5.5（弹性关系）**：
1. 加性：∑_{i=1}^n w_i ε_{ij} = -w_j
2. 恩格尔加性：∑_{i=1}^n w_i ε_{im} = 1
3. 齐次性：∑_{j=1}^n ε_{ij} + ε_{im} = 0

## 本章总结

### 核心要点

1. **偏好关系**：完备性、自反性、传递性、连续性、单调性、凸性是偏好的基本性质。

2. **效用函数**：德布鲁定理保证连续偏好存在连续效用函数表示；效用函数是序数概念，唯一性是严格单调递增变换。

3. **效用最大化问题（UMP）**：在预算约束下最大化效用，解是马歇尔需求函数x(p, m)；值函数是间接效用函数V(p, m)。

4. **支出最小化问题（EMP）**：在效用约束下最小化支出，解是希克斯需求函数h(p, u)；值函数是支出函数e(p, u)。

5. **对偶性**：UMP和EMP是对偶问题；x(p, m) = h(p, V(p, m))和h(p, u) = x(p, e(p, u))；e(p, V(p, m)) = m和V(p, e(p, u)) = u。

6. **重要定理**：罗伊恒等式、谢泼德引理、斯卢茨基方程、霍特林引理。

7. **斯卢茨基分解**：价格变化的总效应 = 替代效应 + 收入效应；替代效应总是负的（或零），收入效应可正可负。

8. **需求函数性质**：齐次性、瓦尔拉斯定律、连续性、对称半负定的斯卢茨基替代矩阵。

### 重要概念表格

| 概念 | 定义/性质 | 经济学含义 |
|------|-----------|------------|
| 马歇尔需求 | x(p, m) = argmax u(x) s.t. p·x ≤ m | 给定价格和收入的最优消费 |
| 希克斯需求 | h(p, u) = argmin p·x s.t. u(x) ≥ u | 给定价格和效用的最小成本消费 |
| 间接效用 | V(p, m) = u(x(p, m)) | 可达到的最大效用 |
| 支出函数 | e(p, u) = p·h(p, u) | 达到给定效用的最小支出 |
| 罗伊恒等式 | x_i = - (∂V/∂p_i)/(∂V/∂m) | 从间接效用函数导出需求 |
| 谢泼德引理 | h_i = ∂e/∂p_i | 从支出函数导出需求 |
| 斯卢茨基方程 | ∂x_i/∂p_j = ∂h_i/∂p_j + x_j(∂x_i/∂m) | 价格效应分解 |
| 替代效应 | ∂h_i/∂p_j | 保持效用不变的价格效应 |
| 收入效应 | x_j(∂x_i/∂m) | 实际收入变化的价格效应 |
