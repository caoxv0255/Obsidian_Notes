## 13.1 局部均衡与一般均衡

### 13.1.1 局部均衡的定义

**局部均衡（Partial Equilibrium）**：
假设其他市场不变，分析一个市场的均衡。

**特点**：
- 只分析一个市场
- 假设其他市场不变
- 不考虑市场之间的相互影响

### 13.1.2 一般均衡的定义

**一般均衡（General Equilibrium）**：
考虑所有市场之间的相互影响，分析所有市场的均衡。

**特点**：
- 分析所有市场
- 考虑市场之间的相互影响
- 所有市场同时均衡

### 13.1.3 局部均衡与一般均衡的区别

**局部均衡与一般均衡的区别**：

| 特征 | 局部均衡 | 一般均衡 |
|------|----------|----------|
| 分析范围 | 一个市场 | 所有市场 |
| 假设 | 其他市场不变 | 考虑市场之间的相互影响 |
| 复杂性 | 简单 | 复杂 |
| 准确性 | 近似 | 精确 |

## 13.2 一般均衡理论的基本概念

### 13.2.1 一般均衡的定义

**一般均衡（General Equilibrium）**：
所有市场同时达到均衡的状态。

### 13.2.2 一般均衡的条件

**一般均衡的条件**：
1. 每个市场的供给等于需求
2. 所有市场同时达到均衡

**数学表达式**：
```
Si(Pi, P1, P2, ..., Pn) = Di(Pi, P1, P2, ..., Pn)  (i = 1, 2, ..., n)
```

**其中**：
- Si：第i个市场的供给
- Di：第i个市场的需求
- Pi：第i个市场的价格
- P1, P2, ..., Pn：其他市场的价格

### 13.2.3 一般均衡的存在性

**一般均衡的存在性（Existence of General Equilibrium）**：
在一定条件下，一般均衡存在。

**条件**：
1. 连续性：供给和需求函数是连续的
2. 凸性：偏好集和生产集是凸的
3. 非饱和性：消费者对商品的偏好是非饱和的

**瓦尔拉斯（Walras）**：
证明了在一定的条件下，一般均衡存在。

## 13.3 两个市场的一般均衡

### 13.3.1 两个市场的模型

**两个市场的模型（Two-Market Model）**：
考虑两个市场之间的相互影响，分析两个市场的一般均衡。

### 13.3.2 两个市场的相互影响

**两个市场的相互影响（Interaction Between Two Markets）**：

**例子**：汽车市场和汽油市场

**汽车市场需求**：
- 汽油价格上升，汽车需求下降
- 汽油价格下降，汽车需求上升

**汽油市场需求**：
- 汽车价格上升，汽油需求下降
- 汽车价格下降，汽油需求上升

### 13.3.3 两个市场的一般均衡

**两个市场的一般均衡（General Equilibrium in Two Markets）**：
汽车市场和汽油市场同时达到均衡。

## 13.4 瓦尔拉斯定律

### 13.4.1 瓦尔拉斯定律的定义

**瓦尔拉斯定律（Walras' Law）**：
所有市场的超额价值之和等于零。

**数学表达式**：
```
Σ Pi × (Di - Si) = 0
```

**其中**：
- Pi：第i个市场的价格
- Di：第i个市场的需求
- Si：第i个市场的供给

### 13.4.2 瓦尔拉斯定律的含义

**瓦尔拉斯定律的含义**：

1. **所有市场的超额价值之和等于零**：
   - 如果一个市场存在超额需求，另一个市场必然存在超额供给
   - 所有市场的超额价值之和等于零

2. **如果n-1个市场达到均衡，第n个市场必然达到均衡**：
   - 如果n-1个市场的超额需求为零，第n个市场的超额需求必然为零
   - 因此，只需要求解n-1个市场的均衡

### 13.4.3 瓦尔拉斯定律的证明

**瓦尔拉斯定律的证明（Proof of Walras' Law）**：

**预算约束**：
```
Σ Pi × Xi = Σ Pi × ωi
```

**其中**：
- Xi：消费者对第i种商品的需求
- ωi：消费者对第i种商品的初始禀赋

**市场需求**：
```
Di = Σ Xi
```

**市场供给**：
```
Si = Σ ωi
```

**超额需求**：
```
Zi = Di - Si
```

**超额价值**：
```
Σ Pi × Zi = Σ Pi × (Di - Si) = Σ Pi × Di - Σ Pi × Si = Σ Pi × Σ Xi - Σ Pi × Σ ωi = 0
```

## 13.5 本章总结

### 核心要点

1. **局部均衡与一般均衡**：
   - 局部均衡：假设其他市场不变，分析一个市场的均衡
   - 一般均衡：考虑所有市场之间的相互影响，分析所有市场的均衡

2. **一般均衡理论的基本概念**：
   - 一般均衡：所有市场同时达到均衡的状态
   - 一般均衡的条件：每个市场的供给等于需求，所有市场同时达到均衡
   - 一般均衡的存在性：在一定条件下，一般均衡存在

3. **两个市场的一般均衡**：
   - 两个市场的模型：考虑两个市场之间的相互影响，分析两个市场的一般均衡
   - 两个市场的相互影响：市场之间存在相互影响

4. **瓦尔拉斯定律**：
   - 瓦尔拉斯定律：所有市场的超额价值之和等于零
   - 瓦尔拉斯定律的含义：如果n-1个市场达到均衡，第n个市场必然达到均衡
