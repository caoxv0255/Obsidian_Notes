## 6.1 完全竞争市场的特征

### 6.1.1 完全竞争市场的定义

**完全竞争市场（Perfectly Competitive Market）**：
一种竞争不受任何阻碍和干扰的市场结构。

### 6.1.2 完全竞争市场的特征

**完全竞争市场的特征**：

1. **大量的买者和卖者（Large Number of Buyers and Sellers）**：
   - 买者和卖者众多
   - 每个买者和卖者只占市场的很小份额
   - 没有单个买者或卖者可以影响价格

2. **产品同质（Homogeneous Product）**：
   - 所有卖者提供相同的产品
   - 产品没有差异
   - 消费者不在意从哪个卖者购买

3. **自由进入和退出（Free Entry and Exit）**：
   - 企业可以自由进入或退出市场
   - 没有进入或退出障碍
   - 资源可以自由流动

4. **完全信息（Perfect Information）**：
   - 买者和卖者了解所有相关信息
   - 没有信息不对称
   - 市场价格透明

5. **价格接受者（Price Taker）**：
   - 买者和卖者不能影响市场价格
   - 只能接受市场价格
   - 市场价格由市场供求决定

## 6.2 厂商的短期均衡

### 6.2.1 厂商的需求曲线

**厂商的需求曲线（Demand Curve Facing the Firm）**：
在完全竞争市场中，厂商面临的需求曲线是水平的。

**原因**：
- 厂商是价格接受者
- 厂商的需求曲线等于市场价格
- 厂商的边际收益 = 价格

### 6.2.2 利润最大化条件

**利润最大化条件（Profit Maximization Condition）**：
边际收益 = 边际成本

**数学推导**：

**利润函数**：
```
π(Q) = TR(Q) - TC(Q)
```

**利润最大化的一阶条件**：
```
dπ/dQ = dTR/dQ - dTC/dQ = 0
MR - MC = 0
MR = MC
```

**含义**：
- 如果 MR > MC，增加产量可以增加利润
- 如果 MR < MC，减少产量可以增加利润
- 如果 MR = MC，利润最大化

**厂商的短期供给曲线（Short-Run Supply Curve）**：
边际成本曲线位于平均可变成本曲线之上的部分。

### 6.2.4 停止营业点

**停止营业点（Shutdown Point）**：
价格等于平均可变成本的最小值。

**停止营业条件**：
- 如果 P < AVC，厂商停止营业
- 如果 P ≥ AVC，厂商继续营业

**原因**：
- 如果 P < AVC，厂商无法弥补可变成本
- 停止营业可以减少损失

## 6.3 厂商的长期均衡

### 6.3.1 进入与退出

**进入（Entry）**：
如果短期利润为正，新厂商进入市场。

**进入条件**：
- 如果 P > ATC，厂商获得正利润
- 新厂商进入
- 供给增加
- 价格下降
- 直到 P = ATC

**退出（Exit）**：
如果短期利润为负，厂商退出市场。

**退出条件**：
- 如果 P < ATC，厂商获得负利润
- 厂商退出
- 供给减少
- 价格上升
- 直到 P = ATC

### 6.3.2 长期均衡

**长期均衡（Long-Run Equilibrium）**：
- P = min(ATC)
- 厂商获得零利润
- 没有进入或退出

## 6.4 完全竞争市场的效率

### 6.4.1 配置效率

**配置效率（Allocative Efficiency）**：
资源配置使总剩余最大化。

**完全竞争市场的配置效率**：
- 完全竞争市场是有效率的
- 市场均衡使总剩余最大化
- 这是福利经济学第一定理

### 6.4.2 生产效率

**生产效率（Productive Efficiency）**：
以最低成本生产商品。

**完全竞争市场的生产效率**：
- 完全竞争市场是有效率的
- 厂商在最低成本点生产
- P = min(ATC)

- 需求曲线下方、价格线上方的区域

**生产者剩余（Producer Surplus, PS）**：
- 价格线下方、供给曲线上方的区域

**总剩余（Total Surplus, TS）**：
- 消费者剩余 + 生产者剩余
- 总剩余最大

**无谓损失（Deadweight Loss, DWL）**：
- 无

## 6.5 本章总结

### 核心要点

1. **完全竞争市场的特征**：
   - 大量的买者和卖者
   - 产品同质
   - 自由进入和退出
   - 完全信息
   - 价格接受者

2. **厂商的短期均衡**：
   - 厂商的需求曲线是水平的
   - 利润最大化条件：MR = MC
   - 厂商的短期供给曲线：MC曲线在AVC曲线之上的部分
   - 停止营业点：P = min(AVC)

3. **厂商的长期均衡**：
   - P = min(ATC)
   - 厂商获得零利润
   - 没有进入或退出

4. **完全竞争市场的效率**：
   - 配置效率：资源配置使总剩余最大化
   - 生产效率：以最低成本生产商品
   - 完全竞争市场是有效率的
