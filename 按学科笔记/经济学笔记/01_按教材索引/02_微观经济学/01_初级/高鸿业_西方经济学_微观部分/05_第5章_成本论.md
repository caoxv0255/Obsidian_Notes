# 第5章：成本论

> 本章学习目标：
> - 理解成本的概念和分类
> - 掌握短期成本和长期成本
> - 学会分析成本曲线
> - 理解规模经济与规模不经济

---

## 5.1 成本的概念

### 5.1.1 成本的分类

**成本的分类**：

1. **显性成本（Explicit Cost）**：
   - 企业需要直接支付的成本
   - 例如：工资、租金、原材料

2. **隐性成本（Implicit Cost）**：
   - 企业使用自有资源的机会成本
   - 例如：所有者放弃的工资、资本收益

3. **机会成本（Opportunity Cost）**：
   - 为了得到某种东西而放弃的其他东西中价值最高的那一个
   - 机会成本 = 显性成本 + 隐性成本

### 5.1.2 会计成本与经济成本

**会计成本（Accounting Cost）**：
显性成本。

**经济成本（Economic Cost）**：
机会成本 = 显性成本 + 隐性成本。

**例子**：

**开餐厅**：
- 会计成本：租金10,000元/月，工资20,000元/月，原材料5,000元/月
- 隐性成本：所有者放弃的工资30,000元/月
- 经济成本 = 10,000 + 20,000 + 5,000 + 30,000 = 65,000元/月

## 5.2 短期成本

### 5.2.1 短期成本的分类

**短期成本的分类**：

1. **固定成本（Fixed Cost, FC）**：
   - 不随产量变化的成本
   - 例如：租金、机器折旧、管理人员工资

2. **可变成本（Variable Cost, VC）**：
   - 随产量变化的成本
   - 例如：原材料、工人工资、能源

3. **总成本（Total Cost, TC）**：
   - 固定成本和可变成本的总和
   - TC = FC + VC

4. **平均固定成本（Average Fixed Cost, AFC）**：
   - 每单位产量的固定成本
   - AFC = FC / Q

5. **平均可变成本（Average Variable Cost, AVC）**：
   - 每单位产量的可变成本
   - AVC = VC / Q

6. **平均总成本（Average Total Cost, ATC）**：
   - 每单位产量的总成本
   - ATC = TC / Q
   - ATC = (FC + VC) / Q = AFC + AVC

7. **边际成本（Marginal Cost, MC）**：
   - 多生产一单位产量的成本
   - MC = ΔTC / ΔQ = ΔVC / ΔQ

## 5.3 长期成本

### 5.3.1 长期成本曲线

**长期成本曲线（Long-Run Cost Curve）**：
所有投入都是可变的成本曲线。

**长期成本曲线的类型**：

1. **长期平均成本曲线（Long-Run Average Cost Curve, LRAC）**：
   - 表示长期中各种产量下的平均成本
   - LRAC曲线是短期平均成本曲线的包络线

2. **长期边际成本曲线（Long-Run Marginal Cost Curve, LRMC）**：
   - 表示长期中多生产一单位产量的成本
   - LRMC曲线通过LRAC曲线的最低点

## 5.4 规模经济与规模不经济

### 5.4.1 规模经济

**规模经济（Economies of Scale）**：
随着产量增加，平均总成本下降。

**原因**：
1. **专业化（Specialization）**：
   - 工人专业化
   - 提高效率

2. **规模效率（Scale Efficiency）**：
   - 大规模生产
   - 降低平均成本

3. **技术效率（Technological Efficiency）**：
   - 大规模使用先进技术
   - 降低平均成本

### 5.4.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
随着产量增加，平均总成本上升。

**原因**：
1. **管理问题（Management Problems）**：
   - 企业规模过大
   - 管理困难

2. **沟通问题（Communication Problems）**：
   - 企业规模过大
   - 沟通困难

3. **协调问题（Coordination Problems）**：
   - 企业规模过大
   - 协调困难

### 5.4.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
随着产量增加，平均总成本不变。

## 5.5 本章总结

### 核心要点

1. **成本的概念**：
   - 显性成本：企业需要直接支付的成本
   - 隐性成本：企业使用自有资源的机会成本
   - 机会成本：为了得到某种东西而放弃的其他东西中价值最高的那一个
   - 会计成本：显性成本
   - 经济成本：机会成本 = 显性成本 + 隐性成本

2. **短期成本**：
   - 固定成本、可变成本、总成本
   - 平均固定成本、平均可变成本、平均总成本
   - 边际成本
   - 成本曲线

3. **长期成本**：
   - 长期平均成本曲线：短期平均成本曲线的包络线
   - 长期边际成本曲线：通过长期平均成本曲线的最低点

4. **规模经济与规模不经济**：
   - 规模经济：随着产量增加，平均总成本下降
   - 规模不经济：随着产量增加，平均总成本上升
   - 规模报酬不变：随着产量增加，平均总成本不变
