# 第2章：需求、供给和均衡价格

## 2.1 需求

### 2.1.1 需求的定义

**需求（Demand）**：
买者在各种价格水平上愿意并且能够购买的商品数量。

**需求量（Quantity Demanded）**：
在特定价格下，买者愿意并且能够购买的商品数量。

**关键要素**：
1. **购买意愿（Willingness to Buy）**：
   - 有购买欲望

2. **购买能力（Ability to Buy）**：
   - 有支付能力

**需求与需求量的区别**：
- 需求：整个需求曲线
- 需求量：需求曲线上的一点

### 2.1.2 需求定理

**需求定理（Law of Demand）**：
在其他条件不变的情况下，商品的价格与需求量呈反方向变动关系。

**表述**：
- 价格上升，需求量减少
- 价格下降，需求量增加

**原因**：
1. **替代效应（Substitution Effect）**：
   - 价格上升 → 相对价格上升 → 消费者转向替代品
   - 价格下降 → 相对价格下降 → 消费者转向该商品

2. **收入效应（Income Effect）**：
   - 价格上升 → 实际收入下降 → 消费者减少购买
   - 价格下降 → 实际收入上升 → 消费者增加购买

### 2.1.3 需求曲线

**需求曲线（Demand Curve）**：

**需求表（Demand Schedule）**：

**需求函数（Demand Function）**：
```
Qd = a - bP
```

其中：
- Qd：需求量
- P：价格
- a、b：常数（a > 0, b > 0）

### 2.1.4 需求曲线的移动

**需求曲线的移动**：
需求曲线整体向左或向右移动。

**需求曲线移动的原因**：

1. **收入（Income）**：
   - 正常商品（Normal Goods）：收入增加，需求增加，需求曲线右移
   - 低档商品（Inferior Goods）：收入增加，需求减少，需求曲线左移

**例子**：
- 正常商品：收入增加 → 多买衣服 → 需求曲线右移
- 低档商品：收入增加 → 少买方便面 → 需求曲线左移

2. **相关商品的价格**：
   - 替代品（Substitutes）：替代品价格上升，需求增加，需求曲线右移
   - 互补品（Complements）：互补品价格上升，需求减少，需求曲线左移

**例子**：
- 替代品：咖啡价格上升 → 多买茶 → 茶的需求曲线右移
- 互补品：汽油价格上升 → 少买汽车 → 汽车的需求曲线左移

3. **偏好（Tastes）**：
   - 偏好增加，需求增加，需求曲线右移
   - 偏好减少，需求减少，需求曲线左移

**例子**：
- 健康意识增强 → 多买蔬菜 → 蔬菜的需求曲线右移
- 吸烟有害健康 → 少买香烟 → 香烟的需求曲线左移

4. **预期（Expectations）**：
   - 预期价格上升，当前需求增加，需求曲线右移
   - 预期价格下降，当前需求减少，需求曲线左移

**例子**：
- 预期房价上涨 → 当前多买房 → 房屋的需求曲线右移
- 预期汽车降价 → 当前少买车 → 汽车的需求曲线左移

5. **买者数量（Number of Buyers）**：
   - 买者数量增加，需求增加，需求曲线右移
   - 买者数量减少，需求减少，需求曲线左移

**例子**：
- 人口增加 → 买者数量增加 → 需求曲线右移
- 人口减少 → 买者数量减少 → 需求曲线左移

## 2.2 供给

### 2.2.1 供给的定义

**供给（Supply）**：
卖者在各种价格水平上愿意并且能够出售的商品数量。

**供给量（Quantity Supplied）**：
在特定价格下，卖者愿意并且能够出售的商品数量。

**关键要素**：
1. **出售意愿（Willingness to Sell）**：
   - 有出售欲望

2. **出售能力（Ability to Sell）**：
   - 有生产能力

**供给与供给量的区别**：
- 供给：整个供给曲线
- 供给量：供给曲线上的一点

### 2.2.2 供给定理

**供给定理（Law of Supply）**：
在其他条件不变的情况下，商品的价格与供给量呈同方向变动关系。

**表述**：
- 价格上升，供给量增加
- 价格下降，供给量减少

**原因**：
1. **利润动机（Profit Motive）**：
   - 价格上升 → 利润增加 → 企业增加生产

2. **边际成本递增（Increasing Marginal Cost）**：
   - 增加生产 → 边际成本上升 → 需要更高价格

### 2.2.4 供给曲线的移动

**供给曲线的移动**：
供给曲线整体向左或向右移动。

**供给曲线移动的原因**：

1. **投入价格（Input Prices）**：
   - 投入价格上升，成本增加，供给减少，供给曲线左移
   - 投入价格下降，成本减少，供给增加，供给曲线右移

**例子**：
- 工资上升 → 成本增加 → 供给曲线左移
- 原材料价格下降 → 成本减少 → 供给曲线右移

2. **技术（Technology）**：
   - 技术进步，生产效率提高，供给增加，供给曲线右移
   - 技术倒退，生产效率降低，供给减少，供给曲线左移

**例子**：
- 新技术发明 → 生产效率提高 → 供给曲线右移
- 设备老化 → 生产效率降低 → 供给曲线左移

3. **预期（Expectations）**：
   - 预期价格上升，当前供给减少，供给曲线左移
   - 预期价格下降，当前供给增加，供给曲线右移

**例子**：
- 预期房价上涨 → 当前少卖房 → 房屋的供给曲线左移
- 预期汽车降价 → 当前多卖车 → 汽车的供给曲线右移

4. **卖者数量（Number of Sellers）**：
   - 卖者数量增加，供给增加，供给曲线右移
   - 卖者数量减少，供给减少，供给曲线左移

**例子**：
- 新企业进入 → 卖者数量增加 → 供给曲线右移
- 企业退出 → 卖者数量减少 → 供给曲线左移

## 2.3 市场均衡

### 2.3.1 均衡的定义

**均衡（Equilibrium）**：
市场价格达到使供给量和需求量相等的状态。

**均衡价格（Equilibrium Price）**：
供给量和需求量相等时的价格。

**均衡数量（Equilibrium Quantity）**：
供给量和需求量相等时的数量。

**图形表示**：

**图中标记**：

- P*：均衡价格
- Q*：均衡数量

### 2.3.3 均衡的数学求解

**例子**：

**需求函数**：Qd = 600 - 100P

**供给函数**：Qs = -100 + 200P

**均衡条件**：
```
Qd = Qs
600 - 100P = -100 + 200P
700 = 300P
P* = 700/300 = 2.33
```

**均衡数量**：
```
Q* = Qd = Qs = 600 - 100×2.33 = 367
```

### 2.3.4 均衡的动态调整

**过剩（Surplus）**：
供给量大于需求量的情况。

**短缺（Shortage）**：
需求量大于供给量的情况。

**动态调整过程**：

**情况1：价格高于均衡价格**
- 价格高于均衡价格 → 供给量 > 需求量 → 过剩
- 卖者发现商品卖不出去 → 降低价格
- 价格下降 → 供给量减少，需求量增加
- 直到价格回到均衡价格

**情况2：价格低于均衡价格**
- 价格低于均衡价格 → 需求量 > 供给量 → 短缺
- 买者发现商品短缺 → 愿意支付更高价格
- 价格上升 → 需求量减少，供给量增加
- 直到价格回到均衡价格

## 2.4 弹性理论

### 2.4.1 需求价格弹性

**需求价格弹性（Price Elasticity of Demand）**：
衡量需求量对价格变化反应程度的指标。

**计算公式**：
```
需求价格弹性 = 需求量变化的百分比 / 价格变化的百分比
Ed = (%ΔQd) / (%ΔP)
Ed = (ΔQd / Qd) / (ΔP / P)
Ed = (ΔQd / ΔP) × (P / Qd)
```

**需求价格弹性的类型**：

1. **完全缺乏弹性（Perfectly Inelastic）**：
   - 弹性 = 0
   - 需求量不随价格变化

2. **缺乏弹性（Inelastic）**：
   - 0 < 弹性 < 1
   - 需求量变化小于价格变化

3. **单位弹性（Unit Elastic）**：
   - 弹性 = 1
   - 需求量变化等于价格变化

4. **富有弹性（Elastic）**：
   - 弹性 > 1
   - 需求量变化大于价格变化

5. **完全富有弹性（Perfectly Elastic）**：
   - 弹性 = ∞
   - 价格微小变化，需求量变化巨大

### 2.4.2 影响需求价格弹性的因素

**影响因素**：

1. **替代品的可得性**：
   - 替代品越多，弹性越大
   - 替代品越少，弹性越小

2. **商品的性质**：
   - 必需品：弹性小
   - 奢侈品：弹性大

3. **商品在预算中的比重**：
   - 比重大，弹性大
   - 比重小，弹性小

4. **时间跨度**：
   - 短期：弹性小
   - 长期：弹性大

### 2.4.3 供给价格弹性

**供给价格弹性（Price Elasticity of Supply）**：
衡量供给量对价格变化反应程度的指标。

**计算公式**：
```
供给价格弹性 = 供给量变化的百分比 / 价格变化的百分比
Es = (%ΔQs) / (%ΔP)
Es = (ΔQs / Qs) / (ΔP / P)
Es = (ΔQs / ΔP) × (P / Qs)
```

**供给价格弹性的类型**：

1. **完全缺乏弹性（Perfectly Inelastic）**：
   - 弹性 = 0
   - 供给量不随价格变化

2. **缺乏弹性（Inelastic）**：
   - 0 < 弹性 < 1
   - 供给量变化小于价格变化

3. **单位弹性（Unit Elastic）**：
   - 弹性 = 1
   - 供给量变化等于价格变化

4. **富有弹性（Elastic）**：
   - 弹性 > 1
   - 供给量变化大于价格变化

5. **完全富有弹性（Perfectly Elastic）**：
   - 弹性 = ∞
   - 价格微小变化，供给量变化巨大

### 2.4.4 影响供给价格弹性的因素

**影响因素**：

1. **生产要素的流动性**：
   - 流动性大，弹性大
   - 流动性小，弹性小

2. **时间的长短**：
   - 短期：弹性小
   - 长期：弹性大

3. **生产能力的利用程度**：
   - 利用程度低，弹性大
   - 利用程度高，弹性小

4. **进入和退出行业的难易程度**：
   - 容易进入和退出，弹性大
   - 难以进入和退出，弹性小

## 2.6 本章总结

### 核心要点

1. **需求**：
   - 需求的定义和需求定理
   - 需求曲线：向下倾斜
   - 需求曲线移动的原因：收入、相关商品价格、偏好、预期、买者数量

2. **供给**：
   - 供给的定义和供给定理
   - 供给曲线：向上倾斜
   - 供给曲线移动的原因：投入价格、技术、预期、卖者数量

3. **市场均衡**：
   - 均衡的定义
   - 均衡价格和均衡数量
   - 动态调整过程

4. **弹性理论**：
   - 需求价格弹性：类型、影响因素
   - 供给价格弹性：类型、影响因素

5. **均衡的变化**：
   - 需求增加：价格上升，数量增加
   - 供给增加：价格下降，数量增加
