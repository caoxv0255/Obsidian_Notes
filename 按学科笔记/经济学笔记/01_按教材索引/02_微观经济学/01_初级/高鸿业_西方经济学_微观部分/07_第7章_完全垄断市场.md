# 第7章：完全垄断市场

> 本章学习目标：
> - 理解完全垄断市场的特征
> - 掌握垄断厂商的短期均衡和长期均衡
> - 学会分析垄断的效率损失
> - 理解价格歧视

---

## 7.1 完全垄断市场的特征

### 7.1.1 完全垄断市场的定义

**完全垄断市场（Monopoly）**：
只有唯一一个卖者的市场结构。

### 7.1.2 完全垄断市场的特征

**完全垄断市场的特征**：

1. **唯一的卖者（Single Seller）**：
   - 市场上只有一个卖者
   - 卖者就是市场
   - 卖者可以影响市场价格

2. **没有替代品（No Close Substitutes）**：
   - 垄断产品没有替代品
   - 消费者没有其他选择

3. **进入障碍（Barriers to Entry）**：
   - 其他厂商无法进入市场
   - 进入障碍是垄断形成的原因

4. **价格制定者（Price Maker）**：
   - 垄断厂商可以制定价格
   - 垄断厂商不是价格接受者

5. **信息不对称（Information Asymmetry）**：
   - 垄断厂商比消费者更了解产品
   - 消费者缺乏信息

### 7.1.3 垄断的形成原因

**垄断的形成原因**：

1. **资源垄断（Resource Monopoly）**：
   - 控制关键资源
   - 例如：钻石、石油

2. **法律垄断（Legal Monopoly）**：
   - 政府授予垄断权
   - 例如：专利、版权、特许经营权

3. **自然垄断（Natural Monopoly）**：
   - 一个厂商以最低成本提供整个市场的产品
   - 例如：电力、自来水、铁路

4. **规模经济（Economies of Scale）**：
   - 大规模生产降低平均成本
   - 小厂商无法竞争

## 7.2 垄断厂商的需求曲线

### 7.2.1 垄断厂商的需求曲线

**垄断厂商的需求曲线（Demand Curve Facing the Monopoly）**：
垄断厂商面临市场需求曲线，是向右下方倾斜的。

**原因**：
- 垄断厂商是唯一卖者
- 垄断厂商面临市场需求曲线
- 垄断厂商可以影响市场价格

### 7.2.2 垄断厂商的收益曲线

**收益曲线**：

1. **总收益（Total Revenue, TR）**：
   - TR = P × Q

2. **平均收益（Average Revenue, AR）**：
   - AR = TR / Q = P
   - 平均收益等于价格

3. **边际收益（Marginal Revenue, MR）**：
   - MR = ΔTR / ΔQ
   - 边际收益小于价格

## 7.3 垄断厂商的短期均衡

### 7.3.1 利润最大化条件

**利润最大化条件（Profit Maximization Condition）**：
边际收益 = 边际成本

**数学推导**：

**利润函数**：
```
π(Q) = TR(Q) - TC(Q)
```

**利润最大化的一阶条件**：
```
dπ/dQ = dTR/dQ - dTC/dQ = 0
MR - MC = 0
MR = MC
```

### 7.3.2 垄断厂商的短期均衡

**图形表示**：

**垄断厂商的决策**：
1. 根据MR = MC确定产量Q*
2. 根据需求曲线确定价格P*
3. 计算利润：π = TR - TC

**垄断厂商的利润**：
- 如果 P > ATC，垄断厂商获得正利润
- 如果 P = ATC，垄断厂商获得零利润
- 如果 P < ATC，垄断厂商获得负利润

## 7.4 垄断厂商的长期均衡

### 7.4.1 长期均衡

**垄断厂商的长期均衡（Long-Run Equilibrium）**：
- MR = LMC
- 垄断厂商可能获得正利润

**原因**：
- 进入障碍阻止了新厂商进入
- 垄断厂商可以长期维持正利润

1. 根据MR = LMC确定产量Q*
2. 根据需求曲线确定价格P*
3. 计算利润：π = TR - LTC

**与完全竞争的区别**：
- 完全竞争：长期零利润
- 垄断：长期正利润

## 7.5 垄断的效率损失

### 7.5.1 配置效率损失

**配置效率损失（Allocative Inefficiency）**：
垄断导致资源配置无效率。

**原因**：
- 垄断价格高于完全竞争价格
- 垄断产量低于完全竞争产量
- 存在无谓损失

**图形表示**：

```plotly

```

**分析**：

**完全竞争均衡**：
- 价格：Pc
- 产量：Qc
- 总剩余最大

**垄断均衡**：
- 价格：Pm > Pc
- 产量：Qm < Qc
- 总剩余减少

**无谓损失（Deadweight Loss, DWL）**：
- 垄断导致的无谓损失
- 由于产量不足造成

### 7.5.2 生产效率损失

**生产效率损失（Productive Inefficiency）**：
垄断厂商不以最低成本生产。

**原因**：
- 垄断厂商缺乏竞争压力
- 垄断厂商可能不以最低成本生产

## 7.6 价格歧视

### 7.6.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
垄断厂商对不同消费者收取不同价格。

### 7.6.2 价格歧视的条件

**价格歧视的条件**：

1. **市场势力（Market Power）**：
   - 垄断厂商有市场势力
   - 可以影响价格

2. **消费者差异（Consumer Differences）**：
   - 消费者有不同的支付意愿
   - 可以区分消费者

3. **防止套利（Prevent Arbitrage）**：
   - 防止消费者转卖商品
   - 消费者不能从低价购买、高价出售

### 7.6.3 价格歧视的类型

**价格歧视的类型**：

**1. 一级价格歧视（First-Degree Price Discrimination）**：
- 也称为完全价格歧视
- 对每个消费者收取其支付意愿的价格
- 垄断厂商获得全部消费者剩余

**2. 二级价格歧视（Second-Degree Price Discrimination）**：
- 根据购买数量制定不同价格
- 例如：批量折扣

**3. 三级价格歧视（Third-Degree Price Discrimination）**：
- 根据消费者特征制定不同价格
- 例如：学生折扣、老年人折扣、地区价格差异

### 7.6.4 价格歧视的效率

**价格歧视的效率**：

**效率影响**：
- 一级价格歧视：配置有效率，但垄断厂商获得全部剩余
- 二级价格歧视：可能增加配置效率
- 三级价格歧视：可能增加配置效率

## 7.7 政府对垄断的管制

### 7.7.1 政府管制的原因

**政府管制的原因**：
- 垄断导致配置效率损失
- 垄断导致生产效率损失
- 垄断导致消费者剩余减少

### 7.7.2 政府管制的方法

**政府管制的方法**：

**1. 价格管制（Price Regulation）**：
- 政府规定垄断厂商的价格
- 例如：自然垄断的价格管制

**2. 拆分垄断（Break Up Monopoly）**：
- 政府拆分垄断厂商
- 增加竞争

**3. 反垄断法（Antitrust Law）**：
- 政府通过法律限制垄断
- 例如：《反垄断法》

**4. 公有化（Nationalization）**：
- 政府收购垄断厂商
- 成为国有企业

## 7.8 本章总结

### 核心要点

1. **完全垄断市场的特征**：
   - 唯一的卖者
   - 没有替代品
   - 进入障碍
   - 价格制定者
   - 信息不对称

2. **垄断厂商的需求曲线**：
   - 垄断厂商面临市场需求曲线
   - 需求曲线向右下方倾斜
   - 边际收益小于价格

3. **垄断厂商的短期均衡**：
   - 利润最大化条件：MR = MC
   - 垄断厂商可能获得正利润

4. **垄断厂商的长期均衡**：
   - MR = LMC
   - 垄断厂商可能获得正利润
   - 与完全竞争不同，垄断厂商可以长期维持正利润

5. **垄断的效率损失**：
   - 配置效率损失：垄断价格高于完全竞争价格，垄断产量低于完全竞争产量，存在无谓损失
   - 生产效率损失：垄断厂商不以最低成本生产

6. **价格歧视**：
   - 一级价格歧视：对每个消费者收取其支付意愿的价格
   - 二级价格歧视：根据购买数量制定不同价格
   - 三级价格歧视：根据消费者特征制定不同价格

7. **政府对垄断的管制**：
   - 价格管制
   - 拆分垄断
   - 反垄断法
   - 公有化
