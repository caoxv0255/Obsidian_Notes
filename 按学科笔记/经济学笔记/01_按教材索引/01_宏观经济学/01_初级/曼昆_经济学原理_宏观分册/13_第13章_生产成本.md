## 13.1 什么是成本

### 13.1.1 成本的定义

**成本（Cost）**：
生产者为生产商品和服务所付出的代价。

**成本的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **显性成本（Explicit Cost）** | 需要货币支出的成本 | 工资、租金、原材料 |
| **隐性成本（Implicit Cost）** | 不需要货币支出的成本 | 机会成本、折旧 |

### 13.1.2 机会成本

**机会成本（Opportunity Cost）**：
为了得到某种东西而放弃的其他东西的价值。

**机会成本的两个组成部分**：

1. **显性成本（Explicit Cost）**：
   - 需要货币支出
   - 例如：工资、租金

2. **隐性成本（Implicit Cost）**：
   - 不需要货币支出
   - 例如：企业主放弃的工资

**例子**：

假设某人放弃年薪50,000美元的工作，创办自己的企业。

**显性成本**：
- 租金：20,000美元
- 工资：30,000美元
- 原材料：10,000美元
- **总计**：60,000美元

**隐性成本**：
- 放弃的工资：50,000美元
- **总计**：50,000美元

**总成本**：
```
总成本 = 显性成本 + 隐性成本
       = 60,000 + 50,000
       = 110,000美元
```

### 13.1.3 经济利润与会计利润

**经济利润（Economic Profit）**：
总收入减去总成本（包括显性成本和隐性成本）。

**会计利润（Accounting Profit）**：
总收入减去显性成本。

**区别**：

| 利润类型 | 公式 | 说明 |
|----------|------|------|
| **会计利润** | 总收入 - 显性成本 | 财务报表上的利润 |
| **经济利润** | 总收入 - 总成本 | 考虑所有成本 |

**例子**：

假设企业收入为150,000美元。

**会计利润**：
```
会计利润 = 150,000 - 60,000 = 90,000美元
```

**经济利润**：
```
经济利润 = 150,000 - 110,000 = 40,000美元
```

**含义**：
- 会计利润为正：企业盈利
- 经济利润为正：企业比其他选择更好
- 经济利润为零：正常利润（Normal Profit）

## 13.2 生产函数

### 13.2.1 生产函数的定义

**生产函数（Production Function）**：
表示投入和产出之间关系的函数。

**数学表达**：
```
Q = f(K, L)
```

其中：
- Q：产出
- K：资本
- L：劳动

**简化**：
- 假设资本固定，只考虑劳动
- Q = f(L)

### 13.2.2 总产量、边际产量、平均产量

**总产量（Total Product, TP）**：
一定数量的可变投入生产的总产出。

**边际产量（Marginal Product, MP）**：
增加一单位可变投入所增加的产出。

**定义公式**：
```
MP = ΔQ / ΔL
```

**平均产量（Average Product, AP）**：
每单位可变投入的产出。

**定义公式**：
```
AP = Q / L
```

**例子**：

| 劳动（L） | 总产量（Q） | 边际产量（MP） | 平均产量（AP） |
|-----------|-------------|----------------|----------------|
| 0 | 0 | - | - |
| 1 | 10 | 10 | 10 |
| 2 | 25 | 15 | 12.5 |
| 3 | 45 | 20 | 15 |
| 4 | 60 | 15 | 15 |
| 5 | 70 | 10 | 14 |
| 6 | 75 | 5 | 12.5 |
| 7 | 75 | 0 | 10.7 |
| 8 | 70 | -5 | 8.75 |

### 13.2.3 边际产量递减规律

**边际产量递减规律（Law of Diminishing Marginal Product）**：
在其他投入不变的情况下，随着可变投入的增加，边际产量最终会递减。

**关键点**：
- 边际产量先上升后下降

- 当边际产量 = 0时，总产量最大

**原因**：
- 资本固定
- 劳动增加，但资本不足
- 边际产量递减

## 13.3 成本的各种衡量

### 13.3.1 固定成本与可变成本

**固定成本（Fixed Cost, FC）**：
不随产量变化的成本。

**可变成本（Variable Cost, VC）**：
随产量变化的成本。

**总成本（Total Cost, TC）**：
固定成本与可变成本之和。

**定义公式**：
```
TC = FC + VC
```

**例子**：

| 产量（Q） | 固定成本（FC） | 可变成本（VC） | 总成本（TC） |
|-----------|----------------|----------------|--------------|
| 0 | 100 | 0 | 100 |
| 1 | 100 | 50 | 150 |
| 2 | 100 | 90 | 190 |
| 3 | 100 | 120 | 220 |
| 4 | 100 | 160 | 260 |
| 5 | 100 | 220 | 320 |
| 6 | 100 | 300 | 400 |

### 13.3.2 平均成本

**平均固定成本（Average Fixed Cost, AFC）**：
每单位产量的固定成本。

**定义公式**：
```
AFC = FC / Q
```

**平均可变成本（Average Variable Cost, AVC）**：
每单位产量的可变成本。

**定义公式**：
```
AVC = VC / Q
```

**平均总成本（Average Total Cost, ATC）**：
每单位产量的总成本。

**定义公式**：
```
ATC = TC / Q = AFC + AVC
```

**例子**：

| 产量（Q） | FC | VC | TC | AFC | AVC | ATC |
|-----------|----|----|----|-----|-----|-----|
| 1 | 100 | 50 | 150 | 100 | 50 | 150 |
| 2 | 100 | 90 | 190 | 50 | 45 | 95 |
| 3 | 100 | 120 | 220 | 33.3 | 40 | 73.3 |
| 4 | 100 | 160 | 260 | 25 | 40 | 65 |
| 5 | 100 | 220 | 320 | 20 | 44 | 64 |
| 6 | 100 | 300 | 400 | 16.7 | 50 | 66.7 |

### 13.3.3 边际成本

**边际成本（Marginal Cost, MC）**：
多生产一单位产品所增加的成本。

**定义公式**：
```
MC = ΔTC / ΔQ
```

**边际成本与边际产量的关系**：
```
MC = w / MP
```

其中：
- w：工资率
- MP：边际产量

**含义**：
- 边际产量高 → 边际成本低
- 边际产量低 → 边际成本高

**例子**：

| 产量（Q） | TC | MC |
|-----------|----|----|
| 0 | 100 | - |
| 1 | 150 | 50 |
| 2 | 190 | 40 |
| 3 | 220 | 30 |
| 4 | 260 | 40 |
| 5 | 320 | 60 |
| 6 | 400 | 80 |

## 13.4 短期成本与长期成本

### 13.4.1 短期与长期

**短期（Short Run）**：
至少有一种投入是固定的。

**长期（Long Run）**：
所有投入都是可变的。

**区别**：

| 特征 | 短期 | 长期 |
|------|------|------|
| 资本 | 固定 | 可变 |
| 劳动 | 可变 | 可变 |
| 固定成本 | 有 | 无 |
| 规模 | 不变 | 可变 |

## 13.5 规模经济与规模不经济

### 13.5.1 规模经济

**规模经济（Economies of Scale）**：
长期平均成本随产量增加而下降。

**原因**：
1. **专业化（Specialization）**：
   - 规模扩大，分工更细
   - 提高效率

2. **技术因素（Technological Factors）**：
   - 规模扩大，可以采用更先进的技术
   - 降低平均成本

3. **采购优势（Purchasing Advantages）**：
   - 大批量采购，获得折扣
   - 降低成本

4. **融资优势（Financing Advantages）**：
   - 大企业融资成本更低
   - 资金成本降低

### 13.5.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
长期平均成本随产量增加而上升。

**原因**：
1. **管理复杂性（Management Complexity）**：
   - 规模过大，管理困难
   - 协调成本上升

2. **信息不对称（Information Asymmetry）**：
   - 规模过大，信息传递困难
   - 决策效率降低

3. **员工激励（Employee Motivation）**：
   - 规模过大，员工激励困难
   - 效率下降

### 13.5.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
长期平均成本不随产量变化。

## 13.6 成本曲线的应用

### 13.6.1 利润最大化

**利润最大化条件**：
```
MR = MC
```

其中：
- MR：边际收益
- MC：边际成本

**图形分析**：

**利润**：
```
利润 = (P - ATC) × Q
```

### 13.6.2 成本最小化

**成本最小化条件**：
```
MPL / w = MPK / r
```

其中：
- MPL：劳动的边际产量
- w：工资率
- MPK：资本的边际产量
- r：资本价格

**含义**：
- 每美元的边际产量相等
- 成本最小化
