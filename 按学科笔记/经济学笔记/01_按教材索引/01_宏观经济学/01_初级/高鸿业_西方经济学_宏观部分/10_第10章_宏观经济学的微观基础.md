## 10.1 消费理论

### 10.1.1 消费理论的定义

**消费理论（Consumption Theory）**：
研究消费行为和消费决策的理论。

### 10.1.2 绝对收入假说

**绝对收入假说（Absolute Income Hypothesis）**：
消费取决于当期收入。

**凯恩斯消费函数（Keynesian Consumption Function）**：
```
C = a + bY
```

**其中**：
- C：消费
- a：自主消费
- b：边际消费倾向（0 < b < 1）
- Y：当期收入

**特征**：
1. 消费随收入增加而增加
2. 边际消费倾向小于1
3. 平均消费倾向随收入增加而下降

### 10.1.3 相对收入假说

**相对收入假说（Relative Income Hypothesis）**：
消费取决于相对收入。

**杜森贝里消费函数（Duesenberry Consumption Function）**：
```
C_t = a + bY_t + cY_{t-1}
```

**其中**：
- C_t：第t期消费
- a：自主消费
- b：当期收入的边际消费倾向
- Y_t：第t期收入
- c：过去收入的边际消费倾向
- Y_{t-1}：第t-1期收入

**特征**：
1. 消费不仅取决于当期收入，还取决于过去收入
2. 棘轮效应（Ratchet Effect）：收入增加，消费增加；收入减少，消费减少较慢
3. 示范效应（Demonstration Effect）：消费取决于与他人收入的比较

### 10.1.4 生命周期假说

**生命周期假说（Life Cycle Hypothesis）**：
消费取决于一生的收入。

**莫迪利安尼消费函数（Modigliani Consumption Function）**：
```
C = (W + RY) / T
```

**其中**：
- C：消费
- W：财富
- R：年收入
- Y：工作年限
- T：预期寿命

**含义**：
1. 消费者平滑一生的消费
2. 工作时储蓄，退休时负储蓄
3. 财富增加，消费增加

### 10.1.5 持久收入假说

**持久收入假说（Permanent Income Hypothesis）**：
消费取决于持久收入。

**弗里德曼消费函数（Friedman Consumption Function）**：
```
C = a + bY_p
```

**其中**：
- C：消费
- a：自主消费
- b：持久收入的边际消费倾向
- Y_p：持久收入

**持久收入（Permanent Income）**：
```
Y_p = λY_t + (1 - λ)Y_{p, t-1}
```

**其中**：
- Y_p：持久收入
- λ：权重
- Y_t：当期收入
- Y_{p, t-1}：上期持久收入

**含义**：
1. 消费者将收入分为持久收入和暂时收入
2. 消费取决于持久收入，而不是暂时收入
3. 暂时收入的边际消费倾向小于持久收入的边际消费倾向

## 10.2 投资理论

### 10.2.1 投资理论的定义

**投资理论（Investment Theory）**：
研究投资行为和投资决策的理论。

### 10.2.2 投资的类型

**投资的类型（Types of Investment）**：

**1. 企业固定投资（Business Fixed Investment）**：
- 企业在厂房、设备上的投资
- 取决于资本成本和预期收益

**2. 住房投资（Residential Investment）**：
- 家庭在新住房上的投资
- 取决于住房需求和住房价格

**3. 存货投资（Inventory Investment）**：
- 企业存货的变动
- 取决于销售和预期

### 10.2.3 新古典投资理论

**新古典投资理论（Neoclassical Investment Theory）**：
投资取决于资本边际收益和资本成本。

**投资函数（Investment Function）**：
```
I = I(PK, r, δ)
```

**其中**：
- I：投资
- PK：资本价格
- r：利率
- δ：折旧率

**含义**：
1. 资本边际收益大于资本成本，投资增加
2. 资本边际收益小于资本成本，投资减少
3. 资本边际收益等于资本成本，投资不变

### 10.2.4 托宾的q理论

**托宾的q理论（Tobin's q Theory）**：
投资取决于q值。

**q值（q Value）**：
```
q = 企业的市场价值 / 资本的重置成本
```

**投资函数（Investment Function）**：
```
I = f(q)
```

**含义**：
1. q > 1，投资增加
2. q < 1，投资减少
3. q = 1，投资不变

### 10.2.5 加速原理

**加速原理（Acceleration Principle）**：
投资取决于产出的变化。

**投资函数（Investment Function）**：
```
I = v(ΔY)
```

**其中**：
- I：投资
- v：加速系数
- ΔY：产出的变化

**含义**：
1. 产出增加，投资增加
2. 产出减少，投资减少
3. 投资的变化大于产出的变化

## 10.3 货币需求理论

### 10.3.1 货币需求的定义

**货币需求（Money Demand）**：
人们愿意持有的货币数量。

### 10.3.2 货币的交易需求

**货币的交易需求（Transaction Demand for Money）**：
货币用于交易的动机。

**公式**：
```
L1 = kY
```

**其中**：
- L1：货币的交易需求
- k：货币的交易需求对收入的敏感度
- Y：收入

**含义**：
1. 收入增加，货币的交易需求增加
2. 交易需求与收入成正比

### 10.3.3 货币的预防需求

**货币的预防需求（Precautionary Demand for Money）**：
货币用于预防的动机。

**公式**：
```
L2 = kY
```

**其中**：
- L2：货币的预防需求
- k：货币的预防需求对收入的敏感度
- Y：收入

**含义**：
1. 收入增加，货币的预防需求增加
2. 预防需求与收入成正比

### 10.3.4 货币的投机需求

**货币的投机需求（Speculative Demand for Money）**：
货币用于投机的动机。

**公式**：
```
L3 = -hr
```

**其中**：
- L3：货币的投机需求
- h：货币的投机需求对利率的敏感度
- r：利率

**含义**：
1. 利率上升，货币的投机需求减少
2. 投机需求与利率成反比

### 10.3.5 凯恩斯货币需求函数

**凯恩斯货币需求函数（Keynesian Money Demand Function）**：
```
L = L1 + L2 + L3 = kY - hr
```

**其中**：
- L：货币需求
- k：货币需求对收入的敏感度
- Y：收入
- h：货币需求对利率的敏感度
- r：利率

**含义**：
1. 收入增加，货币需求增加
2. 利率上升，货币需求减少
3. 货币需求与收入成正比，与利率成反比

### 10.3.6 弗里德曼货币需求函数

**弗里德曼货币需求函数（Friedman Money Demand Function）**：
```
L/P = f(Y, r, π^e)
```

**其中**：
- L：货币需求
- P：价格水平
- Y：收入
- r：利率
- π^e：预期通货膨胀率

**含义**：
1. 实际货币需求取决于收入、利率和预期通货膨胀率
2. 收入增加，实际货币需求增加
3. 利率上升，实际货币需求减少
4. 预期通货膨胀率上升，实际货币需求减少

## 10.4 本章总结

### 核心要点

1. **消费理论**：
   - 绝对收入假说：消费取决于当期收入
   - 相对收入假说：消费取决于相对收入
   - 生命周期假说：消费取决于一生的收入
   - 持久收入假说：消费取决于持久收入

2. **投资理论**：
   - 投资的类型：企业固定投资、住房投资、存货投资
   - 新古典投资理论：投资取决于资本边际收益和资本成本
   - 托宾的q理论：投资取决于q值
   - 加速原理：投资取决于产出的变化

3. **货币需求理论**：
   - 货币的交易需求：货币用于交易的动机
   - 货币的预防需求：货币用于预防的动机
   - 货币的投机需求：货币用于投机的动机
   - 凯恩斯货币需求函数：L = kY - hr
   - 弗里德曼货币需求函数：L/P = f(Y, r, π^e)
