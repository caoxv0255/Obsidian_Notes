# 第5章：投资理论

> 本章学习目标：
> - 理解投资的基本决定因素
> - 掌握q理论
> - 学会分析调整成本

---

## 5.1 投资的基本决定因素

### 5.1.1 投资的定义

**投资（Investment）**：
购买新的资本品（机器、设备、建筑物等）的行为。

**类型**：
1. **商业固定投资（Business Fixed Investment）**：
   - 企业购买机器、设备、建筑物

2. **住宅投资（Residential Investment）**：
   - 家庭购买房屋

3. **存货投资（Inventory Investment）**：
   - 企业增加存货

### 5.1.2 投资的决定因素

**投资的决定因素（Determinants of Investment）**：

1. **资本边际产出（Marginal Product of Capital, MPK）**：
   - MPK越高，投资回报越高，投资越多

2. **实际利率（Real Interest Rate, r）**：
   - r越高，融资成本越高，投资越少

3. **预期利润（Expected Profits）**：
   - 预期利润越高，投资越多

4. **税收政策（Tax Policy）**：
   - 税收优惠可以刺激投资

5. **不确定性（Uncertainty）**：
   - 不确定性越高，投资越少

## 5.2 q理论

### 5.2.1 q的定义

**q（Tobin's q）**：
企业的市场价值与资本重置成本的比值。

**公式**：
```
q = V / (p_K·K)
```
**其中**：
- V：企业的市场价值
- p_K：资本品的价格
- K：资本存量

### 5.2.2 q理论的设定

**q理论（q Theory of Investment）**：
James Tobin提出，投资取决于q。

**基本假设**：
1. 企业是价格接受者
2. 资本市场完美
3. 调整成本凸

**企业价值最大化问题**：
```
max V = ∫₀^∞ e^(-rt)[F(K, L) - wL - p_K I - C(I, K)] dt
```
**约束条件**：
```
K̇ = I - δK
```
**其中**：
- F(K, L)：生产函数
- w：工资
- p_K：资本品价格
- I：投资
- C(I, K)：调整成本
- r：实际利率
- δ：折旧率

### 5.2.3 q理论的推导

**q理论的推导（Derivation of q Theory）**：

**拉格朗日函数**：
```
ℒ = ∫₀^∞ e^(-rt)[F(K, L) - wL - p_K I - C(I, K) + q(I - δK)] dt
```

**一阶条件**：
```
∂ℒ/∂I = e^(-rt)[-p_K - ∂C/∂I + q] = 0
⇒ q = p_K + ∂C/∂I
```

**调整成本函数**：
```
C(I, K) = K·φ(I/K)
```
**其中**：
- φ(·)：调整成本函数
- I/K：投资率

**一阶条件**：
```
q = p_K + φ'(I/K)
```

**投资函数**：
```
I/K = φ'^(-1)(q - p_K)
```

### 5.2.4 q理论的含义

**q理论的含义（Implications of q Theory）**：

1. **投资取决于q**：
   - q > 1 ⇒ 投资 > 折旧 ⇒ 资本存量上升
   - q = 1 ⇒ 投资 = 折旧 ⇒ 资本存量不变
   - q < 1 ⇒ 投资 < 折旧 ⇒ 资本存量下降

2. **q的经济学含义**：
   - q > 1 ⇒ 企业市场价值 > 资本重置成本 ⇒ 投资有利
   - q < 1 ⇒ 企业市场价值 < 资本重置成本 ⇒ 投资不利

3. **投资调整**：
   - q上升 ⇒ 投资上升
   - q下降 ⇒ 投资下降

## 5.3 调整成本

### 5.3.1 调整成本的定义

**调整成本（Adjustment Costs）**：
企业调整资本存量时发生的成本。

**类型**：
1. **内部调整成本（Internal Adjustment Costs）**：
   - 安装新机器的成本
   - 培训工人的成本

2. **外部调整成本（External Adjustment Costs）**：
   - 资本品价格上升
   - 利率上升

### 5.3.2 调整成本的函数

**调整成本函数（Adjustment Cost Function）**：
```
C(I, K) = K·φ(I/K)
```
**性质**：
1. φ(·)递增：I/K越高，调整成本越高
2. φ(·)凸：φ''(·) > 0，边际调整成本递增

**具体形式**：
```
φ(I/K) = a(I/K) + b(I/K)²/2
```
**其中**：
- a, b：参数

### 5.3.3 调整成本的影响

**调整成本的影响（Effects of Adjustment Costs）**：

1. **投资滞后（Investment Lag）**：
   - 调整成本导致投资滞后
   - 投资对经济冲击的反应较慢

2. **投资平滑（Investment Smoothing）**：
   - 调整成本导致投资平滑
   - 企业避免投资的大幅波动

3. **投资波动性（Investment Volatility）**：
   - 调整成本降低投资波动性
   - 投资调整较慢

## 5.4 投资的波动性

### 5.4.1 投资波动的经验事实

**投资波动的经验事实（Empirical Facts of Investment Volatility）**：

1. **投资的波动性（Volatility of Investment）**：
   - 投资的波动性大于消费的波动性
   - 投资的波动性大于产出的波动性

2. **投资的周期性（Cyclical Behavior of Investment）**：
   - 投资是顺周期的（Procyclical）
   - 投资的波动与经济周期同步

3. **投资的持久性（Persistence of Investment）**：
   - 投资的变化具有持久性
   - 投资的一阶自相关系数较高

### 5.4.2 投资波动性的解释

**投资波动性的解释（Explanation of Investment Volatility）**：

1. **q的波动性（Volatility of q）**：
   - q的波动性较大
   - q的波动导致投资的波动

2. **调整成本（Adjustment Costs）**：
   - 调整成本导致投资滞后
   - 投资对经济冲击的反应较慢

3. **不确定性（Uncertainty）**：
   - 不确定性导致投资波动
   - 不确定性增加 ⇒ 投资下降

4. **融资约束（Financing Constraints）**：
   - 企业无法获得足够的融资
   - 融资约束 ⇒ 投资下降

### 5.4.3 投资波动性的模型

**投资波动性的模型（Models of Investment Volatility）**：

1. **加速原理（Accelerator Principle）**：
   - I = v(Ẏ)
   - 投资取决于产出的变化
   - 产出变化 ⇒ 投资变化

2. **利润理论（Profit Theory）**：
   - I = f(π)
   - 投资取决于利润
   - 利润上升 ⇒ 投资上升

3. **新古典投资理论（Neoclassical Investment Theory）**：
   - I = g(MPK, r)
   - 投资取决于资本边际产出和实际利率
   - MPK上升 ⇒ 投资上升；r上升 ⇒ 投资下降

## 5.5 本章总结

### 核心要点

1. **投资的基本决定因素**：
   - 投资的定义：购买新的资本品的行为
   - 投资的类型：商业固定投资、住宅投资、存货投资
   - 投资的决定因素：资本边际产出、实际利率、预期利润、税收政策、不确定性

2. **q理论**：
   - q的定义：q = V / (p_K·K)
   - q理论的设定：企业价值最大化问题
   - q理论的推导：q = p_K + ∂C/∂I
   - q理论的含义：投资取决于q，q > 1 ⇒ 投资 > 折旧

3. **调整成本**：
   - 调整成本的定义：企业调整资本存量时发生的成本
   - 调整成本的类型：内部调整成本、外部调整成本
   - 调整成本的影响：投资滞后、投资平滑、投资波动性

4. **投资的波动性**：
   - 投资波动的经验事实：投资的波动性大于消费的波动性、投资是顺周期的
   - 投资波动性的解释：q的波动性、调整成本、不确定性、融资约束
   - 投资波动性的模型：加速原理、利润理论、新古典投资理论
