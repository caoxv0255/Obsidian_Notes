## 4.1 持久收入假说

### 4.1.1 生命周期假说

**生命周期假说（Life-Cycle Hypothesis, LCH）**：
Franco Modigliani提出，个人根据其整个生命周期的收入和财富来安排消费。

**基本思想**：
- 个人在年轻时期储蓄，在年老时期消费储蓄
- 消费取决于持久收入（Permanent Income）
- 暂时性收入（Transitory Income）对消费的影响较小

**消费函数**：
```
C_t = α + β·Y_t^P + γ·W_t
```
**其中**：
- C_t：第t期的消费
- Y_t^P：第t期的持久收入
- W_t：第t期的财富
- α, β, γ：参数

### 4.1.2 持久收入假说

**持久收入假说（Permanent Income Hypothesis, PIH）**：
Milton Friedman提出，消费取决于持久收入，而不是暂时性收入。

**基本思想**：
- 持久收入（Permanent Income）：长期平均收入
- 暂时性收入（Transitory Income）：短期收入波动
- 消费取决于持久收入
- 暂时性收入对消费的影响较小

**收入分解**：
```
Y_t = Y_t^P + Y_t^T
```
**其中**：
- Y_t：第t期的收入
- Y_t^P：第t期的持久收入
- Y_t^T：第t期的暂时性收入

**消费函数**：
```
C_t = k·Y_t^P
```
**其中**：
- C_t：第t期的消费
- Y_t^P：第t期的持久收入
- k：边际消费倾向（MPC）

### 4.1.3 持久收入的估计

**持久收入的估计（Estimation of Permanent Income）**：

**适应性预期（Adaptive Expectations）**：
```
Y_t^P = Y_(t-1)^P + λ(Y_t - Y_(t-1)^P)
```
**其中**：
- Y_t^P：第t期的持久收入
- Y_(t-1)^P：第t-1期的持久收入
- λ：调整速度（0 < λ < 1）

**含义**：
- 持久收入是过去收入的加权平均
- 近期收入的权重较大

## 4.2 随机游走假说

### 4.2.1 随机游走假说的设定

**随机游走假说（Random Walk Hypothesis）**：
Robert Hall提出，如果消费者是理性的，消费遵循随机游走。

**基本假设**：
1. 消费者理性
2. 资本市场完美
3. 效用函数二次型
4. 无限期界

**欧拉方程**：
```
E_t[C_(t+1)] = C_t
```
**其中**：
- C_t：第t期的消费
- E_t[C_(t+1)]：第t期对第t+1期消费的预期

### 4.2.2 随机游走假说的推导

**随机游走假说的推导（Derivation of Random Walk Hypothesis）**：

**目标函数**：
```
max E[Σ β^t U(C_t)]
```
**约束条件**：
```
A_(t+1) = (1 + r)(A_t + Y_t - C_t)
```

**一阶条件**：
```
U'(C_t) = β(1 + r)E_t[U'(C_(t+1))]
```

**假设U(·)二次型**：
```
U(C) = aC - bC²/2
```
**边际效用**：
```
U'(C) = a - bC
```

**一阶条件**：
```
a - bC_t = β(1 + r)(a - bE_t[C_(t+1)])
```

**假设β(1 + r) = 1（跨期替代弹性为1）**：
```
a - bC_t = a - bE_t[C_(t+1)]
```
**化简**：
```
E_t[C_(t+1)] = C_t
```

### 4.2.3 随机游走假说的含义

**随机游走假说的含义（Implications of Random Walk Hypothesis）**：

1. **消费不可预测（Consumption Unpredictable）**：
   - 消费的变化是随机的
   - 无法用过去的收入预测未来的消费

2. **平滑消费（Consumption Smoothing）**：
   - 消费者平滑消费
   - 暂时性收入冲击对消费的影响较小

3. **消费对收入变化的反应**：
   - 持久性收入变化 ⇒ 消费变化
   - 暂时性收入变化 ⇒ 消费几乎不变

## 4.3 消费的波动性

### 4.3.1 消费波动的经验事实

**消费波动的经验事实（Empirical Facts of Consumption Volatility）**：

1. **消费的波动性（Volatility of Consumption）**：
   - 总消费的波动性小于收入的波动性
   - 耐用品消费的波动性大于非耐用品消费的波动性

2. **消费的周期性（Cyclical Behavior of Consumption）**：
   - 消费是顺周期的（Procyclical）
   - 消费的波动滞后于收入的波动

3. **消费的持久性（Persistence of Consumption）**：
   - 消费的变化具有持久性
   - 消费的一阶自相关系数较高

### 4.3.2 消费波动性假说

**消费波动性假说（Consumption Volatility Hypotheses）**：

1. **持久收入假说（Permanent Income Hypothesis）**：
   - 消费波动性应该小于收入波动性
   - 消费应该平滑

2. **随机游走假说（Random Walk Hypothesis）**：
   - 消费的变化是随机的
   - 消费不可预测

3. **过度平滑假说（Excess Smoothness Hypothesis）**：
   - 消费过于平滑
   - 消费对收入变化的反应不足

4. **过度敏感性假说（Excess Sensitivity Hypothesis）**：
   - 消费对收入变化过于敏感
   - 消费对暂时性收入变化的反应过大

### 4.3.3 消费波动性的解释

**消费波动性的解释（Explanation of Consumption Volatility）**：

1. **流动性约束（Liquidity Constraints）**：
   - 消费者无法借债
   - 消费受当前收入约束

2. **预防性储蓄（Precautionary Savings）**：
   - 消费者面临收入不确定性
   - 消费者增加储蓄以应对风险

3. **行为因素（Behavioral Factors）**：
   - 消费者非理性
   - 消费者短视

## 4.4 消费的横截面差异

### 4.4.1 消费的生命周期差异

**消费的生命周期差异（Lifecycle Differences in Consumption）**：

1. **年轻时期（Young Age）**：
   - 收入较低
   - 消费低于收入
   - 储蓄率较低

2. **中年时期（Middle Age）**：
   - 收入较高
   - 消费低于收入
   - 储蓄率较高

3. **老年时期（Old Age）**：
   - 收入较低
   - 消费高于收入
   - 储蓄率较低

### 4.4.2 消费的收入差异

**消费的收入差异（Income Differences in Consumption）**：

1. **高收入家庭（High-Income Households）**：
   - 边际消费倾向较低
   - 储蓄率较高

2. **低收入家庭（Low-Income Households）**：
   - 边际消费倾向较高
   - 储蓄率较低

### 4.4.3 消费的其他差异

**消费的其他差异（Other Differences in Consumption）**：

1. **教育差异（Education Differences）**：
   - 教育程度高的家庭储蓄率较高
   - 教育程度高的家庭消费较平滑

2. **财富差异（Wealth Differences）**：
   - 财富高的家庭消费较平滑
   - 财富高的家庭边际消费倾向较低

3. **家庭结构差异（Family Structure Differences）**：
   - 有子女的家庭消费较高
   - 单身家庭消费较低

## 4.5 本章总结

### 核心要点

1. **持久收入假说**：
   - 生命周期假说：个人根据整个生命周期的收入和财富来安排消费
   - 持久收入假说：消费取决于持久收入，而不是暂时性收入
   - 持久收入的估计：Y_t^P = Y_(t-1)^P + λ(Y_t - Y_(t-1)^P)

2. **随机游走假说**：
   - 随机游走假说：E_t[C_(t+1)] = C_t
   - 含义：消费不可预测、平滑消费、消费对暂时性收入变化的反应较小

3. **消费的波动性**：
   - 消费波动的经验事实：消费的波动性小于收入的波动性、消费是顺周期的
   - 消费波动性假说：持久收入假说、随机游走假说、过度平滑假说、过度敏感性假说
   - 消费波动性的解释：流动性约束、预防性储蓄、行为因素

4. **消费的横截面差异**：
   - 消费的生命周期差异：年轻时期、中年时期、老年时期
   - 消费的收入差异：高收入家庭、低收入家庭
   - 消费的其他差异：教育差异、财富差异、家庭结构差异
