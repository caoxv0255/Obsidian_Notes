## 8.1 失业

### 8.1.1 失业的定义

**失业（Unemployment）**：
有劳动能力且有工作意愿的人没有工作的状态。

### 8.1.2 失业的类型

**失业的类型（Types of Unemployment）**：

1. **摩擦性失业（Frictional Unemployment）**：
   - 工人寻找合适工作的时间
   - 原因：信息不对称、劳动力流动性
   - 特点：短期失业、自愿失业

2. **结构性失业（Structural Unemployment）**：
   - 工人的技能与需求不匹配
   - 原因：技术进步、产业结构调整
   - 特点：长期失业、非自愿失业

3. **周期性失业（Cyclical Unemployment）**：
   - 经济周期引起的失业
   - 原因：经济衰退
   - 特点：随经济周期波动、非自愿失业

### 8.1.3 失业率

**失业率（Unemployment Rate）**：
失业人口占劳动力人口的比率。

**公式**：
```
u = U / L
```
**其中**：
- u：失业率
- U：失业人口
- L：劳动力人口

### 8.1.4 自然失业率

**自然失业率（Natural Rate of Unemployment）**：
经济处于长期均衡时的失业率。

**组成部分**：
- 摩擦性失业
- 结构性失业

**决定因素**：
- 劳动力市场的流动性
- 失业救济金
- 劳动力的技能水平
- 产业结构

## 8.2 菲利普斯曲线

### 8.2.1 原始菲利普斯曲线

**原始菲利普斯曲线（Original Phillips Curve）**：
失业率与通货膨胀率之间存在负相关关系。

**公式**：
```
π_t = -β(u_t - u*) + ε_t
```
**其中**：
- π_t：第t期的通货膨胀率
- u_t：第t期的失业率
- u*：自然失业率
- β：参数
- ε_t：供给冲击

**含义**：
- 失业率低于自然失业率 ⇒ 通货膨胀率上升
- 失业率高于自然失业率 ⇒ 通货膨胀率下降

### 8.2.2 附加预期的菲利普斯曲线

**附加预期的菲利普斯曲线（Expectations-Augmented Phillips Curve）**：
失业率与通货膨胀率之间的负相关关系取决于预期通货膨胀率。

**公式**：
```
π_t = π_t^e - β(u_t - u*) + ε_t
```
**其中**：
- π_t：第t期的通货膨胀率
- π_t^e：第t期的预期通货膨胀率
- u_t：第t期的失业率
- u*：自然失业率
- β：参数
- ε_t：供给冲击

**含义**：
- 预期通货膨胀率上升 ⇒ 菲利普斯曲线向上移动
- 失业率低于自然失业率 ⇒ 通货膨胀率上升

### 8.2.3 适应性预期

**适应性预期（Adaptive Expectations）**：
预期通货膨胀率基于过去的通货膨胀率。

**公式**：
```
π_t^e = π_(t-1)
```
**其中**：
- π_t^e：第t期的预期通货膨胀率
- π_(t-1)：第t-1期的实际通货膨胀率

**菲利普斯曲线**：
```
π_t = π_(t-1) - β(u_t - u*) + ε_t
```

**含义**：
- 通货膨胀具有惯性
- 降低通货膨胀率需要短期成本（失业率上升）

### 8.2.4 理性预期

**理性预期（Rational Expectations）**：
预期通货膨胀率基于所有可得信息。

**公式**：
```
π_t^e = E_t[π_t]
```
**其中**：
- π_t^e：第t期的预期通货膨胀率
- E_t[π_t]：第t期对第t期通货膨胀率的预期

**菲利普斯曲线**：
```
π_t = E_t[π_t] - β(u_t - u*) + ε_t
```

**含义**：
- 预期通货膨胀率等于实际通货膨胀率（如果预期理性）
- 失业率等于自然失业率（如果预期理性）

## 8.3 通货膨胀

### 8.3.1 通货膨胀的定义

**通货膨胀（Inflation）**：
价格水平持续上升。

### 8.3.2 通货膨胀的类型

**通货膨胀的类型（Types of Inflation）**：

1. **需求拉动型通货膨胀（Demand-Pull Inflation）**：
   - 总需求上升 ⇒ 价格水平上升
   - 原因：扩张性财政政策、扩张性货币政策
   - 特点：经济过热

2. **成本推动型通货膨胀（Cost-Push Inflation）**：
   - 成本上升 ⇒ 价格水平上升
   - 原因：工资上升、原材料价格上升
   - 特点：滞胀（stagflation）

3. **结构型通货膨胀（Structural Inflation）**：
   - 产业结构调整 ⇒ 价格水平上升
   - 原因：需求结构变化、供给结构变化
   - 特点：结构性问题

### 8.3.3 通货膨胀的成本

**通货膨胀的成本（Costs of Inflation）**：

1. **鞋底成本（Shoeleather Costs）**：
   - 通货膨胀率高 ⇒ 货币持有成本高 ⇒ 减少货币持有 ⇒ 交易成本上升

2. **菜单成本（Menu Costs）**：
   - 通货膨胀率高 ⇒ 调整价格频率高 ⇒ 调整价格成本上升

3. **相对价格扭曲（Relative Price Distortions）**：
   - 通货膨胀率高 ⇒ 相对价格变化 ⇒ 资源配置扭曲

4. **税收扭曲（Tax Distortions）**：
   - 通货膨胀率高 ⇒ 名义收入上升 ⇒ 税收增加 ⇒ 实际收入下降

5. **财富再分配（Wealth Redistribution）**：
   - 通货膨胀率高 ⇒ 债权人受损，债务人受益

## 8.4 失业与通货膨胀的关系

### 8.4.1 短期菲利普斯曲线

**短期菲利普斯曲线（Short-Run Phillips Curve）**：
失业率与通货膨胀率之间存在负相关关系。

**公式**：
```
π_t = π_t^e - β(u_t - u*) + ε_t
```

**含义**：
- 短期内，存在失业与通货膨胀的权衡（trade-off）
- 降低失业率 ⇒ 通货膨胀率上升

### 8.4.2 长期菲利普斯曲线

**长期菲利普斯曲线（Long-Run Phillips Curve）**：
失业率等于自然失业率，通货膨胀率由货币供给增长率决定。

**公式**：
```
u_t = u*
```

**含义**：
- 长期内，不存在失业与通货膨胀的权衡
- 失业率等于自然失业率
- 通货膨胀率由货币供给增长率决定

### 8.4.3 滞胀

**滞胀（Stagflation）**：
高通货膨胀率与高失业率并存。

**原因**：
- 负向供给冲击（例如：石油危机）
- 预期通货膨胀率上升
- 政策失误（例如：过度紧缩货币政策）

**解决方法**：
- 稳定预期
- 改善供给

## 8.5 本章总结

### 核心要点

1. **失业**：
   - 失业的定义：有劳动能力且有工作意愿的人没有工作的状态
   - 失业的类型：摩擦性失业、结构性失业、周期性失业
   - 失业率：u = U / L
   - 自然失业率：经济处于长期均衡时的失业率

2. **菲利普斯曲线**：
   - 原始菲利普斯曲线：π_t = -β(u_t - u*) + ε_t
   - 附加预期的菲利普斯曲线：π_t = π_t^e - β(u_t - u*) + ε_t
   - 适应性预期：π_t^e = π_(t-1)
   - 理性预期：π_t^e = E_t[π_t]

3. **通货膨胀**：
   - 通货膨胀的定义：价格水平持续上升
   - 通货膨胀的类型：需求拉动型、成本推动型、结构型
   - 通货膨胀的成本：鞋底成本、菜单成本、相对价格扭曲、税收扭曲、财富再分配

4. **失业与通货膨胀的关系**：
   - 短期菲利普斯曲线：失业率与通货膨胀率之间存在负相关关系
   - 长期菲利普斯曲线：失业率等于自然失业率
   - 滞胀：高通货膨胀率与高失业率并存
