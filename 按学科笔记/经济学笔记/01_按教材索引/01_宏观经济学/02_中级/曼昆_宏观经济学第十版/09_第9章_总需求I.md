# 第9章 总需求I：建立IS-LM模型

## 章节学习目标

1. **理解IS-LM模型的基本框架**：掌握IS曲线和LM曲线的含义和推导
2. **理解产品市场均衡（IS曲线）**：掌握产品市场均衡的条件和IS曲线的性质
3. **理解货币市场均衡（LM曲线）**：掌握货币市场均衡的条件和LM曲线的性质
4. **理解IS-LM模型的均衡**：掌握产品市场和货币市场同时均衡的分析
5. **理解政策分析**：掌握财政政策和货币政策在IS-LM模型中的效果

## 9.1 产品市场与IS曲线

### 9.1.1 凯恩斯交叉模型

**基本假设：**
- 价格水平固定（短期）
- 利率外生给定
- 总需求决定收入

**总需求函数：**
```
Y = C + I + G
```

**消费函数：**
```
C = C(Y - T)
```

**投资函数：**
```
I = I(r)
```

**凯恩斯交叉：**

**图形分析：**

```
总支出
  ↑
  │         45度线
  │        /
  │       /
  │      /
  │     /  总支出曲线
  │    /   E = C + I + G
  │   /
  │  /
  └───┴──────→ 收入 Y
      均衡收入
```

**均衡条件：**
```
Y = E = C(Y - T) + I(r) + G
```

### 9.1.2 IS曲线的推导

**IS曲线（Investment-Saving Curve）**描述产品市场均衡时利率与收入的关系。

**从凯恩斯交叉到IS曲线：**

**步骤1：给定利率r₁**
- I(r₁)给定
- 计算均衡收入Y₁

**步骤2：利率下降到r₂**
- I(r₂) > I(r₁)
- 均衡收入上升到Y₂

**步骤3：连接点(Y₁, r₁)和(Y₂, r₂)**
- 得到IS曲线

**图形分析：**

```
利率 r
  ↑
  │    r₁
  │    *
  │     \
  │      \ IS
  │       \
  │        \
  └─────────*────→ 收入 Y
          Y₂ Y₁
            r₂

利率下降 → 投资增加 → 收入增加
```

**IS曲线的性质：**
- 向下倾斜
- 反映利率与收入的负相关关系

**数学推导：**

**均衡条件：**
```
Y = C(Y - T) + I(r) + G
```

**微分：**
```
dY = C'(Y - T)(dY - dT) + I'(r)dr + dG
dY = MPC(dY - dT) + I'(r)dr + dG
dY(1 - MPC) = -MPC dT + I'(r)dr + dG
```

**IS曲线斜率：**
```
dr/dY = (1 - MPC)/I'(r) < 0
```

**含义：**
- IS曲线向下倾斜
- 斜率取决于MPC和投资对利率的敏感性

### 9.1.3 IS曲线的移动

**IS曲线右移因素：**

**1. 财政扩张（G增加）**
- 直接增加总需求
- 收入增加
- IS右移

**2. 税收减少（T减少）**
- 可支配收入增加
- 消费增加
- IS右移

**3. 消费者信心上升**
- 自发消费增加
- IS右移

**4. 投资者信心上升**
- 自发投资增加
- IS右移

**图形分析：**

```
利率 r
  ↑
  │    IS₁
  │   ↘
  │    ↘
  │     ↘
  │      ↘
  └───────IS₂────→ 收入 Y
```

**IS曲线左移因素：**
- 财政紧缩（G减少）
- 税收增加（T增加）
- 消费者信心下降
- 投资者信心下降

### 9.1.4 案例研究：财政政策的效果

**财政扩张（G增加）：**

**图形分析：**

```
利率 r
  ↑
  │    LM
  │   /
  │  / IS₁
  │ /  ↘
  │/    IS₂
  └────────→ 收入 Y
```

**效果：**
- 收入增加
- 利率上升
- 挤出效应

**挤出效应（Crowding Out）：**
- 政府支出增加
- 利率上升
- 私人投资减少
- 部分抵消财政扩张效果

**挤出效应的大小：**
- LM越陡峭，挤出效应越大
- IS越平坦，挤出效应越大

## 9.2 货币市场与LM曲线

### 9.2.1 货币需求理论

**货币需求函数：**
```
(M/P)^d = L(r, Y)
```

**性质：**
- ∂L/∂r < 0：利率上升，货币需求下降
- ∂L/∂Y > 0：收入上升，货币需求上升

**动机：**

**1. 交易动机**
- 与收入正相关
- L₁(Y)

**2. 预防动机**
- 与收入正相关
- L₂(Y)

**3. 投机动机**
- 与利率负相关
- L₃(r)

**总货币需求：**
```
L(r, Y) = L₁(Y) + L₂(Y) + L₃(r)
```

### 9.2.2 LM曲线的推导

**LM曲线（Liquidity-Money Curve）**描述货币市场均衡时利率与收入的关系。

**货币市场均衡：**
```
(M/P)^d = (M/P)^s
L(r, Y) = M/P
```

**LM曲线的推导：**

**步骤1：给定收入Y₁**
- 货币需求L(r, Y₁)
- 均衡利率r₁

**步骤2：收入上升到Y₂**
- 货币需求增加
- 利率上升到r₂以维持均衡

**步骤3：连接点(Y₁, r₁)和(Y₂, r₂)**
- 得到LM曲线

**图形分析：**

```
利率 r
  ↑
  │
  │    LM
  │   /
  │  /
  │ /
  └───────→ 收入 Y

收入上升 → 货币需求增加 → 利率上升
```

**LM曲线的性质：**
- 向上倾斜
- 反映利率与收入的正相关关系

**数学推导：**

**均衡条件：**
```
L(r, Y) = M/P
```

**微分：**
```
L_r dr + L_Y dY = 0
dr/dY = -L_Y / L_r > 0
```

**含义：**
- LM曲线向上倾斜
- 斜率取决于货币需求对收入和利率的敏感性

### 9.2.3 LM曲线的移动

**LM曲线右移因素：**

**1. 货币供给增加（M增加）**
- 货币供给曲线右移
- 利率下降
- LM右移

**2. 物价水平下降（P下降）**
- 实际货币供给增加
- LM右移

**3. 货币需求减少**
- 例如：金融创新降低货币需求
- LM右移

**图形分析：**

```
利率 r
  ↑
  │    LM₁
  │   ↗
  │  ↗
  │ ↗
  └───────LM₂────→ 收入 Y
```

**LM曲线左移因素：**
- 货币供给减少
- 物价水平上升
- 货币需求增加

### 9.2.4 LM曲线的斜率

**极端情况：**

**1. 古典情况（垂直LM）**
- 货币需求与利率无关（L_r = 0）
- LM垂直
- 货币政策有效，财政政策无效

**2. 流动性陷阱（水平LM）**
- 货币需求对利率无限敏感（L_r = ∞）
- LM水平
- 货币政策无效，财政政策有效

**图形分析：**

```
利率 r
  ↑
  │
  │    | 古典
  │    |
  └────┼────→ Y

利率 r
  ↑
  │
  │________ 流动性陷阱
  └────────→ Y
```

### 9.2.5 案例研究：大萧条与流动性陷阱

**大萧条特征：**

**1. 产出急剧下降**
- GDP下降30%
- 失业率上升到25%

**2. 利率接近零**
- 短期利率接近零
- 货币政策无效

**3. 流动性陷阱**
- 货币需求极高
- LM水平

**凯恩斯的解释：**

**1. 需求不足**
- 消费下降
- 投资下降
- 总需求不足

**2. 货币政策失灵**
- 流动性陷阱
- 利率已近零
- 无法进一步刺激

**3. 需要财政政策**
- 政府支出增加
- 直接刺激需求
- 避免流动性陷阱

## 9.3 IS-LM模型

### 9.3.1 均衡

**IS-LM模型均衡：**
```
IS: Y = C(Y - T) + I(r) + G
LM: M/P = L(r, Y)
```

**图形分析：**

```
利率 r
  ↑
  │    LM
  │   /
  │  / IS
  │ /  *
  └────┼────→ 收入 Y
      Y*
      r*
```

**均衡含义：**
- IS和LM交点
- 产品市场和货币市场同时均衡
- 唯一均衡收入Y*和利率r*

### 9.3.2 均衡的调整

**非均衡调整：**

**区域I：Y < Y*, r < r***
- 产品市场：需求 > 供给 → Y增加
- 货币市场：需求 > 供给 → r上升
- 调整方向：Y上升，r上升

**区域II：Y > Y*, r < r***
- 产品市场：需求 < 供给 → Y减少
- 货币市场：需求 > 供给 → r上升
- 调整方向：Y下降，r上升

**区域III：Y > Y*, r > r***
- 产品市场：需求 < 供给 → Y减少
- 货币市场：需求 < 供给 → r下降
- 调整方向：Y下降，r下降

**区域IV：Y < Y*, r > r***
- 产品市场：需求 > 供给 → Y增加
- 货币市场：需求 < 供给 → r下降
- 调整方向：Y上升，r下降

**图形分析：**

```
利率 r
  ↑
  │    LM
  │   /I   \II
  │  /      \
  │ /IV  \   III
  └────┼────→ Y
  IS

四个区域的调整方向
```

### 9.3.3 稳定性

**IS-LM模型是稳定的：**
- 调整过程收敛到均衡
- 产品市场和货币市场相互作用

**假设：**
- 产品市场调整较快（存货调整）
- 货币市场调整很快（资产市场调整）

## 9.4 政策分析

### 9.4.1 财政政策

**财政扩张（G增加）：**

**图形分析：**

```
利率 r
  ↑
  │    LM
  │   /
  │  / IS₁
  │ /  ↘
  │/    IS₂
  └────────→ 收入 Y
```

**效果：**
- IS右移
- 收入增加
- 利率上升
- 挤出效应

**挤出效应：**
- 政府支出增加
- 利率上升
- 投资减少
- 部分抵消财政扩张

**挤出效应的大小：**
```
ΔY = ΔY_fiscal - ΔY_crowding
```

**决定因素：**
- LM越陡峭 → 挤出效应越大
- IS越平坦 → 挤出效应越大
- 货币需求对利率越不敏感 → 挤出效应越大

**极端情况：**

**1. 古典情况（LM垂直）**
- 完全挤出
- 财政政策无效
- 货币政策有效

**2. 流动性陷阱（LM水平）**
- 无挤出
- 财政政策完全有效
- 货币政策无效

### 9.4.2 货币政策

**货币扩张（M增加）：**

**图形分析：**

```
利率 r
  ↑
  │    LM₁
  │   / ↘
  │  /   LM₂
  │ /     \
  └/       \ IS
  └────────→ 收入 Y
```

**效果：**
- LM右移
- 利率下降
- 收入增加

**传导机制：**
```
M增加 → 利率下降 → 投资增加 → 收入增加
```

**货币乘数：**
```
ΔY/ΔM = ?（取决于IS和LM的斜率）
```

**极端情况：**

**1. 古典情况（LM垂直）**
- 货币政策完全有效
- 财政政策无效

**2. 流动性陷阱（LM水平）**
- 货币政策无效
- 财政政策完全有效

### 9.4.3 政策组合

**财政-货币政策组合：**

**1. 扩张性财政+扩张性货币**
- IS和LM都右移
- 收入大幅增加
- 利率不确定（取决于移动幅度）

**2. 扩张性财政+紧缩性货币**
- IS右移，LM左移
- 利率上升
- 收入不确定

**3. 紧缩性财政+扩张性货币**
- IS左移，LM右移
- 利率下降
- 收入不确定

**4. 紧缩性财政+紧缩性货币**
- IS和LM都左移
- 收入大幅减少
- 利率不确定

**图形分析（组合1）：**

```
利率 r
  ↑
  │    LM₁ → LM₂
  │   / ↘
  │  /   \
  │ /     \
  └/       \ IS₁ → IS₂
  └────────→ 收入 Y

Y大幅增加，r不确定
```

### 9.4.4 案例研究：2008年金融危机的政策应对

**政策组合：**

**1. 财政政策（扩张性）**
- 美国复苏与再投资法案（ARRA）
- 8310亿美元刺激
- IS右移

**2. 货币政策（扩张性）**
- 降息至零
- 量化宽松（QE）
- LM右移

**效果：**

**IS-LM分析：**

```
利率 r
  ↑
  │    LM₁ → LM₂
  │   / ↘
  │  /   \
  │ /     \
  └/       \ IS₁ → IS₂
  └────────→ 收入 Y
```

**结果：**
- 避免了更严重的衰退
- 但复苏缓慢
- 利率接近零（流动性陷阱）

**挑战：**
- 财政刺激有限（政治约束）
- 货币政策接近零下限
- 需要非常规政策（QE）

## 本章总结

### 核心要点

1. **IS曲线**
   - 产品市场均衡
   - 向下倾斜
   - 财政政策移动IS

2. **LM曲线**
   - 货币市场均衡
   - 向上倾斜
   - 货币政策移动LM

3. **IS-LM均衡**
   - 产品市场和货币市场同时均衡
   - 唯一均衡收入和利率
   - 稳定性

4. **财政政策**
   - 移动IS曲线
   - 收入增加，利率上升
   - 挤出效应

5. **货币政策**
   - 移动LM曲线
   - 利率下降，收入增加
   - 传导机制

6. **政策组合**
   - 财政和货币政策配合
   - 不同组合有不同效果
