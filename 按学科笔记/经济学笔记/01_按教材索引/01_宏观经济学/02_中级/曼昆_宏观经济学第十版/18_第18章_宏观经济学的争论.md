# 第18章 宏观经济学的争论

## 本章学习目标

通过本章学习，你应该能够：
1. 理解宏观经济学的主要争论
2. 掌握主动政策与被动政策的分歧
3. 分析规则与相机抉择的选择
4. 理解零利率下限与经济危机的政策应对
5. 综合全书的核心理论框架

## 18.1 政策应该积极还是被动？

### 18.1.1 主动政策的论据

**主动政策（Active Policy）**：
```
定义：政府主动使用财政和货币政策来稳定经济

支持者：凯恩斯主义者、新凯恩斯主义者

核心论点：
1. 市场失灵需要政府干预
2. 逆周期政策可以减少波动
3. 政策可以改善社会福利
4. 自动稳定器不够用
```

**理论基础**：

**1. 价格和工资黏性**：
```
如果价格和工资调整缓慢：
- 市场可能长期偏离均衡
- 失业可能持续
- 产出可能低于潜在水平

政策可以：
- 刺激总需求
- 加速调整过程
- 减少痛苦
```

**2. 乘数效应**：
```
政府支出增加 → 产出倍数增加

乘数：k = 1/(1-MPC)

如果MPC = 0.8：
k = 5

ΔG = 100亿 → ΔY = 500亿

政策效果显著
```

**3. 外部性**：
```
经济衰退有负外部性：
- 失业导致技能损失
- 萧条导致投资不足
- 不确定性增加

政策可以内部化这些外部性
```

**支持证据**：

**历史案例**：
```
大萧条（1930年代）：
- 被动政策：经济深度衰退，失业率25%
- 主动政策：罗斯福新政，经济开始恢复

2008年金融危机：
- 主动政策：财政刺激、量化宽松
- 结果：避免了大萧条重演
```

**实证研究**：
```
Romer and Romer (2010)：
- 识别外生税收变化
- 发现税收政策对产出有显著影响
- 支持主动政策

Blanchard and Perotti (2002)：
- 识别政府购买冲击
- 发现乘数约1.5
- 支持财政政策有效性
```

### 18.1.2 被动政策的论据

**被动政策（Passive Policy）**：
```
定义：政府避免主动干预，让经济自我调整

支持者：新古典主义者、实际经济周期理论家

核心论点：
1. 市场是有效的
2. 政策干预可能适得其反
3. 政策时滞导致失败
4. 政策不确定性损害经济
```

**理论基础**：

**1. 市场出清**：
```
如果价格和工资灵活调整：
- 市场迅速出清
- 资源有效配置
- 非自愿失业不存在

政策干预：
- 干扰市场机制
- 造成无谓损失
- 降低效率
```

**2. 卢卡斯批判**：
```
传统政策分析的问题：
- 假设政策参数不变
- 忽略政策对预期的影响

卢卡斯批判：
- 政策变化改变预期
- 预期改变改变行为
- 行为改变改变政策参数

结论：
- 不能用历史数据预测政策效果
- 政策效果不确定
```

**3. 政策时滞**：
```
内部时滞：9-24个月
外部时滞：6-24个月
总时滞：15-48个月

问题：
- 到政策生效时，经济状况可能已改变
- 政策可能加剧而非减少波动
- 政策时滞导致时机错误
```

**支持证据**：

**政策失误案例**：
```
1970年代的滞胀：
- 政策刺激导致通胀
- 通胀预期上升
- 菲利普斯曲线失效
- 政策适得其反

1930年代的货币政策：
- 货币政策被动
- 货币供给大幅下降
- 加剧大萧条
```

**实证研究**：
```
Romer and Romer (2004)：
- 美联储政策冲击在18-24个月后生效
- 时滞长且不确定
- 支持被动政策

Barro (1979)：
- 税收和债务融资等价
- 财政政策无效
- 支持李嘉图等价
```

### 18.1.3 争论的解决？

**共识领域**：
```
1. 长期：
   - 货币政策中性
   - 产出由供给决定
   - 通胀由货币供给决定

2. 自动稳定器：
   - 所得税
   - 失业保险
   - 都是有益的

3. 政策可信度：
   - 影响预期
   - 影响政策效果
   - 需要建立信誉
```

**分歧领域**：
```
1. 短期：
   - 货币政策是否有效？
   - 财政政策乘数多大？

2. 政策时机：
   - 政策时滞是否致命？
   - 能否准确预测经济？

3. 政策范围：
   - 政府应该干预多深？
   - 哪些工具最有效？
```

**实际政策实践**：
```
大多数国家采用混合策略：
- 日常：被动规则（泰勒规则）
- 危机：主动干预（财政刺激、QE）
- 工具：多种政策配合

原因：
- 完全被动：危机时无能为力
- 完全主动：平时可能过度干预
- 混合策略：平衡风险和收益
```

## 18.2 政策应该按规则还是相机抉择？

### 18.2.1 政策规则的论据

**政策规则（Policy Rules）**：
```
定义：事先规定的、可预测的政策反应函数

代表人物：Milton Friedman, John Taylor

核心论点：
1. 规则提供可预测性
2. 规则避免时间不一致性
3. 规则限制政治压力
4. 规则建立政策信誉
```

**著名规则**：

**1. 货币数量规则（Friedman）**：
```
规则：货币供给每年增长k%

k = 实际GDP长期增长率 + 目标通胀率

如果实际GDP增长3%，目标通胀2%：
k = 3% + 2% = 5%

规则：M每年增长5%
```

**2. 泰勒规则（Taylor）**：
```
规则：i = r* + π + 0.5(π-π*) + 0.5(y-y*)

其中：
i：名义利率
r*：均衡实际利率（2%）
π：通胀率
π*：目标通胀率（2%）
y：实际产出
y*：潜在产出

数值例子：
如果π = 3%，y = y*：
i = 2% + 3% + 0.5×1% + 0.5×0% = 5.5%

如果π = 5%，y = 1.02y*：
i = 2% + 5% + 0.5×3% + 0.5×2% = 8.5%
```

**3. 通胀目标制（Inflation Targeting）**：
```
规则：明确宣布通胀目标并致力于实现

特征：
- 明确的数值目标（如2%±1%）
- 透明的沟通策略
- 问责机制

实施框架：
1. 设定目标（如2%）
2. 发布通胀报告
3. 前瞻性指引
4. 政策解释
```

**规则的优点**：

**1. 可预测性**：
```
规则：i = 2% + π + 0.5(π-π*) + 0.5(y-y*)

公众可以：
- 预测政策行为
- 形成稳定预期
- 减少不确定性

好处：
- 投资决策更确定
- 消费更稳定
- 经济波动减少
```

**2. 时间一致性**：
```
时间不一致性问题：
- 事前最优：承诺某政策
- 事后最优：偏离承诺

规则避免：
- 事先承诺遵循规则
- 减少偏离诱惑
- 增强政策可信度
```

**3. 限制政治压力**：
```
规则可以：
- 限制短期政治考虑
- 避免选举周期政策
- 减少利益集团影响

例子：
- 央行独立
- 通胀目标制
- 限制财政赤字
```

**成功案例**：

**新西兰通胀目标制**：
```
实施：1990年
框架：
- 通胀目标：0-2%
- 央行独立
- 透明度提升

结果：
- 通胀从11%降至2%
- 经济增长稳定
- 政策信誉建立
```

**欧洲央行**：
```
实施：1999年
框架：
- 通胀目标：<2%
- 独立性
- 规则基础

结果：
- 通胀稳定
- 政策可信
- 经济稳定
```

### 18.2.2 相机抉择的论据

**相机抉择（Discretion）**：
```
定义：政策制定者根据经济状况灵活调整政策

代表人物：Keynes, Bernanke, Yellen

核心论点：
1. 灵活性应对未预期冲击
2. 使用所有可得信息
3. 特殊情况需要特殊政策
4. 规则过于僵化
```

**相机抉择的优点**：

**1. 灵活性**：
```
相机抉择可以：
- 应对未预期的冲击
- 使用最新数据
- 调整政策参数

例子：
COVID-19疫情：
- 未预期的冲击
- 需要立即行动
- 非常规政策工具

相机抉择允许：
- 快速政策响应
- 创新政策工具
- 大规模干预
```

**2. 信息利用**：
```
相机抉择可以：
- 使用最新数据
- 修正模型错误
- 调整政策判断

优势：
- 政策更准确
- 减少模型不确定性
- 提高政策效果
```

**3. 特殊情况处理**：
```
相机抉择可以处理：
- 金融危机
- 战争
- 自然灾害
- 技术冲击

这些情况：
- 不符合常规模式
- 需要特殊政策
- 难以事先规定
```

**成功案例**：

**2008年金融危机**：
```
相机抉择政策：
1. 财政政策：
   - TARP法案
   - ARRA法案
   - 失业救济

2. 货币政策：
   - 零利率
   - 量化宽松
   - 前瞻性指引

3. 非常规政策：
   - 商业票据便利
   - 主街贷款
   - 货币市场基金流动性便利

结果：
- 避免了大萧条
- 经济开始恢复
- 政策创新成功
```

### 18.2.3 时间不一致性问题

**定义**：
```
时间不一致性（Time Inconsistency）：
事前最优的政策在事后不再是最优的
```

**经典例子：通胀-失业权衡**：

**设定**：
```
央行目标：最大化社会福利

福利函数：
W = -[α(π-π*)² + (y-y*)²]

约束：
短期菲利普斯曲线：
y = y* + β(π-πᵉ)
```

**博弈过程**：

**第一期：央行承诺**
```
央行宣布：π = π* = 0%

公众预期：πᵉ = 0%
```

**第二期：央行决策**
```
如果央行遵循承诺：
π = 0%
y = y* + 0 = y*
W = -[0 + 0] = 0

如果央行偏离承诺（如π = 2%）：
y = y* + β×2%
W = -[α×(2%)² + (β×2%)²]

如果β > 0, α > 0：
偏离承诺可能改善短期福利
```

**第三期：预期调整**
```
公众理性预期：
- 知道央行会偏离
- 预期通胀πᵉ > 0
- 均衡通胀π = πᵉ > 0

结果：
- 通胀高于目标
- 产出回到自然水平
- 福利损失
```

**解决方法**：

**1. 独立央行**：
```
立法保障：
- 央行独立于政府
- 独立制定政策
- 长期任期

效果：
- 减少短期政治压力
- 增强政策承诺能力
- 建立政策信誉
```

**2. 明确规则**：
```
通胀目标制：
- 明确通胀目标
- 透明沟通
- 问责机制

效果：
- 约束央行行为
- 增强承诺可信度
- 锚定预期
```

**3. 透明度**：
```
政策透明：
- 定期报告
- 前瞻性指引
- 决策解释

效果：
- 公众理解政策
- 减少不确定性
- 建立信誉
```

## 18.3 零利率下限与经济危机

### 18.3.1 零利率下限

**定义**：
```
零利率下限（Zero Lower Bound, ZLB）：
名义利率不能降至零以下

原因：
- 如果利率为负，人们宁愿持有现金
- 现金利率为0，是下限
- 实际利率下限 = -通胀率
```

**问题**：
```
传统货币政策：
i = r* + π + 0.5(π-π*) + 0.5(y-y*)

如果经济深度衰退：
y < y*, π < π*
→ 政策要求i < 0

但i不能 < 0
→ 传统货币政策空间耗尽
```

**历史案例**：

**日本（1990-2000年代）**：
```
问题：
- 泡沫破裂
- 长期停滞
- 利率接近零

应对：
- 零利率政策
- 量化宽松
- 非常规政策

结果：
- 通胀率接近0
- 经济增长缓慢
- 难以走出困境
```

**美国（2008-2009）**：
```
问题：
- 金融危机
- 利率降至0-0.25%
- 传统政策耗尽

应对：
- 量化宽松（QE）
- 前瞻性指引
- 信贷宽松

结果：
- 避免了通货紧缩
- 经济开始恢复
- 但恢复缓慢
```

### 18.3.2 非常规货币政策

**1. 量化宽松（Quantitative Easing）**：
```
定义：央行大规模购买长期资产

机制：
央行购买长期债券
→ 长期债券价格↑
→ 长期利率↓
→ 刺激投资和消费

美国QE：
QE1（2008-2010）：1.7万亿美元
QE2（2010-2011）：6000亿美元
QE3（2012-2014）：开放购买
```

**2. 前瞻性指引（Forward Guidance）**：
```
定义：央行沟通未来政策意图

类型：
- 时间指引："低利率至少到2015年"
- 状态指引："低利率直到失业率降至6%"
- 定性指引："宽松政策直到强劲复苏"

效果：
→ 影响预期
→ 降低长期利率
→ 增强政策效果
```

**3. 信贷宽松（Credit Easing）**：
```
定义：央行直接支持特定市场

工具：
- 商业票据便利
- 资产支持证券购买
- 主街贷款计划

效果：
→ 改善信贷可得性
→ 稳定金融市场
→ 降低借贷成本
```

**4. 负利率政策**：
```
定义：央行将政策利率降至零以下

实施国家：
- 日本：-0.1%
- 欧元区：-0.5%
- 瑞士：-0.75%

效果：
→ 对银行准备金收费
→ 鼓励放贷
→ 刺激经济

问题：
→ 挤压银行利润
→ 可能影响金融稳定
```

### 18.3.3 危机时的政策协调

**2008年金融危机的政策协调**：

**1. 货币政策**：
```
传统政策：
- 利率降至零
- 流动性支持

非常规政策：
- QE
- 前瞻性指引
- 信贷宽松
```

**2. 财政政策**：
```
自动稳定器：
- 税收收入下降
- 失业救济增加

主动政策：
- TARP法案（7000亿）
- ARRA法案（7870亿）
- 汽车业救助
```

**3. 监管政策**：
```
金融稳定：
- 压力测试
- 资本要求提高
- 流动性要求

结构改革：
- 多德-弗兰克法案
- 沃尔克规则
```

**政策效果评估**：
```
GDP变化：
2009Q2：-8.4%（年化）
2010Q4：+2.8%（年化）

政策贡献估计：
- 财政政策：1.5-2.0个百分点
- 货币政策：2.0-3.0个百分点
- 总计：3.5-5.0个百分点

结论：
政策成功避免了大萧条
但恢复缓慢
```

## 18.4 宏观经济学的共识与分歧

### 18.4.1 核心共识

**1. 长期增长理论**：
```
共识：
- 长期增长由供给因素决定
- 技术进步是主要驱动力
- 资本积累和人力资本重要
- 制度质量影响增长

代表模型：
- 索洛模型
- 内生增长模型
```

**2. 货币长期中性**：
```
共识：
- 货币政策在长期中性
- 货币数量决定通胀
- 通胀由货币供给增长决定

证据：
- 长期货币与通胀关系
- 跨国数据
```

**3. 自然率假说**：
```
共识：
- 存在自然失业率
- 存在自然产出水平
- 长期失业率回归自然率
- 长期菲利普斯曲线垂直
```

**4. 理性预期的重要性**：
```
共识：
- 预期影响经济行为
- 理性预期是合理假设
- 政策需要考虑预期
- 预期管理很重要
```

**5. 开放经济的相互依赖**：
```
共识：
- 汇率影响经济
- 资本流动重要
- 政策溢出效应
- 国际协调需要
```

### 18.4.2 持续分歧

**1. 短期政策有效性**：
```
分歧：
- 货币政策短期是否有效？
- 财政政策乘数多大？
- 乘数受什么影响？

证据：
- 新凯恩斯主义：政策有效
- RBC：政策无效
- 实证：混合证据
```

## 18.5 全书总结

### 18.5.1 宏观经济学的核心框架

**长期 vs. 短期**：
```
长期（古典理论）：
- 产出由供给决定
- 价格灵活调整
- 货币中性
- 政策无效

短期（凯恩斯理论）：
- 产出由需求决定
- 价格黏性
- 货币非中性
- 政策有效
```

**总供给-总需求模型**：
```
长期：
AS垂直于潜在产出
AD决定价格水平

短期：
AS向上倾斜
AD决定产出和价格

过渡：
短期 → 长期
价格调整
产出回归潜在
```

**IS-LM模型**：
```
产品市场：IS曲线
货币市场：LM曲线

均衡：
两条曲线交点
决定利率和产出

政策分析：
财政政策：移动IS
货币政策：移动LM
```
