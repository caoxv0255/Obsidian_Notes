# 第5章 开放的经济

## 章节学习目标

1. **理解国际收支平衡**：掌握经常账户、资本账户的构成及相互关系
2. **理解汇率及其决定**：掌握名义汇率、实际汇率的定义、购买力平价理论
3. **理解小型开放经济模型**：掌握储蓄、投资、汇率的关系及政策影响
4. **理解大型开放经济**：掌握大型开放经济与小型开放经济的区别
5. **理解国际政策协调**：掌握国际经济政策协调的重要性

## 5.1 资本与物品的国际流动

### 5.1.1 开放经济的定义

**开放经济（Open Economy）**是指与其他国家进行商品、服务和资本交易的经济。

**与封闭经济（Closed Economy）的区别：**

| 特征 | 封闭经济 | 开放经济 |
|------|---------|---------|
| 对外贸易 | 无 | 有 |
| 资本流动 | 无 | 有 |
| GDP恒等式 | Y = C + I + G | Y = C + I + G + NX |
| 利率决定 | 国内储蓄和投资 | 世界利率 |

### 5.1.2 国际收支平衡表

**国际收支平衡表（Balance of Payments, BOP）**记录一个国家与世界其他国家的所有经济交易。

**主要组成部分：**

**1. 经常账户（Current Account, CA）**

**构成：**
- 商品和服务的净出口（NX）
- 净要素收入（来自国外的收入减去支付给国外的收入）
- 净转移支付（来自国外的转移支付减去支付给国外的转移支付）

**公式：**
```
CA = NX + NFP + NTR
```

**2. 资本账户（Capital Account, KA）**

**构成：**
- 净资本流入
- 对外资产购买减去外国对本国资产的购买

**公式：**
```
KA = KA_in - KA_out
```

**3. 金融账户（Financial Account, FA）**

**构成：**
- 直接投资
- 证券投资
- 其他投资
- 储备资产

**平衡条件：**
```
CA + KA + FA = 0
```

**简化模型（忽略资本账户）：**
```
CA + FA = 0
NX = NCO
```

其中：
- NX = 净出口
- NCO = 净资本流出

**含义：**
- 净出口等于净资本流出
- 贸易盈余对应资本流出
- 贸易赤字对应资本流入

### 5.1.3 储蓄、投资与净出口

**开放经济GDP恒等式：**
```
Y = C + I + G + NX
```

**重新排列：**
```
Y - C - G = I + NX
S = I + NX
NX = S - I
```

**国民储蓄：**
```
S = Y - C - G
  = (Y - T - C) + (T - G)
  = S_private + S_public
```

**净出口与储蓄投资的关系：**
```
NX = S - I
NX = NCO
```

**含义：**

**1. 如果 S > I**
- 国民储蓄超过国内投资
- NX > 0，贸易顺差
- 资本净流出

**2. 如果 S < I**
- 国内投资超过国民储蓄
- NX < 0，贸易逆差
- 资本净流入

**3. 如果 S = I**
- 储蓄等于投资
- NX = 0，贸易平衡

**图形分析：储蓄、投资与净出口**

```
实际利率 r
  ↑
  │    S
  │   /
  │  /
  │ /  I
  │/
  └────────→ S, I, NX

S > I: NX = S - I > 0 (贸易顺差)
S < I: NX = S - I < 0 (贸易逆差)
```

### 5.1.4 案例研究：美国贸易赤字

**美国数据（示例）：**

| 时期 | 储蓄率 | 投资率 | 净出口率 |
|------|--------|--------|---------|
| 1980s | 18-20% | 20-22% | -2%至-4% |
| 1990s | 16-18% | 18-20% | -2%至-4% |
| 2000s | 14-16% | 18-20% | -4%至-6% |
| 2010s | 17-19% | 17-19% | -2%至-3% |
| 2020s | 18-20% | 18-20% | -2%至-3% |

**特征：**

**1. 长期贸易逆差**
- 自1980年代以来持续逆差
- 反映储蓄不足

**2. 储蓄率相对较低**
- 与其他国家相比
- 原因：消费文化、社会保障不足

**3. 投资率相对较高**
- 吸引外国资本
- 经济活力

**原因分析：**

**1. 储蓄不足**
- 高消费倾向
- 政府赤字
- 老龄化预期

**2. 投资吸引力**
- 资本回报率高
- 市场规模大
- 技术领先

**3. 全球因素**
- 全球储蓄过剩
- 美元作为储备货币
- 亚洲国家购买美国国债

**影响：**

**1. 正面影响**
- 消费者享受低价进口商品
- 低利率支持投资
- 资本流入支持增长

**2. 负面影响**
- 外债积累
- 汇率风险
- 制造业萎缩

**政策含义：**
- 提高国民储蓄
- 减少政府赤字
- 提高竞争力

## 5.2 汇率

### 5.2.1 汇率的定义

**汇率（Exchange Rate）**是指一种货币与另一种货币的交换比率。

**两种表示方式：**

**1. 直接标价法**
- 以本币表示外币的价格
- 例如：1美元 = 7人民币
- 汇率上升，本币贬值

**2. 间接标价法**
- 以外币表示本币的价格
- 例如：1人民币 = 0.14美元
- 汇率上升，本币升值

### 5.2.2 名义汇率与实际汇率

**名义汇率（Nominal Exchange Rate, e）：**

**定义：**
两种货币的相对价格。

**公式：**
```
e = E_domestic / E_foreign
```

**例如：**
1美元 = 7人民币
e = 7人民币/美元

**实际汇率（Real Exchange Rate, ε）：**

**定义：**
用两国物价水平调整后的汇率，反映两国商品的相对价格。

**公式：**
```
ε = (e × P) / P*
```

其中：
- e = 名义汇率（本币/外币）
- P = 国内物价水平
- P* = 国外物价水平

**含义：**
- ε上升：国内商品相对国外商品变贵，本币实际贬值
- ε下降：国内商品相对国外商品变便宜，本币实际升值

**例子：**

假设：
- 名义汇率：1美元 = 7人民币
- 中国物价水平：P = 7
- 美国物价水平：P* = 1

实际汇率：
```
ε = (7 × 7) / 1 = 49
```

含义：1单位美国商品可以购买49单位中国商品

### 5.2.3 购买力平价

**购买力平价理论（Purchasing Power Parity, PPP）**认为汇率应该使相同商品在不同国家具有相同的购买力。

**绝对购买力平价：**

**公式：**
```
e = P / P*
```

**含义：**
- 汇率等于两国物价水平的比率
- 汇率调整使购买力相等

**一价定律（Law of One Price）：**
相同商品在不同国家的价格（用同一种货币表示）应该相同。

**公式：**
```
P = e × P*
```

**相对购买力平价：**

**公式：**
```
(Δe/e) ≈ (ΔP/P) - (ΔP*/P*)
```

或：
```
(Δe/e) ≈ π - π*
```

其中：
- π = 国内通胀率
- π* = 国外通胀率

**含义：**
- 汇率变化率等于两国通胀率之差
- 高通胀国家货币贬值

**例子：**

如果：
- 中国通胀率：π = 5%
- 美国通胀率：π* = 2%

汇率变化：
```
Δe/e ≈ 5% - 2% = 3%
```

人民币对美元贬值3%

### 5.2.4 汇率的决定

**外汇市场（Foreign Exchange Market）：**

**参与者：**
- 进出口商
- 投资者
- 投机者
- 央行

**需求：**
- 外汇需求：进口、对外投资
- 与净出口正相关

**供给：**
- 外汇供给：出口、外国投资
- 与净出口负相关

**均衡：**
```
外汇需求 = 外汇供给
NX(e) = NCO(e)
```

**图形分析：外汇市场**

```
汇率 e
  ↑
  │
  │    供给
  │   /
  │  /
  │ /  均衡点
  │/   *
  └───────需求────→ 外汇数量
      e*
      NX = NCO
```

**汇率调整机制：**

**如果 NX > NCO（供给过剩）：**
- 汇率下降（本币升值）
- 出口减少，进口增加
- 回到均衡

**如果 NX < NCO（需求过剩）：**
- 汇率上升（本币贬值）
- 出口增加，进口减少
- 回到均衡

### 5.2.5 汇率制度

**固定汇率制（Fixed Exchange Rate）：**

**定义：**
央行将汇率固定在特定水平。

**机制：**
- 央行干预外汇市场
- 买卖外汇维持汇率
- 使用外汇储备

**优点：**
- 稳定国际贸易和投资
- 减少不确定性
- 防止竞争性贬值

**缺点：**
- 丧失货币政策独立性
- 需要大量外汇储备
- 容易受投机攻击

**浮动汇率制（Flexible/Floating Exchange Rate）：**

**定义：**
汇率由市场供求决定，央行不干预。

**机制：**
- 市场供求决定汇率
- 央行不干预（或仅有限干预）
- 货币政策独立

**优点：**
- 货币政策独立
- 自动调节外部失衡
- 减少外汇储备需求

**缺点：**
- 汇率波动
- 增加不确定性
- 可能过度波动

**有管理的浮动汇率制（Managed Float）：**

**定义：**
汇率主要由市场决定，但央行会进行干预以避免过度波动。

**特点：**
- 介于固定和浮动之间
- 央行有限干预
- "肮脏浮动"

### 5.2.6 案例研究：人民币汇率

**历史演变：**

**1. 1994-2005：固定汇率**
- 实际钉住美元
- 汇率稳定在8.28人民币/美元
- 外汇储备快速增长

**2. 2005-2015：有管理的浮动**
- 2005年7月改革
- 参考一篮子货币
- 逐步升值
- 2005-2015年升值约35%

**3. 2015-至今：更加灵活**
- 市场化程度提高
- 双向波动加大
- 参考篮子货币

**汇率与经济的关系：**

**1. 汇率与出口**
- 人民币贬值 → 出口增加
- 人民币升值 → 出口减少

**2. 汇率与通胀**
- 人民币贬值 → 进口商品价格上涨 → 通胀上升
- 人民币升值 → 进口商品价格下降 → 通胀下降

**3. 汇率与资本流动**
- 人民币升值预期 → 资本流入
- 人民币贬值预期 → 资本流出

**政策挑战：**

**1. 三难选择（Trilemma）**
- 货币政策独立性
- 汇率稳定
- 资本自由流动
- 三者只能同时实现两个

**2. 汇率管理**
- 避免过度波动
- 保持竞争力
- 应对资本流动

**3. 国际压力**
- 贸易摩擦
- 汇率操纵指控
- 国际协调

## 5.3 小型开放经济

### 5.3.1 小型开放经济的定义

**小型开放经济（Small Open Economy）**是指经济规模小，不能影响世界利率的经济。

**假设：**
- 世界利率r*外生给定
- 经济是世界利率的接受者
- 资本完全流动

### 5.3.2 小型开放经济模型

**投资函数：**
```
I = I(r*)
```

**投资与世界利率：**
- 如果r*上升，投资下降
- 如果r*下降，投资上升

**储蓄函数：**
```
S = S(r*)
```

**储蓄与世界利率：**
- 通常假设S不取决于r（简化）

**净出口：**
```
NX = S - I
   = S - I(r*)
```

**净出口与世界利率：**
- r*上升 → I下降 → NX上升
- r*下降 → I上升 → NX下降

**图形分析：小型开放经济**

```
实际利率 r
  ↑
  │    S
  │   /|
  │  / |
  │ /  | I(r)
  │/   |
  └────┴──→ S, I
      r*
      I
      S
      NX = S - I
```

### 5.3.3 政策分析

**政策1：财政政策**

**扩张性财政政策（G增加）：**

**效果：**
- 储蓄减少（S下降）
- 投资不变（r*不变）
- NX下降（贸易逆差增加）

**图形分析：**

```
实际利率 r
  ↑
  │    S₁ → S₂
  │   /|
  │  / |  ↘
  │ /  |   ↘
  │/   |    I(r)
  └────┴──→ S, I
      r*  I
         NX₁ → NX₂ (下降)
```

**含义：**
- 财政扩张减少国民储蓄
- 净出口减少
- 贸易逆差增加
- 资本流入增加

**政策2：贸易政策**

**贸易限制（进口配额、关税）：**

**效果：**
- NX函数上移
- 但均衡NX不变
- 实际汇率上升（本币升值）

**图形分析：**

```
实际汇率 ε
  ↑
  │
  │    NX₁ → NX₂
  │   ↗
  │  ↗
  │ ↗
  │↗
  └────────→ NX
      NX*
```

**含义：**
- 贸易限制不影响贸易余额
- 只影响贸易构成
- 本币升值抵消贸易限制

**政策3：投资激励**

**投资税收抵免：**

**效果：**
- 投资函数右移
- 投资增加
- NX下降

**图形分析：**

```
实际利率 r
  ↑
  │    S
  │   /|
  │  / | I₁ → I₂
  │ /  |  ↘
  │/   |   ↘
  └────┴──→ S, I
      r*  I₁  I₂
         NX₁ → NX₂ (下降)
```

**含义：**
- 投资激励增加投资
- 净出口减少
- 贸易逆差增加

### 5.3.4 汇率的决定

**汇率与净出口：**

**关系：**
```
NX = NX(ε)
ε上升（本币贬值）→ NX上升
```

**均衡汇率：**
```
S - I(r*) = NX(ε)
```

**图形分析：汇率决定**

```
实际汇率 ε
  ↑
  │
  │    NX(ε)
  │   ↘
  │    ↘
  │     ↘
  │      ↘
  └───────┴──→ NX
          S - I(r*)
          ε*
```

**政策对汇率的影响：**

**财政扩张（G增加）：**
- S - I下降
- ε下降（本币升值）

**投资激励：**
- S - I下降
- ε下降（本币升值）

**世界利率上升：**
- I下降
- S - I上升
- ε上升（本币贬值）

### 5.3.5 案例研究：德国统一

**背景：**
1990年，东西德统一。

**政策：**

**1. 财政扩张**
- 大规模基础设施投资
- 社会保障体系整合
- 转移支付

**2. 货币政策**
- 保持德国马克稳定
- 高利率政策

**效果：**

**1. 储蓄和投资**
- 储蓄下降（财政扩张）
- 投资需求增加
- 利率上升

**2. 净出口**
- NX下降
- 贸易顺差减少

**3. 汇率**
- 马克升值
- 出口竞争力下降

**4. 对欧洲的影响**
- 德国利率上升带动欧洲利率上升
- 其他欧洲国家受影响
- 欧洲货币体系（EMS）压力

**启示：**
- 小型开放经济受外部影响大
- 政策溢出效应
- 国际协调的重要性

## 5.4 大型开放经济

### 5.4.1 大型开放经济的定义

**大型开放经济（Large Open Economy）**是指经济规模大，能够影响世界利率的经济。

**特点：**
- 国内储蓄和投资影响世界利率
- 世界利率由所有国家的储蓄和投资决定
- 资本流动不完全

### 5.4.2 大型开放经济模型

**世界市场均衡：**

**世界储蓄：**
```
S_world = S_domestic + S_foreign
```

**世界投资：**
```
I_world = I_domestic + I_foreign
```

**均衡条件：**
```
S_world = I_world
S_domestic + S_foreign = I_domestic + I_foreign
(S_domestic - I_domestic) + (S_foreign - I_foreign) = 0
NX_domestic + NX_foreign = 0
```

**世界利率决定：**
```
r* 决定于 S_world(r) = I_world(r)
```

**图形分析：世界资本市场**

```
实际利率 r
  ↑
  │    S_world
  │   /
  │  /
  │ /  I_world
  │/
  └────────→ S, I
      r*
      S* = I*
```

### 5.4.3 大国与小国的区别

| 特征 | 小国 | 大国 |
|------|------|------|
| 世界利率 | 外生给定 | 内生决定 |
| 政策效果 | 无溢出效应 | 有溢出效应 |
| 资本流动 | 完全流动 | 不完全流动 |
| 贸易政策 | 不影响贸易余额 | 影响贸易余额 |

### 5.4.4 政策分析

**财政政策：**

**扩张性财政政策：**

**对国内：**
- 储蓄下降
- 投资不变（短期）
- 净出口下降

**对世界：**
- 世界储蓄下降
- 世界利率上升
- 其他国家投资下降

**贸易政策：**

**贸易限制：**

**与小国不同：**
- 大国贸易限制影响世界价格
- 净出口可能增加
- 实际汇率变化

**图形分析：**


### 5.4.5 案例研究：美国与全球储蓄过剩

**背景：**
2000年后，美国贸易赤字持续，全球储蓄过剩。

**全球储蓄过剩（Global Saving Glut）：**

**来源：**
- 亚洲国家高储蓄
- 石油出口国高储蓄
- 老龄化社会储蓄

**影响：**

**1. 美国利率**
- 全球资本流入美国
- 美国利率下降
- 投资增加

**2. 美国贸易赤字**
- 进口增加
- 净出口下降

**3. 全球失衡**
- 美国赤字，亚洲盈余
- 资本流动失衡

**政策含义：**

**1. 美国政策**
- 提高储蓄
- 减少赤字

**2. 亚洲政策**
- 增加消费
- 减少储蓄

**3. 全球协调**
- 重新平衡全球经济
- 避免危机

## 本章总结

### 核心要点

1. **国际收支平衡**
   - 经常账户、资本账户、金融账户
   - NX = NCO
   - 储蓄、投资与净出口的关系

2. **汇率**
   - 名义汇率与实际汇率
   - 购买力平价理论
   - 汇率制度：固定、浮动、有管理的浮动

3. **小型开放经济**
   - 世界利率外生给定
   - S - I = NX
   - 政策效果：财政扩张、贸易政策、投资激励

4. **大型开放经济**
   - 影响世界利率
   - 政策有溢出效应
   - 全球资本市场的均衡

5. **政策协调**
   - 三难选择
   - 国际政策协调的重要性

### 重要概念表格

| 概念 | 定义 | 公式/关系 | 重要性 |
|------|------|----------|--------|
| 经常账户 | 记录商品、服务、收入、转移支付的交易 | CA = NX + NFP + NTR | 国际收支 |
| 净出口 | 出口减进口 | NX = X - M | 贸易余额 |

| 购买力平价 | 汇率使购买力相等 | e = P/P* | 汇率理论 |
