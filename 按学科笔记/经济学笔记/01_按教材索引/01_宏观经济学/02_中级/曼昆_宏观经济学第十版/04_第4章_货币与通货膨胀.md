## 4.1 什么是货币

### 4.1.1 货币的定义

**货币（Money）**是指在商品和服务交换中被广泛接受的支付手段，是经济中的资产，可以用于购买商品和服务、偿还债务。

**货币与财富、收入的区别：**

**1. 货币（Money）**
- 流动性最强的资产
- 可以直接用于交易
- 狭义概念

**2. 财富（Wealth）**
- 所有资产减去负债的净值
- 包括货币、股票、债券、房产等
- 广义概念

**3. 收入（Income）**
- 一定时期内的收入流量
- 与存量概念（货币、财富）不同
- 流量概念

**关系：**
```
财富 = 货币 + 其他资产 - 负债
收入是增加财富的来源
```

### 4.1.2 货币的功能

**1. 交易媒介（Medium of Exchange）**
- 在商品和服务交换中被广泛接受
- 解决物物交换的"需求的双重巧合"难题
- 促进专业化分工和贸易

**2. 计价单位（Unit of Account）**
- 衡量商品和服务价值的共同标准
- 便于价格比较和价值衡量
- 简化经济计算

**3. 价值储藏（Store of Value）**
- 将购买力从现在转移到未来
- 相对其他资产，流动性高但收益低
- 帮助人们平滑消费

**例子：**

**交易媒介：**
- 用货币购买商品
- 用货币支付服务

**计价单位：**
- 所有商品都用货币标价
- 便于比较不同商品的价值

**价值储藏：**
- 存钱以备未来使用
- 应对紧急情况

### 4.1.3 货币的类型

**1. 商品货币（Commodity Money）**
- 本身具有内在价值
- 可以作为商品使用
- 历史形式：黄金、白银、贝壳

**2. 法定货币（Fiat Money）**
- 本身没有内在价值
- 由政府法令确立为货币
- 价值来源于政府法令和公众信心
- 现代货币的主要形式

**3. 部分准备金银行货币（Fractional Reserve Banking Money）**
- 银行创造的货币
- 基于部分准备金制度
- 超过准备金的存款作为货币

**4. 数字货币（Digital Currency）**
- 电子形式的货币
- 包括银行卡、电子支付、加密货币
- 未来发展方向

### 4.1.4 货币供给的衡量

**美国货币供给统计：**

**1. M1（最狭义货币）**
- 流通中的现金
- 活期存款（可开支票的存款）
- 旅行支票

**特点：**
- 流动性最强
- 直接用于交易
- 央行主要监控指标

**2. M2（广义货币）**
- M1
- 储蓄存款
- 小额定期存款（10万美元以下）
- 货币市场存款账户
- 货币市场共同基金

**特点：**
- 流动性较强
- 包括近似货币
- 更全面的货币衡量

**3. M3（最广义货币）**
- M2
- 大额定期存款
- 机构货币市场基金
- 回购协议

**特点：**
- 覆盖最广
- 包括机构性存款
- 较少使用

**中国货币供给统计：**

**1. M0**
- 流通中现金
- 狭义货币

**2. M1**
- M0
- 企业活期存款
- 机关团体部队存款
- 农村存款
- 个人持有的信用卡类存款

**3. M2**
- M1
- 准货币（定期存款、储蓄存款、其他存款）

**比较：**

| 货币统计 | 美国 | 中国 |
|---------|------|------|
| 最狭义 | M1 | M0 |
| 广义 | M2 | M1, M2 |
| 最广义 | M3 | （通常不用） |

**货币供给的决定因素：**

**1. 基础货币（High-Powered Money, H）**
```
H = C + R
其中：C = 流通中的现金
     R = 银行准备金
```

**2. 货币乘数（Money Multiplier, m）**
```
M = m × H
```

**3. 准备金率（Reserve Ratio, rr）**
```
rr = R / D
其中：D = 存款
```

**4. 现金存款比（Currency-Deposit Ratio, c）**
```
c = C / D
```

**货币乘数推导：**
```
M = C + D = (c + 1)D
R = rr × D
H = C + R = cD + rrD = (c + rr)D
M = (c + 1)/(c + rr) × H
m = (c + 1)/(c + rr)
```

### 4.1.5 货币创造过程

**部分准备金银行制度（Fractional Reserve Banking）：**

**前提：**
- 银行只需保留部分存款作为准备金
- 剩余部分可以贷出
- 创造货币

**简单例子：**

**初始状态：**
- A银行：准备金100，存款100
- 准备金率：10%

**步骤1：**
A银行贷款90（保留10准备金）
- 存款者收到90，存入B银行

**步骤2：**
B银行：存款90，准备金9，贷款81
- 借款者收到81，存入C银行

**步骤3：**
C银行：存款81，准备金8.1，贷款72.9

**无限继续...**

**货币创造总和：**
```
初始存款：100
第一轮：90
第二轮：81
第三轮：72.9
...
总和 = 100 × (1 + 0.9 + 0.9² + 0.9³ + ...)
    = 100 × 1/(1 - 0.9)
    = 100 × 10
    = 1000
```

**货币乘数：**
```
m = 1/rr = 1/0.1 = 10
```

**货币供给：**
```
M = m × H = 10 × 100 = 1000
```

**限制因素：**

**1. 准备金率**
- rr越高，m越小
- 央行通过准备金要求控制

**2. 现金存款比**
- c越高，m越小
- 公众的持币行为

**3. 银行借贷意愿**
- 经济信心
- 风险偏好
- 监管要求

### 4.1.6 央行与货币政策

**中央银行（Central Bank）的角色：**

**1. 货币发行者**
- 发行货币
- 控制货币供给

**2. 最后贷款人**
- 向银行提供流动性
- 防止银行挤兑

**3. 政策制定者**
- 制定货币政策
- 稳定经济

**4. 监管者**
- 监管银行系统
- 维护金融稳定

**美联储（Federal Reserve）的结构：**

**1. 联邦储备委员会**
- 7名成员
- 由总统任命，参议院确认
- 任期14年

**2. 联邦公开市场委员会（FOMC）**
- 12名成员
- 制定货币政策
- 决定利率

**3. 12家地区联邦储备银行**
- 私有制
- 为地区银行服务

**货币政策工具：**

**1. 公开市场操作（Open Market Operations）**
- 买入/卖出政府债券
- 最常用的工具
- 灵活，可精确控制

**2. 贴现率（Discount Rate）**
- 央行向银行贷款的利率
- 信号作用
- 影响银行借贷

**3. 法定准备金率（Reserve Requirements）**
- 银行必须保留的准备金比例
- 影响货币乘数
- 不常用

**4. 利率目标**
- 联邦基金利率
- 现代主要工具
- 通过公开市场操作实现

### 4.1.7 案例研究：量化宽松（QE）

**背景：**
2008年金融危机后，传统货币政策（利率降至零）失效，央行采取非常规政策。

**量化宽松（Quantitative Easing, QE）：**

**定义：**
央行大规模购买长期资产，增加货币供给，降低长期利率。

**操作：**
- 美联储购买长期国债
- 购买抵押贷款支持证券（MBS）
- 扩大资产负债表

**规模：**

| 轮次 | 时期 | 规模 | 资产类型 |
|------|------|------|---------|
| QE1 | 2008-2010 | 1.75万亿美元 | 国债、MBS |
| QE2 | 2010-2011 | 6000亿美元 | 国债 |
| QE3 | 2012-2014 | 1.6万亿美元 | MBS、国债 |

**目标：**
- 降低长期利率
- 刺激投资和消费
- 避免通缩
- 支持经济复苏

**效果：**
- 长期利率下降
- 股市上涨
- 避免通缩
- 经济逐步复苏

**争议：**
- 可能导致资产泡沫
- 货币过度扩张
- 财政货币化
- 退出难题

## 4.2 货币的价值

### 4.2.1 货币的价值衡量

**货币价值可以用两种方式衡量：**

**1. 实际价值（Real Value）**
- 货币能购买的商品和服务的数量
- 与物价水平成反比

**2. 名义价值（Nominal Value）**
- 货币的面值
- 固定不变

**公式：**
```
货币的实际价值 = 1 / P
其中：P = 物价水平
```

**例子：**
如果物价水平为2（基年=1）：
- 1元货币的实际价值 = 1/2 = 0.5单位商品
- 货币贬值一半

### 4.2.2 货币需求

**人们为什么持有货币？**

**1. 交易动机（Transactions Motive）**
- 用于日常交易
- 与收入水平正相关
- 与利率负相关

**2. 预防动机（Precautionary Motive）**
- 应对不确定的未来支出
- 与收入水平正相关
- 与利率负相关

**3. 投机动机（Speculative Motive）**
- 等待更好的投资机会
- 与利率负相关
- 与利率波动性正相关

**货币需求函数：**
```
(M/P)^d = L(Y, i)
其中：M/P = 实际货币余额
     Y = 收入
     i = 名义利率
```

**性质：**
- ∂L/∂Y > 0：收入增加，货币需求增加
- ∂L/∂i < 0：利率上升，货币需求减少

**图形分析：货币需求曲线**

```
名义利率 i
  ↑
  │    L(Y)
  │   ↘
  │    ↘
  │     ↘
  │      ↘
  └────────→ 实际货币余额 M/P

向下倾斜：利率与货币需求负相关
```

### 4.2.3 货币市场均衡

**货币供给：**
```
(M/P)^s = M/P
```

- 由央行控制
- 外生变量
- 垂直曲线

**货币需求：**
```
(M/P)^d = L(Y, i)
```

- 内生变量
- 取决于收入和利率

**均衡条件：**
```
(M/P)^s = (M/P)^d
M/P = L(Y, i)
```

**图形分析：货币市场均衡**

```
名义利率 i
  ↑
  │    (M/P)^s
  │    |
  │    |
  │    | 均衡点
  │    *
  │   /|
  │  / | (M/P)^d
  └───┴──→ 实际货币余额 M/P
      i*
      M/P
```

**均衡调整：**

**如果 (M/P)^s > (M/P)^d（供给过剩）：**
- 利率下降
- 货币需求增加
- 回到均衡

**如果 (M/P)^s < (M/P)^d（需求过剩）：**
- 利率上升
- 货币需求减少
- 回到均衡

### 4.2.4 货币流通速度

**定义：**
货币流通速度是衡量货币在经济中流通速度的指标。

**公式：**
```
V = PY / M
其中：V = 货币流通速度
     P = 物价水平
     Y = 实际产出
     M = 货币供给
     PY = 名义GDP
```

**含义：**
- 平均每单位货币在一定时期内被使用的次数
- 货币周转速度

**货币数量论（Quantity Theory of Money）：**

**费雪方程式（Fisher Equation）：**
```
MV = PY
```

**假设：**
- 货币流通速度V相对稳定
- 由制度因素决定（支付习惯、金融技术）

**推论：**
```
M增加 → PY同比例增加
```

### 4.2.5 货币数量论与通货膨胀

**从货币数量论推导通胀：**

**1. 写出费雪方程式：**
```
MV = PY
```

**2. 转换为增长率形式：**
```
ΔM/M + ΔV/V = ΔP/P + ΔY/Y
```

**3. 假设V稳定（ΔV/V = 0）：**
```
ΔM/M = ΔP/P + ΔY/Y
```

**4. 得到通胀率：**
```
π = ΔP/P = ΔM/M - ΔY/Y
```

**其中：**
- π = 通货膨胀率
- ΔM/M = 货币增长率
- ΔY/Y = 实际GDP增长率

**含义：**
- 通胀率 = 货币增长率 - 产出增长率
- 如果货币增长超过产出增长，就会出现通胀
- 如果货币增长低于产出增长，就会出现通缩

**图形分析：货币增长与通胀**

```
通胀率 π
  ↑
  │
  │          /
  │         /
  │        /
  │       /
  │      /
  └────────→ 货币增长率 ΔM/M

斜率 = 1
截距 = -ΔY/Y
```

**例子：**
如果货币增长率为10%，产出增长率为3%：
通胀率 = 10% - 3% = 7%

### 4.2.6 实证证据

**历史数据：**

| 国家 | 时期 | 货币增长率 | 通胀率 | 产出增长率 |
|------|------|-----------|--------|-----------|
| 美国 | 1960-2020 | 5-7% | 3-4% | 3% |
| 德国 | 1920-1923 | >1000%/月 | >1000%/月 | 负 |
| 日本 | 1990-2020 | 2-4% | 0-1% | 1% |
| 委内瑞拉 | 2010-2020 | >1000%/年 | >1000%/年 | 负 |

**观察：**
- 高货币增长对应高通胀
- 超高通胀（恶性通胀）对应超高货币增长
- 货币增长与通胀长期正相关

**长期关系：**
- 货币数量论在长期成立
- 短期可能偏离（V不稳定）
- 制度变化影响V

### 4.2.7 案例研究：恶性通胀

**定义：**
月通胀率超过50%的极端通货膨胀。

**历史案例：**

**1. 德国魏玛共和国（1921-1923）**
- 1923年11月月通胀率 > 25000%
- 物价每几小时翻倍
- 面包价格从0.6马克涨到2000亿马克
- 货币体系崩溃

**2. 津巴布韦（2000-2009）**
- 2008年11月月通胀率 > 79,000,000,000%
- 发行100万亿津元纸币
- 最终放弃本国货币

**3. 委内瑞拉（2010-2020）**
- 年通胀率超过1000%
- 货币大幅贬值
- 经济崩溃

**恶性通胀的特征：**

**1. 货币价值急剧下降**
- 钱几乎无用
- 物物交换增加

**2. 经济混乱**
- 价格信号失效
- 资源配置扭曲
- 生产下降

**3. 社会动荡**
- 财富再分配
- 固定收入者受损
- 社会不稳定

**原因：**

**1. 货币过度扩张**
- 政府印钞支付支出
- 财政赤字货币化
- 货币供给失控

**2. 预期失控**
- 通胀预期自我实现
- 人们提前消费
- 货币流通速度上升

**3. 政策失败**
- 央行失去独立性
- 政治干预
- 政策不一致

**解决方案：**

**1. 货币改革**
- 停止印钞
- 控制货币供给
- 发行新货币

**2. 财政整顿**
- 减少赤字
- 削减支出
- 增加税收

**3. 制度改革**
- 央行独立
- 货币规则
- 恢复信心

**4. 国际援助**
- 外汇储备支持
- 国际货币基金组织援助
- 技术支持

## 4.3 通货膨胀

### 4.3.1 通货膨胀的定义

**通货膨胀（Inflation）**是指物价水平持续上涨的现象。

**关键要素：**
1. 物价水平上涨
2. 持续性（非一次性）
3. 一般物价水平（非个别商品）

**通货膨胀率：**
```
π = (P_t - P_{t-1}) / P_{t-1} × 100%
```

### 4.3.2 通货膨胀的类型

**1. 温和通货膨胀（Moderate Inflation）**
- 年通胀率：3-10%
- 常见情况
- 通常不会严重损害经济

**2. 奔腾式通货膨胀（Galloping Inflation）**
- 年通胀率：10-1000%
- 严重但可控
- 需要政策干预

**3. 恶性通货膨胀（Hyperinflation）**
- 月通胀率超过50%
- 极端情况
- 货币体系崩溃

**4. 停滞膨胀（Stagflation）**
- 高通胀 + 高失业 + 低增长
- 1970年代石油危机后出现
- 政策难题

### 4.3.3 通货膨胀的原因

**需求拉动型通货膨胀（Demand-Pull Inflation）：**

**原因：**
总需求超过总供给，拉动物价上涨。

**机制：**
```
AD增加 → Y增加 → 产出接近潜在水平 → 物价上涨
```

**图形分析：**

```
物价水平 P
  ↑
  │    AS
  │   /
  │  /
  │ /  AD₁ → AD₂
  │/
  └────────→ 产出 Y

AD右移：产出增加，物价上涨
```

**成本推动型通货膨胀（Cost-Push Inflation）：**

**原因：**
生产成本上升，推动物价上涨。

**机制：**
```
成本上升 → AS左移 → 产出减少，物价上涨
```

**图形分析：**

```
物价水平 P
  ↑
  │    AS₁ → AS₂
  │   ↘
  │    ↘
  │     ↘
  │      ↘ AD
  └────────→ 产出 Y

AS左移：产出减少，物价上涨
```

**货币性通货膨胀（Monetary Inflation）：**

**原因：**
货币供给过度增长，导致物价上涨。

**机制：**
```
M增加 → 需求增加 → 物价上涨
```

**结构性通货膨胀（Structural Inflation）：**

**原因：**
经济结构变化，某些部门需求增加导致物价上涨。

**预期性通货膨胀（Expectational Inflation）：**

**原因：**
通胀预期导致价格和工资提前上涨。

### 4.3.4 通货膨胀的预期

**适应性预期（Adaptive Expectations）：**

**定义：**
人们根据过去的通胀形成预期。

**公式：**
```
π^e_t = π^e_{t-1} + λ(π_{t-1} - π^e_{t-1})
```

**特点：**
- 向后看
- 缓慢调整
- 可能存在系统误差

**理性预期（Rational Expectations）：**

**定义：**
人们利用所有可获得的信息理性地预测通胀。

**公式：**
```
π^e_t = E[π_t | 信息集_t]
```

**特点：**
- 向前看
- 快速调整
- 无系统误差

**预期的形成：**
```
实际通胀 = 预期通胀 + 意外冲击
π = π^e + ε
```

**预期的作用：**
- 影响工资谈判
- 影响价格设定
- 影响储蓄和投资决策
- 影响政策效果

### 4.3.5 通货膨胀与失业的关系

**短期菲利普斯曲线（Short-Run Phillips Curve）：**

**定义：**
短期内通胀与失业之间存在权衡关系。

**公式：**
```
π = π^e - β(u - u^n) + v
```

**其中：**
- π = 实际通胀率
- π^e = 预期通胀率
- u = 实际失业率
- u^n = 自然失业率
- β > 0
- v = 供给冲击

**图形分析：**

```
通胀率 π
  ↑
  │
  │   /
  │  /  SRPC₁ (π^e = 5%)
  │ /
  │/
  └────────→ 失业率 u
      u^n
```

**含义：**
- 要降低失业率，必须接受更高的通胀
- 通胀率取决于预期通胀和失业缺口

**长期菲利普斯曲线（Long-Run Phillips Curve）：**

**定义：**
长期内通胀与失业不存在权衡关系，菲利普斯曲线垂直。

**公式：**
```
u = u^n
```

**图形分析：**

```
通胀率 π
  ↑
  │
  │    LRPC
  │    |
  │    |
  │    |
  └────────→ 失业率 u
         u^n

垂直：长期失业率 = 自然失业率
```

**含义：**
- 长期内，任何通胀率都对应自然失业率
- 货币政策只能影响通胀，不能影响失业
- 通胀中性

**菲利普斯曲线的移动：**

**1. 预期变化：**
预期通胀上升 → 短期菲利普斯曲线右移

**2. 供给冲击：**
不利供给冲击 → 短期菲利普斯曲线右移

**图形分析：**

```
通胀率 π
  ↑
  │    SRPC₁ (π^e = 5%)
  │   /
  │  /
  │ /  SRPC₂ (π^e = 10%)
  │/
  └────────→ 失业率 u
      u^n
```

### 4.3.6 案例研究：1970年代大通胀

**背景：**
1970年代，美国和西方国家经历了高通胀。

**数据：**

| 年份 | 通胀率 | 失业率 | 石油价格变化 |
|------|--------|--------|-------------|
| 1970 | 5.7% | 4.9% | 正常 |
| 1973 | 6.2% | 4.9% | +300% |
| 1974 | 11.0% | 5.6% | +100% |
| 1975 | 9.1% | 8.5% | 正常 |
| 1979 | 11.3% | 5.8% | +200% |
| 1980 | 13.5% | 7.1% | 正常 |
| 1981 | 10.3% | 7.6% | 正常 |

**原因：**

**1. 供给冲击**
- 1973年和1979年石油危机
- 石油价格暴涨
- 成本推动型通胀

**2. 需求因素**
- 越南战争支出
- 大社会项目支出
- 需求拉动型通胀

**3. 货币政策**
- 货币过度扩张
- 美联储反应迟缓
- 货币增长率高

**4. 预期因素**
- 通胀预期上升
- 工资-物价螺旋
- 自我实现的通胀

**应对：**

**美联储政策：**
- 沃尔克（Paul Volcker）担任主席（1979）
- 紧缩货币政策
- 提高利率（联邦基金利率超过20%）
- 控制货币增长

**结果：**
- 通胀率从13.5%降到3.2%（1983）
- 失业率上升到10.8%（1982）
- 严重的经济衰退
- 长期来看，通胀预期锚定

**启示：**

**1. 通胀预期的重要性**
- 一旦预期上升，难以控制
- 需要强硬政策重新锚定

**2. 央行的信誉**
- 央行需要可信的承诺
- 独立性和透明度重要

**3. 短期代价**
- 降低通胀需要付出代价
- 失业和衰退不可避免

**4. 政策规则**
- 确定性和一致性
- 避免相机抉择的过度反应

## 4.4 通货膨胀的成本

### 4.4.1 预期到的通胀成本

**1. 皮鞋成本（Shoeleather Costs）**

**定义：**
为避免持有现金而产生的成本。

**原因：**
- 通胀时，现金贬值快
- 人们减少现金持有
- 增加去银行取款的次数
- 产生时间和交易成本

**计算：**
```
成本 ≈ 持有现金的机会成本
    ≈ 通胀率 × 现金持有量
```

**2. 菜单成本（Menu Costs）**

**定义：**
企业调整价格的成本。

**内容：**
- 重新印刷菜单
- 更新价格标签
- 通知客户
- 计算新价格

**影响：**
- 资源浪费
- 价格调整不频繁
- 价格刚性

**3. 相对价格扭曲**

**问题：**
- 通胀时，不同商品价格调整频率不同
- 导致相对价格偏离最优水平
- 资源配置效率下降

**例子：**
- 餐厅价格每月调整
- 汽油价格每天调整
- 相对价格不断变化
- 消费者和企业难以决策

**4. 税收扭曲**

**问题：**
- 税制通常未考虑通胀
- 名义收入增加导致实际税负增加

**例子：**
- 资本利得税
- 利息收入税
- 累进税率

**计算：**
```
名义利率 = 10%
通胀率 = 5%
实际利率 = 5%
如果税率 = 25%
税后名义利率 = 7.5%
税后实际利率 = 7.5% - 5% = 2.5%
实际税负 = (2.5% / 5%) = 50%
```

**5. 通胀税（Inflation Tax）**

**定义：**
政府通过发行货币获取的收入，相当于对货币持有者征税。

**公式：**
```
通胀税收入 = (M/P) × π
通胀税率 = π
```

**影响：**
- 实际上是政府收入
- 对持有现金的人征税
- 累退性质（穷人持有更多现金）

### 4.4.2 未预期到的通胀成本

**1. 任意的财富再分配**

**借贷关系：**
- 贷款者：预期回报低于实际
- 借款者：实际支付低于预期

**例子：**
```
贷款100元，利率5%，预期通胀3%
预期实际回报 = 5% - 3% = 2%

实际通胀 = 5%
实际回报 = 5% - 5% = 0%

贷款者受损，借款者受益
```

**再分配方向：**
- 从债权人到债务人
- 从固定收入者到可变收入者
- 从现金持有者到实物资产持有者

**2. 预测困难**

**问题：**
- 企业难以定价
- 投资者难以决策
- 工人难以谈判工资

**影响：**
- 增加不确定性
- 降低经济效率
- 减少投资

**3. 资源配置扭曲**

**问题：**
- 通胀不确定性导致错误的资源配置
- 企业过度关注短期
- 长期投资减少

**4. 政策可信度**

**问题：**
- 政府和央行信誉受损
- 政策效果下降
- 预期更难控制

### 4.4.3 恶性通胀的成本

**1. 货币体系崩溃**
- 货币失去价值
- 物物交换增加
- 交易成本急剧上升

**2. 经济混乱**
- 价格信号失效
- 资源配置极度扭曲
- 生产活动减少

**3. 社会动荡**
- 财富急剧再分配
- 社会不满
- 政治不稳定

**4. 制度破坏**
- 市场机制失效
- 政府信誉丧失
- 长期影响严重

### 4.4.4 通货紧缩的成本

**定义：**
物价水平持续下降（通胀率 < 0）。

**成本：**

**1. 实际债务负担增加**
- 名义债务不变
- 物价下降，实际债务增加
- 债务人受损，债权人受益

**2. 消费延迟**
- 预期物价继续下降
- 延迟消费
- 总需求减少

**3. 利率下限（Zero Lower Bound）**
- 名义利率不能低于零
- 货币政策空间受限
- 难以刺激经济

**4. 通缩螺旋**
- 需求减少 → 物价下降 → 实际债务增加 → 需求进一步减少
- 恶性循环

**案例：日本"失去的几十年"**
- 1990年代后长期通缩或低通胀
- 经济增长缓慢
- 货币政策效果有限
- 陷入流动性陷阱

### 4.4.5 最优通胀率

**通胀目标制的兴起：**

**各国通胀目标：**

| 央行 | 通胀目标 | 时间 |
|------|---------|------|
| 美联储 | 2% | 2012 |
| 欧央行 | 2% | 1998 |
| 英格兰银行 | 2% | 1992 |
| 日本央行 | 2% | 2013 |
| 中国人民银行 | 3%左右 | 近年 |

**为什么是2%？**

**支持2%的理由：**

**1. 为负利率提供缓冲**
- 理想实际利率可能为负
- 需要2%通胀提供空间

**2. 避免通缩风险**
- 提供安全边际
- 防止意外通缩

**3. 调整灵活性**
- 相对工资调整更容易
- 价格调整更频繁

**4. 测量误差**
- CPI测量可能有向上偏差
- 目标略高于零考虑这一点

**其他观点：**

**支持更高通胀（4-5%）的理由：**
- 更多的政策空间
- 降低实际债务负担
- 避免零下限风险

**支持零通胀的理由：**
- 减少通胀成本
- 保持购买力
- 长期稳定性

**结论：**
- 目前2%是国际主流共识
- 但存在学术和政策讨论
- 可根据情况调整

### 4.4.6 案例研究：日本通缩困境

**背景：**
1990年代初泡沫经济破灭后，日本进入长期低增长和通缩。

**数据：**

| 时期 | 年均GDP增长 | 年均通胀 | 年均失业率 |
|------|------------|---------|-----------|
| 1990-2000 | 1.5% | 0.8% | 3.5% |
| 2000-2010 | 0.7% | -0.3% | 4.5% |
| 2010-2020 | 0.9% | 0.6% | 3.5% |

**通缩特征：**

**1. 物价长期低迷**
- CPI接近零或负值
- 核心通胀率低
- 通胀预期低

**2. 需求不足**
- 消费增长缓慢
- 投资低迷
- 出口波动

**3. 债务问题**
- 公共债务高企
- 企业债务负担
- 通缩增加实际债务

**4. 流动性陷阱**
- 利率接近零
- 货币政策效果有限
- 量化宽松效果有限

**政策应对：**

**1. 货币政策**
- 零利率政策（ZIRP）
- 量化宽松（QE）
- 质量化宽松（QQE）
- 负利率政策（NIRP）

**2. 财政政策**
- 大规模财政刺激
- 公共投资
- 消费税调整

**3. 结构改革**
- 安倍三箭（三支箭）
- 货币宽松
- 财政刺激
- 结构改革

**效果评估：**

**有限效果：**
- 通胀率仍低于目标
- 经济增长缓慢
- 政策空间有限

**部分成功：**
- 避免了严重衰退
- 就业相对稳定
- 企业盈利改善

**挑战：**
- 通缩预期难以改变
- 人口结构问题
- 劳动生产率增长缓慢

**启示：**

**1. 通缩难以逆转**
- 一旦形成，难以改变
- 需要持续和有力的政策

**2. 预期的重要性**
- 预期自我实现
- 需要改变预期

**3. 政策组合**
- 货币政策不够
- 需要财政和结构政策配合

**4. 长期挑战**
- 人口老龄化
- 劳动力减少
- 生产率增长缓慢

## 本章总结

### 核心要点

1. **货币的定义和功能**
   - 货币是交易媒介、计价单位、价值储藏
   - M1、M2等货币供给统计
   - 部分准备金银行制度创造货币

2. **货币数量论**
   - MV = PY
   - 通胀率 = 货币增长率 - 产出增长率
   - 长期货币增长决定通胀

3. **通货膨胀的原因**
   - 需求拉动型
   - 成本推动型
   - 货币性
   - 预期性

4. **通货膨胀的成本**
   - 预期到的通胀：皮鞋成本、菜单成本、相对价格扭曲、税收扭曲
   - 未预期到的通胀：财富再分配、预测困难
   - 恶性通胀：货币体系崩溃

5. **货币政策**
   - 央行通过公开市场操作控制货币供给
   - 通胀目标制
   - 最优通胀率约2%
