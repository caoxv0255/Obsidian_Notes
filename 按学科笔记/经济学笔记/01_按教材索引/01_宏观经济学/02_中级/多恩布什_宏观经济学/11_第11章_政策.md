# 第11章：政策

> 本章学习目标：

## 11.1 政策目标

### 11.1.1 政策目标的定义

**政策目标（Policy Goals）**：
政府希望通过政策实现的目标。

### 11.1.2 政策目标的类型

**政策目标的类型（Types of Policy Goals）**：

**1. 充分就业（Full Employment）**：
- 失业率等于自然失业率
- 充分就业不等于零失业

**2. 价格稳定（Price Stability）**：
- 通货膨胀率低且稳定
- 通常指通货膨胀率约为2%

**3. 经济增长（Economic Growth）**：
- 经济持续增长
- 提高生活水平

**4. 国际收支平衡（Balance of Payments Equilibrium）**：
- 国际收支平衡
- 汇率稳定

### 11.1.3 政策目标的权衡

**政策目标的权衡（Trade-off among Policy Goals）**：

**1. 通货膨胀与失业的权衡（Inflation-Unemployment Trade-off）**：
- 短期：菲利普斯曲线，通货膨胀与失业负相关
- 长期：垂直菲利普斯曲线，通货膨胀与失业无关

**2. 通货膨胀与经济增长的权衡（Inflation-Growth Trade-off）**：
- 通货膨胀率过高 → 不确定性增加 → 投资减少 → 经济增长放缓
- 通货膨胀率过低 → 名义利率接近零 → 货币政策无效 → 经济增长放缓

**3. 失业与经济增长的权衡（Unemployment-Growth Trade-off）**：
- 奥肯定律：失业率变化与产出增长负相关
- 经济增长 → 失业率下降
- 经济衰退 → 失业率上升

## 11.2 政策工具

### 11.2.1 政策工具的定义

**政策工具（Policy Instruments）**：
政府用于实现政策目标的工具。

### 11.2.2 财政政策

**财政政策（Fiscal Policy）**：
政府通过政府支出和税收来影响经济的政策。

**财政政策工具（Fiscal Policy Tools）**：

**1. 政府支出（Government Spending, G）**：
- 政府支出增加 → 总需求增加 → 产出增加
- 政府支出减少 → 总需求减少 → 产出减少

**2. 税收（Taxes, T）**：
- 税收增加 → 可支配收入减少 → 消费减少 → 总需求减少 → 产出减少
- 税收减少 → 可支配收入增加 → 消费增加 → 总需求增加 → 产出增加

**财政乘数（Fiscal Multiplier）**：
```
ΔY = (1 / (1 - MPC))·ΔG
```
**其中**：
- ΔY：产出变化
- ΔG：政府支出变化
- MPC：边际消费倾向

**含义**：
- 政府支出增加 → 产出增加的倍数大于1
- 财政乘数 = 1 / (1 - MPC)

### 11.2.3 货币政策

**货币政策（Monetary Policy）**：
中央银行通过控制货币供给和利率来影响经济的政策。

**货币政策工具（Monetary Policy Tools）**：

**1. 公开市场操作（Open Market Operations）**：
- 购买债券 → 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 出售债券 → 货币供给减少 → 利率上升 → 投资减少 → 产出减少

**2. 准备金率（Reserve Requirements）**：
- 准备金率下降 → 货币乘数上升 → 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 准备金率上升 → 货币乘数下降 → 货币供给减少 → 利率上升 → 投资减少 → 产出减少

**3. 贴现率（Discount Rate）**：
- 贴现率下降 → 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 贴现率上升 → 货币供给减少 → 利率上升 → 投资减少 → 产出减少

**4. 联邦基金利率（Federal Funds Rate）**：
- 联邦基金利率下降 → 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 联邦基金利率上升 → 货币供给减少 → 利率上升 → 投资减少 → 产出减少

## 11.3 政策规则

### 11.3.1 政策规则的定义

**政策规则（Policy Rules）**：
政策按照预先确定的规则执行。

### 11.3.2 政策规则与相机抉择

**政策规则与相机抉择（Policy Rules vs. Discretion）**：

**1. 政策规则（Policy Rules）**：
- 政策按照预先确定的规则执行
- 例如：泰勒规则、货币增长率规则

**2. 相机抉择（Discretion）**：
- 政策根据具体情况灵活调整
- 例如：凯恩斯主义政策

### 11.3.3 泰勒规则

**泰勒规则（Taylor Rule）**：
联邦基金利率的设定规则。

**公式**：
```
i = r* + π + 0.5·(π - π*) + 0.5·(Y - Y*)
```
**其中**：
- i：联邦基金利率
- r*：实际均衡利率（约为2%）
- π：通货膨胀率
- π*：目标通货膨胀率（约为2%）
- Y：实际产出
- Y*：潜在产出

**含义**：
- 通货膨胀率高于目标 → 联邦基金利率上升
- 产出高于潜在产出 → 联邦基金利率上升
- 通货膨胀率低于目标 → 联邦基金利率下降
- 产出低于潜在产出 → 联邦基金利率下降

### 11.3.4 货币增长率规则

**货币增长率规则（Money Growth Rate Rule）**：
货币供给按照固定增长率增长。

**公式**：
```
ΔM/M = g
```
**其中**：
- ΔM/M：货币供给增长率
- g：固定增长率（例如：5%）

**含义**：
- 货币供给按照固定增长率增长
- 货币政策可预测

## 11.4 政策不确定性

### 11.4.1 政策不确定性的定义

**政策不确定性（Policy Uncertainty）**：
经济主体对未来政策的不确定性。

### 11.4.2 政策不确定性的来源

**政策不确定性的来源（Sources of Policy Uncertainty）**：

**1. 政策变化（Policy Changes）**：
- 政府换届 → 政策变化
- 政策调整 → 政策变化

**2. 政策不透明（Policy Opacity）**：
- 政策不清晰 → 不确定性增加
- 政策不透明 → 不确定性增加

**3. 政策不一致（Policy Inconsistency）**：
- 政策不一致 → 不确定性增加
- 政策冲突 → 不确定性增加

### 11.4.3 政策不确定性的影响

**政策不确定性的影响（Effects of Policy Uncertainty）**：

**1. 投资减少（Decreased Investment）**：
- 政策不确定性 → 未来收益不确定 → 投资减少

**2. 消费减少（Decreased Consumption）**：
- 政策不确定性 → 未来收入不确定 → 消费减少

**3. 经济增长放缓（Slower Economic Growth）**：
- 投资减少 → 经济增长放缓
- 消费减少 → 经济增长放缓

## 11.5 政策协调

### 11.5.1 政策协调的定义

**政策协调（Policy Coordination）**：
不同政策之间的协调。

### 11.5.2 财政政策与货币政策的协调

**财政政策与货币政策的协调（Coordination between Fiscal Policy and Monetary Policy）**：

**1. 扩张性财政政策 + 扩张性货币政策**：
- 政府支出增加 → 总需求增加 → 产出增加
- 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 产出大幅增加

**2. 紧缩性财政政策 + 紧缩性货币政策**：
- 政府支出减少 → 总需求减少 → 产出减少
- 货币供给减少 → 利率上升 → 投资减少 → 产出减少
- 产出大幅减少

**3. 扩张性财政政策 + 紧缩性货币政策**：
- 政府支出增加 → 总需求增加 → 产出增加
- 货币供给减少 → 利率上升 → 投资减少 → 产出减少
- 利率上升，产出不确定

**4. 紧缩性财政政策 + 扩张性货币政策**：
- 政府支出减少 → 总需求减少 → 产出减少
- 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 利率下降，产出不确定

### 11.5.3 国际政策协调

**国际政策协调（International Policy Coordination）**：
不同国家之间的政策协调。

**例子**：
- 汇率协调
- 贸易政策协调
- 宏观经济政策协调

## 11.6 本章总结

### 核心要点

1. **政策目标**：
   - 政策目标：充分就业、价格稳定、经济增长、国际收支平衡
   - 政策目标的权衡：通货膨胀与失业的权衡、通货膨胀与经济增长的权衡、失业与经济增长的权衡

2. **政策工具**：
   - 财政政策：政府通过政府支出和税收来影响经济的政策
   - 财政政策工具：政府支出、税收
   - 财政乘数：ΔY = (1 / (1 - MPC))·ΔG
   - 货币政策：中央银行通过控制货币供给和利率来影响经济的政策
   - 货币政策工具：公开市场操作、准备金率、贴现率、联邦基金利率

3. **政策规则**：
   - 政策规则：政策按照预先确定的规则执行
   - 政策规则与相机抉择：政策规则 vs. 相机抉择
   - 泰勒规则：i = r* + π + 0.5·(π - π*) + 0.5·(Y - Y*)
   - 货币增长率规则：ΔM/M = g

4. **政策不确定性**：
   - 政策不确定性：经济主体对未来政策的不确定性
   - 政策不确定性的来源：政策变化、政策不透明、政策不一致
   - 政策不确定性的影响：投资减少、消费减少、经济增长放缓

5. **政策协调**：
   - 政策协调：不同政策之间的协调
   - 财政政策与货币政策的协调：扩张性 + 扩张性、紧缩性 + 紧缩性、扩张性 + 紧缩性、紧缩性 + 扩张性
   - 国际政策协调：不同国家之间的政策协调
