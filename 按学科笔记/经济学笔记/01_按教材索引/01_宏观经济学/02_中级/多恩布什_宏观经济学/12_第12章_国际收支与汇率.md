## 12.1 国际收支

### 12.1.1 国际收支的定义

**国际收支（Balance of Payments, BOP）**：
一个国家与世界其他国家之间的交易记录。

### 12.1.2 国际收支的组成

**国际收支的组成（Components of Balance of Payments）**：

**1. 经常账户（Current Account, CA）**：
```
CA = X - M + NY + NCT
```
**其中**：
- CA：经常账户
- X：出口
- M：进口
- NY：净要素收入（国外收入支付）
- NCT：净转移支付（单方面转移）

**经常账户平衡（Current Account Balance）**：
```
CA = S - I
```
**其中**：
- CA：经常账户
- S：储蓄
- I：投资

**含义**：
- 经常账户盈余 = 储蓄 > 投资
- 经常账户赤字 = 储蓄 < 投资

**2. 资本账户（Capital Account, KA）**：
- 记录资本转移和非生产、非金融资产交易
- 通常规模很小

**3. 金融账户（Financial Account, FA）**：
```
FA = KI + FI + R
```
**其中**：
- FA：金融账户
- KI：直接投资
- FI：证券投资
- R：储备资产

### 12.1.3 国际收支平衡

**国际收支平衡（Balance of Payments Equilibrium）**：
```
CA + KA + FA = 0
```
**含义**：
- 国际收支必须平衡
- 经常账户赤字 = 金融账户盈余
- 经常账户盈余 = 金融账户赤字

## 12.2 汇率

### 12.2.1 汇率的定义

**汇率（Exchange Rate）**：
一种货币与另一种货币的兑换比率。

### 12.2.2 汇率的表示方法

**汇率的表示方法（Representation of Exchange Rate）**：

**1. 直接标价法（Direct Quotation）**：
- 以本国货币表示外国货币的价格
- 例如：1美元 = 7人民币

**2. 间接标价法（Indirect Quotation）**：
- 以外国货币表示本国货币的价格
- 例如：1人民币 = 0.14美元

### 12.2.3 名义汇率与实际汇率

**名义汇率（Nominal Exchange Rate, e）**：
一种货币与另一种货币的名义兑换比率。

**实际汇率（Real Exchange Rate, ε）**：
一种货币与另一种货币的实际兑换比率，考虑了价格水平。

**公式**：
```
ε = e·P* / P
```
**其中**：
- ε：实际汇率
- e：名义汇率
- P*：外国价格水平
- P：本国价格水平

**含义**：
- 实际汇率上升 → 本国货币实际贬值 → 出口增加 → 进口减少
- 实际汇率下降 → 本国货币实际升值 → 出口减少 → 进口增加

### 12.2.4 汇率的决定

**汇率的决定（Determination of Exchange Rate）**：
- 供给和需求决定汇率
- 外汇供给：外国人购买本国商品、投资本国
- 外汇需求：本国购买外国商品、投资外国

- 外汇需求增加 → 汇率上升（本国货币贬值）

## 12.3 蒙代尔-弗莱明模型

### 12.3.1 蒙代尔-弗莱明模型的定义

**蒙代尔-弗莱明模型（Mundell-Fleming Model）**：
开放经济中的IS-LM模型。

### 12.3.2 蒙代尔-弗莱明模型的假设

**蒙代尔-弗莱明模型的假设（Assumptions of Mundell-Fleming Model）**：

**1. 小型开放经济（Small Open Economy）**：
- 本国利率等于世界利率
- r = r*

**2. 价格水平固定（Fixed Price Level）**：
- 短期分析
- 价格水平不变

**3. 资本完全流动（Perfect Capital Mobility）**：
- 资本自由流动
- 本国利率等于世界利率

### 12.3.3 蒙代尔-弗莱明模型的方程

**蒙代尔-弗莱明模型的方程（Equations of Mundell-Fleming Model）**：

**1. IS*曲线（IS* Curve）**：
```
Y = C(Y - T) + I(r*) + G + NX(e)
```
**其中**：
- Y：产出
- C：消费
- T：税收
- I：投资
- r*：世界利率
- G：政府购买
- NX：净出口
- e：汇率

**含义**：
- IS*曲线：向右下方倾斜
- 汇率上升 → 净出口减少 → 产出减少

**2. LM*曲线（LM* Curve）**：
```
M/P = L(r*, Y)
```
**其中**：
- M：货币供给
- P：价格水平
- L：货币需求
- r*：世界利率
- Y：产出

**含义**：
- LM*曲线：垂直
- 货币供给变化 → 产出变化

### 12.3.4 浮动汇率下的蒙代尔-弗莱明模型

**浮动汇率下的蒙代尔-弗莱明模型（Mundell-Fleming Model under Floating Exchange Rates）**：

**1. 财政政策（Fiscal Policy）**：
- 政府支出增加 → IS*曲线向右移动 → 汇率上升 → 净出口减少 → 产出不变
- 财政政策无效

**2. 货币政策（Monetary Policy）**：
- 货币供给增加 → LM*曲线向右移动 → 汇率下降 → 净出口增加 → 产出增加
- 货币政策有效

- 财政政策无效，货币政策有效

### 12.3.5 固定汇率下的蒙代尔-弗莱明模型

**固定汇率下的蒙代尔-弗莱明模型（Mundell-Fleming Model under Fixed Exchange Rates）**：

**1. 财政政策（Fiscal Policy）**：
- 政府支出增加 → IS*曲线向右移动 → 汇率上升 → 中央银行增加货币供给 → LM*曲线向右移动 → 产出增加
- 财政政策有效

**2. 货币政策（Monetary Policy）**：
- 货币供给增加 → LM*曲线向右移动 → 汇率下降 → 中央银行减少货币供给 → LM*曲线回到原位 → 产出不变
- 货币政策无效

- 财政政策有效，货币政策无效

## 12.4 汇率制度

### 12.4.1 汇率制度的定义

**汇率制度（Exchange Rate Regime）**：
政府决定汇率的方法。

### 12.4.2 汇率制度的类型

**汇率制度的类型（Types of Exchange Rate Regime）**：

**1. 固定汇率制度（Fixed Exchange Rate Regime）**：
- 汇率固定
- 政府干预外汇市场
- 优点：减少不确定性
- 缺点：货币政策无效

**2. 浮动汇率制度（Floating Exchange Rate Regime）**：
- 汇率由市场决定
- 政府不干预外汇市场
- 优点：货币政策有效
- 缺点：汇率波动大

**3. 管理浮动汇率制度（Managed Float Exchange Rate Regime）**：
- 汇率由市场决定，但政府偶尔干预
- 介于固定汇率制度和浮动汇率制度之间

**4. 货币局制度（Currency Board）**：
- 货币供给与外汇储备挂钩
- 例如：香港的联系汇率制度

**5. 美元化（Dollarization）**：
- 使用外国货币作为本国货币
- 例如：巴拿马使用美元

### 12.4.3 三元悖论

**三元悖论（Trilemma）**：
货币政策独立性、汇率稳定、资本自由流动三者不能同时实现。

- 选择汇率稳定和资本自由流动 → 货币政策无效

## 12.5 本章总结

### 核心要点

1. **国际收支**：
   - 国际收支：一个国家与世界其他国家之间的交易记录
   - 国际收支的组成：经常账户、资本账户、金融账户
   - 国际收支平衡：CA + KA + FA = 0
   - 经常账户平衡：CA = S - I

2. **汇率**：
   - 汇率：一种货币与另一种货币的兑换比率
   - 名义汇率与实际汇率：ε = e·P* / P
   - 汇率的决定：供给和需求决定汇率

3. **蒙代尔-弗莱明模型**：
   - 蒙代尔-弗莱明模型：开放经济中的IS-LM模型
   - 蒙代尔-弗莱明模型的假设：小型开放经济、价格水平固定、资本完全流动
   - 浮动汇率下的蒙代尔-弗莱明模型：财政政策无效，货币政策有效
   - 固定汇率下的蒙代尔-弗莱明模型：财政政策有效，货币政策无效

4. **汇率制度**：
   - 汇率制度：政府决定汇率的方法
   - 汇率制度的类型：固定汇率制度、浮动汇率制度、管理浮动汇率制度、货币局制度、美元化
   - 三元悖论：货币政策独立性、汇率稳定、资本自由流动三者不能同时实现
