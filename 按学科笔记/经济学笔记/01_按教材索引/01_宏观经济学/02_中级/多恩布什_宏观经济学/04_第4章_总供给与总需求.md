# 第4章：总供给与总需求

## 4.1 总供给

### 4.1.1 总供给的定义

**总供给（Aggregate Supply, AS）**：
给定价格水平，经济中的总产出。

### 4.1.2 总供给曲线

**总供给曲线（Aggregate Supply Curve）**：
表示价格水平与总产出之间的关系。

**总供给曲线的类型（Types of Aggregate Supply Curve）**：

**1. 长期总供给曲线（Long-Run Aggregate Supply Curve, LRAS）**：
- 垂直于横轴
- 位于潜在产出水平
- 价格水平不影响总产出

**图形表示**：

```plotly

```

**分析**：
- LRAS：长期总供给曲线，垂直
- Y*：潜在产出（充分就业产出）
- 价格水平不影响总产出

**2. 短期总供给曲线（Short-Run Aggregate Supply Curve, SRAS）**：
- 向右上方倾斜
- 价格水平上升 → 总产出增加
- 价格水平下降 → 总产出减少

**图形表示**：

**分析**：
- SRAS：短期总供给曲线，向右上方倾斜
- 价格水平上升 → 总产出增加
- 价格水平下降 → 总产出减少

### 4.1.3 短期总供给曲线的斜率

**短期总供给曲线的斜率（Slope of Short-Run Aggregate Supply Curve）**：
- 价格水平上升 → 名义工资上升 → 实际工资下降 → 企业增加劳动需求 → 产出增加
- 价格水平下降 → 名义工资下降 → 实际工资上升 → 企业减少劳动需求 → 产出减少

**原因**：
1. 粘性工资：工资在短期内调整缓慢
2. 粘性价格：价格在短期内调整缓慢
3. 错觉理论：企业混淆价格水平变化和相对价格变化

## 4.2 总需求

### 4.2.1 总需求的定义

**总需求（Aggregate Demand, AD）**：
给定价格水平，经济中的总支出。

### 4.2.2 总需求的构成

**总需求的构成（Components of Aggregate Demand）**：
```
Y = C + I + G + (X - M)
```
**其中**：
- C：消费
- I：投资
- G：政府购买
- X：出口
- M：进口

### 4.2.3 总需求曲线

**总需求曲线（Aggregate Demand Curve）**：
表示价格水平与总支出之间的关系。

**图形表示**：

```plotly

  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "产出Y"
  yaxis:
    title:

    range:
      - 0
      - 25
  template: "plotly_white"

  displayModeBar: false
  responsive: true
```

**分析**：
- AD：总需求曲线，向右下方倾斜
- 价格水平上升 → 总支出减少
- 价格水平下降 → 总支出增加

### 4.2.4 总需求曲线的斜率

**总需求曲线的斜率（Slope of Aggregate Demand Curve）**：
- 价格水平上升 → 实际货币供给减少 → 利率上升 → 投资减少 → 总支出减少
- 价格水平下降 → 实际货币供给增加 → 利率下降 → 投资增加 → 总支出增加

**原因**：
1. 凯恩斯效应：价格水平变化 → 实际货币供给变化 → 利率变化 → 投资变化
2. 庇古效应：价格水平变化 → 实际财富变化 → 消费变化
3. 蒙代尔-弗莱明效应：价格水平变化 → 实际汇率变化 → 净出口变化

## 4.3 总供给与总需求的均衡

### 4.3.1 均衡的定义

**均衡（Equilibrium）**：
总供给等于总需求的状态。

**条件**：
```
AS = AD
```

### 4.3.2 短期均衡

**短期均衡（Short-Run Equilibrium）**：

- P₀：短期均衡价格水平

### 4.3.3 长期均衡

**长期均衡（Long-Run Equilibrium）**：
长期总供给等于总需求，且位于潜在产出水平。

**分析**：
- LRAS：长期总供给曲线，垂直
- SRAS：短期总供给曲线，向右上方倾斜
- AD：总需求曲线，向右下方倾斜
- 交点：长期均衡点
- Y*：长期均衡产出（潜在产出）
- P*：长期均衡价格水平

**条件**：
```
SRAS = AD = LRAS
Y = Y*
```

## 4.4 总供给与总需求的移动

### 4.4.1 总供给的移动

**总供给的移动（Shift of Aggregate Supply）**：

**1. 短期总供给的移动（Shift of Short-Run Aggregate Supply）**：
- 原材料价格上升 → SRAS向左移动
- 原材料价格下降 → SRAS向右移动
- 工资上升 → SRAS向左移动
- 工资下降 → SRAS向右移动

**2. 长期总供给的移动（Shift of Long-Run Aggregate Supply）**：
- 技术进步 → LRAS向右移动
- 资本增加 → LRAS向右移动
- 劳动增加 → LRAS向右移动
- 制度改善 → LRAS向右移动

### 4.4.2 总需求的移动

**总需求的移动（Shift of Aggregate Demand）**：
- 消费增加 → AD向右移动
- 投资增加 → AD向右移动
- 政府购买增加 → AD向右移动
- 净出口增加 → AD向右移动
- 货币供给增加 → AD向右移动

## 4.5 供给冲击

### 4.5.1 供给冲击的定义

**供给冲击（Supply Shock）**：
影响总供给的事件。

### 4.5.2 供给冲击的类型

**供给冲击的类型（Types of Supply Shock）**：

**1. 负面供给冲击（Negative Supply Shock）**：
- 例如：石油价格上涨、自然灾害、战争
- 影响：SRAS向左移动 → 价格水平上升 → 产出下降

**图形表示**：

```plotly
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true

  displayModeBar: false
  responsive: true
```

**分析**：
- SRAS₁：初始短期总供给曲线
- SRAS₂：负面供给冲击后的短期总供给曲线
- AD：总需求曲线
- 均衡从E₁移动到E₂
- 价格水平从P₁上升到P₂
- 产出从Y₁下降到Y₂

**2. 正面供给冲击（Positive Supply Shock）**：
- 例如：技术进步、资源发现
- 影响：SRAS向右移动 → 价格水平下降 → 产出上升

**图形表示**：

```plotly

  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "产出Y"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- SRAS₁：初始短期总供给曲线
- SRAS₂：正面供给冲击后的短期总供给曲线
- AD：总需求曲线
- 均衡从E₁移动到E₂
- 价格水平从P₁下降到P₂
- 产出从Y₁上升到Y₂

### 4.5.3 供给冲击的影响

**供给冲击的影响（Effects of Supply Shock）**：

**1. 负面供给冲击的影响**：
- 价格水平上升 → 通货膨胀
- 产出下降 → 失业
- 滞胀：高通货膨胀和高失业并存

**2. 正面供给冲击的影响**：
- 价格水平下降 → 通货紧缩
- 产出上升 → 就业增加
- 高增长和低通胀并存

## 4.6 本章总结

### 核心要点

1. **总供给**：
   - 总供给：给定价格水平，经济中的总产出
   - 长期总供给曲线：垂直于横轴，位于潜在产出水平
   - 短期总供给曲线：向右上方倾斜，价格水平上升 → 产出增加
   - 短期总供给曲线的斜率：粘性工资、粘性价格、错觉理论

2. **总需求**：
   - 总需求：给定价格水平，经济中的总支出
   - 总需求的构成：Y = C + I + G + (X - M)
   - 总需求曲线：向右下方倾斜，价格水平上升 → 总支出减少
   - 总需求曲线的斜率：凯恩斯效应、庇古效应、蒙代尔-弗莱明效应

3. **总供给与总需求的均衡**：
   - 均衡：总供给等于总需求的状态
   - 短期均衡：SRAS = AD
   - 长期均衡：SRAS = AD = LRAS，且Y = Y*

4. **总供给与总需求的移动**：
   - 总供给的移动：原材料价格、工资、技术进步、资本、劳动、制度
   - 总需求的移动：消费、投资、政府购买、净出口、货币供给

5. **供给冲击**：
   - 供给冲击：影响总供给的事件
   - 负面供给冲击：石油价格上涨、自然灾害、战争 → SRAS向左移动 → 价格水平上升 → 产出下降
   - 正面供给冲击：技术进步、资源发现 → SRAS向右移动 → 价格水平下降 → 产出上升

### 重要概念
