# 第4章：分治策略

## 概述

分治策略（Divide and Conquer）是算法设计中最重要和最常用的方法之一。它的核心思想是将一个复杂的问题分解成多个较小、相互独立、与原问题形式相同的子问题，**递归**地解决这些子问题，然后将子问题的解组合起来得到原问题的解。

### 分治法的基本步骤

1. **分解（Divide）**：将原问题分解为若干个规模较小的子问题
2. **解决（Conquer）**：递归地求解各子问题。若子问题规模足够小，则直接求解
3. **合并（Combine）**：将子问题的解组合成原问题的解

### 典型应用

- 归并排序（Merge Sort）
- 快速排序（Quick Sort）
- 二分查找（Binary Search）
- 大整数乘法（Karatsuba算法）
- 矩阵乘法（Strassen算法）
- 最大子数组问题（Maximum Subarray Problem）

### 递归式分析

分治算法的运行时间通常由递归式（Recurrence）描述。对于形如 $T(n) = aT(n/b) + f(n)$ 的递归式，可以使用以下方法分析：

- **代入法（Substitution Method）**：猜测解的形式，然后用数学归纳法验证
- **递归树法（Recursion Tree Method）**：将递归式展开成树形结构，逐层求和
- **主定理（Master Theorem）**：提供求解特定形式递归式的公式

## 核心问题：最大子数组问题

### 问题描述

给定一个数组 `A[1..n]`，寻找连续的子数组，使得该子数组中所有元素的和最大。如果数组中所有元素都是负数，则最大子数组是空数组，和为0。

这个问题在实际应用中非常重要，例如：
- 金融：寻找股票收益最佳的连续时间段
- 图像处理：检测图像中最亮的连续区域
- 生物信息学：寻找DNA序列中特征最强的片段

### 暴力解法

最直观的方法是枚举所有可能的子数组，计算其和，然后找出最大值。

```cpp
#include <vector>
#include <climits>
#include <algorithm>

// 暴力解法：O(n³)
std::pair<int, std::pair<int, int>> max_subarray_brute_force(const std::vector<int>& A) {
    int max_sum = INT_MIN;
    int left = 0, right = 0;
    
    for (int i = 0; i < A.size(); ++i) {
        for (int j = i; j < A.size(); ++j) {
            int sum = 0;
            for (int k = i; k <= j; ++k) {
                sum += A[k];
            }
            if (sum > max_sum) {
                max_sum = sum;
                left = i;
                right = j;
            }
        }
    }
    
    return {max_sum, {left, right}};
}

// 改进的暴力解法：O(n²)
std::pair<int, std::pair<int, int>> max_subarray_brute_force_optimized(const std::vector<int>& A) {
    int max_sum = INT_MIN;
    int left = 0, right = 0;
    
    for (int i = 0; i < A.size(); ++i) {
        int sum = 0;
        for (int j = i; j < A.size(); ++j) {
            sum += A[j];
            if (sum > max_sum) {
                max_sum = sum;
                left = i;
                right = j;
            }
        }
    }
    
    return {max_sum, {left, right}};
}
```

```cpp
// ============ 数据结构定义 ============

/**
 * @brief 最大子数组的结果结构体
 */
struct SubarrayResult {
    int max_sum;      // 最大子数组的和
    int left;         // 左边界索引
    int right;        // 右边界索引
    
    SubarrayResult() : max_sum(0), left(-1), right(-1) {}
    SubarrayResult(int sum, int l, int r) : max_sum(sum), left(l), right(r) {}
    
    // 比较运算符重载
    bool operator<(const SubarrayResult& other) const {
        return max_sum < other.max_sum;
    }
    
    bool operator>(const SubarrayResult& other) const {
        return max_sum > other.max_sum;
    }
};

// ============ 分治法实现 ============

/**
 * @brief 寻找跨越中点的最大子数组
 * 
 * @param A 输入数组
 * @param low 左边界
 * @param mid 中点
 * @param high 右边界
 * @return SubarrayResult 跨越中点的最大子数组
 * 
 * @note 时间复杂度: O(n)
 * @note 空间复杂度: O(1)
 */
SubarrayResult max_crossing_subarray(const std::vector<int>& A, int low, int mid, int high) {
    // 从中点向左找最大和
    int left_sum = INT_MIN;
    int sum = 0;
    int max_left = mid;
    
    for (int i = mid; i >= low; --i) {
        sum += A[i];
        if (sum > left_sum) {
            left_sum = sum;
            max_left = i;
        }
    }
    
    // 从中点+1向右找最大和
    int right_sum = INT_MIN;
    sum = 0;
    int max_right = mid + 1;
    
    for (int j = mid + 1; j <= high; ++j) {
        sum += A[j];
        if (sum > right_sum) {
            right_sum = sum;
            max_right = j;
        }
    }
    
    return SubarrayResult(left_sum + right_sum, max_left, max_right);
}

/**
 * @brief 分治法求解最大子数组
 * 
 * @param A 输入数组
 * @param low 左边界
 * @param high 右边界
 * @return SubarrayResult 最大子数组
 * 
 * @note 时间复杂度: O(n log n)
 * @note 空间复杂度: O(log n) 递归栈空间
 */
SubarrayResult max_subarray_divide_conquer(const std::vector<int>& A, int low, int high) {
    // 基本情况：只有一个元素
    if (low == high) {
        return SubarrayResult(A[low], low, high);
    }
    
    // 分解：计算中点
    int mid = low + (high - low) / 2;
    
    // 解决：递归求解左半部分和右半部分
    SubarrayResult left_result = max_subarray_divide_conquer(A, low, mid);
    SubarrayResult right_result = max_subarray_divide_conquer(A, mid + 1, high);
    
    // 解决：求解跨越中点的最大子数组
    SubarrayResult cross_result = max_crossing_subarray(A, low, mid, high);
    
    // 合并：返回三个结果中的最大值
    if (left_result.max_sum >= right_result.max_sum && 
        left_result.max_sum >= cross_result.max_sum) {
        return left_result;
    } else if (right_result.max_sum >= left_result.max_sum && 
               right_result.max_sum >= cross_result.max_sum) {
        return right_result;
    } else {
        return cross_result;
    }
}

// ============ Kadane算法（线性时间解法） ============

/**
 * @brief Kadane算法求解最大子数组
 * 
 * @param A 输入数组
 * @return SubarrayResult 最大子数组
 * 
 * @note 时间复杂度: O(n)
 * @note 空间复杂度: O(1)
 * 
 * 算法思想：
 * - 维护当前子数组的和（current_sum）
 * - 维护到目前为止找到的最大和（max_sum）
 * - 如果当前和小于0，则从下一个元素重新开始
 */
SubarrayResult max_subarray_kadane(const std::vector<int>& A) {
    if (A.empty()) {
        return SubarrayResult();
    }
    
    int max_sum = A[0];
    int current_sum = A[0];
    int start = 0, end = 0, temp_start = 0;
    
    for (int i = 1; i < A.size(); ++i) {
        // 如果当前和加上当前元素比当前元素本身小，则重新开始
        if (current_sum + A[i] < A[i]) {
            current_sum = A[i];
            temp_start = i;
        } else {
            current_sum += A[i];
        }
        
        // 更新最大和
        if (current_sum > max_sum) {
            max_sum = current_sum;
            start = temp_start;
            end = i;
        }
    }
    
    return SubarrayResult(max_sum, start, end);
}

// ============ 辅助函数 ============

/**
 * @brief 打印子数组结果
 */
void print_subarray_result(const SubarrayResult& result, const std::vector<int>& A) {
    if (result.left == -1 || result.right == -1) {
        std::cout << "最大子数组和: " << result.max_sum << " (空数组)" << std::endl;
        return;
    }
    
    std::cout << "最大子数组和: " << result.max_sum << std::endl;
    std::cout << "索引范围: [" << result.left << ", " << result.right << "]" << std::endl;
    std::cout << "子数组内容: [";
    for (int i = result.left; i <= result.right; ++i) {
        std::cout << A[i];
        if (i < result.right) std::cout << ", ";
    }
    std::cout << "]" << std::endl;
}
```

## 算法分析

### 时间复杂度分析

#### 分治法的时间复杂度

对于最大子数组问题的分治解法，我们可以建立递归式：

$$T(n) = \begin{cases}
\Theta(1) & \text{if } n = 1 \\
2T(n/2) + \Theta(n) & \text{if } n > 1
\end{cases}$$

其中：
- $2T(n/2)$：递归求解左半部分和右半部分
- $\Theta(n)$：求解跨越中点的最大子数组

根据主定理（Master Theorem）：
- $a = 2$, $b = 2$, $f(n) = \Theta(n)$
- $n^{\log_b a} = n^{\log_2 2} = n$
- $f(n) = \Theta(n^{\log_b a})$，因此 $T(n) = \Theta(n \log n)$

#### Kadane算法的时间复杂度

Kadane算法只需要遍历数组一次，因此时间复杂度为 $O(n)$。

### 空间复杂度分析

#### 分治法的空间复杂度

- 递归深度为 $\log n$
- 每层递归使用 $O(1)$ 额外空间
- 总空间复杂度：$O(\log n)$（递归栈空间）

#### Kadane算法的空间复杂度

- 只使用常数级别的额外空间
- 空间复杂度：$O(1)$

### 正确性证明

#### 分治法的正确性证明

使用数学归纳法证明：

**归纳假设**：对于所有规模小于 $n$ 的数组，`max_subarray_divide_conquer` 函数都能正确返回最大子数组。

**归纳基础**：当 $n = 1$ 时，函数直接返回该元素，显然正确。

**归纳步骤**：假设对于规模小于 $n$ 的数组假设成立。对于规模为 $n$ 的数组：

1. 最大子数组要么完全在左半部分
2. 要么完全在右半部分
3. 要么跨越中点

根据归纳假设，左半部分和右半部分的最大子数组可以正确找到。`max_crossing_subarray` 函数可以正确找到跨越中点的最大子数组。最后，函数返回这三个结果中的最大值，因此必定是整个数组的最大子数组。

#### Kadane算法的正确性证明

Kadane算法的正确性可以通过以下性质证明：

**性质1**：在任何位置 $i$，`current_sum` 存储的是以 $i$ 结尾的最大子数组的和。

**证明**：
- 当 `current_sum + A[i] < A[i]` 时，说明包含前面元素的子数组和小于从 $i$ 开始的子数组和，因此重新开始
- 否则，扩展当前的子数组是更好的选择

**性质2**：`max_sum` 始终存储到目前为止找到的最大子数组的和。

**证明**：
- 在每一步，我们都会比较 `current_sum` 和 `max_sum`
- 如果 `current_sum` 更大，则更新 `max_sum`
- 根据性质1，`current_sum` 是以当前位置结尾的最大子数组的和，因此 `max_sum` 会遍历所有可能的结束位置

因此，算法结束时，`max_sum` 就是整个数组的最大子数组的和。

### 1. 归并排序

```cpp
/**
 * @brief 归并排序的分治实现
 */
void merge(std::vector<int>& A, int low, int mid, int high) {
    std::vector<int> L(A.begin() + low, A.begin() + mid + 1);
    std::vector<int> R(A.begin() + mid + 1, A.begin() + high + 1);
    
    int i = 0, j = 0, k = low;
    while (i < L.size() && j < R.size()) {
        if (L[i] <= R[j]) {
            A[k++] = L[i++];
        } else {
            A[k++] = R[j++];
        }
    }
    
    while (i < L.size()) A[k++] = L[i++];
    while (j < R.size()) A[k++] = R[j++];
}

void merge_sort(std::vector<int>& A, int low, int high) {
    if (low < high) {
        int mid = low + (high - low) / 2;
        merge_sort(A, low, mid);
        merge_sort(A, mid + 1, high);
        merge(A, low, mid, high);
    }
}
```

```cpp
int binary_search(const std::vector<int>& A, int target, int low, int high) {
    if (low > high) return -1;
    
    int mid = low + (high - low) / 2;
    
    if (A[mid] == target) {
        return mid;
    } else if (A[mid] > target) {

    }
}
```

```cpp
/**
 * @brief Karatsuba大整数乘法
 */
long long karatsuba(long long x, long long y) {
    if (x < 10 || y < 10) return x * y;
    
    int n = std::max(std::to_string(x).length(), std::to_string(y).length());
    int m = n / 2;
    
    long long high1 = x / static_cast<long long>(std::pow(10, m));
    long long low1 = x % static_cast<long long>(std::pow(10, m));
    long long high2 = y / static_cast<long long>(std::pow(10, m));
    long long low2 = y % static_cast<long long>(std::pow(10, m));
    
    long long z0 = karatsuba(low1, low2);
    long long z1 = karatsuba((low1 + high1), (low2 + high2));
    long long z2 = karatsuba(high1, high2);
    
    return (z2 * static_cast<long long>(std::pow(10, 2 * m))) + 
           ((z1 - z2 - z0) * static_cast<long long>(std::pow(10, m))) + z0;
}
```

## 主定理详解

主定理（Master Theorem）提供了一种求解形如 $T(n) = aT(n/b) + f(n)$ 的递归式的快速方法。

### 主定理的形式

$$T(n) = aT\left(\frac{n}{b}\right) + f(n)$$

其中：
- $a \geq 1$：子问题的数量
- $b > 1$：每个子问题的规模缩小为原问题的 $1/b$
- $f(n)$：分解和合并子问题的工作量

### 三种情况

设 $c = \log_b a$，比较 $f(n)$ 与 $n^c$ 的增长率：

1. **情况1**：$f(n) = O(n^{c-\epsilon})$，其中 $\epsilon > 0$
   - 解：$T(n) = \Theta(n^c)$

2. **情况2**：$f(n) = \Theta(n^c \log^k n)$，其中 $k \geq 0$
   - 解：$T(n) = \Theta(n^c \log^{k+1} n)$

3. **情况3**：$f(n) = \Omega(n^{c+\epsilon})$，其中 $\epsilon > 0$，且满足正则性条件
   - 解：$T(n) = \Theta(f(n))$

### 应用示例

**示例1**：归并排序
- $T(n) = 2T(n/2) + \Theta(n)$
- $a=2, b=2, c=1, f(n)=\Theta(n)$
- 属于情况2，$k=0$
- 解：$T(n) = \Theta(n \log n)$

**示例2**：二分查找
- $T(n) = T(n/2) + \Theta(1)$
- $a=1, b=2, c=0, f(n)=\Theta(1)$
- 属于情况2，$k=0$
- 解：$T(n) = \Theta(\log n)$

**示例3**：Strassen矩阵乘法
- $T(n) = 7T(n/2) + \Theta(n^2)$
- $a=7, b=2, c=\log_2 7 \approx 2.81, f(n)=\Theta(n^2)$
- 属于情况1，$\epsilon = 0.81$
- 解：$T(n) = \Theta(n^{2.81})$
