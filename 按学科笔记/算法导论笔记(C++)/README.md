## 代码风格

### 现代C++ (C++17/20) 特性

1. **智能指针**
   - 使用 `std::unique_ptr` 管理独占所有权
   - 使用 `std::shared_ptr` 管理共享所有权
   - 使用 `std::weak_ptr` 避免循环引用

2. **类型推导**
   - 使用 `auto` 进行类型推导
   - 使用 `decltype` 推导表达式类型

3. **范围和视图**
   - 使用 `std::span` (C++20) 避免不必要的拷贝
   - 使用 `std::ranges` 进行函数式编程

4. **概念和约束** (C++20)
   - 使用 `concepts` 约束模板参数
   - 使用 `requires` 子句

5. **其他特性**
   - 结构化绑定（C++17）
   - `std::optional` 表示可能为空的值
   - `std::variant` 和 `std::any` 处理多态
   - `if constexpr` 编译时条件判断

### STL使用原则

- **优先使用STL容器和算法**
- **展示STL底层实现原理**（在STL专题章节中）
- **关键算法手动实现**（展示核心逻辑）
- **辅助功能使用STL**（提高实用性）

### 代码格式

每个章节包含：

1. **完整的算法实现**
   - 详细的注释
   - 时间/空间复杂度分析
   - 正确性说明

2. **测试用例**
   - 基本功能测试
   - 边界情况测试
   - 性能测试

3. **编译和运行说明**
   - 编译命令
   - 预期输出

## 编译环境

### 要求

- C++17 或 C++20 编译器
  - GCC 9.0+
  - Clang 10.0+
  - MSVC 19.28+

### 编译命令

```bash
# 使用C++20编译
g++ -std=c++20 -O2 -Wall -Wextra your_file.cpp -o your_program

# 使用Clang编译
clang++ -std=c++20 -O2 -Wall -Wextra your_file.cpp -o your_program

# 使用MSVC编译（Windows）
cl /std:c++20 /O2 /W4 your_file.cpp
```
