- 理解并行计算与现代计算机架构

## 📊 关键术语索引

### A-C
- **ABI (Application Binary Interface)**：应用二进制接口
- **ALU (Arithmetic Logic Unit)**：算术逻辑单元
- **CPU (Central Processing Unit)**：中央处理器
- **Cache**：缓存
- **Compiler**：编译器

### D-G
- **Datapath**：数据通路
- **DRAM (Dynamic Random Access Memory)**：动态随机存取存储器
- **GPU (Graphics Processing Unit)**：图形处理器
- **ISA (Instruction Set Architecture)**：指令集架构

### I-M
- **Instruction**：指令
- **Memory Hierarchy**：存储层次结构
- **MIPS**：百万指令每秒
- **Multicore**：多核

### P-S
- **Pipeline**：流水线
- **PMD (Personal Mobile Device)**：个人移动设备
- **Processor**：处理器
- **SRAM (Static Random Access Memory)**：静态随机存取存储器
- **Supercomputer**：超级计算机

### T-Z
- **Throughput**：吞吐量
- **Virtual Memory**：虚拟内存
- **WSC (Warehouse Scale Computer)**：仓库级计算机

## 💡 学习提示

### 理论与实践结合
- 每学习一章后，尝试编写RISC-V汇编程序
- 使用RISC-V模拟器（如RISC-V Spike）验证代码
- 绘制处理器数据通路图
- 分析实际处理器的架构文档

### 深入理解关键概念
- 每个关键概念都要理解其"为什么"而不仅仅是"是什么"
- 思考设计权衡（性能 vs 功耗 vs 成本）
- 联系实际应用场景

### 建立知识体系
- 理解各章节之间的联系
- 绘制知识思维导图
- 定期回顾与总结
