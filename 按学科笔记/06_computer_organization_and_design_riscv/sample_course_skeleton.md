# 示例：课程骨架（使用模板）

本文件演示如何基于 `记录` 中的模板快速创建课程骨架。

1. 在课程根目录添加 `00_Index.md`（可复制自 `记录/course_index_template.md`）。
2. 创建子目录：`01_LectureNotes/`、`02_Problems/`、`03_Labs/`、`05_Refs/`。
3. 使用 `记录/lecture_note_template.md` 编写 `01_LectureNotes/01_第1章.md`。
4. 使用 `记录/problem_template.md` 编写练习：`02_Problems/01_第1章习题.md`。
5. 使用 `记录/lab_template.md` 编写实验：`03_Labs/01_第1章实验.md`。

示例索引（放入 `00_Index.md`）：

- 第1章：计算机抽象与技术 — [01_LectureNotes/01_第1章.md](01_LectureNotes/01_第1章.md)
- 第1章习题 — [02_Problems/01_第1章习题.md](02_Problems/01_第1章习题.md)
- 第1章实验 — [03_Labs/01_第1章实验.md](03_Labs/01_第1章实验.md)

请按 `CONTRIBUTING.md` 的规范命名并使用模板填充内容。
