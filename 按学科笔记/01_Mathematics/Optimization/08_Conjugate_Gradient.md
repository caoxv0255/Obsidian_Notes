## 1. 共轭方向

### 1.1 共轭的定义

**定义**：设 Q 为 n×n 对称正定矩阵，若向量 d₁, d₂, ..., dₖ 满足：
$$d_i^T Q d_j = 0, \quad \forall i \neq j$$

则称这组向量关于 Q **共轭**（或 Q-正交）。

**性质**：
- 共轭向量组必线性无关
- 最多有 n 个相互共轭的向量

### 1.2 几何意义

对于二次函数 $f(x) = \frac{1}{2}x^T Q x - b^T x$：
- 等高线是椭球
- 共轭方向是椭球主轴方向
- 沿共轭方向搜索可避免"锯齿"现象

## 2. 共轭梯度法原理

### 2.1 二次函数情形

**目标函数**：
$$f(x) = \frac{1}{2}x^T Q x - b^T x + c$$

其中 Q 对称正定，唯一极小点 x* = Q⁻¹b。

**关键定理**：若 d₀, d₁, ..., d_{k-1} 关于 Q 共轭，从任意 x₀ 出发，依次沿这些方向精确搜索，最多 n 步达到最优解。

### 2.2 共轭方向的构造

**Fletcher-Reeves 公式**：
$$d_k = -g_k + \beta_k d_{k-1}$$
$$\beta_k^{FR} = \frac{g_k^T g_k}{g_{k-1}^T g_{k-1}}$$

**Polak-Ribière 公式**：
$$\beta_k^{PR} = \frac{g_k^T (g_k - g_{k-1})}{g_{k-1}^T g_{k-1}}$$

**Hestenes-Stiefel 公式**：
$$\beta_k^{HS} = \frac{g_k^T (g_k - g_{k-1})}{d_{k-1}^T (g_k - g_{k-1})}$$

## 3. 共轭梯度算法

### 3.1 线性共轭梯度法（求解 Qx = b）

```
算法：共轭梯度法（线性方程组）
初始化：x₀, r₀ = b - Qx₀, d₀ = r₀
for k = 0, 1, 2, ..., n-1:
    if ‖r_k‖ < ε:
        break
    α_k = (r_k^T r_k) / (d_k^T Q d_k)
    x_{k+1} = x_k + α_k d_k
    r_{k+1} = r_k - α_k Q d_k
    β_{k+1} = (r_{k+1}^T r_{k+1}) / (r_k^T r_k)
    d_{k+1} = r_{k+1} + β_{k+1} d_k
返回 x_{k+1}
```

**性质**：
- 理论上最多 n 步收敛
- 无需存储矩阵，只需矩阵-向量乘法
- 适用于大型稀疏系统

### 3.2 非线性共轭梯度法

```
算法：非线性共轭梯度法（FR方法）
初始化：x₀, g₀ = ∇f(x₀), d₀ = -g₀
for k = 0, 1, 2, ...:
    线搜索确定步长 α_k
    x_{k+1} = x_k + α_k d_k
    g_{k+1} = ∇f(x_{k+1})
    if ‖g_{k+1}‖ < ε:
        break
    β_{k+1} = (g_{k+1}^T g_{k+1}) / (g_k^T g_k)  # FR公式
    d_{k+1} = -g_{k+1} + β_{k+1} d_k
返回 x_{k+1}
```

**重启策略**：每 n 步重置 d_k = -g_k，避免累积误差。

## 4. 收敛性分析

### 4.1 收敛速度

**二次函数情形**：
$$\|x_k - x^*\|_Q \leq 2 \left(\frac{\sqrt{\kappa} - 1}{\sqrt{\kappa} + 1}\right)^k \|x_0 - x^*\|_Q$$

其中 κ = λ_max/λ_min 为条件数，$\|x\|_Q = \sqrt{x^T Q x}$。

**对比**：
| 方法 | 收敛速度 |
|------|----------|
| 梯度下降 | O((κ-1)/(κ+1))^k |
| 共轭梯度 | O((√κ-1)/(√κ+1))^k |

条件数 κ 大时，共轭梯度法显著优于梯度下降。

### 4.2 迭代次数估计

达到精度 ε 所需迭代次数约为：
$$k \approx \frac{1}{2}\sqrt{\kappa} \ln\frac{2}{\epsilon}$$

## 5. 预处理共轭梯度法

### 5.1 预处理思想

**问题**：条件数大时收敛慢

**方法**：选择预处理矩阵 M，使 M⁻¹Q 的条件数更小

**预处理系统**：
$$M^{-1} Q x = M^{-1} b$$

### 5.2 常用预处理子

1. **Jacobi 预处理**：M = diag(Q)
2. **不完全 Cholesky 分解**：M ≈ LL^T
3. **SSOR 预处理**：M = (D + ωL)D⁻¹(D + ωU)

### 5.3 预处理共轭梯度算法

```
算法：PCG
初始化：x₀, r₀ = b - Qx₀, z₀ = M⁻¹r₀, d₀ = z₀
for k = 0, 1, 2, ...:
    α_k = (r_k^T z_k) / (d_k^T Q d_k)
    x_{k+1} = x_k + α_k d_k
    r_{k+1} = r_k - α_k Q d_k
    if ‖r_{k+1}‖ < ε:
        break
    z_{k+1} = M⁻¹r_{k+1}
    β_{k+1} = (r_{k+1}^T z_{k+1}) / (r_k^T z_k)
    d_{k+1} = z_{k+1} + β_{k+1} d_k
返回 x_{k+1}
```

## 6. 与其他方法的关系

### 6.1 与Krylov子空间方法

共轭梯度法是Krylov子空间方法的一种：
$$\mathcal{K}_k(Q, r_0) = \text{span}\{r_0, Qr_0, Q^2r_0, \ldots, Q^{k-1}r_0\}$$

x_k 是在 K_k(Q, r₀) 上使 ‖x - x*‖_Q 最小的点。

### 6.2 与Lanczos方法的关系

- CG产生三对角矩阵
- Lanczos过程产生正交矩阵
- 二者等价，只是表达方式不同

## 7. 应用实例

### 7.1 求解大型稀疏线性方程组

```python
import numpy as np
from scipy.sparse import diags
from scipy.sparse.linalg import cg

# 构造大型稀疏正定矩阵
n = 1000
diagonals = [2*np.ones(n), -np.ones(n-1), -np.ones(n-1)]
A = diags(diagonals, [0, -1, 1], format='csr')
b = np.ones(n)

# 共轭梯度法求解
x, info = cg(A, b, tol=1e-10)
print(f"收敛: {info == 0}")
print(f"残差: {np.linalg.norm(A @ x - b)}")
```

### 7.2 机器学习中的应用

**逻辑回归**：Hessian矩阵稀疏时可用CG求解牛顿方程

**深度学习**：
- 自然梯度下降中的Fisher矩阵求逆
- 大规模线性系统求解

## 8. 总结

| 特点 | 说明 |
|------|------|
| 存储需求 | O(n)，只需存储几个向量 |
| 每步计算量 | 一次梯度计算，两次向量运算 |
| 收敛速度 | 超线性收敛，介于梯度法和牛顿法之间 |
| 适用问题 | 大型稀疏正定系统、一般非线性优化 |
| 缺点 | 对线搜索精度要求高 |

**与其他方法对比**：
- 比梯度下降快，尤其条件数大时
- 比牛顿法内存需求低
- 不需要存储Hessian矩阵
