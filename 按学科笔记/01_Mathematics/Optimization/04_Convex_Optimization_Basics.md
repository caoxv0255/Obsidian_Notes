## 1. 凸优化问题的标准形式

### 1.1 标准形式

**凸优化问题**：
$$\min_x f_0(x)$$
$$\text{s.t.} \quad f_i(x) \leq 0, \quad i = 1, \ldots, m$$
$$\quad\quad\; h_j(x) = 0, \quad j = 1, \ldots, p$$

其中：
- f₀(x)：凸函数（目标函数）
- fᵢ(x)：凸函数（不等式约束）
- hⱼ(x)：仿射函数（等式约束）

### 1.2 凸优化的关键性质

**定理**：凸优化问题具有以下重要性质：

1. **局部最优 = 全局最优**：
   若 x* 是局部最优解，则 x* 是全局最优解。

2. **可行域是凸集**：
   由凸约束定义的区域是凸集。

3. **最优值集是凸集**：
   若最优解不唯一，所有最优解构成凸集。

### 1.3 判定凸优化

**判定步骤**：
1. 目标函数是否为凸函数？
2. 不等式约束是否为凸函数？
3. 等式约束是否为仿射函数？

**例**：
$$\min_x \frac{1}{2}\|Ax-b\|^2$$
$$\text{s.t.} \quad \|x\|_1 \leq 1$$

**判定**：
- 目标函数：二次函数，凸
- 约束：l₁ 范数，凸
- 结论：凸优化问题

## 2. 凸优化问题的分类

### 2.1 线性规划（LP）

$$\min_x c^T x$$
$$\text{s.t.} \quad Ax \preceq b$$

**特点**：
- 目标函数：线性（凸）
- 约束：线性（凸）
- 可行域：多面体

### 2.2 二次规划（QP）

$$\min_x \frac{1}{2} x^T P x + q^T x$$
$$\text{s.t.} \quad Ax \preceq b$$

**凸性条件**：P ≽ 0（半正定）

**例**：投资组合优化、支持向量机

### 2.3 二次约束二次规划（QCQP）

$$\min_x \frac{1}{2} x^T P_0 x + q_0^T x$$
$$\text{s.t.} \quad \frac{1}{2} x^T P_i x + q_i^T x + r_i \leq 0$$

**凸性条件**：P₀, Pᵢ ≽ 0

### 2.4 二阶锥规划（SOCP）

$$\min_x c^T x$$
$$\text{s.t.} \quad \|A_i x + b_i\| \leq c_i^T x + d_i$$

**特点**：
- 包含线性规划、二次规划
- 可有效求解（内点法）

### 2.5 半定规划（SDP）

$$\min_x c^T x$$
$$\text{s.t.} \quad \sum_{i=1}^n x_i F_i + G \preceq 0$$

**特点**：
- 约束涉及矩阵不等式
- 包含线性规划、二阶锥规划

## 3. 凸优化的对偶理论

### 3.1 Lagrange函数

**原问题**：
$$p^* = \min_x f_0(x) \quad \text{s.t.} \quad f_i(x) \leq 0, \quad h_j(x) = 0$$

**Lagrange函数**：
$$L(x, \lambda, \nu) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{j=1}^p \nu_j h_j(x)$$

**对偶函数**：
$$g(\lambda, \nu) = \inf_x L(x, \lambda, \nu)$$

### 3.2 对偶问题

$$\max_{\lambda \geq 0, \nu} g(\lambda, \nu)$$

### 3.3 强对偶性

**定理**（Slater条件）：若凸优化问题存在严格可行点：
$$\exists x_0: f_i(x_0) < 0, \quad h_j(x_0) = 0$$

则强对偶性成立：
$$p^* = d^*$$

**意义**：可通过求解对偶问题得到原问题的解。

## 4. 最优性条件

### 4.1 无约束情形

**定理**：x* 是无约束凸优化问题的最优解 ⟺ ∇f₀(x*) = 0

### 4.2 约束情形（KKT条件）

**定理**：若 x* 是最优解，且 Slater 条件成立，则存在 λ*, ν* 满足：

1. **站点条件**：
   $$\nabla f_0(x^*) + \sum_i \lambda_i^* \nabla f_i(x^*) + \sum_j \nu_j^* \nabla h_j(x^*) = 0$$

2. **原始可行性**：
   $$f_i(x^*) \leq 0, \quad h_j(x^*) = 0$$

3. **对偶可行性**：
   $$\lambda_i^* \geq 0$$

4. **互补松弛**：
   $$\lambda_i^* f_i(x^*) = 0$$

**充分性**：若凸优化问题的 KKT 点存在，则它是全局最优解。

## 5. 几何解释

### 5.1 可行域

**凸优化问题的可行域**：
$$\mathcal{F} = \{x : f_i(x) \leq 0, h_j(x) = 0\}$$

是凸集（由凸函数定义）。

### 5.2 水平集

**目标函数的水平集**：
$$\mathcal{L}_\alpha = \{x : f_0(x) \leq \alpha\}$$

是凸集（f₀ 是凸函数）。

### 5.3 最优解的几何

**最优解**：目标函数等值面与可行域边界的"切点"。

**梯度方向**：最优解处，梯度指向可行域外部。

## 6. 实例分析

### 6.1 最小二乘问题

**问题**：
$$\min_x \|Ax - b\|^2$$

**解**：正规方程
$$A^T A x = A^T b$$
$$x^* = (A^T A)^{-1} A^T b$$

**约束情形**：
$$\min_x \|Ax - b\|^2 \quad \text{s.t.} \quad Cx = d$$

用 Lagrange 乘子法求解。

### 6.2 线性规划

**问题**：
$$\min_x c^T x \quad \text{s.t.} \quad Ax = b, \quad x \geq 0$$

**对偶问题**：
$$\max_y b^T y \quad \text{s.t.} \quad A^T y \leq c$$

### 6.3 支持向量机

**问题**：
$$\min_{w,b} \frac{1}{2}\|w\|^2 + C \sum_i \xi_i$$
$$\text{s.t.} \quad y_i(w^T x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

**凸性**：
- 目标函数：凸（二次 + 线性）
- 约束：线性（凸）
- 结论：凸优化问题

### 6.4 LASSO

**问题**：
$$\min_x \frac{1}{2}\|Ax - b\|^2 + \lambda \|x\|_1$$

**凸性**：
- 目标函数：凸（二次 + l₁范数）
- 无约束
- 结论：凸优化问题

## 7. 数值方法简介

### 7.1 无约束优化

- **梯度下降法**：通用、简单
- **牛顿法**：收敛快、需Hessian
- **拟牛顿法**：折中方案

### 7.2 约束优化

- **罚函数法**：转化为无约束
- **障碍函数法**：保持严格可行
- **增广拉格朗日法**：改进罚函数
- **内点法**：多项式时间

### 7.3 特殊结构

- **线性规划**：单纯形法、内点法
- **二次规划**：有效集法、内点法
- **二阶锥规划**：内点法

## 8. 小结

### 8.1 核心概念

| 概念 | 定义 | 重要性 |
|------|------|--------|
| 凸优化 | f₀, fᵢ 凸，hⱼ 仿射 | 局部最优=全局最优 |
| KKT条件 | 最优性必要条件 | 理论基础、算法设计 |
| 强对偶性 | p* = d* | 对偶方法可用 |

### 8.2 问题分类

```
凸优化
├── 线性规划（LP）
│   └── 目标和约束都是线性
├── 二次规划（QP）
│   └── 目标二次，约束线性
├── 二阶锥规划（SOCP）
│   └── 包含锥约束
└── 半定规划（SDP）
    └── 矩阵不等式约束
```

### 8.3 关键定理

1. **局部最优 ⟹ 全局最优**：凸优化的核心优势
2. **KKT充分性**：凸问题的KKT点即最优解
3. **强对偶**：Sliter条件下成立
