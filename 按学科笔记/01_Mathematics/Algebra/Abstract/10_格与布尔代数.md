## 一、格的定义

### 1.1 偏序集

**定义**：**偏序集**是一个集合 $P$ 配合二元关系 $\leq$，满足：
1. 自反性：$a \leq a$
2. 反对称性：$a \leq b, b \leq a \Rightarrow a = b$
3. 传递性：$a \leq b, b \leq c \Rightarrow a \leq c$

### 1.2 格的定义（偏序集视角）

**定义**：偏序集 $(L, \leq)$ 称为**格**，若任意两个元素都有上确界和下确界。

- **上确界（并）**：$a \vee b = \sup\{a, b\}$
- **下确界（交）**：$a \wedge b = \inf\{a, b\}$

### 1.3 格的定义（代数视角）

**定义**：**格**是一个集合 $L$ 配合两个运算 $\vee, \wedge$，满足：

1. **交换律**：$a \vee b = b \vee a$，$a \wedge b = b \wedge a$
2. **结合律**：$(a \vee b) \vee c = a \vee (b \vee c)$，$(a \wedge b) \wedge c = a \wedge (b \wedge c)$
3. **吸收律**：$a \vee (a \wedge b) = a$，$a \wedge (a \vee b) = a$

### 1.4 等价性

**定理**：两种定义等价。

给定偏序格，定义 $a \vee b = \sup\{a, b\}$，$a \wedge b = \inf\{a, b\}$，满足代数公理。

给定代数格，定义 $a \leq b \Leftrightarrow a \vee b = b$，满足偏序公理。

## 二、格的例子

### 2.1 基本例子

**例1**：$(\mathcal{P}(X), \subseteq)$
- $A \vee B = A \cup B$
- $A \wedge B = A \cap B$

**例2**：$(\mathbb{N}, \mid)$（整除关系）
- $a \vee b = \text{lcm}(a, b)$
- $a \wedge b = \gcd(a, b)$

**例3**：子群格 $(\text{Sub}(G), \subseteq)$
- $H \vee K = \langle H \cup K \rangle$
- $H \wedge K = H \cap K$

**例4**：理想格 $(\text{Id}(R), \subseteq)$

### 2.2 Hasse图

用图表示有限偏序集：
- 元素用点表示
- $a < b$ 且无 $c$ 使 $a < c < b$ 时，画 $a$ 到 $b$ 的线段

**例**：$\{1, 2, 3, 6\}$ 在整除下的Hasse图：
```
    6
   / \
  2   3
   \ /
    1
```

## 三、分配格

### 3.1 定义

**定义**：格 $L$ 称为**分配格**，若满足分配律：
$$a \wedge (b \vee c) = (a \wedge b) \vee (a \wedge c)$$
$$a \vee (b \wedge c) = (a \vee b) \wedge (a \vee c)$$

### 3.2 性质

**定理**：两条分配律等价。

**定理**：分配格中不含 $M_3$ 和 $N_5$ 作为子格。

```
M_3:      N_5:
  1         1
 /|\        |
a b c       a
 \|/       / \
  0       b   c
           \ /
            0
```

### 3.3 例子

**分配格**：
- $(\mathcal{P}(X), \subseteq)$
- $(\mathbb{N}, \mid)$
- 全序集

**非分配格**：
- 子群格（一般不是分配格）
- $M_3$（"钻石格"）
- $N_5$（"五边形格"）

## 四、有界格与有补格

### 4.1 有界格

**定义**：格 $L$ 称为**有界的**，若存在：
- **最大元** $1$：$a \leq 1$ 对所有 $a$
- **最小元** $0$：$0 \leq a$ 对所有 $a$

**等价条件**：$a \vee 1 = 1$，$a \wedge 0 = 0$ 对所有 $a$。

### 4.2 补元

**定义**：在有界格中，元素 $a$ 的**补**是元素 $b$ 使得
$$a \vee b = 1, \quad a \wedge b = 0$$

**注意**：补可能不唯一。

### 4.3 有补格

**定义**：有界格 $L$ 称为**有补的**，若每个元素都有补。

**例**：$(\mathcal{P}(X), \subseteq)$ 是有补格，$A$ 的补是 $A^c$。

---

## 五、布尔代数

### 5.1 定义

**定义**：**布尔代数**是有补分配格。

**等价定义**：布尔代数是集合 $B$ 配合运算 $\vee, \wedge, '$ 和元素 $0, 1$，满足：
1. 格公理
2. 分配律
3. 有界性
4. 补元：$a \vee a' = 1$，$a \wedge a' = 0$

### 5.2 基本性质

**定理**：在布尔代数中，
1. 补唯一：若 $a \vee b = 1$ 且 $a \wedge b = 0$，则 $b = a'$
2. 对合律：$(a')' = a$
3. De Morgan律：$(a \vee b)' = a' \wedge b'$，$(a \wedge b)' = a' \vee b'$

### 5.3 例子

**例1**：$(\mathcal{P}(X), \cup, \cap, ^c, \emptyset, X)$

**例2**：$(\{0, 1\}, \vee, \wedge, ', 0, 1)$，其中 $0' = 1$，$1' = 0$

**例3**：命题逻辑真值表构成布尔代数

---

## 六、Stone表示定理

### 6.1 定理

**定理（Stone表示定理）**：每个布尔代数同构于某个集合的幂集布尔代数的子代数。

即：布尔代数 $B \cong (\mathcal{P}(X)$ 的子代数$)$，对某集合 $X$。

### 6.2 原子

**定义**：布尔代数中的**原子**是覆盖 $0$ 的元素（$a > 0$ 且无 $x$ 使 $0 < x < a$）。

**定理**：有限布尔代数同构于某有限集的幂集布尔代数。

### 6.3 Stone空间

**定理**：布尔代数与Stone空间（紧致全不连通Hausdorff空间）对偶。

---

## 七、布尔环

### 7.1 定义

**定义**：**布尔环**是满足 $a^2 = a$ 对所有 $a$ 的环。

### 7.2 与布尔代数的关系

**定理**：布尔代数与布尔环等价。

**对应关系**：
- 布尔代数 $\to$ 布尔环：
  - 加法：$a + b = (a \wedge b') \vee (a' \wedge b)$（对称差）
  - 乘法：$a \cdot b = a \wedge b$
- 布尔环 $\to$ 布尔代数：
  - 并：$a \vee b = a + b + ab$
  - 交：$a \wedge b = ab$
  - 补：$a' = 1 + a$

---

## 八、应用

### 8.1 逻辑电路设计

布尔代数是数字电路设计的数学基础：
- 逻辑门：AND, OR, NOT
- 电路化简：利用布尔代数定律

### 8.2 开关电路

开关的串联和并联对应布尔代数的运算：
- 串联 $\leftrightarrow \wedge$
- 并联 $\leftrightarrow \vee$

### 8.3 数据库理论

关系代数基于布尔代数：
- 选择、投影、连接

### 8.4 程序验证

命题逻辑的可满足性问题是布尔代数的核心问题。

## 重要定理汇总

| 定理 | 内容 |
|------|------|
| 格的定义等价性 | 偏序视角 $\leftrightarrow$ 代数视角 |
| 分配格判定 | 不含 $M_3, N_5$ |
| 补的唯一性 | 分配格中补唯一 |
| Stone表示 | 布尔代数 $\cong$ 幂集子代数 |

## 总结

- 格论提供了描述“最小上界/最大下界”的统一语言。
- 布尔代数是有补分配格，在逻辑、集合和电路中都扮演核心角色。
- 布尔环和 Stone 表示把代数、拓扑和逻辑联系起来。

## 易错点

- 混淆偏序和全序。
- 以为补元一定唯一，实际上在一般有界格中未必如此。
- 将分配格与布尔代数等同。
- 忽略布尔环和布尔代数之间的加法/乘法对应。
