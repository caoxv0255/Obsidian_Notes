## 一、整环

### 1.1 定义

**定义**：**整环**是含幺、交换、无零因子的环。

**等价条件**：$ab = 0 \Rightarrow a = 0$ 或 $b = 0$

### 1.2 例子

- $\mathbb{Z}$：整数环
- $\mathbb{Z}[i]$：Gauss整数环
- $F[x]$：域上的多项式环
- $\mathbb{Z}[\sqrt{2}]$：$\{a + b\sqrt{2} : a, b \in \mathbb{Z}\}$

### 1.3 消去律

**定理**：在整环中，$ab = ac$ 且 $a \neq 0 \Rightarrow b = c$。

**证明**：$ab = ac \Rightarrow a(b - c) = 0$。由无零因子，$a \neq 0 \Rightarrow b - c = 0$。

---

## 二、域

### 2.1 定义

**定义**：**域**是每个非零元素都有乘法逆元的交换环。

**等价条件**：域的非零元构成乘法群。

### 2.2 例子

- $\mathbb{Q}, \mathbb{R}, \mathbb{C}$：常见的数域
- $\mathbb{Z}_p$：模素数 $p$ 的剩余类域
- $\mathbb{Q}(\sqrt{2})$：$\{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$
- $\mathbb{F}_{p^n}$：有限域

### 2.3 域的性质

**定理**：域没有非平凡理想。

**证明**：设 $I$ 是域 $F$ 的理想，$a \in I$，$a \neq 0$。则 $1 = a \cdot a^{-1} \in I$，故 $I = F$。

**推论**：域到环的同态要么是零同态，要么是单射。

### 2.4 域扩张

**定义**：设 $F$ 是域，$K \supseteq F$ 也是域，则称 $K$ 是 $F$ 的**扩域**。

**例**：$\mathbb{C}$ 是 $\mathbb{R}$ 的扩域，$\mathbb{R}$ 是 $\mathbb{Q}$ 的扩域。

---

## 三、多项式环

### 3.1 定义

**定义**：设 $R$ 是环，**多项式环** $R[x]$ 是所有形式多项式
$$f(x) = a_n x^n + \cdots + a_1 x + a_0$$
构成的环，其中 $a_i \in R$。

### 3.2 多项式环的性质

**定理**：若 $R$ 是整环，则 $R[x]$ 也是整环。

**证明**：设 $f, g \neq 0$，首项分别为 $a_n x^n$, $b_m x^m$。则 $fg$ 的首项为 $a_n b_m x^{n+m} \neq 0$（因为 $R$ 无零因子）。

**定理**：$\deg(fg) = \deg f + \deg g$

### 3.3 多元多项式环

$$R[x_1, \ldots, x_n] = R[x_1][x_2]\cdots[x_n]$$

**例**：$\mathbb{R}[x, y]$ 是二元实系数多项式环。

---

## 四、唯一分解整环（UFD）

### 4.1 定义

**定义**：整环 $R$ 称为**唯一分解整环**（UFD），若：
1. 每个非零非单位元可分解为不可约元素的乘积
2. 分解在相伴意义下唯一（$a = ub$，$u$ 是单位）

### 4.2 例子

- $\mathbb{Z}$ 是 UFD（算术基本定理）
- $F[x]$ 是 UFD（多项式因式分解）
- $\mathbb{Z}[x]$ 是 UFD（Gauss引理）

### 4.3 反例

**例**：$\mathbb{Z}[\sqrt{-5}]$ 不是 UFD。

$6 = 2 \cdot 3 = (1 + \sqrt{-5})(1 - \sqrt{-5})$

这是两种本质上不同的分解。

---

## 五、主理想整环（PID）

### 5.1 定义

**定义**：整环 $R$ 称为**主理想整环**（PID），若每个理想都是主理想。

### 5.2 例子

- $\mathbb{Z}$ 是 PID
- $F[x]$ 是 PID
- $\mathbb{Z}[x]$ 不是 PID（理想 $\langle 2, x \rangle$ 不是主理想）

### 5.3 PID的性质

**定理**：PID 是 UFD。

**定理**：在 PID 中，不可约元素生成极大理想。

**证明**：设 $p$ 不可约，$\langle p \rangle \subseteq I = \langle a \rangle$。则 $p = ab$，由 $p$ 不可约，$a$ 或 $b$ 是单位。若 $a$ 是单位，$I = R$；若 $b$ 是单位，$\langle p \rangle = I$。

---

## 六、Euclid整环（ED）

### 6.1 定义

**定义**：整环 $R$ 称为**Euclid整环**（ED），若存在函数 $d: R \setminus \{0\} \to \mathbb{N}$ 使得：
1. $a \mid b \Rightarrow d(a) \leq d(b)$
2. 对任意 $a, b \neq 0$，存在 $q, r$ 使 $a = bq + r$，其中 $r = 0$ 或 $d(r) < d(b)$

### 6.2 例子

- $\mathbb{Z}$ 是 ED（$d(n) = |n|$）
- $F[x]$ 是 ED（$d(f) = \deg f$）
- $\mathbb{Z}[i]$ 是 ED（$d(a + bi) = a^2 + b^2$）

### 6.3 包含关系

$$\text{ED} \subseteq \text{PID} \subseteq \text{UFD} \subseteq \text{整环}$$

**严格包含**：
- $\mathbb{Z}[x]$ 是 UFD 但不是 PID
- $\mathbb{Z}[\sqrt{-19}]$ 是 PID 但不是 ED

---

## 七、Gauss引理

### 7.1 本原多项式

**定义**：多项式 $f \in \mathbb{Z}[x]$ 称为**本原的**，若其系数的最大公约数为 1。

### 7.2 Gauss引理

**定理（Gauss引理）**：本原多项式的乘积是本原的。

### 7.3 应用

**定理**：$f \in \mathbb{Z}[x]$ 在 $\mathbb{Q}$ 上可约 $\Leftrightarrow$ 在 $\mathbb{Z}$ 上可约。

**推论**：$\mathbb{Z}[x]$ 是 UFD。

---

## 八、Eisenstein判别法

### 8.1 定理

**定理（Eisenstein判别法）**：设 $f(x) = a_n x^n + \cdots + a_0 \in \mathbb{Z}[x]$。若存在素数 $p$ 使得：
1. $p \nmid a_n$
2. $p \mid a_i$，$i = 0, 1, \ldots, n-1$
3. $p^2 \nmid a_0$

则 $f$ 在 $\mathbb{Q}$ 上不可约。

### 8.2 应用

**例1**：$x^n - 2$ 在 $\mathbb{Q}$ 上不可约（取 $p = 2$）。

**例2**：$x^4 + 2x^3 + 4x^2 + 6x + 2$ 在 $\mathbb{Q}$ 上不可约（取 $p = 2$）。

### 8.3 分圆多项式

**例**：分圆多项式 $\Phi_p(x) = x^{p-1} + x^{p-2} + \cdots + x + 1$ 在 $\mathbb{Q}$ 上不可约。

**证明**：$\Phi_p(x+1) = \frac{(x+1)^p - 1}{x} = x^{p-1} + \binom{p}{1}x^{p-2} + \cdots + \binom{p}{p-1}$

用 Eisenstein 判别法（$p \mid \binom{p}{k}$，$1 \leq k \leq p-1$）。

---

## 重要定理汇总

| 定理 | 内容 |
|------|------|
| 整环消去律 | $ab = ac, a \neq 0 \Rightarrow b = c$ |
| PID $\Rightarrow$ UFD | 主理想整环可唯一分解 |
| Gauss引理 | 本原多项式乘积本原 |
| Eisenstein | 判定多项式不可约 |

---

## 总结

- 整环是研究消去律、因式分解与不可约性的基础。
- PID、UFD、Euclid 整环构成了强度递增的分解性层次。
- Gauss 引理与 Eisenstein 判别法是多项式不可约性的核心工具。

## 易错点

- 把整环与域混淆，忘记域要求每个非零元可逆。
- 误把不可约和素元完全等同，忽略它们在一般整环中的差别。
- 混淆 PID、UFD、ED 的包含关系。
- 使用 Eisenstein 判别法时忘记检查所有条件。
