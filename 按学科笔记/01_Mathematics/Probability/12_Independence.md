## 1. 独立性的定义

### 1.1 两个随机变量的独立性

**定义**：随机变量 X 和 Y 相互独立，当且仅当对所有 x, y：
$$F(x, y) = F_X(x) \cdot F_Y(y)$$

其中 F(x, y) 为联合分布函数，F_X, F_Y 为边缘分布函数。

### 1.2 等价条件

**离散型**：X, Y 独立 ⟺ 对所有 i, j：
$$P(X = x_i, Y = y_j) = P(X = x_i) \cdot P(Y = y_j)$$
$$p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$$

**连续型**：X, Y 独立 ⟺ 对所有 x, y：
$$f(x, y) = f_X(x) \cdot f_Y(y)$$

**等价表述**：条件分布等于边缘分布：
$$f_{X|Y}(x|y) = f_X(x)$$

### 1.3 直观理解

**独立性意味着**：
- 知道 Y 的取值不影响 X 的分布
- 联合分布可由边缘分布唯一确定
- 两个随机变量"各管各的"

## 2. 独立性的判定

### 2.1 充要条件

**定理**：X, Y 独立 ⟺ 存在函数 g(x) 和 h(y) 使得：
$$f(x, y) = g(x) \cdot h(y)$$

（连续型）或类似离散型表述。

**注意**：支撑集（取非零值的区域）必须是矩形！

### 2.2 非独立的判定

**非独立的情况**：
1. 联合密度/分布律不能分解为因子乘积
2. 条件分布不等于边缘分布
3. 支撑集非矩形

**示例**：
- f(x, y) = 2, 0 < x < y < 1（支撑集是三角形，非独立）
- f(x, y) = 6xy, 0 < x < 1, 0 < y < √(1-x)（非矩形支撑，非独立）

### 2.3 判定步骤

1. 检查支撑集是否为矩形
2. 若为矩形，检查联合密度能否分解
3. 验证边缘密度的乘积是否等于联合密度

## 3. 多个随机变量的独立性

### 3.1 定义

**相互独立**：X₁, X₂, ..., Xₙ 相互独立，当且仅当：
$$F(x_1, x_2, \ldots, x_n) = \prod_{i=1}^{n} F_{X_i}(x_i)$$

**等价条件**：
$$f(x_1, x_2, \ldots, x_n) = \prod_{i=1}^{n} f_{X_i}(x_i)$$

### 3.2 两两独立与相互独立

**两两独立**：任意两个变量独立。

**相互独立**：任意子集的联合分布等于边缘分布的乘积。

**重要**：相互独立 ⟹ 两两独立，但反之不成立！

### 3.3 反例：两两独立但非相互独立

设 X₁, X₂ 独立同分布 P(Xᵢ = ±1) = 1/2，X₃ = X₁X₂。

可以验证：
- X₁, X₂ 独立 ✓
- X₁, X₃ 独立 ✓
- X₂, X₃ 独立 ✓
- 但 X₁, X₂, X₃ 不相互独立

## 4. 独立随机变量的性质

### 4.1 期望的性质

**定理**：若 X, Y 独立，则：
$$E[XY] = E[X] \cdot E[Y]$$

**推广**：若 X₁, ..., Xₙ 相互独立，则：
$$E\left[\prod_{i=1}^{n} X_i\right] = \prod_{i=1}^{n} E[X_i]$$

**注意**：E[XY] = E[X]E[Y] 是独立的必要条件，但不是充分条件。

### 4.2 方差的性质

**定理**：若 X, Y 独立，则：
$$\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)$$

**推广**：若 X₁, ..., Xₙ 相互独立，则：
$$\text{Var}\left(\sum_{i=1}^{n} X_i\right) = \sum_{i=1}^{n} \text{Var}(X_i)$$

### 4.3 协方差与独立性

**定理**：若 X, Y 独立，则 Cov(X, Y) = 0。

**注意**：Cov(X, Y) = 0 不意味着独立！（零相关不等于独立）

**反例**：设 X ~ N(0,1)，Y = X²，则 Cov(X, Y) = 0，但 X, Y 不独立。

### 4.4 独立性与不相关

| 关系 | 说明 |
|------|------|
| 独立 ⟹ 不相关 | 总是成立 |
| 不相关 ⟹ 独立 | 一般不成立 |
| 特例 | 二维正态分布：不相关 ⟺ 独立 |

## 5. 独立随机变量的函数

### 5.1 函数的独立性

**定理**：若 X, Y 独立，则对任意函数 g, h：
- g(X) 与 h(Y) 独立

**应用**：若 X, Y 独立，则 X² 与 e^Y 独立。

### 5.2 和的分布（卷积公式）

**定理**：若 X, Y 独立，其密度分别为 f_X, f_Y，则 Z = X + Y 的密度为：
$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) dx$$

这称为**卷积**，记为 f_Z = f_X * f_Y。

### 5.3 积与商的分布

**积 Z = XY 的密度**：
$$f_Z(z) = \int_{-\infty}^{+\infty} \frac{1}{|x|} f_X(x) f_Y(z/x) dx$$

**商 Z = X/Y 的密度**：
$$f_Z(z) = \int_{-\infty}^{+\infty} |y| f_X(zy) f_Y(y) dy$$

## 6. 独立同分布 (i.i.d.)

### 6.1 定义

X₁, X₂, ..., Xₙ **独立同分布**（independent and identically distributed），如果：
1. 它们相互独立
2. 它们具有相同的分布函数 F

### 6.2 样本均值与方差

设 X₁, ..., Xₙ i.i.d.，均值 μ，方差 σ²。

**样本均值**：
$$\bar{X} = \frac{1}{n}\sum_{i=1}^{n} X_i$$
- E[X̄] = μ
- Var(X̄) = σ²/n

**样本方差**：
$$S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$$
- E[S²] = σ²

### 6.3 重要结论

**样本均值的性质**：
- 无偏性：E[X̄] = μ
- 一致性：X̄ → μ (概率收敛)
- 方差减小：Var(X̄) = σ²/n

## 7. 独立性的应用

### 7.1 统计推断的基础

独立样本是许多统计方法的基础：
- 参数估计
- 假设检验
- 置信区间

### 7.2 极限定理

独立同分布序列是大数定律和中心极限定理的基础：
- 弱大数定律：X̄ → μ (概率收敛)
- 中心极限定理：√n(X̄ - μ)/σ → N(0,1)

### 7.3 随机过程

独立性是许多随机过程的基础假设：
- 泊松过程
- 随机游走
- 马尔可夫链

## 8. 特殊分布的独立性

### 8.1 正态分布

**定理**：若 X ~ N(μ₁, σ₁²), Y ~ N(μ₂, σ₂²) 独立，则：
- X + Y ~ N(μ₁+μ₂, σ₁²+σ₂²)
- aX + bY ~ N(aμ₁+bμ₂, a²σ₁²+b²σ₂²)

**二维正态**：相关系数 ρ = 0 ⟺ X, Y 独立。

### 8.2 指数分布

**定理**：若 X₁, ..., Xₙ i.i.d. Exp(λ)，则：
$$\sum_{i=1}^{n} X_i \sim \text{Gamma}(n, \lambda)$$

### 8.3 卡方分布

**定理**：若 X₁, ..., Xₙ i.i.d. N(0,1)，则：
$$\sum_{i=1}^{n} X_i^2 \sim \chi^2(n)$$

## 9. Python验证独立性

```python
import numpy as np
from scipy import stats

# 离散型独立性检验
joint_pmf = np.array([[0.1, 0.2],
                      [0.3, 0.4]])

p_x = joint_pmf.sum(axis=1)
p_y = joint_pmf.sum(axis=0)

# 检验 p_ij = p_i * p_j
expected = np.outer(p_x, p_y)
is_independent = np.allclose(joint_pmf, expected)
print(f"离散型独立性: {is_independent}")

# 连续型独立性检验（支撑集检验）
# f(x,y) = 6xy, 0<x<1, 0<y<√(1-x)
# 支撑集 {(x,y): 0<x<1, 0<y<√(1-x)} 不是矩形
# 因此 X, Y 不独立

# 正态分布独立性检验
np.random.seed(42)
n = 10000

# 独立情况
X = np.random.normal(0, 1, n)
Y = np.random.normal(0, 1, n)
corr = np.corrcoef(X, Y)[0, 1]
print(f"独立样本相关系数: {corr:.4f}")  # 接近0

# 相关情况
rho = 0.5
X = np.random.normal(0, 1, n)
Y = rho * X + np.sqrt(1-rho**2) * np.random.normal(0, 1, n)
corr = np.corrcoef(X, Y)[0, 1]
print(f"相关样本相关系数: {corr:.4f}")  # 接近0.5
```

## 10. 总结

| 判定方法 | 适用情况 |
|----------|----------|
| 分布函数分解 | 一般情况 |
| 密度函数分解 | 连续型 |
| 支撑集形状 | 快速判断非独立 |
| 条件分布=边缘分布 | 理论分析 |

**关键要点**：
1. 独立性需要验证所有可能的取值组合
2. 两两独立 ≠ 相互独立
3. 独立 ⟹ 不相关，反之一般不成立
4. 二维正态分布：独立 ⟺ 不相关

**独立性 vs 不相关性**：
- 独立：分布层面的"无关"
- 不相关：线性关系层面的"无关"
- 独立更强，不相关更弱
