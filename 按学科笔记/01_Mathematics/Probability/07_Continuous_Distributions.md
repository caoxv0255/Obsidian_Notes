## 1. 连续型随机变量的定义

### 1.1 基本概念

**定义**：若存在非负函数 f(x) 使得：
$$F(x) = \int_{-\infty}^{x} f(t) dt$$

则称 X 为**连续型随机变量**，f(x) 为**概率密度函数（PDF）**。

### 1.2 概率密度的性质

1. f(x) ≥ 0
2. ∫₋∞^∞ f(x)dx = 1
3. P(a < X ≤ b) = ∫ₐᵇ f(x)dx
4. P(X = x) = 0（对任意单点）
5. F'(x) = f(x)（在连续点处）

**重要说明**：
- f(x) 本身不是概率
- f(x) 可以大于 1
- 只有积分才有概率意义

## 2. 均匀分布

### 2.1 定义

**定义**：X ~ U(a, b)：
$$f(x) = \begin{cases} \frac{1}{b-a}, & a \leq x \leq b \\ 0, & \text{其他} \end{cases}$$

$$F(x) = \begin{cases} 0, & x < a \\ \frac{x-a}{b-a}, & a \leq x \leq b \\ 1, & x > b \end{cases}$$

### 2.2 数字特征

- E[X] = (a+b)/2
- Var(X) = (b-a)²/12

### 2.3 性质

**线性变换**：若 X ~ U(a, b)，则：
$$\frac{X-a}{b-a} \sim U(0, 1)$$

**例**：设 X ~ U(0, 1)，求 Y = -ln X 的分布。

**解**：当 y > 0：
$$F_Y(y) = P(Y \leq y) = P(-\ln X \leq y) = P(X \geq e^{-y}) = 1 - e^{-y}$$

故 Y ~ Exp(1)。

## 3. 指数分布

### 3.1 定义

**定义**：X ~ Exp(λ)：
$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \geq 0 \\ 0, & x < 0 \end{cases}$$

$$F(x) = \begin{cases} 1 - e^{-\lambda x}, & x \geq 0 \\ 0, & x < 0 \end{cases}$$

### 3.2 数字特征

- E[X] = 1/λ
- Var(X) = 1/λ²
- 中位数：ln 2 / λ
- 矩母函数：M(t) = λ/(λ-t)，t < λ

### 3.3 无记忆性

**定理**：X 具有无记忆性 ⟺ X ~ Exp(λ)

**无记忆性**：
$$P(X > s+t | X > s) = P(X > t)$$

**证明**：
$$P(X > s+t | X > s) = \frac{P(X > s+t)}{P(X > s)} = \frac{e^{-\lambda(s+t)}}{e^{-\lambda s}} = e^{-\lambda t} = P(X > t)$$

### 3.4 应用

**例**：某元件寿命 X ~ Exp(0.1)（单位：年），求已使用 5 年后再使用 5 年的概率。

**解**：由无记忆性：
$$P(X > 10 | X > 5) = P(X > 5) = e^{-0.5} \approx 0.6065$$

### 3.5 与泊松过程的关系

**定理**：在泊松过程 {N(t)} 中：
- 到达时间间隔 ~ Exp(λ)
- 第 n 个事件到达时间 ~ Gamma(n, λ)

## 4. 正态分布

### 4.1 定义

**定义**：X ~ N(μ, σ²)：
$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad x \in \mathbb{R}$$

**标准正态分布** N(0, 1)：
$$\phi(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}$$

$$\Phi(x) = \int_{-\infty}^{x} \phi(t) dt$$

### 4.2 数字特征

- E[X] = μ
- Var(X) = σ²
- 矩母函数：M(t) = exp(μt + σ²t²/2)
- 特征函数：φ(t) = exp(iμt - σ²t²/2)

### 4.3 性质

**对称性**：
$$\Phi(-x) = 1 - \Phi(x)$$

**线性变换**：若 X ~ N(μ, σ²)，则：
$$aX + b \sim N(a\mu + b, a^2\sigma^2)$$

**标准化**：
$$Z = \frac{X - \mu}{\sigma} \sim N(0, 1)$$

**可加性**：若 X₁ ~ N(μ₁, σ₁²)，X₂ ~ N(μ₂, σ₂²)，且独立，则：
$$X_1 + X_2 \sim N(\mu_1 + \mu_2, \sigma_1^2 + \sigma_2^2)$$

### 4.4 3σ 原则

$$P(\mu - \sigma < X < \mu + \sigma) = 2\Phi(1) - 1 \approx 0.6827$$
$$P(\mu - 2\sigma < X < \mu + 2\sigma) = 2\Phi(2) - 1 \approx 0.9545$$
$$P(\mu - 3\sigma < X < \mu + 3\sigma) = 2\Phi(3) - 1 \approx 0.9973$$

### 4.5 分位数

**常用分位数**：
- Φ⁻¹(0.9) ≈ 1.282
- Φ⁻¹(0.95) ≈ 1.645
- Φ⁻¹(0.975) ≈ 1.960
- Φ⁻¹(0.99) ≈ 2.326

## 5. 伽马分布

### 5.1 伽马函数

**定义**：
$$\Gamma(\alpha) = \int_0^{\infty} t^{\alpha-1} e^{-t} dt, \quad \alpha > 0$$

**性质**：
1. Γ(α+1) = αΓ(α)
2. Γ(n) = (n-1)!（n 为正整数）
3. Γ(1/2) = √π

### 5.2 伽马分布

**定义**：X ~ Gamma(α, β)：
$$f(x) = \begin{cases} \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$$

### 5.3 数字特征

- E[X] = α/β
- Var(X) = α/β²
- 矩母函数：M(t) = (1 - t/β)^{-α}，t < β

### 5.4 特殊情形

- **指数分布**：Gamma(1, β) = Exp(β)
- **卡方分布**：Gamma(n/2, 1/2) = χ²(n)
- **Erlang分布**：Gamma(n, β)，n 为正整数

### 5.5 可加性

**定理**：若 X₁ ~ Gamma(α₁, β)，X₂ ~ Gamma(α₂, β)，且独立，则：
$$X_1 + X_2 \sim \text{Gamma}(\alpha_1 + \alpha_2, \beta)$$

## 6. 贝塔分布

### 6.1 贝塔函数

**定义**：
$$B(\alpha, \beta) = \int_0^1 t^{\alpha-1} (1-t)^{\beta-1} dt = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}$$

### 6.2 贝塔分布

**定义**：X ~ Beta(α, β)：
$$f(x) = \begin{cases} \frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha, \beta)}, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

### 6.3 数字特征

- E[X] = α/(α + β)
- Var(X) = αβ/[(α+β)²(α+β+1)]

### 6.4 特殊情形

- **均匀分布**：Beta(1, 1) = U(0, 1)
- **对称贝塔**：α = β 时对称

### 6.5 应用

- 先验分布（贝叶斯统计）
- 随机概率建模

## 7. 对数正态分布

### 7.1 定义

**定义**：X ~ LN(μ, σ²)：
$$f(x) = \begin{cases} \frac{1}{x\sqrt{2\pi}\sigma} e^{-\frac{(\ln x - \mu)^2}{2\sigma^2}}, & x > 0 \\ 0, & x \leq 0 \end{cases}$$

### 7.2 性质

**关系**：X ~ LN(μ, σ²) ⟺ ln X ~ N(μ, σ²)

**数字特征**：
- E[X] = e^{μ + σ²/2}
- Var(X) = e^{2μ + σ²}(e^{σ²} - 1)

### 7.3 应用

- 收入分布
- 股票价格

## 8. 小结

### 8.1 常见连续分布

| 分布 | 记号 | 期望 | 方差 | 特点 |
|------|------|------|------|------|
| 均匀 | U(a,b) | (a+b)/2 | (b-a)²/12 | 等可能性 |
| 指数 | Exp(λ) | 1/λ | 1/λ² | 无记忆性 |
| 正态 | N(μ,σ²) | μ | σ² | 中心极限定理 |
| 伽马 | Gamma(α,β) | α/β | α/β² | 指数和 |
| 贝塔 | Beta(α,β) | α/(α+β) | αβ/[(α+β)²(α+β+1)] | [0,1]上分布 |

### 8.2 关系图

```
指数分布 → 伽马分布 (可加)
    ↓
对数正态 (取对数→正态)

正态分布
    ↓
卡方分布 (平方和)

均匀分布 → 贝塔分布 (变换)
```
