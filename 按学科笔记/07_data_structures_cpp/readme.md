## 目录结构

- `00_index.md`：总入口，负责串联讲义、练习、实验和参考资料。
- `01_lecture_notes/`：按章节整理的讲义摘要页与原始详细笔记归档。
- `02_problems/`：按章节整理的练习页、补充习题和试卷资料。
- `03_labs/`：按章节整理的实验页、代码示例和调试记录。
- `04_references/`：教材摘录、复习提纲、错题本和原始资料归档。

## 学习顺序

1. 先从 [总入口](00_index.md) 开始，确认课程地图与学习路径。
2. 再按 [讲义索引](01_lecture_notes/00_index.md) 逐章阅读。
3. 配合 [练习索引](02_problems/00_index.md) 做题和复盘。
4. 结合 [实验索引](03_labs/00_index.md) 查看代码与实现。
5. 最后通过 [参考资料索引](04_references/00_index.md) 汇总试卷、提纲和原始材料。
