- 哈夫曼树（HuffmanTree）

**遍历算法**：
- 前序遍历：根 → 左 → 右
- 中序遍历：左 → 根 → 右
- 后序遍历：左 → 右 → 根
- 层序遍历：从上到下，从左到右

**复杂度分析**：

| 操作 | BST | AVL | 红黑树 |
|------|-----|-----|--------|
| 查找 | O(h) | O(log n) | O(log n) |
| 插入 | O(h) | O(log n) | O(log n) |
| 删除 | O(h) | O(log n) | O(log n) |

- Prim算法（最小生成树）
- Kruskal算法（最小生成树）

- 拓扑排序

**图的表示**：
- 邻接矩阵：O(V²) 空间
- 邻接表：O(V+E) 空间

**复杂度对比**：

| 算法 | 时间复杂度 | 适用场景 |
|------|-----------|----------|
| DFS | O(V+E) | 路径查找、连通性 |
| BFS | O(V+E) | 最短路径（无权图） |
| Dijkstra | O((V+E)log V) | 单源最短路径（非负权） |
| Floyd | O(V³) | 全源最短路径 |
| Prim | O((V+E)log V) | 稠密图MST |
| Kruskal | O(Elog E) | 稀疏图MST |
