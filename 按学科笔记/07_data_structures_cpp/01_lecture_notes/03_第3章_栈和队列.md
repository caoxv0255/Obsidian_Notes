# 第3章：栈和队列

> 这一页用于快速复盘 LIFO 与 FIFO 两类受限结构。

## 本章要点

- 栈是后进先出结构，只能在栈顶操作。
- 队列是先进先出结构，通常在队尾进、队头出。
- 循环队列是避免“假溢出”的常见实现。
- 栈和队列常用于控制流程、任务调度和表达式处理。

## 你应该掌握

- 能区分入栈、出栈、入队、出队的语义。
- 能解释递归为何天然适合栈。
- 能用队列理解层次遍历和缓冲调度。
- 能判断什么时候应该用栈而不是队列。

## 复习提示

- 栈解决“最近发生的事先处理”。
- 队列解决“先到先服务”。
- 很多题目表面是字符串或表达式题，实质是栈题。
