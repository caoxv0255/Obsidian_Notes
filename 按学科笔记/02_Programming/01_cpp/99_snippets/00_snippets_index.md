## 建议补齐（按优先级）

- Fast IO / Debug
- 基础工具：离散化、前缀和/差分、二分答案
- 数据结构：DSU、Fenwick(BIT)、Segment Tree、Sparse Table、单调栈/队列
- 图：Dijkstra、0-1 BFS、拓扑、LCA(倍增)、SCC、Dinic
- 字符串：KMP、Z、Rolling Hash
- 数论：快速幂、逆元、组合数、筛法（需要时再扩展 CRT/NTT）
- 线段树：点改区查、区加区查、Sparse Table
- 图论进阶：Tarjan SCC、桥、割点
- 数论进阶：矩阵快速幂、NTT
- 单调结构：单调栈、单调队列、柱状图面积
- 图论补充：最小生成树、二分图匹配
- 计算几何：叉积、线段相交、凸包
- 字符串进阶：AC 自动机
- 图论进阶：最小费用最大流
- DP 进阶：数位 DP
- 字符串进阶：后缀数组 / LCP
- 树上结构：树链剖分
- 数据结构进阶：主席树 / 可持久化线段树
- 树上基础：LCA、树上差分
- 数论/DP 交叉：矩阵快速幂
- 状压 DP：子集枚举、TSP
- 离线查询：莫队
- 图论进阶：双连通分量、割点割边
- 图论进阶：2-SAT
- 离线查询进阶：带修改莫队
- 数论进阶：NTT
- 线性代数：高斯消元
- 数论进阶：FFT
- 异或结构：线性基
- 数据结构进阶：动态开点线段树
- 字符串进阶：后缀自动机
- 树分治：点分治
- 几何/函数优化：李超线段树
- 二维结构：树套树、二维树状数组
