# 输入输出 (Input Output)

> C++ 提供了丰富的输入输出功能，包括控制台 IO 和文件 IO

## 📌 核心定义

输入输出（IO）是程序与外部世界交互的方式。C++ 使用 iostream 库提供控制台 IO，使用 fstream 库提供文件 IO。

## 💡 直觉理解

IO 就像"对话"：
- 输入 = 程序接收信息（听）
- 输出 = 程序发送信息（说）
- 控制台 = 与用户对话
- 文件 = 与文件对话

## 📖 详细说明

### 1. 控制台输出

```cpp
std::cout << "Hello" << std::endl;  // 输出并换行
std::cerr << "Error" << std::endl;  // 错误输出
std::clog << "Log" << std::endl;    // 日志输出
```
### 2. 控制台输入

```cpp
int x;
std::cin >> x;  // 从键盘读取
```
### 3. 文件输入输出

```cpp
#include <fstream>
std::ifstream inFile("input.txt");   // 读取文件
std::ofstream outFile("output.txt"); // 写入文件
```
## 💻 代码示例

### 示例 1: 基本输入输出

```cpp
#include <iostream>

int main() {
    int age;
    std::string name;
    
    std::cout << "请输入你的名字: ";
    std::cin >> name;
    
    std::cout << "请输入你的年龄: ";
    std::cin >> age;
    
    std::cout << "你好，" << name << "！" << std::endl;
    std::cout << "你今年 " << age << " 岁。" << std::endl;
    
    return 0;
}
```
### 示例 2: 格式化输出

```cpp
#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159265;
    
    // 设置精度
    std::cout << std::fixed << std::setprecision(2);
    std::cout << "Pi: " << pi << std::endl;
    
    // 科学计数法
    std::cout << std::scientific << pi << std::endl;
    
    // 设置宽度
    std::cout << std::setw(10) << "Hello" << std::endl;
    
    return 0;
}
```
### 示例 3: 文件操作

```cpp
#include <iostream>
#include <fstream>

int main() {
    // 写入文件
    std::ofstream outFile("data.txt");
    if (outFile.is_open()) {
        outFile << "Hello, File!" << std::endl;
        outFile << "42" << std::endl;
        outFile.close();
    }
    
    // 读取文件
    std::ifstream inFile("data.txt");
    if (inFile.is_open()) {
        std::string line;
        while (std::getline(inFile, line)) {
            std::cout << line << std::endl;
        }
        inFile.close();
    }
    
    return 0;
}
```
### 示例 4: ACM 竞赛快速输入输出

```cpp
#include <iostream>

// 快速输入
inline int readInt() {
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + ch - '0';
        ch = getchar();
    }
    return x * f;
}

int main() {
    // 关闭同步以提高速度
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);
    
    int n = readInt();
    std::cout << "Read: " << n << std::endl;
    
    return 0;
}
```
## 🎯 应用场景

### ACM 竞赛中的应用
- 快速输入输出：处理大量数据
- 文件读写：从文件读取测试数据
- 调试输出：输出中间结果

### 机器学习中的应用
- 数据加载：从文件读取数据集
- 结果保存：保存训练结果
- 日志记录：记录训练过程
