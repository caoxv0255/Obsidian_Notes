# 第4章 实验：分治策略

## 实验目标

- 理解分治法的三个步骤：分、解、合
- 掌握最大子数组问题的两种解法
- 对比分治法与动态规划（Kadane）的性能

## 问题定义

**最大子数组问题**：给定一个整数数组，找出一个连续子数组，使其和最大。

### 例子
```
输入: [13, -3, -25, 20, -3, -16, -23, 18, 20, -7, 12, -5, -22, 15, -4, 7]
输出: 最大和=43，起点=7（值20），终点=10（值12）
```

## 算法对比

### 1. 分治法（Divide-and-Conquer）

**时间复杂度**: O(n log n)

**步骤**：
1. **分**：将数组分为左、右、跨越三部分
2. **解**：递归求解左右部分的最大子数组
3. **合**：比较三个结果，返回最大

```cpp
SubarrayResult max_subarray_divide_conquer(A, low, high) {
    if low == high: return A[low]
    
    mid = (low + high) / 2
    left = max_subarray_divide_conquer(A, low, mid)
    right = max_subarray_divide_conquer(A, mid+1, high)
    cross = max_crossing_subarray(A, low, mid, high)
    
    return max(left, right, cross)
}
```

### 2. Kadane算法（动态规划）

**时间复杂度**: O(n)

**思想**：
- 跟踪当前最大和与全局最大和
- 若当前和为负，则重新开始

```cpp
max_sum = A[0]
current_sum = A[0]

for i = 1 to n:
    current_sum = max(A[i], current_sum + A[i])
    max_sum = max(max_sum, current_sum)
```
