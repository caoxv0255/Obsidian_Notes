# 第3章 实验：函数的增长

## 实验目标

- 理解渐进记号（O、Ω、Θ）的含义
- 学习如何分析算法的时间复杂度
- 比较不同复杂度函数的增长速度

## 关键概念

### 1. 渐进记号

| 记号 | 定义 | 含义 |
|------|------|------|
| O(g(n)) | 上界 | 最坏情况 |
| Ω(g(n)) | 下界 | 最好情况 |
| Θ(g(n)) | 紧界 | 平均情况 |

### 2. 常见复杂度排序

```
O(1) < O(log n) < O(√n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!)
```

### 3. 复杂度增长速度

在 n=1000000 时：
- O(1) ≈ 1
- O(log n) ≈ 20
- O(n) ≈ 1,000,000
- O(n log n) ≈ 20,000,000
- O(n²) ≈ 10¹²（不可接受）
- O(2ⁿ)：计算不了

## 算法复杂度参考

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|----------|------|
| 插入排序 | O(n²) | O(1) | ✅ |
| 归并排序 | O(n log n) | O(n) | ✅ |
| 快速排序 | O(n log n)~O(n²) | O(log n) | ❌ |
| 堆排序 | O(n log n) | O(1) | ❌ |
| 线性搜索 | O(n) | O(1) | - |
| 二分查找 | O(log n) | O(1) | - |
