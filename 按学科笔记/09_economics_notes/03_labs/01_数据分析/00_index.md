# 01 数据分析

学习经济学数据的处理、计算与基础分析方法。

## 📝 实验列表

### [01 CPI与通胀分析](01_CPI与通胀分析.md)
**目标**：理解价格指数的计算与通胀率分析

**数据**：2020-2024年商品价格表
- 食品、能源、房租、医疗等

**任务**：
1. 计算年度CPI（加权平均）
2. 计算同比、环比通胀率
3. 绘制趋势图表
4. 分析变化原因与政策影响

**工具**：Excel、Python(Pandas)

**预期产出**：
- CPI计算表
- 趋势分析图表
- 200字分析报告

---

### [02 供给需求实证](02_供给需求实证.md)
**目标**：用实际数据拟合供给与需求曲线

**案例选择**（三选一）：
- 农产品市场（粮食、蔬菜价格）
- 房地产市场（房价与供应量）
- 劳动力市场（工资与失业率）

**分析步骤**：
1. 收集3-5年价格与数量数据
2. 绘制供需图表
3. 进行回归拟合
4. 计算价格与收入弹性
5. 解释市场变化成因

**工具**：Excel、Python(Scipy)、R(lm)

**预期产出**：
- 供给需求函数
- 弹性系数
- 政策模拟分析

---

### [03 价格与收入弹性计算](03_价格与收入弹性计算.md)
**目标**：掌握弹性概念与计算方法

**内容**：
- 点弹性与弧弹性
- 交叉价格弹性（商品替代性）
- 收入弹性（正常商品与劣商品）
- 应用案例

**计算练习**：
- 不同商品的需求弹性
