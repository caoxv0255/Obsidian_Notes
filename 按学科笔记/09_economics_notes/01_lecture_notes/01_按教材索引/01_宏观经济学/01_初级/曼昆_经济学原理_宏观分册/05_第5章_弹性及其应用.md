# 第5章：弹性及其应用

> 本章学习目标：
> - 深入理解弹性的概念和计算
> - 掌握需求价格弹性的类型和影响因素
> - 理解供给价格弹性的类型和影响因素
> - 学会分析弹性对总收益的影响
> - 了解弹性在实际政策和商业中的应用

---

## 5.1 弹性的概念

### 5.1.1 弹性的定义

**弹性（Elasticity）**：
一个变量对另一个变量变化的敏感程度的度量。

**数学表达**：
```
弹性 = (Y的变化百分比) / (X的变化百分比)
   = (ΔY/Y) / (ΔX/X)
```

其中：
- Y：因变量
- X：自变量
- ΔY、ΔX：变量的变化量

**弹性的性质**：
1. **无量纲**：没有单位，便于比较
2. **可正可负**：取决于变量关系
3. **比较相对大小**：绝对值越大，敏感度越高

### 5.1.2 为什么弹性重要

**弹性的重要性**：
1. **政策制定**：
   - 税收政策的效果取决于弹性
   - 价格管制的后果取决于弹性

2. **商业决策**：
   - 定价策略需要考虑弹性
   - 市场预测需要了解弹性

3. **经济分析**：
   - 预测经济变量的变化
   - 评估政策效果

**实际应用**：
- **税收政策**：弹性大的商品税收导致消费大幅减少
- **价格歧视**：根据弹性差异制定不同价格
- **需求预测**：预测价格变化对需求的影响

## 5.2 需求价格弹性

### 5.2.1 需求价格弹性的定义

**需求价格弹性（Price Elasticity of Demand, Ed）**：
需求量对价格变化的敏感程度。

**定义公式**：
```
Ed = (需求量变化的百分比) / (价格变化的百分比)
   = (ΔQ/Q) / (ΔP/P)
   = (ΔQ/ΔP) × (P/Q)
```

其中：
- Ed：需求价格弹性
- ΔQ：需求量的变化
- ΔP：价格的变化
- Q：初始需求量
- P：初始价格

**注意**：Ed通常是负数（因为需求曲线向下倾斜）

### 5.2.2 弹性的类型

根据弹性值的大小，需求可以分为：

| 弹性值 | 弹性类型 | 符号 | 特点 | 例子 |
|--------|----------|------|------|------|
| |Ed| > 1 | 富有弹性 | 负 | 需求量对价格变化敏感 | 奢侈品、有替代品的商品 |
| |Ed| = 1 | 单位弹性 | 负 | 需求量变化比例等于价格变化比例 | - |
| |Ed| < 1 | 缺乏弹性 | 负 | 需求量对价格变化不敏感 | 必需品、无替代品的商品 |
| |Ed| = 0 | 完全无弹性 | - | 需求量不随价格变化 | 生命必需药品 |
| |Ed| → -∞ | 完全有弹性 | - | 价格微小变化导致需求量巨大变化 | 两个完全替代的商品 |

### 5.2.3 弹性的计算方法

**中点公式（Midpoint Method）**：
避免使用不同的初始值导致不同的弹性值。

**公式**：
```
Ed = [(Q2 - Q1) / ((Q1 + Q2)/2)] / [(P2 - P1) / ((P1 + P2)/2)]
   = (Q2 - Q1) / (Q2 + Q1) × (P1 + P2) / (P2 - P1)
```

**计算示例**：

**例1**：
- 价格从4元上升到5元（P1 = 4, P2 = 5）
- 需求量从120个下降到80个（Q1 = 120, Q2 = 80）

**计算**：
```
Ed = (80 - 120) / (80 + 120) × (4 + 5) / (5 - 4)
   = (-40) / 200 × 9 / 1
   = -0.2 × 9
   = -1.8
```

**解释**：弹性为-1.8（富有弹性），需求量对价格变化非常敏感。

**例2**：
- 价格从5元上升到6元（P1 = 5, P2 = 6）
- 需求量从100个下降到95个（Q1 = 100, Q2 = 95）

**计算**：
```
Ed = (95 - 100) / (95 + 100) × (5 + 6) / (6 - 5)
   = (-5) / 195 × 11 / 1
   = -0.0256 × 11
   = -0.28
```

**解释**：弹性为-0.28（缺乏弹性），需求量对价格变化不敏感。

### 5.2.4 影响需求价格弹性的因素

**1. 替代品的可获得性（Availability of Substitutes）**
- **替代品多** → 弹性大
- **替代品少** → 弹性小

**例子**：
- **黄油 vs 人造黄油**：替代品多 → 弹性大
- **胰岛素**：无替代品 → 弹性小

**2. 必需程度（Necessity vs Luxury）**
- **必需品** → 弹性小
- **奢侈品** → 弹性大

**例子**：
- **食盐**：必需品 → 弹性小
- **游艇**：奢侈品 → 弹性大

**3. 市场定义的广度（Definition of the Market）**
- **市场定义越窄** → 弹性越大
- **市场定义越宽** → 弹性越小

**例子**：
- **特定品牌的食品**：替代品多 → 弹性大
- **所有食品**：替代品少 → 弹性小

**4. 时间跨度（Time Horizon）**
- **短期**：弹性小
- **长期**：弹性大

**原因**：
- 短期：难以找到替代品、调整消费习惯
- 长期：可以找到替代品、调整消费习惯

**例子**：
- **汽油价格上升**：
  - 短期：弹性小（-0.2）
  - 长期：弹性大（-0.7）

**5. 预算占比（Share of Budget）**
- **占比小**：弹性大
- **占比大**：弹性小

**原因**：
- 占比小：价格变化影响小，不敏感
- 占比大：价格变化影响大，敏感

**例子**：
- **口香糖**：占比小 → 弹性大
- **住房**：占比大 → 弹性小

**总结表**：

| 因素 | 弹性大 | 弹性小 |
|------|--------|--------|
| 替代品 | 多 | 少 |
| 必需程度 | 奢侈品 | 必需品 |
| 市场定义 | 窄 | 宽 |
| 时间跨度 | 长期 | 短期 |
| 预算占比 | 小 | 大 |

### 5.2.5 需求价格弹性的应用

**应用1：预测需求量的变化**

**问题**：
如果价格上升10%，弹性为-2，需求量会变化多少？

**解答**：
```
需求量变化百分比 = Ed × 价格变化百分比
                   = -2 × 10%
                   = -20%
```

**结论**：需求量减少20%。

**应用2：税收对需求的影响**

**问题**：
政府对某商品征收20%的税，弹性为-1.5，对需求量有什么影响？

**解答**：
```
需求量变化百分比 = Ed × 价格变化百分比
                   = -1.5 × 20%
                   = -30%
```

**结论**：需求量减少30%。

**应用3：价格歧视**

**条件**：
- 不同消费者有不同的弹性
- 可以区分消费者
- 防止套利

**策略**：
- 对弹性大的消费者：降低价格，增加销量
- 对弹性小的消费者：提高价格，增加利润

**例子**：
- **航空公司**：商务旅客（弹性小）→ 高价
- **航空公司**：休闲旅客（弹性大）→ 低价

## 5.3 其他需求弹性

### 5.3.1 需求收入弹性

**需求收入弹性（Income Elasticity of Demand, Ei）**：
需求量对收入变化的敏感程度。

**定义公式**：
```
Ei = (需求量变化的百分比) / (收入变化的百分比)
   = (ΔQ/Q) / (ΔI/I)
   = (ΔQ/ΔI) × (I/Q)
```

**弹性类型**：

| 弹性值 | 弹性类型 | 符号 | 特点 | 例子 |
|--------|----------|------|------|------|
| Ei > 0 | 正常商品 | 正 | 收入增加，需求增加 | 食品、新衣服 |
| Ei < 0 | 低档商品 | 负 | 收入增加，需求减少 | 旧衣服、公共交通 |
| Ei = 0 | 收入中性商品 | 0 | 需求不随收入变化 | 基本生活必需品 |

**计算示例**：

**例**：
- 收入从1000元上升到1500元（增加50%）
- 需求量从10个上升到15个（增加50%）

**计算**：
```
Ei = (15 - 10) / 10 / (1500 - 1000) / 1000
   = 0.5 / 0.5
   = 1
```

**结论**：这是正常商品，单位弹性。

### 5.3.2 需求交叉弹性

**需求交叉弹性（Cross-Price Elasticity of Demand, Exy）**：
一种商品的需求量对另一种商品价格变化的敏感程度。

**定义公式**：
```
Exy = (商品X需求量变化的百分比) / (商品Y价格变化的百分比)
    = (ΔQx/Qx) / (ΔPy/Py)
    = (ΔQx/ΔPy) × (Py/Qx)
```

**弹性类型**：

| 弹性值 | 弹性类型 | 关系 | 例子 |
|--------|----------|------|------|
| Exy > 0 | 替代品 | 正 | 可口可乐 vs 百事可乐 |
| Exy < 0 | 互补品 | 负 | 汽车 vs 汽油 |
| Exy = 0 | 无关 | 无 | 书籍 vs 香蕉 |

**计算示例**：

**例1：替代品**
- 可口可乐价格从3元上升到3.3元（上升10%）
- 百事可乐需求量从100上升到110（增加10%）

**计算**：
```
Exy = (110 - 100) / 100 / (3.3 - 3) / 3
    = 0.1 / 0.1
    = 1
```

**结论**：可口可乐和百事可乐是替代品。

**例2：互补品**
- 汽车价格从20万元上升到22万元（上升10%）
- 汽油需求量从1000下降到900（下降10%）

**计算**：
```
Exy = (900 - 1000) / 1000 / (22 - 20) / 20
    = -0.1 / 0.1
    = -1
```

**结论**：汽车和汽油是互补品。

## 5.4 供给价格弹性

### 5.4.1 供给价格弹性的定义

**供给价格弹性（Price Elasticity of Supply, Es）**：
供给量对价格变化的敏感程度。

**定义公式**：
```
Es = (供给量变化的百分比) / (价格变化的百分比)
   = (ΔQ/Q) / (ΔP/P)
   = (ΔQ/ΔP) × (P/Q)
```

其中：
- Es：供给价格弹性
- ΔQ：供给量的变化
- ΔP：价格的变化
- Q：初始供给量
- P：初始价格

**注意**：Es通常是正数（因为供给曲线向上倾斜）

### 5.4.2 供给弹性的类型

根据弹性值的大小，供给可以分为：

| 弹性值 | 弹性类型 | 符号 | 特点 | 例子 |
|--------|----------|------|------|------|
| Es > 1 | 富有弹性 | 正 | 供给量对价格变化敏感 | 容易调整产量的行业 |
| Es = 1 | 单位弹性 | 正 | 供给量变化比例等于价格变化比例 | - |
| Es < 1 | 缺乏弹性 | 正 | 供给量对价格变化不敏感 | 难以调整产量的行业 |
| Es = 0 | 完全无弹性 | 正 | 供给量不随价格变化 | 古董、艺术品 |
| Es → ∞ | 完全有弹性 | 正 | 价格微小变化导致供给量巨大变化 | 失业率最低时的劳动力 |

### 5.4.3 影响供给价格弹性的因素

**1. 生产调整的难易程度（Ease of Adjustment）**
- **容易调整** → 弹性大
- **难以调整** → 弹性小

**例子**：
- **制造业**：容易调整 → 弹性大
- **农业**：难以调整（生产周期长）→ 弹性小

**2. 时间跨度（Time Horizon）**
- **短期**：弹性小
- **长期**：弹性大

**原因**：
- 短期：难以调整生产规模
- 长期：可以调整生产规模

**例子**：
- **农产品**：
  - 短期：弹性小（种植周期固定）
  - 长期：弹性大（可以调整种植面积）

**3. 生产能力的限制（Production Capacity Constraints）**
- **有闲置产能** → 弹性大
- **满负荷生产** → 弹性小

**例子**：
- **经济衰退期**：有闲置产能 → 弹性大
- **经济繁荣期**：满负荷生产 → 弹性小

**4. 生产要素的流动性（Mobility of Factors of Production）**
- **要素流动性强** → 弹性大
- **要素流动性弱** → 弹性小

**例子**：
- **服务业**：要素流动性强 → 弹性大
- **重工业**：要素流动性弱 → 弹性小

**5. 库存的存在（Inventory）**
- **有库存** → 弹性大
- **无库存** → 弹性小

**例子**：
- **制造业**：有库存 → 弹性大
- **服务业**：无库存 → 弹性小

**总结表**：

| 因素 | 弹性大 | 弹性小 |
|------|--------|--------|
| 调整难度 | 容易 | 困难 |
| 时间跨度 | 长期 | 短期 |
| 产能限制 | 有闲置产能 | 满负荷 |
| 要素流动性 | 强 | 弱 |
| 库存 | 有库存 | 无库存 |

### 5.4.4 供给弹性的应用

**应用1：预测供给量的变化**

**问题**：
如果价格上升15%，弹性为0.5，供给量会变化多少？

**解答**：
```
供给量变化百分比 = Es × 价格变化百分比
                   = 0.5 × 15%
                   = 7.5%
```

**结论**：供给量增加7.5%。

**应用2：农业生产周期**

**农产品供给弹性的特点**：
- **短期弹性小**：因为种植周期固定
- **长期弹性大**：可以调整种植面积

**蛛网模型（Cobweb Model）**：
- 农产品价格和产量的周期性波动
- 原因：短期供给弹性小，调整存在时滞

**制造业供给弹性的特点**：
- **短期弹性中等**：可以加班、调整班次
- **长期弹性大**：可以建新工厂、购买新设备

## 5.5 弹性与总收益

### 5.5.1 总收益的概念

**总收益（Total Revenue, TR）**：
卖者从销售商品中获得的总收入。

**定义公式**：
```
TR = P × Q
```

其中：
- TR：总收益
- P：价格
- Q：销售数量

### 5.5.2 弹性与总收益的关系

**关键关系**：
> **总收益的变化取决于需求价格弹性**

**情况1：Ed = 1（单位弹性）**

**分析**：
- 价格上升10%，需求量下降10%
- 总收益变化：TR' = 1.1P × 0.9Q = 0.99PQ ≈ PQ

**结论**：总收益基本不变。

**情况2：|Ed| > 1（富有弹性）**

**分析**：
- 价格上升10%，需求量下降20%（Ed = -2）
- 总收益变化：TR' = 1.1P × 0.8Q = 0.88PQ

**结论**：价格上升，总收益减少。

**情况3：|Ed| < 1（缺乏弹性）**

**分析**：
- 价格上升10%，需求量下降5%（Ed = -0.5）
- 总收益变化：TR' = 1.1P × 0.95Q = 1.045PQ

**结论**：价格上升，总收益增加。

**总结表**：

| 弹性 | 价格变化 | 总收益变化 | 价格策略 |
|------|----------|-----------|----------|
| Ed = -1 | 上升 | 不变 | 无影响 |
| |Ed| > 1 | 上升 | 减少 | 降价增加收益 |
| |Ed| < 1 | 上升 | 增加 | 提价增加收益 |
| Ed = -1 | 下降 | 不变 | 无影响 |
| |Ed| > 1 | 下降 | 增加 | 降价增加收益 |
| |Ed| < 1 | 下降 | 减少 | 提价增加收益 |

### 5.5.3 总收益曲线

**图形分析**：

**总收益曲线的特点**：
- 从原点出发
- 先上升后下降
- 在弹性等于-1的点达到最大值

**数学推导**：

给定需求函数：Q = a - bP

**总收益**：
```
TR = P × Q = P × (a - bP) = aP - bP²
```

**总收益最大化**：
```
dTR/dP = a - 2bP = 0
P* = a/(2b)
```

**弹性在最大化点的值**：
```
Ed = dQ/dP × P/Q = -b × (a/(2b)) / (a - b×(a/(2b)))
   = -b × (a/(2b)) / (a - a/2)
   = -b × (a/(2b)) / (a/2)
   = -1
```

**重要结论**：
> **总收益在需求价格弹性等于-1的点达到最大值**

### 5.5.4 实际应用：定价策略

**应用1：航空公司定价**

**情况**：
- 商务旅客：弹性小（-0.5）
- 休闲旅客：弹性大（-2.0）

**策略**：
- 商务旅客：高价，总收益增加
- 休闲旅客：低价，总收益增加

**应用2：奢侈品定价**

**情况**：
- 奢侈品：弹性大（-2.5）
- 价格上升10% → 需求量下降25%

**策略**：
- 降价策略：价格下降10% → 需求量增加25%
- 总收益变化：0.9P × 1.25Q = 1.125PQ（增加12.5%）

**应用3：必需品定价**

**情况**：
- 必需品：弹性小（-0.3）
- 价格上升10% → 需求量下降3%

**策略**：
- 提价策略：价格上升10% → 需求量下降3%
- 总收益变化：1.1P × 0.97Q = 1.067PQ（增加6.7%）

## 5.6 弹性的实际应用

### 5.6.1 农业政策

**问题**：
- 农产品价格波动大
- 农民收入不稳定

**原因**：
- 需求价格弹性大（替代品多）
- 供给价格弹性小（生产周期长）
- 天气等外部因素影响

**政策建议**：
1. **价格支持政策**：保护农民收入
2. **农业保险**：分散风险
3. **农产品期货市场**：锁定价格
4. **研发新品种**：提高质量

### 5.6.2 税收政策

**问题**：
- 政府如何设计税收政策才能有效？
- 不同商品的税收效果不同

**分析**：

**税收对需求的影响**：
- 税收导致价格上升
- 需求量下降的幅度取决于弹性

**税收对总收益的影响**：
- 弹性大：税收导致需求量大幅下降，总收益可能减少
- 弹性小：税收导致需求量小幅下降，总收益可能增加

**最优税收策略**：
- 对弹性小的商品征收高税率
- 对弹性大的商品征收低税率
- 避免对弹性大的商品征收高税率

**例子**：
- **奢侈品**（弹性大）→ 低税率
- **必需品**（弹性小）→ 可以征收较高税率（但有公平性考虑）

### 5.6.3 反垄断政策

**问题**：
- 垄断企业如何定价？
- 政府如何监管？

**分析**：

**垄断定价策略**：
- 如果需求弹性小 → 高价策略
- 如果需求弹性大 → 低价策略

**政府监管**：
- 对弹性小的垄断企业进行价格管制
- 防止垄断企业滥用市场力量

### 5.6.4 最低工资政策

**问题**：
- 最低工资对就业的影响？
- 取决于劳动需求弹性

**分析**：

**劳动需求弹性小**：
- 工人缺乏技能
- 替代品少
- 最低工资对就业影响小

**劳动需求弹性大**：
- 工人容易替代
- 替代品多
- 最低工资对就业影响大

**结论**：
- 最低工资政策的效果取决于劳动需求弹性
- 需要具体分析具体情况

## 5.7 本章总结

### 核心要点

1. **弹性的概念**
   - 弹性是敏感度的度量
   - 无量纲，便于比较
   - 对政策制定很重要

2. **需求价格弹性**
   - 富有弹性：|Ed| > 1
   - 缺乏弹性：|Ed| < 1
   - 取决于：替代品、必需程度、市场定义、时间、预算占比

3. **其他需求弹性**
   - 收入弹性：正常商品、低档商品
   - 交叉弹性：替代品、互补品、无关商品

4. **供给价格弹性**
   - 富有弹性：Es > 1
   - 缺乏弹性：Es < 1
   - 取决于：调整难度、时间、产能、要素流动性、库存

5. **弹性与总收益**
   - |Ed| > 1：降价增加总收益
   - |Ed| < 1：提价增加总收益
   - |Ed| = 1：总收益最大

6. **弹性应用**
   - 定价策略：根据弹性制定不同价格
   - 税收政策：根据弹性设计税率
   - 农业政策：应对价格波动
   - 反垄断：防止滥用市场力量

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 弹性 | 对变化的敏感程度 | 无量纲 |
| 需求价格弹性 | 需求量对价格的敏感度 | 向下倾斜 |
| 供给价格弹性 | 供给量对价格的敏感度 | 向上倾斜 |
| 富有弹性 | |Ed| > 1 或 Es > 1 | 敏感度高 |
| 缺乏弹性 | |Ed| < 1 或 Es < 1 | 敏感度低 |
| 正常商品 | 收入弹性为正 | Ei > 0 |
| 低档商品 | 收入弹性为负 | Ei < 0 |
| 替代品 | 交叉弹性为正 | Exy > 0 |
| 互补品 | 交叉弹性为负 | Exy < 0 |
| 总收益 | TR = P × Q | 价格×数量 |
| 价格歧视 | 根据弹性定价 | 利润最大化 |
