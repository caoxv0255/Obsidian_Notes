# 第8章 经济波动导论

## 章节学习目标

1. **理解经济波动的事实**：掌握经济波动的基本特征和规律
2. **理解时间维度中的经济波动**：掌握经济周期的阶段和特征
3. **理解宏观经济的时间不一致性**：掌握政策制定的时间不一致性问题
4. **理解总需求和总供给**：掌握AD-AS模型的基本框架
5. **理解稳定化政策**：掌握财政政策和货币政策的稳定作用

## 8.1 经济波动的事实

### 8.1.1 经济波动的定义

**经济波动（Economic Fluctuations）**或商业周期（Business Cycle）是指经济活动围绕长期趋势的波动。

**特征：**
- 非规律性：周期长度和幅度变化
- 持续性：繁荣和衰退会持续一段时间
- 协动性：宏观经济变量一起波动

### 8.1.2 经济波动的特征

**事实1：经济波动是不规则的**

**特点：**
- 没有固定的周期长度
- 幅度变化很大
- 难以预测

**图形分析：**

```
GDP
  ↑
  │     /
  │    /  \
  │   /    \
  │  /      \
  │ /        \
  └─────────→ 时间
    衰退  扩张  衰退
```

**事实2：大多数宏观经济变量同时波动**

**顺周期变量（Pro-cyclical）：**
- GDP
- 消费
- 投资
- 就业
- 物价水平

**逆周期变量（Counter-cyclical）：**
- 失业率

**无周期变量（A-cyclical）：**
- 实际利率（有时）

**事实3：投资波动最大**

**数据（美国，1950-2020）：**

| 变量 | 标准差/平均（%） |
|------|----------------|
| GDP | 1.6 |
| 消费 | 1.2 |
| 投资 | 4.5 |
| 政府支出 | 1.5 |

**含义：**
- 投资波动性远大于消费
- 投资是经济波动的主要来源

**事实4：货币和价格是顺周期的**

**观察：**
- 货币供给在经济扩张时增加
- 价格水平在经济扩张时上升
- 但滞后于产出

**事实5：经济衰退时失业率上升**

**观察：**
- 衰退开始时失业率上升
- 衰退结束后失业率继续上升一段时间
- 失业率是滞后指标

### 8.1.3 经济波动的阶段

**商业周期的阶段：**

**1. 扩张（Expansion）**
- 产出增加
- 就业增加
- 消费和投资增加

**2. 峰值（Peak）**
- 经济达到最高点
- 转折点
- 开始进入衰退

**3. 衰退（Recession）**
- 产出连续两个季度下降
- 就业减少
- 消费和投资减少

**4. 谷底（Trough）**
- 经济达到最低点
- 转折点
- 开始进入扩张

**图形分析：**

```
GDP
  ↑
  │      峰值
  │     /\
  │    /  \
  │   /    \    扩张
  │  /      \  /
  │ /        \/
  │/       谷底
  └─────────→ 时间
    衰退
```

### 8.1.4 案例研究：美国经济周期

**美国衰退（二战后）：**

| 衰退 | 开始 | 结束 | 持续（月） | GDP下降（%） |
|------|------|------|-----------|-------------|
| 1973-75 | 1973年11月 | 1975年3月 | 16 | 3.2 |
| 1980 | 1980年1月 | 1980年7月 | 6 | 2.2 |
| 1981-82 | 1981年7月 | 1982年11月 | 16 | 2.7 |
| 1990-91 | 1990年7月 | 1991年3月 | 8 | 1.4 |
| 2001 | 2001年3月 | 2001年11月 | 8 | 0.5 |
| 2007-09 | 2007年12月 | 2009年6月 | 18 | 4.3 |
| 2020 | 2020年2月 | 2020年4月 | 2 | 5.0 |

**观察：**
- 衰退持续时间：6-18个月
- 2007-09年：大衰退，持续时间长，下降幅度大
- 2020年：COVID-19衰退，快速但剧烈

**特征：**

**1. 2008年金融危机**
- 金融系统危机
- 房地产泡沫破灭
- 全球同步衰退

**2. 2020年疫情衰退**
- 外部冲击
- 供给侧和需求侧同时冲击
- 快速反弹

## 8.2 时间维度中的经济波动

### 8.2.1 冲击类型

**需求冲击（Demand Shocks）：**

**定义：**
影响总需求的冲击。

**类型：**
- 消费冲击：消费者信心变化
- 投资冲击：投资者预期变化
- 政府支出冲击：财政政策变化
- 货币冲击：货币政策变化
- 净出口冲击：汇率、贸易条件变化

**图形分析：**

```
物价水平 P
  ↑
  │    AS
  │   /
  │  /
  │ /  AD₁ → AD₂
  │/
  └────────→ 产出 Y

需求冲击使AD移动
```

**供给冲击（Supply Shocks）：**

**定义：**
影响总供给的冲击。

**类型：**
- 技术冲击：技术创新或破坏
- 要素价格冲击：石油价格
- 自然灾害：地震、洪水
- 制度冲击：政策变化、战争

**图形分析：**

```
物价水平 P
  ↑
  │    AS₁ → AS₂
  │   ↘
  │    ↘
  │     ↘
  │      ↘ AD
  └────────→ 产出 Y

供给冲击使AS移动
```

### 8.2.2 传导机制

**传导渠道：**

**1. 乘数效应**
- 初始冲击被放大
- 消费、投资相互作用

**2. 加速效应**
- 产出变化影响投资
- 投资进一步影响产出

**3. 预期效应**
- 预期影响当前行为
- 自我实现

**4. 金融渠道**
- 信贷条件变化
- 资产价格变化

**5. 国际渠道**
- 汇率变化
- 贸易变化

### 8.2.3 波动的持续性

**为什么波动会持续？**

**1. 调整成本**
- 雇佣和解雇成本
- 投资的不可逆性

**2. 合同刚性**
- 长期合同
- 工资和价格粘性

**3. 金融摩擦**
- 信贷约束
- 资产负债表效应

**4. 预期缓慢调整**
- 信息不完全
- 学习过程

**5. 政策时滞**
- 认识时滞
- 决策时滞
- 效果时滞

### 8.2.4 案例研究：石油价格冲击

**1970年代石油危机：**

**背景：**
- 1973年：石油输出国组织（OPEC）石油禁运
- 1979年：伊朗革命

**影响：**

**1. 供给冲击**
- 石油价格上涨400%
- 生产成本上升
- 总供给左移

**2. 滞胀**
- 产出下降（衰退）
- 物价上升（通胀）
- 失业和通胀同时高

**3. 政策困境**
- 刺激需求 → 通胀加剧
- 紧缩需求 → 衰退加剧

**图形分析：滞胀**

```
物价水平 P
  ↑
  │    AS₁ → AS₂
  │   ↘
  │    ↘
  │     ↘ P₂
  │      ↘
  │       \ AD
  │        \
  └─────────→ 产出 Y
          Y₂ < Y₁

AS左移：P上升，Y下降
```

**应对：**
- 紧缩货币政策（沃尔克）
- 通胀下降
- 但衰退加剧

## 8.3 宏观经济的时间不一致性

### 8.3.1 时间不一致性的定义

**时间不一致性（Time Inconsistency）**是指政策制定者在当前宣布的最优政策，在未来执行时不再是最优的。

**原因：**
- 未来与现在的约束不同
- 预期已经形成
- 事后激励变化

### 8.3.2 例子：通胀偏差

**场景：**

**1. 当前（t=0）**
- 央行宣布零通胀
- 公众相信，预期通胀=0

**2. 未来（t=1）**
- 预期通胀已经形成（0）
- 央行的激励：制造意外通胀
  - 降低实际工资
  - 增加产出
  - 减少失业

**3. 结果**
- 公众预见到央行行为
- 调整预期（通胀>0）
- 通胀偏差（系统性通胀）

**图形分析：**

```
通胀率 π
  ↑
  │    短期菲利普斯曲线
  │   /
  │  /  最优点（π=0, u=u^n）
  │ /
  └────────→ 失业率 u
      u^n

央行事后最优：π>0
均衡：π>0, u=u^n
```

### 8.3.3 解决方案

**1. 委托（Delegation）**
- 将政策委托给保守的央行
- 央行偏好低通胀

**2. 规则（Rules）**
- 明确的政策规则
- 限制自由裁量
- 例如：泰勒规则

**3. 声誉（Reputation）**
- 建立可信度
- 违背声誉的代价
- 重复博弈

**4. 通胀目标制（Inflation Targeting）**
- 明确通胀目标
- 透明度和问责制
- 锚定预期

### 8.3.4 案例研究：美联储的政策框架

**历史演变：**

**1. 1970年代**
- 通胀高企
- 信誉缺失
- 预期脱锚

**2. 1980年代**
- 沃尔克紧缩
- 重建信誉
- 预期重新锚定

**3. 1990年代**
- 通胀目标（隐性）
- 透明度提高
- 预期稳定

**4. 2012年**
- 明确2%通胀目标
- 沟通改善
- 预期管理

**5. 2020年**
- 平均通胀目标制
- 允许通胀暂时高于目标
- 更灵活

**启示：**
- 信誉和预期重要
- 明确目标有助于锚定预期
- 政策框架需要适应新情况

## 8.4 总需求和总供给

### 8.4.1 总需求

**总需求（Aggregate Demand, AD）**是在给定价格水平下，对商品和服务的总需求。

**推导：**

从IS-LM模型：
```
Y = C(Y - T) + I(r) + G + NX(e)
M/P = L(r, Y)
```

**AD曲线：**
```
Y = Y(M/P, G, T, ...)
```

**性质：**
- 向下倾斜
- P上升 → M/P下降 → 利率上升 → 投资下降 → Y下降

**图形分析：**

```
物价水平 P
  ↑
  │
  │      AD
  │     /
  │    /
  │   /
  │  /
  └────────→ 产出 Y

向下倾斜：P与Y负相关
```

**AD移动：**

**右移因素：**
- 货币供给增加（M上升）
- 政府支出增加（G上升）
- 税收减少（T下降）
- 消费者信心上升
- 净出口增加

**左移因素：**
- 货币供给减少
- 政府支出减少
- 税收增加
- 消费者信心下降
- 净出口减少

### 8.4.2 总供给

**总供给（Aggregate Supply, AS）**是在给定价格水平下，企业愿意提供的商品和服务的总量。

**长期总供给（Long-Run AS, LRAS）：**

**定义：**
在长期，产出等于潜在产出，不受价格水平影响。

**性质：**
- 垂直曲线
- Y = Y*（潜在产出）

**图形分析：**

```
物价水平 P
  ↑
  │    LRAS
  │    |
  │    |
  │    |
  └────┼────→ 产出 Y
        Y*

垂直：Y=Y*，不受P影响
```

**短期总供给（Short-Run AS, SRAS）：**

**定义：**
在短期，价格水平上升导致产出增加。

**性质：**
- 向上倾斜
- 价格粘性

**图形分析：**

```
物价水平 P
  ↑
  │      SRAS
  │     /
  │    /
  │   /
  └────────→ 产出 Y

向上倾斜：P与Y正相关
```

**SRAS移动：**

**右移因素：**
- 技术进步
- 要素价格下降
- 预期通胀下降
- 有利供给冲击

**左移因素：**
- 技术退步
- 要素价格上升
- 预期通胀上升
- 不利供给冲击

### 8.4.3 AD-AS模型

**均衡：**
```
AD = SRAS = LRAS
```

**图形分析：**

```
物价水平 P
  ↑
  │    LRAS  SRAS
  │    |    /
  │    |   /
  │    |  /  *
  │    | /  /
  └────┼──┼─→ 产出 Y
         Y*  均衡
```

**短期均衡：**
- AD与SRAS交点
- 可能偏离Y*

**长期均衡：**
- AD、SRAS、LRAS三线交点
- Y = Y*

**调整过程：**

**如果 Y < Y*（衰退）：**
- 资源闲置
- 工资和价格下降
- SRAS右移
- 回到Y*

**如果 Y > Y*（过热）：**
- 资源紧张
- 工资和价格上升
- SRAS左移
- 回到Y*

### 8.4.4 需求冲击的影响

**需求冲击（AD右移）：**

**图形分析：**

```
物价水平 P
  ↑
  │    LRAS  SRAS₁
  │    |    /
  │    |   /
  │    |  /
  │    | /
  └────┼──┼───────→ 产出 Y
         Y*  Y₁
              \
               SRAS₂
                \
                 \
                  \
  短期：P上升，Y上升
  长期：P上升，Y回到Y*
```

**影响：**
- 短期：P上升，Y上升
- 长期：P上升，Y回到Y*
- 货币中性在长期成立

### 8.4.5 供给冲击的影响

**供给冲击（SRAS左移）：**

**图形分析：**

```
物价水平 P
  ↑
  │    LRAS
  │    |    SRAS₂
  │    |   /
  │    |  /
  │    | /
  └────┼──┼───────→ 产出 Y
         Y*  Y₁
          \
           SRAS₁
            \
             \
              \
  短期：P上升，Y下降（滞胀）
  长期：P下降，Y回到Y*
```

**影响：**
- 短期：P上升，Y下降（滞胀）
- 长期：P下降，Y回到Y*
- 政策困境

### 8.4.6 案例研究：2008年金融危机

**冲击：**

**1. 供给冲击**
- 金融危机
- 信贷紧缩
- SRAS左移

**2. 需求冲击**
- 消费者信心下降
- 投资下降
- AD左移

**图形分析：**

```
物价水平 P
  ↑
  │    LRAS
  │    |
  │    |
  │    |   SRAS₂
  │    |  /
  │    | /
  └────┼─┼───────→ 产出 Y
         Y*  Y₁
          \
           AD₁
            \
             \
              \
  AD和SRAS同时左移：Y大幅下降，P不确定
```

**政策应对：**

**1. 财政政策**
- 刺激需求
- AD右移

**2. 货币政策**
- 降息
- 量化宽松
- AD右移

**结果：**
- 避免了更严重的衰退
- 但复苏缓慢
- 政策空间有限

## 本章总结

### 核心要点

1. **经济波动的事实**
   - 不规则、非周期性
   - 大多数变量协动
   - 投资波动最大
   - 货币和价格顺周期
   - 失业率逆周期

2. **经济波动的阶段**
   - 扩张、峰值、衰退、谷底
   - 非规律性
   - 持续性

3. **时间不一致性**
   - 政策的事后最优与事前承诺不同
   - 通胀偏差
   - 解决方案：委托、规则、声誉、通胀目标制

4. **AD-AS模型**
   - AD向下倾斜
   - SRAS向上倾斜
   - LRAS垂直
   - 长期均衡在潜在产出

5. **稳定化政策**
   - 财政政策和货币政策
   - 应对需求冲击和供给冲击
   - 政策时滞和权衡

### 思考题

**理解性问题：**
1. 经济波动有哪些基本事实？
