# 第2章：内生增长模型

> 本章学习目标：
> - 理解内生增长理论的基本思想
> - 掌握AK模型
> - 学会分析研发模型

---

## 2.1 内生增长理论概述

### 2.1.1 索洛模型的局限性

**索洛模型的局限性（Limitations of Solow Model）**：

1. **技术进步外生（Exogenous Technological Progress）**：
   - 索洛模型假设技术进步是外生的
   - 无法解释技术进步的来源和机制

2. **长期经济增长取决于外生参数（Long-Run Growth Depends on Exogenous Parameters）**：
   - 长期经济增长率取决于g（技术进步率）
   - 无法通过政策影响长期经济增长率

3. **条件收敛（Conditional Convergence）**：
   - 索洛模型预测条件收敛
   - 但经验证据显示某些国家持续高速增长

### 2.1.2 内生增长理论的基本思想

**内生增长理论（Endogenous Growth Theory）**：
将技术进步内生化，解释长期经济增长的来源。

**基本思想**：
- 知识积累（Knowledge Accumulation）
- 人力资本积累（Human Capital Accumulation）
- 研发投资（R&D Investment）

**代表人物**：
- Paul Romer（保罗·罗默）
- Robert Lucas（罗伯特·卢卡斯）

---

## 2.2 AK模型

### 2.2.1 AK模型的设定

**AK模型（AK Model）**：
生产函数对资本（包括物质资本和人力资本）的规模报酬不变。

**生产函数**：
```
Y(t) = AK(t)
```
**其中**：
- A：技术水平
- K(t)：总资本（包括物质资本和人力资本）

**资本积累方程**：
```
K̇(t) = sY(t) - δK(t)
```

### 2.2.2 AK模型的动态

**AK模型的动态（Dynamics of AK Model）**：

**资本增长率**：
```
g_K = K̇/K = sA - δ
```

**产出增长率**：
```
g_Y = Ẏ/Y = sA - δ
```

**结论**：
- 长期经济增长率 = sA - δ
- 长期经济增长率是内生的，取决于储蓄率s和技术水平A
- 不存在收敛性（因为边际报酬不递减）

### 2.2.3 AK模型的政策含义

**AK模型的政策含义（Policy Implications of AK Model）**：

1. **提高储蓄率可以提高长期经济增长率**：
   - 储蓄率s上升 ⇒ 长期经济增长率上升

2. **提高技术水平可以提高长期经济增长率**：
   - 技术水平A上升 ⇒ 长期经济增长率上升

3. **不存在条件收敛**：
   - 经济不会自动收敛到平衡增长路径

## 2.3 研发模型

### 2.3.1 研发模型的基本设定

**研发模型（R&D Model）**：
通过研发投资实现技术进步，从而推动经济增长。

**生产函数**：
```
Y(t) = A(t)·F(K(t), L(t))
```
**其中**：
- A(t)：技术水平
- F(·)：生产函数

**技术进步方程**：
```
Ȧ(t) = γ·K_A(t)^β·A(t)^θ·L_A(t)^λ
```
**其中**：
- K_A(t)：研发资本
- L_A(t)：研发劳动力
- γ：研发效率
- β, θ, λ：参数

### 2.3.2 研发模型的平衡增长路径

**研发模型的平衡增长路径（BGP of R&D Model）**：

**技术进步率**：
```
g_A = Ȧ/A = γ·(K_A/A)^β·L_A^λ
```

**平衡增长路径**：
- 技术进步率恒定
- 产出增长率恒定

**平衡增长路径上的增长率**：
```
g_Y = g_A + αg_K + (1 - α)g_L
```
**其中**：
- α：资本产出弹性

### 2.3.3 研发模型的政策含义

**研发模型的政策含义（Policy Implications of R&D Model）**：

1. **增加研发投资可以提高技术进步率**：
   - 研发投资增加 ⇒ 技术进步率上升 ⇒ 经济增长率上升

2. **知识产权保护可以激励研发**：
   - 知识产权保护 ⇒ 研发激励增强 ⇒ 技术进步率上升

3. **人力资本积累可以提高研发效率**：
   - 人力资本积累 ⇒ 研发效率上升 ⇒ 技术进步率上升

## 2.4 人力资本模型

### 2.4.1 人力资本模型的基本设定

**人力资本模型（Human Capital Model）**：
人力资本积累推动经济增长。

**生产函数**：
```
Y(t) = K(t)^α [H(t)L(t)]^(1-α)
```
**其中**：
- K(t)：物质资本
- H(t)：人均人力资本
- L(t)：劳动力

**人力资本积累方程**：
```
Ḣ(t) = B·[1 - u(t)]·H(t) - δ_H·H(t)
```
**其中**：
- B：人力资本积累效率
- u(t)：用于生产的时间比例
- 1 - u(t)：用于人力资本积累的时间比例
- δ_H：人力资本折旧率

### 2.4.2 人力资本模型的平衡增长路径

**人力资本模型的平衡增长路径（BGP of Human Capital Model）**：

**人力资本增长率**：
```
g_H = Ḣ/H = B(1 - u) - δ_H
```

**平衡增长路径**：
- 人力资本增长率恒定
- 产出增长率恒定

**平衡增长路径上的增长率**：
```
g_Y = αg_K + (1 - α)(g_H + g_L)
```

### 2.4.3 人力资本模型的政策含义

**人力资本模型的政策含义（Policy Implications of Human Capital Model）**：

1. **教育投资可以提高人力资本积累**：
   - 教育投资增加 ⇒ 人力资本积累 ⇒ 经济增长率上升

2. **时间分配可以影响人力资本积累**：
   - 增加人力资本积累时间 ⇒ 人力资本增长率上升 ⇒ 经济增长率上升

3. **人力资本外溢效应**：
   - 人力资本积累具有正外部性
   - 整体人力资本水平提高 ⇒ 个人生产率提高

## 2.5 本章总结

### 核心要点

1. **内生增长理论概述**：
   - 索洛模型的局限性：技术进步外生、长期经济增长取决于外生参数、条件收敛
   - 内生增长理论的基本思想：知识积累、人力资本积累、研发投资

2. **AK模型**：
   - 生产函数：Y = AK
   - 长期经济增长率：g = sA - δ
   - 政策含义：提高储蓄率和技术水平可以提高长期经济增长率

3. **研发模型**：
   - 技术进步方程：Ȧ = γ·K_A^β·A^θ·L_A^λ
   - 技术进步率：g_A = γ·(K_A/A)^β·L_A^λ
   - 政策含义：增加研发投资、知识产权保护、人力资本积累可以提高技术进步率

4. **人力资本模型**：
   - 生产函数：Y = K^α(HL)^(1-α)
   - 人力资本积累方程：Ḣ = B(1 - u)H - δ_HH
   - 人力资本增长率：g_H = B(1 - u) - δ_H
   - 政策含义：教育投资、时间分配、人力资本外溢效应
