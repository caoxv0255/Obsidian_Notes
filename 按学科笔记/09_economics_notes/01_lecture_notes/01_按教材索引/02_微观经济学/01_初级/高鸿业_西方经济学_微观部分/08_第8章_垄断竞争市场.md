# 第8章：垄断竞争市场

> 本章学习目标：
> - 理解垄断竞争市场的特征
> - 掌握垄断竞争厂商的短期均衡和长期均衡
> - 学会分析垄断竞争的效率
> - 理解非价格竞争

---

## 8.1 垄断竞争市场的特征

### 8.1.1 垄断竞争市场的定义

**垄断竞争市场（Monopolistic Competition）**：
一种既有垄断又有竞争的市场结构。

### 8.1.2 垄断竞争市场的特征

**垄断竞争市场的特征**：

1. **许多卖者（Many Sellers）**：
   - 市场上有很多卖者
   - 每个卖者只占市场的很小份额

2. **产品差异（Product Differentiation）**：
   - 每个卖者提供略有不同的产品
   - 产品有差异但可以相互替代

3. **自由进入和退出（Free Entry and Exit）**：
   - 企业可以自由进入或退出市场
   - 没有进入或退出障碍

4. **非价格竞争（Non-Price Competition）**：
   - 厂商通过广告、品牌等方式竞争
   - 不只是通过价格竞争

5. **部分价格控制权（Limited Price Control）**：
   - 厂商有一定的价格控制权
   - 但价格控制权有限

## 8.2 垄断竞争厂商的需求曲线

### 8.2.1 垄断竞争厂商的需求曲线

**垄断竞争厂商的需求曲线（Demand Curve Facing the Firm）**：
垄断竞争厂商面临的需求曲线是向右下方倾斜的。

**原因**：
- 产品有差异
- 厂商有一定的价格控制权

### 8.2.2 垄断竞争厂商的收益曲线

**收益曲线**：

1. **总收益（Total Revenue, TR）**：
   - TR = P × Q

2. **平均收益（Average Revenue, AR）**：
   - AR = TR / Q = P
   - 平均收益等于价格

3. **边际收益（Marginal Revenue, MR）**：
   - MR = ΔTR / ΔQ
   - 边际收益小于价格

## 8.3 垄断竞争厂商的短期均衡

### 8.3.1 利润最大化条件

**利润最大化条件（Profit Maximization Condition）**：
边际收益 = 边际成本

**数学推导**：

**利润函数**：
```
π(Q) = TR(Q) - TC(Q)
```

**利润最大化的一阶条件**：
```
dπ/dQ = dTR/dQ - dTC/dQ = 0
MR - MC = 0
MR = MC
```

### 8.3.2 垄断竞争厂商的短期均衡

**图形表示**：


**分析**：

**垄断竞争厂商的决策**：
1. 根据MR = MC确定产量Q*
2. 根据需求曲线确定价格P*
3. 计算利润：π = TR - TC

**垄断竞争厂商的利润**：
- 如果 P > ATC，垄断竞争厂商获得正利润
- 如果 P = ATC，垄断竞争厂商获得零利润
- 如果 P < ATC，垄断竞争厂商获得负利润

## 8.4 垄断竞争厂商的长期均衡

### 8.4.1 进入与退出

**进入（Entry）**：
如果短期利润为正，新厂商进入市场。

**进入条件**：
- 如果 P > ATC，厂商获得正利润
- 新厂商进入
- 每个厂商的需求曲线向左移动
- 利润减少
- 直到 P = ATC

**退出（Exit）**：
如果短期利润为负，厂商退出市场。

**退出条件**：
- 如果 P < ATC，厂商获得负利润
- 厂商退出
- 每个厂商的需求曲线向右移动
- 利润增加
- 直到 P = ATC

### 8.4.2 长期均衡

**长期均衡（Long-Run Equilibrium）**：
- MR = LMC
- P = LATC
- 厂商获得零利润

**分析**：

**垄断竞争厂商的长期决策**：
1. 根据MR = LMC确定产量Q*
2. 根据需求曲线确定价格P*
3. 计算利润：π = TR - LTC

**与完全竞争的区别**：
- 完全竞争：P = min(ATC)，生产效率
- 垄断竞争：P > min(ATC)，生产无效率

**与垄断的区别**：
- 垄断：P > ATC，正利润
- 垄断竞争：P = ATC，零利润

## 8.5 垄断竞争的效率

### 8.5.1 配置效率损失

**配置效率损失（Allocative Inefficiency）**：
垄断竞争导致配置效率损失。

**原因**：
- 垄断竞争价格高于完全竞争价格
- 垄断竞争产量低于完全竞争产量
- 存在无谓损失

**图形表示**：

```plotly

  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]

    line:
      color: "#1f77b4"
      width: 4
      shape: spline

  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "数量Q"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：

**完全竞争均衡**：
- 价格：Pc
- 产量：Qc
- 总剩余最大

**垄断竞争均衡**：
- 价格：Pm > Pc
- 产量：Qm < Qc
- 总剩余减少

**无谓损失（Deadweight Loss, DWL）**：
- 垄断竞争导致的无谓损失
- 由于产量不足造成

### 8.5.2 生产效率损失

**生产效率损失（Productive Inefficiency）**：
垄断竞争厂商不以最低成本生产。

**原因**：
- P > min(ATC)
- 厂商不以最低成本生产

### 8.5.3 多样性与效率的权衡

**多样性与效率的权衡（Diversity-Efficiency Tradeoff）**：
垄断竞争提供了产品多样性，但损失了效率。

**多样性**：
- 产品多样性增加了消费者选择
- 消费者可以选择符合自己偏好的产品

**效率损失**：
- 配置效率损失
- 生产效率损失

**权衡**：
- 多样性的收益可能超过效率损失
- 社会可能偏好产品多样性

## 8.6 非价格竞争

### 8.6.1 非价格竞争的定义

**非价格竞争（Non-Price Competition）**：
厂商通过广告、品牌、产品差异等方式竞争，不只是通过价格竞争。

### 8.6.2 非价格竞争的类型

**非价格竞争的类型**：

**1. 广告（Advertising）**：
- 通过广告提高品牌知名度
- 通过广告吸引消费者

**2. 品牌（Branding）**：
- 通过品牌建立消费者忠诚度
- 通过品牌提高产品差异化

**3. 产品差异（Product Differentiation）**：
- 通过产品差异吸引消费者
- 例如：功能、设计、包装

**4. 服务（Service）**：
- 通过服务提高产品价值
- 例如：售后服务、送货

### 8.6.3 广告的效果

**广告的效果**：

**正面效果**：
- 提供信息
- 减少信息不对称
- 促进竞争

**负面效果**：
- 增加成本
- 可能误导消费者
- 可能形成进入障碍

## 8.7 本章总结

### 核心要点

1. **垄断竞争市场的特征**：
   - 许多卖者
   - 产品差异
   - 自由进入和退出
   - 非价格竞争
   - 部分价格控制权

2. **垄断竞争厂商的需求曲线**：
   - 垄断竞争厂商面临的需求曲线是向右下方倾斜的
   - 边际收益小于价格

3. **垄断竞争厂商的短期均衡**：
   - 利润最大化条件：MR = MC
   - 垄断竞争厂商可能获得正利润、零利润或负利润

4. **垄断竞争厂商的长期均衡**：
   - MR = LMC
   - P = LATC
   - 厂商获得零利润

5. **垄断竞争的效率**：
   - 配置效率损失：垄断竞争价格高于完全竞争价格，垄断竞争产量低于完全竞争产量，存在无谓损失
   - 生产效率损失：垄断竞争厂商不以最低成本生产
   - 多样性与效率的权衡：垄断竞争提供了产品多样性，但损失了效率

6. **非价格竞争**：
   - 广告
   - 品牌
   - 产品差异
   - 服务
