- 无谓损失随税收规模的增加而增加
- 无谓损失的增加速度比税收收入的增加速度快
- 原因：无谓损失是税收规模的二次函数

**图形**：

```plotly

    title:

  yaxis:
    title:

    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**数学证明**：

**假设**：
- 需求函数：Qd = a - bP
- 供给函数：Qs = c + dP

**均衡**：
```
a - bP = c + dP
P* = (a - c) / (b + d)
Q* = a - b × (a - c) / (b + d)
```

**税收**：
```
Qd = a - b(P + t)
Qs = c + dP
a - b(P + t) = c + dP
a - bP - bt = c + dP
(a - c - bt) / (b + d) = P
Q = c + d × (a - c - bt) / (b + d)
```

**无谓损失**：
```
DWL = 0.5 × t × (Q* - Q)
    = 0.5 × t × [Q* - (c + d × (a - c - bt) / (b + d))]
    = 0.5 × t × [(a - c) / (b + d) × (b + d) - (c × (b + d) + d × (a - c - bt)) / (b + d)]
    = 0.5 × t × [(ab - cb + ad - cd - cb - cd - da + cd + dbt) / (b + d)]
    = 0.5 × t × [(ab - 2cb + ad - 2cd + dbt) / (b + d)]
```

**简化**：
```
DWL = 0.5 × t × [bt / (b + d)]
    = 0.5 × b × t² / (b + d)
```

**结论**：
- 无谓损失是税收规模的二次函数
- 税收规模增加1倍，无谓损失增加4倍

- 拉弗曲线：先上升后下降

### 8.4.3 拉弗曲线的解释

**拉弗曲线的解释**：

**情况1：税率很低**：
- 提高税率 → 税收收入增加
- 原因：税基减少的幅度小于税率提高的幅度

**情况2：税率很高**：
- 提高税率 → 税收收入减少
- 原因：税基减少的幅度大于税率提高的幅度
- 避税和逃税增加

**最优税率**：
- 税收收入最大化的税率
- 位于拉弗曲线的顶点

### 8.4.4 拉弗曲线的应用

**应用1：税收政策**

**问题**：政府应该设定多高的税率？

**分析**：
- 如果税率低于最优税率，提高税率可以增加税收收入
- 如果税率高于最优税率，降低税率可以增加税收收入

**应用2：供给侧经济学**

**观点**：
- 降低税率可以增加税收收入
- 原因：刺激经济活动，扩大税基

**争议**：
- 最优税率难以确定
- 短期和长期效应不同

## 8.5 最优税收理论

### 8.5.1 最优税收的定义

**最优税收（Optimal Taxation）**：
在给定的税收收入目标下，使无谓损失最小的税收制度。

### 8.5.2 拉姆齐法则

**拉姆齐法则（Ramsey Rule）**：
为了使无谓损失最小，对需求弹性不同的商品应该征收不同的税率。

**拉姆齐法则的表述**：
```
税率 / (1 - 税率) = 常数 / 需求价格弹性
```

**含义**：
- 需求弹性大的商品，税率应该低
- 需求弹性小的商品，税率应该高

**例子**：
- 香烟（需求弹性小）→ 高税率
- 汽车（需求弹性大）→ 低税率

### 8.5.3 等价边际超额负担

**等价边际超额负担（Equal Marginal Excess Burden）**：
为了使总无谓损失最小，每种商品的边际超额负担应该相等。

**边际超额负担（Marginal Excess Burden）**：
额外征收1单位税收导致的额外无谓损失。

**条件**：
```
MEB1 = MEB2 = ... = MEBn
```

其中：
- MEBi：第i种商品的边际超额负担

### 8.5.4 税收公平

**税收公平（Tax Equity）**：

**纵向公平（Vertical Equity）**：
收入高的人应该多纳税。

**横向公平（Horizontal Equity）**：
收入相同的人应该缴纳相同的税。

**税收公平与效率的权衡**：
- 拉姆齐法则：效率
- 累进所得税：公平
- 两者可能冲突

## 8.6 税收的实际应用

### 8.6.1 香烟税

**香烟税的效果**：

**分析**：
- 香烟需求缺乏弹性
- 香烟税导致价格上升
- 买者承担大部分税负
- 数量减少较少
- 无谓损失较小

**政策建议**：
- 对香烟征收高税率
- 原因：
  1. 香烟有负外部性（健康损害）
  2. 香烟需求缺乏弹性，税收收入高
  3. 无谓损失小

### 8.6.2 汽油税

**汽油税的效果**：

**分析**：
- 汽油需求缺乏弹性
- 汽油税导致价格上升
- 买者承担大部分税负
- 数量减少较少
- 无谓损失较小

**政策建议**：
- 对汽油征收高税率
- 原因：
  1. 汽油有负外部性（污染）
  2. 汽油需求缺乏弹性，税收收入高
  3. 无谓损失小

### 8.6.3 奢侈品税

**奢侈品税的效果**：

**分析**：
- 奢侈品需求富有弹性
- 奢侈品税导致价格上升
- 卖者承担大部分税负
- 数量减少较多
- 无谓损失较大

**政策建议**：
- 对奢侈品征收低税率
- 原因：
  1. 奢侈品需求富有弹性，税收收入低
  2. 无谓损失大

### 8.6.4 所得税

**所得税的效果**：

**分析**：
- 劳动供给弹性中等
- 所得税导致工资下降
- 劳动供给减少
- 无谓损失中等

**政策建议**：
- 对收入高的人征收高税率
- 原因：
  1. 纵向公平
  2. 劳动供给弹性可能较小

## 8.7 本章总结

### 核心要点

1. **税收与市场**：
   - 税收导致价格上升，数量减少
   - 税负归宿取决于弹性
   - 弹性小的一方承担大部分税负

2. **税收的福利分析**：
   - 税收导致消费者剩余减少
   - 税收导致生产者剩余减少
   - 政府获得税收收入
   - 税收导致无谓损失

3. **无谓损失的决定因素**：
   - 弹性越大，无谓损失越大
   - 税收规模越大，无谓损失越大
   - 无谓损失是税收规模的二次函数

4. **拉弗曲线**：
   - 表示税收收入与税率的关系
   - 税率很低时，提高税率增加税收收入
   - 税率很高时，提高税率减少税收收入
   - 存在最优税率

5. **最优税收理论**：
   - 拉姆齐法则：需求弹性大的商品税率低
   - 等价边际超额负担：边际超额负担相等
   - 税收公平与效率的权衡

| 税收收入 | 税收 × 数量 | 政府的收入 |
| 无谓损失 | 由于税收导致的市场低效率造成的福利损失 | 取决于弹性和税收规模 |
| 拉弗曲线 | 表示税收收入与税率之间关系的曲线 | 税率与税收收入的关系 |
| 最优税率 | 税收收入最大化的税率 | 位于拉弗曲线的顶点 |
| 最优税收 | 在给定的税收收入目标下，使无谓损失最小的税收制度 | 效率与公平的权衡 |
| 拉姆齐法则 | 为了使无谓损失最小，对需求弹性不同的商品应该征收不同的税率 | 需求弹性大的商品税率低 |
| 等价边际超额负担 | 为了使总无谓损失最小，每种商品的边际超额负担应该相等 | 边际超额负担相等 |
| 纵向公平 | 收入高的人应该多纳税 | 公平 |
| 横向公平 | 收入相同的人应该缴纳相同的税 | 公平 |
