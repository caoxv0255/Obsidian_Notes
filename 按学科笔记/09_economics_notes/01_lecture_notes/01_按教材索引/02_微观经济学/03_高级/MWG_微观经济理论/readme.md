# 微观经济理论 - MWG

## 教材信息

- **书名**：微观经济理论（Microeconomic Theory）
- **作者**：安德鲁·马斯-克莱尔、迈克尔·温斯顿、杰里·格林
- **出版社**：牛津大学出版社
- **类别**：高级微观经济学
- **章节**：21章

## 内容简介

本书是国际公认的高级微观经济学权威教材，共21章。本书全面系统地涵盖了微观经济学的核心内容，包括消费者理论、生产者理论、不确定性、一般均衡、博弈论、信息经济学、机制设计等。本书数学严谨，适合经济学专业研究生和博士生学习。
