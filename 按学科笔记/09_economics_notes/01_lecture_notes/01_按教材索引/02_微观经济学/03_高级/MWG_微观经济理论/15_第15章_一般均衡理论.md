## 15.1 纯交换经济

### 15.1.1 纯交换经济的定义

**纯交换经济（Pure Exchange Economy）**：
E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ}
- Xᵏ：消费者k的消费集
- ≿ᵏ：消费者k的偏好关系
- ωᵏ：消费者k的禀赋

### 15.1.2 纯交换经济的均衡

**纯交换经济的均衡（Equilibrium of Pure Exchange Economy）**：
价格向量p和配置{xᵏ}ᵏ，使得：
1. 消费者最大化效用：xᵏ ∈ xᵏ(p, p·ωᵏ)
2. 市场出清：Σₖ xᵏ = Σₖ ωᵏ

**其中**：
- p：价格向量
- xᵏ(p, p·ωᵏ)：消费者k的需求
- ωᵏ：消费者k的禀赋

### 15.1.3 超额需求函数

**超额需求函数（Excess Demand Function, Z(p)）**：
```
Z(p) = Σₖ xᵏ(p, p·ωᵏ) - Σₖ ωᵏ
```
**其中**：
- Z(p)：超额需求函数
- xᵏ(p, p·ωᵏ)：消费者k的需求
- ωᵏ：消费者k的禀赋

**性质**：
1. **零次齐次性（Homogeneity of Degree Zero）**：
   - Z(αp) = Z(p)，对于所有α > 0

2. **瓦尔拉斯法则（Walras' Law）**：
   - p·Z(p) = 0

3. **连续性（Continuity）**：
   - Z(p)是连续的

### 15.1.4 均衡的存在性

**均衡的存在性（Existence of Equilibrium）**：

**定理（Arrow-Debreu）**：
如果满足以下条件，则纯交换经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 禀赋严格为正

**证明思路**：
1. 构造超额需求函数Z(p)
2. 证明Z(p)连续
3. 应用不动点定理
4. 证明存在p*，使得Z(p*) = 0

## 15.2 生产经济

### 15.2.1 生产经济的定义

**生产经济（Production Economy）**：
E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ, (Yⱼ)ⱼ₌₁ᴶ}
- Xᵏ：消费者k的消费集
- ≿ᵏ：消费者k的偏好关系
- ωᵏ：消费者k的禀赋
- Yⱼ：企业j的生产集

### 15.2.2 生产经济的均衡

**生产经济的均衡（Equilibrium of Production Economy）**：
价格向量p和配置{xᵏ, yⱼ}ᵏⱼ，使得：
1. 消费者最大化效用：xᵏ ∈ xᵏ(p, p·ωᵏ + Σⱼ θⱼₖ·p·yⱼ)
2. 生产者最大化利润：yⱼ ∈ yⱼ(p)
3. 市场出清：Σₖ xᵏ = Σⱼ yⱼ + Σₖ ωᵏ

**其中**：
- p：价格向量
- xᵏ：消费者k的消费
- yⱼ：企业j的生产
- ωᵏ：消费者k的禀赋
- θⱼₖ：消费者k在企业j中的股权

### 15.2.3 生产集的性质

**生产集的性质（Properties of Production Set）**：

1. **闭集（Closedness）**：
   - Y是闭集

2. **凸集（Convexity）**：
   - Y是凸集（规模报酬不递增）

3. **包含零（Contains Zero）**：
   - 0 ∈ Y（可以不生产）

4. **无免费午餐（No Free Lunch）**：
   - Y ∩ (-Y) = {0}

5. **不可逆性（Irreversibility）**：
   - 如果y ∈ Y且y ≠ 0，则-y ∉ Y

6. **自由处置（Free Disposal）**：
   - 如果y ∈ Y且y' ≤ y，则y' ∈ Y

### 15.2.4 生产经济的均衡存在性

**生产经济的均衡存在性（Existence of Equilibrium in Production Economy）**：

**定理**：
如果满足以下条件，则生产经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 生产集是凸集、闭集、0 ∈ Y、Y ∩ (-Y) = {0}
4. 禀赋严格为正

**证明思路**：
1. 构造超额需求函数Z(p)
2. 证明Z(p)连续
3. 应用不动点定理
4. 证明存在p*，使得Z(p*) = 0

## 15.3 均衡的存在性

### 15.3.1 不动点定理

**不动点定理（Fixed Point Theorem）**：

**Brouwer不动点定理（Brouwer's Fixed Point Theorem）**：
如果f: S → S是连续的，S是紧凸集，则存在x* ∈ S，使得f(x*) = x*。

**Kakutani不动点定理（Kakutani's Fixed Point Theorem）**：
如果F: S → S是上半连续的、紧凸值的，S是紧凸集，则存在x* ∈ S，使得x* ∈ F(x*)。

### 15.3.2 纯交换经济的存在性

**纯交换经济的存在性（Existence in Pure Exchange Economy）**：

**定理**：
如果满足以下条件，则纯交换经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 禀赋严格为正

**证明思路**：
1. 定义价格空间的映射
2. 证明映射满足不动点定理的条件
3. 应用不动点定理
4. 证明均衡价格存在

### 15.3.3 生产经济的存在性

**生产经济的存在性（Existence in Production Economy）**：

**定理**：
如果满足以下条件，则生产经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 生产集是凸集、闭集、0 ∈ Y、Y ∩ (-Y) = {0}
4. 禀赋严格为正

**证明思路**：
1. 定义价格空间的映射
2. 证明映射满足不动点定理的条件
3. 应用不动点定理
4. 证明均衡价格存在

## 15.4 均衡的唯一性

### 15.4.1 唯一性条件

**唯一性条件（Uniqueness Conditions）**：
- 如果偏好是严格凸的，生产函数是严格凹的，则均衡唯一

**定理**：
如果满足以下条件，则竞争均衡唯一：
1. 偏好是严格凸的（无差异曲线严格凸）
2. 生产函数是严格凹的（规模报酬严格递减）
3. 超额需求函数满足强总替代性（Strong Gross Substitutability）

### 15.4.2 总替代性

**总替代性（Gross Substitutability）**：
商品i的价格上升，对其他商品j的需求增加。

**定义**：
超额需求函数Z(p)满足总替代性，如果对于所有i ≠ j，∂Zⱼ(p)/∂pᵢ > 0。

**强总替代性（Strong Gross Substitutability）**：
对于所有i ≠ j，∂Zⱼ(p)/∂pᵢ > 0，且∂Zᵢ(p)/∂pᵢ < 0。

**定理**：
如果超额需求函数满足强总替代性，则均衡唯一。

### 15.4.3 唯一性的反例

**唯一性的反例（Counterexamples to Uniqueness）**：
- 如果偏好不是严格凸的，则可能存在多个均衡
- 如果生产函数不是严格凹的，则可能存在多个均衡

**例子**：
- 同质商品的市场：可能有多个均衡
- 规模报酬递增的生产：可能有多个均衡

## 15.5 本章总结

### 核心要点

1. **纯交换经济**：
   - 纯交换经济：E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ}
   - 纯交换经济的均衡：价格向量p和配置{xᵏ}ᵏ
   - 超额需求函数：Z(p) = Σₖ xᵏ - Σₖ ωᵏ
   - 均衡的存在性：偏好连续、凸，消费集满足一定条件

2. **生产经济**：
   - 生产经济：E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ, (Yⱼ)ⱼ₌₁ᴶ}
   - 生产经济的均衡：价格向量p和配置{xᵏ, yⱼ}ᵏⱼ
   - 生产集的性质：闭集、凸集、包含零、无免费午餐、不可逆性、自由处置
   - 生产经济的均衡存在性：偏好和生产集满足一定条件

3. **均衡的存在性**：
   - 不动点定理：Brouwer不动点定理、Kakutani不动点定理
   - 纯交换经济的存在性：偏好连续、凸，消费集满足一定条件
   - 生产经济的存在性：偏好和生产集满足一定条件

4. **均衡的唯一性**：
   - 唯一性条件：偏好严格凸，生产函数严格凹
   - 总替代性：∂Zⱼ(p)/∂pᵢ > 0，对于i ≠ j
   - 唯一性的反例：偏好不严格凸，生产函数不严格凹

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 纯交换经济 | E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ} | 没有生产的经济 |

| 零次齐次性 | Z(αp) = Z(p) | 价格标准化 |

| Brouwer不动点定理 | 连续映射在紧凸集上有不动点 | 不动点定理 |
| Kakutani不动点定理 | 上半连续、紧凸值映射在紧凸集上有不动点 | 不动点定理 |
