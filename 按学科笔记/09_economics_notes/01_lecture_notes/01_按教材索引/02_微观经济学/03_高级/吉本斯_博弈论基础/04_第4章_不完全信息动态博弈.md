# 第4章 不完全信息动态博弈

## 章节学习目标

本章主要讨论不完全信息动态博弈的基本理论，学生完成本章学习后应能够：

1. 掌握精炼贝叶斯均衡的定义和性质
2. 理解信念的形成和更新过程
3. 掌握信号博弈的基本概念和均衡类型
4. 理解信息传递的条件和限制
5. 掌握声誉模型的基本原理和应用
6. 能够运用不完全信息动态博弈分析现实中的信息不对称问题

## 4.1 精炼贝叶斯均衡

### 4.1.1 不完全信息动态博弈的特点

**不完全信息动态博弈（Dynamic Game of Incomplete Information）**：
- 参与者对某些信息不完全了解
- 博弈有多个阶段，参与者序贯行动
- 参与者可以根据观察到的行为更新信念

**特点**：
1. 行动可以传递信息
2. 信念可以动态更新
3. 策略是历史和信念的函数
4. 均衡需要满足一致性

### 4.1.2 信念的定义

**信念（Belief）**：
- 参与者在每个信息集中对博弈状态的判断
- 通常是概率分布

**符号**：
- μ(x) 表示参与者在到达节点 x 时，认为博弈处于节点 x 的概率
- 对于信息集 H，μ(h) 表示在信息集 H 上的概率分布

**信念的性质**：
1. 非负性：μ(x) ≥ 0
2. 归一性：Σ_{x∈H} μ(x) = 1
3. 一致性：符合贝叶斯法则（在均衡路径上）

### 4.1.3 贝叶斯法则

**贝叶斯法则（Bayes' Rule）**：
在观察到事件 E 后，对假设 H 的后验概率为：
P(H|E) = P(E|H)P(H) / P(E)

**在博弈中的应用**：
参与者根据观察到的行动更新对对手类型的信念：
P(tᵢ|aᵢ) = P(aᵢ|tᵢ)P(tᵢ) / Σ_{t'ᵢ} P(aᵢ|t'ᵢ)P(t'ᵢ)

**例子**：
假设对手有两种类型：强类型（概率0.4）和弱类型（概率0.6）。
- 强类型选择攻击的概率为0.8
- 弱类型选择攻击的概率为0.3

观察到攻击后：
P(强|攻击) = 0.8 × 0.4 / (0.8 × 0.4 + 0.3 × 0.6) = 0.32 / 0.50 = 0.64

### 4.1.4 精炼贝叶斯均衡的定义

**定义**：精炼贝叶斯均衡（Perfect Bayesian Equilibrium, PBE）由两部分组成：
1. 策略组合 s = (s₁, s₂, ..., sₙ)
2. 信念系统 μ

满足：
1. **序贯理性（Sequential Rationality）**：
   给定信念，每个参与者在每个信息集都选择最优行动
2. **信念一致性（Consistency of Beliefs）**：
   - 在均衡路径上：信念通过贝叶斯法则从策略推导
   - 非均衡路径上：信念是合理的

**理解精炼贝叶斯均衡**：
- 是贝叶斯纳什均衡在动态博弈中的扩展
- 是子博弈完美纳什均衡在不完全信息博弈中的扩展
- 考虑了信念的动态更新

### 4.1.5 精炼贝叶斯均衡的求解

**步骤**：
1. 猜测均衡策略和信念
2. 验证序贯理性：给定信念，策略是否最优
3. 验证信念一致性：信念是否符合贝叶斯法则
4. 调整猜测，直到满足所有条件

**例子：不完全信息进入博弈（动态版）**

**博弈描述**：
企业A（在位者）的成本可能是高成本（t = H，概率0.5）或低成本（t = L，概率0.5）。
企业B（进入者）不知道A的成本。
B先决定是否进入，如果B进入，A决定是否对抗。

**博弈树**：

```
       自然
      /     \
   H(0.5)   L(0.5)
     |       |
     B       B
    / \     / \
  进入 不进入 进入 不进入
   |    |     |    |
   A    |     A    |
  / \   |    / \   |
 容忍 对抗 |  容忍 对抗
```

**求解**：

**步骤1**：猜测策略
假设A的策略：
- 高成本：容忍
- 低成本：对抗

**步骤2**：B的决策
如果B进入，观察到对抗时，认为A是低成本的概率为1。
如果B进入，观察到容忍时，认为A是高成本的概率为1。

B的期望支付：
- 进入：0.5 × 1（A容忍） + 0.5 × (-1)（A对抗） = 0
- 不进入：0

B对两种选择无差异。

**步骤3**：考虑混合策略
假设B以概率p进入。

A为高成本时：
- 容忍：E = p × 1 + (1-p) × 0 = p
- 对抗：E = p × 0 + (1-p) × 2 = 2(1-p)
最优：p = 2/3 时无差异，p > 2/3 时容忍，p < 2/3 时对抗

A为低成本时：
- 容忍：E = p × 1 + (1-p) × 0 = p
- 对抗：E = p × 3 + (1-p) × 2 = 2 + p
最优：总是对抗

**精炼贝叶斯均衡**：
策略：
- A：高成本时容忍，低成本时对抗
- B：以2/3概率进入，1/3概率不进入

信念：
- B进入前：P(A=高) = 0.5，P(A=低) = 0.5
- 如果A对抗：P(A=低) = 1
- 如果A容忍：P(A=高) = 1

### 4.1.6 完美贝叶斯均衡与其他均衡概念

**均衡概念的比较**：

| 均衡概念 | 适用范围 | 要求 |
|:---:|:---:|:---:|
| 纳什均衡 | 完全信息静态博弈 | 互为最佳反应 |
| 子博弈完美纳什均衡 | 完全信息动态博弈 | 每个子博弈都纳什均衡 |
| 贝叶斯纳什均衡 | 不完全信息静态博弈 | 每种类型最佳反应 |
| 精炼贝叶斯均衡 | 不完全信息动态博弈 | 序贯理性+信念一致性 |

**包含关系**：
- 完美贝叶斯均衡 ⇒ 贝叶斯纳什均衡
- 子博弈完美纳什均衡 ⇒ 完美贝叶斯均衡（在完全信息情况下）
- 完美贝叶斯均衡不一定满足所有精炼要求

## 4.2 信号博弈

### 4.2.1 信号博弈的定义

**信号博弈（Signaling Game）**是不完全信息动态博弈的特殊形式，包括：

1. **发送者（Sender）**：
   - 有私人类型 t ∈ T
   - 发送信号 m ∈ M

2. **接收者（Receiver）**：
   - 观察信号 m
   - 选择行动 a ∈ A

3. **自然（Nature）**：
   - 首先选择发送者的类型 t
   - 先验分布 p(t)

4. **支付函数**：
   - u_S(m, a, t)：发送者的支付
   - u_R(m, a, t)：接收者的支付

**博弈的顺序**：
1. 自然选择发送者的类型 t
2. 发送者观察到 t，选择信号 m
3. 接收者观察到 m，更新信念，选择行动 a
4. 双方获得支付

### 4.2.2 信号博弈的精炼贝叶斯均衡

**精炼贝叶斯均衡的组成**：
1. 发送者的策略：m(t): T → M
2. 接收者的策略：a(m): M → A
3. 接收者的信念：μ(t|m): T → [0, 1]

**条件**：
1. **发送者的序贯理性**：
   给定接收者的策略 a(m)，m(t) 最大化 u_S(m, a(m), t)

2. **接收者的序贯理性**：
   给定信念 μ(t|m)，a(m) 最大化 E[u_R(m, a, t)]

3. **信念一致性**：
   如果存在 t' 使得 m(t') = m，则：
   μ(t|m) = p(t) / Σ_{t': m(t')=m} p(t')

### 4.2.3 信号博弈的均衡类型

**均衡类型**：

1. **分离均衡（Separating Equilibrium）**：
   - 不同类型选择不同信号
   - 信号可以完全揭示类型
   - μ(t|m) = 1 或 0

2. **混同均衡（Pooling Equilibrium）**：
   - 所有类型选择相同信号
   - 信号不提供任何信息
   - μ(t|m) = p(t)

3. **半分离均衡（Semi-Separating Equilibrium）**：
   - 某些类型随机选择信号
   - 部分揭示信息
   - 0 < μ(t|m) < 1

### 4.2.4 劳动力市场信号博弈

**Spence (1973) 的模型**：

**博弈描述**：
1. 自然选择工人的能力：高能力（t = H，概率p）或低能力（t = L，概率1-p）
2. 工人选择教育水平 e ∈ ℝ₊
3. 雇主观察到 e，决定工资 w
4. 支付：
   - 工人：u = w - e/t
   - 雇主：u = t - w

**解释**：
- 教育成本与能力成反比：高能力的教育成本低
- 教育作为能力的信号
- 雇主根据教育推断能力，决定工资

### 4.2.5 分离均衡

**假设**：高能力选择 e_H，低能力选择 e_L

**发送者的激励相容约束**：
- 高能力选择 e_H：
  w(e_H) - e_H/H ≥ w(e_L) - e_L/H
- 低能力选择 e_L：
  w(e_L) - e_L/L ≥ w(e_H) - e_H/L

**接收者的最优反应**：
- 观察到 e = e_H，认为 t = H，支付 w = H
- 观察到 e = e_L，认为 t = L，支付 w = L

**代入**：
- 高能力：H - e_H/H ≥ L - e_L/H
- 低能力：L - e_L/L ≥ H - e_H/L

**解**：
整理得：
- 高能力：H - L ≥ (e_H - e_L)/H
- 低能力：L - H ≥ (e_L - e_H)/L

即：
- e_H - e_L ≤ H(H - L)
- e_H - e_L ≥ L(H - L)

因此：
L(H - L) ≤ e_H - e_L ≤ H(H - L)

**分离均衡的条件**：
存在 e_H > e_L 满足上述不等式。

**最小分离均衡**：
选择最小的教育差异：
e_H = L(H - L)
e_L = 0

### 4.2.6 混同均衡

**假设**：高能力和低能力都选择相同的 e*

**信念**：
观察到 e* 时，μ(H|e*) = p，μ(L|e*) = 1-p

**接收者的最优反应**：
w(e*) = pH + (1-p)L

**发送者的激励相容约束**：
- 高能力：pH + (1-p)L - e*/H ≥ w(e') - e'/H，对所有 e' ≠ e*
- 低能力：pH + (1-p)L - e*/L ≥ w(e') - e'/L，对所有 e' ≠ e*

**最偏离的激励**：
考虑分离偏离：
- 高能力不偏离到 e_L = 0：
  pH + (1-p)L - e*/H ≥ L
- 低能力不偏离到 e_H：
  pH + (1-p)L - e*/L ≥ H - e_H/L

假设 e_H 很大，第二式自动满足（低能力不会偏离到高教育）。

**解**：
从第一式：
pH + (1-p)L - e*/H ≥ L
pH - pL ≥ e*/H
p(H - L) ≥ e*/H
e* ≤ pH(H - L)

**混同均衡的条件**：
e* ≤ pH(H - L)

### 4.2.7 均衡的精炼

**直觉标准（Intuitive Criterion）**：
剔除不合理的均衡。

**标准**：
如果某个信号 m 不是某些类型的有利偏离，但接收者会将其视为这些类型发出的，则这个均衡不满足直觉标准。

**应用**：
在劳动力市场中，高教育可能只有高能力会选择，低能力不会选择（即使接收者支付高工资）。因此，某些混同均衡可能被剔除。

## 4.3 信息传递

### 4.3.1 信息传递的条件

**信息传递（Information Transmission）**：
- 通过行动向其他参与者传递私人信息
- 需要满足一定条件才能有效传递

**条件**：
1. **信号的可观察性**：其他参与者能够观察到信号
2. **信号的成本不对称性**：不同类型发送信号的成本不同
3. **激励相容性**：类型愿意通过信号揭示自己
4. **接收者的理性**：接收者能够正确解读信号

### 4.3.2 信号传递的效率

**分离均衡**：
- 信息完全传递
- 类型被完全识别
- 效率最高

**混同均衡**：
- 信息不传递
- 类型不被识别
- 效率最低

**半分离均衡**：
- 部分信息传递
- 类型部分被识别
- 效率居中

### 4.3.3 信号传递的例子

**例子1：企业定价**

**博弈描述**：
企业知道自己的产品质量（高质量或低质量），消费者不知道。
企业选择价格，消费者根据价格推断质量，决定是否购买。

**信号**：
- 高质量：高价格（信号质量）
- 低质量：低价格

**成本不对称**：
- 高质量企业维持高价格的成本低（需求稳定）
- 低质量企业维持高价格的成本高（需求不稳定）

**例子2：投资决策**

**博弈描述**：
企业知道自己的项目前景，投资者不知道。
企业选择投资水平，投资者根据投资推断前景，决定是否投资。

**信号**：
- 高投资：表明信心
- 低投资：表明担忧

**成本不对称**：
- 好项目的投资成本低（回报高）
- 坏项目的投资成本高（回报低）

### 4.3.4 信息泄露

**信息泄露（Information Leakage）**：
- 即使参与者试图隐藏信息，信息仍可能通过行动泄露
- 是不完全信息动态博弈中的重要现象

**例子**：
- 在谈判中，参与者的要价和还价可能泄露其底线
- 在拍卖中，竞拍者的出价可能泄露其估价

**防止信息泄露**：
- 使用随机策略
- 保持行动的一致性
- 避免可预测的模式

## 4.4 声誉模型

### 4.4.1 声誉的定义

**声誉（Reputation）**：
- 参与者在博弈历史中建立的形象
- 影响其他参与者对其类型的信念
- 是不完全信息动态博弈中的重要概念

**声誉的作用**：
1. 传递信息
2. 影响未来博弈
3. 维持合作
4. 建立信任

### 4.4.2 Kreps-Milgrom-Roberts-Wilson (KMRW) 模型

**模型描述**：
两个参与者重复博弈 T 期。
每个参与者有两种类型：
- 理性型（Rational）：最大化自身利益
- 合作型（Cooperative）：总是合作

参与者不知道对方的类型，只知道对方是合作型的概率为 ε。

**结论**：
对于任意有限 T，存在 ε̄ > 0 使得如果 ε < ε̄，在前 T-K 期合作可以维持（K 依赖于 T）。

**解释**：
- 参与者有建立"合作型"声誉的动机
- 即使是理性型也会在前期合作
- 合作在最后几期可能崩溃

### 4.4.3 声誉的形成

**声誉的形成机制**：
1. **初始信念**：基于先验信息
2. **行为观察**：观察参与者的行动
3. **信念更新**：使用贝叶斯法则更新
4. **声誉积累**：持续的行为塑造声誉

**影响声誉形成的因素**：
1. 信息的透明度
2. 博弈的重复次数
3. 对手的理性程度
4. 外部环境的变化

### 4.4.4 声誉的维持

**声誉维持的成本**：
- 需要持续的行为
- 可能需要牺牲短期利益
- 需要抵抗诱惑

**声誉破坏的后果**：
- 信任丧失
- 未来合作困难
- 可能无法挽回

**维持声誉的策略**：
1. 一致性行为
2. 及时回应
3. 透明沟通
4. 承诺履行

### 4.4.5 声誉的应用

**应用1：企业声誉**

**问题**：
企业可能选择欺骗消费者以获得短期利益，但这会破坏声誉。

**均衡**：
- 长期企业会选择诚实以维持声誉
- 新进入企业需要时间建立声誉
- 声誉可以形成进入壁垒

**应用2：政府声誉**

**问题**：
政府可能选择违背承诺以获得短期利益，但这会破坏声誉。

**均衡**：
- 长期政府会选择遵守承诺
- 新政府面临建立或继承声誉的问题
- 声誉影响政策效果

**应用3：个人声誉**

**问题**：
个人可能选择欺骗他人以获得短期利益，但这会破坏声誉。

**均衡**：
- 长期个人会选择诚实
- 新人需要时间建立声誉
- 声誉影响职业发展

### 4.4.6 声誉与信息

**声誉与信息的关系**：
1. 声誉是对类型的信念
2. 信息更新影响声誉
3. 声誉影响信息解读

**声誉的不完全性**：
- 声誉永远不完全确定
- 可能存在误解
- 可能被操纵

**声誉的脆弱性**：
- 一次错误可能破坏长期建立的声誉
- 恢复声誉需要时间和努力
- 声誉破坏的后果可能不对称

### 4.4.7 有限博弈中的声誉

**逆向归纳与声誉**：
- 在有限博弈的最后阶段，声誉不再重要
- 参与者在最后阶段会最大化当期利益
- 这可能导致声誉的崩溃

**KMRW 的启示**：
- 即使博弈有限，合作也可能维持
- 不完全信息是关键
- 声誉的作用在有限博弈中仍然重要

## 本章总结

### 核心要点

1. **精炼贝叶斯均衡**是不完全信息动态博弈的均衡概念，要求序贯理性和信念一致性。

2. **信念更新**是精炼贝叶斯均衡的关键，参与者根据观察到的行动使用贝叶斯法则更新信念。

3. **信号博弈**是不完全信息动态博弈的重要形式，包括发送者、接收者和信号传递。

4. **分离均衡、混同均衡和半分离均衡**是信号博弈的三种主要均衡类型，分别对应完全传递、不传递和部分传递信息。

5. **信息传递**需要满足可观察性、成本不对称性、激励相容性和接收者理性等条件。

6. **声誉**是参与者建立的形象，影响其他参与者的信念和未来的博弈结果。

7. **KMRW模型**表明即使在有限博弈中，声誉也可以维持合作。

### 重要概念表格

| 概念 | 定义 | 性质 |
|:---:|:---:|:---:|
| 精炼贝叶斯均衡 | 序贯理性+信念一致性 | 不完全信息动态博弈的均衡 |
| 信念 | 对博弈状态的判断 | 通过贝叶斯法则更新 |
| 序贯理性 | 每个信息集都最优 | 考虑未来影响 |
| 信念一致性 | 符合贝叶斯法则 | 均衡路径上 |
| 信号博弈 | 发送者-接收者博弈 | 传递私人信息 |
| 分离均衡 | 不同类型不同信号 | 完全揭示信息 |
| 混同均衡 | 所有类型相同信号 | 不传递信息 |
| 半分离均衡 | 部分类型随机信号 | 部分传递信息 |
| 声誉 | 建立的形象 | 影响未来博弈 |
| 信息泄露 | 信息通过行动泄露 | 可能不可控 |
