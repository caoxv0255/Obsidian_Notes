# 第5章 合作博弈

## 章节学习目标

本章主要讨论合作博弈的基本理论，学生完成本章学习后应能够：

1. 掌握合作博弈与非合作博弈的区别
2. 理解联盟的概念和表示方法
3. 掌握核心的定义和性质
4. 理解稳定集的概念
5. 掌握夏普利值的定义和计算方法
6. 理解谈判解的基本原理
7. 能够运用合作博弈分析联盟形成和收益分配问题

## 5.1 合作博弈的基本概念

### 5.1.1 合作博弈与非合作博弈

**非合作博弈（Non-Cooperative Game）**：
- 参与者独立做出决策
- 不能达成有约束力的协议
- 强调个人理性
- 关注策略和均衡

**合作博弈（Cooperative Game）**：
- 参与者可以组成联盟
- 可以达成有约束力的协议
- 强调集体理性
- 关注联盟和收益分配

**区别**：
1. 是否可以组成联盟
2. 是否可以达成有约束力的协议
3. 关注的重点不同
4. 分析方法不同

**联系**：
- 合作博弈可以看作是非合作博弈的精炼
- Nash 程序：通过非合作博弈实现合作结果
- 某些合作博弈可以用非合作博弈解释

### 5.1.2 联盟的特征函数

**联盟（Coalition）**：
- 参与者集合 N 的子集 S ⊆ N
- S 中的参与者可以合作

**特征函数（Characteristic Function）**：
- v: 2^N → ℝ
- v(S) 表示联盟 S 可以获得的最大总收益
- v(∅) = 0

**超可加性（Superadditivity）**：
如果 v(S ∪ T) ≥ v(S) + v(T)，对所有不相交的 S, T ⊆ N
则称特征函数 v 是超可加的。

**超可加性的含义**：
- 合并联盟不会减少总收益
- 大联盟至少和各部分之和一样好
- 激励形成大联盟

### 5.1.3 合作博弈的表示

**特征型博弈（Cooperative Game in Characteristic Function Form）**：
- 由参与者集合 N 和特征函数 v 构成
- 记为 (N, v)

**例子：三人博弈**

参与者集合：N = {1, 2, 3}

特征函数：
- v(∅) = 0
- v({1}) = 1, v({2}) = 2, v({3}) = 3
- v({1,2}) = 4, v({1,3}) = 5, v({2,3}) = 6
- v({1,2,3}) = 7

**解释**：
- 每个人单独行动的收益
- 两人合作的收益
- 三人合作的收益

### 5.1.4 转移效用博弈

**可转移效用（Transferable Utility, TU）**：
- 联盟的收益可以在成员之间自由转移
- 通常用货币表示
- 收益是可加的

**不可转移效用（Non-Transferable Utility, NTU）**：
- 联盟的收益不能自由转移
- 收益是不可加的
- 需要考虑个人理性

**本章重点**：
主要讨论可转移效用博弈

### 5.1.5 合作博弈的应用

**应用1：投票**
- 联盟通过投票获得权力
- 特征函数表示联盟是否能赢得投票

**应用2：成本分摊**
- 联盟共同承担成本
- 特征函数表示联盟需要支付的成本

**应用3：市场**
- 参与者通过合作获得更好的交易条件
- 特征函数表示联盟的市场价值

**应用4：公共物品**
- 联盟共同提供公共物品
- 特征函数表示联盟的总收益

## 5.2 核心与稳定集

### 5.2.1 分配的定义

**分配（Imputation）**：
- 支付向量 x = (x₁, x₂, ..., xₙ)
- 满足：
  1. 集体理性：Σ_{i∈N} xᵢ = v(N)
  2. 个人理性：xᵢ ≥ v({i})，对所有 i ∈ N

**解释**：
- 集体理性：总收益等于大联盟的收益
- 个人理性：每个人的收益不低于单独行动的收益

**分配集合（I(N, v)）**：
所有分配的集合

### 5.2.2 核心的定义

**核心（Core）**：
- 分配的集合
- 没有联盟可以改进

**核心的正式定义**：
x ∈ Core(N, v) 当且仅当：
1. Σ_{i∈N} xᵢ = v(N)（集体理性）
2. Σ_{i∈S} xᵢ ≥ v(S)，对所有 S ⊆ N（联盟理性）

**联盟理性的含义**：
- 任何联盟 S 的收益至少等于联盟单独行动的收益
- 没有联盟有动机离开大联盟

### 5.2.3 核心的性质

**性质1：核心可能是空的**
- 某些博弈没有核心
- 例如：某些投票博弈

**性质2：核心可能不唯一**
- 某些博弈有多个核心分配
- 需要进一步精炼

**性质3：核心是凸集**
- 如果 x, y ∈ Core，则 αx + (1-α)y ∈ Core
- 0 ≤ α ≤ 1

**性质4：核心满足匿名性**
- 如果改变参与者的标签，核心相应改变

### 5.2.4 核心的例子

**例子1：二人博弈**

N = {1, 2}
v({1}) = 1, v({2}) = 2, v({1,2}) = 4

核心条件：
1. x₁ + x₂ = 4
2. x₁ ≥ 1, x₂ ≥ 2

解：
x₁ + x₂ = 4
x₁ ∈ [1, 2]
x₂ = 4 - x₁ ∈ [2, 3]

核心 = {(x₁, x₂): x₁ ∈ [1, 2], x₂ = 4 - x₁}

**例子2：三人博弈**

N = {1, 2, 3}
v({1}) = 0, v({2}) = 0, v({3}) = 0
v({1,2}) = 1, v({1,3}) = 1, v({2,3}) = 1
v({1,2,3}) = 1

核心条件：
1. x₁ + x₂ + x₃ = 1
2. x₁ + x₂ ≥ 1, x₁ + x₃ ≥ 1, x₂ + x₃ ≥ 1

从前两个不等式：
x₁ + x₂ ≥ 1
x₁ + x₃ ≥ 1
相加：2x₁ + x₂ + x₃ ≥ 2

代入 x₂ + x₃ = 1 - x₁：
2x₁ + 1 - x₁ ≥ 2
x₁ ≥ 1

同理：x₂ ≥ 1, x₃ ≥ 1

但 x₁ + x₂ + x₃ = 1，不可能有 x₁ ≥ 1, x₂ ≥ 1, x₃ ≥ 1

因此，核心是空的。

### 5.2.5 Bondareva-Shapley 定理

**定理**：核心非空当且仅当特征函数是平衡的。

**平衡（Balanced）**：
对于所有平衡权重的集合 {δ_S}_{S⊆N}，满足：
Σ_{S⊆N} δ_S 1_S ≥ 1_N
Σ_{S: i∈S} δ_S = 1，对所有 i ∈ N

则：Σ_{S⊆N} δ_S v(S) ≤ v(N)

**理解**：
- 平衡权重表示联盟的"强度"
- 核心非空的条件是特征函数满足平衡不等式
- 这是核心非空的充要条件

### 5.2.6 稳定集的定义

**稳定集（Stable Set）**：
- 分配的集合 K
- 满足：
  1. 内部稳定性：K 中任何两个分配互不占优
  2. 外部稳定性：任何不在 K 中的分配被 K 中的某个分配占优

**占优（Domination）**：
分配 y 占优分配 x，如果：
1. 存在联盟 S 使得 Σ_{i∈S} yᵢ ≤ v(S)
2. yᵢ > xᵢ，对所有 i ∈ S

**理解**：
- 内部稳定性：稳定集中的分配相互之间不可改进
- 外部稳定性：稳定集外的分配都可以被稳定集内的分配改进

### 5.2.7 稳定集与核心的关系

**关系**：
1. 如果核心非空，则核心是稳定集的子集
2. 稳定集可能不存在
3. 稳定集可能不唯一

**例子**：
在某些博弈中，存在多个稳定集，核心是所有稳定集的交集。

## 5.3 夏普利值

### 5.3.1 夏普利值的定义

**夏普利值（Shapley Value）**：
- 一种分配方法
- 基于参与者的边际贡献
- 唯一满足特定公理的分配

**边际贡献（Marginal Contribution）**：
参与者 i 对联盟 S \ {i} 的边际贡献：
Δᵢ(S) = v(S) - v(S \ {i})

**夏普利值的公式**：
φᵢ(v) = Σ_{S⊆N: i∈S} [(|S|-1)!(n-|S|)!/n!] · [v(S) - v(S \ {i})]

**解释**：
- 对所有包含参与者 i 的联盟 S 求和
- 权重是联盟 S 形成的概率
- 贡献是参与者 i 对联盟 S 的边际贡献

### 5.3.2 夏普利值的公理

**公理1：有效性（Efficiency）**
Σ_{i∈N} φᵢ(v) = v(N)

**公理2：对称性（Symmetry）**
如果 v(S ∪ {i}) = v(S ∪ {j}) 对所有 S ⊆ N \ {i, j}
则 φᵢ(v) = φⱼ(v)

**公理3：哑元性（Dummy Player）**
如果 v(S ∪ {i}) = v(S) 对所有 S ⊆ N
则 φᵢ(v) = 0

**公理4：可加性（Additivity）**
φᵢ(v + w) = φᵢ(v) + φᵢ(w)

**Shapley (1953) 证明**：
夏普利值是唯一满足这四个公理的分配方法。

### 5.3.3 夏普利值的计算

**例子1：二人博弈**

N = {1, 2}
v({1}) = 1, v({2}) = 2, v({1,2}) = 4

参与者1的夏普利值：
φ₁ = [(0!1!/2!)(v({1}) - v(∅))] + [(1!0!/2!)(v({1,2}) - v({2}))]
    = [(1/2)(1 - 0)] + [(1/2)(4 - 2)]
    = 0.5 + 1
    = 1.5

参与者2的夏普利值：
φ₂ = [(0!1!/2!)(v({2}) - v(∅))] + [(1!0!/2!)(v({1,2}) - v({1}))]
    = [(1/2)(2 - 0)] + [(1/2)(4 - 1)]
    = 1 + 1.5
    = 2.5

验证：φ₁ + φ₂ = 1.5 + 2.5 = 4 = v(N) ✓

**例子2：三人博弈**

N = {1, 2, 3}
v({1}) = 0, v({2}) = 0, v({3}) = 0
v({1,2}) = 1, v({1,3}) = 1, v({2,3}) = 1
v({1,2,3}) = 1

参与者1的夏普利值：
φ₁ = [(0!2!/3!)(v({1}) - v(∅))]
    + [(1!1!/3!)(v({1,2}) - v({2}))]
    + [(1!1!/3!)(v({1,3}) - v({3}))]
    + [(2!0!/3!)(v({1,2,3}) - v({2,3}))]
    = [(1/3)(0 - 0)] + [(1/6)(1 - 0)] + [(1/6)(1 - 0)] + [(1/3)(1 - 1)]
    = 0 + 1/6 + 1/6 + 0
    = 1/3

同理：φ₂ = 1/3，φ₃ = 1/3

验证：φ₁ + φ₂ + φ₃ = 1/3 + 1/3 + 1/3 = 1 = v(N) ✓

### 5.3.4 夏普利值的性质

**性质1：唯一性**
- 满足四个公理的分配是唯一的

**性质2：核心中的分配**
- 如果核心非空，夏普利值不一定在核心中
- 但在某些特殊博弈中，夏普利值在核心中

**性质3：边际性**
- 夏普利值基于边际贡献
- 体现了"按贡献分配"的思想

**性质4：公平性**
- 对称参与者获得相同收益
- 哑元参与者获得零收益

### 5.3.5 夏普利值的应用

**应用1：成本分摊**

**问题**：
三个城镇共同建设污水处理厂，需要分摊成本。

**数据**：
- 城镇1单独建设：成本100
- 城镇2单独建设：成本140
- 城镇3单独建设：成本180
- 1和2共同建设：成本180
- 1和3共同建设：成本220
- 2和3共同建设：成本240
- 1、2和3共同建设：成本240

**转换特征函数**（节省）：
v({1}) = 0
v({2}) = 0
v({3}) = 0
v({1,2}) = 100 + 140 - 180 = 60
v({1,3}) = 100 + 180 - 220 = 60
v({2,3}) = 140 + 180 - 240 = 80
v({1,2,3}) = 100 + 140 + 180 - 240 = 180

**夏普利值**：
φ₁ = [(0!2!/3!)(0)] + [(1!1!/3!)(60)] + [(1!1!/3!)(60)] + [(2!0!/3!)(180 - 80)]
    = 0 + 10 + 10 + 33.33
    = 53.33

φ₂ = [(0!2!/3!)(0)] + [(1!1!/3!)(60)] + [(1!1!/3!)(80)] + [(2!0!/3!)(180 - 60)]
    = 0 + 10 + 13.33 + 40
    = 63.33

φ₃ = [(0!2!/3!)(0)] + [(1!1!/3!)(60)] + [(1!1!/3!)(80)] + [(2!0!/3!)(180 - 60)]
    = 0 + 10 + 13.33 + 40
    = 63.33

**成本分摊**：
- 城镇1：100 - 53.33 = 46.67
- 城镇2：140 - 63.33 = 76.67
- 城镇3：180 - 63.33 = 116.67

**应用2：投票权力**

**问题**：
计算投票者的夏普利-舒比克指数。

**模型**：
- 简单多数投票
- 夏普利值表示参与者成为"关键投票者"的概率

**应用3：专利合作**

**问题**：
多个企业合作开发专利，如何分配收益？

**模型**：
- 使用夏普利值分配
- 基于各企业的边际贡献

## 5.4 谈判解

### 5.4.1 谈判问题的定义

**谈判问题（Bargaining Problem）**：
- 两个参与者 i = 1, 2
- 可达收益集合 S ⊆ ℝ²
- 谈判破裂点 d = (d₁, d₂)

**假设**：
1. S 是凸的
2. S 是闭的
3. S 是有界的
4. 存在 s ∈ S 使得 s > d

**目标**：
找到"合理"的谈判结果

### 5.4.2 纳什谈判解

**纳什谈判解（Nash Bargaining Solution）**：
- 最大化乘积的解
- 唯一满足特定公理的解

**公式**：
max (s₁ - d₁)(s₂ - d₂)
s.t. s ∈ S, s ≥ d

**公理**：
1. 帕累托有效性（Pareto Efficiency）
2. 对称性（Symmetry）
3. 不相关方案的独立性（Independence of Irrelevant Alternatives）
4. 线性变换的不变性（Invariance to Affine Transformations）

### 5.4.3 纳什谈判解的例子

**例子1：简单的谈判问题**

S = {(s₁, s₂): s₁ ≥ 0, s₂ ≥ 0, s₁ + s₂ ≤ 10}
d = (0, 0)

求解：
max s₁s₂
s.t. s₁ + s₂ ≤ 10, s₁ ≥ 0, s₂ ≥ 0

使用拉格朗日乘数：
L = s₁s₂ + λ(10 - s₁ - s₂)

一阶条件：
∂L/∂s₁ = s₂ - λ = 0 ⇒ s₂ = λ
∂L/∂s₂ = s₁ - λ = 0 ⇒ s₁ = λ
∂L/∂λ = 10 - s₁ - s₂ = 0 ⇒ s₁ + s₂ = 10

因此：s₁ = s₂ = 5

纳什谈判解：(5, 5)

**例子2：不对称的谈判问题**

S = {(s₁, s₂): s₁ ≥ 0, s₂ ≥ 0, s₁ + s₂ ≤ 10}
d = (2, 0)

求解：
max (s₁ - 2)s₂
s.t. s₁ + s₂ ≤ 10, s₁ ≥ 2, s₂ ≥ 0

使用拉格朗日乘数：
L = (s₁ - 2)s₂ + λ(10 - s₁ - s₂)

一阶条件：
∂L/∂s₁ = s₂ - λ = 0 ⇒ s₂ = λ
∂L/∂s₂ = s₁ - 2 - λ = 0 ⇒ s₁ - 2 = λ
∂L/∂λ = 10 - s₁ - s₂ = 0 ⇒ s₁ + s₂ = 10

因此：s₁ - 2 = s₂, s₁ + s₂ = 10
s₁ - 2 = 10 - s₁
2s₁ = 12
s₁ = 6, s₂ = 4

纳什谈判解：(6, 4)

### 5.4.4 其他谈判解

**Kalai-Smorodinsky 解**：
- 基于理想点
- 最大化比例
- 满足单调性

**Egalitarian 解**：
- 平均分配剩余
- 强调公平

**效用主义解**：
- 最大化总效用
- 强调效率

### 5.4.5 谈判解的应用

**应用1：工资谈判**
- 工人和雇主谈判工资
- 谈判破裂点是双方不接受合同
- 纳什谈判解给出合理的工资水平

**应用2：国际协议**
- 国家之间谈判合作协议
- 谈判破裂点是无协议
- 纳什谈判解给出合理的协议条款

**应用3：纠纷解决**
- 双方在法庭外解决纠纷
- 谈判破裂点是上法庭
- 纳什谈判解给出合理的和解方案

### 核心概念汇总

#### 1. 纳什均衡
**定义**：互为最佳反应的策略组合
**意义**：自我执行的稳定状态
**求解**：划线法、最佳反应函数、不动点定理

#### 2. 子博弈完美纳什均衡
**定义**：在每个子博弈中都构成纳什均衡
**意义**：消除不可信威胁
**求解**：逆向归纳法

#### 3. 贝叶斯纳什均衡
**定义**：每种类型的最佳反应
**意义**：考虑不确定性下的决策
**求解**：猜测策略，验证一致性

#### 4. 精炼贝叶斯均衡
**定义**：序贯理性 + 信念一致性
**意义**：动态信息更新
**求解**：猜测策略和信念，验证

#### 5. 核心与夏普利值
**定义**：合作博弈的分配方法
**意义**：公平与效率
**求解**：优化问题、边际贡献

### 重要定理

#### 1. 纳什均衡存在性定理
**内容**：每个有限博弈至少存在一个纳什均衡
**意义**：保证博弈模型有解
**证明**：Kakutani 不动点定理

#### 2. 无名氏定理
**内容**：可行且个人理性的支付可以由重复博弈实现
**意义**：合作可以维持
**条件**：贴现因子足够大

#### 3. 收益等价定理
**内容**：标准拍卖的期望收益相同
**意义**：拍卖形式不影响收益
**条件**：独立私人价值、风险中性、对称

#### 4. Bondareva-Shapley 定理
**内容**：核心非空当且仅当特征函数是平衡的
**意义**：核心非空的充要条件
**应用**：联盟形成分析

### 关键工具

#### 1. 逆向归纳法
**用途**：求解有限完美信息博弈
**步骤**：从最后阶段向前求解
**局限**：要求有限、完美信息

#### 2. 海萨尼转换
**用途**：将不完全信息转化为不完美信息
**步骤**：引入自然选择类型
**意义**：使贝叶斯博弈可以使用纳什均衡

#### 3. 贝叶斯法则
**用途**：更新信念
**公式**：P(H|E) = P(E|H)P(H) / P(E)
**应用**：精炼贝叶斯均衡

#### 4. 显示原理
**用途**：简化机制设计
**内容**：任何机制等价于直接机制
**意义**：大大简化设计问题

### 经典模型

#### 1. 囚徒困境
**特点**：占优策略均衡，非帕累托最优
**启示**：个人理性与集体理性的冲突
**扩展**：重复囚徒困境，KMRW模型

#### 2. 古诺模型
**特点**：数量竞争，静态博弈
**均衡**：q* = (a-c)/3（双寡头）
**扩展**：斯塔克尔伯格模型，动态古诺

#### 3. 伯川德模型
**特点**：价格竞争，静态博弈
**均衡**：p* = c（边际成本定价）
**悖论**：两个企业导致完全竞争

#### 4. 斯塔克尔伯格模型
**特点**：领导者-跟随者，动态博弈
**均衡**：领导者获得先动优势
**方法**：逆向归纳法

#### 5. 拍卖模型
**类型**：一级价格、二级价格、英式、荷兰式
**均衡**：如实出价（二级价格）
**定理**：收益等价定理

#### 6. 信号博弈
**应用**：劳动力市场、产品质量
**均衡**：分离、混同、半分离
**精炼**：直觉标准

## 本章总结

### 核心要点

1. **合作博弈**关注联盟形成和收益分配，与非合作博弈形成对比。

2. **核心**是满足联盟理性的分配集合，描述了稳定的联盟结构。

3. **夏普利值**基于边际贡献，是满足特定公理的唯一分配方法。

4. **谈判解**描述了参与者如何通过谈判达成协议。

5. **合作博弈**在投票、成本分摊、公共物品等领域有广泛应用。

6. **全书**系统介绍了博弈论的基本理论，构建了完整的知识体系。

### 重要概念表格

| 概念 | 定义 | 性质 |
|:---:|:---:|:---:|

| 特征函数 | 联盟的收益函数 | v(∅) = 0 |
| 超可加性 | v(S∪T) ≥ v(S)+v(T) | 激励形成大联盟 |
| 核心 | 联盟理性的分配集合 | 可能空，可能不唯一 |
| 稳定集 | 内外部稳定的分配 | 不一定存在 |
