# 第5章：不确定性下的选择

## 章节学习目标

本章介绍不确定性下的选择理论，这是微观经济学的重要组成部分，广泛应用于金融、保险和投资决策分析。通过本章的学习，读者应该能够：

1. 理解不确定性下的选择问题的数学表述
2. 掌握期望效用理论的公理体系
3. 理解冯·诺伊曼-摩根斯坦效用函数
4. 分析风险厌恶的概念及其测度
5. 掌握确定性等价和风险溢价
6. 理解风险资产定价的基本原理
7. 分析保险需求及其决定因素
8. 掌握随机占优理论
9. 理解均值-方差分析
10. 分析投资组合选择问题

## 5.1 期望效用理论

### 5.1.1 不确定性下的选择问题

**定义5.1.1（状态）**：状态s ∈ S = {1, ..., S}表示可能的世界状态，每种状态以概率π_s发生，π_s > 0，∑_{s=1}^S π_s = 1。

**定义5.1.2（结果）**：结果x ∈ X是消费者在每个状态下获得的消费或财富水平。

**定义5.1.3（彩票/概率分布）**：彩票L是在结果空间X上的概率分布，记作L = (x₁, π₁; x₂, π₂; ...; x_S, π_S)，表示结果x_s以概率π_s实现。

**定义5.1.4（简单彩票）**：简单彩票是有限个结果的概率分布。

**定义5.1.5（复合彩票）**：复合彩票是简单彩票的概率分布。例如，L = (L₁, α; L₂, 1-α)，其中L₁和L₂是简单彩票。

**定理5.1.1（复合彩票的简化）**：任何复合彩票都可以简化为等价的简单彩票。例如，如果L₁ = (x₁, π₁¹; x₂, π₂¹)，L₂ = (x₁, π₁²; x₂, π₂²)，则L简化为(x₁, απ₁¹ + (1-α)π₁²; x₂, απ₂¹ + (1-α)π₂²)。

**例5.1.1**：考虑投资决策。状态s = 1表示经济繁荣，概率π₁ = 0.6；状态s = 2表示经济衰退，概率π₂ = 0.4。投资A在繁荣时回报100，衰退时回报50；投资B在繁荣时回报80，衰退时回报70。这两个投资可以表示为彩票：L_A = (100, 0.6; 50, 0.4)，L_B = (80, 0.6; 70, 0.4)。

### 5.1.2 冯·诺伊曼-摩根斯坦公理

**定义5.1.6（偏好关系）**：在彩票空间上定义偏好关系≽，满足以下公理：

**公理1（完备性）**：对于任意L, L'，要么L ≽ L'，要么L' ≽ L（或两者都成立）。

**公理2（自反性）**：对于任意L，L ≽ L。

**公理3（传递性）**：对于任意L, L', L''，如果L ≽ L'且L' ≽ L''，则L ≽ L''。

**公理4（连续性）**：对于任意L, L', L''，如果L ≻ L' ≻ L''，则存在α, β ∈ (0, 1)使得：
αL + (1-α)L'' ≻ L' ≻ βL + (1-β)L''

**公理5（独立性）**：对于任意L, L', L''和α ∈ (0, 1)：
L ≽ L' ⇔ αL + (1-α)L'' ≽ αL' + (1-α)L''

**定理5.1.2（冯·诺伊曼-摩根斯坦期望效用定理）**：如果偏好关系≽满足完备性、自反性、传递性、连续性和独立性，则存在效用函数U: L → ℝ使得：
1. L ≽ L' ⇔ U(L) ≥ U(L')
2. U(L) = ∑_{s=1}^S π_s u(x_s) = E[u(x)]

其中u: X → ℝ是冯·诺伊曼-摩根斯坦效用函数（基数效用函数）。

**证明概要**：
1. 构造效用尺度：选择最好的结果x̄和最差的结果x̲，定义U(x̄) = 1, U(x̲) = 0。
2. 对于任意结果x，定义U(x) = α，其中α满足x ~ αx̄ + (1-α)x̲。
3. 对于任意彩票L = (x₁, π₁; ...; x_S, π_S)，定义U(L) = ∑_{s=1}^S π_s U(x_s)。
4. 验证这个效用函数满足期望效用性质。
5. 验证这个效用函数表示偏好关系。□

**解释5.1.1（独立性公理的经济学含义）**：独立性公理意味着偏好不受无关替代品的影响。如果我在彩票L和L'之间偏好L，则无论L是否与第三个彩票L''混合，这种偏好关系不变。

**例5.1.2（阿莱悖论）**：考虑以下两对选择：
选择A：100%概率获得100万美元
选择B：89%概率获得100万美元，10%概率获得500万美元，1%概率获得0

选择C：11%概率获得100万美元，89%概率获得0
选择D：10%概率获得500万美元，90%概率获得0

许多人在A和B之间偏好A，在C和D之间偏好D。这违反了独立性公理。

### 5.1.3 效用函数的性质

**定理5.1.3（期望效用函数的唯一性）**：期望效用函数U(L) = E[u(x)]是基数效用函数。如果u(x)是冯·诺伊曼-摩根斯坦效用函数，则任何正仿射变换v(x) = a + b·u(x)（b > 0）也是冯·诺伊曼-摩根斯坦效用函数。

**证明**：如果U(L) = E[u(x)]表示偏好，则V(L) = E[v(x)] = a + bE[u(x)] = a + bU(L)也表示相同的偏好，因为b > 0。□

**定理5.1.4（效用函数的构造）**：给定冯·诺伊曼-摩根斯坦效用函数u(x)，可以构造期望效用函数U(L) = E[u(x)]。反之，给定期望效用函数U(L)，可以恢复冯·诺伊曼-摩根斯坦效用函数u(x) = U(确定性结果x)。

**例5.1.3（构造期望效用函数）**：假设冯·诺伊曼-摩根斯坦效用函数u(x) = √x。对于彩票L = (100, 0.6; 50, 0.4)，期望效用为：
U(L) = 0.6√100 + 0.4√50 = 0.6·10 + 0.4·7.07 = 6 + 2.83 = 8.83

### 5.1.4 期望效用的连续性

**定理5.1.5（期望效用函数的连续性）**：如果u(x)是连续的，则期望效用函数U(L)是连续的。

**证明**：由于U(L) = E[u(x)]是连续函数u的积分，期望算子保持连续性。□

**定理5.1.6（期望效用函数的凸性）**：如果u(x)是凹函数，则期望效用函数U(L)对概率分布是凹的。

**证明**：对于任意两个彩票L, L'和λ ∈ [0, 1)：
U(λL + (1-λ)L') = E[u(λx + (1-λ)x')] ≥ λE[u(x)] + (1-λ)E[u(x')] = λU(L) + (1-λ)U(L')
其中不等式由u的凹性保证。□

## 5.2 风险厌恶

### 5.2.1 风险厌恶的定义

**定义5.2.1（风险厌恶）**：消费者是风险厌恶的，如果对于任何彩票L，有：
U(L) ≤ u(E[x])

即期望效用不超过效用的期望。

**定义5.2.2（风险中性）**：消费者是风险中性的，如果对于任何彩票L，有：
U(L) = u(E[x])

**定义5.2.3（风险偏好）**：消费者是风险偏好的，如果对于任何彩票L，有：
U(L) ≥ u(E[x])

**定理5.2.1（风险厌恶与效用函数的关系）**：
1. 消费者是风险厌恶的 ⇔ u(x)是凹函数。
2. 消费者是风险中性的 ⇔ u(x)是线性函数。
3. 消费者是风险偏好的 ⇔ u(x)是凸函数。

**证明（风险厌恶 ⇒ u凹）**：如果消费者是风险厌恶的，对于任意x, x'和λ ∈ (0, 1)，定义彩票L = (x, λ; x', 1-λ)。风险厌恶意味着：
U(L) = λu(x) + (1-λ)u(x') ≤ u(λx + (1-λ)x')
这正是凹函数的定义。□

**图形分析5.2.1**：
```
风险厌恶的效用函数:

效用u(x)
  |
  |        /|
  |       / |
  |      /  |
  |     /   |
  |    /    |
  |   /     |
  |  /      |
  | /       |
  |/________|______ 财富x
  x₁   E[x]  x₂

效用函数u(x)是凹的
U(L) = E[u(x)] < u(E[x])
```

### 5.2.2 风险厌恶的测度

**定义5.2.4（阿罗-普拉特绝对风险厌恶系数）**：
A(x) = -u''(x)/u'(x)

**定义5.2.5（阿罗-普拉特相对风险厌恶系数）**：
R(x) = -x·u''(x)/u'(x) = x·A(x)

**定理5.2.2（风险厌恶系数的性质）**：
1. A(x) > 0 ⇔ u(x)是严格凹的（风险厌恶）
2. A(x) = 0 ⇔ u(x)是线性的（风险中性）
3. A(x) < 0 ⇔ u(x)是凸的（风险偏好）

**例5.2.1（常用效用函数的风险厌恶系数）**：
1. 线性效用：u(x) = x ⇒ A(x) = 0（风险中性）
2. 二次效用：u(x) = x - bx² ⇒ A(x) = 2b/(1-2bx)
3. 负指数效用：u(x) = -e^{-ax} ⇒ A(x) = a（常数绝对风险厌恶）
4. 对数效用：u(x) = ln(x) ⇒ A(x) = 1/x, R(x) = 1（常数相对风险厌恶）
5. 幂效用：u(x) = x^{1-γ}/(1-γ) ⇒ A(x) = γ/x, R(x) = γ（常数相对风险厌恶）

**定理5.2.3（普拉特定理）**：对于两个效用函数u₁(x)和u₂(x)，以下条件等价：
1. A₁(x) ≥ A₂(x) 对所有x
2. u₁(x)是u₂(x)的凹变换，即存在凹函数φ使得u₁(x) = φ(u₂(x))

**解释5.2.1（普拉特定理的经济学含义）**：如果A₁(x) ≥ A₂(x)，则消费者1比消费者2更厌恶风险。消费者1的效用函数是消费者2效用函数的凹变换，这意味着消费者1对风险的厌恶程度更高。

### 5.2.3 确定性等价与风险溢价

**定义5.2.6（确定性等价）**：彩票L的确定性等价CE是满足u(CE) = U(L)的确定性财富水平。

**定义5.2.7（风险溢价）**：彩票L的风险溢价RP是期望财富与确定性等价的差额：
RP = E[x] - CE

**定理5.2.4（风险溢价与风险厌恶的关系）**：
1. 如果消费者是风险厌恶的，则RP > 0。
2. 如果消费者是风险中性的，则RP = 0。
3. 如果消费者是风险偏好的，则RP < 0。

**证明**：由定义，u(CE) = U(L) = E[u(x)]。如果u是凹的，由詹森不等式，E[u(x)] ≤ u(E[x])，因此CE ≤ E[x]，即RP ≥ 0。如果u是严格凹的，则严格不等式成立。□

**定理5.2.5（普拉特定理（风险溢价形式））**：对于小风险（方差σ²很小），风险溢价近似为：
RP ≈ 0.5·A(E[x])·σ²

**证明**：对u(x)在E[x]处进行泰勒展开：
u(x) ≈ u(E[x]) + u'(E[x])(x - E[x]) + 0.5·u''(E[x])(x - E[x])²
取期望：E[u(x)] ≈ u(E[x]) + 0.5·u''(E[x])·σ²
由u(CE) = E[u(x)]，对u(CE)在E[x]处展开：
u(CE) ≈ u(E[x]) + u'(E[x])(CE - E[x])
因此：u'(E[x])(CE - E[x]) ≈ 0.5·u''(E[x])·σ²
CE - E[x] ≈ 0.5·[u''(E[x])/u'(E[x])]·σ² = -0.5·A(E[x])·σ²
所以：RP = E[x] - CE ≈ 0.5·A(E[x])·σ² □

**例5.2.2（风险溢价计算）**：设u(x) = ln(x)，初始财富W = 100。考虑彩票L = (120, 0.5; 80, 0.5)。

期望财富：E[x] = 0.5·120 + 0.5·80 = 100
期望效用：U(L) = 0.5·ln(120) + 0.5·ln(80) ≈ 0.5·4.79 + 0.5·4.38 = 4.59
确定性等价：CE = e^{4.59} ≈ 98.5
风险溢价：RP = 100 - 98.5 = 1.5

风险厌恶系数：A(100) = 1/100 = 0.01
方差：σ² = 0.5·(120-100)² + 0.5·(80-100)² = 400
普拉特近似：RP ≈ 0.5·0.01·400 = 2

近似值与真实值接近但不完全相等，因为风险不是很小。

### 5.2.4 比较风险厌恶

**定理5.2.6（绝对风险厌恶的比较）**：如果A₁(x) ≥ A₂(x)对所有x，则对于任何彩票L，消费者1的风险溢价大于或等于消费者2的风险溢价。

**定理5.2.7（相对风险厌恶的比较）**：如果R₁(x) ≥ R₂(x)对所有x，则对于任何彩票L，消费者1的相对风险溢价（RP/E[x]）大于或等于消费者2的相对风险溢价。

**例5.2.3（比较不同效用函数的风险厌恶）**：
u₁(x) = ln(x), u₂(x) = x^{0.5}/0.5
A₁(x) = 1/x, A₂(x) = 0.5/x
因此，A₁(x) > A₂(x)，消费者1比消费者2更厌恶风险。

## 5.3 不确定性下的消费者理论

### 5.3.1 期望效用最大化

**问题5.3.1（期望效用最大化）**：
max_{x_s ≥ 0} ∑_{s=1}^S π_s u_s(x_s)
s.t. ∑_{s=1}^S p_s x_s ≤ W

其中x_s是状态s下的消费，p_s是状态s下的价格，W是初始财富。

**定义5.3.1（或有商品）**：或有商品是在特定状态下可获得的商品。例如，"失业时的收入"和"就业时的收入"是不同的或有商品。

**定理5.3.1（一阶条件）**：如果内点解存在，则：
π_s u_s'(x_s*) = λp_s, s = 1, ..., S

其中λ是拉格朗日乘数（财富的边际效用）。

**解释5.3.1（一阶条件的经济学含义）**：等式左边是状态s下消费的边际期望效用，右边是状态s下消费的边际成本乘以财富的边际效用。最优条件是边际期望效用等于边际成本。

**推论5.3.1**：由一阶条件，任意两个状态s和t的边际替代率为：
MRS_{st} = (π_s u_s'(x_s*))/(π_t u_t'(x_t*)) = p_s/p_t

即状态价格比等于边际期望效用比。

### 5.3.2 状态依存偏好

**定义5.3.2（状态依存效用函数）**：u_s(x)表示在状态s下获得消费x的效用。如果u_s(x)不依赖于s，则偏好是状态独立的；如果依赖于s，则偏好是状态依存的。

**例5.3.1（状态依存偏好的例子）**：如果状态s表示"生病"，则消费的边际效用可能更高（因为生病时更需要医疗支出）。因此，u_s'(x) > u_t'(x)对于"生病"状态s和"健康"状态t。

### 5.3.3 完全市场

**定义5.3.3（完全市场）**：市场是完全的，如果对于每种状态s，都存在或有商品x_s，且价格p_s已知。

**定理5.3.2（完全市场的均衡）**：在完全市场中，竞争均衡是帕累托最优的。

**证明**：类似于福利经济学第一定理的证明。由于市场是完全的，消费者可以对每种状态下的消费进行最优选择，均衡配置是帕累托最优的。□

### 5.3.4 不完全市场

**定义5.3.4（不完全市场）**：市场是不完全的，如果存在某些状态无法通过市场交易实现风险分担。

**定理5.3.3（不完全市场的无效率）**：在不完全市场中，竞争均衡可能不是帕累托最优的。

**解释5.3.2（无效率的来源）**：由于某些风险无法通过市场分担，消费者可能承担不必要的风险，导致配置无效率。

## 5.4 资产定价

### 5.4.1 资产定价的基本原理

**定义5.4.1（资产）**：资产是在未来不同状态下产生收益的金融工具。资产j可以表示为收益向量r^j = (r^j_1, ..., r^j_S)，其中r^j_s是资产j在状态s下的收益。

**定义5.4.2（无风险资产）**：无风险资产是在所有状态下产生相同收益的资产，即r^j_s = r_f对所有s。

**定义5.4.3（风险资产）**：风险资产是在不同状态下产生不同收益的资产。

**定理5.4.1（资产定价基本方程）**：在均衡时，资产j的价格q_j满足：
q_j = ∑_{s=1}^S π_s m_s r^j_s = E[m·r^j]

其中m_s = u_s'(x_s*)/λ是随机折现因子（或状态价格密度）。

**证明**：消费者的问题可以表示为在预算约束下选择资产组合。一阶条件给出上述定价方程。□

**定义5.4.4（随机折现因子）**：m_s = u_s'(x_s*)/λ是将未来状态s下的收益折现到当前的因子。

**推论5.4.1**：对于无风险资产，r^j_s = 1对所有s，因此：
q_j = ∑_{s=1}^S π_s m_s = E[m]

因此，无风险利率r_f满足1 + r_f = 1/E[m]。

### 5.4.2 资本资产定价模型（CAPM）

**定义5.4.5（市场组合）**：市场组合是所有风险资产的按市值加权的组合。

**定义5.4.6（β系数）**：资产j的β系数是：
β_j = Cov(r^j, r^m)/Var(r^m)

其中r^m是市场组合的收益。

**定理5.4.2（CAPM）**：在均衡时，资产j的期望收益满足：
E[r^j] = r_f + β_j (E[r^m] - r_f)

即资产j的期望收益等于无风险利率加上风险溢价（β_j乘以市场风险溢价）。

**证明概要**：
1. 假设所有投资者具有相同的期望效用函数，且市场是完全的。
2. 投资者选择资产组合以最大化期望效用。
3. 一阶条件给出了资产定价关系。
4. 通过市场组合的收益，可以推导出CAPM。□

**解释5.4.1（CAPM的经济学含义）**：资产的期望收益只与系统性风险（β）相关，与非系统性风险无关。这是因为非系统性风险可以通过分散化消除。

### 5.4.3 套利定价理论（APT）

**定理5.4.3（APT）**：如果资产收益由k个因子驱动，则资产j的期望收益满足：
E[r^j] = r_f + ∑_{k=1}^K λ_k β_{jk}

其中β_{jk}是资产j对因子k的敏感度，λ_k是因子风险溢价。

**解释5.4.2（APT的经济学含义）**：资产的期望收益由多个系统性风险因子决定，而不仅仅是市场组合的收益。

### 5.4.4 风险中性定价

**定义5.4.7（风险中性概率）**：风险中性概率π̃_s定义为：
π̃_s = π_s m_s / E[m] = π_s u_s'(x_s*) / ∑_{t=1}^S π_t u_t'(x_t*)

**定理5.4.4（风险中性定价）**：资产j的价格可以表示为：
q_j = (1/(1+r_f)) ∑_{s=1}^S π̃_s r^j_s = Ẽ[r^j]/(1+r_f)

其中Ẽ[·]表示在风险中性概率下的期望。

**解释5.4.3（风险中性定价的经济学含义）**：在风险中性概率下，资产的期望收益等于无风险利率。风险中性概率调整了真实概率，使得风险厌恶的影响被包含在概率中。

## 5.5 保险需求

### 5.5.1 保险的基本模型

**问题5.5.1（保险需求）**：
max_{c ≥ 0} π_L u(W - D + c - p·c) + π_H u(W - p·c)
s.t. c ≥ 0

其中W是初始财富，D是损失额，c是保险覆盖额，p是保险费率（每单位保险的价格），π_L和π_H分别是损失和不损失的概率（π_L + π_H = 1）。

**定义5.5.1（精算公平保费）**：精算公平保费率p* = π_L，即保险费等于期望赔付。

**定理5.5.1（最优保险）**：如果保费率是精算公平的（p = π_L），则风险厌恶的消费者会选择完全保险（c = D）。

**证明**：一阶条件为：
-π_L u'(W - D + c - p·c) + π_H p u'(W - p·c) = 0

如果p = π_L，则：
-π_L u'(W - D + c - π_L c) + π_H π_L u'(W - π_L c) = 0
⇒ u'(W - D + c - π_L c) = u'(W - π_L c)

由于u'是严格递减的（u是严格凹的），这意味着：
W - D + c - π_L c = W - π_L c
⇒ -D + c = 0
⇒ c = D

因此，消费者选择完全保险。□

**定理5.5.2（不完全保险）**：如果保费率高于精算公平保费率（p > π_L），则风险厌恶的消费者会选择不完全保险（c < D）。

**证明**：由一阶条件，如果p > π_L，则：
π_L u'(W - D + c - p·c) = π_H p u'(W - p·c)
由于p > π_L = 1 - π_H，右边大于左边，因此u'(W - D + c - p·c) > u'(W - p·c)
由于u'递减，这意味着W - D + c - p·c < W - p·c，即c < D。□

**图形分析5.5.1**：
```
保险需求:

效用
  |
  |      •（完全保险，精算公平保费）
  |     /|
  |    / |
  |   /  |
  |  /   |
  | /    |
  |/     |
  |      |
  |      |
  |______|______ 保险覆盖额c
  0     D

精算公平保费：c = D
保费率 > 精算公平：c < D
```

### 5.5.2 保险与风险厌恶

**定理5.5.3（风险厌恶与保险需求）**：如果A₁(x) ≥ A₂(x)（消费者1比消费者2更厌恶风险），则消费者1的保险需求大于或等于消费者2的保险需求。

**证明**：由普拉特定理，更厌恶风险的消费者愿意支付更高的风险溢价，因此更倾向于购买保险。□

### 5.5.3 信息不对称与保险

**定义5.5.2（逆向选择）**：如果保险公司无法区分高风险和低风险消费者，则高风险消费者更可能购买保险，导致保险市场失灵。

**定义5.5.3（道德风险）**：如果消费者购买了保险，可能减少预防措施，导致损失概率增加。

**定理5.5.4（罗斯柴尔德-斯蒂格利茨模型）**：在存在逆向选择的保险市场中，可能不存在竞争均衡，或者均衡是分离均衡（不同风险类型的消费者购买不同数量的保险）。

**解释5.5.1（逆向选择的经济学含义）**：由于保险公司无法区分风险类型，低风险消费者可能被挤出保险市场，导致只有高风险消费者购买保险（市场失灵）。

## 5.6 随机占优

### 5.6.1 一阶随机占优

**定义5.6.1（一阶随机占优，FSD）**：彩票L一阶随机占优于彩票L'，记作L ≽_FSD L'，如果对于所有单调递增的效用函数u，有：
E[u(L)] ≥ E[u(L')]

**定理5.6.1（FSD的等价条件）**：L ≽_FSD L'当且仅当L的累积分布函数F(x) ≤ F'(x)对所有x，即F(x)处处不大于F'(x)。

**解释5.6.1（FSD的经济学含义）**：如果L ≽_FSD L'，则L的分布向右偏移，意味着L在任何状态下都提供至少与L'一样高的结果（概率上）。

**图形分析5.6.1**：
```
一阶随机占优:

CDF
  |
  |   F'(x)
  |  /
  | /
  |/     F(x)
  |      /
  |     /
  |    /
  |   /
  |  /
  | /
  |/______________ 结果x
    0

F(x) ≤ F'(x)对所有x
L ≽_FSD L'
```

### 5.6.2 二阶随机占优

**定义5.6.2（二阶随机占优，SSD）**：彩票L二阶随机占优于彩票L'，记作L ≽_SSD L'，如果对于所有单调递增的凹效用函数u，有：
E[u(L)] ≥ E[u(L')]

**定理5.6.2（SSD的等价条件）**：L ≽_SSD L'当且仅当∫_{-∞}^x [F'(t) - F(t)]dt ≥ 0对所有x，且E[L] ≥ E[L']。

**解释5.6.2（SSD的经济学含义）**：如果L ≽_SSD L'，则L比L'更少风险（在均值至少相等的条件下）。

**图形分析5.6.2**：
```
二阶随机占优:

累积概率差
  |
  |   ∫(F' - F)dt ≥ 0
  |  /
  | /
  |/
  |
  |
  |
  |________________ 结果x
    0

L ≽_SSD L'
```

### 5.6.3 随机占优的应用

**应用5.6.1（投资组合选择）**：如果投资组合A一阶或二阶随机占优于投资组合B，则所有风险厌恶的投资者都偏好A而非B。

**应用5.6.2（保险评估）**：如果保险合同随机占优于不保险，则消费者会购买保险。

## 5.7 均值-方差分析

### 5.7.1 均值-方差偏好

**定义5.7.1（均值-方差效用函数）**：
U(L) = E[x] - 0.5·A·Var(x)

其中A是绝对风险厌恶系数。

**定理5.7.1（均值-方差效用与期望效用的关系）**：如果效用函数是二次的，或者正态分布，则均值-方差分析等价于期望效用分析。

**证明（二次效用）**：设u(x) = x - bx²。则：
E[u(x)] = E[x] - bE[x²] = E[x] - b[Var(x) + (E[x])²]

如果考虑固定均值E[x]，则最大化E[u(x)]等价于最小化Var(x)。□

### 5.7.2 马科维茨投资组合理论

**问题5.7.1（马科维茨投资组合选择）**：
min_w Var(w'r)
s.t. E[w'r] = μ, w'1 = 1

其中w是资产权重向量，r是资产收益向量，μ是目标期望收益。

**定理5.7.2（有效前沿）**：最小方差投资组合的集合构成有效前沿，即给定期望收益下的最小方差投资组合。

**定理5.7.3（两基金定理）**：任何有效前沿上的投资组合都可以表示为两个基准投资组合的线性组合。

**证明**：有效前沿是二次规划的解，由于约束是线性的，解集是凸的，任何有效投资组合都可以表示为两个极端投资组合的凸组合。□

### 5.7.3 资本市场线

**定义5.7.2（资本市场线，CML）**：资本市场线表示有效投资组合的期望收益与风险（标准差）的关系：
E[r_p] = r_f + [E[r_m] - r_f]/σ_m · σ_p

其中r_p是投资组合收益，r_m是市场组合收益，σ_p和σ_m分别是投资组合和市场组合的标准差。

**解释5.7.1（CML的经济学含义）**：CML表示最优投资组合的权衡：更高的期望收益需要承担更高的风险。

**图形分析5.7.1**：
```
资本市场线:

期望收益
  |
  |          有效前沿
  |         /
  |        /  CML
  |       /
  |      /
  |     /
  |    /
  |   /
  |  /
  | /
  |/____________ 标准差
  0   r_f

CML通过无风险资产和市场组合
```

## 本章总结

### 核心要点

1. **期望效用理论**：冯·诺伊曼-摩根斯坦公理（完备性、自反性、传递性、连续性、独立性）保证存在期望效用函数U(L) = E[u(x)]。

2. **风险厌恶**：风险厌恶 ⇔ u(x)是凹函数；风险厌恶测度包括阿罗-普拉特绝对和相对风险厌恶系数A(x)和R(x)。

3. **确定性等价与风险溢价**：CE满足u(CE) = E[u(x)]；RP = E[x] - CE；对于小风险，RP ≈ 0.5·A(E[x])·σ²。

4. **资产定价**：资产定价基本方程q_j = E[m·r^j]；CAPM：E[r^j] = r_f + β_j(E[r^m] - r_f)；风险中性定价：q_j = Ẽ[r^j]/(1+r_f)。

5. **保险需求**：精算公平保费下，风险厌恶消费者选择完全保险；保费率高于精算公平时，选择不完全保险。

6. **随机占优**：一阶随机占优（所有单调递增效用函数）；二阶随机占优（所有单调递增凹效用函数）。

7. **均值-方差分析**：均值-方差效用函数U = E[x] - 0.5·A·Var(x)；马科维茨投资组合理论；有效前沿和资本市场线。
