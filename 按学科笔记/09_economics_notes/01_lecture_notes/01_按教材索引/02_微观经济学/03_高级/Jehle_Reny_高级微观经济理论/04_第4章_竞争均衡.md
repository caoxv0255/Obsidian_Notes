# 第4章：竞争均衡

## 章节学习目标

本章介绍竞争均衡理论，这是微观经济学的核心内容之一，连接了个体行为和市场结果。通过本章的学习，读者应该能够：

1. 理解竞争均衡的概念和定义
2. 掌握瓦尔拉斯均衡的基本要素
3. 理解福利经济学第一和第二定理
4. 分析竞争均衡的存在性条件
5. 理解竞争均衡的唯一性条件
6. 掌握竞争均衡的稳定性分析
7. 理解超额需求函数及其性质
8. 掌握瓦尔拉斯定律
9. 分析竞争均衡的比较静态性质
10. 理解核与竞争均衡的关系

## 4.1 竞争均衡的概念

### 4.1.1 交换经济

**定义4.1.1（交换经济）**：交换经济E由以下要素组成：
1. 商品集合：n种商品，商品空间为ℝ^n
2. 消费者集合：I个消费者，i = 1, ..., I
3. 每个消费者i的偏好关系≽_i和初始禀赋ω_i ∈ ℝ^n_+

总禀赋为ω = ∑_{i=1}^I ω_i。

**定义4.1.2（可行分配）**：分配(x₁, ..., x_I)是可行的，如果：
1. x_i ∈ ℝ^n_+ 对所有i = 1, ..., I
2. ∑_{i=1}^I x_i = ω

**定义4.1.3（帕累托最优）**：可行分配(x₁, ..., x_I)是帕累托最优的，如果不存在另一个可行分配(x₁', ..., x_I')使得：
- x_i' ≽_i x_i 对所有i
- x_j' ≻_j x_j 对某个j

即不存在帕累托改进。

**定义4.1.4（竞争均衡）**：价格向量p ∈ ℝ^n_++和分配(x₁, ..., x_I)构成竞争均衡，如果：
1. x_i是消费者i在价格p和财富m_i = p·ω_i下的最优选择：x_i = x_i(p, m_i)
2. 市场出清：∑_{i=1}^I x_i = ω

**定理4.1.1（瓦尔拉斯定律）**：在均衡时，对所有商品k，超额需求为零：
z_k(p) = ∑_{i=1}^I (x_{ik} - ω_{ik}) = 0, k = 1, ..., n

**证明**：由消费者最优化，p·x_i = p·ω_i对所有i。求和得p·∑_{i=1}^I x_i = p·∑_{i=1}^I ω_i ⇒ p·z(p) = 0。如果价格p > 0，则z(p) = 0。□

### 4.1.2 生产经济

**定义4.1.5（生产经济）**：生产经济E由以下要素组成：
1. 商品集合：n种商品
2. 消费者集合：I个消费者，i = 1, ..., I
3. 厂商集合：J个厂商，j = 1, ..., J
4. 每个消费者i的偏好关系≽_i、初始禀赋ω_i ∈ ℝ^n_+和利润份额θ_{ij} ≥ 0
5. 每个厂商j的生产可能性集Y_j ⊆ ℝ^n

其中θ_{ij}表示消费者i拥有厂商j的利润份额，满足∑_{i=1}^I θ_{ij} = 1对所有j，∑_{j=1}^J θ_{ij} = 1对所有i。

**定义4.1.6（生产经济的可行分配）**：分配((x_i)_{i=1}^I, (y_j)_{j=1}^J)是可行的，如果：
1. x_i ∈ ℝ^n_+ 对所有i
2. y_j ∈ Y_j 对所有j
3. ∑_{i=1}^I x_i = ∑_{i=1}^I ω_i + ∑_{j=1}^J y_j

即总消费等于总禀赋加总净产出。

**定义4.1.7（生产经济的竞争均衡）**：价格向量p、分配((x_i), (y_j))构成竞争均衡，如果：
1. x_i是消费者i在价格p和财富m_i = p·ω_i + ∑_{j=1}^J θ_{ij} p·y_j下的最优选择
2. y_j是厂商j在价格p下的利润最大化生产计划
3. 市场出清：∑_{i=1}^I x_i = ∑_{i=1}^I ω_i + ∑_{j=1}^J y_j

**定理4.1.2（瓦尔拉斯定律（生产经济））**：在均衡时，p·z(p) = 0，其中z(p) = ∑_{i=1}^I x_i(p) - ∑_{i=1}^I ω_i - ∑_{j=1}^J y_j(p)是超额需求函数。

### 4.1.3 超额需求函数

**定义4.1.8（超额需求函数）**：在交换经济中，超额需求函数z(p) = ∑_{i=1}^I x_i(p, p·ω_i) - ω。

**定义4.1.9（归一化价格）**：由于价格是零次齐次的（只影响名义值，不影响实际分配），可以对价格进行归一化。常见的归一化方法包括：
1. ∑_{k=1}^n p_k = 1（价格和为1）
2. p₁ = 1（选择计价物）

**定理4.1.3（超额需求函数的性质）**：
1. 连续性：如果偏好是连续的且严格凸的，则z(p)是连续的。
2. 零次齐次性：z(αp) = z(p) 对所有α > 0。
3. 瓦尔拉斯定律：p·z(p) = 0 对所有p。
4. 正则性：如果p_k → 0，则z_k(p) → +∞（如果商品k是必需品）。

**证明（瓦尔拉斯定律）**：z(p) = ∑_{i=1}^I x_i(p, p·ω_i) - ω。由消费者最优化，p·x_i(p, p·ω_i) = p·ω_i。因此，p·z(p) = ∑_{i=1}^I p·x_i(p, p·ω_i) - p·ω = ∑_{i=1}^I p·ω_i - p·ω = 0。□

### 4.1.4 竞争均衡的几何解释

**定义4.1.10（价格线）**：在埃奇沃斯盒中，价格线是通过初始禀赋ω、斜率为-p₁/p₂的直线。

**定理4.1.4（竞争均衡的几何性质）**：在竞争均衡处，两个消费者的无差异曲线相切，且相切点位于价格线上。

**证明**：在竞争均衡处，x₁是消费者1的最优选择，因此价格线与消费者1的无差异曲线相切。同理，价格线也与消费者2的无差异曲线相切。因此，两条无差异曲线在均衡点相切。□

## 4.2 福利定理

### 4.2.1 福利经济学第一定理

**定理4.2.1（福利经济学第一定理）**：如果偏好是局部非饱和的，则任何竞争均衡都是帕累托最优的。

**证明**：使用反证法。假设竞争均衡(p*, x*)不是帕累托最优的，则存在可行分配x'使得x_i' ≽_i x_i*对所有i，且x_j' ≻_j x_j*对某个j。

由于x_i'是可行的，∑_{i=1}^I x_i' = ω。由局部非饱和性，如果x_i' ≽_i x_i*且x_i' ≠ x_i*，则p*·x_i' ≥ p*·x_i*。如果x_i' = x_i*，则p*·x_i' = p*·x_i*。因此，p*·x_i' ≥ p*·x_i*对所有i，且至少有一个严格不等式。

求和得：p*·∑_{i=1}^I x_i' > p*·∑_{i=1}^I x_i* = p*·ω

但x'是可行的，∑_{i=1}^I x_i' = ω，因此p*·ω > p*·ω，矛盾。因此，竞争均衡必须是帕累托最优的。□

**解释4.2.1（第一定理的经济学含义）**：竞争市场能够实现资源的有效配置，无需中央计划。这是"看不见的手"的数学表述。

**推论4.2.1**：竞争均衡不一定是公平的。帕累托最优性不涉及分配的公平性。

**例4.2.1（帕累托最优但不公平）**：考虑两个消费者，消费者1拥有所有禀赋，消费者2没有禀赋。竞争均衡是消费者1消费所有商品，消费者2消费零商品。这是帕累托最优的（无法在不损害消费者1的情况下改善消费者2），但显然不公平。

### 4.2.2 福利经济学第二定理

**定理4.2.2（福利经济学第二定理）**：如果：
1. 偏好是凸的（严格凸或凸）
2. 消费集合是凸的
3. 初始禀赋重新配置是可能的（通过一次性转移）

则任何帕累托最优分配都可以通过竞争均衡实现。

**证明概要**：
1. 设x*是帕累托最优分配。
2. 构造价格向量p*使得x*在给定价格和财富下是每个消费者的最优选择。
3. 使用分离超平面定理，找到支持上轮廓集的价格向量。
4. 验证这个价格向量支持x*作为竞争均衡。
5. 通过重新配置初始禀赋（ Lump-sum transfer），使x*成为竞争均衡。□

**解释4.2.2（第二定理的经济学含义）**：效率和分配可以分离。政府可以通过一次性转移实现公平的分配，然后让竞争市场实现效率。

**图形分析4.2.1**：
```

  |        •

```

**推论4.2.2**：如果偏好不是凸的，则帕累托最优分配可能无法通过竞争均衡实现。

**例4.2.2（非凸偏好与第二定理）**：如果无差异曲线不是凸的（例如凹向原点），则竞争均衡可能不存在，或存在但不是帕累托最优的。

### 4.2.3 福利定理的应用

**应用4.2.1（市场失灵）**：福利第一定理的假设不满足时，市场失灵可能发生：
1. 外部性：私人成本不等于社会成本
2. 公共物品：非排他性和非竞争性
3. 信息不对称：道德风险和逆向选择
4. 垄断：不完全竞争

**应用4.2.2（政策含义）**：福利第二定理表明，政府可以通过一次性转移实现公平，而不损害效率。这与扭曲性税收（如所得税、消费税）不同，后者会扭曲市场行为。

## 4.3 竞争均衡的存在性

### 4.3.1 存在性定理

**定理4.3.1（竞争均衡存在性（瓦尔拉斯））**：在交换经济中，如果满足：
1. 偏好是完备的、传递的、连续的、严格凸的、单调的
2. 消费集合X_i = ℝ^n_+
3. 总禀赋ω > 0

则竞争均衡存在。

**证明概要**（使用角谷静夫不动点定理）：
1. 构造价格调整函数（瓦尔拉斯拍卖人）。
2. 证明价格调整函数是连续的、值在单纯形内。
3. 由角谷静夫不动点定理，存在不动点。
4. 证明这个不动点就是竞争均衡。□

**定理4.3.2（竞争均衡存在性（德布鲁-斯卡夫））**：在有生产的竞争经济中，如果满足：
1. 偏好是连续的、严格凸的、单调的
2. 生产集合Y_j是闭的、凸的，且满足自由处置
3. 存在可行分配

则竞争均衡存在。

### 4.3.2 不动点定理的应用

**定理4.3.3（角谷静夫不动点定理）**：设Γ: Δ → Δ是对应，其中Δ是单纯形。如果Γ满足：
1. 对每个p ∈ Δ，Γ(p)是非空凸集。
2. Γ是上半连续的。

则Γ在Δ中至少有一个不动点，即存在p* ∈ Δ使得p* ∈ Γ(p*)。

**应用4.3.1（构造价格调整对应）**：
定义Γ(p) = { p' ∈ Δ | p'·z(p) ≥ 0 }

解释：如果商品k的超额需求z_k(p) > 0（供不应求），则价格p_k应该上升（p'_k ≥ p_k）。这个价格调整对应满足角谷静夫定理的条件，因此存在不动点p*，使得p*·z(p*) ≥ 0。由瓦尔拉斯定律p*·z(p*) = 0，得z(p*) = 0，即竞争均衡。

**图形分析4.3.1**：
```
价格单纯形（三种商品）:

价格3
  |
  |     Δ
  |    /|\
  |   / | \
  |  /  |  \
  | /   |   \
  |/____|____\ 价格2
  p*    |      \
         |       \
         |        \
         |_________ 价格1

p*是价格调整对应Γ的不动点
```

### 4.3.3 存在性条件的讨论

**条件4.3.1（偏好的连续性）**：如果偏好不连续，效用函数可能不存在，或者需求函数可能不连续，导致均衡不存在。

**条件4.3.2（偏好的凸性）**：如果偏好不凸，需求对应可能不是凸值的，导致价格调整对应不满足角谷静夫定理的条件。

**条件4.3.3（生产集合的凸性）**：如果生产集合不凸，供给对应可能不连续，导致均衡不存在。

**例4.3.1（不可分生产技术）**：如果生产技术存在固定成本或规模报酬递增，生产集合可能不凸，竞争均衡可能不存在。

## 4.4 竞争均衡的唯一性

### 4.4.1 唯一性条件

**定理4.4.1（总替代条件下的唯一性）**：如果超额需求函数z(p)满足总替代条件（gross substitutes），即∂z_k/∂p_l > 0对所有k ≠ l，则竞争均衡唯一。

**证明概要**：使用反证法。假设存在两个不同的均衡p*和p**。由于价格可以归一化，假设p*_1/p*_2 > p**_1/p**_2。由总替代条件，价格比率的变化导致替代效应，使得超额需求不为零，与均衡定义矛盾。□

**定理4.4.2（弱公理下的唯一性）**：如果超额需求函数满足弱公理（WA），即对于任意p, p'，如果p·z(p') ≤ 0，则p'·z(p) ≤ 0，则竞争均衡唯一。

**解释4.4.1（弱公理的经济学含义）**：弱公理禁止了显示偏好循环，确保了需求行为的理性性。

### 4.4.2 多重均衡

**例4.4.1（多重均衡）**：考虑两个消费者，两种商品。消费者1的效用函数u₁(x₁, x₂) = min{x₁, x₂}，消费者2的效用函数u₂(x₁, x₂) = min{x₁, x₂}。初始禀赋ω₁ = (1, 0)，ω₂ = (0, 1)。

在这个经济中，存在多个竞争均衡。例如，p = (1, 1)是一个均衡，x₁ = (0.5, 0.5)，x₂ = (0.5, 0.5)。另一个均衡可能是p = (α, 1-α)，其中α ∈ (0, 1)，分配x₁ = (α, α)，x₂ = (1-α, 1-α)。

**图形分析4.4.1**：

**定理4.4.3（均衡指数定理）**：在某些条件下，均衡的个数是奇数（如果有限）。这是指数定理的一个应用。

## 4.5 竞争均衡的稳定性

### 4.5.1 塔顿拍卖过程

**定义4.5.1（塔顿拍卖过程）**：价格调整过程：
p_{k}(t+1) = p_{k}(t) + α_k z_k(p(t)), k = 1, ..., n

其中α_k > 0是调整速度，z_k(p(t))是商品k的超额需求。

**解释4.5.1（塔顿过程的经济学含义）**：如果商品k供不应求（z_k(p(t)) > 0），价格p_k上升；如果供过于求（z_k(p(t)) < 0），价格p_k下降。这是"瓦尔拉斯拍卖人"的价格调整机制。

**定理4.5.1（塔顿过程的稳定性）**：如果满足：
1. 超额需求函数z(p)是连续的
2. 总替代条件成立
3. 价格调整速度α_k足够小

则塔顿过程收敛于竞争均衡。

**证明概要**：构造李雅普诺夫函数（势函数），证明在总替代条件下，这个李雅普诺夫函数单调递减且有下界，因此过程收敛。□

### 4.5.2 全局稳定性

**定义4.5.2（全局稳定性）**：竞争均衡p*是全局稳定的，如果对于任意初始价格p(0)，价格调整过程都收敛于p*。

**定理4.5.2（Arrow-Hahn稳定性定理）**：如果超额需求函数z(p)满足：
1. 连续性
2. 瓦尔拉斯定律
3. 正则性（如果p_k → 0，则z_k(p) → +∞）
4. 弱公理

则竞争均衡是全局稳定的。

### 4.5.3 局部稳定性

**定义4.5.3（局部稳定性）**：竞争均衡p*是局部稳定的，如果存在p*的邻域U，使得对于任意p(0) ∈ U，价格调整过程都收敛于p*。

**定理4.5.3（局部稳定性条件）**：线性化价格调整过程在均衡点p*处：
p(t+1) - p* ≈ Dz(p*) · (p(t) - p*)

均衡p*局部稳定的充要条件是矩阵Dz(p*)的所有特征值的实部小于零（负实部）。

**证明**：线性化后，系统p(t+1) - p* = Dz(p*) · (p(t) - p*)的解收敛于零当且仅当Dz(p*)的特征值的模小于1（对于离散时间）或实部小于零（对于连续时间）。□

**图形分析4.5.1**：
```
价格调整过程的相图:

价格p₂
  |
  |      p*（均衡）
```

### 4.5.4 稳定性与总替代

**定理4.5.4（总替代与稳定性）**：如果超额需求函数满足总替代条件，则竞争均衡是全局稳定的。

**解释4.5.2（总替代的经济学含义）**：总替代意味着价格上升导致对该商品需求下降，对其他商品需求上升（替代效应）。这确保了价格调整过程单调收敛。

## 4.6 比较静态分析

### 4.6.1 比较静态的一般方法

**定理4.6.1（均衡比较静态）**：设均衡条件为F(p, θ) = 0，其中θ是外生参数。均衡价格p*(θ)由隐函数定义。如果∂F/∂p可逆，则：
dp*/dθ = - (∂F/∂p)^{-1} (∂F/∂θ)

**应用4.6.1（禀赋变化的比较静态）**：考虑交换经济，总禀赋从ω变为ω'。均衡价格p*(ω)由z(p*(ω), ω) = 0定义。对ω求导：
D_p z(p*, ω) · (dp*/dω) + D_ω z(p*, ω) = 0

因此，dp*/dω = - [D_p z(p*, ω)]^{-1} D_ω z(p*, ω)

### 4.6.2 斯卢茨基方程的加总

**定理4.6.2（斯卢茨基方程的加总）**：总超额需求函数满足：
S(p, m) = ∑_{i=1}^I S_i(p, m_i)

其中S_i(p, m_i)是消费者i的斯卢茨基替代矩阵。

**解释4.6.1（加总的经济学含义）**：总替代效应是个人替代效应的加总。这允许我们从个人需求加总到市场超额需求。

### 4.6.3 对称性

**定理4.6.3（总超额需求的对称性）**：如果所有消费者都具有相同的位似偏好，则总超额需求函数满足对称性：
∂z_k/∂p_l = ∂z_l/∂p_k

**证明**：如果所有消费者具有相同的位似偏好，则总超额需求函数可以表示为z(p) = f(p)·ω，其中f(p)是价格向量p的函数。由对称性，∂f_k/∂p_l = ∂f_l/∂p_k，因此∂z_k/∂p_l = ∂z_l/∂p_k。□

### 4.6.4 齐次性与归一化

**定理4.6.4（超额需求的零次齐次性）**：z(αp) = z(p) 对所有α > 0。

**推论4.6.1**：由于价格只影响名义值，不影响实际分配，可以对价格进行归一化。常见的归一化包括：
1. ∑_{k=1}^n p_k = 1
2. p₁ = 1（计价物）

**应用4.6.2（计价物选择）**：选择商品1作为计价物，p₁ = 1。比较静态分析在(n-1)维价格空间进行。

## 4.7 核与竞争均衡

### 4.7.1 核的概念

**定义4.7.1（联盟）**：联盟S是消费者集合I的子集，S ⊆ I。

**定义4.7.2（联盟的可行分配）**：分配(x_i)_{i∈S}对联盟S是可行的，如果∑_{i∈S} x_i ≤ ∑_{i∈S} ω_i。

**定义4.7.3（联盟S改进分配(x_i)）**：如果存在联盟S和可行分配(x_i')_{i∈S}使得x_i' ≽_i x_i对所有i∈S，且x_j' ≻_j x_j对某个j∈S，则联盟S可以改进分配(x_i)。

**定义4.7.4（核）**：核是无法被任何联盟改进的可行分配集合。

**定理4.7.1（竞争均衡在核中）**：任何竞争均衡分配都在核中。

**证明**：使用反证法。假设竞争均衡分配x*不在核中，则存在联盟S和可行分配x'使得x_i' ≽_i x_i*对所有i∈S，且x_j' ≻_j x_j*对某个j∈S。由消费者最优化，p*·x_i* = p*·ω_i对所有i∈S。由于x_i' ≽_i x_i*，且至少有一个严格偏好，有p*·x_i' ≥ p*·ω_i对所有i∈S，且至少有一个严格不等式。求和得p*·∑_{i∈S} x_i' > p*·∑_{i∈S} ω_i，与x'对S的可行性矛盾。□

### 4.7.2 德布鲁-斯卡夫极限定理

**定理4.7.2（德布鲁-斯卡夫极限定理）**：考虑经济复制I次（每个消费者类型复制I次）。随着I → ∞，核收敛于竞争均衡分配集合。

**解释4.7.1（极限定理的经济学含义）**：当经济规模增大时，非竞争均衡的核分配逐渐消失，只剩下竞争均衡分配。这支持了竞争市场在大规模经济中的有效性。

### 4.7.3 核的性质

**定理4.7.3（核的非空性）**：在交换经济中，如果偏好是连续的、单调的，则核非空。

**定理4.7.4（核与帕累托最优）**：核是帕累托最优分配集合的子集。

**证明**：如果分配x*不是帕累托最优的，则存在可行分配x'使得x_i' ≽_i x_i*对所有i，且x_j' ≻_j x_j*对某个j。取联盟S = I（所有消费者），则S可以改进x*，因此x*不在核中。□

**推论4.7.1**：竞争均衡 ⇒ 帕累托最优 ⇒ 核分配

## 本章总结

### 核心要点

1. **竞争均衡**：价格向量p和分配(x_i)使得每个消费者在给定价格和财富下最优，市场出清。

2. **福利经济学第一定理**：如果偏好是局部非饱和的，则任何竞争均衡都是帕累托最优的。

3. **福利经济学第二定理**：如果偏好是凸的，则任何帕累托最优分配都可以通过竞争均衡实现（配合禀赋重新配置）。

4. **瓦尔拉斯定律**：p·z(p) = 0，即超额需求的价值为零。

5. **竞争均衡存在性**：在偏好连续、严格凸、单调的条件下，竞争均衡存在（使用角谷静夫不动点定理）。

6. **竞争均衡唯一性**：在总替代条件下，竞争均衡唯一。

7. **竞争均衡稳定性**：在总替代条件下，塔顿拍卖过程全局收敛于竞争均衡。

8. **核与竞争均衡**：竞争均衡在核中；随着经济规模增大，核收敛于竞争均衡。

### 重要概念表格

| 概念 | 定义/性质 | 经济学含义 |
|------|-----------|------------|
