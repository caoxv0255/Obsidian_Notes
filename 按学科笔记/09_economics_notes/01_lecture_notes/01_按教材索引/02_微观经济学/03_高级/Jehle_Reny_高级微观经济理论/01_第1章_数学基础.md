# 第1章：数学基础

## 章节学习目标

本章旨在为学习高级微观经济学提供必要的数学工具和理论基础。通过本章的学习，读者应该能够：

1. 掌握集合论的基本概念和运算
2. 理解函数与对应的关系及其在经济分析中的应用
3. 熟悉拓扑学基础，特别是开集、闭集、紧性等概念
4. 掌握凸集和凸函数的理论及其在经济模型中的重要性
5. 理解优化理论，包括无约束优化和约束优化
6. 掌握拉格朗日乘数法和库恩-塔克条件

## 1.1 集合论基础

### 1.1.1 集合的基本概念

**定义1.1.1（集合）**：集合是具有某种共同性质的元素的汇集。我们用大写字母A, B, C...表示集合，用小写字母a, b, c...表示集合的元素。如果元素x属于集合A，记作x ∈ A；如果x不属于A，记作x ∉ A。

**定义1.1.2（子集）**：如果集合A的所有元素都是集合B的元素，即∀x ∈ A ⇒ x ∈ B，则称A是B的子集，记作A ⊆ B。如果A ⊆ B且B ⊆ A，则A = B。

**定义1.1.3（空集）**：空集∅是不包含任何元素的集合。空集是任何集合的子集。

**定义1.1.4（幂集）**：集合A的幂集P(A)是A的所有子集构成的集合。

**例1.1.1**：设A = {1, 2, 3}，则P(A) = {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}。

### 1.1.2 集合的运算

**定义1.1.5（并集）**：集合A和B的并集A ∪ B = {x | x ∈ A 或 x ∈ B}。

**定义1.1.6（交集）**：集合A和B的交集A ∩ B = {x | x ∈ A 且 x ∈ B}。如果A ∩ B = ∅，则称A和B不相交。

**定义1.1.7（差集）**：集合A和B的差集A \ B = {x | x ∈ A 且 x ∉ B}。

**定义1.1.8（补集）**：设全集为X，集合A的补集A^c = X \ A = {x ∈ X | x ∉ A}。

**德摩根定律（De Morgan's Laws）**：
- (A ∪ B)^c = A^c ∩ B^c
- (A ∩ B)^c = A^c ∪ B^c

**定理1.1.1（分配律）**：
- A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
- A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)

**证明**：我们证明第一个等式。设x ∈ A ∩ (B ∪ C)，则x ∈ A且x ∈ (B ∪ C)。这意味着x ∈ A且（x ∈ B或x ∈ C）。如果x ∈ B，则x ∈ A ∩ B；如果x ∈ C，则x ∈ A ∩ C。因此，x ∈ (A ∩ B) ∪ (A ∩ C)。反向类似可证。□

**例1.1.2**：设消费集合为X = {x ∈ ℝ^n_+ | x₁ + x₂ ≤ 100}，预算集合为B = {x ∈ X | p₁x₁ + p₂x₂ ≤ m}，其中p₁, p₂ > 0为价格，m > 0为收入。则可行消费集合为C = X ∩ B。

### 1.1.3 笛卡尔积

**定义1.1.9（笛卡尔积）**：集合A和B的笛卡尔积A × B = {(a, b) | a ∈ A, b ∈ B}。

推广到有限个集合：A₁ × A₂ × ... × A_n = {(a₁, a₂, ..., a_n) | a_i ∈ A_i, i = 1, ..., n}。

特别地，ℝ^n = ℝ × ℝ × ... × ℝ（n次），表示n维欧几里得空间。

**例1.1.3**：设商品空间为ℝ²，消费束x = (x₁, x₂)表示两种商品的消费量。如果商品1的取值范围是[0, 10]，商品2的取值范围是[0, 20]，则消费空间为[0, 10] × [0, 20]。

### 1.1.4 关系与等价关系

**定义1.1.10（二元关系）**：集合X上的二元关系R是X × X的子集。如果(x, y) ∈ R，我们记作xRy。

**定义1.1.11（等价关系）**：集合X上的关系R称为等价关系，如果满足：
1. 自反性：∀x ∈ X, xRx
2. 对称性：∀x, y ∈ X, xRy ⇒ yRx
3. 传递性：∀x, y, z ∈ X, xRy且yRz ⇒ xRz

**定义1.1.12（等价类）**：设R是X上的等价关系，x ∈ X的等价类[x] = {y ∈ X | xRy}。

**定理1.1.2（划分定理）**：X上的等价关系将X划分为若干个不相交的等价类。

**例1.1.4**：在消费者理论中，"无差异"关系是一个等价关系。两个消费束x和y无差异（记作x ~ y）意味着消费者对它们的偏好相同。无差异关系满足自反性（x与自身无差异）、对称性（如果x与y无差异，则y与x无差异）和传递性（如果x与y无差异，y与z无差异，则x与z无差异）。

### 1.1.5 偏序关系

**定义1.1.13（偏序关系）**：集合X上的关系≲称为偏序关系，如果满足：
1. 自反性：∀x ∈ X, x ≲ x
2. 反对称性：∀x, y ∈ X, x ≲ y且y ≲ x ⇒ x = y
3. 传递性：∀x, y, z ∈ X, x ≲ y且y ≲ z ⇒ x ≲ z

**定义1.1.14（全序关系）**：如果偏序关系≲满足∀x, y ∈ X，x ≲ y或y ≲ x，则称为全序关系。

**例1.1.5**：在ℝ^n上，定义x ≲ y当且仅当x_i ≤ y_i对所有i = 1, ..., n成立。这是一个偏序关系，但不是全序关系（除非n = 1）。例如，(1, 2)和(2, 1)是不可比较的。

## 1.2 函数与对应

### 1.2.1 函数的定义

**定义1.2.1（函数）**：从集合X到集合Y的函数f是一个规则，使得对于每个x ∈ X，都有唯一的y ∈ Y与之对应，记作y = f(x)。X称为定义域，Y称为值域。

**定义1.2.2（像）**：函数f: X → Y的像（或值域）为f(X) = {f(x) | x ∈ X}。

**定义1.2.3（单射）**：函数f: X → Y称为单射（或一对一），如果f(x₁) = f(x₂) ⇒ x₁ = x₂。

**定义1.2.4（满射）**：函数f: X → Y称为满射（或映上），如果f(X) = Y。

**定义1.2.5（双射）**：如果f既是单射又是满射，则称为双射（或一一对应）。

**定义1.2.6（逆函数）**：如果f: X → Y是双射，则存在逆函数f⁻¹: Y → X，使得f⁻¹(f(x)) = x和f(f⁻¹(y)) = y对所有x ∈ X, y ∈ Y成立。

**例1.2.1**：效用函数u: X → ℝ将消费束映射为实数，表示消费者的满足程度。如果效用函数是严格单调的，则它是单射（不同的消费束有不同的效用值）。

### 1.2.2 对应

**定义1.2.7（对应）**：从集合X到集合Y的对应Γ是一个规则，使得对于每个x ∈ X，都有Y的一个子集Γ(x)与之对应。Γ(x)称为x的像。

**定义1.2.8（对应与函数的关系）**：函数是对应的一种特殊情况，即Γ(x)只包含一个元素。

**定义1.2.9（对应的上半连续）**：设Γ: X → Y是对应，x₀ ∈ X。Γ在x₀处上半连续，如果对于每个满足Γ(x₀) ⊆ O的开集O，存在x₀的邻域U，使得Γ(x) ⊆ O对所有x ∈ U成立。

**定义1.2.10（对应的下半连续）**：设Γ: X → Y是对应，x₀ ∈ X。Γ在x₀处下半连续，如果对于每个满足Γ(x₀) ∩ O ≠ ∅的开集O，存在x₀的邻域U，使得Γ(x) ∩ O ≠ ∅对所有x ∈ U成立。

**定义1.2.11（对应的连续性）**：如果Γ在x₀处既是上半连续又是下半连续，则称Γ在x₀处连续。

**例1.2.2**：需求对应x(p, m)表示在价格p和收入m下的最优消费束集合。如果效用函数是严格凸的，需求对应退化为需求函数（唯一解）。如果效用函数是凸的但不是严格凸的，需求可能是一个集合（对应）。

### 1.2.3 复合函数

**定义1.2.12（复合函数）**：设f: X → Y和g: Y → Z是两个函数，则复合函数g ∘ f: X → Z定义为(g ∘ f)(x) = g(f(x))。

**定理1.2.1（复合函数的单调性）**：
- 如果f和g都是单调递增的，则g ∘ f也是单调递增的。
- 如果f是单调递增的，g是单调递减的，则g ∘ f是单调递减的。

**例1.2.3**：间接效用函数V(p, m) = u(x(p, m))是效用函数u(x)和马歇尔需求函数x(p, m)的复合。

### 1.2.4 连续函数

**定义1.2.13（连续函数）**：函数f: X → Y在点x₀ ∈ X处连续，如果对于任意ε > 0，存在δ > 0，使得当d_X(x, x₀) < δ时，有d_Y(f(x), f(x₀)) < ε。如果f在X的每一点都连续，则称f在X上连续。

**定理1.2.2（连续函数的性质）**：
- 两个连续函数的和、差、积仍是连续函数（在定义域内）。
- 连续函数的复合仍是连续函数。
- 连续函数的像保持连通性和紧性。

**例1.2.4**：效用函数通常是连续的，这反映了偏好的连续性假设。

## 1.3 拓扑学基础

### 1.3.1 度量空间

**定义1.3.1（度量）**：集合X上的度量d: X × X → ℝ_+是一个函数，满足：
1. 非负性：d(x, y) ≥ 0，且d(x, y) = 0当且仅当x = y
2. 对称性：d(x, y) = d(y, x)
3. 三角不等式：d(x, z) ≤ d(x, y) + d(y, z)

**定义1.3.2（度量空间）**：集合X配备度量d称为度量空间，记作(X, d)。

**例1.3.1**：在ℝ^n上，欧几里得度量定义为d(x, y) = √[∑_{i=1}^n (x_i - y_i)²]。曼哈顿度量定义为d(x, y) = ∑_{i=1}^n |x_i - y_i|。

**定义1.3.3（开球）**：在度量空间(X, d)中，以x₀为中心、半径为r > 0的开球为B(x₀, r) = {x ∈ X | d(x, x₀) < r}。

### 1.3.2 开集与闭集

**定义1.3.4（开集）**：集合A ⊆ X是开集，如果对于每个x ∈ A，存在r > 0，使得B(x, r) ⊆ A。

**定义1.3.5（闭集）**：集合A ⊆ X是闭集，如果其补集A^c是开集。

**定理1.3.1（开集和闭集的性质）**：
1. 任意个开集的并集是开集，有限个开集的交集是开集。
2. 任意个闭集的交集是闭集，有限个闭集的并集是闭集。
3. X和∅既是开集又是闭集。

**定义1.3.6（内部）**：集合A的内部int(A)是A包含的最大开集。

**定义1.3.7（闭包）**：集合A的闭包cl(A)是包含A的最小闭集。

**定义1.3.8（边界）**：集合A的边界∂A = cl(A) \ int(A)。

**例1.3.2**：在ℝ中，开区间(a, b)是开集，闭区间[a, b]是闭集。半开区间[a, b)既不是开集也不是闭集。

**图形分析1.3.1**：
```
闭区间[a, b]的边界:
     a         b
     |---------|
     ↑         ↑
   边界点   边界点
```

### 1.3.3 紧性

**定义1.3.9（紧集）**：集合A ⊆ X是紧集，如果A的每个开覆盖都有有限子覆盖。即，如果{O_α}是一族开集，使得A ⊆ ∪_α O_α，则存在有限个O_α₁, ..., O_α_k，使得A ⊆ ∪_{i=1}^k O_αᵢ。

**定理1.3.2（海涅-博雷尔定理）**：在ℝ^n中，集合A是紧集当且仅当A是有界闭集。

**定理1.3.3（紧集的性质）**：
- 紧集的闭子集是紧集。
- 紧集的有限并集是紧集。
- 紧集在连续函数下的像是紧集。
- 紧集上的连续函数达到最大值和最小值（极值定理）。

**例1.3.3**：在消费者理论中，消费集合通常假设为有界闭集（紧集），以确保最优解的存在性。例如，预算集合B = {x ∈ ℝ^n_+ | p·x ≤ m}是紧集。

**证明1.3.1（预算集合的紧性）**：
预算集合B = {x ∈ ℝ^n_+ | p·x ≤ m}，其中p > 0, m > 0。
1. 有界性：对于任意x ∈ B，p·x = ∑_{i=1}^n p_i x_i ≤ m。由于p_i > 0，x_i ≤ m/p_i对所有i成立。因此，0 ≤ x_i ≤ m/p_i，B是有界的。
2. 闭性：考虑连续函数f(x) = p·x。B = {x ∈ ℝ^n_+ | f(x) ≤ m}是f的上水平集，而连续函数的上水平集是闭集。因此，B是闭集。
根据海涅-博雷尔定理，B是紧集。□

### 1.3.4 连通性

**定义1.3.10（连通集）**：集合A是连通集，如果A不能表示为两个不相交的非空开集的并集。

**定理1.3.4**：在ℝ中，区间[a, b]是连通集。

**例1.3.4**：在消费者理论中，偏好无差异曲线通常是连通的，这反映了偏好的连续性。

### 1.3.5 序列与收敛

**定义1.3.11（序列）**：集合X中的序列是一个函数x: ℕ → X，记作{x_k}_{k=1}^∞。

**定义1.3.12（收敛）**：序列{x_k}在度量空间(X, d)中收敛于x ∈ X，如果对于任意ε > 0，存在N ∈ ℕ，使得对所有k ≥ N，有d(x_k, x) < ε。记作x_k → x。

**定理1.3.5（收敛与闭集）**：集合A是闭集当且仅当A中每个收敛序列的极限都在A中。

**定义1.3.13（柯西序列）**：序列{x_k}是柯西序列，如果对于任意ε > 0，存在N ∈ ℕ，使得对所有k, l ≥ N，有d(x_k, x_l) < ε。

**定义1.3.14（完备度量空间）**：度量空间(X, d)是完备的，如果X中每个柯西序列都收敛于X中的点。

**定理1.3.6**：ℝ^n配备欧几里得度量是完备度量空间。

## 1.4 凸性理论

### 1.4.1 凸集

**定义1.4.1（凸集）**：集合A ⊆ ℝ^n是凸集，如果对于任意x, y ∈ A和λ ∈ [0, 1]，有λx + (1-λ)y ∈ A。

**图形分析1.4.1**：
```
凸集 vs 非凸集:

凸集:          非凸集:
  ●───●          ●───●
 /     \        |     |
●       ●      ●       ●
 \     /        |     |
  ●───●          ●───●

凸集：任意两点的连线都在集合内
非凸集：某些点的连线不在集合内
```

**定理1.4.1（凸集的性质）**：
- 凸集的交集是凸集。
- 凸集的卡氏积是凸集。
- 凸集的线性组合是凸集。

**证明1.4.1**：设A₁, A₂是凸集。令x, y ∈ A₁ ∩ A₂，λ ∈ [0, 1]。由于x, y ∈ A₁且A₁是凸集，λx + (1-λ)y ∈ A₁。同理，λx + (1-λ)y ∈ A₂。因此，λx + (1-λ)y ∈ A₁ ∩ A₂，A₁ ∩ A₂是凸集。□

**定义1.4.2（凸包）**：集合A的凸包co(A)是包含A的最小凸集。

**定理1.4.2（凸包的表示）**：A的凸包可以表示为：
co(A) = {∑_{i=1}^k λ_i x_i | k ∈ ℕ, x_i ∈ A, λ_i ≥ 0, ∑_{i=1}^k λ_i = 1}

**例1.4.1**：消费集合X = ℝ^n_+（所有非负消费束的集合）是凸集。如果x, y ∈ ℝ^n_+，则对于λ ∈ [0, 1]，λx + (1-λ)y的每个分量都是非负的，因此属于ℝ^n_+。

**例1.4.2**：预算集合B = {x ∈ ℝ^n_+ | p·x ≤ m}是凸集。证明：设x, y ∈ B，即p·x ≤ m, p·y ≤ m。对于λ ∈ [0, 1]，p·(λx + (1-λ)y) = λp·x + (1-λ)p·y ≤ λm + (1-λ)m = m。因此，λx + (1-λ)y ∈ B，B是凸集。

### 1.4.2 凸函数与凹函数

**定义1.4.3（凸函数）**：函数f: A → ℝ（A ⊆ ℝ^n是凸集）是凸函数，如果对于任意x, y ∈ A和λ ∈ [0, 1]，有：
f(λx + (1-λ)y) ≤ λf(x) + (1-λ)f(y)

**定义1.4.4（严格凸函数）**：函数f是严格凸的，如果对于任意x ≠ y和λ ∈ (0, 1)，有：
f(λx + (1-λ)y) < λf(x) + (1-λ)f(y)

**定义1.4.5（凹函数）**：函数f: A → ℝ是凹函数，如果-f是凸函数，即：
f(λx + (1-λ)y) ≥ λf(x) + (1-λ)f(y)

**定义1.4.6（拟凸函数）**：函数f: A → ℝ是拟凸函数，如果其上水平集{x ∈ A | f(x) ≥ t}对所有t ∈ ℝ都是凸集。

**定义1.4.7（拟凹函数）**：函数f: A → ℝ是拟凹函数，如果其下水平集{x ∈ A | f(x) ≤ t}对所有t ∈ ℝ都是凸集。

**图形分析1.4.2**：
```
凸函数的图形:     凹函数的图形:
      /|           |\ 
     / |            |\
    /  |            | \
   /   |            |  \
  /____|            |___\
  
凸函数：图形位于弦的下方
凹函数：图形位于弦的上方
```

**定理1.4.3（一阶条件）**：设f: ℝ^n → ℝ是连续可微函数，则：
- f是凸函数当且仅当f(y) ≥ f(x) + ∇f(x)·(y-x)对所有x, y成立。
- f是凹函数当且仅当f(y) ≤ f(x) + ∇f(x)·(y-x)对所有x, y成立。

**定理1.4.4（二阶条件）**：设f: ℝ^n → ℝ是二阶连续可微函数，则：
- f是凸函数当且仅当海塞矩阵∇²f(x)对所有x都是半正定的。
- f是凹函数当且仅当海塞矩阵∇²f(x)对所有x都是半负定的。

**证明1.4.2（二阶条件的充分性）**：设∇²f(x)对所有x都是半正定的。由泰勒展开，f(y) = f(x) + ∇f(x)·(y-x) + (1/2)(y-x)ᵀ∇²f(ξ)(y-x)，其中ξ位于x和y之间。由于∇²f(ξ)半正定，第二项非负，因此f(y) ≥ f(x) + ∇f(x)·(y-x)。由定理1.4.3，f是凸函数。□

**例1.4.3**：效用函数u(x) = ∑_{i=1}^n α_i ln(x_i)（α_i > 0）是严格拟凹的，因为它是严格凹函数（海塞矩阵负定）。这意味着无差异曲线是严格凸向原点的。

**例1.4.4**：成本函数c(w, y)通常是凸函数，因为边际成本递增。

### 1.4.3 分离超平面定理

**定义1.4.8（超平面）**：ℝ^n中的超平面是集合H = {x ∈ ℝ^n | p·x = c}，其中p ≠ 0是法向量，c是常数。

**定义1.4.9（半空间）**：超平面H = {x | p·x = c}将空间分为两个闭半空间：{x | p·x ≥ c}和{x | p·x ≤ c}。

**定理1.4.5（分离超平面定理）**：设A和B是ℝ^n中两个不相交的凸集，且A是紧集，B是闭集。则存在p ≠ 0和c ∈ ℝ，使得：
p·x ≤ c < p·y 对所有x ∈ A, y ∈ B成立。

**定理1.4.6（支撑超平面定理）**：设A是ℝ^n中的凸集，x₀是A的边界点。则存在p ≠ 0，使得p·x ≤ p·x₀对所有x ∈ A成立。超平面H = {x | p·x = p·x₀}称为A在x₀处的支撑超平面。

**图形分析1.4.3**：
```
支撑超平面:

        H (支撑超平面)
        | 
        | p·x = p·x₀
        | 
   _____|_____ 
  /     |     \
 |  A   |   x₀|
 |      |      |
 \_____|_____/
       |
       |
```

**应用1.4.1（消费者理论中的价格支撑）**：在最优消费束x*处，预算超平面p·x = m是上轮廓集{x | u(x) ≥ u(x*)}的支撑超平面。这表明价格向量p支撑了偏好集合。

### 1.4.4 次微分

**定义1.4.10（次梯度）**：设f: ℝ^n → ℝ是凸函数，x₀ ∈ ℝ^n。向量p ∈ ℝ^n称为f在x₀处的次梯度，如果：
f(x) ≥ f(x₀) + p·(x - x₀) 对所有x ∈ ℝ^n成立。

**定义1.4.11（次微分）**：f在x₀处的次微分∂f(x₀)是所有次梯度的集合。

**定理1.4.7（次微分的性质）**：
- 如果f在x₀处可微，则∂f(x₀) = {∇f(x₀)}。
- ∂f(x₀)总是非空、闭集和凸集。
- f在x₀处取得最小值当且仅当0 ∈ ∂f(x₀)。

**应用1.4.2（成本函数的对偶性）**：成本函数c(w, y)在w处的次微分与条件要素需求对应相关。

## 1.5 优化理论

### 1.5.1 无约束优化

**定义1.5.1（局部极大值）**：函数f: ℝ^n → ℝ在x*处取得局部极大值，如果存在δ > 0，使得对所有满足‖x - x*‖ < δ的x，有f(x) ≤ f(x*)。

**定义1.5.2（全局极大值）**：函数f在x*处取得全局极大值，如果对所有x ∈ ℝ^n，有f(x) ≤ f(x*)。

**定理1.5.1（费马定理-一阶必要条件）**：设f: ℝ^n → ℝ在x*处可微，且x*是f的局部极值点，则∇f(x*) = 0。

**证明1.5.1**：假设x*是局部极大值点。考虑方向d ∈ ℝ^n，定义φ(t) = f(x* + td)。由于x*是局部极大值，φ(t)在t = 0处取得局部极大值。因此，φ'(0) = ∇f(x*)·d = 0。由于这对所有方向d都成立，必有∇f(x*) = 0。□

**定理1.5.2（二阶充分条件）**：设f: ℝ^n → ℝ是二阶连续可微函数，∇f(x*) = 0。如果∇²f(x*)是负定的，则x*是f的严格局部极大值点。如果∇²f(x*)是正定的，则x*是f的严格局部极小值点。

**定理1.5.3（二阶必要条件）**：设f: ℝ^n → ℝ是二阶连续可微函数，x*是f的局部极大值点。如果∇f(x*) = 0，则∇²f(x*)是半负定的。

**例1.5.1**：考虑效用最大化问题max u(x) s.t. x ∈ X。如果X是开集，且u是可微的，则在最优解x*处，∇u(x*) = 0（如果存在内部解）。

### 1.5.2 等式约束优化

**问题1.5.1（等式约束优化）**：
max_x f(x)
s.t. g_i(x) = 0, i = 1, ..., m

其中f: ℝ^n → ℝ, g_i: ℝ^n → ℝ是连续可微函数。

**定义1.5.3（约束规范）**：在可行点x*处，约束规范满足，如果约束函数的梯度∇g₁(x*), ..., ∇g_m(x*)是线性无关的。

**定理1.5.4（拉格朗日乘数定理）**：设x*是问题1.5.1的局部最优解，且约束规范在x*处成立。则存在拉格朗日乘数λ₁, ..., λ_m ∈ ℝ，使得：
∇f(x*) + ∑_{i=1}^m λ_i ∇g_i(x*) = 0

这些方程称为拉格朗日条件。

**证明1.5.2（拉格朗日乘数定理的证明）**：证明概要：
1. 定义拉格朗日函数L(x, λ) = f(x) + ∑_{i=1}^m λ_i g_i(x)。
2. 由于约束规范成立，切空间是约束梯度张成空间的正交补。
3. 任何可行的方向d必须满足∇g_i(x*)·d = 0对所有i。
4. 由于x*是局部最优解，∇f(x*)·d = 0对所有可行方向d成立。
5. 这意味着∇f(x*)位于约束梯度张成的空间中，即存在λ使得∇f(x*) + ∑λ_i ∇g_i(x*) = 0。□

**定义1.5.4（拉格朗日函数）**：L(x, λ) = f(x) + ∑_{i=1}^m λ_i g_i(x)，其中λ = (λ₁, ..., λ_m)是拉格朗日乘数向量。

**二阶条件（等式约束）**：
- 充分条件：如果拉格朗日函数关于x的海塞矩阵在约束切空间上负定，则x*是局部极大值。
- 必要条件：如果x*是局部极大值，则拉格朗日函数关于x的海塞矩阵在约束切空间上半负定。

**例1.5.2**：考虑效用最大化问题max u(x₁, x₂) s.t. p₁x₁ + p₂x₂ = m。拉格朗日函数为L(x₁, x₂, λ) = u(x₁, x₂) + λ(m - p₁x₁ - p₂x₂)。一阶条件为：
∂L/∂x₁ = ∂u/∂x₁ - λp₁ = 0
∂L/∂x₂ = ∂u/∂x₂ - λp₂ = 0
∂L/∂λ = m - p₁x₁ - p₂x₂ = 0

这给出了经济学中的基本等式：边际替代率等于价格比率。

### 1.5.3 不等式约束优化

**问题1.5.2（不等式约束优化）**：
max_x f(x)
s.t. g_i(x) ≤ 0, i = 1, ..., k

**定义1.5.5（有效约束）**：在可行点x*处，约束i是有效的（或起作用的），如果g_i(x*) = 0；否则称为无效约束。

**定义1.5.6（斯莱特条件）**：存在x使得g_i(x) < 0对所有i = 1, ..., k成立（即约束在某个点严格满足）。

**定理1.5.5（库恩-塔克定理）**：设x*是问题1.5.2的局部最优解，f和g_i是连续可微函数，且斯莱特条件成立。则存在拉格朗日乘数λ₁, ..., λ_k ≥ 0，使得：
1. ∇f(x*) + ∑_{i=1}^k λ_i ∇g_i(x*) = 0（平稳性条件）
2. λ_i g_i(x*) = 0, i = 1, ..., k（互补松弛条件）
3. g_i(x*) ≤ 0, i = 1, ..., k（可行性条件）
4. λ_i ≥ 0, i = 1, ..., k（非负性条件）

**证明1.5.3（库恩-塔克定理的证明）**：证明概要：
1. 定义拉格朗日函数L(x, λ) = f(x) + ∑_{i=1}^k λ_i g_i(x)。
2. 斯莱特条件保证了约束规范成立。
3. 对于有效约束，使用拉格朗日乘数定理。
4. 对于无效约束，互补松弛条件确保λ_i = 0。
5. 非负性条件来自于目标函数是最大化问题。□

**解释1.5.1（互补松弛条件的经济学含义）**：
- λ_i > 0 ⇒ g_i(x*) = 0：如果约束的影子价格为正，则该约束在最优解处是紧的（有效）。
- g_i(x*) < 0 ⇒ λ_i = 0：如果约束在最优解处不是紧的，则该约束的影子价格为0（松弛约束不影响最优解）。

**例1.5.3**：考虑效用最大化问题max u(x) s.t. p·x ≤ m, x ≥ 0。拉格朗日函数为L(x, λ, μ) = u(x) + λ(m - p·x) + μ·x，其中λ ≥ 0是收入约束的乘数，μ ≥ 0是非负约束的乘数向量。一阶条件为：
∇u(x*) - λp + μ = 0
λ(m - p·x*) = 0
μ_i x_i* = 0, i = 1, ..., n

如果x_i* > 0，则μ_i = 0，得到∂u/∂x_i = λp_i（边际效用等于价格乘以收入边际效用）。如果x_i* = 0，则∂u/∂x_i ≤ λp_i（消费零商品的边际效用不超过价格乘以收入边际效用）。

### 1.5.4 包络定理

**问题1.5.3（参数化优化）**：
max_x f(x, α)
s.t. g_i(x, α) ≤ 0, i = 1, ..., k

其中α是参数向量。设x*(α)是最优解，V(α) = f(x*(α), α)是值函数。

**定理1.5.6（包络定理）**：设f和g_i是连续可微函数，x*(α)是连续可微的，且库恩-塔克条件成立。则值函数V(α)关于α的导数为：
∇V(α) = ∇_α L(x*(α), λ(α), α) = ∂f/∂α + ∑_{i=1}^k λ_i ∂g_i/∂α

即在求值函数的导数时，只需要考虑参数α对目标函数和约束函数的直接影响，不需要考虑α通过最优解x*(α)的间接影响。

**证明1.5.4（包络定理的证明）**：证明概要：
1. V(α) = f(x*(α), α)
2. 对α求导：∇V(α) = ∇_x f(x*(α), α)·∇x*(α) + ∂f/∂α
3. 由库恩-塔克条件，∇_x f(x*(α), α) = -∑λ_i ∇_x g_i(x*(α), α)
4. 代入得到：∇V(α) = -∑λ_i ∇_x g_i·∇x* + ∂f/∂α
5. 由互补松弛条件，λ_i g_i(x*(α), α) = 0，对α求导得：∇_x g_i·∇x* + ∂g_i/∂α = 0（对于有效约束）
6. 因此，-∑λ_i ∇_x g_i·∇x* = ∑λ_i ∂g_i/∂α
7. 最终得到：∇V(α) = ∂f/∂α + ∑λ_i ∂g_i/∂α □

**应用1.5.1（霍特林引理）**：利润函数π(p, w)的导数给出供给和要素需求函数：
∂π/∂p = y(p, w)（供给函数）
∂π/∂w = -x(p, w)（要素需求函数）

**应用1.5.2（谢泼德引理）**：成本函数c(w, y)的导数给出条件要素需求函数：
∂c/∂w_i = x_i(w, y)（条件要素需求函数）

**应用1.5.3（罗伊恒等式）**：间接效用函数V(p, m)的导数给出马歇尔需求函数：
x_i(p, m) = - (∂V/∂p_i) / (∂V/∂m)

### 1.5.5 对偶理论

**定义1.5.7（原问题）**：
max_x f(x)
s.t. g_i(x) ≤ 0, i = 1, ..., k

**定义1.5.8（对偶问题）**：
min_λ L^*(λ) = max_x [f(x) + ∑_{i=1}^k λ_i g_i(x)]
s.t. λ_i ≥ 0, i = 1, ..., k

其中L^*(λ)是拉格朗日对偶函数。

**定理1.5.7（弱对偶性）**：对偶问题的最优值不超过原问题的最优值。

**证明1.5.5**：设x是原问题的可行解，λ ≥ 0。由于g_i(x) ≤ 0且λ_i ≥ 0，有∑λ_i g_i(x) ≤ 0。因此，f(x) ≤ f(x) + ∑λ_i g_i(x) ≤ max_x [f(x) + ∑λ_i g_i(x)] = L^*(λ)。对所有可行的x和λ取极值，得到原问题最优值 ≤ 对偶问题最优值。□

**定理1.5.8（强对偶性）**：在某些条件下（如凸优化问题满足斯莱特条件），原问题和对偶问题的最优值相等。

**应用1.5.4（消费者理论的对偶性）**：
- 原问题：效用最大化 max u(x) s.t. p·x ≤ m
- 对偶问题：支出最小化 min p·x s.t. u(x) ≥ u

这两个问题的对偶性关系是微观经济学的核心内容之一。

### 1.5.6 不动点定理

**定义1.5.9（不动点）**：设f: X → X是函数。如果存在x* ∈ X使得f(x*) = x*，则x*称为f的不动点。

**定理1.5.9（布劳威尔不动点定理）**：设f: D → D是连续函数，其中D ⊆ ℝ^n是紧凸集。则f在D中至少有一个不动点。

**证明1.5.6（布劳威尔不动点定理的证明）**：证明概要（使用反证法和拓扑度理论）：
1. 假设f没有不动点。
2. 构造一个从D到其边界的连续映射g。
3. 这与D的收缩性质矛盾。
4. 因此，f必须至少有一个不动点。□

**定理1.5.10（角谷静夫不动点定理）**：设Γ: D → D是对应，其中D ⊆ ℝ^n是紧凸集。如果Γ满足：
1. 对每个x ∈ D，Γ(x)是非空凸集。
2. Γ是上半连续的。

则Γ在D中至少有一个不动点，即存在x* ∈ D使得x* ∈ Γ(x*)。

**应用1.5.5（一般均衡存在性）**：角谷静夫不动点定理是证明竞争均衡存在性的核心工具。

## 本章总结

### 核心要点

1. **集合论**：集合、子集、运算、笛卡尔积、关系（等价关系、偏序关系）是数学分析的基础。

2. **函数与对应**：函数、单射、满射、双射、逆函数；对应、上半连续、下半连续。

3. **拓扑学**：度量空间、开集、闭集、紧性、连通性、序列收敛。紧集在经济分析中特别重要，因为它们保证了最优解的存在性。

4. **凸性理论**：凸集、凸函数、凹函数、拟凸函数、拟凹函数；分离超平面定理、支撑超平面定理、次微分。凸性是微观经济学中的关键假设。

5. **优化理论**：无约束优化、等式约束优化（拉格朗日乘数法）、不等式约束优化（库恩-塔克条件）、包络定理、对偶理论、不动点定理。优化理论是经济学分析的核心工具。

|------|------|------------|

| 凸集 | 任意两点连线仍在集合内 | 消费集合、生产集合 |
