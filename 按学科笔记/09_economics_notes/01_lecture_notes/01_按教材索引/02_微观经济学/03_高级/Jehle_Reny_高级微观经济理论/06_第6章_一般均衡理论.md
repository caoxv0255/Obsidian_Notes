# 第6章：一般均衡理论

本章系统介绍一般均衡理论，这是微观经济学的核心内容之一，研究多个市场同时出清时的价格和资源配置。通过本章的学习，读者应该能够：

1. 理解纯交换经济的一般均衡概念
2. 掌握瓦尔拉斯均衡的定义和性质
3. 理解生产经济的一般均衡
4. 掌握一般均衡存在性的证明方法
5. 理解一般均衡的比较静态分析
6. 掌握核与竞争均衡的关系
7. 理解德布鲁-斯卡夫极限定理

10. 掌握不确定性下的一般均衡理论

## 6.1 纯交换经济

### 6.1.1 纯交换经济的定义

**定义6.1.1（纯交换经济）**：纯交换经济E由以下要素组成：
1. 商品集合：n种商品，商品空间为ℝ^n
2. 消费者集合：I个消费者，i = 1, ..., I
3. 每个消费者i的偏好关系≽_i和初始禀赋ω_i ∈ ℝ^n_+

总禀赋为ω = ∑_{i=1}^I ω_i。

**定义6.1.2（可行分配）**：分配(x₁, ..., x_I)是可行的，如果：
1. x_i ∈ ℝ^n_+ 对所有i = 1, ..., I
2. ∑_{i=1}^I x_i = ω

**定义6.1.3（瓦尔拉斯均衡）**：价格向量p ∈ ℝ^n_++和分配(x₁, ..., x_I)构成瓦尔拉斯均衡，如果：
1. x_i是消费者i在价格p和财富m_i = p·ω_i下的最优选择：x_i = x_i(p, p·ω_i)
2. 市场出清：∑_{i=1}^I x_i = ω

**定理6.1.1（瓦尔拉斯定律）**：在均衡时，p·z(p) = 0，其中z(p) = ∑_{i=1}^I x_i(p, p·ω_i) - ω是超额需求函数。

**证明**：由消费者最优化，p·x_i(p, p·ω_i) = p·ω_i对所有i。求和得p·∑_{i=1}^I x_i = p·ω ⇒ p·z(p) = 0。□

### 6.1.2 超额需求函数

**定义6.1.4（超额需求函数）**：z(p) = ∑_{i=1}^I x_i(p, p·ω_i) - ω

**定理6.1.2（超额需求函数的性质）**：
1. 连续性：如果偏好是连续的且严格凸的，则z(p)是连续的。
2. 零次齐次性：z(αp) = z(p) 对所有α > 0。
3. 瓦尔拉斯定律：p·z(p) = 0 对所有p。
4. 边界条件：如果p_k → 0，则z_k(p) → +∞（如果商品k是必需品）。

**证明（瓦尔拉斯定律）**：同定理6.1.1。□

**例6.1.1（两个消费者，两种商品）**：
消费者1：效用函数u₁(x₁, x₂) = x₁^α x₂^{1-α}，禀赋ω₁ = (1, 0)
消费者2：效用函数u₂(x₁, x₂) = x₁^β x₂^{1-β}，禀赋ω₂ = (0, 1)

需求函数：
x₁(p) = (α·p·ω₁/p₁, (1-α)·p·ω₁/p₂) = (αp₁/p₁, (1-α)p₁/p₂) = (α, (1-α)p₁/p₂)
x₂(p) = (β·p·ω₂/p₁, (1-β)·p·ω₂/p₂) = (βp₂/p₁, (1-β)p₂/p₂) = (βp₂/p₁, 1-β)

超额需求：
z₁(p) = x₁₁(p) + x₂₁(p) - ω₁₁ - ω₂₁ = α + βp₂/p₁ - 1
z₂(p) = x₁₂(p) + x₂₂(p) - ω₁₂ - ω₂₂ = (1-α)p₁/p₂ + 1-β - 1 = (1-α)p₁/p₂ - β

均衡条件：z₁(p) = 0 ⇒ α + βp₂/p₁ = 1 ⇒ p₂/p₁ = (1-α)/β
验证：z₂(p) = (1-α)·β/(1-α) - β = β - β = 0

均衡价格：p = (1, (1-α)/β)

### 6.1.3 埃奇沃斯盒

**定义6.1.5（埃奇沃斯盒）**：埃奇沃斯盒是纯交换经济的几何表示，盒子的宽度为总禀赋ω₁，高度为总禀赋ω₂。盒内每点表示一个可行分配(x₁, x₂) = ((x₁₁, x₁₂), (x₂₁, x₂₂))。

**定理6.1.3（埃奇沃斯盒中的均衡）**：在埃奇沃斯盒中，瓦尔拉斯均衡对应于两条无差异曲线相切的点，且切点位于通过初始禀赋ω的直线上。

### 6.1.4 契约曲线

**定义6.1.6（契约曲线）**：契约曲线是所有帕累托最优分配的集合，在埃奇沃斯盒中表示为两条无差异曲线相切的点的轨迹。

**定理6.1.4（契约曲线的性质）**：
1. 契约曲线上的点都是帕累托最优的。
2. 每个瓦尔拉斯均衡分配都在契约曲线上。
3. 如果偏好是凸的，则契约曲线是连续的。

**证明（第二部分）**：由福利经济学第一定理，瓦尔拉斯均衡是帕累托最优的，因此在契约曲线上。□

## 6.2 生产经济

### 6.2.1 生产经济的定义

**定义6.2.1（生产经济）**：生产经济E由以下要素组成：
1. 商品集合：n种商品
2. 消费者集合：I个消费者，i = 1, ..., I
3. 厂商集合：J个厂商，j = 1, ..., J
4. 每个消费者i的偏好关系≽_i、初始禀赋ω_i ∈ ℝ^n_+和利润份额θ_{ij} ≥ 0
5. 每个厂商j的生产可能性集Y_j ⊆ ℝ^n

其中θ_{ij}表示消费者i拥有厂商j的利润份额。

**定义6.2.2（生产经济的可行分配）**：分配((x_i)_{i=1}^I, (y_j)_{j=1}^J)是可行的，如果：
1. x_i ∈ ℝ^n_+ 对所有i
2. y_j ∈ Y_j 对所有j
3. ∑_{i=1}^I x_i = ∑_{i=1}^I ω_i + ∑_{j=1}^J y_j

**定义6.2.3（生产经济的瓦尔拉斯均衡）**：价格向量p、分配((x_i), (y_j))构成瓦尔拉斯均衡，如果：
1. x_i是消费者i在价格p和财富m_i = p·ω_i + ∑_{j=1}^J θ_{ij} p·y_j下的最优选择
2. y_j是厂商j在价格p下的利润最大化生产计划
3. 市场出清：∑_{i=1}^I x_i = ∑_{i=1}^I ω_i + ∑_{j=1}^J y_j

### 6.2.2 生产集合的性质

**定义6.2.4（生产集合）**：生产集合Y_j ⊆ ℝ^n表示厂商j的所有可行生产计划。

**定理6.2.1（生产集合的标准假设）**：
1. 闭性：Y_j是闭集。
2. 凸性：Y_j是凸集。
3. 自由处置：如果y ∈ Y_j且y' ≤ y，则y' ∈ Y_j。
4. 不可逆性：0 ∈ Y_j，且Y_j ∩ (-Y_j) = {0}。
5. 没有免费的午餐：Y_j ∩ ℝ^n_+ = {0}。

**解释6.2.1（标准假设的经济学含义）**：
- 闭性：极限生产计划是可行的。
- 凸性：生产技术满足边际替代率递减。
- 自由处置：可以免费处置多余产出。
- 不可逆性：不能逆转生产过程。
- 没有免费的午餐：不能无中生有创造价值。

### 6.2.3 利润最大化

**问题6.2.1（利润最大化）**：
π_j(p) = max_{y_j ∈ Y_j} p·y_j

**定理6.2.2（利润函数的性质）**：
1. 一次齐次性：π_j(αp) = απ_j(p) 对所有α > 0。
2. 对价格非减：如果p' ≥ p，则π_j(p') ≥ π_j(p)。
3. 对价格凸：π_j(p)是p的凸函数。
4. 霍特林引理：y_j(p) = ∇_p π_j(p)。

### 6.2.4 生产经济的均衡条件

**定理6.2.3（生产经济的瓦尔拉斯定律）**：在均衡时，p·z(p) = 0，其中z(p) = ∑_{i=1}^I x_i(p) - ∑_{i=1}^I ω_i - ∑_{j=1}^J y_j(p)。

**证明**：由消费者最优化，p·x_i = p·ω_i + ∑_{j=1}^J θ_{ij} p·y_j。求和得：
p·∑_{i=1}^I x_i = p·∑_{i=1}^I ω_i + ∑_{i=1}^I ∑_{j=1}^J θ_{ij} p·y_j = p·∑_{i=1}^I ω_i + ∑_{j=1}^J p·y_j
因此，p·z(p) = 0。□

## 6.3 一般均衡的存在性

### 6.3.1 瓦尔拉斯均衡的存在性定理

**定理6.3.1（瓦尔拉斯均衡存在性（纯交换经济））**：在纯交换经济中，如果满足：
1. 偏好是完备的、传递的、连续的、严格凸的、单调的
2. 消费集合X_i = ℝ^n_+
3. 总禀赋ω > 0

则瓦尔拉斯均衡存在。

**证明概要**（使用角谷静夫不动点定理）：
1. 构造价格单纯形Δ = { p ∈ ℝ^n_++ | ∑_{k=1}^n p_k = 1 }。
2. 构造价格调整对应Γ: Δ → Δ，定义Γ(p) = { p' ∈ Δ | p'_k ≥ p_k · max{0, z_k(p)} / ∑_{l=1}^n p_l · max{0, z_l(p)} }。
3. 验证Γ满足角谷静夫定理的条件：
   - 对每个p，Γ(p)是非空凸集。
   - Γ是上半连续的。
4. 由角谷静夫定理，存在不动点p* ∈ Γ(p*)。
5. 证明p*满足z(p*) = 0，即瓦尔拉斯均衡。□

**定理6.3.2（瓦尔拉斯均衡存在性（生产经济））**：在生产经济中，如果满足：
1. 偏好是连续的、严格凸的、单调的
2. 生产集合Y_j是闭的、凸的，且满足自由处置
3. 存在可行分配

则瓦尔拉斯均衡存在。

### 6.3.2 角谷静夫不动点定理

**定理6.3.3（角谷静夫不动点定理）**：设Γ: Δ → Δ是对应，其中Δ是单纯形。如果Γ满足：
1. 对每个p ∈ Δ，Γ(p)是非空凸集。
2. Γ是上半连续的。

则Γ在Δ中至少有一个不动点，即存在p* ∈ Δ使得p* ∈ Γ(p*)。

**证明概要**：使用布劳威尔不动点定理的推广版本。证明涉及构造近似映射和使用逼近论。□

**应用6.3.1（价格调整对应）**：
定义Γ(p) = { p' ∈ Δ | p'·z(p) ≥ 0 }

解释：如果商品k的超额需求z_k(p) > 0，则价格p_k应该上升。这个价格调整对应满足角谷静夫定理的条件。

### 6.3.3 存在性条件的讨论

**条件6.3.1（偏好的连续性）**：如果偏好不连续，需求函数可能不连续，导致均衡不存在。

**条件6.3.2（偏好的凸性）**：如果偏好不凸，需求对应可能不是凸值的，导致价格调整对应不满足角谷静夫定理的条件。

**条件6.3.3（生产集合的凸性）**：如果生产集合不凸，供给对应可能不连续，导致均衡不存在。

**例6.3.1（不可分生产技术）**：如果生产技术存在固定成本或规模报酬递增，生产集合可能不凸，瓦尔拉斯均衡可能不存在。

## 6.4 一般均衡的比较静态

### 6.4.1 比较静态的一般方法

**定理6.4.1（均衡比较静态）**：设均衡条件为z(p) = 0，其中z(p)是超额需求函数。均衡价格p*由z(p*) = 0定义。如果D_z(p*)可逆，则：
dp* = - [D_z(p*)]^{-1} · dz

**应用6.4.1（禀赋变化的比较静态）**：考虑总禀赋从ω变为ω'。均衡价格p*(ω)由z(p*(ω), ω) = 0定义。对ω求导：
D_p z(p*, ω) · (dp*/dω) + D_ω z(p*, ω) = 0

因此，dp*/dω = - [D_p z(p*, ω)]^{-1} D_ω z(p*, ω)

### 6.4.2 斯卢茨基方程的加总

**定理6.4.2（斯卢茨基方程的加总）**：总超额需求函数满足：
S(p, m) = ∑_{i=1}^I S_i(p, m_i)

其中S_i(p, m_i)是消费者i的斯卢茨基替代矩阵。

**解释6.4.1（加总的经济学含义）**：总替代效应是个人替代效应的加总。

### 6.4.3 对称性与齐次性

**定理6.4.3（总超额需求的对称性）**：如果所有消费者都具有相同的位似偏好，则总超额需求函数满足对称性：
∂z_k/∂p_l = ∂z_l/∂p_k

**定理6.4.4（总超额需求的齐次性）**：z(αp) = z(p) 对所有α > 0。

## 6.5 核与竞争均衡

### 6.5.1 核的概念

**定义6.5.1（联盟）**：联盟S是消费者集合I的子集，S ⊆ I。

**定义6.5.2（联盟的可行分配）**：分配(x_i)_{i∈S}对联盟S是可行的，如果∑_{i∈S} x_i ≤ ∑_{i∈S} ω_i。

**定义6.5.3（核）**：核是无法被任何联盟改进的可行分配集合。

**定理6.5.1（瓦尔拉斯均衡在核中）**：任何瓦尔拉斯均衡分配都在核中。

**证明**：使用反证法。假设瓦尔拉斯均衡分配x*不在核中，则存在联盟S和可行分配x'使得x_i' ≽_i x_i*对所有i∈S，且x_j' ≻_j x_j*对某个j∈S。由消费者最优化，p*·x_i* = p*·ω_i对所有i∈S。由于x_i' ≽_i x_i*，且至少有一个严格偏好，有p*·x_i' ≥ p*·ω_i对所有i∈S，且至少有一个严格不等式。求和得p*·∑_{i∈S} x_i' > p*·∑_{i∈S} ω_i，与x'对S的可行性矛盾。□

### 6.5.2 德布鲁-斯卡夫极限定理

**定理6.5.2（德布鲁-斯卡夫极限定理）**：考虑经济复制I次（每个消费者类型复制I次）。随着I → ∞，核收敛于瓦尔拉斯均衡分配集合。

**解释6.5.1（极限定理的经济学含义）**：当经济规模增大时，非竞争均衡的核分配逐渐消失，只剩下竞争均衡分配。

### 6.5.3 核的性质

**定理6.5.3（核的非空性）**：在交换经济中，如果偏好是连续的、单调的，则核非空。

**定理6.5.4（核与帕累托最优）**：核是帕累托最优分配集合的子集。

**证明**：如果分配x*不是帕累托最优的，则存在可行分配x'使得x_i' ≽_i x_i*对所有i，且x_j' ≻_j x_j*对某个j。取联盟S = I，则S可以改进x*，因此x*不在核中。□

## 6.6 测度论与一般均衡

### 6.6.1 连续统经济

**定义6.6.1（连续统经济）**：连续统经济是指消费者集合是连续统[0, 1]的经济，其中每个消费者类型t ∈ [0, 1]具有偏好≽_t和禀赋ω_t。

**定义6.6.2（测度）**：测度μ是定义在[0, 1]的子集上的函数，表示消费者类型的分布。

**定理6.6.1（埃奇沃斯猜想）**：在连续统经济中，核与瓦尔拉斯均衡分配集合重合。

**解释6.6.1（埃奇沃斯猜想的经济学含义）**：当经济规模无限大时，核完全收敛于竞争均衡分配。

### 6.6.2 测度论的应用

**应用6.6.1（大数定律）**：在连续统经济中，个体风险可以通过加总消除，使得总禀赋是确定的。

**应用6.6.2（无原子的测度）**：无原子的测度确保没有单个消费者可以影响市场价格，支持了价格接受者的假设。

## 6.7 不确定性下的一般均衡

### 6.7.1 状态依存商品

**定义6.7.1（状态依存商品）**：状态依存商品x_s是在状态s ∈ S下可获得的商品。

**定义6.7.2（阿罗-德布鲁证券）**：阿罗-德布鲁证券是在特定状态下支付1单位货币，在其他状态下支付0的证券。

**定理6.7.1（阿罗-德布鲁均衡）**：在不确定性下的一般均衡中，每种状态依存商品都有价格p_s，均衡配置满足：
1. 消费者在预算约束下最大化期望效用。
2. 市场出清：∑_{i=1}^I x_{i,s} = ∑_{i=1}^I ω_{i,s} 对所有s。

### 6.7.2 完全市场与不完全市场

**定义6.7.3（完全市场）**：市场是完全的，如果对于每种状态s，都存在状态依存商品x_s，且价格p_s已知。

**定义6.7.4（不完全市场）**：市场是不完全的，如果存在某些状态无法通过市场交易实现风险分担。

**定理6.7.2（不完全市场的无效率）**：在不完全市场中，竞争均衡可能不是帕累托最优的。

## 本章总结

### 核心要点

1. **纯交换经济**：瓦尔拉斯均衡（价格和分配）、超额需求函数、瓦尔拉斯定律、埃奇沃斯盒、契约曲线。

2. **生产经济**：生产集合的性质、利润最大化、生产经济的均衡条件、瓦尔拉斯定律。

3. **一般均衡存在性**：瓦尔拉斯均衡存在性定理（使用角谷静夫不动点定理）、存在性条件（偏好的连续性、凸性；生产集合的凸性）。

4. **比较静态分析**：均衡价格对外生参数的导数、斯卢茨基方程的加总、对称性与齐次性。

5. **核与竞争均衡**：核的定义、瓦尔拉斯均衡在核中、德布鲁-斯卡夫极限定理、核与帕累托最优的关系。

6. **测度论与一般均衡**：连续统经济、埃奇沃斯猜想、测度论的应用。

7. **不确定性下的一般均衡**：状态依存商品、阿罗-德布鲁证券、完全市场与不完全市场。
