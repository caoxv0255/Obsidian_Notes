# 第14章：消费者剩余

> 本章学习目标：
> - 理解消费者剩余的定义
> - 掌握消费者剩余的计算
> - 学会分析消费者剩余的变化
> - 理解生产者剩余和总剩余

---

## 14.1 消费者剩余的定义

### 14.1.1 消费者剩余的定义

**消费者剩余（Consumer Surplus）**：
消费者愿意支付的最高价格与实际支付价格之间的差额。

### 14.1.2 消费者剩余的含义

**消费者剩余的含义（Meaning of Consumer Surplus）**：
- 消费者从消费中获得的净收益
- 衡量消费者福利的指标

---

## 14.2 消费者剩余的计算

### 14.2.1 离散情况

**离散情况（Discrete Case）**：

**消费者剩余的公式（Formula of Consumer Surplus）**：
```
CS = Σᵢ (pᵢ - p) × xᵢ
```

**其中**：
- CS：消费者剩余
- pᵢ：消费者i愿意支付的最高价格
- p：实际支付价格
- xᵢ：消费者i的消费量

**含义**：
- 消费者剩余等于每个消费者愿意支付的最高价格与实际支付价格的差额乘以消费量的总和

### 14.2.2 连续情况

**连续情况（Continuous Case）**：

**消费者剩余的公式（Formula of Consumer Surplus）**：
```
CS = ∫₀^ˣ D(q) dq - p × x
```

**其中**：
- CS：消费者剩余
- D(q)：需求函数
- p：实际支付价格
- x：消费量

**含义**：
- 消费者剩余等于需求曲线下方、价格线上方的面积

## 14.3 消费者剩余的变化

### 14.3.1 价格下降

**价格下降（Price Decrease）**：

**消费者剩余的变化（Change in Consumer Surplus）**：
```
ΔCS = ∫ₚ'ᵖ D(q) dq
```

**其中**：
- ΔCS：消费者剩余的变化
- D(q)：需求函数
- p'：新价格
- p：原价格

**含义**：
- 价格下降，消费者剩余增加
- 消费者剩余的变化等于需求曲线在原价格和新价格之间的面积

### 14.3.2 价格上升

**价格上升（Price Increase）**：

**消费者剩余的变化（Change in Consumer Surplus）**：
```
ΔCS = -∫ₚᵖ' D(q) dq
```

**其中**：
- ΔCS：消费者剩余的变化
- D(q)：需求函数
- p'：新价格
- p：原价格

**含义**：
- 价格上升，消费者剩余减少
- 消费者剩余的变化等于需求曲线在原价格和新价格之间的面积的负值

## 14.4 生产者剩余

### 14.4.1 生产者剩余的定义

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

### 14.4.2 生产者剩余的计算

**生产者剩余的公式（Formula of Producer Surplus）**：
```
PS = p × x - ∫₀^ˣ S(q) dq
```

**其中**：
- PS：生产者剩余
- S(q)：供给函数
- p：实际获得价格
- x：供给量

**含义**：
- 生产者剩余等于价格线上方、供给曲线下方的面积

## 14.5 总剩余

### 14.5.1 总剩余的定义

**总剩余（Total Surplus）**：
消费者剩余和生产者剩余的总和。

### 14.5.2 总剩余的计算

**总剩余的公式（Formula of Total Surplus）**：
```
TS = CS + PS
```

**其中**：
- TS：总剩余
- CS：消费者剩余
- PS：生产者剩余

**含义**：
- 总剩余等于消费者剩余和生产者剩余的总和
- 总剩余衡量社会总福利

## 14.6 福利分析

### 14.6.1 价格上限

**价格上限（Price Ceiling）**：
政府规定的最高价格。

**效果**：
- 价格低于均衡价格
- 消费者剩余可能增加或减少
- 生产者剩余减少
- 总剩余减少（无谓损失）

- 消费者剩余减少

## 14.7 本章总结

### 核心要点

1. **消费者剩余的定义**：
   - 消费者剩余：消费者愿意支付的最高价格与实际支付价格之间的差额
   - 消费者剩余的含义：消费者从消费中获得的净收益

2. **消费者剩余的计算**：
   - 离散情况：CS = Σᵢ (pᵢ - p) × xᵢ
   - 连续情况：CS = ∫₀^ˣ D(q) dq - p × x
   - 消费者剩余等于需求曲线下方、价格线上方的面积

3. **消费者剩余的变化**：
   - 价格下降：消费者剩余增加
   - 价格上升：消费者剩余减少

4. **生产者剩余**：
   - 生产者剩余：生产者实际获得的价格与愿意接受的最低价格之间的差额
   - 生产者剩余的计算：PS = p × x - ∫₀^ˣ S(q) dq
   - 生产者剩余等于价格线上方、供给曲线下方的面积

5. **总剩余**：
   - 总剩余：消费者剩余和生产者剩余的总和
   - 总剩余的计算：TS = CS + PS
   - 总剩余衡量社会总福利

6. **福利分析**：
   - 价格上限：消费者剩余可能增加或减少，生产者剩余减少，总剩余减少（无谓损失）
   - 价格下限：消费者剩余减少，生产者剩余可能增加或减少，总剩余减少（无谓损失）
