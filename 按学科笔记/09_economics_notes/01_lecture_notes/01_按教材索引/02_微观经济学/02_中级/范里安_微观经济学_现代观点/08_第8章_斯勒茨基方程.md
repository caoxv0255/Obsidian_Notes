# 第8章：斯勒茨基方程

> 本章学习目标：

> - 掌握斯勒茨基方程的推导

---

## 8.1 斯勒茨基方程的定义

### 8.1.1 斯勒茨基方程的定义

**斯勒茨基方程（Slutsky Equation）**：
描述价格变化的总效应如何分解为替代效应和收入效应的方程。

### 8.1.2 斯勒茨基方程的公式

**斯勒茨基方程的公式（Formula of Slutsky Equation）**：
```
∂x₁/∂p₁ = ∂x₁^h/∂p₁ - x₁(∂x₁/∂m)
```

**其中**：
- ∂x₁/∂p₁：马歇尔需求对价格的变化率（总效应）
- ∂x₁^h/∂p₁：希克斯需求对价格的变化率（替代效应）
- -x₁(∂x₁/∂m)：收入效应

---

## 8.2 斯勒茨基方程的推导

### 8.2.1 斯勒茨基方程的推导

**斯勒茨基方程的推导（Derivation of Slutsky Equation）**：

**步骤1：定义马歇尔需求函数**：
```
x₁ = x₁(p₁, p₂, m)
```

**步骤2：定义希克斯需求函数**：
```
x₁^h = x₁^h(p₁, p₂, u)
```

**步骤3：定义收入补偿**：
```
m' = p₁x₁ + p₂x₂
```

**步骤4：在(p₁', p₂, m')下，消费者选择(x₁', x₂')**：
```
x₁' = x₁(p₁', p₂, m')
```

**步骤5：在(p₁', p₂, u)下，消费者选择(x₁'^h, x₂'^h)**：
```
x₁'^h = x₁^h(p₁', p₂, u)
```

**步骤6：斯勒茨基方程**：
```
∂x₁/∂p₁ = ∂x₁^h/∂p₁ - x₁(∂x₁/∂m)
```

---

## 8.3 替代效应和收入效应

### 8.3.1 替代效应

**替代效应（Substitution Effect）**：
由于商品相对价格变化，消费者调整商品组合，保持效用不变。

**公式**：
```
∂x₁^h/∂p₁
```

**含义**：
- 替代效应总是负的
- 价格上升，替代效应导致需求量减少
- 价格下降，替代效应导致需求量增加

### 8.3.2 收入效应

**收入效应（Income Effect）**：
由于商品价格变化，消费者的实际收入变化，从而调整商品组合。

**公式**：
```
-x₁(∂x₁/∂m)
```

**含义**：
- 收入效应的符号取决于商品类型
- 正常商品：收入效应是负的
- 劣等商品：收入效应是正的

### 8.3.3 总效应

**总效应（Total Effect）**：
替代效应和收入效应的总和。

**公式**：
```
∂x₁/∂p₁ = ∂x₁^h/∂p₁ - x₁(∂x₁/∂m)
```

**含义**：
- 总效应 = 替代效应 + 收入效应
- 正常商品：替代效应和收入效应同方向
- 劣等商品：替代效应和收入效应反方向

---

## 8.4 正常商品和劣等商品

### 8.4.1 正常商品

**正常商品（Normal Goods）**：
- ∂x₁/∂m > 0
- 收入增加，需求量增加
- 收入效应是负的

**斯勒茨基方程**：
```
∂x₁/∂p₁ = ∂x₁^h/∂p₁ - x₁(∂x₁/∂m)
```

**分析**：
- 替代效应：∂x₁^h/∂p₁ < 0
- 收入效应：-x₁(∂x₁/∂m) < 0
- 总效应：∂x₁/∂p₁ < 0

**含义**：
- 价格上升，需求量减少
- 价格下降，需求量增加
- 需求曲线向右下方倾斜

### 8.4.2 劣等商品

**劣等商品（Inferior Goods）**：
- ∂x₁/∂m < 0
- 收入增加，需求量减少
- 收入效应是正的

**斯勒茨基方程**：
```
∂x₁/∂p₁ = ∂x₁^h/∂p₁ - x₁(∂x₁/∂m)
```

**分析**：
- 替代效应：∂x₁^h/∂p₁ < 0
- 收入效应：-x₁(∂x₁/∂m) > 0
- 总效应：取决于替代效应和收入效应的大小

**情况1：替代效应 > 收入效应**：
- 总效应：∂x₁/∂p₁ < 0
- 需求曲线向右下方倾斜

**情况2：替代效应 < 收入效应**：
- 总效应：∂x₁/∂p₁ > 0
- 需求曲线向右上方倾斜（吉芬商品）

## 8.5 吉芬商品

### 8.5.1 吉芬商品的定义

**吉芬商品（Giffen Goods）**：
劣等商品的一种特殊情况，收入效应的绝对值大于替代效应。

### 8.5.2 吉芬商品的特征

**吉芬商品的特征（Characteristics of Giffen Goods）**：

**1. 劣等商品（Inferior Good）**：
- ∂x₁/∂m < 0
- 收入增加，需求量减少

**2. 收入效应的绝对值大于替代效应（|Income Effect| > |Substitution Effect|）**：
- |x₁(∂x₁/∂m)| > |∂x₁^h/∂p₁|
- 总效应：∂x₁/∂p₁ > 0

**3. 需求曲线向右上方倾斜（Upward Sloping Demand Curve）**：
- 价格上升，需求量增加
- 价格下降，需求量减少

### 8.5.3 吉芬商品的例子

**吉芬商品的例子（Examples of Giffen Goods）**：

**1. 土豆饥荒（Potato Famine）**：
- 19世纪爱尔兰土豆饥荒
- 土豆价格上涨，土豆需求量增加
- 原因：土豆是劣等商品，价格上涨导致实际收入下降，消费者减少其他商品消费，增加土豆消费

**2. 大米和小麦（Rice and Wheat）**：
- 大米价格上涨，大米需求量增加
- 原因：大米是劣等商品，价格上涨导致实际收入下降，消费者减少小麦消费，增加大米消费

## 8.6 斯勒茨基方程的矩阵形式

### 8.6.1 斯勒茨基方程的矩阵形式

**斯勒茨基方程的矩阵形式（Matrix Form of Slutsky Equation）**：
```
S = X - (∂x/∂m)(x')^T
```

**其中**：
- S：斯卢茨基替代矩阵
- X：马歇尔需求对价格的导数矩阵
- ∂x/∂m：马歇尔需求对收入的导数向量
- x'：马歇尔需求向量的转置

### 8.6.2 斯卢茨基替代矩阵的性质

**斯卢茨基替代矩阵的性质（Properties of Slutsky Substitution Matrix）**：

**1. 对称性（Symmetry）**：
- S = S^T
- 斯卢茨基替代矩阵是对称的

**2. 负半定性（Negative Semidefiniteness）**：
- z^T Sz ≤ 0
- 斯卢茨基替代矩阵是负半定的

**3. 零次齐次性（Homogeneity of Degree Zero）**：
- S p = 0
- 斯卢茨基替代矩阵乘以价格向量等于零

## 8.7 斯勒茨基方程的应用

### 8.7.1 分析价格变化的影响

**分析价格变化的影响（Analyzing the Effect of Price Change）**：

**步骤1：计算总效应**：
```
∂x₁/∂p₁
```

**步骤2：计算替代效应**：
```
∂x₁^h/∂p₁
```

**步骤3：计算收入效应**：
```
-x₁(∂x₁/∂m)
```

**步骤4：分析替代效应和收入效应**：
- 比较替代效应和收入效应的大小
- 判断商品类型

### 8.7.2 检验需求理论

**检验需求理论（Testing Demand Theory）**：

**方法1：检验弱公理**：
- 检验是否满足弱公理
- 如果满足，需求理论是正确的

**方法2：检验斯卢茨基矩阵**：
- 检验斯卢茨基矩阵是否对称
- 检验斯卢茨基矩阵是否负半定
- 如果满足，需求理论是正确的

1. **斯勒茨基方程的定义**：
