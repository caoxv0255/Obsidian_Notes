# 第16章：成本最小化

> 本章学习目标：
> - 理解成本最小化的定义
> - 掌握成本最小化的条件
> - 学会分析成本函数
> - 理解成本函数的性质

---

## 16.1 成本最小化的定义

### 16.1.1 成本最小化的定义

**成本最小化（Cost Minimization）**：
在给定产量下，选择要素投入组合，使成本最小。

### 16.1.2 成本最小化问题的数学表述

**成本最小化问题的数学表述（Mathematical Formulation of Cost Minimization Problem）**：
```
min C = w₁x₁ + w₂x₂
s.t. f(x₁, x₂) = y
```

**其中**：
- C：成本
- w₁：要素1的价格
- w₂：要素2的价格
- x₁：要素1的投入量
- x₂：要素2的投入量
- f(x₁, x₂)：生产函数
- y：产量

---

## 16.2 成本最小化的条件

### 16.2.1 一阶条件

**一阶条件（First-Order Condition, FOC）**：
使用拉格朗日方法，建立拉格朗日函数：
```
L = w₁x₁ + w₂x₂ + λ(y - f(x₁, x₂))
```

**一阶条件**：
```
∂L/∂x₁ = w₁ - λ∂f/∂x₁ = 0
```
```
∂L/∂x₂ = w₂ - λ∂f/∂x₂ = 0
```
```
∂L/∂λ = y - f(x₁, x₂) = 0
```

**从一阶条件得到**：
```
w₁/∂f/∂x₁ = λ
```
```
w₂/∂f/∂x₂ = λ
```

**因此**：
```
w₁/∂f/∂x₁ = w₂/∂f/∂x₂
```

**或者**：
```
MP₁/w₁ = MP₂/w₂
```

**其中**：
- MP₁：要素1的边际产量
- MP₂：要素2的边际产量

**含义**：
- 技术替代率等于要素价格比
- 每单位货币的边际产量相等

### 16.2.2 二阶条件

**二阶条件（Second-Order Condition, SOC）**：
等产量曲线是凸的。

**含义**：
- 边际技术替代率递减
- 等产量曲线凸向原点

---

## 16.3 条件要素需求

### 16.3.1 条件要素需求的定义

**条件要素需求（Conditional Factor Demand）**：
在给定产量和要素价格下，成本最小化的要素需求。

### 16.3.2 条件要素需求的函数

**条件要素需求的函数（Function of Conditional Factor Demand）**：
```
x₁ = x₁(w₁, w₂, y)
```
```
x₂ = x₂(w₁, w₂, y)
```

**其中**：
- x₁：要素1的条件需求
- x₂：要素2的条件需求
- w₁：要素1的价格
- w₂：要素2的价格
- y：产量

**含义**：
- 条件要素需求是要素价格和产量的函数
- 条件要素需求依赖于产量

---

## 16.4 成本函数

### 16.4.1 成本函数的定义

**成本函数（Cost Function）**：
在给定产量和要素价格下，最小成本的函数。

### 16.4.2 成本函数的公式

**成本函数的公式（Formula of Cost Function）**：
```
C(w₁, w₂, y) = w₁x₁(w₁, w₂, y) + w₂x₂(w₁, w₂, y)
```

**其中**：
- C(w₁, w₂, y)：成本函数
- w₁：要素1的价格
- w₂：要素2的价格
- y：产量
- x₁(w₁, w₂, y)：要素1的条件需求
- x₂(w₁, w₂, y)：要素2的条件需求

**含义**：
- 成本函数是要素价格和产量的函数
- 成本函数等于要素价格乘以条件要素需求的总和

---

## 16.5 成本函数的性质

### 16.5.1 成本函数的性质

**成本函数的性质（Properties of Cost Function）**：

**1. 齐次性（Homogeneity）**：
- 成本函数是要素价格的一次齐次函数
- C(tw₁, tw₂, y) = tC(w₁, w₂, y)

**2. 单调性（Monotonicity）**：
- 成本函数是要素价格和产量的增函数
- ∂C/∂w₁ ≥ 0，∂C/∂w₂ ≥ 0，∂C/∂y ≥ 0

**3. 凹性（Concavity）**：
- 成本函数是要素价格的凹函数
- 成本函数的海塞矩阵是负半定的

**4. 谢泼德引理（Shephard's Lemma）**：
```
∂C/∂w₁ = x₁
```
```
∂C/∂w₂ = x₂
```

**含义**：
- 成本函数对要素价格的偏导数等于条件要素需求

## 16.6 成本最小化的图形分析

### 16.6.1 等成本线

**等成本线（Isocost Line）**：
表示给定成本下要素组合的轨迹。

**公式**：
```
w₁x₁ + w₂x₂ = C
```

**图形表示**：

```plotly

  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "x₁"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- 等成本线向右下方倾斜
- 斜率：-w₁/w₂

- 最优投入：等产量曲线和等成本线的切点

## 16.7 本章总结

### 核心要点

1. **成本最小化的定义**：
   - 成本最小化：在给定产量下，选择要素投入组合，使成本最小
   - 成本最小化问题的数学表述：min C = w₁x₁ + w₂x₂ s.t. f(x₁, x₂) = y

2. **成本最小化的条件**：
   - 一阶条件：w₁/∂f/∂x₁ = w₂/∂f/∂x₂，或MP₁/w₁ = MP₂/w₂
   - 二阶条件：等产量曲线是凸的

3. **条件要素需求**：
   - 条件要素需求：在给定产量和要素价格下，成本最小化的要素需求
   - 条件要素需求的函数：x₁ = x₁(w₁, w₂, y)，x₂ = x₂(w₁, w₂, y)

4. **成本函数**：
   - 成本函数：在给定产量和要素价格下，最小成本的函数
   - 成本函数的公式：C(w₁, w₂, y) = w₁x₁(w₁, w₂, y) + w₂x₂(w₁, w₂, y)

5. **成本函数的性质**：
   - 齐次性：C(tw₁, tw₂, y) = tC(w₁, w₂, y)
   - 单调性：∂C/∂w₁ ≥ 0，∂C/∂w₂ ≥ 0，∂C/∂y ≥ 0
   - 凹性：成本函数是要素价格的凹函数
   - 谢泼德引理：∂C/∂w₁ = x₁，∂C/∂w₂ = x₂

6. **成本最小化的图形分析**：
   - 等成本线：w₁x₁ + w₂x₂ = C
   - 最优投入：等产量曲线和等成本线的切点
