# 第21章：寡头

> 本章学习目标：
> - 理解寡头的定义

> - 学会分析寡头市场的均衡

---

## 21.1 寡头的定义

### 21.1.1 寡头的定义

**寡头（Oligopoly）**：
少数几个卖者控制的市场。

### 21.1.2 寡头的特征

**寡头的特征（Characteristics of Oligopoly）**：

**1. 少数卖者（Few Sellers）**：
- 市场上只有少数几个卖者
- 卖者之间存在相互依赖

**2. 相互依赖（Interdependence）**：
- 每个厂商的决策影响其他厂商
- 厂商需要考虑竞争对手的反应

**3. 进入壁垒（Barriers to Entry）**：
- 其他厂商难以进入市场
- 进入壁垒包括：规模经济、专利、品牌

**4. 产品同质或异质（Homogeneous or Differentiated Products）**：
- 产品可以是同质的（如石油）
- 产品也可以是异质的（如汽车）

## 21.2 寡头市场的模型

### 21.2.1 古诺模型

**古诺模型（Cournot Model）**：
同时选择产量的寡头模型。

**假设**：
- 两个厂商（双寡头）
- 同时选择产量
- 厂商知道市场需求

**厂商i的利润**：
```
πᵢ = p(y₁ + y₂) × yᵢ - Cᵢ(yᵢ)
```

**其中**：
- πᵢ：厂商i的利润
- p(y₁ + y₂)：价格函数
- yᵢ：厂商i的产量
- Cᵢ(yᵢ)：厂商i的成本函数

**古诺均衡（Cournot Equilibrium）**：
```
∂π₁/∂y₁ = 0
```
```
∂π₂/∂y₂ = 0
```

**含义**：
- 每个厂商在其他厂商产量给定的情况下，选择自己的产量
- 古诺均衡是纳什均衡

### 21.2.2 伯特兰模型

**伯特兰模型（Bertrand Model）**：
同时选择价格的寡头模型。

**假设**：
- 两个厂商（双寡头）
- 同时选择价格
- 产品同质
- 消费者购买价格最低的产品

**厂商i的利润**：
```
如果 pᵢ < pⱼ，πᵢ = (pᵢ - c) × D(pᵢ)
如果 pᵢ = pⱼ，πᵢ = (pᵢ - c) × D(pᵢ)/2
如果 pᵢ > pⱼ，πᵢ = 0
```

**其中**：
- πᵢ：厂商i的利润
- pᵢ：厂商i的价格
- c：边际成本
- D(p)：需求函数

**伯特兰均衡（Bertrand Equilibrium）**：
```
p₁ = p₂ = c
```

**含义**：
- 伯特兰均衡价格等于边际成本
- 伯特兰均衡产量等于完全竞争产量
- 厂商获得零利润

### 21.2.3 斯塔克伯格模型

**斯塔克伯格模型（Stackelberg Model）**：
先动和后动的寡头模型。

**假设**：
- 两个厂商（双寡头）
- 厂商1先选择产量（领导者）
- 厂商2后选择产量（追随者）
- 厂商2知道厂商1的产量

**厂商2的利润最大化**：
```
max π₂ = p(y₁ + y₂) × y₂ - C₂(y₂)
```

**一阶条件**：
```
∂π₂/∂y₂ = 0
```

**得到反应函数**：
```
y₂ = R₂(y₁)
```

**厂商1的利润最大化**：
```
max π₁ = p(y₁ + R₂(y₁)) × y₁ - C₁(y₁)
```

**一阶条件**：
```
∂π₁/∂y₁ = 0
```

**斯塔克伯格均衡（Stackelberg Equilibrium）**：
- 厂商1选择产量y₁
- 厂商2根据y₁选择产量y₂ = R₂(y₁)

**含义**：
- 领导者获得优势
- 领导者的产量高于古诺均衡产量
- 追随者的产量低于古诺均衡产量

## 21.3 寡头市场的均衡

### 21.3.1 纳什均衡

**纳什均衡（Nash Equilibrium）**：
在其他厂商策略给定的情况下，每个厂商选择最优策略。

**定义**：
```
对于所有厂商i，sᵢ* 是厂商i对 s_{-i}* 的最优反应
```

**其中**：
- sᵢ*：厂商i的策略
- s_{-i}*：其他厂商的策略

**含义**：
- 在纳什均衡处，没有任何厂商有动力单方面改变策略
- 纳什均衡是策略相互依赖的结果

### 21.3.2 寡头市场的效率

**寡头市场的效率（Efficiency of Oligopoly Market）**：

**与完全竞争市场比较**：
- 寡头市场的价格高于完全竞争价格
- 寡头市场的产量低于完全竞争产量
- 寡头市场的无谓损失大于零

**与垄断市场比较**：
- 寡头市场的价格低于垄断价格
- 寡头市场的产量高于垄断产量
- 寡头市场的无谓损失小于垄断市场的无谓损失

## 21.4 寡头市场的策略行为

### 21.4.1 价格领导

**价格领导（Price Leadership）**：
一个厂商（领导者）制定价格，其他厂商（追随者）跟随。

**类型**：

**1. 支配型价格领导（Dominant Firm Price Leadership）**：
- 支配型厂商制定价格
- 小厂商作为价格接受者

**2. 晴雨表型价格领导（Barometric Price Leadership）**：
- 晴雨表型厂商制定价格
- 其他厂商跟随

### 21.4.2 卡特尔

**卡特尔（Cartel）**：
厂商之间合谋，共同决定产量和价格。

**特征**：
- 厂商之间合作
- 分配市场份额
- 制定统一价格

**卡特尔的不稳定性（Instability of Cartel）**：
- 每个厂商有动力背叛卡特尔
- 增加产量，获得更高利润
- 卡特尔容易崩溃

## 21.5 本章总结

### 核心要点

1. **寡头的定义**：
   - 寡头：少数几个卖者控制的市场
   - 寡头的特征：少数卖者、相互依赖、进入壁垒、产品同质或异质

2. **寡头市场的模型**：
   - 古诺模型：同时选择产量的寡头模型
   - 伯特兰模型：同时选择价格的寡头模型
   - 斯塔克伯格模型：先动和后动的寡头模型

3. **寡头市场的均衡**：
   - 纳什均衡：在其他厂商策略给定的情况下，每个厂商选择最优策略
   - 寡头市场的效率：介于完全竞争和垄断之间

4. **寡头市场的策略行为**：
   - 价格领导：一个厂商制定价格，其他厂商跟随
   - 卡特尔：厂商之间合谋，共同决定产量和价格

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 寡头 | 少数几个卖者控制的市场 | 少数卖者、相互依赖、进入壁垒 |
