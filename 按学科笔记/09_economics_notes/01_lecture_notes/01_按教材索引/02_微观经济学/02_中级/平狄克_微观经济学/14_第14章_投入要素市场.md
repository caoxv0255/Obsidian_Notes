# 第14章：投入要素市场

> 本章学习目标：
> - 理解要素市场的定义
> - 掌握竞争性要素市场
> - 学会分析垄断性要素市场
> - 理解买方垄断

---

## 14.1 要素市场的定义

### 14.1.1 要素市场的定义

**要素市场（Factor Market）**：
生产要素的市场。

### 14.1.2 要素的类型

**要素的类型（Types of Factors）**：

**1. 劳动（Labor）**：
- 工人的劳动服务
- 例如：工资

**2. 资本（Capital）**：
- 机器、设备、建筑物
- 例如：租金

**3. 土地（Land）**：
- 自然资源
- 例如：地租

**4. 企业家才能（Entrepreneurship）**：
- 企业家的才能
- 例如：利润

---

## 14.2 竞争性要素市场

### 14.2.1 竞争性要素市场的定义

**竞争性要素市场（Competitive Factor Market）**：
许多买者和卖者的要素市场。

### 14.2.2 竞争性要素市场的特征

**竞争性要素市场的特征（Characteristics of Competitive Factor Market）**：

**1. 许多买者和卖者（Many Buyers and Sellers）**：
- 市场上有许多买者和卖者
- 每个买者和卖者市场份额小

**2. 要素同质（Homogeneous Factors）**
- 要素是同质的
- 买者和卖者对要素无偏好

**3. 自由进入和退出（Free Entry and Exit）**
- 买者和卖者可以自由进入和退出市场
- 没有进入壁垒

**4. 完全信息（Perfect Information）**
- 买者和卖者拥有完全信息
- 没有信息不对称

### 14.2.3 要素需求

**要素需求（Factor Demand）**：
企业对要素的需求。

**利润最大化问题**：
```
max π = p × f(K, L) - r × K - w × L
```

**其中**：
- π：利润
- p：产品价格
- f(K, L)：生产函数
- K：资本
- L：劳动
- r：资本价格
- w：劳动价格

**利润最大化条件**：
```
MPₖ × p = r
MPₗ × p = w
```

**其中**：
- MPₖ：资本的边际产量
- MPₗ：劳动的边际产量
- p：产品价格
- r：资本价格
- w：劳动价格

**含义**：
- 资本的边际产品价值等于资本价格
- 劳动的边际产品价值等于劳动价格

### 14.2.4 边际产品价值

**边际产品价值（Value of Marginal Product, VMP）**：
边际产量乘以产品价格。

**公式**：
```
VMPₖ = MPₖ × p
VMPₗ = MPₗ × p
```

**其中**：
- VMPₖ：资本的边际产品价值
- VMPₗ：劳动的边际产品价值
- MPₖ：资本的边际产量
- MPₗ：劳动的边际产量
- p：产品价格

### 14.2.5 要素供给

**要素供给（Factor Supply）**：
要素所有者对要素的供给。

**利润最大化问题**：
```
max U = U(C, L)
```

**其中**：
- U：效用
- C：消费
- L：闲暇

**约束条件**：
```
C = w × (T - L) + π
```

**其中**：
- C：消费
- w：劳动价格
- T：总时间
- L：闲暇
- π：非劳动收入

**效用最大化条件**：
```
MUₗ/MU𝑐 = w
```

**其中**：
- MUₗ：闲暇的边际效用
- MU𝑐：消费的边际效用
- w：劳动价格

**含义**：
- 闲暇和消费的边际替代率等于劳动价格

## 14.3 垄断性要素市场

### 14.3.1 垄断性要素市场的定义

**垄断性要素市场（Monopolistic Factor Market）**：
要素市场是垄断的。

### 14.3.2 卖方垄断

**卖方垄断（Monopoly）**：
要素市场只有一个卖者。

**利润最大化问题**：
```
max π = r(K) × K - C(K)
```

**其中**：
- π：利润
- r(K)：资本价格函数
- K：资本
- C(K)：成本函数

**利润最大化条件**：
```
MR = MC
```

**其中**：
- MR：边际收益
- MC：边际成本

### 14.3.3 买方垄断

**买方垄断（Monopsony）**：
要素市场只有一个买者。

**利润最大化问题**：
```
max π = p × f(K, L) - w(L) × L
```

**其中**：
- π：利润
- p：产品价格
- f(K, L)：生产函数
- K：资本
- L：劳动
- w(L)：劳动价格函数

**利润最大化条件**：
```
MV = MFC
```

**其中**：
- MV：边际价值
- MFC：边际要素成本

**边际要素成本（Marginal Factor Cost, MFC）**：
增加一单位要素带来的成本变化。

**公式**：
```
MFC = w(L) + w'(L) × L
```

**其中**：
- MFC：边际要素成本
- w(L)：劳动价格函数
- w'(L)：劳动价格对劳动的导数
- L：劳动

**含义**：
- 边际要素成本大于劳动价格
- 因为增加劳动会导致劳动价格上升

## 14.4 本章总结

### 核心要点

1. **要素市场的定义**：
   - 要素市场：生产要素的市场
   - 要素的类型：劳动、资本、土地、企业家才能

2. **竞争性要素市场**：
   - 竞争性要素市场：许多买者和卖者的要素市场
   - 竞争性要素市场的特征：许多买者和卖者、要素同质、自由进入和退出、完全信息
   - 要素需求：MPₖ × p = r，MPₗ × p = w
   - 边际产品价值：VMPₖ = MPₖ × p，VMPₗ = MPₗ × p
   - 要素供给：MUₗ/MU𝑐 = w

3. **垄断性要素市场**：
   - 垄断性要素市场：要素市场是垄断的
   - 卖方垄断：要素市场只有一个卖者
   - 买方垄断：要素市场只有一个买者
   - 买方垄断的利润最大化条件：MV = MFC
   - 边际要素成本：MFC = w(L) + w'(L) × L
