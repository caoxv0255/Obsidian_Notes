# 第9章：竞争性市场的分析

> 本章学习目标：
> - 理解市场效率的定义
> - 掌握消费者剩余和生产者剩余
> - 学会分析市场均衡的变化

---

## 9.1 市场效率

### 9.1.1 市场效率的定义

**市场效率（Market Efficiency）**：
市场有效配置资源的能力。

### 9.1.2 市场效率的类型

**市场效率的类型（Types of Market Efficiency）**：

**1. 配置效率（Allocative Efficiency）**：
- 资源配置使总剩余最大化
- 价格等于边际成本

**2. 生产效率（Productive Efficiency）：
- 以最低成本生产商品
- 在生产可能性边界上生产

---

## 9.2 消费者剩余和生产者剩余

### 9.2.1 消费者剩余

**消费者剩余（Consumer Surplus）**：
消费者愿意支付的最高价格与实际支付价格之间的差额。

**公式**：
```
CS = ∫₀^Q D(p) dp - P × Q
```

**其中**：
- CS：消费者剩余
- D(p)：需求函数
- P：实际支付价格
- Q：消费量

### 9.2.2 生产者剩余

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

**公式**：
```
PS = P × Q - ∫₀^Q S(p) dp
```

**其中**：
- PS：生产者剩余
- S(p)：供给函数
- P：实际获得价格
- Q：供给量

### 9.2.3 总剩余

**总剩余（Total Surplus）**：
消费者剩余和生产者剩余的总和。

**公式**：
```
TS = CS + PS
```

**其中**：
- TS：总剩余
- CS：消费者剩余
- PS：生产者剩余

**含义**：
- 总剩余衡量社会总福利
- 总剩余最大化表示市场效率

## 9.3 市场均衡的变化

### 9.3.1 需求变化

**需求增加（Increase in Demand）**：
- 需求曲线向右移动
- 均衡价格上升
- 均衡数量增加

**需求减少（Decrease in Demand）**：
- 需求曲线向左移动
- 均衡价格下降
- 均衡数量减少

### 9.3.2 供给变化

**供给增加（Increase in Supply）**：
- 供给曲线向右移动
- 均衡价格下降
- 均衡数量增加

**供给减少（Decrease in Supply）**：
- 供给曲线向左移动
- 均衡价格上升
- 均衡数量减少

## 9.4 政府干预

### 9.4.1 价格上限

**价格上限（Price Ceiling）**：
政府规定的最高价格。

**效果**：
- 价格低于均衡价格
- 消费者剩余可能增加或减少
- 生产者剩余减少
- 总剩余减少（无谓损失）

**分析**：
- DWL：无谓损失（Deadweight Loss）
- 价格上限导致无谓损失

### 9.4.2 价格下限

**价格下限（Price Floor）**：
政府规定的最低价格。

**效果**：
- 价格高于均衡价格
- 消费者剩余减少
- 生产者剩余可能增加或减少
- 总剩余减少（无谓损失）

## 9.5 本章总结

### 核心要点

1. **市场效率**：
   - 市场效率：市场有效配置资源的能力
   - 市场效率的类型：配置效率、生产效率

2. **消费者剩余和生产者剩余**：
   - 消费者剩余：CS = ∫₀^Q D(p) dp - P × Q
   - 生产者剩余：PS = P × Q - ∫₀^Q S(p) dp
   - 总剩余：TS = CS + PS

3. **市场均衡的变化**：
   - 需求增加：需求曲线向右移动，均衡价格上升，均衡数量增加
   - 需求减少：需求曲线向左移动，均衡价格下降，均衡数量减少
   - 供给增加：供给曲线向右移动，均衡价格下降，均衡数量增加
   - 供给减少：供给曲线向左移动，均衡价格上升，均衡数量减少

4. **政府干预**：
   - 价格上限：政府规定的最高价格，导致无谓损失
   - 价格下限：政府规定的最低价格，导致无谓损失
