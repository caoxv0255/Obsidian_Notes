# 第8章：利润最大化和竞争性供给

> 本章学习目标：
> - 理解利润的定义
> - 掌握利润最大化的条件
> - 学会分析竞争性企业的供给
> - 理解竞争性市场的长期均衡

---

## 8.1 利润的定义

### 8.1.1 利润的定义

**利润（Profit）**：
总收入减去总成本。

### 8.1.2 利润的公式

**利润的公式（Formula of Profit）**：
```
π = TR - TC
```

**其中**：
- π：利润
- TR：总收入
- TC：总成本

### 8.1.3 利润的类型

**利润的类型（Types of Profit）**：

**1. 会计利润（Accounting Profit）**：
- 总收入减去显性成本
- 公式：πₐ = TR - TCₐ

**2. 经济利润（Economic Profit）**：
- 总收入减去总成本（包括机会成本）
- 公式：πₑ = TR - TCₑ

**3. 正常利润（Normal Profit）**：
- 经济利润为零
- 企业获得正常利润

---

## 8.2 利润最大化

### 8.2.1 利润最大化问题

**利润最大化问题（Profit Maximization Problem）**：
选择产量，使利润最大化。

**数学表述**：
```
max π = p × Q - C(Q)
```

**其中**：
- π：利润
- p：价格
- Q：产量
- C(Q)：成本函数

### 8.2.2 利润最大化的条件

**利润最大化的条件（Condition of Profit Maximization）**：
```
MR = MC
```

**其中**：
- MR：边际收益
- MC：边际成本

**含义**：
- 边际收益等于边际成本
- 利润最大化的必要条件

### 8.2.3 利润最大化的二阶条件

**利润最大化的二阶条件（Second-Order Condition of Profit Maximization）**：
```
dMC/dQ > 0
```

**含义**：
- 边际成本递增
- 利润最大化的充分条件

## 8.3 竞争性企业的供给

### 8.3.1 竞争性企业的定义

**竞争性企业（Competitive Firm）**：
在完全竞争市场上的企业。

### 8.3.2 竞争性企业的特征

**竞争性企业的特征（Characteristics of Competitive Firm）**：

**1. 价格接受者（Price Taker）**：
- 企业不能影响市场价格
- 企业只能接受市场价格

**2. 产品同质（Homogeneous Product）**
- 企业生产的产品是同质的
- 消费者对产品无偏好

**3. 自由进入和退出（Free Entry and Exit）**
- 企业可以自由进入和退出市场
- 没有进入壁垒

**4. 完全信息（Perfect Information）**
- 企业和消费者拥有完全信息
- 没有信息不对称

### 8.3.3 竞争性企业的供给曲线

**竞争性企业的供给曲线（Supply Curve of Competitive Firm）**：
表示价格和供给量之间关系的曲线。

**图形表示**：

## 8.4 竞争性市场的长期均衡

### 8.4.1 竞争性市场的长期均衡

**竞争性市场的长期均衡（Long-Run Equilibrium of Competitive Market）**：
- 价格等于边际成本
- 价格等于平均成本
- 经济利润为零
- 企业获得正常利润

### 8.4.2 竞争性市场的长期均衡条件

**竞争性市场的长期均衡条件（Condition of Long-Run Equilibrium of Competitive Market）**：
```
P = MC = AC
```

**其中**：
- P：价格
- MC：边际成本
- AC：平均成本

**含义**：
- 价格等于边际成本
- 价格等于平均成本
- 经济利润为零
- 企业获得正常利润

## 8.5 生产者剩余

### 8.5.1 生产者剩余的定义

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

### 8.5.2 生产者剩余的计算

**生产者剩余的公式（Formula of Producer Surplus）**：
```
PS = P × Q - ∫₀^Q MC(q) dq
```

**其中**：
- PS：生产者剩余
- P：实际获得价格
- Q：供给量
- MC(q)：边际成本函数

**含义**：
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

**图形表示**：

```plotly

      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "Q"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true

  displayModeBar: false
  responsive: true
```

**分析**：
- MC：边际成本曲线
- PS：生产者剩余
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

## 8.6 本章总结

1. **利润的定义**：

2. **利润最大化**：
   - 利润最大化问题：max π = p × Q - C(Q)
   - 利润最大化的条件：MR = MC
   - 利润最大化的二阶条件：dMC/dQ > 0

3. **竞争性企业的供给**：
   - 竞争性企业：在完全竞争市场上的企业
   - 竞争性企业的特征：价格接受者、产品同质、自由进入和退出、完全信息
   - 竞争性企业的供给曲线：表示价格和供给量之间关系的曲线

4. **竞争性市场的长期均衡**：
   - 竞争性市场的长期均衡：价格等于边际成本，价格等于平均成本，经济利润为零
   - 竞争性市场的长期均衡条件：P = MC = AC

5. **生产者剩余**：
   - 生产者剩余：生产者实际获得的价格与愿意接受的最低价格之间的差额
   - 生产者剩余的计算：PS = P × Q - ∫₀^Q MC(q) dq
   - 生产者剩余等于价格线上方、边际成本曲线下方的面积
