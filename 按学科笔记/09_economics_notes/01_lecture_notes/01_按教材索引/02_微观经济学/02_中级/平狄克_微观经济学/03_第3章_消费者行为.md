> - 理解消费者行为的基本理论

---

## 3.1 消费者偏好

### 3.1.1 偏好的定义

**偏好（Preferences）**：
消费者对不同商品组合的排序。

### 3.1.2 偏好的性质

**偏好的性质（Properties of Preferences）**：

**1. 完备性（Completeness）**：
- 对于任意两个商品组合，消费者能够比较

**2. 传递性（Transitivity）**：
- 如果A优于B，B优于C，那么A优于C

**3. 良好性状（More is Better）**：
- 消费者偏好更多的商品

**4. 凸性（Convexity）**：
- 平均商品组合至少和极端商品组合一样好

### 3.1.3 无差异曲线

**无差异曲线（Indifference Curve）**：
给消费者带来相同满足程度的所有商品组合。

**图形表示**：

- 横轴：商品1的数量x₁
- 纵轴：商品2的数量x₂
- 无差异曲线向右下方倾斜
- 无差异曲线凸向原点

### 3.1.4 边际替代率

**边际替代率（Marginal Rate of Substitution，MRS）**：
在保持效用不变的前提下，消费者愿意用一种商品替代另一种商品的比率。

**公式**：
```
MRS₁₂ = -dx₂/dx₁ |du=0 = MU₁/MU₂
```

**其中**：
- MRS₁₂：商品1对商品2的边际替代率
- dx₂/dx₁：商品1增加一单位时，商品2需要减少的数量
- 负号：表明两种商品呈替代关系
- MU₁：商品1的边际效用
- MU₂：商品2的边际效用

**图形表示**：


**无差异曲线斜率与边际替代率的关系**：
- 无差异曲线的斜率 = -MRS
- 切线斜率越陡，边际替代率越大
- 切线斜率越平缓，边际替代率越小

**边际替代率递减规律（Diminishing MRS）**：
- 随着商品1的消费量增加，消费者愿意用商品1替代商品2的数量逐渐减少
- 原因：商品的边际效用递减

**举例**：
- 假设消费者有5个苹果和1个香蕉，MRS=3（愿意用3个苹果换1个香蕉）
- 当消费者只有1个苹果和5个香蕉时，MRS=1/3（只愿意用1/3个苹果换1个香蕉）

## 3.2 效用理论

### 3.2.1 效用的定义

**效用（Utility）**：
消费者从消费商品组合中获得的满足程度。

### 3.2.2 效用函数

**效用函数（Utility Function）**：
给每个商品组合分配一个效用值的函数。

**公式**：
```
u(x₁, x₂)
```

**其中**：
- u：效用
- x₁：商品1的数量
- x₂：商品2的数量

### 3.2.3 边际效用

**边际效用（Marginal Utility）**：
增加一单位商品消费带来的效用变化。

**公式**：
```
MU₁ = ∂u/∂x₁
```
```
MU₂ = ∂u/∂x₂
```

**其中**：
- MU₁：商品1的边际效用
- MU₂：商品2的边际效用
- ∂u/∂x₁：效用对x₁的偏导数
- ∂u/∂x₂：效用对x₂的偏导数

## 3.3 消费者选择

### 3.3.1 预算约束

**预算约束（Budget Constraint）**：
消费者能够购买的商品组合受到收入的限制。

**公式**：
```
p₁x₁ + p₂x₂ = m
```

**其中**：
- p₁：商品1的价格
- p₂：商品2的价格
- x₁：商品1的数量
- x₂：商品2的数量
- m：收入

### 3.3.2 消费者最优化

**消费者最优化（Consumer Optimization）**：
在预算约束下，选择最优商品组合，使效用最大化。

**数学表述**：
```
max u(x₁, x₂)
s.t. p₁x₁ + p₂x₂ = m
```

### 3.3.3 消费者最优选择的条件

**消费者最优选择的条件（Condition of Optimal Choice）**：
```
MU₁/p₁ = MU₂/p₂
```

**含义**：
- 每单位货币的边际效用相等
- 边际替代率等于价格比

---

## 3.4 收入效应和替代效应

### 3.4.1 替代效应

**替代效应（Substitution Effect）**：
由于商品相对价格变化，消费者调整商品组合，保持效用不变。

### 3.4.2 收入效应

**收入效应（Income Effect）**：
由于商品价格变化，消费者的实际收入变化，从而调整商品组合。

### 3.4.3 总效应

**总效应（Total Effect）**：
替代效应和收入效应的总和。

**公式**：
```
总效应 = 替代效应 + 收入效应
```

## 3.5 本章总结

### 核心要点

1. **消费者偏好**：
   - 偏好：消费者对不同商品组合的排序
   - 偏好的性质：完备性、传递性、良性性状、凸性
   - 无差异曲线：给消费者带来相同满足程度的所有商品组合

2. **效用理论**：
   - 效用：消费者从消费商品组合中获得的满足程度
   - 效用函数：给每个商品组合分配一个效用值的函数
   - 边际效用：增加一单位商品消费带来的效用变化

3. **消费者选择**：
   - 预算约束：消费者能够购买的商品组合受到收入的限制
   - 消费者最优化：在预算约束下，选择最优商品组合，使效用最大化
   - 消费者最优选择的条件：MU₁/p₁ = MU₂/p₂

4. **收入效应和替代效应**：
   - 替代效应：由于商品相对价格变化，消费者调整商品组合，保持效用不变
   - 收入效应：由于商品价格变化，消费者的实际收入变化，从而调整商品组合
   - 总效应：替代效应和收入效应的总和
