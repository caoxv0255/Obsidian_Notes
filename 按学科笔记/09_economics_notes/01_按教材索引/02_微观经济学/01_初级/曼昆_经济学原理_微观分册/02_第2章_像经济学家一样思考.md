# 第2章：像经济学家一样思考

> 本章学习目标：
> - 理解经济学作为一门科学的性质
> - 掌握经济学模型的作用
> - 学会区分实证分析和规范分析
> - 理解经济学家的分歧来源

---

## 2.1 经济学家作为科学家

### 2.1.1 经济学作为一门科学

**经济学作为科学**：
经济学是一门研究社会如何管理其稀缺资源的科学。

**经济学的科学方法**：

1. **观察（Observation）**：
   - 观察经济现象
   - 收集数据

2. **提出理论（Theorize）**：
   - 提出假设
   - 建立模型

3. **检验理论（Test）**：
   - 收集数据
   - 统计分析
   - 验证或拒绝理论

**经济学与其他科学的区别**：

| 特征 | 自然科学 | 经济学 |
|------|----------|--------|
| **研究对象** | 自然现象 | 人类行为 |
| **实验方法** | 可控实验 | 很难进行实验 |
| **确定性** | 高确定性 | 不确定性高 |
| **道德判断** | 价值中性 | 涉及价值判断 |

### 2.1.2 假设的作用

**假设（Assumption）**：
简化复杂现实，使问题易于分析。

**假设的作用**：
1. **简化现实**：
   - 忽略不重要因素
   - 专注于核心关系

2. **建立模型**：
   - 基于假设建立模型
   - 分析核心机制

3. **得出结论**：
   - 基于假设推理
   - 得出可检验的预测

**经济学的常见假设**：

1. **理性人假设**：
   - 人们追求效用最大化或利润最大化
   - 系统地追求目标

2. **完全信息假设**：
   - 买者和卖者了解所有相关信息
   - 没有信息不对称

3. **其他条件不变（Ceteris Paribus）**：
   - 其他因素保持不变
   - 只分析一个因素的变化

**例子**：

**需求曲线的假设**：
- 假设：收入、偏好、相关商品价格不变
- 分析：价格变化对需求量的影响
- 结论：需求曲线向下倾斜

### 2.1.3 经济学模型

**经济学模型（Economic Model）**：
对现实经济的简化表述，用于分析经济问题。

**模型的类型**：

1. **文字模型（Verbal Models）**：
   - 用文字描述经济关系
   - 例如：需求定理

2. **数学模型（Mathematical Models）**：
   - 用数学方程描述经济关系
   - 例如：需求函数 Qd = a - bP

3. **图形模型（Graphical Models）**：
   - 用图形描述经济关系
   - 例如：需求曲线

**模型的作用**：
- 解释经济现象
- 预测经济变化
- 指导政策制定

**模型的局限性**：
- 简化现实
- 假设可能不成立
- 预测可能不准确

## 2.2 第一个模型：循环流量图

### 2.2.1 循环流量图的定义

**循环流量图（Circular-Flow Diagram）**：
一个说明货币如何通过市场在家庭和企业之间流动的直观经济模型。

### 2.2.2 循环流量图的组成部分

**两类决策者**：

1. **家庭（Households）**：
   - 拥有生产要素（劳动、土地、资本）
   - 消费商品和服务

2. **企业（Firms）**：
   - 雇佣生产要素
   - 生产商品和服务

**两类市场**：

1. **生产要素市场（Markets for Factors of Production）**：
   - 家庭出售生产要素
   - 企业购买生产要素

2. **商品和服务市场（Markets for Goods and Services）**：
   - 企业出售商品和服务
   - 家庭购买商品和服务

### 2.2.3 循环流量图的图形

**图形表示**：

```
            生产要素市场
                ↑
                │ 生产要素
                │
        家庭 ────────→ 企业
        │              │
        │              │
        │商品和服务    │
        │              │
        ←──────────────
            商品和服务市场
```

**流程说明**：

**内圈（实物流）**：
1. 家庭 → 生产要素市场 → 企业：家庭提供生产要素
2. 企业 → 商品和服务市场 → 家庭：企业提供商品和服务

**外圈（货币流）**：
1. 企业 → 生产要素市场 → 家庭：企业支付工资、租金、利润
2. 家庭 → 商品和服务市场 → 企业：家庭支付购买商品和服务的费用

### 2.2.4 循环流量图的意义

**意义**：
1. **展示经济的相互联系**：
   - 家庭和企业相互依赖
   - 市场连接家庭和企业

2. **理解收入循环**：
   - 家庭的收入来自企业
   - 企业的收入来自家庭
   - 收入在经济中循环

3. **分析经济问题**：
   - 失业：劳动力市场
   - 通货膨胀：商品和服务市场
   - 收入分配：要素市场和商品市场

## 2.3 第二个模型：生产可能性边界

### 2.3.1 生产可能性边界的定义

**生产可能性边界（Production Possibilities Frontier, PPF）**：
表示在给定资源和技术下，一个经济可以生产的两种商品的所有可能组合。

**图中标记**：
- A点：全生产商品B
- B点：全生产商品A
- PPF：生产可能性边界
- PPF上的点：有效率的资源配置
- PPF内的点：资源未充分利用
- PPF外的点：无法实现

### 2.3.3 生产可能性边界的含义

**1. 效率（Efficiency）**：
- PPF上的点：有效率的资源配置
- PPF内的点：资源未充分利用（失业、低效率）
- PPF外的点：无法实现（资源不足）

**2. 机会成本（Opportunity Cost）**：
- 从A点移动到B点：放弃的商品B = 增加的商品A
- 机会成本递增：沿着PPF移动，机会成本递增

**3. 权衡取舍（Trade-off）**：
- 多生产商品A，必须减少商品B
- 无法同时增加两种商品

## 2.4 微观经济学与宏观经济学

### 2.4.1 微观经济学

**微观经济学（Microeconomics）**：
研究家庭和企业如何做出决策，以及它们如何在市场上相互交易。

**研究范围**：
1. **消费者行为**：
   - 消费者选择
   - 需求理论
   - 效用最大化

2. **生产者行为**：
   - 生产决策
   - 供给理论
   - 利润最大化

3. **市场结构**：
   - 完全竞争
   - 垄断
   - 垄断竞争
   - 寡头

4. **要素市场**：
   - 劳动市场
   - 资本市场
   - 土地市场

### 2.4.2 宏观经济学

**宏观经济学（Macroeconomics）**：
研究整体经济现象，包括通货膨胀、失业和经济增长。

**研究范围**：
1. **经济增长**：
   - GDP增长
   - 生产率
   - 生活水平

2. **经济周期**：
   - 经济波动
   - 繁荣和衰退
   - 经济政策

3. **通货膨胀**：
   - 物价水平
   - 货币政策

4. **失业**：
   - 失业率
   - 劳动市场

### 2.4.3 微观与宏观的关系

**关系**：
- 宏观是微观的总和
- 宏观现象基于微观行为
- 两者相互联系

**例子**：
- 微观：企业决定投资
- 宏观：总投资影响经济增长

## 2.5 实证分析与规范分析

### 2.5.1 实证分析

**实证分析（Positive Analysis）**：
描述世界是什么样子的陈述。

**特点**：
- 客观描述
- 可以检验
- 不涉及价值判断

**例子**：
- "最低工资法导致失业增加"
- "减税可以刺激经济增长"
- "通货膨胀导致货币贬值"

### 2.5.2 规范分析

**规范分析（Normative Analysis）**：
描述世界应该是什么样子的陈述。

**特点**：
- 主观判断
- 涉及价值观
- 难以检验

**例子**：
- "政府应该提高最低工资"
- "政府应该减税"
- "政府应该控制通货膨胀"

### 2.5.3 两者的区别

**区别**：

| 特征 | 实证分析 | 规范分析 |
|------|----------|----------|
| **性质** | 描述性 | 指令性 |
| **可检验性** | 可以检验 | 难以检验 |
| **客观性** | 客观 | 主观 |
| **目的** | 解释现象 | 指导政策 |

**例子**：

**实证陈述**：
- "提高最低工资会使失业率上升5%"
- 这是可以检验的

**规范陈述**：
- "政府应该提高最低工资"
- 这涉及价值判断

### 2.5.4 经济学家的分歧

**经济学家的分歧来源**：

1. **实证分歧（科学分歧）**：
   - 对经济现象的不同理解
   - 对理论的不同检验结果
   - 例子：减税对经济增长的影响

2. **规范分歧（价值分歧）**：
   - 不同的价值观
   - 不同的政策目标
   - 例子：公平 vs 效率

**实证分歧的例子**：
- 问题：减税是否刺激经济增长？
- 经济学家A：是，减税刺激投资和消费
- 经济学家B：否，减税导致预算赤字

**规范分歧的例子**：
- 问题：是否应该提高最低工资？
- 经济学家A：应该，保护低收入工人
- 经济学家B：不应该，导致失业增加

## 2.6 经济学家的政策建议

### 2.6.1 政策制定的过程

**政策制定的过程**：
1. **实证分析**：
   - 分析现状
   - 收集数据
   - 建立模型

2. **规范分析**：
   - 确定目标
   - 评估不同政策
   - 选择最优政策

3. **政策实施**：
   - 设计政策
   - 实施政策
   - 监测效果

### 2.6.2 经济学家的角色

**经济学家的角色**：

1. **科学家（Scientist）**：
   - 进行实证分析
   - 检验理论
   - 提供客观信息

2. **政策顾问（Policy Advisor）**：
   - 进行规范分析
   - 提供政策建议
   - 影响政策制定

**例子**：

**作为科学家**：
- "数据显示最低工资上升1%，失业率上升0.5%"
- 客观描述，不涉及价值判断

**作为政策顾问**：
- "基于数据，政府应该谨慎提高最低工资"
- 提供政策建议，涉及价值判断

1. **经济学家作为科学家**

   - 使用假设简化现实
   - 使用经济模型分析问题

4. **微观经济学与宏观经济学**
   - 微观：家庭和企业的决策
   - 宏观：整体经济现象
   - 两者相互联系

5. **实证分析与规范分析**
   - 实证分析：描述世界是什么
   - 规范分析：描述世界应该是什么
   - 经济学家的分歧来源

6. **经济学家的政策建议**
   - 科学家角色：提供客观信息
   - 政策顾问角色：提供政策建议
