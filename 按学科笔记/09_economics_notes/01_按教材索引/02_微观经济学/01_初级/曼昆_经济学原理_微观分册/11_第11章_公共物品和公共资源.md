## 11.1 物品的类型

### 11.1.1 物品的分类

**物品的分类**：

根据两个特征：
1. **排他性（Excludability）**：
   - 可以阻止某人使用该物品
   - 例如：收费道路、私人物品

2. **竞争性（Rivalry）**：
   - 一个人的使用会减少其他人的使用
   - 例如：食品、私人物品

### 11.1.2 四种类型

**四种类型**：

| 类型 | 排他性 | 竞争性 | 例子 |
|------|--------|--------|------|
| **私人物品（Private Goods）** | 有 | 有 | 食品、衣服、汽车 |
| **公共物品（Public Goods）** | 无 | 无 | 国防、路灯、知识 |
| **公共资源（Common Resources）** | 无 | 有 | 海洋鱼类、草原、空气 |
| **俱乐部物品（Club Goods）** | 有 | 无 | 有线电视、收费公路 |

## 11.2 公共物品

### 11.2.1 公共物品的特征

**公共物品的特征**：

1. **非排他性（Non-Excludability）**：
   - 无法阻止某人使用该物品
   - 例如：国防、路灯

2. **非竞争性（Non-Rivalry）**：
   - 一个人的使用不会减少其他人的使用
   - 例如：国防、知识

### 11.2.2 搭便车问题

**搭便车问题（Free-Rider Problem）**：
人们不付费却享受公共物品的好处。

**原因**：
- 公共物品的非排他性
- 无法阻止不付费的人使用
- 人们有激励不付费

**结果**：
- 私人市场供给不足
- 市场失灵

**例子**：

**国防**：
- 国防保护所有人
- 无法阻止不付费的人享受保护
- 人们有激励不付费
- 私人市场无法提供国防

**路灯**：
- 路灯照亮所有人
- 无法阻止不付费的人享受照明
- 人们有激励不付费
- 私人市场无法提供路灯

**分析**：

**关键点**：
- 公共物品的需求是个人需求的垂直加总
- 原因：公共物品具有非竞争性
- 所有人消费相同数量
- 愿意支付的价格 = 个人愿意支付的价格之和

**例子**：

**国防**：
- 个人A愿意为国防支付100元
- 个人B愿意为国防支付200元
- 社会愿意为国防支付300元

### 11.2.4 公共物品的最优供给

**公共物品的最优供给**：

**条件**：
- 边际收益 = 边际成本
- 边际收益 = 所有人的边际收益之和

**例子**：

**国防**：
- 个人A的边际收益 = 100元
- 个人B的边际收益 = 200元
- 社会边际收益 = 300元
- 边际成本 = 300元
- 最优供给：国防

### 11.2.5 解决公共物品问题的政策

**解决公共物品问题的政策**：

1. **政府提供**：
   - 政府提供公共物品
   - 通过税收筹资
   - 例如：国防、路灯

2. **成本收益分析**：
   - 评估公共物品的成本和收益
   - 决定是否提供公共物品
   - 例如：基础设施、教育

3. **私有化**：
   - 将公共物品私有化
   - 使其具有排他性
   - 例如：收费道路、有线电视

**例子**：

**国防**：
- 政府提供国防
- 通过税收筹资
- 无法私有化

**路灯**：
- 政府提供路灯
- 通过税收筹资
- 可以私有化（路灯广告）

## 11.3 公共资源

### 11.3.1 公共资源的特征

**公共资源的特征**：

1. **非排他性（Non-Excludability）**：
   - 无法阻止某人使用该资源
   - 例如：海洋鱼类、草原

2. **竞争性（Rivalry）**：
   - 一个人的使用会减少其他人的使用
   - 例如：海洋鱼类、草原

### 11.3.2 公地悲剧

**公地悲剧（Tragedy of the Commons）**：
由于公共资源的非排他性和竞争性，导致资源过度使用。

**原因**：
- 公共资源的非排他性
- 无法阻止人们使用
- 人们有激励过度使用
- 导致资源枯竭

**结果**：
- 资源过度使用
- 资源枯竭
- 市场失灵

**例子**：

**海洋鱼类**：
- 海洋鱼类是公共资源
- 无法阻止人们捕鱼
- 渔民有激励过度捕鱼
- 导致鱼类资源枯竭

**草原**：
- 草原是公共资源
- 无法阻止人们放牧
- 牧民有激励过度放牧
- 导致草原退化

- 私人市场数量 Qm > 社会最优数量 Q*
- 过度使用
- 无谓损失

### 11.3.4 解决公共资源问题的政策

**解决公共资源问题的政策**：

1. **产权界定**：
   - 将公共资源私有化
   - 使其具有排他性
   - 例如：土地私有化

2. **管制**：
   - 政府限制使用
   - 例如：捕鱼配额、放牧限制

3. **税收**：
   - 对使用征收税收
   - 例如：污染税、资源税

4. **可交易许可证**：
   - 政府发放使用许可证
   - 允许交易许可证
   - 例如：捕鱼许可证、碳排放许可证

**例子**：

**海洋鱼类**：
- 政府发放捕鱼许可证
- 限制总捕捞量
- 允许交易许可证
- 避免过度捕鱼

**草原**：
- 政府限制放牧
- 规定放牧数量
- 避免过度放牧

## 11.4 物品类型的比较

### 11.4.1 四种类型的比较

**四种类型的比较**：

| 类型 | 排他性 | 竞争性 | 市场失灵 | 解决方案 |
|------|--------|--------|----------|----------|
| **私人物品** | 有 | 有 | 无 | 市场有效 |
| **公共物品** | 无 | 无 | 供给不足 | 政府提供 |
| **公共资源** | 无 | 有 | 过度使用 | 产权界定、管制、税收、许可证 |
| **俱乐部物品** | 有 | 无 | 潜在供给不足 | 私人提供、政府管制 |

### 11.4.2 实际例子

**实际例子**：

1. **食品**：
   - 类型：私人物品
   - 市场有效

2. **国防**：
   - 类型：公共物品
   - 市场失灵：供给不足
   - 解决方案：政府提供

3. **海洋鱼类**：
   - 类型：公共资源
   - 市场失灵：过度使用
   - 解决方案：管制、许可证

4. **有线电视**：
   - 类型：俱乐部物品
   - 市场有效

## 11.5 本章总结

### 核心要点

1. **物品的类型**：
   - 根据排他性和竞争性分为四种类型
   - 私人物品、公共物品、公共资源、俱乐部物品

2. **公共物品**：
   - 非排他性、非竞争性
   - 搭便车问题
   - 市场供给不足
   - 解决方案：政府提供、成本收益分析、私有化

3. **公共资源**：
   - 非排他性、竞争性
   - 公地悲剧
   - 过度使用
   - 解决方案：产权界定、管制、税收、可交易许可证

4. **物品类型的比较**：
   - 不同类型有不同的市场失灵
   - 需要不同的解决方案

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 排他性 | 可以阻止某人使用该物品 | 有排他性：私人物品、俱乐部物品 |
| 竞争性 | 一个人的使用会减少其他人的使用 | 有竞争性：私人物品、公共资源 |
| 私人物品 | 有排他性、有竞争性 | 市场有效 |
| 公共物品 | 无排他性、无竞争性 | 搭便车问题，供给不足 |
| 公共资源 | 无排他性、有竞争性 | 公地悲剧，过度使用 |
| 俱乐部物品 | 有排他性、无竞争性 | 潜在供给不足 |
| 搭便车问题 | 人们不付费却享受公共物品的好处 | 公共物品的市场失灵 |

| 成本收益分析 | 评估公共物品的成本和收益 | 决定是否提供公共物品 |
