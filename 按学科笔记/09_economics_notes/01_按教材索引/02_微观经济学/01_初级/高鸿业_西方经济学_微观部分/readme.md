## 重要概念

### 基本概念
- 稀缺性（Scarcity）
- 机会成本（Opportunity Cost）
- 边际分析（Marginal Analysis）
- 激励（Incentives）
- 市场经济（Market Economy）

### 市场分析
- 供给（Supply）
- 需求（Demand）
- 均衡（Equilibrium）
- 弹性（Elasticity）
- 市场失灵（Market Failure）

### 消费者行为
- 效用（Utility）
- 预算约束（Budget Constraint）
- 消费者均衡（Consumer Equilibrium）
- 替代效应（Substitution Effect）
- 收入效应（Income Effect）

### 生产者行为
- 生产函数（Production Function）
- 成本函数（Cost Function）
- 利润最大化（Profit Maximization）
- 市场结构（Market Structure）
- 垄断（Monopoly）
