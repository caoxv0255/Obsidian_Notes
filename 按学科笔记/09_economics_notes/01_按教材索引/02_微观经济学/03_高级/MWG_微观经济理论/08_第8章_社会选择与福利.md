## 8.1 社会福利函数

### 8.1.1 社会福利函数的定义

**社会福利函数（Social Welfare Function, SWF）**：
将个体效用映射为社会福利的函数。

**公式**：
```
W = W(u¹, u², ..., uᴷ)
```
**其中**：
- W：社会福利
- uᵏ：消费者k的效用

### 8.1.2 社会福利函数的类型

**社会福利函数的类型（Types of Social Welfare Functions）**：

1. **功利主义社会福利函数（Utilitarian Social Welfare Function）**：
   ```
   W = Σₖ uᵏ
   ```
   - 简单加总
   - 不关注分配
   - 边际社会福利：∂W/∂uᵏ = 1

2. **罗尔斯社会福利函数（Rawlsian Social Welfare Function）**：
   ```
   W = minₖ uᵏ
   ```
   - 关注最贫困者
   - 关注分配
   - 边际社会福利：∂W/∂uᵏ = 1，如果uᵏ = minₖ uᵏ；否则0

3. **伯格森-萨缪尔森社会福利函数（Bergson-Samuelson Social Welfare Function）**：
   ```
   W = W(u¹, u², ..., uᴷ)
   ```
   - 一般形式
   - 可以关注分配
   - 边际社会福利：∂W/∂uᵏ = Wₖ

4. **纳什社会福利函数（Nash Social Welfare Function）**：
   ```
   W = Πₖ uᵏ
   ```
   - 效用乘积
   - 关注分配
   - 边际社会福利：∂W/∂uᵏ = (Πⱼ uʲ) / uᵏ

### 8.1.3 社会福利函数的性质

**社会福利函数的性质（Properties of Social Welfare Function）**：

1. **非减性（Non-decreasing）**：
   - 对于所有k，∂W/∂uᵏ ≥ 0

2. **严格递增（Strictly Increasing）**：
   - 对于所有k，∂W/∂uᵏ > 0

3. **对称性（Symmetry）**：
   - 对于任意排列π，W(u¹, ..., uᴷ) = W(u^{π(1)}, ..., u^{π(K)})

4. **凹性（Concavity）**：
   - W(λu + (1-λ)u') ≥ λW(u) + (1-λ)W(u')

## 8.2 阿罗不可能定理

### 8.2.1 社会选择问题

**社会选择问题（Social Choice Problem）**：
给定个体偏好，如何选择社会偏好。

**符号**：
- 个体偏好：≿₁, ..., ≿ᴷ
- 社会偏好：≿
- 社会选择函数：f(≿₁, ..., ≿ᴷ) = ≿

### 8.2.2 阿罗的条件

**阿罗的条件（Arrow's Conditions）**：

1. **无限制域（Unrestricted Domain, U）**：
   - 社会选择函数定义在所有可能的个体偏好上
   - 对于任意个体偏好组合，都有定义

2. **帕累托原则（Pareto Principle, P）**：
   - 如果所有个体都偏好x > y，则社会偏好x > y
   - 弱帕累托原则：如果所有个体都弱偏好x ≥ y，则社会弱偏好x ≥ y

3. **无关选项的独立性（Independence of Irrelevant Alternatives, IIA）**：
   - 社会对x和y的偏好只取决于个体对x和y的偏好
   - 不取决于个体对其他选项的偏好

4. **非独裁性（Non-dictatorship, D）**：
   - 不存在个体独裁者，其偏好决定社会偏好
   - 即不存在k，使得对于所有x, y，如果x ≿ₖ y，则x ≿ y

### 8.2.3 阿罗不可能定理

**阿罗不可能定理（Arrow's Impossibility Theorem）**：

**定理**：
不存在同时满足以下四个条件的社会选择函数：
1. 无限制域（U）
2. 帕累托原则（P）
3. 无关选项的独立性（IIA）
4. 非独裁性（D）

**证明思路**：
1. 构造关键个体（Pivotal Agent）
2. 证明关键个体是独裁者
3. 与非独裁性矛盾

### 8.2.4 阿罗不可能定理的含义

**阿罗不可能定理的含义（Implications of Arrow's Impossibility Theorem）**：
- 不能同时满足所有四个条件
- 必须放弃至少一个条件
- 常见的解决方案：
  - 限制个体偏好域（例如：单峰偏好）
  - 放弃无关选项的独立性
  - 引入基数效用（例如：社会福利函数）

## 8.3 福利评价

### 8.3.1 福利评价的方法

**福利评价的方法（Methods of Welfare Evaluation）**：

1. **帕累托标准（Pareto Criterion）**：
   - 如果配置x帕累托优于配置y，则x的福利高于y
   - 问题：不能比较所有配置

2. **卡尔多-希克斯标准（Kaldor-Hicks Criterion）**：
   - 如果从配置y到配置x的受益者可以补偿受损者，且仍有剩余，则x的福利高于y
   - 问题：补偿不一定实际发生

3. **社会福利函数（Social Welfare Function）**：
   - 使用社会福利函数评价配置
   - 问题：社会福利函数的选择是主观的

### 8.3.2 等价变分和补偿变分

**等价变分（Equivalent Variation, EV）**：
使消费者在价格变化前后的效用相同的收入变化。

**公式**：
```
EV = e(p₀, u(p₁, w)) - w
```
**其中**：
- p₀：初始价格
- p₁：新价格
- w：初始收入
- e(p₀, u(p₁, w))：在新价格下的效用，在初始价格下的支出

**补偿变分（Compensating Variation, CV）**：
使消费者在价格变化后的效用等于初始效用的收入变化。

**公式**：
```
CV = w - e(p₁, u(p₀, w))
```
**其中**：
- p₀：初始价格
- p₁：新价格
- w：初始收入
- e(p₁, u(p₀, w))：在初始价格下的效用，在新价格下的支出

### 8.3.3 消费者剩余

**消费者剩余（Consumer Surplus, CS）**：
消费者愿意支付的价格与实际支付的价格之差。

**公式**：
```
CS = ∫₀ᵠ p(q)dq - p·q
```
**其中**：
- p(q)：反需求函数
- q：数量
- p：价格

**性质**：
- 消费者剩余是效用的货币度量
- 对于拟线性效用函数，消费者剩余等于效用的精确度量
- 对于一般效用函数，消费者剩余是效用的近似度量

## 8.4 不平等度量

### 8.4.1 不平等的度量

**不平等的度量（Measurement of Inequality）**：

1. **基尼系数（Gini Coefficient）**：
   ```
   G = (1 / (2n²μ)) Σᵢ Σⱼ |xᵢ - xⱼ|
   ```
   **其中**：
   - G：基尼系数
   - n：人口数量
   - μ：平均收入
   - xᵢ：个体i的收入

   - G ∈ [0, 1]
   - G = 0：完全平等
   - G = 1：完全不平等

2. **泰尔指数（Theil Index）**：
   ```
   T = (1 / n) Σᵢ (xᵢ / μ)·ln(xᵢ / μ)
   ```
   **其中**：
   - T：泰尔指数
   - xᵢ：个体i的收入
   - μ：平均收入

   - T ∈ [0, ln(n)]
   - T = 0：完全平等
   - T = ln(n)：完全不平等

3. **方差系数（Coefficient of Variation, CV）**：
   ```
   CV = σ / μ
   ```
   **其中**：
   - CV：方差系数
   - σ：标准差
   - μ：平均收入

   - CV ∈ [0, +∞)
   - CV = 0：完全平等

### 8.4.2 不平等的分解

**不平等的分解（Decomposition of Inequality）**：
- 按组分解（Between-Group Inequality）
- 组内分解（Within-Group Inequality）

**泰尔指数的分解（Theil Index Decomposition）**：
```
T = T_B + T_W
```
**其中**：
- T：总体不平等
- T_B：组间不平等
- T_W：组内不平等

**公式**：
```
T_B = (1 / n) Σ_g n_g·(μ_g / μ)·ln(μ_g / μ)
T_W = (1 / n) Σ_g Σᵢ∈g (xᵢ / μ_g)·ln(xᵢ / μ_g)
```
**其中**：
- g：组
- n_g：组g的人口数量
- μ_g：组g的平均收入
- μ：总体平均收入

### 8.4.3 不平等与福利

**不平等与福利（Inequality and Welfare）**：
- 不平等影响社会福利
- 社会福利函数可以反映对不平等的态度

**例子**：
- 功利主义社会福利函数：不关注不平等
- 罗尔斯社会福利函数：极端关注不平等
- 伯格森-萨缪尔森社会福利函数：可以反映对不平等的态度

## 8.5 本章总结

### 核心要点

1. **社会福利函数**：
   - 社会福利函数：W = W(u¹, u², ..., uᴷ)
   - 社会福利函数的类型：功利主义、罗尔斯、伯格森-萨缪尔森、纳什
   - 社会福利函数的性质：非减性、严格递增、对称性、凹性

2. **阿罗不可能定理**：
   - 阿罗的条件：无限制域、帕累托原则、无关选项的独立性、非独裁性
   - 阿罗不可能定理：不存在同时满足四个条件的社会选择函数
   - 阿罗不可能定理的含义：必须放弃至少一个条件

3. **福利评价**：
   - 福利评价的方法：帕累托标准、卡尔多-希克斯标准、社会福利函数
   - 等价变分：EV = e(p₀, u(p₁, w)) - w
   - 补偿变分：CV = w - e(p₁, u(p₀, w))
   - 消费者剩余：CS = ∫₀ᵠ p(q)dq - p·q

4. **不平等度量**：
   - 不平等的度量：基尼系数、泰尔指数、方差系数
   - 不平等的分解：T = T_B + T_W
   - 不平等与福利：不平等影响社会福利
