# 第9章：博弈论：静态博弈

> 本章学习目标：
> - 理解纳什均衡
> - 掌握混合策略纳什均衡
> - 学会分析重复剔除被占优策略

## 9.1 博弈的表示

### 9.1.1 博弈的定义

**博弈（Game）**：
G = {I, (Sᵢ)ᵢ∈I, (uᵢ)ᵢ∈I}
- I：参与人集合
- Sᵢ：参与人i的策略空间
- uᵢ：参与人i的收益函数

### 9.1.2 策略型博弈

**策略型博弈（Strategic-Form Game）**：
参与人同时选择策略，收益取决于所有参与人的策略组合。

**表示**：
- 策略组合：s = (s₁, ..., sₙ)
- 参与人i的收益：uᵢ(s) = uᵢ(s₁, ..., sₙ)

### 9.1.3 收益矩阵

**收益矩阵（Payoff Matrix）**：
表示双人博弈的收益。

**例子（囚徒困境）**：

```
      囚徒2
      坦白    沉默
坦白  -3, -3  0, -6
囚徒1
沉默  -6, 0   -1, -1
```

**分析**：
- （坦白，坦白）：纳什均衡
- （沉默，沉默）：帕累托最优
- 囚徒困境：个人理性导致集体非理性

## 9.2 纳什均衡

### 9.2.1 纳什均衡的定义

**纳什均衡（Nash Equilibrium）**：
策略组合s*是纳什均衡，如果对于所有i和所有sᵢ ∈ Sᵢ，有：
```
uᵢ(s*ᵢ, s*₋ᵢ) ≥ uᵢ(sᵢ, s*₋ᵢ)
```
**其中**：
- s*ᵢ：参与人i的均衡策略
- s*₋ᵢ：其他参与人的均衡策略
- sᵢ：参与人i的任意策略

**含义**：
- 没有参与人有动力单方面偏离均衡
- 均衡策略是最佳应对

### 9.2.2 纳什均衡的存在性

**纳什均衡的存在性（Existence of Nash Equilibrium）**：

**定理（Nash, 1950）**：
如果满足以下条件，则纳什均衡存在：
1. 参与人集合I是有限的
2. 策略空间Sᵢ是有限的
3. 或者策略空间Sᵢ是紧凸集，收益函数uᵢ是连续的、拟凹的

**证明思路**：
1. 构造最佳应对映射
2. 证明最佳应对映射是上半连续的、紧凸值的
3. 应用Kakutani不动点定理
4. 证明不动点是纳什均衡

### 9.2.3 纳什均衡的计算

**纳什均衡的计算（Computation of Nash Equilibrium）**：

1. **纯策略纳什均衡（Pure Strategy Nash Equilibrium）**：
   - 寻找策略组合，使得每个参与人的策略都是其他参与人策略的最佳应对
   - 方法：枚举所有策略组合，验证是否为纳什均衡

2. **混合策略纳什均衡（Mixed Strategy Nash Equilibrium）**：
   - 寻找概率分布，使得每个参与人的策略都是其他参与人策略的最佳应对
   - 方法：求解线性方程组

### 9.2.4 纳什均衡的例子

**纳什均衡的例子（Examples of Nash Equilibrium）**：

**1. 囚徒困境（Prisoner's Dilemma）**：
- 纳什均衡：（坦白，坦白）
- 帕累托最优：（沉默，沉默）
- 囚徒困境：纳什均衡不是帕累托最优

**2. 性别战（Battle of the Sexes）**：
- 纳什均衡：（歌剧，歌剧）和（电影，电影）
- 协调问题：两个纳什均衡

**3. 猜硬币（Matching Pennies）**：
- 纯策略纳什均衡：不存在
- 混合策略纳什均衡：（(0.5, 0.5), (0.5, 0.5)）

## 9.3 混合策略

### 9.3.1 混合策略的定义

**混合策略（Mixed Strategy）**：
参与人i的混合策略σᵢ是纯策略空间Sᵢ上的概率分布。

**符号**：
- σᵢ(sᵢ)：选择纯策略sᵢ的概率
- Σₛᵢ∈Sᵢ σᵢ(sᵢ) = 1
- σᵢ(sᵢ) ≥ 0，对于所有sᵢ ∈ Sᵢ

### 9.3.2 混合策略收益

**混合策略收益（Mixed Strategy Payoff）**：
参与人i的期望收益：
```
uᵢ(σ) = Σₛ∈S Πⱼ∈I σⱼ(sⱼ)·uᵢ(s)
```
**其中**：
- σ = (σ₁, ..., σₙ)：混合策略组合
- s = (s₁, ..., sₙ)：纯策略组合
- uᵢ(s)：参与人i在纯策略组合s下的收益

### 9.3.3 混合策略纳什均衡

**混合策略纳什均衡（Mixed Strategy Nash Equilibrium）**：
混合策略组合σ*是纳什均衡，如果对于所有i和所有σᵢ，有：
```
uᵢ(σ*ᵢ, σ*₋ᵢ) ≥ uᵢ(σᵢ, σ*₋ᵢ)
```
**其中**：
- σ*ᵢ：参与人i的均衡混合策略
- σ*₋ᵢ：其他参与人的均衡混合策略
- σᵢ：参与人i的任意混合策略

### 9.3.4 混合策略的性质

**混合策略的性质（Properties of Mixed Strategies）**：

1. **存在性（Existence）**：
   - 如果参与人集合I和策略空间Sᵢ是有限的，则混合策略纳什均衡存在

2. **无差异性质（Indifference Property）**：
   - 在混合策略纳什均衡中，参与人对所有被赋予正概率的纯策略无差异
   - 即uᵢ(sᵢ, σ*₋ᵢ) = uᵢ(s'ᵢ, σ*₋ᵢ)，对于所有σ*ᵢ(sᵢ) > 0和σ*ᵢ(s'ᵢ) > 0

3. **支持集性质（Support Property）**：
   - 在混合策略纳什均衡中，参与人只在最佳应对上赋予正概率
   - 即如果σ*ᵢ(sᵢ) > 0，则sᵢ是σ*₋ᵢ的最佳应对

## 9.4 重复剔除被占优策略

### 9.4.1 被占优策略的定义

**被占优策略（Dominated Strategy）**：
纯策略sᵢ被纯策略s'ᵢ占优，如果对于所有s₋ᵢ，有：
```
uᵢ(sᵢ, s₋ᵢ) ≤ uᵢ(s'ᵢ, s₋ᵢ)
```
且存在至少一个s₋ᵢ，使得：
```
uᵢ(sᵢ, s₋ᵢ) < uᵢ(s'ᵢ, s₋ᵢ)
```

**强被占优策略（Strictly Dominated Strategy）**：
纯策略sᵢ被纯策略s'ᵢ强占优，如果对于所有s₋ᵢ，有：
```
uᵢ(sᵢ, s₋ᵢ) < uᵢ(s'ᵢ, s₋ᵢ)
```

### 9.4.2 重复剔除被占优策略

**重复剔除被占优策略（Iterated Elimination of Dominated Strategies, IEDS）**：
1. 剔除所有被占优策略
2. 在剩余博弈中，剔除所有被占优策略
3. 重复步骤2，直到没有策略可以剔除

**结果**：
- 剩余策略组合是重复剔除被占优策略的解（IEDS解）

### 9.4.3 重复剔除被占优策略的性质

**重复剔除被占优策略的性质（Properties of IEDS）**：

1. **与纳什均衡的关系（Relation to Nash Equilibrium）**：
   - 如果策略组合s*是纳什均衡，则s*在重复剔除被占优策略后仍然存在
   - 即纳什均衡不会被剔除

2. **唯一性（Uniqueness）**：
   - 如果重复剔除被占优策略后有唯一解，则该解是唯一的纳什均衡

3. **一致性（Consistency）**：
   - 重复剔除被占优策略的顺序不影响最终结果（如果剔除顺序合理）

## 9.5 相关均衡

### 9.5.1 相关均衡的定义

**相关均衡（Correlated Equilibrium）**：
策略组合上的联合概率分布，使得给定信号后，每个参与人的策略都是最佳应对。

**符号**：
- σ：策略组合s上的联合概率分布
- σ(s) ≥ 0，对于所有s ∈ S
- Σₛ∈S σ(s) = 1

### 9.5.2 相关均衡的条件

**相关均衡的条件（Conditions for Correlated Equilibrium）**：
σ是相关均衡，如果对于所有i和所有sᵢ, s'ᵢ，有：
```
Σₛ₋ᵢ σ(sᵢ, s₋ᵢ)·uᵢ(sᵢ, s₋ᵢ) ≥ Σₛ₋ᵢ σ(sᵢ, s₋ᵢ)·uᵢ(s'ᵢ, s₋ᵢ)
```

**含义**：
- 给定信号sᵢ，参与人i选择sᵢ是最优的
- 没有参与人有动力单方面偏离均衡

### 9.5.3 相关均衡与纳什均衡的关系

**相关均衡与纳什均衡的关系（Relation between Correlated Equilibrium and Nash Equilibrium）**：

1. **纳什均衡是相关均衡（Nash Equilibrium is Correlated Equilibrium）**：
   - 任何纳什均衡都是相关均衡
   - 即如果s*是纳什均衡，则定义σ(s*) = 1，其他σ(s) = 0，则σ是相关均衡

2. **相关均衡集合包含纳什均衡集合（Set of Correlated Equilibria Contains Set of Nash Equilibria）**：
   - 相关均衡集合包含纳什均衡集合
   - 相关均衡可能不是纳什均衡

### 9.5.4 相关均衡的性质

**相关均衡的性质（Properties of Correlated Equilibrium）**：

1. **存在性（Existence）**：
   - 如果参与人集合I和策略空间Sᵢ是有限的，则相关均衡存在

2. **凸性（Convexity）**：
   - 相关均衡集合是凸集
   - 即如果σ₁和σ₂是相关均衡，则λσ₁ + (1-λ)σ₂也是相关均衡

3. **与纳什均衡的关系（Relation to Nash Equilibrium）**：
   - 相关均衡集合包含纳什均衡集合
   - 相关均衡的收益不低于纳什均衡的收益

## 9.6 本章总结

### 核心要点

1. **博弈的表示**：
   - 博弈：G = {I, (Sᵢ)ᵢ∈I, (uᵢ)ᵢ∈I}
   - 策略型博弈：参与人同时选择策略
   - 收益矩阵：表示双人博弈的收益

2. **纳什均衡**：
   - 纳什均衡：没有参与人有动力单方面偏离均衡
   - 纳什均衡的存在性：Nash定理
   - 纳什均衡的计算：纯策略、混合策略

3. **混合策略**：
   - 混合策略：纯策略空间上的概率分布
   - 混合策略收益：uᵢ(σ) = Σₛ∈S Πⱼ σⱼ(sⱼ)·uᵢ(s)
   - 混合策略纳什均衡：uᵢ(σ*ᵢ, σ*₋ᵢ) ≥ uᵢ(σᵢ, σ*₋ᵢ)
   - 混合策略的性质：存在性、无差异性质、支持集性质

4. **重复剔除被占优策略**：
   - 被占优策略：uᵢ(sᵢ, s₋ᵢ) ≤ uᵢ(s'ᵢ, s₋ᵢ)
   - 重复剔除被占优策略：迭代剔除被占优策略
   - 重复剔除被占优策略的性质：与纳什均衡的关系、唯一性、一致性

5. **相关均衡**：
   - 相关均衡：策略组合上的联合概率分布
   - 相关均衡的条件：Σₛ₋ᵢ σ(sᵢ, s₋ᵢ)·uᵢ(sᵢ, s₋ᵢ) ≥ Σₛ₋ᵢ σ(sᵢ, s₋ᵢ)·uᵢ(s'ᵢ, s₋ᵢ)
   - 相关均衡与纳什均衡的关系：纳什均衡是相关均衡
   - 相关均衡的性质：存在性、凸性
