## 6.1 成本函数的性质

### 6.1.1 成本函数的定义

**成本函数（Cost Function）**：
表示生产给定产量的最小成本的函数。

### 6.1.2 成本最小化问题

**成本最小化问题（Cost Minimization Problem）**：
给定产量，选择投入组合，使成本最小。

**数学表述**：
```
min w₁x₁ + w₂x₂
s.t. f(x₁, x₂) = y
```

**其中**：
- w₁, w₂：投入1和投入2的价格
- x₁, x₂：投入1和投入2的数量
- f(x₁, x₂)：生产函数
- y：产量

**拉格朗日函数**：
```
L = w₁x₁ + w₂x₂ + λ[y - f(x₁, x₂)]
```

**一阶条件**：
```
∂L/∂x₁ = w₁ - λMP₁ = 0
∂L/∂x₂ = w₂ - λMP₂ = 0
∂L/∂λ = y - f(x₁, x₂) = 0
```

**均衡条件**：
```
MP₁/w₁ = MP₂/w₂ = 1/λ
```

**其中**：
- MP₁, MP₂：投入1和投入2的边际产量
- w₁, w₂：投入1和投入2的价格
- λ：拉格朗日乘数（边际成本）

**含义**：
- 每种投入的边际产量与价格之比相等
- 每种投入的边际技术替代率等于价格之比
- MRTS₁₂ = MP₁/MP₂ = w₁/w₂

### 6.1.3 成本函数的性质

**成本函数的性质（Properties of Cost Function）**：

**1. 单调性（Monotonicity）**：
- 产量增加，成本增加
- ∂C/∂y > 0

**2. 凹性（Concavity）**：
- 成本函数是凹的
- ∂²C/∂y² ≥ 0

**3. 齐次性（Homogeneity）**：
- 成本函数是投入价格的一次齐次函数
- C(tw₁, tw₂, y) = tC(w₁, w₂, y)

**4. 谢泼德引理（Shephard's Lemma）**：
- 成本函数对投入价格的偏导数等于条件要素需求
- ∂C/∂wᵢ = xᵢ(w, y)

## 6.2 短期成本

### 6.2.1 短期成本的定义

**短期成本（Short-Run Cost）**：
在资本固定的情况下，生产给定产量的成本。

### 6.2.2 短期总成本

**短期总成本（Short-Run Total Cost, STC）**：
在资本固定的情况下，生产给定产量的总成本。

**公式**：
```
STC = wL + rK̅
```

**其中**：
- STC：短期总成本
- w：劳动价格
- L：劳动
- r：资本价格
- K̅：固定资本

### 6.2.3 短期固定成本

**短期固定成本（Short-Run Fixed Cost, SFC）**：
不随产量变化的成本。

**公式**：
```
SFC = rK̅
```

**其中**：
- SFC：短期固定成本
- r：资本价格
- K̅：固定资本

### 6.2.4 短期可变成本

**短期可变成本（Short-Run Variable Cost, SVC）**：
随产量变化的成本。

**公式**：
```
SVC = wL
```

**其中**：
- SVC：短期可变成本
- w：劳动价格
- L：劳动

### 6.2.5 短期平均成本

**短期平均成本（Short-Run Average Cost, SAC）**：
单位产量的短期成本。

**公式**：
```
SAC = STC/y = SFC/y + SVC/y = AFC + AVC
```

**其中**：
- SAC：短期平均成本
- STC：短期总成本
- y：产量
- SFC：短期固定成本
- SVC：短期可变成本
- AFC：平均固定成本
- AVC：平均可变成本

### 6.2.6 短期边际成本

**短期边际成本（Short-Run Marginal Cost, SMC）**：
增加一单位产量带来的成本变化。

**公式**：
```
SMC = ∂STC/∂y = ∂SVC/∂y
```

**其中**：
- SMC：短期边际成本
- STC：短期总成本
- SVC：短期可变成本
- y：产量

## 6.3 长期成本

### 6.3.1 长期成本的定义

**长期成本（Long-Run Cost）**：
在所有投入都可变的情况下，生产给定产量的最小成本。

### 6.3.2 长期总成本

**长期总成本（Long-Run Total Cost, LTC）**：
在所有投入都可变的情况下，生产给定产量的最小成本。

**公式**：
```
LTC = min_{K,L} [wL + rK]
s.t. f(K, L) = y
```

**其中**：
- LTC：长期总成本
- w：劳动价格
- L：劳动
- r：资本价格
- K：资本
- f(K, L)：生产函数
- y：产量

### 6.3.3 长期平均成本

**长期平均成本（Long-Run Average Cost, LAC）**：
单位产量的长期成本。

**公式**：
```
LAC = LTC/y
```

**其中**：
- LAC：长期平均成本
- LTC：长期总成本
- y：产量

### 6.3.4 长期边际成本

**长期边际成本（Long-Run Marginal Cost, LMC）**：
增加一单位产量带来的长期成本变化。

**公式**：
```
LMC = ∂LTC/∂y
```

**其中**：
- LMC：长期边际成本
- LTC：长期总成本
- y：产量

---

## 6.4 规模经济

### 6.4.1 规模经济的定义

**规模经济（Economies of Scale）**：
长期平均成本随产量增加而下降。

### 6.4.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
长期平均成本随产量增加而上升。

### 6.4.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
长期平均成本不随产量变化。

### 6.4.4 规模经济的度量

**规模经济的度量（Measure of Economies of Scale）**：

**成本弹性（Cost Elasticity）**：
```
Ec = (∂LTC/∂y) × (y/LTC) = LMC/LAC
```

**其中**：
- Ec：成本弹性
- LTC：长期总成本
- LMC：长期边际成本
- LAC：长期平均成本

**含义**：
- Ec < 1：规模经济
- Ec = 1：规模报酬不变
- Ec > 1：规模不经济

## 6.5 本章总结

### 核心要点

1. **成本函数的性质**：
   - 成本函数：表示生产给定产量的最小成本的函数
   - 成本最小化问题：min w₁x₁ + w₂x₂ s.t. f(x₁, x₂) = y
   - 均衡条件：MP₁/w₁ = MP₂/w₂ = 1/λ
   - 成本函数的性质：单调性、凹性、齐次性、谢泼德引理

2. **短期成本**：
   - 短期总成本：STC = wL + rK̅
   - 短期固定成本：SFC = rK̅
   - 短期可变成本：SVC = wL
   - 短期平均成本：SAC = STC/y = AFC + AVC
   - 短期边际成本：SMC = ∂STC/∂y = ∂SVC/∂y

3. **长期成本**：
   - 长期总成本：LTC = min_{K,L} [wL + rK] s.t. f(K, L) = y
   - 长期平均成本：LAC = LTC/y
   - 长期边际成本：LMC = ∂LTC/∂y

4. **规模经济**：
   - 规模经济：长期平均成本随产量增加而下降
   - 规模不经济：长期平均成本随产量增加而上升
   - 规模报酬不变：长期平均成本不随产量变化
   - 规模经济的度量：成本弹性Ec = LMC/LAC
