## 教材信息

- **书名**：微观经济学十八讲
- **作者**：平新乔
- **出版社**：北京大学出版社
- **类别**：中级微观经济学
- **章节**：18讲

## 内容简介

本书是北京大学平新乔教授编著的中级微观经济学教材，共18讲。本书涵盖了微观经济学的核心内容，包括消费者理论、生产者理论、市场结构、博弈论、信息经济学、一般均衡理论等。本书注重数学推导和经济学直觉的结合，适合经济学专业本科生和研究生学习。
