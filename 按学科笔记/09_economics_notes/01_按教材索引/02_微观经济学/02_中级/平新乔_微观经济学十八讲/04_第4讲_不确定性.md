# 第四讲：不确定性

> 本章学习目标：
> - 理解期望效用理论

## 4.1 期望效用理论

### 4.1.1 期望效用理论的定义

**期望效用理论（Expected Utility Theory）**：
在不确定性下，消费者根据期望效用来做出决策。

### 4.1.2 期望效用函数

**期望效用函数（Expected Utility Function）**：
表示不确定结果的期望效用。

**公式**：
```
EU = Σᵢ pᵢu(xᵢ)
```

**其中**：
- EU：期望效用
- pᵢ：状态i的概率
- u(xᵢ)：状态i的效用
- xᵢ：状态i的收益

**含义**：
- 期望效用是各状态效用的加权平均
- 权重是各状态的概率

### 4.1.3 期望值的效用

**期望值的效用（Utility of Expected Value）**：
期望值的效用。

**公式**：
```
u(E[x]) = u(Σᵢ pᵢxᵢ)
```

**其中**：
- E[x]：期望值
- pᵢ：状态i的概率
- xᵢ：状态i的收益

### 4.1.4 期望效用与期望值的效用

**期望效用与期望值的效用（Expected Utility vs. Utility of Expected Value）**：

**1. 风险中性（Risk Neutral）**：
- 期望效用等于期望值的效用
- EU = u(E[x])
- 效用函数是线性的

**2. 风险规避（Risk Averse）**：
- 期望效用小于期望值的效用
- EU < u(E[x])
- 效用函数是严格凹的

**3. 风险爱好（Risk Loving）**：
- 期望效用大于期望值的效用
- EU > u(E[x])
- 效用函数是严格凸的

## 4.2 风险态度

### 4.2.1 风险态度的定义

**风险态度（Risk Attitude）**：
消费者对风险的态度。

### 4.2.2 风险态度的类型

**风险态度的类型（Types of Risk Attitude）**：

**1. 风险中性（Risk Neutral）**：
- 对风险持中性态度
- 只关心期望值
- 效用函数是线性的

**2. 风险规避（Risk Averse）**：
- 不喜欢风险
- 愿意支付风险溢价
- 效用函数是严格凹的

**3. 风险爱好（Risk Loving）**：
- 喜欢风险
- 愿意承担风险
- 效用函数是严格凸的

### 4.2.3 风险规避的度量

**风险规避的度量（Measure of Risk Aversion）**：

**1. 阿罗-普拉特绝对风险规避系数（Arrow-Pratt Absolute Risk Aversion Coefficient）**：
```
A(x) = -u''(x)/u'(x)
```

**其中**：
- A(x)：绝对风险规避系数
- u'(x)：效用函数的一阶导数
- u''(x)：效用函数的二阶导数

**含义**：
- A(x) > 0：风险规避
- A(x) = 0：风险中性
- A(x) < 0：风险爱好

**2. 阿罗-普拉特相对风险规避系数（Arrow-Pratt Relative Risk Aversion Coefficient）**：
```
R(x) = -xu''(x)/u'(x)
```

**其中**：
- R(x)：相对风险规避系数
- x：财富
- u'(x)：效用函数的一阶导数
- u''(x)：效用函数的二阶导数

**含义**：
- R(x) > 0：风险规避
- R(x) = 0：风险中性
- R(x) < 0：风险爱好

## 4.3 风险溢价

### 4.3.1 风险溢价的定义

**风险溢价（Risk Premium）**：
风险规避者愿意支付的金额，以避免风险。

### 4.3.2 风险溢价的公式

**风险溢价的公式（Formula of Risk Premium）**：
```
π = E[x] - CE
```

**其中**：
- π：风险溢价
- E[x]：期望值
- CE：确定性等价

### 4.3.3 确定性等价

**确定性等价（Certainty Equivalent）**：
使消费者获得与不确定结果相同效用的确定性收益。

**公式**：
```
u(CE) = EU = Σᵢ pᵢu(xᵢ)
```

**其中**：
- CE：确定性等价
- EU：期望效用
- pᵢ：状态i的概率
- u(xᵢ)：状态i的效用
- xᵢ：状态i的收益

**含义**：
- 确定性等价是消费者愿意接受的确定性收益
- 风险溢价 = 期望值 - 确定性等价

## 4.4 保险

### 4.4.1 保险的定义

**保险（Insurance）**：
风险规避者通过支付保费，将风险转移给保险公司。

### 4.4.2 保险的基本原理

**保险的基本原理（Basic Principles of Insurance）**：

**1. 风险共担（Risk Sharing）**：
- 保险公司承担风险
- 消费者支付保费

**2. 大数定律（Law of Large Numbers）**：
- 保险公司通过大量保单分散风险
- 保险公司的赔付支出相对稳定

**3. 公平保费（Fair Premium）**：
- 公平保费等于期望赔付
- 公平保费 = p × L
- 其中：p是损失概率，L是损失金额

### 4.4.3 最优保险

**最优保险（Optimal Insurance）**：
消费者选择最优保险，使期望效用最大化。

**数学表述**：
```
max pU(W - L + x - πx) + (1-p)U(W - πx)
s.t. 0 ≤ x ≤ L
```

**其中**：
- p：损失概率
- U(·)：效用函数
- W：初始财富
- L：损失金额
- x：保险金额
- π：保费率

**最优解**：
- 如果保费率等于损失概率（π = p），消费者选择全额保险（x = L）
- 如果保费率大于损失概率（π > p），消费者选择部分保险（x < L）

### 4.4.4 道德风险

**道德风险（Moral Hazard）**：
投保后，消费者更加不谨慎，导致损失概率增加。

**解决方法**：
- 免赔额（Deductible）
- 共保（Co-payment）
- 限额（Limit）

## 4.5 本章总结

### 核心要点

1. **期望效用理论**：
   - 期望效用理论：在不确定性下，消费者根据期望效用来做出决策
   - 期望效用函数：EU = Σᵢ pᵢu(xᵢ)
   - 期望值的效用：u(E[x]) = u(Σᵢ pᵢxᵢ)
   - 风险中性：EU = u(E[x])，效用函数是线性的
   - 风险规避：EU < u(E[x])，效用函数是严格凹的
   - 风险爱好：EU > u(E[x])，效用函数是严格凸的

2. **风险态度**：
   - 风险态度：消费者对风险的态度
   - 风险态度的类型：风险中性、风险规避、风险爱好
   - 阿罗-普拉特绝对风险规避系数：A(x) = -u''(x)/u'(x)
   - 阿罗-普拉特相对风险规避系数：R(x) = -xu''(x)/u'(x)

3. **风险溢价**：
   - 风险溢价：风险规避者愿意支付的金额，以避免风险
   - 风险溢价的公式：π = E[x] - CE
   - 确定性等价：使消费者获得与不确定结果相同效用的确定性收益
   - 确定性等价的公式：u(CE) = EU = Σᵢ pᵢu(xᵢ)

4. **保险**：
   - 保险：风险规避者通过支付保费，将风险转移给保险公司
   - 保险的基本原理：风险共担、大数定律、公平保费
   - 最优保险：消费者选择最优保险，使期望效用最大化
   - 道德风险：投保后，消费者更加不谨慎，导致损失概率增加
