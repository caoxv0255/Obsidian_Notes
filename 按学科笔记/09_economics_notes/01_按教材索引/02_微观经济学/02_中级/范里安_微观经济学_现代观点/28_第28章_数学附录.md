## 28.1 数学基础

### 28.1.1 函数

**函数（Function）**：
将一个集合的元素映射到另一个集合的元素。

**表示方法**：
```
y = f(x)
```

**其中**：
- x：自变量
- y：因变量
- f(·)：函数关系

### 28.1.2 导数

**导数（Derivative）**：
函数在某一点的变化率。

**公式**：
```
f'(x) = lim(Δx→0) [f(x + Δx) - f(x)]/Δx
```

**含义**：
- 导数表示函数的斜率
- 导数表示函数的变化率

### 28.1.3 偏导数

**偏导数（Partial Derivative）**：
多元函数对其中一个变量的导数。

**公式**：
```
∂f/∂x = lim(Δx→0) [f(x + Δx, y) - f(x, y)]/Δx
```

**其中**：
- ∂f/∂x：函数f对x的偏导数
- x：自变量
- y：其他自变量

**含义**：
- 偏导数表示函数对其中一个变量的变化率
- 其他变量保持不变

## 28.2 最优化

### 28.2.1 无约束最优化

**无约束最优化（Unconstrained Optimization）**：
没有约束条件的最优化问题。

**问题**：
```
max f(x)
```

**一阶条件（First-Order Condition, FOC）**：
```
f'(x) = 0
```

**二阶条件（Second-Order Condition, SOC）**：
```
f''(x) < 0
```

**含义**：
- 一阶条件：导数等于零
- 二阶条件：二阶导数小于零（最大值）

### 28.2.2 有约束最优化

**有约束最优化（Constrained Optimization）**：
有约束条件的最优化问题。

**问题**：
```
max f(x)
s.t. g(x) = 0
```

**拉格朗日函数（Lagrangian Function）**：
```
L = f(x) + λg(x)
```

**其中**：
- L：拉格朗日函数
- f(x)：目标函数
- g(x)：约束条件
- λ：拉格朗日乘数

**一阶条件**：
```
∂L/∂x = f'(x) + λg'(x) = 0
```
```
∂L/∂λ = g(x) = 0
```

**含义**：
- 拉格朗日乘数表示约束条件的影子价格
- 拉格朗日乘数表示约束条件对目标函数的边际影响

## 28.3 积分

### 28.3.1 定积分

**定积分（Definite Integral）：
函数在某个区间内的面积。

**公式**：
```
∫ₐᵇ f(x) dx
```

**其中**：
- a：积分下限
- b：积分上限
- f(x)：被积函数
- dx：积分变量

**含义**：
- 定积分表示函数在区间[a, b]内的面积
- 定积分表示函数在区间[a, b]内的累积

### 28.3.2 不定积分

**不定积分（Indefinite Integral）**：
函数的原函数。

**公式**：
```
∫ f(x) dx = F(x) + C
```

**其中**：
- F(x)：原函数
- C：常数

**含义**：
- 不定积分表示函数的原函数
- 不定积分表示函数的反导数

## 28.4 概率

### 28.4.1 概率的定义

**概率（Probability）**：
事件发生的可能性。

**性质**：
- 0 ≤ P ≤ 1
- P(Ω) = 1（全集的概率为1）
- P(∅) = 0（空集的概率为0）

### 28.4.2 期望值

**期望值（Expected Value）**：
随机变量的平均值。

**公式**：
```
E[X] = Σₜ pₜ xₜ
```

**其中**：
- E[X]：期望值
- pₜ：结果t的概率
- xₜ：结果t的值

**含义**：
- 期望值表示随机变量的平均值
- 期望值表示随机变量的长期平均值

### 28.4.3 方差

**方差（Variance）**：
随机变量的离散程度。

**公式**：
```
Var[X] = E[(X - E[X])²] = Σₜ pₜ (xₜ - E[X])²
```

**其中**：
- Var[X]：方差
- E[X]：期望值
- pₜ：结果t的概率
- xₜ：结果t的值

**含义**：
- 方差表示随机变量的离散程度
- 方差表示随机变量的波动性

## 28.5 矩阵

### 28.5.1 矩阵的定义

**矩阵（Matrix）**：
按行和列排列的数字集合。

**表示方法**：
```
A = [a₁₁ a₁₂
     a₂₁ a₂₂]
```

**其中**：
- aᵢⱼ：矩阵第i行第j列的元素

### 28.5.2 矩阵的运算

**矩阵加法（Matrix Addition）**：
```
A + B = [a₁₁ + b₁₁ a₁₂ + b₁₂
        a₂₁ + b₂₁ a₂₂ + b₂₂]
```

**矩阵乘法（Matrix Multiplication）**：
```
A × B = [a₁₁b₁₁ + a₁₂b₂₁ a₁₁b₁₂ + a₁₂b₂₂
         a₂₁b₁₁ + a₂₂b₂₁ a₂₁b₁₂ + a₂₂b₂₂]
```

**矩阵转置（Matrix Transpose）**：
```
A^T = [a₁₁ a₂₁
       a₁₂ a₂₂]
```

**其中**：
- A^T：矩阵A的转置

## 28.6 数学方法在经济学中的应用

### 28.6.1 效用最大化

**效用最大化（Utility Maximization）**：
```
max u(x₁, x₂)
s.t. p₁x₁ + p₂x₂ = m
```

**拉格朗日函数**：
```
L = u(x₁, x₂) + λ(m - p₁x₁ - p₂x₂)
```

**一阶条件**：
```
∂L/∂x₁ = MU₁ - λp₁ = 0
```
```
∂L/∂x₂ = MU₂ - λp₂ = 0
```
```
∂L/∂λ = m - p₁x₁ - p₂x₂ = 0
```

**求解**：
```
MU₁/MU₂ = p₁/p₂
```

### 28.6.2 利润最大化

**利润最大化（Profit Maximization）**：
```
max π = p × y - C(y)
```

**一阶条件**：
```
dπ/dy = p - C'(y) = 0
```

**求解**：
```
p = MC(y)
```

### 28.6.3 成本最小化

**成本最小化（Cost Minimization）**：
```
min C = w₁x₁ + w₂x₂
s.t. f(x₁, x₂) = y
```

**拉格朗日函数**：
```
L = w₁x₁ + w₂x₂ + λ(y - f(x₁, x₂))
```

**一阶条件**：
```
∂L/∂x₁ = w₁ - λ∂f/∂x₁ = 0
```
```
∂L/∂x₂ = w₂ - λ∂f/∂x₂ = 0
```

**求解**：
```
w₁/∂f/∂x₁ = w₂/∂f/∂x₂
```

## 28.7 本章总结

### 核心要点

1. **数学基础**：
   - 函数：将一个集合的元素映射到另一个集合的元素
   - 导数：函数在某一点的变化率
   - 偏导数：多元函数对其中一个变量的导数

2. **最优化**：
   - 无约束最优化：没有约束条件的最优化问题
   - 有约束最优化：有约束条件的最优化问题
   - 拉格朗日函数：L = f(x) + λg(x)

3. **积分**：
   - 定积分：函数在某个区间内的面积
   - 不定积分：函数的原函数

4. **概率**：
   - 概率：事件发生的可能性
   - 期望值：随机变量的平均值
   - 方差：随机变量的离散程度

5. **矩阵**：
   - 矩阵：按行和列排列的数字集合
   - 矩阵运算：矩阵加法、矩阵乘法、矩阵转置

6. **数学方法在经济学中的应用**：
   - 效用最大化：MU₁/MU₂ = p₁/p₂
   - 利润最大化：p = MC(y)
   - 成本最小化：w₁/∂f/∂x₁ = w₂/∂f/∂x₂
