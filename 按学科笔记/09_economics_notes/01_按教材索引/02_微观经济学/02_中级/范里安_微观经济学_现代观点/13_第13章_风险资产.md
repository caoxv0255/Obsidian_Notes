# 第13章：风险资产

> 本章学习目标：
> - 理解风险资产的定义
> - 掌握风险资产组合的选择
> - 学会分析风险分散化

---

## 13.1 风险资产的定义

### 13.1.1 风险资产的定义

**风险资产（Risky Asset）**：
收益不确定的资产。

### 13.1.2 风险资产的收益

**风险资产的收益（Return of Risky Asset）**：
```
R = (P₁ - P₀ + D)/P₀
```

**其中**：
- R：收益
- P₀：期初价格
- P₁：期末价格
- D：股息或利息

**期望收益**：
```
E[R] = Σₜ pₜ Rₜ
```

**方差**：
```
Var[R] = Σₜ pₜ(Rₜ - E[R])²
```

---

## 13.2 风险资产组合

### 13.2.1 风险资产组合的定义

**风险资产组合（Portfolio of Risky Assets）**：
多种风险资产的组合。

### 13.2.2 风险资产组合的收益

**风险资产组合的收益（Return of Portfolio）**：
```
Rₚ = Σᵢ wᵢ Rᵢ
```

**其中**：
- Rₚ：组合收益
- wᵢ：资产i的权重
- Rᵢ：资产i的收益

**期望收益**：
```
E[Rₚ] = Σᵢ wᵢ E[Rᵢ]
```

**方差**：
```
Var[Rₚ] = Σᵢ Σⱼ wᵢ wⱼ Cov(Rᵢ, Rⱼ)
```

**其中**：
- Cov(Rᵢ, Rⱼ)：资产i和资产j收益的协方差

---

## 13.3 风险分散化

### 13.3.1 风险分散化的定义

**风险分散化（Diversification）**：
通过投资多种资产降低风险。

### 13.3.2 风险分散化的原理

**风险分散化的原理（Principle of Diversification）**：

**1. 非系统性风险（Unsystematic Risk）**：
- 特定资产的风险
- 可以通过分散化消除

**2. 系统性风险（Systematic Risk）**：
- 市场整体的风险
- 不能通过分散化消除

### 13.3.3 风险分散化的效果

**风险分散化的效果（Effect of Diversification）**：

**两种资产组合的方差**：
```
Var[Rₚ] = w₁² Var[R₁] + w₂² Var[R₂] + 2w₁w₂ Cov(R₁, R₂)
```

**如果资产收益不相关（Cov(R₁, R₂) = 0）**：
```
Var[Rₚ] = w₁² Var[R₁] + w₂² Var[R₂]
```

**如果等权重投资（w₁ = w₂ = 0.5）**：
```
Var[Rₚ] = 0.25 Var[R₁] + 0.25 Var[R₂] = 0.5 Var[R₁]
```

**含义**：
- 分散化可以降低组合风险
- 资产数量越多，风险越低

**有效前沿（Efficient Frontier）**：

## 13.5 最优资产组合选择

### 13.5.1 最优资产组合选择问题

**最优资产组合选择问题（Optimal Portfolio Choice Problem）**：
在期望收益和风险之间权衡，选择最优资产组合。

**数学表述**：
```
max U(E[Rₚ], Var[Rₚ])
s.t. Σᵢ wᵢ = 1
```

**其中**：
- U(·, ·)：效用函数
- E[Rₚ]：组合期望收益
- Var[Rₚ]：组合方差
- wᵢ：资产i的权重

### 13.5.2 最优资产组合的条件

**最优资产组合的条件（Condition of Optimal Portfolio）**：
无差异曲线与有效前沿相切。

## 13.6 分离定理

### 13.6.1 分离定理的定义

**分离定理（Separation Theorem）**：
最优资产组合可以分为两部分：风险资产组合和无风险资产。

### 13.6.2 分离定理的内容

**分离定理的内容（Content of Separation Theorem）**：

**1. 所有投资者持有相同的风险资产组合（市场组合）**：
- 风险资产组合的选择与投资者的风险偏好无关
- 所有投资者持有市场组合

**2. 投资者根据风险偏好在无风险资产和市场组合之间分配资金**：
- 风险规避的投资者：投资更多无风险资产
- 风险偏好的投资者：投资更多市场组合

## 13.7 资本资产定价模型（再次）

### 13.7.1 资本资产定价模型（再次）

**资本资产定价模型（Capital Asset Pricing Model, CAPM）**：
描述风险资产期望收益与市场风险之间关系的模型。

**公式**：
```
E[Rᵢ] = r + βᵢ(E[Rₘ] - r)
```

**其中**：
- E[Rᵢ]：资产i的期望收益
- r：无风险利率
- βᵢ：资产i的贝塔系数
- E[Rₘ]：市场组合的期望收益

### 13.7.2 证券市场线

**证券市场线（Security Market Line, SML）**：
表示贝塔系数和期望收益之间关系的直线。
