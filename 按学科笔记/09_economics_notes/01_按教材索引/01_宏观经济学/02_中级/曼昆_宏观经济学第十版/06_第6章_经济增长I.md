# 第6章 经济增长I：资本积累与人口增长

## 章节学习目标

1. **理解经济增长的事实**：掌握世界各国经济增长的差异和特征
2. **掌握索洛增长模型**：理解资本积累、人口增长对稳态的影响
3. **理解稳态的性质**：掌握稳态的决定因素和收敛性质
4. **理解储蓄率的影响**：掌握储蓄率对稳态收入和增长的影响
5. **理解人口增长的影响**：掌握人口增长对稳态收入和增长的影响

## 6.1 世界各国的经济增长

### 6.1.1 经济增长的事实

**事实1：各国收入水平差异巨大**

**数据（2020年人均GDP，PPP调整，美元）：**

| 国家 | 人均GDP | 相对美国 |
|------|--------|---------|
| 美国 | 63,500 | 100% |
| 日本 | 41,700 | 66% |
| 德国 | 53,800 | 85% |
| 中国 | 17,200 | 27% |
| 印度 | 6,300 | 10% |
| 刚果民主共和国 | 1,300 | 2% |

**含义：**
- 最富国家的人均收入是最穷国家的50倍
- 收入差距巨大且持续存在

**事实2：经济增长率差异巨大**

**数据（1960-2020年年均增长率）：**

| 国家 | 年均增长率 | 60年倍数 |
|------|-----------|---------|
| 韩国 | 5.8% | 33倍 |
| 中国 | 5.5% | 29倍 |
| 日本 | 3.5% | 8倍 |
| 美国 | 2.0% | 3.3倍 |
| 尼日利亚 | 0.7% | 1.5倍 |
| 尼日尔 | 0.1% | 1.06倍 |

**含义：**
- 持续的小增长率差异累积成巨大差异
- 韩国60年间收入增长33倍
- 尼日尔几乎没有增长

**70法则（Rule of 70）：**

```
收入翻倍时间 ≈ 70 / 增长率（%）
```

**例子：**
- 增长率2%：翻倍时间 ≈ 35年
- 增长率5%：翻倍时间 ≈ 14年
- 增长率10%：翻倍时间 ≈ 7年

**事实3：经济增长率随时间变化**

**历史数据：**

| 时期 | 世界平均增长率 |
|------|---------------|
| 1-1000年 | 0.0% |
| 1000-1500年 | 0.1% |
| 1500-1820年 | 0.3% |
| 1820-1950年 | 0.9% |
| 1950-2000年 | 2.1% |

**含义：**
- 经济增长是现代现象
- 工业革命后才开始持续增长
- 不同时期增长速度不同

**事实4：收敛性**

**σ-收敛（σ-convergence）：**
- 收入差距随时间缩小
- 发展中国家增长更快

**β-收敛（β-convergence）：**
- 贫穷国家增长更快
- 向发达国家追赶

**但有条件：**
- 绝对收敛不完全成立
- 需要控制其他因素（储蓄率、人口增长、制度）

### 6.1.2 增长的决定因素

**生产函数：**
```
Y = F(K, L, H, N, A)
```

其中：
- K = 实物资本
- L = 劳动
- H = 人力资本
- N = 自然资源
- A = 技术（全要素生产率）

**人均形式：**
```
y = f(k, h, n, a)
```

**增长来源：**

**1. 资本积累**
- 实物资本投资
- 基础设施
- 机器设备

**2. 人力资本**
- 教育
- 健康
- 培训

**3. 技术进步**
- 创新
- 研发
- 技术扩散

**4. 制度质量**
- 产权保护
- 法治
- 政府效率

**5. 资源配置**
- 市场机制
- 对外开放
- 贸易

### 6.1.3 案例研究：东亚奇迹

**东亚新兴经济体：**
- 日本、韩国、台湾、香港、新加坡
- 中国、马来西亚、泰国、印度尼西亚

**增长特征：**

**1. 高增长率**
- 1960-2000年年均增长率5-8%
- 远超世界平均

**2. 持续时间长**
- 30-40年持续增长
- 避免了中等收入陷阱

**3. 结构转变**
- 农业 → 制造业 → 服务业
- 出口导向工业化

**成功因素：**

**1. 高储蓄率**
- 储蓄率30-40%
- 资本积累快

**2. 教育投资**
- 普及基础教育
- 高等教育普及
- 技能培训

**3. 技术吸收**
- 引进外国技术
- 学习和模仿
- 技术升级

**4. 出口导向**
- 利用全球市场
- 参与国际竞争
- 获取规模经济

**5. 政府作用**
- 产业政策
- 基础设施投资
- 教育投资

**启示：**
- 高储蓄和投资是增长的关键
- 教育和人力资本重要
- 对外开放有益
- 政府可以发挥作用

## 6.2 索洛增长模型

### 6.2.1 模型假设

**索洛模型（Solow Growth Model）**是描述长期经济增长的标准模型。

**生产函数：**
```
Y = F(K, L)
```

**假设：**

**1. 规模报酬不变**
```
F(zK, zL) = zF(K, L)
```

**2. 边际产量递减**
```
∂²F/∂K² < 0, ∂²F/∂L² < 0
```

**3. 柯布-道格拉斯形式**
```
Y = K^α L^(1-α)
```

**4. 外生增长率**
- 人口增长率：n
- 技术增长率：g（本章暂不考虑，g=0）

### 6.2.2 人均生产函数

**人均形式：**
```
y = Y/L = F(K/L, 1) = f(k)
```

其中：
- y = 人均产出
- k = 资本劳动比（K/L）

**柯布-道格拉斯人均形式：**
```
y = k^α
```

**图形分析：人均生产函数**

```
人均产出 y
  ↑
  │
  │     /
  │    /
  │   /  y = f(k)
  │  /
  │ /
  └────────→ 人均资本 k

形状：边际产量递减，f'(k) > 0, f''(k) < 0
```

**边际产量：**
```
MPK = f'(k) = αk^(α-1)
MPL = f(k) - kf'(k)
```

### 6.2.3 资本积累

**资本存量变化：**
```
ΔK = I - δK
```

其中：
- I = 投资
- δ = 折旧率

**投资函数：**
```
I = sY
```

其中：
- s = 储蓄率
- 0 < s < 1

**人均资本变化：**

**推导：**
```
k = K/L
Δk = ΔK/L - k×ΔL/L
    = (sY - δK)/L - k×n
    = sy - δk - nk
    = sf(k) - (δ + n)k
```

**基本方程：**
```
Δk = sf(k) - (δ + n)k
```

**含义：**
- 人均资本变化 = 人均投资 - (折旧 + 人口增长)对资本的稀释
- sf(k) = 实际投资
- (δ + n)k = 平衡投资（维持k不变所需的投资）

### 6.2.4 稳态

**稳态（Steady State）**是指人均资本不再变化的状态。

**条件：**
```
Δk = 0
sf(k*) = (δ + n)k*
```

其中：
- k* = 稳态人均资本

**图形分析：稳态**

```
人均投资
  ↑
  │    sf(k)
  │   /
  │  /      * 稳态
  │ /      /
  │/      /
  └──────/(δ + n)k────→ 人均资本 k
        k*

稳态：sf(k*) = (δ + n)k*
```

**稳态的性质：**

**1. 稳定：**
- 如果 k < k*：sf(k) > (δ + n)k → k增加
- 如果 k > k*：sf(k) < (δ + n)k → k减少
- 最终收敛到k*

**2. 唯一：**
- 在给定参数下，只有一个稳态

**3. 增长率：**

**稳态增长率：**
- 人均资本增长率：0%
- 人均产出增长率：0%
- 总资本增长率：n%
- 总产出增长率：n%

**含义：**
- 稳态时，人均经济不增长
- 总经济按人口增长率n增长

**柯布-道格拉斯的稳态：**

**条件：**
```
s(k*)^α = (δ + n)k*
(k*)^(α-1) = (δ + n)/s
k* = [(s)/(δ + n)]^[1/(1-α)]
```

**稳态人均产出：**
```
y* = (k*)^α
    = [(s)/(δ + n)]^[α/(1-α)]
```

### 6.2.5 收敛到稳态

**收敛机制：**

**如果初始k < k*：**
```
sf(k) > (δ + n)k → k增加 → k趋向k*
```

**如果初始k > k*：**
```
sf(k) < (δ + n)k → k减少 → k趋向k*
```

**收敛速度：**

**近似公式：**
```
dy/dt ≈ -λ(y - y*)
```

其中：
- λ = 收敛速度
- λ ≈ (1 - α)(δ + n)

**含义：**
- 经济趋向稳态的速度取决于α、δ、n
- 典型收敛速度：每年2-3%

**图形分析：收敛路径**

```
人均资本 k
  ↑
  │ k*
  │   ↘
  │    ↘
  │     ↘
  │      ↘
  └───────→ 时间 t

从k₀收敛到k*
```

### 6.2.6 案例研究：德国和日本的战后增长

**背景：**
二战后，德国和日本经济被摧毁。

**特征：**

**1. 初始资本低**
- 资本存量被战争摧毁
- k远低于稳态k*

**2. 增长速度快**
- 1950-1970年代增长率很高
- 德国：经济奇迹
- 日本：经济奇迹

**3. 收敛到稳态**
- 1970年代后增长放缓
- 趋向美国水平

**索洛模型解释：**

**1. 资本积累**
- 从低k开始
- 高储蓄率
- 资本快速增长

**2. 边际产量高**
- k低时，f'(k)高
- 资本回报率高
- 激励投资

**3. 收敛效应**
- 贫穷国家增长更快
- 追赶发达国家

**启示：**
- 战争摧毁资本后，恢复很快
- 收敛是经济增长的重要特征
- 政策可以加速收敛

## 6.3 储蓄率的影响

### 6.3.1 储蓄率与稳态

**储蓄率s增加的影响：**

**图形分析：**

```
人均投资
  ↑
  │    sf₁(k)
  │   /
  │  /      * k₁*
  │ /      /
  │/      /
  └──────/(δ + n)k────→ k
        k₁*

储蓄率增加到s₂：
  │    sf₂(k)
  │   /
  │  /      * k₂*
  │ /      /
  │/      /
  └──────/(δ + n)k────→ k
        k₂* > k₁*
```

**影响：**

**1. 稳态资本：**
- k*增加
- 储蓄率越高，k*越高

**2. 稳态产出：**
- y*增加
- 储蓄率越高，y*越高

**3. 稳态消费：**
- c* = (1 - s)y*
- 不一定增加
- 存在最优储蓄率

**柯布-道格拉斯：**
```
k* = [(s)/(δ + n)]^[1/(1-α)]
y* = [(s)/(δ + n)]^[α/(1-α)]
```

**影响的大小：**

**对k*的弹性：**
```
∂ln(k*)/∂ln(s) = 1/(1-α)
```

如果α = 1/3：
```
∂ln(k*)/∂ln(s) = 1/(1-1/3) = 1.5
```

**含义：**
- 储蓄率增加1%，k*增加1.5%
- 储蓄率的影响较大

### 6.3.2 黄金律

**黄金律（Golden Rule）**是指使稳态人均消费最大化的储蓄率。

**稳态消费：**
```
c* = y* - i*
    = f(k*) - sf(k*)
    = f(k*) - (δ + n)k*
```

**最大化c*：**
```
dc*/dk* = f'(k*) - (δ + n) = 0
f'(k*) = δ + n
```

**黄金律资本：**
```
k_g: f'(k_g) = δ + n
```

**黄金律储蓄率：**
```
s_g = f(k_g)/k_g = (δ + n)/f'(k_g) = (δ + n)/(δ + n) = 1?
不对，重新计算：
s_g = f(k_g)/k_g
```

**柯布-道格拉斯的黄金律：**

**条件：**
```
f'(k_g) = α(k_g)^(α-1) = δ + n
k_g = [α/(δ + n)]^[1/(1-α)]
```

**黄金律储蓄率：**
```
s_g = f(k_g)/k_g = (k_g)^α/k_g = (k_g)^(α-1)
     = [α/(δ + n)]^[(α-1)/(1-α)]
     = [α/(δ + n)]^[-1]
     = (δ + n)/α
```

**含义：**
- 如果α = 1/3，δ + n = 0.05：
  - s_g = 0.05/(1/3) = 0.15
  - 黄金律储蓄率为15%

**图形分析：黄金律**

```
人均产出/投资
  ↑
  │    f(k)
  │   /
  │  /      * 黄金律点
  │ /      /
  │/      /
  └──────/(δ + n)k────→ k
        k_g

f'(k_g) = (δ + n)
```

**黄金律的经济学含义：**

**1. 资本边际产量等于折旧加人口增长**
- 在这一点，消费最大化
- 资本回报正好补偿资本损耗和人口增长

**2. 最优储蓄率**
- 储蓄率太低：k*太低，消费低
- 储蓄率太高：k*太高，消费低（太多投资）
- 黄金律储蓄率使消费最大化

### 6.3.3 储蓄率与增长率

**过渡期增长率：**

**储蓄率增加后：**
- 人均资本k从k₁*增加到k₂*
- 过渡期，人均增长率 > 0

**收敛后：**
- 达到新稳态k₂*
- 人均增长率 = 0

**图形分析：增长路径**

```
人均产出 y
  ↑
  │        y₂*
  │       /
  │      /
  │     /
  │    /
  │   /
  │  y₁*
  │  ────────
  └────────→ 时间 t
      储蓄率增加

过渡期：增长率为正
稳态：增长率为零（人均）
```

**含义：**
- 储蓄率增加只能暂时提高增长率
- 长期增长率由技术进步决定（本章不考虑）
- 索洛模型预测：资本积累不能带来长期增长

### 6.3.4 案例研究：高储蓄国家的增长

**东亚国家：**

**储蓄率：**
- 韩国：30-40%
- 新加坡：40-50%
- 中国：40-50%

**增长率：**
- 1960-2000年：5-8%
- 远高于世界平均

**索洛模型解释：**

**1. 高储蓄率**
- 高k*
- 高y*

**2. 资本积累**
- 从低k开始
- 高增长（收敛效应）
- 赶上高k*

**3. 长期影响**
- 高储蓄率提高稳态收入
- 但不能维持高增长率

**启示：**
- 高储蓄率有助于提高收入水平
- 但长期增长需要技术进步
- 中国需要从投资驱动转向创新驱动

## 6.4 人口增长的影响

### 6.4.1 人口增长与稳态

**人口增长率n增加的影响：**

**图形分析：**

```
人均投资
  ↑
  │    sf(k)
  │   /
  │  /      * k₁*
  │ /      /
  │/      /
  └──────/(δ + n₁)k────→ k
        k₁*

人口增长率增加到n₂：
  │    sf(k)
  │   /
  │  /      * k₂*
  │ /      /
  │/      /
  └──────/(δ + n₂)k────→ k
        k₂* < k₁*
```

**影响：**

**1. 稳态资本：**
- k*减少
- 人口增长越快，k*越低

**2. 稳态产出：**
- y*减少
- 人口增长越快，y*越低

**3. 稳态消费：**
- c* = (1 - s)y*
- 减少

**柯布-道格拉斯：**
```
k* = [(s)/(δ + n)]^[1/(1-α)]
y* = [(s)/(δ + n)]^[α/(1-α)]
```

**影响的大小：**

**对k*的弹性：**
```
∂ln(k*)/∂ln(δ + n) = -1/(1-α)
```

如果α = 1/3：
```
∂ln(k*)/∂ln(δ + n) = -1.5
```

**含义：**
- 人口增长率增加1%，k*减少1.5%
- 人口增长的影响较大

### 6.4.2 人口增长与总产出

**总产出增长率：**

**稳态时：**
- 人均产出增长率：0%
- 人口增长率：n%
- 总产出增长率：n%

**含义：**
- 人口增长带来总产出增长
- 但不带来人均产出增长

**人均收入：**
- 不随人口增长而增加
- 反而可能减少（资本稀释）

**图形分析：人均产出与人口增长**

```
人均产出 y
  ↑
  │
  │        /  n₁
  │       /
  │      /
  │     /
  │    /   n₂ > n₁
  │   /
  │  /
  └────────→ 人口增长 n

人口增长越高，人均产出越低
```

### 6.4.3 马尔萨斯陷阱

**马尔萨斯（Malthus）的预测：**

**理论：**
- 人口增长导致人均收入下降到生存水平
- 贫困陷阱

**机制：**
1. 人均收入增加
2. 人口增加
3. 资本稀释
4. 人均收入下降
5. 回到生存水平

**历史现实：**
- 工业革命后，马尔萨斯预测失效
- 人口增长但人均收入持续增长
- 技术进步抵消了人口增长

**现代观点：**
- 人口增长不是贫困的原因
- 技术进步可以抵消人口增长
- 人口结构比人口总量更重要

### 6.4.4 案例研究：人口增长与经济增长

**数据：**

| 国家 | 人口增长率（%） | 人均GDP增长率（%） |
|------|---------------|------------------|
| 日本 | 0.1 | 1.5 |
| 中国 | 0.5 | 6.0 |
| 印度 | 1.0 | 5.0 |
| 尼日利亚 | 2.5 | 0.5 |

**观察：**
- 人口增长与人均增长负相关
- 但不是因果关系
- 其他因素更重要

**复杂关系：**

**人口增长的积极影响：**
- 劳动力增加
- 创新潜力增加
- 市场规模扩大

**人口增长的消极影响：**
- 资本稀释
- 人均资源减少
- 教育、医疗压力

**最优人口增长：**
- 需要平衡积极和消极影响
- 考虑人口结构（年龄结构）

## 本章总结

### 核心要点

1. **经济增长的事实**
   - 各国收入差距巨大
   - 增长率差异巨大
   - 经济增长是现代现象
   - 收敛性存在但有条件

2. **索洛模型**
   - 生产函数：y = f(k)
   - 资本积累：Δk = sf(k) - (δ + n)k
   - 稳态：sf(k*) = (δ + n)k*

3. **储蓄率的影响**
   - 提高稳态收入
   - 只能暂时提高增长率
   - 存在黄金律储蓄率

4. **人口增长的影响**
   - 降低稳态收入
   - 增加总产出增长
   - 人均收入不增长

5. **收敛**
   - 经济趋向稳态
   - 收敛速度约2-3%/年
   - 贫穷国家增长更快
