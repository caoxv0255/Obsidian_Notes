# 宏观经济学 - 多恩布什

## 教材信息

- **书名**：宏观经济学（第10版）
- **作者**：鲁迪格·多恩布什
- **出版社**：中国人民大学出版社
- **类别**：中级宏观经济学
- **章节**：12章

## 内容简介

本书是国际公认的经典宏观经济学教材，共12章。本书涵盖了宏观经济学的核心内容，包括国民收入核算、经济增长、总供给与总需求、消费与投资、货币需求、货币政策、国际收支与汇率等。本书注重理论与实践的结合，适合经济学专业本科生和研究生学习。
