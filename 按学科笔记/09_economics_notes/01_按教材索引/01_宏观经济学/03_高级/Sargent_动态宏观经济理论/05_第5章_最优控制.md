# 第5章：最优控制

## 本章学习目标

通过本章的学习，学生应该能够：
1. 理解最优控制问题的基本设定和经济学意义
2. 掌握庞特里亚金最大值原理的推导和应用
3. 熟悉连续时间和离散时间最优控制问题的求解方法
4. 应用最优控制理论分析投资决策、消费选择等经济问题
5. 理解最优控制与动态规划的关系和区别

## 5.1 最优控制问题基础

### 5.1.1 问题设定

**定义：连续时间最优控制问题**

一个标准的连续时间最优控制问题可以表示为：

$$\max_{u(t)} \int_{t_0}^{t_f} e^{-\rho t} F(x(t), u(t), t) dt + \Phi(x(t_f), t_f)$$

$$s.t. \quad \dot{x}(t) = f(x(t), u(t), t)$$

$$x(t_0) = x_0 \text{ 给定}$$

$$u(t) \in U \text{（控制约束集）}$$

其中：
- $x(t)$ 是状态变量（$n$ 维）
- $u(t)$ 是控制变量（$m$ 维）
- $F(x, u, t)$ 是即时收益函数
- $\Phi(x, t)$ 是终值函数
- $\rho$ 是贴现率
- $\dot{x}(t) = f(x, u, t)$ 是状态方程（运动方程）

**经济学解释**
- **状态变量**：描述经济系统状态的存量（如资本、财富）
- **控制变量**：决策者可以控制的流量（如消费、投资）
- **目标函数**：跨期效用或利润的现值
- **状态方程**：描述状态变量如何随控制变量变化

### 5.1.2 离散时间最优控制

**定义：离散时间最优控制问题**

$$\max_{\{u_t\}_{t=0}^{T-1}} \sum_{t=0}^{T-1} \beta^t F(x_t, u_t) + \beta^T \Phi(x_T)$$

$$s.t. \quad x_{t+1} = f(x_t, u_t)$$

$$x_0 \text{ 给定}$$

$$u_t \in U \text{ 对于所有 } t$$

其中 $\beta = e^{-\rho \Delta t}$ 是离散时间贴现因子。

### 5.1.3 与动态规划的关系

最优控制和动态规划都是解决跨期最优化问题的方法，但有重要区别：

| 特征 | 最优控制 | 动态规划 |
|------|----------|----------|
| 时间设定 | 连续时间 | 离散时间（可扩展到连续） |
| 数学工具 | 变分法、庞特里亚金原理 | 贝尔曼方程、递归方法 |
| 计算方法 | 解微分方程组 | 迭代算法（价值迭代、策略迭代） |
| 适用场景 | 连续时间问题、分析解 | 离散问题、数值解 |
| 理论基础 | 拉格朗日乘子法 | 不动点定理 |

**联系**：
- 连续时间动态规划中的哈密顿-雅可比-贝尔曼方程等价于庞特里亚金原理
- 离散时间最优控制可以看作是动态规划的特例

### 5.1.4 图形分析

**最优控制问题的结构**

```
时间 t₀ → t₁ → t₂ → ... → t_T
  │     │     │          │
  │     │     │          │
状态 x₀ → x₁ → x₂ → ... → x_T
  │     │     │          │
  │     │     │          │
控制 u₀  u₁  u₂  ...     u_{T-1}
  │     │     │          │
  │     │     │          │
收益 F(x₀,u₀) F(x₁,u₁) ... Φ(x_T)
  │     │     │          │
  └─────┴─────┴──────────┴─────► 目标函数
```

在每一期，决策者选择控制变量 $u_t$，这影响：
1. 即时收益 $F(x_t, u_t)$
2. 下一期状态 $x_{t+1} = f(x_t, u_t)$

最优控制问题的核心是权衡当前收益和未来收益。

### 5.1.5 例子

**资源开采问题**

一个公司开采不可再生资源，选择开采率 $u(t)$ 最大化利润现值：

$$\max_{u(t)} \int_{0}^{T} e^{-rt} [P u(t) - C(u(t), S(t))] dt$$

$$s.t. \quad \dot{S}(t) = -u(t)$$

$$S(0) = S_0 \text{ 给定}$$

$$u(t) \geq 0, \quad S(t) \geq 0$$

其中：
- $S(t)$ 是资源储量（状态变量）
- $u(t)$ 是开采率（控制变量）
- $P$ 是资源价格
- $C(u, S)$ 是开采成本，通常随开采率增加、储量减少而增加

**经济含义**：
- 快速开采获得即时利润，但减少未来储量
- 缓慢开采保存资源，但放弃即时利润
- 最优开采策略需要平衡这两个目标

## 5.2 庞特里亚金最大值原理

### 5.2.1 哈密顿函数

**定义：现值哈密顿函数**

$$\mathcal{H}(x, u, \lambda, t) = F(x, u, t) + \lambda(t)' f(x, u, t)$$

其中 $\lambda(t)$ 是协状态变量（影子价格）。

**定义：现值哈密顿函数**

$$\mathcal{H}_c(x, u, \mu, t) = e^{-\rho t} F(x, u, t) + \mu(t)' f(x, u, t)$$

其中 $\mu(t) = e^{-\rho t}\lambda(t)$ 是现值协状态变量。

**经济学解释**
- $F(x, u, t)$ 是即时收益
- $\lambda'(t)f(x, u, t)$ 是状态变化的边际价值的现值
- 哈密顿函数是"总边际价值"：当前收益加上状态变化的价值

### 5.2.2 必要条件（庞特里亚金原理）

对于最优控制问题，如果 $u^*(t)$ 和 $x^*(t)$ 是最优解，则存在协状态变量 $\lambda(t)$ 满足：

**条件1：状态方程**

$$\dot{x}^*(t) = \frac{\partial \mathcal{H}}{\partial \lambda} = f(x^*(t), u^*(t), t)$$

**条件2：协状态方程**

$$-\dot{\lambda}(t) = \frac{\partial \mathcal{H}}{\partial x} = \frac{\partial F}{\partial x} + \lambda(t)' \frac{\partial f}{\partial x}$$

**条件3：最大值条件**

$$u^*(t) = \arg\max_{u \in U} \mathcal{H}(x^*(t), u, \lambda(t), t)$$

**条件4：横截条件**

$$\lambda(t_f) = \frac{\partial \Phi(x^*(t_f), t_f)}{\partial x}$$

（如果有终值函数 $\Phi$）

### 5.2.3 经济学解释

**协状态变量 $\lambda(t)$**
- $\lambda(t)$ 是状态变量 $x(t)$ 的影子价格
- 表示状态变量增加一个单位对目标函数的边际贡献
- 类似于静态最优化中的拉格朗日乘子

**协状态方程**
$$-\dot{\lambda}(t) = \frac{\partial \mathcal{H}}{\partial x}$$

这个方程说明：影子价格的变化率等于哈密顿函数对状态变量的偏导数的负值。
- 左边：$-\dot{\lambda}(t)$ 是影子价格的贬值率
- 右边：$\frac{\partial F}{\partial x} + \lambda(t)' \frac{\partial f}{\partial x}$ 是状态变量对即时收益和未来状态的边际影响

**最大值条件**
$$u^*(t) = \arg\max_{u \in U} \mathcal{H}$$

这意味着在每一时刻，最优控制使哈密顿函数最大化，即权衡当前收益和未来收益。

### 5.2.4 求解步骤

**步骤1：建立哈密顿函数**

$$\mathcal{H}(x, u, \lambda, t) = F(x, u, t) + \lambda' f(x, u, t)$$

**步骤2：应用最大值条件**

从 $\frac{\partial \mathcal{H}}{\partial u} = 0$（如果内部解）或边界条件找到最优控制 $u^*(x, \lambda)$。

**步骤3：写出状态方程和协状态方程**

$$\dot{x} = \frac{\partial \mathcal{H}}{\partial \lambda} = f(x, u^*, t)$$
$$-\dot{\lambda} = \frac{\partial \mathcal{H}}{\partial x} = \frac{\partial F}{\partial x} + \lambda' \frac{\partial f}{\partial x}$$

**步骤4：应用边界条件**

初始条件：$x(t_0) = x_0$
横截条件：$\lambda(t_f) = \frac{\partial \Phi}{\partial x}$（如果适用）

**步骤5：解微分方程组**

解 $2n$ 个微分方程（$n$ 个状态方程，$n$ 个协状态方程）的边值问题。

### 5.2.5 特殊情况

**自治问题**

如果目标函数和状态方程不显含时间 $t$：

$$\max \int_{t_0}^{t_f} e^{-\rho t} F(x, u) dt$$

$$s.t. \quad \dot{x} = f(x, u)$$

则哈密顿函数在最优路径上为常数：
$$\mathcal{H}(x^*(t), u^*(t), \lambda(t)) = \text{常数}$$

这是哈密顿系统的守恒定律。

**无限期问题**

如果 $t_f \to \infty$，横截条件变为：

$$\lim_{t \to \infty} e^{-\rho t} \lambda(t) x(t) = 0$$

这个条件排除了"爆炸式"路径，确保目标函数有限。

### 5.2.6 图形分析

**相图分析（二维系统）**

考虑二维系统 $(x_1, x_2)$，我们可以在相平面中分析动态：

```
x₂
│
│       ṡ = 0
│      ••••••••
│     •       •
│    •   ẋ = 0 •
│   •       •
│  •       •
│ •       •
│•       •
└──────────────────► x₁
```

- $\dot{x}_1 = 0$ 曲线：$x_1$ 的等斜线
- $\dot{x}_2 = 0$ 曲线：$x_2$ 的等斜线
- 箭头表示运动方向
- 两条曲线的交点是稳态

**鞍点稳定性**

```
x₂
│
│       稳态
│        •
│       ╱│╲
│      ╱ │ ╲
│     ╱  │  ╲
│    •   │   •
│   ╱    │    ╲
│  •     │     •
└──────────────────► x₁
```

鞍点的特征：
- 两条稳定臂（收敛到稳态）
- 两条不稳定臂（发散）
- 最优路径位于稳定臂上

### 5.2.7 例子

**最优消费（Ramsey模型）**

考虑代表性家庭的消费选择：

$$\max_{c(t)} \int_{0}^{\infty} e^{-\rho t} \frac{c(t)^{1-\sigma}}{1-\sigma} dt$$

$$s.t. \quad \dot{k}(t) = f(k(t)) - c(t) - \delta k(t)$$

$$k(0) = k_0 \text{ 给定}$$

$$c(t) \geq 0, \quad k(t) \geq 0$$

其中：
- $c(t)$ 是消费（控制变量）
- $k(t)$ 是资本（状态变量）
- $f(k)$ 是生产函数
- $\delta$ 是折旧率

**哈密顿函数**：
$$\mathcal{H} = \frac{c^{1-\sigma}}{1-\sigma} + \lambda[f(k) - c - \delta k]$$

**最大值条件**：
$$\frac{\partial \mathcal{H}}{\partial c} = c^{-\sigma} - \lambda = 0 \Rightarrow \lambda = c^{-\sigma}$$

**协状态方程**：
$$-\dot{\lambda} = \frac{\partial \mathcal{H}}{\partial k} = \lambda[f'(k) - \delta]$$

**状态方程**：
$$\dot{k} = f(k) - c - \delta k$$

**欧拉方程**：
将 $\lambda = c^{-\sigma}$ 代入协状态方程：
$$-\frac{d}{dt}(c^{-\sigma}) = c^{-\sigma}[f'(k) - \delta]$$
$$\sigma c^{-\sigma-1} \dot{c} = c^{-\sigma}[f'(k) - \delta]$$
$$\frac{\dot{c}}{c} = \frac{1}{\sigma}[f'(k) - \delta - \rho]$$

**经济含义**：
- 消费增长率取决于资本的边际产出 $f'(k)$
- 如果 $f'(k) > \rho + \delta$，资本回报高，家庭增加消费
- 如果 $f'(k) < \rho + \delta$，资本回报低，家庭减少消费

**稳态**：
稳态时 $\dot{k} = 0$ 且 $\dot{c} = 0$：
$$c^* = f(k^*) - \delta k^*$$
$$f'(k^*) = \rho + \delta$$

这就是修正黄金律：资本的边际产出等于主观贴现率加折旧率。

## 5.3 离散时间最优控制

### 5.3.1 拉格朗日方法

对于离散时间最优控制问题，我们可以使用拉格朗日方法：

$$\mathcal{L} = \sum_{t=0}^{T-1} \left[\beta^t F(x_t, u_t) + \lambda_{t+1}' (f(x_t, u_t) - x_{t+1})\right] + \beta^T \Phi(x_T) + \lambda_0'(x_0 - x_0)$$

其中 $\lambda_t$ 是拉格朗日乘子。

**一阶条件**：

对 $u_t$：
$$\frac{\partial \mathcal{L}}{\partial u_t} = \beta^t \frac{\partial F}{\partial u_t} + \lambda_{t+1}' \frac{\partial f}{\partial u_t} = 0$$

对 $x_t$（$t = 1, \ldots, T-1$）：
$$\frac{\partial \mathcal{L}}{\partial x_t} = \beta^t \frac{\partial F}{\partial x_t} + \lambda_{t+1}' \frac{\partial f}{\partial x_t} - \lambda_t' = 0$$

对 $x_T$：
$$\frac{\partial \mathcal{L}}{\partial x_T} = \beta^T \frac{\partial \Phi}{\partial x_T} - \lambda_T' = 0$$

### 5.3.2 欧拉方程

从一阶条件可以得到欧拉方程：

$$\frac{\partial F}{\partial u_t} + \beta \frac{\partial F}{\partial x_{t+1}} \frac{\partial f}{\partial u_t} + \beta \frac{\partial F}{\partial u_{t+1}} \frac{\partial f}{\partial x_t} = 0$$

或者更简洁地：
$$\frac{\partial F}{\partial u_t} = -\beta \left[\frac{\partial F}{\partial x_{t+1}} \frac{\partial f}{\partial u_t} + \frac{\partial F}{\partial u_{t+1}} \frac{\partial f}{\partial x_t}\right]$$

**经济学解释**：
- 左边：当前控制变量的边际收益
- 右边：当前控制对未来的边际成本的现值

### 5.3.3 离散时间哈密顿系统

**定义：离散时间哈密顿函数**

$$\mathcal{H}_t = \beta^t F(x_t, u_t) + \lambda_{t+1}' f(x_t, u_t)$$

**必要条件**：

1. **状态方程**：
$$x_{t+1} = \frac{\partial \mathcal{H}_t}{\partial \lambda_{t+1}} = f(x_t, u_t)$$

2. **协状态方程**：
$$\lambda_t = \frac{\partial \mathcal{H}_t}{\partial x_t} = \beta^t \frac{\partial F}{\partial x_t} + \lambda_{t+1}' \frac{\partial f}{\partial x_t}$$

3. **最大值条件**：
$$\frac{\partial \mathcal{H}_t}{\partial u_t} = \beta^t \frac{\partial F}{\partial u_t} + \lambda_{t+1}' \frac{\partial f}{\partial u_t} = 0$$

4. **横截条件**：
$$\lambda_T = \beta^T \frac{\partial \Phi}{\partial x_T}$$

### 5.3.4 与连续时间的关系

当时间间隔 $\Delta t \to 0$ 时，离散时间最优控制收敛到连续时间最优控制：

$$\beta = e^{-\rho \Delta t} \approx 1 - \rho \Delta t$$
$$\dot{x} \approx \frac{x_{t+1} - x_t}{\Delta t}$$
$$\lambda_{t+1} \approx \lambda_t + \dot{\lambda} \Delta t$$

在此极限下，离散时间的一阶条件收敛到庞特里亚金原理。

### 5.3.5 图形分析

**跨期权衡**

```
t    期    t+1期
│     │      │
│     │      │
决策  u_t    u_{t+1}
│     │      │
│     │      │
收益  F(x_t, u_t)  F(x_{t+1}, u_{t+1})
│     │      │
│     │      │
状态  x_t → x_{t+1} → x_{t+2}
│     │      │
│     │      │
影响  - 影响本期收益
       - 影响下期状态
       - 从而影响下期收益
```

在时期 $t$ 选择 $u_t$ 时，需要考虑：
1. 对本期收益的直接影响
2. 通过改变 $x_{t+1}$ 对未来收益的间接影响

### 5.3.6 例子

**最优投资（离散时间）**

考虑企业的投资决策：

$$\max_{\{i_t\}_{t=0}^{T-1}} \sum_{t=0}^{T-1} \beta^t [F(k_t) - C(i_t, k_t)]$$

$$s.t. \quad k_{t+1} = (1-\delta)k_t + i_t$$

$$k_0 \text{ 给定}$$

其中：
- $k_t$ 是资本（状态变量）
- $i_t$ 是投资（控制变量）
- $F(k)$ 是生产函数
- $C(i, k)$ 是调整成本函数

**拉格朗日函数**：
$$\mathcal{L} = \sum_{t=0}^{T-1} \left[\beta^t(F(k_t) - C(i_t, k_t)) + \lambda_{t+1}((1-\delta)k_t + i_t - k_{t+1})\right]$$

**一阶条件**：

对 $i_t$：
$$-\beta^t \frac{\partial C}{\partial i_t} + \lambda_{t+1} = 0$$

对 $k_t$（$t = 1, \ldots, T-1$）：
$$\beta^t \left[\frac{\partial F}{\partial k_t} - \frac{\partial C}{\partial k_t}\right] + \lambda_{t+1}(1-\delta) - \lambda_t = 0$$

**欧拉方程**：
将 $\lambda_{t+1} = \beta^t \frac{\partial C}{\partial i_t}$ 代入对 $k_t$ 的条件：
$$\frac{\partial F}{\partial k_t} - \frac{\partial C}{\partial k_t} + \frac{\partial C}{\partial i_t}(1-\delta) = \beta \frac{\partial C}{\partial i_{t+1}}$$

**特殊情形：调整成本为零**

如果 $C(i, k) = i$，则 $\frac{\partial C}{\partial i_t} = 1$，$\frac{\partial C}{\partial k_t} = 0$：

$$\frac{\partial F}{\partial k_t} + (1-\delta) = \beta^{-1} = 1 + r$$

其中 $r$ 是利率。这得到标准的投资规则：投资的边际收益等于边际成本。

## 5.4 最优控制的应用

### 5.4.1 环境经济学：污染控制

**问题设定**

一个社会选择污染排放 $E(t)$ 最大化社会福利：

$$\max_{E(t)} \int_{0}^{\infty} e^{-\rho t} [B(E(t)) - D(S(t))] dt$$

$$s.t. \quad \dot{S}(t) = \gamma E(t) - \delta S(t)$$

$$S(0) = S_0 \text{ 给定}$$

其中：
- $S(t)$ 是污染存量（状态变量）
- $E(t)$ 是排放水平（控制变量）
- $B(E)$ 是排放的经济收益
- $D(S)$ 是污染的损害成本
- $\gamma$ 是排放到存量的转化率
- $\delta$ 是污染的自然衰减率

**哈密顿函数**：
$$\mathcal{H} = B(E) - D(S) + \lambda[\gamma E - \delta S]$$

**一阶条件**：

最大值条件：
$$\frac{\partial \mathcal{H}}{\partial E} = B'(E) + \lambda \gamma = 0 \Rightarrow \lambda = -\frac{B'(E)}{\gamma}$$

协状态方程：
$$-\dot{\lambda} = \frac{\partial \mathcal{H}}{\partial S} = -D'(S) - \lambda \delta$$

状态方程：
$$\dot{S} = \gamma E - \delta S$$

**最优排放规则**：
将 $\lambda = -\frac{B'(E)}{\gamma}$ 代入协状态方程：
$$\frac{d}{dt}\left(\frac{B'(E)}{\gamma}\right) = D'(S) - \frac{B'(E)}{\gamma}\delta$$

如果 $B(E)$ 和 $D(S)$ 是凸函数，这个微分方程描述了最优排放路径。

**稳态**：
稳态时 $\dot{S} = 0$ 且 $\dot{\lambda} = 0$：
$$E^* = \frac{\delta}{\gamma} S^*$$
$$D'(S^*) = \frac{\delta}{\gamma} B'(E^*)$$

这个条件表明：污染的边际损害等于减少排放的边际成本（经过贴现调整）。

### 5.4.2 财政学：最优税收

**问题设定**

政府选择税率 $\tau(t)$ 最大化社会福利：

$$\max_{\tau(t)} \int_{0}^{\infty} e^{-\rho t} u(c(t), 1-l(t)) dt$$

$$s.t. \quad \dot{k}(t) = f(k(t), l(t)) - c(t) - \tau(t) f(k(t), l(t))$$

$$c(t) = (1-\tau(t))f(k(t), l(t))$$

$$k(0) = k_0 \text{ 给定}$$

其中：
- $k(t)$ 是资本（状态变量）
- $\tau(t)$ 是税率（控制变量）
- $c(t)$ 是消费
- $l(t)$ 是劳动供给
- $f(k, l)$ 是生产函数

**哈密顿函数**：
$$\mathcal{H} = u(c, 1-l) + \lambda[f(k, l) - c - \tau f(k, l)]$$

利用预算约束 $c = (1-\tau)f(k, l)$：
$$\mathcal{H} = u((1-\tau)f(k, l), 1-l) + \lambda[k - (1-\tau)f(k, l)]$$

**一阶条件**：

对 $\tau$：
$$\frac{\partial \mathcal{H}}{\partial \tau} = \frac{\partial u}{\partial c}(-f) + \lambda f = 0$$

这得到标准的拉姆齐法则：税率应该是常数（如果生产函数是科布-道格拉斯的）。

### 5.4.3 货币经济学：最优货币增长

**问题设定**

中央银行选择货币增长率 $\mu(t)$ 最大化社会福利：

$$\max_{\mu(t)} \int_{0}^{\infty} e^{-\rho t} u(c(t), m(t)) dt$$

$$s.t. \quad \dot{m}(t) = \mu(t) - \pi(t) m(t)$$

$$\pi(t) = \mu(t)$$（假设完全预见的通胀）

$$m(0) = m_0 \text{ 给定}$$

其中：
- $m(t)$ 是实际货币余额（状态变量）
- $\mu(t)$ 是货币增长率（控制变量）
- $\pi(t)$ 是通货膨胀率
- $u(c, m)$ 是效用函数

**哈密顿函数**：
$$\mathcal{H} = u(c, m) + \lambda[\mu - \pi m]$$

**一阶条件**：

对 $\mu$：
$$\frac{\partial \mathcal{H}}{\partial \mu} = \lambda = 0$$

这意味着 $\lambda(t) = 0$ 对于所有 $t$，所以协状态方程简化为：
$$\frac{\partial u}{\partial m} - \pi \lambda = \frac{\partial u}{\partial m} = 0$$

如果 $\frac{\partial u}{\partial m} > 0$（货币提供正效用），这要求 $\lambda = 0$ 意味着最优货币增长应该使实际货币余额的边际效用为零。

**弗里德曼法则**：
在某些情况下，最优货币增长率是负的（通货紧缩），使得名义利率为零。

### 5.4.5 例子

**最优开采石油**

假设一家石油公司拥有储量 $S_0 = 100$ 单位石油，选择开采路径 $u(t)$ 最大化利润：

$$\max_{u(t)} \int_{0}^{T} e^{-0.05t} (10 u(t) - u(t)^2) dt$$

$$s.t. \quad \dot{S}(t) = -u(t)$$

$$S(0) = 100$$

$$u(t) \geq 0, \quad S(t) \geq 0$$

**哈密顿函数**：
$$\mathcal{H} = e^{-0.05t}(10u - u^2) + \lambda(-u)$$

**最大值条件**：
$$\frac{\partial \mathcal{H}}{\partial u} = e^{-0.05t}(10 - 2u) - \lambda = 0$$
$$2u = 10 - e^{0.05t}\lambda$$
$$u = 5 - \frac{1}{2}e^{0.05t}\lambda$$

**协状态方程**：
$$-\dot{\lambda} = \frac{\partial \mathcal{H}}{\partial S} = 0$$

这意味着 $\lambda(t) = \text{常数} = \lambda^*$。

**最优开采路径**：
$$u(t) = 5 - \frac{1}{2}e^{0.05t}\lambda^*$$

**横截条件**：
$$S(T) = 0 \Rightarrow \int_{0}^{T} u(t) dt = 100$$

假设 $T = 20$ 年，解这个积分方程：
$$\int_{0}^{20} \left(5 - \frac{1}{2}e^{0.05t}\lambda^*\right) dt = 100$$
$$5 \times 20 - \frac{\lambda^*}{2} \int_{0}^{20} e^{0.05t} dt = 100$$
$$100 - \frac{\lambda^*}{2} \times \frac{e^{1} - 1}{0.05} = 100$$
$$-\frac{\lambda^*}{2} \times \frac{1.718 - 1}{0.05} = 0$$
$$\lambda^* = 0$$

**结果**：
如果 $\lambda^* = 0$，则 $u(t) = 5$（常数开采率）。

验证：$\int_{0}^{20} 5 dt = 100$，正好用完储量。

**经济含义**：
- 由于没有稀缺性租金（$\lambda^* = 0$），最优开采率是常数
- 如果需求函数不同或有限期更长，开采率可能随时间变化

## 本章总结

### 核心要点

1. **最优控制问题**描述了在动态环境中如何选择控制变量以最大化跨期目标函数

2. **庞特里亚金最大值原理**提供了求解最优控制问题的系统方法，包括状态方程、协状态方程、最大值条件和横截条件

3. **协状态变量**是状态变量的影子价格，表示状态变量对目标函数的边际贡献

4. **欧拉方程**是跨期最优化的核心条件，描述了当前决策和未来决策之间的权衡

5. **最优控制理论**广泛应用于投资决策、消费选择、环境政策、税收政策等经济学问题

### 重要概念表格

| 概念 | 定义/公式 | 经济学意义 |
|------|-----------|------------|
| 状态变量 | $x(t)$，描述系统状态的存量 | 资本、财富、污染存量等 |
| 控制变量 | $u(t)$，决策者可以选择的流量 | 消费、投资、排放等 |
| 哈密顿函数 | $\mathcal{H} = F(x, u, t) + \lambda' f(x, u, t)$ | 总边际价值（当前+未来） |
| 协状态变量 | $\lambda(t)$，状态变量的影子价格 | 状态变量的边际价值 |
| 庞特里亚金原理 | 一组必要条件（状态、协状态、最大值、横截） | 求解最优控制的系统方法 |
| 欧拉方程 | 跨期最优化的一阶条件 | 描述当前和未来决策的权衡 |
| 横截条件 | $\lambda(t_f) = \partial \Phi / \partial x$ | 终值条件，确保最优性 |
| 鞍点 | 稳定臂和不稳定臂的交点 | 最优路径位于稳定臂上 |
| 稳态 | $\dot{x} = 0$ 且 $\dot{\lambda} = 0$ | 长期均衡状态 |
| 贴现因子 | $\beta = e^{-\rho \Delta t}$ | 折扣未来效用的因子 |
