# 第4章：随机过程

## 本章学习目标

通过本章的学习，学生应该能够：
1. 理解随机过程的基本概念和分类
2. 掌握马尔可夫过程的性质和应用
3. 熟悉ARIMA模型的设定、估计和预测
4. 理解状态空间模型的构建和卡尔曼滤波算法
5. 应用随机过程分析宏观经济波动和预测

## 4.1 随机过程基础

### 4.1.1 基本定义

**定义：随机过程**

随机过程是一个随机变量的集合 $\{X_t, t \in T\}$，其中 $T$ 是索引集（通常是时间）。对于每个固定的 $t$，$X_t$ 是一个随机变量；对于每个固定的样本路径，$\{X_t, t \in T\}$ 是一个时间序列。

**离散时间与连续时间**
- 离散时间：$T = \{0, 1, 2, \ldots\}$ 或 $T = \{\ldots, -1, 0, 1, \ldots\}$
- 连续时间：$T = [0, \infty)$ 或 $T = (-\infty, \infty)$

在宏观经济学中，我们主要关注离散时间过程。

**状态空间**
状态空间 $S$ 是随机变量可能取值的集合：
- 离散状态空间：$S$ 是有限或可数集（如 $\{0, 1\}$ 或 $\mathbb{Z}$）
- 连续状态空间：$S$ 是连续集（如 $\mathbb{R}$ 或 $\mathbb{R}^n$）

### 4.1.2 统计性质

**均值函数**

$$\mu_t = E[X_t]$$

**自协方差函数**

$$\gamma(t, s) = Cov(X_t, X_s) = E[(X_t - \mu_t)(X_s - \mu_s)]$$

**自相关函数**

$$\rho(t, s) = \frac{\gamma(t, s)}{\sqrt{\gamma(t, t)\gamma(s, s)}}$$

**平稳性**

**弱平稳（协方差平稳）**：
1. $E[X_t] = \mu$（常数）
2. $Var(X_t) = \gamma_0$（常数）
3. $Cov(X_t, X_{t-k}) = \gamma_k$（仅依赖于滞后 $k$）

**强平稳（严格平稳）**：
对于任意 $n$ 和任意时间点 $t_1, t_2, \ldots, t_n$，随机向量 $(X_{t_1}, X_{t_2}, \ldots, X_{t_n})$ 的联合分布与 $(X_{t_1+h}, X_{t_2+h}, \ldots, X_{t_n+h})$ 的联合分布相同。

### 4.1.3 白噪声过程

**定义：白噪声过程**

一个过程 $\{\varepsilon_t\}$ 是白噪声，如果：
1. $E[\varepsilon_t] = 0$
2. $Var(\varepsilon_t) = \sigma^2$
3. $Cov(\varepsilon_t, \varepsilon_s) = 0$ 对于 $t \neq s$

**高斯白噪声**：如果白噪声过程的每个 $\varepsilon_t$ 服从正态分布 $N(0, \sigma^2)$，则称为高斯白噪声。

### 4.1.4 图形分析

**白噪声过程的时间序列图**

```
εₜ
│
│   •      •         •    •
│  • •   •   •     • •  •
│ •   • •     • • •    •
│•     •       •       •
└────────────────────────► t
```

白噪声过程表现出无规律、不相关的随机波动，没有明显的趋势或周期。

**平稳过程 vs 非平稳过程**

```
平稳过程              非平稳过程
│                    │
│  ••••••••••••••   │      •
│  •  •  •  •  •    │     •
│ •  •  •  •  •    │    •
│•  •  •  •  •     │   •
└─────────────────► └─────────►
   t                   t
```

平稳过程围绕常数均值波动，而非平稳过程有明显的趋势或漂移。

### 4.1.5 例子

**GDP增长率序列**

考虑美国季度GDP增长率 $g_t = \frac{GDP_t - GDP_{t-1}}{GDP_{t-1}}$：

$$g_t = 0.02 + \varepsilon_t + 0.5\varepsilon_{t-1}$$

其中 $\varepsilon_t$ 是方差为 $\sigma^2 = 0.001$ 的白噪声。

这个过程的性质：
- 均值：$E[g_t] = 0.02$
- 方差：$Var(g_t) = \sigma^2(1 + 0.5^2) = 0.00125$
- 自协方差：$\gamma_0 = 0.00125$, $\gamma_1 = 0.5\sigma^2 = 0.0005$, $\gamma_k = 0$ 对于 $k \geq 2$

这是一个MA(1)过程，是平稳的，因为其自协方差只依赖于滞后，不随时间变化。

## 4.2 马尔可夫过程

### 4.2.1 定义与性质

**定义：马尔可夫过程**

一个随机过程 $\{X_t\}$ 是马尔可夫过程，如果对于任意 $n$ 和任意时间点 $t_1 < t_2 < \ldots < t_n$：

$$P(X_{t_{n+1}} \leq x|X_{t_1}, X_{t_2}, \ldots, X_{t_n}) = P(X_{t_{n+1}} \leq x|X_{t_n})$$

即未来的状态只依赖于当前状态，与历史无关。

**转移概率**

对于离散状态空间，转移概率为：
$$P_{ij}(t) = P(X_{t+1} = j|X_t = i)$$

对于连续状态空间，转移密度为：
$$f(x_{t+1}|x_t) = \frac{d}{dx_{t+1}} P(X_{t+1} \leq x_{t+1}|X_t = x_t)$$

### 4.2.2 离散马尔可夫链

**定义**：状态空间 $S = \{1, 2, \ldots, N\}$ 是有限集的马尔可夫过程。

**转移矩阵**

$$\mathbf{P} = \begin{bmatrix}
p_{11} & p_{12} & \cdots & p_{1N} \\
p_{21} & p_{22} & \cdots & p_{2N} \\
\vdots & \vdots & \ddots & \vdots \\
p_{N1} & p_{N2} & \cdots & p_{NN}
\end{bmatrix}$$

其中 $p_{ij} = P(X_{t+1} = j|X_t = i)$，满足：
1. $p_{ij} \geq 0$ 对于所有 $i, j$
2. $\sum_{j=1}^{N} p_{ij} = 1$ 对于所有 $i$

**n步转移概率**

$$\mathbf{P}^{(n)} = \mathbf{P}^n$$

其中 $p_{ij}^{(n)} = P(X_{t+n} = j|X_t = i)$。

** Chapman-Kolmogorov方程**

$$p_{ij}^{(m+n)} = \sum_{k=1}^{N} p_{ik}^{(m)} p_{kj}^{(n)}$$

用矩阵表示：$\mathbf{P}^{(m+n)} = \mathbf{P}^{(m)}\mathbf{P}^{(n)}$。

### 4.2.3 稳态分布

**定义**：一个概率分布 $\boldsymbol{\pi} = (\pi_1, \pi_2, \ldots, \pi_N)$ 是稳态分布，如果：

$$\boldsymbol{\pi} = \boldsymbol{\pi}\mathbf{P}$$

即 $\pi_j = \sum_{i=1}^{N} \pi_i p_{ij}$ 对于所有 $j$。

**存在性条件**：如果马尔可夫链是不可约的且非周期的，则存在唯一的稳态分布。

**不可约性**：从任意状态可以到达任意其他状态。

**非周期性**：不存在周期性的状态转移模式。

### 4.2.4 连续马尔可夫过程

**AR(1)过程**

$$X_{t+1} = \rho X_t + \varepsilon_{t+1}$$
$$\varepsilon_{t+1} \sim N(0, \sigma^2)$$

其中 $|\rho| < 1$ 保证平稳性。

**转移密度**

$$f(x_{t+1}|x_t) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x_{t+1} - \rho x_t)^2}{2\sigma^2}\right)$$

**稳态分布**

如果 $|\rho| < 1$，$X_t$ 收敛到稳态分布 $N(0, \frac{\sigma^2}{1-\rho^2})$。

### 4.2.5 图形分析

**马尔可夫链的转移图**

```
    状态1        状态2        状态3
      │            │            │
      │ 0.7        │ 0.6        │ 0.5
      ▼            ▼            ▼
   [ 1 ] ───0.3──► [ 2 ] ───0.4──► [ 3 ]
      ▲            ▲            ▲
      │ 0.2        │ 0.3        │ 0.5
      └────────────┴────────────┘
```

每个箭头表示状态转移，数字表示转移概率。

**AR(1)过程的样本路径**

```
Xₜ
│
│           •••••••••••••
│        ••             ••
│      ••                 ••
│    ••                     ••
│  ••                         ••
│••                             ••
└────────────────────────────────► t
```

AR(1)过程表现出持续性：当 $X_t$ 为正时，$X_{t+1}$ 也倾向于为正。

### 4.2.6 例子

**经济周期状态转移**

考虑一个简化的经济周期模型，经济有两个状态：
- 状态1：扩张（E）
- 状态2：衰退（R）

转移矩阵：
$$\mathbf{P} = \begin{bmatrix}
0.95 & 0.05 \\
0.10 & 0.90
\end{bmatrix}$$

**解释**：
- 如果当前是扩张，下一期有95%概率继续扩张，5%概率进入衰退
- 如果当前是衰退，下一期有90%概率继续衰退，10%概率进入扩张

**稳态分布**：
设 $\boldsymbol{\pi} = (\pi_E, \pi_R)$，则：

$$\pi_E = 0.95\pi_E + 0.10\pi_R$$
$$\pi_R = 0.05\pi_E + 0.90\pi_R$$
$$\pi_E + \pi_R = 1$$

解得：$\pi_E = 2/3$, $\pi_R = 1/3$

**经济含义**：
- 长期来看，经济有2/3的时间处于扩张状态
- 衰退的平均持续时间：$1/(1-0.90) = 10$ 个时期
- 扩张的平均持续时间：$1/(1-0.95) = 20$ 个时期

## 4.3 ARIMA模型

### 4.3.1 自回归模型（AR）

**定义：AR(p)过程**

$$X_t = c + \phi_1 X_{t-1} + \phi_2 X_{t-2} + \cdots + \phi_p X_{t-p} + \varepsilon_t$$

其中 $\varepsilon_t$ 是白噪声。

**滞后算子表示**

$$\phi(L)X_t = c + \varepsilon_t$$

其中 $\phi(L) = 1 - \phi_1 L - \phi_2 L^2 - \cdots - \phi_p L^p$。

**平稳性条件**

AR(p)过程平稳当且仅当特征方程的根都在单位圆外：

$$\phi(z) = 1 - \phi_1 z - \phi_2 z^2 - \cdots - \phi_p z^p = 0$$

的所有根 $|z| > 1$。

**AR(1)特例**

$$X_t = c + \phi X_{t-1} + \varepsilon_t$$

平稳性条件：$|\phi| < 1$

均值：$\mu = \frac{c}{1-\phi}$

方差：$\gamma_0 = \frac{\sigma^2}{1-\phi^2}$

自相关：$\rho_k = \phi^k$（几何衰减）

### 4.3.2 移动平均模型（MA）

**定义：MA(q)过程**

$$X_t = \mu + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \theta_2 \varepsilon_{t-2} + \cdots + \theta_q \varepsilon_{t-q}$$

其中 $\varepsilon_t$ 是白噪声。

**滞后算子表示**

$$X_t = \mu + \theta(L)\varepsilon_t$$

其中 $\theta(L) = 1 + \theta_1 L + \theta_2 L^2 + \cdots + \theta_q L^q$。

**可逆性条件**

MA(q)过程可逆当且仅当 $\theta(z) = 0$ 的所有根 $|z| > 1$。

**MA(1)特例**

$$X_t = \mu + \varepsilon_t + \theta \varepsilon_{t-1}$$

均值：$\mu$

方差：$\gamma_0 = (1+\theta^2)\sigma^2$

自协方差：$\gamma_1 = \theta\sigma^2$, $\gamma_k = 0$ 对于 $k \geq 2$

自相关：$\rho_1 = \frac{\theta}{1+\theta^2}$, $\rho_k = 0$ 对于 $k \geq 2$

### 4.3.3 自回归移动平均模型（ARMA）

**定义：ARMA(p,q)过程**

$$X_t = c + \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t + \theta_1 \varepsilon_{t-1} + \cdots + \theta_q \varepsilon_{t-q}$$

滞后算子表示：
$$\phi(L)X_t = c + \theta(L)\varepsilon_t$$

平稳性：$\phi(z) = 0$ 的根都在单位圆外

可逆性：$\theta(z) = 0$ 的根都在单位圆外

### 4.3.4 自回归积分移动平均模型（ARIMA）

**定义：ARIMA(p,d,q)过程**

如果 $X_t$ 需要d阶差分才能平稳，则称 $X_t \sim ARIMA(p,d,q)$：

$$\phi(L)(1-L)^d X_t = c + \theta(L)\varepsilon_t$$

其中：
- $(1-L)^d$ 是d阶差分算子
- $\phi(L)$ 是p阶AR多项式
- $\theta(L)$ 是q阶MA多项式

**特例**：
- ARIMA(0,1,0) = 随机游走：$X_t = X_{t-1} + \varepsilon_t$
- ARIMA(0,0,1) = MA(1)
- ARIMA(1,0,0) = AR(1)

### 4.3.5 模型识别与估计

**Box-Jenkins方法**

1. **识别**：通过ACF和PACF确定p, d, q
   - AR(p)：ACF衰减，PACF在p阶后截尾
   - MA(q)：ACF在q阶后截尾，PACF衰减
   - ARMA：ACF和PACF都衰减

2. **估计**：使用极大似然法或最小二乘法估计参数

3. **诊断**：检验残差是否为白噪声

**信息准则**
- AIC：$AIC = \ln(\hat{\sigma}^2) + \frac{2(p+q)}{T}$
- BIC：$BIC = \ln(\hat{\sigma}^2) + \frac{(p+q)\ln(T)}{T}$

### 4.3.6 图形分析

**ACF和PACF的典型模式**

```
AR(1)                MA(1)                ARMA(1,1)
│                    │                    │
│ •  •  •  •         │ •                  │ •  •  •
│  •  •  •  •        │  •                 │  •  •  •
│   •  •  •  •       │                   │   •  •  •
│____▲______│        │__▲_____________   │    •  •  •
  ACF                │  PACF              │____▲____│
                       ACF                    ACF
│                    │                    │
│ •                  │ •  •  •  •         │ •
│  ▲                 │  •  •  •  •        │  ▲
│   │                │   •  •  •  •       │   │
│___│                │____▲______│        │___│
  PACF                  PACF                PACF
```

AR过程：ACF衰减，PACF截尾
MA过程：ACF截尾，PACF衰减
ARMA过程：ACF和PACF都衰减

### 4.3.7 例子

**美国GDP增长率建模**

假设我们观察美国季度GDP增长率 $g_t$：

**数据特征**：
- 样本容量：T = 200
- 均值：$\bar{g} = 0.02$
- 标准差：$s_g = 0.015$

**ACF和PACF**：
- ACF：$\hat{\rho}_1 = 0.60$, $\hat{\rho}_2 = 0.40$, $\hat{\rho}_3 = 0.25$, 然后缓慢衰减
- PACF：$\hat{\phi}_{11} = 0.60$, $\hat{\phi}_{22} = 0.10$, $\hat{\phi}_{33} = 0.05$, 一阶后截尾

**模型选择**：
根据ACF和PACF模式，选择AR(1)模型：
$$g_t = c + \phi g_{t-1} + \varepsilon_t$$

**估计结果**：
$$\hat{g}_t = 0.008 + 0.58 g_{t-1} + \hat{\varepsilon}_t$$
$$(0.002) \quad (0.07)$$

**诊断**：
- 残差ACF在所有滞后都不显著
- Ljung-Box检验：Q(10) = 8.5, p值 = 0.58（不拒绝白噪声假设）

**预测**：
一阶预测：
$$E[g_{T+1}|I_T] = 0.008 + 0.58 g_T$$

如果 $g_T = 0.03$，则：
$$E[g_{T+1}|I_T] = 0.008 + 0.58 \times 0.03 = 0.0254$$

预测误差方差：$\sigma^2(1 + 0.58^2) = 1.34\sigma^2$

## 4.4 状态空间模型

### 4.4.1 基本设定

**定义：状态空间模型**

一个状态空间模型由两个方程组成：

**状态方程（转移方程）**：
$$\boldsymbol{\alpha}_t = \mathbf{T}_t \boldsymbol{\alpha}_{t-1} + \mathbf{R}_t \boldsymbol{\eta}_t$$

**观测方程（量测方程）**：
$$\mathbf{y}_t = \mathbf{Z}_t \boldsymbol{\alpha}_t + \mathbf{d}_t + \boldsymbol{\varepsilon}_t$$

其中：
- $\boldsymbol{\alpha}_t$ 是 $m \times 1$ 状态向量
- $\mathbf{y}_t$ 是 $n \times 1$ 观测向量
- $\mathbf{T}_t$ 是 $m \times m$ 转移矩阵
- $\mathbf{Z}_t$ 是 $n \times m$ 量测矩阵
- $\mathbf{d}_t$ 是 $n \times 1$ 确定性项
- $\mathbf{R}_t$ 是 $m \times r$ 选择矩阵
- $\boldsymbol{\eta}_t$ 是 $r \times 1$ 状态扰动，$E[\boldsymbol{\eta}_t] = \mathbf{0}$, $Var(\boldsymbol{\eta}_t) = \mathbf{Q}_t$
- $\boldsymbol{\varepsilon}_t$ 是 $n \times 1$ 观测误差，$E[\boldsymbol{\varepsilon}_t] = \mathbf{0}$, $Var(\boldsymbol{\varepsilon}_t) = \mathbf{H}_t$
- $\boldsymbol{\eta}_t$ 和 $\boldsymbol{\varepsilon}_t$ 相互独立，且独立于历史

### 4.4.2 结构时间序列模型

**趋势分解模型**

$$y_t = \mu_t + \gamma_t + \varepsilon_t$$

其中：
- $\mu_t$ 是趋势成分
- $\gamma_t$ 是周期成分
- $\varepsilon_t$ 是不规则成分

**趋势成分（局部线性趋势）**：
$$\mu_t = \mu_{t-1} + \beta_{t-1} + \eta_t$$
$$\beta_t = \beta_{t-1} + \zeta_t$$

其中 $\eta_t$ 和 $\zeta_t$ 是独立的白噪声。

**周期成分**：
$$\gamma_t = \sum_{j=1}^{[s/2]} \gamma_{j,t}$$

对于偶数s：
$$\gamma_{j,t} = \gamma_{j,t-1}\cos\lambda_j + \gamma_{j,t-1}^*\sin\lambda_j + \omega_{j,t}$$
$$\gamma_{j,t}^* = -\gamma_{j,t-1}\sin\lambda_j + \gamma_{j,t-1}^*\cos\lambda_j + \omega_{j,t}^*$$

对于奇数s，还有最后一项：
$$\gamma_{s/2,t} = \gamma_{s/2,t-1}\cos\pi + \omega_{s/2,t} = -\gamma_{s/2,t-1} + \omega_{s/2,t}$$

其中 $\lambda_j = 2\pi j/s$。

### 4.4.3 卡尔曼滤波

**目的**：给定观测数据 $\mathbf{y}_1, \mathbf{y}_2, \ldots, \mathbf{y}_T$，估计状态 $\boldsymbol{\alpha}_t$。

**符号**：
- $\mathbf{a}_{t|t-1} = E[\boldsymbol{\alpha}_t|\mathbf{Y}_{t-1}]$（预测）
- $\mathbf{P}_{t|t-1} = Var(\boldsymbol{\alpha}_t|\mathbf{Y}_{t-1})$（预测误差方差）
- $\mathbf{a}_{t|t} = E[\boldsymbol{\alpha}_t|\mathbf{Y}_{t}]$（滤波）
- $\mathbf{P}_{t|t} = Var(\boldsymbol{\alpha}_t|\mathbf{Y}_{t})$（滤波误差方差）

**初始化**：
$$\mathbf{a}_{1|0} = \boldsymbol{\alpha}_0$$
$$\mathbf{P}_{1|0} = \mathbf{P}_0$$

**预测步骤**：
$$\mathbf{a}_{t|t-1} = \mathbf{T}_t \mathbf{a}_{t-1|t-1}$$
$$\mathbf{P}_{t|t-1} = \mathbf{T}_t \mathbf{P}_{t-1|t-1} \mathbf{T}_t' + \mathbf{R}_t \mathbf{Q}_t \mathbf{R}_t'$$

**更新步骤**：
$$\mathbf{v}_t = \mathbf{y}_t - \mathbf{Z}_t \mathbf{a}_{t|t-1} - \mathbf{d}_t$$（新息）
$$\mathbf{F}_t = \mathbf{Z}_t \mathbf{P}_{t|t-1} \mathbf{Z}_t' + \mathbf{H}_t$$（新息方差）
$$\mathbf{K}_t = \mathbf{P}_{t|t-1} \mathbf{Z}_t' \mathbf{F}_t^{-1}$$（卡尔曼增益）
$$\mathbf{a}_{t|t} = \mathbf{a}_{t|t-1} + \mathbf{K}_t \mathbf{v}_t$$
$$\mathbf{P}_{t|t} = \mathbf{P}_{t|t-1} - \mathbf{K}_t \mathbf{Z}_t \mathbf{P}_{t|t-1}$$

**对数似然函数**：

$$\ln L = -\frac{nT}{2}\ln(2\pi) - \frac{1}{2}\sum_{t=1}^{T}\left[\ln|\mathbf{F}_t| + \mathbf{v}_t'\mathbf{F}_t^{-1}\mathbf{v}_t\right]$$

### 4.4.4 平滑（Kalman Smoother）

**目的**：基于所有观测数据 $\mathbf{Y}_T = \{\mathbf{y}_1, \ldots, \mathbf{y}_T\}$ 估计状态 $\boldsymbol{\alpha}_t$。

**递归平滑**：

$$\mathbf{a}_{t|T} = \mathbf{a}_{t|t} + \mathbf{P}_{t|t}\mathbf{T}_{t+1}'\mathbf{P}_{t+1|t}^{-1}(\mathbf{a}_{t+1|T} - \mathbf{a}_{t+1|t})$$

$$\mathbf{P}_{t|T} = \mathbf{P}_{t|t} + \mathbf{P}_{t|t}\mathbf{T}_{t+1}'\mathbf{P}_{t+1|t}^{-1}(\mathbf{P}_{t+1|T} - \mathbf{P}_{t+1|t})\mathbf{P}_{t+1|t}^{-1}\mathbf{T}_{t+1}\mathbf{P}_{t|t}$$

从 $t = T-1, T-2, \ldots, 1$ 递归计算。

### 4.4.5 图形分析

**状态空间模型的结构**

```
ηₜ → 状态方程 → αₜ → 观测方程 → yₜ
     ↑                        ↑
     │                        │
   状态扰动                  观测误差
     │                        │
     └────────────────────────┘
```

状态变量 $\alpha_t$ 隐藏在观测数据 $y_t$ 背后，卡尔曼滤波通过观测数据推断状态。

**卡尔曼滤波的递归过程**

```
时间 t=0          t=1          t=2          t=3
   │             │             │             │
   │             │             │             │
α₀ → 预测 → a₁|₀ → 更新 → a₁|₁ → 预测 → a₂|₁ → ...
   │             │             │
   │             │             │
预测P₁|₀      新息v₁        预测P₂|₁
   │             │             │
   │             │             │
   └─────────────┴─────────────┘
                观测 y₁
```

每一步都包括预测和更新两个阶段，新息 $v_t$ 是预测误差，用于更新状态估计。

### 4.4.6 例子

**潜在GDP的估计**

考虑美国实际GDP $y_t$ 的分解：

$$y_t = \mu_t + \varepsilon_t$$

其中：
- $\mu_t$ 是潜在GDP（趋势）
- $\varepsilon_t$ 是产出缺口（周期）

**状态方程**：
$$\mu_t = \mu_{t-1} + \beta_{t-1} + \eta_t$$
$$\beta_t = \beta_{t-1} + \zeta_t$$

**参数**：
- $\eta_t \sim N(0, \sigma_\eta^2)$
- $\zeta_t \sim N(0, \sigma_\zeta^2)$
- $\varepsilon_t \sim N(0, \sigma_\varepsilon^2)$

**状态空间表示**：

状态向量：$\boldsymbol{\alpha}_t = (\mu_t, \beta_t)'$

状态方程：
$$\begin{bmatrix}
\mu_t \\
\beta_t
\end{bmatrix} = \begin{bmatrix}
1 & 1 \\
0 & 1
\end{bmatrix} \begin{bmatrix}
\mu_{t-1} \\
\beta_{t-1}
\end{bmatrix} + \begin{bmatrix}
\eta_t \\
\zeta_t
\end{bmatrix}$$

观测方程：
$$y_t = \begin{bmatrix}1 & 0\end{bmatrix} \begin{bmatrix}
\mu_t \\
\beta_t
\end{bmatrix} + \varepsilon_t$$

**估计结果**（示例）：
- $\hat{\sigma}_\eta^2 = 0.002$
- $\hat{\sigma}_\zeta^2 = 0.0001$
- $\hat{\sigma}_\varepsilon^2 = 0.001$

**经济含义**：
- 较大的 $\sigma_\eta^2$ 表明潜在GDP有显著的不确定性
- 较小的 $\sigma_\zeta^2$ 表明趋势增长率相对稳定
- 产出缺口围绕潜在GDP波动，反映了经济周期

## 本章总结

### 核心要点

1. **随机过程**是随时间演变的随机变量集合，其统计性质（均值、方差、自协方差）描述了时间序列的特征

2. **马尔可夫过程**的核心是"无记忆性"：未来只依赖于当前，不依赖于历史

3. **ARIMA模型**是时间序列分析的核心工具，通过AR、MA和差分操作描述各种时间序列模式

4. **状态空间模型**提供了一个统一的框架来处理不可观测的状态变量，**卡尔曼滤波**是估计这些状态的有效算法

5. **模型选择**需要结合理论和数据，使用ACF、PACF、信息准则等工具

### 重要概念表格

| 概念 | 定义/公式 | 经济学意义 |
|------|-----------|------------|
| 平稳性 | 均值、方差不随时间变化 | 保证模型的可预测性 |
| 白噪声 | 零均值、常方差、无自相关 | 时间序列的"基础砖块" |
| 马尔可夫过程 | $P(X_{t+1}|X_t, X_{t-1}, \ldots) = P(X_{t+1}|X_t)$ | 简化动态系统的建模 |
| AR(p)过程 | $\phi(L)X_t = c + \varepsilon_t$ | 描述持续性 |
| MA(q)过程 | $X_t = \mu + \theta(L)\varepsilon_t$ | 描述短期冲击 |
| ARIMA(p,d,q) | $\phi(L)(1-L)^d X_t = c + \theta(L)\varepsilon_t$ | 描述非平稳过程 |
| 状态空间模型 | $\alpha_t = T_t\alpha_{t-1} + R_t\eta_t$, $y_t = Z_t\alpha_t + \varepsilon_t$ | 处理不可观测变量 |
| 卡尔曼滤波 | 递归估计状态的算法 | 从观测数据推断隐藏状态 |
| 新息 | $v_t = y_t - Z_t a_{t|t-1}$ | 预测误差，包含新信息 |
| 平滑 | 利用所有数据估计历史状态 | 改进历史状态估计 |
