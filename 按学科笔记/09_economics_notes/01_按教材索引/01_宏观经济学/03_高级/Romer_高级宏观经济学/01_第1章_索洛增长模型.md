## 1.1 索洛增长模型的基本框架

### 1.1.1 模型的基本假设

**索洛增长模型（Solow Growth Model）**：
描述长期经济增长的基本模型，由罗伯特·索洛于1956年提出。

**基本假设**：

1. **生产函数（Production Function）**：
   ```
   Y(t) = F(K(t), A(t)L(t))
   ```
   **其中**：
   - Y(t)：产出
   - K(t)：资本
   - L(t)：劳动力
   - A(t)：技术水平

2. **规模报酬不变（Constant Returns to Scale）**：
   ```
   F(cK, cAL) = cF(K, AL), 对于任意c ≥ 0
   ```

3. **边际报酬递减（Diminishing Marginal Returns）**：
   ```
   ∂F/∂K > 0, ∂²F/∂K² < 0
   ∂F/∂(AL) > 0, ∂²F/∂(AL)² < 0
   ```

4. **稻田条件（Inada Conditions）**：
   ```
   lim_{K→0} ∂F/∂K = ∞
   lim_{K→∞} ∂F/∂K = 0
   lim_{AL→0} ∂F/∂(AL) = ∞
   lim_{AL→∞} ∂F/∂(AL) = 0
   ```

### 1.1.2 密集型生产函数

**密集型生产函数（Intensive Form of Production Function）**：
由于规模报酬不变，可以写成：
```
Y(t) = A(t)L(t)f(k(t))
```
**其中**：
```
k(t) = K(t) / [A(t)L(t)]：人均有效资本
f(k) = F(k, 1)：人均有效产出
```

**性质**：
- f'(k) > 0：边际产出为正
- f''(k) < 0：边际报酬递减

### 1.1.3 柯布-道格拉斯生产函数

**柯布-道格拉斯生产函数（Cobb-Douglas Production Function）**：
```
Y(t) = K(t)^α [A(t)L(t)]^(1-α), 0 < α < 1
```
**密集型形式**：
```
f(k) = k^α
```

## 1.2 资本积累

### 1.2.1 资本积累方程

**资本积累方程（Capital Accumulation Equation）**：
```
K̇(t) = sY(t) - δK(t)
```
**其中**：
- K̇(t)：资本变化量
- s：储蓄率
- δ：折旧率

### 1.2.2 人均有效资本积累方程

**人均有效资本积累方程（Equation for the Evolution of k）**：
```
k̇(t) = sf(k(t)) - (n + g + δ)k(t)
```
**其中**：
- k̇(t)：人均有效资本变化量
- n：人口增长率
- g：技术增长率
- n + g + δ：有效折旧率

**推导**：
```
k = K / (AL)
k̇ = K̇/(AL) - K/(AL)²·(ȦL + AL̇)
    = [sY - δK]/(AL) - K/(AL)·(Ȧ/A + L̇/L)
    = sf(k) - (n + g + δ)k
```

### 1.2.3 k的动态分析

**k的动态分析（Dynamics of k）**：

- 当sf(k) > (n + g + δ)k时，k上升
- 当sf(k) < (n + g + δ)k时，k下降
- 当sf(k) = (n + g + δ)k时，k不变

## 1.3 平衡增长路径

### 1.3.1 平衡增长路径的定义

**平衡增长路径（Balanced Growth Path, BGP）**：
k、y、c按恒定速率增长的路径。

### 1.3.2 平衡增长路径的条件

**平衡增长路径的条件（Conditions for BGP）**：
```
sf(k*) = (n + g + δ)k*
```
**其中**：
- k*：平衡增长路径上的人均有效资本

**人均有效产出**：
```
y* = f(k*)
```

**人均有效消费**：
```
c* = (1 - s)f(k*)
```

### 1.3.3 平衡增长路径上的增长率

**平衡增长路径上的增长率（Growth Rates on BGP）**：

| 变量 | 增长率 |
|------|--------|
| 人均有效资本 | 0 |
| 人均有效产出 | 0 |
| 人均资本 | g |
| 人均产出 | g |
| 资本 | n + g |
| 产出 | n + g |
| 劳动力 | n |
| 技术 | g |

## 1.4 收敛性

### 1.4.1 绝对收敛

**绝对收敛（Absolute Convergence）**：
贫穷国家比富裕国家增长更快，最终收敛到相同的平衡增长路径。

**条件**：
- 所有国家有相同的s、n、g、δ、f(·)

### 1.4.2 条件收敛

**条件收敛（Conditional Convergence）**：
贫穷国家比富裕国家增长更快，最终收敛到各自的平衡增长路径。

**条件**：
- 国家有不同的s、n、g、δ、f(·)

### 1.4.3 收敛速度

**收敛速度（Speed of Convergence）**：
k向k*收敛的速度。

**线性近似**：
```
k̇ ≈ -λ(k - k*)
```
**其中**：
- λ：收敛速度
```
λ = (n + g + δ)(1 - α)
```
**半衰期（Half-Life）**：
```
T = ln(2) / λ
```

## 1.5 比较静态分析

### 1.5.1 储蓄率变化

**储蓄率上升（Increase in Savings Rate）**：
- 平衡增长路径上的人均有效资本k*上升
- 平衡增长路径上的人均有效产出y*上升
- 短期经济增长率上升
- 长期经济增长率不变

- 人均产出增长率在调整期间高于g
- 调整完成后，人均产出增长率回到g

### 1.5.2 人口增长率变化

**人口增长率上升（Increase in Population Growth Rate）**：
- 平衡增长路径上的人均有效资本k*下降
- 平衡增长路径上的人均有效产出y*下降
- 短期经济增长率下降
- 长期经济增长率不变

### 1.5.3 技术进步率变化

**技术进步率上升（Increase in Technological Progress Rate）**：
- 平衡增长路径上的人均有效资本k*不变
- 平衡增长路径上的人均有效产出y*不变
- 长期经济增长率上升

## 1.6 本章总结

### 核心要点

1. **索洛增长模型的基本框架**：
   - 基本假设：规模报酬不变、边际报酬递减、稻田条件
   - 密集型生产函数：y = f(k)
   - 柯布-道格拉斯生产函数：f(k) = k^α

2. **资本积累**：
   - 资本积累方程：K̇ = sY - δK
   - 人均有效资本积累方程：k̇ = sf(k) - (n + g + δ)k
   - k的动态分析：k趋向于平衡增长路径

3. **平衡增长路径**：
   - 平衡增长路径的条件：sf(k*) = (n + g + δ)k*
   - 平衡增长路径上的增长率：人均资本和人均产出增长率为g

4. **收敛性**：
   - 绝对收敛：所有国家收敛到相同的平衡增长路径
   - 条件收敛：各国收敛到各自的平衡增长路径
   - 收敛速度：λ = (n + g + δ)(1 - α)

5. **比较静态分析**：
   - 储蓄率上升：k*和y*上升，短期经济增长率上升，长期经济增长率不变
   - 人口增长率上升：k*和y*下降，长期经济增长率不变
   - 技术进步率上升：k*和y*不变，长期经济增长率上升
