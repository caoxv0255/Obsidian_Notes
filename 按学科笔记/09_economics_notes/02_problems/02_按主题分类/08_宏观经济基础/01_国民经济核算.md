# 第1章 国民经济核算

## 本章目标

国民经济核算是宏观经济学的起点。本章要解决的不是“背公式”，而是理解为什么 GDP、GNP、NI、增加值这些概念要同时存在，以及它们各自能说明什么。

### 学完本章你应该能够
- 说清 GDP 的定义和统计口径
- 区分支出法、收入法和生产法
- 区分名义变量和实际变量
- 判断一个数据属于流量还是存量
- 解释 GDP 为什么不能完全代表福利

## 1. 核心概念

### GDP
GDP 是一定时期内一国境内生产的最终产品和服务的市场价值总和。它强调三个关键词：
- 一定时期内
- 一国境内
- 最终产品与服务

### GNP / GNI
GNP 和 GNI 更强调“国民”而不是“地域”。
- GDP 看的是在哪里生产
- GNP / GNI 看的是谁拥有生产要素

### 增加值
增加值 = 总产出 - 中间投入。它是避免重复计算的关键概念。

## 2. 三种核算方法

### 支出法
GDP = C + I + G + NX

- C：消费
- I：投资
- G：政府购买
- NX：净出口

### 收入法
GDP = 工资 + 利息 + 租金 + 利润 + 间接税 + 折旧

### 生产法
GDP = 各部门增加值之和

## 3. 常见辨析题

### GDP 和 GNP 的区别
- GDP 关注地理范围
- GNP 关注国民归属
- 当本国居民在国外生产很多时，GNP 可能高于 GDP

### 最终产品与中间产品的区别
- 最终产品用于最终消费或投资
- 中间产品用于进一步生产
- GDP 只统计最终产品，避免重复计算

### 名义 GDP 和实际 GDP 的区别
- 名义 GDP 按当年价格计算
- 实际 GDP 按基期价格计算
- 实际 GDP 更适合衡量真实产出变化
