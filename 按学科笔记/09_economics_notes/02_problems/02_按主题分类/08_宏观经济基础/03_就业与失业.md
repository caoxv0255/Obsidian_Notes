# 第3章 就业与失业

## 本章目标

本章解决劳动力市场的基本测量问题，重点是失业率、就业率和自然失业率的概念，以及不同失业类型的经济含义。

### 学完本章你应该能够
- 计算失业率和就业率
- 区分摩擦性、结构性和周期性失业
- 解释自然失业率和 NAIRU
- 分析失业率变化背后的经济原因

## 1. 基础概念

### 劳动力人口
劳动力人口 = 就业人口 + 失业人口。

### 失业率
失业率 = 失业人口 / 劳动力人口。

### 就业率
就业率 = 就业人口 / 适龄人口。

## 2. 失业类型

### 摩擦性失业
求职匹配过程中的短期失业。

### 结构性失业
劳动者技能与岗位需求不匹配导致的失业。

### 周期性失业
总需求不足引起的失业，通常与经济衰退有关。

## 3. 自然失业率与 NAIRU

### 自然失业率
经济长期稳定状态下的失业率，包含摩擦性和结构性失业。

### NAIRU
不加速通胀失业率，强调失业与通胀稳定之间的关系。

## 4. 常见辨析题

### 失业率下降一定是好事吗？
不一定。可能存在劳动力退出、低质量就业增加或统计口径变化。

### 为什么结构性失业更难解决？
因为它涉及技能培训、产业转型和岗位匹配，不能只靠短期刺激解决。
