# 第1章 凯恩斯交叉模型

## 本章目标

凯恩斯交叉模型是短期宏观分析的起点，用来说明计划支出如何决定均衡产出，以及为什么财政支出、税收和投资变化会被乘数放大。

### 学完本章你应该能够
- 写出计划支出函数
- 说明 45 度线的经济含义
- 求解均衡产出
- 解释乘数效应的来源
- 分析政府支出变化带来的短期影响

## 1. 核心概念

### 计划支出
计划支出是家庭、企业、政府和国外部门计划购买的总量。

### 45 度线
45 度线表示计划支出等于实际产出，是寻找均衡点的基准线。

### 均衡条件
计划支出 = 实际产出。

## 2. 基本结构

常见表达式：

- 总需求 = C + I + G + NX
- 简单模型中可先忽略净出口，聚焦 C、I、G

### 消费函数
常写成：

C = a + bY

其中：
- a 是自主消费
- b 是边际消费倾向
- Y 是收入

## 3. 乘数效应

当政府支出或投资增加时，收入上升，消费进一步上升，从而使总产出被放大。

### 简单乘数
如果消费函数为 C = a + bY，则简单乘数近似为：

1 / (1 - b)

## 4. 常见例题

### 例题（详细）

### 例题 1 — 概念与机制（第1章 凯恩斯交叉模型）
问题：请定义“第1章 凯恩斯交叉模型”中的两个核心概念，并说明二者如何通过一个关键机制发生联系。

答案与解析：先给出概念的边界定义，再写出变量间的方向关系（正向/负向/非线性），最后给出一句可检验命题。

### 例题 2 — 计算与推导（第1章 凯恩斯交叉模型）
问题：围绕“第1章 凯恩斯交叉模型”构造一个可计算的小算例（给定参数、写出公式、计算结果），并解释结果的经济含义。

答案与解析：列出已知条件 -> 写核心公式 -> 逐步代入 -> 报告结果 -> 解释“结果为何成立”和“何时会失效”。

### 例题 3 — 图形分析（第1章 凯恩斯交叉模型）
问题：请画出与“第1章 凯恩斯交叉模型”最相关的分析图（曲线/边界/均衡点），并标注冲击前后关键点变化。

答案与解析：明确横纵轴含义，先标初始均衡，再加外生冲击，最后解释图形移动与变量变化的一致性。

### 例题 4 — 政策情景（第1章 凯恩斯交叉模型）
问题：针对“第1章 凯恩斯交叉模型”设计一个政策冲击（税收/补贴/规制/信息披露等），分析其对效率与分配的双重影响。

答案与解析：按“政策工具 -> 行为响应 -> 均衡变化 -> 福利分配”链条分析，并指出至少一条副作用路径。

### 例题 5 — 现实案例（第1章 凯恩斯交叉模型）
问题：选择一个现实案例，使用“第1章 凯恩斯交叉模型”的模型语言做结构化解释，并给出结论的边界条件。

答案与解析：先把文字案例映射为变量，再做机制推理，最后写出结论只在何种假设下成立。

### 例题 6 — 识别与验证（第1章 凯恩斯交叉模型）
问题：如果要验证“第1章 凯恩斯交叉模型”中的一个关键命题，你会如何设计经验检验（数据、识别策略、稳健性）？

答案与解析：明确被解释变量与核心解释变量，提出可执行识别策略（如对照、准实验、工具变量），并给至少两个稳健性检查。
