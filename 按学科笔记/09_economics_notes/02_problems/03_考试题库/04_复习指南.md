# 复习指南与重点总结

系统化复习策略与学期关键考点整理。

## 🗓️ 学期复习规划

### 第一轮：基础巩固（第1-4周）
**目标**：掌握基本概念与模型框架

**任务**：
- 逐章阅读讲义，整理关键概念
- 完成相应章节的初级练习题
- 制作概念卡片（重点概念、公式、常见错误）

**时间配置**：
- 讲义阅读：60%
- 概念整理：20%
- 基础练习：20%

**检验**：能否用自己的语言解释核心概念

---

### 第二轮：深化理解（第5-8周）
**目标**：理解模型与方法，进行跨章节应用

**任务**：
- 学习中级练习题与图形分析
- 进行小型实证项目（CPI分析等）
- 比较不同模型的应用场景

**时间配置**：
- 理论深化：40%
- 实证分析：30%
- 中级练习：30%

**检验**：能否用模型分析现实经济现象

---

### 第三轮：综合应用（第9-13周）
**目标**：整合知识体系，提高应考能力

**任务**：
- 进行完整的实证研究项目
- 做高级练习题与历年考题
- 进行知识点间的逻辑联系

**时间配置**：
- 综合练习：40%
- 实证项目：30%
- 错题分析：30%

**检验**：能否在有限时间内完成综合题

---

### 第四轮：冲刺突破（最后2周）
**目标**：消除知识死角，适应考试节奏

**任务**：
- 集中做历年真题或模拟题
- 重点关注高频考点与易错题
- 进行全真模拟考试

**时间配置**：
- 模拟考试：60%
- 查漏补缺：30%
- 答题技巧训练：10%

**检验**：模拟考试成绩与进步空间

---

## 📊 考点归纳

### 微观经济学核心考点

**高频考点**（出题率 > 70%）：
1. 供给需求与市场均衡
2. 消费者选择与需求函数
3. 企业成本与市场结构
4. 市场失灵与政府干预
5. 完全竞争市场的效率性

**中等频率考点**（出题率 40-70%）：
1. 收入与替代效应分解
2. 垄断与价格歧视
3. 外部性与公共物品
4. 一般均衡框架
5. 博弈论与寡头竞争

**低频但关键考点**（出题率 20-40%）：
1. 显示偏好理论
2. 要素市场与收入分配
3. 信息不对称问题
4. 福利经济学定理

---

### 宏观经济学核心考点

**高频考点**（出题率 > 70%）：
1. GDP核算与国民收入
2. 凯恩斯收入决定模型
3. 货币与通货膨胀
4. 财政与货币政策
5. 国际收支与汇率

**中等频率考点**（出题率 40-70%）：
1. IS-LM模型
2. AD-AS模型
3. 失业与通胀的权衡
4. 开放经济与蒙代尔-弗莱明模型
5. 经济增长理论

**低频但关键考点**（出题率 20-40%）：
1. 政策时滞与有效性
2. 商业周期理论
3. 预期与理性预期
4. 金融市场与资产定价

## 🎯 答题策略

### 名词解释 (5-10分)

**答题框架**：
1. **定义**（1句）- 简洁、准确的定义
2. **含义**（2-3句）- 解释为什么重要、反映什么现象
3. **应用或联系**（1-2句）- 与其他概念的关系或应用举例

**常见考点**：
- 机会成本、比较优势、帕累托有效
- 通货膨胀、失业率、自然失业率
- 垄断权力、价格歧视、市场失灵

---

### 简答题 (10-20分)

**答题步骤**：
1. **确认问题** - 题目问的具体是什么？
2. **提出框架** - 选用哪个模型？为什么？
3. **逻辑分析** - 按步骤推导/分析
4. **总结结论** - 最终答案是什么？

**常见陷阱**：
- 用错模型（如用IS-LM分析长期效应）
- 遗漏重要条件（如政策时滞、预期）
- 只有结论，没有过程

---

### 计算题 (15-20分)

**答题要点**：
1. **清晰列式** - 写出计算式，不要心算
2. **标注单位** - 人民币、百分比、万元等
3. **中间步骤** - 即使答案错，过程分能得多
4. **合理性检验** - 结果是否合理？（如失业率> 100%是错的）

**高频题型**：
- 均衡价格与产量计算
- 消费者剩余与生产者剩余
- CPI与通胀率计算
- 乘数与均衡产出

---

### 分析题 (20-30分)

**答题框架**（"分析法"）：
1. **现象描述** - 题目给出或可观察的事实
2. **理论分析** - 用经济学理论解释
3. **图形说明** - 画出相关经济学图形
4. **政策建议** - 应对或改进的办法
5. **可能影响** - 预期的后果与风险

**失分原因**：
- 只有观点，缺乏逻辑
- 图形不规范或标注不清
- 忽视时间因素（短期vs长期）
- 政策建议不切实际

## 📚 关键记忆点

### 必须背诵的公式

```
经济增长率 = [(GDP_t - GDP_{t-1}) / GDP_{t-1}] × 100%
CPI = Σ(某年价格×基期权重) / Σ(基期价格×基期权重) × 100
通胀率 = (CPI_t - CPI_{t-1}) / CPI_{t-1}

边际消费倾向 (MPC) = ΔC / ΔY
储蓄倾向 = 1 - MPC
乘数 = 1 / [1 - MPC(1-t) + m]   【m=边际进口倾向】

需求价格弹性 = (ΔQ/Q) / (ΔP/P) = (ΔQ·P) / (ΔP·Q)
消费者剩余 = ∫D(P)dP - P·Q
生产者剩余 = P·Q - ∫S(P)dP
```

### 关键定义与区分

| 概念对 | 区分要点 |
|-------|--------|
| 正常商品 vs 劣商品 | 收入弹性的正负 |
| 替代品 vs 互补品 | 交叉价格弹性的正负 |
| 完全竞争 vs 垄断 | 是否有市场势力 |
| 短期 vs 长期 | 生产要素是否完全可调整 |
| 流量 vs 存量 | 是否是时间段的概念 |
| 实际 vs 名义 | 是否排除了物价变化影响 |

## 💪 冲刺建议

### 考前一周
- 每天做2-3套模拟题
- 回顾易错题与重点考点
- 保证充足睡眠（8小时以上）

### 考试当天
- 提前15分钟进考场，熟悉环境
- 先做容易的题建立信心
- 合理分配时间（通常为：题目审读3分钟，作答85分钟，检查12分钟）
- 草稿纸整洁（便于检查）

### 常见失分原因与对策

| 失分原因 | 对策 |
|---------|------|
| 审题不清 | 重点圈出题目关键词，确认问的是什么 |
| 遗漏条件 | 重读题目，列出所有给定条件 |
| 计算错误 | 重新计算，数字代入检验 |
| 时间不够 | 放弃得分率<30%的题，优先完成其他 |
| 逻辑混乱 | 按分点答题，用子标题结构清晰 |
