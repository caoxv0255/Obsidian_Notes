## 1. 线性规划的标准形式

### 1.1 标准形式

$$\min c^T x$$
$$\text{s.t.} \quad Ax = b$$
$$\quad\quad x \geq 0$$

其中：
- c ∈ ℝⁿ 为目标函数系数
- A ∈ ℝᵐˣⁿ 为约束矩阵
- b ∈ ℝᵐ 为右端项（通常要求 b ≥ 0）
- x ∈ ℝⁿ 为决策变量

### 1.2 其他形式转化

**不等式约束**：引入松弛变量
$$a^T x \leq b \Rightarrow a^T x + s = b, s \geq 0$$

**自由变量**：分解为正负部分
$$x \in \mathbb{R} \Rightarrow x = x^+ - x^-, x^+, x^- \geq 0$$

**最大化问题**：
$$\max c^T x = -\min(-c)^T x$$

## 2. 几何基础

### 2.1 可行域的性质

**定理**：线性规划的可行域是凸多面体。

**极点（顶点）**：可行域的"角点"，不能表示为可行域内两点的凸组合。

**基本可行解**：对应极点的代数表示。

### 2.2 最优解的位置

**定理**：若线性规划有有限最优解，则必在某个极点处取得。

**意义**：只需检查极点（有限个），不必搜索整个可行域。

## 3. 基本可行解

### 3.1 定义

**基**：矩阵 A 的 m 个线性无关列，记为 B。

**基变量**：对应于基 B 的变量 x_B。

**非基变量**：其余 n-m 个变量 x_N，设为零。

**基本解**：
$$x_B = B^{-1}b, \quad x_N = 0$$

**基本可行解**：基本解满足 x ≥ 0。

### 3.2 基本可行解的性质

**定理**：基本可行解与可行域极点一一对应。

**定理**：若可行域非空，则存在基本可行解。

## 4. 单纯形法原理

### 4.1 基本思想

从一个基本可行解出发，通过**换基操作**转移到相邻的更好的基本可行解，直到找到最优解。

### 4.2 转移的代数描述

设当前基为 B，目标函数值：
$$z = c_B^T x_B + c_N^T x_N = c_B^T B^{-1}b + (c_N^T - c_B^T B^{-1}N)x_N$$

**检验数（Reduced Cost）**：
$$\bar{c}_N = c_N - N^T(B^{-1})^T c_B = c_N - N^T y$$

其中 y = (B⁻¹)ᵀc_B 为对偶变量。

**最优性条件**：若所有检验数 ≥ 0（min问题），则当前解最优。

### 4.3 选择进基变量

选择检验数最小（最负）的非基变量进基：
$$j^* = \arg\min_j \bar{c}_j$$

### 4.4 选择出基变量

**最小比值法则**：
$$i^* = \arg\min_{i: \bar{a}_{ij^*} > 0} \frac{\bar{b}_i}{\bar{a}_{ij^*}}$$

其中：
- ā = B⁻¹A
- b̄ = B⁻¹b

若所有 ā_{ij*} ≤ 0，问题无界。

## 5. 单纯形表

### 5.1 表格形式

| 基变量 | x₁ | x₂ | ... | xₙ | RHS |
|--------|----|----|-----|-----|------|
| z | -c₁ | -c₂ | ... | -cₙ | 0 |
| s₁ | a₁₁ | a₁₂ | ... | a₁ₙ | b₁ |
| ... | ... | ... | ... | ... | ... |
| sₘ | aₘ₁ | aₘ₂ | ... | aₘₙ | bₘ |

### 5.2 迭代过程

1. **选主元列**：检验数最负的列
2. **选主元行**：最小比值的行
3. **旋转运算**：将主元化为1，主元列其他位置化为0
4. **更新**：重复直到所有检验数 ≥ 0

### 5.3 表格迭代示例

```
问题：min z = -3x₁ - 2x₂
      s.t. x₁ + x₂ + s₁ = 4
           2x₁ + x₂ + s₂ = 6
           x₁, x₂, s₁, s₂ ≥ 0

初始表：
|   | x₁ | x₂ | s₁ | s₂ | RHS |
|---|----|----|----|----|-----|
| z | 3  | 2  | 0  | 0  | 0   |
| s₁| 1  | 1  | 1  | 0  | 4   |
| s₂| 2  | 1  | 0  | 1  | 6   |

x₁ 进基（检验数最大为3）
最小比值：min(4/1, 6/2) = 3，s₂ 出基

迭代后：
|   | x₁ | x₂ | s₁ | s₂ | RHS |
|---|----|----|----|----|-----|
| z | 0  | 0.5| 0  |-1.5| -9  |
| s₁| 0  | 0.5| 1  |-0.5| 1   |
| x₁| 1  | 0.5| 0  | 0.5| 3   |

x₂ 进基，s₁ 出基

最终表：
|   | x₁ | x₂ | s₁ | s₂ | RHS |
|---|----|----|----|----|-----|
| z | 0  | 0  | -1 | -1 | -10 |
| x₂| 0  | 1  | 2  |-1  | 2   |
| x₁| 1  | 0  |-1  | 1  | 2   |

最优解：x₁=2, x₂=2, z=-10
```

## 6. 改进单纯形法

### 6.1 基本思想

不存储整个单纯形表，只维护：
- 基矩阵的逆 B⁻¹
- 当前基变量值 x_B = B⁻¹b
- 对偶变量 y = (B⁻¹)ᵀc_B

### 6.2 计算步骤

```
算法：改进单纯形法
初始化：找初始基B，计算B⁻¹
while 存在负检验数:
    # 计算检验数
    y = (B⁻¹)ᵀc_B
    c̄_N = c_N - Nᵀy
    
    # 选择进基变量
    j* = argmin c̄_j
    
    # 计算进基列
    ā_{j*} = B⁻¹a_{j*}
    
    # 选择出基变量
    i* = argmin{b̄_i/ā_{ij*} : ā_{ij*} > 0}
    
    # 更新基
    更新B⁻¹（使用修正公式）
```

### 6.3 基逆的更新

**修正公式**：
$$B_{new}^{-1} = E B^{-1}$$

其中 E 为初等矩阵，仅在第 i* 列与单位矩阵不同。

## 7. 两阶段法与大M法

### 7.1 初始基本可行解的获取

**问题**：标准单纯形法需要初始基本可行解，但通常不显然。

### 7.2 两阶段法

**第一阶段**：求解人工问题
$$\min \sum_{i=1}^{m} a_i$$
$$\text{s.t.} \quad Ax + Ia = b, \quad x, a \geq 0$$

若最优值为0，得到原问题的初始基本可行解。

**第二阶段**：从初始解出发，求解原问题。

### 7.3 大M法

在目标函数中对人工变量施加大惩罚：
$$\min c^T x + M \sum_{i=1}^{m} a_i$$

其中 M 为很大的正数。

## 8. 单纯形法的复杂性

### 8.1 最坏情况

**Klee-Minty例子**：存在 n 维问题，需要遍历 2ⁿ 个顶点。

**结论**：单纯形法不是多项式时间算法。

### 8.2 实际表现

**经验观察**：大多数实际问题迭代次数为 O(m) 到 O(3m)。

**平均情况**：在概率模型下是多项式时间的。

## 9. 对偶单纯形法

### 9.1 对偶问题的单纯形法

**对偶问题**：
$$\max b^T y$$
$$\text{s.t.} \quad A^T y \leq c$$

**对偶单纯形法**：保持对偶可行性（检验数 ≥ 0），逐步满足原始可行性。

### 9.2 应用场景

- 灵敏度分析
- 整数规划的割平面法
- 原问题无可行解时

## 10. Python实现

```python
import numpy as np

def simplex(c, A, b, max_iter=1000):
    """
    单纯形法求解标准型线性规划
    min c'x, s.t. Ax = b, x >= 0
    假设已有初始基本可行解
    """
    m, n = A.shape
    
    # 找初始基（简化：假设单位矩阵存在）
    basis = list(range(n-m, n))
    B = A[:, basis]
    
    for iteration in range(max_iter):
        # 计算基变量值
        try:
            x_B = np.linalg.solve(B, b)
        except np.linalg.LinAlgError:
            return None, None, "奇异矩阵"
        
        if np.any(x_B < -1e-10):
            return None, None, "无可行解"
        
        # 计算对偶变量和检验数
        c_B = c[basis]
        try:
            y = np.linalg.solve(B.T, c_B)
        except np.linalg.LinAlgError:
            return None, None, "奇异矩阵"
        
        reduced_costs = c - A.T @ y
        
        # 找进基变量
        non_basis = [j for j in range(n) if j not in basis]
        min_rc_idx = np.argmin(reduced_costs[non_basis])
        
        if reduced_costs[non_basis[min_rc_idx]] >= -1e-10:
            # 最优
            x = np.zeros(n)
            x[basis] = x_B
            return x, c @ x, "最优"
        
        j_enter = non_basis[min_rc_idx]
        
        # 计算方向
        d = np.linalg.solve(B, A[:, j_enter])
        
        # 找出基变量（最小比值）
        ratios = []
        for i in range(m):
            if d[i] > 1e-10:
                ratios.append(x_B[i] / d[i])
            else:
                ratios.append(np.inf)
        
        if all(r == np.inf for r in ratios):
            return None, None, "无界"
        
        i_leave = np.argmin(ratios)
        j_leave = basis[i_leave]
        
        # 更新基
        basis[i_leave] = j_enter
        B = A[:, basis]
    
    return None, None, "迭代超限"

# 示例
if __name__ == "__main__":
    # min -3x1 - 2x2, s.t. x1+x2<=4, 2x1+x2<=6, x1,x2>=0
    # 标准化：min -3x1 -2x2, s.t. x1+x2+s1=4, 2x1+x2+s2=6
    c = np.array([-3, -2, 0, 0])
    A = np.array([[1, 1, 1, 0], [2, 1, 0, 1]])
    b = np.array([4, 6])
    
    x, obj, status = simplex(c, A, b)
    print(f"状态: {status}")
    print(f"解: {x}")
    print(f"目标值: {obj}")
```

## 11. 总结

| 特点 | 说明 |
|------|------|
| 复杂度 | 最坏指数，实际多项式 |
| 适用 | 中小规模线性规划 |
| 优势 | 直观，可用热启动 |
| 劣势 | 大规模问题不如内点法 |

**与其他方法比较**：
- **内点法**：大规模问题更快
- **对偶单纯形法**：灵敏度分析更方便
- **修正单纯形法**：稀疏问题内存效率高
