# 10 - 线性规划

## 1. 线性规划问题

### 1.1 标准形式

**线性规划（LP）**：目标函数和约束都是线性的优化问题。

**标准形式**：
$$\min_x c^T x$$
$$\text{s.t.} \quad Ax = b$$
$$\quad\quad\; x \geq 0$$

其中：
- c ∈ ℝⁿ：成本向量
- A ∈ ℝᵐˣⁿ：约束矩阵
- b ∈ ℝᵐ：右端向量
- x ∈ ℝⁿ：决策变量

### 1.2 其他形式的转化

**不等式约束**：
- g(x) ≤ h ⟺ g(x) + s = h, s ≥ 0（松弛变量 s）

**自由变量**：
- x 自由 ⟺ x = x⁺ - x⁻, x⁺, x⁻ ≥ 0

**最大化问题**：
- max cᵀx = -min (-c)ᵀx

### 1.3 几何性质

**可行域**：
- 多面体（由超平面围成的凸多面体）
- 顶点（极点）是关键
- 最优解必在顶点处（若存在）

**定理**：若线性规划问题有最优解，则必在可行域的某个顶点处取得。

## 2. 对偶理论

### 2.1 拉格朗日对偶

**原问题（P）**：
$$\min_x c^T x$$
$$\text{s.t.} \quad Ax = b$$
$$\quad\quad\; x \geq 0$$

**拉格朗日函数**：
$$L(x, \lambda, \nu) = c^T x - \nu^T (Ax - b) - \lambda^T x$$
$$= (c - A^T \nu - \lambda)^T x + b^T \nu$$

**对偶函数**：
$$g(\nu, \lambda) = \inf_x L(x, \nu, \lambda) = \begin{cases}
b^T \nu, & c - A^T \nu - \lambda = 0 \\
-\infty, & \text{其他}
\end{cases}$$

**对偶问题（D）**：
$$\max_\nu b^T \nu$$
$$\text{s.t.} \quad A^T \nu \leq c$$

### 2.2 对偶变量

**对偶变量 ν 的意义**：
- 代表资源的"价格"或"影子价格"
- 对偶问题的最优解 = 原问题约束的边际价值

**经济解释**：
- 原问题：生产成本最小化
- 对偶问题：资源价值最大化

### 2.3 弱对偶定理

**定理**：设 x 为原问题可行解，ν 为对偶问题可行解，则：
$$c^T x \geq b^T \nu$$

**证明**：
$$c^T x - b^T \nu = c^T x - \nu^T b = c^T x - \nu^T Ax$$
$$= x^T c - x^T A^T \nu = x^T (c - A^T \nu) \geq 0$$

（因为 x ≥ 0，c - Aᵀν ≥ 0）

**意义**：
- 对偶目标值 ≤ 原目标值
- 对偶问题提供下界

### 2.4 强对偶定理

**定理**：若原问题或对偶问题有有限最优解，则：
$$\min c^T x = \max b^T \nu$$

**意义**：
- 强对偶成立
- 对偶间隙为 0
- 可通过对偶问题求解原问题

### 2.5 互补松弛条件

**定理**：设 x* 和 ν* 分别为原问题和对偶问题的最优解，则：
$$x_i^* (c - A^T \nu^*)_i = 0, \quad i = 1, \ldots, n$$

即：
$$x_i^* > 0 \Rightarrow (A^T \nu^*)_i = c_i$$
$$(A^T \nu^*)_i < c_i \Rightarrow x_i^* = 0$$

**经济解释**：
- 若资源完全利用（xᵢ > 0），则边际价值等于成本
- 若边际价值小于成本，则资源不被利用（xᵢ = 0）

## 3. 单纯形法

### 3.1 基本思想

**关键观察**：
- 最优解必在顶点处
- 顶点由基变量和非基变量确定
- 迭代：从一个顶点移动到相邻顶点

### 3.2 基本可行解

**定义**：
- 设 A 的秩为 m，选 m 个线性独立的列构成基矩阵 B
- 剩余 n-m 列构成非基矩阵 N
- 对应变量分为基变量 x_B 和非基变量 x_N

**基本解**：x_N = 0，x_B = B⁻¹b

**基本可行解**：基本解且 x_B ≥ 0

**定理**：基本可行解 ⟺ 可行域的顶点

### 3.3 单纯形算法

**算法框架**：
```
初始化：找到一个基本可行解
while 未达到最优:
    1. 选择入基变量（目标函数下降方向）
    2. 选择出基变量（保持可行性）
    3. 执行基变换
    4. 更新解
```

**检验数**：检验非基变量是否能使目标函数下降。

### 3.4 收敛性

**定理**：若存在最优解，单纯形法在有限步内终止。

**退化问题**：
- 多个基本可行解对应同一顶点
- 可能循环
- 解决：Bland规则（最小下标规则）

## 4. 敏感性分析

### 4.1 目标函数系数变化

**问题**：成本系数 c 变化多少，最优解不变？

**方法**：
- 最优基不变 ⟺ 检验数不变号
- 计算允许变化范围

### 4.2 右端向量变化

**问题**：右端向量 b 变化多少，最优基不变？

**方法**：
- 基本可行解要求 x_B = B⁻¹(b + Δb) ≥ 0
- 计算允许变化范围

### 4.3 影子价格

**定义**：约束右端增加一个单位时，目标函数的变化量。

**关系**：影子价格 = 对偶变量的最优值

## 5. 应用实例

### 5.1 生产计划问题

**问题**：某工厂生产两种产品，资源有限，求最优生产方案。

**数据**：
- 产品1：利润 3 元/单位，消耗资源 A 1 单位，资源 B 2 单位
- 产品2：利润 5 元/单位，消耗资源 A 2 单位，资源 B 1 单位
- 资源 A 总量：8 单位
- 资源 B 总量：6 单位

**模型**：
$$\max \; 3x_1 + 5x_2$$
$$\text{s.t.} \quad x_1 + 2x_2 \leq 8$$
$$\quad\quad\; 2x_1 + x_2 \leq 6$$
$$\quad\quad\; x_1, x_2 \geq 0$$

**标准形式**：
$$\min \; -3x_1 - 5x_2$$
$$\text{s.t.} \quad x_1 + 2x_2 + s_1 = 8$$
$$\quad\quad\; 2x_1 + x_2 + s_2 = 6$$
$$\quad\quad\; x_1, x_2, s_1, s_2 \geq 0$$

### 5.2 运输问题

**问题**：将货物从多个产地运往多个销地，最小化运输成本。

**模型**：
$$\min \sum_{i=1}^m \sum_{j=1}^n c_{ij} x_{ij}$$
$$\text{s.t.} \quad \sum_{j=1}^n x_{ij} = a_i, \quad i = 1, \ldots, m$$
$$\quad\quad\; \sum_{i=1}^m x_{ij} = b_j, \quad j = 1, \ldots, n$$
$$\quad\quad\; x_{ij} \geq 0$$

**特点**：
- 总供给 = 总需求时可行
- 特殊结构，有专门算法

### 5.3 指派问题

**问题**：n 项任务分配给 n 个人，每人一项，最小化总成本。

**模型**：
$$\min \sum_{i=1}^n \sum_{j=1}^n c_{ij} x_{ij}$$
$$\text{s.t.} \quad \sum_{j=1}^n x_{ij} = 1, \quad i = 1, \ldots, n$$
$$\quad\quad\; \sum_{i=1}^n x_{ij} = 1, \quad j = 1, \ldots, n$$
$$\quad\quad\; x_{ij} \in \{0, 1\}$$

**松弛**：xᵢⱼ ≥ 0，则松弛问题的最优解自动为 0-1 解。

### 5.4 网络流问题

**最大流问题**：
$$\max \; f$$
$$\text{s.t.} \quad \sum_{j:(i,j) \in E} x_{ij} - \sum_{j:(j,i) \in E} x_{ji} = \begin{cases} f, & i = s \\ -f, & i = t \\ 0, & \text{其他} \end{cases}$$
$$\quad\quad\; 0 \leq x_{ij} \leq c_{ij}$$

## 6. 数值实验

### 6.1 Python实现

```python
import numpy as np
from scipy.optimize import linprog

# 定义问题
c = np.array([-3, -5])  # 取负号转为最小化
A_ub = np.array([[1, 2], [2, 1]])
b_ub = np.array([8, 6])
bounds = [(0, None), (0, None)]

# 求解
result = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=bounds, method='highs')

print(f"最优解: {result.x}")
print(f"最优值: {-result.fun}")  # 取负号还原
```

## 7. 线性规划的对偶形式总结

### 7.1 标准对偶关系

**原问题**：
$$\min c^T x \quad \text{s.t.} \quad Ax = b, \quad x \geq 0$$

**对偶问题**：
$$\max b^T \nu \quad \text{s.t.} \quad A^T \nu \leq c$$

### 7.2 一般形式的对偶

| 原问题 | 对偶问题 |
|--------|----------|
| min cᵀx | max bᵀν |
| Aᵢx = bᵢ | νᵢ 自由 |
| Aᵢx ≤ bᵢ | νᵢ ≥ 0 |
| Aᵢx ≥ bᵢ | νᵢ ≤ 0 |
| xⱼ ≥ 0 | (Aᵀν)ⱼ ≤ cⱼ |
| xⱼ ≤ 0 | (Aᵀν)ⱼ ≥ cⱼ |
| xⱼ 自由 | (Aᵀν)ⱼ = cⱼ |

## 8. 小结

### 8.1 核心概念

| 概念 | 定义 | 性质 |
|------|------|------|
| 基本可行解 | x_N = 0, x_B = B⁻¹b ≥ 0 | 对应顶点 |
| 对偶问题 | 原问题的对偶 | 强对偶性 |
| 影子价格 | 对偶最优解 | 边际价值 |

### 8.2 重要定理

1. **顶点定理**：最优解必在顶点
2. **强对偶定理**：min = max
3. **互补松弛**：xᵢ(c - Aᵀν)ᵢ = 0

### 8.3 求解方法

- **单纯形法**：经典算法，实践中高效
- **内点法**：多项式时间，适合大规模问题
- **商业求解器**：Gurobi, CPLEX, MOSEK
