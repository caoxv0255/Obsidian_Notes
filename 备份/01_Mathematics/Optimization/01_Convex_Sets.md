# 01 - 凸集

## 📌 学习目标

- 能判断集合是否凸（用定义/保凸运算）
- 会写出常见凸集（半空间、多面体、球/椭球、范数球/范数锥、PSD 锥）的“为什么凸”
- 理解分离超平面、支撑超平面、对偶锥在后续对偶/KKT中的作用

## 难度分层

- **基础**：凸集/仿射集定义 + 典型例子 + 保凸运算
- **进阶**：分离/支撑超平面定理 + 对偶锥性质
- **拓展**：用分离定理推导 Farkas 引理/强对偶的直觉

## 1. 凸集的基本概念

### 1.1 仿射集

**定义**：若集合 C ⊆ ℝⁿ 中任意两点的连线仍在 C 中，即对任意 x₁, x₂ ∈ C 和任意 θ ∈ ℝ，有：
$$\theta x_1 + (1-\theta) x_2 \in C$$

则称 C 为**仿射集**（affine set）。

**等价定义**：C 是仿射集 ⟺ 对任意 k 个点 x₁, ..., xₖ ∈ C 和任意满足 θ₁ + ... + θₖ = 1 的 θᵢ ∈ ℝ，有：
$$\theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_k x_k \in C$$

**仿射组合**：
$$\theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_k x_k, \quad \text{其中} \sum_{i=1}^{k} \theta_i = 1$$

**仿射包**：集合 S 的所有仿射组合构成的集合，记为 aff(S)：
$$\text{aff}(S) = \left\{ \sum_{i=1}^{k} \theta_i x_i : k \geq 1, x_i \in S, \sum_{i=1}^{k} \theta_i = 1 \right\}$$

**例**：
- 直线、平面、超平面
- 线性方程组的解集 {x : Ax = b}
- 单点集

### 1.2 凸集的定义

**定义**：若集合 C ⊆ ℝⁿ 中任意两点的**线段**仍在 C 中，即对任意 x₁, x₂ ∈ C 和任意 θ ∈ [0, 1]，有：
$$\theta x_1 + (1-\theta) x_2 \in C$$

则称 C 为**凸集**（convex set）。

**凸组合**：
$$\theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_k x_k, \quad \text{其中} \theta_i \geq 0, \sum_{i=1}^{k} \theta_i = 1$$

**凸包**：集合 S 的所有凸组合构成的集合，记为 conv(S)：
$$\text{conv}(S) = \left\{ \sum_{i=1}^{k} \theta_i x_i : k \geq 1, x_i \in S, \theta_i \geq 0, \sum_{i=1}^{k} \theta_i = 1 \right\}$$

**几何意义**：凸包是用凸多面体"包裹"集合 S 的最小凸集。

**关系**：仿射集 ⟹ 凸集（反之不成立）

### 1.3 凸集的例子

**凸集**：
1. **超平面**：{x : aᵀx = b}
2. **半空间**：{x : aᵀx ≤ b}
3. **多面体**：{x : Ax ≤ b}
4. **球**：{x : ‖x - x₀‖₂ ≤ r}
5. **椭球**：{x : (x-x₀)ᵀP⁻¹(x-x₀) ≤ 1}，P 正定
6. **范数球**：{x : ‖x‖ ≤ 1}
7. **锥**：{θx : θ ≥ 0, x ∈ C}，其中 C 为凸集

**非凸集**：
1. 两个不相交集合的并
2. 有"凹痕"的集合
3. 某些非凸区域

**具体例子**：
- 单位球盘 $\{x \in \mathbb{R}^2 : \|x\|_2 \leq 1\}$ 是凸集，因为任意两点的线段仍留在圆盘内。
- 圆环 $\{x \in \mathbb{R}^2 : 1 \leq \|x\|_2 \leq 2\}$ 不是凸集，因为取内外两点，其连线可能穿过中间空洞。
- 二个分离的圆盘并集不是凸集，因为跨圆盘的线段会离开集合。

### 1.4 凸锥

**定义**：若集合 C 满足：
1. 凸性：对任意 x₁, x₂ ∈ C 和 θ₁, θ₂ ≥ 0，有 θ₁x₁ + θ₂x₂ ∈ C
2. 锥性：对任意 x ∈ C 和 θ ≥ 0，有 θx ∈ C

则称 C 为**凸锥**。

**锥组合**：
$$\theta_1 x_1 + \theta_2 x_2 + \cdots + \theta_k x_k, \quad \text{其中} \theta_i \geq 0$$

**锥包**：
$$\left\{ \sum_{i=1}^{k} \theta_i x_i : k \geq 1, x_i \in S, \theta_i \geq 0 \right\}$$

**例**：
1. 非负象限：ℝ₊ⁿ = {x : x ≥ 0}
2. 范数锥：{(x, t) : ‖x‖₂ ≤ t, x ∈ ℝⁿ, t ∈ ℝ}
3. 半正定锥：S₊ⁿ = {X ∈ Sⁿ : X ≽ 0}

## 2. 凸集的重要例子

### 2.1 多面体

**定义**：线性不等式和等式约束的解集：
$$\mathcal{P} = \{x : a_i^T x \leq b_i, i = 1, \ldots, m, \; c_j^T x = d_j, j = 1, \ldots, p\}$$

等价地：
$$\mathcal{P} = \{x : Ax \preceq b, \; Cx = d\}$$

**性质**：
- 多面体是凸集
- 多面体的交集仍是多面体
- 仿射映射保持多面体性质

**特殊情况**：
- **单纯形**：{x : x ≥ 0, 1ᵀx = 1}
- **盒子**：{x : l ≤ x ≤ u}

### 2.2 半正定锥

**定义**：
$$S_+^n = \{X \in S^n : X \succeq 0\} = \{X \in S^n : z^T X z \geq 0, \forall z \in \mathbb{R}^n\}$$

其中 Sⁿ 表示 n × n 对称矩阵的集合。

**性质**：
1. S₊ⁿ 是凸锥
2. X ≽ 0 ⟺ X 的所有特征值非负
3. X ≽ 0 ⟺ X = LLᵀ（Cholesky分解）

**严格半正定锥**：
$$S_{++}^n = \{X \in S^n : X \succ 0\} = \{X \in S^n : z^T X z > 0, \forall z \neq 0\}$$

### 2.3 范数球与范数锥

**范数球**：
$$B_p = \{x \in \mathbb{R}^n : \|x\|_p \leq 1\}$$

**常用范数**：
- l₁ 范数：‖x‖₁ = Σ|xᵢ|
- l₂ 范数：‖x‖₂ = √(Σxᵢ²)
- l∞ 范数：‖x‖∞ = max|xᵢ|

**范数锥**：
$$K_p = \{(x, t) \in \mathbb{R}^{n+1} : \|x\|_p \leq t\}$$

### 2.4 保凸运算

**定理**：以下运算保持集合的凸性：

1. **交集**：若 C₁, C₂ 为凸集，则 C₁ ∩ C₂ 为凸集
   - 推广：任意多个凸集的交集仍为凸集

2. **仿射变换**：若 C 为凸集，A ∈ ℝᵐˣⁿ，b ∈ ℝᵐ，则
   $$f(C) = \{Ax + b : x \in C\}$$
   为凸集

3. **线性分式变换**：若 C 为凸集，则
   $$f(C) = \left\{ \frac{Ax + b}{c^T x + d} : x \in C, c^T x + d > 0 \right\}$$
   为凸集

4. **透视函数**：若 C ⊆ ℝⁿˣ⁽¹⁾ 为凸集，则
   $$P(C) = \left\{ \left(\frac{x}{t}, \frac{1}{t}\right) : (x, t) \in C, t > 0 \right\}$$
   为凸集

**例**：椭球
$$\mathcal{E} = \{x : (x-x_c)^T P^{-1} (x-x_c) \leq 1\}$$

是单位球的仿射变换：
$$\mathcal{E} = \{P^{1/2} u + x_c : \|u\|_2 \leq 1\}$$

## 3. 分离超平面定理

### 3.1 超平面分离

**定理**（分离超平面定理）：设 C 和 D 为 ℝⁿ 中的两个不相交凸集，则存在 a ≠ 0 和 b 使得：
$$a^T x \leq b, \quad \forall x \in C$$
$$a^T x \geq b, \quad \forall x \in D$$

即超平面 {x : aᵀx = b} 分离 C 和 D。

**几何意义**：两个不相交凸集可以用一个超平面"隔开"。

### 3.2 严格分离

**定理**（严格分离定理）：设 C 为闭凸集，D = {x₀} 为单点集，若 x₀ ∉ C，则存在 a ≠ 0 和 b 使得：
$$a^T x < b, \quad \forall x \in C$$
$$a^T x_0 > b$$

**证明思路**：取 C 中距离 x₀ 最近的点 x*，则超平面 a = x₀ - x*, b = aᵀx* + aᵀa/2 严格分离 C 和 x₀。

### 3.3 支撑超平面

**定义**：设 C 为凸集，x₀ 为 C 的边界点。若超平面 {x : aᵀx = aᵀx₀} 满足：
$$a^T x \leq a^T x_0, \quad \forall x \in C$$

则称其为 C 在 x₀ 处的**支撑超平面**。

**定理**（支撑超平面定理）：凸集边界的每一点都存在支撑超平面。

**应用**：构造对偶问题、最优性条件证明。

### 3.4 应用实例

**例1**（Farkas引理）：设 A ∈ ℝᵐˣⁿ, b ∈ ℝᵐ，则以下恰有一者成立：
1. 存在 x ≥ 0 使得 Ax = b
2. 存在 y 使得 Aᵀy ≥ 0 且 bᵀy < 0

**证明**：利用凸锥分离定理。

**例2**（强对偶性）：在某些条件下，凸优化问题满足强对偶性，即原问题与对偶问题的最优值相等。分离超平面定理是其证明的关键工具。

**例3**（二维分离超平面）
设
$$C = \{x \in \mathbb{R}^2 : \|x\|_2 \leq 1\}, \qquad D = \{(2,0)\}$$
则超平面
$$x_1 = 1$$
把 C 与 D 分开，因为对任意 $x \in C$ 都有 $x_1 \leq 1$，而点 $(2,0)$ 满足 $x_1 = 2 > 1$。

这个例子说明：分离超平面不只是抽象定理，也可以直接在几何图形上看出来。

## 4. 对偶锥

### 4.1 对偶锥的定义

**定义**：锥 K 的**对偶锥**定义为：
$$K^* = \{y : x^T y \geq 0, \forall x \in K\}$$

**性质**：
1. K* 是闭凸锥
2. K ⊆ K**（双对偶锥）
3. 若 K 为闭凸锥，则 K** = K

### 4.2 对偶锥的例子

1. **非负象限的对偶**：(ℝ₊ⁿ)* = ℝ₊ⁿ

2. **半正定锥的对偶**：(S₊ⁿ)* = S₊ⁿ

3. **范数锥的对偶**：l₂ 范数锥的对偶锥是其自身

4. **锥的组合**：
   - (K₁ ∩ K₂)* = K₁* + K₂*
   - (K₁ + K₂)* = K₁* ∩ K₂*

### 4.3 广义不等式

**定义**：设 K 为闭凸锥且具有内点（正规锥），定义**广义不等式**：
$$x \preceq_K y \iff y - x \in K$$
$$x \prec_K y \iff y - x \in \text{int}(K)$$

**性质**：
1. 自反性：x ⪯_K x
2. 反对称性：x ⪯_K y, y ⪯_K x ⟹ x = y
3. 传递性：x ⪯_K y, y ⪯_K z ⟹ x ⪯_K z
4. 加法保序性：x ⪯_K y ⟹ x + z ⪯_K y + z
5. 非负数乘保序性：x ⪯_K y, α ≥ 0 ⟹ αx ⪯_K αy

**例**：
- K = ℝ₊ⁿ：x ⪯_K y ⟺ x ≤ y（分量）
- K = S₊ⁿ：X ⪯_K Y ⟺ Y - X ≽ 0（矩阵）

## 5. 几何性质

### 5.1 内点与相对内部

**定义**：集合 C 的**相对内部**定义为：
$$\text{relint}(C) = \{x \in C : \exists r > 0, B(x, r) \cap \text{aff}(C) \subseteq C\}$$

**相对边界**：
$$\text{relbd}(C) = \text{cl}(C) \setminus \text{relint}(C)$$

**例**：
- 单位圆盘在 ℝ² 中的内点：开圆盘
- 单位圆盘在 ℝ³ 中的相对内部：开圆盘（没有内点）

### 5.2 凸集的几何特征

**定理**：凸集具有以下几何特征：

1. **连通性**：凸集是道路连通的

2. **连通分量**：凸集只有一个连通分量

### 5.3 Carathéodory定理

**定理**（Carathéodory）：设 S ⊆ ℝⁿ，则 S 的凸包中任意一点可以表示为 S 中至多 n + 1 个点的凸组合。

**证明要点**：利用线性相关性证明。

**推论**：若 S 是紧集，则 conv(S) 也是紧集。

## 6. 应用

### 6.1 凸优化中的凸集

**可行域**：凸优化问题的可行域是凸集（由凸约束定义）。

**水平集**：凸函数的水平集 {x : f(x) ≤ α} 是凸集。

**定义域**：凸函数的定义域是凸集。

### 6.2 几何概率

**例**：在凸集 C 中随机选取两点，求两点连线仍在 C 中的概率。

**解**：凸集定义保证概率为 1。

### 6.3 控制理论

**例**（可达集）：线性系统 x(t+1) = Ax(t) + Bu(t), ‖u‖ ≤ 1 的可达集是凸集。

## 7. 小结

### 7.1 核心概念

| 概念 | 定义 | 特点 |
|------|------|------|
| 凸集 | 线段在集合内 | 连通、有良好的几何性质 |
| 仿射集 | 直线在集合内 | 更强的条件 |
| 凸锥 | 非负线性组合封闭 | 无界、包含原点 |
| 对偶锥 | 正交于原锥的半空间 | 凸、闭 |

### 7.2 重要定理

1. **分离超平面定理**：不相交凸集可分离
2. **支撑超平面定理**：边界点存在支撑超平面
3. **Carathéodory定理**：凸组合的点数上界

### 7.3 凸集的判断

**直接判断**：
- 检查定义：线段是否在集合内
- 检查约束：是否由凸函数定义

**间接判断**：
- 保凸运算：交集、仿射变换等
- 已知凸集的组合
