# 14 - 数学期望

## ✅ 先修

- [[../00_Symbols_Conventions|符号与约定总表]]

## 📌 学习目标

- 掌握离散/连续情形下期望的定义与存在条件（绝对可积）
- 熟练使用线性性、独立时乘积期望、全期望公式
- 能用“期望是最小均方误差预测”解释条件期望的意义

## 难度分层

- **基础**：定义 + 常见分布期望表 + 线性性
- **进阶**：条件期望与全期望公式 + 期望变换技巧
- **拓展**：矩母函数/特征函数视角与推断应用

## 1. 数学期望的定义

### 1.1 离散型随机变量的期望

**定义**：设离散型随机变量 X 的分布律为 P(X = xₖ) = pₖ，若级数 Σ|xₖ|pₖ 收敛，则 X 的**数学期望**（均值）定义为：
$$E[X] = \sum_{k=1}^{\infty} x_k p_k$$

**说明**：
- 要求级数**绝对收敛**，否则期望可能不存在
- 期望是随机变量的"中心位置"
- 反映随机变量取值的平均水平

**例**：掷骰子，X 为点数，E[X] = 1×(1/6) + 2×(1/6) + ... + 6×(1/6) = 3.5

### 1.2 连续型随机变量的期望

**定义**：设连续型随机变量 X 的概率密度函数为 f(x)，若积分 ∫|x|f(x)dx 收敛，则 X 的数学期望定义为：
$$E[X] = \int_{-\infty}^{+\infty} x f(x) dx$$

**例**：X ~ U(a, b)，E[X] = ∫ₐᵇ x/(b-a) dx = (a+b)/2

### 1.3 期望的物理意义

**质心类比**：
- 概率分布 ↔ 质量分布
- 期望 ↔ 质心位置
- 方差 ↔ 转动惯量

**直观理解**：
- 期望是随机变量取值的"加权平均"
- 权重为概率
- 反映分布的"中心趋势"

## 2. 期望的性质

### 2.1 基本性质

**定理**：数学期望具有以下基本性质：

1. **常数性质**：E[c] = c（c 为常数）

2. **线性性质**：E[aX + bY] = aE[X] + bE[Y]
   
   **证明**：
   $$E[aX + bY] = \sum (ax_i + by_i) p(x_i, y_i) = a\sum x_i p(x_i) + b\sum y_i p(y_i) = aE[X] + bE[Y]$$

3. **非负性**：若 X ≥ 0，则 E[X] ≥ 0

4. **单调性**：若 X₁ ≤ X₂，则 E[X₁] ≤ E[X₂]

5. **保序性**：|E[X]| ≤ E[|X|]

### 2.2 独立随机变量的期望

**定理**：若 X, Y 独立，且 E[|XY|] < ∞，则：
$$E[XY] = E[X] E[Y]$$

**证明**：
$$E[XY] = \iint xy f_X(x) f_Y(y) dx dy = \int x f_X(x) dx \cdot \int y f_Y(y) dy = E[X] E[Y]$$

**注意**：
- 逆命题不成立：E[XY] = E[X]E[Y] 不保证 X, Y 独立
- 例如：X 对称分布，Y = X²

### 2.3 随机变量函数的期望

**定理**（期望的变量替换）：设 Y = g(X)，则：
- **离散型**：E[Y] = Σ g(xₖ)pₖ
- **连续型**：E[Y] = ∫ g(x)f(x)dx

**重要**：不需要先求 Y 的分布！

**例**：X ~ U(0, 1)，求 E[X²]。

**解**：E[X²] = ∫₀¹ x² dx = 1/3

**对比**：若先求 Y = X² 的分布，则 Y ~ 某分布，再求 E[Y]，更复杂。

### 2.4 多元函数的期望

**定理**：设 Z = g(X₁, ..., Xₙ)，则：
$$E[Z] = \int \cdots \int g(x_1, \ldots, x_n) f(x_1, \ldots, x_n) dx_1 \cdots dx_n$$

## 3. 条件期望

### 3.1 条件期望的定义

**定义**：设 X, Y 为随机变量，在 Y = y 条件下 X 的**条件期望**定义为：
$$E[X|Y = y] = \begin{cases}
\sum_x x P(X = x|Y = y), & \text{离散型} \\
\int x f_{X|Y}(x|y) dx, & \text{连续型}
\end{cases}$$

**条件期望函数**：E[X|Y] = φ(Y)，其中 φ(y) = E[X|Y = y]

### 3.2 条件期望的性质

**定理**：
1. E[E[X|Y]] = E[X]（全期望公式）
2. E[Xg(Y)|Y] = g(Y)E[X|Y]
3. E[g(Y)|Y] = g(Y)
4. 若 X, Y 独立，则 E[X|Y] = E[X]

**全期望公式证明**：
$$E[E[X|Y]] = \int E[X|Y=y] f_Y(y) dy = \int \int x f_{X|Y}(x|y) dx f_Y(y) dy$$
$$= \int x \int f_{X|Y}(x|y) f_Y(y) dy dx = \int x f_X(x) dx = E[X]$$

### 3.3 条件期望的应用

**预测问题**：给定 Y，预测 X 的最优值？

**定理**：E[X|Y] 是给定 Y 时 X 的**最小均方误差预测**。

**证明**：设 g(Y) 为任意函数，则：
$$E[(X - g(Y))^2] = E[E[(X - g(Y))^2|Y]]$$

对固定的 Y = y：
$$E[(X - g(y))^2|Y = y] \geq E[(X - E[X|Y = y])^2|Y = y]$$

等号成立 ⟺ g(y) = E[X|Y = y]。

**意义**：条件期望是最优预测器。

### 3.4 全期望公式的应用

**例**：一天内到达商店的顾客数 N ~ P(λ)，每位顾客消费金额 Yᵢ i.i.d.，求一天总消费 S = Y₁ + ... + Yₙ 的期望。

**解**：由全期望公式：
$$E[S] = E[E[S|N]] = E[N \cdot E[Y_1]] = E[N] \cdot E[Y_1] = \lambda E[Y_1]$$

**复合泊松分布**：S 的分布称为复合泊松分布。

## 4. 常见分布的期望

### 4.1 离散分布

| 分布 | 期望 | 推导 |
|------|------|------|
| Bernoulli(p) | p | 0×q + 1×p = p |
| Binomial(n, p) | np | np（可加性） |
| Poisson(λ) | λ | λe^{-λ} Σkλ^{k-1}/k! = λ |
| Geometric(p) | 1/p | Σk p(1-p)^{k-1} = 1/p |
| NB(r, p) | r/p | r/p（可加性） |

### 4.2 连续分布

| 分布 | 期望 | 推导 |
|------|------|------|
| Uniform(a, b) | (a+b)/2 | ∫x dx/(b-a) |
| Exponential(λ) | 1/λ | ∫xλe^{-λx}dx = 1/λ |
| Normal(μ, σ²) | μ | 对称性 |
| Gamma(α, β) | α/β | 由矩母函数 |
| Beta(α, β) | α/(α+β) | 由Beta函数性质 |

### 4.3 期望的计算技巧

**技巧1**：利用对称性
- 正态分布 N(μ, σ²)：E[X] = μ（由对称性）
- 均匀分布 U(-a, a)：E[X] = 0

**技巧2**：变量替换
- 例：X ~ Exp(λ)，求 E[e^X]
- 解：E[e^X] = ∫₀^∞ e^x λe^{-λx} dx = λ/(λ-1)（λ > 1）

**技巧3**：利用矩母函数
- φ(t) = E[e^{tX}]
- E[X] = φ'(0)
- E[X^n] = φ^{(n)}(0)

**技巧4**：分部积分
- 例：X ~ Exp(λ)，求 E[X]
- 解：E[X] = ∫₀^∞ xλe^{-λx}dx = [−xe^{-λx}]₀^∞ + ∫₀^∞ e^{-λx}dx = 1/λ

## 5. 矩母函数

### 5.1 矩母函数的定义

**定义**：随机变量 X 的**矩母函数**（moment generating function）定义为：
$$M_X(t) = E[e^{tX}] = \begin{cases}
\sum_k e^{t x_k} p_k, & \text{离散型} \\
\int e^{tx} f(x) dx, & \text{连续型}
\end{cases}$$

**注意**：
- 矩母函数不一定对所有 t 都存在
- 需要在 t = 0 的某邻域内存在

### 5.2 矩母函数的性质

**定理**：
1. M(0) = 1
2. M^{(n)}(0) = E[X^n]（n 阶矩）
3. 若 X, Y 独立，则 M_{X+Y}(t) = M_X(t) M_Y(t)
4. M_{aX+b}(t) = e^{bt} M_X(at)

### 5.3 常见分布的矩母函数

| 分布 | 矩母函数 |
|------|----------|
| Bernoulli(p) | q + pe^t |
| Binomial(n, p) | (q + pe^t)^n |
| Poisson(λ) | exp(λ(e^t - 1)) |
| Uniform(a, b) | (e^{tb} - e^{ta})/(t(b-a)) |
| Exponential(λ) | λ/(λ-t), t < λ |
| Normal(μ, σ²) | exp(μt + σ²t²/2) |
| Gamma(α, β) | (1 - t/β)^{-α}, t < β |

### 5.4 应用

**例**：X ~ P(λ₁)，Y ~ P(λ₂)，独立，求 X + Y 的分布。

**解**：
$$M_{X+Y}(t) = M_X(t) M_Y(t) = e^{\lambda_1(e^t-1)} \cdot e^{\lambda_2(e^t-1)} = e^{(\lambda_1+\lambda_2)(e^t-1)}$$

这是 P(λ₁ + λ₂) 的矩母函数，故 X + Y ~ P(λ₁ + λ₂)。

## 6. 期望的几何解释

### 6.1 重心

**类比**：概率分布可看作质量分布，期望是重心。

**例**：离散分布 P(X = xᵢ) = pᵢ
- 想象在位置 xᵢ 放置质量为 pᵢ 的质点
- 重心位置 = Σxᵢpᵢ = E[X]

### 6.2 势能最小化

**定理**：E[(X - c)²] 在 c = E[X] 时取得最小值。

**证明**：
$$E[(X-c)^2] = E[X^2] - 2cE[X] + c^2$$

对 c 求导：-2E[X] + 2c = 0 ⟹ c = E[X]

**意义**：期望是使均方误差最小的点。

## 7. 期望的应用

### 7.1 决策论

**期望效用**：选择使期望效用最大的决策。

**例**：投资选择
- 方案A：确定性收益 100
- 方案B：50%收益200，50%收益0，期望 = 100

若效用函数 u(x) = x，则期望效用相同。
若 $u(x)=\sqrt{x}$（风险厌恶），则：
- 方案A：$u(100)=10$
- 方案B：$\mathbb{E}[u]=0.5\times\sqrt{200}+0.5\times0\approx 7.07$
- 因为 10 > 7.07，风险厌恶者更偏好方案A

**结论**：即使两个方案的期望收益相同，风险偏好不同也会导致不同选择。

### 7.1.1 期望作为“最小均方误差”

若要用一个常数 $c$ 去预测随机变量 $X$，并以均方误差 $\mathbb{E}[(X-c)^2]$ 为损失，则最优选择是 $c=\mathbb{E}[X]$。

这说明期望不仅是“平均值”，还是平方损失下的最佳预测点。

## 8. 小结

### 8.1 核心概念

| 概念 | 定义 | 性质 |
|------|------|------|
| 期望 | E[X] = Σxₖpₖ 或 ∫xf(x)dx | 线性、单调 |
| 条件期望 | E[X|Y] | 全期望公式 |
| 矩母函数 | M(t) = E[e^{tX}] | 唯一性、乘积性 |

### 8.2 重要公式

1. **线性性质**：E[aX + bY] = aE[X] + bE[Y]
2. **独立乘积**：E[XY] = E[X]E[Y]（X, Y 独立）
3. **全期望公式**：E[E[X|Y]] = E[X]
4. **变量替换**：E[g(X)] = ∫g(x)f(x)dx

### 8.3 应用领域

- **决策论**：期望效用
- **金融**：期望收益、风险度量
- **统计**：参数估计（矩估计）
- **机器学习**：损失函数的期望
