# 17 - 矩与特征函数

## 1. 矩的概念

### 1.1 原点矩

**定义**：设 X 为随机变量，k 阶**原点矩**定义为：
$$\mu_k' = E[X^k]$$

**离散型**：$\mu_k' = \sum_i x_i^k p_i$

**连续型**：$\mu_k' = \int_{-\infty}^{+\infty} x^k f(x) dx$

### 1.2 中心矩

**定义**：k 阶**中心矩**定义为：
$$\mu_k = E[(X - E[X])^k]$$

**重要关系**：
- μ₁ = 0
- μ₂ = Var(X) = E[X²] - (E[X])²
- μ₃ = E[X³] - 3E[X]E[X²] + 2(E[X])³

### 1.3 矩与数字特征

| 矩 | 名称 | 意义 |
|----|------|------|
| μ₁' | 期望 | 位置 |
| μ₂ | 方差 | 离散度 |
| μ₃/σ³ | 偏度 | 对称性 |
| μ₄/σ⁴ | 峰度 | 尾部厚度 |

## 2. 矩母函数

### 2.1 定义

**定义**：随机变量 X 的**矩母函数**（MGF）定义为：
$$M_X(t) = E[e^{tX}]$$

**离散型**：$M_X(t) = \sum_i e^{tx_i} p_i$

**连续型**：$M_X(t) = \int_{-\infty}^{+\infty} e^{tx} f(x) dx$

### 2.2 矩的计算

**定理**：若 M_X(t) 在 t = 0 的某邻域内存在，则：
$$E[X^k] = M_X^{(k)}(0) = \left.\frac{d^k M_X(t)}{dt^k}\right|_{t=0}$$

### 2.3 性质

1. M(0) = 1
2. M_{aX+b}(t) = e^{bt} M_X(at)
3. 若 X, Y 独立，M_{X+Y}(t) = M_X(t)M_Y(t)
4. M_X(t) 唯一确定分布（若存在）

### 2.4 常见分布的矩母函数

| 分布 | 矩母函数 |
|------|----------|
| Bernoulli(p) | q + pe^t |
| B(n, p) | (q + pe^t)^n |
| P(λ) | exp(λ(e^t - 1)) |
| N(μ, σ²) | exp(μt + σ²t²/2) |
| Exp(λ) | λ/(λ-t), t < λ |
| Gamma(α, β) | (1-t/β)^{-α}, t < β |

## 3. 特征函数

### 3.1 定义

**定义**：随机变量 X 的**特征函数**定义为：
$$\varphi_X(t) = E[e^{itX}] = E[\cos(tX) + i\sin(tX)]$$

### 3.2 性质

1. φ(0) = 1
2. |φ(t)| ≤ 1
3. φ(-t) = φ̄(t)（共轭）
4. φ 一致连续
5. φ(t) 总是存在

### 3.3 矩的计算

$$E[X^k] = \frac{1}{i^k} \varphi^{(k)}(0)$$

（若 k 阶矩存在）

### 3.4 常见分布的特征函数

| 分布 | 特征函数 |
|------|----------|
| Bernoulli(p) | q + pe^{it} |
| B(n, p) | (q + pe^{it})^n |
| P(λ) | exp(λ(e^{it} - 1)) |
| N(μ, σ²) | exp(iμt - σ²t²/2) |
| Exp(λ) | λ/(λ-it) |
| Gamma(α, β) | (1-it/β)^{-α} |

## 4. 特征函数的重要定理

### 4.1 唯一性定理

**定理**：特征函数唯一确定分布。

**意义**：
- 若 φ₁(t) = φ₂(t)，则对应分布相同
- 证明分布相同 ⟺ 特征函数相同

### 4.2 连续性定理（Lévy）

**定理**：设 Xₙ 和 X 的特征函数分别为 φₙ(t) 和 φ(t)，则：
$$X_n \xrightarrow{d} X \iff \varphi_n(t) \to \varphi(t)$$

**应用**：证明中心极限定理的关键工具。

### 4.3 独立和的特征函数

**定理**：若 X, Y 独立，则：
$$\varphi_{X+Y}(t) = \varphi_X(t) \varphi_Y(t)$$

**推论**：若 X₁, ..., Xₙ i.i.d.，则：
$$\varphi_{S_n}(t) = [\varphi_{X_1}(t)]^n$$

## 5. 半不变量

### 5.1 定义

**定义**：矩母函数的对数称为**累积量母函数**：
$$K_X(t) = \ln M_X(t)$$

k 阶**半不变量**（累积量）定义为：
$$\kappa_k = \left.\frac{d^k K_X(t)}{dt^k}\right|_{t=0}$$

### 5.2 与矩的关系

- κ₁ = μ₁' = E[X]
- κ₂ = μ₂ = Var(X)
- κ₃ = μ₃
- κ₄ = μ₄ - 3μ₂²

### 5.3 性质

**可加性**：若 X, Y 独立，则：
$$\kappa_k(X+Y) = \kappa_k(X) + \kappa_k(Y)$$

**正态分布**：
- κ₁ = μ, κ₂ = σ²
- κ₃ = κ₄ = ... = 0

## 6. 矩生成函数的应用

### 6.1 证明分布性质

**例**：证明正态分布的可加性。

**证明**：设 X ~ N(μ₁, σ₁²), Y ~ N(μ₂, σ₂²)，独立。

$$M_{X+Y}(t) = M_X(t) M_Y(t) = e^{\mu_1 t + \sigma_1^2 t^2/2} \cdot e^{\mu_2 t + \sigma_2^2 t^2/2}$$
$$= e^{(\mu_1+\mu_2)t + (\sigma_1^2+\sigma_2^2)t^2/2}$$

这是 N(μ₁+μ₂, σ₁²+σ₂²) 的矩母函数。

### 6.2 计算期望和方差

**例**：X ~ P(λ)，求 E[X] 和 Var(X)。

**解**：M(t) = exp(λ(e^t - 1))
$$M'(t) = \lambda e^t M(t)$$
$$M'(0) = \lambda$$
$$E[X] = \lambda$$

$$M''(t) = (\lambda e^t + \lambda^2 e^{2t}) M(t)$$
$$M''(0) = \lambda + \lambda^2$$
$$E[X^2] = \lambda + \lambda^2$$
$$Var(X) = E[X^2] - (E[X])^2 = \lambda$$

## 7. 小结

### 7.1 核心概念

| 概念 | 定义 | 性质 |
|------|------|------|
| 矩母函数 | M(t) = E[e^{tX}] | 可能不存在 |
| 特征函数 | φ(t) = E[e^{itX}] | 总是存在 |
| 半不变量 | ln M(t) 的导数 | 可加性 |

### 7.2 重要定理

- **唯一性**：特征函数唯一确定分布
- **连续性**：特征函数收敛 ⟺ 分布收敛
- **独立性**：独立和的特征函数 = 特征函数之积
