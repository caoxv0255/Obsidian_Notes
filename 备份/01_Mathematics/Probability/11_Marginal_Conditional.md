## 1. 边缘分布

### 1.1 边缘分布函数

**定义**：设 (X, Y) 的联合分布函数为 F(x, y)，则：

**X 的边缘分布函数**：
$$F_X(x) = F(x, +\infty) = \lim_{y \to +\infty} F(x, y) = P(X \leq x)$$

**Y 的边缘分布函数**：
$$F_Y(y) = F(+\infty, y) = \lim_{x \to +\infty} F(x, y) = P(Y \leq y)$$

**解释**：边缘分布是联合分布的"投影"，只关注单个随机变量的分布。

### 1.2 离散型的边缘分布律

**定义**：设 (X, Y) 为二维离散型随机变量，联合分布律为 p_{ij}。

**X 的边缘分布律**：
$$P(X = x_i) = p_{i\cdot} = \sum_{j} p_{ij}$$

**Y 的边缘分布律**：
$$P(Y = y_j) = p_{\cdot j} = \sum_{i} p_{ij}$$

**计算方法**：将联合分布表的行（列）求和。

**示例**：

| X\Y | 1 | 2 | 3 | P(X=xᵢ) |
|-----|---|---|---|---------|
| 1 | 0.1 | 0.2 | 0.1 | 0.4 |
| 2 | 0.2 | 0.1 | 0.3 | 0.6 |
| P(Y=yⱼ) | 0.3 | 0.3 | 0.4 | 1.0 |

- X 的边缘分布：P(X=1)=0.4, P(X=2)=0.6
- Y 的边缘分布：P(Y=1)=0.3, P(Y=2)=0.3, P(Y=3)=0.4

### 1.3 连续型的边缘概率密度

**定义**：设 (X, Y) 为二维连续型随机变量，联合概率密度为 f(x, y)。

**X 的边缘概率密度**：
$$f_X(x) = \int_{-\infty}^{+\infty} f(x, y) dy$$

**Y 的边缘概率密度**：
$$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y) dx$$

**几何意义**：联合密度函数沿某轴"积分掉"另一变量。

**示例**：设联合密度为：
$$f(x, y) = \begin{cases} 2, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$$

X 的边缘密度：
$$f_X(x) = \int_x^1 2 dy = 2(1-x), \quad 0 < x < 1$$

Y 的边缘密度：
$$f_Y(y) = \int_0^y 2 dx = 2y, \quad 0 < y < 1$$

### 1.4 边缘分布与联合分布的关系

**关键点**：
- 联合分布 → 边缘分布：可确定
- 边缘分布 → 联合分布：**不能确定**

**反例**：不同的联合分布可能有相同的边缘分布。

## 2. 条件分布

### 2.1 离散型的条件分布律

**定义**：设 P(Y = y_j) > 0，则给定 Y = y_j 条件下 X 的条件分布律为：
$$P(X = x_i | Y = y_j) = \frac{P(X = x_i, Y = y_j)}{P(Y = y_j)} = \frac{p_{ij}}{p_{\cdot j}}$$

**同理**：
$$P(Y = y_j | X = x_i) = \frac{p_{ij}}{p_{i\cdot}}$$

**性质**：
1. P(X = xᵢ | Y = yⱼ) ≥ 0
2. $\sum_i P(X = x_i | Y = y_j) = 1$

**示例**：接上表，给定 Y=1 时 X 的条件分布：
- P(X=1|Y=1) = 0.1/0.3 = 1/3
- P(X=2|Y=1) = 0.2/0.3 = 2/3

### 2.2 连续型的条件概率密度

**定义**：设 f_Y(y) > 0，则给定 Y = y 条件下 X 的条件概率密度为：
$$f_{X|Y}(x|y) = \frac{f(x, y)}{f_Y(y)}$$

**同理**：
$$f_{Y|X}(y|x) = \frac{f(x, y)}{f_X(x)}$$

**条件分布函数**：
$$F_{X|Y}(x|y) = P(X \leq x | Y = y) = \int_{-\infty}^x f_{X|Y}(u|y) du$$

### 2.3 条件密度的计算步骤

1. 计算联合密度 f(x, y)
2. 计算边缘密度 f_Y(y)
3. 相除得到条件密度 f_{X|Y}(x|y)
4. 确定条件密度的定义域

**示例**：设联合密度 f(x, y) = 2，0 < x < y < 1。

给定 Y = y（0 < y < 1）时 X 的条件密度：
$$f_{X|Y}(x|y) = \frac{2}{2y} = \frac{1}{y}, \quad 0 < x < y$$

即 X|Y=y 服从 (0, y) 上的均匀分布。

## 3. 多维随机变量的边缘分布和条件分布

### 3.1 三维情况

设 (X₁, X₂, X₃) 的联合密度为 f(x₁, x₂, x₃)。

**边缘分布**：
- f_{X₁}(x₁) = ∫∫ f(x₁, x₂, x₃) dx₂ dx₃
- f_{X₁,X₂}(x₁, x₂) = ∫ f(x₁, x₂, x₃) dx₃

**条件分布**：
- f_{X₁|X₂,X₃}(x₁|x₂, x₃) = f(x₁, x₂, x₃)/f_{X₂,X₃}(x₂, x₃)
- f_{X₁,X₂|X₃}(x₁, x₂|x₃) = f(x₁, x₂, x₃)/f_{X₃}(x₃)

### 3.2 一般n维情况

**边缘密度**：对 n-k 个变量积分得到 k 维边缘密度。

**条件密度**：联合密度除以条件变量的边缘密度。

## 4. 条件期望

### 4.1 定义

**离散型**：给定 Y = yⱼ 条件下 X 的条件期望：
$$E[X|Y = y_j] = \sum_i x_i P(X = x_i | Y = y_j)$$

**连续型**：给定 Y = y 条件下 X 的条件期望：
$$E[X|Y = y] = \int_{-\infty}^{+\infty} x f_{X|Y}(x|y) dx$$

### 4.2 条件期望的性质

**性质1**：E[X|Y] 是 Y 的函数，仍为随机变量。

**性质2（全期望公式）**：
$$E[X] = E[E[X|Y]]$$

**性质3**：
$$E[g(X)|Y] = \int g(x) f_{X|Y}(x|Y) dx$$

**性质4**：若 X, Y 独立，则 E[X|Y] = E[X]。

### 4.3 条件方差

**定义**：
$$\text{Var}(X|Y) = E[(X - E[X|Y])^2|Y] = E[X^2|Y] - (E[X|Y])^2$$

**方差分解公式**：
$$\text{Var}(X) = E[\text{Var}(X|Y)] + \text{Var}(E[X|Y])$$

**解释**：
- E[Var(X|Y)]：组内方差的期望
- Var(E[X|Y])：组间方差

## 5. 条件分布在统计中的应用

### 5.1 贝叶斯推断

**后验分布**：
$$P(\theta|X) = \frac{P(X|\theta)P(\theta)}{P(X)} \propto P(X|\theta)P(\theta)$$

这是给定数据 X 条件下参数 θ 的条件分布。

### 5.2 回归分析

**回归函数**：E[Y|X = x] 描述 Y 对 X 的回归关系。

**最优预测**：条件期望是给定 X 时 Y 的最优预测（最小均方误差）。

## 6. 计算实例

### 6.1 二维正态分布的边缘分布和条件分布

**联合密度**：
$$f(x, y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left(-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right)$$

**边缘分布**：
- X ~ N(μ₁, σ₁²)
- Y ~ N(μ₂, σ₂²)

**条件分布**：
- X|Y=y ~ N(μ₁ + ρσ₁/σ₂(y - μ₂), σ₁²(1-ρ²))
- Y|X=x ~ N(μ₂ + ρσ₂/σ₁(x - μ₁), σ₂²(1-ρ²))

### 6.2 指数分布的条件分布

设 X, Y 独立同分布 Exp(λ)，令 U = X + Y, V = X/(X+Y)。

则：
- U ~ Gamma(2, λ)
- V ~ Uniform(0, 1)
- U, V 独立

## 7. Python计算

```python
import numpy as np
from scipy import stats

# 离散型边缘分布和条件分布
joint_pmf = np.array([[0.1, 0.2, 0.1],
                      [0.2, 0.1, 0.3]])

# 边缘分布
p_x = joint_pmf.sum(axis=1)  # [0.4, 0.6]
p_y = joint_pmf.sum(axis=0)  # [0.3, 0.3, 0.4]

print("X的边缘分布:", p_x)
print("Y的边缘分布:", p_y)

# 条件分布 P(X|Y=1)
p_x_given_y1 = joint_pmf[:, 0] / p_y[0]
print("P(X|Y=1):", p_x_given_y1)

# 二维正态分布的条件分布
mu = [0, 0]
cov = [[1, 0.5], [0.5, 1]]  # 相关系数 rho = 0.5

# 给定 Y=1 时 X 的条件分布
y_given = 1
mu_x_given_y = mu[0] + cov[0][1]/cov[1][1] * (y_given - mu[1])
var_x_given_y = cov[0][0] - cov[0][1]**2 / cov[1][1]

print(f"X|Y=1 ~ N({mu_x_given_y}, {var_x_given_y})")
```

## 8. 总结

| 概念 | 离散型 | 连续型 |
|------|--------|--------|
| 边缘分布 | 对联合分布求和 | 对联合密度积分 |
| 条件分布 | p_{ij}/p_{·j} | f(x,y)/f_Y(y) |
| 条件期望 | ΣxᵢP(X=xᵢ\|Y=yⱼ) | ∫xf_{X\|Y}(x\|y)dx |

**重要公式**：
- 全期望公式：E[X] = E[E[X|Y]]
- 方差分解：Var(X) = E[Var(X|Y)] + Var(E[X|Y])
