# 09 - 常见概率分布族

## 1. 指数分布族

### 1.1 定义

**定义**：若随机变量 X 的 PDF/PMF 可表示为：
$$f(x|\theta) = h(x) \exp(\eta(\theta)^T T(x) - A(\theta))$$

则称 X 服从**指数分布族**，其中：
- θ：自然参数
- η(θ)：自然参数映射
- T(x)：充分统计量
- h(x)：基础测度
- A(θ)：归一化常数（对数配分函数）

### 1.2 常见指数分布族成员

**伯努利分布**：
$$f(x|p) = p^x(1-p)^{1-x} = \exp\left(x \ln\frac{p}{1-p} + \ln(1-p)\right)$$

**二项分布**：B(n, p)
**泊松分布**：P(λ)
**正态分布**：N(μ, σ²)
**指数分布**：Exp(λ)
**伽马分布**：Gamma(α, β)
**贝塔分布**：Beta(α, β)

### 1.3 性质

**充分统计量**：T(X) 包含了关于 θ 的所有信息。

**共轭先验**：指数分布族的共轭先验仍在同一族内。

**极大似然估计**：
$$\nabla A(\theta) = E[T(X)]$$

## 2. 皮尔逊分布族

### 2.1 定义

**皮尔逊分布族**：由微分方程定义的分布族：
$$\frac{d \ln f(x)}{dx} = \frac{x - a}{b_0 + b_1 x + b_2 x^2}$$

### 2.2 分类

根据分母根的情况，分为：
- **I型**：贝塔分布
- **III型**：伽马分布
- **IV型**：学生t分布的变形
- **V型**：逆伽马分布
- **VI型**：F分布
- **VII型**：学生t分布
- **正态分布**：特殊情况

## 3. 位置-尺度分布族

### 3.1 定义

**位置-尺度族**：设 Z 为标准随机变量，则 X = μ + σZ 构成的分布族。

### 3.2 性质

- E[X] = μ + σE[Z]
- Var(X) = σ²Var(Z)
- F_X(x) = F_Z((x-μ)/σ)

### 3.3 例子

**正态分布族**：N(μ, σ²) = μ + σN(0, 1)

**均匀分布族**：U(a, b) = (a+b)/2 + (b-a)/2 × U(-1, 1)

**指数分布族**：Exp(λ) = (1/λ)Exp(1)

**逻辑斯谛分布**：Logistic(μ, σ) = μ + σLogistic(0, 1)

## 4. 稳定分布族

### 4.1 定义

**稳定分布**：若 X₁, X₂ i.i.d.，则对任意常数 a, b，存在 c, d 使得：
$$aX_1 + bX_2 \overset{d}{=} cX + d$$

### 4.2 特征函数

$$\varphi(t) = \exp\left(i\delta t - \gamma^\alpha |t|^\alpha (1 - i\beta \text{sign}(t) \Phi(t,\alpha))\right)$$

其中 α ∈ (0, 2] 为稳定指数。

### 4.3 特例

- **α = 2**：正态分布
- **α = 1, β = 0**：Cauchy分布
- **α = 1/2, β = 1**：Lévy分布

### 4.4 性质

- 稳定分布的和仍是稳定分布
- 中心极限定理的推广
- 金融中的重尾建模

## 5. 重尾分布族

### 5.1 重尾的定义

**定义**：若 X 的尾部概率比指数分布衰减更慢：
$$\lim_{x \to \infty} \frac{P(X > x)}{e^{-\lambda x}} = \infty, \quad \forall \lambda > 0$$

则称 X 具有重尾。

### 5.2 幂律分布

**Pareto分布**：
$$f(x) = \frac{\alpha x_m^\alpha}{x^{\alpha+1}}, \quad x \geq x_m$$

**性质**：
- P(X > x) = (x_m/x)^α
- 若 α ≤ 2，方差无限
- 若 α ≤ 1，期望无限

### 5.3 应用

- 收入分布
- 城市规模
- 网络度分布
- 金融风险

## 6. 截断分布族

### 6.1 定义

**截断分布**：将原分布限制在某区间 [a, b] 上，重新归一化。

$$f_{\text{trunc}}(x) = \frac{f(x)}{F(b) - F(a)}, \quad a \leq x \leq b$$

### 6.2 例：截断正态分布

$$f(x) = \frac{\phi((x-\mu)/\sigma)}{\sigma(\Phi((b-\mu)/\sigma) - \Phi((a-\mu)/\sigma))}$$

## 7. 混合分布族

### 7.1 定义

**混合分布**：
$$f(x) = \sum_{i=1}^{k} \pi_i f_i(x)$$

其中 πᵢ ≥ 0, Σπᵢ = 1。

### 7.2 高斯混合模型（GMM）

$$f(x) = \sum_{i=1}^{k} \pi_i \mathcal{N}(x|\mu_i, \sigma_i^2)$$

**应用**：
- 聚类分析
- 密度估计
- 语音识别

### 7.3 EM算法

**期望最大化**：求解混合模型的参数估计。

**步骤**：
1. **E步**：计算后验概率
2. **M步**：更新参数

## 8. 球面分布族

### 8.1 定义

**球面分布**：若 X 的分布只依赖于 ‖X‖，则称 X 服从球面分布。

### 8.2 性质

- 球对称性
- 多元正态分布的推广
- 椭球分布的特例

## 9. 椭球分布族

### 9.1 定义

**椭球分布**：X ~ ECₙ(μ, Σ, ψ)，其特征函数为：
$$\varphi(t) = e^{i\mu^T t} \psi(t^T \Sigma t)$$

### 9.2 性质

- E[X] = μ（若存在）
- Cov(X) = -2ψ'(0)Σ（若存在）

### 9.3 例子

- 多元正态分布
- 多元t分布
- 多元Cauchy分布

## 10. 小结

### 10.1 分布族分类

| 分布族 | 特点 | 代表分布 |
|--------|------|----------|
| 指数族 | 统一形式 | 正态、泊松、伽马 |
| 位置-尺度 | X = μ + σZ | 正态、均匀 |
| 稳定分布 | 线性组合稳定 | 正态、Cauchy |
| 重尾分布 | 尾部概率大 | Pareto、t分布 |

### 10.2 重要概念

- **充分统计量**：包含所有信息
- **共轭先验**：后验与先验同族
- **稳定性**：线性组合的封闭性
