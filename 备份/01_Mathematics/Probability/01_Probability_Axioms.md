## 1. 概率论发展史

### 1.1 早期发展
- **起源**：17世纪，帕斯卡与费马的通信（赌博问题）
- **古典概型**：拉普拉斯的古典概率定义
- **几何概型**：布丰投针问题

### 1.2 公理化体系
- **Kolmogorov公理**（1933）：概率论的数学基础
- **测度论基础**：概率论与测度论的结合
- **现代概率论**：基于测度论的严格公理化体系

## 2. 随机试验与样本空间

### 2.1 随机试验
**定义**：满足以下三个条件的试验称为随机试验：
1. 可以在相同条件下重复进行
2. 所有可能结果事先已知
3. 每次试验的结果事先不能确定

**示例**：
- 抛硬币：{正面, 反面}
- 掷骰子：{1, 2, 3, 4, 5, 6}
- 测量灯泡寿命：(0, +∞)

### 2.2 样本空间
**定义**：随机试验的所有可能结果组成的集合称为样本空间，记为 Ω。

**分类**：
1. **有限样本空间**：Ω = {ω₁, ω₂, ..., ωₙ}
   - 例：掷骰子 Ω = {1, 2, 3, 4, 5, 6}

2. **可数无限样本空间**：Ω = {ω₁, ω₂, ...}
   - 例：抛硬币直到出现正面 Ω = {1, 2, 3, ...}

3. **不可数样本空间**：Ω 为不可数集合
   - 例：测量寿命 Ω = (0, +∞)

### 2.3 样本点与事件
- **样本点**：样本空间的元素，记为 ω
- **事件**：样本空间的子集，记为 A, B, C 等
- **基本事件**：只包含一个样本点的事件
- **必然事件**：样本空间 Ω 本身
- **不可能事件**：空集 ∅

## 3. 事件的关系与运算

### 3.1 事件的关系

设 A, B 为两个事件：

| 关系 | 符号表示 | 含义 | 概率意义 |
|------|----------|------|----------|
| 包含 | A ⊂ B | A 发生蕴含 B 发生 | P(A) ≤ P(B) |
| 相等 | A = B | A ⊂ B 且 B ⊂ A | P(A) = P(B) |
| 互斥 | AB = ∅ | A 与 B 不能同时发生 | P(AB) = 0 |
| 对立 | B = Aᶜ | A 与 B 互斥且并集为 Ω | P(A) + P(B) = 1 |

### 3.2 事件的运算

**基本运算**：
1. **并**：A ∪ B = {ω : ω ∈ A 或 ω ∈ B}
2. **交**：A ∩ B = AB = {ω : ω ∈ A 且 ω ∈ B}
3. **差**：A - B = A\B = {ω : ω ∈ A 且 ω ∉ B}
4. **补**：Aᶜ = Ω\A = {ω : ω ∉ A}

**运算律**：
1. **交换律**：A ∪ B = B ∪ A，AB = BA
2. **结合律**：(A ∪ B) ∪ C = A ∪ (B ∪ C)
3. **分配律**：A(B ∪ C) = AB ∪ AC
4. **De Morgan律**：
   - (A ∪ B)ᶜ = AᶜBᶜ
   - (AB)ᶜ = Aᶜ ∪ Bᶜ

**De Morgan律的推广**：
$$\left(\bigcup_{i=1}^{n} A_i\right)^c = \bigcap_{i=1}^{n} A_i^c$$
$$\left(\bigcap_{i=1}^{n} A_i\right)^c = \bigcup_{i=1}^{n} A_i^c$$

### 3.3 完备事件组
**定义**：若事件 A₁, A₂, ..., Aₙ 满足：
1. 两两互斥：AᵢAⱼ = ∅ (i ≠ j)
2. 并集为 Ω：⋃ᵢAᵢ = Ω

则称 A₁, A₂, ..., Aₙ 为样本空间的一个**完备事件组**（或**分割**）。

## 4. 概率的公理化定义

### 4.1 σ-代数
**定义**：设 Ω 为样本空间，ℱ 为 Ω 的某些子集组成的集合，若满足：
1. Ω ∈ ℱ
2. 若 A ∈ ℱ，则 Aᶜ ∈ ℱ（对补封闭）
3. 若 Aₙ ∈ ℱ (n = 1, 2, ...)，则 ⋃ₙAₙ ∈ ℱ（对可数并封闭）

则称 ℱ 为 Ω 上的一个**σ-代数**。

**性质**：
- ∅ ∈ ℱ
- 若 Aₙ ∈ ℱ (n = 1, 2, ...)，则 ⋂ₙAₙ ∈ ℱ
- ℱ 对有限并、有限交、差运算封闭

**例**：
1. **平凡σ-代数**：ℱ = {∅, Ω}
2. **最大σ-代数**：ℱ = 2^Ω（Ω 的所有子集）
3. **Borel σ-代数**：ℝ 上由开集生成的σ-代数，记为 ℬ(ℝ)

### 4.2 Kolmogorov公理

**定义**：设 Ω 为样本空间，ℱ 为 Ω 上的σ-代数。若映射 P: ℱ → [0, 1] 满足：
1. **非负性**：P(A) ≥ 0，∀ A ∈ ℱ
2. **规范性**：P(Ω) = 1
3. **可列可加性**：若 A₁, A₂, ... 两两互斥，则
   $$P\left(\bigcup_{n=1}^{\infty} A_n\right) = \sum_{n=1}^{\infty} P(A_n)$$

则称 P 为 **概率测度**，三元组 (Ω, ℱ, P) 称为**概率空间**。

### 4.3 概率的基本性质

**定理**：概率 P 具有以下基本性质：

1. **不可能事件的概率**：P(∅) = 0

2. **有限可加性**：若 A₁, A₂, ..., Aₙ 两两互斥，则
   $$P\left(\bigcup_{i=1}^{n} A_i\right) = \sum_{i=1}^{n} P(A_i)$$

3. **对立事件的概率**：P(Aᶜ) = 1 - P(A)

4. **包含关系**：若 A ⊂ B，则 P(A) ≤ P(B)，且 P(B - A) = P(B) - P(A)

5. **单调性**：若 A ⊂ B，则 P(A) ≤ P(B)

6. **加法公式**：
   $$P(A \cup B) = P(A) + P(B) - P(AB)$$

**证明**（加法公式）：
$$A \cup B = A \cup (B - A)$$
由于 A 与 B - A 互斥，故：
$$P(A \cup B) = P(A) + P(B - A) = P(A) + P(B) - P(AB)$$

### 4.4 概率的连续性

**定理**（概率的连续性）：
1. **下连续**：若 A₁ ⊂ A₂ ⊂ ...，则
   $$P\left(\bigcup_{n=1}^{\infty} A_n\right) = \lim_{n \to \infty} P(A_n)$$

2. **上连续**：若 A₁ ⊃ A₂ ⊃ ...，则
   $$P\left(\bigcap_{n=1}^{\infty} A_n\right) = \lim_{n \to \infty} P(A_n)$$

## 5. 古典概型

### 5.1 古典概型的定义
**定义**：若随机试验满足：
1. **有限性**：样本空间 Ω 只包含有限个样本点
2. **等可能性**：每个基本事件发生的概率相等

则称此概率模型为**古典概型**。

### 5.2 古典概率计算公式
$$P(A) = \frac{A \text{中样本点的个数}}{\Omega \text{中样本点的总数}} = \frac{|A|}{|\Omega|}$$

**说明**：
- |A| 表示集合 A 的元素个数
- 本质是计数问题

### 5.3 排列组合基础

**排列**：
- **全排列**：n 个不同元素的排列数 n!
- **选排列**：从 n 个不同元素中选 m 个的排列数
  $$P_n^m = \frac{n!}{(n-m)!} = n(n-1)...(n-m+1)$$

**组合**：
- 从 n 个不同元素中选 m 个的组合数
  $$C_n^m = \binom{n}{m} = \frac{n!}{m!(n-m)!}$$

**组合数的性质**：
1. $\binom{n}{m} = \binom{n}{n-m}$
2. $\binom{n}{m} = \binom{n-1}{m-1} + \binom{n-1}{m}$
3. $\sum_{k=0}^{n} \binom{n}{k} = 2^n$
4. $\binom{n+m}{k} = \sum_{i=0}^{k} \binom{n}{i}\binom{m}{k-i}$

**重复排列**：从 n 个不同元素中可重复地选取 m 个的排列数 n^m

**重复组合**：从 n 个不同元素中可重复地选取 m 个的组合数
$$\overline{C_n^m} = C_{n+m-1}^m = \binom{n+m-1}{m}$$

### 5.4 古典概型例题

**例1**（抽球问题）：袋中有 a 个白球，b 个黑球，从中不放回地抽取 k 个球，求恰好有 m 个白球的概率。

**解**：
$$P = \frac{C_a^m \cdot C_b^{k-m}}{C_{a+b}^k}$$

**例2**（配对问题）：n 个人随机排列，求至少有一人排在自己原来位置的概率。

**解**：设 Aᵢ 为"第 i 个人排在自己原来的位置"，则：
$$P(\bigcup_{i=1}^n A_i) = \sum_{k=1}^{n} (-1)^{k-1} \binom{n}{k} \frac{(n-k)!}{n!} = \sum_{k=1}^{n} \frac{(-1)^{k-1}}{k!}$$

当 n → ∞ 时，概率趋向于 1 - e⁻¹ ≈ 0.632。

**例3**（生日问题）：n 个人中至少有两人生日相同的概率。

**解**：
$$P(\text{至少两人生日相同}) = 1 - P(\text{n人生日各不相同}) = 1 - \frac{365 \times 364 \times \cdots \times (365-n+1)}{365^n}$$

当 n = 23 时，概率已超过 50%。

## 6. 几何概型

### 6.1 几何概型的定义
**定义**：若样本空间 Ω 为 ℝ^n 中的有界区域，且每个区域发生的概率与其测度（长度、面积、体积）成正比，则称此概率模型为**几何概型**。

### 6.2 几何概率计算公式
$$P(A) = \frac{m(A)}{m(\Omega)}$$

其中 m(·) 表示测度（长度、面积、体积等）。

### 6.3 几何概型的特点
- 样本空间 Ω 是无限集（不可数）
- 样本点的概率为 0
- 需要测度论基础

### 6.4 几何概型例题

**例1**（会面问题）：两人约定在 [0, T] 时段内会面，先到者等待时间 t 后离开，求两人能会面的概率。

**解**：设两人到达时间为 x, y ∈ [0, T]，则样本空间 Ω = [0, T]²。
能会面的条件：|x - y| ≤ t
能会面的事件 A = {(x, y) : |x - y| ≤ t}
$$P(A) = \frac{m(A)}{m(\Omega)} = \frac{T^2 - (T-t)^2}{T^2} = 1 - \left(1 - \frac{t}{T}\right)^2$$

**例2**（布丰投针问题）：平面上画有间距为 a 的平行线，向平面随机投掷长度为 l (l < a) 的针，求针与平行线相交的概率。

**解**：设针的中点到最近平行线的距离为 x，针与平行线的夹角为 θ。
样本空间：Ω = [0, a/2] × [0, π]
相交条件：x ≤ (l/2)sin θ
$$P = \frac{\int_0^{\pi} \frac{l}{2}\sin\theta \, d\theta}{\frac{a}{2} \cdot \pi} = \frac{l}{a\pi} \cdot 2 = \frac{2l}{\pi a}$$

可用于估算 π 值：π ≈ 2l/(aP)

## 7. 统计概率

### 7.1 频率与概率
**频率的定义**：在 n 次独立重复试验中，事件 A 发生的次数 n_A 与试验总次数 n 的比值：
$$f_n(A) = \frac{n_A}{n}$$

**频率的性质**：
1. 0 ≤ fₙ(A) ≤ 1
2. fₙ(Ω) = 1
3. 若 A₁, A₂, ..., Aₘ 两两互斥，则
   $$f_n\left(\bigcup_{i=1}^{m} A_i\right) = \sum_{i=1}^{m} f_n(A_i)$$

### 7.2 频率的稳定性
当试验次数 n 很大时，频率 fₙ(A) 会稳定在某个常数附近，这个常数就是事件 A 的概率。

**大数定律**保证：
$$P\left(\lim_{n \to \infty} f_n(A) = P(A)\right) = 1$$

## 8. 概率空间的例子

### 8.1 有限样本空间
**例**：掷骰子
- Ω = {1, 2, 3, 4, 5, 6}
- ℱ = 2^Ω（所有子集）
- P(A) = |A|/6（均匀分布）

### 8.2 可数样本空间
**例**：几何分布
- Ω = {1, 2, 3, ...}
- ℱ = 2^Ω
- P({n}) = p(1-p)^{n-1}，其中 0 < p < 1

### 8.3 连续样本空间
**例**：[0, 1] 上的均匀分布
- Ω = [0, 1]
- ℱ = ℬ([0, 1])（Borel σ-代数）
- P(A) = ∫_A dx（Lebesgue测度）

## 9. 重要定理与推论

### 9.1 Bonferroni不等式
$$P(AB) \leq P(A) \leq P(A \cup B) \leq P(A) + P(B)$$

### 9.2 Boole不等式
$$P\left(\bigcup_{i=1}^{n} A_i\right) \leq \sum_{i=1}^{n} P(A_i)$$

### 9.3 容斥原理
$$P\left(\bigcup_{i=1}^{n} A_i\right) = \sum_{i} P(A_i) - \sum_{i<j} P(A_i A_j) + \sum_{i<j<k} P(A_i A_j A_k) - \cdots + (-1)^{n-1} P(A_1 A_2 \cdots A_n)$$

## 10. 应用实例

### 10.1 彩票中奖概率
**双色球**：从 33 个红球中选 6 个，从 16 个蓝球中选 1 个。
- 一等奖（6+1）：$P = \frac{1}{C_{33}^6 \times C_{16}^1} \approx 1.77 \times 10^{-8}$

### 10.2 抽样检验
**不合格品率估计**：从 N 件产品中抽取 n 件，发现 k 件不合格品。
$$P(\text{恰有} k \text{件不合格}) = \frac{C_M^k C_{N-M}^{n-k}}{C_N^n}$$

其中 M 为总不合格品数。

## 11. 小结

### 11.1 核心概念
- **概率空间**：(Ω, ℱ, P) 三元组
- **σ-代数**：事件域
- **概率测度**：满足 Kolmogorov 公理的映射

### 11.2 重要性质
- 非负性、规范性、可列可加性
- 有限可加性、加法公式
- 连续性、单调性

### 11.3 计算方法
- **古典概型**：计数方法
- **几何概型**：测度方法
- **统计概率**：频率方法
