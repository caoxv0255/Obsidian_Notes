## 一、广义逆的概念

### 1.1 动机

矩阵 $A$ 可逆时，方程 $Ax = b$ 有唯一解 $x = A^{-1}b$。

当 $A$ 不可逆或非方阵时，我们希望找到一种"逆"使得：
- $Ax = b$ 有解时给出一个解
- $Ax = b$ 无解时给出"最佳近似解"

### 1.2 Moore-Penrose广义逆

**定义**：矩阵 $A \in \mathbb{C}^{m \times n}$ 的 **Moore-Penrose广义逆** $A^+ \in \mathbb{C}^{n \times m}$ 满足：

1. $AA^+A = A$
2. $A^+AA^+ = A^+$
3. $(AA^+)^* = AA^+$
4. $(A^+A)^* = A^+A$

**定理**：对任意矩阵 $A$，Moore-Penrose广义逆存在且唯一。

---

## 二、广义逆的性质

### 2.1 基本性质

1. $(A^+)^+ = A$
2. $(A^*)^+ = (A^+)^*$
3. $(A^T)^+ = (A^+)^T$
4. $(\lambda A)^+ = \lambda^{-1} A^+$（$\lambda \neq 0$）
5. $(A^*A)^+ = A^+(A^+)^*$
6. $\text{rank}(A^+) = \text{rank}(A)$

### 2.2 特殊情况

| 矩阵类型 | 广义逆 |
|----------|--------|
| 可逆方阵 | $A^{-1}$ |
| 列满秩 ($m \geq n$) | $(A^*A)^{-1}A^*$ |
| 行满秩 ($m \leq n$) | $A^*(AA^*)^{-1}$ |
| 酉矩阵 | $A^*$ |
| 对角矩阵 $\text{diag}(\lambda_i)$ | $\text{diag}(\lambda_i^+)$，$\lambda^+ = 1/\lambda$ 若 $\lambda \neq 0$，否则 $0$ |

### 2.3 与SVD的关系

**定理**：设 $A = U\Sigma V^*$ 是SVD，则
$$A^+ = V \Sigma^+ U^*$$
其中 $\Sigma^+$ 将 $\Sigma$ 的非零元素取倒数后转置。

---

## 三、广义逆的计算

### 3.1 SVD方法

```
输入：矩阵 A
输出：广义逆 A^+

1. 计算 SVD: A = U Σ V^*
2. 构造 Σ^+: 将 Σ 的非零元素取倒数
3. 计算 A^+ = V Σ^+ U^*
```

### 3.2 满秩分解法

**定理**：设 $A = BC$ 是满秩分解（$B$ 列满秩，$C$ 行满秩），则
$$A^+ = C^*(CC^*)^{-1}(B^*B)^{-1}B^*$$

### 3.3 迭代方法

**Newton迭代**：
$$X_{k+1} = X_k(2I - AX_k)$$
收敛到 $A^+$（在适当初值下）

## 四、矩阵方程 Ax = b

### 4.1 解的存在性与唯一性

| 情况 | 解的存在性 | 解的唯一性 |
|------|------------|------------|
| $Ax = b$ 相容 | 存在 | 当 $\ker A = \{0\}$ 时唯一 |
| $Ax = b$ 不相容 | 无解 | — |

### 4.2 通解公式

**定理**：若 $Ax = b$ 相容，则通解为
$$x = A^+b + (I - A^+A)y$$
其中 $y$ 是任意向量。

**解释**：
- $A^+b$ 是一个特解
- $I - A^+A$ 是投影到 $\ker A$ 的投影矩阵

### 4.3 最小范数解

**定理**：若 $Ax = b$ 相容，则 $x^* = A^+b$ 是唯一的最小范数解：
$$\|A^+b\| = \min\{\|x\| : Ax = b\}$$

---

## 五、最小二乘问题

### 5.1 问题定义

**最小二乘问题**：求 $x$ 使 $\|Ax - b\|^2$ 最小。

### 5.2 正规方程

**定理**：$x$ 是最小二乘解当且仅当满足正规方程
$$A^*Ax = A^*b$$

### 5.3 最小范数最小二乘解

**定理**：$x^* = A^+b$ 是唯一的最小范数最小二乘解：
- $\|Ax^* - b\| = \min_x \|Ax - b\|$
- $\|x^*\| = \min\{\|x\| : \|Ax - b\| \text{ 最小}\}$

### 5.4 与投影的关系

最小二乘解 $x^*$ 满足
$$Ax^* = P_{\text{Im}\,A}(b)$$
其中 $P_{\text{Im}\,A}$ 是投影到 $\text{Im}\,A$ 的正交投影。

---

## 六、其他矩阵方程

### 6.1 Lyapunov方程

**方程**：$AX + XA^* = B$

**解的存在性**：若 $A$ 的特征值 $\lambda_i + \bar{\lambda}_j \neq 0$ 对所有 $i, j$，则解唯一。

**应用**：控制理论中的稳定性分析

### 6.2 Sylvester方程

**方程**：$AX + XB = C$

**解的存在性**：若 $A, B$ 的特征值满足 $\lambda_i(A) + \lambda_j(B) \neq 0$，则解唯一。

**向量化解**：$\text{vec}(X) = (I \otimes A + B^T \otimes I)^{-1} \text{vec}(C)$

### 6.3 Riccati方程

**方程**：$A^*X + XA - XBX + C = 0$

**应用**：最优控制

---

## 七、Penrose条件详解

### 7.1 四个条件的几何意义

| 条件 | 含义 |
|------|------|
| $AA^+A = A$ | $A^+$ 是 $A$ 的"内逆" |
| $A^+AA^+ = A^+$ | $A$ 是 $A^+$ 的"内逆" |
| $(AA^+)^* = AA^+$ | $AA^+$ 是Hermite投影 |
| $(A^+A)^* = A^+A$ | $A^+A$ 是Hermite投影 |

### 7.2 投影的几何

- $AA^+$：投影到 $\text{Im}\,A$ 的正交投影
- $A^+A$：投影到 $(\ker A)^\perp$ 的正交投影
- $I - A^+A$：投影到 $\ker A$ 的正交投影

---

## 八、应用

### 8.1 线性回归

**模型**：$y = X\beta + \epsilon$

**最小二乘估计**：$\hat{\beta} = (X^TX)^+X^Ty = X^+y$

### 8.2 图像处理

**图像恢复**：$A^+$ 用于求解病态方程，实现图像去模糊

### 8.3 控制系统

**状态估计**：利用广义逆从不完全观测恢复状态

## 重要公式汇总

| 公式 | 说明 |
|------|------|
| $A^+ = V\Sigma^+U^*$ | SVD计算 |
| $x = A^+b + (I - A^+A)y$ | $Ax = b$ 通解 |
| $A^+b$ | 最小范数解/最小二乘解 |
| $AA^+$ | 投影到 $\text{Im}\,A$ |
| $A^+A$ | 投影到 $(\ker A)^\perp$ |

## 总结

- 广义逆把“不可逆矩阵的逆”扩展成可用于最小二乘与最小范数问题的标准工具。
- Moore-Penrose 条件保证了广义逆的唯一性和几何意义。
- SVD 是计算广义逆的核心算法入口。

## 易错点

- 把广义逆误认为普通逆的直接替代，不考虑维数和非满秩情形。
- 忽略四个 Penrose 条件的必要性。
- 把最小二乘解和最小范数解混为一谈。
- 忘记广义逆的投影几何意义。
