# 第1章 完全信息静态博弈

## 章节学习目标

本章主要讨论完全信息静态博弈的基本理论，学生完成本章学习后应能够：

1. 掌握策略型博弈的基本概念和表示方法
2. 深入理解纳什均衡的定义、性质和求解方法
3. 掌握混合策略纳什均衡的概念和计算方法
4. 理解纳什均衡存在性定理及其应用
5. 了解纳什均衡的选择与精炼的基本方法
6. 能够运用博弈论分析现实中的策略互动问题

## 1.1 策略型博弈

### 1.1.1 策略型博弈的定义

**定义**：策略型博弈（Strategic-Form Game），也称为标准型博弈（Normal-Form Game），是指所有参与者同时选择策略，或者虽非同时但后行动者不知道前行动者所选策略的博弈。策略型博弈是描述静态博弈的基本形式。

一个策略型博弈可以表示为三元组 G = {S₁, S₂, ..., Sₙ; u₁, u₂, ..., uₙ}，其中：

- N = {1, 2, ..., n} 是参与者的有限集合
- Sᵢ 是参与者 i 的策略空间，即参与者 i 所有可选策略的集合
- uᵢ: S₁ × S₂ × ... × Sₙ → ℝ 是参与者 i 的支付函数

**符号说明**：
- 策略组合 s = (s₁, s₂, ..., sₙ) ∈ S₁ × S₂ × ... × Sₙ 表示每个参与者选择一个策略
- S = S₁ × S₂ × ... × Sₙ 表示所有可能的策略组合构成的集合
- 对于任意参与者 i，用 sᵢ 表示参与者 i 的策略
- 用 s₋ᵢ 表示除参与者 i 之外所有其他参与者的策略组合，即 s₋ᵢ = (s₁, ..., sᵢ₋₁, sᵢ₊₁, ..., sₙ)
- 因此，策略组合可以表示为 s = (sᵢ, s₋ᵢ)

### 1.1.2 囚徒困境

囚徒困境是博弈论中最著名的例子之一，它展示了个人理性与集体理性之间的冲突。

**博弈描述**：
两个嫌疑人被警方逮捕，被分别关在不同的审讯室，无法相互交流。警方告诉每个嫌疑人：
- 如果两人都认罪，各判5年
- 如果一人认罪，另一人不认罪，认罪者判1年，不认罪者判10年
- 如果两人都不认罪，各判2年

**博弈矩阵**：

| 嫌疑人A \ 嫌疑人B | 认罪 | 不认罪 |
|:---:|:---:|:---:|
| **认罪** | (-5, -5) | (-1, -10) |
| **不认罪** | (-10, -1) | (-2, -2) |

**分析**：
- 对于嫌疑人A：如果B认罪，A的最佳反应是认罪（-5 > -10）；如果B不认罪，A的最佳反应仍是认罪（-1 > -2）
- 对于嫌疑人B：同样地，无论A选择什么，B的最佳反应都是认罪
- 因此，（认罪，认罪）是这个博弈的纳什均衡
- 然而，（不认罪，不认罪）给两人带来的收益都更好（-2 > -5）
- 这就是囚徒困境：个人理性的选择导致集体非优的结果

### 1.1.3 占优策略均衡

**定义**：策略 sᵢ* ∈ Sᵢ 是参与者 i 的（弱）占优策略，如果对于所有 sᵢ ∈ Sᵢ 和所有 s₋ᵢ ∈ S₋ᵢ，都有：
uᵢ(sᵢ*, s₋ᵢ) ≥ uᵢ(sᵢ, s₋ᵢ)
并且至少存在一个 sᵢ 和 s₋ᵢ 使得严格不等式成立。

**定义**：策略组合 s* = (s₁*, s₂*, ..., sₙ*) 是（弱）占优策略均衡，如果对于每个参与者 i，sᵢ* 都是参与者 i 的占优策略。

**囚徒困境中的占优策略**：
- 在囚徒困境中，"认罪"是每个嫌疑人的占优策略
- 因此，（认罪，认罪）是占优策略均衡
- 占优策略均衡一定是纳什均衡，但纳什均衡不一定是占优策略均衡

### 1.1.4 重复剔除劣策略

**定义**：策略 sᵢ' ∈ Sᵢ 是参与者 i 的（弱）劣策略，如果存在另一个策略 sᵢ ∈ Sᵢ，使得对于所有 s₋ᵢ ∈ S₋ᵢ，都有：
uᵢ(sᵢ, s₋ᵢ) ≥ uᵢ(sᵢ', s₋ᵢ)
并且至少存在一个 s₋ᵢ 使得严格不等式成立。

**重复剔除劣策略**：通过重复剔除参与者的劣策略，可以缩小博弈的均衡范围。如果经过剔除后，每个参与者只剩下一个策略，那么这个唯一的策略组合就是博弈的均衡。

**例子：性别战博弈**

**博弈描述**：
丈夫和妻子想一起看电影，但偏好不同。

**博弈矩阵**：

| 丈夫 \ 妻子 | 动作片 | 爱情片 |
|:---:|:---:|:---:|
| **动作片** | (2, 1) | (0, 0) |
| **爱情片** | (0, 0) | (1, 2) |

**分析**：
- 没有占优策略
- 两个纳什均衡：(动作片, 动作片) 和 (爱情片, 爱情片)
- 需要其他方法来选择均衡

### 1.1.5 纳什均衡的定义

**定义**：策略组合 s* = (s₁*, s₂*, ..., sₙ*) 是策略型博弈 G = {S₁, S₂, ..., Sₙ; u₁, u₂, ..., uₙ} 的纳什均衡，如果对于每个参与者 i，sᵢ* 是对其他参与者策略组合 s₋ᵢ* 的最佳反应，即：
uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)，对所有 sᵢ ∈ Sᵢ

**理解纳什均衡**：
- 纳什均衡是一种自我执行、自我强化的协议
- 在纳什均衡处，没有任何参与者有单方面偏离的动机
- 纳什均衡是博弈的一种稳定状态
- 纳什均衡是博弈的"预测"，但不一定是最优的结果

**纳什均衡的等价表述**：
令 BRᵢ(s₋ᵢ) = {sᵢ ∈ Sᵢ: uᵢ(sᵢ, s₋ᵢ) ≥ uᵢ(sᵢ', s₋ᵢ), ∀sᵢ' ∈ Sᵢ} 表示参与者 i 对 s₋ᵢ 的最佳反应集合。

策略组合 s* 是纳什均衡，当且仅当：
sᵢ* ∈ BRᵢ(s₋ᵢ*)，对所有 i ∈ N

### 1.1.6 求解纳什均衡的方法

**方法1：检查法**
对于有限博弈，可以逐一检查每个策略组合是否是纳什均衡。

**方法2：划线法**
在博弈矩阵中，在每个参与者对对手策略的最佳反应对应的支付下划线。某个策略组合下两个支付都被划线，该策略组合就是纳什均衡。

**例子：使用划线法求解囚徒困境**

| 嫌疑人A \ 嫌疑人B | 认罪 | 不认罪 |
|:---:|:---:|:---:|
| **认罪** | <u>(-5)</u>, <u>(-5)</u> | <u>(-1)</u>, (-10) |
| **不认罪** | (-10), <u>(-1)</u> | (-2), <u>(-2)</u> |

划线后，只有（认罪，认罪）两个支付都被划线，因此这是唯一的纳什均衡。

**方法3：通过最佳反应函数求解**
对于每个参与者，构造最佳反应函数，然后找到所有最佳反应函数的交点。

## 1.2 纳什均衡

### 1.2.1 纳什均衡的性质

**性质1：纳什均衡的不唯一性**
一个博弈可能有多个纳什均衡，也可能没有纳什均衡（纯策略意义上）。

**性质2：纳什均衡可能不是帕累托最优的**
囚徒困境中的纳什均衡（认罪，认罪）不是帕累托最优的，因为（不认罪，不认罪）让两人都更好。

**性质3：纳什均衡可能不是帕累托有效的**
帕累托有效是指没有任何参与者能在不损害其他参与者的情况下改善自己的状况。

**性质4：纳什均衡是自我实施的**
如果所有参与者都预期某策略组合会被实施，那么没有参与者有动机偏离。

### 1.2.2 纳什均衡的例子

**例子1：智猪博弈**

**博弈描述**：
一头大猪和一头小猪在猪圈里，食槽在一边，按钮在另一边。按按钮需要花费2单位能量，但能产生10单位食物。食物分配取决于谁去按按钮。

**博弈矩阵**：

| 大猪 \ 小猪 | 按按钮 | 等待 |
|:---:|:---:|:---:|
| **按按钮** | (5, 1) | (4, 4) |
| **等待** | (9, -1) | (0, 0) |

**分析**：
- 小猪：如果大猪按按钮，小猪的最佳反应是等待（4 > 1）；如果大猪等待，小猪的最佳反应还是等待（0 > -1）
- 因此，"等待"是小猪的占优策略
- 大猪：给定小猪等待，大猪的最佳反应是按按钮（4 > 0）
- 纳什均衡：（按按钮，等待）

**例子2：猎鹿博弈**

**博弈描述**：
两个猎人可以合作猎鹿（每人得到3单位肉），也可以单独猎兔（每人得到1单位肉）。猎鹿需要两人合作，猎兔可以独自进行。

**博弈矩阵**：

| 猎人1 \ 猎人2 | 猎鹿 | 猎兔 |
|:---:|:---:|:---:|
| **猎鹿** | (3, 3) | (0, 1) |
| **猎兔** | (1, 0) | (1, 1) |

**分析**：
- 两个纳什均衡：（猎鹿，猎鹿）和（猎兔，猎兔）
- （猎鹿，猎鹿）帕累托优于（猎兔，猎兔）
- 但存在协调问题：如果一方怀疑另一方不合作，就会选择猎兔

**例子3：胆小鬼博弈**

**博弈描述**：
两个司机相向而驶，谁先转向谁就是"胆小鬼"，两人都不转向则相撞。

**博弈矩阵**：

| 司机1 \ 司机2 | 转向 | 直行 |
|:---:|:---:|:---:|
| **转向** | (0, 0) | (-1, 1) |
| **直行** | (1, -1) | (-10, -10) |

**分析**：
- 两个纳什均衡：（转向，直行）和（直行，转向）
- （转向，直行）和（直行，转向）都不是对称的
- 存在混合策略纳什均衡（后面讨论）

### 1.2.3 纳什均衡与理性

**共同知识假设**：
纳什均衡的存在依赖于共同知识假设：
- 所有参与者都是理性的
- 所有参与者知道博弈的结构（支付函数）
- 所有参与者知道其他参与者也是理性的
- 这个知识结构是无限的

**纳什均衡与理性的关系**：
- 纳什均衡是理性参与者在相互理解对方策略的基础上达成的一致
- 纳什均衡是一种稳定状态，一旦达成，没有人有动机偏离
- 但纳什均衡不一定是"最优"的，也不一定会被达成

### 1.2.4 纳什均衡的应用

**应用1：古诺双寡头模型**

**博弈描述**：
两个企业同时决定产量 q₁ 和 q₂，市场价格为 P = a - q₁ - q₂，成本函数为 Cᵢ(qᵢ) = c·qᵢ。

**利润函数**：
π₁(q₁, q₂) = (a - q₁ - q₂)q₁ - cq₁ = (a - c)q₁ - q₁² - q₁q₂
π₂(q₁, q₂) = (a - q₁ - q₂)q₂ - cq₂ = (a - c)q₂ - q₂² - q₁q₂

**最佳反应函数**：
∂π₁/∂q₁ = a - c - 2q₁ - q₂ = 0 ⇒ q₁ = (a - c - q₂)/2
∂π₂/∂q₂ = a - c - 2q₂ - q₁ = 0 ⇒ q₂ = (a - c - q₁)/2

**纳什均衡**：
解联立方程：
q₁ = (a - c - q₂)/2
q₂ = (a - c - q₁)/2

得到：q₁* = q₂* = (a - c)/3

**应用2：伯川德双寡头模型**

**博弈描述**：
两个企业同时决定价格 p₁ 和 p₂，产品同质，边际成本为 c，需求函数为 D(p)。

**利润函数**：
πᵢ(pᵢ, pⱼ) = {
  (pᵢ - c)D(pᵢ), 如果 pᵢ < pⱼ
  (pᵢ - c)D(pᵢ)/2, 如果 pᵢ = pⱼ
  0, 如果 pᵢ > pⱼ
}

**纳什均衡**：
唯一的纳什均衡是 p₁* = p₂* = c（边际成本定价）

**伯川德悖论**：
在简单的伯川德模型中，两个企业就足以导致完全竞争的结果，这与现实不符。

## 1.3 混合策略纳什均衡

### 1.3.1 混合策略的定义

**定义**：在策略型博弈中，参与者 i 的混合策略 σᵢ 是其纯策略集合 Sᵢ 上的概率分布：
σᵢ: Sᵢ → [0, 1]
满足：Σ_{sᵢ∈Sᵢ} σᵢ(sᵢ) = 1

**符号说明**：
- Σᵢ 表示参与者 i 所有混合策略的集合
- σ = (σ₁, σ₂, ..., σₙ) 表示混合策略组合
- uᵢ(σ) 表示参与者 i 在混合策略组合 σ 下的期望支付

**期望支付的计算**：
uᵢ(σ) = Σ_{s∈S} uᵢ(s)·Π_{j=1}^{n} σⱼ(sⱼ)

其中 s = (s₁, s₂, ..., sₙ) 是纯策略组合。

### 1.3.2 混合策略纳什均衡的定义

**定义**：混合策略组合 σ* = (σ₁*, σ₂*, ..., σₙ*) 是博弈 G 的混合策略纳什均衡，如果对于每个参与者 i，σᵢ* 是对其他参与者混合策略组合 σ₋ᵢ* 的最佳反应，即：
uᵢ(σᵢ*, σ₋ᵢ*) ≥ uᵢ(σᵢ, σ₋ᵢ*)，对所有 σᵢ ∈ Σᵢ

**混合策略最佳反应**：
σᵢ ∈ BRᵢ(σ₋ᵢ) 当且仅当：
uᵢ(sᵢ, σ₋ᵢ) = uᵢ(σᵢ, σ₋ᵢ)，对所有 sᵢ 使得 σᵢ(sᵢ) > 0

即，被以正概率选择的纯策略必须都是最佳反应。

### 1.3.3 混合策略纳什均衡的求解

**例子：胆小鬼博弈的混合策略纳什均衡**

**博弈矩阵**：

| 司机1 \ 司机2 | 转向 | 直行 |
|:---:|:---:|:---:|
| **转向** | (0, 0) | (-1, 1) |
| **直行** | (1, -1) | (-10, -10) |

**求解过程**：

设司机1以概率 p 选择直行，司机2以概率 q 选择直行。

司机1的期望支付：
- 选择转向：E₁(转向) = 0·(1-q) + (-1)·q = -q
- 选择直行：E₁(直行) = 1·(1-q) + (-10)·q = 1 - 11q

混合策略均衡时，E₁(转向) = E₁(直行)：
-q = 1 - 11q ⇒ 10q = 1 ⇒ q = 0.1

司机2的期望支付：
- 选择转向：E₂(转向) = 0·(1-p) + (-1)·p = -p
- 选择直行：E₂(直行) = 1·(1-p) + (-10)·p = 1 - 11p

混合策略均衡时，E₂(转向) = E₂(直行)：
-p = 1 - 11p ⇒ 10p = 1 ⇒ p = 0.1

**混合策略纳什均衡**：
司机1以 0.1 的概率选择直行，以 0.9 的概率选择转向
司机2以 0.1 的概率选择直行，以 0.9 的概率选择转向

**期望支付**：
u₁(σ*) = u₂(σ*) = -0.1

**例子：配硬币博弈**

**博弈描述**：
两人同时展示硬币，若正面相同则玩家1赢，若反面相同则玩家2赢。

**博弈矩阵**：

| 玩家1 \ 玩家2 | 正面 | 反面 |
|:---:|:---:|:---:|
| **正面** | (1, -1) | (-1, 1) |
| **反面** | (-1, 1) | (1, -1) |

**求解过程**：

设玩家1以概率 p 选择正面，玩家2以概率 q 选择正面。

玩家1的期望支付：
- 选择正面：E₁(正面) = 1·q + (-1)·(1-q) = 2q - 1
- 选择反面：E₁(反面) = (-1)·q + 1·(1-q) = 1 - 2q

混合策略均衡时，E₁(正面) = E₁(反面)：
2q - 1 = 1 - 2q ⇒ 4q = 2 ⇒ q = 0.5

同理，p = 0.5

**混合策略纳什均衡**：
双方都以 0.5 的概率选择正面，0.5 的概率选择反面

**期望支付**：
u₁(σ*) = u₂(σ*) = 0

### 1.3.4 混合策略的解释

**解释1：随机化策略**
参与者真的以一定概率随机选择策略，使对手无法预测自己的行为。

**解释2：群体解释**
在一个大群体中，有一定比例的参与者选择不同策略。

**解释3：信念解释**
混合策略反映了参与者对对手策略的信念，而非真正的随机化。

**解释4：不确定性**
参与者对某些参数不确定，导致策略的随机性。

### 1.3.5 混合策略的应用

**应用1：竞选博弈**

**博弈描述**：
两个候选人可以在左、中、右三个位置中选择一个进行竞选，选民分布使得候选人会选择中间位置。

**应用2：体育博弈**
点球、棒球等体育比赛中，攻击方和防守方的策略选择可以用混合策略模型分析。

**应用3：军事博弈**
军事部署、攻击路线选择等问题中，混合策略可以增加对手的预测难度。

## 1.4 纳什均衡的存在性

### 1.4.1 纳什均衡存在性定理

**定理（纳什，1950）**：每一个有限策略型博弈（参与者有限，每个参与者的策略空间有限）至少存在一个纳什均衡（可能是混合策略）。

**定理的条件**：
1. 参与者集合 N 是有限的
2. 每个参与者的策略空间 Sᵢ 是有限的
3. 支付函数 uᵢ 是连续的（在有限策略情况下自动满足）

**定理的证明思路**：
纳什使用了不动点定理（Kakutani 不动点定理）来证明纳什均衡的存在性。

**Kakutani 不动点定理**：
设 S 是 ℝⁿ 上的非空、有界、闭的凸集，f: S → S 是一个上半连续的凸值对应，则存在 x* ∈ S 使得 x* ∈ f(x*)。

**应用 Kakutani 不动点定理**：
1. 定义混合策略空间 Σ = Σ₁ × Σ₂ × ... × Σₙ
2. Σ 是非空、有界、闭的凸集
3. 定义最佳反应对应 BR: Σ → Σ
4. BR 是上半连续的凸值对应
5. 由 Kakutani 不动点定理，存在 σ* ∈ Σ 使得 σ* ∈ BR(σ*)
6. 这个 σ* 就是混合策略纳什均衡

### 1.4.2 无限博弈的纳什均衡存在性

**Debreu-Glicksberg-Fan 定理**：
在策略型博弈 G = {S₁, S₂, ..., Sₙ; u₁, u₂, ..., uₙ} 中，如果满足：
1. 每个参与者的策略空间 Sᵢ 是非空、有界、闭的凸集
2. 每个参与者的支付函数 uᵢ 在 S 上是连续的
3. 每个参与者的支付函数 uᵢ 在 sᵢ 上是拟凹的

则博弈 G 至少存在一个纯策略纳什均衡。

**Glicksberg 定理**：
在策略型博弈 G = {S₁, S₂, ..., Sₙ; u₁, u₂, ..., uₙ} 中，如果满足：
1. 每个参与者的策略空间 Sᵢ 是非空、有界、闭的凸集
2. 每个参与者的支付函数 uᵢ 在 S 上是连续的

则博弈 G 至少存在一个混合策略纳什均衡。

### 1.4.3 存在性定理的意义

**理论意义**：
- 保证了博弈论模型的良好性质
- 为博弈分析提供了理论基础
- 证明了纳什均衡这一概念的合理性

**实践意义**：
- 在构建博弈模型时，确保至少有一个解
- 为算法求解纳什均衡提供了保证
- 增强了对博弈分析结果的信心

### 1.4.4 不存在纳什均衡的情况

**纯策略不存在的情况**：
有些博弈在纯策略意义上没有纳什均衡，但存在混合策略纳什均衡，如配硬币博弈。

**混合策略也不存在的情况**：
如果策略空间无限且支付函数不满足连续性条件，可能连混合策略纳什均衡也不存在。

**例子**：
考虑两个参与者在 [0, 1] 上选择数字，支付函数不连续的情况。

## 1.5 纳什均衡的选择与精炼

### 1.5.1 多重均衡问题

**问题**：
许多博弈有多个纳什均衡，这导致均衡选择问题：哪个均衡会被实际达成？

**例子：性别战博弈**
两个纳什均衡：（动作片，动作片）和（爱情片，爱情片）

**均衡选择的标准**：
1. 帕累托优势（Pareto Dominance）
2. 风险优势（Risk Dominance）
3. 焦点效应（Focal Point）
4. 历史和传统
5. 学习和演化

### 1.5.2 帕累托优势

**定义**：均衡 s* 帕累托优于均衡 s**，如果对于所有参与者 i，都有 uᵢ(s*) ≥ uᵢ(s**)，并且至少存在一个参与者 j 使得严格不等式成立。

**例子：猎鹿博弈**
（猎鹿，猎鹿）帕累托优于（猎兔，猎兔）

**局限性**：
- 并非所有博弈都有帕累托优势均衡
- 帕累托优势均衡不一定会被选择
- 帕累托优势与风险优势可能冲突

### 1.5.3 风险优势

**定义**：均衡 s* 的风险优势高于均衡 s**，如果偏离 s* 的"激励"更小。

**Harsanyi-Selten 风险优势准则**：
对于 2×2 博弈，计算两个均衡的"风险优势"指标，选择风险优势更大的均衡。

**例子：猎鹿博弈**

| 猎人1 \ 猎人2 | 猎鹿 | 猎兔 |
|:---:|:---:|:---:|
| **猎鹿** | (3, 3) | (0, 1) |
| **猎兔** | (1, 0) | (1, 1) |

计算风险优势：
- 偏离（猎鹿，猎鹿）的损失：3 - 0 = 3
- 偏离（猎兔，猎兔）的损失：1 - 1 = 0

因此，（猎兔，猎兔）的风险优势更大。

**冲突**：
（猎鹿，猎鹿）帕累托优势，但（猎兔，猎兔）风险优势

### 1.5.4 焦点效应

**Schelling 的焦点概念**：
在某些情况下，由于文化、历史、语言等原因，某个均衡会成为"焦点"，被参与者自然选择。

**例子**：
- 在性别战中，如果今天是丈夫的生日，（动作片，动作片）可能成为焦点
- 在协调博弈中，某个明显"更好"的选项可能成为焦点

**焦点效应的条件**：
1. 突出性（Salience）
2. 共同知识
3. 文化背景
4. 历史传统

### 1.5.5 演化稳定策略

**定义**：策略 s* 是演化稳定的，如果：
1. (s*, s*) 是纳什均衡
2. 对于任何 s ≠ s*，存在一个 ε̄ > 0，使得对于所有 ε ∈ (0, ε̄)：
u(s*, (1-ε)s* + εs) > u(s, (1-ε)s* + εs)

**解释**：
演化稳定策略意味着，如果小比例的参与者偏离，采用原策略的参与者仍然比偏离者有更高的适应度。

**例子**：
在囚徒困境中，（认罪，认罪）是演化稳定的。

**与纳什均衡的关系**：
- 演化稳定策略一定是纳什均衡
- 但纳什均衡不一定是演化稳定的

### 1.5.6 相关均衡

**定义**：相关均衡是一种通过相关机制协调参与者策略的均衡概念。

**机制**：
参与者观察一个共同的随机信号，根据信号选择策略。

**例子**：
在配硬币博弈中，引入相关信号可以改善结果。

**优点**：
可以实现比纳什均衡更好的结果
可以通过简单的机制实现

**局限性**：
需要可信的相关机制
不适用于所有情况

## 本章总结

### 核心要点

1. **策略型博弈**是描述静态博弈的基本形式，由参与者、策略空间和支付函数构成。

2. **纳什均衡**是最重要的均衡概念，是指在给定其他参与者策略的情况下，每个参与者都选择了最佳策略的组合。

3. **混合策略**是纯策略的概率分布，许多博弈只有混合策略纳什均衡。

4. **纳什均衡存在性定理**保证了有限博弈至少存在一个纳什均衡。

5. **多重均衡问题**需要通过帕累托优势、风险优势、焦点效应等标准进行选择。

6. **纳什均衡**是博弈论的核心概念，为分析策略互动提供了强有力的工具。

### 重要概念表格

| 概念 | 定义 | 性质 |
|:---:|:---:|:---:|
| 策略型博弈 | 三元组 G = {S₁, S₂, ..., Sₙ; u₁, u₂, ..., uₙ} | 描述静态博弈的基本形式 |
| 纳什均衡 | uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*) | 自我执行的稳定状态 |
