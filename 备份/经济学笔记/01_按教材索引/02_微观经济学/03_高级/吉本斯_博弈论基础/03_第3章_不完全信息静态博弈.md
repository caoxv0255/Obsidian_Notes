# 第3章 不完全信息静态博弈

## 章节学习目标

本章主要讨论不完全信息静态博弈的基本理论，学生完成本章学习后应能够：

1. 掌握贝叶斯博弈的基本概念和表示方法
2. 深入理解贝叶斯纳什均衡的定义和求解方法
3. 掌握海萨尼转换的基本原理
4. 理解混合策略的贝叶斯解释
5. 掌握拍卖理论的基本模型和均衡分析
6. 理解机制设计的基本原理和应用

## 3.1 贝叶斯博弈

### 3.1.1 不完全信息的含义

**不完全信息（Incomplete Information）**：
- 参与者不知道其他参与者的某些特征
- 例如：不知道对手的成本、类型、偏好等

**完全信息（Complete Information）**：
- 所有参与者都知道博弈的结构
- 包括参与者的支付函数

**海萨尼的贡献**：
Harsanyi (1967-1968) 提出将不完全信息转化为不完美信息（海萨尼转换）

### 3.1.2 贝叶斯博弈的定义

**定义**：贝叶斯博弈（Bayesian Game）是描述不完全信息静态博弈的基本形式，表示为：
G = {N, (Tᵢ)ᵢ∈N, (Aᵢ)ᵢ∈N, (pᵢ)ᵢ∈N, (uᵢ)ᵢ∈N}

其中：
- N = {1, 2, ..., n} 是参与者集合
- Tᵢ 是参与者 i 的类型空间
- Aᵢ 是参与者 i 的行动空间
- pᵢ: Tᵢ → Δ(T₋ᵢ) 是参与者 i 关于其他参与者类型的信念
- uᵢ: A₁ × ... × Aₙ × T₁ × ... × Tₙ → ℝ 是参与者 i 的支付函数

**符号说明**：
- tᵢ ∈ Tᵢ 表示参与者 i 的类型
- t = (t₁, t₂, ..., tₙ) 表示类型组合
- t₋ᵢ = (t₁, ..., tᵢ₋₁, tᵢ₊₁, ..., tₙ) 表示其他参与者的类型组合
- aᵢ ∈ Aᵢ 表示参与者 i 的行动
- a = (a₁, a₂, ..., aₙ) 表示行动组合

### 3.1.3 类型与信念

**类型（Type）**：
- 描述参与者的私人信息
- 包括成本、偏好、能力等
- 类型影响参与者的支付

**信念（Belief）**：
- 参与者对其他参与者类型的概率判断
- pᵢ(t₋ᵢ | tᵢ) 表示参与者 i 在自己类型为 tᵢ 时，认为其他参与者类型为 t₋ᵢ 的概率

**共同先验（Common Prior）**：
- 假设存在一个共同的先验分布 p(t₁, t₂, ..., tₙ)
- 参与者的信念通过贝叶斯法则从先验推导
- pᵢ(t₋ᵢ | tᵢ) = p(t₋ᵢ, tᵢ) / p(tᵢ)

### 3.1.4 海萨尼转换

**目的**：
将不完全信息博弈转化为不完美信息博弈，从而可以使用纳什均衡的概念。

**步骤**：

1. **引入自然（Nature）**：
   - 自然首先选择参与者的类型
   - 类型分布由先验 p(t) 给出

2. **信息结构**：
   - 参与者 i 观察到自己的类型 tᵢ
   - 参与者 i 不知道其他参与者的类型

3. **博弈进行**：
   - 参与者根据观察到的类型选择行动
   - 支付取决于行动和类型

**转换后的博弈**：
- 完全信息的扩展型博弈
- 不完美信息（参与者不知道某些决策节点的信息）
- 可以使用贝叶斯纳什均衡的概念

### 3.1.5 贝叶斯博弈的例子

**例子1：不完全信息进入博弈**

**博弈描述**：
企业A（在位者）的成本可能是高成本（t = H）或低成本（t = L），概率各为0.5。企业B（进入者）不知道A的成本。

**类型**：
- A的类型：T_A = {H, L}
- B的类型：T_B = {单一类型}

**先验**：
p(H) = p(L) = 0.5

**行动**：
- A：{容忍, 对抗}
- B：{进入, 不进入}

**支付**：

A为高成本时：
| B \ A | 容忍 | 对抗 |
|:---:|:---:|:---:|
| **进入** | (1, 1) | (-1, 0) |
| **不进入** | (0, 2) | (0, 2) |

A为低成本时：
| B \ A | 容忍 | 对抗 |
|:---:|:---:|:---:|
| **进入** | (1, 2) | (-1, 3) |
| **不进入** | (0, 2) | (0, 2) |

**分析**：
- 如果A是高成本，B的最佳反应：进入（1 > 0）
- 如果A是低成本，B的最佳反应：不进入（0 > -1）
- B不知道A的类型，需要计算期望支付

**例子2：一级价格拍卖**

**博弈描述**：
两个竞拍者对物品进行估价v₁和v₂，v₁和v₂在[0,1]上独立均匀分布。竞拍者同时出价b₁和b₂，出价高者获得物品，支付自己的出价。

**类型**：
- 竞拍者1的类型：v₁ ∈ [0, 1]
- 竞拍者2的类型：v₂ ∈ [0, 1]

**行动**：
- 竞拍者1的出价：b₁ ∈ [0, 1]
- 竞拍者2的出价：b₂ ∈ [0, 1]

**先验**：
v₁和v₂在[0,1]上独立均匀分布

**支付**：
u₁(b₁, b₂, v₁, v₂) = {
  v₁ - b₁, 如果 b₁ > b₂
  (v₁ - b₁)/2, 如果 b₁ = b₂
  0, 如果 b₁ < b₂
}

### 3.1.6 策略的定义

**定义**：在贝叶斯博弈中，参与者 i 的策略是一个从类型到行动的映射：
sᵢ: Tᵢ → Aᵢ

**理解**：
- 策略规定了每种类型应该采取什么行动
- 策略是事前制定的，参与者根据实际观察到的类型执行策略
- 策略必须是完整的，规定所有可能类型的行动

**例子**：
在进入博弈中，A的策略可以表示为：
s_A(H) = 容忍，s_A(L) = 对抗

## 3.2 贝叶斯纳什均衡

### 3.2.1 贝叶斯纳什均衡的定义

**定义**：策略组合 s* = (s₁*, s₂*, ..., sₙ*) 是贝叶斯博弈 G 的贝叶斯纳什均衡，如果对于每个参与者 i 和每种类型 tᵢ ∈ Tᵢ，sᵢ*(tᵢ) 最大化参与者 i 的期望支付：

E[uᵢ(sᵢ*(tᵢ), s₋ᵢ*(t₋ᵢ), tᵢ, t₋ᵢ) | tᵢ] ≥ E[uᵢ(aᵢ, s₋ᵢ*(t₋ᵢ), tᵢ, t₋ᵢ) | tᵢ]

对所有 aᵢ ∈ Aᵢ

**期望支付的计算**：
E[uᵢ(aᵢ, s₋ᵢ*(t₋ᵢ), tᵢ, t₋ᵢ) | tᵢ] = Σ_{t₋ᵢ} pᵢ(t₋ᵢ | tᵢ) uᵢ(aᵢ, s₋ᵢ*(t₋ᵢ), tᵢ, t₋ᵢ)

**理解贝叶斯纳什均衡**：
- 每种类型都选择最佳反应
- 考虑了不确定性下的期望效用最大化
- 是贝叶斯博弈的自我执行状态

### 3.2.2 贝叶斯纳什均衡的求解

**方法**：
1. 猜测均衡策略的形式
2. 计算每种类型的最佳反应
3. 验证猜测是否一致

**例子：不完全信息进入博弈的贝叶斯纳什均衡**

**步骤1**：猜测策略形式
假设A的策略：
- 高成本时：s_A(H) = 容忍
- 低成本时：s_A(L) = 对抗

**步骤2**：计算B的最佳反应
B不知道A的类型，计算期望支付：
- 如果B进入：
  E[支付] = 0.5 × 1（A容忍） + 0.5 × (-1)（A对抗） = 0
- 如果B不进入：
  E[支付] = 0

B对两种选择无差异。

**步骤3**：考虑混合策略
设B以概率p选择进入。

A为高成本时：
- 容忍：E = p × 1 + (1-p) × 0 = p
- 对抗：E = p × 0 + (1-p) × 2 = 2(1-p)
比较：p = 2(1-p) ⇒ p = 2/3

A为低成本时：
- 容忍：E = p × 1 + (1-p) × 0 = p
- 对抗：E = p × 3 + (1-p) × 2 = 2 + p
比较：p < 2 + p，总是对抗

**贝叶斯纳什均衡**：
- A：高成本时容忍，低成本时对抗
- B：以2/3概率进入，1/3概率不进入

### 3.2.3 一级价格拍卖的均衡

**对称线性策略均衡**：
猜测均衡策略形式：bᵢ(vᵢ) = αvᵢ，其中0 < α < 1

**竞拍者1的问题**：
给定b₂(v₂) = αv₂，选择b₁最大化期望支付：
E[v₁ - b₁ | v₁, b₁ > αv₂] × P(b₁ > αv₂)

P(b₁ > αv₂) = P(v₂ < b₁/α) = b₁/α（因为v₂均匀分布于[0,1]）

E[v₁ - b₁ | v₁, b₁ > αv₂] = v₁ - b₁（独立于v₂）

期望支付：
(b₁/α)(v₁ - b₁)

最优化：
max_{b₁} (b₁/α)(v₁ - b₁) = (b₁v₁/α) - (b₁²/α)

一阶条件：
∂/∂b₁: v₁/α - 2b₁/α = 0 ⇒ v₁ - 2b₁ = 0 ⇒ b₁ = v₁/2

因此，α = 1/2

**贝叶斯纳什均衡**：
bᵢ(vᵢ) = vᵢ/2

**性质**：
- 竞拍者出价是估价的一半
- 出价随估价递增
- 估价最高的竞拍者获得物品（效率）

### 3.2.4 二级价格拍卖

**拍卖规则**：
- 竞拍者同时提交出价b₁和b₂
- 出价最高者获得物品
- 支付第二高的出价

**策略**：
 truthful bidding（如实出价）：bᵢ = vᵢ

**为什么如实出价是弱占优策略**：

情况1：bᵢ < vᵢ
- 如果bⱼ < bᵢ < vᵢ：本来可以vᵢ获得，现在bᵢ获得，损失vᵢ - bᵢ
- 如果bᵢ < bⱼ < vᵢ：本来可以vᵢ获得并支付bⱼ，现在不获得，损失vᵢ - bⱼ
- 如果bᵢ < bⱼ > vᵢ：本来也不获得，无损失

情况2：bᵢ > vᵢ
- 如果vᵢ < bⱼ < bᵢ：本来不获得，现在bᵢ获得并支付bⱼ，损失bⱼ - vᵢ
- 如果bⱼ > bᵢ > vᵢ：本来不获得，现在也不获得，无损失
- 如果bⱼ < vᵢ < bᵢ：本来vᵢ获得并支付bⱼ，现在bᵢ获得并支付bⱼ，无损失

**结论**：
如实出价 bᵢ = vᵢ 是弱占优策略

**贝叶斯纳什均衡**：
bᵢ(vᵢ) = vᵢ

**性质**：
- 如实出价是弱占优策略
- 估价最高的竞拍者获得物品（效率）
- 支付等于第二高的估价

### 3.2.5 收益等价定理

**定理**：在满足一定条件下（独立私人价值、风险中性、对称），任何标准拍卖形式（包括一级价格拍卖、二级价格拍卖、荷兰式拍卖、英式拍卖）在贝叶斯纳什均衡下的期望收益相同。

**理解**：
- 拍卖形式不影响卖家的期望收益
- 只影响收益的分配（谁得到剩余）
- 这是机制设计的重要结论

## 3.3 混合策略的重新解释

### 3.3.1 混合策略的不确定性

**问题**：
为什么参与者要随机选择策略？

**解释1**：真正的随机化
- 参与者真的以一定概率随机选择
- 使对手无法预测

**解释2**：对不确定性的反应
- 参与者对某些参数不确定
- 这种不确定性导致策略的随机性

### 3.3.2 混合策略的贝叶斯解释

**Harsanyi (1973) 的观点**：
混合策略纳什均衡可以解释为完全信息博弈的贝叶斯纳什均衡的极限。

**方法**：
1. 在博弈中引入微小的不完全信息
2. 每个参与者对自己的支付有微小的私人扰动
3. 参与者根据私人信息选择纯策略
4. 当扰动趋于零时，贝叶斯纳什均衡收敛于混合策略纳什均衡

### 3.3.3 例子：配硬币博弈

**原始博弈**：

| 玩家1 \ 玩家2 | 正面 | 反面 |
|:---:|:---:|:---:|
| **正面** | (1, -1) | (-1, 1) |
| **反面** | (-1, 1) | (1, -1) |

**混合策略均衡**：双方都以0.5概率选择正面，0.5概率选择反面

**引入微小扰动**：
- 玩家1对正面的偏好有微小扰动ε₁
- 玩家2对正面的偏好有微小扰动ε₂
- ε₁和ε₂在[-ε, ε]上均匀分布

**扰动后的博弈**：

| 玩家1 \ 玩家2 | 正面 | 反面 |
|:---:|:---:|:---:|
| **正面** | (1+ε₁, -1+ε₂) | (-1+ε₁, 1+ε₂) |
| **反面** | (-1, 1) | (1, -1) |

**贝叶斯纳什均衡**：
- 玩家1根据ε₁选择正面或反面
- 玩家2根据ε₂选择正面或反面
- 当ε₁ > 0时选择正面，ε₁ < 0时选择反面
- 当ε₂ < 0时选择正面，ε₂ > 0时选择反面

**极限**：
当ε → 0时，选择正面的概率→ 0.5，选择反面的概率→ 0.5
这就是原始博弈的混合策略纳什均衡

### 3.3.4 解释的意义

**理论意义**：
- 将混合策略转化为纯策略的不确定性
- 提供了混合策略的另一种理解
- 连接了完全信息和不完全信息博弈

**实践意义**：
- 解释了为什么在现实中很少看到真正的随机化
- 参与者的"随机"选择实际上反映了对私人信息的反应
- 更符合行为经济学的观察

### 3.3.5 应用

**应用1：军事策略**
- 军队部署的"随机性"可能反映了对敌人意图的不确定性
- 而不是真正的随机选择

**应用2：体育比赛**
- 点球的随机性可能反映了对守门员意图的不确定性
- 而不是真正的随机选择

**应用3：商业决策**
- 企业的策略选择可能反映了对市场和竞争对手的不确定性
- 而不是真正的随机选择

## 3.4 应用：拍卖与机制设计

### 3.4.1 拍卖的基本概念

**拍卖（Auction）**：
- 一种资源配置机制
- 通过竞争性出价确定资源配置和价格

**拍卖的类型**：
1. 一级价格拍卖（First-Price Auction）
2. 二级价格拍卖（Second-Price Auction）
3. 英式拍卖（English Auction）
4. 荷兰式拍卖（Dutch Auction）

**假设条件**：
- 独立私人价值（Independent Private Values, IPV）
- 风险中性（Risk Neutral）
- 对称（Symmetric）

### 3.4.2 一级价格拍卖（密封拍卖）

**规则**：
- 竞拍者同时提交密封出价
- 出价最高者获得物品
- 支付自己的出价

**均衡分析**（两人情况）：
- 对称线性策略均衡：b(v) = v/2

**n个竞拍者的均衡**：
对称均衡：b(v) = (n-1)v/n

**性质**：
- 出价低于估价
- 竞拍者越多，出价越接近估价
- 卖家的期望收益：E[第二高的估价]

### 3.4.3 二级价格拍卖（Vickrey拍卖）

**规则**：
- 竞拍者同时提交密封出价
- 出价最高者获得物品
- 支付第二高的出价

**均衡分析**：
- 如实出价 b(v) = v 是弱占优策略

**性质**：
- 如实出价是弱占优策略
- 简单且激励相容
- 估价最高者获得物品
- 卖家的期望收益：E[第二高的估价]

### 3.4.4 英式拍卖（升价拍卖）

**规则**：
- 拍卖师逐步提高价格
- 竞拍者可以随时退出
- 最后一个退出的竞拍者获得物品
- 支付第二高的出价

**均衡分析**：
- 策略：留在拍卖中直到价格达到估价
- 估价最高者获得物品
- 支付第二高的估价

**性质**：
- 与二级价格拍卖等价
- 透明度高
- 参与者可以看到其他竞拍者的行为

### 3.4.5 荷兰式拍卖（降价拍卖）

**规则**：
- 拍卖师从高价开始逐步降低价格
- 第一个接受价格的竞拍者获得物品
- 支付接受时的价格

**均衡分析**：
- 与一级价格拍卖等价
- 对称均衡：b(v) = (n-1)v/n

**性质**：
- 与一级价格拍卖等价
- 可以快速完成

### 3.4.6 收益等价定理的证明

**定理**：在独立私人价值、风险中性、对称的假设下，所有标准拍卖的期望收益相同。

**证明思路**：
1. 任何标准拍卖都可以看作是一个机制
2. 在激励相容和参与约束下，卖家的期望收益相同
3. 使用包络定理（Envelope Theorem）

**关键结论**：
- 拍卖形式不影响期望收益
- 只影响收益的分配
- 估价最高者获得物品（效率）

### 3.4.7 机制设计的基本概念

**机制设计（Mechanism Design）**：
- 逆向的博弈论
- 设计规则以实现特定的结果

**要素**：
1. 参与者集合
2. 消息空间（Message Space）
3. 分配规则（Allocation Rule）
4. 支付规则（Payment Rule）

**目标**：
1. 激励相容（Incentive Compatibility）：参与者如实报告
2. 参与约束（Participation Constraint）：参与者愿意参与
3. 效率（Efficiency）：实现帕累托最优
4. 收益最大化（Revenue Maximization）：最大化设计者收益

### 3.4.8 显示原理

**显示原理（Revelation Principle）**：
任何机制的均衡结果都可以通过一个直接机制实现，其中参与者如实报告自己的类型。

**理解**：
- 不需要考虑所有可能的复杂机制
- 只需要考虑直接机制（参与者直接报告类型）
- 大大简化了机制设计问题

**证明思路**：
1. 给定任意机制和均衡
2. 构造一个直接机制
3. 在直接机制中，如实报告是均衡
4. 两种机制的结果相同

### 3.4.9 Myerson拍卖

**Myerson (1981) 的最优拍卖**：
在独立私人价值、风险中性的假设下，最大化卖家期望收益的拍卖。

**关键步骤**：
1. 使用显示原理，考虑直接机制
2. 写出激励相容和参与约束
3. 优化卖家的期望收益
4. 得到最优拍卖形式

**最优拍卖的性质**：
- 保留价格（Reserve Price）：低于保留价格的竞拍者不参与
- 估价最高者（且高于保留价格）获得物品
- 支付取决于分布

### 3.4.10 其他应用

**应用1：公共物品提供**
- 机制设计可以解决搭便车问题
- Clarke-Groves 机制

**应用2：匹配问题**
- 学生-学校匹配
- 稳定匹配

**应用3：监管问题**
- 设计激励相容的监管机制
- 委托-代理问题

## 本章总结

### 核心要点

1. **贝叶斯博弈**是描述不完全信息静态博弈的基本形式，通过类型和信念来描述不确定性。

2. **海萨尼转换**将不完全信息转化为不完美信息，使贝叶斯博弈可以使用纳什均衡的概念。

3. **贝叶斯纳什均衡**是贝叶斯博弈的均衡概念，每种类型都选择最佳反应。

4. **混合策略**可以解释为不完全信息博弈的贝叶斯纳什均衡的极限。

5. **拍卖理论**是贝叶斯博弈的重要应用，不同拍卖形式的均衡分析揭示了拍卖的基本性质。

6. **收益等价定理**表明在标准假设下，不同拍卖形式的期望收益相同。

7. **机制设计**是逆向的博弈论，通过设计规则实现特定的结果。

### 重要概念表格

| 概念 | 定义 | 性质 |
|:---:|:---:|:---:|
| 贝叶斯博弈 | 不完全信息静态博弈 | 使用类型和信念描述不确定性 |
| 类型 | 参与者的私人信息 | 影响支付 |
| 信念 | 对他人类型的概率判断 | 通过贝叶斯法则更新 |
| 海萨尼转换 | 不完全信息→不完美信息 | 引入自然选择类型 |
| 贝叶斯纳什均衡 | 每种类型的最佳反应 | 考虑期望效用最大化 |
| 一级价格拍卖 | 出价最高者支付自己的出价 | 均衡出价低于估价 |
| 二级价格拍卖 | 出价最高者支付第二高出价 | 如实出价是弱占优策略 |
| 收益等价定理 | 标准拍卖期望收益相同 | 不影响效率 |
| 机制设计 | 设计规则实现目标 | 激励相容和参与约束 |
| 显示原理 | 任何机制等价于直接机制 | 大大简化设计问题 |
