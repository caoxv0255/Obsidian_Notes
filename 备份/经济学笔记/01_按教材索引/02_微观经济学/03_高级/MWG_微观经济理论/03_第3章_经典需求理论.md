# 第3章：经典需求理论

> 本章学习目标：
> - 理解需求函数
> - 掌握需求弹性
> - 学会分析需求的加总
> - 理解需求的比较静态分析

---

## 3.1 需求函数

### 3.1.1 马歇尔需求

**马歇尔需求（Marshallian Demand）**：
瓦尔拉斯需求，也称为马歇尔需求。

**符号**：
- x(p, w)：马歇尔需求函数
- p：价格向量
- w：财富

**性质**：
1. 零次齐次性：x(αp, αw) = x(p, w)
2. 瓦尔拉斯法则：p·x(p, w) = w
3. 对称性：替代矩阵对称
4. 负定性：替代矩阵负半定

### 3.1.2 希克斯需求

**希克斯需求（Hicksian Demand）**：
补偿需求，保持效用不变。

**符号**：
- h(p, u)：希克斯需求函数
- p：价格向量
- u：效用水平

**性质**：
1. 零次齐次性：h(αp, u) = h(p, u)
2. 无差异曲线性：u(h(p, u)) = u
3. 对称性：替代矩阵对称
4. 负定性：替代矩阵负半定
5. 补偿需求定律：∂hᵢ/∂pᵢ ≤ 0

### 3.1.3 需求的可积性

**可积性问题（Integrability Problem）**：
给定需求函数，是否存在效用函数生成该需求函数？

**可积性条件（Integrability Conditions）**：
1. 零次齐次性：x(αp, αw) = x(p, w)
2. 瓦尔拉斯法则：p·x(p, w) = w
3. 对称性：S(p, w) = S(p, w)ᵀ
4. 负定性：S(p, w)是负半定的

**定理**：
如果需求函数满足可积性条件，则存在效用函数生成该需求函数。

## 3.2 需求弹性

### 3.2.1 价格弹性

**价格弹性（Price Elasticity）**：
```
εᵢⱼ = (∂xᵢ/∂pⱼ)·(pⱼ/xᵢ)
```

**分类**：
1. **自价格弹性（Own-Price Elasticity）**：
   - εᵢᵢ = (∂xᵢ/∂pᵢ)·(pᵢ/xᵢ)
   - 通常εᵢᵢ < 0
   - |εᵢᵢ| > 1：富有弹性（Elastic）
   - |εᵢᵢ| < 1：缺乏弹性（Inelastic）
   - |εᵢᵢ| = 1：单位弹性（Unit Elastic）

2. **交叉价格弹性（Cross-Price Elasticity）**：
   - εᵢⱼ = (∂xᵢ/∂pⱼ)·(pⱼ/xᵢ)，i ≠ j
   - εᵢⱼ > 0：替代品（Substitutes）
   - εᵢⱼ < 0：互补品（Complements）
   - εᵢⱼ = 0：独立品（Independent）

### 3.2.2 收入弹性

**收入弹性（Income Elasticity）**：
```
εᵢᵂ = (∂xᵢ/∂w)·(w/xᵢ)
```

**分类**：
- εᵢᵂ > 0：正常商品（Normal Goods）
- εᵢᵂ < 0：劣等商品（Inferior Goods）
- εᵢᵂ > 1：奢侈品（Luxury Goods）
- 0 < εᵢᵂ < 1：必需品（Necessity Goods）

### 3.2.3 弹性矩阵

**弹性矩阵（Elasticity Matrix）**：
```
E = [εᵢⱼ]ᵢⱼ
```
**其中**：
- Eᵢᵢ：自价格弹性
- Eᵢⱼ（i ≠ j）：交叉价格弹性

**性质**：
- 对称性：如果偏好是位似的，则弹性矩阵对称
- 齐次性：Σⱼ εᵢⱼ + εᵢᵂ = 0

## 3.3 需求的加总

### 3.3.1 总体需求的定义

**总体需求（Aggregate Demand）**：
所有消费者需求的总和。

**公式**：
```
X(p, w) = Σₖ xᵏ(p, wᵏ)
```
**其中**：
- X(p, w)：总体需求
- xᵏ(p, wᵏ)：消费者k的需求
- w = (w¹, ..., wᴷ)：所有消费者的财富向量

### 3.3.2 总体需求的性质

**总体需求的性质（Properties of Aggregate Demand）**：

1. **零次齐次性（Homogeneity of Degree Zero）**：
   - X(αp, αw) = X(p, w)
   - 对于所有α > 0

2. **瓦尔拉斯法则（Walras' Law）**：
   - p·X(p, w) = Σₖ wᵏ
   - 总支出等于总收入

3. **连续性（Continuity）**：
   - 如果每个消费者的需求是连续的，则总体需求是连续的

4. **可微性（Differentiability）**：
   - 如果每个消费者的需求是可微的，则总体需求是可微的

### 3.3.3 总体需求的存在性问题

**总体需求的存在性问题（Existence Problem of Aggregate Demand）**：
- 给定总体需求函数，是否存在个体需求函数加总得到该总体需求？

**问题**：
- 总体需求函数不一定由代表性消费者生成
- 需要满足一定的条件

### 3.3.4 代表性消费者

**代表性消费者（Representative Consumer）**：
能够生成总体需求函数的假设消费者。

**存在条件**：
1. **偏好相同（Identical Preferences）**：
   - 所有消费者有相同的偏好

2. **位似偏好（Homothetic Preferences）**：
   - 偏好是位似的
   - 需求函数形式为x(p, w) = w·x(p, 1)

3. **拟线性偏好（Quasilinear Preferences）**：
   - 效用函数为u(x₀, x₁, ..., xₙ) = x₀ + Σᵢ vᵢ(xᵢ)
   - 对商品0的需求不依赖于财富

## 3.4 需求的比较静态分析

### 3.4.1 价格变化的效应

**价格变化的效应（Effects of Price Changes）**：
```
∂xᵢ/∂pⱼ = ∂hᵢ/∂pⱼ - xⱼ·(∂xᵢ/∂w)
```
- **替代效应（Substitution Effect）**：∂hᵢ/∂pⱼ
- **收入效应（Income Effect）**：-xⱼ·(∂xᵢ/∂w)

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "商品1" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 价格p₁下降
- 替代效应：从x₁'到x₁''
- 收入效应：从x₁''到x₁
- 总效应：从x₁'到x₁

### 3.4.2 收入变化的效应

**收入变化的效应（Effects of Income Changes）**：
```
∂xᵢ/∂w
```
- **正常商品（Normal Goods）**：∂xᵢ/∂w > 0
- **劣等商品（Inferior Goods）**：∂xᵢ/∂w < 0

**恩格尔曲线（Engel Curve）**：
- 描述需求与收入之间的关系
- 恩格尔曲线向上倾斜：正常商品
- 恩格尔曲线向下倾斜：劣等商品

### 3.4.3 恩格尔定律

**恩格尔定律（Engel's Law）**：
随着收入增加，食品支出占总支出的比例下降。

**公式**：
```
∂(p₁·x₁/w)/∂w < 0
```
**其中**：
- p₁·x₁：食品支出
- w：总收入

**含义**：
- 食品是必需品
- 食品的收入弹性小于1

## 3.5 需求的实证分析

### 3.5.1 需求估计

**需求估计（Demand Estimation）**：
估计需求函数的参数。

**方法**：
1. **线性需求函数（Linear Demand Function）**：
   - xᵢ = αᵢ + Σⱼ βᵢⱼ·pⱼ + γᵢ·w
   - 使用OLS估计参数

2. **对数线性需求函数（Log-Linear Demand Function）**：
   - ln xᵢ = αᵢ + Σⱼ βᵢⱼ·ln pⱼ + γᵢ·ln w
   - βᵢⱼ：价格弹性
   - γᵢ：收入弹性
   - 使用OLS估计参数

3. **几乎理想需求系统（Almost Ideal Demand System, AIDS）**：
   - Deaton和Muellbauer（1980）
   - 支出份额sᵢ = αᵢ + Σⱼ γᵢⱼ·ln pⱼ + βᵢ·ln(w/P)
   - P：价格指数

### 3.5.2 需求检验

**需求检验（Demand Testing）**：
检验需求函数是否满足理论约束。

**检验**：
1. **齐次性检验（Homogeneity Test）**：
   - H₀：Σⱼ βᵢⱼ + γᵢ = 0
   - 检验需求是否零次齐次

2. **对称性检验（Symmetry Test）**：
   - H₀：βᵢⱼ = βⱼᵢ
   - 检验替代矩阵是否对称

3. **负定性检验（Negative Semidefiniteness Test）**：
   - 检验替代矩阵是否负半定

## 3.6 本章总结

### 核心要点

1. **需求函数**：
   - 马歇尔需求：x(p, w)
   - 希克斯需求：h(p, u)
   - 需求的可积性：给定需求函数，是否存在效用函数生成该需求函数

2. **需求弹性**：
   - 价格弹性：εᵢⱼ = (∂xᵢ/∂pⱼ)·(pⱼ/xᵢ)
   - 收入弹性：εᵢᵂ = (∂xᵢ/∂w)·(w/xᵢ)
   - 弹性矩阵：E = [εᵢⱼ]ᵢⱼ

3. **需求的加总**：
   - 总体需求：X(p, w) = Σₖ xᵏ(p, wᵏ)
   - 总体需求的性质：零次齐次性、瓦尔拉斯法则、连续性、可微性
   - 代表性消费者：能够生成总体需求函数的假设消费者

4. **需求的比较静态分析**：
   - 价格变化的效应：替代效应 + 收入效应
   - 收入变化的效应：正常商品、劣等商品
   - 恩格尔定律：食品支出占总支出的比例随收入增加而下降

5. **需求的实证分析**：
   - 需求估计：线性需求函数、对数线性需求函数、几乎理想需求系统
   - 需求检验：齐次性检验、对称性检验、负定性检验
