## 18.1 帕累托最优

### 18.1.1 帕累托最优的定义

**帕累托最优（Pareto Optimality）**：
配置{xᵏ}ᵏ是帕累托最优的，如果不存在其他配置{x'ᵏ}ᵏ，使得：
1. 对于所有k，uᵏ(x'ᵏ) ≥ uᵏ(xᵏ)
2. 存在至少一个k，使得uᵏ(x'ᵏ) > uᵏ(xᵏ)

### 18.1.2 帕累托改进

**帕累托改进（Pareto Improvement）**：
配置{x'ᵏ}ᵏ是配置{xᵏ}ᵏ的帕累托改进，如果：
1. 对于所有k，uᵏ(x'ᵏ) ≥ uᵏ(xᵏ)
2. 存在至少一个k，使得uᵏ(x'ᵏ) > uᵏ(xᵏ)

### 18.1.3 帕累托最优的条件

**帕累托最优的条件（Conditions for Pareto Optimality）**：

**1. 交换经济（Exchange Economy）**：
```
MRSᵢⱼᵏ = MRSᵢⱼᵏ', 对于所有k, k'和i, j
```
**其中**：
- MRSᵢⱼᵏ：消费者k的边际替代率

**2. 生产经济（Production Economy）**：
```
MRSᵢⱼ = MRTᵢⱼ, 对于所有i, j
```
**其中**：
- MRSᵢⱼ：边际替代率
- MRTᵢⱼ：边际转换率

### 18.1.4 帕累托最优的存在性

**帕累托最优的存在性（Existence of Pareto Optimality）**：
- 如果偏好是连续的、凸的，消费集是紧集，则帕累托最优配置存在

**定理**：
如果偏好关系≿ᵏ是连续的、凸的，消费集Xᵏ是紧集，则帕累托最优配置存在。

## 18.2 福利定理

### 18.2.1 第一福利定理

**第一福利定理（First Welfare Theorem）**：
竞争均衡是帕累托最优的。

**定理**：
如果{(p, xᵏ)ᵏ}是竞争均衡，且偏好是局部非饱和的，则{xᵏ}ᵏ是帕累托最优的。

**条件**：
1. 局部非饱和性（Local Nonsatiation）
2. 完全市场（Complete Markets）
3. 价格接受者（Price Taking）

**含义**：
- 市场机制可以实现帕累托最优
- 不需要政府干预

### 18.2.2 第二福利定理

**第二福利定理（Second Welfare Theorem）**：
任何帕累托最优配置都可以通过适当的一次性转移支付实现为竞争均衡。

**定理**：
如果{xᵏ}ᵏ是帕累托最优配置，偏好是连续的、凸的，消费集是凸集，则存在价格向量p和转移支付{tᵏ}ᵏ，使得{(p, xᵏ)ᵏ}是竞争均衡。

**条件**：
1. 偏好是连续的、凸的
2. 消费集是凸集
3. 一次性转移支付（Lump-Sum Transfers）

**含义**：
- 可以通过再分配实现任何帕累托最优配置
- 效率与公平可以分离

## 18.3 社会选择

### 18.3.1 社会福利函数

**社会福利函数（Social Welfare Function, SWF）**：
将个体效用映射为社会福利的函数。

**公式**：
```
W = W(u¹, u², ..., uᴷ)
```
**其中**：
- W：社会福利
- uᵏ：消费者k的效用

### 18.3.2 社会福利函数的类型

**社会福利函数的类型（Types of Social Welfare Function）**：

1. **功利主义社会福利函数（Utilitarian Social Welfare Function）**：
   ```
   W = Σₖ uᵏ
   ```

2. **罗尔斯社会福利函数（Rawlsian Social Welfare Function）**：
   ```
   W = minₖ uᵏ
   ```

3. **伯格森-萨缪尔森社会福利函数（Bergson-Samuelson Social Welfare Function）**：
   ```
   W = W(u¹, u², ..., uᴷ)
   ```

4. **纳什社会福利函数（Nash Social Welfare Function）**：
   ```
   W = Πₖ uᵏ
   ```

### 18.3.3 社会最优

**社会最优（Social Optimum）**：
配置{xᵏ}ᵏ使得社会福利最大化。

**问题**：
```
max_{xᵏ} W(u¹(x¹), ..., uᴷ(xᴷ))
s.t. Σₖ xᵏ ≤ Σⱼ yⱼ + Σₖ ωᵏ
```
**其中**：
- W：社会福利函数
- uᵏ(xᵏ)：消费者k的效用
- yⱼ：企业j的生产
- ωᵏ：消费者k的禀赋

## 18.4 不平等

### 18.4.1 不平等的度量

**不平等的度量（Measurement of Inequality）**：

1. **基尼系数（Gini Coefficient）**：
   ```
   G = (1 / (2n²μ)) Σᵢ Σⱼ |xᵢ - xⱼ|
   ```
   **其中**：
   - G：基尼系数
   - n：人口数量
   - μ：平均收入
   - xᵢ：个体i的收入

2. **泰尔指数（Theil Index）**：
   ```
   T = (1 / n) Σᵢ (xᵢ / μ)·ln(xᵢ / μ)
   ```
   **其中**：
   - T：泰尔指数
   - xᵢ：个体i的收入
   - μ：平均收入

3. **方差系数（Coefficient of Variation, CV）**：
   ```
   CV = σ / μ
   ```
   **其中**：
   - CV：方差系数
   - σ：标准差
   - μ：平均收入

### 18.4.2 不平等的分解

**不平等的分解（Decomposition of Inequality）**：
- 按组分解（Between-Group Inequality）
- 组内分解（Within-Group Inequality）

**泰尔指数的分解（Theil Index Decomposition）**：
```
T = T_B + T_W
```
**其中**：
- T：总体不平等
- T_B：组间不平等
- T_W：组内不平等

## 18.5 本章总结

### 核心要点

1. **帕累托最优**：
   - 帕累托最优：不存在其他配置使所有人至少不差，且至少一人更好
   - 帕累托最优的条件：交换经济（MRSᵢⱼᵏ = MRSᵢⱼᵏ'），生产经济（MRSᵢⱼ = MRTᵢⱼ）
   - 帕累托最优的存在性：偏好连续、凸，消费集紧

2. **福利定理**：
   - 第一福利定理：竞争均衡是帕累托最优的
   - 第二福利定理：任何帕累托最优配置都可以通过适当的一次性转移支付实现为竞争均衡

3. **社会选择**：
   - 社会福利函数：W = W(u¹, u², ..., uᴷ)
   - 社会福利函数的类型：功利主义、罗尔斯、伯格森-萨缪尔森、纳什
   - 社会最优：配置{xᵏ}ᵏ使得社会福利最大化

4. **不平等**：
   - 不平等的度量：基尼系数、泰尔指数、方差系数
   - 不平等的分解：T = T_B + T_W
