## 12.1 古诺模型

### 12.1.1 古诺模型的定义

**古诺模型（Cournot Model）**：
企业同时选择产量，市场价格由总产量决定。

**假设**：
1. 企业i选择产量qᵢ
2. 市场价格p = P(Q)，其中Q = Σᵢ qᵢ
3. 企业i的成本函数cᵢ(qᵢ)
4. 企业i的收益πᵢ = p·qᵢ - cᵢ(qᵢ)

### 12.1.2 古诺均衡

**古诺均衡（Cournot Equilibrium）**：
产量组合q*是古诺均衡，如果对于所有i和所有qᵢ，有：
```
πᵢ(q*ᵢ, q*₋ᵢ) ≥ πᵢ(qᵢ, q*₋ᵢ)
```
**其中**：
- q*ᵢ：企业i的均衡产量
- q*₋ᵢ：其他企业的均衡产量
- πᵢ(q*ᵢ, q*₋ᵢ)：企业i在产量组合(q*ᵢ, q*₋ᵢ)下的收益

### 12.1.3 古诺均衡的计算

**古诺均衡的计算（Computation of Cournot Equilibrium）**：

**一阶条件（First-Order Conditions）**：
对于内点解：
```
∂πᵢ/∂qᵢ = P(Q) + P'(Q)·qᵢ - c'ᵢ(qᵢ) = 0
```
**其中**：
- P(Q)：市场价格
- P'(Q)：价格对总产量的导数
- c'ᵢ(qᵢ)：边际成本

**含义**：
- 边际收益等于边际成本
- 企业i的产量选择影响市场价格

### 12.1.4 古诺均衡的性质

**古诺均衡的性质（Properties of Cournot Equilibrium）**：

1. **存在性（Existence）**：
   - 如果需求函数P(Q)是连续的、递减的，成本函数cᵢ(qᵢ)是凸的，则古诺均衡存在

2. **唯一性（Uniqueness）**：
   - 如果需求函数P(Q)是严格凹的，成本函数cᵢ(qᵢ)是严格凸的，则古诺均衡唯一

3. **对称性（Symmetry）**：
   - 如果企业对称（相同的需求和成本），则古诺均衡对称
   - 即q*₁ = q*₂ = ... = q*ₙ

### 12.1.5 古诺均衡与完全竞争、垄断的比较

**古诺均衡与完全竞争、垄断的比较（Comparison with Perfect Competition and Monopoly）**：

**1. 完全竞争（Perfect Competition）**：
- 企业是价格接受者
- p = MC
- 产量最大，价格最低

**2. 垄断（Monopoly）**：
- 企业选择垄断产量
- MR = MC
- 产量最小，价格最高

**3. 古诺均衡（Cournot Equilibrium）**：
- 企业同时选择产量
- MR = MC（考虑其他企业的产量）
- 产量和价格介于完全竞争和垄断之间

## 12.2 伯特兰模型

### 12.2.1 伯特兰模型的定义

**伯特兰模型（Bertrand Model）**：
企业同时选择价格，产量由需求决定。

**假设**：
1. 企业i选择价格pᵢ
2. 企业i的产量qᵢ = Dᵢ(p₁, ..., pₙ)
3. 企业i的成本函数cᵢ(qᵢ)
4. 企业i的收益πᵢ = pᵢ·qᵢ - cᵢ(qᵢ)

### 12.2.2 伯特兰均衡

**伯特兰均衡（Bertrand Equilibrium）**：
价格组合p*是伯特兰均衡，如果对于所有i和所有pᵢ，有：
```
πᵢ(p*ᵢ, p*₋ᵢ) ≥ πᵢ(pᵢ, p*₋ᵢ)
```
**其中**：
- p*ᵢ：企业i的均衡价格
- p*₋ᵢ：其他企业的均衡价格
- πᵢ(p*ᵢ, p*₋ᵢ)：企业i在价格组合(p*ᵢ, p*₋ᵢ)下的收益

### 12.2.3 伯特兰悖论

**伯特兰悖论（Bertrand Paradox）**：
如果企业生产同质产品，且边际成本相同，则伯特兰均衡是完全竞争均衡。

**结果**：
- p* = MC
- Q* = D(MC)
- π* = 0

**含义**：
- 即使只有两个企业，伯特兰均衡也是完全竞争均衡
- 与现实不符（现实中寡头垄断企业的利润为正）

### 12.2.4 伯特兰悖论的解决方案

**伯特兰悖论的解决方案（Solutions to Bertrand Paradox）**：

1. **产品差异化（Product Differentiation）**：
   - 企业生产差异化产品
   - 伯特兰均衡不是完全竞争均衡
   - 价格高于边际成本

2. **生产能力约束（Capacity Constraints）**：
   - 企业有生产能力约束
   - 伯特兰均衡不是完全竞争均衡
   - 价格高于边际成本

3. **重复博弈（Repeated Game）**：
   - 企业重复博弈
   - 可以维持合谋
   - 价格高于边际成本

4. **不完全信息（Incomplete Information）**：
   - 企业有私有信息（例如：成本）
   - 伯特兰均衡不是完全竞争均衡
   - 价格高于边际成本

## 12.3 斯塔克尔伯格模型

### 12.3.1 斯塔克尔伯格模型的定义

**斯塔克尔伯格模型（Stackelberg Model）**：
企业依次选择产量（领导者-跟随者博弈）。

**假设**：
1. 领导者企业1先选择产量q₁
2. 跟随者企业2观察到q₁后选择产量q₂
3. 市场价格p = P(Q)，其中Q = q₁ + q₂
4. 企业i的成本函数cᵢ(qᵢ)

### 12.3.2 斯塔克尔伯格均衡

**斯塔克尔伯格均衡（Stackelberg Equilibrium）**：
产量组合(q*₁, q*₂)是斯塔克尔伯格均衡，如果：
1. 跟随者企业2给定领导者产量q*₁，选择最优产量q*₂(q*₁)
2. 领导者企业1预测跟随者企业2的反应，选择最优产量q*₁

**方法**：
1. 求解跟随者企业2的反应函数q₂(q₁)
2. 将q₂(q₁)代入领导者企业1的收益函数
3. 求解领导者企业1的最优产量q*₁
4. 计算跟随者企业2的最优产量q*₂ = q₂(q*₁)

### 12.3.3 斯塔克尔伯格均衡的性质

**斯塔克尔伯格均衡的性质（Properties of Stackelberg Equilibrium）**：

1. **先动优势（First-Mover Advantage）**：
   - 领导者企业1的产量高于跟随者企业2的产量
   - 领导者企业1的收益高于跟随者企业2的收益

2. **与古诺均衡的比较（Comparison with Cournot Equilibrium）**：
   - 领导者企业1的产量高于古诺均衡产量
   - 跟随者企业2的产量低于古诺均衡产量
   - 总产量高于古诺均衡总产量
   - 价格低于古诺均衡价格

## 12.4 合谋

### 12.4.1 合谋的定义

**合谋（Collusion）**：
企业协调行动，实现联合利润最大化。

### 12.4.2 卡特尔

**卡特尔（Cartel）**：
企业组成的联盟，协调产量和价格，实现联合利润最大化。

**问题**：
```
max_{q₁, ..., qₙ} Σᵢ [P(Q)·qᵢ - cᵢ(qᵢ)]
s.t. qᵢ ≥ 0
```
**其中**：
- Q = Σᵢ qᵢ：总产量
- P(Q)：市场价格
- cᵢ(qᵢ)：企业i的成本函数

**一阶条件**：
```
P(Q) + P'(Q)·Q - c'ᵢ(qᵢ) = 0, 对于所有i
```
**含义**：
- 联合边际收益等于边际成本
- 所有企业的边际成本相等

### 12.4.3 卡特尔的不稳定性

**卡特尔的不稳定性（Instability of Cartel）**：
- 每个企业有动力偏离卡特尔协议
- 偏离卡特尔协议可以增加企业利润
- 除非有有效的监督和惩罚机制

**例子**：
- 如果其他企业遵守卡特尔协议，企业i可以增加产量，增加利润
- 如果所有企业都偏离，则回到古诺均衡

### 12.4.4 重复博弈与合谋

**重复博弈与合谋（Repeated Game and Collusion）**：
- 如果企业重复博弈，可以维持合谋
- 使用触发策略（Trigger Strategy）
- 如果企业偏离合谋，其他企业惩罚（回到古诺均衡）

**无名氏定理（Folk Theorem）**：
如果企业足够耐心（折现因子接近1），则任何严格大于古诺均衡利润的利润都可以作为重复博弈的均衡。

## 12.5 本章总结

### 核心要点

1. **古诺模型**：
   - 古诺模型：企业同时选择产量，市场价格由总产量决定
   - 古诺均衡：∂πᵢ/∂qᵢ = P(Q) + P'(Q)·qᵢ - c'ᵢ(qᵢ) = 0
   - 古诺均衡的性质：存在性、唯一性、对称性
   - 古诺均衡与完全竞争、垄断的比较

2. **伯特兰模型**：
   - 伯特兰模型：企业同时选择价格，产量由需求决定
   - 伯特兰均衡：πᵢ(p*ᵢ, p*₋ᵢ) ≥ πᵢ(pᵢ, p*₋ᵢ)
   - 伯特兰悖论：同质产品、相同边际成本 ⇒ 完全竞争均衡
   - 伯特兰悖论的解决方案：产品差异化、生产能力约束、重复博弈、不完全信息

3. **斯塔克尔伯格模型**：
   - 斯塔克尔伯格模型：企业依次选择产量（领导者-跟随者博弈）
   - 斯塔克尔伯格均衡：跟随者给定领导者产量，选择最优产量；领导者预测跟随者反应，选择最优产量
   - 斯塔克尔伯格均衡的性质：先动优势
   - 与古诺均衡的比较

4. **合谋**：
   - 合谋：企业协调行动，实现联合利润最大化
   - 卡特尔：企业组成的联盟，协调产量和价格
   - 卡特尔的不稳定性：每个企业有动力偏离卡特尔协议
   - 重复博弈与合谋：使用触发策略维持合谋
