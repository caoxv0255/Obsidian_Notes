# 第6章：竞争均衡

> 本章学习目标：
> - 理解帕累托最优
> - 掌握福利定理
> - 学会分析竞争均衡的性质
> - 理解均衡的存在性

---

## 6.1 帕累托最优

### 6.1.1 帕累托最优的定义

**帕累托最优（Pareto Optimal）**：
配置{xᵏ}ᵏ是帕累托最优的，如果不存在其他配置{x'ᵏ}ᵏ，使得：
1. 对于所有k，uᵏ(x'ᵏ) ≥ uᵏ(xᵏ)
2. 存在至少一个k，使得uᵏ(x'ᵏ) > uᵏ(xᵏ)

### 6.1.2 帕累托最优的条件

**帕累托最优的条件（Conditions for Pareto Optimality）**：

**1. 交换经济（Exchange Economy）**：
```
MRSᵢⱼᵏ = MRSᵢⱼᵏ', 对于所有k, k'和i, j
```
**其中**：
- MRSᵢⱼᵏ：消费者k的边际替代率

**含义**：
- 所有消费者的边际替代率相等
- 资源配置达到帕累托最优

**2. 生产经济（Production Economy）**：
```
MRSᵢⱼ = MRTᵢⱼ, 对于所有i, j
```
**其中**：
- MRSᵢⱼ：边际替代率
- MRTᵢⱼ：边际转换率

**含义**：
- 边际替代率等于边际转换率
- 资源配置达到帕累托最优

### 6.1.3 帕累托最优的存在性

**帕累托最优的存在性（Existence of Pareto Optimality）**：
- 如果偏好是连续的、凸的，消费集是紧集，则帕累托最优配置存在

**定理**：
如果偏好关系≿ᵏ是连续的、凸的，消费集Xᵏ是紧集，则帕累托最优配置存在。

## 6.2 福利定理

### 6.2.1 第一福利定理

**第一福利定理（First Welfare Theorem）**：
竞争均衡是帕累托最优的。

**定理**：
如果{(p, xᵏ)ᵏ}是竞争均衡，且偏好是局部非饱和的，则{xᵏ}ᵏ是帕累托最优的。

**条件**：
1. 局部非饱和性（Local Nonsatiation）
2. 完全市场（Complete Markets）
3. 价格接受者（Price Taking）

**含义**：
- 市场机制可以实现帕累托最优
- 不需要政府干预

### 6.2.2 第二福利定理

**第二福利定理（Second Welfare Theorem）**：
任何帕累托最优配置都可以通过适当的一次性转移支付实现为竞争均衡。

**定理**：
如果{xᵏ}ᵏ是帕累托最优配置，偏好是连续的、凸的，消费集是凸集，则存在价格向量p和转移支付{tᵏ}ᵏ，使得{(p, xᵏ)ᵏ}是竞争均衡。

**条件**：
1. 偏好是连续的、凸的
2. 消费集是凸集
3. 一次性转移支付（Lump-Sum Transfers）

**含义**：
- 可以通过再分配实现任何帕累托最优配置
- 效率与公平可以分离

## 6.3 竞争均衡的性质

### 6.3.1 竞争均衡的定义

**竞争均衡（Competitive Equilibrium, CE）**：
价格向量p和配置{xᵏ, yⱼ}ᵏⱼ，使得：
1. 消费者最大化效用：xᵏ ∈ xᵏ(p, p·ωᵏ + Σⱼ θⱼₖ·p·yⱼ)
2. 生产者最大化利润：yⱼ ∈ yⱼ(p)
3. 市场出清：Σₖ xᵏ = Σⱼ yⱼ + Σₖ ωᵏ

**其中**：
- p：价格向量
- xᵏ：消费者k的消费
- yⱼ：企业j的生产
- ωᵏ：消费者k的禀赋
- θⱼₖ：消费者k在企业j中的股权

### 6.3.2 竞争均衡的性质

**竞争均衡的性质（Properties of Competitive Equilibrium）**：

1. **有效性（Efficiency）**：
   - 第一福利定理：竞争均衡是帕累托最优的

2. **存在性（Existence）**：
   - 如果偏好是连续的、凸的，消费集和生产集是紧集，则竞争均衡存在

3. **唯一性（Uniqueness）**：
   - 如果偏好是严格凸的，生产集是严格凸的，则竞争均衡唯一

4. **稳定性（Stability）：
   - 如果偏好和生产函数满足一定条件，则竞争均衡稳定

### 6.3.3 均衡的比较静态分析

**均衡的比较静态分析（Comparative Statics of Equilibrium）**：
- 分析均衡价格和配置随参数变化的变化

**方法**：
1. **隐性函数定理（Implicit Function Theorem）**：
   - 假设均衡方程可微
   - 计算均衡对参数的导数

2. **数值方法（Numerical Methods）**：
   - 使用数值方法求解均衡
   - 分析均衡对参数变化的敏感性

## 6.4 均衡的存在性

### 6.4.1 存在性定理

**存在性定理（Existence Theorem）**：

**定理（Arrow-Debreu）**：
如果满足以下条件，则竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 生产集是凸集、闭集
4. 0 ∈ Yⱼ（每个企业可以不生产）
5. Y ∩ (-Y) = {0}（没有免费午餐）
6. 禀赋严格为正

**证明思路**：
1. 构造超额需求函数Z(p)
2. 证明Z(p)连续
3. 应用不动点定理（Brouwer不动点定理或Kakutani不动点定理）
4. 证明存在p*，使得Z(p*) = 0

### 6.4.2 超额需求函数

**超额需求函数（Excess Demand Function, Z(p)）**：
```
Z(p) = Σₖ xᵏ(p, p·ωᵏ) - Σⱼ yⱼ(p) - Σₖ ωᵏ
```
**其中**：
- Z(p)：超额需求函数
- xᵏ(p, p·ωᵏ)：消费者k的需求
- yⱼ(p)：企业j的供给
- ωᵏ：消费者k的禀赋

**性质**：
1. **零次齐次性（Homogeneity of Degree Zero）**：
   - Z(αp) = Z(p)，对于所有α > 0

2. **瓦尔拉斯法则（Walras' Law）**：
   - p·Z(p) = 0

3. **连续性（Continuity）**：
   - Z(p)是连续的

### 6.4.3 不动点定理

**不动点定理（Fixed Point Theorem）**：

**Brouwer不动点定理（Brouwer's Fixed Point Theorem）**：
如果f: S → S是连续的，S是紧凸集，则存在x* ∈ S，使得f(x*) = x*。

**Kakutani不动点定理（Kakutani's Fixed Point Theorem）**：
如果F: S → S是上半连续的、紧凸值的，S是紧凸集，则存在x* ∈ S，使得x* ∈ F(x*)。

**应用**：
- 使用不动点定理证明竞争均衡的存在性
- 定义价格空间的映射
- 证明映射满足不动点定理的条件
- 得到均衡价格p*

## 6.5 本章总结

### 核心要点

1. **帕累托最优**：
   - 帕累托最优：不存在其他配置使所有人至少不差，且至少一人更好
   - 帕累托最优的条件：交换经济（MRSᵢⱼᵏ = MRSᵢⱼᵏ'），生产经济（MRSᵢⱼ = MRTᵢⱼ）
   - 帕累托最优的存在性：偏好连续、凸，消费集紧

2. **福利定理**：
   - 第一福利定理：竞争均衡是帕累托最优的
   - 第二福利定理：任何帕累托最优配置都可以通过适当的一次性转移支付实现为竞争均衡

3. **竞争均衡的性质**：
   - 竞争均衡的定义：价格向量p和配置{xᵏ, yⱼ}ᵏⱼ
   - 竞争均衡的性质：有效性、存在性、唯一性、稳定性
   - 均衡的比较静态分析

4. **均衡的存在性**：
   - 存在性定理（Arrow-Debreu）：偏好连续、凸，消费集和生产集满足一定条件
   - 超额需求函数：Z(p) = Σₖ xᵏ - Σⱼ yⱼ - Σₖ ωᵏ
   - 不动点定理：Brouwer不动点定理、Kakutani不动点定理

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 帕累托最优 | 不存在其他配置使所有人至少不差，且至少一人更好 | 效率标准 |
| 第一福利定理 | 竞争均衡是帕累托最优的 | 市场机制有效 |
| 第二福利定理 | 任何帕累托最优配置都可以通过适当的一次性转移支付实现为竞争均衡 | 效率与公平分离 |
| 竞争均衡 | 价格向量p和配置{xᵏ, yⱼ}ᵏⱼ | 市场均衡 |
| 超额需求函数 | Z(p) = Σₖ xᵏ - Σⱼ yⱼ - Σₖ ωᵏ | 市场超额需求 |
| 瓦尔拉斯法则 | p·Z(p) = 0 | 总超额需求的价值为零 |
| 零次齐次性 | Z(αp) = Z(p) | 价格标准化 |
| Arrow-Debreu定理 | 偏好连续、凸，消费集和生产集满足一定条件 | 均衡存在性 |
| Brouwer不动点定理 | 连续映射在紧凸集上有不动点 | 不动点定理 |
| Kakutani不动点定理 | 上半连续、紧凸值映射在紧凸集上有不动点 | 不动点定理 |
