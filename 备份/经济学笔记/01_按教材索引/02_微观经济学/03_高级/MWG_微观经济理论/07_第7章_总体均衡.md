## 7.1 纯交换经济

### 7.1.1 纯交换经济的定义

**纯交换经济（Pure Exchange Economy）**：
E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ}
- Xᵏ：消费者k的消费集
- ≿ᵏ：消费者k的偏好关系
- ωᵏ：消费者k的禀赋

### 7.1.2 纯交换经济的均衡

**纯交换经济的均衡（Equilibrium of Pure Exchange Economy）**：
价格向量p和配置{xᵏ}ᵏ，使得：
1. 消费者最大化效用：xᵏ ∈ xᵏ(p, p·ωᵏ)
2. 市场出清：Σₖ xᵏ = Σₖ ωᵏ

**其中**：
- p：价格向量
- xᵏ(p, p·ωᵏ)：消费者k的需求
- ωᵏ：消费者k的禀赋

### 7.1.3 埃奇沃斯盒

**埃奇沃斯盒（Edgeworth Box）**：
表示纯交换经济的图形工具。

**构造**：
- 横轴：商品1
- 纵轴：商品2
- 消费者1的原点在左下角
- 消费者2的原点在右上角
- 盒子的宽度 = 商品1的总禀赋
- 盒子的高度 = 商品2的总禀赋

**性质**：
- 盒子内的每一点表示一个配置
- 从消费者1的原点到点的向量表示消费者1的消费
- 从消费者2的原点到点的向量表示消费者2的消费

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "商品1" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- ω = (ω₁, ω₂)：总禀赋
- 契约曲线（Contract Curve）：帕累托最优配置的集合
- 核心合约集（Core）：帕累托最优且个人理性的配置集合

## 7.2 生产经济

### 7.2.1 生产经济的定义

**生产经济（Production Economy）**：
E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ, (Yⱼ)ⱼ₌₁ᴶ}
- Xᵏ：消费者k的消费集
- ≿ᵏ：消费者k的偏好关系
- ωᵏ：消费者k的禀赋
- Yⱼ：企业j的生产集

### 7.2.2 生产经济的均衡

**生产经济的均衡（Equilibrium of Production Economy）**：
价格向量p和配置{xᵏ, yⱼ}ᵏⱼ，使得：
1. 消费者最大化效用：xᵏ ∈ xᵏ(p, p·ωᵏ + Σⱼ θⱼₖ·p·yⱼ)
2. 生产者最大化利润：yⱼ ∈ yⱼ(p)
3. 市场出清：Σₖ xᵏ = Σⱼ yⱼ + Σₖ ωᵏ

**其中**：
- p：价格向量
- xᵏ：消费者k的消费
- yⱼ：企业j的生产
- ωᵏ：消费者k的禀赋
- θⱼₖ：消费者k在企业j中的股权

### 7.2.3 生产集的性质

**生产集的性质（Properties of Production Set）**：

1. **闭集（Closedness）**：
   - Y是闭集

2. **凸集（Convexity）**：
   - Y是凸集（规模报酬不递增）

3. **包含零（Contains Zero）**：
   - 0 ∈ Y（可以不生产）

4. **无免费午餐（No Free Lunch）**：
   - Y ∩ (-Y) = {0}

5. **不可逆性（Irreversibility）**：
   - 如果y ∈ Y且y ≠ 0，则-y ∉ Y

6. **自由处置（Free Disposal）**：
   - 如果y ∈ Y且y' ≤ y，则y' ∈ Y

## 7.3 均衡的存在性

### 7.3.1 纯交换经济的存在性

**纯交换经济的存在性（Existence in Pure Exchange Economy）**：

**定理**：
如果满足以下条件，则纯交换经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 禀赋严格为正

**证明思路**：
1. 构造超额需求函数Z(p)
2. 证明Z(p)连续
3. 应用不动点定理
4. 证明存在p*，使得Z(p*) = 0

### 7.3.2 生产经济的存在性

**生产经济的存在性（Existence in Production Economy）**：

**定理**：
如果满足以下条件，则生产经济的竞争均衡存在：
1. 偏好是连续的、凸的、局部非饱和的
2. 消费集是凸集、闭集、下有界
3. 生产集是凸集、闭集、0 ∈ Y、Y ∩ (-Y) = {0}
4. 禀赋严格为正

**证明思路**：
1. 构造超额需求函数Z(p)
2. 证明Z(p)连续
3. 应用不动点定理
4. 证明存在p*，使得Z(p*) = 0

### 7.3.3 存在性条件的讨论

**存在性条件的讨论（Discussion of Existence Conditions）**：

1. **连续性（Continuity）**：
   - 偏好和需求函数的连续性
   - 保证超额需求函数连续

2. **凸性（Convexity）**：
   - 偏好和生产集的凸性
   - 保证需求和生产集的凸性

3. **边界条件（Boundary Conditions）**：
   - 消费集和生产集的有界性
   - 保证均衡存在

4. **正禀赋（Positive Endowments）**：
   - 每个消费者拥有正禀赋
   - 保证消费者有购买力

## 7.4 均衡的唯一性

### 7.4.1 唯一性条件

**唯一性条件（Uniqueness Conditions）**：
- 如果偏好是严格凸的，生产函数是严格凹的，则均衡唯一

**定理**：
如果满足以下条件，则竞争均衡唯一：
1. 偏好是严格凸的（无差异曲线严格凸）
2. 生产函数是严格凹的（规模报酬严格递减）
3. 超额需求函数满足强总替代性（Strong Gross Substitutability）

### 7.4.2 总替代性

**总替代性（Gross Substitutability）**：
商品i的价格上升，对其他商品j的需求增加。

**定义**：
超额需求函数Z(p)满足总替代性，如果对于所有i ≠ j，∂Zⱼ(p)/∂pᵢ > 0。

**强总替代性（Strong Gross Substitutability）**：
对于所有i ≠ j，∂Zⱼ(p)/∂pᵢ > 0，且∂Zᵢ(p)/∂pᵢ < 0。

**定理**：
如果超额需求函数满足强总替代性，则均衡唯一。

### 7.4.3 唯一性的反例

**唯一性的反例（Counterexamples to Uniqueness）**：
- 如果偏好不是严格凸的，则可能存在多个均衡
- 如果生产函数不是严格凹的，则可能存在多个均衡

**例子**：
- 同质商品的市场：可能有多个均衡
- 规模报酬递增的生产：可能有多个均衡

## 7.5 均衡的稳定性

### 7.5.1 稳定性的定义

**稳定性（Stability）**：
经济系统受到扰动后，能够回到均衡状态。

### 7.5.2 瓦尔拉斯调整过程

**瓦尔拉斯调整过程（Walrasian Tatonnement）**：
价格根据超额需求调整。

**调整规则**：
```
dp/dt = λ·Z(p)
```
**其中**：
- p：价格向量
- Z(p)：超额需求函数
- λ：调整速度（λ > 0）

**含义**：
- 如果商品i的超额需求为正，则价格pᵢ上升
- 如果商品i的超额需求为负，则价格pᵢ下降
- 超额需求为零时，价格不变

### 7.5.3 稳定性条件

**稳定性条件（Stability Conditions）**：
- 如果超额需求函数满足总替代性，则瓦尔拉斯调整过程是稳定的

**定理**：
如果超额需求函数Z(p)满足总替代性，则瓦尔拉斯调整过程全局稳定。

**证明思路**：
1. 构造李雅普诺夫函数
2. 证明李雅普诺夫函数沿调整路径递减
3. 证明系统收敛到均衡

## 7.6 本章总结

### 核心要点

1. **纯交换经济**：
   - 纯交换经济：E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ}
   - 纯交换经济的均衡：价格向量p和配置{xᵏ}ᵏ
   - 埃奇沃斯盒：表示纯交换经济的图形工具

2. **生产经济**：
   - 生产经济：E = {(Xᵏ, ≿ᵏ, ωᵏ)ₖ₌₁ᴷ, (Yⱼ)ⱼ₌₁ᴶ}
   - 生产经济的均衡：价格向量p和配置{xᵏ, yⱼ}ᵏⱼ
   - 生产集的性质：闭集、凸集、包含零、无免费午餐、不可逆性、自由处置

3. **均衡的存在性**：
   - 纯交换经济的存在性：偏好连续、凸，消费集满足一定条件
   - 生产经济的存在性：偏好和生产集满足一定条件
   - 存在性条件的讨论：连续性、凸性、边界条件、正禀赋

4. **均衡的唯一性**：
   - 唯一性条件：偏好严格凸，生产函数严格凹
   - 总替代性：∂Zⱼ(p)/∂pᵢ > 0，对于i ≠ j
   - 唯一性的反例：偏好不严格凸，生产函数不严格凹

5. **均衡的稳定性**：
   - 稳定性：经济系统受到扰动后，能够回到均衡状态
   - 瓦尔拉斯调整过程：dp/dt = λ·Z(p)
   - 稳定性条件：超额需求函数满足总替代性
