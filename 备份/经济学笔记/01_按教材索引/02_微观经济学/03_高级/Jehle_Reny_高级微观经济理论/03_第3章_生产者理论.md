# 第3章：生产者理论

本章系统介绍生产者理论，这是微观经济学的另一个核心组成部分。通过本章的学习，读者应该能够：

1. 理解生产技术的数学表述及其性质
2. 掌握生产函数的概念及其与生产可能性集的关系
3. 分析利润最大化问题及其一阶和二阶条件
4. 推导供给函数和要素需求函数及其性质
5. 理解利润函数及其性质
6. 掌握成本最小化问题与利润最大化问题的对偶性

10. 分析规模报酬及其对成本函数的影响

## 3.1 生产技术

### 3.1.1 生产可能性集

**定义3.1.1（生产可能性集）**：生产可能性集Y ⊆ ℝ^n是厂商所有可行生产计划的集合。每个y ∈ Y是一个生产向量，其中y_i > 0表示产出，y_i < 0表示投入。

**定义3.1.2（净产出向量）**：y = (y₁, ..., y_n) ∈ ℝ^n，其中：
- y_i > 0：商品i的净产出
- y_i < 0：商品i的净投入

**定义3.1.3（单产单入技术）**：设商品1是产出，商品2,...,n是投入。生产可能性集可以表示为Y = {y ∈ ℝ^n | y₁ ≤ f(-y₂, ..., -y_n)}，其中f: ℝ^{n-1}_+ → ℝ_+是生产函数。

**定义3.1.4（有效生产计划）**：生产计划y ∈ Y是有效的，如果不存在y' ∈ Y使得y' ≥ y且y' ≠ y。

**定理3.1.1（有效生产计划的存在性）**：如果生产可能性集Y是闭集且满足自由处置（即如果y ∈ Y且y' ≤ y，则y' ∈ Y），则对于每个产出水平都存在有效生产计划。

### 3.1.2 生产函数

**定义3.1.5（生产函数）**：生产函数f: ℝ^{m}_+ → ℝ_+表示在给定投入向量x = (x₁, ..., x_m)下可生产的最大产出y，即：
y = f(x) = max{ y | (y, -x) ∈ Y }

其中x_i ≥ 0表示投入i的数量。

**例3.1.1（科布-道格拉斯生产函数）**：
f(x) = A∏_{i=1}^m x_i^{α_i}, A > 0, α_i > 0

**例3.1.2（CES生产函数）**：
f(x) = A(∑_{i=1}^m α_i x_i^ρ)^{1/ρ}, A > 0, α_i > 0, ρ ≠ 0

替代弹性：σ = 1/(1-ρ)

**例3.1.3（里昂惕夫生产函数）**：
f(x) = min_{i=1,...,m} {x_i/α_i}, α_i > 0

这是完全互补的技术。

**例3.1.4（线性生产函数）**：
f(x) = ∑_{i=1}^m α_i x_i, α_i > 0

这是完全替代的技术。

**图形分析3.1.1**：
```
不同生产技术的等产量线:

科布-道格拉斯:    里昂惕夫:         线性:

投入2           投入2             投入2
  |              |                 |
  |             /|                 |
  |            / |                 |
  |           /  |                 |
  |          /   |                 |
  |         /    |                 |
  |        /     |                 |
  |_______/      |_________        |________________ 投入1
投入1          投入1            投入1

凸向原点       直角              直线
```

### 3.1.3 生产函数的性质

**定义3.1.6（单调性）**：生产函数f是单调的，如果对于任意x, x' ∈ ℝ^m_+，x ≥ x' ⇒ f(x) ≥ f(x')。

**定义3.1.7（严格单调性）**：生产函数f是严格单调的，如果对于任意x, x' ∈ ℝ^m_+，x > x' ⇒ f(x) > f(x')。

**定义3.1.8（拟凹性）**：生产函数f是拟凹的，如果对于任意x, x' ∈ ℝ^m_+和λ ∈ [0, 1]：
f(λx + (1-λ)x') ≥ min{f(x), f(x')}

这等价于等产量线是凸向原点的。

**定理3.1.2（生产函数的性质）**：
1. 如果生产可能性集Y是凸集，则生产函数f是拟凹的。
2. 如果生产函数f是拟凹的，则等产量线是凸向原点的。

**证明（第一部分）**：设x, x'是投入向量，令y = f(x), y' = f(x')。假设y ≥ y'。对于λ ∈ [0, 1]，定义x̄ = λx + (1-λ)x'。由于Y是凸集，(y, -x), (y', -x') ∈ Y ⇒ (λy + (1-λ)y', -x̄) ∈ Y。由单调性，(y', -x̄) ∈ Y，因此f(x̄) ≥ y' = min{y, y'}。□

### 3.1.4 边际产量与技术替代率

**定义3.1.9（边际产量）**：投入i的边际产量MP_i = ∂f/∂x_i表示在其他投入不变的情况下，投入i增加一单位时产出的增加量。

**定义3.1.10（技术替代率）**：投入i对投入j的技术替代率TRS_{ij} = MP_i/MP_j = (∂f/∂x_i)/(∂f/∂x_j)表示在保持产出不变的情况下，可以用多少单位的投入j替代一单位的投入i。

**定理3.1.3（技术替代率递减）**：如果生产函数f是严格拟凹的，则技术替代率沿着等产量线递减。

**解释3.1.1（技术替代率递减的经济学含义）**：随着投入i的使用量增加，它变得更加稀缺（相对），因此需要用更多的投入j来替代相同单位的投入i以保持产出不变。

### 3.1.5 规模报酬

**定义3.1.11（规模报酬）**：生产函数f(x)具有：
1. **规模报酬不变（CRS）**：如果f(αx) = αf(x)对所有α > 0
2. **规模报酬递增（IRS）**：如果f(αx) > αf(x)对所有α > 1
3. **规模报酬递减（DRS）**：如果f(αx) < αf(x)对所有α > 1

**定理3.1.4（欧拉定理）**：如果f是连续可微的，则：
- f是规模报酬不变的 ⇔ ∑_{i=1}^m (∂f/∂x_i)x_i = f(x)
- f是规模报酬递增的 ⇔ ∑_{i=1}^m (∂f/∂x_i)x_i > f(x)
- f是规模报酬递减的 ⇔ ∑_{i=1}^m (∂f/∂x_i)x_i < f(x)

**证明（CRS情形）**：如果f(αx) = αf(x)，对α求导得∑_{i=1}^m (∂f/∂x_i)(αx)x_i = f(x)。令α = 1，得∑_{i=1}^m (∂f/∂x_i)x_i = f(x)。反向类似可证。□

**例3.1.5（科布-道格拉斯的规模报酬）**：
f(x) = A∏_{i=1}^m x_i^{α_i}

计算f(αx) = A∏_{i=1}^m (αx_i)^{α_i} = α^{∑α_i} A∏_{i=1}^m x_i^{α_i} = α^{∑α_i} f(x)

因此：
- ∑α_i = 1 ⇒ CRS
- ∑α_i > 1 ⇒ IRS
- ∑α_i < 1 ⇒ DRS

### 3.1.6 距离函数

**定义3.1.12（产出距离函数）**：产出距离函数d_o(y, x)表示给定投入x时，产出向量y的效率因子：
d_o(y, x) = max{ α > 0 | (αy, x) 是可行的 }

**定义3.1.13（投入距离函数）**：投入距离函数d_i(x, y)表示给定产出y时，投入向量x的效率因子：
d_i(x, y) = max{ α > 0 | (y, x/α) 是可行的 }

**定理3.1.5（距离函数的性质）**：
1. d_o(y, x) ≥ 1当且仅当(y, x)是可行的。
2. d_o(y, x)是y的一次齐次函数，是x的减函数。
3. d_i(x, y) ≥ 1当且仅当(y, x)是可行的。
4. d_i(x, y)是x的一次齐次函数，是y的增函数。

## 3.2 利润最大化

### 3.2.1 利润最大化问题（PMP）

**问题3.2.1（利润最大化问题）**：
max_{y ∈ Y} p·y

其中p ∈ ℝ^n_++是价格向量，Y是生产可能性集。

**问题3.2.2（利润最大化问题（生产函数形式））**：
max_{x ∈ ℝ^m_+, y ≥ 0} py - w·x
s.t. y ≤ f(x)

其中p > 0是产出价格，w ∈ ℝ^m_++是投入价格向量。

**定义3.2.1（供给函数）**：利润最大化问题的解y(p)称为供给函数。

**定义3.2.2（要素需求函数）**：利润最大化问题的解x(p, w)称为要素需求函数。

**定义3.2.3（利润函数）**：π(p, w) = max_{x ≥ 0} pf(x) - w·x是利润最大化问题的值函数，称为利润函数。

### 3.2.2 利润最大化的一阶条件

**定理3.2.1（一阶必要条件）**：设生产函数f是连续可微的，x*是利润最大化问题的内点解。则存在λ ≥ 0，使得：
1. p∂f/∂x_i(x*) = w_i, i = 1, ..., m
2. λ(y* - f(x*)) = 0
3. y* ≤ f(x*)

如果约束是紧的（y* = f(x*)），则p∂f/∂x_i(x*) = w_i对所有i成立。

**解释3.2.1（一阶条件的经济学含义）**：p∂f/∂x_i = w_i表示投入i的边际收益等于其边际成本。这是利润最大化的必要条件。

**例3.2.1（科布-道格拉斯利润最大化）**：
f(x₁, x₂) = x₁^α x₂^β, α, β ∈ (0, 1)

利润函数：π = px₁^α x₂^β - w₁x₁ - w₂x₂

一阶条件：
pαx₁^{α-1}x₂^β = w₁
pβx₁^α x₂^{β-1} = w₂

两式相除：αx₂/(βx₁) = w₁/w₂ ⇒ x₂ = (βw₁/αw₂)x₁

代入第一式：pαx₁^{α-1}((βw₁/αw₂)x₁)^β = w₁
⇒ pα(βw₁/αw₂)^β x₁^{α+β-1} = w₁
⇒ x₁ = [pα(βw₁/αw₂)^β/w₁]^{1/(1-α-β)}

因此，要素需求函数为：
x₁(p, w) = [pα^{1-β}β^β w₁^{-(1-β)} w₂^{-β}]^{1/(1-α-β)}
x₂(p, w) = [pα^α β^{1-α} w₁^{-α} w₂^{-(1-α)}]^{1/(1-α-β)}

供给函数：y(p, w) = f(x₁, x₂)

### 3.2.3 利润函数的性质

**定理3.2.2（利润函数的性质）**：
1. 一次齐次性：π(αp, αw) = απ(p, w) 对所有α > 0
2. 对价格非减：如果p' ≥ p，则π(p', w) ≥ π(p, w)
3. 对要素价格非增：如果w' ≥ w，则π(p, w') ≤ π(p, w)
4. 对价格凸：π(p, w)是p的凸函数
5. 对要素价格凹：π(p, w)是w的凹函数
6. 霍特林引理：∂π/∂p = y(p, w), ∂π/∂w_i = -x_i(p, w)

**证明（霍特林引理）**：由包络定理，π(p, w)关于p的导数等于拉格朗日函数关于p的偏导数，即∂π/∂p = y(p, w)。类似地，∂π/∂w_i = -x_i(p, w)。□

**证明（利润函数对价格凸）**：设p = λp' + (1-λ)p''，λ ∈ [0, 1]。则：
π(p, w) ≥ p·y(p', w) = λp'·y(p', w) + (1-λ)p''·y(p', w)
≥ λπ(p', w) + (1-λ)π(p'', w)
因为y(p', w)不是p的最优解。□

**定理3.2.3（供给函数和要素需求函数的性质）**：
1. 零次齐次性：y(αp, αw) = y(p, w), x(αp, αw) = x(p, w) 对所有α > 0
2. 对价格非减：如果p' ≥ p，则y(p', w) ≥ y(p, w)
3. 对要素价格非增：如果w' ≥ w，则x(p, w') ≤ x(p, w)
4. 对称性：∂y/∂p_i = ∂y/∂p_j, ∂x_i/∂w_j = ∂x_j/∂w_i
5. 半正定性：供给函数的价格导数矩阵是半正定的，要素需求函数的要素价格导数矩阵是半负定的

### 3.2.4 利润最大化与规模报酬

**定理3.2.4（利润最大化与规模报酬的关系）**：
1. 如果生产函数具有规模报酬递增，利润最大化问题可能无解（利润可以无限增大）。
2. 如果生产函数具有规模报酬不变，利润最大化问题有解当且仅当最大利润为零。
3. 如果生产函数具有规模报酬递减，利润最大化问题通常有唯一解。

**证明（CRS情形）**：如果f具有CRS，则f(αx) = αf(x)。利润为π(αx) = pαf(x) - w·αx = α(pf(x) - w·x)。如果pf(x) - w·x > 0，令α → ∞，利润无限增大；如果pf(x) - w·x < 0，令α → 0，利润趋近于零（最优是不生产）。因此，唯一均衡是pf(x) - w·x = 0，即利润为零。□

**例3.2.2（CRS科布-道格拉斯的利润最大化）**：
f(x₁, x₂) = x₁^{1/2} x₂^{1/2}

一阶条件：p(1/2)x₁^{-1/2}x₂^{1/2} = w₁, p(1/2)x₁^{1/2}x₂^{-1/2} = w₂

两式相除：x₂/x₁ = w₁/w₂ ⇒ x₂ = (w₁/w₂)x₁

代入第一式：p(1/2)x₁^{-1/2}(w₁/w₂)^{1/2}x₁^{1/2} = p(1/2)(w₁/w₂)^{1/2} = w₁
⇒ p = 2√(w₁w₂)

如果p ≠ 2√(w₁w₂)，则利润最大化问题无界。如果p = 2√(w₁w₂)，则利润为零，任何满足x₂ = (w₁/w₂)x₁的(x₁, x₂)都是最优解。

## 3.3 成本最小化

### 3.3.1 成本最小化问题（CMP）

**问题3.3.1（成本最小化问题）**：
min_{x ∈ ℝ^m_+} w·x
s.t. f(x) ≥ y

其中w ∈ ℝ^m_++是要素价格向量，y ≥ 0是目标产出水平。

**定义3.3.1（条件要素需求函数）**：成本最小化问题的解x(w, y)称为条件要素需求函数。

**定义3.3.2（成本函数）**：c(w, y) = w·x(w, y)是成本最小化问题的值函数，称为成本函数。

**定理3.3.1（成本最小化问题的存在性）**：如果满足：
1. 生产函数f是连续的
2. 上轮廓集{ x | f(x) ≥ y }是非空紧集

则成本最小化问题有解。

### 3.3.2 成本函数的性质

**定理3.3.2（成本函数的性质）**：
1. 对要素价格一次齐次：c(αw, y) = αc(w, y) 对所有α > 0
2. 对要素价格非减：如果w' ≥ w，则c(w', y) ≥ c(w, y)
3. 对产出非减：如果y' ≥ y，则c(w, y') ≥ c(w, y)
4. 对要素价格凹：c(w, y)是w的凹函数
5. 谢泼德引理：∂c/∂w_i = x_i(w, y)
6. 对产出的性质：
   - 如果f是CRS，则c(w, y) = yc(w, 1)（对产出一次齐次）
   - 如果f是IRS，则c(w, y)对产出是凹的（平均成本递减）
   - 如果f是DRS，则c(w, y)对产出是凸的（平均成本递增）

**证明（谢泼德引理）**：由包络定理，c(w, y)关于w的导数等于拉格朗日函数关于w的偏导数，即∂c/∂w_i = x_i(w, y)。□

**证明（成本函数对要素价格凹）**：设w = λw' + (1-λ)w''，λ ∈ [0, 1]。则：
c(w, y) = w·x(w, y) = λw'·x(w, y) + (1-λ)w''·x(w, y)
≥ λc(w', y) + (1-λ)c(w'', y)
因为x(w, y)不是w'或w''的最优解，但满足约束f(x(w, y)) ≥ y。□

### 3.3.3 条件要素需求函数的性质

**定理3.3.3（条件要素需求函数的性质）**：
1. 对要素价格零次齐次：x(αw, y) = x(w, y) 对所有α > 0
2. 对产出非减：如果y' ≥ y，则x(w, y') ≥ x(w, y)
3. 对称性：∂x_i/∂w_j = ∂x_j/∂w_i
4. 半负定性：要素替代矩阵[∂x_i/∂w_j]是半负定的

**证明（对称性）**：由谢泼德引理，x_i = ∂c/∂w_i。因此，∂x_i/∂w_j = ∂²c/∂w_i∂w_j = ∂²c/∂w_j∂w_i = ∂x_j/∂w_i。□

**解释3.3.1（对称性的经济学含义）**：要素价格w_j的变化对要素i需求的影响等于要素价格w_i的变化对要素j需求的影响。这反映了要素替代的对称性。

### 3.3.4 成本函数与规模报酬

**定理3.3.4（成本函数与规模报酬的关系）**：
1. f是CRS ⇔ c(w, y) = yc(w, 1)
2. f是IRS ⇔ c(w, y)/y对y递减（平均成本递减）
3. f是DRS ⇔ c(w, y)/y对y递增（平均成本递增）

**证明（CRS情形）**：如果f是CRS，则f(x) ≥ y ⇔ f(x/y) ≥ 1。因此：
c(w, y) = min{w·x | f(x) ≥ y} = min{w·(y·(x/y)) | f(x/y) ≥ 1}
= y·min{w·z | f(z) ≥ 1} = yc(w, 1)
其中z = x/y。□

**例3.3.1（科布-道格拉斯成本函数）**：
f(x₁, x₂) = x₁^α x₂^β, α, β ∈ (0, 1)

成本最小化问题：min w₁x₁ + w₂x₂ s.t. x₁^α x₂^β ≥ y

拉格朗日函数：L = w₁x₁ + w₂x₂ + λ(y - x₁^α x₂^β)

一阶条件：
w₁ = λαx₁^{α-1}x₂^β
w₂ = λβx₁^α x₂^{β-1}
x₁^α x₂^β = y

两式相除：w₁/w₂ = (α/β)(x₂/x₁) ⇒ x₂ = (βw₁/αw₂)x₁

代入约束：x₁^α ((βw₁/αw₂)x₁)^β = y
⇒ x₁^{α+β} (βw₁/αw₂)^β = y
⇒ x₁ = [y/(βw₁/αw₂)^β]^{1/(α+β)}

成本函数：
c(w, y) = w₁x₁ + w₂x₂ = w₁x₁ + w₂(βw₁/αw₂)x₁
= w₁x₁(1 + β/α) = w₁(1 + β/α)[y/(βw₁/αw₂)^β]^{1/(α+β)}

简化后：
c(w, y) = (α + β)[(w₁/α)^α (w₂/β)^β]^{1/(α+β)} y^{1/(α+β)}

如果α + β = 1（CRS），则：
c(w, y) = y(w₁/α)^α (w₂/β)^β

条件要素需求：
x₁(w, y) = y(αw₂/βw₁)^β
x₂(w, y) = y(βw₁/αw₂)^α

## 3.4 对偶性

### 3.4.1 利润最大化与成本最小化的对偶性

**定理3.4.1（对偶性关系）**：
1. x(p, w) = x(w, y(p, w))
2. y(p, w) = f(x(p, w))

即非条件要素需求可以通过条件要素需求得到，供给函数通过生产函数得到。

**定理3.4.2（利润函数与成本函数的对偶性）**：
π(p, w) = max_{y ≥ 0} [py - c(w, y)]

即利润函数可以通过成本函数表示。

**证明**：利润最大化问题可以表示为：
π(p, w) = max_{x ≥ 0, y ≥ 0} [py - w·x] s.t. f(x) ≥ y
= max_{y ≥ 0} [py - min_{x ≥ 0} w·x s.t. f(x) ≥ y]
= max_{y ≥ 0} [py - c(w, y)] □

**定理3.4.3（一阶条件）**：如果c(w, y)是可微的，利润最大化的一阶条件为：
p = ∂c/∂y(w, y)

即产出价格等于边际成本。

**证明**：由π(p, w) = max_{y ≥ 0} [py - c(w, y)]，一阶条件为p - ∂c/∂y(w, y) = 0，即p = ∂c/∂y(w, y)。□

### 3.4.2 生产函数与成本函数的对偶性

**定理3.4.4（成本函数导出生产函数）**：给定成本函数c(w, y)，可以恢复生产函数：
f(x) = max{ y | w·x ≥ c(w, y) 对所有w ∈ ℝ^m_++ }

**证明概要**：生产函数表示给定投入x的最大产出y。成本函数c(w, y)表示给定要素价格w和产出y的最小成本。如果w·x ≥ c(w, y)对所有w成立，则给定投入x，生产产出y是可行的。最大化y得到生产函数。□

**例3.4.1（从成本函数恢复生产函数）**：
c(w, y) = y(w₁w₂)^{1/2}

计算：w·x = w₁x₁ + w₂x₂ ≥ y(w₁w₂)^{1/2} 对所有w₁, w₂ > 0

令w₁ = 1/x₁, w₂ = 1/x₂：
(1/x₁)x₁ + (1/x₂)x₂ = 2 ≥ y(1/(x₁x₂))^{1/2}
⇒ y ≤ 2(x₁x₂)^{1/2}

因此，f(x) = 2(x₁x₂)^{1/2}（科布-道格拉斯生产函数）

### 3.4.3 对偶性的应用

**应用3.4.1（比较静态分析）**：使用对偶性可以简化比较静态分析。例如，供给函数对价格的导数可以通过成本函数的二阶导数得到：
∂y/∂p = 1/(∂²c/∂y²)

由于∂²c/∂y² ≥ 0（成本函数对产出凸），供给曲线向上倾斜。

**应用3.4.2（生产率测度）**：全要素生产率（TFP）可以通过成本函数测度：
TFP = y/f(x) = y/f(x(w, y))

如果f(x) = y，则TFP = 1。如果TFP > 1，则生产率高于基准。

## 3.5 成本曲线与生产决策

### 3.5.1 成本曲线

**定义3.5.1（总成本）**：TC(y) = c(w, y)是生产y单位的总成本。

**定义3.5.2（平均成本）**：AC(y) = TC(y)/y = c(w, y)/y是每单位产出的平均成本。

**定义3.5.3（边际成本）**：MC(y) = dTC/dy = ∂c/∂y(w, y)是生产额外一单位产出的成本。

**定理3.5.1（成本曲线的性质）**：
1. MC曲线通过AC曲线的最低点。
2. 当MC < AC时，AC递减；当MC > AC时，AC递增。
3. 在竞争性市场中，利润最大化要求p = MC（如果p ≥ AVC）。

**证明（第二部分）**：AC(y) = TC(y)/y。对y求导：
dAC/dy = (MC·y - TC)/y² = (MC - AC)/y
因此，当MC < AC时，dAC/dy < 0（AC递减）；当MC > AC时，dAC/dy > 0（AC递增）。□

**图形分析3.5.1**：
```
成本曲线:

成本
  |
  |        MC
  |       /|\
  |      / | \
  |     /  |  \
  |    /   |   \
  |   /    |    \
  |  /     |     \
  | /      |      \
  |/_______|_______\_____ 产出
          y*  AC最低点

MC曲线通过AC曲线的最低点
```

### 3.5.2 短期与长期

**定义3.5.4（短期成本函数）**：c_s(w, y; k) = min_{x_v} w_v·x_v s.t. f(x_v, k) ≥ y，其中k是固定投入，x_v是可变投入。

**定义3.5.5（长期成本函数）**：c_l(w, y) = min_{k} [w_k·k + c_s(w, y; k)]

**定理3.5.2（包络定理）**：长期成本曲线是短期成本曲线的包络线。

**图形分析3.5.2**：
```
短期与长期成本曲线:

成本
  |
  |   c_s(w, y; k₁)
  |  /
  | /   c_s(w, y; k₂)
  |/   /
  |   /   c_s(w, y; k₃)
  |  /   /
  | /   /   c_s(w, y; k₄)
  |/   /   /
  |   /   /
  |  /   /   c_l(w, y)
  | /   /   /
  |/   /   /
  |___/___/____________ 产出

长期成本曲线是短期成本曲线的包络线
```

### 3.5.3 生产者剩余

**定义3.5.6（生产者剩余）**：生产者剩余是供给曲线以上的面积：
PS = ∫_{0}^{y*} [p - MC(y)] dy = p*y* - TC(y*)

**定理3.5.3（生产者剩余与利润）**：在竞争性市场中，生产者剩余等于利润（如果不存在固定成本）。

**证明**：PS = p*y* - TC(y*) = π(y*)（假设固定成本为零）。□

### 3.5.4 要素份额

**定义3.5.7（要素份额）**：要素i的份额为s_i = w_i x_i / c(w, y)。

**定理3.5.4（欧拉定理与要素份额）**：如果生产函数f是CRS的，则要素份额之和为1，且s_i = (∂f/∂x_i)(x_i/f)。

**证明**：由欧拉定理，f(x) = ∑_{i=1}^m (∂f/∂x_i)x_i。两边除以f：
1 = ∑_{i=1}^m (∂f/∂x_i)(x_i/f)

由成本最小化一阶条件，w_i = λ∂f/∂x_i，其中λ是拉格朗日乘数。因此：
s_i = w_i x_i / c(w, y) = λ∂f/∂x_i x_i / (λf) = (∂f/∂x_i)(x_i/f) □

## 本章总结

### 核心要点

1. **生产技术**：生产可能性集Y、生产函数f(x)、有效生产计划、距离函数。

2. **生产函数性质**：单调性、拟凹性、边际产量、技术替代率、规模报酬（CRS、IRS、DRS）。

3. **利润最大化**：在给定价格下最大化利润，解是供给函数y(p, w)和要素需求函数x(p, w)；值函数是利润函数π(p, w)。

4. **成本最小化**：在给定产出和要素价格下最小化成本，解是条件要素需求函数x(w, y)；值函数是成本函数c(w, y)。

5. **对偶性**：利润最大化与成本最小化是对偶问题；π(p, w) = max_{y ≥ 0} [py - c(w, y)]；一阶条件p = ∂c/∂y。

6. **重要定理**：霍特林引理、谢泼德引理、欧拉定理、成本函数的性质。

7. **成本曲线**：总成本、平均成本、边际成本；MC曲线通过AC曲线的最低点。

8. **规模报酬与成本**：CRS ⇒ AC恒定；IRS ⇒ AC递减；DRS ⇒ AC递增。

### 重要概念表格

| 概念 | 定义/性质 | 经济学含义 |
|------|-----------|------------|
| 生产函数 | f(x) = max{ y | (y, -x) ∈ Y } | 给定投入的最大产出 |
| 利润函数 | π(p, w) = max_x [pf(x) - w·x] | 给定价格的最大利润 |
| 成本函数 | c(w, y) = min_x [w·x s.t. f(x) ≥ y] | 给定产出的最小成本 |
| 霍特林引理 | ∂π/∂p = y, ∂π/∂w_i = -x_i | 从利润函数导出供给和需求 |
| 谢泼德引理 | ∂c/∂w_i = x_i | 从成本函数导出要素需求 |
| 规模报酬 | f(αx) = αf(x) (CRS) | 投入增加α倍，产出增加α倍 |
| 技术替代率 | TRS_{ij} = MP_i/MP_j | 投入i替代投入j的比率 |
| 边际成本 | MC = ∂c/∂y | 额外一单位产出的成本 |
| 平均成本 | AC = c/y | 每单位产出的成本 |
