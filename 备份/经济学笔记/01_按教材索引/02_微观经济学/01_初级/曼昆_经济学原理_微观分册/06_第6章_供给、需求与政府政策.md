# 第6章：供给、需求与政府政策

> 本章学习目标：
> - 理解政府干预市场的理由
> - 掌握价格上限和价格下限的效果
> - 学会分析税收对市场的影响
> - 理解政府政策的福利分析

---

## 6.1 政府干预市场的理由

### 6.1.1 市场失灵

**市场失灵（Market Failure）**：
市场机制不能有效配置资源的情况。

**市场失灵的类型**：

1. **垄断（Monopoly）**：
   - 单个卖者控制市场
   - 导致价格高于边际成本
   - 产出低于社会最优水平

2. **外部性（Externalities）**：
   - 交易对第三方的影响
   - 正外部性：教育、研发
   - 负外部性：污染、噪音

3. **公共物品（Public Goods）**：
   - 非竞争性和非排他性
   - 例如：国防、路灯
   - 市场供给不足

4. **信息不对称（Information Asymmetry）**：
   - 买者和卖者掌握的信息不同
   - 例如：二手车市场、保险市场

### 6.1.2 政府干预的目的

**政府干预的目的**：

1. **效率（Efficiency）**：
   - 纠正市场失灵
   - 提高资源配置效率

2. **公平（Equity）**：
   - 促进收入分配公平
   - 保护弱势群体

3. **稳定（Stability）**：
   - 稳定经济
   - 防止经济波动

4. **特定目标（Specific Goals）**：
   - 促进特定行业发展
   - 实现特定政策目标

### 6.1.3 政府干预的方式

**政府干预的方式**：

1. **价格管制（Price Controls）**：
   - 价格上限（Price Ceiling）
   - 价格下限（Price Floor）

2. **数量管制（Quantity Controls）**：
   - 配额（Quota）
   - 许可证（License）

3. **税收（Taxation）**：
   - 从量税（Specific Tax）
   - 从价税（Ad Valorem Tax）

4. **补贴（Subsidy）**：
   - 生产补贴
   - 消费补贴

5. **直接干预（Direct Intervention）**：
   - 国有化
   - 直接提供

## 6.2 价格上限

### 6.2.1 价格上限的定义

**价格上限（Price Ceiling）**：
政府规定的某种商品或服务的最高价格。

**特点**：
- 价格上限必须低于均衡价格才能生效
- 价格上限高于均衡价格时，无效果

### 6.2.2 价格上限的效果

**情况1：价格上限高于均衡价格（非约束性）**

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 价格上限 > 均衡价格
- 市场在均衡价格运行
- 价格上限无效

**情况2：价格上限低于均衡价格（约束性）**

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 价格上限 < 均衡价格
- 供给量 < 需求量
- 短缺（Shortage）
- 短缺 = Qd - Qs

**短缺的后果**：

1. **非价格配给（Non-Price Rationing）**：
   - 排队
   - 配给制
   - 优惠卡

2. **黑市（Black Market）**：
   - 在黑市上以高于价格上限的价格交易
   - 买者和卖者都受益

3. **质量下降（Quality Deterioration）**：
   - 卖者降低质量以减少成本
   - 消费者受害

4. **搜寻成本（Search Costs）**：
   - 买者花时间寻找愿意卖的人
   - 浪费资源

### 6.2.3 价格上限的福利分析

**福利分析**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**价格上限前**：
- 消费者剩余：A + B
- 生产者剩余：C + D + E
- 总剩余：A + B + C + D + E

**价格上限后**：
- 消费者剩余：A + B + C（部分消费者获益）
- 生产者剩余：E
- 总剩余：A + B + C + E
- 无谓损失：D

**无谓损失（Deadweight Loss）**：
由于价格上限导致的市场低效率造成的福利损失。

### 6.2.4 价格上限的例子

**例子1：汽油价格管制**

**背景**：
- 20世纪70年代石油危机
- 政府对汽油设定价格上限

**结果**：
- 汽油短缺
- 排队加油
- 黑市交易

**教训**：
- 价格上限导致短缺
- 短缺导致资源浪费

**例子2：租金管制**

**背景**：
- 政府对租金设定价格上限
- 目的：帮助低收入家庭

**结果**：
- 短缺：租房需求增加，供给减少
- 房东减少维护：质量下降
- 黑市：收取额外费用

**教训**：
- 价格上限有助于低收入家庭
- 但导致效率损失

## 6.3 价格下限

### 6.3.1 价格下限的定义

**价格下限（Price Floor）**：
政府规定的某种商品或服务的最低价格。

**特点**：
- 价格下限必须高于均衡价格才能生效
- 价格下限低于均衡价格时，无效果

### 6.3.2 价格下限的效果

**情况1：价格下限低于均衡价格（非约束性）**

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 价格下限 < 均衡价格
- 市场在均衡价格运行
- 价格下限无效

**情况2：价格下限高于均衡价格（约束性）**

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 价格下限 > 均衡价格
- 供给量 > 需求量
- 过剩（Surplus）
- 过剩 = Qs - Qd

**过剩的后果**：

1. **政府购买（Government Purchase）**：
   - 政府购买过剩商品
   - 成本高昂

2. **黑市（Black Market）**：
   - 在黑市上以低于价格下限的价格交易
   - 买者和卖者都受益

3. **质量提高（Quality Improvement）**：
   - 卖者提高质量以吸引买者
   - 成本上升

4. **限制供给（Supply Restriction）**：
   - 政府限制供给
   - 例如：限制种植面积

### 6.3.3 价格下限的福利分析

**福利分析**：

**图形**：

```mermaid

    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**价格下限前**：
- 消费者剩余：A + B + C
- 生产者剩余：D + E
- 总剩余：A + B + C + D + E

**价格下限后**：
- 消费者剩余：A
- 生产者剩余：B + D + E（部分生产者获益）
- 总剩余：A + B + D + E
- 无谓损失：C

**无谓损失（Deadweight Loss）**：
由于价格下限导致的市场低效率造成的福利损失。

### 6.3.4 价格下限的例子

**例子1：最低工资**

**背景**：
- 政府设定最低工资
- 目的：保护低收入工人

**结果**：
- 劳动供给 > 劳动需求
- 失业增加
- 部分工人受益，部分工人受损

**教训**：
- 最低工资有助于低收入工人
- 但导致失业增加

**例子2：农产品价格支持**

**背景**：
- 政府对农产品设定价格下限
- 目的：保护农民

**结果**：
- 农产品过剩
- 政府购买过剩农产品
- 成本高昂

**教训**：
- 价格支持有助于农民
- 但导致效率损失

## 6.4 税收

### 6.4.1 税收的定义

**税收（Tax）**：
政府向个人或企业征收的强制性支付。

**税收的类型**：

1. **从量税（Specific Tax）**：
   - 按数量征收
   - 例如：每包烟征收2元

2. **从价税（Ad Valorem Tax）**：
   - 按价值征收
   - 例如：增值税10%

3. **所得税（Income Tax）**：
   - 按收入征收
   - 例如：个人所得税

4. **财产税（Property Tax）**：
   - 按财产价值征收
   - 例如：房产税

### 6.4.2 税收对市场的影响

**税收对市场的影响**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**税收前**：
- 均衡价格：P1
- 均衡数量：Q1

**税收后**：
- 供给曲线左移（S1 → S2）
- 买者支付的价格：P2
- 卖者收到的价格：P2 - 税收
- 均衡数量：Q2

**税收的影响**：

1. **价格上升**：
   - 买者支付的价格上升
   - 但上升幅度小于税收

2. **数量减少**：
   - 均衡数量减少
   - 市场规模缩小

3. **税负归宿（Tax Incidence）**：
   - 买者和卖者分担税负
   - 分担比例取决于弹性

### 6.4.3 税负归宿

**税负归宿（Tax Incidence）**：
税收在买者和卖者之间的分配。

**弹性与税负归宿**：

**情况1：需求缺乏弹性，供给富有弹性**

**分析**：
- 买者承担大部分税负
- 卖者承担小部分税负

**原因**：
- 需求缺乏弹性：买者对价格变化不敏感
- 供给富有弹性：卖者对价格变化敏感

**例子**：
- 香烟税：买者承担大部分税负

**情况2：需求富有弹性，供给缺乏弹性**

**分析**：
- 卖者承担大部分税负
- 买者承担小部分税负

**原因**：
- 需求富有弹性：买者对价格变化敏感
- 供给缺乏弹性：卖者对价格变化不敏感

**例子**：
- 奢侈品税：卖者承担大部分税负

### 6.4.4 税收的福利分析

**福利分析**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**税收前**：
- 消费者剩余：A + B + C
- 生产者剩余：D + E + F
- 总剩余：A + B + C + D + E + F

**税收后**：
- 消费者剩余：A
- 生产者剩余：F
- 政府税收：B + D
- 总剩余：A + B + D + F
- 无谓损失：C + E

**无谓损失（Deadweight Loss）**：
由于税收导致的市场低效率造成的福利损失。

### 6.4.5 税收的例子

**例子1：汽油税**

**背景**：
- 政府对汽油征收税收
- 目的：减少污染，增加收入

**结果**：
- 汽油价格上升
- 汽油消费量减少
- 政府获得税收收入

**教训**：
- 税收可以改变行为
- 税收可以增加收入

**例子2：香烟税**

**背景**：
- 政府对香烟征收税收
- 目的：减少吸烟，增加收入

**结果**：
- 香烟价格上升
- 香烟消费量减少
- 政府获得税收收入

**教训**：
- 对缺乏弹性的商品征税可以获得更多收入
- 税收可以减少不良行为

## 6.5 补贴

### 6.5.1 补贴的定义

**补贴（Subsidy）**：
政府向个人或企业提供的支付。

**补贴的类型**：

1. **生产补贴（Production Subsidy）**：
   - 对生产者提供补贴
   - 例如：农业补贴

2. **消费补贴（Consumption Subsidy）**：
   - 对消费者提供补贴
   - 例如：教育补贴

3. **出口补贴（Export Subsidy）**：
   - 对出口商品提供补贴
   - 例如：出口退税

### 6.5.2 补贴对市场的影响

**补贴对市场的影响**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**补贴前**：
- 均衡价格：P1
- 均衡数量：Q1

**补贴后**：
- 供给曲线右移（S1 → S2）
- 买者支付的价格：P2
- 卖者收到的价格：P2 + 补贴
- 均衡数量：Q2

**补贴的影响**：

1. **价格下降**：
   - 买者支付的价格下降
   - 下降幅度小于补贴

2. **数量增加**：
   - 均衡数量增加
   - 市场规模扩大

3. **补贴归宿**：
   - 买者和卖者分享补贴
   - 分享比例取决于弹性

### 6.5.3 补贴的福利分析

**福利分析**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**补贴前**：
- 消费者剩余：A
- 生产者剩余：F
- 总剩余：A + F

**补贴后**：
- 消费者剩余：A + B + C
- 生产者剩余：F + D + E
- 政府成本：B + C + D + E + G
- 总剩余：A + B + C + D + E + F - (B + C + D + E + G) = A + F - G
- 无谓损失：G

**无谓损失（Deadweight Loss）**：
由于补贴导致的市场低效率造成的福利损失。

### 6.5.4 补贴的例子

**例子1：农业补贴**

**背景**：
- 政府对农业提供补贴
- 目的：保护农民，确保粮食安全

**结果**：
- 农产品价格下降
- 农产品数量增加
- 政府支出增加

**教训**：
- 补贴有助于农民
- 但导致效率损失

**例子2：教育补贴**

**背景**：
- 政府对教育提供补贴
- 目的：促进教育，提高人力资本

**结果**：
- 教育价格下降
- 教育数量增加
- 政府支出增加

**教训**：
- 补贴有助于教育
- 有正外部性

## 6.6 本章总结

### 核心要点

1. **政府干预市场的理由**：
   - 市场失灵：垄断、外部性、公共物品、信息不对称
   - 目的：效率、公平、稳定、特定目标

2. **价格上限**：
   - 定义：最高价格
   - 效果：短缺、非价格配给、黑市、质量下降
   - 福利分析：无谓损失

3. **价格下限**：
   - 定义：最低价格
   - 效果：过剩、政府购买、黑市、质量提高
   - 福利分析：无谓损失

4. **税收**：
   - 定义：强制性支付
   - 效果：价格上升、数量减少
   - 税负归宿：取决于弹性
   - 福利分析：无谓损失

5. **补贴**：
   - 定义：政府提供的支付
   - 效果：价格下降、数量增加
   - 补贴归宿：取决于弹性
   - 福利分析：无谓损失

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 市场失灵 | 市场机制不能有效配置资源 | 垄断、外部性、公共物品、信息不对称 |
| 价格上限 | 政府规定的最高价格 | 导致短缺 |
| 价格下限 | 政府规定的最低价格 | 导致过剩 |
| 短缺 | 需求量 > 供给量 | 价格上限的后果 |
| 过剩 | 供给量 > 需求量 | 价格下限的后果 |
| 黑市 | 在管制价格之外交易 | 价格管制和价格下限的后果 |
| 税收 | 政府征收的强制性支付 | 从量税、从价税 |
| 税负归宿 | 税收在买者和卖者之间的分配 | 取决于弹性 |
| 补贴 | 政府提供的支付 | 生产补贴、消费补贴 |
| 无谓损失 | 由于政策导致的市场低效率造成的福利损失 | 价格管制、税收、补贴的后果 |
