# 第15章：垄断

> 本章学习目标：
> - 理解垄断产生的原因
> - 掌握垄断企业的利润最大化决策
> - 学会分析垄断的福利损失
> - 理解价格歧视及其类型

## 15.1 垄断产生的原因

### 15.1.1 垄断的定义

**垄断（Monopoly）**：
只有一个卖者的市场。

**垄断的特征**：
1. **唯一卖者**：
   - 只有一个企业
   - 没有竞争对手

2. **没有替代品**：
   - 商品没有接近的替代品
   - 消费者必须购买

3. **价格制定者**：
   - 企业可以影响价格
   - 不是价格接受者

4. **进入障碍**：
   - 其他企业难以进入
   - 市场存在进入障碍

### 15.1.2 进入障碍

**进入障碍（Barriers to Entry）**：
阻止新企业进入市场的因素。

**进入障碍的类型**：

1. **垄断资源（Monopoly Resources）**：
   - 企业拥有关键资源
   - 其他企业无法获得
   - 例如：钻石、稀土

2. **政府管制（Government Regulation）**：
   - 政府给予专利或版权
   - 其他企业无法进入
   - 例如：药品、软件

3. **生产流程（Production Process）**：
   - 企业拥有独特技术
   - 其他企业无法模仿
   - 例如：可口可乐配方

4. **自然垄断（Natural Monopoly）**：
   - 一个企业能以低于两个或更多企业的成本提供整个市场的商品
   - 原因：规模经济
   - 例如：自来水、电力

## 15.2 垄断企业的利润最大化

### 15.2.1 垄断企业的需求曲线

**垄断企业的需求曲线**：
向右下方倾斜的市场需求曲线。

**含义**：
- 垄断企业是价格制定者
- 价格取决于产量
- 增加产量必须降低价格

### 15.2.2 垄断企业的收益

**垄断企业的收益**：

1. **总收益（Total Revenue, TR）**：
   - TR = P × Q

2. **边际收益（Marginal Revenue, MR）**：
   - MR = ΔTR / ΔQ

3. **平均收益（Average Revenue, AR）**：
   - AR = TR / Q = P

**关键点**：
- 垄断企业的边际收益小于价格
- MR < P

**原因**：
- 垄断企业要增加产量，必须降低价格
- 不仅新增单位的价格降低，所有单位的价格都降低
- 因此，边际收益小于价格

### 15.2.3 利润最大化条件

**利润最大化条件**：
边际收益 = 边际成本

**数学推导**：

**利润函数**：
```
π(Q) = TR(Q) - TC(Q)
```

**利润最大化的一阶条件**：
```
dπ/dQ = dTR/dQ - dTC/dQ = 0
MR - MC = 0
MR = MC
```

**含义**：
- 如果 MR > MC，增加产量可以增加利润
- 如果 MR < MC，减少产量可以增加利润
- 如果 MR = MC，利润最大化

### 15.2.4 利润最大化的图形分析

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- MC：边际成本曲线
- ATC：平均总成本曲线
- D = AR：需求曲线（等于平均收益）
- MR：边际收益曲线
- Q*：利润最大化的产量
- P*：垄断价格

**分析**：
- MR = MC → Q*
- 价格P*在需求曲线上对应Q*
- P* > MC（垄断价格高于边际成本）

## 15.3 垄断的福利损失

### 15.3.1 垄断与完全竞争的比较

**垄断与完全竞争的比较**：

**完全竞争**：
- P = MC
- 产量 = Qc
- 总剩余最大

**垄断**：
- P > MC
- 产量 = Qm < Qc
- 总剩余减少

### 15.3.2 垄断的无谓损失

**垄断的无谓损失**：

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**垄断**：
- 产量：Qm
- 价格：Pm
- 消费者剩余：A
- 生产者剩余：B + C
- 总剩余：A + B + C

**完全竞争**：
- 产量：Qc
- 价格：Pc
- 消费者剩余：A + B + D
- 生产者剩余：C + E
- 总剩余：A + B + C + D + E

**无谓损失**：
- DWL = D + E
- 垄断导致无谓损失

### 15.3.3 垄断的低效率

**垄断的低效率**：

1. **价格高于边际成本**：
   - P > MC
   - 导致消费不足

2. **产量低于社会最优**：
   - Qm < Qc
   - 导致无谓损失

3. **缺乏创新激励**：
   - 没有竞争压力
   - 缺乏创新激励

## 15.4 价格歧视

### 15.4.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
对相同商品向不同消费者收取不同价格。

**条件**：
1. **市场势力**：
   - 企业有定价能力
   - 不是价格接受者

2. **可以区分消费者**：
   - 可以识别不同消费者
   - 可以防止套利

3. **消费者支付意愿不同**：
   - 不同消费者有不同的支付意愿
   - 企业可以获取消费者剩余

### 15.4.2 价格歧视的类型

**价格歧视的类型**：

1. **一级价格歧视（Perfect Price Discrimination）**：
   - 对每个消费者收取其支付意愿
   - 企业获取所有消费者剩余
   - 产量 = 社会最优产量

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 企业对每个消费者收取其支付意愿
- 消费者剩余 = 0
- 生产者剩余 = 总剩余
- 无谓损失 = 0

2. **二级价格歧视（Second-Degree Price Discrimination）**：
   - 根据购买数量收取不同价格
   - 例如：批量折扣

**例子**：
- 买1-10个：10元/个
- 买11-20个：9元/个
- 买21-30个：8元/个

3. **三级价格歧视（Third-Degree Price Discrimination）**：
   - 根据消费者群体收取不同价格
   - 例如：学生折扣、老年折扣

**例子**：
- 成人票：100元
- 学生票：50元

### 15.4.3 价格歧视的影响

**价格歧视的影响**：

1. **增加利润**：
   - 企业获取更多消费者剩余
   - 利润增加

2. **增加产量**：
   - 价格歧视可能增加产量
   - 接近社会最优产量

3. **改变福利分配**：
   - 消费者剩余减少
   - 生产者剩余增加

## 15.5 政府对垄断的管制

### 15.5.1 管制的方式

**管制的方式**：

1. **价格管制（Price Regulation）**：
   - 设定最高价格
   - 例如：P = MC

2. **利润管制（Profit Regulation）**：
   - 限制利润
   - 例如：P = ATC

3. **国有化（Nationalization）**
   - 政府拥有垄断企业
   - 例如：自来水、电力

4. **拆分垄断（Break Up Monopoly）**：
   - 将垄断企业拆分为多个企业
   - 例如：AT&T

### 15.5.2 管制的优缺点

**管制的优缺点**：

**优点**：
1. **降低价格**：
   - 价格管制降低价格
   - 消费者受益

2. **增加产量**：
   - 价格管制增加产量
   - 接近社会最优产量

3. **减少无谓损失**：
   - 价格管制减少无谓损失
   - 提高效率

**缺点**：
1. **信息不对称**：
   - 政府不了解企业的成本
   - 难以设定合适的价格

2. **激励问题**：
   - 价格管制降低利润
   - 企业缺乏创新激励

3. **官僚低效**：
   - 政府管制可能低效
   - 浪费资源

## 15.6 本章总结

### 核心要点

1. **垄断产生的原因**：
   - 垄断资源
   - 政府管制
   - 生产流程
   - 自然垄断

2. **垄断企业的利润最大化**：
   - 垄断企业的需求曲线向右下方倾斜
   - 边际收益小于价格
   - 利润最大化条件：MR = MC
   - 垄断价格高于边际成本

3. **垄断的福利损失**：
   - 垄断导致价格高于边际成本
   - 垄断导致产量低于社会最优
   - 垄断导致无谓损失
   - 垄断是低效率的

4. **价格歧视**：
   - 对相同商品向不同消费者收取不同价格
   - 类型：一级、二级、三级
   - 增加利润、增加产量、改变福利分配

5. **政府对垄断的管制**：
   - 价格管制、利润管制、国有化、拆分垄断
   - 优点：降低价格、增加产量、减少无谓损失
   - 缺点：信息不对称、激励问题、官僚低效
