## 10.1 外部性概述

### 10.1.1 外部性的定义

**外部性（Externality）**：
一个人的行为对旁观者福利的影响，而这种影响没有通过价格机制反映出来。

**关键要素**：
1. **影响旁观者**：
   - 不是直接影响交易双方
   - 而是影响第三方

2. **没有通过价格机制反映**：
   - 市场价格没有考虑外部性的影响
   - 导致市场失灵

### 10.1.2 外部性的类型

**外部性的类型**：

1. **正外部性（Positive Externality）**：
   - 行为对旁观者有正面影响
   - 例如：教育、研发、疫苗接种

2. **负外部性（Negative Externality）**：
   - 行为对旁观者有负面影响
   - 例如：污染、噪音、吸烟

**分类方式**：

| 分类标准 | 类型 | 例子 |
|----------|------|------|
| **影响性质** | 正外部性 | 教育、研发 |
| | 负外部性 | 污染、噪音 |
| **产生者** | 生产外部性 | 工厂污染 |
| | 消费外部性 | 吸烟、噪音 |

### 10.2.2 负外部性的例子

**例子1：工厂污染**

**描述**：
- 工厂生产商品
- 排放污染物
- 污染影响居民健康

**外部性**：
- 工厂的生产对居民产生负外部性
- 工厂不考虑污染成本
- 私人成本 < 社会成本

**结果**：
- 工厂生产过多
- 污染过多

**例子2：汽车尾气**

**描述**：
- 人们驾驶汽车
- 排放尾气
- 尾气污染空气

**外部性**：
- 驾驶汽车对他人产生负外部性
- 驾驶者不考虑尾气成本
- 私人成本 < 社会成本

**结果**：
- 汽车过多
- 尾气过多

**例子3：吸烟**

**描述**：
- 人们在公共场所吸烟
- 二手烟危害他人健康

**外部性**：
- 吸烟对他人产生负外部性
- 吸烟者不考虑二手烟成本
- 私人成本 < 社会成本

**结果**：
- 吸烟过多
- 二手烟过多

## 10.4 解决外部性的私人方案

### 10.4.1 私人谈判

**私人谈判（Private Bargaining）**：
受影响的各方通过谈判解决外部性问题。

**条件**：
1. **产权明确**：
   - 产权归属清晰
   - 没有争议

2. **交易成本低**：
   - 谈判成本低
   - 容易达成协议

3. **人数少**：
   - 受影响的人数少
   - 容易协调

**例子**：

**工厂污染**：
- 工厂污染河流
- 影响下游居民
- 居民和工厂谈判
- 工厂减少污染，居民补偿工厂

### 10.4.2 科斯定理

**科斯定理（Coase Theorem）**：
如果产权明确且交易成本低，无论产权归谁，私人市场都可以有效解决外部性问题。

**科斯定理的含义**：
1. **产权明确**：
   - 产权归属清晰
   - 是解决外部性的前提

2. **交易成本低**：
   - 谈判成本低
   - 容易达成协议

3. **无论产权归谁**：
   - 产权归属不影响效率
   - 只影响收入分配

**例子**：

**工厂污染**：

**情况1：产权归工厂**
- 工厂有权排放污染
- 居民付钱给工厂，减少污染
- 结果：污染减少到社会最优水平

**情况2：产权归居民**
- 居民有权享受清洁环境
- 工厂付钱给居民，排放污染
- 结果：污染减少到社会最优水平

**结论**：
- 无论产权归谁，污染都减少到社会最优水平
- 产权归属只影响收入分配

### 10.4.3 私人方案的局限性

**私人方案的局限性**：

1. **产权不明确**：
   - 产权归属不清晰
   - 难以谈判

2. **交易成本高**：
   - 谈判成本高
   - 难以达成协议

3. **人数多**：
   - 受影响的人数多
   - 难以协调

4. **搭便车问题**：
   - 有些人不愿意付费
   - 导致谈判失败

**例子**：

**大气污染**：
- 污染影响所有人
- 人数众多
- 难以谈判
- 私人方案难以实施

## 10.5 解决外部性的公共政策

### 10.5.1 命令与控制管制

**命令与控制管制（Command-and-Control Regulation）**：
政府直接规定排放标准或技术要求。

**类型**：

1. **排放标准**：
   - 规定最高排放量
   - 例如：汽车尾气排放标准

2. **技术标准**：
   - 规定使用特定技术
   - 例如：安装污染控制设备

**优点**：
- 简单直接
- 容易实施

**缺点**：
- 不灵活
- 不考虑成本差异
- 效率低

**例子**：

**汽车尾气排放标准**：
- 政府规定汽车尾气排放标准
- 汽车制造商必须达标
- 所有汽车制造商采用相同技术
- 成本可能不同

- 政府对汽油征收庇古税
- 汽油价格上升
- 汽油消费减少
- 污染减少

### 10.5.4 可交易许可证

**可交易许可证（Tradable Permits）**：
政府发放排放许可证，允许企业交易许可证。

**可交易许可证的效果**：

**分析**：
1. **政府发放许可证**：
   - 总排放量 = 许可证总量
   - 社会最优排放量

2. **企业交易许可证**：
   - 减少排放成本低的企业：减少排放，出售许可证
   - 减少排放成本高的企业：购买许可证，排放更多

3. **结果**：
   - 总排放量 = 许可证总量
   - 减少排放的成本最小化

**可交易许可证的优点**：
1. **效率高**：
   - 实现社会最优
   - 成本最小化

2. **灵活**：
   - 考虑成本差异
   - 企业可以选择最优方式

3. **激励创新**：
   - 企业有动力减少排放
   - 出售许可证获得收入

**例子**：

**碳排放交易**：
- 政府发放碳排放许可证
- 企业交易许可证
- 减少碳排放成本低的企业：减少排放，出售许可证
- 减少碳排放成本高的企业：购买许可证，排放更多
- 结果：总碳排放量 = 许可证总量，成本最小化

## 10.6 公共政策比较

### 10.6.1 命令与控制管制 vs 市场化政策

**命令与控制管制 vs 市场化政策**：

| 特征 | 命令与控制管制 | 市场化政策 |
|------|----------------|------------|
| **效率** | 低 | 高 |
| **灵活性** | 低 | 高 |
| **成本** | 高 | 低 |
| **激励** | 弱 | 强 |
| **实施难度** | 容易 | 困难 |

### 10.6.2 庇古税 vs 可交易许可证

**庇古税 vs 可交易许可证**：

| 特征 | 庇古税 | 可交易许可证 |
|------|--------|--------------|
| **效率** | 高 | 高 |
| **不确定性** | 不确定排放量 | 确定排放量 |
| **成本** | 不确定成本 | 确定成本 |
| **实施难度** | 困难 | 困难 |
| **适用情况** | 适合排放监测困难的情况 | 适合排放监测容易的情况 |

## 10.7 本章总结

### 核心要点

1. **外部性概述**：
   - 外部性：一个人的行为对旁观者福利的影响
   - 类型：正外部性、负外部性
   - 原因：没有通过价格机制反映

2. **负外部性**：
   - 私人成本 < 社会成本
   - 导致过度生产
   - 无谓损失

3. **正外部性**：
   - 私人收益 < 社会收益
   - 导致生产不足
   - 无谓损失

4. **解决外部性的私人方案**：
   - 私人谈判
   - 科斯定理
   - 局限性：产权不明确、交易成本高、人数多、搭便车

5. **解决外部性的公共政策**：
   - 命令与控制管制
   - 庇古税
   - 补贴
   - 可交易许可证

### 重要概念

| 概念 | 定义 | 关键要点 |

| 外部性 | 一个人的行为对旁观者福利的影响，而这种影响没有通过价格机制反映出来 | 市场失灵 |
