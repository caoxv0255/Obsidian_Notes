# 第13章：生产成本

> 本章学习目标：
> - 理解成本的定义和类型
> - 掌握生产函数的概念
> - 学会计算各种成本
> - 理解短期成本与长期成本
> - 掌握规模经济与规模不经济

---

## 13.1 成本的定义

### 13.1.1 显性成本

**显性成本（Explicit Cost）**：
企业需要直接支付的成本。

**例子**：
- 工资
- 租金
- 原材料
- 利息

### 13.1.2 隐性成本

**隐性成本（Implicit Cost）**：
企业使用自有资源的机会成本。

**例子**：
- 所有者放弃的工资
- 所有者放弃的资本收益
- 所有者放弃的租金

### 13.1.3 机会成本

**机会成本（Opportunity Cost）**：
为了得到某种东西而放弃的其他东西中价值最高的那一个。

**机会成本 = 显性成本 + 隐性成本**

**例子**：

**开餐厅**：
- 显性成本：租金10,000元/月，工资20,000元/月，原材料5,000元/月
- 隐性成本：所有者放弃的工资30,000元/月
- 机会成本 = 10,000 + 20,000 + 5,000 + 30,000 = 65,000元/月

## 13.2 生产函数

### 13.2.1 生产函数的定义

**生产函数（Production Function）**：
表示投入与产出之间关系的函数。

**生产函数的表示**：
```
Q = f(L, K)
```

其中：
- Q：产出
- L：劳动
- K：资本

### 13.2.2 总产量、边际产量、平均产量

**总产量（Total Product, TP）**：
使用一定投入生产的总产出。

**边际产量（Marginal Product, MP）**：
增加一单位投入导致的产出增加。

**平均产量（Average Product, AP）**：
每单位投入的产出。

**计算公式**：
```
TP = f(L, K)
MPL = ΔTP / ΔL
APL = TP / L
```

其中：
- TP：总产量
- MPL：劳动的边际产量
- APL：劳动的平均产量
- L：劳动
- K：资本

### 13.2.3 边际产量递减

**边际产量递减（Diminishing Marginal Product）**：
随着一种投入的增加，其边际产量递减。

**原因**：
1. **固定投入限制**：
   - 资本固定
   - 劳动增加
   - 每个工人分到的资本减少

2. **拥挤效应**：
   - 工人过多
   - 相互干扰
   - 效率下降

**例子**：

**餐厅**：
- 1个工人：生产10份餐
- 2个工人：生产25份餐（MPL = 15）
- 3个工人：生产35份餐（MPL = 10）
- 4个工人：生产40份餐（MPL = 5）
- 5个工人：生产42份餐（MPL = 2）

**边际产量递减**：
- MPL从15降到2

## 13.3 成本的各种衡量

### 13.3.1 固定成本

**固定成本（Fixed Cost, FC）**：
不随产量变化的成本。

**例子**：
- 租金
- 机器折旧
- 管理人员工资

### 13.3.2 可变成本

**可变成本（Variable Cost, VC）**：
随产量变化的成本。

**例子**：
- 原材料
- 工人工资
- 能源

### 13.3.3 总成本

**总成本（Total Cost, TC）**：
固定成本和可变成本的总和。

**计算公式**：
```
TC = FC + VC
```

### 13.3.4 平均固定成本

**平均固定成本（Average Fixed Cost, AFC）**：
每单位产量的固定成本。

**计算公式**：
```
AFC = FC / Q
```

### 13.3.5 平均可变成本

**平均可变成本（Average Variable Cost, AVC）**：
每单位产量的可变成本。

**计算公式**：
```
AVC = VC / Q
```

### 13.3.6 平均总成本

**平均总成本（Average Total Cost, ATC）**：
每单位产量的总成本。

**计算公式**：
```
ATC = TC / Q
ATC = (FC + VC) / Q
ATC = AFC + AVC
```

### 13.3.7 边际成本

**边际成本（Marginal Cost, MC）**：
多生产一单位产量的成本。

**计算公式**：
```
MC = ΔTC / ΔQ
MC = ΔVC / ΔQ
```

### 13.3.8 成本曲线

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- ATC：平均总成本曲线（U型）
- AVC：平均可变成本曲线（U型）
- MC：边际成本曲线（U型）
- MC曲线通过ATC和AVC的最低点

**关键点**：
- MC曲线通过ATC的最低点
- MC曲线通过AVC的最低点
- ATC = AFC + AVC
- AFC随产量增加而下降

## 13.4 短期成本与长期成本

### 13.4.1 短期与长期

**短期（Short Run）**：
至少有一种投入是固定的。

**长期（Long Run）**：
所有投入都是可变的。

### 13.4.2 短期成本曲线

**短期成本曲线**：
- 至少有一种投入是固定的
- 固定成本存在

**例子**：
- 短期：资本固定，劳动可变
- 固定成本：租金、机器折旧
- 可变成本：工资、原材料

### 13.4.3 长期成本曲线

**长期成本曲线**：
- 所有投入都是可变的
- 没有固定成本
- 是短期成本曲线的包络线

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- LRATC：长期平均总成本曲线（U型）
- SRATC1、SRATC2、SRATC3：短期平均总成本曲线
- LRATC是SRATC的包络线

## 13.5 规模经济与规模不经济

### 13.5.1 规模经济

**规模经济（Economies of Scale）**：
随着产量增加，平均总成本下降。

**原因**：
1. **专业化（Specialization）**：
   - 工人专业化
   - 提高效率

2. **规模效率（Scale Efficiency）**：
   - 大规模生产
   - 降低平均成本

3. **技术效率（Technological Efficiency）**：
   - 大规模使用先进技术
   - 降低平均成本

### 13.5.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
随着产量增加，平均总成本上升。

**原因**：
1. **管理问题（Management Problems）**：
   - 企业规模过大
   - 管理困难

2. **沟通问题（Communication Problems）**：
   - 企业规模过大
   - 沟通困难

3. **协调问题（Coordination Problems）**：
   - 企业规模过大
   - 协调困难

### 13.5.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
随着产量增加，平均总成本不变。

### 13.5.4 长期平均总成本曲线

**图形表示**：


- 规模不经济：产量 > Q2

## 13.6 本章总结

### 核心要点

1. **成本的定义**：
   - 显性成本：需要直接支付的成本
   - 隐性成本：使用自有资源的机会成本
   - 机会成本 = 显性成本 + 隐性成本

2. **生产函数**：
   - 表示投入与产出之间的关系
   - 总产量、边际产量、平均产量
   - 边际产量递减

3. **成本的各种衡量**：
   - 固定成本、可变成本、总成本
   - 平均固定成本、平均可变成本、平均总成本
   - 边际成本
   - 成本曲线

4. **短期成本与长期成本**：
   - 短期：至少有一种投入是固定的
   - 长期：所有投入都是可变的
   - 长期成本曲线是短期成本曲线的包络线

5. **规模经济与规模不经济**：
   - 规模经济：产量增加，平均总成本下降
   - 规模不经济：产量增加，平均总成本上升
   - 规模报酬不变：产量增加，平均总成本不变
