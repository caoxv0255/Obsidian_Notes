# 第14章：竞争市场上的企业

> 本章学习目标：
> - 理解完全竞争市场的特征
> - 掌握利润最大化的条件
> - 学会分析企业的短期供给决策
> - 学会分析企业的长期供给决策

## 14.1 完全竞争市场

### 14.1.1 完全竞争市场的特征

**完全竞争市场的特征**：

1. **买者和卖者众多**：
   - 没有单个买者或卖者可以影响价格
   - 每个买者和卖者都是价格接受者

2. **商品同质**：
   - 所有卖者提供相同的商品
   - 消费者不在意从哪个卖者购买

3. **自由进出**：
   - 企业可以自由进入或退出市场
   - 没有进入或退出障碍

4. **完全信息**：
   - 买者和卖者了解所有相关信息
   - 没有信息不对称

### 14.1.2 价格接受者

**价格接受者（Price Taker）**：
买者和卖者不能影响市场价格，只能接受市场价格。

**原因**：
- 买者和卖者众多
- 商品同质
- 单个企业的产量相对于市场规模很小

**含义**：
- 企业的需求曲线是水平的
- 企业的边际收益 = 价格

## 14.2 利润最大化

### 14.2.1 利润的定义

**利润（Profit）**：
总收益减去总成本。

**计算公式**：
```
π = TR - TC
```

其中：
- π：利润
- TR：总收益
- TC：总成本

### 14.2.2 总收益、边际收益、平均收益

**总收益（Total Revenue, TR）**：
企业出售商品获得的总收入。

**边际收益（Marginal Revenue, MR）**：
多出售一单位商品获得的收入。

**平均收益（Average Revenue, AR）**：
每单位商品的收益。

**计算公式**：
```
TR = P × Q
MR = ΔTR / ΔQ = P
AR = TR / Q = P
```

其中：
- P：价格
- Q：数量

### 14.2.3 利润最大化条件

**利润最大化条件**：
边际收益 = 边际成本

**数学推导**：

**利润函数**：
```
π(Q) = TR(Q) - TC(Q)
```

**利润最大化的一阶条件**：
```
dπ/dQ = dTR/dQ - dTC/dQ = 0
MR - MC = 0
MR = MC
```

**含义**：
- 如果 MR > MC，增加产量可以增加利润
- 如果 MR < MC，减少产量可以增加利润
- 如果 MR = MC，利润最大化

### 14.2.4 利润最大化的图形分析

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- MC：边际成本曲线
- ATC：平均总成本曲线
- AVC：平均可变成本曲线
- MR = P：边际收益（等于价格）
- Q*：利润最大化的产量

**分析**：
- MR = P（价格接受者）
- MR = MC → P = MC
- Q*：利润最大化的产量

## 14.3 短期供给决策

### 14.3.1 企业的短期供给曲线

**企业的短期供给曲线**：
边际成本曲线位于平均可变成本曲线之上的部分。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- MC：边际成本曲线
- AVC：平均可变成本曲线
- 停止营业点：P = min(AVC)
- 企业的短期供给曲线：MC曲线在AVC曲线之上的部分

### 14.3.2 停止营业点

**停止营业点（Shutdown Point）**：
价格等于平均可变成本的最小值。

**停止营业条件**：
- 如果 P < AVC，企业停止营业
- 如果 P ≥ AVC，企业继续营业

**原因**：
- 如果 P < AVC，企业无法弥补可变成本
- 停止营业可以减少损失

### 14.3.3 短期利润

**短期利润**：
- 如果 P > ATC，企业获得正利润
- 如果 P = ATC，企业获得零利润
- 如果 P < ATC，企业获得负利润（亏损）

**图形分析**：

**情况1：P > ATC（正利润）**

```mermaid

    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- P > ATC
- 正利润 = (P - ATC) × Q*

**情况2：P = ATC（零利润）**

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- P = ATC
- 零利润

**情况3：P < ATC（负利润）**

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- P < ATC
- 负利润 = (P - ATC) × Q*

## 14.4 长期供给决策

### 14.4.1 市场的长期供给曲线

**市场的长期供给曲线**：
表示长期中各种价格下市场供给量的曲线。

**市场的长期供给曲线的类型**：

1. **成本不变行业（Constant-Cost Industry）**：
   - 长期供给曲线是水平的
   - 行业扩张不改变成本

2. **成本递增行业（Increasing-Cost Industry）**：
   - 长期供给曲线向上倾斜
   - 行业扩张增加成本

3. **成本递减行业（Decreasing-Cost Industry）**：
   - 长期供给曲线向下倾斜
   - 行业扩张降低成本

### 14.4.2 进入与退出

**进入（Entry）**：
新企业进入市场。

**进入条件**：
- 如果 P > ATC，企业获得正利润
- 新企业进入
- 供给增加
- 价格下降
- 直到 P = ATC

**退出（Exit）**：
现有企业退出市场。

**退出条件**：
- 如果 P < ATC，企业获得负利润
- 企业退出
- 供给减少
- 价格上升
- 直到 P = ATC

### 14.4.3 长期均衡

**长期均衡**：
- P = min(ATC)
- 企业获得零利润
- 没有进入或退出

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- S：市场供给曲线
- D：市场需求曲线
- P*：均衡价格
- Q*：均衡数量

**分析**：
- 长期均衡：P = min(ATC)
- 企业获得零利润
- 没有进入或退出

## 14.5 竞争市场的效率

### 14.5.1 配置效率

**配置效率（Allocative Efficiency）**：
资源配置使总剩余最大化。

**完全竞争市场的配置效率**：
- 完全竞争市场是有效率的
- 市场均衡使总剩余最大化
- 这是福利经济学第一定理

### 14.5.2 生产效率

**生产效率（Productive Efficiency）**：
以最低成本生产商品。

**完全竞争市场的生产效率**：
- 完全竞争市场是有效率的
- 企业在最低成本点生产
- P = min(ATC)

### 14.5.3 福利分析

**福利分析**：

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**消费者剩余**：
- 需求曲线下方、价格线上方的区域

**生产者剩余**：
- 价格线下方、供给曲线上方的区域

**总剩余**：
- 消费者剩余 + 生产者剩余
- 总剩余最大

**无谓损失**：
- 无

## 14.6 本章总结

### 核心要点

1. **完全竞争市场**：
   - 买者和卖者众多
   - 商品同质
   - 自由进出
   - 完全信息
   - 价格接受者

2. **利润最大化**：
   - 利润 = 总收益 - 总成本
   - 利润最大化条件：MR = MC
   - 完全竞争市场：P = MC

3. **短期供给决策**：
   - 企业的短期供给曲线：MC曲线在AVC曲线之上的部分
   - 停止营业点：P = min(AVC)
   - 短期利润：P > ATC（正）、P = ATC（零）、P < ATC（负）

4. **长期供给决策**：
   - 市场的长期供给曲线：成本不变行业（水平）、成本递增行业（向上）、成本递减行业（向下）
   - 进入与退出
   - 长期均衡：P = min(ATC)

5. **竞争市场的效率**：
   - 配置效率：资源配置使总剩余最大化
   - 生产效率：以最低成本生产商品
   - 完全竞争市场是有效率的
