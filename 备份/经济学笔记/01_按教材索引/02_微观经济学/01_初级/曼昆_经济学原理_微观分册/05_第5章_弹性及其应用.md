# 第5章：弹性及其应用

> 本章学习目标：
> - 理解弹性的概念和计算方法
> - 掌握需求价格弹性的类型和影响因素
> - 掌握供给价格弹性的类型和影响因素
> - 学会分析弹性与总收益的关系
> - 理解弹性在实际中的应用

## 5.1 弹性的概念

### 5.1.1 弹性的定义

**弹性（Elasticity）**：
衡量一个变量对另一个变量变化的反应程度的指标。

**弹性的计算**：
```
弹性 = 因变量的变化率 / 自变量的变化率
弹性 = (ΔY / Y) / (ΔX / X)
弹性 = (ΔY / ΔX) × (X / Y)
```

其中：
- Y：因变量
- X：自变量
- ΔY：Y的变化
- ΔX：X的变化

### 5.1.2 弹性的重要性

**弹性的重要性**：
1. **预测反应**：
   - 预测需求量或供给量对价格变化的反应

2. **制定政策**：
   - 制定税收政策
   - 制定价格管制政策

3. **企业决策**：
   - 定价决策
   - 产量决策

4. **市场分析**：
   - 分析市场结构
   - 分析竞争程度

## 5.2 需求价格弹性

### 5.2.1 需求价格弹性的定义

**需求价格弹性（Price Elasticity of Demand）**：
衡量需求量对价格变化反应程度的指标。

**需求价格弹性的计算**：

**方法1：百分比变化法**
```
需求价格弹性 = 需求量变化的百分比 / 价格变化的百分比
Ed = (%ΔQd) / (%ΔP)
Ed = (ΔQd / Qd) / (ΔP / P)
Ed = (ΔQd / ΔP) × (P / Qd)
```

**方法2：中点法（推荐）**
```
需求价格弹性 = (Q2 - Q1) / [(Q2 + Q1) / 2] ÷ (P2 - P1) / [(P2 + P1) / 2]
Ed = [(Q2 - Q1) / ((Q2 + Q1) / 2)] / [(P2 - P1) / ((P2 + P1) / 2)]
Ed = [(Q2 - Q1) / (Q2 + Q1)] × [(P2 + P1) / (P2 - P1)]
```

其中：
- Q1、Q2：变化前后的需求量
- P1、P2：变化前后的价格

**注意**：
- 需求价格弹性通常为负数（因为价格和需求量反方向变动）
- 但通常取绝对值进行分析

### 5.2.2 需求价格弹性的计算示例

**示例1：使用百分比变化法**

**数据**：
- 初始价格P1 = 4元
- 初始需求量Q1 = 200个
- 新价格P2 = 5元
- 新需求量Q2 = 160个

**计算**：
```
价格变化的百分比 = (P2 - P1) / P1 × 100%
                    = (5 - 4) / 4 × 100%
                    = 25%

需求量变化的百分比 = (Q2 - Q1) / Q1 × 100%
                    = (160 - 200) / 200 × 100%
                    = -20%

需求价格弹性 = 需求量变化的百分比 / 价格变化的百分比
             = -20% / 25%
             = -0.8
```

**示例2：使用中点法**

**数据**：
- 初始价格P1 = 4元
- 初始需求量Q1 = 200个
- 新价格P2 = 5元
- 新需求量Q2 = 160个

**计算**：
```
需求价格弹性 = [(Q2 - Q1) / (Q2 + Q1)] × [(P2 + P1) / (P2 - P1)]
             = [(160 - 200) / (160 + 200)] × [(5 + 4) / (5 - 4)]
             = [-40 / 360] × [9 / 1]
             = -0.11 × 9
             = -1.0
```

### 5.2.3 需求价格弹性的类型

根据需求价格弹性的绝对值大小，可以将需求分为五种类型：

**类型1：完全缺乏弹性（Perfectly Inelastic）**

- 需求曲线：陡峭

**例子**：

- 盐
- 电力

- 弹性 = 1
- 需求量变化等于价格变化
- 需求曲线：等轴双曲线

### 5.2.4 影响需求价格弹性的因素

**因素1：替代品的可得性**
- 替代品越多，弹性越大
- 替代品越少，弹性越小

**例子**：
- 黄油：替代品多（人造黄油、植物油）→ 弹性大
- 胰岛素：替代品少 → 弹性小

**因素2：商品的性质**
- 必需品：弹性小
- 奢侈品：弹性大

**例子**：
- 粮食：必需品 → 弹性小
- 汽车：奢侈品 → 弹性大

**因素3：商品在预算中的比重**
- 比重大，弹性大
- 比重小，弹性小

**例子**：
- 汽车：比重大 → 弹性大
- 盐：比重小 → 弹性小

**因素4：时间跨度**
- 短期：弹性小
- 长期：弹性大

**例子**：
- 汽油：短期弹性小，长期弹性大
  - 短期：很难改变开车习惯
  - 长期：可以购买节能汽车或使用公共交通

### 5.2.5 需求价格弹性的总结

| 弹性类型 | 弹性值 | 特征 | 例子 |
|----------|--------|------|------|
| 完全缺乏弹性 | 0 | 需求量不随价格变化 | 胰岛素 |
| 缺乏弹性 | 0 < |Ed| < 1 | 需求量变化小于价格变化 | 粮食 |
| 单位弹性 | \|Ed\| = 1 | 需求量变化等于价格变化 | 某些奢侈品 |
| 富有弹性 | \|Ed\| > 1 | 需求量变化大于价格变化 | 汽车 |
| 完全富有弹性 | ∞ | 价格微小变化，需求量变化巨大 | 完全竞争市场的产品 |

## 5.3 其他需求弹性

### 5.3.1 需求收入弹性

**需求收入弹性（Income Elasticity of Demand）**：
衡量需求量对收入变化反应程度的指标。

**需求收入弹性的计算**：
```
需求收入弹性 = 需求量变化的百分比 / 收入变化的百分比
Ei = (%ΔQd) / (%ΔI)
Ei = (ΔQd / Qd) / (ΔI / I)
Ei = (ΔQd / ΔI) × (I / Qd)
```

其中：
- I：收入
- ΔI：收入的变化

**需求收入弹性的类型**：

**类型1：正常商品（Normal Goods）**
- Ei > 0
- 收入增加，需求量增加

**子类型**：
1. 必需品（Necessities）：0 < Ei < 1
   - 收入增加，需求量增加，但增加幅度小于收入
   - 例子：粮食、衣服

2. 奢侈品（Luxuries）：Ei > 1
   - 收入增加，需求量增加，增加幅度大于收入
   - 例子：汽车、珠宝

**类型2：低档商品（Inferior Goods）**
- Ei < 0
- 收入增加，需求量减少
- 例子：方便面、二手车

### 5.3.2 需求交叉弹性

**需求交叉弹性（Cross-Price Elasticity of Demand）**：
衡量一种商品的需求量对另一种商品价格变化反应程度的指标。

**需求交叉弹性的计算**：
```
需求交叉弹性 = 商品X需求量变化的百分比 / 商品Y价格变化的百分比
Exy = (%ΔQx) / (%ΔPy)
Exy = (ΔQx / Qx) / (ΔPy / Py)
Exy = (ΔQx / ΔPy) × (Py / Qx)
```

其中：
- Qx：商品X的需求量
- Py：商品Y的价格
- ΔQx：商品X需求量的变化
- ΔPy：商品Y价格的变化

**需求交叉弹性的类型**：

**类型1：替代品（Substitutes）**
- Exy > 0
- 商品Y价格上升，商品X需求量增加
- 例子：咖啡和茶

**类型2：互补品（Complements）**
- Exy < 0
- 商品Y价格上升，商品X需求量减少
- 例子：汽车和汽油

**类型3：无关商品（Independent Goods）**
- Exy = 0
- 商品Y价格变化，商品X需求量不变
- 例子：面包和衣服

## 5.4 供给价格弹性

### 5.4.1 供给价格弹性的定义

**供给价格弹性（Price Elasticity of Supply）**：
衡量供给量对价格变化反应程度的指标。

**供给价格弹性的计算**：

**方法1：百分比变化法**
```
供给价格弹性 = 供给量变化的百分比 / 价格变化的百分比
Es = (%ΔQs) / (%ΔP)
Es = (ΔQs / Qs) / (ΔP / P)
Es = (ΔQs / ΔP) × (P / Qs)
```

**方法2：中点法（推荐）**
```
供给价格弹性 = [(Q2 - Q1) / (Q2 + Q1)] × [(P2 + P1) / (P2 - P1)]
```

其中：
- Q1、Q2：变化前后的供给量
- P1、P2：变化前后的价格

**注意**：
- 供给价格弹性通常为正数（因为价格和供给量同方向变动）

### 5.4.3 影响供给价格弹性的因素

**因素1：生产要素的流动性**
- 流动性大，弹性大
- 流动性小，弹性小

**例子**：
- 服务业：流动性大 → 弹性大
- 农业：流动性小 → 弹性小

**因素2：时间的长短**
- 短期：弹性小
- 长期：弹性大

**例子**：
- 房地产：短期弹性小，长期弹性大
  - 短期：难以增加供给
  - 长期：可以建设新房

**因素3：生产能力的利用程度**
- 利用程度低，弹性大
- 利用程度高，弹性小

**例子**：
- 工厂闲置：弹性大
- 工厂满负荷：弹性小

**因素4：进入和退出行业的难易程度**
- 容易进入和退出，弹性大
- 难以进入和退出，弹性小

**例子**：
- 餐饮业：容易进入和退出 → 弹性大
- 电力行业：难以进入和退出 → 弹性小

## 5.5 弹性与总收益

### 5.5.1 总收益的定义

**总收益（Total Revenue）**：
企业出售商品获得的总收入。

**总收益的计算**：
```
总收益 = 价格 × 数量
TR = P × Q
```

其中：
- TR：总收益
- P：价格
- Q：数量

### 5.5.2 弹性与总收益的关系

**弹性与总收益的关系**：

**情况1：需求缺乏弹性（|Ed| < 1）**

**分析**：
- 价格上升（%）× 1 > 需求量下降（%）× |Ed|
- 总收益增加

**例子**：
- 初始：P = 4, Q = 200, TR = 800
- 价格上升25%：P = 5
- 需求量下降20%：Q = 160（弹性 = 0.8）
- 新总收益：TR = 5 × 160 = 800
- 等等，这个例子不对

**重新计算**：
- 初始：P = 4, Q = 200, TR = 800
- 价格上升25%：P = 5
- 假设弹性 = 0.5
- 需求量下降12.5%：Q = 175
- 新总收益：TR = 5 × 175 = 875
- 总收益增加

**结论**：
- 如果需求缺乏弹性，价格上升 → 总收益增加
- 如果需求缺乏弹性，价格下降 → 总收益减少

**情况2：需求富有弹性（|Ed| > 1）**

**分析**：
- 价格上升（%）× 1 < 需求量下降（%）× |Ed|
- 总收益减少

**例子**：
- 初始：P = 4, Q = 200, TR = 800
- 价格上升25%：P = 5
- 假设弹性 = 2
- 需求量下降50%：Q = 100
- 新总收益：TR = 5 × 100 = 500
- 总收益减少

**结论**：
- 如果需求富有弹性，价格上升 → 总收益减少
- 如果需求富有弹性，价格下降 → 总收益增加

**情况3：需求单位弹性（|Ed| = 1）**

**分析**：
- 价格上升（%）× 1 = 需求量下降（%）× |Ed|
- 总收益不变

**结论**：
- 如果需求单位弹性，价格变化 → 总收益不变

### 5.5.3 弹性与总收益的总结

| 弹性类型 | 价格上升 | 价格下降 |
|----------|----------|----------|
| 缺乏弹性（|Ed| < 1） | 总收益增加 | 总收益减少 |
| 单位弹性（|Ed| = 1） | 总收益不变 | 总收益不变 |
| 富有弹性（|Ed| > 1） | 总收益减少 | 总收益增加 |

## 5.6 弹性的应用

### 5.6.1 农业政策

**问题**：
- 农业技术进步 → 供给增加 → 供给曲线右移
- 农产品需求缺乏弹性
- 结果：价格下降，总收益减少

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 供给增加（S1 → S2）
- 均衡价格下降（P1 → P2）
- 均衡数量增加（Q1 → Q2）
- 总收益变化：
  - 如果需求缺乏弹性，价格下降的百分比 > 数量增加的百分比
  - 总收益减少

**政策建议**：
1. **价格支持**：
   - 政府设定最低价格
   - 购买剩余农产品
   - 缺点：成本高

2. **供给限制**：
   - 限制种植面积
   - 限制产量
   - 缺点：效率低

3. **补贴**：
   - 给农民补贴
   - 补偿收入损失
   - 缺点：财政负担

### 5.6.2 税收政策

**问题**：
- 对商品征税
- 税负由谁承担？
- 取决于弹性

**分析**：

**情况1：需求缺乏弹性，供给富有弹性**

**图形**：

```mermaid

    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 税收导致供给曲线左移（S1 → S2）
- 买者支付的价格上升（P1 → P2）
- 卖者收到的价格下降（P1 - 税收）
- 税负主要由买者承担

**情况2：需求富有弹性，供给缺乏弹性**

**图形**：

```mermaid

    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 税收导致供给曲线左移（S1 → S2）
- 买者支付的价格上升（P1 → P2）
- 卖者收到的价格下降（P1 - 税收）
- 税负主要由卖者承担

**结论**：
- 如果需求缺乏弹性，供给富有弹性，税负主要由买者承担
- 如果需求富有弹性，供给缺乏弹性，税负主要由卖者承担

**政策建议**：
- 对需求缺乏弹性的商品征税（如香烟、汽油）
- 税负主要由买者承担
- 减少需求量

### 5.6.3 反垄断政策

**问题**：
- 垄断企业抬高价格
- 消费者受损
- 政府应该干预吗？

**分析**：
- 垄断企业面临的需求富有弹性吗？
- 如果需求富有弹性，企业不能大幅提价
- 如果需求缺乏弹性，企业可以大幅提价

**政策建议**：
1. **价格管制**：
   - 设定最高价格
   - 限制垄断权力

2. **拆分垄断**：
   - 将垄断企业拆分为多个企业
   - 增加竞争

3. **促进竞争**：
   - 降低进入壁垒
   - 鼓励新企业进入

### 5.6.4 最低工资政策

**问题**：
- 最低工资法提高工资
- 对就业有什么影响？

**分析**：

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 最低工资高于均衡工资
- 劳动供给量 > 劳动需求量
- 失业

**影响**：
- 如果劳动需求缺乏弹性，就业减少较少
- 如果劳动需求富有弹性，就业减少较多

**政策建议**：
- 最低工资应该适中
- 避免对就业造成太大影响

## 5.7 本章总结

### 核心要点

1. **弹性的概念**：
   - 衡量反应程度的指标
   - 因变量变化率 / 自变量变化率

2. **需求价格弹性**：
   - 衡量需求量对价格变化的反应
   - 五种类型：完全缺乏弹性、缺乏弹性、单位弹性、富有弹性、完全富有弹性
   - 影响因素：替代品、商品性质、预算比重、时间

3. **其他需求弹性**：
   - 需求收入弹性：衡量需求量对收入变化的反应
   - 需求交叉弹性：衡量一种商品需求量对另一种商品价格变化的反应

4. **供给价格弹性**：
   - 衡量供给量对价格变化的反应
   - 五种类型：完全缺乏弹性、缺乏弹性、单位弹性、富有弹性、完全富有弹性
   - 影响因素：生产要素流动性、时间、生产能力、进入退出难易程度

5. **弹性与总收益**：
   - 缺乏弹性：价格上升 → 总收益增加
   - 单位弹性：价格变化 → 总收益不变
   - 富有弹性：价格上升 → 总收益减少

6. **弹性的应用**：
   - 农业政策：技术进步导致价格下降
   - 税收政策：税负取决于弹性
   - 反垄断政策：限制垄断权力
   - 最低工资政策：对就业的影响

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 弹性 | 衡量反应程度的指标 | 因变量变化率 / 自变量变化率 |
