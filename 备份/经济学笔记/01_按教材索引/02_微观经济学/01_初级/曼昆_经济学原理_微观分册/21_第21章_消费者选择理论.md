## 21.1 预算约束

### 21.1.1 预算约束的定义

**预算约束（Budget Constraint）**：
消费者有限的收入限制了能购买的物品和服务的数量。

### 21.1.2 预算约束线

**预算约束线（Budget Constraint Line）**：
表示消费者在给定收入下能够购买的两种商品的所有组合。

**假设**：
- 消费者购买两种商品：X和Y
- 收入：I
- 商品X的价格：Px
- 商品Y的价格：Py

**预算约束方程**：
```
Px × X + Py × Y = I
```

## 21.2 消费者偏好

### 21.2.1 无差异曲线

**无差异曲线（Indifference Curve）**：
表示给消费者带来相同满足程度的两种商品的所有组合。

**无差异曲线的特征**：

1. **向右下方倾斜**：
   - 为了保持满足程度不变，增加一种商品必须减少另一种商品

2. **凸向原点**：
   - 边际替代率递减

3. **不相交**：
   - 两条无差异曲线不相交

4. **离原点越远的无差异曲线代表更高的满足程度**：
   - 消费者更喜欢离原点远的无差异曲线

### 21.2.2 边际替代率

**边际替代率（Marginal Rate of Substitution, MRS）**：
消费者愿意用一种商品交换另一种商品的比率。

**计算公式**：
```
MRS = MUx / MUy = -ΔY / ΔX
```

其中：
- MUx：商品X的边际效用
- MUy：商品Y的边际效用
- ΔY：商品Y的变化
- ΔX：商品X的变化

**含义**：
- MRS表示消费者愿意放弃多少商品Y来获得一单位商品X
- MRS递减：随着商品X增加，MRS递减

### 21.2.3 无差异曲线的图形

**图形表示**：

```mermaid
flowchart TD
    +----------/---- --> 商品X
```

**图中标记**：
- IC1、IC2：无差异曲线
- IC2代表比IC1更高的满足程度

## 21.3 消费者的最优选择

### 21.3.1 最优选择条件

**最优选择条件**：
预算约束线与无差异曲线相切。

**数学条件**：
```
MRS = Px / Py
```

**含义**：
- 消费者选择的点，无差异曲线与预算约束线相切
- 在该点，消费者达到最大满足程度

## 21.4 收入和价格变化的影响

### 21.4.1 收入变化

**收入变化的影响**：

**图形**：

```mermaid
flowchart TD
    +----------/---- --> 商品X
```

**分析**：
- 收入增加
- 预算约束线向外平移
- 最优选择从X1移动到X2
- 商品X和Y的需求量都增加

**正常商品（Normal Goods）**：
- 收入增加，需求量增加

**低档商品（Inferior Goods）**：
- 收入增加，需求量减少

### 21.4.2 价格变化

**价格变化的影响**：

**图形**：

```mermaid
flowchart TD
    +----------/---- --> 商品X
```

**分析**：
- 商品X价格上升
- 预算约束线向内旋转
- 最优选择从X1移动到X2
- 商品X的需求量减少
- 商品Y的需求量可能增加或减少

## 21.5 收入效应和替代效应

### 21.5.1 价格变化的总效应

**价格变化的总效应**：
商品X价格变化导致商品X需求量的变化。

**总效应 = 收入效应 + 替代效应**

### 21.5.2 替代效应

**替代效应（Substitution Effect）**：
由于相对价格变化导致的需求量变化。

**分析**：
- 商品X价格上升 → 相对价格上升 → 消费者转向商品Y → 商品X需求量减少
- 替代效应总是负的：价格上升，需求量减少

### 21.5.3 收入效应

**收入效应（Income Effect）**：
由于实际收入变化导致的需求量变化。

**分析**：
- 商品X价格上升 → 实际收入下降 → 商品X需求量变化
- 收入效应的符号取决于商品是正常商品还是低档商品
- 正常商品：收入效应为负
- 低档商品：收入效应为正

### 21.5.4 正常商品

**正常商品（Normal Goods）**：
收入增加，需求量增加。

**总效应**：
- 替代效应：负
- 收入效应：负
- 总效应：负（替代效应 + 收入效应）

### 21.5.5 低档商品

**低档商品（Inferior Goods）**：
收入增加，需求量减少。

**总效应**：
- 替代效应：负
- 收入效应：正
- 总效应：取决于替代效应和收入效应的相对大小

**情况1：吉芬商品（Giffen Goods）**：
- 低档商品的一种特殊情况
- 收入效应为正，且大于替代效应
- 总效应：正（价格上升，需求量增加）

**情况2：普通低档商品**：
- 收入效应为正，但小于替代效应
- 总效应：负（价格上升，需求量减少）

## 21.6 本章总结

### 核心要点

1. **预算约束**：
   - 预算约束：消费者有限的收入限制了能购买的物品和服务的数量
   - 预算约束线：表示消费者在给定收入下能够购买的两种商品的所有组合
   - 收入变化：预算约束线平移
   - 价格变化：预算约束线旋转

2. **消费者偏好**：
   - 无差异曲线：表示给消费者带来相同满足程度的两种商品的所有组合
   - 边际替代率：消费者愿意用一种商品交换另一种商品的比率
   - 无差异曲线的特征：向右下方倾斜、凸向原点、不相交、离原点越远满足程度越高

3. **消费者的最优选择**：
   - 最优选择条件：MRS = Px / Py
   - 无差异曲线与预算约束线相切

4. **收入和价格变化的影响**：
   - 收入变化：预算约束线平移，最优选择移动
   - 价格变化：预算约束线旋转，最优选择移动

5. **收入效应和替代效应**：
   - 替代效应：由于相对价格变化导致的需求量变化
   - 收入效应：由于实际收入变化导致的需求量变化
   - 总效应 = 收入效应 + 替代效应

6. **正常商品和低档商品**：
   - 正常商品：收入增加，需求量增加
   - 低档商品：收入增加，需求量减少
   - 吉芬商品：价格上升，需求量增加

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 预算约束 | 消费者有限的收入限制了能购买的物品和服务的数量 | Px × X + Py × Y = I |
| 预算约束线 | 表示消费者在给定收入下能够购买的两种商品的所有组合 | 斜率 = -Px / Py |
| 无差异曲线 | 表示给消费者带来相同满足程度的两种商品的所有组合 | 向右下方倾斜、凸向原点、不相交 |
| 边际替代率 | 消费者愿意用一种商品交换另一种商品的比率 | MRS = MUx / MUy = -ΔY / ΔX |
| 最优选择条件 | MRS = Px / Py | 无差异曲线与预算约束线相切 |
| 替代效应 | 由于相对价格变化导致的需求量变化 | 总是负的 |
| 收入效应 | 由于实际收入变化导致的需求量变化 | 正常商品：负，低档商品：正 |
