# 第10章：生产要素价格决定的需求方面

> 本章学习目标：
> - 理解生产要素市场的特征
> - 掌握生产要素需求的决定因素
> - 学会分析完全竞争厂商的要素需求
> - 理解不完全竞争厂商的要素需求

---

## 10.1 生产要素市场的特征

### 10.1.1 生产要素市场的定义

**生产要素市场（Factor Market）**：
买卖生产要素的市场。

### 10.1.2 生产要素市场的特征

**生产要素市场的特征**：

1. **需求来自厂商（Demand from Firms）**：
   - 厂商是生产要素的需求者
   - 厂商需要生产要素来生产产品

2. **供给来自家庭（Supply from Households）**：
   - 家庭是生产要素的供给者
   - 家庭提供劳动、资本、土地等

3. **派生需求（Derived Demand）**：
   - 生产要素需求是从产品需求派生出来的
   - 产品需求增加，生产要素需求增加

4. **相互依赖（Interdependence）**：
   - 生产要素之间相互依赖
   - 例如：劳动和资本

### 10.1.3 生产要素的类型

**生产要素的类型**：

1. **劳动（Labor）**：
   - 工人提供的劳动
   - 例如：工资

2. **资本（Capital）**：
   - 厂商提供的资本
   - 例如：利息

3. **土地（Land）**：
   - 土地所有者提供的土地
   - 例如：地租

4. **企业家才能（Entrepreneurship）**：
   - 企业家提供的才能
   - 例如：利润

---

## 10.2 生产要素需求的决定因素

### 10.2.1 生产要素需求的定义

**生产要素需求（Factor Demand）**：
厂商在不同价格水平下愿意购买的生产要素数量。

### 10.2.2 生产要素需求的决定因素

**生产要素需求的决定因素**：

1. **产品价格（Product Price）**：
   - 产品价格上升，生产要素需求增加
   - 产品价格下降，生产要素需求减少

2. **生产要素价格（Factor Price）**：
   - 生产要素价格上升，生产要素需求减少
   - 生产要素价格下降，生产要素需求增加

3. **生产技术（Production Technology）**：
   - 生产技术进步，生产要素需求可能增加或减少
   - 例如：自动化技术可能减少劳动需求

4. **其他生产要素价格（Other Factor Prices）**：
   - 其他生产要素价格上升，该生产要素需求可能增加或减少
   - 例如：资本价格上升，劳动需求可能增加

---

## 10.3 完全竞争厂商的要素需求

### 10.3.1 完全竞争厂商的定义

**完全竞争厂商（Perfectly Competitive Firm）**：
在产品市场和要素市场都是完全竞争的厂商。

### 10.3.2 边际产品价值

**边际产品价值（Value of Marginal Product, VMP）**：
边际产品的市场价值。

**公式**：
```
VMP = MP × P
```

**其中**：
- MP：边际产品
- P：产品价格

**含义**：
- 多使用一单位生产要素增加的产品价值
- 厂商愿意支付的最高价格

### 10.3.3 边际收益产品

**边际收益产品（Marginal Revenue Product, MRP）**：
多使用一单位生产要素增加的收益。

**公式**：
```
MRP = MP × MR
```

**其中**：
- MP：边际产品
- MR：边际收益

**含义**：
- 多使用一单位生产要素增加的收益
- 厂商愿意支付的最高价格

**完全竞争厂商**：
- MR = P
- MRP = VMP = MP × P

### 10.3.4 完全竞争厂商的要素需求曲线

**完全竞争厂商的要素需求曲线（Factor Demand Curve）**：
VMP曲线或MRP曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "要素数量L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- VMP = MRP：边际产品价值 = 边际收益产品
- 要素需求曲线：VMP曲线或MRP曲线

### 10.3.5 完全竞争厂商的要素使用原则

**完全竞争厂商的要素使用原则（Factor Employment Principle）**：
边际产品价值 = 要素价格

**公式**：
```
VMP = w
```

**其中**：
- VMP：边际产品价值
- w：要素价格

**含义**：
- 如果 VMP > w，增加要素使用
- 如果 VMP < w，减少要素使用
- 如果 VMP = w，要素使用最优

**推导**：

**利润最大化**：
```
max π = TR(Q(L)) - TC(Q(L)) = P × Q(L) - w × L
```

**利润最大化的一阶条件**：
```
dπ/dL = P × dQ/dL - w = 0
P × MP - w = 0
VMP - w = 0
VMP = w
```

## 10.4 不完全竞争厂商的要素需求

### 10.4.1 不完全竞争厂商的定义

**不完全竞争厂商（Imperfectly Competitive Firm）**：
在产品市场是不完全竞争的厂商。

### 10.4.2 边际收益产品

**边际收益产品（Marginal Revenue Product, MRP）**：
多使用一单位生产要素增加的收益。

**公式**：
```
MRP = MP × MR
```

**其中**：
- MP：边际产品
- MR：边际收益

**不完全竞争厂商**：
- MR < P
- MRP < VMP

### 10.4.3 不完全竞争厂商的要素需求曲线

**不完全竞争厂商的要素需求曲线（Factor Demand Curve）**：
MRP曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "要素数量L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- VMP：边际产品价值
- MRP：边际收益产品
- 要素需求曲线：MRP曲线
- MRP < VMP

### 10.4.4 不完全竞争厂商的要素使用原则

**不完全竞争厂商的要素使用原则（Factor Employment Principle）**：
边际收益产品 = 要素价格

**公式**：
```
MRP = w
```

**其中**：
- MRP：边际收益产品
- w：要素价格

**含义**：
- 如果 MRP > w，增加要素使用
- 如果 MRP < w，减少要素使用
- 如果 MRP = w，要素使用最优

**推导**：

**利润最大化**：
```
max π = TR(Q(L)) - TC(Q(L)) = P × Q(L) - w × L
```

**利润最大化的一阶条件**：
```
dπ/dL = dTR/dL - w = 0
MR × dQ/dL - w = 0
MR × MP - w = 0
MRP - w = 0
MRP = w
```

## 10.5 市场的要素需求曲线

### 10.5.1 市场的要素需求曲线的推导

**市场的要素需求曲线（Market Factor Demand Curve）**：
所有厂商的要素需求曲线的水平加总。

**推导**：
1. 每个厂商的要素需求曲线
2. 水平加总所有厂商的要素需求曲线
3. 得到市场的要素需求曲线

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "要素数量L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- Dm：市场的要素需求曲线

### 10.5.2 市场的要素需求曲线的移动

**市场的要素需求曲线的移动**：

1. **产品需求变化（Change in Product Demand）**：
   - 产品需求增加，市场的要素需求曲线向右移动
   - 产品需求减少，市场的要素需求曲线向左移动

2. **生产技术变化（Change in Production Technology）**：
   - 生产技术进步，市场的要素需求曲线可能向右或向左移动
   - 例如：自动化技术可能使劳动需求曲线向左移动

3. **其他生产要素价格变化（Change in Other Factor Prices）**：
   - 其他生产要素价格上升，市场的要素需求曲线可能向右或向左移动
   - 例如：资本价格上升，劳动需求曲线可能向右移动

## 10.6 本章总结

### 核心要点

1. **生产要素市场的特征**：
   - 需求来自厂商
   - 供给来自家庭
   - 派生需求
   - 相互依赖

2. **生产要素需求的决定因素**：
   - 产品价格
   - 生产要素价格
   - 生产技术
   - 其他生产要素价格

3. **完全竞争厂商的要素需求**：
   - 边际产品价值：VMP = MP × P
   - 边际收益产品：MRP = MP × MR
   - 完全竞争厂商：MR = P，MRP = VMP = MP × P
   - 完全竞争厂商的要素需求曲线：VMP曲线或MRP曲线
   - 完全竞争厂商的要素使用原则：VMP = w

4. **不完全竞争厂商的要素需求**：
   - 边际收益产品：MRP = MP × MR
   - 不完全竞争厂商：MR < P，MRP < VMP
   - 不完全竞争厂商的要素需求曲线：MRP曲线
   - 不完全竞争厂商的要素使用原则：MRP = w

5. **市场的要素需求曲线**：
   - 市场的要素需求曲线：所有厂商的要素需求曲线的水平加总
   - 市场的要素需求曲线的移动：产品需求变化、生产技术变化、其他生产要素价格变化
