# 第12章：生产要素价格决定的供求方面

> 本章学习目标：
> - 理解生产要素市场均衡
> - 掌握工资、利息、地租的决定
> - 学会分析收入分配
> - 理解洛伦兹曲线和基尼系数

---

## 12.1 生产要素市场均衡

### 12.1.1 生产要素市场均衡的定义

**生产要素市场均衡（Factor Market Equilibrium）**：
生产要素供给和需求相等的状态。

### 12.1.2 生产要素市场均衡的条件

**生产要素市场均衡的条件**：
生产要素供给 = 生产要素需求

**数学表达式**：
```
S(w) = D(w)
```

**其中**：
- S(w)：生产要素供给
- D(w)：生产要素需求
- w：要素价格

### 12.1.3 生产要素市场均衡的图形

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "要素数量L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- S：生产要素供给曲线
- D：生产要素需求曲线
- 均衡：供给和需求曲线的交点
- 均衡价格：w*
- 均衡数量：L*

### 12.1.4 生产要素市场均衡的调整

**生产要素市场均衡的调整（Adjustment to Equilibrium）**：

**如果 w > w***：
- 供给 > 需求
- 存在过剩
- 价格下降
- 向均衡调整

**如果 w < w***：
- 供给 < 需求
- 存在短缺
- 价格上升
- 向均衡调整

## 12.2 工资的决定

### 12.2.1 工资的定义

**工资（Wage）**：
劳动的价格，劳动者提供劳动服务的报酬。

### 12.2.2 完全竞争市场的工资决定

**完全竞争市场的工资决定（Wage Determination in Perfect Competition）**：
劳动供给和劳动需求决定工资。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**劳动供给曲线（S）**：
- 向右上方倾斜（如果替代效应 > 收入效应）
- 或向后弯曲（如果收入效应 > 替代效应）

**劳动需求曲线（D）**：
- 向右下方倾斜

**均衡**：
- 工资：w*
- 劳动：L*

### 12.2.3 不完全竞争市场的工资决定

**不完全竞争市场的工资决定（Wage Determination in Imperfect Competition）**：
工会、政府等因素影响工资。

**工会（Labor Union）**：
- 通过集体谈判提高工资
- 可能导致失业

**最低工资法（Minimum Wage Law）**：
- 政府规定最低工资
- 可能导致失业

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**最低工资（w_min）**：
- w_min > w*
- 劳动供给增加
- 劳动需求减少
- 失业 = 劳动供给 - 劳动需求

## 12.3 利息的决定

### 12.3.1 利息的定义

**利息（Interest）**：
资本的价格，资本所有者提供资本的报酬。

### 12.3.2 完全竞争市场的利息决定

**完全竞争市场的利息决定（Interest Determination in Perfect Competition）**：
资本供给和资本需求决定利息。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "资本K" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**资本供给曲线（S）**：
- 向右上方倾斜

**资本需求曲线（D）**：
- 向右下方倾斜

**均衡**：
- 利率：r*
- 资本：K*

## 12.4 地租的决定

### 12.4.1 地租的定义

**地租（Rent）**：
土地的价格，土地所有者提供土地的报酬。

### 12.4.2 完全竞争市场的地租决定

**完全竞争市场的地租决定（Rent Determination in Perfect Competition）**：
土地供给和土地需求决定地租。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "土地N" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**土地供给曲线（S）**：
- 垂直

**土地需求曲线（D）**：
- 向右下方倾斜

**均衡**：
- 地租：R*
- 土地：N*

### 12.4.3 经济租金

**经济租金（Economic Rent）**：
生产要素的实际收入超过其机会成本的部分。

**特征**：
- 土地的全部收入都是经济租金
- 经济租金不影响要素供给

## 12.5 收入分配

### 12.5.1 收入分配的定义

**收入分配（Income Distribution）**：
总收入在生产要素所有者之间的分配。

### 12.5.2 功能性收入分配

**功能性收入分配（Functional Income Distribution）**：
收入在生产要素之间的分配。

**分类**：
1. **工资（Wage）**：劳动收入
2. **利息（Interest）**：资本收入
3. **地租（Rent）**：土地收入
4. **利润（Profit）**：企业家才能收入

### 12.5.3 个人收入分配

**个人收入分配（Personal Income Distribution）**：
收入在个人之间的分配。

### 12.5.4 洛伦兹曲线

**洛伦兹曲线（Lorenz Curve）**：
表示收入分配不平等的曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "累计收入比例%" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**45度线**：
- 完全平等
- 每个人获得相同收入

**洛伦兹曲线**：
- 表示实际收入分配
- 离45度线越远，收入分配越不平等

**对角线**：
- 完全不平等
- 一个人获得全部收入

### 12.5.5 基尼系数

**基尼系数（Gini Coefficient）**：
衡量收入分配不平等的指标。

**计算公式**：
```
G = A / (A + B)
```

**其中**：
- A：洛伦兹曲线与45度线之间的面积
- B：洛伦兹曲线与对角线之间的面积

**含义**：
- G = 0：完全平等
- G = 1：完全不平等
- 0 < G < 1：收入分配不平等

**基尼系数的值**：
- 0 < G < 0.3：收入分配比较平等
- 0.3 < G < 0.4：收入分配合理
- 0.4 < G < 0.5：收入分配差距较大
- G > 0.5：收入分配差距悬殊

## 12.6 本章总结

### 核心要点

1. **生产要素市场均衡**：
   - 生产要素市场均衡：生产要素供给和需求相等的状态
   - 生产要素市场均衡的条件：生产要素供给 = 生产要素需求

2. **工资的决定**：
   - 完全竞争市场的工资决定：劳动供给和劳动需求决定工资
   - 不完全竞争市场的工资决定：工会、最低工资法等因素影响工资

3. **利息的决定**：
   - 完全竞争市场的利息决定：资本供给和资本需求决定利息

4. **地租的决定**：
   - 完全竞争市场的地租决定：土地供给和土地需求决定地租
   - 经济租金：生产要素的实际收入超过其机会成本的部分

5. **收入分配**：
   - 功能性收入分配：收入在生产要素之间的分配
   - 个人收入分配：收入在个人之间的分配
   - 洛伦兹曲线：表示收入分配不平等的曲线
   - 基尼系数：衡量收入分配不平等的指标

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 生产要素市场均衡 | 生产要素供给和需求相等的状态 | S(w) = D(w) |
| 工资 | 劳动的价格，劳动者提供劳动服务的报酬 | 劳动供给和劳动需求决定 |
| 利息 | 资本的价格，资本所有者提供资本的报酬 | 资本供给和资本需求决定 |
| 地租 | 土地的价格，土地所有者提供土地的报酬 | 土地供给和土地需求决定 |
| 经济租金 | 生产要素的实际收入超过其机会成本的部分 | 不影响要素供给 |
| 收入分配 | 总收入在生产要素所有者之间的分配 | 功能性收入分配、个人收入分配 |
| 功能性收入分配 | 收入在生产要素之间的分配 | 工资、利息、地租、利润 |
| 个人收入分配 | 收入在个人之间的分配 | 洛伦兹曲线、基尼系数 |
| 洛伦兹曲线 | 表示收入分配不平等的曲线 | 离45度线越远，收入分配越不平等 |
| 基尼系数 | 衡量收入分配不平等的指标 | G = A / (A + B) |
