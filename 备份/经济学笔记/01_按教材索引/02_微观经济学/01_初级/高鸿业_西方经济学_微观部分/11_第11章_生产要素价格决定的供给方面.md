# 第11章：生产要素价格决定的供给方面

> 本章学习目标：
> - 理解生产要素供给的特征
> - 掌握劳动供给曲线和工资决定
> - 学会分析资本供给和利息决定
> - 理解土地供给和地租决定

---

## 11.1 生产要素供给的特征

### 11.1.1 生产要素供给的定义

**生产要素供给（Factor Supply）**：
生产要素所有者在不同价格水平下愿意提供的生产要素数量。

### 11.1.2 生产要素供给的特征

**生产要素供给的特征**：

1. **供给来自家庭（Supply from Households）**：
   - 家庭是生产要素的供给者
   - 家庭提供劳动、资本、土地等

2. **有限性（Limited Supply）**：
   - 生产要素的数量是有限的
   - 例如：劳动时间有限、土地数量有限

3. **灵活性（Flexibility）**：
   - 生产要素供给具有一定的灵活性
   - 例如：劳动时间可以调整

4. **相互替代性（Substitutability）**：
   - 生产要素之间可以相互替代
   - 例如：劳动和资本可以相互替代

### 11.1.3 生产要素供给的决定因素

**生产要素供给的决定因素**：

1. **要素价格（Factor Price）**：
   - 要素价格上升，要素供给增加
   - 要素价格下降，要素供给减少

2. **非收入因素（Non-Income Factors）**：
   - 偏好、收入、预期等
   - 影响要素供给

---

## 11.2 劳动供给曲线

### 11.2.1 劳动供给的定义

**劳动供给（Labor Supply）**：
劳动者在不同工资水平下愿意提供的劳动时间。

### 11.2.2 劳动供给的决定因素

**劳动供给的决定因素**：

1. **工资（Wage）**：
   - 工资上升，劳动供给可能增加或减少
   - 取决于收入效应和替代效应

2. **非工资因素（Non-Wage Factors）**：
   - 偏好、非劳动收入、家庭责任等
   - 影响劳动供给

### 11.2.3 收入效应和替代效应

**收入效应（Income Effect）**：
工资上升使劳动者收入增加，劳动者减少劳动供给，增加闲暇。

**替代效应（Substitution Effect）**：
工资上升使闲暇价格上升，劳动者增加劳动供给，减少闲暇。

**总效应（Total Effect）**：
- 如果替代效应 > 收入效应，劳动供给增加
- 如果替代效应 < 收入效应，劳动供给减少

### 11.2.4 劳动供给曲线

**劳动供给曲线（Labor Supply Curve）**：
工资和劳动供给之间的关系。

**图形表示**：

**图中标记**：
- S：劳动供给曲线
- 向右上方倾斜（替代效应 > 收入效应）或向后弯曲（替代效应 < 收入效应）

**向后弯曲的劳动供给曲线**：

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 低工资：替代效应 > 收入效应，劳动供给增加
- 高工资：收入效应 > 替代效应，劳动供给减少
- 劳动供给曲线向后弯曲

## 11.3 工资决定

### 11.3.1 工资的定义

**工资（Wage）**：
劳动的价格，劳动者提供劳动服务的报酬。

### 11.3.2 完全竞争市场的工资决定

**完全竞争市场的工资决定（Wage Determination in Perfect Competition）**：
劳动供给和劳动需求决定工资。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**劳动供给曲线（S）**：
- 向右上方倾斜

**劳动需求曲线（D）**：
- 向右下方倾斜

**均衡**：
- 工资：w*
- 劳动：L*

### 11.3.3 不完全竞争市场的工资决定

**不完全竞争市场的工资决定（Wage Determination in Imperfect Competition）**：
工会、政府等因素影响工资。

**工会（Labor Union）**：
- 通过集体谈判提高工资
- 可能导致失业

**最低工资法（Minimum Wage Law）**：
- 政府规定最低工资
- 可能导致失业

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "劳动L" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**最低工资（w_min）**：
- w_min > w*
- 劳动供给增加
- 劳动需求减少
- 失业

## 11.4 资本供给和利息决定

### 11.4.1 资本供给的定义

**资本供给（Capital Supply）**：
资本所有者在不同利率水平下愿意提供的资本数量。

### 11.4.2 资本供给的决定因素

**资本供给的决定因素**：

1. **利率（Interest Rate）**：
   - 利率上升，资本供给增加
   - 利率下降，资本供给减少

2. **非利率因素（Non-Interest Factors）**：
   - 偏好、收入、预期等
   - 影响资本供给

### 11.4.3 资本供给曲线

**利息的定义**：

**完全竞争市场的利息决定（Interest Rate Determination in Perfect Competition）**：

## 11.5 土地供给和地租决定

### 11.5.1 土地供给的定义

**土地供给（Land Supply）**：
土地所有者在不同地租水平下愿意提供的土地数量。

### 11.5.2 土地供给的特征

**土地供给的特征**：

1. **固定性（Fixed Supply）**：
   - 土地供给是固定的
   - 土地数量不随地租变化

2. **无弹性（Inelastic）**：
   - 土地供给完全无弹性
   - 土地供给曲线是垂直的

### 11.5.3 土地供给曲线

**土地供给曲线（Land Supply Curve）**：
地租和土地供给之间的关系。

**图形表示**：

```mermaid
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- S：土地供给曲线
- 垂直

### 11.5.4 地租决定

**地租的定义**：
土地的价格，土地所有者提供土地的报酬。

**完全竞争市场的地租决定（Rent Determination in Perfect Competition）**：
土地供给和土地需求决定地租。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "土地N" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：

**土地供给曲线（S）**：
- 垂直

**土地需求曲线（D）**：
- 向右下方倾斜

**均衡**：
- 地租：R*
- 土地：N*

### 11.5.5 经济租金

**经济租金（Economic Rent）**：
生产要素的实际收入超过其机会成本的部分。

**特征**：
- 土地的全部收入都是经济租金
- 经济租金不影响要素供给

## 11.6 本章总结

### 核心要点

1. **生产要素供给的特征**：
   - 供给来自家庭
   - 有限性
   - 灵活性
   - 相互替代性

2. **劳动供给曲线**：
   - 工资和劳动供给之间的关系
   - 收入效应和替代效应
   - 劳动供给曲线可能向后弯曲

3. **工资决定**：
   - 完全竞争市场的工资决定：劳动供给和劳动需求决定工资
   - 不完全竞争市场的工资决定：工会、最低工资法等因素影响工资

4. **资本供给和利息决定**：
   - 资本供给：利率和资本供给之间的关系
   - 利息决定：资本供给和资本需求决定利率

5. **土地供给和地租决定**：
   - 土地供给：土地供给是固定的，土地供给曲线是垂直的
   - 地租决定：土地供给和土地需求决定地租
   - 经济租金：生产要素的实际收入超过其机会成本的部分
