# 第十四讲：一般均衡

## 14.1 一般均衡概述

### 14.1.1 一般均衡的定义

**一般均衡（General Equilibrium）**：
所有市场同时达到均衡的状态。

### 14.1.2 一般均衡与局部均衡

**一般均衡与局部均衡（General Equilibrium vs. Partial Equilibrium）**：

**1. 局部均衡（Partial Equilibrium）**：
- 只考虑单个市场的均衡
- 假设其他市场不变
- 简化分析

**2. 一般均衡（General Equilibrium）**：
- 考虑所有市场的均衡
- 考虑市场之间的相互影响
- 更全面的分析

## 14.2 交换经济

### 14.2.1 交换经济的定义

**交换经济（Exchange Economy）**：
消费者交换商品的经济。

### 14.2.2 交换经济的模型

**交换经济的模型（Model of Exchange Economy）**：

**消费者**：
- 两个消费者：A和B
- 两种商品：X和Y
- 初始禀赋：A的禀赋为(Xᴀ₀, Yᴀ₀)，B的禀赋为(Xʙ₀, Yʙ₀)

**效用函数**：
```
Uᴀ = Uᴀ(Xᴀ, Yᴀ)
Uʙ = Uʙ(Xʙ, Yʙ)
```

**其中**：
- Uᴀ：消费者A的效用
- Uʙ：消费者B的效用
- Xᴀ：消费者A消费的商品X
- Yᴀ：消费者A消费的商品Y
- Xʙ：消费者B消费的商品X
- Yʙ：消费者B消费的商品Y

**市场均衡条件**：
```
Xᴀ + Xʙ = Xᴀ₀ + Xʙ₀
Yᴀ + Yʙ = Yᴀ₀ + Yʙ₀
```

**其中**：
- Xᴀ + Xʙ：商品X的总消费
- Xᴀ₀ + Xʙ₀：商品X的总禀赋
- Yᴀ + Yʙ：商品Y的总消费
- Yᴀ₀ + Yʙ₀：商品Y的总禀赋

**含义**：
- 商品X的总消费等于商品X的总禀赋
- 商品Y的总消费等于商品Y的总禀赋

### 14.2.3 埃奇沃思盒状图

**埃奇沃思盒状图（Edgeworth Box）**：
表示两个消费者之间的商品交换的图形。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "X" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 横轴：商品X
- 纵轴：商品Y
- 盒状图表示两个消费者之间的商品分配
- 原点A表示消费者A的坐标原点
- 原点B表示消费者B的坐标原点

### 14.2.4 契约曲线

**契约曲线（Contract Curve）**：
所有帕累托最优点的集合。

**定义**：
```
在埃奇沃思盒状图中，两个消费者的无差异曲线切点的轨迹
```

**含义**：
- 契约曲线上的点都是帕累托最优的
- 在契约曲线上，无法再通过交换使一个人的效用增加而不减少另一个人的效用

### 14.2.5 竞争性均衡

**竞争性均衡（Competitive Equilibrium）**：
价格使得每个消费者选择最优消费，市场出清。

**数学表述**：
```
对于所有消费者i，有：xᵢ(p, p · ωᵢ) ∈ Xᵢ(p, p · ωᵢ)
Σᵢ xᵢ(p, p · ωᵢ) = Σᵢ ωᵢ
```

**其中**：
- xᵢ：消费者i的消费
- p：价格向量
- ωᵢ：消费者i的禀赋
- Xᵢ(p, p · ωᵢ)：消费者i的需求集合

**含义**：
- 每个消费者在给定价格和收入下选择最优消费
- 市场出清

## 14.3 生产经济

### 14.3.1 生产经济的定义

**生产经济（Production Economy）**：
包含生产的经济。

### 14.3.2 生产经济的模型

**生产经济的模型（Model of Production Economy）**：

**生产函数**：
```
X = f₁(K₁, L₁)
Y = f₂(K₂, L₂)
```

**其中**：
- X：商品X的产量
- Y：商品Y的产量
- K₁：生产商品X使用的资本
- L₁：生产商品X使用的劳动
- K₂：生产商品Y使用的资本
- L₂：生产商品Y使用的劳动

**资源约束**：
```
K₁ + K₂ = K
L₁ + L₂ = L
```

**其中**：
- K：总资本
- L：总劳动
- K₁ + K₂：资本的总使用
- L₁ + L₂：劳动的总使用

**含义**：
- 资本的总使用等于总资本
- 劳动的总使用等于总劳动

### 14.3.3 生产可能性边界

**生产可能性边界（Production Possibility Frontier, PPF）**：
给定资源和技术，经济能够生产的商品组合。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "X" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- PPF：生产可能性边界
- 横轴：商品X
- 纵轴：商品Y
- 生产可能性边界向右下方倾斜
- 生产可能性边界表示给定资源和技术，经济能够生产的最大商品组合

### 14.3.4 竞争性均衡

**竞争性均衡（Competitive Equilibrium）**：
价格使得每个企业选择最优生产，每个消费者选择最优消费，市场出清。

**数学表述**：
```
对于所有企业j，有：yⱼ ∈ Yⱼ(p)
对于所有消费者i，有：xᵢ(p, p · ωᵢ + Σⱼ θᵢⱼ p · yⱼ) ∈ Xᵢ(p, p · ωᵢ + Σⱼ θᵢⱼ p · yⱼ)
Σᵢ xᵢ = Σᵢ ωᵢ + Σⱼ yⱼ
```

**其中**：
- yⱼ：企业j的生产
- Yⱼ(p)：企业j的生产集合
- xᵢ：消费者i的消费
- p：价格向量
- ωᵢ：消费者i的禀赋
- θᵢⱼ：消费者i在企业j中的股份
- Xᵢ(p, p · ωᵢ + Σⱼ θᵢⱼ p · yⱼ)：消费者i的需求集合

**含义**：
- 每个企业在给定价格下选择最优生产
- 每个消费者在给定价格和收入下选择最优消费
- 市场出清

## 14.4 福利定理

### 14.4.1 帕累托效率

**帕累托效率（Pareto Efficiency）**：
资源配置使得无法再通过重新配置使一个人的效用增加而不减少另一个人的效用。

**定义**：
```
如果不存在其他配置，使得至少一个人的效用增加，而其他人的效用不减少，则该配置是帕累托有效的
```

**含义**：
- 帕累托效率是资源配置的理想状态
- 帕累托效率不考虑公平性

### 14.4.2 福利经济学第一定理

**福利经济学第一定理（First Welfare Theorem）**：
任何竞争性均衡都是帕累托有效的。

**条件**：
- 完全竞争
- 无外部性
- 无信息不对称

**含义**：
- 市场经济能够实现帕累托效率
- 市场经济是有效的

### 14.4.3 福利经济学第二定理

**福利经济学第二定理（Second Welfare Theorem）**：
任何帕累托有效的配置都可以通过适当的初始禀赋分配和竞争性市场实现。

**条件**：
- 完全竞争
- 无外部性
- 无信息不对称
- 凸偏好

**含义**：
- 市场经济可以实现任何帕累托有效的配置
- 通过收入再分配可以实现公平

## 14.5 本章总结

### 核心要点

1. **一般均衡概述**：
   - 一般均衡：所有市场同时达到均衡的状态
   - 一般均衡与局部均衡：局部均衡只考虑单个市场的均衡，一般均衡考虑所有市场的均衡

2. **交换经济**：
   - 交换经济：消费者交换商品的经济
   - 交换经济的模型：两个消费者、两种商品、初始禀赋
   - 埃奇沃思盒状图：表示两个消费者之间的商品交换的图形
   - 契约曲线：所有帕累托最优点的集合
   - 竞争性均衡：价格使得每个消费者选择最优消费，市场出清

3. **生产经济**：
   - 生产经济：包含生产的经济
   - 生产经济的模型：生产函数、资源约束
   - 生产可能性边界：给定资源和技术，经济能够生产的商品组合
   - 竞争性均衡：价格使得每个企业选择最优生产，每个消费者选择最优消费，市场出清

4. **福利定理**：
   - 帕累托效率：资源配置使得无法再通过重新配置使一个人的效用增加而不减少另一个人的效用
   - 福利经济学第一定理：任何竞争性均衡都是帕累托有效的
   - 福利经济学第二定理：任何帕累托有效的配置都可以通过适当的初始禀赋分配和竞争性市场实现
