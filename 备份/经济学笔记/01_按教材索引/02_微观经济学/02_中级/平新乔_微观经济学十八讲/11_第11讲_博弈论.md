# 第十一讲：博弈论

## 11.1 博弈的基本要素

### 11.1.1 博弈的定义

**博弈（Game）**：
决策者之间的策略互动。

### 11.1.2 博弈的基本要素

**博弈的基本要素（Basic Elements of Game）**：

**1. 参与人（Players）**：
- 博弈的决策者
- 例如：企业、消费者

**2. 策略（Strategies）**：
- 参与人的行动方案
- 例如：产量、价格

**3. 收益（Payoffs）**：
- 参与人获得的收益
- 例如：利润

### 11.1.3 博弈的表示

**博弈的表示（Representation of Game）**：

**1. 策略型博弈（Strategic Form Game）**：
- 参与人集合：N = {1, 2, ..., n}
- 策略集合：Sᵢ，对于所有i ∈ N
- 收益函数：uᵢ(s₁, s₂, ..., sₙ)，对于所有i ∈ N

**2. 扩展型博弈（Extensive Form Game）**：
- 博弈树（Game Tree）
- 参与人集合：N = {1, 2, ..., n}
- 行动顺序
- 信息集（Information Sets）
- 收益函数：uᵢ(s₁, s₂, ..., sₙ)，对于所有i ∈ N

## 11.2 纳什均衡

### 11.2.1 纳什均衡的定义

**纳什均衡（Nash Equilibrium）**：
在其他参与人的策略给定的情况下，每个参与人的策略都是最优的。

### 11.2.2 纳什均衡的数学定义

**纳什均衡的数学定义（Mathematical Definition of Nash Equilibrium）**：
```
对于所有参与人i，有：uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)，对于所有sᵢ ∈ Sᵢ
```

**其中**：
- uᵢ：参与人i的收益函数
- sᵢ*：参与人i的纳什均衡策略
- s₋ᵢ*：其他参与人的纳什均衡策略
- Sᵢ：参与人i的策略集合

**含义**：
- 给定其他参与人的策略，没有参与人有动机改变策略
- 纳什均衡是自我实施的（Self-Enforcing）

### 11.2.3 纳什均衡的存在性

**纳什均衡的存在性（Existence of Nash Equilibrium）**：

**纳什定理（Nash's Theorem）**：
每个有限博弈（有限参与人、有限策略）至少存在一个纳什均衡（可能是混合策略）。

**条件**：
- 参与人集合是有限的
- 策略集合是有限的
- 收益函数是连续的

## 11.3 混合策略

### 11.3.1 混合策略的定义

**混合策略（Mixed Strategy）**：
参与人以一定概率选择各个纯策略。

### 11.3.2 混合策略的表示

**混合策略的表示（Representation of Mixed Strategy）**：
```
σᵢ = (σᵢ(sᵢ₁), σᵢ(sᵢ₂), ..., σᵢ(sᵢₙᵢ))
```

**其中**：
- σᵢ：参与人i的混合策略
- σᵢ(sᵢⱼ)：参与人i选择纯策略sᵢⱼ的概率
- σᵢ(sᵢⱼ) ≥ 0，对于所有j
- Σⱼ σᵢ(sᵢⱼ) = 1

### 11.3.3 混合策略纳什均衡

**混合策略纳什均衡（Mixed Strategy Nash Equilibrium）**：
每个参与人的混合策略都是对其他参与人混合策略的最优反应。

**数学定义**：
```
对于所有参与人i，有：uᵢ(σᵢ*, σ₋ᵢ*) ≥ uᵢ(σᵢ, σ₋ᵢ*)，对于所有σᵢ
```

**其中**：
- uᵢ：参与人i的期望收益函数
- σᵢ*：参与人i的混合策略纳什均衡
- σ₋ᵢ*：其他参与人的混合策略纳什均衡

### 11.3.4 混合策略均衡的例子

**混合策略均衡的例子（Example of Mixed Strategy Equilibrium）**：

**猜硬币游戏（Matching Pennies）**：
- 参与人A和参与人B同时选择正面或反面
- 如果匹配，A赢，B输
- 如果不匹配，A输，B赢

## 11.4 重复博弈

### 11.4.1 重复博弈的定义

**重复博弈（Repeated Game）**：
同一博弈重复进行多次。

### 11.4.2 重复博弈的类型

**重复博弈的类型（Types of Repeated Game）**：

**1. 有限重复博弈（Finite Repeated Game）**：
- 博弈重复有限次
- 重复次数已知

**2. 无限重复博弈（Infinite Repeated Game）**：
- 博弈重复无限次
- 重复次数未知

### 11.4.3 重复博弈的均衡

**重复博弈的均衡（Equilibrium of Repeated Game）**：

**1. 有限重复博弈的均衡**：
- 有限重复囚徒困境的纳什均衡是不合作
- 最后一期不合作，倒推第一期也不合作
- 唯一子博弈完美纳什均衡

**2. 无限重复博弈的均衡**：
- 无限重复囚徒困境可能存在合作均衡
- 如果折现因子足够大，合作是纳什均衡
- 威胁策略（Trigger Strategy）

### 11.4.4 威胁策略

**威胁策略（Trigger Strategy）**：
如果对方不合作，则永远不合作。

**形式化定义**：
```
第一期：合作
后续期：如果上一期对方合作，则合作；如果上一期对方不合作，则永远不合作
```

**含义**：
- 威胁策略维持合作
- 如果折现因子足够大，威胁策略是纳什均衡

## 11.5 本章总结

### 核心要点

1. **博弈的基本要素**：
   - 博弈：决策者之间的策略互动
   - 博弈的基本要素：参与人、策略、收益
   - 博弈的表示：策略型博弈、扩展型博弈

2. **纳什均衡**：
   - 纳什均衡：在其他参与人的策略给定的情况下，每个参与人的策略都是最优的
   - 纳什均衡的数学定义：对于所有参与人i，有uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)
   - 纳什均衡的存在性：每个有限博弈至少存在一个纳什均衡（纳什定理）

3. **混合策略**：
   - 混合策略：参与人以一定概率选择各个纯策略
   - 混合策略的表示：σᵢ = (σᵢ(sᵢ₁), σᵢ(sᵢ₂), ..., σᵢ(sᵢₙᵢ))
   - 混合策略纳什均衡：每个参与人的混合策略都是对其他参与人混合策略的最优反应
   - 混合策略均衡的例子：猜硬币游戏

4. **重复博弈**：
   - 重复博弈：同一博弈重复进行多次
   - 重复博弈的类型：有限重复博弈、无限重复博弈
   - 重复博弈的均衡：有限重复博弈的均衡是不合作；无限重复博弈可能存在合作均衡
   - 威胁策略：如果对方不合作，则永远不合作
