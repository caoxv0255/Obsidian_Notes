# 第十讲：寡头垄断

> 本章学习目标：

## 10.1 寡头垄断的定义

### 10.1.1 寡头垄断的定义

**寡头垄断（Oligopoly）**：
少数几家大企业控制的市场结构。

### 10.1.2 寡头垄断的特征

**寡头垄断的特征（Characteristics of Oligopoly）**：

**1. 少数企业（Few Firms）**：
- 市场上只有少数几家大企业
- 每个企业市场份额大

**2. 产品同质或差异化（Homogeneous or Differentiated Products）**
- 企业生产同质产品或差异化产品
- 产品可能不完全替代

**3. 进入壁垒（Barriers to Entry）**
- 其他企业难以进入市场
- 市场进入成本高

**4. 相互依存（Interdependence）**
- 企业决策相互影响
- 企业需要考虑竞争对手的反应

## 10.2 古诺模型

### 10.2.1 古诺模型的定义

**古诺模型（Cournot Model）**：
企业同时选择产量，使利润最大化。

### 10.2.2 古诺模型的假设

**古诺模型的假设（Assumptions of Cournot Model）**：

**1. 同时决策（Simultaneous Decision）**：
- 企业同时选择产量
- 不知道其他企业的产量

**2. 同质产品（Homogeneous Products）**：
- 企业生产同质产品
- 消费者对产品无偏好

**3. 市场需求已知（Known Market Demand）**：
- 企业知道市场需求
- 市场需求函数为p(y)

**4. 成本函数已知（Known Cost Functions）**：
- 企业知道其他企业的成本函数
- 企业i的成本函数为Cᵢ(yᵢ)

### 10.2.3 古诺模型的利润最大化

**古诺模型的利润最大化（Profit Maximization of Cournot Model）**：
企业选择产量，使利润最大化。

**数学表述**：
```
max πᵢ = p(yᵢ + y₋ᵢ) × yᵢ - Cᵢ(yᵢ)
```

**其中**：
- πᵢ：企业i的利润
- p(yᵢ + y₋ᵢ)：价格函数
- yᵢ：企业i的产量
- y₋ᵢ：其他企业的产量
- Cᵢ(yᵢ)：企业i的成本函数

### 10.2.4 古诺模型的利润最大化条件

**古诺模型的利润最大化条件（Condition of Profit Maximization of Cournot Model）**：
```
MRᵢ = MCᵢ
```

**其中**：
- MRᵢ：企业i的边际收益
- MCᵢ：企业i的边际成本

**推导**：
```
πᵢ = p(yᵢ + y₋ᵢ) × yᵢ - Cᵢ(yᵢ)
∂πᵢ/∂yᵢ = p(yᵢ + y₋ᵢ) + p'(yᵢ + y₋ᵢ) × yᵢ - Cᵢ'(yᵢ) = 0
p(yᵢ + y₋ᵢ) + p'(yᵢ + y₋ᵢ) × yᵢ = Cᵢ'(yᵢ)
MRᵢ = MCᵢ
```

**含义**：
- 边际收益等于边际成本
- 利润最大化的必要条件

### 10.2.5 古诺模型的反应函数

**古诺模型的反应函数（Reaction Function of Cournot Model）**：
企业i的最优产量是其他企业产量的函数。

**公式**：
```
yᵢ = Rᵢ(y₋ᵢ)
```

**其中**：
- yᵢ：企业i的最优产量
- Rᵢ(·)：企业i的反应函数
- y₋ᵢ：其他企业的产量

**含义**：
- 反应函数表示企业i的最优产量如何依赖于其他企业的产量
- 反应函数的斜率为负

### 10.2.6 古诺模型的均衡

**古诺模型的均衡（Equilibrium of Cournot Model）**：
所有企业的产量选择都是对其他企业产量的最优反应。

**纳什均衡**：
```
yᵢ* = Rᵢ(y₋ᵢ*)，对所有i成立
```

**其中**：
- yᵢ*：企业i的均衡产量
- Rᵢ(·)：企业i的反应函数
- y₋ᵢ*：其他企业的均衡产量

**含义**：
- 每个企业选择产量，使利润最大化
- 给定其他企业的产量，没有企业有动机改变产量

## 10.3 伯特兰模型

### 10.3.1 伯特兰模型的定义

**伯特兰模型（Bertrand Model）**：
企业同时选择价格，使利润最大化。

### 10.3.2 伯特兰模型的假设

**伯特兰模型的假设（Assumptions of Bertrand Model）**：

**1. 同时决策（Simultaneous Decision）**：
- 企业同时选择价格
- 不知道其他企业的价格

**2. 同质产品（Homogeneous Products）**：
- 企业生产同质产品
- 消费者购买价格最低的产品

**3. 市场需求已知（Known Market Demand）**：
- 企业知道市场需求
- 市场需求函数为y(p)

**4. 成本函数已知（Known Cost Functions）**：
- 企业知道其他企业的成本函数
- 企业i的成本函数为Cᵢ(yᵢ)

### 10.3.3 伯特兰模型的利润最大化

**伯特兰模型的利润最大化（Profit Maximization of Bertrand Model）**：
企业选择价格，使利润最大化。

**数学表述**：
```
max πᵢ = pᵢ × yᵢ(pᵢ, p₋ᵢ) - Cᵢ(yᵢ)
```

**其中**：
- πᵢ：企业i的利润
- pᵢ：企业i的价格
- yᵢ(pᵢ, p₋ᵢ)：企业i的需求函数
- Cᵢ(yᵢ)：企业i的成本函数

### 10.3.4 伯特兰模型的利润最大化条件

**伯特兰模型的利润最大化条件（Condition of Profit Maximization of Bertrand Model）**：
```
pᵢ = MCᵢ
```

**其中**：
- pᵢ：企业i的价格
- MCᵢ：企业i的边际成本

**推导**：
- 如果pᵢ > MCᵢ，企业可以降低价格，获得更多利润
- 如果pᵢ < MCᵢ，企业会亏损
- 均衡时，pᵢ = MCᵢ

**含义**：
- 价格等于边际成本
- 完全竞争均衡

### 10.3.5 伯特兰悖论

**伯特兰悖论（Bertrand Paradox）**：
即使只有少数企业，价格也会等于边际成本，就像完全竞争一样。

**解释**：
- 企业之间激烈的价格竞争
- 每个企业都有动机降价
- 均衡时，价格等于边际成本

## 10.4 斯塔克尔伯格模型

### 10.4.1 斯塔克尔伯格模型的定义

**斯塔克尔伯格模型（Stackelberg Model）**：
企业依次选择产量，使利润最大化。

### 10.4.2 斯塔克尔伯格模型的假设

**斯塔克尔伯格模型的假设（Assumptions of Stackelberg Model）**：

**1. 依次决策（Sequential Decision）**：
- 领导者先选择产量
- 跟随者后选择产量

**2. 同质产品（Homogeneous Products）**：
- 企业生产同质产品
- 消费者对产品无偏好

**3. 市场需求已知（Known Market Demand）**：
- 企业知道市场需求
- 市场需求函数为p(y)

**4. 成本函数已知（Known Cost Functions）**：
- 企业知道其他企业的成本函数
- 企业i的成本函数为Cᵢ(yᵢ)

### 10.4.3 斯塔克尔伯格模型的利润最大化

**斯塔克尔伯格模型的利润最大化（Profit Maximization of Stackelberg Model）**：

**领导者**：
```
max π₁ = p(y₁ + y₂) × y₁ - C₁(y₁)
s.t. y₂ = R₂(y₁)
```

**跟随者**：
```
max π₂ = p(y₁ + y₂) × y₂ - C₂(y₂)
```

**其中**：
- π₁, π₂：领导者1和跟随者2的利润
- p(y₁ + y₂)：价格函数
- y₁, y₂：领导者1和跟随者2的产量
- C₁(y₁), C₂(y₂)：领导者1和跟随者2的成本函数
- R₂(y₁)：跟随者2的反应函数

### 10.4.4 斯塔克尔伯格模型的利润最大化条件

**斯塔克尔伯格模型的利润最大化条件（Condition of Profit Maximization of Stackelberg Model）**：

**跟随者**：
```
MR₂ = MC₂
```

**领导者**：
```
MR₁ = MC₁
```

**其中**：
- MR₁, MR₂：领导者1和跟随者2的边际收益
- MC₁, MC₂：领导者1和跟随者2的边际成本

### 10.4.5 斯塔克尔伯格模型的均衡

**斯塔克尔伯格模型的均衡（Equilibrium of Stackelberg Model）**：

**跟随者**：
```
y₂ = R₂(y₁)
```

**领导者**：
```
y₁* 满足 MR₁ = MC₁
```

**含义**：
- 跟随者根据领导者的产量选择最优产量
- 领导者知道跟随者的反应函数，选择最优产量
- 先动优势（First-Mover Advantage）

## 10.5 本章总结

### 核心要点

1. **寡头垄断的定义**：
   - 寡头垄断：少数几家大企业控制的市场结构
   - 寡头垄断的特征：少数企业、产品同质或差异化、进入壁垒、相互依存

2. **古诺模型**：
   - 古诺模型：企业同时选择产量，使利润最大化
   - 古诺模型的假设：同时决策、同质产品、市场需求已知、成本函数已知
   - 古诺模型的利润最大化条件：MRᵢ = MCᵢ
   - 古诺模型的反应函数：yᵢ = Rᵢ(y₋ᵢ)
   - 古诺模型的均衡：yᵢ* = Rᵢ(y₋ᵢ*)

3. **伯特兰模型**：
   - 伯特兰模型：企业同时选择价格，使利润最大化
   - 伯特兰模型的假设：同时决策、同质产品、市场需求已知、成本函数已知
   - 伯特兰模型的利润最大化条件：pᵢ = MCᵢ
   - 伯特兰悖论：价格等于边际成本，就像完全竞争一样

4. **斯塔克尔伯格模型**：
   - 斯塔克尔伯格模型：企业依次选择产量，使利润最大化
   - 斯塔克尔伯格模型的假设：依次决策、同质产品、市场需求已知、成本函数已知
   - 斯塔克尔伯格模型的利润最大化条件：MR₁ = MC₁, MR₂ = MC₂
   - 斯塔克尔伯格模型的均衡：跟随者根据领导者的产量选择最优产量，领导者选择最优产量
   - 先动优势
