# 第九讲：垄断

> 本章学习目标：
> - 理解垄断定价
> - 掌握垄断的无效率
> - 学会分析价格歧视

---

## 9.1 垄断定价

### 9.1.1 垄断的定义

**垄断（Monopoly）**：
市场上只有一个卖者的市场结构。

### 9.1.2 垄断企业的特征

**垄断企业的特征（Characteristics of Monopoly）**：

**1. 价格制定者（Price Maker）**：
- 企业可以影响市场价格
- 企业可以制定价格

**2. 产品独特（Unique Product）**
- 企业生产的产品是独特的
- 没有替代品

**3. 进入壁垒（Barriers to Entry）**
- 其他企业无法进入市场
- 市场进入成本高

**4. 完全信息（Perfect Information）**
- 企业和消费者拥有完全信息
- 没有信息不对称

### 9.1.3 垄断企业的利润最大化

**垄断企业的利润最大化（Profit Maximization of Monopoly）**：
选择产量，使利润最大化。

**数学表述**：
```
max π = p(y)y - C(y)
```

**其中**：
- π：利润
- p(y)：价格函数
- y：产量
- C(y)：成本函数

### 9.1.4 垄断企业的利润最大化条件

**垄断企业的利润最大化条件（Condition of Profit Maximization of Monopoly）**：
```
MR(y) = MC(y)
```

**其中**：
- MR(y)：边际收益
- MC(y)：边际成本

**推导**：
```
π = p(y)y - C(y)
dπ/dy = p(y) + p'(y)y - C'(y) = 0
p(y) + p'(y)y = C'(y)
MR(y) = MC(y)
```

**含义**：
- 边际收益等于边际成本
- 利润最大化的必要条件

### 9.1.5 垄断企业的边际收益

**垄断企业的边际收益（Marginal Revenue of Monopoly）**：
增加一单位产量带来的收益变化。

**公式**：
```
MR(y) = p(y) + p'(y)y
```

**其中**：
- MR(y)：边际收益
- p(y)：价格函数
- p'(y)：价格对产量的导数
- y：产量

**含义**：
- 边际收益小于价格
- 因为增加产量会导致价格下降

### 9.1.6 垄断企业的定价

**垄断企业的定价（Pricing of Monopoly）**：
垄断企业制定价格，使边际收益等于边际成本。

**公式**：
```
p(y) > MC(y)
```

**其中**：
- p(y)：价格
- MC(y)：边际成本

**含义**：
- 垄断价格高于边际成本
- 垄断价格高于竞争价格

## 9.2 垄断的无效率

### 9.2.1 垄断的无效率

**垄断的无效率（Inefficiency of Monopoly）**：
垄断导致无谓损失。

### 9.2.2 垄断的无效率分析

**垄断的无效率分析（Analysis of Inefficiency of Monopoly）**：

**1. 价格高于边际成本（Price > Marginal Cost）**：
- 垄断价格高于竞争价格
- 垄断产量低于竞争产量

**2. 消费者剩余减少（Consumer Surplus Decreases）**：
- 消费者剩余减少
- 生产者剩余可能增加

**3. 无谓损失（Deadweight Loss）**：
- 总剩余减少
- 无谓损失

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "y" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- MC：边际成本曲线
- DWL：无谓损失（Deadweight Loss）
- 垄断导致无谓损失

## 9.3 价格歧视

### 9.3.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
企业对同一商品向不同消费者收取不同价格。

### 9.3.2 价格歧视的条件

**价格歧视的条件（Conditions of Price Discrimination）**：

**1. 市场势力（Market Power）**：
- 企业必须有市场势力
- 企业可以影响价格

**2. 市场分割（Market Segmentation）**：
- 消费者可以被分割成不同市场
- 不同市场有不同的需求价格弹性

**3. 防止套利（Prevent Arbitrage）**：
- 消费者不能在不同市场之间套利
- 套利成本高

### 9.3.3 一级价格歧视

**一级价格歧视（First-Degree Price Discrimination）**：
企业对每个消费者收取其愿意支付的最高价格。

**效果**：
- 消费者剩余为零
- 生产者剩余最大化
- 无谓损失为零

### 9.3.4 二级价格歧视

**二级价格歧视（Second-Degree Price Discrimination）**：
企业根据消费者购买数量收取不同价格。

**例子**：
- 数量折扣
- 两部收费

### 9.3.5 三级价格歧视

**三级价格歧视（Third-Degree Price Discrimination）**：
企业根据消费者特征收取不同价格。

**利润最大化条件**：
```
MR₁ = MR₂ = ... = MRₙ = MC
```

**其中**：
- MRᵢ：市场i的边际收益
- MC：边际成本

**含义**：
- 所有市场的边际收益相等
- 边际收益等于边际成本

## 9.4 自然垄断

### 9.4.1 自然垄断的定义

**自然垄断（Natural Monopoly）**：
规模经济显著，一个企业比多个企业更有效率。

### 9.4.2 自然垄断的特征

**自然垄断的特征（Characteristics of Natural Monopoly）**：

**1. 规模经济（Economies of Scale）**：
- 平均成本随产量增加而下降
- 一个企业比多个企业更有效率

**2. 固定成本高（High Fixed Costs）**：
- 固定成本很高
- 边际成本很低

**3. 进入壁垒（Barriers to Entry）**：
- 其他企业难以进入市场
- 市场进入成本高

### 9.4.3 自然垄断的管制

**自然垄断的管制（Regulation of Natural Monopoly）**：

**1. 价格上限管制（Price Cap Regulation）**：
- 政府规定最高价格
- 例如：p = AC

**2. 回报率管制（Rate of Return Regulation）**：
- 政府规定企业的回报率
- 例如：利润率

**3. 激励管制（Incentive Regulation）**：
- 政府设计激励契约
- 例如：绩效激励

## 9.5 本章总结

### 核心要点

1. **垄断定价**：
   - 垄断：市场上只有一个卖者的市场结构
   - 垄断企业的特征：价格制定者、产品独特、进入壁垒、完全信息
   - 垄断企业的利润最大化条件：MR(y) = MC(y)
   - 垄断企业的边际收益：MR(y) = p(y) + p'(y)y
   - 垄断企业的定价：p(y) > MC(y)

2. **垄断的无效率**：
   - 垄断的无效率：垄断导致无谓损失
   - 价格高于边际成本：p(y) > MC(y)
   - 消费者剩余减少
   - 无谓损失

3. **价格歧视**：
   - 价格歧视：企业对同一商品向不同消费者收取不同价格
   - 价格歧视的条件：市场势力、市场分割、防止套利
   - 一级价格歧视：企业对每个消费者收取其愿意支付的最高价格
   - 二级价格歧视：企业根据消费者购买数量收取不同价格
   - 三级价格歧视：企业根据消费者特征收取不同价格
   - 三级价格歧视的利润最大化条件：MR₁ = MR₂ = ... = MRₙ = MC

4. **自然垄断**：
   - 自然垄断：规模经济显著，一个企业比多个企业更有效率
   - 自然垄断的特征：规模经济、固定成本高、进入壁垒
   - 自然垄断的管制：价格上限管制、回报率管制、激励管制
