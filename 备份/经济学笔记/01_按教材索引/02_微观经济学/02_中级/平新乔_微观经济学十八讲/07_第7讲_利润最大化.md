## 7.1 利润最大化问题

### 7.1.1 利润的定义

**利润（Profit）**：
总收入减去总成本。

### 7.1.2 利润的公式

**利润的公式（Formula of Profit）**：
```
π = py - C(y)
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

### 7.1.3 利润最大化问题

**利润最大化问题（Profit Maximization Problem）**：
选择产量，使利润最大化。

**数学表述**：
```
max π = py - C(y)
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

### 7.1.4 利润最大化条件

**利润最大化条件（Condition of Profit Maximization）**：
```
p = MC(y)
```

**其中**：
- p：价格
- MC(y)：边际成本

**推导**：
```
π = py - C(y)
dπ/dy = p - C'(y) = 0
p = C'(y) = MC(y)
```

**含义**：
- 价格等于边际成本
- 利润最大化的必要条件

### 7.1.5 利润最大化的二阶条件

**利润最大化的二阶条件（Second-Order Condition of Profit Maximization）**：
```
dMC/dy > 0
```

**其中**：
- MC：边际成本
- y：产量

**推导**：
```
d²π/dy² = -C''(y) < 0
C''(y) > 0
dMC/dy > 0
```

**含义**：
- 边际成本递增
- 利润最大化的充分条件

## 7.2 竞争性企业

### 7.2.1 竞争性企业的定义

**竞争性企业（Competitive Firm）**：
在完全竞争市场上的企业。

### 7.2.2 竞争性企业的特征

**竞争性企业的特征（Characteristics of Competitive Firm）**：

**1. 价格接受者（Price Taker）**：
- 企业不能影响市场价格
- 企业只能接受市场价格

**2. 产品同质（Homogeneous Product）**
- 企业生产的产品是同质的
- 消费者对产品无偏好

**3. 自由进入和退出（Free Entry and Exit）**
- 企业可以自由进入和退出市场
- 没有进入壁垒

**4. 完全信息（Perfect Information）**
- 企业和消费者拥有完全信息
- 没有信息不对称

### 7.2.3 竞争性企业的利润最大化

**竞争性企业的利润最大化（Profit Maximization of Competitive Firm）**：
选择产量，使利润最大化。

**数学表述**：
```
max π = py - C(y)
s.t. p 给定
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

**利润最大化条件**：
```
p = MC(y)
```

**其中**：
- p：价格
- MC(y)：边际成本

## 7.3 供给函数

### 7.3.1 供给函数的定义

**供给函数（Supply Function）**：
表示供给量与价格之间关系的函数。

### 7.3.2 竞争性企业的供给函数

**竞争性企业的供给函数（Supply Function of Competitive Firm）**：
表示竞争性企业的供给量与价格之间关系的函数。

**公式**：
```
y = S(p)
```

**其中**：
- y：供给量
- S(p)：供给函数
- p：价格

### 7.3.3 竞争性企业的供给曲线

**竞争性企业的供给曲线（Supply Curve of Competitive Firm）**：
表示竞争性企业的供给量与价格之间关系的曲线。

**图形表示**：

```plotly
data:
  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "p"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- S：供给曲线
- 横轴：供给量y
- 纵轴：价格p
- 供给曲线向右上方倾斜

### 7.3.4 竞争性企业的供给函数的性质

**竞争性企业的供给函数的性质（Properties of Supply Function of Competitive Firm）**：

**1. 单调性（Monotonicity）**：
- 价格上升，供给量增加
- dy/dp > 0

**2. 零次齐次性（Homogeneity of Degree Zero）**：
- 供给函数是价格的零次齐次函数
- S(tp) = S(p)

**3. 连续性（Continuity）**：
- 供给函数是连续的

## 7.4 生产者剩余

### 7.4.1 生产者剩余的定义

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

### 7.4.2 生产者剩余的计算

**生产者剩余的公式（Formula of Producer Surplus）**：
```
PS = py - ∫₀^y MC(q) dq
```

**其中**：
- PS：生产者剩余
- p：实际获得价格
- y：供给量
- MC(q)：边际成本函数

**含义**：
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

**图形表示**：

```plotly
data:
  -
    type: scatter
    mode: lines
    name: 曲线 1
    x: [0, 20, 40, 60, 80, 100]
    y: [25, 20, 15, 10, 5, 0]
    line:
      color: "#1f77b4"
      width: 4
      shape: spline
layout:
  title:
    text: "需求曲线"
  xaxis:
    title:
      text: "y"
  yaxis:
    title:
      text: "价格"
    range:
      - 0
      - 25
  template: "plotly_white"
  showlegend: true
config:
  displayModeBar: false
  responsive: true
```

**分析**：
- MC：边际成本曲线
- PS：生产者剩余
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

## 7.5 本章总结

1. **利润最大化问题**：
   - 利润：π = py - C(y)

   - 利润最大化条件：p = MC(y)
   - 利润最大化的二阶条件：dMC/dy > 0

2. **竞争性企业**：
   - 竞争性企业：在完全竞争市场上的企业
   - 竞争性企业的特征：价格接受者、产品同质、自由进入和退出、完全信息
   - 竞争性企业的利润最大化条件：p = MC(y)

3. **供给函数**：
   - 供给函数：表示供给量与价格之间关系的函数
   - 竞争性企业的供给函数：y = S(p)
   - 竞争性企业的供给曲线：表示竞争性企业的供给量与价格之间关系的曲线
   - 竞争性企业的供给函数的性质：单调性、零次齐次性、连续性

4. **生产者剩余**：
   - 生产者剩余：生产者实际获得的价格与愿意接受的最低价格之间的差额
   - 生产者剩余的计算：PS = py - ∫₀^y MC(q) dq
   - 生产者剩余等于价格线上方、边际成本曲线下方的面积
