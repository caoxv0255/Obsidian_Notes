# 第二讲：消费者选择

> 本章学习目标：
> - 理解消费者均衡
> - 掌握收入效应和替代效应
> - 学会分析斯卢茨基方程

---

## 2.1 消费者均衡

### 2.1.1 消费者均衡的定义

**消费者均衡（Consumer Equilibrium）**：
在给定预算约束下，消费者选择使其效用最大化的商品组合。

### 2.1.2 预算约束

**预算约束（Budget Constraint）**：
消费者的支出不能超过其收入。

**公式**：
```
p₁x₁ + p₂x₂ = m
```

**其中**：
- p₁, p₂：商品1和商品2的价格
- x₁, x₂：商品1和商品2的消费量
- m：收入

**预算线方程**：
```
x₂ = m/p₂ - (p₁/p₂)x₁
```

**斜率**：
- 预算线的斜率为-p₁/p₂

**图形表示**：


**分析**：
- 纵截距：m/p₂
- 横截距：m/p₁
- 斜率：-p₁/p₂

### 2.1.3 消费者均衡条件

**消费者均衡条件（Condition of Consumer Equilibrium）**：
选择商品组合，使效用最大化。

**数学表述**：
```
max U(x₁, x₂)
s.t. p₁x₁ + p₂x₂ = m
```

**拉格朗日函数**：
```
L = U(x₁, x₂) + λ(m - p₁x₁ - p₂x₂)
```

**一阶条件**：
```
∂L/∂x₁ = MU₁ - λp₁ = 0
∂L/∂x₂ = MU₂ - λp₂ = 0
∂L/∂λ = m - p₁x₁ - p₂x₂ = 0
```

**均衡条件**：
```
MU₁/p₁ = MU₂/p₂ = λ
```

**其中**：
- MU₁, MU₂：商品1和商品2的边际效用
- p₁, p₂：商品1和商品2的价格
- λ：拉格朗日乘数（收入的边际效用）

**含义**：
- 每种商品的边际效用与价格之比相等
- 每种商品的边际替代率等于价格之比
- MRS₁₂ = MU₁/MU₂ = p₁/p₂

## 2.2 收入效应和替代效应

### 2.2.1 价格变化的分解

**价格变化的分解（Decomposition of Price Change）**：
价格变化可以分为替代效应和收入效应。

**公式**：
```
Δx = Δx^s + Δx^m
```

**其中**：
- Δx：价格变化导致的需求变化
- Δx^s：替代效应
- Δx^m：收入效应

### 2.2.2 替代效应

**替代效应（Substitution Effect）**：
在保持效用不变的情况下，价格变化导致的需求变化。

**希克斯替代效应（Hicksian Substitution Effect）**：
- 保持效用不变
- 替代效应总是负的

**公式**：
```
∂xᵢ/∂pⱼ|U=U̅
```

**含义**：
- 价格上升，替代效应为负
- 价格下降，替代效应为正

### 2.2.3 收入效应

**收入效应（Income Effect）**：
价格变化导致实际收入变化，从而引起需求变化。

**公式**：
```
∂xᵢ/∂m × xⱼ
```

**其中**：
- ∂xᵢ/∂m：商品i的收入效应
- xⱼ：商品j的消费量

**含义**：
- 对于正常商品：收入效应为正
- 对于劣等商品：收入效应为负

### 2.2.4 正常商品和劣等商品

**正常商品（Normal Goods）**：
- 收入增加，需求增加
- 收入效应为正

**劣等商品（Inferior Goods）**：
- 收入增加，需求减少
- 收入效应为负

**吉芬商品（Giffen Goods）**：
- 价格上升，需求增加
- 收入效应为负，且绝对值大于替代效应

## 2.3 斯卢茨基方程

### 2.3.1 斯卢茨基方程的定义

**斯卢茨基方程（Slutsky Equation）**：
描述价格变化对需求的影响的分解方程。

### 2.3.2 斯卢茨基方程的公式

**斯卢茨基方程的公式（Formula of Slutsky Equation）**：
```
∂xᵢ/∂pⱼ = ∂xᵢ/∂pⱼ|U=U̅ - xⱼ(∂xᵢ/∂m)
```

**其中**：
- ∂xᵢ/∂pⱼ：价格变化的总效应
- ∂xᵢ/∂pⱼ|U=U̅：替代效应（希克斯）
- -xⱼ(∂xᵢ/∂m)：收入效应

### 2.3.3 斯卢茨基方程的含义

**斯卢茨基方程的含义（Meaning of Slutsky Equation）**：
- 总效应 = 替代效应 + 收入效应
- 替代效应总是负的
- 收入效应可以是正的或负的

### 2.3.4 自身价格效应

**自身价格效应（Own-Price Effect）**：
商品自身价格变化对该商品需求的影响。

**斯卢茨基方程**：
```
∂xᵢ/∂pᵢ = ∂xᵢ/∂pᵢ|U=U̅ - xᵢ(∂xᵢ/∂m)
```

**含义**：
- 替代效应为负
- 收入效应取决于商品性质
- 对于正常商品：总效应为负
- 对于劣等商品：总效应可能为正（吉芬商品）

## 2.4 需求弹性

### 2.4.1 需求弹性的定义

**需求弹性（Elasticity of Demand）**：
需求对价格或收入变化的敏感程度。

### 2.4.2 需求价格弹性

**需求价格弹性（Price Elasticity of Demand）**：
需求对价格变化的敏感程度。

**公式**：
```
ε = (∂x/∂p) × (p/x)
```

**其中**：
- ε：需求价格弹性
- ∂x/∂p：需求对价格的导数
- p：价格
- x：需求量

**分类**：
- |ε| > 1：富有弹性
- |ε| = 1：单位弹性
- |ε| < 1：缺乏弹性

### 2.4.3 需求收入弹性

**需求收入弹性（Income Elasticity of Demand）**：
需求对收入变化的敏感程度。

**公式**：
```
η = (∂x/∂m) × (m/x)
```

**其中**：
- η：需求收入弹性
- ∂x/∂m：需求对收入的导数
- m：收入
- x：需求量

**分类**：
- η > 0：正常商品
- η < 0：劣等商品
- η > 1：奢侈品
- 0 < η < 1：必需品

## 2.5 本章总结

### 核心要点

1. **消费者均衡**：
   - 消费者均衡：在给定预算约束下，消费者选择使其效用最大化的商品组合
   - 预算约束：p₁x₁ + p₂x₂ = m
   - 消费者均衡条件：MU₁/p₁ = MU₂/p₂ = λ

2. **收入效应和替代效应**：
   - 价格变化的分解：Δx = Δx^s + Δx^m
   - 替代效应：在保持效用不变的情况下，价格变化导致的需求变化
   - 收入效应：价格变化导致实际收入变化，从而引起需求变化
   - 正常商品：收入效应为正
   - 劣等商品：收入效应为负
   - 吉芬商品：价格上升，需求增加

3. **斯卢茨基方程**：
   - 斯卢茨基方程：∂xᵢ/∂pⱼ = ∂xᵢ/∂pⱼ|U=U̅ - xⱼ(∂xᵢ/∂m)
   - 总效应 = 替代效应 + 收入效应
   - 替代效应总是负的
   - 收入效应可以是正的或负的

4. **需求弹性**：
   - 需求价格弹性：ε = (∂x/∂p) × (p/x)
   - 需求收入弹性：η = (∂x/∂m) × (m/x)
   - 正常商品：η > 0
   - 劣等商品：η < 0
