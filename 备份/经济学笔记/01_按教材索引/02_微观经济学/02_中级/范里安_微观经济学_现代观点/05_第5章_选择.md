## 5.1 消费者选择的基本问题

### 5.1.1 消费者选择的基本问题

**消费者选择的基本问题（Consumer's Basic Problem）**：
在预算约束下，选择最优商品组合，使效用最大化。

### 5.1.2 消费者选择的数学表述

**消费者选择的数学表述（Mathematical Formulation of Consumer's Problem）**：

**目标函数（Objective Function）**：
```
max u(x₁, x₂)
```

**约束条件（Constraint）**：
```
p₁x₁ + p₂x₂ = m
```

**其中**：
- u(x₁, x₂)：效用函数
- p₁：商品1的价格
- p₂：商品2的价格
- x₁：商品1的数量
- x₂：商品2的数量
- m：收入

---

## 5.2 消费者最优选择的条件

### 5.2.1 消费者最优选择的几何条件

**消费者最优选择的几何条件（Geometric Condition of Optimal Choice）**：
无差异曲线与预算线相切。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "x₁" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IC：无差异曲线
- 预算线：p₁x₁ + p₂x₂ = m
- 最优选择：无差异曲线与预算线相切的点

### 5.2.2 消费者最优选择的代数条件

**消费者最优选择的代数条件（Algebraic Condition of Optimal Choice）**：
```
MRS = p₁/p₂
```

**证明**：
- 无差异曲线：u(x₁, x₂) = 常数
- 无差异曲线的斜率：MRS = -dx₂/dx₁ = MU₁/MU₂
- 预算线的斜率：-p₁/p₂
- 无差异曲线与预算线相切：MRS = p₁/p₂

**含义**：
- 边际替代率等于价格比
- 效用最大化条件

### 5.2.3 消费者最优选择的求解

**消费者最优选择的求解（Solving for Optimal Choice）**：

**步骤1：写出消费者选择问题**：
```
max u(x₁, x₂)
s.t. p₁x₁ + p₂x₂ = m
```

**步骤2：建立拉格朗日函数**：
```
L = u(x₁, x₂) + λ(m - p₁x₁ - p₂x₂)
```

**步骤3：求一阶条件**：
```
∂L/∂x₁ = MU₁ - λp₁ = 0
```
```
∂L/∂x₂ = MU₂ - λp₂ = 0
```
```
∂L/∂λ = m - p₁x₁ - p₂x₂ = 0
```

**步骤4：求解一阶条件**：
```
MU₁ = λp₁
```
```
MU₂ = λp₂
```
```
p₁x₁ + p₂x₂ = m
```

**步骤5：求解最优选择**：
```
MU₁/MU₂ = p₁/p₂
```
```
MRS = p₁/p₂
```

## 5.3 消费者选择的变化

### 5.3.1 收入变化

**收入变化（Change in Income）**：

**收入增加**：
- 预算线向外平移
- 最优选择向外移动
- 效用增加

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "x₁" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IC₁：无差异曲线1
- IC₂：无差异曲线2
- 预算线₁：收入m₁
- 预算线₂：收入m₂ > m₁
- 最优选择从预算线₁和IC₁的切点移动到预算线₂和IC₂的切点

**收入减少**：
- 预算线向内平移
- 最优选择向内移动
- 效用减少

### 5.3.2 价格变化

**价格变化（Change in Price）**：

**商品1价格下降**：
- 预算线向外旋转
- 最优选择移动
- 效用增加

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "x₁" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IC₁：无差异曲线1
- IC₂：无差异曲线2
- 预算线₁：价格p₁₁
- 预算线₂：价格p₁₂ < p₁₁
- 最优选择从预算线₁和IC₁的切点移动到预算线₂和IC₂的切点

**商品1价格上升**：
- 预算线向内旋转
- 最优选择移动
- 效用减少

- 收入提供曲线：最优选择移动的轨迹

### 5.4.2 恩格尔曲线

**恩格尔曲线（Engel Curve）**：
收入与商品需求量之间的关系。

**图形表示**：


- 横轴：收入m
- 纵轴：商品1的需求量x₁

**恩格尔曲线的类型**：

**1. 正常商品（Normal Goods）**：
- 恩格尔曲线向右上方倾斜
- 收入增加，需求量增加

**2. 劣等商品（Inferior Goods）**：
- 恩格尔曲线向右下方倾斜
- 收入增加，需求量减少

## 5.6 替代效应和收入效应

### 5.6.1 替代效应

**替代效应（Substitution Effect）**：
由于商品相对价格变化，消费者调整商品组合，保持效用不变。

### 5.6.2 收入效应

**收入效应（Income Effect）**：
由于商品价格变化，消费者的实际收入变化，从而调整商品组合。

### 5.6.3 总效应

**总效应（Total Effect）**：
替代效应和收入效应的总和。

**公式**：
```
总效应 = 替代效应 + 收入效应
```

### 5.6.4 正常商品的替代效应和收入效应

**正常商品（Normal Goods）**：

**替代效应**：
- 价格下降，相对价格变化
- 增加价格下降的商品的消费
- 减少价格上升的商品的消费

**收入效应**：
- 价格下降，实际收入增加
- 增加正常商品的消费

**总效应**：
- 替代效应和收入效应同方向
- 总效应：增加价格下降的商品的消费

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "x₁" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IC₁：无差异曲线1
- IC₂：无差异曲线2
- 预算线₁：价格p₁₁
- 预算线₂：价格p₁₂ < p₁₁
- 替代效应：从IC₁和预算线₁的切点移动到IC₁和补偿预算线的切点
- 收入效应：从IC₁和补偿预算线的切点移动到IC₂和预算线₂的切点
- 总效应：从IC₁和预算线₁的切点移动到IC₂和预算线₂的切点

### 5.6.5 劣等商品的替代效应和收入效应

**劣等商品（Inferior Goods）**：

**替代效应**：
- 价格下降，相对价格变化
- 增加价格下降的商品的消费
- 减少价格上升的商品的消费

**收入效应**：
- 价格下降，实际收入增加
- 减少劣等商品的消费

**总效应**：
- 替代效应和收入效应反方向
- 如果替代效应 > 收入效应，总效应：增加价格下降的商品的消费
- 如果替代效应 < 收入效应，总效应：减少价格下降的商品的消费（吉芬商品）

## 5.7 本章总结

### 核心要点

1. **消费者选择的基本问题**：
   - 消费者选择的基本问题：在预算约束下，选择最优商品组合，使效用最大化
   - 数学表述：max u(x₁, x₂) s.t. p₁x₁ + p₂x₂ = m

2. **消费者最优选择的条件**：
   - 几何条件：无差异曲线与预算线相切
   - 代数条件：MRS = p₁/p₂
   - 求解方法：拉格朗日方法

3. **消费者选择的变化**：
   - 收入变化：预算线平移，最优选择移动
   - 价格变化：预算线旋转，最优选择移动

4. **收入提供曲线和恩格尔曲线**：
   - 收入提供曲线：收入变化时，最优选择移动的轨迹
   - 恩格尔曲线：收入与商品需求量之间的关系
   - 正常商品：恩格尔曲线向右上方倾斜
   - 劣等商品：恩格尔曲线向右下方倾斜

5. **价格提供曲线和需求曲线**：
   - 价格提供曲线：价格变化时，最优选择移动的轨迹
   - 需求曲线：价格与需求量之间的关系
   - 正常需求曲线：需求曲线向右下方倾斜
   - 吉芬商品：需求曲线向右上方倾斜

6. **替代效应和收入效应**：
   - 替代效应：由于商品相对价格变化，消费者调整商品组合，保持效用不变
   - 收入效应：由于商品价格变化，消费者的实际收入变化，从而调整商品组合
   - 总效应：替代效应和收入效应的总和
   - 正常商品：替代效应和收入效应同方向
   - 劣等商品：替代效应和收入效应反方向
