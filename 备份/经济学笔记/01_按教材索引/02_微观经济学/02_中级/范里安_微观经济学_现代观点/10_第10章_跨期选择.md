# 第10章：跨期选择

> 本章学习目标：
> - 理解跨期选择的定义
> - 掌握跨期预算约束
> - 学会分析跨期消费选择
> - 理解利率变化的影响

---

## 10.1 跨期选择的定义

### 10.1.1 跨期选择的定义

**跨期选择（Intertemporal Choice）**：
消费者在不同时期之间分配消费的选择。

### 10.1.2 跨期选择的假设

**跨期选择的假设（Assumptions of Intertemporal Choice）**：

**1. 两期模型（Two-Period Model）**：
- 时期1：现在（Today）
- 时期2：未来（Tomorrow）

**2. 完全确定（Certainty）**：
- 消费者确切知道未来收入

**3. 完全市场（Perfect Markets）**：
- 消费者可以以利率r借款或储蓄

---

## 10.2 跨期预算约束

### 10.2.1 跨期预算约束的定义

**跨期预算约束（Intertemporal Budget Constraint）**：
消费者在两期之间的消费受到收入的限制。

### 10.2.2 跨期预算约束的公式

**跨期预算约束的公式（Formula of Intertemporal Budget Constraint）**：
```
c₁ + c₂/(1 + r) = m₁ + m₂/(1 + r)
```

**其中**：
- c₁：时期1的消费
- c₂：时期2的消费
- r：利率
- m₁：时期1的收入
- m₂：时期2的收入

**含义**：
- 左边：两期消费的现值
- 右边：两期收入的现值

### 10.2.3 现值

**现值（Present Value, PV）**：
未来收入或消费的当前价值。

**公式**：
```
PV = FV/(1 + r)^t
```

**其中**：
- PV：现值
- FV：未来值
- r：利率
- t：时期

**含义**：
- 1单位的未来价值等于1/(1 + r)^t单位的现值
- 利率越高，现值越低

---

## 10.3 消费者选择

### 10.3.1 消费者选择问题

**消费者选择问题（Consumer Choice Problem）**：
在跨期预算约束下，选择最优跨期消费组合，使效用最大化。

**数学表述**：
```
max u(c₁, c₂)
s.t. c₁ + c₂/(1 + r) = m₁ + m₂/(1 + r)
```

### 10.3.2 消费者最优选择的条件

**消费者最优选择的条件（Condition of Optimal Choice）**：
```
MRS = 1 + r
```

**证明**：
- 无差异曲线：u(c₁, c₂) = 常数
- 无差异曲线的斜率：MRS = -dc₂/dc₁ = MU₁/MU₂
- 跨期预算约束线的斜率：-(1 + r)
- 无差异曲线与跨期预算约束线相切：MRS = 1 + r

**含义**：
- 边际替代率等于1加利率
- 时期1消费的边际效用等于时期2消费的边际效用乘以1加利率

---

## 10.4 利率变化的影响

### 10.4.1 利率上升的影响

**利率上升的影响（Effect of Interest Rate Increase）**：

**替代效应（Substitution Effect）**：
- 利率上升，未来消费变得更便宜
- 减少时期1消费，增加时期2消费
- 储蓄增加

**收入效应（Income Effect）**：
- 利率上升，储蓄者的实际收入增加
- 储蓄者：增加时期1消费和时期2消费
- 借款者：减少时期1消费和时期2消费

**总效应（Total Effect）**：
- 储蓄者：时期1消费可能增加或减少，取决于替代效应和收入效应的大小
- 借款者：时期1消费减少

### 10.4.2 利率下降的影响

**利率下降的影响（Effect of Interest Rate Decrease）**：

**替代效应（Substitution Effect）**：
- 利率下降，未来消费变得更贵
- 增加时期1消费，减少时期2消费
- 储蓄减少

**收入效应（Income Effect）**：
- 利率下降，储蓄者的实际收入减少
- 储蓄者：减少时期1消费和时期2消费
- 借款者：增加时期1消费和时期2消费

**总效应（Total Effect）**：
- 储蓄者：时期1消费减少
- 借款者：时期1消费可能增加或减少，取决于替代效应和收入效应的大小

---

## 10.5 储蓄和借款

### 10.5.1 储蓄者

**储蓄者（Saver）**：
时期1消费小于时期1收入。

**公式**：
```
c₁ < m₁
```

**含义**：
- 消费者在时期1储蓄
- 时期2消费大于时期2收入

### 10.5.2 借款者

**借款者（Borrower）**：
时期1消费大于时期1收入。

**公式**：
```
c₁ > m₁
```

**含义**：
- 消费者在时期1借款
- 时期2消费小于时期2收入

---

**储蓄供给曲线（Saving Supply Curve）**：

## 10.8 本章总结

### 核心要点

1. **跨期选择的定义**：
   - 跨期选择：消费者在不同时期之间分配消费的选择
   - 跨期选择的假设：两期模型、完全确定、完全市场

2. **跨期预算约束**：
   - 跨期预算约束：消费者在两期之间的消费受到收入的限制
   - 跨期预算约束的公式：c₁ + c₂/(1 + r) = m₁ + m₂/(1 + r)
   - 现值：未来收入或消费的当前价值，PV = FV/(1 + r)^t

3. **消费者选择**：
   - 消费者选择问题：在跨期预算约束下，选择最优跨期消费组合，使效用最大化
   - 消费者最优选择的条件：MRS = 1 + r

4. **利率变化的影响**：
   - 利率上升：替代效应（减少时期1消费，增加时期2消费）+ 收入效应（储蓄者增加消费，借款者减少消费）
   - 利率下降：替代效应（增加时期1消费，减少时期2消费）+ 收入效应（储蓄者减少消费，借款者增加消费）

5. **储蓄和借款**：
   - 储蓄者：时期1消费小于时期1收入，c₁ < m₁
   - 借款者：时期1消费大于时期1收入，c₁ > m₁

6. **储蓄供给曲线**：
   - 储蓄供给曲线：表示利率和储蓄之间关系的曲线
   - 储蓄供给曲线向右上方倾斜

7. **借款需求曲线**：
   - 借款需求曲线：表示利率和借款之间关系的曲线
   - 借款需求曲线向右下方倾斜
