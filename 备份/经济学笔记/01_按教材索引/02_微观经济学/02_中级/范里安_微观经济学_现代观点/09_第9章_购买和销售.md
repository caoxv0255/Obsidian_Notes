# 第9章：购买和销售

> 本章学习目标：
> - 理解禀赋的定义
> - 掌握禀赋约束
> - 学会分析禀赋收入
> - 理解禀赋效应

---

## 9.1 禀赋的定义

### 9.1.1 禀赋的定义

**禀赋（Endowment）**：
消费者初始拥有的商品数量。

### 9.1.2 禀赋的表示

**禀赋的表示（Representation of Endowment）**：
```
(ω₁, ω₂)
```

**其中**：
- ω₁：消费者初始拥有的商品1的数量
- ω₂：消费者初始拥有的商品2的数量

---

## 9.2 禀赋约束

### 9.2.1 禀赋约束的定义

**禀赋约束（Endowment Constraint）**：
消费者的预算约束基于禀赋。

### 9.2.2 禀赋约束的公式

**禀赋约束的公式（Formula of Endowment Constraint）**：
```
p₁x₁ + p₂x₂ = p₁ω₁ + p₂ω₂
```

**其中**：
- p₁：商品1的价格
- p₂：商品2的价格
- x₁：商品1的消费量
- x₂：商品2的消费量
- ω₁：消费者初始拥有的商品1的数量
- ω₂：消费者初始拥有的商品2的数量

**含义**：
- 左边：消费者的支出
- 右边：消费者的收入（禀赋的价值）

### 9.2.3 禀赋收入

**禀赋收入（Endowment Income）**：
消费者通过出售禀赋获得的收入。

**公式**：
```
m = p₁ω₁ + p₂ω₂
```

**其中**：
- m：禀赋收入
- p₁：商品1的价格
- p₂：商品2的价格
- ω₁：消费者初始拥有的商品1的数量
- ω₂：消费者初始拥有的商品2的数量

---

## 9.3 消费者选择

### 9.3.1 消费者选择问题

**消费者选择问题（Consumer Choice Problem）**：
在禀赋约束下，选择最优商品组合，使效用最大化。

**数学表述**：
```
max u(x₁, x₂)
s.t. p₁x₁ + p₂x₂ = p₁ω₁ + p₂ω₂
```

### 9.3.2 消费者最优选择的条件

**消费者最优选择的条件（Condition of Optimal Choice）**：
```
MRS = p₁/p₂
```

**证明**：
- 无差异曲线：u(x₁, x₂) = 常数
- 无差异曲线的斜率：MRS = -dx₂/dx₁ = MU₁/MU₂
- 禀赋约束线的斜率：-p₁/p₂
- 无差异曲线与禀赋约束线相切：MRS = p₁/p₂

---

## 9.4 禀赋效应

### 9.4.1 禀赋效应的定义

**禀赋效应（Endowment Effect）**：
由于拥有某种商品，消费者对该商品的评价提高。

### 9.4.2 禀赋效应的原因

**禀赋效应的原因（Causes of Endowment Effect）**：

**1. 损失厌恶（Loss Aversion）**：
- 消费者对损失的敏感度高于对收益的敏感度
- 失去禀赋的痛苦大于获得禀赋的快乐

**2. 心理账户（Mental Accounting）**：
- 消费者对禀赋和购买的商品分别对待
- 禀赋被视为"我自己的商品"

**3. 身份认同（Identity）**：
- 禀赋与消费者的身份认同相关
- 消费者不愿意失去与身份认同相关的商品

### 9.4.3 禀赋效应的影响

**禀赋效应的影响（Impact of Endowment Effect）**：

**1. 减少交易（Reduce Trading）**：
- 消费者不愿意出售禀赋
- 市场交易减少

**2. 提高保留价格（Increase Reservation Price）**：
- 消费者对禀赋的保留价格高于市场价格
- 消费者不愿意以市场价格出售禀赋

**3. 价格粘性（Price Stickiness）**：
- 市场价格调整缓慢
- 价格粘性

---

## 9.5 净需求

### 9.5.1 净需求的定义

**净需求（Net Demand）**：
消费者的消费量减去禀赋。

### 9.5.2 净需求的公式

**净需求的公式（Formula of Net Demand）**：
```
x₁ - ω₁
```
```
x₂ - ω₂
```

**其中**：
- x₁：商品1的消费量
- ω₁：商品1的禀赋
- x₂：商品2的消费量
- ω₂：商品2的禀赋

**含义**：
- 净需求 > 0：净买者（Net Buyer）
- 净需求 < 0：净卖者（Net Seller）
- 净需求 = 0：净平衡（Net Balanced）

---

## 9.6 禀赋供给曲线

### 9.6.1 禀赋供给曲线的定义

**禀赋供给曲线（Endowment Supply Curve）**：
表示价格和净供给之间关系的曲线。

### 9.6.2 禀赋供给曲线的图形

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "p₁" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- S：禀赋供给曲线
- 横轴：净供给
- 纵轴：价格p₁
- 禀赋供给曲线向右上方倾斜
- 价格上升，净供给增加

---

## 9.7 劳动供给

### 9.7.1 劳动供给的定义

**劳动供给（Labor Supply）**：
消费者提供的劳动时间。

### 9.7.2 劳动供给的模型

**劳动供给的模型（Model of Labor Supply）**：

**禀赋**：
- 总时间：T
- 闲暇：L
- 劳动：l = T - L

**预算约束**：
```
C = w(T - L) + V
```

**其中**：
- C：消费
- w：工资率
- T：总时间
- L：闲暇
- V：非劳动收入

**消费者选择问题**：
```
max u(C, L)
s.t. C = w(T - L) + V
```

### 9.7.3 劳动供给曲线

**劳动供给曲线（Labor Supply Curve）**：
表示工资率和劳动供给之间关系的曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "w" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- LS：劳动供给曲线
- 横轴：劳动供给l
- 纵轴：工资率w
- 劳动供给曲线可能向右上方倾斜（正常情况）或向后弯曲（高工资情况）

## 9.8 本章总结

### 核心要点

1. **禀赋的定义**：
   - 禀赋：消费者初始拥有的商品数量
   - 禀赋的表示：(ω₁, ω₂)

2. **禀赋约束**：
   - 禀赋约束：消费者的预算约束基于禀赋
   - 禀赋约束的公式：p₁x₁ + p₂x₂ = p₁ω₁ + p₂ω₂
   - 禀赋收入：m = p₁ω₁ + p₂ω₂

3. **消费者选择**：
   - 消费者选择问题：在禀赋约束下，选择最优商品组合，使效用最大化
   - 消费者最优选择的条件：MRS = p₁/p₂

4. **禀赋效应**：
   - 禀赋效应：由于拥有某种商品，消费者对该商品的评价提高
   - 禀赋效应的原因：损失厌恶、心理账户、身份认同
   - 禀赋效应的影响：减少交易、提高保留价格、价格粘性

5. **净需求**：
   - 净需求：消费者的消费量减去禀赋
   - 净需求的公式：x₁ - ω₁，x₂ - ω₂
   - 净需求 > 0：净买者；净需求 < 0：净卖者；净需求 = 0：净平衡

6. **禀赋供给曲线**：
   - 禀赋供给曲线：表示价格和净供给之间关系的曲线
   - 禀赋供给曲线向右上方倾斜

7. **劳动供给**：
   - 劳动供给：消费者提供的劳动时间
   - 劳动供给的模型：max u(C, L) s.t. C = w(T - L) + V
   - 劳动供给曲线：表示工资率和劳动供给之间关系的曲线
