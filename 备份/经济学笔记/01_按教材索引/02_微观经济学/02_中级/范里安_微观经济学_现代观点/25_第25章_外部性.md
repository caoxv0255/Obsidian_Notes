# 第25章：外部性

> 本章学习目标：
> - 理解外部性的定义
> - 掌握外部性的类型
> - 学会分析外部性的影响
> - 理解外部性的解决方法

---

## 25.1 外部性的定义

### 25.1.1 外部性的定义

**外部性（Externality）**：
一个经济主体的行为对另一个经济主体产生影响，但没有通过价格机制反映出来。

### 25.1.2 外部性的类型

**外部性的类型（Types of Externalities）**：

**1. 正外部性（Positive Externality）**：
- 行为产生有益的影响
- 例如：教育、研究

**2. 负外部性（Negative Externality）**：
- 行为产生有害的影响
- 例如：污染、噪音

---

## 25.2 外部性的影响

### 25.2.1 负外部性

**负外部性（Negative Externality）**：

**例子**：工厂污染

**私人成本（Private Cost, PC）**：
```
PC(y) = C(y)
```

**社会成本（Social Cost, SC）**：
```
SC(y) = C(y) + EC(y)
```

**其中**：
- C(y)：私人成本
- EC(y)：外部成本

**含义**：
- 社会成本大于私人成本
- 市场产量高于社会最优产量

**图形表示**：

**分析**：

- y*：社会最优产量
- yₘ：市场产量
- 市场产量高于社会最优产量
- 无谓损失：外部性导致的福利损失

### 25.2.2 正外部性

**正外部性（Positive Externality）**：

**例子**：教育

**私人收益（Private Benefit, PB）**：
```
PB(y) = B(y)
```

**社会收益（Social Benefit, SB）**：
```
SB(y) = B(y) + EB(y)
```

**其中**：
- B(y)：私人收益
- EB(y)：外部收益

**含义**：
- 社会收益大于私人收益
- 市场产量低于社会最优产量

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "y" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- SB：社会收益曲线
- PB：私人收益曲线
- MC：边际成本曲线
- y*：社会最优产量
- yₘ：市场产量
- 市场产量低于社会最优产量
- 无谓损失：外部性导致的福利损失

## 25.3 外部性的解决方法

### 25.3.1 庇古税

**庇古税（Pigouvian Tax）**：
政府征收等于外部成本的税。

**公式**：
```
t = EC(y*)
```

**其中**：
- t：庇古税
- EC(y*)：社会最优产量下的外部成本

**含义**：
- 庇古税使私人成本等于社会成本
- 实现社会最优产量

### 25.3.2 补贴

**补贴（Subsidy）**：
政府给予等于外部收益的补贴。

**公式**：
```
s = EB(y*)
```

**其中**：
- s：补贴
- EB(y*)：社会最优产量下的外部收益

**含义**：
- 补贴使私人收益等于社会收益
- 实现社会最优产量

### 25.3.3 产权和科斯定理

**产权（Property Rights）**：
明确界定资源的所有权。

**科斯定理（Coase Theorem）**：
如果产权明确界定且交易成本为零，外部性可以通过私人谈判解决。

**含义**：
- 无论产权归谁，资源配置都是有效的
- 政府干预不是必需的

### 25.3.4 数量管制

**数量管制（Quantity Regulation）**：
政府规定污染数量或产量上限。

**方法**：
- 发放可交易许可证（ Tradable Permits）
- 建立污染许可证市场

**含义**：
- 可交易许可证市场可以实现成本有效的污染控制
- 污染控制成本低的厂商出售许可证
- 污染控制成本高的厂商购买许可证

## 25.4 本章总结

### 核心要点

1. **外部性的定义**：
   - 外部性：一个经济主体的行为对另一个经济主体产生影响，但没有通过价格机制反映出来
   - 外部性的类型：正外部性、负外部性

2. **外部性的影响**：
   - 负外部性：社会成本大于私人成本，市场产量高于社会最优产量
   - 正外部性：社会收益大于私人收益，市场产量低于社会最优产量

3. **外部性的解决方法**：
   - 庇古税：政府征收等于外部成本的税
   - 补贴：政府给予等于外部收益的补贴
   - 产权和科斯定理：明确产权，通过私人谈判解决外部性
   - 数量管制：政府规定污染数量或产量上限
