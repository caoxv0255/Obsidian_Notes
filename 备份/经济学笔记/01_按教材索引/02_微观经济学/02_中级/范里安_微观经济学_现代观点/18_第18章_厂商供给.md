# 第18章：厂商供给

> 本章学习目标：
> - 理解厂商供给的定义
> - 掌握完全竞争厂商的供给决策
> - 学会分析厂商供给曲线
> - 理解生产者剩余

---

## 18.1 厂商供给的定义

### 18.1.1 厂商供给的定义

**厂商供给（Firm Supply）**：
厂商在不同价格下愿意供给的产量。

### 18.1.2 完全竞争市场

**完全竞争市场（Perfectly Competitive Market）**：
- 价格接受者
- 产品同质化
- 自由进入和退出
- 完全信息

---

## 18.2 完全竞争厂商的供给决策

### 18.2.1 利润最大化

**利润最大化（Profit Maximization）**：
```
max π = p × y - C(y)
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

### 18.2.2 利润最大化的条件

**利润最大化的条件（Condition of Profit Maximization）**：
```
p = MC(y)
```

**其中**：
- p：价格
- MC(y)：边际成本

**含义**：
- 价格等于边际成本
- 利润最大化的必要条件

**二阶条件**：
```
dMC/dy > 0
```

**含义**：
- 边际成本递增
- 利润最大化的充分条件

---

## 18.3 厂商供给曲线

### 18.3.1 厂商供给曲线的定义

**厂商供给曲线（Firm Supply Curve）**：
表示价格和供给量之间关系的曲线。

### 18.3.2 厂商供给曲线的推导

**厂商供给曲线的推导（Derivation of Firm Supply Curve）**：

**利润最大化条件**：
```
p = MC(y)
```

**反供给函数**：
```
y = y(p)
```

**含义**：
- 供给函数是边际成本曲线的上升部分
- 价格越高，供给量越大

## 18.4 短期供给曲线

### 18.4.1 短期供给曲线的定义

**短期供给曲线（Short-Run Supply Curve）**：
固定成本存在，可变成本变化的供给曲线。

### 18.4.2 短期供给曲线的推导

**短期供给曲线的推导（Derivation of Short-Run Supply Curve）**：

**利润最大化条件**：
```
p = SRMC(y)
```

**关停条件**：
```
p < AVC(y)
```

**含义**：
- 如果价格低于平均可变成本，厂商关停
- 短期供给曲线是SRMC曲线在AVC上方的部分

### 18.4.3 短期供给曲线的图形

**图形表示**：

    y-axis "价格" 0 --> 25

```

**分析**：
- SRMC：短期边际成本曲线
- AVC：平均可变成本曲线
- 短期供给曲线是SRMC曲线在AVC上方的部分

## 18.5 长期供给曲线

### 18.5.1 长期供给曲线的定义

**长期供给曲线（Long-Run Supply Curve）**：
固定成本不存在，所有成本都是可变的供给曲线。

### 18.5.2 长期供给曲线的推导

**长期供给曲线的推导（Derivation of Long-Run Supply Curve）**：

**利润最大化条件**：
```
p = LRMC(y)
```

**零利润条件**：
```
p = LRAC(y)
```

**含义**：
- 长期利润为零
- 长期供给曲线是LRMC曲线在LRAC上方的部分

### 18.5.3 长期供给曲线的图形

**图形表示**：

```

**分析**：
- LRMC：长期边际成本曲线
- LRAC：长期平均成本曲线
- 长期供给曲线是LRMC曲线在LRAC上方的部分

## 18.6 生产者剩余

### 18.6.1 生产者剩余的定义

**生产者剩余（Producer Surplus）**：
生产者实际获得的价格与愿意接受的最低价格之间的差额。

### 18.6.2 生产者剩余的计算

**生产者剩余的公式（Formula of Producer Surplus）**：
```
PS = p × y - ∫₀^ˣ MC(q) dq
```

**其中**：
- PS：生产者剩余
- p：实际获得价格
- y：供给量
- MC(q)：边际成本函数

**含义**：
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

**图形表示**：

```mermaid

    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- MC：边际成本曲线
- PS：生产者剩余
- 生产者剩余等于价格线上方、边际成本曲线下方的面积

## 18.7 本章总结

### 核心要点

1. **厂商供给的定义**：
   - 厂商供给：厂商在不同价格下愿意供给的产量
   - 完全竞争市场：价格接受者、产品同质化、自由进入和退出、完全信息

2. **完全竞争厂商的供给决策**：
   - 利润最大化：max π = p × y - C(y)
   - 利润最大化的条件：p = MC(y)
   - 二阶条件：dMC/dy > 0

3. **厂商供给曲线**：
   - 厂商供给曲线：表示价格和供给量之间关系的曲线
   - 厂商供给曲线的推导：p = MC(y)，y = y(p)
   - 厂商供给曲线向右上方倾斜

4. **短期供给曲线**：
   - 短期供给曲线：固定成本存在，可变成本变化的供给曲线
   - 短期供给曲线的推导：p = SRMC(y)，关停条件：p < AVC(y)
   - 短期供给曲线是SRMC曲线在AVC上方的部分

5. **长期供给曲线**：
   - 长期供给曲线：固定成本不存在，所有成本都是可变的供给曲线
   - 长期供给曲线的推导：p = LRMC(y)，零利润条件：p = LRAC(y)
   - 长期供给曲线是LRMC曲线在LRAC上方的部分

6. **生产者剩余**：
   - 生产者剩余：生产者实际获得的价格与愿意接受的最低价格之间的差额
   - 生产者剩余的计算：PS = p × y - ∫₀^ˣ MC(q) dq
   - 生产者剩余等于价格线上方、边际成本曲线下方的面积
