# 第15章：利润最大化

> 本章学习目标：
> - 理解利润的定义
> - 掌握利润最大化的条件
> - 学会分析利润最大化的决策
> - 理解利润最大化的性质

---

## 15.1 利润的定义

### 15.1.1 利润的定义

**利润（Profit）**：
总收入减去总成本。

### 15.1.2 利润的公式

**利润的公式（Formula of Profit）**：
```
π = R - C
```

**其中**：
- π：利润
- R：收入
- C：成本

**收入**：
```
R = p × y
```

**成本**：
```
C = C(y)
```

**利润**：
```
π = p × y - C(y)
```

**其中**：
- p：价格
- y：产量
- C(y)：成本函数

---

## 15.2 利润最大化问题

### 15.2.1 利润最大化问题的定义

**利润最大化问题（Profit Maximization Problem）**：
选择产量，使利润最大化。

### 15.2.2 利润最大化问题的数学表述

**利润最大化问题的数学表述（Mathematical Formulation of Profit Maximization Problem）**：
```
max π = p × y - C(y)
```

**其中**：
- π：利润
- p：价格
- y：产量
- C(y)：成本函数

---

## 15.3 利润最大化的条件

### 15.3.1 一阶条件

**一阶条件（First-Order Condition, FOC）**：
```
dπ/dy = p - C'(y) = 0
```

**其中**：
- dπ/dy：利润对产量的导数
- p：价格
- C'(y)：边际成本

**含义**：
- 价格等于边际成本
- 利润最大化的必要条件

### 15.3.2 二阶条件

**二阶条件（Second-Order Condition, SOC）**：
```
d²π/dy² = -C''(y) < 0
```

**其中**：
- d²π/dy²：利润对产量的二阶导数
- C''(y)：边际成本对产量的导数

**含义**：
- 边际成本递增
- 利润最大化的充分条件

## 15.5 供给函数

### 15.5.1 供给函数的定义

**供给函数（Supply Function）**：
表示价格和供给量之间关系的函数。

### 15.5.2 供给函数的推导

**供给函数的推导（Derivation of Supply Function）**：

**利润最大化条件**：
```
p = MC(y)
```

**反供给函数**：
```
y = y(p)
```

**含义**：
- 供给函数是边际成本曲线的上升部分
- 价格越高，供给量越大

### 15.5.3 供给曲线

**供给曲线（Supply Curve）**：
表示价格和供给量之间关系的曲线。

**图形表示**：

```mermaid

    y-axis "价格" 0 --> 25

```

**分析**：
- S：供给曲线
- 供给曲线向右上方倾斜
- 价格上升，供给量增加

## 15.6 利润最大化的性质

### 15.6.1 利润函数的性质

**利润函数的性质（Properties of Profit Function）**：

**1. 利润函数是价格的增函数（Profit Function is Increasing in Price）**：
```
∂π/∂p = y ≥ 0
```

**含义**：
- 价格上升，利润增加

**2. 利润函数是价格的凸函数（Profit Function is Convex in Price）**：
```
∂²π/∂p² = ∂y/∂p ≥ 0
```

**含义**：
- 利润函数是凸的
- 霍特林引理（Hotelling's Lemma）

### 15.6.2 霍特林引理

**霍特林引理（Hotelling's Lemma）**：
```
∂π/∂p = y
```

**其中**：
- ∂π/∂p：利润对价格的偏导数
- y：供给量

**含义**：
- 利润对价格的偏导数等于供给量
- 供给函数是利润函数对价格的导数

## 15.7 短期和长期

### 15.7.1 短期利润最大化

**短期利润最大化（Short-Run Profit Maximization）**：
固定成本存在，可变成本变化。

**利润最大化条件**：
```
p = MC(y)
```

**含义**：
- 短期利润最大化条件与长期相同
- 价格等于边际成本

### 15.7.2 长期利润最大化

**长期利润最大化（Long-Run Profit Maximization）**：
固定成本不存在，所有成本都是可变的。

**利润最大化条件**：
```
p = MC(y) = AC(y)
```

**含义**：
- 长期利润最大化条件比短期更严格
- 价格等于边际成本，也等于平均成本
- 长期利润为零（正常利润）

## 15.8 本章总结

### 核心要点

1. **利润的定义**：
   - 利润：总收入减去总成本
   - 利润的公式：π = p × y - C(y)

2. **利润最大化问题**：
   - 利润最大化问题：选择产量，使利润最大化
   - 利润最大化问题的数学表述：max π = p × y - C(y)

3. **利润最大化的条件**：
   - 一阶条件：dπ/dy = p - C'(y) = 0，价格等于边际成本
   - 二阶条件：d²π/dy² = -C''(y) < 0，边际成本递增

4. **利润最大化的图形分析**：
   - 利润曲线是倒U形
   - 最优产量：边际成本曲线和价格线的交点

5. **供给函数**：
   - 供给函数：表示价格和供给量之间关系的函数
   - 供给函数的推导：p = MC(y)，y = y(p)
   - 供给曲线向右上方倾斜

6. **利润最大化的性质**：
   - 利润函数是价格的增函数：∂π/∂p = y ≥ 0
   - 利润函数是价格的凸函数：∂²π/∂p² = ∂y/∂p ≥ 0
   - 霍特林引理：∂π/∂p = y

7. **短期和长期**：
   - 短期利润最大化：p = MC(y)
   - 长期利润最大化：p = MC(y) = AC(y)，长期利润为零
