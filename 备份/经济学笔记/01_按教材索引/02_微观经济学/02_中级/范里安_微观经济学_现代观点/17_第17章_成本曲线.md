## 17.1 成本曲线的定义

### 17.1.1 成本曲线的定义

**成本曲线（Cost Curves）**：
表示成本和产量之间关系的曲线。

### 17.1.2 成本曲线的类型

**成本曲线的类型（Types of Cost Curves）**：

**1. 总成本曲线（Total Cost Curve, TC）**：
```
TC(y) = C(w₁, w₂, y)
```

**2. 固定成本曲线（Fixed Cost Curve, FC）**：
```
FC = C(w₁, w₂, 0)
```

**3. 可变成本曲线（Variable Cost Curve, VC）**：
```
VC(y) = TC(y) - FC
```

**4. 平均成本曲线（Average Cost Curve, AC）**：
```
AC(y) = TC(y)/y
```

**5. 平均固定成本曲线（Average Fixed Cost Curve, AFC）**：
```
AFC(y) = FC/y
```

**6. 平均可变成本曲线（Average Variable Cost Curve, AVC）**：
```
AVC(y) = VC(y)/y
```

**7. 边际成本曲线（Marginal Cost Curve, MC）**：
```
MC(y) = dTC(y)/dy = dVC(y)/dy
```

## 17.2 短期成本曲线

### 17.2.1 短期成本曲线的定义

**短期成本曲线（Short-Run Cost Curves）**：
固定成本存在，可变成本变化的成本曲线。

- VC：可变成本曲线
- FC：固定成本曲线

### 17.2.3 短期平均成本曲线

**短期平均成本曲线（Short-Run Average Cost Curve, SRAC）**：
```
SRAC(y) = TC(y)/y = AFC(y) + AVC(y)
```

- SRAC = AFC + AVC

## 17.4 成本曲线之间的关系

### 17.4.1 平均成本和边际成本的关系

**平均成本和边际成本的关系（Relationship between Average Cost and Marginal Cost）**：

**当MC < AC时**：
- AC递减

**当MC = AC时**：
- AC达到最小值

**当MC > AC时**：
- AC递增

**图形表示**：

```plotly

```

**分析**：
- MC：边际成本曲线
- AC：平均成本曲线
- MC通过AC的最低点

### 17.4.2 短期成本曲线和长期成本曲线的关系

**短期成本曲线和长期成本曲线的关系（Relationship between Short-Run and Long-Run Cost Curves）**：

**LRAC是SRAC的包络线**：
```
LRAC(y) = minₖ SRACₖ(y)
```

**LRMC与SRMC的关系**：
```
LRMC(y) = MCₖ(y)
```

**其中**：
- k：最优工厂规模

**含义**：
- 在产量y处，长期边际成本等于最优工厂规模的短期边际成本

## 17.5 规模经济和规模不经济

### 17.5.1 规模经济

**规模经济（Economies of Scale）**：
长期平均成本随产量增加而下降。

**条件**：
```
dLRAC/dy < 0
```

**含义**：
- 产量增加，长期平均成本下降
- 大规模生产更有效率

### 17.5.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
长期平均成本随产量增加而上升。

**条件**：
```
dLRAC/dy > 0
```

**含义**：
- 产量增加，长期平均成本上升
- 大规模生产效率下降

### 17.5.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
长期平均成本不随产量变化。

**条件**：
```
dLRAC/dy = 0
```

**含义**：
- 产量变化，长期平均成本不变
- 规模报酬不变

## 17.6 本章总结

### 核心要点

1. **成本曲线的定义**：
   - 成本曲线：表示成本和产量之间关系的曲线
   - 成本曲线的类型：TC、FC、VC、AC、AFC、AVC、MC

2. **短期成本曲线**：
   - 短期成本曲线：固定成本存在，可变成本变化的成本曲线
   - 短期平均成本曲线：SRAC = AFC + AVC
   - 短期边际成本曲线：SRMC通过SRAC和AVC的最低点

3. **长期成本曲线**：
   - 长期成本曲线：固定成本不存在，所有成本都是可变的成本曲线
   - 长期平均成本曲线：LRAC是SRAC的包络线
   - 长期边际成本曲线：LRMC通过LRAC的最低点

4. **成本曲线之间的关系**：
   - 平均成本和边际成本的关系：MC < AC时AC递减，MC = AC时AC达到最小值，MC > AC时AC递增
   - 短期成本曲线和长期成本曲线的关系：LRAC是SRAC的包络线，LRMC = MCₖ

5. **规模经济和规模不经济**：
   - 规模经济：长期平均成本随产量增加而下降
   - 规模不经济：长期平均成本随产量增加而上升
   - 规模报酬不变：长期平均成本不随产量变化
