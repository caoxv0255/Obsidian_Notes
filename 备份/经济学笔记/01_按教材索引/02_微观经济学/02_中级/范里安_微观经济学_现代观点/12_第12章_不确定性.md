# 第12章：不确定性

> 本章学习目标：
> - 理解不确定性的定义
> - 掌握期望效用理论
> - 学会分析风险态度
> - 理解风险规避和风险偏好

---

## 12.1 不确定性的定义

### 12.1.1 不确定性的定义

**不确定性（Uncertainty）**：
结果不确定的情况。

### 12.1.2 不确定性的表示

**不确定性的表示（Representation of Uncertainty）**：

**1. 概率分布（Probability Distribution）**：
```
P = {p₁, p₂, ..., pₙ}
```
其中，pᵢ是结果i的概率，Σpᵢ = 1

**2. 期望值（Expected Value）**：
```
E[X] = Σₜ pₜ xₜ
```
其中，xₜ是结果t的值

**3. 方差（Variance）**：
```
Var[X] = Σₜ pₜ(xₜ - E[X])²
```

---

## 12.2 期望效用理论

### 12.2.1 期望效用理论的基本假设

**期望效用理论的基本假设（Basic Assumptions of Expected Utility Theory）**：

**1. 完备性（Completeness）**：
- 对于任意两个结果，消费者能够比较

**2. 传递性（Transitivity）**：
- 如果A优于B，B优于C，那么A优于C

**3. 连续性（Continuity）**：
- 偏好关系是连续的

**4. 独立性（Independence）**：
- 消费者对某个结果的偏好独立于其他结果

### 12.2.2 期望效用函数

**期望效用函数（Expected Utility Function）**：
```
U(P) = Σₜ pₜ u(xₜ)
```

**其中**：
- U(P)：概率分布P的期望效用
- pₜ：结果t的概率
- u(xₜ)：结果t的效用

**含义**：
- 期望效用等于各结果效用的期望值
- 期望效用函数是冯·诺伊曼-摩根斯坦效用函数

## 12.3 风险态度

### 12.3.1 风险规避

**风险规避（Risk Aversion）**：
消费者不喜欢不确定性。

**条件**：
```
u(E[X]) > E[u(X)]
```

**含义**：
- 确定结果的效用大于期望效用
- 效用函数是凹的（Concave）

### 12.3.2 风险中性

**风险中性（Risk Neutrality）**：
消费者对不确定性无所谓。

**条件**：
```
u(E[X]) = E[u(X)]
```

**含义**：
- 确定结果的效用等于期望效用
- 效用函数是线性的（Linear）

### 12.3.3 风险偏好

**风险偏好（Risk Loving）**：
消费者喜欢不确定性。

**条件**：
```
u(E[X]) < E[u(X)]
```

**含义**：
- 确定结果的效用小于期望效用
- 效用函数是凸的（Convex）

## 12.4 风险规避的度量

### 12.4.1 阿罗-普拉特绝对风险规避度量

**阿罗-普拉特绝对风险规避度量（Arrow-Pratt Measure of Absolute Risk Aversion）**：
```
A(x) = -u''(x)/u'(x)
```

**其中**：
- A(x)：绝对风险规避度量
- u''(x)：效用函数的二阶导数
- u'(x)：效用函数的一阶导数

**含义**：
- 绝对风险规避度量衡量消费者对绝对风险的态度
- A(x) > 0：风险规避
- A(x) = 0：风险中性
- A(x) < 0：风险偏好

### 12.4.2 阿罗-普拉特相对风险规避度量

**阿罗-普拉特相对风险规避度量（Arrow-Pratt Measure of Relative Risk Aversion）**：
```
R(x) = -x u''(x)/u'(x)
```

**其中**：
- R(x)：相对风险规避度量
- x：财富
- u''(x)：效用函数的二阶导数
- u'(x)：效用函数的一阶导数

**含义**：
- 相对风险规避度量衡量消费者对相对风险的态度
- R(x) > 0：风险规避
- R(x) = 0：风险中性
- R(x) < 0：风险偏好

## 12.5 保险

### 12.5.1 保险的定义

**保险（Insurance）**：
转移风险的机制。

### 12.5.2 保险的原理

**保险的原理（Principle of Insurance）**：

**1. 风险共担（Risk Sharing）**：
- 多个个体共同承担风险
- 降低每个个体的风险

**2. 大数定律（Law of Large Numbers）**：
- 大量独立风险事件的总风险趋于稳定
- 保险公司可以预测赔付

### 12.5.3 保险需求

**保险需求（Insurance Demand）**：
风险规避的消费者愿意支付保险费。

**条件**：
```
u(w - p) ≥ E[u(w - L)]
```

**其中**：
- w：财富
- p：保险费
- L：损失
- u(·)：效用函数

**含义**：
- 支付保险费后的效用大于不保险的期望效用
- 风险规避的消费者愿意购买保险

## 12.6 本章总结

### 核心要点

1. **不确定性的定义**：
   - 不确定性：结果不确定的情况
   - 不确定性的表示：概率分布、期望值、方差

2. **期望效用理论**：
   - 期望效用理论的基本假设：完备性、传递性、连续性、独立性
   - 期望效用函数：U(P) = Σₜ pₜ u(xₜ)

3. **风险态度**：
   - 风险规避：u(E[X]) > E[u(X)]，效用函数是凹的
   - 风险中性：u(E[X]) = E[u(X)]，效用函数是线性的
   - 风险偏好：u(E[X]) < E[u(X)]，效用函数是凸的

4. **风险规避的度量**：
   - 阿罗-普拉特绝对风险规避度量：A(x) = -u''(x)/u'(x)
   - 阿罗-普拉特相对风险规避度量：R(x) = -x u''(x)/u'(x)

5. **保险**：
   - 保险：转移风险的机制
   - 保险的原理：风险共担、大数定律
   - 保险需求：风险规避的消费者愿意支付保险费
