## 4.1 个别需求

### 4.1.1 个别需求的定义

**个别需求（Individual Demand）**
单个消费者在不同价格下愿意购买的商品数量。

### 4.1.2 个别需求曲线

**个别需求曲线（Individual Demand Curve）**
表示单个消费者的价格和需求量之间关系的曲线。

**图形表示**：


- 横轴：需求量Q
- 纵轴：价格P
- 个别需求曲线向右下方倾斜

## 4.2 市场需求

### 4.2.1 市场需求的定义

**市场需求（Market Demand）**
所有消费者需求的加总。

### 4.2.2 市场需求的公式

**市场需求的公式（Formula of Market Demand）**：
```
Q = Σᵢ qᵢ(p)
```

**其中**：
- Q：市场需求
- qᵢ(p)：第i个消费者的需求
- p：价格

**含义**：
- 市场需求等于所有消费者需求的总和
- 市场需求是价格的函数

### 4.2.3 市场需求曲线

**市场需求曲线（Market Demand Curve）**

## 4.3 需求弹性

### 4.3.1 需求价格弹性

**需求价格弹性（Price Elasticity of Demand）**
需求量对价格变化的敏感程度。

**公式**：
```
Eₚ = (ΔQ/Q)/(ΔP/P)
```

**含义**：
- |Eₚ| > 1：富有弹性
- |Eₚ| < 1：缺乏弹性
- |Eₚ| = 1：单位弹性

### 4.3.2 需求收入弹性

**需求收入弹性（Income Elasticity of Demand）**
需求量对收入变化的敏感程度。

**公式**：
```
Eᵢ = (ΔQ/Q)/(ΔI/I)
```

**其中**：
- Eᵢ：需求收入弹性
- I：收入

**含义**：
- Eᵢ > 0：正常商品
- Eᵢ < 0：劣等商品

## 4.4 消费者剩余

### 4.4.1 消费者剩余的定义

**消费者剩余（Consumer Surplus）**
消费者愿意支付的最高价格与实际支付价格之间的差额。

### 4.4.2 消费者剩余的计算

**消费者剩余的公式（Formula of Consumer Surplus）**：
```
CS = ∫₀^Q D(p) dp - P × Q
```

**其中**：
- CS：消费者剩余
- D(p)：需求函数
- P：实际支付价格
- Q：消费量

**含义**：
- 消费者剩余等于需求曲线下方、价格线上方的面积

## 4.5 本章总结

### 核心要点

1. **个别需求**：
   - 个别需求：单个消费者在不同价格下愿意购买的商品数量
   - 个别需求曲线：表示单个消费者的价格和需求量之间关系的曲线

2. **市场需求**：
   - 市场需求：所有消费者需求的加总
   - 市场需求的公式：Q = Σᵢ qᵢ(p)
   - 市场需求曲线：表示市场价格和市场需求量之间关系的曲线

3. **需求弹性**：
   - 需求价格弹性：需求量对价格变化的敏感程度
   - 需求收入弹性：需求量对收入变化的敏感程度
   - 弹性的类型：富有弹性、缺乏弹性、单位弹性

4. **消费者剩余**：
   - 消费者剩余：消费者愿意支付的最高价格与实际支付价格之间的差额
   - 消费者剩余的计算：CS = ∫₀^Q D(p) dp - P × Q
   - 消费者剩余等于需求曲线下方、价格线上方的面积

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 个别需求 | 单个消费者在不同价格下愿意购买的商品数量 | qᵢ(p) |
| 个别需求曲线 | 表示单个消费者的价格和需求量之间关系的曲线 | 向右下方倾斜 |
| 市场需求 | 所有消费者需求的加总 | Q = Σᵢ qᵢ(p) |
| 市场需求曲线 | 表示市场价格和市场需求量之间关系的曲线 | 向右下方倾斜 |
| 需求价格弹性 | 需求量对价格变化的敏感程度 | Eₚ = (ΔQ/Q)/(ΔP/P) |
| 需求收入弹性 | 需求量对收入变化的敏感程度 | Eᵢ = (ΔQ/Q)/(ΔI/I) |
