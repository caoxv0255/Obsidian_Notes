# 第2章：供给与需求的基本原理

> 本章学习目标：
> - 理解供给和需求的定义
> - 掌握供给曲线和需求曲线
> - 学会分析市场均衡
> - 理解价格机制的作用

---

## 2.1 供给

### 2.1.1 供给的定义

**供给（Supply）**：
在其他条件不变的情况下，生产者在不同价格下愿意并且能够提供的商品数量。

### 2.1.2 供给曲线

**供给曲线（Supply Curve）**：
表示价格和供给量之间关系的曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- S：供给曲线
- 横轴：供给量Q
- 纵轴：价格P
- 供给曲线向右上方倾斜

### 2.1.3 供给曲线的移动

**供给曲线的移动（Shift of Supply Curve）**：

**供给增加（Increase in Supply）**：
- 供给曲线向右移动
- 在相同价格下，供给量增加

**供给减少（Decrease in Supply）**：
- 供给曲线向左移动
- 在相同价格下，供给量减少

**影响供给的因素**：
- 生产成本
- 技术进步
- 生产要素价格
- 政府政策
- 预期

---

## 2.2 需求

### 2.2.1 需求的定义

**需求（Demand）**：
在其他条件不变的情况下，消费者在不同价格下愿意并且能够购买的商品数量。

### 2.2.2 需求曲线

**需求曲线（Demand Curve）**：
表示价格和需求量之间关系的曲线。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- D：需求曲线
- 横轴：需求量Q
- 纵轴：价格P
- 需求曲线向右下方倾斜

### 2.2.3 需求曲线的移动

**需求曲线的移动（Shift of Demand Curve）**：

**需求增加（Increase in Demand）**：
- 需求曲线向右移动
- 在相同价格下，需求量增加

**需求减少（Decrease in Demand）**：
- 需求曲线向左移动
- 在相同价格下，需求量减少

**影响需求的因素**：
- 收入
- 偏好
- 相关商品价格
- 预期
- 人口

---

## 2.3 市场均衡

### 2.3.1 市场均衡的定义

**市场均衡（Market Equilibrium）**：
供给等于需求的状态。

### 2.3.2 市场均衡的条件

**市场均衡的条件（Condition of Market Equilibrium）**：
```
Qs = Qd
```

**其中**：
- Qs：供给量
- Qd：需求量

### 2.3.3 市场均衡的图形

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- S：供给曲线
- D：需求曲线
- 均衡：供给曲线和需求曲线的交点
- 均衡价格：P*
- 均衡数量：Q*

### 2.3.4 市场均衡的调整

**市场均衡的调整（Adjustment to Market Equilibrium）**：

**如果 P > P*：
- 供给 > 需求
- 存在过剩（Surplus）
- 价格下降
- 向均衡调整

**如果 P < P*：
- 供给 < 需求
- 存在短缺（Shortage）
- 价格上升
- 向均衡调整

## 2.4 弹性

### 2.4.1 需求价格弹性

**需求价格弹性（Price Elasticity of Demand）**：
需求量对价格变化的敏感程度。

**公式**：
```
Eₚ = (ΔQ/Q)/(ΔP/P)
```

**其中**：
- Eₚ：需求价格弹性
- ΔQ：需求量的变化
- Q：需求量
- ΔP：价格的变化
- P：价格

**需求价格弹性的类型**：

**1. 富有弹性（Elastic）**：
- |Eₚ| > 1
- 需求量对价格变化非常敏感

**2. 缺乏弹性（Inelastic）**：
- |Eₚ| < 1
- 需求量对价格变化不敏感

**3. 单位弹性（Unit Elastic）**：
- |Eₚ| = 1
- 需求量对价格变化的敏感程度适中

### 2.4.2 供给价格弹性

**供给价格弹性（Price Elasticity of Supply）**：
供给量对价格变化的敏感程度。

**公式**：
```
Eₛ = (ΔQ/Q)/(ΔP/P)
```

**其中**：
- Eₛ：供给价格弹性
- ΔQ：供给量的变化
- Q：供给量
- ΔP：价格的变化
- P：价格

## 2.5 本章总结

### 核心要点

1. **供给**：
   - 供给：在其他条件不变的情况下，生产者在不同价格下愿意并且能够提供的商品数量
   - 供给曲线：表示价格和供给量之间关系的曲线，向右上方倾斜
   - 供给曲线的移动：供给增加向右移动，供给减少向左移动

2. **需求**：
   - 需求：在其他条件不变的情况下，消费者在不同价格下愿意并且能够购买的商品数量
   - 需求曲线：表示价格和需求量之间关系的曲线，向右下方倾斜
   - 需求曲线的移动：需求增加向右移动，需求减少向左移动

3. **市场均衡**：
   - 市场均衡：供给等于需求的状态
   - 市场均衡的条件：Qs = Qd
   - 市场均衡的调整：价格调节市场向均衡调整

4. **弹性**：
   - 需求价格弹性：需求量对价格变化的敏感程度
   - 供给价格弹性：供给量对价格变化的敏感程度
   - 弹性的类型：富有弹性、缺乏弹性、单位弹性
