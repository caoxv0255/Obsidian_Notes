# 第7章：生产成本

> 本章学习目标：
> - 理解成本的定义
> - 掌握短期成本和长期成本
> - 学会分析成本函数
> - 理解规模经济

---

## 7.1 成本的定义

### 7.1.1 成本的定义

**成本（Cost）**：
生产者购买生产要素的支出。

### 7.1.2 成本的类型

**成本的类型（Types of Costs）**：

**1. 机会成本（Opportunity Cost）**：
- 放弃的最佳替代方案的价值
- 包括显性成本和隐性成本

**2. 显性成本（Explicit Cost）**：
- 实际货币支付的成本
- 例如：工资、租金、材料成本

**3. 隐性成本（Implicit Cost）**：
- 不涉及实际货币支付的成本
- 例如：企业主自己投入的资本和劳动的机会成本

---

## 7.2 短期成本

### 7.2.1 短期的定义

**短期（Short Run）**：
至少有一种生产要素（通常是资本）固定不变的时期。

### 7.2.2 总成本

**总成本（Total Cost, TC）**：
生产者购买生产要素的总支出。

**公式**：
```
TC = TFC + TVC
```

**其中**：
- TC：总成本
- TFC：总固定成本
- TVC：总可变成本

### 7.2.3 固定成本

**固定成本（Fixed Cost, FC）**：
不随产量变化的成本。

**公式**：
```
FC = rK̅
```

**其中**：
- FC：固定成本
- r：资本的价格
- K̅：固定资本

### 7.2.4 可变成本

**可变成本（Variable Cost, VC）**：
随产量变化的成本。

**公式**：
```
VC = wL
```

**其中**：
- VC：可变成本
- w：劳动的价格
- L：劳动

### 7.2.5 平均成本

**平均成本（Average Cost, AC）**：
单位产量的成本。

**公式**：
```
AC = TC/Q = AFC + AVC
```

**其中**：
- AC：平均成本
- TC：总成本
- Q：产量
- AFC：平均固定成本
- AVC：平均可变成本

### 7.2.6 平均固定成本

**平均固定成本（Average Fixed Cost, AFC）**：
单位产量的固定成本。

**公式**：
```
AFC = FC/Q = TFC/Q
```

**其中**：
- AFC：平均固定成本
- FC：固定成本
- Q：产量

### 7.2.7 平均可变成本

**平均可变成本（Average Variable Cost, AVC）**：
单位产量的可变成本。

**公式**：
```
AVC = VC/Q = TVC/Q
```

**其中**：
- AVC：平均可变成本
- VC：可变成本
- Q：产量

### 7.2.8 边际成本

**边际成本（Marginal Cost, MC）**：
增加一单位产量带来的成本变化。

**公式**：
```
MC = ΔTC/ΔQ = ΔTVC/ΔQ
```

**其中**：
- MC：边际成本
- ΔTC：总成本的变化
- ΔQ：产量的变化
- ΔTVC：总可变成本的变化

---

## 7.3 长期成本

### 7.3.1 长期的定义

**长期（Long Run）**：
所有生产要素都可变的时期。

### 7.3.2 长期总成本

**长期总成本（Long-Run Total Cost, LTC）**：
在长期，所有生产要素都可变时的总成本。

### 7.3.3 长期平均成本

**长期平均成本（Long-Run Average Cost, LAC）**：
单位产量的长期成本。

**公式**：
```
LAC = LTC/Q
```

**其中**：
- LAC：长期平均成本
- LTC：长期总成本
- Q：产量

### 7.3.4 长期边际成本

**长期边际成本（Long-Run Marginal Cost, LMC）**：
在长期，增加一单位产量带来的成本变化。

**公式**：
```
LMC = ΔLTC/ΔQ
```

**其中**：
- LMC：长期边际成本
- ΔLTC：长期总成本的变化
- ΔQ：产量的变化

---

## 7.4 规模经济

### 7.4.1 规模经济的定义

**规模经济（Economies of Scale）**：
长期平均成本随产量增加而下降。

### 7.4.2 规模不经济

**规模不经济（Diseconomies of Scale）**：
长期平均成本随产量增加而上升。

### 7.4.3 规模报酬不变

**规模报酬不变（Constant Returns to Scale）**：
长期平均成本不随产量变化。

---

## 7.5 本章总结

### 核心要点

1. **成本的定义**：
   - 成本：生产者购买生产要素的支出
   - 成本的类型：机会成本、显性成本、隐性成本

2. **短期成本**：
   - 短期：至少有一种生产要素（通常是资本）固定不变的时期
   - 总成本：TC = TFC + TVC
   - 固定成本：FC = rK̅
   - 可变成本：VC = wL
   - 平均成本：AC = TC/Q = AFC + AVC
   - 边际成本：MC = ΔTC/ΔQ

3. **长期成本**：
   - 长期：所有生产要素都可变的时期
   - 长期总成本：LTC
   - 长期平均成本：LAC = LTC/Q
   - 长期边际成本：LMC = ΔLTC/ΔQ

4. **规模经济**：
   - 规模经济：长期平均成本随产量增加而下降
   - 规模不经济：长期平均成本随产量增加而上升
   - 规模报酬不变：长期平均成本不随产量变化
