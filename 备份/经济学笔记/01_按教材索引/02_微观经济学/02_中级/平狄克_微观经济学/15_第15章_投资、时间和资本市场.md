# 第15章：投资、时间和资本市场

## 15.1 投资的决定

### 15.1.1 投资的定义

**投资（Investment）**：
当前支出以获得未来收益。

### 15.1.2 投资的类型

**投资的类型（Types of Investment）**：

**1. 物质投资（Physical Investment）**：
- 购买机器、设备、建筑物
- 例如：工厂投资

**2. 人力资本投资（Human Capital Investment）**：
- 教育和培训
- 例如：大学教育

**3. 金融投资（Financial Investment）**：
- 购买股票、债券
- 例如：股票投资

---

## 15.2 跨期选择

### 15.2.1 跨期选择的定义

**跨期选择（Intertemporal Choice）**：
消费者在不同时期之间的选择。

### 15.2.2 跨期选择模型

**跨期选择模型（Intertemporal Choice Model）**：

**效用函数**：
```
U = U(C₁, C₂)
```

**其中**：
- U：效用
- C₁：第一期消费
- C₂：第二期消费

**预算约束**：
```
C₁ + C₂/(1 + r) = Y₁ + Y₂/(1 + r)
```

**其中**：
- C₁：第一期消费
- C₂：第二期消费
- Y₁：第一期收入
- Y₂：第二期收入
- r：利率

**含义**：
- 第一期消费加上第二期消费的现值等于第一期收入加上第二期收入的现值

### 15.2.3 跨期选择的均衡

**跨期选择的均衡（Equilibrium of Intertemporal Choice）**：
选择第一期消费和第二期消费，使效用最大化。

**效用最大化条件**：
```
MU₁/MU₂ = 1 + r
```

**其中**：
- MU₁：第一期消费的边际效用
- MU₂：第二期消费的边际效用
- r：利率

**含义**：
- 第一期消费和第二期消费的边际替代率等于1加上利率

## 15.3 利率

### 15.3.1 利率的定义

**利率（Interest Rate）**：
使用资本的价格。

### 15.3.2 利率的类型

**利率的类型（Types of Interest Rate）**：

**1. 名义利率（Nominal Interest Rate）**：
- 未考虑通货膨胀的利率
- 例如：银行利率

**2. 实际利率（Real Interest Rate）**：
- 考虑通货膨胀的利率
- 公式：实际利率 = 名义利率 - 通货膨胀率

### 15.3.3 利率的决定

**利率的决定（Determination of Interest Rate）**：
利率由资本市场的供给和需求决定。

**图形表示**：


- 均衡利率：资本供给等于资本需求时的利率

## 15.4 资本市场

### 15.4.1 资本市场的定义

**资本市场（Capital Market）**：
资本借贷的市场。

### 15.4.2 资本市场的类型

**资本市场的类型（Types of Capital Market）**：

**1. 债券市场（Bond Market）**：
- 债券发行和交易的市场
- 例如：国债、公司债

**2. 股票市场（Stock Market）**：
- 股票发行和交易的市场
- 例如：股票

**3. 银行市场（Bank Market）**：
- 银行借贷的市场
- 例如：银行贷款

## 15.5 现值和净现值

### 15.5.1 现值的定义

**现值（Present Value, PV）**：
未来现金流在今天的价值。

### 15.5.2 现值的公式

**现值的公式（Formula of Present Value）**：
```
PV = C₁/(1 + r) + C₂/(1 + r)² + ... + Cₙ/(1 + r)ⁿ
```

**其中**：
- PV：现值
- Cᵢ：第i期现金流
- r：利率
- n：期数

### 15.5.3 净现值的定义

**净现值（Net Present Value, NPV）**：
未来现金流的现值减去初始投资。

### 15.5.4 净现值的公式

**净现值的公式（Formula of Net Present Value）**：
```
NPV = -I + C₁/(1 + r) + C₂/(1 + r)² + ... + Cₙ/(1 + r)ⁿ
```

**其中**：
- NPV：净现值
- I：初始投资
- Cᵢ：第i期现金流
- r：利率
- n：期数

### 15.5.5 投资决策准则

**投资决策准则（Investment Decision Criterion）**：

**1. 净现值准则（NPV Criterion）**：
- 如果NPV > 0，接受投资
- 如果NPV < 0，拒绝投资
- 如果NPV = 0，无所谓

**2. 内部收益率准则（IRR Criterion）**：
- 内部收益率：使NPV = 0的利率
- 如果IRR > r，接受投资
- 如果IRR < r，拒绝投资
- 如果IRR = r，无所谓

## 15.6 本章总结

### 核心要点

1. **投资的决定**：
   - 投资：当前支出以获得未来收益
   - 投资的类型：物质投资、人力资本投资、金融投资

2. **跨期选择**：
   - 跨期选择：消费者在不同时期之间的选择
   - 跨期选择模型：U = U(C₁, C₂)，C₁ + C₂/(1 + r) = Y₁ + Y₂/(1 + r)
   - 跨期选择的均衡：MU₁/MU₂ = 1 + r

3. **利率**：
   - 利率：使用资本的价格
   - 利率的类型：名义利率、实际利率
   - 利率的决定：利率由资本市场的供给和需求决定

4. **资本市场**：
   - 资本市场：资本借贷的市场
   - 资本市场的类型：债券市场、股票市场、银行市场

5. **现值和净现值**：
   - 现值：PV = C₁/(1 + r) + C₂/(1 + r)² + ... + Cₙ/(1 + r)ⁿ
   - 净现值：NPV = -I + C₁/(1 + r) + C₂/(1 + r)² + ... + Cₙ/(1 + r)ⁿ
   - 投资决策准则：净现值准则、内部收益率准则
