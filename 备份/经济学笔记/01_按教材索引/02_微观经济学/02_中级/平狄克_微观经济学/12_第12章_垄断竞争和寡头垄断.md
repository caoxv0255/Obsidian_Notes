## 12.1 垄断竞争

### 12.1.1 垄断竞争的定义

**垄断竞争（Monopolistic Competition）**：
许多企业生产差异化产品的市场结构。

### 12.1.2 垄断竞争的特征

**垄断竞争的特征（Characteristics of Monopolistic Competition）**：

**1. 许多企业（Many Firms）**：
- 市场上有许多企业
- 每个企业市场份额小

**2. 产品差异化（Product Differentiation）**
- 企业生产差异化产品
- 产品不完全替代

**3. 自由进入和退出（Free Entry and Exit）**
- 企业可以自由进入和退出市场
- 长期经济利润为零

**4. 完全信息（Perfect Information）**
- 企业和消费者拥有完全信息
- 没有信息不对称

### 12.1.3 垄断竞争的短期均衡

**垄断竞争的短期均衡（Short-Run Equilibrium of Monopolistic Competition）**：
- 企业选择产量，使利润最大化
- 边际收益等于边际成本
- 企业可能获得正利润、零利润或负利润

**条件**：
```
MR = MC
```

**其中**：
- MR：边际收益
- MC：边际成本

### 12.1.4 垄断竞争的长期均衡

**垄断竞争的长期均衡（Long-Run Equilibrium of Monopolistic Competition）**：
- 企业选择产量，使利润最大化
- 边际收益等于边际成本
- 价格等于平均成本
- 经济利润为零

**条件**：
```
MR = MC
P = AC
```

**其中**：
- MR：边际收益
- MC：边际成本
- P：价格
- AC：平均成本

## 12.2 寡头垄断

### 12.2.1 寡头垄断的定义

**寡头垄断（Oligopoly）**：
少数几家大企业控制的市场结构。

### 12.2.2 寡头垄断的特征

**寡头垄断的特征（Characteristics of Oligopoly）**：

**1. 少数企业（Few Firms）**：
- 市场上只有少数几家大企业
- 每个企业市场份额大

**2. 产品同质或差异化（Homogeneous or Differentiated Products）**
- 企业生产同质产品或差异化产品
- 产品可能不完全替代

**3. 进入壁垒（Barriers to Entry）**
- 其他企业难以进入市场
- 市场进入成本高

**4. 相互依存（Interdependence）**
- 企业决策相互影响
- 企业需要考虑竞争对手的反应

## 12.3 寡头竞争模型

### 12.3.1 古诺模型

**古诺模型（Cournot Model）**：
企业同时选择产量，使利润最大化。

**假设**：
- 企业同时选择产量
- 企业知道市场需求
- 企业知道其他企业的产量

**利润最大化问题**：
```
max πᵢ = p(Q₁ + Q₂ + ... + Qₙ) × Qᵢ - Cᵢ(Qᵢ)
```

**其中**：
- πᵢ：企业i的利润
- p(Q₁ + Q₂ + ... + Qₙ)：价格函数
- Qᵢ：企业i的产量
- Cᵢ(Qᵢ)：企业i的成本函数

**利润最大化条件**：
```
MRᵢ = MCᵢ
```

**其中**：
- MRᵢ：企业i的边际收益
- MCᵢ：企业i的边际成本

### 12.3.2 伯特兰模型

**伯特兰模型（Bertrand Model）**：
企业同时选择价格，使利润最大化。

**假设**：
- 企业同时选择价格
- 企业生产同质产品
- 消费者购买价格最低的产品

**利润最大化问题**：
```
max πᵢ = pᵢ × Qᵢ(pᵢ, p₋ᵢ) - Cᵢ(Qᵢ)
```

**其中**：
- πᵢ：企业i的利润
- pᵢ：企业i的价格
- Qᵢ(pᵢ, p₋ᵢ)：企业i的需求函数
- Cᵢ(Qᵢ)：企业i的成本函数

**利润最大化条件**：
```
pᵢ = MCᵢ
```

**其中**：
- pᵢ：企业i的价格
- MCᵢ：企业i的边际成本

**均衡**：
- 价格等于边际成本
- 完全竞争均衡

### 12.3.3 斯塔克尔伯格模型

**斯塔克尔伯格模型（Stackelberg Model）**：
企业依次选择产量，使利润最大化。

**假设**：
- 企业依次选择产量
- 领导者先选择产量
- 跟随者后选择产量

**利润最大化问题**：
- 领导者：`max π₁ = p(Q₁ + Q₂) × Q₁ - C₁(Q₁)`
- 跟随者：`max π₂ = p(Q₁ + Q₂) × Q₂ - C₂(Q₂)`

**利润最大化条件**：
- 领导者：`MR₁ = MC₁`
- 跟随者：`MR₂ = MC₂`

## 12.4 博弈论基础

### 12.4.1 博弈论的定义

**博弈论（Game Theory）**：
研究决策者策略选择的数学理论。

### 12.4.2 博弈的基本要素

**博弈的基本要素（Basic Elements of Game）**：

**1. 参与人（Players）**：
- 博弈的决策者
- 例如：企业、消费者

**2. 策略（Strategies）**：
- 参与人的行动方案
- 例如：产量、价格

**3. 收益（Payoffs）**：
- 参与人获得的收益
- 例如：利润

### 12.4.3 纳什均衡

**纳什均衡（Nash Equilibrium）**：
在其他参与人的策略给定的情况下，每个参与人的策略都是最优的。

**定义**：
```
对于所有参与人i，有：uᵢ(sᵢ*, s₋ᵢ*) ≥ uᵢ(sᵢ, s₋ᵢ*)
```

**其中**：
- uᵢ：参与人i的收益
- sᵢ*：参与人i的纳什均衡策略
- s₋ᵢ*：其他参与人的纳什均衡策略
- sᵢ：参与人i的任意策略

### 12.4.4 囚徒困境

**囚徒困境（Prisoner's Dilemma）**：
两个囚徒同时选择是否合作，最优策略是都不合作。

## 12.5 本章总结

### 核心要点

1. **垄断竞争**：
   - 垄断竞争：许多企业生产差异化产品的市场结构
   - 垄断竞争的特征：许多企业、产品差异化、自由进入和退出、完全信息
   - 垄断竞争的短期均衡：MR = MC
   - 垄断竞争的长期均衡：MR = MC，P = AC

2. **寡头垄断**：
   - 寡头垄断：少数几家大企业控制的市场结构
   - 寡头垄断的特征：少数企业、产品同质或差异化、进入壁垒、相互依存

3. **寡头竞争模型**：
   - 古诺模型：企业同时选择产量，使利润最大化
   - 伯特兰模型：企业同时选择价格，使利润最大化
   - 斯塔克尔伯格模型：企业依次选择产量，使利润最大化

4. **博弈论基础**：
   - 博弈论：研究决策者策略选择的数学理论
   - 博弈的基本要素：参与人、策略、收益
   - 纳什均衡：在其他参与人的策略给定的情况下，每个参与人的策略都是最优的
   - 囚徒困境：两个囚徒同时选择是否合作，最优策略是都不合作
