# 第15章 投资

## 本章学习目标

通过本章学习，你应该能够：
1. 理解企业固定投资的新古典理论
2. 掌握投资的决定因素：利率、税收、预期
3. 理解住宅投资的决定机制
4. 分析存货投资的行为模式
5. 应用投资理论分析实际经济问题

## 15.1 企业固定投资

### 15.1.1 租赁价格模型

**基本思想**：
企业通过比较资本租赁价格和资本成本来决定投资。

**资本租赁价格（R/P）**：
```
定义：企业使用一单位资本一年愿意支付的价格

决定因素：
1. 边际产出（MPK）：资本边际产品
2. 资本存量（K）：资本数量
3. 生产技术：生产函数形式

公式：
R/P = MPK(K)
```

**资本成本（User Cost of Capital）**：
```
定义：拥有一单位资本一年的成本

构成：
1. 利息成本：rK（机会成本）
2. 折旧成本：δK（物理损耗）
3. 价格变化：-ΔP/P·K（资本价格变化）

公式：
UC = (r + δ - ΔP/P)P_K

其中：
r：实际利率
δ：折旧率
ΔP/P：资本品价格变化率
P_K：资本品价格
```

**投资决策**：
```
企业投资条件：
如果 R/P > UC → 增加投资
如果 R/P < UC → 减少投资
如果 R/P = UC → 资本存量最优

最优资本存量K*：
MPK(K*) = r + δ - ΔP/P
```

**图形分析**：

```
MPK, UC
  ↑
  |    MPK
  |   /
  |  /
  | /     UC
  |/__________→ K
    K*

均衡点：MPK = UC
K*是最优资本存量
```

### 15.1.2 投资函数

**从最优资本存量推导投资**：
```
意愿资本存量：K* = K*(MPK, r, δ, P_K)

实际资本存量：K_{-1}

投资需求：
I = K* - K_{-1} + δK_{-1}
  = ΔK* + δK_{-1}

其中：
I：总投资
ΔK*：净投资
δK_{-1}：重置投资（补偿折旧）
```

**投资函数的简化形式**：
```
I = I(r, Y)

其中：
I_r < 0（利率与投资负相关）
I_Y > 0（收入与投资正相关）

具体形式：
I = I₀ - br + cY

其中：
I₀：自发投资
b：投资对利率的敏感度
c：投资对收入的敏感度
```


```
情况2：r = 10%
I = 100 - 5×10 + 0.2×1000 = 100 - 50 + 200 = 250

利率上升5%，投资下降25
```

### 15.1.3 税收与投资激励

**投资税抵免（Investment Tax Credit）**：
```
定义：政府为投资支出提供税收抵免

效果：
实际资本成本：
UC' = (1 - τ)UC

其中τ是税抵免率

如果τ = 10%：
UC' = 0.9UC
资本成本降低10%

投资增加：
I' = I(UC') > I(UC)
```

**公司所得税的影响**：
```
税前收益：π = R(K) - rK - δK

税后收益：π_after = (1 - t)π

其中t是公司所得税率

对投资的影响：
如果允许利息抵扣：
有效资本成本不变
投资不变

如果不允许利息抵扣：
有效资本成本上升
投资下降
```

**加速折旧**：
```
定义：允许企业更快扣除折旧

效果：
减少当前税收负担
提高投资回报率
增加投资

例子：
标准折旧：每年10%，10年
加速折旧：第一年30%，然后每年...

加速折旧降低资本成本的现值
```

### 15.1.4 新古典投资模型

**托宾的q理论**：
```
定义：q = 企业市场价值 / 资本重置成本

经济含义：
- q > 1：企业价值 > 资本成本 → 投资
- q < 1：企业价值 < 资本成本 → 减资
- q = 1：企业价值 = 资本成本 → 均衡

投资函数：
I = I(q)
I_q > 0（q与投资正相关）
```

**q的组成部分**：
```
q = (预期未来利润现值) / (资本重置成本)

预期未来利润现值 = Σ(π_t / (1+r)ᵗ)

其中：
π_t：t期预期利润
r：贴现率

如果预期利润高 → q高 → 投资高
```

**投资调整成本**：
```
现实中的投资调整需要成本：
- 安装成本
- 培训成本
- 生产中断

调整成本函数：
C = c(I/K)·K

其中c(I/K)是调整成本函数

优化问题：
max Σ[(利润 - 投资 - 调整成本)/(1+r)ᵗ]

结果：
投资是渐进的，不是瞬间完成
```


```
- 投资波动大于GDP波动
- 投资是顺周期的

投资波动标准差：8.5%
```

## 15.2 住宅投资

### 15.2.1 住宅投资的决定

**基本模型**：
```
住宅供给：
长期：垂直（住房存量固定）
短期：向上倾斜（新建筑需要时间）

住宅需求：
取决于：
- 住房价格
- 收入
- 人口
- 利率（抵押贷款利率）
```

**住房需求函数**：
```
D_H = D_H(P_H, Y, r, N)

其中：
P_H：住房价格
Y：收入
r：利率
N：人口

需求特征：
D_P < 0（价格与需求负相关）
D_Y > 0（收入与需求正相关）
D_r < 0（利率与需求负相关）
D_N > 0（人口与需求正相关）
```

### 15.2.2 住房价格与投资

**住房市场均衡**：
```
均衡条件：
D_H(P_H, Y, r, N) = S_H(P_H)

决定均衡住房价格P_H*

如果Y↑, r↓, N↑：
→ 需求增加
→ P_H*上升
→ 住房投资增加
```

**住房投资的决定**：
```
住房投资I_H取决于：
1. 住房价格P_H
2. 建筑成本C
3. 利率r

投资函数：
I_H = I_H(P_H, C, r)

特征：
I_P > 0（价格与投资正相关）
I_C < 0（成本与投资负相关）
I_r < 0（利率与投资负相关）
```

### 15.2.3 住宅投资周期

**住房投资特征**：
```
1. 长周期：
   - 住房寿命长（50-70年）
   - 存量调整缓慢
   - 周期长度15-20年

2. 高波动性：
   - 对利率敏感
   - 对收入敏感
   - 对预期敏感

3. 地域差异：
   - 不同地区供需不同
   - 人口流动影响
   - 政策差异
```

**案例：美国住房市场（2000-2010）**
```
繁荣期（2000-2006）：
- 利率下降
- 收入增长
- 人口增长
- 价格上升：2000年至2006年上升100%

衰退期（2006-2009）：
- 次贷危机
- 价格下跌：2006年至2009年下跌30%
- 投资下降：2007年至2009年下降50%

恢复期（2009-2013）：
- 政策支持
- 价格稳定
- 投资缓慢恢复
```

### 15.2.4 住房政策

**主要政策工具**：
```
1. 抵押贷款利率政策：
   - 央行政策利率影响抵押贷款利率
   - 直接影响住房需求和投资

2. 首付要求：
   - 高首付降低需求
   - 低首付刺激需求

3. 税收政策：
   - 抵押贷款利息抵扣
   - 房产税
   - 资本利得税

4. 供给政策：
   - 土地使用法规
   - 建筑标准
   - 开发许可
```

**案例：中国住房市场**
```
特征：
- 快速城市化
- 投资需求旺盛
- 政策调控频繁

政策工具：
- 限购
- 限贷
- 限价
- 房产税试点

效果：
- 2016-2017：一线城市房价暴涨
- 2017-2018：政策收紧，房价趋稳
- 2020-2021：疫情后反弹
- 2021-2023：政策收紧，市场降温
```

## 15.3 存货投资

### 15.3.1 存货的类型

**存货类型**：
```
1. 原材料存货：
   - 投入生产的原材料
   - 应对供应不确定性
   - 批量购买优惠

2. 在产品存货（Work-in-Process）：
   - 生产过程中的产品
   - 取决于生产过程
   - 生产技术决定

3. 成品存货：
   - 已完成待售产品
   - 应对需求不确定性
   - 生产平滑

4. 季节性存货：
   - 应对季节性需求
   - 生产提前准备
```

**存货持有成本**：
```
1. 仓储成本：
   - 仓库租金
   - 管理费用
   - 保险费用

2. 资金成本：
   - 占用资金的机会成本
   - 利率影响
   - 流动性考虑

3. 过期风险：
   - 产品过期
   - 技术过时
   - 款式过时
```

### 15.3.2 存货的动机

**交易动机**：
```
定义：保证生产和销售的连续进行

例子：
- 工厂需要原材料存货
- 零售商需要成品存货
- 生产过程需要在产品存货

决定因素：
- 生产和销售的节奏
- 供应可靠性
- 需求稳定性
```

**预防性动机**：
```
定义：应对需求和供应的不确定性

例子：
- 供应中断时使用存货
- 需求激增时快速响应

决定因素：
- 不确定性程度
- 缺货成本
- 持有成本
```

**投机动机**：
```
定义：预期价格变化而持有存货

机制：
如果预期价格上涨：
→ 当前增加存货
→ 未来减少存货
→ 赚取价差

例子：
- 原材料价格预期上涨
→ 企业囤积原材料
```

### 15.3.3 存货投资模型

**存货的加速模型**：
```
基本思想：
存货投资与销售正相关

公式：
I_inv = ΔS

其中：
I_inv：存货投资
ΔS：销售变化

扩展模型：
I_inv = λS - S_{-1}

其中：
λ：目标存货销售比
S：销售
```

### 15.3.4 存货投资波动

**存货投资特征**：
```
1. 高波动性：
   - 存货投资波动大于GDP
   - 波动系数约2-3倍

2. 顺周期：
   - 繁荣期：存货投资增加
   - 衰退期：存货投资减少

3. 调整快速：
   - 比企业固定投资调整快
   - 短期内大幅变化
```

**存货投资与经济周期**：
```
机制：

扩张期：
1. 销售增加
2. 存货销售比下降
3. 企业增加存货
4. 存货投资增加

衰退期：
1. 销售下降
2. 存货销售比上升
3. 企业减少存货
4. 存货投资减少
```

**案例：2008年金融危机**
```
存货投资变化：
2008Q1：销售开始下降
2008Q2：存货仍增加（预期调整滞后）
2008Q3：开始减少存货
2008Q4：存货大幅减少（预期调整）

存货投资对GDP的贡献：
2008Q1：+0.5%
2008Q2：+0.3%
2008Q3：-1.0%
2008Q4：-2.5%

存货投资加剧衰退
```

### 15.3.5 存货与生产波动

**生产平滑假说**：
```
假设：企业平滑生产，吸收需求波动

机制：
需求波动 → 存货调整 → 生产稳定

现实：
- 部分企业平滑生产
- 部分企业跟随需求
- 取决于行业特征
```

**生产与存货的关系**：
```
会计恒等式：
Y = C + I + G + NX

存货投资：
I_inv = Y - (C_d + G_d + NX_d)

其中d表示需求

如果Y > 需求 → 存货增加
如果Y < 需求 → 存货减少
```

**案例：COVID-19疫情**
```
初期（2020Q1）：
- 需求骤降
- 生产未及时调整
- 存货意外增加

中期（2020Q2）：
- 生产大幅削减
- 存货减少
- 调整过猛

后期（2020Q3-Q4）：
- 需求恢复
- 生产跟不上
- 存货短缺

教训：
- 供应链脆弱性
- 存货管理的重要性
```

## 15.4 投资的总体分析

### 15.4.1 投资的构成

**总投资的组成部分**：
```
I = I_business + I_residential + I_inventory + I_government

其中：
I_business：企业固定投资（约70%）
I_residential：住宅投资（约15%）
I_inventory：存货投资（约5%）
I_government：政府投资（约10%）

各部分特征：
1. 企业固定投资：主导部分，长期决定
2. 住宅投资：对利率敏感，周期性
3. 存货投资：波动最大，短期调整
4. 政府投资：政策驱动，反周期
```

**投资占GDP比重（国际比较）**：

| 国家 | 投资/GDP | 企业固定 | 住宅 | 存货 |
|------|----------|----------|------|------|
| 美国 | 18% | 12% | 4% | 2% |
| 中国 | 45% | 35% | 8% | 2% |
| 日本 | 20% | 13% | 5% | 2% |
| 德国 | 19% | 12% | 5% | 2% |
| 印度 | 30% | 20% | 7% | 3% |

### 15.4.2 投资的决定因素

**利率**：
```
影响机制：
r↑ → 资本成本↑ → 投资↓

传导渠道：
1. 企业固定投资：资本成本
2. 住宅投资：抵押贷款利率
3. 存货投资：资金成本

敏感度：
- 企业固定投资：中等
- 住宅投资：高
- 存货投资：低（短期）
```

**收入/产出**：
```
影响机制：
Y↑ → 需求↑ → 投资↑

加速原理：
I = v(Y - Y_{-1})

收入变化决定投资变化
```

**预期**：
```
影响机制：
E[π]↑ → q↑ → 投资↑

预期形成：
- 基于历史数据
- 基于政策信号
- 基于经济环境

预期的作用：
- 支配长期投资决策
- 影响投资时机
- 决定投资规模
```

**不确定性**：
```
影响机制：
不确定性↑ → 投资↓

机制：
- 期权价值：等待更多信息
- 风险溢价：要求更高回报
- 信贷约束：融资困难

证据：
- 政治不确定性降低投资
- 政策不确定性降低投资
- 经济不确定性降低投资
```

**税收政策**：
```
影响机制：
税收激励↑ → 投资↑

政策工具：
1. 投资税抵免
2. 加速折旧
3. 公司所得税
4. 房产税

效果：
- 短期：刺激投资
- 长期：可能扭曲投资结构
```

### 15.4.3 投资与经济波动

**投资波动来源**：
```
1. 需求冲击：
   - 消费变化
   - 政府支出变化
   - 净出口变化

2. 供给冲击：
   - 技术冲击
   - 资源价格变化
   - 生产率变化

3. 政策冲击：
   - 货币政策
   - 财政政策
   - 监管政策

4. 金融冲击：
   - 信贷紧缩
   - 资产价格波动
   - 金融危机
```

**投资的放大效应**：
```
机制：
冲击 → 投资变化 → 乘数效应 → GDP变化

乘数效应：
ΔY = k·ΔI

其中k = 1/(1-MPC)

如果MPC = 0.8：
k = 5
ΔI = 100亿 → ΔY = 500亿

投资变化被放大
```

### 15.4.4 案例研究：投资政策的国际经验

**案例1：日本的投资衰退（1990-2010）**
```
背景：
- 1990年代泡沫破裂
- 资产价格暴跌
- 银行不良贷款

投资变化：
- 企业投资从GDP的20%降至15%
- 住宅投资从GDP的8%降至4%
- 持续20年低投资

原因：
1. 需求不足
2. 利率接近零
3. 不确定性高
4. 人口老龄化

政策应对：
- 零利率政策
- 量化宽松
- 财政刺激
- 结构性改革

效果：
- 投资恢复缓慢
- 经济增长低迷
```

**案例2：中国的投资驱动增长**
```
特征：
- 投资率全球最高（45%）
- 政府主导投资
- 基础设施投资巨大

优点：
- 快速工业化
- 基础设施改善
- 就业创造

问题：
- 投资效率下降
- 债务积累
- 产能过剩
- 资源错配

转型：
- 从投资驱动转向消费驱动
- 提高投资效率
- 减少政府干预
```

## 15.5 本章总结

### 核心要点

1. **企业固定投资**：
   - 取决于资本租赁价格和资本成本
   - 利率、税收、预期影响投资
   - 投资调整需要成本和时间

2. **住宅投资**：
   - 对抵押贷款利率敏感
   - 住房价格决定投资
   - 周期性强，波动性大

3. **存货投资**：
   - 应对不确定性
   - 与销售变化相关
   - 波动性最大

4. **投资的决定因素**：
   - 利率：资本成本
   - 收入：需求拉动
   - 预期：未来前景
   - 不确定性：等待价值
   - 税收：政策激励

5. **投资与经济波动**：
   - 投资波动大于GDP
   - 投资有放大效应
   - 政策可以影响投资
