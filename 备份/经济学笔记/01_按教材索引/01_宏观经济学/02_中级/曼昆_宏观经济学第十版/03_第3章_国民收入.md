# 第3章 国民收入：源自何处，去向何方

## 章节学习目标

1. **理解生产函数的作用**：掌握生产函数的性质、边际产量递减规律及其对产出决定的影响
2. **掌握要素市场的均衡**：理解生产要素价格的决定机制和收入分配理论
3. **理解物品与劳务的需求**：掌握消费、投资、政府购买和净出口的决定因素
4. **理解储蓄与投资的关系**：掌握国民储蓄、私人储蓄、公共储蓄的概念及计算
5. **理解可贷资金市场**：掌握可贷资金市场的均衡机制和政策影响

## 3.1 生产要素

### 3.1.1 生产函数

**定义：**

生产函数描述了生产要素投入与产出之间的技术关系，表示在给定技术水平下，使用一定数量的生产要素能够生产出多少产出。

**一般形式：**
```
Y = F(K, L)
```

其中：
- Y = 产出（GDP）
- K = 资本存量
- L = 劳动投入
- F(·) = 生产函数

**生产函数的性质：**

**1. 正边际产量**
- 资本边际产量（MPK）> 0
- 劳动边际产量（MPL）> 0
- 增加要素投入会增加产出

**2. 边际产量递减**
- 资本边际产量递减：MPK递减
- 劳动边际产量递减：MPL递减
- 保持一种要素不变，增加另一种要素，边际产量最终会下降

**3. 规模报酬不变**
- 增加所有要素投入一定比例，产出也增加相同比例
- F(zK, zL) = zF(K, L)

**数学表示：**

**资本边际产量（MPK）：**
```
MPK = ∂F/∂K = F_K(K, L)
```

**劳动边际产量（MPL）：**
```
MPL = ∂F/∂L = F_L(K, L)
```

**边际产量递减：**
```
∂²F/∂K² < 0, ∂²F/∂L² < 0
```

### 3.1.2 具体生产函数形式

**柯布-道格拉斯生产函数（Cobb-Douglas Production Function）：**

**形式：**
```
Y = A K^α L^(1-α)
```

其中：
- A = 全要素生产率（技术水平）
- α = 资本产出弹性（0 < α < 1）
- 1-α = 劳动产出弹性

**性质：**

**1. 边际产量：**
```
MPK = α A K^(α-1) L^(1-α) = αY/K
MPL = (1-α) A K^α L^(-α) = (1-α)Y/L
```

**2. 边际产量递减：**
- MPK随K增加而递减
- MPL随L增加而递减

**3. 规模报酬不变：**
```
F(zK, zL) = A (zK)^α (zL)^(1-α) = z A K^α L^(1-α) = zY
```

**4. 要素份额：**
- 资本收入份额 = α
- 劳动收入份额 = 1-α

**图形分析：生产函数**

```
产出 Y
  ↑
  │
  │        /
  │       /
  │      /
  │     /
  │    /
  │   /
  │  /
  │ /
  └──────────→ 资本 K（保持L不变）
  曲线形状：边际产量递减
```

### 3.1.3 要素供给

**资本供给：**

**特点：**
- 短期：资本存量固定
- 长期：资本可以通过投资积累

**决定因素：**
- 储蓄率
- 投资回报率
- 折旧率
- 税收政策

**劳动供给：**

**特点：**
- 劳动力数量：人口、劳动参与率
- 劳动时间：工作时间
- 劳动质量：教育、培训、健康

**决定因素：**
- 人口结构
- 工资水平
- 社会保障制度
- 文化因素

**全要素生产率：**

**决定因素：**
- 技术进步
- 管理效率
- 制度质量
- 基础设施
- 人力资本

**图形分析：生产可能性边界**

```
资本密集型商品
  ↑
  │
  │   /
  │  /  PPF（生产可能性边界）
  │ /
  │/
  └────────→ 劳动密集型商品

形状：凹向原点（边际转换率递增）
```

### 3.1.4 竞争性企业

**假设：**
- 企业在竞争性要素市场和产品市场经营
- 企业是价格接受者
- 企业追求利润最大化

**利润函数：**
```
利润 = 收益 - 成本
     = P·Y - (r·K + w·L)
     = P·F(K, L) - r·K - w·L
```

其中：
- P = 产品价格
- r = 资本租金价格
- w = 工资率

**利润最大化条件：**

**1. 资本需求：**
企业雇佣资本直到资本边际产品价值等于资本价格：
```
P × MPK = r
或：MPK = r/P
```

**2. 劳动需求：**
企业雇佣劳动直到劳动边际产品价值等于工资：
```
P × MPL = w
或：MPL = w/P
```

**含义：**
- 实际工资等于劳动边际产量
- 实际资本租金等于资本边际产量

**图形分析：劳动需求**

```
实际工资 w/P
  ↑
  │
  │    MPL
  │   ↘
  │    ↘
  │     ↘
  │      ↘
  └───────→ 劳动 L

劳动需求曲线 = MPL曲线
```

### 3.1.5 案例研究：技术进步与生产函数

**技术进步类型：**

**1. 中性技术进步**
- 技术进步同等提高资本和劳动的生产率
- 生产函数形式：Y = A(t)F(K, L)

**2. 劳动增强型技术进步**
- 技术进步提高劳动生产率
- 生产函数形式：Y = F(K, A(t)L)

**3. 资本增强型技术进步**
- 技术进步提高资本生产率
- 生产函数形式：Y = F(A(t)K, L)

**例子：信息技术革命**

**影响：**
- 1990年代以来，信息技术快速发展
- 提高了全要素生产率（A）
- 改变了生产方式和企业组织

**数据证据：**
- 美国生产率增长率从1995年前的1.4%上升到1995-2004年的2.5%
- 信息技术密集型行业生产率增长更快

**机制：**
- 计算机和通讯设备成为重要生产要素
- 信息处理效率大幅提高
- 协调和交易成本下降

**启示：**
- 技术进步是经济增长的重要源泉
- 技术进步影响要素价格和收入分配
- 政策应鼓励技术创新和扩散

## 3.2 国民收入的分配

### 3.2.1 要素价格的决定

**新古典分配理论（Neoclassical Theory of Distribution）：**
在竞争性市场经济中，每种生产要素的价格等于该要素的边际产品价值。

**资本价格：**
```
r/P = MPK = ∂F/∂K
```

**劳动价格：**
```
w/P = MPL = ∂F/∂L
```

**要素收入：**

**资本收入：**
```
资本收入 = (r/P) × K = MPK × K
```

**劳动收入：**
```
劳动收入 = (w/P) × L = MPL × L
```

**总收入：**
```
Y = 资本收入 + 劳动收入
Y = MPK × K + MPL × L
```

### 3.2.2 柯布-道格拉斯生产函数与要素份额

**使用柯布-道格拉斯生产函数：Y = A K^α L^(1-α)**

**边际产量：**
```
MPK = αY/K
MPL = (1-α)Y/L
```

**要素价格：**
```
r/P = αY/K → rK = αY
w/P = (1-α)Y/L → wL = (1-α)Y
```

**要素收入份额：**

**资本收入份额：**
```
资本收入份额 = 资本收入 / 总收入 = rK / Y = α
```

**劳动收入份额：**
```
劳动收入份额 = 劳动收入 / 总收入 = wL / Y = 1-α
```

**含义：**
- 在柯布-道格拉斯生产函数下，要素收入份额是常数
- 这与长期经济数据相符（劳动收入份额约2/3，资本收入份额约1/3）

**图形分析：要素收入分配**

```
收入
  ↑
  │
  │███████
  │███████ 资本收入 (αY)
  │███████
  │███████
  │███████
  │███████ 劳动收入 ((1-α)Y)
  │███████
  │███████
  └────────→ 收入类型
```

### 3.2.3 要素份额的稳定性

**历史数据：**

| 国家 | 时期 | 劳动收入份额 | 资本收入份额 |
|------|------|-------------|-------------|
| 美国 | 1960-2020 | 65-70% | 30-35% |
| 英国 | 1960-2020 | 60-70% | 30-40% |
| 日本 | 1960-2020 | 55-65% | 35-45% |
| 中国 | 1990-2020 | 50-60% | 40-50% |

**观察：**
- 劳动收入份额相对稳定
- 资本收入份额相对稳定
- 不同国家有所差异

**理论解释：**
- 柯布-道格拉斯生产函数的假设
- 技术进步保持要素份额稳定
- 竞争性市场的调节作用

**近期变化：**
- 2000年后，部分国家劳动收入份额下降
- 原因：全球化、技术进步、劳动力市场制度变化

### 3.2.4 实际工资与实际租金

**实际工资（Real Wage）：**
```
实际工资 = w/P = MPL
```

**决定因素：**
- 劳动边际产量
- 技术水平
- 资本劳动比

**实际租金（Real Rental Price）：**
```
实际租金 = r/P = MPK
```

**决定因素：**
- 资本边际产量
- 技术水平
- 资本劳动比

**长期趋势：**

**实际工资：**
- 随技术进步而上升
- 随资本积累而上升
- 长期增长趋势

**实际租金：**
- 取决于资本积累速度
- 边际产量递减导致长期稳定或下降
- 技术进步可能提高

**图形分析：要素价格与要素投入**

```
实际工资/租金
  ↑
  │    MPL₁
  │   /|
  │  / |
  │ /  |
  │/   |
  │    | MPL₂
  │   /|
  │  / |
  └───┴──→ 劳动 L

资本积累 → MPL曲线右移 → 实际工资上升
```

### 3.2.5 收入不平等

**要素收入与个人收入：**
- 要素收入是收入分配的第一层次
- 个人收入取决于拥有的要素数量
- 收入不平等源于要素分配不均

**不平等的来源：**

**1. 资本分配不均**
- 财富分配不均
- 金融资产持有不均

**2. 劳动收入不均**
- 人力资本差异
- 技能差异
- 职业差异

**3. 其他因素**
- 继承
- 运气
- 歧视

**衡量指标：**

**基尼系数（Gini Coefficient）：**
- 0 = 完全平等
- 1 = 完全不平等

**收入十等分比：**
- 最富10%与最贫10%的收入比

**图形分析：洛伦兹曲线**

```
累计收入百分比
  ↑
100│
   │            ↗
   │          ↗
   │        ↗
   │      ↗
   │    ↗
   │  ↗
   │↗
   └────────→ 累计人口百分比
    0       100

45度线 = 完全平等
洛伦兹曲线 = 实际分配
```

**全球收入分配数据：**

| 收入十等分 | 占全球收入比重 | 年收入（美元，2018年） |
|-----------|---------------|----------------------|
| 最贫10% | 0.7% | 600 |
| 第2-4个10% | 7.5% | 1,600 |
| 第5个10% | 2.2% | 3,500 |
| 第6个10% | 3.8% | 6,000 |
| 第7个10% | 6.0% | 9,000 |
| 第8个10% | 9.5% | 14,000 |
| 第9个10% | 15.5% | 23,000 |
| 最富10% | 54.8% | 81,000 |

**不平等趋势：**

**过去几十年：**
- 国家间不平等下降（新兴市场增长）
- 国家内不平等上升（发达国家和发展中国家）
- 全球不平等略有下降

**驱动因素：**

**减少不平等的因素：**
- 新兴市场经济快速增长
- 全球化扩散技术和知识
- 贫困人口脱贫

**增加不平等的因素：**
- 技术进步偏向高技能劳动力
- 资本收入份额上升
- 全球化影响不同群体不同

**政策含义：**
- 教育和培训：提高人力资本
- 税收和转移支付：再分配
- 社会保障：提供安全网
- 促进机会平等：打破代际传递

## 3.3 物品与劳务的需求

### 3.3.1 总需求恒等式

**回顾：**
```
Y = C + I + G + NX
```

**本章假设：**
- 封闭经济：NX = 0
- Y = C + I + G

**含义：**
产出用于消费、投资和政府购买

### 3.3.2 消费

**消费函数（Consumption Function）：**

**一般形式：**
```
C = C(Y - T)
```

其中：
- C = 消费
- Y = 收入
- T = 税收
- (Y - T) = 可支配收入

**线性消费函数：**
```
C = C₀ + c(Y - T)
```

其中：
- C₀ = 自发消费（与收入无关的消费）
- c = 边际消费倾向（MPC）
- 0 < c < 1

**边际消费倾向（MPC）：**
```
MPC = ∂C/∂(Y - T) = c
```

**含义：**
可支配收入每增加1单位，消费增加c单位。

**平均消费倾向（APC）：**
```
APC = C/(Y - T)
```

**边际储蓄倾向（MPS）：**
```
MPS = 1 - MPC
```

**图形分析：消费函数**

```
消费 C
  ↑
  │    /
  │   /  C = C₀ + c(Y - T)
  │  /
  │ /
  │/
  └──────────→ 可支配收入 (Y - T)

斜率 = c = MPC
截距 = C₀
```

**消费的决定因素：**

**1. 可支配收入**
- 主要决定因素
- 正相关关系

**2. 利率**
- 影响跨期消费决策
- 利率上升可能减少消费（替代效应）
- 利率上升可能增加消费（收入效应，对于净储蓄者）

**3. 财富**
- 财富增加可能增加消费
- 财富效应

**4. 预期**
- 预期未来收入影响当前消费
- 预期未来价格影响当前消费

**5. 信贷约束**
- 借贷限制影响消费能力
- 流动性约束

**生命周期假说（Life-Cycle Hypothesis）：**

**代表人物：** 莫迪利安尼（Modigliani）

**核心思想：**
人们基于一生收入进行消费决策，平滑消费。

**模型：**
```
消费 = (W + R×Y)/T
其中：W = 财富
     R = 工作年限
     Y = 年收入
     T = 预期寿命
```

**含义：**
- 年轻人：借贷消费（收入低）
- 中年人：储蓄（收入高）
- 老年人：负储蓄（退休）

**持久收入假说（Permanent-Income Hypothesis）：**

**代表人物：** 弗里德曼（Friedman）

**核心思想：**
消费取决于持久收入而非暂时收入。

**模型：**
```
持久收入 = Yₚ
暂时收入 = Yₜ
当前收入 = Y = Yₚ + Yₜ
消费 = C = c × Yₚ
```

**含义：**
- 暂时性收入变化对消费影响小
- 持久性收入变化对消费影响大

### 3.3.3 投资

**投资函数（Investment Function）：**

**一般形式：**
```
I = I(r)
```

其中：
- I = 投资
- r = 实际利率

**投资与利率：**
- 负相关关系
- 利率上升，投资下降

**图形分析：投资函数**

```
投资 I
  ↑
  │
  │
  │      I = I(r)
  │     ↘
  │      ↘
  │       ↘
  │        ↘
  └────────→ 实际利率 r

向下倾斜：利率与投资负相关
```

**投资的决定因素：**

**1. 利率**
- 投资成本
- 主要决定因素
- 负相关

**2. 预期盈利能力**
- 预期未来收益
- 预期不确定性
- 正相关

**3. 税收政策**
- 投资税收抵免
- 折旧政策
- 影响投资回报

**4. 技术进步**
- 新技术创造投资机会
- 提高资本生产率
- 正相关

**5. 资本存量**
- 现有资本存量
- 资本边际产量
- 负相关（边际产量递减）

**新古典投资理论：**

**企业投资决策：**
企业投资直到资本边际收益等于资本成本。

**资本成本：**
```
资本成本 = r + δ
其中：r = 实际利率
     δ = 折旧率
```

**投资条件：**
```
MPK > r + δ → 增加投资
MPK < r + δ → 减少投资
MPK = r + δ → 最优资本存量
```

**托宾q理论（Tobin's q）：**

**定义：**
```
q = 企业市场价值 / 资本重置成本
```

**投资决策：**
- q > 1：投资（市场价值 > 重置成本）
- q < 1：不投资（市场价值 < 重置成本）
- q = 1：均衡状态

**含义：**
- 反映投资机会
- 股市是投资的风向标

### 3.3.4 政府购买

**政府购买（Government Purchases，G）：**

**定义：**
各级政府对商品和服务的购买。

**性质：**
- 外生变量（本章假设）
- 由政治过程决定
- 不取决于收入

**不包括：**
- 转移支付（社会保障、福利）
- 政府债务利息

**政府购买的影响：**

**1. 总需求**
- 直接增加总需求
- 通过乘数效应进一步影响

**2. 资源配置**
- 公共物品提供
- 基础设施投资

**3. 收入分配**
- 再分配功能
- 社会保障

**政府支出的决定因素：**

**1. 经济周期**
- 逆周期政策
- 稳定经济

**2. 人口结构**
- 老龄化：医疗、养老支出增加
- 教育：人口增长

**3. 政治因素**
- 选民偏好
- 利益集团

**4. 历史因素**
- 路径依赖
- 既得利益

- 收入分配改革

## 3.4 均衡收入与利率

### 3.4.1 可贷资金市场

**定义：**
可贷资金市场是储蓄供给和投资需求相互作用的市场，利率起到调节作用。

**国民储蓄：**
```
S = Y - C - G
```

**推导：**
```
Y = C + I + G
Y - C - G = I
S = I
```

**均衡条件：**
```
S(r) = I(r)
```

即：可贷资金供给等于可贷资金需求

**图形分析：可贷资金市场**

```
实际利率 r
  ↑
  │    S
  │   /
  │  /
  │ /  I
  │/
  └────────→ 可贷资金 S = I

均衡利率：S和I交点处的r
均衡数量：均衡S和I
```

### 3.4.2 可贷资金供给

**可贷资金供给 = 国民储蓄**

**组成部分：**
```
S = Y - C - G
  = 私人储蓄 + 公共储蓄
```

**私人储蓄：**
```
S_private = Y - T - C
```

**公共储蓄：**
```
S_public = T - G
```

**国民储蓄：**
```
S = S_private + S_public = (Y - T - C) + (T - G) = Y - C - G
```

**储蓄与利率：**

**替代效应：**
利率上升 → 储蓄回报增加 → 储蓄增加

**收入效应：**
利率上升 → 储蓄者更富有 → 储蓄可能减少

**净效应：**
通常替代效应大于收入效应，利率与储蓄正相关

**储蓄函数：**
```
S = S(r), S' > 0
```

**图形分析：储蓄曲线**

```
储蓄 S
  ↑
  │
  │      S(r)
  │     /
  │    /
  │   /
  │  /
  └────────→ 利率 r

向上倾斜：利率与储蓄正相关
```

### 3.4.3 可贷资金需求

**可贷资金需求 = 投资**

**投资函数：**
```
I = I(r), I' < 0
```

**投资与利率：**
利率上升 → 投资成本增加 → 投资减少

**图形分析：投资曲线**

```
投资 I
  ↑
  │
  │      I(r)
  │     ↘
  │      ↘
  │       ↘
  │        ↘
  └────────→ 利率 r

向下倾斜：利率与投资负相关
```

### 3.4.4 可贷资金市场均衡

**均衡条件：**
```
S(r) = I(r)
```

**均衡决定：**
- 利率调整使储蓄等于投资
- 利率是可贷资金的价格

**图形分析：市场均衡**

```
实际利率 r
  ↑
  │    S
  │   /
  │  / 
  │ /  均衡点
  │/   *
  └───────I────→ 可贷资金
      r*
      S* = I*
```

**调整机制：**

**如果 S > I（供给过剩）：**
- 利率下降
- 储蓄减少，投资增加
- 回到均衡

**如果 S < I（需求过剩）：**
- 利率上升
- 储蓄增加，投资减少
- 回到均衡

### 3.4.5 政策分析

**政策1：投资税收抵免**

**效果：**
- 投资增加（投资曲线右移）
- 均衡利率上升
- 储蓄增加
- 投资增加

**图形分析：**

```
实际利率 r
  ↑
  │    S
  │   /|
  │  / | I₁ → I₂
  │ /  |  ↘
  │/   |   ↘
  └────┴────→ 可贷资金
      r₁  r₂
```

**政策2：财政赤字增加**

**效果：**
- 公共储蓄减少（S_public下降）
- 国民储蓄减少（储蓄曲线左移）
- 均衡利率上升
- 投资减少（挤出效应）

**图形分析：**

```
实际利率 r
  ↑
  │    S₁ → S₂
  │   /|
  │  / |  ↘
  │ /  |   ↘
  │/   |    I
  └────┴────→ 可贷资金
      r₁  r₂
```

**挤出效应（Crowding Out）：**
政府赤字增加导致利率上升，从而挤出私人投资。

**政策3：储蓄激励**

**效果：**
- 储蓄增加（储蓄曲线右移）
- 均衡利率下降
- 投资增加

**图形分析：**

```
实际利率 r
  ↑
  │    S₁ → S₂
  │   /|
  │  / |  ↘
  │ /  |   ↘
  │/   |    I
  └────┴────→ 可贷资金
      r₁  r₂
```

### 3.4.6 案例研究：美国财政赤字与利率

**历史数据：**

| 时期 | 财政赤字率 | 实际利率 | 投资率 |
|------|-----------|---------|--------|
| 1980s | 4-5% | 3-5% | 18-20% |
| 1990s | -2%（盈余） | 2-4% | 18-20% |
| 2000s | 3-4% | 1-3% | 16-18% |
| 2010s | 4-6% | 0-2% | 16-18% |
| 2020s | 10%+ | -1-1% | 18-20% |

**观察：**

**1980s：**
- 里根政府赤字增加
- 实际利率较高
- 投资率较高（可能有其他因素）

**1990s：**
- 克林顿政府实现盈余
- 实际利率下降
- 投资率稳定

**2000s：**
- 布什政府赤字增加
- 实际利率下降（可能有其他因素）
- 投资率下降

**2010s：**
- 奥巴马政府赤字较高
- 实际利率很低
- 投资率稳定

**2020s：**
- COVID-19疫情，赤字飙升
- 实际利率为负
- 投资率较高

**分析：**

**理论预期：**
赤字增加 → 利率上升 → 投资减少

**实际情况：**
- 有时符合理论预期
- 有时不符合
- 原因：其他因素同时作用

**其他因素：**
- 全球储蓄过剩（储蓄曲线右移）
- 央行政策（利率被压低）
- 技术进步（投资需求变化）
- 预期因素

**启示：**
- 理论提供分析框架
- 实际情况更复杂
- 需要考虑多因素

## 本章总结

### 核心要点

1. **生产函数**
   - 描述要素投入与产出的关系
   - 边际产量递减
   - 规模报酬不变
   - 柯布-道格拉斯生产函数

2. **要素价格与收入分配**
   - 要素价格等于边际产品价值
   - 柯布-道格拉斯生产函数的要素份额恒定
   - 实际工资等于劳动边际产量
   - 实际租金等于资本边际产量

3. **消费与投资**
   - 消费取决于可支配收入
   - 边际消费倾向
   - 投资取决于利率
   - 投资与利率负相关

4. **可贷资金市场**
   - 储蓄供给与投资需求
   - 利率调节市场均衡
   - S = I
   - 挤出效应

5. **政策影响**
   - 投资税收抵免：增加投资
   - 财政赤字：挤出投资
   - 储蓄激励：增加储蓄和投资

### 重要概念表格

| 概念 | 定义 | 公式/关系 | 重要性 |
|------|------|----------|--------|
| 生产函数 | 要素投入与产出的关系 | Y = F(K, L) | 决定产出和增长 |
| 边际产量 | 增加一单位要素增加的产出 | MPK = ∂F/∂K, MPL = ∂F/∂L | 决定要素价格 |
| 柯布-道格拉斯生产函数 | 具体的生产函数形式 | Y = AK^αL^(1-α) | 广泛使用的生产函数 |
| 规模报酬不变 | 要素同比例增加，产出同比例增加 | F(zK, zL) = zF(K, L) | 长期增长分析 |
| 要素价格 | 要素的市场价格 | r/P = MPK, w/P = MPL | 收入分配 |
| 要素收入份额 | 要素收入占总收入的比例 | 资本份额=α, 劳动份额=1-α | 收入分配 |
| 消费函数 | 消费与可支配收入的关系 | C = C₀ + c(Y - T) | 总需求 |
| 边际消费倾向 | 可支配收入增加1单位消费增加的数量 | MPC = ∂C/∂(Y - T) | 消费行为 |
| 投资函数 | 投资与利率的关系 | I = I(r), I' < 0 | 总需求 |
| 国民储蓄 | 收入减去消费和政府支出 | S = Y - C - G | 可贷资金供给 |
| 私人储蓄 | 可支配收入减去消费 | S_private = Y - T - C | 储蓄来源 |
| 公共储蓄 | 税收减去政府支出 | S_public = T - G | 财政状况 |
| 可贷资金市场 | 储蓄供给与投资需求的市场 | S(r) = I(r) | 利率决定 |
| 挤出效应 | 政府赤字增加导致私人投资减少 | ΔI < 0 when Δ(G - T) > 0 | 财政政策效果 |
