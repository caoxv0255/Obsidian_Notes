# 第16章 货币供给与货币需求

## 本章学习目标

通过本章学习，你应该能够：
1. 理解货币的定义和功能
2. 掌握银行体系与货币创造过程
3. 分析货币需求的决定因素
4. 理解货币乘数的机制
5. 掌握货币政策工具及其传导

## 16.1 什么是货币

### 16.1.1 货币的定义

**经济学中的货币**：
```
货币的定义：在经济中被广泛接受的支付手段

与传统理解的"财富"不同：
- 货币：流动性高，可用于交易
- 财富：包括所有资产，流动性不同

货币的特征：
1. 流动性：易于转换为交易媒介
2. 可接受性：广泛接受为支付手段
3. 价值储藏：保持购买力
4. 记账单位：衡量商品价值
```

**货币的三大功能**：

**1. 交易媒介（Medium of Exchange）**：
```
定义：用于购买商品和服务的工具

优点：
- 消除物物交换的低效
- 降低交易成本
- 促进专业化分工

例子：
- 购买商品
- 支付服务
- 偿还债务
```

**2. 价值储藏（Store of Value）**：
```
定义：将购买力从现在转移到未来的方式

特征：
- 流动性：可以迅速转换为购买力
- 稳定性：保持价值的能力

比较：
- 货币：高流动性，低收益
- 债券：中等流动性，中等收益
- 股票：低流动性，高收益
```

**3. 记账单位（Unit of Account）**：
```
定义：衡量和记录价值的标准

用途：
- 标价
- 会计
- 契约

例子：
- 商品标价：100元
- 工资：5000元/月
- 合同：100万元
```

### 16.1.2 货币的衡量

**货币供给的层次**：

**M0（基础货币）**：
```
定义：现金 + 银行在央行的准备金

M0 = C + R

其中：
C：流通中的现金
R：银行准备金

特点：
- 央行可以直接控制
- 货币创造的基础
- 又称"高能货币"
```

**M1（狭义货币）**：
```
定义：现金 + 活期存款

M1 = C + D

其中：
D：活期存款（可开支票的存款）

包括：
- 现金（纸币和硬币）
- 旅行支票
- 活期存款
- 其他可开支票的存款

用途：
- 日常交易
- 流动性最强
```

**M2（广义货币）**：
```
定义：M1 + 储蓄存款 + 小额定期存款 + 货币市场基金

M2 = M1 + S + T + MMF

其中：
S：储蓄存款
T：小额定期存款（<10万美元）
MMF：货币市场共同基金

特点：
- 流动性较强
- 更广泛的货币定义
- 与经济活动关系更密切
```

**M3（更广义货币）**：
```
定义：M2 + 大额定期存款 + 机构货币市场基金

特点：
- 流动性相对较弱
- 主要用于投资
- 一些国家不再公布M3
```

**美国货币供给数据（2023年）**：

| 指标 | 数量（十亿美元） | 占GDP比重 |
|------|-----------------|-----------|
| M1   | 18,500          | 8%        |
| M2   | 20,800          | 9%        |
| GDP  | 23,300          | 100%      |

**货币供给的增长率**：

| 年份 | M1增长率 | M2增长率 | GDP增长率 | 通胀率 |
|------|----------|----------|-----------|--------|
| 2019 | 5%       | 5%       | 2.3%      | 1.8%   |
| 2020 | 40%      | 25%      | -3.5%     | 1.2%   |
| 2021 | 15%      | 12%      | 5.7%      | 4.7%   |
| 2022 | -10%     | 2%       | 2.1%      | 8.0%   |
| 2023 | 5%       | 3%       | 2.5%      | 3.2%   |

**货币流通速度**：
```
定义：单位货币在一定时期内用于交易的平均次数

公式：
V = (P×Y)/M

其中：
V：货币流通速度
P：价格水平
Y：实际产出
M：货币供给

交易方程式：
M×V = P×Y

含义：
- 如果V稳定
- M的增加导致P×Y同比例增加
```

**货币流通速度的变化**：

| 时期 | V（基于M1） | V（基于M2） | 特征 |
|------|-------------|-------------|------|
| 1960-1980 | 3-6 | 1.5-2.0 | 稳定上升 |
| 1980-2000 | 6-10 | 1.8-2.0 | 波动较大 |
| 2000-2020 | 5-7 | 1.4-1.7 | 相对稳定 |
| 2020-2023 | 3-4 | 1.0-1.2 | 显著下降 |

## 16.2 银行体系与货币创造

### 16.2.1 银行的资产负债表

**典型商业银行的资产负债表**：

**资产（Assets）**：
```
1. 准备金（Reserves）：
   - 现金
   - 在央行的存款
   - 用以满足提款需求

2. 贷款（Loans）：
   - 商业贷款
   - 消费贷款
   - 抵押贷款
   - 主要收入来源

3. 证券（Securities）：
   - 政府债券
   - 公司债券
   - 流动性和收益平衡

4. 其他资产：
   - 房地产
   - 设备
   - 其他投资
```

**负债（Liabilities）**：
```
1. 存款（Deposits）：
   - 活期存款
   - 储蓄存款
   - 定期存款
   - 主要资金来源

2. 借款（Borrowings）：
   - 同业拆借
   - 央行借款
   - 发行债券

3. 其他负债：
   - 应付款项
   - 其他义务
```

**净值（Equity）**：
```
净值 = 资产 - 负债

作用：
- 缓冲损失
- 满足资本要求
- 吸收风险
```

**资产负债表示例**（单位：百万美元）：
```
资产：
  准备金：100
  贷款：700
  证券：150
  其他：50
  总资产：1000

负债：
  存款：850
  借款：100
  其他负债：20
  总负债：970

净值：30

验证：1000 = 970 + 30 ✓
```

### 16.2.2 准备金制度

**准备金的类型**：

**1. 法定准备金（Required Reserves）**：
```
定义：央行要求银行持有的最低准备金

公式：
RR = r × D

其中：
RR：法定准备金
r：法定准备金率
D：存款总额

例子：
准备金率r = 10%
存款D = 1000万
法定准备金RR = 1000万 × 10% = 100万
```

**2. 超额准备金（Excess Reserves）**：
```
定义：银行自愿持有的超出法定要求的准备金

公式：
ER = R - RR

其中：
ER：超额准备金
R：实际准备金
RR：法定准备金

原因：
- 应对流动性需求
- 安全考虑
- 贷款机会有限
```

**准备金率的作用**：
```
准备金率r决定：
1. 银行的放贷能力
2. 货币乘数的大小
3. 银行的流动性风险

r↑ → 贷款能力↓ → 货币乘数↓ → 货币供给↓

国际比较：
美国：活期存款0-3%（部分取消）
中国：大型银行12.5%，中小银行10.5%
欧元区：1%
日本：0.05%（实际零准备金）
```

### 16.2.3 货币创造过程

**简单货币乘数模型**：

**设定**：
```
假设：
1. 公众只持有存款，不持有现金
2. 银行没有超额准备金
3. 法定准备金率为r
```

**货币创造过程**：

**步骤1**：央行注入基础货币
```
央行购买政府债券100元
→ 向银行账户支付100元
→ 银行准备金增加100元
→ 存款增加100元

资产负债表变化：
银行：
  准备金：+100
  存款：+100
```

**步骤2**：银行放贷
```
法定准备金 = 100 × r = 100 × 0.1 = 10元
超额准备金 = 100 - 10 = 90元

银行可以贷出90元

资产负债表变化：
银行：
  准备金：+100
  贷款：+90
  存款：+100
```

**步骤3**：贷款转为存款**
```
借款人将90元贷款存入银行
→ 银行准备金增加90元
→ 存款增加90元

资产负债表变化：
银行：
  准备金：+190
  贷款：+90
  存款：+190
```

**步骤4**：再次放贷**
```
法定准备金 = 190 × 0.1 = 19元
超额准备金 = 190 - 19 = 171元

银行可以贷出171元
```

**货币创造的数学推导**：
```
存款总额：
D = 100 + 90 + 81 + 72.9 + ...
  = 100 × (1 + 0.9 + 0.9² + 0.9³ + ...)
  = 100 × 1/(1-0.9)
  = 100 × 10
  = 1000元

货币乘数：
m = D/R
  = 1000/100
  = 10
  = 1/r
  = 1/0.1
  = 10

结论：基础货币扩张10倍
```

**一般货币乘数公式**：
```
m = 1/r

其中：
m：货币乘数
r：法定准备金率

货币供给：
M = m × B

其中：
M：货币供给
B：基础货币
```

### 16.2.4 考虑现金和超额准备金的货币乘数

**现实中的货币乘数**：

**考虑现金漏损**：
```
现金持有率（c）：
c = C/D

其中：
C：公众持有的现金
D：存款

货币供给：
M = C + D
  = cD + D
  = (1+c)D

基础货币：
B = C + R
  = cD + rD
  = (c+r)D

货币乘数：
m = M/B
  = (1+c)/(c+r)
```

**考虑超额准备金**：
```
超额准备金率（e）：
e = ER/D

其中：
ER：超额准备金

准备金总额：
R = RR + ER
  = rD + eD
  = (r+e)D

基础货币：
B = C + R
  = cD + (r+e)D
  = (c+r+e)D

货币乘数：
m = (1+c)/(c+r+e)
```

**数值例子**：
```
假设：
c = 0.2（现金持有率20%）
r = 0.1（法定准备金率10%）
e = 0.05（超额准备金率5%）

货币乘数：
m = (1+0.2)/(0.2+0.1+0.05)
  = 1.2/0.35
  ≈ 3.43

如果基础货币B = 1000亿：
M = 3.43 × 1000 = 3430亿
```

**货币乘数的影响因素**：

| 因素 | 变化 | 对乘数的影响 | 原因 |
|------|------|--------------|------|
| 法定准备金率r | 增加 | 下降 | 银行放贷能力减弱 |
| 现金持有率c | 增加 | 下降 | 现金漏损增加 |
| 超额准备金率e | 增加 | 下降 | 银行放贷意愿减弱 |

**美国货币乘数变化**：

| 年份 | 基础货币B | M1 | 乘数m = M1/B | 特征 |
|------|-----------|----|-------------|------|
| 2007 | 850 | 1360 | 1.60 | 正常水平 |
| 2008 | 1650 | 1600 | 0.97 | 金融危机 |
| 2009 | 1700 | 1700 | 1.00 | 超额准备金增加 |
| 2019 | 3400 | 3900 | 1.15 | 逐步恢复 |
| 2020 | 5000 | 5500 | 1.10 | 疫情期间 |

## 16.3 货币需求

### 16.3.1 货币需求理论

**古典货币数量论**：

**费雪的交易方程式**：
```
M × V = P × Y

其中：
M：货币供给
V：货币流通速度
P：价格水平
Y：实际产出

古典假设：
- V恒定
- Y由供给决定
- M决定P

结论：
货币数量决定价格水平
```

**剑桥方程式**：
```
M = k × P × Y

其中：
k = 1/V：货币持有系数

含义：
公众持有k比例的名义收入
```

**凯恩斯的流动性偏好理论**：

**货币需求的三个动机**：

**1. 交易动机（Transactions Motive）**：
```
定义：为日常交易而持有货币

决定因素：
- 收入水平Y
- 交易便利性
- 支付习惯

货币需求：
L_T = L_T(Y)

L_Y > 0（收入增加，需求增加）
```

**2. 预防动机（Precautionary Motive）**：
```
定义：应对未预期支出而持有货币

决定因素：
- 不确定性程度
- 收入水平
- 保险可得性

货币需求：
L_P = L_P(Y, σ)

L_Y > 0, L_σ > 0
```

**3. 投机动机（Speculative Motive）**：
```
定义：为投机获利而持有货币

决定因素：
- 利率r
- 预期利率变化

货币需求：
L_S = L_S(r)

L_r < 0（利率上升，需求下降）

机制：
r↑ → 债券价格↓ → 预期债券价格上升
→ 持有货币等待债券价格上升
→ 货币需求↑

或者：
r↑ → 持有货币的机会成本↑
→ 减少货币持有
→ 货币需求↓

凯恩斯认为后者占主导
```

**总货币需求**：
```
L = L(Y, r)

其中：
L_Y > 0（收入效应）
L_r < 0（利率效应）

货币市场均衡：
M/P = L(Y, r)

其中：
M/P：实际货币供给
L(Y, r)：实际货币需求
```

### 16.3.2 鲍莫尔-托宾模型

**交易性货币需求的最优存货模型**：

**设定**：
```
假设：
1. 消费者定期收入Y
2. 持有货币获得便利
3. 将货币转为债券获得利息
4. 转换有成本F
```

**决策问题**：
```
选择提款次数n，最小化总成本

成本函数：
C = nF + (Y/2n)r

其中：
nF：转换成本
(Y/2n)r：利息损失（平均货币持有量Y/2n）

一阶条件：
dC/dn = F - (Y/2n²)r = 0

解出n：
n* = √(Yr/2F)

最优货币持有量：
M* = Y/(2n*) = √(FY/2r)
```

**关键含义**：

**1. 规模经济**：
```
货币需求的收入弹性：
dM*/dY × (Y/M*) = 0.5

收入增加1%，货币需求增加0.5%

原因：
- 大额交易可以摊薄转换成本
- 持有货币的平均余额相对较小
```

**2. 利率敏感度**：
```
货币需求的利率弹性：
dM*/dr × (r/M*) = -0.5

利率上升1%，货币需求下降0.5%

原因：
- 利率上升，持有货币成本增加
- 减少货币持有，增加债券持有
```

**3. 技术进步**：
```
如果转换成本F下降：
ATM、网上银行降低交易成本
→ 货币需求减少

电子货币发展：
- 支付便捷化
- 转换成本降低
- 货币需求减少
```

### 16.3.3 弗里德曼的现代货币数量论

**货币需求函数**：
```
M^d/P = f(Y_p, r_b - r_m, r_e - r_m, π^e)

其中：
M^d/P：实际货币需求
Y_p：持久收入
r_b：债券利率
r_m：货币利率
r_e：股票预期收益率
π^e：预期通胀率
```

**关键特征**：

**1. 持久收入**：
```
货币需求取决于持久收入，而非当前收入

持久收入：
Y_p = θY + (1-θ)Y_{p,prev}

平滑收入波动
货币需求相对稳定
```

**2. 多种资产替代**：
```
货币与多种资产竞争：
- 债券：r_b - r_m
- 股票：r_e - r_m
- 实物资产：π^e

相对收益率决定货币需求
```

**3. 货币流通速度**：
```
货币流通速度：
V = Y / (M/P)

如果货币需求稳定：
V也相对稳定

实证：
- 长期V相对稳定
- 短期V有波动
```

### 16.3.4 货币需求的实证证据

**货币需求的收入弹性**：
```
研究发现：
- 长期收入弹性接近1
- 短期收入弹性小于1

含义：
- 长期：货币需求与收入同比例变化
- 短期：货币需求调整较慢
```

**货币需求的利率弹性**：
```
研究发现：
- 利率弹性为负
- 弹性大小估计不一：-0.1到-0.5

含义：
- 利率上升，货币需求下降
- 但敏感度有限
```

**货币需求的稳定性**：
```
1980年代以前：
- 货币需求相对稳定
- 货币政策有效

1980年代以后：
- 金融创新
- 货币需求不稳定
- 货币政策目标从货币数量转向利率
```

**货币需求不稳定的来源**：

**1. 金融创新**：
```
新产品：
- 货币市场账户
- NOW账户
- 信用卡

影响：
- 改变货币定义
- 影响货币需求
- 增加预测难度
```

**2. 国际化**：
```
货币跨境流动：
- 外汇交易
- 国际投资
- 跨国经营

影响：
- 国内货币需求变化
- 货币政策复杂化
```

**3. 数字货币**：
```
新型货币：
- 比特币
- 稳定币
- 央行数字货币

影响：
- 改变货币需求
- 影响货币政策传导
- 增加不确定性
```

## 16.4 货币政策工具

### 16.4.1 传统货币政策工具

**1. 公开市场操作（Open Market Operations）**：

**定义**：
央行买卖政府债券以影响基础货币。

**操作机制**：

**买进债券**：
```
央行购买100亿债券
→ 向银行体系支付100亿
→ 银行准备金增加100亿
→ 基础货币增加100亿
→ 货币供给增加

资产负债表变化：
央行：
  资产：债券+100
  负债：准备金+100

银行：
  资产：准备金+100
  负债：存款+100
```

**卖出债券**：
```
央行卖出100亿债券
→ 从银行体系收取100亿
→ 银行准备金减少100亿
→ 基础货币减少100亿
→ 货币供给减少
```

**优点**：
- 灵活主动
- 精确控制
- 可连续操作
- 市场影响小

**2. 贴现率（Discount Rate）**：

**定义**：
央行向商业银行提供贷款的利率。

**作用机制**：
```
贴现率↑
→ 银行向央行借款成本↑
→ 银行准备金减少
→ 货币供给减少

反之：
贴现率↓
→ 货币供给增加
```

**特点**：
- 信号作用强
- 影响银行行为
- 但使用较少

**3. 准备金率（Reserve Requirements）**：

**定义**：
央行要求银行持有的准备金比例。

**作用机制**：
```
准备金率↑
→ 银行放贷能力↓
→ 货币乘数↓
→ 货币供给↓

反之：
准备金率↓
→ 货币供给增加
```

**特点**：
- 效果强烈
- 不频繁使用
- 部分国家已取消

### 16.4.2 非常规货币政策

**量化宽松（Quantitative Easing, QE）**：

**定义**：
央行大规模购买长期资产，影响长期利率。

**操作机制**：
```
央行购买长期债券
→ 长期债券价格↑
→ 长期利率↓
→ 刺激投资和消费
→ 提高通胀预期
```

**例子：美联储QE**：
```
QE1（2008-2010）：
- 购买1.7万亿美元MBS和国债
- 稳定金融市场

QE2（2010-2011）：
- 购买6000亿美元国债
- 刺激经济

QE3（2012-2014）：
- 开放式购买MBS
- 持续宽松
```

**前瞻性指引（Forward Guidance）**：

**定义**：
央行通过沟通影响市场对未来政策的预期。

**类型**：
```
1. 时间指引：
   "将维持低利率直到特定时间"

2. 状态指引：
   "将维持低利率直到失业率降至6%"

3. 定性指引：
   "将维持宽松政策直到经济强劲复苏"
```

**效果**：
```
→ 影响市场预期
→ 降低长期利率
→ 稳定预期
→ 增强政策有效性
```

**负利率政策（Negative Interest Rates）**：
```
定义：
央行将政策利率降至零以下

实施国家：
- 日本：-0.1%
- 欧元区：-0.5%
- 瑞士：-0.75%
- 丹麦：-0.6%

机制：
→ 对银行超额准备金收费
→ 鼓励银行放贷
→ 刺激经济

问题：
→ 挤压银行利润
→ 可能影响金融稳定
```

### 16.4.3 货币政策传导机制

**利率渠道**：
```
货币政策 → 利率 → 投资/消费 → 总需求 → 产出和通胀

具体：
M↑ → r↓ → I↑, C↑ → AD↑ → Y↑, P↑
```

**信贷渠道**：
```
货币政策 → 银行信贷 → 投资 → 总需求

具体：
M↑ → 银行准备金↑ → 银行放贷能力↑ → 信贷↑ → I↑ → AD↑
```

**资产价格渠道**：
```
货币政策 → 资产价格 → 财富 → 消费

具体：
M↑ → r↓ → 股价↑ → 财富↑ → C↑ → AD↑
```

**汇率渠道**：
```
货币政策 → 汇率 → 净出口 → 总需求

具体：
M↑ → r↓ → 汇率↓（本币贬值） → NX↑ → AD↑
```
