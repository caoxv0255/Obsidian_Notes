# 第11章 总供给与通货膨胀和失业之间的短期权衡

## 本章学习目标

通过本章学习，你应该能够：
1. 理解短期总供给曲线的三种理论解释
2. 掌握菲利普斯曲线的概念和推导
3. 分析通货膨胀和失业之间的短期权衡
4. 理解适应性预期和理性预期的含义
5. 评估通胀-失业权衡的政策含义

## 11.1 短期总供给曲线的三种理论

### 11.1.1 黏性工资理论

**定义**：黏性工资理论认为，名义工资在短期内调整缓慢，不能迅速对经济状况变化做出完全反应。

**基本逻辑**：
```
实际工资 = 名义工资 / 价格水平

W/P ≠ (W/P)ᵉ

如果P > Pᵉ → W/P < (W/P)ᵉ → 企业有利可图 → 增加产出
如果P < Pᵉ → W/P > (W/P)ᵉ → 企业利润减少 → 减少产出
```

**工资黏性的原因**：

1. **长期劳动合同**：
   - 工会谈判
   - 1-3年期合同
   - 工资调整滞后

2. **效率工资**：
   - 企业支付高于市场均衡的工资
   - 提高员工士气和生产力
   - 减少员工流动

3. **菜单成本**：
   - 调整工资有行政成本
   - 小幅变化不值得调整
   - 批量调整更有利

4. **协调问题**：
   - 企业担心单方面调整工资
   - 等待其他企业行动
   - 导致集体滞后

**数学表述**：
```
Y = Ȳ + α(P - Pᵉ)

其中：
Y：实际产出
Ȳ：自然产出（充分就业产出）
α：参数，表示产出对价格未预期变化的敏感度
P：实际价格水平
Pᵉ：预期价格水平
```

**图形分析**：

```
价格水平P
  ↑
  |      SRAS
  |     /
  |    /
  |   /
  |  /
  | /
  |/___________→ 产出Y
  Ȳ

当P = Pᵉ时，Y = Ȳ（自然产出）
当P > Pᵉ时，Y > Ȳ（产出高于自然水平）
当P < Pᵉ时，Y < Ȳ（产出低于自然水平）
```

**数值例子**：
假设Ȳ = 1000，α = 100，Pᵉ = 100

```
情况1：P = 100（预期正确）
Y = 1000 + 100×(100-100) = 1000

情况2：P = 105（价格高于预期）
Y = 1000 + 100×(105-100) = 1500（高出50%）

情况3：P = 95（价格低于预期）
Y = 1000 + 100×(95-100) = 500（低于50%）
```

### 11.1.2 黏性价格理论

**定义**：黏性价格理论认为，由于菜单成本和价格调整成本，企业在短期内不会立即调整价格。

**基本逻辑**：
```
价格调整决策：
如果调整收益 > 调整成本 → 调整价格
如果调整收益 < 调整成本 → 保持价格不变

菜单成本（Menu Costs）：
- 印刷新价签
- 通知客户
- 更新系统
- 管理时间
```

**价格黏性的来源**：

1. **菜单成本**：
   - 小幅涨价成本可能超过收益
   - 累积小幅变化后一次性调整
   - 批量调整降低平均成本

2. **顾客关系**：
   - 频繁涨价损害顾客关系
   - 隐性契约和公平性考虑
   - 品牌声誉

3. **协调问题**：
   - 单个企业担心失去市场份额
   - 等待行业领导者调整
   - 集体行动困境

4. **指数化合同**：
   - 长期供货合同
   - 定期调整条款
   - 调整频率有限

**数学表述**：
```
Y = Ȳ + β(P - Pᵉ)

其中：
β：参数，表示黏性价格程度
β越大，价格越黏性，总供给越平坦
```

**机制分析**：

```
当总需求增加时：
1. 部分企业立即提高价格
2. 其他企业保持价格不变
3. 价格不变企业的相对价格下降
4. 相对价格下降企业的需求增加
5. 这些企业增加产出
6. 总产出增加
```

**案例研究：麦当劳的价格调整**

**背景**：麦当劳需要决定是否调整汉堡价格

**成本分析**：
- 菜单印刷成本：每次$5,000
- 系统更新成本：$2,000
- 员工培训成本：$3,000
- 总菜单成本：$10,000

**收益分析**：
- 单个汉堡利润：$1
- 日销量：1,000个
- 每日利润：$1,000

**决策**：
```
如果涨价10%：
- 每个汉堡增加利润$0.5
- 日利润增加：$500
- 收回菜单成本：20天

如果涨价2%：
- 每个汉堡增加利润$0.1
- 日利润增加：$100
- 收回菜单成本：100天

结论：小幅度涨价不划算，等待累积较大变化后一次性调整
```

### 11.1.3 错觉理论

**定义**：错觉理论认为，当价格水平上升时，供给者可能误以为他们产品的相对价格上升，从而增加供给。

**基本逻辑**：
```
供给者的信息不完美：
- 观察到自己产品的价格变化
- 不能立即区分绝对价格和相对价格变化
- 如果认为相对价格上升 → 增加供给
- 如果认为绝对价格上升 → 不改变供给

决策规则：
如果P_i > Pᵉ → 认为相对价格上升 → 增加供给
如果P_i < Pᵉ → 认为相对价格下降 → 减少供给
```

**信息问题**：

1. **信息不对称**：
   - 供给者掌握本地信息
   - 不了解总体价格水平
   - 需要推断整体情况

2. **信号提取**：
   - 从价格变化中提取信息
   - 区分特异冲击和一般冲击
   - 最优信息处理

3. **理性预期**：
   - 供给者根据所有可得信息形成预期
   - 预期可能正确也可能错误
   - 系统性错误会被纠正

**数学表述**：
```
Y = Ȳ + γ(P - Pᵉ)

其中：
γ：参数，表示信息不完美程度
γ越大，信息越不完美，供给者越容易产生错觉
```

**图形分析**：

```
价格水平P
  ↑
  |      SRAS
  |     /
  |    /
  |   /
  |  /
  | /
  |/___________→ 产出Y
  Ȳ

当P = Pᵉ时，供给者认为相对价格不变，Y = Ȳ
当P > Pᵉ时，供给者认为相对价格上升，Y > Ȳ
```

**例子：农民的错觉**

**场景**：小麦价格上涨10%

**农民的思考**：
```
可能性1：所有农产品都涨价10%（一般价格水平上升）
→ 相对价格不变
→ 不改变生产

可能性2：只有小麦涨价10%（特异冲击）
→ 相对价格上升
→ 增加生产

如果农民认为可能性2更大 → 增加生产 → 总供给增加
```

**实证证据**：
- Lucas (1973) 的跨国研究
- 通货膨胀波动大的国家：总供给曲线更陡峭
- 供给者学会区分绝对和相对价格变化
- 错觉效应随时间减弱

### 11.1.4 三种理论的比较

**共同点**：
1. 短期总供给曲线向右上方倾斜
2. 产出偏离自然水平取决于预期误差
3. 长期总供给曲线垂直于自然产出水平

**不同点**：

| 特征 | 黏性工资 | 黏性价格 | 错觉理论 |
|------|----------|----------|----------|
| 主要原因 | 工资调整缓慢 | 价格调整缓慢 | 信息不完美 |
| 关键主体 | 工人和企业 | 企业 | 供给者 |
| 市场类型 | 劳动市场 | 产品市场 | 所有市场 |
| 政策含义 | 财政政策有效 | 货币政策有效 | 预期管理重要 |
| 实证支持 | 中等 | 较强 | 较弱 |

**适用性**：
- 黏性工资：解释劳动市场现象
- 黏性价格：解释产品市场现象
- 错觉理论：解释信息不完美的市场

**综合理解**：
三种理论可能同时起作用，共同解释短期总供给曲线的形状。现实中的黏性程度是多种因素共同作用的结果。

## 11.2 从总供给到菲利普斯曲线

### 11.2.1 菲利普斯曲线的发现

**历史背景**：
- 1958年：A.W. Phillips发表研究
- 分析英国1861-1957年数据
- 发现失业率和工资增长率之间的负相关关系

**原始发现**：
```
失业率  工资增长率
3%     4.5%
4%     3.5%
5%     2.5%
6%     1.5%
7%     0.5%

关系：失业率每增加1%，工资增长率下降1%
```

**原始菲利普斯曲线**：
```
ΔW/W = -β(u - uⁿ)

其中：
ΔW/W：工资增长率
u：实际失业率
uⁿ：自然失业率
β：参数，表示曲线的斜率
```

**后续发展**：
- Samuelson和Solow (1960) 将工资增长率转换为通货膨胀率
- 发现失业率和通货膨胀率之间的负相关关系
- 菲利普斯曲线成为政策工具

**图形分析**：

```
通货膨胀率π
  ↑
  |    PC
  |   /
  |  /
  | /
  |/___________→ 失业率u
    uⁿ

当u = uⁿ时，π = 0
当u < uⁿ时，π > 0（通胀）
当u > uⁿ时，π < 0（通缩）
```

### 11.2.2 从总供给曲线推导菲利普斯曲线

**总供给曲线**：
```
Y = Ȳ + α(P - Pᵉ)
```

**奥肯定律**：
```
u - uⁿ = -β(Y - Ȳ)

其中：
u：实际失业率
uⁿ：自然失业率
β：奥肯系数（约0.5）
```

**价格水平与通货膨胀**：
```
P = P_{-1} (1 + π)
Pᵉ = P_{-1} (1 + πᵉ)

其中：
π：实际通货膨胀率
πᵉ：预期通货膨胀率
P_{-1}：上一期价格水平
```

**推导过程**：

**步骤1**：用奥肯定律替换总供给中的(Y - Ȳ)
```
Y - Ȳ = -(1/β)(u - uⁿ)
```

**步骤2**：代入总供给曲线
```
Y = Ȳ + α(P - Pᵉ)
Y - Ȳ = α(P - Pᵉ)
-(1/β)(u - uⁿ) = α(P - Pᵉ)
```

**步骤3**：用通货膨胀率替换价格水平
```
P - Pᵉ = P_{-1}(1 + π) - P_{-1}(1 + πᵉ)
P - Pᵉ = P_{-1}(π - πᵉ)
```

**步骤4**：代入步骤2的结果
```
-(1/β)(u - uⁿ) = αP_{-1}(π - πᵉ)
```

**步骤5**：解出π
```
π - πᵉ = -(1/αβP_{-1})(u - uⁿ)
π = πᵉ - ε(u - uⁿ)

其中：ε = 1/(αβP_{-1})
```

**预期扩大的菲利普斯曲线**：
```
π = πᵉ - β(u - uⁿ) + v

其中：
π：通货膨胀率
πᵉ：预期通货膨胀率
u：实际失业率
uⁿ：自然失业率
β：参数，表示权衡程度
v：供给冲击
```

### 11.2.3 菲利普斯曲线的经济含义

**核心含义**：
1. **短期权衡**：低失业率与高通胀率之间的权衡
2. **预期的重要性**：只有未预期的失业偏离自然水平才影响通胀
3. **供给冲击**：供给冲击移动菲利普斯曲线

**图形分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "π" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**数值例子**：
假设πᵉ = 2%，β = 0.5，uⁿ = 5%

```
情况1：u = 3%（低于自然失业率）
π = 2% - 0.5×(3% - 5%) = 2% + 1% = 3%

情况2：u = 5%（等于自然失业率）
π = 2% - 0.5×(5% - 5%) = 2%

情况3：u = 7%（高于自然失业率）
π = 2% - 0.5×(7% - 5%) = 2% - 1% = 1%
```

**政策含义**：
- 如果政策制定者试图将失业率压低到自然失业率以下
- 这将导致通货膨胀率上升
- 长期来看，预期通胀会调整，权衡消失

### 11.2.4 案例研究：美国1960年代的菲利普斯曲线

**背景**：
- 1960年代美国经济繁荣
- 肯尼迪和约翰逊政府采取扩张性政策
- 失业率下降，通胀率上升

**数据**：

| 1961 | 6.7%    | 1.0%    |
| 1962 | 5.5%    | 1.0%    |
| 1963 | 5.7%    | 1.3%    |
| 1964 | 5.2%    | 1.3%    |
| 1965 | 4.5%    | 1.6%    |
| 1966 | 3.8%    | 2.9%    |
| 1967 | 3.8%    | 3.1%    |
| 1968 | 3.6%    | 4.2%    |
| 1969 | 3.5%    | 5.5%    |

**图形分析**：

```mermaid

    y-axis "π" 0 --> 100

```

**政策分析**：
- 政策制定者认为菲利普斯曲线是稳定的
- 尝试将失业率压低到4%以下
- 成功降低失业率，但通胀率上升

**教训**：
- 短期菲利普斯曲线似乎存在
- 但长期权衡可能不存在
- 预期调整会改变权衡关系

## 11.3 通货膨胀、失业与菲利普斯曲线

### 11.1.1 适应性预期与短期权衡

**适应性预期定义**：
预期基于过去的经验，随时间逐步调整。

**形成机制**：
```
πᵉ_t = λπ_{t-1} + (1-λ)πᵉ_{t-1}

其中：
πᵉ_t：t期的预期通胀率
π_{t-1}：t-1期的实际通胀率
πᵉ_{t-1}：t-1期的预期通胀率
λ：调整速度（0 < λ ≤ 1）
```

**简化形式**：
```
πᵉ_t = π_{t-1}

预期通胀等于上一期的实际通胀
```

**预期扩大的菲利普斯曲线**：
```
π = π_{t-1} - β(u - uⁿ) + v
```

**动态过程**：

**第一年**：
```
初始状态：π = πᵉ = 2%，u = uⁿ = 5%

政策：扩张性政策将失业率降至3%

结果：
π = 2% - 0.5×(3% - 5%) = 3%
```

**第二年**：
```
预期调整：πᵉ = π_{-1} = 3%

维持政策：u = 3%

结果：
π = 3% - 0.5×(3% - 5%) = 4%
```

**第三年**：
```
预期继续调整：πᵉ = 4%

维持政策：u = 3%

结果：
π = 4% - 0.5×(3% - 5%) = 5%
```

**图形分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "π" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**关键洞察**：
- 只有未预期的失业偏离影响通胀
- 预期调整后，菲利普斯曲线移动
- 长期没有通胀-失业权衡

### 11.3.2 长期菲利普斯曲线

**长期均衡**：
在长期，预期通胀等于实际通胀，失业率等于自然失业率。

**数学推导**：
```
菲利普斯曲线：π = πᵉ - β(u - uⁿ) + v

长期均衡条件：π = πᵉ，v = 0

代入：
π = π - β(u - uⁿ)
β(u - uⁿ) = 0
u = uⁿ
```

**长期菲利普斯曲线**：
```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "垂直于自然失业率uⁿ" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**政策含义**：
- 长期无法用通胀换取低失业
- 货币政策在长期中性
- 稳定通胀是长期目标

### 11.3.3 牺牲率

**定义**：牺牲率是指为降低1%的通货膨胀率所需要经历的GDP损失（通常用百分比表示）。

**计算公式**：
```
牺牲率 = 累计GDP损失百分比 / 通胀率下降百分比

累计GDP损失 = Σ(Y* - Y_t)/Y*
```

**数值例子**：
假设初始通胀率10%，目标通胀率2%，降低8个百分点。

**情况1：快速降低（2年）**
```
第一年：GDP下降4%
第二年：GDP下降2%
累计损失：6%

牺牲率 = 6% / 8% = 0.75
```

**情况2：缓慢降低（4年）**
```
第一年：GDP下降2%
第二年：GDP下降1.5%
第三年：GDP下降1%
第四年：GDP下降0.5%
累计损失：5%

牺牲率 = 5% / 8% = 0.625
```

**图形分析**：

```
GDP增长率
  ↑
  |
  |    ● ● ● ● (缓慢)
  |  ●         (快速)
  |
  |__________________→ 时间
  t₁  t₂  t₃  t₄
```

**实际数据**：
美国1980年代初的反通胀：
- 通胀率从13.5%（1980）降至3.2%（1983）
- GDP累计下降约7.5%
- 通胀下降10.3个百分点
- 牺牲率 ≈ 0.73

### 11.3.4 案例研究：1970年代的滞胀

**背景**：
- 石油价格冲击

### 11.3.6 预期与政策信誉

**政策信誉的定义**：
公众相信政策制定者会执行宣布的政策。

**信誉的作用**：
```
如果公众相信央行会维持低通胀
→ 预期通胀保持在低位
→ 实际通胀更容易维持低位

菲利普斯曲线：π = πᵉ - β(u - uⁿ)

如果πᵉ锚定在低位：
→ 即使u < uⁿ，π也不会大幅上升
```

**建立信誉**：

1. **独立性**：
   - 央行独立于政府
   - 避免政治压力
   - 长期承诺

2. **透明度**：
   - 明确的政策目标
   - 清晰的沟通策略
   - 前瞻性指引

3. **一致性**：
   - 言行一致
   - 历史记录
   - 制度保障

**案例研究：通胀目标制**

**实施国家**：
- 新西兰（1990年率先实施）
- 加拿大（1991年）
- 英国（1992年）
- 欧元区（1999年）

**效果**：

| 国家 | 实施前平均通胀 | 实施后平均通胀 |
|------|----------------|----------------|
| 新西兰 | 11.0%          | 2.0%           |
| 加拿大 | 4.5%           | 2.0%           |
| 英国 | 5.8%           | 2.5%           |
| 欧元区 | 3.5%           | 2.0%           |

**机制**：
```
通胀目标制 → 明确目标 → 预期锚定 → 实际通胀接近目标

菲利普斯曲线：π = πᵉ - β(u - uⁿ)

如果πᵉ = π* = 2%
→ 长期π将收敛于2%
```

## 11.4 本章总结

### 核心要点

1. **短期总供给曲线**：
   - 黏性工资、黏性价格、错觉理论解释短期非垂直
   - 产出偏离自然水平取决于预期误差
   - 长期总供给曲线垂直于自然产出

2. **菲利普斯曲线**：
   - 失业率和通胀率之间的负相关关系
   - 预期扩大的菲利普斯曲线包含预期因素
   - 供给冲击移动菲利普斯曲线

3. **通货膨胀与失业的权衡**：
   - 短期存在权衡关系
   - 长期菲利普斯曲线垂直
   - 预期调整消除长期权衡

4. **政策含义**：
   - 货币政策长期中性
   - 预期管理至关重要
   - 政策信誉影响效果

5. **实证证据**：
   - 1960年代：权衡关系明显
   - 1970年代：滞胀，权衡失效
   - 1990年代：低通胀低失业并存
   - 通胀目标制成功锚定预期
