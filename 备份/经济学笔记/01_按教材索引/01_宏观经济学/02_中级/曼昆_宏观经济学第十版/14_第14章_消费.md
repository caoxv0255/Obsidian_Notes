## 14.1 凯恩斯的猜测

### 14.1.1 凯恩斯消费函数

**凯恩斯的观察**：
在《就业、利息和货币通论》(1936)中，凯恩斯提出了基于内省和观察的消费理论。

**基本公式**：
```
C = a + bY

其中：
C：消费
Y：当前可支配收入
a：自发消费（当Y=0时的消费）
b：边际消费倾向（MPC），0 < b < 1
```

**数值例子**：
```
假设a = 1000，b = 0.8

Y = 10,000：
C = 1000 + 0.8×10000 = 9000
APC = 9000/10000 = 0.9

Y = 20,000：
C = 1000 + 0.8×20000 = 17000
APC = 17000/20000 = 0.85

Y = 40,000：
C = 1000 + 0.8×40000 = 33000
APC = 33000/40000 = 0.825

结论：APC随Y增加而下降
```

**猜测3：消费主要由当前收入决定**
```
含义：
- 消费对当前收入变化敏感
- 当前收入是消费的主要决定因素
- 未来收入和财富影响较小

政策含义：
- 短期财政政策有效
- 乘数效应较大
```

### 14.1.2 凯恩斯消费函数的实证检验

**早期证据（1940-1950年代）**：
```
横截面数据（同一时间不同家庭）：

收入水平    消费    MPC    APC
低收入     8000    0.9    0.95
中等收入   15000   0.8    0.80
高收入     25000   0.7    0.65

支持猜测：
- MPC在0-1之间 ✓
- APC随收入增加而下降 ✓
```

**库兹涅茨悖论（Kuznets, 1946）**：
```
时间序列数据（美国1869-1939）：

时期           平均收入  平均消费  APC
1869-1898      4500     4000     0.89
1899-1928      6000     5300     0.88
1929-1938      6500     5800     0.89

发现：
- 长期APC相对稳定
- 与凯恩斯猜测2矛盾

解释：
- 短期和长期消费函数不同
- 存在消费调整过程
```

**图形解释**：

```
C
  ↑
  |    长期APC稳定
  |    _______
  |   /       \
  |  /         \  短期消费函数
  | /           \
  |/_____________\________→ Y
  t₁    t₂    t₃

长期：APC稳定
短期：APC随收入下降

消费函数随时间向上移动（a增加）
```

### 14.1.3 消费之谜的解释

**三种解释**：

**1. 相对收入假说（Duesenberry, 1949）**
```
基本思想：
- 消费取决于相对收入
- 人们模仿他人消费
- "棘轮效应"：消费易升难降

消费函数：
C = a + bY + cY_{max}

其中Y_{max}是历史最高收入

短期：Y变化，Y_{max}不变
→ APC下降

长期：Y和Y_{max}都变化
→ APC稳定
```

**2. 生命周期假说（Modigliani, 1950s）**
```
基本思想：
- 消费平滑整个生命周期
- 储蓄用于退休

消费函数：
C = αW + βY

其中：
W：财富
Y：当前收入
α, β：参数

短期：Y变化，W不变
→ APC下降

长期：Y和W都变化
→ APC稳定
```

**3. 持久收入假说（Friedman, 1957）**
```
基本思想：
- 消费取决于持久收入
- 暂时性收入变化影响小

收入分解：
Y = Y_p + Y_t

其中：
Y_p：持久收入
Y_t：暂时收入

消费函数：
C = kY_p

暂时收入变化：
MPC_t ≈ 0
MPC_p ≈ k

短期：Y_t变化
→ MPC小

长期：Y_p变化
→ MPC大
```

## 14.2 欧文·费雪的跨期选择

### 14.2.1 两期消费模型

**基本设定**：
```
消费者在两期（第一期和第二期）做出消费决策

预算约束：
C₁ + C₂/(1+r) = Y₁ + Y₂/(1+r)

其中：
C₁：第一期消费
C₂：第二期消费
Y₁：第一期收入
Y₂：第二期收入
r：利率
```

**预算约束的经济含义**：
```
现值形式：
消费的现值 = 收入的现值

C₁ + C₂/(1+r) = Y₁ + Y₂/(1+r)

未来价值形式：
C₁(1+r) + C₂ = Y₁(1+r) + Y₂

两种形式等价
```

```

### 14.2.3 消费者最优选择

**最优条件**：
```

s.t. C₁ + C₂/(1+r) = Y₁ + Y₂/(1+r)

一阶条件：
u'(C₁) = β(1+r)u'(C₂)

经济含义：
第一期消费的边际效用 = 第二期消费的边际效用的现值
```

```

```

→ MPC较低

C₁增加≈50，C₂增加≈50（假设β=1）

如果暂时收入增加100（仅Y₁）：
C₁增加≈25，C₂增加≈25
MPC ≈ 0.5（暂时收入）
MPC ≈ 0.5（持久收入）

但跨期平滑导致暂时收入的MPC更低
```

### 14.2.5 利率变化的影响

**利率上升的影响**：
```
利率r → r+Δr

预算约束变化：
1. 斜率更陡：-(1+r+Δr)更负
2. 可能移动（取决于是借款者还是储蓄者）

替代效应：
- C₁变贵，C₂变便宜
- 减少C₁，增加C₂
- 沿着无差异曲线移动

收入效应：
- 储蓄者：实际收入增加
  → 两期消费都可能增加
- 借款者：实际收入减少
  → 两期消费都可能减少

净效应：
取决于替代效应和收入效应的相对大小
```

## 14.3 持久收入假说

### 14.3.1 理论基础

**米尔顿·弗里德曼（1957）提出持久收入假说**

**核心思想**：
```
消费决策基于长期收入预期，而非当前收入

收入分解：
Y = Y_p + Y_t

其中：
Y_p：持久收入（可预期的长期收入）
Y_t：暂时收入（不可预期的短期波动）

消费函数：
C = kY_p

其中：
k：常数（取决于利率、偏好等）
0 < k < 1
```

**持久收入的定义**：
```
Y_p = θY + (1-θ)Y_{p,prev}

其中：
θ：调整速度（0 < θ < 1）
Y_{p,prev}：上期持久收入

动态过程：
Y_t = Y_p + ε_t
Y_p,t = θY_t + (1-θ)Y_{p,t-1}

如果θ = 1：
Y_p = Y（完全调整）

如果θ = 0：
Y_p不变（完全不变）
```

### 14.3.2 关键含义

**含义1：暂时收入的MPC低**
```
如果Y增加ΔY，且是暂时性的：
Y_p增加θΔY
C增加kθΔY

MPC_temp = dC/dY = kθ

如果k = 0.9，θ = 0.2：
MPC_temp = 0.9 × 0.2 = 0.18

暂时收入对消费影响小
```

**含义2：持久收入的MPC高**
```
如果Y增加ΔY，且是持久性的：
Y_p增加ΔY
C增加kΔY

MPC_perm = dC/dY = k

如果k = 0.9：
MPC_perm = 0.9

持久收入对消费影响大
```

**含义3：消费比收入稳定**
```
收入波动：
Y_t = Y_p + ε_t

消费：
C_t = kY_p,t
     = k[θY_t + (1-θ)Y_{p,t-1}]

如果θ < 1：
消费波动小于收入波动

原因：
暂时收入波动被平滑
```

### 14.3.3 实证检验

**弗里德曼的检验**：
```
方法：按收入分组

假设：
- 高收入组：暂时收入较高
- 低收入组：暂时收入较低

预测：
- 高收入组：APC较低
- 低收入组：APC较高

数据支持：
收入水平     平均收入  平均消费  APC
最低20%     5000     4500     0.90
20-40%      15000    12000    0.80
40-60%      25000    18500    0.74
60-80%      35000    24500    0.70
最高20%     60000    39000    0.65

支持持久收入假说 ✓
```

**现代检验**：
```
研究：Hall (1978)
方法：检验消费是否可预测

结果：
- 消费变化不可预测
- 消费遵循随机游走
- 支持理性预期下的持久收入假说
```

**案例：减税的效果**
```
场景：政府减税1000元

如果是暂时性减税：
- Y_p增加θ×1000
- C增加kθ×1000
- MPC_temp = kθ

如果是持久性减税：
- Y_p增加1000
- C增加k×1000
- MPC_perm = k

例子：
如果k = 0.9，θ = 0.2：
暂时减税：MPC = 0.18，消费增加180元
持久减税：MPC = 0.9，消费增加900元

政策含义：
- 暂时性财政政策效果有限
- 持久性政策效果显著
```

## 14.4 生命周期假说

### 14.4.1 理论基础

**弗兰科·莫迪利安尼（1950s）提出生命周期假说**

**核心思想**：
```
消费平滑整个生命周期
年轻时储蓄，年老时负储蓄
财富和收入都影响消费
```

**生命周期阶段**：
```
1. 年轻期（工作前期）：
   - 收入较低
   - 借债消费
   - 负储蓄

2. 中年期（工作期）：
   - 收入较高
   - 偿还债务
   - 储蓄养老

3. 老年期（退休期）：
   - 收入下降
   - 消费储蓄
   - 负储蓄
```

**消费函数**：
```
C = αW + βY

其中：
W：财富
Y：当前收入
α：财富的边际消费倾向
β：收入的边际消费倾向

α + β < 1（部分收入用于储蓄）
```

### 14.4.2 模型分析

**简化模型**：
```
假设：
- 生活T年
- 工作R年
- 退休T-R年
- 无遗产
- 恒定收入Y
- 恒定消费C

预算约束：
RC = (T-R)C + W₀ + RY
其中W₀是初始财富

解出消费：
C = (W₀ + RY)/T
  = (1/T)W₀ + (R/T)Y

与消费函数对比：
α = 1/T
β = R/T
```

**数值例子**：
```
假设：
- T = 60年（60岁退休，活到80岁）
- R = 40年（20岁开始工作）
- W₀ = 0
- Y = 50000/年

计算：
C = (0 + 40×50000)/60
  = 2000000/60
  = 33333/年

储蓄：
工作期：S = Y - C = 50000 - 33333 = 16667/年
总储蓄：40×16667 = 666667

退休期：消费 = 33333/年
退休长度：20年
总消费：20×33333 = 666667

平衡：储蓄 = 退休期消费 ✓
```

### 14.4.3 财富效应

**财富增加的影响**：
```
消费函数：C = αW + βY

如果W增加ΔW：
C增加αΔW

财富的边际消费倾向：
MPC_W = α = 1/T

如果T = 60：
MPC_W = 1/60 ≈ 0.017

例子：
如果财富增加100000：
消费增加0.017×100000 = 1700
```

**股市波动的影响**：
```
场景：股市上涨20%，家庭财富增加

假设：
- 家庭财富：500000
- 股市占比：60%
- 股市上涨：20%
- MPC_W = 0.04

计算：
股市财富：500000×60% = 300000
股市增值：300000×20% = 60000
消费增加：60000×0.04 = 2400

政策含义：
- 财富效应影响消费
- 股市波动通过消费影响经济
```

### 14.4.4 生命周期假说的扩展

**1. 遗产动机**：
```
如果人们留遗产：
- 储蓄率更高
- 消费更低
- 财富积累更多

修正预算约束：
RC + B = (T-R)C + W₀ + RY

其中B：遗产

消费降低，储蓄增加
```

**2. 不确定性**：
```
退休时间和寿命不确定：
- 预防性储蓄
- 消费更保守
- 储蓄率更高

预防性储蓄取决于：
- 风险厌恶程度
- 收入波动性
- 社会保障程度
```

**3. 借贷约束**：
```
年轻人可能无法自由借贷：
- 消费受当前收入限制
- 消费平滑不完美
- MPC更高

修正：
如果Y_t < C_t（借贷约束）：
C_t = Y_t
MPC = 1（更高）
```

## 14.5 欧拉方程与随机游走假说

### 14.5.1 欧拉方程的推导

**从跨期优化得到欧拉方程**：

**设定**：
```
消费者最大化终身效用：
max E₀[Σβᵗu(C_t)]

约束：
ΣC_t/(1+r)ᵗ = ΣY_t/(1+r)ᵗ + W₀

其中：
E₀：时期0的期望
β：主观贴现因子
r：利率
```

**一阶条件（欧拉方程）**：
```
u'(C_t) = β(1+r)E_t[u'(C_{t+1})]

经济含义：
- 当前消费的边际效用
- 等于未来消费边际效用的预期
- 的现值

如果效用函数为二次型：
u(C) = aC - bC²

则u'(C) = a - 2bC

欧拉方程变为：
a - 2bC_t = β(1+r)[a - 2bE_t(C_{t+1})]

简化：
E_t(C_{t+1}) = C_t + 常数
```

### 14.5.2 随机游走假说

**Hall (1978) 的发现**：
```
如果欧拉方程成立且：
1. 效用函数是二次型的
2. 利率恒定
3. β(1+r) = 1

则：
E_t(C_{t+1}) = C_t

消费遵循随机游走：
C_{t+1} = C_t + ε_{t+1}

其中ε_{t+1}是不可预测的冲击

含义：
- 消费变化不可预测
- 只有未预期的收入变化影响消费
- 预期的收入变化不影响消费
```

### 14.5.3 实证检验

**Hall的检验**：
```
回归方程：
C_{t+1} - C_t = α + β(Y_t - Y_{t-1}) + γZ_t + ε_{t+1}

其中：
- Y_t - Y_{t-1}：收入变化
- Z_t：其他预测变量

随机游走假说预测：
β = 0（收入变化不预测消费变化）
γ = 0（其他变量不预测消费变化）

结果：
β ≈ 0.1（接近0但不完全为0）
γ ≈ 0

结论：
- 消费变化基本不可预测
- 部分支持随机游走假说
```

**后续研究**：
```
研究发现：
1. 流动性约束：低收入家庭的消费对收入变化敏感
2. 预防性储蓄：风险厌恶者消费更保守
3. 习惯形成：消费调整缓慢

修正的随机游走：
C_{t+1} = C_t + θ(Y_{t+1} - Y_t) + ε_{t+1}

其中0 < θ < 1
```

### 14.5.4 政策含义

**对财政政策的含义**：
```
如果消费遵循随机游走：
- 预期的税收变化不影响消费
- 只有未预期的税收变化影响消费
- 财政政策效果有限

例子：
如果政府宣布明年减税：
- 理性消费者预期到
- 提前调整消费
- 政策效果提前体现
```

**对货币政策的含义**：
```
如果消费遵循随机游走：
- 利率变化影响跨期替代
- 但效果取决于利率敏感性
- 不确定性强

传导机制：
r↑ → 未来消费相对便宜 → C₁↓, C₂↑
```

## 14.6 本章总结

### 核心要点

1. **凯恩斯消费函数**：
   - 三个猜测：MPC在0-1之间、APC下降、消费由当前收入决定
   - 消费之谜：长期APC稳定，短期APC下降
   - 需要更复杂的理论解释

2. **跨期选择理论**：
   - 消费者平滑消费
   - 消费取决于收入和利率
   - 最优条件：u'(C₁) = β(1+r)u'(C₂)

3. **持久收入假说**：
   - 消费取决于持久收入
   - 暂时收入MPC低，持久收入MPC高
   - 消费比收入稳定

4. **生命周期假说**：
   - 消费平滑整个生命周期
   - 年轻储蓄，年老负储蓄
   - 财富和收入都影响消费

5. **欧拉方程与随机游走**：
   - 消费变化不可预测
   - 只有未预期冲击影响消费
   - 政策效果受预期影响

### 重要概念表格

| 概念 | 定义 | 关键公式 |
