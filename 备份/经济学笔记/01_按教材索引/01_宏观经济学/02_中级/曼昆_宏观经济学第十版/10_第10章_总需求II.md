# 第10章 总需求II：应用IS-LM模型

## 本章学习目标

通过本章学习，你应该能够：
1. 理解IS-LM模型如何解释短期经济波动
2. 分析总需求冲击对经济的影响
3. 探讨总供给冲击与"滞胀"现象
4. 评估财政政策和货币政策在稳定经济中的作用
5. 应用IS-LM模型分析实际经济问题

## 10.1 用IS-LM模型解释经济波动

### 10.1.1 IS-LM模型的基本框架

**定义**：IS-LM模型是凯恩斯主义宏观经济学的核心分析框架，用于解释短期中产品市场和货币市场的均衡。IS曲线代表产品市场均衡，LM曲线代表货币市场均衡。

**基本公式**：

产品市场均衡（IS曲线）：
```
Y = C(Y-T) + I(r) + G + NX
```

其中：
- Y：总收入/产出
- C：消费函数，取决于可支配收入(Y-T)
- I：投资函数，取决于利率r
- G：政府购买
- T：税收
- NX：净出口

货币市场均衡（LM曲线）：
```
M/P = L(r, Y)
```

其中：
- M：货币供给
- P：价格水平
- L：货币需求函数，取决于利率r和收入Y

    y-axis "利率r" 0 --> 100

**关键理解**：
- IS曲线向右下方倾斜：利率下降→投资增加→收入增加
- LM曲线向右上方倾斜：收入增加→货币需求增加→利率上升
- 两条曲线的交点确定了产品市场和货币市场同时均衡时的利率和收入

### 10.1.3 经济波动的来源

经济波动可以源于：
1. **产品市场冲击**：改变IS曲线的位置
2. **货币市场冲击**：改变LM曲线的位置
3. **总供给冲击**：改变价格水平，进而影响LM曲线

**例子**：消费信心下降

当消费者对未来经济前景变得悲观时，消费减少，IS曲线向左移动：

### 10.1.4 案例研究：2008年金融危机

**背景**：2008年全球金融危机起源于美国次贷危机，迅速蔓延至全球。

**冲击分析**：
1. **金融部门冲击**：银行危机导致信贷紧缩
   - 投资减少（I下降）
   - 消费信贷减少（C下降）
   - IS曲线大幅左移

2. **货币市场冲击**：央行采取宽松货币政策
   - 美联储将联邦基金利率降至接近零
   - LM曲线向右移动

3. **政策应对**：
   - 财政刺激：7870亿美元的经济刺激计划（G增加）
   - 量化宽松：大规模资产购买（M增加）

**IS-LM模型分析**：

```
初始状态 → 危机冲击 → 政策干预
IS1      → IS2       → IS3
LM1      → LM1       → LM2

均衡变化：
Y1       → Y2        → Y3
r1       → r2        → r3

结果：政策部分抵消了危机的影响，但产出仍低于潜在水平
```

**经验教训**：
- 金融危机同时冲击产品市场和货币市场
- 传统货币政策在零利率下限处可能失效
- 财政政策在极端情况下发挥重要作用

## 10.2 总需求冲击分析

### 10.2.1 什么是总需求冲击

**定义**：总需求冲击是指任何引起总需求曲线移动的外部事件或政策变化。总需求冲击可以是正向的（增加总需求）或负向的（减少总需求）。

**主要来源**：
1. **消费冲击**：消费者信心变化、税收政策变化
2. **投资冲击**：企业信心变化、税收优惠、技术进步
3. **政府购买冲击**：战争、基础设施建设
4. **净出口冲击**：国外收入变化、汇率变化

### 10.2.2 财政政策冲击

**扩张性财政政策**：增加政府购买(G)或减少税收(T)

**IS曲线移动**：
```
ΔG > 0 → IS曲线向右移动
ΔT < 0 → IS曲线向右移动
```

**图形分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**挤出效应**：
```
Y增加 → 货币需求增加 → r上升 → I下降

总效应：
ΔY = 乘数效应 - 挤出效应
```

**乘数公式**：
```
财政乘数 = ΔY/ΔG = 1/(1-MPC) · [1/(1 + MPC·k/h)]

其中：
MPC：边际消费倾向
k：货币需求对收入的敏感度
h：货币需求对利率的敏感度
```

**数值例子**：
假设MPC = 0.8, k = 0.5, h = 0.4

```
简单乘数 = 1/(1-0.8) = 5
考虑货币市场的乘数 = 5 · [1/(1 + 0.8×0.5/0.4)] = 5 · 0.5 = 2.5

如果ΔG = 100亿：
简单乘数预测：ΔY = 500亿
IS-LM模型预测：ΔY = 250亿
挤出效应减少了50%的乘数效应
```

### 10.2.3 货币政策冲击

**扩张性货币政策**：增加货币供给(M)

**LM曲线移动**：
```
ΔM > 0 → LM曲线向右移动
```

**图形分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**货币乘数**：
```
货币乘数 = ΔY/ΔM = [1/(k)] · [1/(1 + (1-MPC)·h/(k))]

其中：
k：货币需求对收入的敏感度
h：货币需求对利率的敏感度
MPC：边际消费倾向
```

**传导机制**：
```
ΔM > 0 
→ r下降 
→ I增加 
→ Y增加 
→ C增加（乘数效应）
→ Y进一步增加
```

**数值例子**：
假设k = 0.5, h = 0.4, MPC = 0.8

```
货币乘数 = (1/0.5) · [1/(1 + 0.2×0.4/0.5)] = 2 · 0.84 = 1.68

如果ΔM = 100亿：
ΔY = 168亿
```

### 10.2.4 政策组合分析

**政策组合1：双扩张政策**
```
ΔG > 0 且 ΔM > 0

IS右移，LM右移
结果：Y大幅增加，r可能上升、下降或不变
```

**政策组合2：双紧缩政策**
```
ΔG < 0 且 ΔM < 0

IS左移，LM左移
结果：Y大幅减少，r可能上升、下降或不变
```

**政策组合3：财政扩张+货币紧缩**
```
ΔG > 0 且 ΔM < 0

IS右移，LM左移
结果：r大幅上升，Y可能增加、减少或不变
```

**案例研究：美国的政策组合（1980年代）**

**背景**：里根政府采取扩张性财政政策，沃尔克领导的联邦储备银行采取紧缩性货币政策。

**政策内容**：
1. 财政政策：
   - 大规模减税（1981年经济复苏税收法案）
   - 增加国防支出
   - IS曲线向右移动

2. 货币政策：
   - 紧缩货币供给以抑制高通胀
   - 联邦基金利率从1980年的11.4%升至1981年的20%
   - LM曲线向左移动

**IS-LM分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**实际结果**：
- 1981-1982年：深度衰退（失业率升至10.8%）
- 1983-1989年：强劲复苏（平均GDP增长4%）
- 通胀率从1980年的13.5%降至1983年的3.2%

**政策启示**：
- 紧缩货币政策可以有效抑制通胀，但会带来短期衰退
- 扩张性财政政策可以刺激长期增长，但可能推高利率
- 政策组合的效果取决于时序和力度

### 10.2.5 案例研究：COVID-19疫情的政策应对

**冲击性质**：罕见的负向总需求冲击和总供给冲击同时发生

**负向总需求冲击**：
- 消费减少：社交距离、不确定性
- 投资减少：企业推迟投资计划
- 净出口减少：全球贸易萎缩

**政策应对**：

1. **财政政策**：
   - 美国 CARES 法案：2.2万亿美元
   - 直接支付、失业救济、企业贷款
   - IS曲线向右移动

2. **货币政策**：
   - 美联储将利率降至0-0.25%
   - 无限量化宽松
   - LM曲线向右移动

3. **非常规政策**：
   - 主街贷款计划
   - 公司债券购买
   - 货币市场共同基金流动性便利

**IS-LM分析**：

```
疫情冲击：IS大幅左移，LM左移（流动性需求增加）
政策应对：IS大幅右移，LM大幅右移

净效应：
Y：短期大幅下降，部分恢复
r：降至接近零
```

**政策效果评估**：
- 2020年Q2：美国GDP年化萎缩31.4%
- 2020年Q3：GDP年化增长33.1%
- 2020年全年：GDP萎缩3.5%
- 政策有效防止了更严重的萧条

## 10.3 总供给冲击与滞胀

### 10.3.1 什么是总供给冲击

**定义**：总供给冲击是指任何引起生产成本或生产能力变化的外部事件，导致总供给曲线移动。

**类型**：
1. **正向供给冲击**：降低生产成本，增加供给
   - 技术进步
   - 资源发现
   - 有利的气候变化

2. **负向供给冲击**：提高生产成本，减少供给
   - 石油价格暴涨
   - 自然灾害
   - 工资快速上涨

### 10.3.2 负向供给冲击的IS-LM分析

**机制**：
```
负向供给冲击 → 成本上升 → 价格上升 → 实际货币供给(M/P)下降 → LM曲线左移
```

**图形分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**滞胀**：
```
滞胀 = 停滞 + 通货膨胀
- 停滞：产出下降，失业上升
- 通货膨胀：价格水平上升

政策困境：
- 扩张性政策（右移IS/LM）可以缓解衰退，但会加剧通胀
- 紧缩性政策（左移IS/LM）可以抑制通胀，但会加剧衰退
```

### 10.3.3 案例研究：1970年代的石油危机

**背景**：
- 1973年：石油输出国组织(OPEC)石油禁运
- 1979年：伊朗革命导致石油供应中断
- 石油价格从1972年的3美元/桶升至1981年的35美元/桶

**经济影响**：

1. **生产成本上升**：
   - 能源成本占总成本15-20%
   - 交通运输、制造业受影响最大

2. **价格传导**：
   - 石油价格上升 → 生产成本上升 → 价格上升
   - 通胀率从1972年的3.3%升至1980年的13.5%

3. **产出影响**：
   - 1973-1975年：衰退（GDP下降3.2%）
   - 1980-1982年：衰退（GDP下降2.2%）

**IS-LM分析**：

```mermaid
xychart-beta
    title "曲线图"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 100
    line [80, 60, 45, 30, 15, 5]
```

**政策困境**：

1. **福特政府（1974-1977）**：
   - 政策："WIN"（Whip Inflation Now）
   - 效果：失败，通胀持续

2. **卡特政府（1977-1981）**：
   - 初期：扩张性政策（右移IS）
   - 后期：紧缩性政策（左移LM）
   - 效果：通胀仍未受控

3. **里根政府（1981-1989）**：
   - 沃尔克领导下的紧缩货币政策
   - 联邦基金利率升至20%
   - 效果：通胀下降，但引发深度衰退

**经验教训**：
- 负向供给冲击导致政策两难
- 控制通胀需要付出短期衰退的代价
- 供给冲击需要结构性应对，而非单纯的总量调控

### 10.3.4 正向供给冲击

**例子：1990年代的信息技术革命**

**背景**：
- 互联网商业化
- 计算机技术进步
- 生产力提升

**经济影响**：

1. **生产成本下降**：
   - 信息处理成本大幅降低
   - 供应链效率提升
   - 企业管理效率提高

2. **IS-LM分析**：

```mermaid
xychart-beta
    title "供给曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "利率r" 0 --> 25
    line [0, 5, 10, 15, 20, 25]
```

**实际数据**：
- 1995-2000年：平均GDP增长4.2%
- 通胀率从1995年的2.8%降至1998年的1.6%
- 失业率从1995年的5.6%降至2000年的4.0%

**政策启示**：
- 正向供给冲击带来"金发姑娘"经济（增长强劲、通胀温和）
- 供给冲击是长期经济增长的重要驱动力
- 政策应创造有利于供给冲击的环境（如鼓励创新）

## 10.4 政策分析：稳定经济的工具

### 10.4.1 财政政策工具

**主要工具**：

1. **政府购买(G)**：
   - 直接影响总需求
   - 乘数效应较大
   - 时滞较长（立法、实施）

2. **税收(T)**：
   - 影响可支配收入
   - 乘数效应较小（部分用于储蓄）
   - 时滞较长

3. **转移支付**：
   - 影响可支配收入
   - 自动稳定器功能
   - 实施相对迅速

**财政政策时滞**：

```
识别时滞：3-6个月
  ↓
立法时滞：6-18个月（美国）
  ↓
实施时滞：3-12个月
  ↓
效果时滞：6-12个月

总时滞：18-48个月
```

**优点**：
- 可以直接针对特定部门或人群
- 政治上更易接受（创造就业）
- 可以用于结构性改革

**缺点**：
- 时滞长，可能加剧经济周期
- 政治博弈导致低效
- 挤出效应降低乘数效应
- 可能导致财政赤字和债务积累

### 10.4.2 货币政策工具

**主要工具**：

1. **公开市场操作**：
   - 买卖政府债券
   - 影响基础货币
   - 最常用、最灵活

2. **贴现率**：
   - 央行对商业银行的贷款利率
   - 信号作用
   - 使用较少

3. **准备金率**：
   - 商业银行准备金要求
   - 影响货币乘数
   - 较少调整

**货币政策时滞**：

```
识别时滞：3-6个月
  ↓
决策时滞：1-3个月
  ↓
实施时滞：即时
  ↓
效果时滞：6-18个月

总时滞：10-27个月
```

**传导机制**：

```
货币政策 → 利率 → 投资/消费 → 总需求 → 产出和通胀
    ↓         ↓         ↓            ↓          ↓
  ΔM/P    Δr     ΔI, ΔC      ΔAD    ΔY, ΔP
```

**优点**：
- 时滞较短（相对财政政策）
- 决策迅速，不受立法程序约束
- 技术性强，受政治影响较小
- 可以连续调整

**缺点**：
- 效果不确定（受预期影响）
- 零利率下限问题
- 传导机制可能受阻
- 可能加剧金融不稳定性

### 10.4.3 政策选择原则

**何时使用财政政策**：
1. 货币政策失效时（零利率下限）
2. 需要针对性帮助特定部门时
3. 长期结构性改革需求
4. 货币传导机制受阻时

**何时使用货币政策**：
1. 需要快速响应时
2. 经济处于衰退期但通胀温和
3. 避免挤出效应
4. 日常稳定操作

**政策协调**：
```
理想状态：财政和货币政策协调配合

例子：
- 衰退期：双扩张政策
- 过热期：双紧缩政策
- 滞胀期：结构性财政政策+审慎货币政策
```

### 10.4.4 案例研究：2008年危机后的政策选择

**背景**：
- 房地产泡沫破裂
- 金融危机爆发
- 经济深度衰退

**货币政策困境**：
- 利率降至零下限
- 传统货币政策空间耗尽
- 需要非常规货币政策

**非常规货币政策工具**：

1. **量化宽松（QE）**：
   - 大规模资产购买
   - 压低长期利率
   - 刺激投资和消费

2. **前瞻性指引**：
   - 承诺维持低利率
   - 影响预期
   - 降低长期利率

3. **信贷宽松**：
   - 直接购买私人部门资产
   - 改善信贷可得性
   - 稳定金融市场

**财政政策选择**：
- ARRA法案：7870亿美元刺激
- 财政自动稳定器发挥重要作用
- 财政赤字大幅增加

**政策效果评估**：
```
GDP变化（实际 vs. 潜在）：
2009Q1：-8.4%（实际），0%（潜在）→ 缺口8.4%
2010Q4：+2.8%（实际），2.5%（潜在）→ 缺口接近消除

政策贡献估计：
- 财政政策：贡献1.5-2.0个百分点
- 货币政策：贡献2.0-3.0个百分点
- 总计：3.5-5.0个百分点
```

**经验教训**：
1. 非常规货币政策在零利率下限有效
2. 财政政策在深度衰退中至关重要
3. 政策协调可以增强效果
4. 及时行动比完美行动更重要

### 10.4.5 案例研究：COVID-19疫情的政策创新

**危机特点**：
- 需求和供给双重冲击
- 传统政策工具受限
- 需要创新政策工具

**财政政策创新**：

1. **直接支付**：
   - 成人每人1200美元
   - 直接刺激消费
   - 实施迅速

2. **扩大失业救济**：
   - 每周额外600美元
   - 自动稳定器强化
   - 保护弱势群体

3. **企业贷款计划（PPP）**：
   - 可免除贷款
   - 条件是保留员工
   - 保护就业

**货币政策创新**：

1. **无限量化宽松**：
   - 无限量购买国债和MBS
   - 购买公司债券
   - 直接购买ETF

2. **主街贷款计划**：
   - 向中小企业提供贷款
   - 弥补信贷缺口
   - 创新货币政策工具

3. **市政债券购买**：
   - 支持地方政府
   - 稳定市政债券市场
   - 扩大货币传导范围

**政策效果**：

```
对比2008年和2020年：

维度        2008年          2020年
------------------------------------
政策速度    中等            极快
政策规模    较大            史无前例
政策创新    有限            高度创新
目标群体    金融机构        家庭和企业
政策协调    一般            高度协调
```

**政策启示**：
1. 危机需要快速、大规模政策响应
2. 创新工具可以突破传统限制
3. 直接支持比间接刺激更有效
4. 政策需要覆盖广泛的经济主体

## 10.5 本章总结

### 核心要点

1. **IS-LM模型解释经济波动**：
   - 产品市场均衡由IS曲线描述
   - 货币市场均衡由LM曲线描述
   - 两条曲线的交点确定均衡利率和收入

2. **总需求冲击**：
   - 财政政策冲击移动IS曲线
   - 货币政策冲击移动LM曲线
   - 政策效果取决于挤出效应和乘数效应

3. **总供给冲击**：
   - 负向供给冲击导致滞胀（衰退+通胀）
   - 正向供给冲击带来增长和反通胀
   - 供给冲击创造政策两难

4. **政策工具选择**：
   - 财政政策时滞长但针对性强
   - 货币政策时滞短但效果不确定
   - 政策协调至关重要

5. **经验教训**：
   - 2008年金融危机展示了非常规货币政策的潜力
   - COVID-19疫情证明了创新政策的重要性
   - 政策需要快速、大规模、创新

### 重要概念表格

| 概念 | 定义 | 关键公式/关系 |
|------|------|---------------|
| IS曲线 | 产品市场均衡时的利率与收入组合 | Y = C(Y-T) + I(r) + G + NX |
| LM曲线 | 货币市场均衡时的利率与收入组合 | M/P = L(r, Y) |
| 总需求冲击 | 移动总需求曲线的外部事件 | IS或LM移动 |
| 财政乘数 | 政府购买变化引起的收入变化 | ΔY/ΔG = 1/(1-MPC) · 1/(1+MPC·k/h) |
| 货币乘数 | 货币供给变化引起的收入变化 | ΔY/ΔM = 1/k · 1/(1+(1-MPC)h/k) |
| 挤出效应 | 利率上升减少投资 | ΔI = -β·Δr |
| 滞胀 | 停滞与通货膨胀并存 | Y↓, P↑ |
| 零利率下限 | 利率无法降至零以下 | r ≥ 0 |
