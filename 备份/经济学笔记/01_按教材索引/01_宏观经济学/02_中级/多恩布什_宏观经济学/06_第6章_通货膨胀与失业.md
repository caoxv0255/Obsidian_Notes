# 第6章：通货膨胀与失业

> 本章学习目标：

## 6.1 通货膨胀的动态

### 6.1.1 通货膨胀的定义

**通货膨胀（Inflation）**：
价格水平的持续上升。

**通货膨胀率（Inflation Rate）**：
价格水平的增长率。

**公式**：
```
π = (Pₜ - Pₜ₋₁) / Pₜ₋₁
```
**其中**：
- π：通货膨胀率
- Pₜ：当前时期价格水平
- Pₜ₋₁：上一时期价格水平

### 6.1.2 通货膨胀的动态

**通货膨胀的动态（Dynamics of Inflation）**：
```
π = πᵉ - β·(u - uⁿ)
πᵉ = π₋₁
```
**其中**：
- π：实际通货膨胀率
- πᵉ：预期通货膨胀率
- π₋₁：上一时期的实际通货膨胀率
- u：实际失业率
- uⁿ：自然失业率
- β：参数（β > 0）

**含义**：
- 如果实际失业率低于自然失业率，则实际通货膨胀率高于预期通货膨胀率
- 如果实际失业率高于自然失业率，则实际通货膨胀率低于预期通货膨胀率
- 预期通货膨胀率等于上一时期的实际通货膨胀率（适应性预期）

### 6.1.3 通货膨胀的加速

**通货膨胀的加速（Acceleration of Inflation）**：
```
Δπ = -β·(u - uⁿ)
```
**其中**：
- Δπ：通货膨胀率的变化
- u：实际失业率
- uⁿ：自然失业率
- β：参数（β > 0）

**含义**：
- 如果实际失业率低于自然失业率，则通货膨胀率上升
- 如果实际失业率高于自然失业率，则通货膨胀率下降
- 如果实际失业率等于自然失业率，则通货膨胀率不变

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- PC₁：初始预期通货膨胀率较低的菲利普斯曲线
- 实际失业率低于自然失业率 → 实际通货膨胀率高于预期通货膨胀率
- 预期通货膨胀率上升 → 菲利普斯曲线向上移动到PC₂
- 通货膨胀率加速上升

## 6.2 通货膨胀与失业的权衡

### 6.2.1 短期权衡

**短期权衡（Short-Run Trade-off）**：
在短期内，失业率与通货膨胀率负相关。

**公式**：
```
π = πᵉ - β·(u - uⁿ)
```
**含义**：
- 如果实际失业率低于自然失业率，则实际通货膨胀率高于预期通货膨胀率
- 如果实际失业率高于自然失业率，则实际通货膨胀率低于预期通货膨胀率

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- PC：短期菲利普斯曲线，向右下方倾斜
- 短期权衡：失业率下降 → 通货膨胀率上升
- 失业率上升 → 通货膨胀率下降

### 6.2.2 长期权衡

**长期权衡（Long-Run Trade-off）**：
在长期内，失业率与通货膨胀率无关。

**公式**：
```
u = uⁿ
```
**含义**：
- 在长期内，实际失业率等于自然失业率
- 失业率与通货膨胀率无关

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- LRPC：长期菲利普斯曲线，垂直
- 长期权衡：失业率与通货膨胀率无关
- 在长期内，实际失业率等于自然失业率

## 6.3 滞胀

### 6.3.1 滞胀的定义

**滞胀（Stagflation）**：
高通货膨胀和高失业并存。

### 6.3.2 滞胀的原因

**滞胀的原因（Causes of Stagflation）**：

**1. 负面供给冲击（Negative Supply Shock）**：
- 例如：石油价格上涨、自然灾害、战争
- 影响：SRAS向左移动 → 价格水平上升 → 产出下降 → 失业上升

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产出Y" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- SRAS₁：初始短期总供给曲线
- SRAS₂：负面供给冲击后的短期总供给曲线
- AD：总需求曲线
- 均衡从E₁移动到E₂
- 价格水平从P₁上升到P₂
- 产出从Y₁下降到Y₂
- 失业从u₁上升到u₂

**2. 预期通货膨胀率上升（Increase in Expected Inflation）**：
- 影响：SRAS向左移动 → 价格水平上升 → 产出下降 → 失业上升

**3. 政策失误（Policy Mistakes）**：
- 例如：过度刺激经济 → 预期通货膨胀率上升 → SRAS向左移动 → 价格水平上升 → 产出下降 → 失业上升

### 6.3.3 滞胀的解决

**滞胀的解决（Solutions to Stagflation）**：

**1. 供给政策（Supply Policy）**：
- 例如：减税、放松管制、改善基础设施
- 影响：SRAS向右移动 → 价格水平下降 → 产出上升 → 失业下降

**2. 货币政策（Monetary Policy）**：
- 例如：紧缩货币政策 → 通货膨胀率下降 → 预期通货膨胀率下降 → SRAS向右移动 → 产出上升 → 失业下降

**3. 财政政策（Fiscal Policy）**：
- 例如：紧缩财政政策 → 通货膨胀率下降 → 预期通货膨胀率下降 → SRAS向右移动 → 产出上升 → 失业下降

## 6.4 通货膨胀预期

### 6.4.1 通货膨胀预期的定义

**通货膨胀预期（Inflation Expectations）**：
经济主体对未来通货膨胀率的预期。

### 6.4.2 通货膨胀预期的类型

**通货膨胀预期的类型（Types of Inflation Expectations）**：

**1. 适应性预期（Adaptive Expectations）**：
预期基于过去的经验。

**公式**：
```
πᵉₜ = πₜ₋₁
```
**其中**：
- πᵉₜ：t时期的预期通货膨胀率
- πₜ₋₁：t-1时期的实际通货膨胀率

**含义**：
- 预期通货膨胀率等于上一时期的实际通货膨胀率
- 预期缓慢调整

**2. 理性预期（Rational Expectations）**：
预期基于所有可获得的信息，且与模型的预测一致。

**公式**：
```
πᵉₜ = E(πₜ | Iₜ₋₁)
```
**其中**：
- πᵉₜ：t时期的预期通货膨胀率
- E(πₜ | Iₜ₋₁)：给定t-1时期信息Iₜ₋₁，t时期通货膨胀率的期望

**含义**：
- 预期通货膨胀率等于给定信息下通货膨胀率的期望
- 预期快速调整

### 6.4.3 通货膨胀预期的影响

**通货膨胀预期的影响（Effects of Inflation Expectations）**：

**1. 对菲利普斯曲线的影响**：
```
π = πᵉ - β·(u - uⁿ)
```
- 预期通货膨胀率上升 → 菲利普斯曲线向上移动
- 预期通货膨胀率下降 → 菲利普斯曲线向下移动

**2. 对总供给的影响**：
```
P = (1 + μ)·W/A
W = Pᵉ·F(u, z)
```
- 预期价格水平上升 → 工资上升 → 价格上升
- 预期价格水平下降 → 工资下降 → 价格下降

**3. 对货币政策的影响**：
- 预期通货膨胀率上升 → 实际利率下降 → 刺激经济
- 预期通货膨胀率下降 → 实际利率上升 → 抑制经济

## 6.5 通货膨胀的成本

### 6.5.1 通货膨胀的成本

**通货膨胀的成本（Costs of Inflation）**：

**1. 鞋底成本（Shoeleather Costs）**：
- 通货膨胀率高 → 货币贬值 → 人们减少货币持有量 → 频繁交易 → 交易成本增加

**2. 菜单成本（Menu Costs）**：
- 通货膨胀率高 → 价格频繁变化 → 企业频繁调整价格 → 调整成本增加

**3. 相对价格扭曲（Relative Price Distortions）**：
- 通货膨胀率高 → 价格频繁变化 → 相对价格扭曲 → 资源配置效率降低

**4. 税收扭曲（Tax Distortions）**：
- 通货膨胀率高 → 名义收入上升 → 实际税率上升 → 税收扭曲

**5. 财富再分配（Wealth Redistribution）**：
- 通货膨胀率高 → 债务人受益 → 债权人受损 → 财富再分配

**6. 不确定性增加（Increased Uncertainty）**：
- 通货膨胀率高 → 未来价格不确定 → 投资减少 → 经济增长放缓

### 6.5.2 通货紧缩的成本

**通货紧缩的成本（Costs of Deflation）**：

**1. 债务-通缩螺旋（Debt-Deflation Spiral）**：
- 通货紧缩 → 实际债务增加 → 企业破产 → 投资减少 → 经济衰退 → 通货紧缩

**2. 货币幻觉（Money Illusion）**：
- 通货紧缩 → 名义工资下降 → 工人减少劳动供给 → 失业上升

**3. 流动性陷阱（Liquidity Trap）**：
- 通货紧缩 → 名义利率下降到零 → 货币政策无效 → 经济陷入衰退

## 6.6 本章总结

### 核心要点

1. **通货膨胀的动态**：
   - 通货膨胀：价格水平的持续上升
   - 通货膨胀的动态：π = πᵉ - β·(u - uⁿ)
   - 通货膨胀的加速：Δπ = -β·(u - uⁿ)

2. **通货膨胀与失业的权衡**：
   - 短期权衡：在短期内，失业率与通货膨胀率负相关
   - 长期权衡：在长期内，失业率与通货膨胀率无关

3. **滞胀**：
   - 滞胀：高通货膨胀和高失业并存
   - 滞胀的原因：负面供给冲击、预期通货膨胀率上升、政策失误
   - 滞胀的解决：供给政策、货币政策、财政政策

4. **通货膨胀预期**：
   - 通货膨胀预期：经济主体对未来通货膨胀率的预期
   - 通货膨胀预期的类型：适应性预期、理性预期
   - 通货膨胀预期的影响：对菲利普斯曲线、总供给、货币政策的影响

5. **通货膨胀的成本**：
   - 通货膨胀的成本：鞋底成本、菜单成本、相对价格扭曲、税收扭曲、财富再分配、不确定性增加
   - 通货紧缩的成本：债务-通缩螺旋、货币幻觉、流动性陷阱
