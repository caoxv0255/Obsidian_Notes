## 10.1 联邦储备系统

### 10.1.1 联邦储备系统的定义

**联邦储备系统（Federal Reserve System, Fed）**：
美国的中央银行。

### 10.1.2 联邦储备系统的结构

**联邦储备系统的结构（Structure of Federal Reserve System）**：

**1. 联邦储备委员会（Board of Governors）**：
- 由7名委员组成
- 任期14年
- 由总统任命，参议院确认

**2. 12家联邦储备银行（12 Federal Reserve Banks）**：
- 分布在美国各地
- 私人银行持股

**3. 联邦公开市场委员会（Federal Open Market Committee, FOMC）**：
- 由7名委员 + 12家联邦储备银行行长组成
- 负责制定货币政策

### 10.1.3 联邦储备系统的职能

**联邦储备系统的职能（Functions of Federal Reserve System）**：

**1. 货币政策（Monetary Policy）**：
- 制定和执行货币政策
- 控制货币供给和利率

**2. 银行监管（Bank Regulation）**：
- 监管银行
- 维护金融稳定

**3. 支付系统（Payment System）**：
- 提供支付系统
- 处理支票和电子支付

**4. 政府银行（Government Bank）**：
- 充当政府银行
- 持有政府账户

## 10.2 货币供给

### 10.2.1 货币供给的定义

**货币供给（Money Supply）**：
中央银行控制的货币数量。

### 10.2.2 货币供给的组成部分

**货币供给的组成部分（Components of Money Supply）**：

**1. 基础货币（Monetary Base, B）**：
```
B = C + R
```
**其中**：
- B：基础货币
- C：通货（流通中的现金）
- R：准备金（银行持有的准备金）

**2. 货币供给（Money Supply, M）**：
```
M = C + D
```
**其中**：
- M：货币供给
- C：通货
- D：活期存款

### 10.2.3 货币乘数

**货币乘数（Money Multiplier, m）**：
基础货币变化导致的货币供给变化的倍数。

**公式**：
```
M = m·B
```
**其中**：
- M：货币供给
- m：货币乘数
- B：基础货币

**货币乘数的推导（Derivation of Money Multiplier）**：
```
M = C + D
B = C + R
R = θ·D（准备金率）
C = λ·D（通货-存款比率）

M = λ·D + D = (λ + 1)·D
B = λ·D + θ·D = (λ + θ)·D

M/B = (λ + 1) / (λ + θ) = m
```

**含义**：
- 货币乘数取决于通货-存款比率和准备金率
- 通货-存款比率上升 → 货币乘数下降
- 准备金率上升 → 货币乘数下降

### 10.2.4 货币供给的控制

**货币供给的控制（Control of Money Supply）**：

**1. 公开市场操作（Open Market Operations）**：
- 购买债券 → 基础货币增加 → 货币供给增加
- 出售债券 → 基础货币减少 → 货币供给减少

**2. 准备金率（Reserve Requirements）**：
- 准备金率上升 → 货币乘数下降 → 货币供给减少
- 准备金率下降 → 货币乘数上升 → 货币供给增加

**3. 贴现率（Discount Rate）**：
- 贴现率上升 → 银行借贷减少 → 准备金减少 → 货币供给减少
- 贴现率下降 → 银行借贷增加 → 准备金增加 → 货币供给增加

## 10.3 货币政策

### 10.3.1 货币政策的定义

**货币政策（Monetary Policy）**：
中央银行通过控制货币供给和利率来影响经济的政策。

### 10.3.2 货币政策工具

**货币政策工具（Monetary Policy Tools）**：

**1. 公开市场操作（Open Market Operations）**：
- 最常用的货币政策工具
- 购买债券 → 货币供给增加 → 利率下降
- 出售债券 → 货币供给减少 → 利率上升

**2. 准备金率（Reserve Requirements）**：
- 准备金率上升 → 货币供给减少 → 利率上升
- 准备金率下降 → 货币供给增加 → 利率下降

**3. 贴现率（Discount Rate）**：
- 贴现率上升 → 货币供给减少 → 利率上升
- 贴现率下降 → 货币供给增加 → 利率下降

**4. 联邦基金利率（Federal Funds Rate）**：
- 银行之间的隔夜拆借利率
- 目标利率由联邦公开市场委员会设定
- 通过公开市场操作实现目标利率

### 10.3.3 货币政策的传导机制

**货币政策的传导机制（Transmission Mechanism of Monetary Policy）**：
```
货币政策 → 货币供给变化 → 利率变化 → 投资变化 → 产出变化
```

**图形表示**：


- 货币供给增加 → 利率下降 → 投资增加 → 产出增加

### 10.3.4 货币政策的效果

**货币政策的效果（Effects of Monetary Policy）**：

**1. 扩张性货币政策（Expansionary Monetary Policy）**：
- 货币供给增加 → 利率下降 → 投资增加 → 产出增加
- 用于刺激经济

**2. 紧缩性货币政策（Contractionary Monetary Policy）**：
- 货币供给减少 → 利率上升 → 投资减少 → 产出减少
- 用于抑制通货膨胀

## 10.4 信用配给

### 10.4.1 信用配给的定义

**信用配给（Credit Rationing）**：
即使借款人愿意支付更高的利率，银行也不愿意提供更多的贷款。

### 10.4.2 信用配给的原因

**信用配给的原因（Causes of Credit Rationing）**：

**1. 逆向选择（Adverse Selection）**：
- 利率上升 → 高风险借款人更愿意借款 → 银行风险增加 → 银行减少贷款

**2. 道德风险（Moral Hazard）**：
- 利率上升 → 借款人更愿意冒险 → 银行风险增加 → 银行减少贷款

**3. 信息不对称（Information Asymmetry）**：
- 银行无法完全了解借款人的风险 → 银行限制贷款

### 10.4.3 信用配给的影响

**信用配给的影响（Effects of Credit Rationing）**：
- 信用配给 → 贷款减少 → 投资减少 → 产出减少
- 信用配给 → 货币政策效果减弱

## 10.5 本章总结

### 核心要点

1. **联邦储备系统**：
   - 联邦储备系统：美国的中央银行
   - 联邦储备系统的结构：联邦储备委员会、12家联邦储备银行、联邦公开市场委员会
   - 联邦储备系统的职能：货币政策、银行监管、支付系统、政府银行

2. **货币供给**：
   - 货币供给：中央银行控制的货币数量
   - 基础货币：B = C + R
   - 货币供给：M = C + D
   - 货币乘数：M = m·B，m = (λ + 1) / (λ + θ)
   - 货币供给的控制：公开市场操作、准备金率、贴现率

3. **货币政策**：
   - 货币政策：中央银行通过控制货币供给和利率来影响经济的政策
   - 货币政策工具：公开市场操作、准备金率、贴现率、联邦基金利率
   - 货币政策的传导机制：货币政策 → 货币供给变化 → 利率变化 → 投资变化 → 产出变化
   - 货币政策的效果：扩张性货币政策、紧缩性货币政策

4. **信用配给**：
   - 信用配给：即使借款人愿意支付更高的利率，银行也不愿意提供更多的贷款
   - 信用配给的原因：逆向选择、道德风险、信息不对称
   - 信用配给的影响：贷款减少 → 投资减少 → 产出减少
