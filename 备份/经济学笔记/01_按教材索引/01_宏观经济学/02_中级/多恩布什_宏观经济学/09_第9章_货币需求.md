# 第9章：货币需求

> 本章学习目标：
> - 理解货币的定义
> - 掌握货币需求理论
> - 学会分析货币流通速度

---

## 9.1 货币的定义

### 9.1.1 货币的定义

**货币（Money）**：
用于交易和储藏价值的资产。

### 9.1.2 货币的职能

**货币的职能（Functions of Money）**：

**1. 交易媒介（Medium of Exchange）**：
- 货币用于交易
- 降低交易成本

**2. 计价单位（Unit of Account）**：
- 货币用于计价
- 降低信息成本

**3. 储藏价值（Store of Value）**：
- 货币用于储藏价值
- 可以跨期转移购买力

### 9.1.3 货币的度量

**货币的度量（Measurement of Money）**：

**1. M1（狭义货币）**：
```
M1 = 通货 + 活期存款
```
**其中**：
- 通货：纸币和硬币
- 活期存款：支票账户

**2. M2（广义货币）**：
```
M2 = M1 + 储蓄存款 + 货币市场存款
```
**其中**：
- 储蓄存款：定期存款
- 货币市场存款：货币市场基金

**3. M3（更广义货币）**：
```
M3 = M2 + 大额定期存款 + 回购协议
```

## 9.2 货币需求理论

### 9.2.1 货币需求的定义

**货币需求（Money Demand）**：
人们愿意持有的货币数量。

### 9.2.2 古典货币数量论

**古典货币数量论（Classical Quantity Theory of Money）**：
```
M·V = P·Y
```
**其中**：
- M：货币供给
- V：货币流通速度
- P：价格水平
- Y：实际产出

**含义**：
- 货币供给 × 货币流通速度 = 价格水平 × 实际产出
- 货币流通速度不变 → 货币供给变化导致价格水平成比例变化

**货币需求（Money Demand）**：
```
L = k·P·Y
```
**其中**：
- L：货币需求
- k：参数（k = 1/V）
- P：价格水平
- Y：实际产出

**含义**：
- 货币需求与价格水平和实际产出正相关
- 货币需求与货币流通速度负相关

### 9.2.3 凯恩斯的流动性偏好理论

**流动性偏好理论（Liquidity Preference Theory）**：
人们持有货币的动机有三种：交易动机、预防动机、投机动机。

**货币需求（Money Demand）**：
```
L = L₁(Y) + L₂(r)
```
**其中**：
- L：货币需求
- L₁(Y)：交易动机和预防动机的货币需求，取决于收入Y
- L₂(r)：投机动机的货币需求，取决于利率r

**性质**：
- 收入增加 → 交易动机和预防动机的货币需求增加
- 利率上升 → 投机动机的货币需求减少

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "利率r" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 货币需求：L = L₁(Y) + L₂(r)，向右下方倾斜
- 利率上升 → 货币需求下降
- 利率下降 → 货币需求上升

### 9.2.4 鲍莫尔-托宾模型

**鲍莫尔-托宾模型（Baumol-Tobin Model）**：
交易性货币需求的平方根法则。

**货币需求（Money Demand）**：
```
L = √(c·Y·r / 2)
```
**其中**：
- L：最优货币持有量
- c：每次交易的固定成本
- Y：收入
- r：利率

**含义**：
- 最优货币持有量与收入的平方根正相关
- 最优货币持有量与利率的平方根负相关

**平方根法则（Square-Root Rule）**：
```
L ∝ √Y
```
**含义**：
- 货币需求的收入弹性为0.5
- 货币需求的利率弹性为-0.5

## 9.3 货币流通速度

### 9.3.1 货币流通速度的定义

**货币流通速度（Velocity of Money）**：
货币在经济中流通的快慢。

**公式**：
```
V = (P·Y) / M
```
**其中**：
- V：货币流通速度
- P：价格水平
- Y：实际产出
- M：货币供给

**含义**：
- 货币流通速度 = 名义GDP / 货币供给
- 货币流通速度表示1单位货币在一年内用于交易的次数

### 9.3.2 货币流通速度的决定因素

**货币流通速度的决定因素（Determinants of Velocity of Money）**：

**1. 支付技术（Payment Technology）**：
- 支付技术改进 → 货币流通速度上升
- 例如：信用卡、电子支付

**2. 通货膨胀率（Inflation Rate）**：
- 通货膨胀率上升 → 货币贬值 → 人们减少货币持有 → 货币流通速度上升

**3. 利率（Interest Rate）**：
- 利率上升 → 持有货币的机会成本上升 → 人们减少货币持有 → 货币流通速度上升

**4. 金融创新（Financial Innovation）**：
- 金融创新 → 货币替代品增加 → 货币流通速度上升

### 9.3.3 货币流通速度的稳定性

**货币流通速度的稳定性（Stability of Velocity of Money）**：
- 古典观点：货币流通速度稳定
- 现代观点：货币流通速度不稳定

**原因**：
- 支付技术变化
- 金融创新
- 货币替代品增加

## 9.4 货币需求的经验研究

### 9.4.1 货币需求的收入弹性

**货币需求的收入弹性（Income Elasticity of Money Demand）**：
收入增加1%，货币需求增加的百分比。

**公式**：
```
εᵧ = (ΔL/L) / (ΔY/Y)
```
**其中**：
- εᵧ：货币需求的收入弹性
- L：货币需求
- Y：收入

**经验结果**：
- 货币需求的收入弹性约为1
- 收入增加1%，货币需求增加约1%

### 9.4.2 货币需求的利率弹性

**货币需求的利率弹性（Interest Elasticity of Money Demand）**：
利率上升1%，货币需求减少的百分比。

**公式**：
```
εᵣ = (ΔL/L) / (Δr/r)
```
**其中**：
- εᵣ：货币需求的利率弹性
- L：货币需求
- r：利率

**经验结果**：
- 货币需求的利率弹性约为-0.1到-0.5
- 利率上升1%，货币需求减少约0.1%到0.5%

### 9.4.3 货币需求的稳定性

**货币需求的稳定性（Stability of Money Demand）**：
- 古典观点：货币需求稳定
- 现代观点：货币需求不稳定

**原因**：
- 金融创新
- 货币替代品增加
- 制度变化

## 9.5 本章总结

### 核心要点

1. **货币的定义**：
   - 货币：用于交易和储藏价值的资产
   - 货币的职能：交易媒介、计价单位、储藏价值
   - 货币的度量：M1、M2、M3

2. **货币需求理论**：
   - 古典货币数量论：M·V = P·Y，L = k·P·Y
   - 凯恩斯的流动性偏好理论：L = L₁(Y) + L₂(r)
   - 鲍莫尔-托宾模型：L = √(c·Y·r / 2)，平方根法则

3. **货币流通速度**：
   - 货币流通速度：V = (P·Y) / M
   - 货币流通速度的决定因素：支付技术、通货膨胀率、利率、金融创新
   - 货币流通速度的稳定性：古典观点稳定，现代观点不稳定

4. **货币需求的经验研究**：
   - 货币需求的收入弹性：约为1
   - 货币需求的利率弹性：约为-0.1到-0.5
   - 货币需求的稳定性：古典观点稳定，现代观点不稳定
