## 7.1 消费函数

### 7.1.1 消费函数的定义

**消费函数（Consumption Function）**：
表示消费与收入之间的关系。

### 7.1.2 凯恩斯消费函数

**凯恩斯消费函数（Keynesian Consumption Function）**：
```
C = a + b·Y
```
**其中**：
- C：消费
- Y：收入
- a：自发消费（a > 0）
- b：边际消费倾向（0 < b < 1）

**边际消费倾向（Marginal Propensity to Consume, MPC）**：
收入增加1单位时，消费增加的量。

**公式**：
```
MPC = ΔC / ΔY = b
```

**平均消费倾向（Average Propensity to Consume, APC）**：
消费占收入的比例。

**公式**：
```
APC = C / Y = a/Y + b
```

**分析**：
- 消费函数：C = a + b·Y，向右上方倾斜
- a：自发消费，截距
- b：边际消费倾向，斜率
- 收入增加 → 消费增加

### 7.1.3 凯恩斯消费函数的特征

**凯恩斯消费函数的特征（Characteristics of Keynesian Consumption Function）**：

**1. 边际消费倾向递减（Diminishing Marginal Propensity to Consume）**：
- 收入增加 → 边际消费倾向下降
- 穷人：边际消费倾向高
- 富人：边际消费倾向低

**2. 平均消费倾向递减（Diminishing Average Propensity to Consume）**：
- 收入增加 → 平均消费倾向下降
- 穷人：平均消费倾向高
- 富人：平均消费倾向低

**3. 消费取决于当前收入（Consumption Depends on Current Income）**：
- 消费取决于当前收入，而不是预期收入
- 短期消费函数与长期消费函数不同

## 7.2 生命周期假说

### 7.2.1 生命周期假说的定义

**生命周期假说（Life-Cycle Hypothesis, LCH）**：
莫迪利亚尼提出的消费理论，认为消费取决于整个生命周期的收入。

### 7.2.2 生命周期假说的基本假设

**生命周期假说的基本假设（Basic Assumptions of Life-Cycle Hypothesis）**：

**1. 消费者理性（Rational Consumers）**：
- 消费者最大化整个生命周期的效用
- 消费者平滑消费

**2. 预期收入可预测（Predictable Expected Income）**：
- 消费者可以预测未来收入
- 消费者根据预期收入制定消费计划

**3. 资本市场完善（Perfect Capital Markets）**：
- 消费者可以自由借贷
- 利率相同

### 7.2.3 生命周期假说的消费函数

**生命周期假说的消费函数（Consumption Function of Life-Cycle Hypothesis）**：
```
C = (1/T)·(W + R·Y)
```
**其中**：
- C：消费
- T：生命长度
- W：初始财富
- R：工作年数
- Y：年收入

**含义**：
- 消费取决于初始财富和预期收入
- 消费者在整个生命周期内平滑消费

**图形表示**：

- 消费：平滑，不随年龄变化
- 收入：工作期间高，退休期间低
- 工作期间：储蓄 = 收入 - 消费
- 退休期间：负储蓄 = 收入 - 消费

### 7.2.4 生命周期假说的启示

**生命周期假说的启示（Implications of Life-Cycle Hypothesis）**：

**1. 储蓄取决于年龄（Savings Depend on Age）**：
- 年轻人：负储蓄（借贷）
- 中年人：正储蓄（为退休储蓄）
- 老年人：负储蓄（使用储蓄）

**2. 储蓄取决于财富（Savings Depend on Wealth）**：
- 财富增加 → 消费增加 → 储蓄减少
- 财富减少 → 消费减少 → 储蓄增加

**3. 财富效应（Wealth Effect）**：
- 股票价格上涨 → 财富增加 → 消费增加
- 房产价格上涨 → 财富增加 → 消费增加

## 7.3 持久收入假说

### 7.3.1 持久收入假说的定义

**持久收入假说（Permanent Income Hypothesis, PIH）**：
弗里德曼提出的消费理论，认为消费取决于持久收入。

### 7.3.2 持久收入假说的基本假设

**持久收入假说的基本假设（Basic Assumptions of Permanent Income Hypothesis）**：

**1. 消费者理性（Rational Consumers）**：
- 消费者最大化整个生命周期的效用
- 消费者平滑消费

**2. 预期收入可预测（Predictable Expected Income）**：
- 消费者可以预测未来收入
- 消费者根据预期收入制定消费计划

**3. 持久收入和暂时收入（Permanent and Transitory Income）**：
- 收入 = 持久收入 + 暂时收入
- 消费取决于持久收入，而不是暂时收入

### 7.3.3 持久收入假说的消费函数

**持久收入假说的消费函数（Consumption Function of Permanent Income Hypothesis）**：
```
C = k·Yᵖ
```
**其中**：
- C：消费
- k：边际消费倾向（常数）
- Yᵖ：持久收入

**持久收入（Permanent Income）**：
```
Yᵖ = θ·Y + (1 - θ)·Yᵖ₋₁
```
**其中**：
- Yᵖ：持久收入
- Y：当前收入
- Yᵖ₋₁：上一期的持久收入
- θ：参数（0 < θ < 1）

**含义**：
- 持久收入是当前收入和上一期持久收入的加权平均
- 持久收入平滑变化
- 消费取决于持久收入，而不是暂时收入

### 7.3.4 持久收入假说的启示

**持久收入假说的启示（Implications of Permanent Income Hypothesis）**：

**1. 暂时收入变化不影响消费（Transitory Income Changes Do Not Affect Consumption）**：
- 暂时收入增加 → 持久收入增加很少 → 消费增加很少
- 暂时收入减少 → 持久收入减少很少 → 消费减少很少

**2. 持久收入变化影响消费（Permanent Income Changes Affect Consumption）**：
- 持久收入增加 → 消费增加
- 持久收入减少 → 消费减少

**3. 边际消费倾向取决于收入变化的性质（Marginal Propensity to Consume Depends on Nature of Income Changes）**：
- 暂时收入变化：边际消费倾向低
- 持久收入变化：边际消费倾向高

## 7.4 消费与储蓄

### 7.4.1 储蓄的定义

**储蓄（Savings）**：
收入中未消费的部分。

**公式**：
```
S = Y - C
```
**其中**：
- S：储蓄
- Y：收入
- C：消费

### 7.4.2 储蓄的决定因素

**储蓄的决定因素（Determinants of Savings）**：

**1. 收入（Income）**：
- 收入增加 → 储蓄增加
- 收入减少 → 储蓄减少

**2. 利率（Interest Rate）**：
- 利率上升 → 储蓄增加（替代效应）
- 利率上升 → 储蓄减少（收入效应）
- 净效应：替代效应 > 收入效应 → 储蓄增加

**3. 财富（Wealth）**：
- 财富增加 → 消费增加 → 储蓄减少
- 财富减少 → 消费减少 → 储蓄增加

**4. 预期（Expectations）**：
- 预期收入增加 → 消费增加 → 储蓄减少
- 预期收入减少 → 消费减少 → 储蓄增加

### 7.4.3 储蓄的动机

**储蓄的动机（Motivations for Savings）**：

**1. 生命周期储蓄（Life-Cycle Savings）**：
- 为退休储蓄
- 为大额支出储蓄（例如：购房、子女教育）

**2. 预防性储蓄（Precautionary Savings）**：
- 为不确定事件储蓄（例如：失业、疾病）

**3. 遗产储蓄（Bequest Savings）**：
- 为子女储蓄

**4. 投资储蓄（Investment Savings）**：
- 为投资储蓄（例如：创业、股票投资）

## 7.5 本章总结

### 核心要点

1. **消费函数**：
   - 消费函数：表示消费与收入之间的关系
   - 凯恩斯消费函数：C = a + b·Y
   - 边际消费倾向：MPC = ΔC / ΔY = b
   - 平均消费倾向：APC = C / Y = a/Y + b
   - 凯恩斯消费函数的特征：边际消费倾向递减、平均消费倾向递减、消费取决于当前收入

2. **生命周期假说**：
   - 生命周期假说：消费取决于整个生命周期的收入
   - 生命周期假说的消费函数：C = (1/T)·(W + R·Y)
   - 生命周期假说的启示：储蓄取决于年龄、储蓄取决于财富、财富效应

3. **持久收入假说**：
   - 持久收入假说：消费取决于持久收入
   - 持久收入假说的消费函数：C = k·Yᵖ
   - 持久收入：Yᵖ = θ·Y + (1 - θ)·Yᵖ₋₁
   - 持久收入假说的启示：暂时收入变化不影响消费、持久收入变化影响消费、边际消费倾向取决于收入变化的性质

4. **消费与储蓄**：
   - 储蓄：收入中未消费的部分
   - 储蓄的决定因素：收入、利率、财富、预期
   - 储蓄的动机：生命周期储蓄、预防性储蓄、遗产储蓄、投资储蓄
