## 5.1 菲利普斯曲线

### 5.1.1 菲利普斯曲线的定义

**菲利普斯曲线（Phillips Curve）**：
表示失业率与通货膨胀率之间负相关的曲线。

### 5.1.2 原始菲利普斯曲线

**原始菲利普斯曲线（Original Phillips Curve）**：
失业率与名义工资增长率负相关。

**公式**：
```
ΔW/W = -β·u + γ
```
**其中**：
- ΔW/W：名义工资增长率
- u：失业率
- β：参数（β > 0）
- γ：参数

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 原始菲利普斯曲线：向右下方倾斜
- 失业率上升 → 工资增长率下降
- 失业率下降 → 工资增长率上升

### 5.1.3 修正的菲利普斯曲线

**修正的菲利普斯曲线（Modified Phillips Curve）**：
失业率与通货膨胀率负相关。

**公式**：
```
π = -β·u + γ
```
**其中**：
- π：通货膨胀率
- u：失业率
- β：参数（β > 0）
- γ：参数

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 修正的菲利普斯曲线：向右下方倾斜
- 失业率上升 → 通货膨胀率下降
- 失业率下降 → 通货膨胀率上升

### 5.1.4 附加预期的菲利普斯曲线

**附加预期的菲利普斯曲线（Expectations-Augmented Phillips Curve）**：
失业率与通货膨胀率之间的权衡取决于预期通货膨胀率。

**公式**：
```
π = πᵉ - β·(u - uⁿ)
```
**其中**：
- π：实际通货膨胀率
- πᵉ：预期通货膨胀率
- u：实际失业率
- uⁿ：自然失业率
- β：参数（β > 0）

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- PC₁：预期通货膨胀率较低的菲利普斯曲线
- PC₂：预期通货膨胀率较高的菲利普斯曲线
- uⁿ：自然失业率
- 实际失业率等于自然失业率时，实际通货膨胀率等于预期通货膨胀率

## 5.2 工资-价格机制

### 5.2.1 工资-价格机制的定义

**工资-价格机制（Wage-Price Mechanism）**：
工资与价格之间的相互作用。

### 5.2.2 工资决定

**工资决定（Wage Determination）**：
```
W = Pᵉ·F(u, z)
```
**其中**：
- W：名义工资
- Pᵉ：预期价格水平
- u：失业率
- z：其他因素（例如：失业保险、工会力量）

**性质**：
1. 失业率上升 → 工资下降
2. 预期价格水平上升 → 工资上升
3. 其他因素（例如：失业保险）上升 → 工资上升

### 5.2.3 价格决定

**价格决定（Price Determination）**：
```
P = (1 + μ)·W/A
```
**其中**：
- P：价格水平
- W：名义工资
- A：生产率
- μ：价格加成（mark-up）

**性质**：
1. 工资上升 → 价格上升
2. 生产率上升 → 价格下降
3. 价格加成上升 → 价格上升

### 5.2.4 工资-价格螺旋

**工资-价格螺旋（Wage-Price Spiral）**：
工资上升 → 价格上升 → 工资上升 → 价格上升 → ...

**机制**：
```
工资上升 → 价格上升 → 预期价格水平上升 → 工资上升 → 价格上升 → ...
```

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "时间" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- t₁：初始时间，价格水平为P₁
- t₂：工资上升 → 价格水平上升到P₂
- 工资-价格螺旋导致价格水平持续上升

## 5.3 预期与调整

### 5.3.1 预期的类型

**预期的类型（Types of Expectations）**：

**1. 适应性预期（Adaptive Expectations）**：
预期基于过去的经验。

**公式**：
```
πᵉₜ = πₜ₋₁
```
**其中**：
- πᵉₜ：t时期的预期通货膨胀率
- πₜ₋₁：t-1时期的实际通货膨胀率

**含义**：
- 预期通货膨胀率等于上一时期的实际通货膨胀率
- 预期缓慢调整

**2. 理性预期（Rational Expectations）**：
预期基于所有可获得的信息，且与模型的预测一致。

**公式**：
```
πᵉₜ = E(πₜ | Iₜ₋₁)
```
**其中**：
- πᵉₜ：t时期的预期通货膨胀率
- E(πₜ | Iₜ₋₁)：给定t-1时期信息Iₜ₋₁，t时期通货膨胀率的期望

**含义**：
- 预期通货膨胀率等于给定信息下通货膨胀率的期望
- 预期快速调整

### 5.3.2 预期与菲利普斯曲线

**预期与菲利普斯曲线（Expectations and Phillips Curve）**：
```
π = πᵉ - β·(u - uⁿ)
```
**含义**：
- 如果实际失业率等于自然失业率，则实际通货膨胀率等于预期通货膨胀率
- 如果实际失业率低于自然失业率，则实际通货膨胀率高于预期通货膨胀率
- 如果实际失业率高于自然失业率，则实际通货膨胀率低于预期通货膨胀率

### 5.3.3 预期的调整

**预期的调整（Adjustment of Expectations）**：
- 实际通货膨胀率高于预期通货膨胀率 → 预期通货膨胀率上升
- 实际通货膨胀率低于预期通货膨胀率 → 预期通货膨胀率下降

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "失业率" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- PC₁：初始预期通货膨胀率较低的菲利普斯曲线
- 实际失业率低于自然失业率 → 实际通货膨胀率高于预期通货膨胀率
- 预期通货膨胀率上升 → 菲利普斯曲线向上移动到PC₂

## 5.4 自然失业率

### 5.4.1 自然失业率的定义

**自然失业率（Natural Rate of Unemployment）**：
实际通货膨胀率等于预期通货膨胀率时的失业率。

**公式**：
```
uⁿ
```
**其中**：
- uⁿ：自然失业率

### 5.4.2 自然失业率的决定因素

**自然失业率的决定因素（Determinants of Natural Rate of Unemployment）**：

**1. 摩擦性失业（Frictional Unemployment）**：
- 由于工作转换或初次求职造成的短期失业
- 影响因素：劳动力流动性、信息透明度

**2. 结构性失业（Structural Unemployment）**：
- 由于经济结构变化造成的长期失业
- 影响因素：技术进步、产业结构调整

**3. 制度因素（Institutional Factors）**：
- 失业保险：失业保险慷慨 → 自然失业率上升
- 最低工资：最低工资提高 → 自然失业率上升
- 工会力量：工会力量强 → 自然失业率上升

### 5.4.3 自然失业率的估计

**自然失业率的估计（Estimation of Natural Rate of Unemployment）**：
- 方法1：统计方法（例如：HP滤波）
- 方法2：结构方法（例如：基于劳动市场模型）

**估计结果**：
- 美国：自然失业率约为5-6%
- 欧洲大陆：自然失业率约为8-10%
- 日本：自然失业率约为3-4%

## 5.5 本章总结

### 核心要点

1. **菲利普斯曲线**：
   - 菲利普斯曲线：表示失业率与通货膨胀率之间负相关的曲线
   - 原始菲利普斯曲线：ΔW/W = -β·u + γ
   - 修正的菲利普斯曲线：π = -β·u + γ
   - 附加预期的菲利普斯曲线：π = πᵉ - β·(u - uⁿ)

2. **工资-价格机制**：
   - 工资-价格机制：工资与价格之间的相互作用
   - 工资决定：W = Pᵉ·F(u, z)
   - 价格决定：P = (1 + μ)·W/A
   - 工资-价格螺旋：工资上升 → 价格上升 → 工资上升 → 价格上升 → ...

3. **预期与调整**：
   - 预期的类型：适应性预期、理性预期
   - 适应性预期：πᵉₜ = πₜ₋₁
   - 理性预期：πᵉₜ = E(πₜ | Iₜ₋₁)
   - 预期与菲利普斯曲线：π = πᵉ - β·(u - uⁿ)
   - 预期的调整：实际通货膨胀率高于预期通货膨胀率 → 预期通货膨胀率上升

4. **自然失业率**：
   - 自然失业率：实际通货膨胀率等于预期通货膨胀率时的失业率
   - 自然失业率的决定因素：摩擦性失业、结构性失业、制度因素
   - 自然失业率的估计：统计方法、结构方法
