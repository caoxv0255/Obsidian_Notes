# 第8章：投资

> 本章学习目标：
> - 理解投资理论
> - 掌握企业的投资决策

---

## 8.1 投资理论

### 8.1.1 投资的定义

**投资（Investment）**：
用于增加资本存量的支出。

### 8.1.2 投资的类型

**投资的类型（Types of Investment）**：

**1. 企业固定投资（Business Fixed Investment）**：
- 企业购买机器、设备、厂房等
- 是总投资的主要部分

**2. 住房投资（Residential Investment）**：
- 居民购买新房
- 受利率和房价影响

**3. 存货投资（Inventory Investment）**：
- 企业存货的变化
- 包括原材料、在产品、产成品

### 8.1.3 新古典投资理论

**新古典投资理论（Neoclassical Investment Theory）**：
企业根据资本边际收益等于资本边际成本的原则决定投资。

**投资函数（Investment Function）**：
```
I = I(r)
```
**其中**：
- I：投资
- r：实际利率

**性质**：
- 实际利率上升 → 投资下降
- 实际利率下降 → 投资上升

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "实际利率r" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 投资函数：I = I(r)，向右下方倾斜
- 实际利率上升 → 投资下降
- 实际利率下降 → 投资上升

## 8.2 企业的投资决策

### 8.2.1 企业的投资决策

**企业的投资决策（Firm's Investment Decision）**：
企业比较投资的预期收益和成本，决定是否投资。

### 8.2.2 资本边际收益

**资本边际收益（Marginal Revenue Product of Capital, MRPK）**：
增加1单位资本带来的收益。

**公式**：
```
MRPK = MPK·P
```
**其中**：
- MRPK：资本边际收益
- MPK：资本边际产出
- P：产品价格

**性质**：
- 资本增加 → 资本边际产出下降
- 资本增加 → 资本边际收益下降

### 8.2.3 资本边际成本

**资本边际成本（Marginal Cost of Capital, MCₖ）**：
增加1单位资本的成本。

**公式**：
```
MCₖ = (r + δ)·Pₖ
```
**其中**：
- MCₖ：资本边际成本
- r：实际利率
- δ：折旧率
- Pₖ：资本价格

**性质**：
- 实际利率上升 → 资本边际成本上升
- 折旧率上升 → 资本边际成本上升
- 资本价格上升 → 资本边际成本上升

### 8.2.4 投资决策规则

**投资决策规则（Investment Decision Rule）**：
- 如果MRPK > MCₖ，则投资
- 如果MRPK < MCₖ，则不投资
- 如果MRPK = MCₖ，则资本最优

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "资本K" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- MRPK：资本边际收益，向右下方倾斜
- MCₖ：资本边际成本，水平线
- K*：最优资本存量
- 如果K < K*，则MRPK > MCₖ，投资
- 如果K > K*，则MRPK < MCₖ，负投资

### 8.2.5 最优资本存量

**最优资本存量（Optimal Capital Stock）**：
资本边际收益等于资本边际成本时的资本存量。

**条件**：
```
MRPK = MCₖ
MPK·P = (r + δ)·Pₖ
MPK = (r + δ)·(Pₖ/P)
```

**含义**：
- 最优资本存量取决于实际利率和折旧率
- 实际利率上升 → 最优资本存量下降
- 折旧率上升 → 最优资本存量下降

## 8.3 住房投资

### 8.3.1 住房投资的决定

**住房投资的决定（Determinants of Residential Investment）**：
- 住房投资取决于住房需求
- 住房需求取决于住房价格、收入、利率

### 8.3.2 住房价格

**住房价格（Housing Price）**：
住房的均衡价格。

**决定因素**：
- 住房需求上升 → 住房价格上升
- 住房供给上升 → 住房价格下降

### 8.3.3 住房投资与利率

**住房投资与利率（Residential Investment and Interest Rate）**：
- 利率上升 → 住房抵押贷款利率上升 → 住房需求下降 → 住房价格下降 → 住房投资下降
- 利率下降 → 住房抵押贷款利率下降 → 住房需求上升 → 住房价格上升 → 住房投资上升

**图形表示**：

```mermaid

    x-axis "利率r" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25

```

**分析**：
- 住房投资：I = I(r)，向右下方倾斜
- 利率上升 → 住房投资下降
- 利率下降 → 住房投资上升

## 8.4 存货投资

### 8.4.1 存货投资的决定

**存货投资的决定（Determinants of Inventory Investment）**：
- 存货投资取决于销售、生产、成本

### 8.4.2 存货的动机

**存货的动机（Motivations for Inventory）**：

**1. 生产平滑（Production Smoothing）**：
- 需求波动 → 生产平滑 → 存货波动

**2. 规模经济（Economies of Scale）**：
- 大规模生产 → 单位成本降低 → 存货增加

**3. 预防性动机（Precautionary Motive）**：
- 为不确定需求准备存货

**4. 投机性动机（Speculative Motive）**：
- 预期价格上涨 → 增加存货

### 8.4.3 存货投资与利率

**存货投资与利率（Inventory Investment and Interest Rate）**：
- 利率上升 → 持有存货的机会成本上升 → 存货投资下降
- 利率下降 → 持有存货的机会成本下降 → 存货投资上升

- 利率上升 → 存货投资下降
- 利率下降 → 存货投资上升

## 8.5 投资的波动

### 8.5.1 投资的波动性

**投资的波动性（Volatility of Investment）**：
- 投资是GDP中最波动的组成部分
- 投资波动是经济周期的主要驱动力

### 8.5.2 投资波动的原因

**投资波动的原因（Causes of Investment Volatility）**：

**1. 利率波动（Interest Rate Volatility）**：
- 利率波动 → 资本边际成本波动 → 投资波动

**2. 预期波动（Expectations Volatility）**：
- 预期波动 → 资本边际收益波动 → 投资波动

**3. 不确定性（Uncertainty）**：
- 不确定性增加 → 投资减少

**4. 金融约束（Financial Constraints）**：
- 金融约束紧 → 投资减少

### 8.5.3 投资与经济周期

**投资与经济周期（Investment and Business Cycle）**：
- 经济扩张 → 投资增加 → 经济扩张（加速器效应）
- 经济衰退 → 投资减少 → 经济衰退（减速器效应）

**加速器原理（Accelerator Principle）**：
```
I = α·ΔY
```
**其中**：
- I：投资
- α：加速系数（α > 0）
- ΔY：产出变化

**含义**：
- 产出增加 → 投资增加
- 产出减少 → 投资减少

## 8.6 本章总结

### 核心要点

1. **投资理论**：
   - 投资：用于增加资本存量的支出
   - 投资的类型：企业固定投资、住房投资、存货投资
   - 新古典投资理论：企业根据资本边际收益等于资本边际成本的原则决定投资
   - 投资函数：I = I(r)，向右下方倾斜

2. **企业的投资决策**：
   - 企业的投资决策：企业比较投资的预期收益和成本，决定是否投资
   - 资本边际收益：MRPK = MPK·P
   - 资本边际成本：MCₖ = (r + δ)·Pₖ
   - 投资决策规则：如果MRPK > MCₖ，则投资
   - 最优资本存量：MRPK = MCₖ

3. **住房投资**：
   - 住房投资取决于住房需求
   - 住房投资与利率：利率上升 → 住房投资下降
   - 住房投资函数：I = I(r)，向右下方倾斜

4. **存货投资**：
   - 存货投资取决于销售、生产、成本
   - 存货的动机：生产平滑、规模经济、预防性动机、投机性动机
   - 存货投资与利率：利率上升 → 存货投资下降
   - 存货投资函数：I = I(r)，向右下方倾斜

5. **投资的波动**：
   - 投资是GDP中最波动的组成部分
   - 投资波动的原因：利率波动、预期波动、不确定性、金融约束
   - 投资与经济周期：加速器原理，I = α·ΔY
