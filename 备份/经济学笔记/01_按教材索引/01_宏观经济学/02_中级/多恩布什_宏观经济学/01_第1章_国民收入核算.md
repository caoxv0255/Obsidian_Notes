# 第1章：国民收入核算

> 本章学习目标：
> - 理解国内生产总值（GDP）
> - 掌握价格指数

---

## 1.1 国内生产总值（GDP）

### 1.1.1 GDP的定义

**国内生产总值（Gross Domestic Product, GDP）**：
一个国家在一定时期内生产的所有最终产品和服务的市场价值。

### 1.1.2 GDP的计算方法

**GDP的计算方法（Methods of Calculating GDP）**：

**1. 生产法（Production Approach）**：
```
GDP = 各部门增加值之和
增加值 = 总产出 - 中间投入
```

**2. 收入法（Income Approach）**：
```
GDP = 工资 + 利息 + 租金 + 利润 + 间接税 - 补贴 + 折旧
```

**3. 支出法（Expenditure Approach）**：
```
GDP = C + I + G + (X - M)
```
**其中**：
- C：消费
- I：投资
- G：政府购买
- X：出口
- M：进口

### 1.1.3 GDP与GNP

**GDP与GNP（GDP vs. GNP）**：

**1. 国内生产总值（GDP）**：
- 一个国家境内生产的最终产品和服务的市场价值
- 基于地域原则

**2. 国民生产总值（GNP）**：
- 一个国家国民生产的最终产品和服务的市场价值
- 基于国民原则

**关系**：
```
GNP = GDP + 来自国外的要素收入净额
```

## 1.2 价格指数

### 1.2.1 价格指数的定义

**价格指数（Price Index）**：
衡量价格水平变化的指标。

### 1.2.2 价格指数的类型

**价格指数的类型（Types of Price Index）**：

**1. 消费者价格指数（Consumer Price Index, CPI）**：
- 衡量消费者购买的一篮子商品和服务价格的变化
- 公式：CPI = (当前时期一篮子商品价格 / 基期一篮子商品价格) × 100

**2. 生产者价格指数（Producer Price Index, PPI）**：
- 衡量生产者出售的一篮子商品价格的变化
- 反映生产成本的变化

**3. GDP平减指数（GDP Deflator）**：
- 衡量GDP中所有商品和服务价格的变化
- 公式：GDP平减指数 = (名义GDP / 实际GDP) × 100

### 1.2.3 通货膨胀率

**通货膨胀率（Inflation Rate）**：
价格水平的增长率。

**公式**：
```
π = (Pₜ - Pₜ₋₁) / Pₜ₋₁
```
**其中**：
- π：通货膨胀率
- Pₜ：当前时期价格指数
- Pₜ₋₁：上一时期价格指数

## 1.3 失业

### 1.3.1 失业的定义

**失业（Unemployment）**：
有工作能力且有工作意愿，但没有工作的人。

### 1.3.2 失业率的定义

**失业率（Unemployment Rate）**：
失业人口占劳动力的百分比。

**公式**：
```
失业率 = (失业人口 / 劳动力) × 100%
```
**其中**：
- 劳动力 = 就业人口 + 失业人口

### 1.3.3 失业的类型

**失业的类型（Types of Unemployment）**：

**1. 摩擦性失业（Frictional Unemployment）**：
- 由于工作转换或初次求职造成的短期失业
- 例如：大学毕业生寻找工作

**2. 结构性失业（Structural Unemployment）**：
- 由于经济结构变化造成的长期失业
- 例如：产业衰退导致的失业

**3. 周期性失业（Cyclical Unemployment）**：
- 由于经济周期波动造成的失业
- 例如：经济衰退导致的失业

**4. 自然失业率（Natural Rate of Unemployment）**：
- 摩擦性失业和结构性失业的总和
- 经济处于充分就业时的失业率

## 1.4 名义变量与实际变量

### 1.4.1 名义变量

**名义变量（Nominal Variables）**：
用当年价格衡量的变量。

**例子**：
- 名义GDP
- 名义工资
- 名义利率

### 1.4.2 实际变量

**实际变量（Real Variables）**：
用基期价格衡量的变量。

**例子**：
- 实际GDP
- 实际工资
- 实际利率

### 1.4.3 名义变量与实际变量的关系

**名义变量与实际变量的关系（Nominal vs. Real Variables）**：
```
实际变量 = 名义变量 / 价格指数
```

**例子**：
- 实际GDP = 名义GDP / GDP平减指数
- 实际工资 = 名义工资 / CPI

## 1.5 本章总结

### 核心要点

1. **国内生产总值（GDP）**：
   - GDP：一个国家在一定时期内生产的所有最终产品和服务的市场价值
   - GDP的计算方法：生产法、收入法、支出法
   - GDP与GNP：GNP = GDP + 来自国外的要素收入净额

2. **价格指数**：
   - 价格指数：衡量价格水平变化的指标
   - 价格指数的类型：CPI、PPI、GDP平减指数
   - 通货膨胀率：π = (Pₜ - Pₜ₋₁) / Pₜ₋₁

3. **失业**：
   - 失业：有工作能力且有工作意愿，但没有工作的人
   - 失业率：失业率 = (失业人口 / 劳动力) × 100%
   - 失业的类型：摩擦性失业、结构性失业、周期性失业、自然失业率

4. **名义变量与实际变量**：
   - 名义变量：用当年价格衡量的变量
   - 实际变量：用基期价格衡量的变量
   - 名义变量与实际变量的关系：实际变量 = 名义变量 / 价格指数
