# 第2章：线性回归

## 本章学习目标

通过本章的学习，学生应该能够：
1. 理解线性回归模型的基本假设和数学表述
2. 掌握普通最小二乘法（OLS）的推导过程和性质
3. 熟悉参数估计量的统计性质和假设检验方法
4. 应用线性回归分析实际经济问题
5. 理解回归诊断和模型选择的基本原理

## 2.1 线性回归模型基础

### 2.1.1 基本定义

线性回归模型是计量经济学中最基础、最重要的分析工具之一。它描述了因变量与自变量之间的线性关系，并允许我们研究变量之间的因果联系。

**定义：线性回归模型**

一个标准的一元线性回归模型可以表示为：

$$y_t = \beta_0 + \beta_1 x_t + \varepsilon_t, \quad t = 1, 2, \ldots, n$$

其中：
- $y_t$ 是因变量（被解释变量）
- $x_t$ 是自变量（解释变量）
- $\beta_0$ 是截距项
- $\beta_1$ 是斜率系数
- $\varepsilon_t$ 是随机误差项
- $n$ 是样本容量

多元线性回归模型则可以表示为：

$$y_t = \beta_0 + \beta_1 x_{1t} + \beta_2 x_{2t} + \cdots + \beta_k x_{kt} + \varepsilon_t$$

用矩阵形式表示：

$$\mathbf{y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}$$

其中：
- $\mathbf{y} = (y_1, y_2, \ldots, y_n)'$ 是 $n \times 1$ 的因变量向量
- $\mathbf{X}$ 是 $n \times (k+1)$ 的设计矩阵，第一列全为1
- $\boldsymbol{\beta} = (\beta_0, \beta_1, \ldots, \beta_k)'$ 是 $(k+1) \times 1$ 的参数向量
- $\boldsymbol{\varepsilon} = (\varepsilon_1, \varepsilon_2, \ldots, \varepsilon_n)'$ 是 $n \times 1$ 的误差向量

### 2.1.2 经典假设

为了获得良好的估计性质，线性回归模型通常基于以下经典假设：

**假设1：线性性**
回归模型对参数是线性的。这意味着变量之间的函数关系可以通过参数的线性组合来表示。

**假设2：严格外生性**
误差项的条件期望为零：
$$E[\varepsilon_t|\mathbf{X}] = 0, \quad \forall t$$

这等价于：
$$E[\varepsilon_t] = 0$$
$$Cov(\varepsilon_t, x_{is}) = 0, \quad \forall t, s$$

**假设3：同方差性**
误差项的方差是常数：
$$Var(\varepsilon_t|\mathbf{X}) = \sigma^2, \quad \forall t$$

**假设4：无自相关**
误差项之间不存在序列相关：
$$Cov(\varepsilon_t, \varepsilon_s) = 0, \quad t \neq s$$

**假设5：无多重共线性**
设计矩阵 $\mathbf{X}$ 是满秩的，即 $rank(\mathbf{X}) = k+1$。这意味着解释变量之间不存在完全的线性关系。

**假设6：正态性**
误差项服从正态分布：
$$\boldsymbol{\varepsilon}|\mathbf{X} \sim N(\mathbf{0}, \sigma^2\mathbf{I}_n)$$

### 2.1.3 图形分析

为了直观理解线性回归，我们可以考虑一元回归的情况：

```
y
│
│         •
│      •     •
│   •           •
│•              •
└────────────────── x
```

在这个图形中：
- 黑点表示观测数据点 $(x_t, y_t)$
- 直线表示回归直线 $\hat{y}_t = \hat{\beta}_0 + \hat{\beta}_1 x_t$
- 每个点到直线的垂直距离代表残差 $\hat{\varepsilon}_t = y_t - \hat{y}_t$

### 2.1.4 例子

**消费函数示例**

假设我们研究消费 $C_t$ 与收入 $Y_t$ 之间的关系。一个简单的凯恩斯消费函数可以表示为：

$$C_t = \alpha + \beta Y_t + \varepsilon_t$$

其中：
- $\alpha$ 是自主消费（不依赖于收入的基本消费）
- $\beta$ 是边际消费倾向（MPC），表示每增加一单位收入所增加的消费
- $\varepsilon_t$ 代表其他影响消费的随机因素

在实证分析中，我们收集一组家庭的消费和收入数据，通过线性回归估计 $\alpha$ 和 $\beta$。如果估计结果为 $\hat{\alpha} = 5000$，$\hat{\beta} = 0.75$，这意味着：
- 基本消费为5000元
- 收入每增加1元，消费平均增加0.75元

## 2.2 最小二乘法

### 2.2.1 最小二乘估计的推导

普通最小二乘法（Ordinary Least Squares, OLS）是估计线性回归模型参数最常用的方法。其基本思想是选择使残差平方和最小的参数估计值。

**定义：残差平方和（RSS）**

$$RSS(\boldsymbol{\beta}) = \sum_{t=1}^{n} \hat{\varepsilon}_t^2 = \sum_{t=1}^{n} (y_t - \mathbf{x}_t'\boldsymbol{\beta})^2$$

用矩阵形式表示：

$$RSS(\boldsymbol{\beta}) = (\mathbf{y} - \mathbf{X}\boldsymbol{\beta})'(\mathbf{y} - \mathbf{X}\boldsymbol{\beta})$$

**最小化问题**

我们通过求解以下最小化问题获得OLS估计量：

$$\min_{\boldsymbol{\beta}} RSS(\boldsymbol{\beta})$$

**一阶条件**

对 $\boldsymbol{\beta}$ 求导并令导数为零：

$$\frac{\partial RSS}{\partial \boldsymbol{\beta}} = -2\mathbf{X}'(\mathbf{y} - \mathbf{X}\boldsymbol{\beta}) = \mathbf{0}$$

整理得到：

$$\mathbf{X}'\mathbf{X}\boldsymbol{\beta} = \mathbf{X}'\mathbf{y}$$

**正规方程组**

这被称为正规方程组。假设 $\mathbf{X}'\mathbf{X}$ 可逆（无多重共线性），我们得到OLS估计量：

$$\hat{\boldsymbol{\beta}}_{OLS} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$$

### 2.2.2 一元回归的特殊形式

对于一元线性回归 $y_t = \beta_0 + \beta_1 x_t + \varepsilon_t$，OLS估计量有显式表达式：

$$\hat{\beta}_1 = \frac{\sum_{t=1}^{n}(x_t - \bar{x})(y_t - \bar{y})}{\sum_{t=1}^{n}(x_t - \bar{x})^2}$$

$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}$$

其中 $\bar{x}$ 和 $\bar{y}$ 分别是样本均值。

### 2.2.3 几何解释

从几何角度看，OLS估计可以理解为将因变量向量 $\mathbf{y}$ 投影到由 $\mathbf{X}$ 的列向量张成的空间上。

```
y
│
│     ε̂ (残差向量)
│    ╱
│   ╱
│  ╱
│ ●→ ŷ = Xβ̂ (拟合值向量)
│ ╱
│╱
└─────────────────
    Span(X) (X的列空间)
```

在这个几何解释中：
- $\hat{\mathbf{y}} = \mathbf{X}\hat{\boldsymbol{\beta}}$ 是 $\mathbf{y}$ 在 $\mathbf{X}$ 的列空间上的投影
- $\hat{\boldsymbol{\varepsilon}} = \mathbf{y} - \hat{\mathbf{y}}$ 是残差向量，与 $\mathbf{X}$ 的所有列向量正交
- 这就是正交性条件 $\mathbf{X}'\hat{\boldsymbol{\varepsilon}} = \mathbf{0}$ 的几何含义

### 2.2.4 例子

**工资方程示例**

考虑一个工资回归模型：

$$\ln(wage_t) = \beta_0 + \beta_1 education_t + \beta_2 experience_t + \beta_3 experience_t^2 + \varepsilon_t$$

其中：
- $wage_t$ 是工资
- $education_t$ 是受教育年限
- $experience_t$ 是工作经验年限

假设我们有以下简化数据：

| 个体 | education | experience | wage |
|------|-----------|------------|------|
| 1    | 12        | 5          | 30000 |
| 2    | 16        | 2          | 35000 |
| 3    | 14        | 8          | 40000 |

设计矩阵 $\mathbf{X}$ 和因变量向量 $\mathbf{y}$ 分别为：

$$\mathbf{X} = \begin{bmatrix}
1 & 12 & 5 & 25 \\
1 & 16 & 2 & 4 \\
1 & 14 & 8 & 64
\end{bmatrix}, \quad \mathbf{y} = \begin{bmatrix}
\ln(30000) \\
\ln(35000) \\
\ln(40000)
\end{bmatrix}$$

通过计算 $(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$，我们可以得到参数估计值 $\hat{\boldsymbol{\beta}}$。

## 2.3 参数估计与统计推断

### 2.3.1 OLS估计量的性质

在经典假设下，OLS估计量具有以下重要性质：

**性质1：无偏性**

$$E[\hat{\boldsymbol{\beta}}|\mathbf{X}] = \boldsymbol{\beta}$$

**证明**：
$$\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'(\mathbf{X}\boldsymbol{\beta} + \boldsymbol{\varepsilon}) = \boldsymbol{\beta} + (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\boldsymbol{\varepsilon}$$

$$E[\hat{\boldsymbol{\beta}}|\mathbf{X}] = \boldsymbol{\beta} + (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'E[\boldsymbol{\varepsilon}|\mathbf{X}] = \boldsymbol{\beta} + \mathbf{0} = \boldsymbol{\beta}$$

**性质2：有效性（BLUE性质）**

在所有线性无偏估计量中，OLS估计量具有最小的方差。这被称为高斯-马尔可夫定理。

**性质3：一致性**

当样本容量 $n \to \infty$ 时，$\hat{\boldsymbol{\beta}} \xrightarrow{p} \boldsymbol{\beta}$。

**性质4：渐近正态性**

在适当条件下，$\sqrt{n}(\hat{\boldsymbol{\beta}} - \boldsymbol{\beta}) \xrightarrow{d} N(\mathbf{0}, \mathbf{Q}^{-1}\sigma^2)$，其中 $\mathbf{Q} = \lim_{n\to\infty} \frac{1}{n}\mathbf{X}'\mathbf{X}$。

### 2.3.2 方差-协方差矩阵

**定义：参数估计量的方差-协方差矩阵**

$$Var(\hat{\boldsymbol{\beta}}|\mathbf{X}) = E[(\hat{\boldsymbol{\beta}} - \boldsymbol{\beta})(\hat{\boldsymbol{\beta}} - \boldsymbol{\beta})'|\mathbf{X}] = \sigma^2(\mathbf{X}'\mathbf{X})^{-1}$$

**证明**：
$$\hat{\boldsymbol{\beta}} - \boldsymbol{\beta} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\boldsymbol{\varepsilon}$$

$$Var(\hat{\boldsymbol{\beta}}|\mathbf{X}) = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'Var(\boldsymbol{\varepsilon}|\mathbf{X})\mathbf{X}(\mathbf{X}'\mathbf{X})^{-1} = \sigma^2(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{X}(\mathbf{X}'\mathbf{X})^{-1} = \sigma^2(\mathbf{X}'\mathbf{X})^{-1}$$

### 2.3.3 误差方差的估计

误差方差 $\sigma^2$ 是未知的，需要估计：

**定义：残差**

$$\hat{\boldsymbol{\varepsilon}} = \mathbf{y} - \mathbf{X}\hat{\boldsymbol{\beta}}$$

**定义：残差平方和**

$$RSS = \hat{\boldsymbol{\varepsilon}}'\hat{\boldsymbol{\varepsilon}}$$

**定义：误差方差的无偏估计量**

$$\hat{\sigma}^2 = \frac{RSS}{n - k - 1}$$

其中 $n - k - 1$ 是自由度。

### 2.3.4 假设检验

**单个系数的t检验**

原假设：$H_0: \beta_j = 0$
备择假设：$H_1: \beta_j \neq 0$

检验统计量：

$$t_j = \frac{\hat{\beta}_j}{se(\hat{\beta}_j)} = \frac{\hat{\beta}_j}{\sqrt{\hat{\sigma}^2[(\mathbf{X}'\mathbf{X})^{-1}]_{jj}}}$$

在原假设成立时，$t_j \sim t(n - k - 1)$。

**置信区间**

$\beta_j$ 的 $100(1-\alpha)\%$ 置信区间为：

$$\hat{\beta}_j \pm t_{\alpha/2}(n - k - 1) \cdot se(\hat{\beta}_j)$$

**联合检验（F检验）**

原假设：$H_0: \mathbf{R}\boldsymbol{\beta} = \mathbf{r}$
备择假设：$H_1: \mathbf{R}\boldsymbol{\beta} \neq \mathbf{r}$

其中 $\mathbf{R}$ 是 $q \times (k+1)$ 的约束矩阵，$\mathbf{r}$ 是 $q \times 1$ 的约束向量。

F检验统计量：

$$F = \frac{(RSS_R - RSS_U)/q}{RSS_U/(n - k - 1)}$$

其中 $RSS_R$ 是受限模型的残差平方和，$RSS_U$ 是非受限模型的残差平方和。

在原假设成立时，$F \sim F(q, n - k - 1)$。

### 2.3.5 图形分析

**t检验的拒绝域**

```
       拒绝域          接受域           拒绝域
    ←─────────|────────────────────────|─────────→
      -t_α/2        0              t_α/2
```

如果 $|t_j| > t_{\alpha/2}$，则拒绝原假设，认为 $\beta_j$ 显著不为零。

### 2.3.6 例子

**菲利普斯曲线检验**

考虑菲利普斯曲线回归：

$$\pi_t = \beta_0 + \beta_1 u_t + \varepsilon_t$$

其中：
- $\pi_t$ 是通货膨胀率
- $u_t$ 是失业率

假设估计结果为：
- $\hat{\beta}_1 = -0.5$
- $se(\hat{\beta}_1) = 0.15$
- 样本容量 $n = 50$

检验边际消费倾向是否显著为负：

**假设**：
$H_0: \beta_1 = 0$
$H_1: \beta_1 < 0$

**检验统计量**：
$$t = \frac{-0.5}{0.15} = -3.33$$

**临界值**：
在5%显著性水平下，单侧检验的临界值为 $t_{0.05}(48) \approx -1.68$

**结论**：
由于 $-3.33 < -1.68$，我们拒绝原假设，认为失业率对通货膨胀率有显著的负向影响，这符合传统的菲利普斯曲线理论。

## 2.4 拟合优度与模型选择

### 2.4.1 决定系数R²

**定义：总平方和（TSS）**

$$TSS = \sum_{t=1}^{n}(y_t - \bar{y})^2 = \mathbf{y}'\mathbf{y} - n\bar{y}^2$$

**定义：回归平方和（ESS）**

$$ESS = \sum_{t=1}^{n}(\hat{y}_t - \bar{y})^2 = \hat{\boldsymbol{\beta}}'\mathbf{X}'\mathbf{y} - n\bar{y}^2$$

**定义：残差平方和（RSS）**

$$RSS = \sum_{t=1}^{n}\hat{\varepsilon}_t^2 = \mathbf{y}'\mathbf{y} - \hat{\boldsymbol{\beta}}'\mathbf{X}'\mathbf{y}$$

**平方和分解**

$$TSS = ESS + RSS$$

**决定系数R²**

$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}$$

$R^2$ 表示因变量的变动中可以被解释变量解释的比例，取值范围是 $[0, 1]$。

### 2.4.2 调整后的决定系数

**问题**：$R^2$ 随着解释变量数量的增加而单调递增，即使增加的变量没有解释力。

**解决方法**：调整后的决定系数

$$\bar{R}^2 = 1 - \frac{RSS/(n - k - 1)}{TSS/(n - 1)} = 1 - (1 - R^2)\frac{n - 1}{n - k - 1}$$

$\bar{R}^2$ 考虑了模型自由度，只有在增加的变量对模型有足够的解释力时才会增加。

### 2.4.3 信息准则

**赤池信息准则（AIC）**

$$AIC = \ln(\hat{\sigma}^2) + \frac{2(k+1)}{n}$$

**贝叶斯信息准则（BIC）**

$$BIC = \ln(\hat{\sigma}^2) + \frac{(k+1)\ln(n)}{n}$$

信息准则的值越小，模型越好。BIC对过度参数化的惩罚比AIC更严厉。

### 2.4.4 图形分析

**R²的几何解释**

```
│
│     y
│    ╱│╲
│   ╱ │ ╲
│  ╱  │  ╲
│ ╱   │   ╲
│╱____│____╲
│    ŷ      ε̂
└────────────
```

$R^2$ 表示 $\hat{\mathbf{y}}$ 的长度在 $\mathbf{y} - \bar{y}\mathbf{1}$ 的长度中所占的比例：
$$R^2 = \frac{||\hat{\mathbf{y}} - \bar{y}\mathbf{1}||^2}{||\mathbf{y} - \bar{y}\mathbf{1}||^2}$$

### 2.4.5 例子

**生产函数估计**

考虑柯布-道格拉斯生产函数的对数形式：

$$\ln(Y_t) = \beta_0 + \beta_1\ln(K_t) + \beta_2\ln(L_t) + \beta_3\ln(K_t)\ln(L_t) + \varepsilon_t$$

假设我们比较两个模型：

**模型1**（仅包含主效应）：
$$\ln(Y_t) = \alpha_0 + \alpha_1\ln(K_t) + \alpha_2\ln(L_t) + u_t$$

估计结果：
- $R^2 = 0.85$
- $\bar{R}^2 = 0.84$
- $AIC = -2.45$
- $BIC = -2.30$

**模型2**（包含交互效应）：
$$\ln(Y_t) = \beta_0 + \beta_1\ln(K_t) + \beta_2\ln(L_t) + \beta_3\ln(K_t)\ln(L_t) + \varepsilon_t$$

估计结果：
- $R^2 = 0.86$
- $\bar{R}^2 = 0.85$
- $AIC = -2.50$
- $BIC = -2.28$

**比较**：
- $R^2$：模型2更高（0.86 > 0.85）
- $\bar{R}^2$：模型2更高（0.85 > 0.84）
- $AIC$：模型2更低（-2.50 < -2.45），模型2更好
- $BIC$：模型1更低（-2.30 < -2.28），模型1更好

**结论**：
根据AIC，模型2更好；但根据BIC，模型1更好。由于BIC对过度参数化的惩罚更严厉，如果样本量不大，可能会倾向于选择模型1。

## 2.5 回归诊断与异方差

### 2.5.1 残差分析

**定义：残差**

$$\hat{\varepsilon}_t = y_t - \hat{y}_t = y_t - \mathbf{x}_t'\hat{\boldsymbol{\beta}}$$

**标准化残差**

$$r_t = \frac{\hat{\varepsilon}_t}{\hat{\sigma}\sqrt{1 - h_{tt}}}$$

其中 $h_{tt}$ 是帽子矩阵 $\mathbf{H} = \mathbf{X}(\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'$ 的第 $t$ 个对角元素。

**残差图诊断**

1. **残差vs拟合值图**：检查异方差和模型设定错误
2. **残差vs时间图**：检查自相关
3. **Q-Q图**：检查正态性

### 2.5.2 异方差的检验与处理

**定义：异方差**

误差项的方差不是常数，而是随观测值变化：
$$Var(\varepsilon_t|\mathbf{X}) = \sigma_t^2 \neq \sigma^2$$

**异方差的后果**
- OLS估计量仍然无偏，但不再是有效的
- 标准误估计有偏，导致t检验和F检验不可靠

**White检验**

检验原假设：$H_0: \sigma_t^2 = \sigma^2$（同方差）

辅助回归：
$$\hat{\varepsilon}_t^2 = \gamma_0 + \sum_{i=1}^{k}\gamma_i x_{it} + \sum_{i=1}^{k}\sum_{j=i}^{k}\delta_{ij}x_{it}x_{jt} + v_t$$

检验统计量：$LM = n \cdot R_{aux}^2 \xrightarrow{d} \chi^2(q)$

其中 $R_{aux}^2$ 是辅助回归的决定系数，$q$ 是辅助回归中参数的个数。

**异方差稳健标准误（White标准误）**

$$\hat{Var}_{robust}(\hat{\boldsymbol{\beta}}) = (\mathbf{X}'\mathbf{X})^{-1}(\sum_{t=1}^{n}\hat{\varepsilon}_t^2\mathbf{x}_t\mathbf{x}_t')(\mathbf{X}'\mathbf{X})^{-1}$$

**加权最小二乘法（WLS）**

如果知道异方差的形式，可以使用WLS。假设 $Var(\varepsilon_t|\mathbf{X}) = \sigma^2 h_t$，其中 $h_t$ 已知。

转换模型：
$$\frac{y_t}{\sqrt{h_t}} = \frac{\beta_0}{\sqrt{h_t}} + \beta_1\frac{x_{1t}}{\sqrt{h_t}} + \cdots + \beta_k\frac{x_{kt}}{\sqrt{h_t}} + \frac{\varepsilon_t}{\sqrt{h_t}}$$

转换后的误差项 $\varepsilon_t/\sqrt{h_t}$ 是同方差的，可以使用OLS。

### 2.5.3 图形分析

**同方差情况**

```
ε̂
│  •    •    •
│ •   •     •  •
│•  •   •   •   •
└────────────────── ŷ
```

残差在拟合值周围均匀分布，没有明显的模式。

**异方差情况**

```
ε̂
│         • • •
│       ••   •
│    ••      •
│  •
└────────────────── ŷ
```

残差的离散程度随拟合值增加而增大，呈现漏斗形状。

### 2.5.4 例子

**房价回归的异方差**

考虑房价回归模型：

$$\ln(price_t) = \beta_0 + \beta_1\ln(sqft_t) + \beta_2\ln(lotsize_t) + \beta_3 bedroom_t + \varepsilon_t$$

假设我们进行White检验，得到 $LM = 24.5$，而 $\chi^2(9)$ 在5%显著性水平下的临界值为16.92。

**结论**：$24.5 > 16.92$，拒绝同方差假设，存在异方差。

**处理方法**：使用异方差稳健标准误重新计算t统计量。

**原标准误**（假设同方差）：
- $\hat{\beta}_1 = 0.75$, $se = 0.08$, $t = 9.38$

**稳健标准误**（考虑异方差）：
- $\hat{\beta}_1 = 0.75$, $se_{robust} = 0.12$, $t_{robust} = 6.25$

虽然t统计量变小了，但仍然显著，说明估计结果是稳健的。

## 本章总结

### 核心要点

1. **线性回归模型**是研究变量之间因果关系的基础工具，其核心是 $y_t = \mathbf{x}_t'\boldsymbol{\beta} + \varepsilon_t$

2. **最小二乘法**通过最小化残差平方和得到参数估计量，在经典假设下具有无偏性、有效性等优良性质

3. **统计推断**基于参数估计量的渐近正态性，可以使用t检验、F检验等方法进行假设检验

4. **模型评估**需要综合考虑拟合优度（$R^2$、$\bar{R}^2$）和信息准则（AIC、BIC）

5. **回归诊断**包括残差分析、异方差检验等，确保模型的假设条件得到满足

### 重要概念表格

| 概念 | 定义/公式 | 经济学意义 |
|------|-----------|------------|
| OLS估计量 | $\hat{\boldsymbol{\beta}} = (\mathbf{X}'\mathbf{X})^{-1}\mathbf{X}'\mathbf{y}$ | 使残差平方和最小的参数估计 |
| R² | $R^2 = 1 - \frac{RSS}{TSS}$ | 模型的拟合优度 |
| t统计量 | $t_j = \hat{\beta}_j / se(\hat{\beta}_j)$ | 检验单个系数显著性 |
| F统计量 | $F = \frac{(RSS_R - RSS_U)/q}{RSS_U/(n - k - 1)}$ | 检验多个系数的联合显著性 |
| 异方差 | $Var(\varepsilon_t|\mathbf{X}) \neq$ 常数 | 误差项方差不是常数 |
| AIC | $AIC = \ln(\hat{\sigma}^2) + \frac{2(k+1)}{n}$ | 模型选择的信息准则 |
| BIC | $BIC = \ln(\hat{\sigma}^2) + \frac{(k+1)\ln(n)}{n}$ | 对过度参数化惩罚更严厉的准则 |
