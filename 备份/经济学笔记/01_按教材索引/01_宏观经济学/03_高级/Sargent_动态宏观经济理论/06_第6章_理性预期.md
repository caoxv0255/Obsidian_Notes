# 第6章：理性预期

## 本章学习目标

通过本章的学习，学生应该能够：
1. 理解理性预期的基本假设和经济学意义
2. 掌握理性预期均衡的概念和求解方法
3. 熟悉理性预期模型的设定和分析
4. 理解理性预期对宏观经济政策的影响
5. 应用理性预期理论分析实际经济问题

## 6.1 理性预期假设

### 6.1.1 预期形成理论的发展

**适应性预期**

在理性预期理论出现之前，主流的预期形成理论是适应性预期：

$$E_t[y_{t+1}] = y_t + \alpha(y_t - E_{t-1}[y_t])$$

或者：
$$E_t[y_{t+1}] = \sum_{j=0}^{\infty} \lambda (1-\lambda)^j y_{t-j}$$

其中 $\alpha, \lambda \in (0, 1)$ 是调整速度参数。

**特点**：
- 预期只依赖于过去的实际值
- 调整是渐进的，可能存在系统性误差
- 不考虑经济结构和政策变化

**局限性**：
- 如果经济结构或政策变化，适应性预期会持续产生系统性误差
- 不符合最优化行为：经济主体应该利用所有可用信息

### 6.1.2 理性预期的定义

**定义：理性预期**

经济主体对变量 $x_{t+1}$ 在时期 $t$ 的预期 $E_t[x_{t+1}]$ 是理性的，如果：

$$E_t[x_{t+1}] = E[x_{t+1} | I_t]$$

其中 $I_t$ 是时期 $t$ 可用信息集。

**等价表述**：

1. **条件期望形式**：预期等于给定可用信息的条件期望
2. **无偏性**：$E_t[x_{t+1}] = E[x_{t+1} | I_t]$ 意味着预测是无偏的
3. **正交性**：预测误差与可用信息正交：$E[(x_{t+1} - E_t[x_{t+1}]) \cdot z] = 0$ 对于所有 $z \in I_t$
4. **有效性**：预测误差是不可预测的（白噪声）

**预测误差的性质**：

$$\varepsilon_{t+1} = x_{t+1} - E_t[x_{t+1}]$$

理性预期假设意味着：
1. $E_t[\varepsilon_{t+1}] = 0$（零均值）
2. $Cov(\varepsilon_{t+1}, I_t) = 0$（与历史信息不相关）
3. $\{\varepsilon_t\}$ 是鞅差分序列

### 6.1.3 理性预期的经济学含义

**含义1：信息利用**

理性预期的经济主体利用所有相关信息形成预期，包括：
- 历史数据
- 经济结构和参数
- 政策规则和未来政策预期

**含义2：无系统性误差**

理性预期不会产生系统性预测误差。如果存在系统性误差，经济主体会调整预期形成过程以消除这些误差。

**含义3：市场有效性**

在金融市场中，理性预期意味着资产价格反映了所有可用信息（有效市场假说）。

**含义4：政策含义**

理性预期对政策有重要含义（卢卡斯批评、政策无效性命题等）。

### 6.1.4 信息集的定义

**完全信息**

如果信息集包含所有相关的经济变量：

$$I_t = \{x_{t-j}, j = 0, 1, 2, \ldots\}$$

**有限信息**

如果信息集只包含部分变量：

$$I_t = \{y_{t-j}, z_{t-j}, j = 0, 1, \ldots, k\}$$

**信息集的性质**

1. **嵌套性**：$I_t \subseteq I_{t+1}$（信息随时间累积）
2. **过滤**：$\{\mathcal{F}_t\}$ 是一个过滤，其中 $\mathcal{F}_t = \sigma(I_t)$ 是由 $I_t$ 生成的 $\sigma$-代数
3. **适应性**：预测规则适应新信息

### 6.1.5 图形分析

**适应性预期 vs 理性预期**

```
实际值 yₜ
│
│        ••••••••••••••••••••••••••••••••
│      •    理性预期
│     •
│    •
│   •      适应性预期（滞后）
│  •
│ •
│•
└─────────────────────────────────────────► t
```

- 理性预期紧随实际值，没有系统性滞后
- 适应性预期可能滞后于实际值

**理性预期的预测误差**

```
预测误差 εₜ
│
│     •    •   •     •
│    • •  • • •    •
│   •   ••   •   •  •
│  •         • •    •
│ •          •
│•
└─────────────────────────────────────────► t
```

理性预期的预测误差是白噪声，没有可预测的模式。

### 6.1.6 例子

**通货膨胀预期**

假设实际通货膨胀过程为：

$$\pi_t = \alpha + \beta \pi_{t-1} + \varepsilon_t$$

其中 $\varepsilon_t \sim N(0, \sigma^2)$ 是白噪声。

**适应性预期**：
$$E_t^{AE}[\pi_{t+1}] = \pi_t + \gamma(\pi_t - E_{t-1}[\pi_t])$$

如果 $\gamma = 0.5$，预期调整缓慢。

**理性预期**：
$$E_t^{RE}[\pi_{t+1}] = E[\alpha + \beta \pi_t + \varepsilon_{t+1} | I_t] = \alpha + \beta \pi_t$$

因为 $E_t[\varepsilon_{t+1}] = 0$。

**比较**：

如果 $\beta = 0.8$，$\pi_t = 3\%$，$\pi_{t-1} = 2\%$，$\alpha = 0.5\%$：

- 适应性预期（假设 $E_{t-1}[\pi_t] = 2.5\%$）：
  $$E_t^{AE}[\pi_{t+1}] = 3\% + 0.5 \times (3\% - 2.5\%) = 3.25\%$$

- 理性预期：
  $$E_t^{RE}[\pi_{t+1}] = 0.5\% + 0.8 \times 3\% = 2.9\%$$

理性预期更准确地利用了通货膨胀过程的参数信息。

## 6.2 理性预期均衡

### 6.2.1 基本设定

**定义：理性预期均衡（REE）**

在理性预期均衡中，经济主体的预期与经济模型的预测一致：

$$E_t[x_{t+1}] = E[x_{t+1} | I_t, \text{模型参数}]$$

**要素**：

1. **经济结构**：描述经济变量关系的方程组
2. **预期形成**：理性预期假设
3. **均衡条件**：市场出清或最优化行为

### 6.2.2 简单模型：菲利普斯曲线

**传统菲利普斯曲线**：

$$\pi_t = \pi_t^e - \beta(u_t - u^n) + \varepsilon_t$$

其中：
- $\pi_t$ 是实际通货膨胀
- $\pi_t^e$ 是预期通货膨胀
- $u_t$ 是失业率
- $u^n$ 是自然失业率
- $\varepsilon_t$ 是供给冲击

**理性预期下的菲利普斯曲线**：

$$\pi_t = E_{t-1}[\pi_t] - \beta(u_t - u^n) + \varepsilon_t$$

**含义**：

对两边取条件期望 $E_{t-1}[\cdot]$：

$$E_{t-1}[\pi_t] = E_{t-1}[\pi_t] - \beta E_{t-1}[u_t - u^n] + E_{t-1}[\varepsilon_t]$$

因为 $E_{t-1}[\varepsilon_t] = 0$，所以：

$$\beta E_{t-1}[u_t - u^n] = 0$$

这意味着在理性预期下，系统性货币政策（改变 $E_{t-1}[u_t]$）无法影响实际通货膨胀。

### 6.2.3 Cagan超通胀模型

**模型设定**：

货币需求函数：
$$\frac{M_t}{P_t} = e^{-\alpha \pi_t^e}$$

其中：
- $M_t$ 是名义货币供给
- $P_t$ 是价格水平
- $\pi_t^e = E_t[\pi_{t+1}]$ 是预期通货膨胀
- $\alpha > 0$ 是货币需求的半弹性

货币供给增长：
$$M_{t+1} = (1 + \mu_t) M_t$$

**理性预期均衡**：

取对数：
$$m_t - p_t = -\alpha \pi_t^e$$

其中小写字母表示对数。

因为 $\pi_t^e = p_{t+1} - p_t$，所以：
$$m_t - p_t = -\alpha (p_{t+1} - p_t)$$

$$p_t = \frac{1}{1+\alpha} m_t + \frac{\alpha}{1+\alpha} p_{t+1}$$

向前迭代：
$$p_t = \frac{1}{1+\alpha} \sum_{j=0}^{\infty} \left(\frac{\alpha}{1+\alpha}\right)^j E_t[m_{t+j}]$$

如果货币供给增长率 $\mu_t = \bar{\mu}$ 是常数：
$$E_t[m_{t+j}] = m_t + j \ln(1+\bar{\mu})$$

$$p_t = \frac{1}{1+\alpha} m_t + \frac{\alpha}{1+\alpha} \sum_{j=1}^{\infty} \left(\frac{\alpha}{1+\alpha}\right)^j [m_t + j \ln(1+\bar{\mu})]$$

简化得到：
$$p_t = m_t + \alpha \ln(1+\bar{\mu})$$

实际货币余额：
$$m_t - p_t = -\alpha \ln(1+\bar{\mu})$$

通货膨胀：
$$\pi_t = \ln(1+\bar{\mu})$$

**含义**：

- 货币是中性的：货币供给只影响价格水平，不影响实际变量
- 预期通货膨胀等于实际货币增长率
- 超通胀情况下，货币需求急剧下降

### 6.2.4 线性理性预期模型

**一般形式**：

$$\mathbf{A}_0 \mathbf{x}_t = \mathbf{A}_1 \mathbf{x}_{t-1} + \mathbf{B}_0 \mathbf{E}_t[\mathbf{x}_{t+1}] + \mathbf{C}_0 \mathbf{z}_t$$

其中：
- $\mathbf{x}_t$ 是内生变量向量
- $\mathbf{z}_t$ 是外生变量向量
- $\mathbf{A}_0, \mathbf{A}_1, \mathbf{B}_0, \mathbf{C}_0$ 是参数矩阵

**求解方法（Blanchard-Kahn方法）**：

1. **重写为预期差分形式**：
   $$\mathbf{x}_t = \mathbf{M}_0 \mathbf{x}_{t-1} + \mathbf{M}_1 \mathbf{E}_t[\mathbf{x}_{t+1}] + \mathbf{M}_2 \mathbf{z}_t$$

2. **假设解的形式**：
   $$\mathbf{x}_t = \mathbf{P} \mathbf{x}_{t-1} + \mathbf{Q} \mathbf{z}_t$$

3. **将假设解代入**：
   $$\mathbf{P} \mathbf{x}_{t-1} + \mathbf{Q} \mathbf{z}_t = \mathbf{M}_0 \mathbf{x}_{t-1} + \mathbf{M}_1 \mathbf{P}^2 \mathbf{x}_{t-1} + \mathbf{M}_1 \mathbf{P} \mathbf{Q} \mathbf{z}_t + \mathbf{M}_2 \mathbf{z}_t$$

4. **匹配系数**：
   $$\mathbf{P} = \mathbf{M}_0 + \mathbf{M}_1 \mathbf{P}^2$$
   $$\mathbf{Q} = \mathbf{M}_1 \mathbf{P} \mathbf{Q} + \mathbf{M}_2$$

5. **求解矩阵二次方程**：
   将 $\mathbf{P}$ 的方程重新排列：
   $$\mathbf{M}_1 \mathbf{P}^2 - \mathbf{P} + \mathbf{M}_0 = \mathbf{0}$$

   这个矩阵二次方程的解取决于特征值的分布。

### 6.2.5 稳定性条件

**Blanchard-Kahn条件**：

对于线性理性预期模型，存在唯一稳定解的条件是：

不稳定特征值的数量等于预期变量（前定变量）的数量。

**前定变量 vs 跳跃变量**：

- **前定变量**：由过去决定，不能跳跃调整（如资本存量）
- **跳跃变量**：可以自由调整（如价格、汇率）

**稳定性分析**：

设 $\mathbf{M}_1$ 有 $n$ 个特征值 $\lambda_1, \lambda_2, \ldots, \lambda_n$。

- 如果 $|\lambda_i| > 1$，特征值是不稳定的（爆炸的）
- 如果 $|\lambda_i| < 1$，特征值是稳定的

**唯一均衡的条件**：

不稳定特征值的数量 = 跳跃变量的数量

如果：
- 不稳定特征值 > 跳跃变量：无解（系统爆炸）
- 不稳定特征值 < 跳跃变量：无限多解（过度决定）

### 6.2.6 图形分析

**理性预期均衡的确定**

```
变量 xₜ
│
│   稳定路径
│    •
│   ╱│╲
│  ╱ │ ╲
│ ╱  │  ╲
│╱   │   ╲
└────┴────► 时间 t
     │
    x*
```

- 稳定路径通向稳态
- 不稳定路径发散
- 理性预期选择稳定路径

**Blanchard-Kahn条件**

```
复平面
│
│   |λ| > 1 (不稳定)
│    ••••••••••••••
│   •           •
│  •  稳定      •
│ •  区域      •
│•           •
│••••••••••••••
│   |λ| < 1 (稳定)
└──────────────────► 实部
     │
     │
```

- 单位圆内：稳定特征值
- 单位圆外：不稳定特征值

### 6.2.7 例子

**简单的货币模型**

模型方程：
$$y_t = E_t[y_{t+1}] - (i_t - E_t[\pi_{t+1}]) + \varepsilon_t$$
$$\pi_t = \beta E_t[\pi_{t+1}] + \kappa y_t + \eta_t$$
$$i_t = \phi_\pi \pi_t + \phi_y y_t$$

其中：
- $y_t$ 是产出缺口
- $\pi_t$ 是通货膨胀
- $i_t$ 是名义利率
- $\varepsilon_t, \eta_t$ 是冲击

**矩阵形式**：

设 $\mathbf{x}_t = (y_t, \pi_t)'$，$\mathbf{z}_t = (\varepsilon_t, \eta_t)'$：

$$\mathbf{x}_t = \mathbf{A} E_t[\mathbf{x}_{t+1}] + \mathbf{B} \mathbf{z}_t$$

其中：
$$\mathbf{A} = \begin{bmatrix}
0 & 1 \\
\kappa & \beta
\end{bmatrix}$$

**特征值计算**：

特征方程：$|\lambda \mathbf{I} - \mathbf{A}| = 0$

$$\begin{vmatrix}
\lambda & -1 \\
-\kappa & \lambda - \beta
\end{vmatrix} = \lambda(\lambda - \beta) - \kappa = 0$$

$$\lambda^2 - \beta \lambda - \kappa = 0$$

$$\lambda = \frac{\beta \pm \sqrt{\beta^2 + 4\kappa}}{2}$$

**稳定性分析**：

假设 $\beta = 0.99$, $\kappa = 0.1$：

$$\lambda_1 = \frac{0.99 + \sqrt{0.99^2 + 0.4}}{2} = \frac{0.99 + 1.18}{2} = 1.09$$
$$\lambda_2 = \frac{0.99 - 1.18}{2} = -0.10$$

- $\lambda_1 = 1.09 > 1$：不稳定
- $\lambda_2 = -0.10 < 1$：稳定

**唯一均衡条件**：

因为 $y_t$ 和 $\pi_t$ 都是跳跃变量（可以自由调整），跳跃变量数量 = 2，不稳定特征值数量 = 1。

这不符合唯一均衡条件（不稳定特征值应该等于跳跃变量数量）。这意味着需要额外的约束（如泰勒原理）。

**泰勒原理**：

如果 $\phi_\pi > 1$，系统可以稳定。

## 6.3 理性预期与政策

### 6.3.1 卢卡斯批评

**内容**：

传统的政策评估（如使用传统计量模型）是不可靠的，因为：
1. 政策变化会改变经济主体的行为
2. 传统的估计参数依赖于政策规则
3. 当政策变化时，这些参数也会变化

**例子**：

假设我们估计一个菲利普斯曲线：
$$\pi_t = \alpha + \beta u_t + \varepsilon_t$$

如果我们使用这个方程来评估政策效果（如通过增加失业来降低通货膨胀），这是不可靠的，因为：
- 如果政策变化，经济主体会调整预期
- 预期变化会影响实际通货膨胀
- 估计的参数 $\beta$ 依赖于政策环境

**启示**：

政策评估应该：
1. 考虑经济主体的最优化行为
2. 分析政策如何影响预期
3. 使用结构参数（深参数）而非简化形式参数

### 6.3.2 政策无效性命题

**Sargent-Wallace政策无效性命题**：

在理性预期下，系统性货币政策无法影响实际变量。

**证明思路**：

考虑一个简单的总供给模型：
$$y_t = y_t^n + \alpha(\pi_t - E_{t-1}[\pi_t]) + \varepsilon_t$$

其中 $y_t^n$ 是自然产出水平。

货币政策规则：
$$m_t = \bar{m} + \gamma y_{t-1} + \eta_t$$

预期通货膨胀：
$$E_{t-1}[\pi_t] = E_{t-1}[m_t] - E_{t-1}[m_{t-1}] = \gamma E_{t-1}[y_{t-1}] + E_{t-1}[\eta_t] = \gamma y_{t-1}$$

实际通货膨胀：
$$\pi_t = m_t - m_{t-1} = \gamma y_{t-1} + \eta_t - \gamma y_{t-2} - \eta_{t-1}$$

产出缺口：
$$y_t - y_t^n = \alpha(\pi_t - E_{t-1}[\pi_t]) + \varepsilon_t = \alpha(\gamma y_{t-1} + \eta_t - \gamma y_{t-1} - \eta_{t-1} - \gamma y_{t-1}) + \varepsilon_t$$
$$= \alpha(\eta_t - \eta_{t-1} - \gamma y_{t-1}) + \varepsilon_t$$

**含义**：

系统性政策（$\gamma \neq 0$）无法预期地影响产出，只有未预期到的冲击（$\eta_t, \varepsilon_t$）影响产出。

### 6.3.3 政策承诺与时间不一致性

**问题设定**：

政府在时期 $t$ 选择政策 $a_t$ 最大化：

$$W_t = \sum_{s=t}^{\infty} \beta^{s-t} L(x_s, a_s)$$

其中 $x_s$ 是经济状态，$L$ 是损失函数。

**Kydland-Prescott时间不一致性**：

1. **时期 $t$（承诺）**：政府承诺未来的政策路径 $\{a_{t+1}, a_{t+2}, \ldots\}$
2. **时期 $t+1$（重新优化）**：政府面对新的状态 $x_{t+1}$，可能偏离之前的承诺

**原因**：

- 政府在每个时期都重新最优化
- 承诺不是可信的，因为政府有激励偏离

**解决方法**：

1. **规则 vs 相机抉择**：遵循预先设定的规则而非相机抉择
2. **声誉机制**：建立不偏离承诺的声誉
3. **制度设计**：设计独立的央行等制度

### 6.3.4 最优货币政策规则

**泰勒规则**：

$$i_t = \bar{r} + \pi_t + \phi_\pi(\pi_t - \pi^*) + \phi_y(y_t - y^n)$$

其中：
- $i_t$ 是名义利率
- $\bar{r}$ 是实际利率
- $\pi^*$ 是目标通货膨胀
- $\phi_\pi, \phi_y$ 是政策参数

**最优参数**：

理性预期模型表明，最优泰勒规则应该满足：
- $\phi_\pi > 1$（泰勒原理）：当通货膨胀超过目标时，利率提高幅度超过通货膨胀上升幅度
- $\phi_y > 0$：当产出为正时，提高利率

**含义**：

- 泰勒原理确保系统稳定（满足Blanchard-Kahn条件）
- 系统性政策可以稳定通货膨胀和产出

### 6.3.5 图形分析

**政策无效性**

```
产出 yₜ
│
│     自然率
│       ••••••••••••••••••••••••••••••••••
│      •
│     • 政策冲击
│    •  (仅未预期部分有效)
│   •
│  •
│ •
│•
└─────────────────────────────────────────► t
```

系统性政策无法影响产出，只有未预期到的冲击有效。

**时间不一致性**

```
时间 t            t+1
│                │
│  承诺 a*       │  重新优化
│      ↓         │      ↓
│  福利 W*       │  福利 W' > W*
│                │  但承诺不可信
└────────────────┴──────────►
```

政府在 $t+1$ 期有激励偏离承诺，导致时间不一致性。

### 6.3.6 例子

**最优货币规则**

考虑一个简化的新凯恩斯模型：

$$\pi_t = \beta E_t[\pi_{t+1}] + \kappa y_t + \varepsilon_t$$
$$y_t = E_t[y_{t+1}] - (i_t - E_t[\pi_{t+1}]) + \eta_t$$

央行损失函数：
$$L = \frac{1}{2} \sum_{t=0}^{\infty} \beta^t (\pi_t^2 + \lambda y_t^2)$$

**最优政策**：

理性预期均衡下，最优政策满足：

$$\pi_t = -\frac{\lambda}{\kappa} y_t$$

代入总供给方程：
$$-\frac{\lambda}{\kappa} y_t = \beta E_t\left[-\frac{\lambda}{\kappa} y_{t+1}\right] + \kappa y_t + \varepsilon_t$$

$$y_t = \beta E_t[y_{t+1}] - \frac{\kappa^2}{\lambda} y_t - \frac{\kappa}{\lambda} \varepsilon_t$$

**泰勒规则**：

最优利率规则：
$$i_t = r^* + \pi_t + \frac{\lambda + \kappa^2}{\kappa} \pi_t$$

这对应于 $\phi_\pi = 1 + \frac{\lambda + \kappa^2}{\kappa} > 1$，满足泰勒原理。

**含义**：

- 产出和通货膨胀的权衡（$\pi = -\frac{\lambda}{\kappa} y$）
- 较高的产出稳定权重（$\lambda$）导致对产出波动的容忍度降低
- 泰勒原理确保系统稳定

## 6.4 理性预期的检验

### 6.4.1 检验方法

**方法1：正交性检验**

检验预测误差是否与可用信息正交：

$$\varepsilon_{t+1} = x_{t+1} - E_t[x_{t+1}]$$

回归：
$$\varepsilon_{t+1} = \alpha + \mathbf{\beta}' \mathbf{z}_t + u_{t+1}$$

其中 $\mathbf{z}_t$ 是信息集中的变量。

理性预期假设：
$$H_0: \alpha = 0, \mathbf{\beta} = \mathbf{0}$$

**方法2：无偏性检验**

检验预期是否无偏：

回归：
$$x_{t+1} = \alpha + \beta E_t[x_{t+1}] + \varepsilon_{t+1}$$

理性预期假设：
$$H_0: \alpha = 0, \beta = 1$$

**方法3：效率检验**

检验是否利用了所有可用信息：

比较两个预期：
$$E_t^{RE}[x_{t+1}] \text{ vs } E_t^{AE}[x_{t+1}]$$

如果理性预期更准确，则 $Var(x_{t+1} - E_t^{RE}[x_{t+1}]) \leq Var(x_{t+1} - E_t^{AE}[x_{t+1}])$

### 6.4.2 估计问题

**内生性问题**：

预期通常不可直接观测，需要使用调查数据或模型估计。

**解决方法**：

1. **调查数据**：使用调查问卷获得的预期数据
2. **模型估计**：从模型推导理性预期
3. **工具变量**：使用工具变量解决内生性

### 6.4.3 经验证据

**金融市场**：

理性预期在金融市场得到较强支持：
- 资产价格反映所有可用信息
- 超额收益不可预测（大部分情况下）

**通货膨胀预期**：

证据混合：
- 部分研究表明预期是理性的
- 部分研究发现系统性偏差

**消费者和企业预期**：

证据较弱：
- 预期往往表现出滞后性
- 存在系统性预测误差

### 6.4.4 局限性

**局限性1：信息成本**

获取和处理信息有成本，理性预期假设经济主体免费使用所有信息。

**局限性2：认知限制**

经济主体的认知能力有限，可能无法准确计算复杂模型的预测。

**局限性3：学习过程**

经济主体需要时间学习经济结构，预期可能逐渐收敛到理性预期。

**局限性4：异质性预期**

不同经济主体可能有不同的信息集和预期形成过程。

### 6.4.5 图形分析

**理性预期检验：无偏性**

```
实际值 x_{t+1}
│
│     β = 1 (理性预期)
│      ↗
│     ↗  实际回归
│    ↗    β < 1
│   ↗
│  ↗
│ ↗
│↗
└────────────────────► 预期 E_t[x_{t+1}]
```

如果实际回归斜率等于1，支持理性预期。

**正交性检验**

```
预测误差 ε_{t+1}
│
│     应该无系统模式
│    •  •   •
│   •    •   •
│  •  •    •
│ •   •  •
│•  •   •
└────────────────────► 信息变量 zₜ
```

如果预测误差与信息变量无关，支持理性预期。

### 6.4.6 例子

**检验通货膨胀预期的理性**

使用美国通货膨胀数据和调查预期数据：

回归方程：
$$\pi_{t+1} = \alpha + \beta E_t[\pi_{t+1}] + \varepsilon_{t+1}$$

**假设检验**：
$$H_0: \alpha = 0, \beta = 1$$

**回归结果**（示例）：
$$\pi_{t+1} = 0.002 + 0.85 E_t[\pi_{t+1}] + \hat{\varepsilon}_{t+1}$$
$$(0.001) (0.08)$$

**统计检验**：

检验 $H_0: \beta = 1$：
$$t = \frac{0.85 - 1}{0.08} = -1.875$$

在5%显著性水平下，临界值为 $\pm 1.96$，所以不拒绝 $H_0$。

检验 $H_0: \alpha = 0$：
$$t = \frac{0.002}{0.001} = 2$$

在5%显著性水平下，拒绝 $H_0$，预期有系统性正偏差。

**结论**：

混合证据：
- 斜率接近1，支持无偏性
- 截距显著为正，存在系统性高估偏差

## 本章总结

### 核心要点

1. **理性预期假设**认为经济主体的预期等于给定所有可用信息的条件期望，没有系统性预测误差

2. **理性预期均衡**要求预期与经济模型的预测一致，这通常需要求解预期的差分方程

3. **卢卡斯批评**指出传统的政策评估方法不可靠，因为政策变化会改变经济行为

4. **政策无效性命题**表明在理性预期下，系统性政策无法影响实际变量，只有未预期到的冲击有效

5. **时间不一致性**是承诺问题的重要来源，需要通过规则或制度设计解决

### 重要概念表格

| 概念 | 定义/公式 | 经济学意义 |
|------|-----------|------------|
| 理性预期 | $E_t[x_{t+1}] = E[x_{t+1} | I_t]$ | 预期等于条件期望 |
| 适应性预期 | $E_t[y_{t+1}] = y_t + \alpha(y_t - E_{t-1}[y_t])$ | 基于过去预测的渐进调整 |
| 预测误差 | $\varepsilon_{t+1} = x_{t+1} - E_t[x_{t+1}]$ | 预期偏差 |
| 理性预期均衡 | 预期与模型预测一致 | 理论的自洽性 |
| 卢卡斯批评 | 政策变化改变行为参数 | 政策评估需考虑结构 |
| 政策无效性 | 系统性政策不影响实际变量 | 只有未预期冲击有效 |
| 时间不一致性 | 承诺与重新优化的冲突 | 需要制度约束 |
| 泰勒规则 | $i_t = r + \pi_t + \phi_\pi(\pi_t - \pi^*)$ | 系统性货币政策 |
| Blanchard-Kahn条件 | 不稳定特征值=跳跃变量数量 | 唯一均衡条件 |
| 泰勒原理 | $\phi_\pi > 1$ | 稳定性条件 |
