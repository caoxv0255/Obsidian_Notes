## 8.1 货币政策基本框架

### 8.1.1 货币政策的目标

**主要目标：**

1. **价格稳定**
   - 维持低而稳定的通货膨胀率
   - 避免通货紧缩
   - 典型目标：通胀率2%左右

2. **充分就业**
   - 最大化可持续就业
   - 接近自然失业率
   - 避免周期性失业

3. **经济增长**
   - 支持经济可持续增长
   - 减少经济波动
   - 维持潜在增长率

4. **金融稳定**
   - 防范系统性风险
   - 维护金融体系稳定
   - 避免资产泡沫

**目标冲突：**

短期内，不同目标之间可能存在权衡：
```
菲利普斯曲线（短期）：
失业率 u_t
  │
  │        PC
  │      ／
  │    ／
  │  ／
  │／
  └────────────────── 通货膨胀率 π_t

低通胀 ↔ 高失业
高通胀 ↔ 低失业
```

**时间不一致性问题：**

承诺 vs 执行的冲突：
- 承诺：低通胀，建立信誉
- 执行：短期扩张性政策，推高通胀

### 8.1.2 货币政策工具

**1. 政策利率（主要工具）：**

联邦基金利率（美国）、政策利率（中国）、再融资利率（欧元区）

作用机制：
```
中央银行调整政策利率
    ↓
市场利率跟随调整
    ↓
影响消费、投资决策
    ↓
影响总需求和通胀
```

**2. 公开市场操作（OMO）：**

买入国债 → 增加基础货币 → 降低利率
卖出国债 → 减少基础货币 → 提高利率

**公式：**
```
货币乘数：
M = m × B

其中：
M：货币供应量
m：货币乘数
B：基础货币
```

**3. 存款准备金率：**

提高准备金率 → 减少货币创造 → 紧缩性政策
降低准备金率 → 增加货币创造 → 扩张性政策

**货币乘数与准备金率：**
```
m = 1/rr

其中 rr 为准备金率
```

**4. 贴现窗口/再贷款利率：**

贴现率影响银行借款成本：
- 高贴现率 → 银行借款减少 → 货币供应紧缩
- 低贴现率 → 银行借款增加 → 货币供应扩张

**5. 非传统工具（非常规政策）：**

量化宽松（QE）：
- 大规模购买长期国债和资产支持证券
- 直接影响长期利率
- 增加基础货币

前瞻性指引：
- 公布未来政策利率路径
- 影响预期
- 增强政策透明度

**图形分析：传统vs非传统货币政策**

```
利率曲线
期限
  │
  │    ── 传统政策（调整短期利率）
  │   ／
  │  ／
  │ ／
  │／
  └───────────
    短期

  │    ── QE政策（压低长期利率）
  │  ／
  │ ／
  │／
  └───────────
    长期
```

### 8.1.3 中央银行的独立性

**独立性维度：**

1. **目标独立性**
   - 中央银行自行设定目标
   - 独立于政府决策

2. **工具独立性**
   - 中央银行自由选择政策工具
   - 不受政治干预

**独立性与通胀率的关系：**

实证规律：
```
独立性指数
高         低
  │         │
  │    ╱╲   │
  │   ╱  ╲  │
  │  ╱    ╲ │
  │ ╱      ╲│
  └─────────┴─────
低通胀   高通胀
```

**好处：**
- 降低平均通胀率
- 减少通胀波动
- 提高政策可信度
- 增强经济稳定

**潜在问题：**
- 民主赤字（缺乏问责）
- 与财政政策协调困难
- 可能忽视其他目标（如就业）

## 8.2 货币政策理论模型

### 8.2.1 经典货币数量论

**费雪方程式：**

```
MV = PY
```

其中：
- M：货币供应量
- V：货币流通速度
- P：价格水平
- Y：实际产出

**货币需求函数：**

剑桥方程式：
```
M^d = k × P × Y
```

其中 k 为货币需求比例（k = 1/V）

**古典二分法：**

```
实际部门：由生产函数、要素市场决定
    ↓
Y, w, r, n（实际变量）

货币部门：由货币供求决定
    ↓
P, M, V（名义变量）
```

**长期中性：**

货币供应量变化只影响价格水平，不影响实际变量：
```
ΔM/M = ΔP/P（长期）

实际变量不变：ΔY/Y = 0, Δw/w = 0, Δr/r = 0
```

**图形分析：货币数量论**

```
价格水平 P
  │
  │    MV = PY (给定V, Y)
  │   ／
  │  ／
  │ ／
  │／
  └────────────────── 货币供应量 M

M↑ → P↑（同比例）
```

**例子：货币扩张的效果**

```
初始状态：
M = 1000, V = 5, Y = 500
P = MV/Y = 1000×5/500 = 10

货币扩张20%：
M' = 1200, V = 5, Y = 500
P' = MV/Y = 1200×5/500 = 12

结论：
ΔP/P = (12-10)/10 = 20% = ΔM/M
```

### 8.2.2 IS-LM模型

**模型框架：**

IS曲线（商品市场均衡）：
```
Y = C(Y - T) + I(r) + G
```

LM曲线（货币市场均衡）：
```
M/P = L(Y, r) = kY - hr
```

其中：
- Y：产出
- C：消费
- I：投资
- G：政府支出
- T：税收
- r：利率
- M：货币供应
- P：价格水平
- k：收入货币需求系数
- h：利率货币需求系数

**IS曲线推导：**

从Y = C(Y-T) + I(r) + G

投资函数：I = I₀ - br（b > 0）

消费函数：C = c(Y-T)（c为边际消费倾向）

得到IS曲线方程：
```
Y = c(Y-T) + I₀ - br + G
Y = [1/(1-c)] [I₀ - br + G - cT]
```

斜率：dY/dr = -b/(1-c) < 0

**图形分析：IS曲线**

```
利率 r
  │
  │   IS
  │  ／
  │ ／  斜率 = -b/(1-c)
  │／
  └────────────────── 产出 Y
```

**LM曲线推导：**

从M/P = kY - hr

解出利率：
```
r = (kY - M/P)/h
```

斜率：dr/dY = k/h > 0

**图形分析：LM曲线**

```
利率 r
  │
  │    LM
  │   ／
  │  ／  斜率 = k/h
  │ ／
  │／
  └────────────────── 产出 Y
```

**IS-LM均衡：**

图形分析：

```
利率 r
  │
  │    LM
  │   ／
  │  ／    E (均衡点)
  │ ／    ╱
  │／    ╱
  ╱─────╱─── IS
  ╱    ╱
 └────────────────── 产出 Y
```

均衡条件：
```
IS: Y = [1/(1-c)] [I₀ - br + G - cT]
LM: r = (kY - M/P)/h
```

联立求解得到均衡产出Y*和均衡利率r*

### 8.2.3 货币政策在IS-LM模型中的效应

**扩张性货币政策：**

货币供应量M增加 → LM曲线右移

图形分析：

```
利率 r
  │    LM (原位置)
  │   ／
  │  ／
  │ ／
  │／
  ╱────── LM' (右移)
  ╱     ／
  ╱    ╱
  ╱E  ╱E' (新均衡)
  ╱──╱── IS
 └────────────────── 产出 Y
    Y*   Y'
```

效果：
- 利率下降：r ↓
- 产出增加：Y ↑
- 投资增加（由于利率下降）

**传导机制：**
```
M ↑ → LM右移 → r ↓ → I ↑ → Y ↑
```

**数学推导：**

假设：
- M增加ΔM
- 从LM曲线：r = (kY - M/P)/h
- 从IS曲线：Y = [1/(1-c)] [I₀ - br + G - cT]

将r代入IS：
```
Y = [1/(1-c)] [I₀ - b(kY - M/P)/h + G - cT]

整理：
Y(1 - c + bk/h) = (I₀ + G - cT) + (b/h)(M/P)

解出Y：
Y = [(I₀ + G - cT) + (b/h)(M/P)] / (1 - c + bk/h)

对M求导：
dY/dM = [b/(hP)] / (1 - c + bk/h) = b / [hP(1-c) + bk]

货币乘数（产出对货币供应的弹性）：
k_m = dY/dM = b / [hP(1-c) + bk]

影响因素：
- 投资对利率的敏感度b越大 → k_m越大
- 货币需求对利率的敏感度h越大 → k_m越小
- 边际消费倾向c越大 → k_m越小

**例子：货币政策乘数计算**

```
给定参数：
b = 100（投资对利率敏感度）
h = 50（货币需求对利率敏感度）
P = 1
c = 0.8（边际消费倾向）
k = 0.5（收入货币需求系数）

货币乘数：
k_m = 100 / [50×1×(1-0.8) + 100×0.5]
     = 100 / [10 + 50]
     = 100/60
     ≈ 1.67

含义：货币供应增加100单位，产出增加167单位
```

**紧缩性货币政策：**

货币供应量M减少 → LM曲线左移

效果：
- 利率上升：r ↑
- 产出减少：Y ↓
- 投资减少

**流动性陷阱：**

当利率接近零下限时，LM曲线变为水平

图形分析：

```
利率 r
  │    LM
  │   ／
  │  ／
  │ ╱
  │╱────── 流动性陷阱区域（r ≈ 0）
  │
  └────────────────── 产出 Y
```

在流动性陷阱中：
- 货币政策无效
- LM曲线移动不影响产出
- 需要财政政策或其他非常规政策

**零利率下限（ZLB）约束：**

名义利率不能为负：
```
i_t = max{0, i_t^*}
```

其中 i_t^* 为理想利率

在ZLB环境下：
- 传统货币政策空间受限
- 需要非常规政策（QE、前瞻性指引）

### 8.2.4 动态IS-LM模型

**模型设定：**

动态IS曲线（从消费欧拉方程推导）：
```
y_t = E_t[y_{t+1}] - (1/σ)(i_t - E_t[π_{t+1}] - r_t^n)
```

其中：
- y_t：产出缺口
- i_t：名义利率
- π_t：通货膨胀率
- r_t^n：自然利率（均衡实际利率）
- σ：跨期替代弹性

动态LM曲线（货币需求）：
```
m_t - p_t = η y_t - γ i_t
```

其中：
- m_t：货币供应量（对数）
- p_t：价格水平（对数）
- η：收入弹性
- γ：利率弹性

**动态方程组：**

```
y_t = E_t[y_{t+1}] - (1/σ)(i_t - E_t[π_{t+1}] - r_t^n)
π_t = β E_t[π_{t+1}] + κ y_t + ε_t^π
m_t - p_t = η y_t - γ i_t
```

**图形分析：动态AD-AS模型**

```
通胀率π
  │
  │    AS (NKPC)
  │   ／
  │  ／    E (均衡)
  │ ／    ╱
  │／    ╱
  ╱─────╱─── AD (IS)
  ╱    ╱
 └────────────────── 产出 y
```

**动态调整过程：**

假设正向需求冲击：

```
时期 t=0: 需求冲击 → y_t ↑
         ↓
时期 t=1: 通胀上升 → 实际利率下降（如果名义利率不变）
         ↓
时期 t=2: 中央银行提高名义利率 → y_t 下降
         ↓
时期 t=3: 通胀逐渐回落 → 经济回归均衡
```

**图形分析：动态调整路径**

```
产出缺口 y
  │
  │        ┌───
  │      ／
  │    ／
  │  ／
  │／
  └──────────────────── 时间 t
    0  1  2  3  4  5

通胀率 π
  │
  │          ┌───
  │        ／
  │      ／
  │    ／
  │  ／
  │／
  └──────────────────── 时间 t
```

## 8.3 货币政策规则

### 8.3.1 货币政策规则的类型

**1. 货币供应量规则：**

弗里德曼的k%规则：
```
ΔM/M = k（常数增长率）
```

优点：
- 规则简单，透明
- 避免政策随意性
- 控制长期通胀

缺点：
- 忽视经济状况变化
- 货币流通速度不稳定
- 金融创新影响货币需求

**例子：货币供应量增长规则**

```
假设：
- 长期经济增长率 g = 3%
- 货币流通速度增长率 g_V = 0%
- 目标通胀率 π* = 2%

货币供应量增长率：
ΔM/M = g + π* + g_V = 3% + 2% + 0% = 5%

每年货币供应量增长5%
```

**2. 名义收入规则：**

规则形式：
```
i_t = φ_π (π_t - π*) + φ_y (y_t - y_t^*)
```

其中 y_t^* 为潜在产出增长路径

优点：
- 同时关注通胀和产出
- 自动稳定器作用

**3. 泰勒规则（Taylor Rule）：**

Taylor（1993）提出：
```
i_t = r̄ + π_t + 0.5(π_t - π*) + 0.5(y_t - ȳ_t)
```

其中：
- i_t：联邦基金利率
- r̄：均衡实际利率（约2%）
- π_t：当前通胀率
- π*：目标通胀率（约2%）
- y_t：实际GDP增长率
- ȳ_t：潜在GDP增长率

**一般形式：**
```
i_t = ρ + φ_π π_t + φ_y (y_t - ȳ_t) + ε_t
```

其中：
- ρ = r̄ + π*：稳态名义利率
- φ_π > 1：泰勒原则
- φ_y：对产出缺口的反应系数

**图形分析：泰勒规则**

```
政策利率 i
  │
  │    ╱    (φ_π = 1.5)
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── 通胀率 π
       π*

当 π > π* 时：i > π + r̄（实际利率上升）
当 π < π* 时：i < π + r̄（实际利率下降）
```

**泰勒原则：**

φ_π > 1 是稳定的必要条件

含义：
- 通胀上升时，名义利率上升幅度超过通胀上升幅度
- 实际利率上升，抑制通胀
- 形成稳定的均衡

如果 φ_π ≤ 1：
- 实际利率可能下降
- 通胀自我强化
- 不稳定均衡

**例子：泰勒规则的应用**

```
假设：
- r̄ = 2%
- π* = 2%
- 当前通胀 π_t = 4%
- 产出缺口 y_t - ȳ_t = 2%

泰勒规则：
i_t = 2% + 4% + 0.5(4% - 2%) + 0.5(2%)
    = 2% + 4% + 1% + 1%
    = 8%

政策含义：
实际利率 = i_t - π_t = 8% - 4% = 4%
高于均衡实际利率（2%），紧缩性政策
```

**4. 通货膨胀目标制：**

规则形式：
```
i_t = ρ + φ_π (π_t - π*) + φ_y (y_t - ȳ_t)
```

特点：
- 明确宣布通胀目标
- 透明度高
- 灵活性（关注产出缺口）

**通胀目标制的实施框架：**

```
中央银行设定通胀目标（如2%）
    ↓
定期发布通胀预期和预测
    ↓
基于预测调整政策利率
    ↓
沟通策略增强可信度
    ↓
长期实现通胀稳定
```

**例子：通胀目标制国家的表现**

```
国家      目标通胀  实际通胀(1990-2020)
新西兰     1-3%        2.1%
英国       2%          2.2%
加拿大     2%          1.9%
瑞典       2%          1.8%
巴西       3-6%        5.2%（早期）
```

### 8.3.2 最优货币政策规则

**福利分析框架：**

社会损失函数：
```
L = E₀ Σ β^t [λ (π_t - π*)² + (y_t - ȳ_t)²]
```

其中：
- λ：通胀相对于产出的权重
- π*：目标通胀率（通常设为0）

中央银行选择政策规则来最小化损失函数

**约束条件：**

模型结构（NKPC和动态IS）：
```
π_t = β E_t[π_{t+1}] + κ y_t + ε_t^π
y_t = E_t[y_{t+1}] - (1/σ)(i_t - E_t[π_{t+1}] - r_t^n)
```

**最优解：**

通过动态规划求解

一阶条件：
```
y_t = - (κ/λ) (π_t - π*)
```

这是最优的权衡关系

**最优政策规则：**

最优利率规则形式：
```
i_t = r_t^n + π* + φ_π (π_t - π*) + φ_y (y_t - ȳ_t)
```

最优系数：
```
φ_π* = 1 + (λ/κ)(1 - ρ)
φ_y* = (λ/κ)
```

其中 ρ 为冲击的持续性

**解释：**
- 对通胀的反应系数 φ_π* > 1（满足泰勒原则）
- 权重 λ/κ 越大，对通胀的反应越强

**图形分析：最优政策反应**

```
政策利率 i
  │
  │    最优反应函数
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── 通胀缺口 π - π*
```

**例子：最优政策系数计算**

```
假设：
- 通胀权重 λ = 1
- 菲利普斯曲线斜率 κ = 0.1
- 冲击持续性 ρ = 0.8

最优系数：
φ_π* = 1 + (1/0.1)(1 - 0.8) = 1 + 2 = 3
φ_y* = 1/0.1 = 10

最优政策规则：
i_t = r_t^n + π* + 3(π_t - π*) + 10(y_t - ȳ_t)
```

### 8.3.3 通胀预期管理

**预期的重要性：**

预期影响实际通胀：
```
π_t = β E_t[π_{t+1}] + κ y_t + ε_t^π
```

如果E_t[π_{t+1}]高 → 当前通胀上升

**通胀锚定：**

中央银行通过沟通锚定通胀预期：
```
E_t[π_{t+1}] ≈ π*
```

锚定的好处：
- 降低通胀波动
- 减少实际产出损失
- 提高政策有效性

**前瞻性指引：**

形式：
1. **基于时间的指引：**
   - "政策利率将保持低位至少到2027年"

2. **基于状态的指引：**
   - "政策利率将保持低位，直到失业率低于6%且通胀稳定在2%"

3. **基于结果的指引：**
   - "政策利率将保持低位，直到通胀预期稳定在2%"

**图形分析：前瞻性指引的效果**

```
预期利率路径
  │
  │  ╱
  │ ╱    传统政策（无指引）
  │╱
  │
  │────── 前瞻性指引
  │
  └────────────────── 时间
    当前  未来
```

**例子：美联储的前瞻性指引（2012-2015）**

```
指引内容：
"预计极低的联邦基金利率水平至少会持续到2015年中期"

效果：
- 短期利率不变（已接近零）
- 长期利率下降（预期未来低利率）
- 刺激投资和消费
- 支持经济复苏
```

## 8.4 货币政策传导机制

### 8.4.1 利率渠道

**传统传导渠道：**

```
政策利率 i_t ↓
    ↓
市场利率 r_t ↓
    ↓
投资 I_t ↑ (成本降低)
    ↓
消费 C_t ↑ (跨期替代)
    ↓
总需求 AD_t ↑
    ↓
产出 Y_t ↑
```

**投资函数：**
```
I_t = I₀ - b r_t

其中 b > 0 为投资对利率的敏感度
```

**利率敏感度分析：**

投资对利率的弹性：
```
ε_I,r = (dI/I) / (dr/r) = -b r/I
```

影响弹性的因素：
- 投资项目的盈利能力
- 企业融资约束
- 经济不确定性

**例子：利率传导的量化分析**

```
假设：
- 投资函数：I = 1000 - 50r
- 初始利率：r = 6%
- 政策利率下降：Δr = -2%（从6%到4%）

投资变化：
ΔI = -50 × (-2%) = 100
新投资：I' = 1000 - 50×4% = 980
投资增长率：ΔI/I = 100/1000 = 10%

结论：利率下降2个百分点，投资增加10%
```

**长期利率的影响：**

预期理论：
```
R_t^n = (1/n) Σ E_t[i_{t+k}]
```

其中 R_t^n 为n期长期利率

政策利率变化影响预期 → 影响长期利率

**图形分析：收益率曲线的变动**

```
利率
  │
  │    扩张性政策后
  │   ／╲
  │  ／  ╲
  │ ╱    ╲
  │╱      ╲
  └────────────────── 期限
    短期    长期

短期利率大幅下降，长期利率小幅下降
```

### 8.4.2 信贷渠道

**1. 银行贷款渠道：**

传导机制：
```
政策利率 i_t ↓
    ↓
存款利率 ↓
    ↓
银行存款减少
    ↓
银行贷款供给减少（如果储备金约束）
    ↓
依赖银行贷款的企业投资减少
```

**银行资产负债表：**
```
资产：储备金 + 贷款 + 证券
负债：存款 + 其他负债

准备金要求：
储备金 ≥ rr × 存款
```

**图形分析：银行贷款渠道**

```
贷款供给
  │
  │   供给曲线
  │  ／
  │ ／
  │╱
  └────────────────── 贷款利率
    i

扩张性政策：
存款 ↑ → 贷款供给曲线右移 → 贷款量 ↑
```

**2. 资产负债表渠道：**

传导机制：
```
政策利率 i_t ↓
    ↓
资产价格 ↑
    ↓
企业净值 ↑
    ↓
抵押品价值 ↑
    ↓
融资约束放松
    ↓
投资 ↑
```

**外部融资溢价：**

```
i_t^ext = i_t + ρ_t

其中：
i_t：无风险利率
ρ_t：外部融资溢价

外部融资溢价函数：
ρ_t = f(企业净值) = ρ₀ - λ × 净值
```

净值越高 → 溢价越低

**图形分析：外部融资溢价**

```
溢价 ρ
  │
  │    ρ = ρ₀ - λ × 净值
  │   ╱
  │  ／
  │ ／
  │╱
  └────────────────── 企业净值
    低    高
```

**例子：资产负债表渠道的量化**

```
假设：
- 无风险利率：i = 5%
- 外部融资溢价函数：ρ = 10% - 0.5 × 净值
- 企业初始净值：10%

初始贷款利率：
i^ext = 5% + (10% - 0.5×10%) = 10%

资产价格上涨使净值增加到12%：
新外部融资溢价：ρ' = 10% - 0.5×12% = 4%
新贷款利率：i'^ext = 5% + 4% = 9%

贷款利率下降1个百分点 → 投资激励增强
```

### 8.4.3 资产价格渠道

**1. 托宾q理论：**

q比率定义：
```
q = 企业市场价值 / 资本重置成本
```

投资规则：
```
如果 q > 1 → 投资（购买资本比购买现有资产便宜）
如果 q < 1 → 不投资（购买现有资产更便宜）
```

传导机制：
```
政策利率 i_t ↓
    ↓
折现率 ↓
    ↓
股票价格 ↑
    ↓
q比率 ↑
    ↓
投资 I_t ↑
    ↓
产出 Y_t ↑
```

**q比率的公式：**
```
q = (1/1+r) × E[(收益 + 期末价值)/重置成本]
```

**图形分析：托宾q与投资**

```
投资 I
  │
  │
  │      I = f(q)
  │     ／
  │    ／
  │   ／
  │  ／
  │ ／
  │╱
  └────────────────── q比率
      0   1
```

**例子：托宾q效应**

```
假设：
- 资本重置成本：100
- 企业市场价值：150
- q = 150/100 = 1.5 > 1

含义：
- 投资回报率 > 资本成本
- 企业有激励投资

如果政策利率下降使市场价值增加到180：
q' = 180/100 = 1.8
投资激励进一步增强
```

**2. 财富效应：**

传导机制：
```
政策利率 i_t ↓
    ↓
资产价格（股票、房地产）↑
    ↓
家庭财富 ↑
    ↓
消费 C_t ↑（生命周期假说）
    ↓
产出 Y_t ↑
```

**消费函数（生命周期/持久收入假说）：**
```
C_t = α + β × W_t

其中：
W_t：总财富（人力资本 + 金融财富 + 实物财富）
β：边际消费倾向
```

**财富效应的大小：**
```
ΔC = β × ΔW

总财富变化：
ΔW = Δ金融财富 + Δ实物财富
```

**例子：财富效应**

```
假设：
- 家庭总财富：500,000
- 金融财富占比：30%
- 房地产占比：50%
- 边际消费倾向：β = 0.05

资产价格上涨10%：
金融财富变化：0.3 × 500,000 × 10% = 15,000
房地产财富变化：0.5 × 500,000 × 10% = 25,000
总财富变化：ΔW = 15,000 + 25,000 = 40,000

消费变化：
ΔC = 0.05 × 40,000 = 2,000

结论：资产价格上涨10%，消费增加2,000
```

**3. 汇率渠道：**

传导机制（开放经济）：
```
政策利率 i_t ↓
    ↓
本币汇率 e_t 贬值
    ↓
出口价格 ↓，进口价格 ↑
    ↓
净出口 NX_t ↑
    ↓
产出 Y_t ↑
```

**汇率决定（利率平价）：**

非抛补利率平价：
```
i_t = i*_t + E_t[Δe_{t+1}]
```

其中：
- i_t：本国利率
- i*_t：外国利率
- e_t：汇率（直接标价法，即一单位外币的本币价格）

**图形分析：汇率渠道**

```
汇率 e
  │
  │    本币贬值
  │   ／
  │  ／
  │ ╱
  │╱
  └────────────────── 利率差 i - i*
```

**出口和进口的反应：**

出口需求函数：
```
X_t = X_0 (e_t/P*_t)^(-η_x)
```

进口需求函数：
```
M_t = M_0 (e_t/P_t)^(η_m)
```

其中 η_x 和 η_m 为弹性

**例子：汇率传导**

```
假设：
- 初始汇率：e = 6.5 人民币/美元
- 政策利率下降：Δi = -2%
- 利率平价：Δe = 2%（本币贬值）
- 出口弹性：η_x = 0.5

新汇率：e' = 6.5 × 1.02 = 6.63

出口变化：
ΔX/X = -0.5 × (6.63/6.5 - 1) ≈ -0.5 × 0.02 = -0.01
（等价于出口增加1%）

结论：本币贬值2%，出口增加1%

```
### 8.5.2 案例分析：美联储的货币政策

**案例1：2008年金融危机**

政策演变：
```
2007年：政策利率5.25%（正常环境）
2008年12月：政策利率降至0-0.25%（零利率下限）
2008-2014：量化宽松（QE1, QE2, QE3）
2013-2014：缩减购债（Taper Tantrum）
2015年底：开始加息
2017-2019：缩表
2020年3月：降至零利率 + 无限QE
