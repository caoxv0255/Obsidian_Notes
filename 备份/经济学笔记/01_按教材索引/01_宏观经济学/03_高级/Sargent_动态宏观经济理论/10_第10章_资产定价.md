## 10.1 资产定价基本原理

### 10.1.1 资产定价的基本概念

**资产的定义：**

资产是未来收益流的权益主张。资产定价的核心问题是：如何确定未来不确定收益流的当前价值。

**基本定价公式：**

```
P₀ = E₀ [Σ_{t=1}^T (D_t / (1 + r)^t)]

其中：
P₀：资产当前价格
D_t：t期的收益（分红、利息等）
r：折现率
E₀：期望算子（基于当前信息）
```

**图形分析：资产定价原理**

```
时间线
  │
  │  t=0   t=1   t=2   t=3   ...
  │   │     │     │     │
  │   │     │     │     │
  │  P₀    D₁    D₂    D₃
  │  ↓     ↓     ↓     ↓
  │  └─────────────────┘
  │         未来收益的折现和
```

**折现率的组成：**

```
r = r_f + RP

其中：
r_f：无风险利率
RP：风险溢价
```

**风险溢价的来源：**

1. **时间偏好：**
   - 当前消费 vs 未来消费
   - 等待的补偿

2. **风险厌恶：**
   - 不确定性厌恶
   - 风险的补偿

3. **流动性：**
   - 流动性差异
   - 流动性溢价

4. **税收：**
   - 税收待遇差异
   - 税收溢价

### 10.1.2 无套利原理

**定义：**

无套利原理指在均衡状态下，不存在任何零投资、零风险的正收益机会。

**套利的定义：**

套利是同时买入和卖出相关资产，以从价格差异中获利的策略。

**套利的类型：**

1. **空间套利：**
   - 同一资产在不同市场有不同价格
   - 买入低价市场，卖出高价市场

2. **时间套利：**
   - 当前价格与未来预期价格不一致
   - 买入当前，卖出未来

3. **统计套利：**
   - 基于统计关系的套利策略
   - 需要考虑风险

**无套利条件：**

```
如果存在套利机会：
P₀ ≠ Σ_{t=1}^T E₀[D_t] / (1 + r)^t

均衡时：
P₀ = Σ_{t=1}^T E₀[D_t] / (1 + r)^t
```

**例子：无套利定价**

```
假设：
- 资产A：当前价格100，一年后确定收益110
- 无风险利率：r_f = 5%

检查套利：
无风险债券价格 = 110 / (1 + 0.05) ≈ 104.76

资产A价格（100）< 债券价格（104.76）

套利策略：
1. 卖空债券，获得104.76
2. 买入资产A，支付100
3. 套利利润 = 104.76 - 100 = 4.76

均衡时：
资产A价格必须上升到104.76
```

**图形分析：无套利区域**

```
价格
  │
  │    无套利区域
  │   ╱
  │  ╱
  │ ╱
  │╱────── 公允价格
  │
  │    套利机会（被高估）
  │
  └──────────────────
```

### 10.1.3 效用与风险厌恶

**效用函数：**

投资者基于期望效用进行决策：
```
E[U(W)] = Σ p_i U(W_i)

其中：
W：财富
p_i：状态概率
U：效用函数
```

**风险厌恶：**

定义：U''(W) < 0

风险厌恶投资者：
- 偏好确定性收益
- 愿意支付风险溢价以避免风险

**风险厌恶的度量：**

**绝对风险厌恶：**
```
ARA(W) = -U''(W) / U'(W)
```

**相对风险厌恶：**
```
RRA(W) = -W U''(W) / U'(W)
```

**常见效用函数：**

1. **二次效用：**
   ```
   U(W) = aW - bW² (b > 0)

   特点：
   - 风险厌恶随财富增加而增加
   - 存在财富饱和点
   ```

2. **CRRA效用（常相对风险厌恶）：**
   ```
   U(W) = W^(1-γ) / (1-γ) (γ > 0, γ ≠ 1)

   特点：
   - 相对风险厌恶系数为常数γ
   - 适用于增长资产
   ```

3. **CARA效用（常绝对风险厌恶）：**
   ```
   U(W) = -exp(-aW) (a > 0)

   特点：
   - 绝对风险厌恶系数为常数a
   - 适用于固定投资组合
   ```

**图形分析：风险厌恶效用函数**

```
效用 U
  │
  │    ┌─── 确定性效用
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── 财富 W

期望效用：
E[U] = 0.5 × U(90) + 0.5 × U(110)

确定性等价 CE：
U(CE) = E[U]

风险溢价 RP：
RP = E[W] - CE

图形：
   U
   │
   │         ╱
   │        ╱  E[U]
   │       ╱
   │      ╱
   │     ╱
   │    ╱
   │   ╱
   │  ╱
   │ ╱
   │╱
   └──────────────── W
     90   CE 100  110
         ↑    ↑
      确定性等价 期望财富
```

**例子：风险溢价计算**

```
假设：
- 投资者初始财富：100
- 投资：50%概率获得120，50%概率获得80
- CRRA效用，γ = 2

期望财富：
E[W] = 0.5 × 120 + 0.5 × 80 = 100

期望效用：
U(W) = W^(-1)
E[U] = 0.5 × (120)^(-1) + 0.5 × (80)^(-1)
     = 0.5 × 0.00833 + 0.5 × 0.0125
     = 0.00417 + 0.00625
     = 0.01042

确定性等价 CE：
U(CE) = E[U]
(CE)^(-1) = 0.01042
CE = 1 / 0.01042 ≈ 95.97

风险溢价：
RP = E[W] - CE = 100 - 95.97 = 4.03

含义：投资者愿意放弃4.03的期望收益以避免风险
```

## 10.2 欧拉方程与资产定价

### 10.2.1 消费资产定价模型

**基本框架：**

代表性家庭最优化问题：

```
max E₀ Σ β^t U(c_t)

s.t. c_t + P_t a_{t+1} = Y_t + a_t D_t
```

其中：
- c_t：消费
- a_t：资产持有量
- P_t：资产价格
- D_t：资产收益
- Y_t：劳动收入
- β：主观折现因子

**一阶条件（欧拉方程）：**

从消费和资产的边际效用相等得到：
```
U'(c_t) = β E_t[U'(c_{t+1}) (D_{t+1} + P_{t+1}) / P_t]
```

整理得到定价公式：
```
P_t = E_t[β (U'(c_{t+1}) / U'(c_t)) (D_{t+1} + P_{t+1})]
```

**随机折现因子（SDF）：**

定义随机折现因子（或定价核）：
```
M_{t,t+1} = β (U'(c_{t+1}) / U'(c_t))
```

定价公式简化为：
```
P_t = E_t[M_{t,t+1} (D_{t+1} + P_{t+1})]
```

**无风险利率：**

对于无风险资产（收益 = 1 + r_f）：
```
1 = E_t[M_{t,t+1} (1 + r_f)]

因此：
r_f = (1 / E_t[M_{t,t+1}]) - 1
```

**风险溢价：**

对于风险资产：
```
1 = E_t[M_{t,t+1} R_{t+1}]

其中 R_{t+1} = (D_{t+1} + P_{t+1}) / P_t

风险溢价：
E[R] - r_f = -Cov(M, R) / E[M]

使用对数近似：
E[r] - r_f ≈ γ Var(Δc) / 2

其中 γ 为相对风险厌恶系数
```

**图形分析：消费CAPM**

```
收益 R
  │
  │        证券市场线
  │       ╱
  │      ╱
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── β_M

期望收益与消费风险成正比
```

### 10.2.2 资本资产定价模型（CAPM）

**基本假设：**

1. 投资者具有均值-方差偏好
2. 市场无摩擦（无税收、无交易成本）
3. 投资者对收益分布有相同预期
4. 存在无风险资产
5. 资本市场均衡

**市场组合：**

市场组合包含所有风险资产，权重为市场价值比例：
```
w_i^M = (P_i Q_i) / Σ_j (P_j Q_j)

其中：
P_i：资产i的价格
Q_i：资产i的数量
```

**CAPM公式：**

```
E[R_i] = r_f + β_iM (E[R_M] - r_f)

其中：
E[R_i]：资产i的期望收益
r_f：无风险利率
E[R_M]：市场组合期望收益
β_iM：资产i的市场Beta系数
```

**Beta系数：**

```
β_iM = Cov(R_i, R_M) / Var(R_M)

解释：
- β_iM > 1：资产比市场风险更大
- β_iM = 1：资产风险与市场相同
- β_iM < 1：资产风险小于市场
- β_iM = 0：资产无系统性风险
```

**证券市场线（SML）：**

图形分析：

```
期望收益 E[R]
  │
  │          SML
  │        ╱
  │       ╱  斜率 = E[R_M] - r_f
  │      ╱
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  └──────────────────── β_M
      0    1    2
     无风险  市场组合

证券市场线上的点：定价正确
线上的点：被高估（期望收益低于应有水平）
线下的点：被低估（期望收益高于应有水平）
```

**Alpha：**

实际收益与CAPM预测的差：
```
α_i = R_i - [r_f + β_iM (R_M - r_f)]

如果 α_i > 0：超额收益，资产被低估
如果 α_i < 0：收益不足，资产被高估
```

**例子：CAPM的应用**

```
假设：
- 无风险利率：r_f = 3%
- 市场组合期望收益：E[R_M] = 10%
- 市场方差：Var(R_M) = 0.04

资产A：
- 期望收益：E[R_A] = 12%
- 与市场的协方差：Cov(R_A, R_M) = 0.06

计算Beta：
β_AM = 0.06 / 0.04 = 1.5

CAPM预测收益：
E[R_A]^CAPM = 3% + 1.5 × (10% - 3%) = 3% + 10.5% = 13.5%

实际收益与预测比较：
E[R_A] = 12% < 13.5%

Alpha：
α_A = 12% - 13.5% = -1.5%

结论：资产A被高估（收益低于应有水平）
```

**CAPM的实证检验：**

**Fama-MacBeth回归（1973）：**

两阶段回归：
1. 时间序列回归估计Beta：
   ```
   R_{i,t} = α_i + β_iM R_{M,t} + ε_{i,t}
   ```

2. 横截面回归检验CAPM：
   ```
   R_{i,t} = γ₀,t + γ₁,t β_iM + ε_{i,t}
   ```

CAPM预测：γ₁ = E[R_M] - r_f > 0

**实证发现：**
- Beta与收益正相关（支持CAPM）
- 但存在其他因素（规模、价值、动量）
- 截距γ₀不为零（市场组合不是有效组合）

### 10.2.3 套利定价理论（APT）

**基本思想：**

资产收益由多个系统性风险因素驱动，而非仅市场因素。

**模型设定：**

```
R_i = E[R_i] + Σ_k β_ik f_k + ε_i

其中：
R_i：资产i的收益
E[R_i]：期望收益
β_ik：资产i对因素k的敏感度
f_k：因素k的冲击（零均值）
ε_i：特质风险（与因素无关）
```

**APT定价公式：**

```
E[R_i] = r_f + Σ_k β_ik λ_k

其中：
r_f：无风险利率
λ_k：因素k的风险溢价
```

**无套利证明：**

如果有资产不满足APT定价，可以构造套利组合：
1. 零投资
2. 零风险（对冲所有因素）
3. 正收益

**图形分析：APT定价**

```
期望收益
  │
  │      APT定价平面
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── β₁
     β₂

所有资产应该位于APT平面上
```

**常见风险因素：**

1. **宏观因素：**
   - 通胀风险
   - 利率风险
   - GDP增长风险
   - 石油价格风险

2. **Fama-French三因素模型（1993）：**
   ```
   R_i - r_f = α_i + β_iM (R_M - r_f) + β_iS SMB + β_iH HML + ε_i

   其中：
   - SMB（Small Minus Big）：规模因子
   - HML（High Minus Low）：价值因子
   ```

3. **Carhart四因素模型（1997）：**
   - 增加动量因子

**例子：APT的应用**

```
假设：
- 无风险利率：r_f = 4%
- 两个风险因素：
  - 因素1风险溢价：λ₁ = 3%
  - 因素2风险溢价：λ₂ = 5%

资产A的敏感度：
- β_A1 = 1.2
- β_A2 = 0.8

APT预测收益：
E[R_A] = 4% + 1.2×3% + 0.8×5%
       = 4% + 3.6% + 4%
       = 11.6%

如果实际期望收益为13%：
Alpha = 13% - 11.6% = 1.4%

资产被低估，有正的超额收益
```

## 10.3 期权定价理论

### 10.3.1 期权的基本概念

**期权的定义：**

期权是赋予持有者在特定时间以特定价格买卖资产的权利（而非义务）的合约。

**期权类型：**

1. **看涨期权（Call Option）：**
   - 赋予买方买入资产的权利
   - 行权价格：K
   - 到期时间：T

   收益：
   ```
   C_T = max(S_T - K, 0)

   其中 S_T 为到期时资产价格
   ```

2. **看跌期权（Put Option）：**
   - 赋予买方卖出资产的权利
   - 行权价格：K
   - 到期时间：T

   收益：
   ```
   P_T = max(K - S_T, 0)
   ```

**图形分析：期权收益**

```
看涨期权收益
  │
  │         ╱
  │        ╱
  │       ╱
  │      ╱
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱───────
  └────────────────── 资产价格 S_T
      K

看跌期权收益
  │
  │╱
  │ ╲
  │  ╲
  │   ╲
  │    ╲
  │     ╲
  │      ╲
  │       ╲
  │        ╲
  └────────────────── 资产价格 S_T
          K
```

**期权的影响因素：**

| 因素 | 看涨期权 | 看跌期权 |
|------|---------|---------|
| 标的资产价格（S） | + | - |
| 行权价格（K） | - | + |
| 到期时间（T） | + | + |
| 波动率（σ） | + | + |
| 无风险利率（r） | + | - |

### 10.3.2 二叉树模型

**基本设定：**

单期二叉树模型：

```
S₀：初始资产价格
u：上涨因子（u > 1）
d：下跌因子（d < 1）
p：上涨概率（风险中性概率）

到期价格：
S₁ = S₀ × u（上涨）
S₁ = S₀ × d（下跌）
```

**图形分析：单期二叉树**

```
      S₀
     ／  ＼
   S₀u   S₀d
  ／      ／
 期权   期权
 收益   收益
```

**无套利定价：**

构造无风险组合：
- 买入Δ单位资产
- 卖出1份看涨期权

组合价值：
```
上涨时：V_u = Δ S₀u - C_u
下跌时：V_d = Δ S₀d - C_d
```

无风险条件：V_u = V_d

因此：
```
Δ = (C_u - C_d) / (S₀u - S₀d)
```

组合收益：
```
V = V_u / (1 + r) = V_d / (1 + r)
```

期权价格：
```
C₀ = Δ S₀ - V
```

**风险中性定价：**

定义风险中性概率：
```
p* = [(1 + r) - d] / (u - d)
```

期权价格：
```
C₀ = [p* C_u + (1 - p*) C_d] / (1 + r)
```

**例子：单期二叉树定价**

```
假设：
- 当前资产价格：S₀ = 100
- 上涨因子：u = 1.2
- 下跌因子：d = 0.8
- 无风险利率：r = 10%
- 行权价格：K = 100

到期价格：
S_u = 100 × 1.2 = 120
S_d = 100 × 0.8 = 80

期权收益：
C_u = max(120 - 100, 0) = 20
C_d = max(80 - 100, 0) = 0

风险中性概率：
p* = (1.1 - 0.8) / (1.2 - 0.8) = 0.3 / 0.4 = 0.75

期权价格：
C₀ = [0.75 × 20 + 0.25 × 0] / 1.1
    = 15 / 1.1
    ≈ 13.64
```

**多期二叉树：**

将单期扩展到多期，通过递归定价：

```
C₀ = Σ_{j=0}^n [C(n, j) p*^j (1-p*)^(n-j) C_{i,j}] / (1 + r)^n

其中：
n：期数
C(n, j)：组合数
C_{i,j}：第j个节点的期权收益
```

### 10.3.3 Black-Scholes模型

**基本假设：**

1. 资产价格服从几何布朗运动
2. 无风险利率恒定
3. 无交易成本和税收
4. 资产可连续交易
5. 市场无摩擦

**资产价格动态：**

```
dS_t = μ S_t dt + σ S_t dW_t

其中：
S_t：资产价格
μ：漂移率（期望收益）
σ：波动率
W_t：标准布朗运动
```

**伊藤引理：**

对于函数f(S, t)：
```
df = (∂f/∂t + μ S ∂f/∂S + 0.5 σ² S² ∂²f/∂S²) dt + σ S ∂f/∂S dW
```

**Black-Scholes偏微分方程：**

通过构造无风险组合，得到：
```
∂C/∂t + r S ∂C/∂S + 0.5 σ² S² ∂²C/∂S² = r C
```

**Black-Scholes公式：**

看涨期权价格：
```
C = S N(d₁) - K e^{-rT} N(d₂)

其中：
d₁ = [ln(S/K) + (r + 0.5 σ²)T] / (σ√T)
d₂ = d₁ - σ√T

N(·)：标准正态累积分布函数
```

看跌期权价格（看跌-看涨平价）：
```
P = C - S + K e^{-rT}
   = K e^{-rT} N(-d₂) - S N(-d₁)
```

**希腊字母：**

期权价格对各种因素的敏感度：

1. **Delta（Δ）：**
   ```
   Δ = ∂C/∂S = N(d₁)

   含义：期权价格对标的资产价格的敏感度
   ```

2. **Gamma（Γ）：**
   ```
   Γ = ∂²C/∂S² = N'(d₁) / (S σ √T)

   含义：Delta对资产价格的敏感度
   ```

3. **Theta（Θ）：**
   ```
   Θ = ∂C/∂t = -[S N'(d₁) σ] / (2√T) - r K e^{-rT} N(d₂)

   含义：期权价格对时间的敏感度（时间衰减）
   ```

4. **Vega（ν）：**
   ```
   ν = ∂C/∂σ = S √T N'(d₁)

   含义：期权价格对波动率的敏感度
   ```

5. **Rho（ρ）：**
   ```
   ρ = ∂C/∂r = K T e^{-rT} N(d₂)

   含义：期权价格对利率的敏感度
   ```

**图形分析：期权价格与希腊字母**

```
期权价格
  │
  │       ╱
  │      ╱
  │     ╱
  │    ╱
  │   ╱
  │  ╱
  │ ╱
  │╱
  └────────────────── 标的资产价格 S
      Delta = 斜率
      Gamma = 斜率的变化率

期权价格 vs 时间
  │
  │
  │╱
  │╲
  │ ╲
  │  ╲
  │   ╲
  │    ╲
  │     ╲
  └────────────────── 时间
      Theta = 时间衰减率
```

**例子：Black-Scholes定价**

```
假设：
- 当前资产价格：S = 100
- 行权价格：K = 100
- 到期时间：T = 1年
- 无风险利率：r = 5%
- 波动率：σ = 20%

计算d₁和d₂：
d₁ = [ln(100/100) + (0.05 + 0.5×0.04)×1] / (0.2×1)
   = [0 + 0.07] / 0.2
   = 0.35

d₂ = 0.35 - 0.2 = 0.15

查找标准正态分布表：
N(d₁) = N(0.35) ≈ 0.6368
N(d₂) = N(0.15) ≈ 0.5596

看涨期权价格：
C = 100 × 0.6368 - 100 × e^(-0.05) × 0.5596
  = 63.68 - 95.12 × 0.5596
  = 63.68 - 53.25
  ≈ 10.43

希腊字母：
Δ = N(d₁) = 0.6368
Γ = N'(0.35) / (100 × 0.2 × 1) ≈ 0.0132
Θ ≈ -5.5（每年）
ν ≈ 38.3（每1%波动率）
ρ ≈ 53.2（每1%利率）
```

## 10.4 风险溢价

### 10.4.1 股权溢价之谜

**定义：**

股权溢价是股票平均收益与无风险收益的差。

**历史数据：**

美国市场（1926-2020）：
- 股票平均收益：约10%
- 无风险收益：约3-4%
- 股权溢价：约6-7%

**图形分析：股权溢价**

```
收益（%）
  │
  │    ████ 股票
  │   ████████
  │  ████████████
  │ ████████████████
  │███████████████████
  │    ████ 无风险
  │   ████████
  │  ████████████
  └───────────────────
     1926  1950  1975  2000

股权溢价 = 股票收益 - 无风险收益
```

**谜题：**

**Mehra and Prescott（1985）：**

基于标准CRRA效用模型，预测的股权溢价远小于实际观察值。

**模型预测：**

```
消费CAPM：
E[R] - r_f ≈ γ Var(Δc) / 2

其中：
γ：相对风险厌恶系数
Var(Δc)：消费增长率方差

实际数据：
- Var(Δc) ≈ 0.0001（季度）
- 股权溢价 ≈ 0.015（季度）

需要：
γ = 2 × 0.015 / 0.0001 = 300

但合理的γ范围是1-10
```

**谜题：为什么需要如此高的风险厌恶系数？**

**可能的解释：**

1. **消费CAPM不完善：**
   - 股票收益不仅与消费风险相关
   - 存在其他风险因素

2. **罕见灾难风险：**
   - Barro（2006）：罕见灾难（如战争、金融危机）增加风险溢价
   - 小概率大损失增加风险溢价

3. **习惯形成：**
   - Campbell and Cochrane（1999）：效用取决于相对于习惯的消费
   - 习惯随时间变化，增加风险厌恶

4. **长期风险：**
   - Bansal and Yaron（2004）：长期经济风险增加股权溢价
   - 期望增长和波动的时变

5. **制度摩擦：**
   - 借贷约束
   - 市场分割
   - 交易成本

### 10.4.2 无风险利率之谜

**定义：**

标准模型预测的无风险利率远高于实际水平。

**模型预测：**

```
欧拉方程：
1 = β E[(c_{t+1}/c_t)^(-γ) (1 + r_f)]

对数线性化：
r_f ≈ (1/β) - 1 + γ E[Δc] - 0.5 γ² Var(Δc)

假设：
- β = 0.99（季度）
- γ = 10
- E[Δc] = 0.005（季度增长）
- Var(Δc) = 0.0001

预测：
r_f ≈ (1/0.99 - 1) + 10×0.005 - 0.5×100×0.0001
    ≈ 0.01 + 0.05 - 0.005
    ≈ 5.5%（年化）

实际：约1-2%
```

**解释：**

1. **增长预期：**
   - 市场可能低估增长
   - 或存在增长风险

2. **偏好参数：**
   - 主观折现因子可能更低（更多耐心）
   - 风险厌恶可能更低

3. **生命周期因素：**
   - 年轻人储蓄推低利率
   - 人口老龄化推高利率

4. **安全资产短缺：**
   - 对无风险资产需求高
   - 推低利率

### 10.4.3 风险溢价的分解

**总风险溢价：**

```
E[R] - r_f = 系统性风险溢价 + 特质性风险溢价
                + 流动性溢价 + 其他溢价
```

**系统性风险溢价：**

来自CAPM或APT：
```
RP_systematic = β_iM (E[R_M] - r_f)  (CAPM)
或
RP_systematic = Σ_k β_ik λ_k  (APT)
```

**特质风险溢价：**

来自公司特定风险：
```
RP_idiosyncratic = λ_i σ_i²

其中 σ_i² 为特质风险方差
```

**流动性溢价：**

来自资产流动性差异：
```
RP_liquidity = λ_L L_i

其中 L_i 为流动性指标（如买卖价差）
```

**图形分析：风险溢价分解**

```
总风险溢价
  │
  │    ████████████
  │    ████ 系统性
  │    ██ 特质性
  │    ██ 流动性
  │    ██ 其他
  │
  └──────────────────
```

**例子：风险溢价分解**

```
假设：
- 资产总收益：12%
- 无风险利率：3%
- 总风险溢价：9%

分解：
系统性风险（CAPM）：
β = 1.2, E[R_M] - r_f = 5%
RP_systematic = 1.2 × 5% = 6%

特质风险：
σ² = 0.04, λ = 25
RP_idiosyncratic = 25 × 0.04 = 1%

流动性溢价：
买卖价差 = 2%, λ_L = 1
RP_liquidity = 1 × 2% = 2%

其他溢价：
1%

总溢价：6% + 1% + 2% + 1% = 10%（接近9%）
```

## 10.5 资产定价与宏观经济的联系

### 10.5.1 消费与资产收益

**消费CAPM回顾：**

```
E[R_i] - r_f ≈ γ Cov(R_i, Δc)

其中：
γ：相对风险厌恶系数
Δc：消费增长率
```

**含义：**

- 资产收益与消费增长协动
- 消费平滑决定风险溢价

### 10.5.2 生产与资产收益

**生产资产定价模型：**

资产收益与生产率增长相关：
```
E[R_i] - r_f ≈ Cov(R_i, ΔA)

其中 ΔA 为生产率增长
```

**含义：**

- 生产率冲击影响收益
- 技术进步影响估值

### 10.5.3 商业周期与资产收益

**周期性：**

- 股票收益顺周期：繁荣期收益高
- 债券收益逆周期或弱顺周期
- 期权收益在不确定性增加时上升

**预测性：**

变量预测未来收益：
- 股息价格比（D/P）：反向预测
- 期限利差：预测经济和股票收益
- 波动率（VIX）：预测风险溢价

### 10.5.4 货币政策与资产价格

**货币政策传导：**

```
货币政策利率调整
    ↓
无风险利率变化
    ↓
折现率变化
    ↓
资产价格变化
```

**资产定价反应：**

- 股票：对利率敏感，利率上升 → 价格下降
- 债券：久期影响，长期债券更敏感
- 房地产：长期资产，对利率敏感
- 汇率：影响跨境资产价格

**例子：货币政策与股市**

```
假设：
- 无风险利率上升1%
- 股票市场：久期约为10年
- 股息增长预期不变

股价变化：
ΔP/P ≈ -D × Δr = -10 × 1% = -10%

市场下跌10%

但考虑货币政策含义：
- 利率上升可能反映经济强劲
- 企业盈利预期上升
- 部分抵消负面影响
```

## 本书总结

### 动态宏观经济理论的核心框架

本书系统性地构建了动态宏观经济理论的完整框架，从基础到高级，从理论到应用。核心框架包括：

#### 1. 微观基础与优化

**代表性主体优化：**

家庭最优化：
```
max E₀ Σ β^t U(c_t, n_t)
s.t. c_t + ... ≤ ...

核心概念：
- 时间偏好（β）
- 风险厌恶（U'' < 0）
- 跨期替代（σ = -U'/ (U'' C)）
- 劳动供给决策
```

企业最优化：
```
max Σ β^t Π_t
Π_t = P_t F(K_t, N_t) - W_t N_t - I_t

核心概念：
- 生产函数（规模报酬）
- 要素市场出清
- 投资决策（调整成本）
```

#### 2. 一般均衡框架

**竞争性均衡：**

```
定义：在给定价格下，所有主体最优化，所有市场出清

要素：
1. 家庭最优化（消费、储蓄、劳动供给）
2. 企业最优化（生产、投资）
3. 政府行为（财政政策、货币政策）
4. 外国部门（开放经济）
5. 市场出清条件

均衡类型：
- 短期均衡：给定资本存量
- 长期均衡：资本调整完成
- 稳态：所有变量恒定
- 转移路径：向稳态调整
```

**均衡的存在性与唯一性：**
- 严格凸性保证唯一性
- 布劳威尔不动点定理
- 动态规划递归方法

#### 3. 动态系统分析

**差分方程与动态：**

```
线性系统：X_t = A X_{t-1} + B ε_t
非线性系统：X_t = f(X_{t-1}, ε_t)

分析工具：
- 特征值分析（稳定性）
- 脉冲响应函数
- 方差分解
- 频域分析
```

**稳定性条件：**
```
所有特征值 |λ| < 1：全局稳定
存在 |λ| > 1：不稳定或鞍点稳定
|λ| = 1：单位根（随机游走）
```

#### 4. 随机动态与预期

**理性预期：**

```
定义：E_t[X_{t+1}] = E[X_{t+1} | I_t]

性质：
- 无系统预测误差
- 预期与模型一致
- 信息有效利用

应用：
- 欧拉方程
- 资产定价
- 政策分析
```

**鞅性质：**

```
定义：E_t[X_{t+1}] = X_t

例子：
- 股价（在无套利下）
- 汇率（在随机游走假设下）
- 消费（永久收入假说）
```

#### 5. 经济周期理论

**实际经济周期（RBC）：**

```
核心思想：
- 经济周期源于实际冲击（技术、偏好）
- 市场持续出清
- 经济周期是最优响应

关键要素：
- 技术冲击：ln(A_t) = ρ ln(A_{t-1}) + ε_t
- 跨期优化：家庭和企业
- 动态一般均衡
- 脉冲响应分析

贡献：
- 严格的微观基础
- 强调供给冲击
- 跨期平滑行为

局限：
- 名义刚性缺失
- 劳动市场摩擦不足
- 货币中性过强
```

**凯恩斯主义经济周期：**

```
核心思想：
- 名义刚性导致市场失灵
- 需求冲击和货币冲击重要
- 主动政策有效

关键要素：
- 价格/工资刚性（Calvo定价）
- 新凯恩斯菲利普斯曲线：π_t = β E_t[π_{t+1}] + κ y_t
- 动态IS曲线：y_t = E_t[y_{t+1}] - (1/σ)(i_t - E_t[π_{t+1}] - r_t^n)
- 泰勒规则：i_t = ρ + φ_π π_t + φ_y (y_t - ȳ_t)

贡献：
- 解释名义刚性的重要性
- 货币政策分析框架
- 政策传导机制

局限：
- 刚性的微观基础
- 长期分析不足
```

**现代DSGE模型：**

```
综合框架：
- RBC的微观基础
- 凯恩斯的名义刚性
- 金融部门摩擦
- 不完全竞争

应用：
- 政策分析
- 预测
- 反事实分析
```

#### 6. 货币政策理论

**货币政策目标与工具：**

```
目标：
- 价格稳定（通胀目标制）
- 充分就业
- 经济增长
- 金融稳定

工具：
- 政策利率（主要）
- 公开市场操作
- 准备金率
- 非传统工具（QE、前瞻性指引）
```

**货币政策规则：**

```
泰勒规则：
i_t = ρ + φ_π π_t + φ_y (y_t - ȳ_t)

泰勒原则：φ_π > 1

通胀目标制：
- 明确宣布目标
- 透明沟通
- 责任机制
```

**传导机制：**

```
利率渠道：i ↓ → r ↓ → I, C ↑ → Y ↑
信贷渠道：i ↓ → 银行贷款 ↑ → I ↑ → Y ↑
资产价格渠道：i ↓ → 资产价格 ↑ → 财富 ↑ → C ↑ → Y ↑
汇率渠道：i ↓ → 汇率贬值 → NX ↑ → Y ↑
```

**ZLB与非常规政策：**

```
零利率下限约束：
i_t = max{0, i_t^*}

非常规工具：
- 量化宽松（QE）
- 前瞻性指引
- 负利率政策
- 大规模资产购买
```

#### 7. 财政政策理论

**财政政策乘数：**

```
简单乘数：k_G = 1/(1 - c)
IS-LM乘数：k_G^{IS-LM} = 1/[(1 - c) + (bk/h)]

影响因素：
- 经济周期状态（衰退期乘数大）
- 货币环境（ZLB下乘数大）
- 债务水平（高债务乘数小）
- 经济结构（开放度、金融发展）
```

**债务动态与可持续性：**

```
债务动态方程：
b_{t+1} ≈ (1 + r - π - g) b_t + d_t

可持续性条件：
g > r（实际增长率 > 实际利率）

稳态债务：
b* = d* / (g* - r_real)
```

**李嘉图等价：**

```
定理：在理想条件下，融资方式无关

条件：
- 完全资本markets
- 无限期界
- 一次性总量税
- 理性预期

现实：不完全成立，但部分有效
```

**财政整顿：**

```
策略：
- 支出削减
- 税收增加
- 结构性改革

时机：
- 扩张期整顿效果好
- 衰退期整顿可能加剧衰退

组成：
- 基于支出的整顿更成功
- 对增长影响较小
```

#### 8. 资产定价理论

**欧拉方程与随机折现因子：**

```
欧拉方程：
U'(c_t) = β E_t[U'(c_{t+1}) R_{t+1}]

随机折现因子：
M_{t,t+1} = β (U'(c_{t+1}) / U'(c_t))

定价公式：
P_t = E_t[M_{t,t+1} (D_{t+1} + P_{t+1})]
```

**CAPM与APT：**

```
CAPM：
E[R_i] = r_f + β_iM (E[R_M] - r_f)
β_iM = Cov(R_i, R_M) / Var(R_M)

APT：
E[R_i] = r_f + Σ_k β_ik λ_k
```

**期权定价：**

```
Black-Scholes公式：
C = S N(d₁) - K e^{-rT} N(d₂)

希腊字母：
- Delta：Δ = N(d₁)
- Gamma：Γ = N'(d₁) / (S σ √T)
- Theta：Θ = ∂C/∂t
- Vega：ν = ∂C/∂σ
- Rho：ρ = ∂C/∂r
```

**风险溢价：**

```
股权溢价之谜：
历史溢价6-7%
模型预测需要γ ≈ 300
合理的γ范围1-10

无风险利率之谜：
模型预测5-6%
实际1-2%

解释：
- 罕见灾难风险
- 习惯形成
- 长期风险
- 制度摩擦
```

## 本章总结

### 核心要点

1. **资产定价基本原理**
   - 资产定价 = 未来收益的折现
   - 无套利原理是定价基础
   - 风险溢价补偿风险和时间偏好

   - 消费资产定价模型（CCAPM）

   - Beta系数衡量系统性风险

4. **期权定价**
   - 二叉树模型：离散时间
   - Black-Scholes模型：连续时间
   - 希腊字母：敏感度分析

5. **风险溢价**
   - 股权溢价之谜：需要极高风险厌恶
   - 无风险利率之谜：模型预测过高
   - 多种可能解释

6. **资产定价与宏观**
   - 消费与收益协动
   - 商业周期影响
   - 货币政策传导

### 关键公式汇总

1. **资产定价基本公式：**
   ```
   P₀ = E₀ [Σ D_t / (1 + r)^t]
   ```

2. **欧拉方程：**
   ```
   U'(c_t) = β E_t[U'(c_{t+1}) R_{t+1}]
   ```

3. **随机折现因子：**
   ```
   M_{t,t+1} = β (U'(c_{t+1}) / U'(c_t))
   ```

4. **CAPM：**
   ```
   E[R_i] = r_f + β_iM (E[R_M] - r_f)
   ```

5. **Beta系数：**
   ```
   β_iM = Cov(R_i, R_M) / Var(R_M)
   ```

6. **APT：**
   ```
   E[R_i] = r_f + Σ_k β_ik λ_k
   ```

7. **Black-Scholes看涨期权：**
   ```
   C = S N(d₁) - K e^{-rT} N(d₂)
   d₁ = [ln(S/K) + (r + 0.5σ²)T] / (σ√T)
   d₂ = d₁ - σ√T
   ```

8. **消费CAPM风险溢价：**
   ```
   E[R] - r_f ≈ γ Var(Δc) / 2
   ```

9. **风险中性概率（二叉树）：**
   ```
   p* = [(1 + r) - d] / (u - d)
   ```

10. **看跌-看涨平价：**
    ```
    P = C - S + K e^{-rT}
    ```
