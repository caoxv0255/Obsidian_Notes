# 第3章：动态规划

## 本章学习目标

通过本章的学习，学生应该能够：
1. 理解动态规划的基本原理和贝尔曼最优化原理
2. 掌握贝尔曼方程的推导和求解方法
3. 熟悉值函数和策略函数的性质
4. 应用动态规划解决宏观经济中的最优化问题
5. 理解动态规划在消费-储蓄模型、投资模型中的应用

## 3.1 最优化原理

### 3.1.1 动态最优化问题

动态规划是解决多期最优化问题的强大工具。在宏观经济学中，许多核心问题都涉及跨期决策，如消费-储蓄选择、投资决策、劳动力供给等。

**定义：离散时间动态最优化问题**

一个标准的动态最优化问题可以表示为：

$$\max_{\{c_t, k_{t+1}\}_{t=0}^{\infty}} \sum_{t=0}^{\infty} \beta^t u(c_t)$$

$$s.t. \quad c_t + k_{t+1} = f(k_t) + (1-\delta)k_t$$

$$c_t \geq 0, \quad k_{t+1} \geq 0$$

$$k_0 \text{ 给定}$$

其中：
- $\beta \in (0, 1)$ 是主观贴现因子
- $u(\cdot)$ 是效用函数
- $c_t$ 是第 $t$ 期的消费
- $k_t$ 是第 $t$ 期的资本存量
- $f(\cdot)$ 是生产函数
- $\delta \in [0, 1]$ 是资本折旧率

### 3.1.2 贝尔曼最优化原理

贝尔曼最优化原理是动态规划的核心思想。

**原理陈述**：最优策略具有这样的性质，无论初始状态和初始决策如何，对于第一个决策所形成的状态，其余的决策必须构成一个从该状态出发的最优策略。

这个原理意味着多期最优化问题可以分解为一系列相关的一期最优化问题。

**直观理解**：
考虑一个从时期 $t$ 开始的子问题。如果整体问题是最优的，那么从时期 $t$ 开始的子问题也必须是最优的。否则，我们可以通过改进这个子问题来改进整体问题，这与最优性矛盾。

### 3.1.3 值函数

**定义：值函数**

值函数表示从给定状态出发，遵循最优策略所能获得的最大效用现值：

$$V(k_t) = \max_{\{c_{s}, k_{s+1}\}_{s=t}^{\infty}} \sum_{s=t}^{\infty} \beta^{s-t} u(c_s)$$

$$s.t. \quad c_s + k_{s+1} = f(k_s) + (1-\delta)k_s, \quad s \geq t$$

值函数具有以下重要性质：

**性质1：时间一致性**

$$V(k_t) = \max_{c_t, k_{t+1}} \{u(c_t) + \beta V(k_{t+1})\}$$

这意味着当前的最优值等于当前效用加上未来最优值的贴现值。

**性质2：贝尔曼方程**

$$V(k) = \max_{c, k'} \{u(c) + \beta V(k')\}$$

$$s.t. \quad c + k' = f(k) + (1-\delta)k$$

其中我们省略了时间下标，强调这是静态（递归）问题。

### 3.1.4 策略函数

**定义：策略函数**

策略函数将状态变量映射为控制变量的最优选择：

$$c = g_c(k)$$
$$k' = g_k(k)$$

策略函数描述了在每种可能的状态下应该采取什么行动。

**性质**：
- 策略函数是值函数的隐函数
- 一旦找到策略函数，我们可以通过迭代计算整个最优路径
- 策略函数通常是状态的连续函数（在适当条件下）

### 3.1.5 图形分析

**动态规划问题的结构**

```
时间 t=0          t=1          t=2          t=3          ...
  │             │             │             │
  │             │             │             │
状态 k₀ ──────► k₁ ──────► k₂ ──────► k₃ ──────► ...
  │             │             │             │
  │             │             │             │
决策 c₀,k₁     c₁,k₂       c₂,k₃       c₃,k₄      ...
  │             │             │             │
  │             │             │             │
效用 u(c₀)     βu(c₁)      β²u(c₂)      β³u(c₃)    ...
  │             │             │             │
  │             │             │             │
  └─────────────┴─────────────┴─────────────┴─────► 总和：V(k₀)
```

### 3.1.6 例子

**Ramsey-Cass-Koopmans模型**

考虑一个代表性家庭的消费-储蓄问题：

$$\max_{\{c_t\}_{t=0}^{\infty}} \sum_{t=0}^{\infty} \beta^t \frac{c_t^{1-\sigma}}{1-\sigma}$$

$$s.t. \quad c_t + k_{t+1} = k_t^\alpha + (1-\delta)k_t$$

其中：
- $\sigma > 0$ 是相对风险厌恶系数
- $\alpha \in (0, 1)$ 是资本产出弹性

**值函数**：
$$V(k) = \max_{c, k'} \left\{\frac{c^{1-\sigma}}{1-\sigma} + \beta V(k')\right\}$$

**一阶条件**：
通过拉格朗日方法或包络定理，我们得到欧拉方程：
$$c_t^{-\sigma} = \beta c_{t+1}^{-\sigma}[\alpha k_{t+1}^{\alpha-1} + 1 - \delta]$$

这个方程描述了最优消费路径的特征：边际效用的现值应该相等。

## 3.2 贝尔曼方程

### 3.2.1 贝尔曼方程的推导

从最优化原理出发，我们可以严格推导出贝尔曼方程。

**第一步：定义值函数**

$$V(k_t) = \max_{\{c_s, k_{s+1}\}_{s=t}^{\infty}} \sum_{s=t}^{\infty} \beta^{s-t} u(c_s)$$

**第二步：分离第一期**

$$V(k_t) = \max_{c_t, k_{t+1}} \left\{u(c_t) + \max_{\{c_s, k_{s+1}\}_{s=t+1}^{\infty}} \sum_{s=t+1}^{\infty} \beta^{s-t} u(c_s)\right\}$$

**第三步：提取贴现因子**

$$V(k_t) = \max_{c_t, k_{t+1}} \left\{u(c_t) + \beta \max_{\{c_s, k_{s+1}\}_{s=t+1}^{\infty}} \sum_{s=t+1}^{\infty} \beta^{s-t-1} u(c_s)\right\}$$

**第四步：识别子问题**

注意到大括号内的子问题正是从状态 $k_{t+1}$ 开始的值函数：

$$V(k_{t+1}) = \max_{\{c_s, k_{s+1}\}_{s=t+1}^{\infty}} \sum_{s=t+1}^{\infty} \beta^{s-t-1} u(c_s)$$

**第五步：贝尔曼方程**

$$V(k_t) = \max_{c_t, k_{t+1}} \{u(c_t) + \beta V(k_{t+1})\}$$

### 3.2.2 贝尔曼方程的性质

**性质1：不动点性质**

贝尔曼方程可以看作是一个压缩映射：

$$\mathcal{T}V(k) = \max_{c, k'} \{u(c) + \beta V(k')\}$$

在适当条件下（如 $u(\cdot)$ 有界，$\beta < 1$），存在唯一的不动点：
$$V^*(k) = \mathcal{T}V^*(k)$$

**性质2：Banach不动点定理**

映射 $\mathcal{T}$ 是一个压缩映射，因为：

$$||\mathcal{T}V_1 - \mathcal{T}V_2||_{\infty} \leq \beta ||V_1 - V_2||_{\infty}$$

这意味着我们可以通过迭代方法找到不动点。

### 3.2.3 价值迭代算法

**算法步骤**：

1. **初始化**：选择初始猜测 $V_0(k)$
2. **迭代**：对于 $s = 0, 1, 2, \ldots$
   $$V_{s+1}(k) = \max_{c, k'} \{u(c) + \beta V_s(k')\}$$
3. **收敛**：当 $||V_{s+1} - V_s|| < \varepsilon$ 时停止

**收敛速度**：由于 $\mathcal{T}$ 是压缩映射，价值迭代的误差以 $\beta^s$ 的速度收敛。

### 3.2.4 策略迭代算法

**算法步骤**：

1. **初始化**：选择初始策略函数 $g_0(k)$
2. **策略评估**：给定策略 $g_s(k)$，计算值函数：
   $$V_s(k) = u(g_{c,s}(k)) + \beta V_s(g_{k,s}(k))$$
3. **策略改进**：根据新值函数改进策略：
   $$g_{s+1}(k) = \arg\max_{c, k'} \{u(c) + \beta V_s(k')\}$$
4. **收敛检查**：如果 $g_{s+1} = g_s$，停止；否则重复步骤2-4

策略迭代通常比价值迭代更快收敛，但每次迭代的计算量更大。

### 3.2.5 图形分析

**价值迭代的收敛过程**

```
V(k)
│
│     V₁(k) ──────
│    ╱            ╲
│   ╱              ╲
│  ╱                ╲ V₂(k) ──────
│ ╱                  ╲    ╱          ╲
│╱                    ╲  ╱            ╲ V₃(k) ──────
└─────────────────────┴────────────────────────────► k
```

随着迭代进行，值函数逐渐收敛到最优值函数 $V^*(k)$。

### 3.2.6 例子

**离散状态空间的价值迭代**

考虑一个简化的消费问题：
- 资本存量 $k \in \{1, 2, 3\}$
- 生产函数 $f(k) = k$
- 效用函数 $u(c) = \ln(c)$
- 贴现因子 $\beta = 0.9$
- 无折旧 $\delta = 0$

**迭代过程**：

**第0次迭代**（初始猜测 $V_0(k) = 0$）：
- 对于 $k=1$：$V_1(1) = \max_{c+k'=1} \ln(c) = \ln(1) = 0$
- 对于 $k=2$：$V_1(2) = \max_{c+k'=2} \ln(c) = \ln(2) \approx 0.69$
- 对于 $k=3$：$V_1(3) = \max_{c+k'=3} \ln(c) = \ln(3) \approx 1.10$

**第1次迭代**：
- 对于 $k=1$：$V_2(1) = \max_{c+k'=1} \{\ln(c) + 0.9 V_1(k')\} = \max\{0+0.9 \times 0, \ln(1)+0.9 \times 0\} = 0$
- 对于 $k=2$：$V_2(2) = \max\{\ln(2)+0.9 \times 0, \ln(1)+0.9 \times 0.69\} = \max\{0.69, 0.62\} = 0.69$
- 对于 $k=3$：$V_2(3) = \max\{\ln(3)+0.9 \times 0, \ln(2)+0.9 \times 0.69, \ln(1)+0.9 \times 1.10\}$
  $= \max\{1.10, 0.69+0.62, 0+0.99\} = \max\{1.10, 1.31, 0.99\} = 1.31$

继续迭代，值函数会逐渐收敛到最优值函数。

## 3.3 值函数的性质

### 3.3.1 凸性

**定理**：如果 $u(\cdot)$ 是严格凹的，生产函数 $f(\cdot)$ 是凹的，那么值函数 $V(k)$ 也是严格凹的。

**证明思路**：
1. 凹效用函数意味着对平均值的偏好高于对平均的偏好
2. 凹生产函数意味着资本的平均产出递减
3. 最优化问题的性质决定了值函数的凹性

**经济学含义**：
- 值函数的凹性反映了资本的边际效用递减
- 凹值函数保证了最优策略的存在性和唯一性

### 3.3.2 单调性

**定理**：如果 $u'(\cdot) > 0$ 且 $f'(\cdot) > 0$，那么值函数 $V(k)$ 是严格递增的。

**经济学含义**：
- 初始资本越多，最优路径上的总效用越高
- 这是直观的：更多的资源总是更好的

### 3.3.3 连续性

**定理**：在标准假设下（$u(\cdot)$ 和 $f(\cdot)$ 连续，$\beta < 1$），值函数 $V(k)$ 是连续的。

**证明思路**：
1. 紧性：可行集是紧的
2. 最大值定理：连续函数在紧集上的最大值是连续的

### 3.3.4 可微性

**定理**：在标准假设下，值函数 $V(k)$ 是连续可微的，且导数由包络定理给出：

$$V'(k) = u'(c(k))[f'(k) + 1 - \delta]$$

其中 $c(k)$ 是最优消费策略。

**包络定理的应用**：
值函数对状态变量的导数等于目标函数对状态变量的导数，因为最优选择时对控制变量的一阶条件成立。

### 3.3.5 图形分析

**值函数的形状**

```
V(k)
│
│    •
│   ╱ ╲
│  ╱   ╲
│ ╱     ╲
│╱       ╲
└─────────► k
```

值函数 $V(k)$ 是严格递增、严格凹、连续可微的函数。

**策略函数的形状**

```
c(k)
│
│       •
│      ╱
│     ╱
│    ╱
│   ╱
│  ╱
│ ╱
│╱
└─────────► k
```

消费策略函数 $c(k)$ 通常是递增的（收入越高，消费越高），但增长率递减（边际消费倾向小于1）。

### 3.3.6 例子

**CRRA效用下的值函数性质**

考虑 $u(c) = \frac{c^{1-\sigma}}{1-\sigma}$，$f(k) = k^\alpha$，$\delta = 1$（完全折旧）。

猜测值函数的形式：$V(k) = A k^{1-\sigma\alpha}$

其中 $A$ 是待定常数。

通过贝尔曼方程求解 $A$：
$$A k^{1-\sigma\alpha} = \max_{k'} \left\{\frac{(k^\alpha - k')^{1-\sigma}}{1-\sigma} + \beta A k'^{1-\sigma\alpha}\right\}$$

一阶条件：
$$(k^\alpha - k')^{-\sigma} = \beta A (1-\sigma\alpha) k'^{-\sigma\alpha}$$

猜测最优策略：$k' = \phi k^\alpha$

代入一阶条件求解 $\phi$ 和 $A$，最终得到：
$$V(k) = \frac{(\beta\alpha)^{\frac{1-\sigma\alpha}{1-\sigma}} - 1}{(1-\sigma)\beta^{\frac{1-\sigma\alpha}{1-\sigma}}} k^{1-\sigma\alpha}$$

这证实了值函数是资本 $k$ 的幂函数，具有我们所预期的凹性和单调性。

## 3.4 动态规划的应用

### 3.4.1 消费-储蓄模型

**问题设定**

一个代表性的无限期家庭选择消费 $c_t$ 和储蓄 $s_t$ 来最大化终身效用：

$$\max_{\{c_t, s_t\}_{t=0}^{\infty}} \sum_{t=0}^{\infty} \beta^t u(c_t)$$

$$s.t. \quad c_t + s_t = (1+r)s_{t-1} + y_t$$

$$s_t \geq 0$$

$$s_{-1} \text{ 给定}$$

其中：
- $r$ 是利率
- $y_t$ 是外生的劳动收入

**贝尔曼方程**

$$V(s, y) = \max_{c, s'} \{u(c) + \beta E[V(s', y')|y]\}$$

$$s.t. \quad c + s' = (1+r)s + y$$

**欧拉方程**

通过一阶条件得到：
$$u'(c_t) = \beta (1+r) E[u'(c_{t+1})]$$

这个方程表明，当前消费的边际效用应该等于未来消费边际效用的预期贴现值。

### 3.4.2 投资模型

**问题设定**

一个企业选择投资 $I_t$ 来最大化利润现值：

$$\max_{\{I_t, K_t\}_{t=0}^{\infty}} \sum_{t=0}^{\infty} \beta^t [F(K_t) - C(I_t, K_t)]$$

$$s.t. \quad K_{t+1} = (1-\delta)K_t + I_t$$

$$K_0 \text{ 给定}$$

其中：
- $F(K)$ 是生产函数
- $C(I, K)$ 是调整成本函数
- $K$ 是资本存量

**调整成本函数的例子**
$$C(I, K) = I + \frac{\gamma}{2}\left(\frac{I}{K}\right)^2 K$$

这个形式表示投资不仅有直接成本 $I$，还有随投资强度 $I/K$ 增加的调整成本。

**贝尔曼方程**

$$V(K) = \max_{I, K'} \{F(K) - C(I, K) + \beta V(K')\}$$

$$s.t. \quad K' = (1-\delta)K + I$$

**一阶条件**

$$C_I(I, K) + \beta V'(K') = 0$$

$$V'(K) = F'(K) - C_K(I, K) + \beta V'(K')(1-\delta)$$

结合这两个条件得到投资q理论：
$$q_t = 1 + \gamma \frac{I_t}{K_t}$$

其中 $q_t = -\beta E[V'(K_{t+1})]$ 是资本的影子价格（托宾q）。

### 3.4.3 搜寻模型

**问题设定**

一个失业工人选择是否接受当前的工作机会 $w$，或者继续搜寻：

$$\max_{\{accept, search\}} E\left[\sum_{t=0}^{\infty} \beta^t y_t\right]$$

其中：
- $y_t = w$ 如果接受工作并就业
- $y_t = b$ 如果继续失业搜寻
- $b$ 是失业救济金

**贝尔曼方程**

$$V(w) = \max\left\{\frac{w}{1-\beta}, b + \beta E[V(w')]\right\}$$

其中：
- $\frac{w}{1-\beta}$ 是接受工资 $w$ 的终身效用现值
- $b + \beta E[V(w')]$ 是继续搜寻的期望值

**保留工资**

存在一个保留工资 $w^*$ 使得：
- 如果 $w \geq w^*$，接受工作
- 如果 $w < w^*$，继续搜寻

保留工资满足：
$$\frac{w^*}{1-\beta} = b + \beta E[V(w')]$$

### 3.4.4 图形分析

**投资q理论**

```
q
│
│       •
│      ╱│
│     ╱ │
│    ╱  │
│   ╱   │
│  ╱    │
│ ╱     │
│╱      │
└───────┴──────► I/K
       w^*/γ
```

托宾q与投资强度的关系：$q = 1 + \gamma(I/K)$
- 当 $q > 1$ 时，投资为正
- 当 $q = 1$ 时，投资为零
- 当 $q < 1$ 时，投资为负（出售资本）

**保留工资理论**

```
V(w)
│
│              接受
│            ╱
│          ╱
│        ╱
│      ╱
│    ╱
│  ╱
│╱
└────────────────► w
           w^*
```

值函数 $V(w)$ 在保留工资 $w^*$ 处有一个"拐点"，工资低于 $w^*$ 时继续搜寻，高于 $w^*$ 时接受工作。

### 3.4.5 例子

**消费平滑模型**

考虑一个简化的消费问题：
- 收入 $y_t$ 在 $\{10, 20\}$ 之间随机转移，转移概率为 $0.5$
- 效用函数 $u(c) = \ln(c)$
- 利率 $r = 0$
- 贴现因子 $\beta = 0.95$

**贝尔曼方程**：
$$V(s, y) = \max_{s'} \{\ln((1+r)s + y - s') + 0.95 E[V(s', y')]\}$$

**稳态分析**：
在稳态时，储蓄应该使得未来状态与当前状态相同。通过数值求解，我们可以找到最优储蓄策略。

**数值结果**（示例）：
- 如果 $y = 10$，最优储蓄 $s' \approx 5$
- 如果 $y = 20$，最优储蓄 $s' \approx 10$

这意味着家庭会从高收入时期储蓄，以平滑低收入时期的消费，体现了消费平滑的动机。

## 3.5 不确定性下的动态规划

### 3.5.1 随机贝尔曼方程

在随机环境下，状态变量和控制变量都可能受到随机冲击的影响。

**一般形式**

$$V(x) = \max_{u \in \Gamma(x)} \{F(x, u) + \beta E[V(x')|x, u]\}$$

其中：
- $x$ 是状态变量
- $u$ 是控制变量
- $F(x, u)$ 是当期收益函数
- $\beta$ 是贴现因子
- $x'$ 是下一期的状态，服从转移概率 $P(x'|x, u)$

**经济学的例子**

考虑随机生产函数：

$$\max_{\{c_t, k_{t+1}\}_{t=0}^{\infty}} E\left[\sum_{t=0}^{\infty} \beta^t u(c_t)\right]$$

$$s.t. \quad c_t + k_{t+1} = A_t f(k_t) + (1-\delta)k_t$$

其中 $A_t$ 是随机的全要素生产率（TFP）冲击。

**贝尔曼方程**

$$V(k, A) = \max_{c, k'} \{u(c) + \beta E[V(k', A')|k, A]\}$$

### 3.5.2 马尔可夫过程

**定义**：一个随机过程 $\{X_t\}$ 是马尔可夫过程，如果：

$$P(X_{t+1} \in B|X_t, X_{t-1}, \ldots, X_0) = P(X_{t+1} \in B|X_t)$$

即未来的状态只依赖于当前状态，与历史无关。

**转移概率**：
$$P(A'|A) = \text{Prob}(A_{t+1} \in A'|A_t = A)$$

在连续情况下，我们使用转移密度 $f(A'|A)$。

**AR(1)过程的例子**

$$A_{t+1} = \rho A_t + \varepsilon_{t+1}$$
$$\varepsilon_{t+1} \sim N(0, \sigma^2)$$

转移密度：
$$A'|A \sim N(\rho A, \sigma^2)$$

$$f(A'|A) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(A' - \rho A)^2}{2\sigma^2}\right)$$

### 3.5.3 随机动态规划的求解

**价值迭代**

在随机环境下，价值迭代算法为：

$$V_{s+1}(k, A) = \max_{c, k'} \{u(c) + \beta E[V_s(k', A')|k, A]\}$$

其中期望需要根据转移概率计算：
$$E[V_s(k', A')|k, A] = \int V_s(k', A') f(A'|A) dA'$$

**数值方法**
1. **离散状态空间**：将连续变量离散化为有限网格
2. **插值**：使用线性插值或其他方法近似网格点之间的值函数
3. **数值积分**：使用高斯积分或蒙特卡洛方法计算期望

### 3.5.4 风险溢价

**定义**：风险溢价是为补偿不确定性而要求的额外收益。

**例子**：消费资本资产定价模型（CCAPM）

考虑一个代表性家庭面临随机消费：
$$\max E\left[\sum_{t=0}^{\infty} \beta^t u(c_t)\right]$$

资产定价公式：
$$p_t = E_t\left[\beta \frac{u'(c_{t+1})}{u'(c_t)} (p_{t+1} + d_{t+1})\right]$$

其中 $d_t$ 是资产红利。

**风险溢价**：
$$r^e = \frac{1}{\beta} - 1 + \sigma \gamma \sigma_c$$

其中：
- $\sigma$ 是资产收益的标准差
- $\gamma$ 是相对风险厌恶系数
- $\sigma_c$ 是消费增长的标准差

风险溢价与风险厌恶系数和消费波动性正相关。

### 3.5.5 图形分析

**风险厌恶与值函数**


风险厌恶个体的值函数是凹的。$\bar{c}$ 是确定性的消费水平，$E[c]$ 是随机消费的期望值。由于 $V''(c) < 0$，我们有 $V(E[c]) > E[V(c)]$，这就是詹森不等式。

### 3.5.6 例子

**随机生产函数下的最优投资**

考虑：
$$A_{t+1} = \rho A_t + \varepsilon_{t+1}, \quad \varepsilon_{t+1} \sim N(0, \sigma^2)$$
$$y_t = A_t k_t^\alpha$$
$$u(c) = \ln(c)$$

**贝尔曼方程**：
$$V(k, A) = \max_{k'} \{\ln(A k^\alpha - k') + \beta E[V(k', A')|A]\}$$

**一阶条件**：
$$\frac{1}{A k^\alpha - k'} = \beta E\left[\frac{\alpha A' k'^{\alpha-1}}{A' k'^\alpha - k''}\right]$$

这个方程显示了随机冲击如何影响最优投资决策。当当前生产率 $A$ 较高时，投资回报更高，因此家庭会增加投资。

**数值结果**（示例）：
- 如果 $\rho = 0.9$，$\sigma = 0.1$，$\alpha = 0.33$，$\beta = 0.96$
- 投资率 $k'/y$ 在 $20\%$ 到 $30\%$ 之间波动
- 高生产率时期投资率更高，体现了"预防性储蓄"和"投资时机选择"的权衡

## 本章总结

### 核心要点

1. **最优化原理**是动态规划的基础，它将多期问题分解为一系列相关的一期问题

2. **贝尔曼方程**是动态规划的核心工具，它建立了值函数的递归关系

3. **值函数**具有凹性、单调性、连续性和可微性等重要性质，这些性质保证了最优策略的存在性和唯一性

4. **价值迭代**和**策略迭代**是求解动态规划问题的两种主要算法

5. **不确定性**下的动态规划需要考虑随机冲击和期望计算，这是实际应用中的常见情况

### 重要概念表格

| 概念 | 定义/公式 | 经济学意义 |
|------|-----------|------------|
| 贝尔曼最优化原理 | 子问题必须是最优的 | 保证整体最优性 |
| 值函数 | $V(k) = \max_{c,k'}\{u(c) + \beta V(k')\}$ | 最优路径的效用现值 |
| 策略函数 | $c = g_c(k), k' = g_k(k)$ | 状态到最优决策的映射 |
| 价值迭代 | $V_{s+1} = \mathcal{T}V_s$ | 通过迭代求解值函数 |
| 欧拉方程 | $u'(c_t) = \beta(1+r)E[u'(c_{t+1})]$ | 跨期最优条件 |

| 保留工资 | $w^* = b + \beta E[V(w')](1-\beta)$ | 接受工作的临界工资 |
| 风险溢价 | $r^e = \frac{1}{\beta} - 1 + \sigma \gamma \sigma_c$ | 补偿不确定性的额外收益 |
