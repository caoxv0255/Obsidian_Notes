# 第7章：货币政策

> 本章学习目标：
> - 理解货币的需求和供给
> - 掌握货币政策传导机制

## 7.1 货币需求

### 7.1.1 货币需求的定义

**货币需求（Money Demand）**：
家庭和企业愿意持有的货币数量。

### 7.1.2 古典货币数量论

**货币数量论（Quantity Theory of Money）**：
货币数量与价格水平之间存在关系。

**货币数量方程（Quantity Equation）**：
```
M·V = P·Y
```
**其中**：
- M：货币供给
- V：货币流通速度
- P：价格水平
- Y：实际产出

**费雪方程（Fisher Equation）**：
```
M·V = P·Y
```
**含义**：
- 货币供给上升 ⇒ 价格水平上升（假设V和Y不变）

### 7.1.3 凯恩斯货币需求理论

**凯恩斯货币需求理论（Keynesian Money Demand Theory）**：
货币需求取决于利率和收入。

**货币需求函数（Money Demand Function）**：
```
M^d/P = L(i, Y)
```
**其中**：
- M^d/P：实际货币需求
- i：名义利率
- Y：实际收入

**性质**：
1. L_i < 0：货币需求与利率负相关
2. L_Y > 0：货币需求与收入正相关

**交易动机（Transactions Motive）**：
- 货币用于交易
- 收入上升 ⇒ 货币需求上升

**预防动机（Precautionary Motive）**：
- 货币用于预防不确定性
- 收入上升 ⇒ 货币需求上升

**投机动机（Speculative Motive）**：
- 货币用于投机
- 利率上升 ⇒ 货币需求下降

### 7.1.4 弗里德曼货币需求理论

**弗里德曼货币需求理论（Friedman Money Demand Theory）**：
货币需求取决于永久收入、利率、预期通货膨胀率等。

**货币需求函数（Money Demand Function）**：
```
M^d/P = f(Y^P, r_b - r_m, r_e - r_m, π^e)
```
**其中**：
- M^d/P：实际货币需求
- Y^P：永久收入
- r_b：债券利率
- r_m：货币利率
- r_e：股票利率
- π^e：预期通货膨胀率

**含义**：
- 永久收入上升 ⇒ 货币需求上升
- 债券利率上升 ⇒ 货币需求下降
- 股票利率上升 ⇒ 货币需求下降
- 预期通货膨胀率上升 ⇒ 货币需求下降

## 7.2 货币供给

### 7.2.1 货币供给的定义

**货币供给（Money Supply）**：
中央银行和银行系统提供的货币数量。

### 7.2.2 货币乘数

**货币乘数（Money Multiplier）**：
基础货币与货币供给的比值。

**基础货币（Base Money）**：
```
B = C + R
```
**其中**：
- B：基础货币
- C：通货
- R：准备金

**货币供给（Money Supply）**：
```
M = C + D
```
**其中**：
- M：货币供给
- D：活期存款

**货币乘数（Money Multiplier）**：
```
m = M / B = (C + D) / (C + R)
```

**简化假设**：
- 通货比率：c = C / D
- 准备金比率：r = R / D

**货币乘数**：
```
m = (1 + c) / (c + r)
```

### 7.2.3 货币供给的控制

**货币供给的控制（Control of Money Supply）**：

1. **公开市场操作（Open Market Operations）**：
   - 中央银行买卖政府债券
   - 买入债券 ⇒ 基础货币上升 ⇒ 货币供给上升
   - 卖出债券 ⇒ 基础货币下降 ⇒ 货币供给下降

2. **准备金要求（Reserve Requirements）**：
   - 中央银行规定准备金比率
   - 准备金比率上升 ⇒ 货币乘数下降 ⇒ 货币供给下降

3. **贴现率（Discount Rate）**：
   - 中央银行向银行提供贷款的利率
   - 贴现率下降 ⇒ 银行借款增加 ⇒ 准备金增加 ⇒ 货币供给上升

## 7.3 货币政策传导机制

### 7.3.1 利率传导机制

**利率传导机制（Interest Rate Channel）**：
货币供给变化 ⇒ 利率变化 ⇒ 投资和消费变化 ⇒ 产出变化。

**过程**：
1. 货币供给上升 ⇒ 利率下降
2. 利率下降 ⇒ 投资上升
3. 投资上升 ⇒ 产出上升

### 7.3.2 资产价格传导机制

**资产价格传导机制（Asset Price Channel）**：
货币供给变化 ⇒ 资产价格变化 ⇒ 财富变化 ⇒ 消费变化 ⇒ 产出变化。

**过程**：
1. 货币供给上升 ⇒ 利率下降 ⇒ 股票价格上升
2. 股票价格上升 ⇒ 财富上升 ⇒ 消费上升
3. 消费上升 ⇒ 产出上升

### 7.3.3 汇率传导机制

**汇率传导机制（Exchange Rate Channel）**：
货币供给变化 ⇒ 利率变化 ⇒ 汇率变化 ⇒ 净出口变化 ⇒ 产出变化。

**过程**：
1. 货币供给上升 ⇒ 利率下降 ⇒ 资本流出 ⇒ 汇率下降
2. 汇率下降 ⇒ 净出口上升
3. 净出口上升 ⇒ 产出上升

### 7.3.4 信贷传导机制

**信贷传导机制（Credit Channel）**：
货币供给变化 ⇒ 银行信贷变化 ⇒ 投资和消费变化 ⇒ 产出变化。

**银行信贷渠道（Bank Lending Channel）**：
1. 货币供给上升 ⇒ 银行准备金增加 ⇒ 银行信贷增加
2. 银行信贷增加 ⇒ 投资和消费增加
3. 投资和消费增加 ⇒ 产出增加

**资产负债表渠道（Balance Sheet Channel）**：
1. 货币供给上升 ⇒ 利率下降 ⇒ 企业净值上升
2. 企业净值上升 ⇒ 信贷约束放松 ⇒ 投资增加
3. 投资增加 ⇒ 产出增加

## 7.4 货币政策规则

### 7.4.1 泰勒规则

**泰勒规则（Taylor Rule）**：
货币政策规则，指导中央银行如何设定利率。

**公式**：
```
i_t = r* + π_t + 0.5(π_t - π*) + 0.5(y_t - y*)
```
**其中**：
- i_t：第t期的名义利率
- r*：均衡实际利率
- π_t：第t期的通货膨胀率
- π*：目标通货膨胀率
- y_t：第t期的实际产出
- y*：潜在产出

**含义**：
- 通货膨胀率高于目标 ⇒ 利率上升
- 产出低于潜在 ⇒ 利率下降

### 7.4.2 通货膨胀目标制

**通货膨胀目标制（Inflation Targeting）**：
中央银行设定通货膨胀目标，并以此指导货币政策。

**特点**：
1. 明确的通货膨胀目标（通常为2%）
2. 货币政策透明度
3. 中央银行问责制

### 7.4.3 货币供给目标制

**货币供给目标制（Monetary Targeting）**：
中央银行设定货币供给增长率目标。

**特点**：
1. 明确的货币供给增长率目标
2. 货币供给可测
3. 货币供给与通货膨胀的关系稳定

**局限性**：
- 货币供给与通货膨胀的关系不稳定
- 货币供给难以控制

## 7.5 货币政策的有效性

### 7.5.1 货币政策的长期中性

**货币政策的长期中性（Long-Run Neutrality of Money）**：
货币供给变化在长期只影响价格水平，不影响实际产出。

**含义**：
- 货币供给上升 ⇒ 价格水平上升
- 货币供给上升 ⇒ 实际产出不变（长期）

### 7.5.2 货币政策的短期非中性

**货币政策的短期非中性（Short-Run Non-Neutrality of Money）**：
货币供给变化在短期内影响实际产出。

**原因**：
1. **价格粘性（Price Stickiness）**：
   - 价格调整缓慢
   - 货币供给上升 ⇒ 实际货币供给上升 ⇒ 利率下降 ⇒ 产出上升

2. **信息不对称（Information Asymmetry）**：
   - 个体无法区分货币冲击和实际冲击
   - 货币供给上升 ⇒ 个体误认为实际冲击 ⇒ 产出上升

3. **名义刚性（Nominal Rigidities）**：
   - 工资和价格具有刚性
   - 货币供给上升 ⇒ 实际工资和实际价格下降 ⇒ 产出上升

### 7.5.3 货币政策的局限性

**货币政策的局限性（Limitations of Monetary Policy）**：

1. **流动性陷阱（Liquidity Trap）**：
   - 利率接近零
   - 货币政策无效

2. **预期陷阱（Expectations Trap）**：
   - 预期通货膨胀率上升
   - 货币政策无效

3. **时间滞后（Time Lags）**：
   - 货币政策传导存在时间滞后
   - 货币政策效果不确定

## 7.6 本章总结

### 核心要点

1. **货币需求**：
   - 货币需求的定义：家庭和企业愿意持有的货币数量
   - 古典货币数量论：M·V = P·Y
   - 凯恩斯货币需求理论：M^d/P = L(i, Y)
   - 弗里德曼货币需求理论：M^d/P = f(Y^P, r_b - r_m, r_e - r_m, π^e)

2. **货币供给**：
   - 货币供给的定义：中央银行和银行系统提供的货币数量
   - 货币乘数：m = (1 + c) / (c + r)
   - 货币供给的控制：公开市场操作、准备金要求、贴现率

3. **货币政策传导机制**：
   - 利率传导机制：货币供给变化 ⇒ 利率变化 ⇒ 投资和消费变化 ⇒ 产出变化
   - 资产价格传导机制：货币供给变化 ⇒ 资产价格变化 ⇒ 财富变化 ⇒ 消费变化 ⇒ 产出变化
   - 汇率传导机制：货币供给变化 ⇒ 利率变化 ⇒ 汇率变化 ⇒ 净出口变化 ⇒ 产出变化
   - 信贷传导机制：货币供给变化 ⇒ 银行信贷变化 ⇒ 投资和消费变化 ⇒ 产出变化

4. **货币政策规则**：
   - 泰勒规则：i_t = r* + π_t + 0.5(π_t - π*) + 0.5(y_t - y*)
   - 通货膨胀目标制：中央银行设定通货膨胀目标
   - 货币供给目标制：中央银行设定货币供给增长率目标

5. **货币政策的有效性**：
   - 货币政策的长期中性：货币供给变化在长期只影响价格水平
   - 货币政策的短期非中性：货币供给变化在短期内影响实际产出
   - 货币政策的局限性：流动性陷阱、预期陷阱、时间滞后
