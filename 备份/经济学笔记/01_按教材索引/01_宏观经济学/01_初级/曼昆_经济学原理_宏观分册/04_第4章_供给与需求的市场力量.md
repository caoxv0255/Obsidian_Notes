# 第4章：供给与需求的市场力量

> 本章学习目标：
> - 理解需求、供给和市场均衡的概念
> - 掌握供给和需求曲线的移动因素
> - 学会分析市场均衡的变化
> - 理解价格机制如何配置资源
> - 了解价格管制的经济影响

---

## 4.1 市场类型

### 4.1.1 市场的定义

**市场（Market）**：
某种商品或服务买卖的场所或机制。

**市场的类型**：

| 类型 | 特点 | 例子 |
|------|------|------|
| **完全竞争市场** | 买者和卖者众多，产品同质 | 农产品市场 |
| **垄断市场** | 只有一个卖者 | 公用事业 |
| **垄断竞争市场** | 卖者众多，产品有差异 | 餐饮、零售 |
| **寡头垄断市场** | 几个大卖者 | 汽车、航空 |

**本章重点**：
本章主要研究**完全竞争市场**，这是市场机制的基础。

### 4.1.2 完全竞争市场的特征

**特征**：
1. **众多买者和卖者**：
   - 没有任何一个买者或卖者能够影响市场价格
   - 每个参与者都是"价格接受者"（Price Taker）

2. **产品同质**：
   - 所有卖者提供的商品完全相同
   - 买者不在意从哪个卖者购买

3. **自由进入和退出**：
   - 卖者可以自由进入或退出市场
   - 没有进入壁垒

4. **完全信息**：
   - 买者和卖者都了解商品价格和质量
   - 没有信息不对称

**现实中的近似**：
- **农产品市场**：小麦、玉米、大豆
- **金融市场**：外汇市场、股票市场
- **部分商品市场**：黄金、白银

## 4.2 需求

### 4.2.1 需求的定义

**需求（Demand）**：
在一定时期内，在各种可能的价格水平下，买者愿意并且能够购买的商品数量。

**关键要素**：
1. **愿意（Willing）**：买者有购买意愿
2. **能够（Able）**：买者有支付能力
3. **一定时期内**：特定时间范围
4. **各种价格**：价格和数量的关系

**需求量（Quantity Demanded）**：
在某一特定价格下，买者愿意并且能够购买的数量。

**注意**：需求（D）和需求量（Qd）是不同的概念！

### 4.2.2 需求曲线

**需求曲线（Demand Curve）**：
表示价格和需求量之间关系的图形。

**图形表示**：

```mermaid

    title "需求曲线"
    x-axis "需求量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**特点**：
- **向下倾斜**：价格越高，需求量越低
- **负斜率**：表示价格和需求量呈反比关系

**需求表**：

| 价格（美元） | 需求量（个） |
|-------------|-------------|
| 10 | 0 |
| 8 | 1 |
| 6 | 2 |
| 4 | 3 |
| 2 | 4 |
| 0 | 5 |

### 4.2.3 需求定理

**需求定理（Law of Demand）：
在其他条件不变的情况下，价格上升，需求量减少；价格下降，需求量增加。

**数学表达**：
```
Qd = f(P), dQd/dP < 0
```

其中：
- Qd：需求量
- P：价格
- dQd/dP < 0：需求曲线向下倾斜

**直观理解**：
1. **替代效应（Substitution Effect）**：
   - 价格上升 → 替代品变得更便宜 → 转向替代品
   - 价格下降 → 本商品更便宜 → 从替代品转向本商品

2. **收入效应（Income Effect）**：
   - 价格上升 → 实际收入下降 → 减少购买
   - 价格下降 → 实际收入上升 → 增加购买

**例外情况**：
1. **吉芬商品（Giffen Goods）**：
   - 价格上升 → 需求量增加
   - 原因：收入效应大于替代效应
   - 例子：贫困时期的土豆

2. **凡勃伦商品（Veblen Goods）**：
   - 价格上升 → 需求量增加
   - 原因：炫耀性消费
   - 例子：奢侈品、名表

### 4.2.4 需求曲线的移动

**需求曲线的移动 vs 沿需求曲线移动**：

| 变化类型 | 表现 | 原因 |
|----------|------|------|
| **沿需求曲线移动** | 价格变化，需求量变化 | 价格本身的变化 |
| **需求曲线移动** | 整条曲线移动 | 非价格因素的变化 |

**需求曲线右移（需求增加）**：
```mermaid

    x-axis "需求量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**需求曲线左移（需求减少）**：
```mermaid
xychart-beta
    title "需求曲线"
    x-axis "需求量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**影响需求的因素（需求曲线移动的因素）**：

1. **收入（Income）**：
   - 正常商品（Normal Goods）：收入增加 → 需求增加
   - 低档商品（Inferior Goods）：收入增加 → 需求减少

2. **相关商品的价格**：
   - **替代品（Substitutes）**：
     * 替代品价格上升 → 本商品需求增加
     * 例子：可口可乐 vs 百事可乐
   - **互补品（Complements）**：
     * 互补品价格上升 → 本商品需求减少
     * 例子：汽车 vs 汽油

3. **偏好（Tastes）：
   - 偏好增加 → 需求增加
   - 例子：健康意识增强 → 健康食品需求增加

4. **预期（Expectations）**：
   - 价格预期上升 → 当前需求增加
   - 收入预期下降 → 当前需求减少

5. **买者数量（Number of Buyers）**：
   - 买者增加 → 需求增加
   - 例子：人口增长、季节性变化

**总结表**：

| 因素变化 | 需求曲线移动 | 原因 |
|----------|-------------|------|
| 收入增加（正常商品） | 右移 | 有更多钱购买 |
| 收入增加（低档商品） | 左移 | 转向更好商品 |
| 替代品价格上升 | 右移 | 转向本商品 |
| 互补品价格上升 | 左移 | 互补品使用减少 |
| 偏好增强 | 右移 | 更喜欢该商品 |
| 预期价格上升 | 右移 | 提前购买 |
| 买者数量增加 | 右移 | 更多人购买 |

## 4.3 供给

### 4.3.1 供给的定义

**供给（Supply）**：
在一定时期内，在各种可能的价格水平下，卖者愿意并且能够出售的商品数量。

**关键要素**：
1. **愿意（Willing）**：卖者有出售意愿
2. **能够（Able）**：卖者有生产能力
3. **一定时期内**：特定时间范围
4. **各种价格**：价格和数量的关系

**供给量（Quantity Supplied）**：
在某一特定价格下，卖者愿意并且能够出售的数量。

### 4.3.2 供给曲线

**供给曲线（Supply Curve）**：
表示价格和供给量之间关系的图形。

**图形表示**：

```mermaid

    x-axis "供给量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**特点**：
- **向上倾斜**：价格越高，供给量越高
- **正斜率**：表示价格和供给量呈正比关系

**供给表**：

| 价格（美元） | 供给量（个） |
|-------------|-------------|
| 0 | 0 |
| 2 | 1 |
| 4 | 2 |
| 6 | 3 |
| 8 | 4 |
| 10 | 5 |

### 4.3.3 供给定理

**供给定理（Law of Supply）**：
在其他条件不变的情况下，价格上升，供给量增加；价格下降，供给量减少。

**数学表达**：
```
Qs = g(P), dQs/dP > 0
```

其中：
- Qs：供给量
- P：价格
- dQs/dP > 0：供给曲线向上倾斜

**直观理解**：
1. **利润动机**：
   - 价格上升 → 利润增加 → 生产更多
   - 价格下降 → 利润减少 → 生产更少

2. **边际成本递增**：
   - 产量增加 → 边际成本上升
   - 只有价格更高才能覆盖更高成本

3. **生产能力限制**：
   - 价格低时，只生产低成本的产量
   - 价格高时，可以承担更高的生产成本

### 4.3.4 供给曲线的移动

**供给曲线的移动 vs 沿供给曲线移动**：

| 变化类型 | 表现 | 原因 |
|----------|------|------|
| **沿供给曲线移动** | 价格变化，供给量变化 | 价格本身的变化 |
| **供给曲线移动** | 整条曲线移动 | 非价格因素的变化 |

**供给曲线右移（供给增加）**：
```mermaid

    title "需求曲线"
    x-axis "供给量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**供给曲线左移（供给减少）**：
```mermaid
xychart-beta
    title "需求曲线"
    x-axis "供给量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**影响供给的因素（供给曲线移动的因素）**：

1. **投入价格（Input Prices）**：
   - 投入价格上升 → 成本增加 → 供给减少（左移）
   - 投入价格下降 → 成本减少 → 供给增加（右移）
   - 例子：工资上升 → 劳动密集型产品供给减少

2. **技术（Technology）**：
   - 技术进步 → 生产率提高 → 供给增加（右移）
   - 技术倒退 → 生产率下降 → 供给减少（左移）
   - 例子：自动化技术 → 供给增加

3. **预期（Expectations）**：
   - 预期价格上升 → 当前供给减少（左移）
   - 预期价格下降 → 当前供给增加（右移）
   - 例子：农民预期明年粮价高 → 囤积粮食

4. **卖者数量（Number of Sellers）**：
   - 卖者增加 → 供给增加（右移）
   - 卖者减少 → 供给减少（左移）
   - 例子：新企业进入市场

5. **自然条件（Natural Conditions）**：
   - 自然条件好 → 供给增加（右移）
   - 自然条件差 → 供给减少（左移）
   - 例子：天气好 → 农产品供给增加

**总结表**：

| 因素变化 | 供给曲线移动 | 原因 |
|----------|-------------|------|
| 投入价格上升 | 左移 | 成本增加 |
| 技术进步 | 右移 | 生产率提高 |
| 预期价格上升 | 左移 | 囤积商品 |
| 卖者数量增加 | 右移 | 更多生产者 |
| 自然条件改善 | 右移 | 更高产量 |

## 4.4 供给与需求的结合

### 4.4.1 市场均衡

**市场均衡（Market Equilibrium）**：
供给和需求相等的状态。

**均衡价格（Equilibrium Price, P*）**：
供给量等于需求量时的价格。

**均衡数量（Equilibrium Quantity, Q*）**：
供给量等于需求量时的数量。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**交点（E点）**：供给曲线和需求曲线的交点

**均衡条件**：
```
Qd(P*) = Qs(P*)
```

**计算示例**：

**需求函数**：Qd = 10 - P
**供给函数**：Qs = P

**求解均衡**：
```
Qd = Qs
10 - P = P
2P = 10
P* = 5

Q* = P* = 5
```

**均衡**：P* = 5，Q* = 5

### 4.4.2 均衡的动态调整

**动态调整过程**：

**供过于求（Surplus）**：
- 价格高于均衡价格
- 供给量 > 需求量
- 卖者无法卖出所有商品
- 价格下降
- 调整过程：P → P*

**供不应求（Shortage）**：
- 价格低于均衡价格
- 需求量 > 供给量
- 买者无法买到所有商品
- 价格上升
- 调整过程：P → P*

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**重要性质**：
- **自我稳定（Self-correcting）**：市场倾向于回到均衡
- **价格机制（Price Mechanism）**：价格调节资源配置
- **无政府干预**：市场会自动达到均衡

### 4.4.3 均衡的变化

**分析均衡变化的步骤**：
1. 确定事件影响需求还是供给
2. 确定需求或供给曲线如何移动
3. 画出新均衡
4. 比较新均衡和原均衡

**案例1：需求增加**

**事件**：消费者收入增加（商品是正常商品）

**分析**：
1. 收入增加 → 需求增加 → 需求曲线右移
2. 供给不变 → 供给曲线不移动

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结果**：
- 均衡价格上升（P2 > P1）
- 均衡数量增加（Q2 > Q1）

**案例2：供给减少**

**事件**：投入价格上升

**分析**：
1. 投入价格上升 → 供给减少 → 供给曲线左移
2. 需求不变 → 需求曲线不移动

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结果**：
- 均衡价格上升（P2 > P1）
- 均衡数量减少（Q2 < Q1）

**案例3：需求和供给同时变化**

**事件**：收入增加和技术进步同时发生

**分析**：
1. 收入增加 → 需求增加 → 需求曲线右移
2. 技术进步 → 供给增加 → 供给曲线右移

**图形**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结果**：
- 均衡数量增加（Q2 > Q1）
- 均衡价格不确定：取决于移动幅度

**总结表**：

| 变化 | 需求曲线 | 供给曲线 | 均衡价格 | 均衡数量 |
|------|----------|----------|----------|----------|
| 需求增加 | 右移 | 不变 | 上升 | 增加 |
| 需求减少 | 左移 | 不变 | 下降 | 减少 |
| 供给增加 | 不变 | 右移 | 下降 | 增加 |
| 供给减少 | 不变 | 左移 | 上升 | 减少 |
| 需求供给都增加 | 右移 | 右移 | 不确定 | 增加 |
| 需求增加供给减少 | 右移 | 左移 | 上升 | 不确定 |

## 4.5 弹性及其应用

### 4.5.1 需求价格弹性

**需求价格弹性（Price Elasticity of Demand）**：
需求量对价格变化的敏感程度。

**定义公式**：
```
Ed = (需求量变化的百分比) / (价格变化的百分比)
   = (ΔQ/Q) / (ΔP/P)
   = (ΔQ/ΔP) × (P/Q)
```

**计算示例**：

**例1**：
- 价格从4元上升到5元（ΔP = 1元）
- 需求量从120个下降到80个（ΔQ = -40个）
- 弹性：Ed = (-40/120) / (1/4) = -0.33/0.25 = -1.33

**弹性类型**：

| 弹性值 | 弹性类型 | 例子 |
|--------|----------|------|
| |Ed| > 1 | 富有弹性 | 奢侈品、有替代品的商品 |
| |Ed| = 1 | 单位弹性 | - |
| |Ed| < 1 | 缺乏弹性 | 必需品、无替代品的商品 |

**影响因素**：
1. **替代品的可获得性**：
   - 替代品多 → 弹性大
   - 替代品少 → 弹性小

2. **必需程度**：
   - 必需品 → 弹性小
   - 奢侈品 → 弹性大

3. **时间跨度**：
   - 长期 → 弹性大
   - 短期 → 弹性小

4. **预算占比**：
   - 占比小 → 弹性大
   - 占比大 → 弹性小

### 4.5.2 供给价格弹性

**供给价格弹性（Price Elasticity of Supply）**：
供给量对价格变化的敏感程度。

**定义公式**：
```
Es = (供给量变化的百分比) / (价格变化的百分比)
   = (ΔQ/Q) / (ΔP/P)
   = (ΔQ/ΔP) × (P/Q)
```

**弹性类型**：

| 弹性值 | 弹性类型 | 例子 |
|--------|----------|------|
| |Es| > 1 | 富有弹性 | 容易调整产量的行业 |
| |Es| = 1 | 单位弹性 | - |
| |Es| < 1 | 缺乏弹性 | 难以调整产量的行业 |

**影响因素**：
1. **生产调整的难易程度**：
   - 容易调整 → 弹性大
   - 难以调整 → 弹性小

2. **时间跨度**：
   - 长期 → 弹性大
   - 短期 → 弹性小

3. **生产能力的限制**：
   - 有闲置产能 → 弹性大
   - 已充分利用 → 弹性小

## 4.6 市场效率

### 4.6.1 消费者剩余和生产者剩余

**消费者剩余（Consumer Surplus）**：
买者愿意支付的价格减去实际支付的价格。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**消费者剩余 = 需求曲线下方、均衡价格上方的面积**

**生产者剩余（Producer Surplus）**：
卖者实际获得的价格减去愿意接受的价格。

**生产者剩余 = 供给曲线上方、均衡价格下方的面积**

**总剩余（Total Surplus）**：
消费者剩余 + 生产者剩余

**图形表示**：

```mermaid
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

### 4.6.2 市场效率

**市场效率（Market Efficiency）**：
资源配置使总剩余最大化的状态。

**福利经济学第一定理（First Theorem of Welfare Economics）：
完全竞争市场的均衡是有效率的。

**含义**：
1. **资源配置最优**：总剩余最大化
2. **帕累托最优（Pareto Optimal）**：无法在不使任何人状况变差的情况下使任何人状况变好
3. **无谓损失（Deadweight Loss）**：没有效率损失

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

在均衡点E：
- 消费者剩余最大
- 生产者剩余最大
- 总剩余最大
- 无谓损失为0

## 4.7 价格管制

### 4.7.1 价格上限

**价格上限（Price Ceiling）：
政府规定的商品最高价格。

**约束条件**：
- **有效价格上限**：低于均衡价格
- **无效价格上限**：高于均衡价格（无影响）

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**效果**：
1. **短缺（Shortage）**：Qd > Qs
2. **无谓损失**：三角形区域
3. **黑市（Black Market）**：非法交易

**例子**：
- 租金管制
- 汽油价格管制
- 医疗服务价格上限

### 4.7.2 价格下限

**价格下限（Price Floor）**：
政府规定的商品最低价格。

**约束条件**：
- **有效价格下限**：高于均衡价格
- **无效价格下限**：低于均衡价格（无影响）

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**效果**：
1. **过剩（Surplus）：**Qs > Qd
2. **无谓损失**：三角形区域
3. **政府购买**：政府购买过剩产品

**例子**：
- 最低工资
- 农产品价格支持
- 银行利率下限

## 4.8 本章总结

### 核心要点

1. **市场类型**
   - 完全竞争市场：众多买者和卖者，产品同质
   - 垄断市场：只有一个卖者
   - 本章重点研究完全竞争市场

2. **需求**
   - 需求定理：价格上升，需求量减少
   - 需求曲线向下倾斜
   - 需求曲线移动因素：收入、相关商品价格、偏好、预期、买者数量

3. **供给**
   - 供给定理：价格上升，供给量增加
   - 供给曲线向上倾斜
   - 供给曲线移动因素：投入价格、技术、预期、卖者数量、自然条件

4. **市场均衡**
   - 供给等于需求
   - 价格机制调节资源配置
   - 市场自动回到均衡

5. **弹性**
   - 需求价格弹性：需求量对价格的敏感度
   - 供给价格弹性：供给量对价格的敏感度

6. **市场效率**
   - 消费者剩余和生产者剩余
   - 完全竞争市场是有效率的
   - 总剩余最大化

7. **价格管制**
   - 价格上限：导致短缺和无谓损失
   - 价格下限：导致过剩和无谓损失

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 市场 | 买卖的场所或机制 | 配置资源 |
| 需求 | 买者的购买意愿和能力 | 向下倾斜 |
| 供给 | 卖者的出售意愿和能力 | 向上倾斜 |
| 均衡 | 供给等于需求 | 稳定状态 |
| 价格机制 | 价格调节资源配置 | "看不见的手" |
| 弹性 | 对价格变化的敏感度 | 影响政策效果 |
| 消费者剩余 | 愿意支付 - 实际支付 | 买者福利 |
| 生产者剩余 | 实际获得 - 愿意接受 | 卖者福利 |
| 价格上限 | 最高价格限制 | 导致短缺 |
| 价格下限 | 最低价格限制 | 导致过剩 |
