# 第6章：供给、需求与政府政策

> 本章学习目标：
> - 理解政府干预市场的原因和方式
> - 掌握价格上限的效果和无谓损失
> - 掌握价格下限的效果和无谓损失
> - 理解税收的经济影响和税负归宿
> - 学会分析政府政策的市场效率影响

## 6.1 政府干预市场的理由

### 6.1.1 市场失灵

**市场失灵（Market Failure）**：
市场机制不能有效配置资源的情况。

**市场失灵的类型**：
1. **垄断（Monopoly）**：
   - 市场权力导致高价格和低产量
   - 原因：规模经济、进入壁垒、网络效应

2. **外部性（Externalities）**：
   - 交易对第三方的影响未被市场反映
   - 负外部性：污染、噪音
   - 正外部性：教育、科研

3. **公共物品（Public Goods）**：
   - 非竞争性和非排他性
   - 市场无法有效提供
   - 例子：国防、路灯

4. **信息不对称（Information Asymmetry）**：
   - 买者和卖者掌握的信息不同
   - 导致逆向选择和道德风险

### 6.1.2 政府干预的目的

**政府干预的目的**：
1. **提高效率（Efficiency）**：
   - 减少市场失灵
   - 提高资源配置效率
   - 增加总剩余

2. **促进公平（Equity）**：
   - 减少收入不平等
   - 保护弱势群体
   - 实现社会公正

3. **稳定经济（Stability）**：
   - 减少经济波动
   - 促进充分就业
   - 控制通货膨胀

4. **实现特定目标（Specific Objectives）**：
   - 环境保护
   - 公共安全
   - 社会福利

### 6.1.3 政府干预的方式

**价格管制（Price Controls）**：
- 价格上限（Price Ceiling）
- 价格下限（Price Floor）

**数量管制（Quantity Controls）**：
- 配额（Quotas）
- 生产限制

**税收和补贴（Taxes and Subsidies）**：
- 税收（Taxes）
- 补贴（Subsidies）

**直接干预（Direct Intervention）**：
- 国有化（Nationalization）
- 直接提供公共物品
- 价格支持（Price Supports）

## 6.2 价格上限

### 6.2.1 价格上限的定义

**价格上限（Price Ceiling）**：
政府规定的商品最高价格。

**法律约束**：
- 卖者不得以高于上限的价格出售商品
- 但可以以低于上限的价格出售

**类型**：

| 类型 | 定义 | 效果 |
|------|------|------|
| **约束性价格上限** | 低于均衡价格 | 市场短缺、无谓损失 |
| **非约束性价格上限** | 高于均衡价格 | 无影响、市场均衡 |

**条件**：
- 价格上限 < 均衡价格 → 约束性 → 有影响
- 价格上限 > 均衡价格 → 非约束性 → 无影响

### 6.2.2 价格上限的效果分析

**约束性价格上限的效果**：

1. **短缺（Shortage）**：
   - 需求量 > 供给量
   - 短缺量 = Qd - Qs

2. **无谓损失（Deadweight Loss）**：
   - 总剩余减少
   - 原因：部分交易无法进行

3. **非价格配给（Non-Price Rationing）**：
   - 先到先得
   - 排队（Queues）
   - 配给券（Rationing）
   - 歧视性分配

4. **黑市（Black Market）**：
   - 非法交易
   - 价格高于上限

5. **质量下降（Quality Reduction）**：
   - 卖者降低产品质量
   - 减少服务质量

### 6.2.3 价格上限的福利分析

**消费者剩余（Consumer Surplus）**：
- 原消费者剩余：三角形A + 三角形B
- 新消费者剩余：三角形A + 三角形C
- 变化：-三角形B + 三角形C

**生产者剩余（Producer Surplus）**：
- 原生产者剩余：三角形B + 三角形D
- 新生产者剩余：三角形D
- 变化：-三角形B

**总剩余（Total Surplus）**：
- 原总剩余：三角形A + 三角形B + 三角形C + 三角形D
- 新总剩余：三角形A + 三角形C + 三角形D
- 无谓损失：-三角形B

**图形表示**：

**结论**：
- 消费者剩余可能增加或减少
- 生产者剩余减少
- 总剩余减少（无谓损失）
- 市场效率降低

### 6.2.4 价格上限的实际案例

**案例1：租金管制（Rent Control）**

**背景**：
- 政府对租金设定上限
- 目的：保护低收入租户

**效果**：
1. **短期效果**：
   - 租金降低
   - 租户受益
   - 住房短缺不严重

2. **长期效果**：
   - 房东减少维护
   - 新建住房减少
   - 住房短缺严重
   - 黑市租金

3. **福利影响**：
   - 短期：租户受益
   - 长期：租户和房东都受损

**案例2：汽油价格管制**

**背景**：
- 20世纪70年代石油危机
- 政府对汽油设定价格上限

**效果**：
1. **汽油短缺**：
   - 需求量 > 供给量
   - 加油站排队

2. **非价格配给**：
   - 先到先得
   - 配给券

3. **黑市**：
   - 非法高价汽油
   - 汽油走私

**案例3：自然灾害后的价格管制**

**背景**：
- 自然灾害（如飓风、地震）
- 商品价格暴涨
- 政府设定价格上限

**效果**：
1. **短期缓解**：
   - 价格降低
   - 消费者受益

2. **长期问题**：
   - 供给减少
   - 黑市出现
   - 质量下降

### 6.2.5 价格上限的优缺点

**优点**：
1. **保护消费者**：
   - 降低商品价格
   - 保护低收入群体

2. **政治受欢迎**：
   - 容易获得选民支持
   - 短期内效果明显

**缺点**：
1. **市场扭曲**：
   - 短缺
   - 无谓损失

2. **黑市**：
   - 非法交易
   - 犯罪增加

3. **质量下降**：
   - 产品质量降低
   - 服务质量下降

4. **长期负面影响**：
   - 供给减少
   - 新投资减少

## 6.3 价格下限

### 6.3.1 价格下限的定义

**价格下限（Price Floor）**：
政府规定的商品最低价格。

**法律约束**：
- 买者不得以低于下限的价格购买商品
- 但可以以高于下限的价格购买

**类型**：

| 类型 | 定义 | 效果 |
|------|------|------|
| **约束性价格下限** | 高于均衡价格 | 市场过剩、无谓损失 |
| **非约束性价格下限** | 低于均衡价格 | 无影响、市场均衡 |

**条件**：
- 价格下限 > 均衡价格 → 约束性 → 有影响
- 价格下限 < 均衡价格 → 非约束性 → 无影响

### 6.3.2 价格下限的效果分析

**约束性价格下限的效果**：

1. **过剩（Surplus）**：
   - 供给量 > 需求量
   - 过剩量 = Qs - Qd

2. **无谓损失（Deadweight Loss）**：
   - 总剩余减少
   - 原因：部分交易无法进行

3. **非价格配给（Non-Price Rationing）**：
   - 卖者选择买者
   - 优先给关系好的买者
   - 歧视性分配

4. **政府购买（Government Purchase）**：
   - 政府购买过剩产品
   - 增加政府支出

5. **黑市（Black Market）**：
   - 非法交易
   - 价格低于下限

- E点：市场均衡点（Pe, Qe）

- 过剩：Qs - Qd

### 6.3.3 价格下限的福利分析

**消费者剩余（Consumer Surplus）**：
- 原消费者剩余：三角形A + 三角形B
- 新消费者剩余：三角形A
- 变化：-三角形B

**生产者剩余（Producer Surplus）**：
- 原生产者剩余：三角形C
- 新生产者剩余：三角形B + 三角形C + 三角形D
- 变化：+三角形B + 三角形D

**总剩余（Total Surplus）**：
- 原总剩余：三角形A + 三角形B + 三角形C
- 新总剩余：三角形A + 三角形B + 三角形C + 三角形D
- 无谓损失：-三角形B - 三角形D

**如果政府购买过剩产品**：
- 政府支出：Pf × (Qs - Qd)
- 政府成本：矩形E
- 总社会成本：无谓损失 + 政府成本

```mermaid
    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结论**：
- 消费者剩余减少
- 生产者剩余可能增加
- 总剩余减少（无谓损失）
- 市场效率降低

### 6.3.4 价格下限的实际案例

**案例1：最低工资（Minimum Wage）**

**背景**：
- 政府设定最低工资标准
- 目的：保护低收入工人

**效果**：
1. **正面效果**：
   - 低收入工人工资提高
   - 减少贫困

2. **负面效果**：
   - 失业增加
   - 劳动需求减少
   - 黑市劳动

3. **弹性影响**：
   - 劳动需求弹性小 → 失业影响小
   - 劳动需求弹性大 → 失业影响大

**争议**：
- 经济学家对最低工资的效果有争议
- 实证研究显示效果不确定

**案例2：农产品价格支持（Agricultural Price Supports）**

**背景**：
- 政府对农产品设定价格下限
- 目的：保护农民收入

**效果**：
1. **农民收入增加**：
   - 价格提高
   - 收入增加

2. **农产品过剩**：
   - 供给 > 需求
   - 政府购买过剩产品

3. **政府支出增加**：
   - 购买过剩产品
   - 储存成本

4. **资源配置扭曲**：
   - 过度生产
   - 土地过度使用

**案例3：汇率下限**

**背景**：
- 政府设定汇率下限
- 目的：保护出口

**效果**：
1. **短期效果**：
   - 出口增加
   - 进口减少

2. **长期问题**：
   - 贸易伙伴报复
   - 货币贬值压力
   - 外汇储备减少

### 6.3.5 价格下限的优缺点

**优点**：
1. **保护生产者**：
   - 提高商品价格
   - 保护低收入生产者

2. **政治受欢迎**：
   - 容易获得生产者支持
   - 短期内效果明显

**缺点**：
1. **市场扭曲**：
   - 过剩
   - 无谓损失

2. **政府负担**：
   - 政府购买过剩产品
   - 增加政府支出

3. **黑市**：
   - 非法交易
   - 价格低于下限

4. **长期负面影响**：
   - 需求减少
   - 消费者剩余减少

## 6.4 税收

### 6.4.1 税收的定义和类型

**税收（Tax）**：
政府强制征收的收入，用于提供公共物品和服务。

**税收的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **从量税（Per-Unit Tax）** | 每单位商品征收固定税额 | 汽油税、烟草税 |
| **从价税（Ad Valorem Tax）** | 按商品价格征收一定比例 | 销售税、增值税 |
| **所得税（Income Tax）** | 按收入征收 | 个人所得税、企业所得税 |
| **财产税（Property Tax）** | 按财产价值征收 | 房产税、遗产税 |

**本章重点**：从量税

### 6.4.2 税收的效果分析

**从量税的效果**：

1. **价格上升**：
   - 买者支付的价格上升
   - 卖者收到的价格下降

2. **数量减少**：
   - 均衡数量减少
   - 市场交易减少

3. **税负归宿（Tax Incidence）**：
   - 买者和卖者分担税负
   - 分担比例取决于弹性

4. **无谓损失（Deadweight Loss）**：
   - 总剩余减少
   - 原因：部分交易无法进行

**图形分析**：


- 买者税负：Pb - Pe
- 卖者税负：Pe - Ps
- 总税额：Pb - Ps

### 6.4.3 税收的福利分析

**消费者剩余（Consumer Surplus）**：
- 原消费者剩余：三角形A + 三角形B + 三角形C
- 新消费者剩余：三角形A
- 变化：-三角形B - 三角形C

**生产者剩余（Producer Surplus）**：
- 原生产者剩余：三角形D + 三角形E + 三角形F
- 新生产者剩余：三角形F
- 变化：-三角形D - 三角形E

**政府税收（Government Revenue）**：
- 税收收入：矩形B + 矩形D
- 单位：每单位税收 × 交易数量

**总剩余（Total Surplus）**：
- 原总剩余：三角形A + 三角形B + 三角形C + 三角形D + 三角形E + 三角形F
- 新总剩余：三角形A + 三角形B + 三角形D + 三角形F
- 无谓损失：-三角形C - 三角形E

```mermaid

    x-axis "数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结论**：
- 消费者剩余减少
- 生产者剩余减少
- 政府获得税收收入
- 总剩余减少（无谓损失）
- 市场效率降低

### 6.4.4 税负归宿（Tax Incidence）

**税负归宿的定义**：
税收最终由谁承担，而不是法律规定的谁承担。

**关键原则**：
> **税收归宿取决于供给和需求的相对弹性，而不是法律规定的谁支付**

**情况1：需求富有弹性，供给缺乏弹性**

**分析**：
- 需求富有弹性：买者对价格敏感
- 供给缺乏弹性：卖者对价格不敏感
- 税负主要由卖者承担

**图形**：

```mermaid

    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结论**：
- 买者税负：Pb - Pe（小）
- 卖者税负：Pe - Ps（大）
- 卖者承担大部分税负

**情况2：需求缺乏弹性，供给富有弹性**

**分析**：
- 需求缺乏弹性：买者对价格不敏感
- 供给富有弹性：卖者对价格敏感
- 税负主要由买者承担

**图形**：

```mermaid

    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**结论**：
- 买者税负：Pb - Pe（大）
- 卖者税负：Pe - Ps（小）
- 买者承担大部分税负

**总结表**：

| 情况 | 需求弹性 | 供给弹性 | 买者税负 | 卖者税负 |
|------|----------|----------|----------|----------|
| 1 | 富有弹性 | 缺乏弹性 | 小 | 大 |
| 2 | 缺乏弹性 | 富有弹性 | 大 | 小 |
| 3 | 弹性相同 | 弹性相同 | 相等 | 相等 |

### 6.4.5 税收的实际案例

**案例1：汽油税（Gasoline Tax）**

**背景**：
- 政府对汽油征收从量税
- 目的：减少污染、筹集收入

**效果**：
1. **需求弹性**：
   - 短期弹性小（-0.2）
   - 长期弹性大（-0.7）

2. **税负归宿**：
   - 短期：买者承担大部分
   - 长期：卖者承担更多

3. **环境影响**：
   - 汽油消费减少
   - 污染减少

**案例2：香烟税（Cigarette Tax）**

**背景**：
- 政府对香烟征收高额税
- 目的：减少吸烟、筹集收入

**效果**：
1. **需求弹性**：
   - 弹性中等（-0.4）

2. **税负归宿**：
   - 买者承担大部分税负
   - 卖者承担小部分

3. **健康影响**：
   - 吸烟减少
   - 健康改善

**案例3：奢侈品税（Luxury Tax）**

**背景**：
- 政府对奢侈品征收高税
- 目的：收入再分配

**效果**：
1. **需求弹性**：
   - 弹性大（-2.5）

2. **税负归宿**：
   - 卖者承担大部分税负
   - 买者承担小部分

3. **市场影响**：
   - 奢侈品销量大幅下降
   - 产业受损

### 6.4.6 无谓损失与弹性

**无谓损失（Deadweight Loss）**：
税收导致的效率损失。

**无谓损失与弹性的关系**：
> **弹性越大，无谓损失越大**

**原因**：
- 弹性大 → 税收导致数量变化大
- 数量变化大 → 交易减少多
- 交易减少多 → 无谓损失大

**图形对比**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量
    +---------------- --> 数量" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**政策含义**：
- 对弹性大的商品征税 → 无谓损失大
- 对弹性小的商品征税 → 无谓损失小
- 最优税收：对弹性小的商品征收高税率

## 6.5 补贴

### 6.5.1 补贴的定义

**补贴（Subsidy）**：
政府向生产者或消费者支付的转移支付。

**类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **生产补贴（Production Subsidy）** | 政府支付给生产者 | 农业补贴、能源补贴 |
| **消费补贴（Consumption Subsidy）** | 政府支付给消费者 | 教育补贴、住房补贴 |
| **出口补贴（Export Subsidy）** | 政府补贴出口 | 出口退税 |

### 6.5.2 补贴的效果分析

**生产补贴的效果**：

1. **价格下降**：
   - 买者支付的价格下降
   - 卖者收到的价格上升

2. **数量增加**：
   - 均衡数量增加
   - 市场交易增加

3. **政府支出增加**：
   - 政府支付补贴
   - 财政负担增加

4. **无谓损失（Deadweight Loss）**：
   - 总剩余减少
   - 原因：部分低效率的交易

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**图中标记**：
- S：供给曲线
- D：需求曲线
- S - Subsidy：补贴后的供给曲线
- Pb：买者支付的价格
- Ps：卖者收到的价格
- Q1：原均衡数量
- Q2：新均衡数量

**补贴分担**：
- 买者收益：Pe - Pb
- 卖者收益：Ps - Pe
- 总补贴：Ps - Pb

### 6.5.3 补贴的福利分析

**消费者剩余（Consumer Surplus）**：
- 原消费者剩余：三角形A
- 新消费者剩余：三角形A + 三角形B
- 变化：+三角形B

**生产者剩余（Producer Surplus）**：
- 原生产者剩余：三角形C
- 新生产者剩余：三角形C + 三角形D
- 变化：+三角形D

**政府成本（Government Cost）**：
- 补贴成本：矩形B + 矩形D + 矩形E
- 单位：每单位补贴 × 交易数量

**总剩余（Total Surplus）**：
- 原总剩余：三角形A + 三角形C
- 新总剩余：三角形A + 三角形B + 三角形C + 三角形D
- 净变化：+三角形B + 三角形D - (矩形B + 矩形D + 矩形E)
- 无谓损失：-矩形E

**结论**：
- 消费者剩余增加
- 生产者剩余增加
- 政府成本增加
- 总剩余减少（无谓损失）
- 市场效率降低

### 6.5.4 补贴的实际案例

**案例1：农业补贴（Agricultural Subsidy）**

**背景**：
- 政府向农民支付补贴
- 目的：保护农民收入、确保粮食安全

**效果**：
1. **农民收入增加**：
   - 农产品价格上升
   - 收入增加

2. **农产品过剩**：
   - 供给 > 需求
   - 政府购买过剩产品

3. **政府支出增加**：
   - 支付补贴
   - 储存成本

4. **国际贸易扭曲**：
   - 过度出口
   - 贸易伙伴报复

**案例2：教育补贴（Education Subsidy）**

**背景**：
- 政府向学生支付教育补贴
- 目的：提高教育水平、促进人力资本

**效果**：
1. **教育机会增加**：
   - 更多学生接受教育
   - 人力资本提高

2. **教育质量提高**：
   - 学校收入增加
   - 质量改善

3. **政府支出增加**：
   - 教育支出增加
   - 财政负担

**案例3：可再生能源补贴（Renewable Energy Subsidy）**

**背景**：
- 政府向可再生能源产业支付补贴
- 目的：促进环境保护、应对气候变化

**效果**：
1. **可再生能源增加**：
   - 太阳能、风能等增加
   - 污染减少

2. **技术进步**：
   - 产业投资增加
   - 技术创新

3. **政府支出增加**：
   - 补贴支出增加
   - 财政负担

## 6.6 本章总结

### 核心要点

1. **政府干预的原因**
   - 市场失灵（垄断、外部性、公共物品、信息不对称）
   - 促进公平
   - 稳定经济

2. **价格上限**
   - 低于均衡价格 → 约束性
   - 导致短缺、无谓损失、黑市
   - 消费者剩余可能增加或减少

3. **价格下限**
   - 高于均衡价格 → 约束性
   - 导致过剩、无谓损失、政府购买
   - 生产者剩余可能增加

4. **税收**
   - 导致价格上升、数量减少
   - 买者和卖者分担税负
   - 税负归宿取决于弹性
   - 无谓损失取决于弹性

5. **补贴**
   - 导致价格下降、数量增加
   - 买者和卖者分享收益
   - 政府成本增加
   - 存在无谓损失

6. **政策效果**
   - 所有干预都导致效率损失
   - 无谓损失取决于弹性
   - 政策需要权衡效率和公平

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 市场失灵 | 市场机制不能有效配置资源 | 垄断、外部性、公共物品、信息不对称 |
| 价格上限 | 最高价格限制 | 导致短缺 |
| 价格下限 | 最低价格限制 | 导致过剩 |
| 约束性 | 低于均衡价格（上限）或高于均衡价格（下限） | 有影响 |
| 非约束性 | 高于均衡价格（上限）或低于均衡价格（下限） | 无影响 |
| 短缺 | 需求量 > 供给量 | 价格上限的结果 |
| 过剩 | 供给量 > 需求量 | 价格下限的结果 |
| 无谓损失 | 总剩余减少 | 效率损失 |
| 税负归宿 | 税收最终由谁承担 | 取决于弹性 |
| 补贴 | 政府支付给生产者或消费者 | 导致数量增加 |
