# 第11章：公共物品和公共资源

> 本章学习目标：
> - 理解物品的不同类型
> - 掌握公共物品的特征和市场失灵
> - 掌握公共资源的特征和"公地悲剧"
> - 学会分析公共物品和公共资源的解决方案
> - 理解政府提供公共物品的作用

---

## 11.1 物品的类型

### 11.1.1 物品的分类

**物品的分类标准**：

1. **排他性（Excludability）**：
   - 可以阻止他人使用该物品
   - 例如：食品、衣服

2. **竞争性（Rivalry）**：
   - 一人使用会影响他人使用
   - 例如：食物、停车位

**物品的类型**：

| 类型 | 排他性 | 竞争性 | 例子 |
|------|--------|--------|------|
| **私人物品（Private Goods）** | 是 | 是 | 食品、衣服 |
| **公共物品（Public Goods）** | 否 | 否 | 国防、路灯 |
| **公共资源（Common Resources）** | 否 | 是 | 海洋鱼类、公共牧场 |
| **俱乐部物品（Club Goods）** | 是 | 否 | 有线电视、收费公路 |

### 11.1.2 私人物品

**私人物品（Private Goods）**：
既有排他性又有竞争性的物品。

**特征**：
1. **排他性**：
   - 可以阻止他人使用
   - 只有付费才能使用

2. **竞争性**：
   - 一人使用会影响他人使用
   - 数量有限

**市场机制**：
- 市场可以有效提供私人物品
- 价格机制调节资源配置
- 供需平衡

**例子**：
- 食品、衣服、汽车、房屋

### 11.1.3 公共物品

**公共物品（Public Goods）**：
既无排他性又无竞争性的物品。

**特征**：
1. **非排他性（Non-Excludability）**：
   - 无法阻止他人使用
   - 无法收费

2. **非竞争性（Non-Rivalry）**：
   - 一人使用不影响他人使用
   - 边际成本为零

**市场失灵**：
- 市场无法有效提供公共物品
- 搭便车问题
- 需要政府提供

**例子**：
- 国防、路灯、知识、天气预报

### 11.1.4 公共资源

**公共资源（Common Resources）**：
无排他性但有竞争性的物品。

**特征**：
1. **非排他性**：
   - 无法阻止他人使用
   - 任何人都可以使用

2. **竞争性**：
   - 一人使用会影响他人使用
   - 过度使用会导致枯竭

**市场失灵**：
- 市场无法有效管理公共资源
- 公地悲剧
- 需要政府干预

**例子**：
- 海洋鱼类、公共牧场、清洁空气

### 11.1.5 俱乐部物品

**俱乐部物品（Club Goods）**：
有排他性但无竞争性的物品。

**特征**：
1. **排他性**：
   - 可以阻止他人使用
   - 付费才能使用

2. **非竞争性**：
   - 一人使用不影响他人使用
   - 边际成本低

**市场机制**：
- 市场可以提供，但可能有供给不足
- 需要政府监管

**例子**：
- 有线电视、收费公路、电影

## 11.2 公共物品

### 11.2.1 公共物品的特征

**公共物品的两个关键特征**：

1. **非排他性（Non-Excludability）**：
   - 无法阻止他人使用
   - 一旦提供，所有人都能享受
   - 无法收费

2. **非竞争性（Non-Rivalry）**：
   - 一人使用不影响他人使用
   - 边际成本为零
   - 增加使用者不增加成本

**公共物品的例子**：

| 物品 | 非排他性 | 非竞争性 | 说明 |
|------|----------|----------|------|
| 国防 | 是 | 是 | 保护所有人，一人不影响他人 |
| 路灯 | 是 | 是 | 照亮所有人，一人不影响他人 |
| 知识 | 是 | 是 | 所有人都可以使用知识 |
| 天气预报 | 是 | 是 | 所有人都可以知道天气 |

### 11.2.2 搭便车问题

**搭便车问题（Free Rider Problem）**：
人们不愿为公共物品付费，但希望他人付费，自己受益。

**原因**：
1. **非排他性**：
   - 无法阻止不付费者使用
   - 不付费也能享受

2. **理性选择**：
   - 个人理性：不付费，享受他人付费的成果
   - 集体非理性：无人付费，公共物品无法提供

**数学分析**：

**假设**：
- 公共物品成本：C
- 每人支付意愿：WTP
- 人数：n

**如果所有人都付费**：
- 每人支付：C/n
- 每人收益：WTP
- 净收益：WTP - C/n > 0

**如果某人不付费**：
- 不支付者支付：0
- 不支付者收益：WTP
- 净收益：WPT > WTP - C/n

**结论**：
- 不付费的净收益更高
- 理性选择：不付费
- 结果：所有人都想搭便车
- 最终：公共物品无法提供

### 11.2.3 公共物品的提供

**政府提供公共物品**：

**政府的作用**：
1. **税收筹资**：
   - 强制征税
   - 解决搭便车问题

2. **公共提供**：
   - 政府直接提供
   - 确保公共物品供应

3. **效率考虑**：
   - 提供最优数量
   - 社会福利最大化

**公共物品的最优数量**：

**边际成本 = 边际收益**

```
MSC = MSB
```

其中：
- MSC：社会边际成本
- MSB：社会边际收益

**社会边际收益（MSB）**：
```
MSB = ΣMPBi
```

其中：
- MPBi：个人i的边际支付意愿

**图形分析**：

```mermaid

    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**政府面临的挑战**：

1. **确定社会边际收益**：
   - 难以测量每个人的支付意愿
   - 信息不对称

2. **确定社会边际成本**：
   - 成本可能不确定
   - 技术变化

3. **政治因素**：
   - 利益集团影响
   - 选民偏好

### 11.2.4 公共物品的类型

**公共物品的类型**：

| 类型 | 特点 | 例子 |
|------|------|------|
| **纯公共物品（Pure Public Goods）** | 完全非排他性和非竞争性 | 国防、路灯 |
| **准公共物品（Quasi-Public Goods）** | 部分排他性和部分竞争性 | 教育、医疗 |

**准公共物品**：
- 既有公共物品特征，又有私人物品特征
- 可以部分由市场提供
- 可以部分由政府提供

**例子**：
- **教育**：
  * 正外部性：社会受益
  * 部分排他性：可以选择学校
  * 市场和政府共同提供

- **医疗**：
  * 正外部性：公共健康
  * 部分排他性：可以选择医院
  * 市场和政府共同提供

## 11.3 公共资源

### 11.3.1 公共资源的特征

**公共资源的两个关键特征**：

1. **非排他性（Non-Excludability）**：
   - 无法阻止他人使用
   - 任何人都可以使用

2. **竞争性（Rivalry）**：
   - 一人使用会影响他人使用
   - 资源有限

**公共资源的例子**：

| 资源 | 非排他性 | 竞争性 | 说明 |
|------|----------|--------|------|
| 海洋鱼类 | 是 | 是 | 任何人都可以捕鱼，鱼有限 |
| 公共牧场 | 是 | 是 | 任何人都可以放牧，草有限 |
| 清洁空气 | 是 | 是 | 任何人都可以呼吸，空气有限 |
| 道路 | 是 | 是 | 任何人都可以使用，道路拥堵 |

### 11.3.2 公地悲剧

**公地悲剧（Tragedy of the Commons）**：
当资源是公共资源时，个人理性导致集体非理性，资源被过度使用。

**原因**：
1. **非排他性**：
   - 任何人都可以使用
   - 无法阻止他人使用

2. **竞争性**：
   - 资源有限
   - 一人使用影响他人

3. **个人理性**：
   - 个人：尽可能多地使用资源
   - 集体：资源被过度使用

**数学分析**：

**假设**：
- 公共牧场面积：A
- 每头牛收益：B
- 每头牛成本：C
- 养牛数量：n

**个人收益**：
```
个人收益 = B - C
```

**集体收益**：
```
集体收益 = n × (B - C)
```

**但是**：
- 养牛数量增加 → 牧草减少 → 每头牛收益下降
- B = B(n)，B随n增加而减少

**个人决策**：
- 只要 B(n) > C，就继续养牛
- 导致过度放牧

**社会最优**：
- 社会边际收益 = 社会边际成本
- 限制养牛数量

### 11.3.3 公共资源的解决方案

**方案1：政府管制（Government Regulation）**

**管制的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **数量管制（Quantity Regulation）** | 限制使用数量 | 捕鱼配额 |
| **技术标准（Technology Standard）** | 强制使用特定技术 | 汽车排放标准 |
| **市场管制（Market Regulation）** | 创建交易市场 | 碳交易市场 |

**效果**：
- 直接限制资源使用
- 防止过度使用
- 可能缺乏灵活性

**实际应用**：
- 捕鱼许可证
- 污染排放限制
- 建筑规范

**方案2：私有化（Privatization）**

**私有化的原理**：
- 将公共资源变为私人物品
- 建立产权
- 市场机制有效配置资源

**效果**：
- 产权明确 → 有效使用
- 防止过度使用
- 提高效率

**局限性**：
- 某些资源难以私有化
- 可能导致不公平
- 可能导致资源过度开发

**实际应用**：
- 土地私有化
- 水资源私有化
- 矿产资源私有化

**方案3：配额制度（Quota System）**

**配额制度的原理**：
- 政府设定总使用量
- 分配配额给使用者
- 允许配额交易

**效果**：
- 限制总使用量
- 配额交易提高效率
- 鼓励技术创新

**实际应用**：
- 捕鱼配额
- 碳排放配额
- 水资源配额

**方案4：社区管理（Community Management）**

**社区管理的原理**：
- 社区共同管理公共资源
- 建立社区规则
- 自我监督和执行

**效果**：
- 社区参与度高
- 规则适应性强
- 执行成本低

**实际应用**：
- 农村公共土地
- 森林资源
- 水资源

## 11.4 政府的作用

### 11.4.1 政府提供公共物品

**政府提供公共物品的原因**：

1. **市场失灵**：
   - 市场无法有效提供公共物品
   - 搭便车问题

2. **社会福利**：
   - 公共物品提高社会福利
   - 政府代表公共利益

3. **规模经济**：
   - 政府提供可以降低成本
   - 统一采购和管理

**政府提供公共物品的方式**：

| 方式 | 定义 | 例子 |
|------|------|------|
| **直接提供（Direct Provision）** | 政府直接生产 | 国防、警察 |
| **外包（Contracting Out）** | 政府付费，私人生产 | 垃圾处理、公共交通 |
| **补贴（Subsidy）** | 政府补贴私人提供 | 教育、医疗 |

### 11.4.2 政府管理公共资源

**政府管理公共资源的原因**：

1. **市场失灵**：
   - 市场无法有效管理公共资源
   - 公地悲剧

2. **可持续性**：
   - 防止资源枯竭
   - 保护环境

3. **公平性**：
   - 确保资源公平分配
   - 保护弱势群体

**政府管理公共资源的方式**：

| 方式 | 定义 | 例子 |
|------|------|------|
| **管制（Regulation）** | 设定使用规则 | 捕鱼配额 |
| **税收（Taxation）** | 对使用征税 | 碳税 |
| **产权界定（Property Rights）** | 明确产权 | 土地私有化 |

### 11.4.3 政府的局限性

**政府的局限性**：

1. **信息不对称（Information Asymmetry）**：
   - 政府难以获得完整信息
   - 决策可能不准确

2. **官僚低效（Bureaucratic Inefficiency）**：
   - 政府机构效率低
   - 决策缓慢

3. **政治干预（Political Interference）**：
   - 利益集团影响
   - 决策偏离最优

4. **寻租行为（Rent-Seeking）**：
   - 个人和集团为获得利益而游说政府
   - 导致资源浪费

**结论**：
- 政府干预有必要，但也有局限性
- 需要权衡政府干预的利弊
- 政府和市场相结合

## 11.5 本章总结

### 核心要点

1. **物品的类型**
   - 私人物品：排他性 + 竞争性
   - 公共物品：非排他性 + 非竞争性
   - 公共资源：非排他性 + 竞争性
   - 俱乐部物品：排他性 + 非竞争性

2. **公共物品**
   - 非排他性和非竞争性
   - 搭便车问题
   - 市场无法有效提供
   - 需要政府提供

3. **公共资源**
   - 非排他性和竞争性
   - 公地悲剧
   - 市场无法有效管理
   - 需要政府干预

4. **解决方案**
   - 公共物品：政府提供、税收筹资
   - 公共资源：管制、私有化、配额、社区管理

5. **政府的作用**
   - 提供公共物品
   - 管理公共资源
   - 但也有局限性

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 排他性 | 可以阻止他人使用 | 收费的前提 |
| 竞争性 | 一人使用影响他人使用 | 数量有限 |
| 私人物品 | 排他性 + 竞争性 | 市场有效提供 |
| 公共物品 | 非排他性 + 非竞争性 | 政府提供 |
| 公共资源 | 非排他性 + 竞争性 | 公地悲剧 |
| 俱乐部物品 | 排他性 + 非竞争性 | 市场提供 |
| 搭便车问题 | 不付费但享受公共物品 | 市场失灵 |
| 公地悲剧 | 个人理性导致集体非理性 | 过度使用 |
| 社会边际收益 | 所有人边际收益之和 | MSB = ΣMPBi |
| 最优数量 | 边际成本 = 边际收益 | MSC = MSB |
