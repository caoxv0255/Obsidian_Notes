## 14.1 什么是竞争市场

### 14.1.1 完全竞争市场的定义

**完全竞争市场（Perfectly Competitive Market）**：
有许多买者和卖者，销售相同产品的市场。

**完全竞争市场的特征**：

1. **众多买者和卖者（Many Buyers and Sellers）**：
   - 没有任何一个买者或卖者能够影响市场价格
   - 每个参与者都是"价格接受者"（Price Taker）

2. **产品同质（Identical Products）**：
   - 所有卖者提供的商品完全相同
   - 买者不在意从哪个卖者购买

3. **自由进入和退出（Free Entry and Exit）**：
   - 卖者可以自由进入或退出市场
   - 没有进入壁垒

4. **完全信息（Perfect Information）**：
   - 买者和卖者都了解商品价格和质量
   - 没有信息不对称

**例子**：
- 农产品市场（小麦、玉米、大豆）
- 金融市场（外汇、股票）
- 某些商品市场（黄金、白银）

### 14.1.2 竞争企业的收益

**竞争企业的收益**：

1. **平均收益（Average Revenue, AR）**：
   - 每单位产品的收益
   - AR = TR / Q = P

2. **边际收益（Marginal Revenue, MR）**：
   - 多销售一单位产品所增加的收益
   - MR = ΔTR / ΔQ = P

**关键点**：
- 对于竞争企业：P = AR = MR
- 价格、平均收益、边际收益相等

- 竞争企业面临水平的需求曲线
- 价格由市场决定
- 企业可以按市场价格销售任何数量

## 14.2 利润最大化

### 14.2.1 利润最大化的条件

**利润（Profit, π）**：
总收益减去总成本。

**定义公式**：
```
π = TR - TC
```

**利润最大化的条件**：
```
MR = MC
```

其中：
- MR：边际收益
- MC：边际成本

**直观理解**：
- MR > MC → 增加产量 → 利润增加
- MR < MC → 减少产量 → 利润增加
- MR = MC → 利润最大化

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**关键点**：
- 在MR = MC的点，利润最大化
- 该点对应的产量为Q*

### 14.2.2 利润的计算

**利润的计算**：

**方法1：总收益减去总成本**：
```
π = TR - TC = P × Q - ATC × Q = (P - ATC) × Q
```

**方法2：图形法**：
```
利润 = (P - ATC) × Q
```

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**例子**：

假设：
- 价格 P = 10美元
- 产量 Q* = 100
- 平均总成本 ATC = 8美元

**利润**：
```
π = (P - ATC) × Q*
  = (10 - 8) × 100
  = 2 × 100
  = 200美元
```

### 14.2.3 亏损情况

**当 P < ATC 时，企业亏损**：

**亏损的计算**：
```
亏损 = (ATC - P) × Q
```

## 14.3 竞争企业的短期供给

### 14.3.1 短期供给决策

**短期供给决策**：
企业根据价格决定是否生产以及生产多少。

**三种情况**：

**情况1：P > ATC**
- 企业盈利
- 生产，利润最大化
- 产量：MR = MC

**情况2：AVC < P < ATC**
- 企业亏损，但亏损 < 固定成本
- 生产，亏损最小化
- 产量：MR = MC

**情况3：P < AVC**
- 企业亏损，亏损 > 固定成本
- 停止生产
- 产量：0


### 14.3.2 停止营业点

**停止营业点（Shutdown Point）**：
价格等于平均可变成本的最低点。

**定义**：
```
P = AVC（最低点）
```

**含义**：
- 当价格低于AVC最低点时，企业停止生产
- 因为亏损 > 固定成本

## 14.4 竞争市场的短期供给

### 14.4.1 市场供给曲线

**市场供给曲线（Market Supply Curve）**：
所有企业供给曲线的水平加总。

**数学表达**：
```
Qs = Σqsi
```

其中：
- Qs：市场供给量
- qsi：企业i的供给量

**图形分析**：


### 14.4.2 短期均衡

**短期均衡（Short-Run Equilibrium）**：
市场供给等于市场需求。

**均衡条件**：
```
Qs = Qd
P = 市场价格
```

## 14.5 竞争企业的长期供给

### 14.5.1 长期均衡

**长期均衡（Long-Run Equilibrium）**：
利润为零（正常利润）。

**条件**：
```
P = MR = MC = ATC（最低点）
```

**含义**：
- 利润为零
- 企业只能获得正常利润
- 没有企业进入或退出

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

### 14.5.2 进入与退出

**进入（Entry）**：
- 如果 P > ATC，企业获得正经济利润
- 新企业进入市场
- 市场供给增加
- 价格下降
- 直到 P = ATC

**退出（Exit）**：
- 如果 P < ATC，企业亏损
- 企业退出市场
- 市场供给减少
- 价格上升
- 直到 P = ATC

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

### 14.5.3 长期供给曲线

**长期供给曲线（Long-Run Supply Curve）**：
表示长期中价格与供给量关系的曲线。

**类型**：

1. **成本不变行业（Constant-Cost Industry）**：
   - 长期供给曲线水平
   - 价格不变

2. **成本递增行业（Increasing-Cost Industry）**：
   - 长期供给曲线向上倾斜
   - 价格随产量增加而上升

3. **成本递减行业（Decreasing-Cost Industry）**：
   - 长期供给曲线向下倾斜
   - 价格随产量增加而下降

## 14.6 竞争市场的效率

### 14.6.1 配置效率

**配置效率（Allocative Efficiency）**：
资源配置使总剩余最大化。

**条件**：
```
P = MC
```

**含义**：
- 价格 = 边际成本
- 消费者的支付意愿 = 生产者的边际成本
- 资源配置最优

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

### 14.6.2 生产效率

**生产效率（Productive Efficiency）**：
以最低的平均成本生产。

**条件**：
```
P = ATC（最低点）
```

**含义**：
- 价格 = 平均总成本
- 生产在最低平均成本点
- 资源利用最优

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "产量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

### 14.6.3 完全竞争市场的福利

**消费者剩余**：
```
消费者剩余 = 需求曲线下方、价格上方的面积
```

**生产者剩余**：
```
生产者剩余 = 供给曲线上方、价格下方的面积
```

**总剩余**：
```
总剩余 = 消费者剩余 + 生产者剩余
```

**无谓损失**：
```
无谓损失 = 0
```

**结论**：
> **完全竞争市场是有效率的，实现配置效率和生产效率**

## 14.7 本章总结

### 核心要点

1. **什么是竞争市场**
   - 完全竞争市场的特征
   - 价格接受者
   - 水平的需求曲线

2. **利润最大化**
   - MR = MC
   - 利润 = (P - ATC) × Q
   - 亏损 = (ATC - P) × Q

3. **短期供给**
   - 停止营业点：P = AVC
   - 短期供给曲线：MC在AVC最低点以上
   - 三种情况：盈利、亏损但生产、停止生产

4. **长期供给**
   - 长期均衡：P = MR = MC = ATC
   - 进入与退出
   - 长期供给曲线

5. **竞争市场的效率**
   - 配置效率：P = MC
   - 生产效率：P = ATC
   - 无谓损失为零
