# 第8章：应用：税收的代价

> 本章学习目标：
> - 深入理解税收对市场的影响
> - 掌握税收的无谓损失
> - 理解无谓损失的决定因素
> - 学会分析税收收入与税率的关系
> - 理解拉弗曲线和税收政策

---

## 8.1 税收与市场

### 8.1.1 税收的定义

**税收（Tax）**：
政府强制征收的收入，用于提供公共物品和服务。

**税收的类型**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **从量税（Per-Unit Tax）** | 每单位商品征收固定税额 | 汽油税、烟草税 |
| **从价税（Ad Valorem Tax）** | 按商品价格征收一定比例 | 销售税、增值税 |
| **所得税（Income Tax）** | 按收入征收 | 个人所得税、企业所得税 |
| **财产税（Property Tax）** | 按财产价值征收 | 房产税、遗产税 |
| **消费税（Excise Tax）** | 对特定商品征收 | 酒精税、烟草税 |

**本章重点**：从量税

### 8.1.2 税收的影响

**税收的影响**：
1. **价格上升**：
   - 买者支付的价格上升
   - 卖者收到的价格下降

2. **数量减少**：
   - 均衡数量减少
   - 市场交易减少

3. **税收收入（Tax Revenue）**：
   - 政府获得税收收入
   - 单位：每单位税收 × 交易数量

4. **无谓损失（Deadweight Loss）**：
   - 总剩余减少
   - 原因：部分交易无法进行

### 8.2.3 福利变化

**消费者剩余变化**：
```
ΔCS = 新CS - 原CS
    = 三角形A - (三角形A + 三角形B + 三角形C)
    = -三角形B - 三角形C
```

**生产者剩余变化**：
```
ΔPS = 新PS - 原PS
    = 三角形F - (三角形D + 三角形E + 三角形F)
    = -三角形D - 三角形E
```

**政府税收收入**：
```
税收收入 = 矩形B + 矩形D
```

**总剩余变化**：
```
ΔTS = 新TS - 原TS
    = (三角形A + 三角形F + 矩形B + 矩形D) - (三角形A + 三角形B + 三角形C + 三角形D + 三角形E + 三角形F)
    = -三角形C - 三角形E
```

**无谓损失**：
```
无谓损失 = -ΔTS = 三角形C + 三角形E
```

### 8.2.4 税收负担的分担

**税负归宿（Tax Incidence）**：
税收最终由谁承担。

**买者税负**：
```
买者税负 = Pb - Pe
         = 矩形B
```

**卖者税负**：
```
卖者税负 = Pe - Ps
         = 矩形D
```

**税负分担比例**：
```
买者税负比例 = (Pb - Pe) / (Pb - Ps)
卖者税负比例 = (Pe - Ps) / (Pb - Ps)
```

**关键原则**：
> **税负归宿取决于供给和需求的相对弹性，而不是法律规定的谁支付**

**情况1：需求富有弹性，供给缺乏弹性**
- 买者税负：小
- 卖者税负：大
- 卖者承担大部分税负

**情况2：需求缺乏弹性，供给富有弹性**
- 买者税负：大
- 卖者税负：小
- 买者承担大部分税负

## 8.3 无谓损失

### 8.3.1 无谓损失的定义

**无谓损失（Deadweight Loss, DWL）**：
税收导致的效率损失，即总剩余的减少。

**产生原因**：
1. **交易数量减少**：
   - 税收导致价格上升
   - 需求量减少，供给量减少
   - 均衡数量减少

2. **部分交易无法进行**：
   - 部分买者和卖者的交易被阻止
   - 这些交易的总收益大于总成本
   - 因此导致效率损失

**直观理解**：
- 买者的支付意愿高于卖者的成本
- 但由于税收，这些交易无法进行
- 这些本可以进行的交易代表了效率损失

**无谓损失的数学表达**：

给定需求函数：Qd = a - bP

给定供给函数：Qs = c + dP

**税前均衡**：
```
Qd = Qs
a - bP = c + dP
Pe = (a - c)/(b + d)
Qe = (ad + bc)/(b + d)
```

**税后均衡（税率为t）**：
```
a - bPb = c + dPs
Pb - Ps = t

求解得：
Pb = (a - c + dt)/(b + d) + (bd/b + d)t
Ps = (a - c - bt)/(b + d) + (bd/b + d)t
Q2 = (ad + bc)/(b + d) - (bd/b + d)t
```

**无谓损失**：
```
DWL = ½ × (Qe - Q2) × t
    = ½ × (Qe - Q2) × (Pb - Ps)
```

**特殊情况：线性需求和供给**

给定弹性：|Ed| 和 Es

**无谓损失**：
```
DWL = ½ × ΔQ × ΔP
    = ½ × (Qe - Q2) × (Pb - Ps)
```

### 8.3.3 无谓损失的决定因素

**决定因素**：
1. **需求弹性**：
   - 需求弹性大 → 无谓损失大
   - 需求弹性小 → 无谓损失小

2. **供给弹性**：
   - 供给弹性大 → 无谓损失大
   - 供给弹性小 → 无谓损失小

3. **税率**：
   - 税率高 → 无谓损失大
   - 税率低 → 无谓损失小

**原因**：
- 弹性大 → 税收导致数量变化大
- 数量变化大 → 交易减少多
- 交易减少多 → 无谓损失大

### 8.3.4 无谓损失与税收

**无谓损失与税率的关系**：

**线性需求和无谓损失**：

**税率翻倍，无谓损失翻四倍**

**数学证明**：

给定线性需求：Q = a - bP

**无谓损失**：
```
DWL = ½ × ΔQ × ΔP
    = ½ × (Qe - Q2) × t
```

**当税率翻倍时**：
```
ΔQ' = 2 × ΔQ
ΔP' = 2 × ΔP

DWL' = ½ × ΔQ' × ΔP'
     = ½ × (2 × ΔQ) × (2 × ΔP)
     = 4 × (½ × ΔQ × ΔP)
     = 4 × DWL
```

**结论**：
> **税率翻倍，无谓损失翻四倍**

## 8.4 税收收入

### 8.4.1 税收收入的定义

**税收收入（Tax Revenue）**：
政府从税收中获得的收入。

**定义公式**：
```
税收收入 = 税率 × 交易数量
        = t × Q2
```

其中：
- t：税率（每单位税收）
- Q2：税后交易数量

### 8.4.2 税收收入与税率的关系

**税收收入与税率的关系**：

**低税率**：
- 税收收入随税率上升而增加
- 原因：税率上升 × 数量小幅减少

**中等税率**：
- 税收收入达到最大值
- 原因：税率上升 × 数量减少的平衡

**高税率**：
- 税收收入随税率上升而减少
- 原因：税率上升 × 数量大幅减少

## 8.5 最优税收

### 8.5.1 最优税收的定义

**最优税收（Optimal Tax）**：
使社会福利最大化的税收政策。

**社会福利函数**：
```
社会福利 = 总剩余 - 无谓损失
        = 消费者剩余 + 生产者剩余 + 税收收入 - 无谓损失
```

**最优税收的条件**：
1. **税收收入最大化**：
   - 税率 = 最优税率
   - 税收收入最大

2. **无谓损失最小化**：
   - 对弹性小的商品征收高税率
   - 对弹性大的商品征收低税率

3. **公平性**：
   - 税收负担公平分配
   - 保护低收入群体

### 8.5.2 拉姆齐法则（Ramsey Rule）

**拉姆齐法则（Ramsey Rule）**：
最优税收应该使所有商品的边际无谓损失相等。

**数学表达**：
```
τ1 / τ2 = |Ed2| / |Ed1|
```

其中：
- τ1、τ2：商品1和商品2的税率
- |Ed1|、|Ed2|：商品1和商品2的需求弹性

**含义**：
- 弹性小的商品应该征收高税率
- 弹性大的商品应该征收低税率

**原因**：
- 弹性小 → 无谓损失小 → 可以征收高税率
- 弹性大 → 无谓损失大 → 应该征收低税率

### 8.5.3 最优税收政策

**最优税收政策的原则**：

1. **效率原则**：
   - 对弹性小的商品征收高税率
   - 对弹性大的商品征收低税率
   - 减少无谓损失

2. **公平原则**：
   - 税收负担公平分配
   - 保护低收入群体
   - 考虑收入分配

3. **简化原则**：
   - 税收制度简单透明
   - 减少税收遵从成本
   - 提高税收效率

**实际应用**：

**例子1：消费税设计**
- 对必需品（弹性小）征收低税率
- 对奢侈品（弹性大）征收高税率

**例子2：所得税设计**
- 累进税率：收入高，税率高

## 8.6 税收的实际案例

### 8.6.1 香烟税

**背景**：
- 政府对香烟征收高额税
- 目的：减少吸烟、筹集收入

**效果**：
1. **需求弹性**：
   - 弹性中等（-0.4）
   - 税收导致吸烟减少

2. **税负归宿**：
   - 买者承担大部分税负
   - 卖者承担小部分

3. **无谓损失**：
   - 无谓损失中等
   - 弹性适中

4. **税收收入**：
   - 税收收入高
   - 需求缺乏弹性

5. **健康影响**：
   - 吸烟减少
   - 健康改善

### 8.6.2 汽油税

**背景**：
- 政府对汽油征收从量税
- 目的：减少污染、筹集收入

**效果**：
1. **需求弹性**：
   - 短期弹性小（-0.2）
   - 长期弹性大（-0.7）

2. **税负归宿**：
   - 短期：买者承担大部分
   - 长期：卖者承担更多

3. **无谓损失**：
   - 短期无谓损失小
   - 长期无谓损失大

4. **税收收入**：
   - 短期税收收入高
   - 长期税收收入减少

5. **环境影响**：
   - 汽油消费减少
   - 污染减少

### 8.6.3 奢侈品税

**背景**：
- 政府对奢侈品征收高税
- 目的：收入再分配

**效果**：
1. **需求弹性**：
   - 弹性大（-2.5）

2. **税负归宿**：
   - 卖者承担大部分税负
   - 买者承担小部分

3. **无谓损失**：
   - 无谓损失大
   - 弹性大

4. **税收收入**：
   - 税收收入少
   - 弹性大

5. **市场影响**：
   - 奢侈品销量大幅下降
   - 产业受损

### 8.6.4 所得税

**背景**：
- 政府对收入征收所得税
- 目的：筹集收入、收入再分配

**效果**：
1. **劳动供给弹性**：
   - 弹性中等
   - 税收导致劳动供给减少

2. **税负归宿**：
   - 工人承担税负
   - 雇主转移部分税负

3. **无谓损失**：
   - 无谓损失中等
   - 劳动供给弹性适中

4. **税收收入**：
   - 税收收入高
   - 收入基数大

5. **公平性**：
   - 累进税率
   - 保护低收入群体

## 8.7 本章总结

### 核心要点

1. **税收与市场**
   - 税收导致价格上升、数量减少
   - 买者和卖者分担税负
   - 税负归宿取决于弹性

2. **税收的福利分析**
   - 消费者剩余减少
   - 生产者剩余减少
   - 政府获得税收收入
   - 总剩余减少（无谓损失）

3. **无谓损失**
   - 税收导致的效率损失
   - 取决于需求弹性和供给弹性
   - 弹性越大，无谓损失越大
   - 税率翻倍，无谓损失翻四倍

4. **税收收入**
   - 税收收入 = 税率 × 交易数量
   - 拉弗曲线：税收收入随税率上升先增加后减少
   - 弹性小，税收收入多
   - 弹性大，税收收入少

5. **最优税收**
   - 使社会福利最大化
   - 拉姆齐法则：对弹性小的商品征收高税率
   - 效率、公平、简化原则

### 重要概念

| 概念 | 定义 | 关键要点 |
|------|------|----------|
| 税收 | 政府强制征收的收入 | 从量税、从价税、所得税 |
| 税负归宿 | 税收最终由谁承担 | 取决于弹性 |
| 无谓损失 | 税收导致的效率损失 | 弹性越大，无谓损失越大 |
| 税收收入 | 政府从税收中获得的收入 | 税率 × 交易数量 |
| 拉弗曲线 | 税收收入与税率的关系 | 倒U形曲线 |
| 最优税率 | 使税收收入最大化的税率 | 拉弗曲线的顶点 |
| 拉姆齐法则 | 最优税收的规则 | 对弹性小的商品征收高税率 |
| 税收楔子 | 买者支付价格与卖者收到价格的差额 | Pb - Ps |
