# 第15章：垄断

> 本章学习目标：
> - 理解为什么会产生垄断
> - 掌握垄断企业的利润最大化决策
> - 学会分析垄断的福利损失
> - 理解价格歧视
> - 了解政府对垄断的管制

---

## 15.1 为什么会产生垄断

### 15.1.1 垄断的定义

**垄断（Monopoly）**：
市场上只有一个卖者，且没有相近替代品。

**垄断的特征**：

1. **单一卖者（Single Seller）**：
   - 只有一个生产者
   - 企业就是行业

2. **价格制定者（Price Maker）**：
   - 有市场权力
   - 可以设定价格

3. **无相近替代品（No Close Substitutes）**：
   - 消费者没有其他选择
   - 需求缺乏弹性

4. **进入壁垒（Barriers to Entry）**：
   - 其他企业无法进入
   - 维持垄断地位

### 15.1.2 垄断产生的原因

**垄断产生的原因**：

1. **垄断资源（Monopoly Resources）**：
   - 企业拥有关键资源
   - 其他企业无法获得
   - 例子：钻石（德比尔斯公司）

2. **政府创造的垄断（Government-Created Monopoly）**：
   - 政府授予独家经营权
   - 专利和版权
   - 例子：药品专利

3. **自然垄断（Natural Monopoly）**：
   - 一个企业能以低于多个企业的成本提供整个市场的产品
   - 规模经济
   - 例子：自来水、电力

**进入壁垒（Barriers to Entry）**：

| 类型 | 定义 | 例子 |
|------|------|------|
| **法律壁垒（Legal Barriers）** | 专利、版权、特许经营 | 药品专利 |
| **经济壁垒（Economic Barriers）** | 规模经济、网络效应 | 电信、社交网络 |
| **技术壁垒（Technological Barriers）** | 技术领先、专利 | 半导体、软件 |
| **策略壁垒（Strategic Barriers）** | 掠夺性定价、品牌效应 | 零售、餐饮 |

### 15.1.3 自然垄断

**自然垄断（Natural Monopoly）**：
一个企业能以低于多个企业的成本提供整个市场的产品。

**原因**：
- 高固定成本
- 低边际成本
- 规模经济

**图形分析**：

**关键点**：
- ATC随产量增加而下降
- 一个企业生产整个市场的产量成本最低
- 多个企业生产成本更高

**例子**：
- 自来水：管道建设成本高，边际成本低
- 电力：电网建设成本高，边际成本低
- 铁路：轨道建设成本高，边际成本低

## 15.2 垄断企业的决策

### 15.2.1 垄断企业的收益

**垄断企业的收益**：

1. **总收益（Total Revenue, TR）**：
   - TR = P × Q

2. **平均收益（Average Revenue, AR）**：
   - AR = TR / Q = P
   - 平均收益等于价格

3. **边际收益（Marginal Revenue, MR）**：
   - MR = ΔTR / ΔQ
   - MR < P（向下倾斜的需求曲线）

**边际收益与价格的关系**：
```
MR = P × (1 - 1/|Ed|)
```

其中：
- P：价格
- |Ed|：需求价格弹性

**含义**：
- |Ed| > 1：MR > 0
- |Ed| = 1：MR = 0
- |Ed| < 1：MR < 0

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**关键点**：
- 需求曲线（D）向下倾斜
- 边际收益曲线（MR）也在需求曲线下方
- MR比需求曲线更陡峭

### 15.2.2 利润最大化

**利润最大化条件**：
```
MR = MC
```

**垄断企业的利润最大化**：

**步骤1**：
- 找到MR = MC的点
- 确定最优产量Q*

**步骤2**：
- 在需求曲线上找到对应Q*的价格P*
- 确定最优价格P*

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**例子**：

假设需求函数：P = 100 - 2Q

**总收益**：
```
TR = P × Q = (100 - 2Q) × Q = 100Q - 2Q²
```

**边际收益**：
```
MR = dTR/dQ = 100 - 4Q
```

**假设边际成本**：MC = 20

**利润最大化**：
```
MR = MC
100 - 4Q = 20
4Q = 80
Q* = 20
```

**最优价格**：
```
P* = 100 - 2Q* = 100 - 2 × 20 = 60
```

### 15.2.3 垄断利润

**垄断利润**：
```
π = (P - ATC) × Q
```

**图形分析**：

```mermaid

    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**关键点**：
- 垄断企业可以获得正经济利润
- 长期中，由于进入壁垒，垄断企业可以维持正利润

## 15.3 垄断的福利代价

### 15.3.1 无谓损失

**垄断的福利损失**：

**竞争市场**：
- 价格 = 边际成本
- 产量 = 竞争产量
- 总剩余最大

**垄断市场**：
- 价格 > 边际成本
- 产量 < 竞争产量
- 总剩余减少

**图形分析**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**无谓损失（Deadweight Loss）**：
```
无谓损失 = 三角形
```

**原因**：
- 垄断价格高于竞争价格
- 垄断产量低于竞争产量
- 部分交易无法进行
- 这些交易的总收益大于总成本
- 导致效率损失

### 15.3.2 垄断的低效率

**垄断的低效率**：

1. **配置低效（Allocative Inefficiency）**：
   - P > MC
   - 资源配置不当
   - 消费者剩余减少

2. **生产低效（Productive Inefficiency）**：
   - 不一定在ATC最低点生产
   - 可能存在X-非效率

3. **动态低效（Dynamic Inefficiency）**：
   - 缺乏竞争激励
   - 创新动力不足

4. **寻租行为（Rent-Seeking）**：
   - 获取或维持垄断地位
   - 浪费资源

### 15.3.3 垄断的好处

**垄断的好处**：

1. **规模经济（Economies of Scale）**：
   - 自然垄断可以降低成本
   - 例如：自来水、电力

2. **创新激励（Innovation Incentives）**：
   - 专利保护鼓励创新
   - 例如：药品研发

3. **稳定价格（Price Stability）**：
   - 避免恶性竞争
   - 例如：公共事业

## 15.4 价格歧视

### 15.4.1 价格歧视的定义

**价格歧视（Price Discrimination）**：
对同一种商品向不同消费者收取不同的价格。

**价格歧视的条件**：

1. **市场权力（Market Power）**：
   - 企业有定价能力
   - 不是价格接受者

2. **区分消费者（Identify Consumers）**：
   - 可以区分不同消费者群体
   - 了解不同的支付意愿

3. **防止套利（Prevent Arbitrage）**：
   - 无法低价买入高价卖出
   - 商品无法转售

### 15.4.2 价格歧视的类型

**价格歧视的类型**：

1. **一级价格歧视（Perfect Price Discrimination）**：
   - 对每个消费者收取其支付意愿
   - 完全榨取消费者剩余
   - 无谓损失为零

2. **二级价格歧视（Second-Degree Price Discrimination）**：
   - 根据购买数量定价
   - 例如：批量折扣

3. **三级价格歧视（Third-Degree Price Discrimination）**：
   - 根据消费者群体定价
   - 例如：学生票、老年票

**图形分析**：

**一级价格歧视**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "数量Q" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**关键点**：
- 消费者剩余 = 0
- 生产者剩余 = 总剩余
- 无谓损失 = 0
- 产量 = 竞争产量

### 15.4.3 价格歧视的例子

**价格歧视的例子**：

1. **航空业**：
   - 商务旅客：高价格（弹性小）
   - 休闲旅客：低价格（弹性大）

2. **电影票**：
   - 成人票：高价格
   - 学生票：低价格
   - 老年票：低价格

3. **电费**：
   - 峰时电价：高价格
   - 谷时电价：低价格

4. **软件**：
   - 企业版：高价格
   - 个人版：低价格
   - 学生版：更低价

## 15.5 公共政策

### 15.5.1 对垄断的管制

**对垄断的管制方式**：

1. **价格管制（Price Regulation）**：
   - 设定最高价格
   - 目标：P = MC
   - 问题：自然垄断的亏损

2. **回报率管制（Rate-of-Return Regulation）**：
   - 允许垄断企业获得正常利润
   - P = ATC
   - 问题：缺乏效率激励

3. **国有化（Nationalization）**：
   - 政府接管垄断企业
   - 目标：公共利益
   - 问题：效率低下

**价格管制的图形分析**：

### 15.5.2 反垄断法

**反垄断法（Antitrust Law）**：
防止垄断和促进竞争的法律。

**美国反垄断法**：

1. **谢尔曼法（Sherman Act, 1890）**：
   - 禁止垄断和限制贸易
   - 第1条：禁止限制贸易的合同
   - 第2条：禁止垄断和企图垄断

2. **克莱顿法（Clayton Act, 1914）**：
   - 补充谢尔曼法
   - 禁止价格歧视、排他性交易、兼并

3. **联邦贸易委员会法（FTC Act, 1914）**：
   - 成立联邦贸易委员会
   - 防止不公平竞争

**反垄断政策的目标**：
- 防止垄断
- 促进竞争
- 保护消费者利益

### 15.5.3 公有制

**公有制（Public Ownership）**：
政府拥有和经营企业。

**公有制的例子**：
- 公共事业：自来水、电力
- 邮政服务：邮政
- 交通运输：铁路、地铁

**公有制的优点**：
- 服务公共利益
- 价格稳定
- 普遍服务

**公有制的缺点**：
- 效率低下
- 缺乏创新
- 政治干预

## 15.6 垄断与竞争的比较

### 15.6.1 垄断与竞争的比较

**垄断与竞争的比较**：

| 特征 | 完全竞争 | 垄断 |
|------|----------|------|
| **企业数量** | 众多 | 一个 |
| **产品** | 同质 | 无替代品 |
| **价格** | 价格接受者 | 价格制定者 |
| **进入** | 自由 | 进入壁垒 |
| **需求曲线** | 水平 | 向下倾斜 |
| **边际收益** | MR = P | MR < P |
| **利润最大化** | P = MC | MR = MC |
| **价格** | P = MC | P > MC |
| **产量** | Qc | Qm < Qc |
| **效率** | 配置效率、生产效率 | 配置低效、生产低效 |
| **无谓损失** | 0 | > 0 |
| **长期利润** | 0 | > 0 |

### 15.6.2 垄断的争议

**垄断的争议**：

**支持垄断的论点**：
1. **规模经济**：
   - 自然垄断可以降低成本
   - 提高效率

2. **创新激励**：
   - 专利保护鼓励创新
   - 促进技术进步

3. **稳定价格**：
   - 避免恶性竞争
   - 价格稳定

**反对垄断的论点**：
1. **低效率**：
   - 配置低效、生产低效
   - 无谓损失

2. **剥削消费者**：
   - 高价格、低产量
   - 消费者剩余减少

3. **抑制创新**：
   - 缺乏竞争激励
   - 创新动力不足
