## 3.1 IS-LM模型

### 3.1.1 IS-LM模型的定义

**IS-LM模型（IS-LM Model）**：
分析产品市场和货币市场一般均衡的模型。

### 3.1.2 IS-LM模型的假设

**IS-LM模型的假设**：

1. **价格水平固定（Fixed Price Level）**：
   - 名义变量等于实际变量
   - 不考虑价格变化

2. **短期分析（Short-Run Analysis）**：
   - 供给能够满足需求
   - 不考虑供给约束

3. **封闭经济（Closed Economy）**：
   - 不考虑对外贸易
   - 不考虑净出口

## 3.2 产品市场的均衡：IS曲线

### 3.2.1 产品市场的均衡条件

**产品市场的均衡条件（Equilibrium Condition in Product Market）**：
```
Y = C + I + G
```

**其中**：
- Y：国民收入
- C：消费
- I：投资
- G：政府支出

### 3.2.2 消费函数

**消费函数（Consumption Function）**：
```
C = a + b(Y - T)
```

**其中**：
- C：消费
- a：自主消费
- b：边际消费倾向
- Y：收入
- T：税收

### 3.2.3 投资函数

**投资函数（Investment Function）**：
```
I = e - dr
```

**其中**：
- I：投资
- e：自主投资
- d：投资对利率的敏感度
- r：利率

### 3.2.4 税收函数

**税收函数（Tax Function）**：
```
T = T0 + tY
```

**其中**：
- T：税收
- T0：自主税收
- t：税率
- Y：收入

### 3.2.5 IS曲线的推导

**IS曲线的推导（Derivation of IS Curve）**：

**均衡条件**：
```
Y = C + I + G
Y = a + b(Y - T) + e - dr + G
Y = a + b(Y - T0 - tY) + e - dr + G
Y = a + bY - bT0 - btY + e - dr + G
Y - bY + btY = a - bT0 + e - dr + G
(1 - b + bt)Y = a - bT0 + e + G - dr
Y = (a - bT0 + e + G - dr) / (1 - b + bt)
```

**IS曲线的公式**：
```
Y = (A - dr) / (1 - b + bt)
```

**其中**：
- A = a - bT0 + e + G：自发支出
- d：投资对利率的敏感度
- r：利率
- b：边际消费倾向
- t：税率
- Y：收入

**含义**：
- IS曲线表示产品市场均衡时收入和利率的关系
- 利率上升，收入下降
- IS曲线向右下方倾斜

### 3.2.7 IS曲线的移动

**IS曲线的移动（Shift of IS Curve）**：

**自发支出增加**：
- 自发支出增加，IS曲线向右移动
- 自发支出减少，IS曲线向左移动

**自发支出（A）**：
```
A = a - bT0 + e + G
```

**影响因素**：
1. 自主消费（a）增加，IS曲线向右移动
2. 自主税收（T0）减少，IS曲线向右移动
3. 自主投资（e）增加，IS曲线向右移动
4. 政府支出（G）增加，IS曲线向右移动

## 3.3 货币市场的均衡：LM曲线

### 3.3.1 货币市场的均衡条件

**货币市场的均衡条件（Equilibrium Condition in Money Market）**：
```
M/P = L(r, Y)
```

**其中**：
- M：货币供给
- P：价格水平
- L：货币需求
- r：利率
- Y：收入

### 3.3.2 货币需求

**货币需求（Money Demand）**：
```
L = kY - hr
```

**其中**：
- L：货币需求
- k：货币需求对收入的敏感度
- Y：收入
- h：货币需求对利率的敏感度
- r：利率

**含义**：
- 收入增加，货币需求增加
- 利率上升，货币需求减少

### 3.3.3 货币供给

**货币供给（Money Supply）**：
```
M = M0
```

**其中**：
- M：货币供给
- M0：货币供给量（由中央银行决定）

### 3.3.4 LM曲线的推导

**LM曲线的推导（Derivation of LM Curve）**：

**均衡条件**：
```
M/P = L
M/P = kY - hr
kY = M/P + hr
Y = (M/P + hr) / k
```

**LM曲线的公式**：
```
Y = (M/P) / k + (h/k)r
```

**其中**：
- M：货币供给
- P：价格水平
- k：货币需求对收入的敏感度
- h：货币需求对利率的敏感度
- r：利率
- Y：收入

**含义**：
- LM曲线表示货币市场均衡时收入和利率的关系
- 利率上升，收入上升
- LM曲线向右上方倾斜

**图形表示**：

    x-axis "r" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- LM曲线：货币市场均衡时收入和利率的关系
- LM曲线向右上方倾斜
- 利率上升，货币需求减少，收入上升

### 3.3.6 LM曲线的移动

**LM曲线的移动（Shift of LM Curve）**：

**货币供给增加**：
- 货币供给增加，LM曲线向右移动
- 货币供给减少，LM曲线向左移动

**实际货币供给（M/P）**：
- M/P = M0 / P

**影响因素**：
1. 货币供给（M0）增加，LM曲线向右移动
2. 价格水平（P）下降，LM曲线向右移动

## 3.4 产品市场和货币市场的一般均衡

### 3.4.1 一般均衡的条件

**一般均衡的条件（General Equilibrium Condition）**：
```
Y = (A - dr) / (1 - b + bt)  (IS曲线)
Y = (M/P) / k + (h/k)r  (LM曲线)
```

### 3.4.2 一般均衡的解

**一般均衡的解（Solution of General Equilibrium）**：

**联立方程**：
```
(1 - b + bt)Y + dr = A
kY - hr = M/P
```

**解**：
```
Y = [hA + d(M/P)] / [h(1 - b + bt) + dk]
r = [kA - (1 - b + bt)(M/P)] / [h(1 - b + bt) + dk]
```

**含义**：
- 均衡收入：Y*
- 均衡利率：r*

**图形表示**：

    x-axis "r" [0, 20, 40, 60, 80, 100]

```

**分析**：

- LM曲线：货币市场均衡
- 一般均衡：IS曲线和LM曲线的交点
- 均衡收入：Y*

## 3.5 IS-LM模型的应用

### 3.5.1 财政政策

**财政政策（Fiscal Policy）**：
政府通过调整政府支出和税收来影响经济。

**扩张性财政政策（Expansionary Fiscal Policy）**：
- 增加政府支出（G）
- 减少税收（T）
- IS曲线向右移动
- 收入增加，利率上升

**紧缩性财政政策（Contractionary Fiscal Policy）**：
- 减少政府支出（G）
- 增加税收（T）
- IS曲线向左移动
- 收入减少，利率下降

### 3.5.2 货币政策

**货币政策（Monetary Policy）**：
中央银行通过调整货币供给来影响经济。

**扩张性货币政策（Expansionary Monetary Policy）**：
- 增加货币供给（M）
- LM曲线向右移动
- 收入增加，利率下降

**紧缩性货币政策（Contractionary Monetary Policy）**：
- 减少货币供给（M）
- LM曲线向左移动
- 收入减少，利率上升

### 3.5.3 挤出效应

**挤出效应（Crowding-Out Effect）**：
扩张性财政政策导致利率上升，减少私人投资。

**机制**：
1. 政府支出增加，IS曲线向右移动
2. 收入增加，货币需求增加
3. 利率上升，投资减少
4. 部分收入增加被投资减少抵消

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "r" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IS曲线向右移动
- 利率上升，投资减少
- 收入增加的幅度小于IS曲线移动的幅度

## 3.6 本章总结

### 核心要点

1. **IS-LM模型**：
   - 分析产品市场和货币市场一般均衡的模型
   - 假设：价格水平固定、短期分析、封闭经济

2. **产品市场的均衡（IS曲线）**：
   - 均衡条件：Y = C + I + G
   - IS曲线：Y = (A - dr) / (1 - b + bt)
   - IS曲线向右下方倾斜
   - 自发支出增加，IS曲线向右移动

3. **货币市场的均衡（LM曲线）**：
   - 均衡条件：M/P = L
   - LM曲线：Y = (M/P) / k + (h/k)r
   - LM曲线向右上方倾斜
   - 货币供给增加，LM曲线向右移动

4. **产品市场和货币市场的一般均衡**：
   - 一般均衡：IS曲线和LM曲线的交点
   - 均衡收入：Y*
   - 均衡利率：r*

5. **IS-LM模型的应用**：
   - 财政政策：调整政府支出和税收
   - 货币政策：调整货币供给
   - 挤出效应：扩张性财政政策导致利率上升，减少私人投资
