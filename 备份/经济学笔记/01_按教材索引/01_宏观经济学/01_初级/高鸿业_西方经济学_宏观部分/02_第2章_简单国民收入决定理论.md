# 第2章：简单国民收入决定理论

> 本章学习目标：
> - 理解简单国民收入决定模型
> - 掌握消费函数和储蓄函数
> - 学会分析均衡国民收入的决定

## 2.1 简单国民收入决定模型

### 2.1.1 简单国民收入决定模型的假设

**简单国民收入决定模型（Simple National Income Determination Model）的假设**：

1. **只有产品市场（Only Product Market）**：
   - 不考虑货币市场
   - 不考虑劳动市场

2. **两部门经济（Two-Sector Economy）**：
   - 只有家庭和企业
   - 不考虑政府和对外贸易

3. **固定价格（Fixed Price）**：
   - 价格水平固定
   - 名义变量等于实际变量

4. **供给创造需求（Supply Creates Demand）**：
   - 供给能够满足需求
   - 不考虑供给约束

### 2.1.2 均衡国民收入的定义

**均衡国民收入（Equilibrium National Income）**：
总需求等于总产出时的国民收入。

**条件**：
```
Y = C + I
```

**其中**：
- Y：国民收入
- C：消费
- I：投资

---

## 2.2 消费函数

### 2.2.1 消费函数的定义

**消费函数（Consumption Function）**：
消费与收入之间的关系。

**公式**：
```
C = C(Y)
```

**其中**：
- C：消费
- Y：收入

### 2.2.2 凯恩斯消费函数

**凯恩斯消费函数（Keynesian Consumption Function）**：
消费随收入增加而增加，但增加的幅度递减。

**公式**：
```
C = a + bY
```

**其中**：
- a：自主消费（Autonomous Consumption）
- b：边际消费倾向（Marginal Propensity to Consume, MPC）
- Y：收入

**特征**：
1. **a > 0**：自主消费为正
2. **0 < b < 1**：边际消费倾向在0和1之间
3. **dC/dY = b > 0**：消费随收入增加而增加
4. **d²C/dY² = 0**：消费随收入线性增加

### 2.2.3 边际消费倾向

**边际消费倾向（Marginal Propensity to Consume, MPC）**：
收入增加一单位时消费的增加量。

**公式**：
```
MPC = dC/dY = b
```

**含义**：
- MPC = 0.8：收入增加1元，消费增加0.8元
- 0 < MPC < 1

### 2.2.4 平均消费倾向

**平均消费倾向（Average Propensity to Consume, APC）**：
消费占收入的比例。

**公式**：
```
APC = C/Y
```

**含义**：
- APC > 0：消费为正
- APC可能大于1：消费大于收入

---

## 2.3 储蓄函数

### 2.3.1 储蓄函数的定义

**储蓄函数（Saving Function）**：
储蓄与收入之间的关系。

**公式**：
```
S = Y - C
```

**其中**：
- S：储蓄
- Y：收入
- C：消费

### 2.3.2 凯恩斯储蓄函数

**凯恩斯储蓄函数（Keynesian Saving Function）**：
储蓄随收入增加而增加。

**公式**：
```
S = -a + (1 - b)Y
```

**其中**：
- a：自主消费
- b：边际消费倾向
- (1 - b)：边际储蓄倾向（Marginal Propensity to Save, MPS）
- Y：收入

**推导**：
```
S = Y - C
S = Y - (a + bY)
S = -a + (1 - b)Y
```

### 2.3.3 边际储蓄倾向

**边际储蓄倾向（Marginal Propensity to Save, MPS）**：
收入增加一单位时储蓄的增加量。

**公式**：
```
MPS = dS/dY = 1 - b
```

**含义**：
- MPS = 0.2：收入增加1元，储蓄增加0.2元
- 0 < MPS < 1

### 2.3.4 平均储蓄倾向

**平均储蓄倾向（Average Propensity to Save, APS）**：
储蓄占收入的比例。

**公式**：
```
APS = S/Y
```

**含义**：
- APS可能为负：储蓄为负
- APS可能为正：储蓄为正

### 2.3.5 边际消费倾向与边际储蓄倾向的关系

**边际消费倾向与边际储蓄倾向的关系**：
```
MPC + MPS = 1
b + (1 - b) = 1
```

**含义**：
- 收入增加一单位，要么用于消费，要么用于储蓄
- MPC和MPS之和等于1

## 2.4 均衡国民收入的决定

### 2.4.1 均衡国民收入的条件

**均衡国民收入的条件（Equilibrium Condition）**：
```
Y = C + I
```

**其中**：
- Y：国民收入
- C：消费
- I：投资

### 2.4.2 均衡国民收入的决定

**均衡国民收入的决定（Determination of Equilibrium National Income）**：

**条件**：
```
Y = C + I
Y = a + bY + I
Y - bY = a + I
(1 - b)Y = a + I
Y = (a + I) / (1 - b)
```

**其中**：
- Y：均衡国民收入
- a：自主消费
- b：边际消费倾向
- I：投资

### 2.4.3 均衡国民收入的图形

**图形表示**：

## 2.5 乘数理论

### 2.5.1 乘数的定义

**乘数（Multiplier）**：
均衡国民收入的变化与自发支出变化的比例。

### 2.5.2 投资乘数

**投资乘数（Investment Multiplier）**：
投资增加一单位时均衡国民收入的增加量。

**公式**：
```
k = ΔY / ΔI = 1 / (1 - b) = 1 / MPS
```

**其中**：
- k：投资乘数
- ΔY：均衡国民收入的变化
- ΔI：投资的变化
- b：边际消费倾向
- MPS：边际储蓄倾向

**推导**：
```
Y = (a + I) / (1 - b)
ΔY = ΔI / (1 - b)
k = ΔY / ΔI = 1 / (1 - b) = 1 / MPS
```

**含义**：
- 如果b = 0.8，MPS = 0.2，k = 5
- 投资增加1元，均衡国民收入增加5元

### 2.5.3 乘数的作用机制

**乘数的作用机制（Multiplier Mechanism）**：

**例子**：投资增加100元

**第一轮**：
- 投资增加100元
- 收入增加100元
- 消费增加80元（b = 0.8）
- 储蓄增加20元（MPS = 0.2）

**第二轮**：
- 消费增加80元
- 收入增加80元
- 消费增加64元
- 储蓄增加16元

**第三轮**：
- 消费增加64元
- 收入增加64元
- 消费增加51.2元
- 储蓄增加12.8元

**...**

**总效应**：
- 收入增加：100 + 80 + 64 + 51.2 + ... = 500元
- 乘数：5

### 2.5.4 乘数的局限性

**乘数的局限性**：

1. **时间滞后（Time Lags）**：
   - 乘数效应需要时间
   - 不是立即生效

2. **资源约束（Resource Constraints）**：
   - 如果资源已经充分利用
   - 乘数效应可能减弱

3. **进口漏出（Import Leakage）**：
   - 如果进口增加
   - 乘数效应可能减弱

4. **税收（Taxes）**：
   - 如果税收增加
   - 乘数效应可能减弱

## 2.6 本章总结

### 核心要点

1. **简单国民收入决定模型**：
   - 假设：只有产品市场、两部门经济、固定价格、供给创造需求
   - 均衡国民收入：总需求等于总产出时的国民收入

2. **消费函数**：
   - 凯恩斯消费函数：C = a + bY
   - 边际消费倾向：MPC = dC/dY = b
   - 平均消费倾向：APC = C/Y

3. **储蓄函数**：
   - 凯恩斯储蓄函数：S = -a + (1 - b)Y
   - 边际储蓄倾向：MPS = dS/dY = 1 - b
   - 平均储蓄倾向：APS = S/Y
   - MPC + MPS = 1

4. **均衡国民收入的决定**：
   - 条件：Y = C + I
   - 均衡国民收入：Y = (a + I) / (1 - b)

5. **乘数理论**：
   - 投资乘数：k = 1 / (1 - b) = 1 / MPS
   - 乘数的作用机制：投资增加，收入增加，消费增加，收入增加，...
   - 乘数的局限性：时间滞后、资源约束、进口漏出、税收
