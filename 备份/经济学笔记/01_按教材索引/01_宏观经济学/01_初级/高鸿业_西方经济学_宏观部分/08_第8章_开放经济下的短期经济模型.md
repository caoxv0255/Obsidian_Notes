# 第8章：开放经济下的短期经济模型

> 本章学习目标：
> - 理解开放经济的特征

## 8.1 开放经济的特征

### 8.1.1 开放经济的定义

**开放经济（Open Economy）**：
与国外进行贸易和资本流动的经济。

### 8.1.2 开放经济的特征

**开放经济的特征（Characteristics of Open Economy）**：

1. **国际贸易（International Trade）**：
   - 商品和服务的进出口
   - 贸易余额 = 出口 - 进口

2. **国际资本流动（International Capital Flows）**：
   - 资本的流入和流出
   - 资本账户余额 = 资本流入 - 资本流出

3. **汇率（Exchange Rate）**：
   - 货币之间的交换比例
   - 影响进出口和资本流动

### 8.1.3 国际收支平衡表

**国际收支平衡表（Balance of Payments）**：
记录一国对外经济往来的会计报表。

**国际收支平衡表的构成**：

**1. 经常账户（Current Account）**：
- 商品和服务贸易
- 收益（投资收益）
- 经常转移（例如：汇款）

**2. 资本账户（Capital Account）**：
- 资本转移
- 非生产、非金融资产的取得和处置

**3. 金融账户（Financial Account）**：
- 直接投资
- 证券投资
- 其他投资
- 储备资产

**国际收支平衡**：
```
经常账户 + 资本账户 + 金融账户 = 0
```

## 8.2 汇率

### 8.2.1 汇率的定义

**汇率（Exchange Rate）**：
一种货币与另一种货币的交换比例。

### 8.2.2 汇率的标价方法

**汇率的标价方法（Quotation of Exchange Rate）**：

**1. 直接标价法（Direct Quotation）**：
- 以一定单位的外国货币为标准，折算为若干单位的本国货币
- 例如：1美元 = 7.0人民币

**2. 间接标价法（Indirect Quotation）**：
- 以一定单位的本国货币为标准，折算为若干单位的外国货币
- 例如：1人民币 = 0.143美元

### 8.2.3 名义汇率与实际汇率

**名义汇率（Nominal Exchange Rate）**：
两种货币的交换比例。

**实际汇率（Real Exchange Rate）**：
考虑物价水平的汇率。

**公式**：
```
实际汇率 = (名义汇率 × 国外物价水平) / 国内物价水平
ε = e × P* / P
```

**其中**：
- ε：实际汇率
- e：名义汇率
- P*：国外物价水平
- P：国内物价水平

**含义**：
- 实际汇率上升，国内商品相对便宜，出口增加
- 实际汇率下降，国内商品相对昂贵，出口减少

### 8.2.4 汇率的决定

**汇率的决定（Determination of Exchange Rate）**：

**1. 外汇供给和需求（Supply and Demand for Foreign Exchange）**：
- 外汇供给：出口、资本流入
- 外汇需求：进口、资本流出
- 外汇供给和需求决定汇率

**2. 购买力平价（Purchasing Power Parity, PPP）**：
- 一价定律：同一种商品在不同国家的价格应该相同
- 购买力平价：汇率应该使两国货币的购买力相等
- 公式：e = P / P*

**3. 利率平价（Interest Rate Parity）**：
- 利率平价：汇率应该反映两国利率的差异
- 公式：(1 + r) = (1 + r*) × (e' / e)
- 含义：高利率货币预期贬值

### 8.2.5 汇率制度

**汇率制度（Exchange Rate Regimes）**：

**1. 固定汇率制（Fixed Exchange Rate Regime）**：
- 汇率固定
- 中央银行干预外汇市场
- 例如：金本位制、布雷顿森林体系

**2. 浮动汇率制（Floating Exchange Rate Regime）**：
- 汇率由市场决定
- 中央银行不干预外汇市场
- 例如：现代国际货币体系

**3. 管理浮动汇率制（Managed Float Exchange Rate Regime）**：
- 汇率主要由市场决定
- 中央银行适当干预外汇市场
- 例如：大多数发展中国家

## 8.3 蒙代尔-弗莱明模型

### 8.3.1 蒙代尔-弗莱明模型的定义

**蒙代尔-弗莱明模型（Mundell-Fleming Model）**：
分析开放经济下短期经济均衡的模型。

### 8.3.2 蒙代尔-弗莱明模型的假设

**蒙代尔-弗莱明模型的假设**：

1. **价格水平固定（Fixed Price Level）**：
   - 名义变量等于实际变量
   - 不考虑价格变化

2. **短期分析（Short-Run Analysis）**：
   - 供给能够满足需求
   - 不考虑供给约束

3. **小型开放经济（Small Open Economy）**：
   - 该国是世界价格的接受者
   - 不能影响世界利率

4. **资本完全流动（Perfect Capital Mobility）**：
   - 资本可以自由流动
   - 国内利率等于世界利率

### 8.3.3 IS*曲线

**IS*曲线（IS* Curve）**：
表示开放经济下产品市场均衡时收入和汇率的曲线。

**公式**：
```
Y = C(Y - T) + I(r*) + G + NX(e)
```

**其中**：
- Y：收入
- C：消费
- T：税收
- I：投资
- r*：世界利率
- G：政府支出
- NX：净出口
- e：汇率

**含义**：
- 汇率上升，净出口增加，收入增加
- IS*曲线向右上方倾斜

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "e" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- IS*曲线：产品市场均衡时收入和汇率的曲线
- IS*曲线向右上方倾斜
- 汇率上升，净出口增加，收入增加

### 8.3.4 LM*曲线

**LM*曲线（LM* Curve）**：
表示开放经济下货币市场均衡时收入和汇率的曲线。

**公式**：
```
M/P = L(r*, Y)
```

**其中**：
- M：货币供给
- P：价格水平
- L：货币需求
- r*：世界利率
- Y：收入

**含义**：
- 货币市场均衡与汇率无关
- LM*曲线垂直

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "e" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- LM*曲线：货币市场均衡时收入和汇率的曲线
- LM*曲线垂直
- 货币市场均衡与汇率无关

### 8.3.5 均衡

**均衡（Equilibrium）**：
IS*曲线和LM*曲线的交点。

**图形表示**：

```mermaid
xychart-beta
    title "需求曲线"
    x-axis "e" [0, 20, 40, 60, 80, 100]
    y-axis "价格" 0 --> 25
    line [25, 20, 15, 10, 5, 0]
```

**分析**：
- 均衡：IS*曲线和LM*曲线的交点
- 均衡收入：Y*
- 均衡汇率：e*

## 8.4 开放经济下的宏观经济政策

### 8.4.1 浮动汇率下的财政政策

**浮动汇率下的财政政策（Fiscal Policy under Floating Exchange Rate）**：

**扩张性财政政策**：
- 政府支出增加，IS*曲线向右移动
- 收入增加，货币需求增加
- 利率上升，资本流入
- 汇率升值，净出口减少
- 收入减少
- 收入不变

**结论**：
- 浮动汇率下，财政政策无效

### 8.4.2 浮动汇率下的货币政策

**浮动汇率下的货币政策（Monetary Policy under Floating Exchange Rate）**：

**扩张性货币政策**：
- 货币供给增加，LM*曲线向右移动
- 利率下降，资本流出
- 汇率贬值，净出口增加
- 收入增加

**结论**：
- 浮动汇率下，货币政策有效

### 8.4.3 固定汇率下的财政政策

**固定汇率下的财政政策（Fiscal Policy under Fixed Exchange Rate）**：

**扩张性财政政策**：
- 政府支出增加，IS*曲线向右移动
- 收入增加，货币需求增加
- 利率上升，资本流入
- 汇率升值压力
- 中央银行干预，买入外汇，卖出本国货币
- 货币供给增加，LM*曲线向右移动
- 收入进一步增加

**结论**：
- 固定汇率下，财政政策非常有效

### 8.4.4 固定汇率下的货币政策

**固定汇率下的货币政策（Monetary Policy under Fixed Exchange Rate）**：

**扩张性货币政策**：
- 货币供给增加，LM*曲线向右移动
- 利率下降，资本流出
- 汇率贬值压力
- 中央银行干预，卖出外汇，买入本国货币
- 货币供给减少，LM*曲线回到原位
- 收入不变

**结论**：
- 固定汇率下，货币政策无效

## 8.5 本章总结

### 核心要点

1. **开放经济的特征**：
   - 国际贸易：商品和服务的进出口
   - 国际资本流动：资本的流入和流出
   - 汇率：货币之间的交换比例

2. **汇率**：
   - 定义：一种货币与另一种货币的交换比例
   - 标价方法：直接标价法、间接标价法
   - 名义汇率与实际汇率：实际汇率 = (名义汇率 × 国外物价水平) / 国内物价水平
   - 汇率的决定：外汇供给和需求、购买力平价、利率平价
   - 汇率制度：固定汇率制、浮动汇率制、管理浮动汇率制

3. **蒙代尔-弗莱明模型**：
   - 定义：分析开放经济下短期经济均衡的模型
   - 假设：价格水平固定、短期分析、小型开放经济、资本完全流动
   - IS*曲线：向右上方倾斜
   - LM*曲线：垂直
   - 均衡：IS*曲线和LM*曲线的交点

4. **开放经济下的宏观经济政策**：
   - 浮动汇率下的财政政策：无效
   - 浮动汇率下的货币政策：有效
   - 固定汇率下的财政政策：非常有效
   - 固定汇率下的货币政策：无效
