# 经济学笔记 (Economics Notes)

> 基于15本经典经济学教材的完整学习笔记系统  
> 覆盖宏观经济学和微观经济学，从初级到高级的完整知识体系

---

## 📚 项目简介

本笔记系统基于15本经典经济学教材编写，旨在提供一套完整、详细、系统的经济学学习资料。每本教材的笔记都包含：

- ✅ **完整概念体系**：清晰的概念定义和术语解释
- ✅ **详细理论推导**：严格的数学推导和证明过程
- ✅ **丰富图形分析**：使用Mermaid图表展示结构关系，平滑曲线图优先用 Plotly，并配合统一样式优化图表与表格的阅读体验
- 🔎 **图表显示示例**：[00_图表显示示例.md](00_图表显示示例.md)
- ✅ **实际应用案例**：结合现实经济现象和政策分析
- ✅ **完整习题解答**：教材习题的详细解答和分析
- ✅ **跨教材整合**：按主题分类，整合不同教材的观点
- ✅ **知识图谱**：展示概念之间的内在联系

---

## 🎯 学习目标

通过学习本笔记系统，您将能够：

1. **掌握经济学基本概念**：理解经济学的基本原理和核心概念
2. **运用经济理论分析**：能够运用经济学理论分析现实经济问题
3. **进行数学建模**：掌握经济学的数学建模和分析方法
4. **理解经济政策**：深入理解各种经济政策的理论基础和效果
5. **应对学术挑战**：为经济学专业的学习、考试和研究打下坚实基础

## 💡 学习建议

### 对于初学者

1. **从初级教材开始**：曼昆《经济学原理》是最好的入门教材
2. **注重概念理解**：先理解概念，再深入学习理论
3. **多做图形分析**：经济学很多理论都可以通过图形理解
4. **联系实际**：关注新闻和经济政策，理论联系实际
5. **循序渐进**：不要急于求成，按部就班地学习

### 对于进阶学习者

1. **重视数学推导**：中级和高级教材需要扎实的数学基础
2. **比较不同教材**：理解不同教材的观点和方法
3. **完成习题**：习题是检验理解程度的最好方式
4. **建立知识体系**：将分散的知识整合成完整的体系
5. **深入研究**：对感兴趣的主题进行深入阅读和研究

### 对于研究者

1. **掌握高级理论**：深入研究高级教材
2. **阅读经典文献**：阅读经济学经典研究论文
3. **学习数学工具**：掌握必要的数学工具和方法
4. **关注前沿研究**：了解经济学研究的最新进展
5. **独立研究**：尝试用经济学方法分析实际问题
