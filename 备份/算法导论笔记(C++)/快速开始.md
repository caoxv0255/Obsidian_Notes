### 编译模板

```bash
# 基础编译
g++ -std=c++20 filename.cpp -o filename

# 优化编译
g++ -std=c++20 -O2 -Wall -Wextra filename.cpp -o filename
```

## 常见问题

### Q1: 编译错误 "span is not a member of std"

**A**: 确保使用C++20标准

```bash
g++ -std=c++20 filename.cpp -o filename
```

### Q2: 链接错误

**A**: 确保包含所有必要的库

```cpp
#include <vector>
#include <algorithm>
#include <functional>
```

### Q3: 性能问题

**A**:
1. 使用 `-O2` 或 `-O3` 优化
2. 避免不必要的拷贝（使用引用或移动语义）
3. 使用合适的容器
4. 预分配内存（reserve）

### Q4: 内存泄漏

**A**:
1. 使用智能指针（std::unique_ptr, std::shared_ptr）
2. 使用Valgrind检测（Linux）

```bash
valgrind --leak-check=full ./filename
```
