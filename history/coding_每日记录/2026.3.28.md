## 今日主题：图论基础——拓扑排序（Topological Sorting）

### 学习内容总结

**核心知识点：**
- 有向无环图（DAG）：有向边 + 无环，描述偏序关系
- 拓扑排序：将DAG的顶点排成线性序列，满足所有边约束（u → v 则 u 在 v 前）
- 环检测：若图存在环，则无法完成拓扑排序
- Kahn算法：基于入度的BFS，不断移除入度为0的顶点
- DFS算法：三色标记法检测环（0=未访问，1=访问中，2=已访问）

**两种算法对比：**

| 维度 | Kahn算法 | DFS算法 |
|------|---------|---------|
| 数据结构 | 队列 | 递归栈 |
| 核心思路 | 入度为0优先 | 后序遍历逆序 |
| 环检测 | 结果长度≠顶点数 | 遇到访问中状态 |
| 适用场景 | 需显式拓扑序列 | 深度遍历信息 |
| 实现难度 | 直观易实现 | 代码简洁 |

## 今日LeetCode题目

### LeetCode 207 - 课程表

**题目描述：**
- 输入：课程总数 numCourses 和先修关系列表 prerequisites
- prerequisites[i] = [a_i, b_i] 表示学习课程 a_i 必须先完成课程 b_i
- 输出：判断是否可能完成所有课程（图中是否存在环）

**建模思路：**
- 课程 = 顶点，先修关系 = 有向边 b_i → a_i
- 问题转化为：判断有向图是否存在环
- 存在环 = 循环依赖，无法完成所有课程

## 方法1：Kahn算法（推荐）

```cpp
#include <vector>
#include <queue>
using namespace std;

class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        // 邻接表存储图
        vector<vector<int>> graph(numCourses);
        // 入度数组
        vector<int> indegree(numCourses, 0);
        
        // 建图并计算入度
        for (const auto& pre : prerequisites) {
            int course = pre[0];      // 目标课程
            int prereq = pre[1];      // 先修课程
            graph[prereq].push_back(course);
            indegree[course]++;
        }
        
        // 队列存储入度为0的顶点
        queue<int> q;
        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) {
                q.push(i);
            }
        }
        
        int processed = 0;
        while (!q.empty()) {
            int u = q.front();
            q.pop();
            processed++;
            
            for (int v : graph[u]) {
                indegree[v]--;
                if (indegree[v] == 0) {
                    q.push(v);
                }
            }
        }
        
        return processed == numCourses;
    }
};
```

**复杂度：**
- 时间复杂度：O(V + E)
- 空间复杂度：O(V + E)

## 方法2：DFS算法（三色标记）

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> graph(numCourses);
        for (const auto& pre : prerequisites) {
            graph[pre[1]].push_back(pre[0]);
        }
        
        // 0=未访问，1=访问中，2=已访问
        vector<int> visited(numCourses, 0);
        
        for (int i = 0; i < numCourses; i++) {
            if (visited[i] == 0) {
                if (!dfs(i, graph, visited)) {
                    return false;
                }
            }
        }
        return true;
    }
    
private:
    bool dfs(int u, vector<vector<int>>& graph, vector<int>& visited) {
        visited[u] = 1; // 标记为访问中
        
        for (int v : graph[u]) {
            if (visited[v] == 1) {
                return false; // 发现环
            } else if (visited[v] == 0) {
                if (!dfs(v, graph, visited)) {
                    return false;
                }
            }
        }
        
        visited[u] = 2; // 标记为已访问
        return true;
    }
};
```

**复杂度：**
- 时间复杂度：O(V + E)
- 空间复杂度：O(V + E)

## 核心代码模板

**Kahn算法模板：**
```cpp
vector<int> topoSort(int n, vector<vector<int>>& edges) {
    vector<vector<int>> graph(n);
    vector<int> indegree(n, 0);
    
    for (auto& e : edges) {
        graph[e[0]].push_back(e[1]);
        indegree[e[1]]++;
    }
    
    queue<int> q;
    for (int i = 0; i < n; i++) {
        if (indegree[i] == 0) q.push(i);
    }
    
    vector<int> result;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        result.push_back(u);
        for (int v : graph[u]) {
            if (--indegree[v] == 0) q.push(v);
        }
    }
    
    return result.size() == n ? result : vector<int>();
}
```

**DFS环检测模板：**
```cpp
bool hasCycle = false;

void dfs(int u, vector<vector<int>>& graph, vector<int>& visited) {
    visited[u] = 1; // 访问中
    
    for (int v : graph[u]) {
        if (visited[v] == 1) { hasCycle = true; return; }
        if (visited[v] == 0) dfs(v, graph, visited);
    }
    
    visited[u] = 2; // 已访问
}
```
