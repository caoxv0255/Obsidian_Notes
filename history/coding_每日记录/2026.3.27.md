## 今日主题：图论基础——深度优先搜索（DFS）与广度优先搜索（BFS）遍历

### 学习内容总结

**核心知识点：**
- 图遍历的本质：系统性地访问图中所有顶点，每个顶点访问且仅访问一次
- 访问标记机制：使用visited数组避免重复访问与死循环
- 深度优先搜索（DFS）：纵向深入优先，一条路走到底
- 广度优先搜索（BFS）：横向扩散优先，层层向外扩展
- 连通分量：图中相互连接的顶点集合
- 原地修改法：直接修改原数组标记访问状态

**DFS vs BFS核心差异：**

| 维度 | DFS（深度优先） | BFS（广度优先） |
|------|----------------|----------------|
| 数据结构 | 栈（递归/显式） | 队列 |
| 遍历顺序 | 纵向深入优先 | 横向扩散优先 |
| 空间占用 | O(depth) | O(width) |
| 路径特性 | 不一定最短 | 首次访问即最短（无权图） |
| 实现方式 | 递归简单，显式栈稍复杂 | 队列简单直接 |

## 今日LeetCode题目

### LeetCode 200 - 岛屿数量

**题目描述：**
- 给定由'1'（陆地）和'0'（水）组成的二维网格
- 计算网格中岛屿的数量（连通的'1'区域）
- 岛屿只能由水平或垂直方向相邻的陆地连接

**核心思路：**
- 遍历整个网格，遇到'1'时启动DFS/BFS标记整个连通分量
- 每启动一次遍历，岛屿计数+1
- 使用原地修改法将访问过的'1'改为'0'

---

## 方法1：DFS递归（推荐）

```cpp
#include <vector>
using namespace std;

class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        if (grid.empty() || grid[0].empty()) return 0;
        
        int m = grid.size(), n = grid[0].size();
        int islands = 0;
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    islands++;
                    dfs(grid, i, j);
                }
            }
        }
        
        return islands;
    }
    
private:
    void dfs(vector<vector<char>>& grid, int i, int j) {
        int m = grid.size(), n = grid[0].size();
        
        // 边界检查与访问控制
        if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != '1') {
            return;
        }
        
        // 标记为已访问（淹没岛屿）
        grid[i][j] = '0';
        
        // 递归探索四个方向
        dfs(grid, i - 1, j); // 上
        dfs(grid, i + 1, j); // 下
        dfs(grid, i, j - 1); // 左
        dfs(grid, i, j + 1); // 右
    }
};
```

**复杂度：**
- 时间复杂度：O(m×n)，每个单元格最多访问一次
- 空间复杂度：O(m×n)，最坏情况下递归深度达到m×n

## 方法2：BFS队列

```cpp
#include <vector>
#include <queue>
using namespace std;

class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        if (grid.empty() || grid[0].empty()) return 0;
        
        int m = grid.size(), n = grid[0].size();
        int islands = 0;
        
        // 方向数组
        vector<pair<int, int>> directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    islands++;
                    
                    // BFS队列初始化
                    queue<pair<int, int>> q;
                    q.push({i, j});
                    grid[i][j] = '0'; // 入队即标记
                    
                    while (!q.empty()) {
                        auto [x, y] = q.front();
                        q.pop();
                        
                        // 探索四个方向
                        for (auto& dir : directions) {
                            int nx = x + dir.first;
                            int ny = y + dir.second;
                            
                            if (nx >= 0 && nx < m && ny >= 0 && ny < n && grid[nx][ny] == '1') {
                                q.push({nx, ny});
                                grid[nx][ny] = '0'; // 关键：入队时立即标记
                            }
                        }
                    }
                }
            }
        }
        
        return islands;
    }
};
```

**复杂度：**
- 时间复杂度：O(m×n)，每个单元格最多访问一次
- 空间复杂度：O(min(m, n))，队列大小最多为网格宽度或高度

## 方法3：DFS迭代（显式栈）

```cpp
#include <vector>
#include <stack>
using namespace std;

class Solution {
public:
    int numIslands(vector<vector<char>>& grid) {
        if (grid.empty() || grid[0].empty()) return 0;
        
        int m = grid.size(), n = grid[0].size();
        int islands = 0;
        
        vector<pair<int, int>> directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
        
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (grid[i][j] == '1') {
                    islands++;
                    
                    stack<pair<int, int>> stk;
                    stk.push({i, j});
                    
                    while (!stk.empty()) {
                        auto [x, y] = stk.top();
                        stk.pop();
                        
                        if (x < 0 || x >= m || y < 0 || y >= n || grid[x][y] != '1') {
                            continue;
                        }
                        
                        grid[x][y] = '0';
                        
                        for (auto& dir : directions) {
                            stk.push({x + dir.first, y + dir.second});
                        }
                    }
                }
            }
        }
        
        return islands;
    }
};
```

## 三种方法对比

| 方法 | 数据结构 | 空间复杂度 | 优势 | 劣势 | 适用场景 |
|------|---------|-----------|------|------|---------|
| DFS递归 | 递归栈 | O(m×n) | 代码极简，易于理解 | 可能栈溢出 | 网格规模适中，代码简洁优先 |
| BFS队列 | 队列 | O(min(m,n)) | 避免栈溢出，内存可控 | 需要队列管理 | 大规模网格，需要最短路径 |
| DFS迭代 | 显式栈 | O(m×n) | 避免递归溢出 | 代码稍复杂 | 需要DFS特性但避免递归 |

## 核心代码模板

**DFS递归模板（网格遍历）：**
```cpp
void dfs(vector<vector<char>>& grid, int i, int j) {
    int m = grid.size(), n = grid[0].size();
    
    // 终止条件
    if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] != '1') {
        return;
    }
    
    // 标记访问
    grid[i][j] = '0';
    
    // 递归探索（上下左右）
    dfs(grid, i - 1, j);
    dfs(grid, i + 1, j);
    dfs(grid, i, j - 1);
    dfs(grid, i, j + 1);
}
```

**BFS队列模板（网格遍历）：**
```cpp
void bfs(vector<vector<char>>& grid, int startI, int startJ) {
    int m = grid.size(), n = grid[0].size();
    vector<pair<int, int>> directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
    
    queue<pair<int, int>> q;
    q.push({startI, startJ});
    grid[startI][startJ] = '0'; // 入队即标记
    
    while (!q.empty()) {
        auto [x, y] = q.front();
        q.pop();
        
        for (auto& dir : directions) {
            int nx = x + dir.first, ny = y + dir.second;
            if (nx >= 0 && nx < m && ny >= 0 && ny < n && grid[nx][ny] == '1') {
                q.push({nx, ny});
                grid[nx][ny] = '0';
            }
        }
    }
}
```

## visited数组优化技巧

### 1. 原地修改法（推荐）
```cpp
grid[i][j] = '0'; // 空间O(1)，代码简洁
```
- 优势：无需额外空间，代码简洁
- 注意：会破坏原始数据

### 2. 独立布尔数组
```cpp
vector<vector<bool>> visited(m, vector<bool>(n, false));
visited[i][j] = true;
```
- 适用：需要保留原始网格数据

### 3. 时间戳优化法（高级）
```cpp
vector<vector<int>> visited(m, vector<int>(n, -1));
int timestamp = 0;

void dfs(int i, int j) {
    if (visited[i][j] == timestamp) return;
    visited[i][j] = timestamp;
    // ...
}
```
- 优势：避免O(m×n)重置成本，适合多次遍历
