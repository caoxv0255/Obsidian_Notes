# 课程：{{课程名称}}

简短描述：课程简介和目标。

## 结构说明

- `01_LectureNotes/`：课堂笔记（按章节编号）。
- `02_Problems/`：练习题与解答。每题单独 Markdown 文件，使用题目模板。
- `03_Labs/`：实验与动手练习（仅包含实验说明与代码片段，不包含可执行文件）。
- `04_Slides/`：讲义或幻灯片链接/摘要。
- `05_Refs/`：参考资料与教材（指向 `电子书附件/` 的条目）。

## 初始章节目录（示例）

1. 课程概览
2. 第1章：示例章节 — [01_LectureNotes/01_第1章.md](01_LectureNotes/01_第1章.md)
3. 第2章：示例章节 — [01_LectureNotes/02_第2章.md](01_LectureNotes/02_第2章.md)
4. 练习：第1章习题 — [02_Problems/01_第1章习题.md](02_Problems/01_第1章习题.md)
5. 实验：第1章实验 — [03_Labs/01_第1章实验.md](03_Labs/01_第1章实验.md)

## 使用说明
复制本文件到课程根目录并替换 `{{课程名称}}`，然后创建对应子目录与笔记文件，使用相应模板填充内容。
